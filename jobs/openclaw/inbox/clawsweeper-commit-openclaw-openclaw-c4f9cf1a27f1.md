## Summary

Found one low-severity reliability regression in the Slack preview edit recovery path. The edit payload is now truncated, but readback verification still expects the untruncated fallback text.

## Findings

### Low: Slack preview edit readback compares against the pre-truncation fallback

- Kind: reliability
- File: `extensions/slack/src/monitor/message-handler/preview-finalize.ts`
- Line: 22
- Evidence: `editSlackMessage` now sends `truncateSlackText(buildSlackBlocksFallbackText(blocks), SLACK_TEXT_LIMIT)` for blocks-only edits at `extensions/slack/src/actions.ts:241`. The preview recovery helper still builds expected readback text with raw `buildSlackBlocksFallbackText(params.blocks)` at `extensions/slack/src/monitor/message-handler/preview-finalize.ts:22`. A read-only probe with the same 3x3000-char context block pattern produced `expectedLength=9002`, `actualLength=8000`, and `didSlackPreviewEditApplyAfterError(...)=false` even when readback contained the exact truncated text plus matching blocks.
- Impact: if a blocks-only final preview edit with long fallback text is applied by Slack but the Web API call throws afterward, the readback guard fails to recognize success. `deliverFinalizableDraftPreview` then falls back to normal delivery, so users can receive a duplicate final reply after the preview message was already updated.
- Suggested fix: make `buildExpectedSlackEditText` mirror `editSlackMessage` for blocks-only fallback text, ideally by sharing one helper that computes the exact Slack edit `text` payload, including `truncateSlackText(..., SLACK_TEXT_LIMIT)`.
- Confidence: high

- Do we have a high-confidence way to reproduce the issue? yes. Use a fake readback message whose `text` is the 8000-char truncated fallback generated by `editSlackMessage` and whose `blocks` match; `didSlackPreviewEditApplyAfterError` returns `false`.
- Is this the best way to solve the issue? yes. The recovery comparator should match the actual payload sent to Slack; sharing the edit-text helper avoids this class of drift.
