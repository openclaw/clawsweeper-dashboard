## Summary

Keep open: this is a fresh product feature request for stronger WhatsApp backup and retention semantics. Current main already has encrypted Git backup for the wacrawl archive, but the code still mirrors the current WhatsApp Desktop-derived archive and does not clearly provide an append-only alternative to WhatsApp iCloud/GDrive backups.

## What This Changes

Request to turn wacrawl into a stable WhatsApp backup alternative that can preserve chat history beyond the current local WhatsApp Desktop SQLite archive sync.

## Best Possible Solution

Define a dedicated backup/retention design that keeps current mirror-style sync safe while adding explicit deletion policy, restore behavior, threat-model docs, and tests for WhatsApp Desktop data disappearing upstream.
