Full review comments:

- **[P2] Ignore stale GameBridge manifests before bypassing AX:** `Core/PeekabooAutomationKit/Sources/PeekabooAutomationKit/Services/GameBridge/GameBridgeDetectionService.swift:112-113`
  - body: `readManifest` accepts any decodable file from `~/.firestaff/accessibility.json`, and the hook returns it before the AX path runs. If Firestaff crashes or leaves an old manifest behind, future detections will keep using stale coordinates instead of falling back; add a freshness check and a stale-manifest fallback test.
  - confidence: 0.86
- **[P2] Keep static text out of textFields:** `Core/PeekabooAutomationKit/Sources/PeekabooAutomationKit/Services/GameBridge/GameBridgeDetectionIntegration.swift:63-64`
  - body: Manifest `text` is mapped to `.staticText`, but this switch stores those elements in `DetectedElements.textFields`. Existing grouping only puts true `.textField` elements there, so consumers can mis-treat labels as inputs; group by the mapped `ElementType` or reuse the existing builder semantics.
  - confidence: 0.78
