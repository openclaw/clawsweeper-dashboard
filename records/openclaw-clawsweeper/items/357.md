- **[P1] Preserve the existing login-method fallback:** `src/clawsweeper.ts:15782`
  - body: This gives normal hosted reviews a default `forcedLoginMethod` of `api`, so `runCodex()` never reaches the existing `codexLoginConfig()` fallback that reads `CLAWSWEEPER_CODEX_LOGIN_METHOD`. In that setup `pnpm run codex:local:check` can pass while `pnpm run review:local` still throws `Unable to resolve Windows Codex command: codex`; make the runner use the same extensionless lookup or stop advertising that path.
