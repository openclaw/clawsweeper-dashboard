## Summary

Keep open; current main still renders PR reviews in the compact durable-comment style and does not implement the requested contributor/maintainer sections, evidence-status split, zero-finding wording, or expanded snapshot coverage. ## What This Changes

The issue requests reorganizing PR review comments into contributor action items, maintainer review, review findings, evidence/proof status, and collapsed audit details while preserving existing markers and findings.

## Best Possible Solution

Adopt a maintainer-approved two-audience PR comment renderer that reuses the existing typed review data, preserves durable markers and inline findings, and adds focused snapshot coverage for low-risk, findings, merge-risk, maintainer-only decision, and missing-proof cases.
