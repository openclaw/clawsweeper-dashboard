  - body: A standalone `--local-range` run still passes `preserveCodexAuth: localOnly`, so unless the user also adds `--local-only`, `runCodex` uses hosted auth defaults and strips local Codex credentials. That breaks the advertised pre-submission local workflow for users following the local review setup.
  - confidence: 0.87

  - body: `git diff --name-status` emits rename/copy rows with both old and new paths, but this parser keeps everything after the first tab as one filename. The follow-up `git diff -- <old\tnew>` can produce an empty patch, so local-range reviews can miss renamed or copied files.
  - confidence: 0.92
