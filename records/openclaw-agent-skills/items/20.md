Full review comments:

- **[P1] Preserve OpenCode's .env read denies:** `skills/autoreview/scripts/autoreview:112`
  - body: OpenCode's default read permission explicitly denies `.env` and `.env.*`, but this config replaces `read` with a blanket allow.   - confidence: 0.9
- **[P2] Send the review bundle off the argv path:** `skills/autoreview/scripts/autoreview:684`
  - body: The full autoreview prompt contains the diff and review context, but `build_opencode_cmd` appends it as a command-line argument. `opencode run` supports piped stdin, and argv can expose the bundle in process listings or fail on large PRs due command-line length limits; feed it through stdin or a protected temp file instead.
  - confidence: 0.86
