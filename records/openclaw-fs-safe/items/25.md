The issue asks for the test suite to be deterministic under restrictive umask settings when asserting POSIX permission modes.

## Best Possible Solution

Make the affected test fixture explicitly establish the intended POSIX mode before asserting preservation, while leaving production mode-preservation semantics unchanged.

## Reproduction Assessment

## Solution Assessment
