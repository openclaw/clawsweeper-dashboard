## Best Possible Solution

Land the narrow typed-sentinel and bounded async retry once redacted real behavior proof confirms atomic rename races recover without weakening non-file or symlink rejection.

## Reproduction Assessment

Yes, the proposed `FsSafeError("path-mismatch")` sentinel plus bounded retry only in async JSON readers is a narrow maintainable fix; merge should wait for real behavior proof.
