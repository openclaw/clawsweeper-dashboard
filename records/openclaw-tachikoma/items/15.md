The PR changes `GoogleProvider.convertTool` to filter `required` to declared properties, warn on dropped orphan entries, and omit empty `required` before sending Gemini schemas.

## Best Possible Solution

Land the Google provider sanitization, ideally with a focused serialization regression test, while keeping upstream schema-conversion fixes separate.

## Solution Assessment
