## What This Changes

The branch changes `GoogleProvider.convertTool` to filter `required` names to declared properties, warn on dropped names, and omit empty `required` before serializing Gemini tool schemas.

## Best Possible Solution

Land the Google provider sanitization with focused request-serialization regression coverage so malformed external tool schemas cannot make Gemini requests fail.
