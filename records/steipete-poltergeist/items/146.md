## Summary

## Best Possible Solution

Broaden argv normalization to recognize the Bun virtual script basenames produced by both local and Homebrew release builds, then add regression coverage and a native smoke that exercises target execution.

## Solution Assessment

No, not as submitted; the helper is the right narrow area, but it needs to cover the Homebrew arch-suffixed slice names or the release workflow needs an equivalent target-execution proof.
