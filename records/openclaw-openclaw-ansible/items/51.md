  - confidence: 0.88

  - body: The RedHat section says the new Quadlet gateway is available on `127.0.0.1:18789`, and the Quadlet publishes that host port, but this shared access section still tells every user to tunnel and browse port 3000. Fedora/RHEL users following it will tunnel the wrong port; split the Docker and Podman access examples or use the runtime variables.
  - confidence: 0.84
- **[P2] Remove host CLI claims from the shared README path:** `README.md:81`
  - body: The Podman path skips the release/development host CLI install and the role explicitly says there is no host `openclaw` binary, but the shared README install summary still says the host CLI is installed and later shows host CLI commands without a Debian/Ubuntu-only boundary.
