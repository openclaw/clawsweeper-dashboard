## Best Possible Solution

## Reproduction Assessment

Yes, from source inspection: current main returns `All systems operational` for `pageStatus: "UP"` with no incidents and a maintenance entry. The PR regression is also source-reproducible because the public API exposes `NOTSTARTEDYET` maintenance statuses and the client does not filter them.

## Solution Assessment

No, not as written. The fix should distinguish scheduled maintenance notices from maintenance that is actually underway instead of treating every maintenance entry as non-operational.
