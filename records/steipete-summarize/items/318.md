## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Yes. Delegating file URL conversion to Node is the narrowest maintainable fix for the parsing contract; the only remaining decision is whether the stricter invalid-URL behavior is acceptable for existing users.
