## Best Possible Solution

## Reproduction Assessment

Yes. Source inspection shows `parseOnOffFlags` uses `Flags().Changed("on")`, so a command such as `groups announce-only --jid ... --on=false` selects the on branch and returns true.

## Solution Assessment
