## What This Changes

The issue requests preserving nested or indented Markdown list structure inside Google Docs table cells for both markdown import and `docs cell-update`.

## Best Possible Solution

Reuse the existing leading-tab plus `CreateParagraphBullets` strategy for table-cell Markdown content, with focused tests for both markdown-table import and `docs cell-update`, followed by live Google Docs proof.

## Solution Assessment
