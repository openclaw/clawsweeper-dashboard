## What This Changes

This PR turns `gog sheets links` into a command group and adds `links set` for single-cell, multi-run, and batch hyperlink writes via Sheets `UpdateCells`.

## Best Possible Solution

Make `links set` replace hyperlink state deterministically, including clearing stale whole-cell link formatting, then verify with a redacted live write/read round-trip.

## Reproduction Assessment

Yes, from source for the review finding: current `links get` reads whole-cell links, while the PR update mask does not clear that field. I did not run a live Sheets round-trip in this read-only review.

## Solution Assessment

No; the chosen `UpdateCells` approach is appropriate, but the request should also clear stale whole-cell hyperlink formatting before this is the best implementation.
