## What This Changes

Add a `--since=<RFC3339>` server-side modified-time filter to `gog drive comments list` and `gog docs comments list`.

## Best Possible Solution

Add a shared modified-time filter for both comment-list commands while preserving existing pagination, resolved-comment filtering, JSON shape, and generated command docs.

## Reproduction Assessment

Not applicable as a bug reproduction: this is a feature request for a new CLI flag. Source inspection confirms the current command structs and shared API call do not expose the requested filter.

## Solution Assessment

Yes, the proposed narrow `--since` flag wired to Drive `comments.list` is the maintainable path. It should use strict RFC3339-with-timezone validation rather than the existing flexible tracking parser that accepts local or date-only inputs.
