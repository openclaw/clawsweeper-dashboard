## Summary

Keep open: current main still has only partial data-validation copy support, with no first-class `sheets validation get/set/clear` surface and no table-managed dropdown round-trip behavior.

## What This Changes

Request adds Sheets data-validation get/set/clear commands and makes validation copy/restore work for validation rules, including table-managed dropdowns.

## Best Possible Solution

Keep this open and design a focused `gog sheets validation` command family that dumps effective rules, sets/clears cell rules through Sheets batchUpdate, and explicitly defines table-column dropdown behavior with live provider proof.

## Solution Assessment

Unclear until maintainers settle the command shape and table-dropdown semantics. A first-class validation subcommand looks like the narrow maintainable direction, but the effective-rule/table behavior should be approved and live-tested.
