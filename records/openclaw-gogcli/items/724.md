The patch is focused and mechanically sound, and the exact-head live Google Docs readback now provides sufficient after-fix proof. This PR should remain open for maintainer acceptance of the intentional formatting-contract change across every incremental workflow using the shared renderer.

## What This Changes

The PR changes locally rendered fenced Markdown blocks from Courier New with default text color to Roboto Mono with dark-green text, preserves paragraph shading, and updates formatter, write, and find-replace tests.

## Best Possible Solution

Adopt Roboto Mono and `#188038` as the documented shared style for locally rendered fenced blocks, while treating broader convergence with the Drive-import renderer as separate follow-up work if additional differences remain.

## Solution Assessment
