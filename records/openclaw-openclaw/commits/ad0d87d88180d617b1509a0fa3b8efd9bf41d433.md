### Low: CLI channel-options test now reuses stale startup metadata

- Kind: regression
- File: `src/cli/startup-metadata.ts`
- Line: 16
- Evidence: `readCliStartupMetadata()` now caches parsed metadata by path. `src/cli/channel-options.test.ts` resets only `channel-options`’ local `precomputedChannelOptions`, not the new shared `startupMetadataByPath` cache. - Confidence: high
