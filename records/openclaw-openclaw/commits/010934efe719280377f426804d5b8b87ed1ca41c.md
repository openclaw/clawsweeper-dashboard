# Commit 010934e

## Summary

Found one low-severity test portability regression. The new exact remediation snapshot is not stable when Node is installed through Homebrew, because production formatting conditionally adds a cert bundle path based on `process.execPath`.

## Findings

### Low: Exact TLS remediation snapshot fails under Homebrew Node

- Kind: compatibility
- File: `src/commands/oauth-tls-preflight.test.ts`
- Line: 67
- Evidence: The test stubs `HOMEBREW_PREFIX` to `""`, then expects an exact string without the optional cert-bundle line. Current implementation still derives a Homebrew prefix first from `process.execPath` when it contains `/Cellar/`, and then appends `- Verify cert bundle exists: <prefix>/etc/openssl@3/cert.pem` in `src/plugins/provider-openai-codex-oauth-tls.ts:61` and `src/plugins/provider-openai-codex-oauth-tls.ts:142`. A focused run passes on the hosted Linux Node path, but fails deterministically when `process.execPath` is simulated as a Homebrew-style `.../Cellar/node/.../bin/node`.
- Impact: Developers or CI lanes running the commands tests with Homebrew-installed Node will fail this test even though the production output is valid. This makes the new test platform-dependent.
- Suggested fix: Avoid asserting the whole string for the environment-dependent formatter output, or make the formatter test control the exec path through an injected resolver. At minimum, include/allow the optional cert-bundle line when `process.execPath` resolves to Homebrew.
- Confidence: high

## Details

- Do we have a high-confidence way to reproduce the issue? yes. Run `pnpm test src/commands/oauth-tls-preflight.test.ts -- --reporter=verbose` with `process.execPath` patched to a Homebrew-style path containing `/Cellar/`; the final assertion fails because the received output contains the extra cert-bundle verification line.
- Is this the best way to solve the issue? yes. The production behavior is intentionally environment-sensitive, so the test should either assert stable fragments or inject/control every environment input used by the formatter.
