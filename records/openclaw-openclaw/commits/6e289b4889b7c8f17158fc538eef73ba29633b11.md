# Commit 6e289b4

## Summary

Found one user-visible cost reporting regression in `/usage full`: the changed footer path now disables Gateway pricing-cache lookup, so models that previously got prices from the background OpenRouter/LiteLLM pricing bootstrap lose the estimated cost suffix unless they also have an exact local model cost entry.

## Findings

### Medium: `/usage full` no longer uses cached Gateway pricing

- Kind: regression
- File: `src/auto-reply/reply/agent-runner.ts`
- Line: 1867
- Evidence: The new call to `resolveModelCostConfig` passes `allowPluginNormalization: false`. In `src/utils/usage-format.ts:329`, that option returns `undefined` before the normal fallback to `getCachedGatewayModelPricing(params)` at `src/utils/usage-format.ts:348`. A focused probe seeded Gateway pricing for `anthropic/claude-sonnet-4-6`; the default resolver produced `Usage: ... · est $0.04`, while the new raw-only resolver produced the same footer without `est $`.
- Impact: Users who enabled `/usage full` and rely on the Gateway pricing bootstrap for configured models without explicit `models.providers.*.models[].cost` entries will still see tokens, but no estimated dollar cost. That regresses the existing full usage footer behavior for API-key providers with cached remote pricing.
- Suggested fix: Keep the new `responseUsageMode === "full"` gate, but do not pass `allowPluginNormalization: false` on the response footer path, or otherwise allow the Gateway pricing-cache fallback while avoiding plugin-heavy normalization if that was the concern.
- Confidence: high

## Details

- Do we have a high-confidence way to reproduce the issue? yes. Seed `setGatewayModelPricingForTest` for a model with no explicit local config, then compare `resolveModelCostConfig({ provider, model, config })` with the new `allowPluginNormalization: false` call; only the default path returns pricing and formats `est $0.04`.
- Is this the best way to solve the issue? yes. The intended behavior change is auth-mode gating, not removal of the established Gateway pricing fallback for `/usage full`.
