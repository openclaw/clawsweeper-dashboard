- Impact: The changed diagnostics plugin cannot pass the npm package preview/release lane, so the OTLP hardening in this commit is at risk of being unpublishable for the external `@openclaw/diagnostics-otel` package.
- Suggested fix: Regenerate/fix `extensions/diagnostics-otel/npm-shrinkwrap.json` so `npm ci --omit=dev --omit=peer --legacy-peer-deps --ignore-scripts` succeeds for the plugin package, then rerun `bash scripts/plugin-npm-publish.sh --pack-dry-run extensions/diagnostics-otel`.
