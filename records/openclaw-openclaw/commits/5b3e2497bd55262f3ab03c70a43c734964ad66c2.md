# Commit 5b3e249

## Summary

Found one low-severity regression in timeout classification for the new phase-specific cron timeout errors.

## Findings

### Low: Phase-specific cron timeouts are recorded as generic task failures

- Kind: regression
- File: `src/cron/service/timer.ts`
- Line: 389
- Evidence: This commit adds timeout errors such as `cron: isolated agent setup timed out before runner start` and `cron: isolated agent run stalled before first model call`, but task completion still marks a cron task as `timed_out` only when `normalizeCronRunErrorText(...) === "cron: job execution timed out"`. The same exact-string check remains in manual cron task completion at `src/cron/service/ops.ts:589`, and task recovery from cron run logs still only recognizes the generic string at `src/tasks/task-registry.maintenance.ts:317`.
- Impact: The cron run itself still records `status: error`, but background task records for the new setup/pre-model timeout paths are labeled `failed` instead of `timed_out`. That changes user-visible task messages and task recovery/status summaries for the exact timeout cases this commit introduces.
- Suggested fix: Add a shared `isCronTimeoutError()` helper that recognizes the generic timeout plus the new setup/pre-model timeout strings, including `Error: ...` wrappers, and use it in timer task completion, manual cron task completion, and task-registry maintenance recovery.
- Confidence: high

## Details

- Do we have a high-confidence way to reproduce the issue? yes. The focused timer tests confirm the new phase-specific timeout paths run, and a direct normalizer check showed `cron: isolated agent run stalled before first model call (last phase: context-engine)` normalizes to itself and does not equal the generic timeout string used for `timed_out` classification.
- Is this the best way to solve the issue? yes. The bug is not in timeout firing; it is in duplicated exact-string timeout classification. A shared predicate keeps the new diagnostic text while preserving existing `timed_out` semantics.
