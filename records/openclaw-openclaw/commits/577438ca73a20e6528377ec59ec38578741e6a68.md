- Kind: test_gap

- Line: 1
- Evidence: The parent file had `allows direct and channel sessions in the default qmd scope`, asserting `agent:main:discord:channel:chan-123` is allowed. This commit replaces the whole source test with `import "../../../packages/memory-host-sdk/src/host/backend-config.test.js";`, but the package test does not contain that assertion. If direct-only is intended, update `docs/concepts/memory-qmd.md` and keep/adjust a package test that locks the direct-only default.
