# Commit 7729e6c

## Summary

Found one actionable supply-chain hardening regression in the ClawHub trust predicate. The focused touched-surface tests pass, but they do not cover malformed ClawHub metadata.

## Findings

### Medium: Non-boolean `isOfficial` can enable the trusted ClawHub install bypass

- Kind: supply_chain
- File: `src/plugins/clawhub.ts`
- Line: 168
- Evidence: `isTrustedSourceLinkedOfficialPackage` changed from `pkg.isOfficial === true` to `pkg.isOfficial`. ClawHub package detail is external JSON cast without runtime validation in `src/infra/clawhub.ts:463`, so a malformed truthy value such as `"false"` or `"0"` would satisfy this branch when the other metadata fields match. That result is passed to `installPluginFromArchive` as `trustedSourceLinkedOfficialInstall` at `src/plugins/clawhub.ts:959`, and the install scanner treats any truthy value as permission to skip blocking critical dangerous-code findings at `src/plugins/install-security-scan.runtime.ts:571`.
- Impact: A malformed or compromised ClawHub response could mark a package as trusted source-linked official without a strict boolean `true`, allowing critical install-scan findings to be downgraded to warnings.
- Suggested fix: Restore `pkg.isOfficial === true` and add a regression test with `isOfficial: "false" as unknown as boolean` proving `trustedSourceLinkedOfficialInstall` is not enabled. Longer term, validate ClawHub API responses at the boundary instead of relying on TypeScript casts.
- Confidence: medium

## Details

- Do we have a high-confidence way to reproduce the issue? Yes: mock `fetchClawHubPackageDetail` with the existing source-linked official metadata shape but set `isOfficial` to a truthy non-boolean string, then assert `installPluginFromArchive` receives a truthy trusted flag and critical scan blocking is bypassed.
- Is this the best way to solve the issue? Yes for the immediate regression: the trust decision should require the external metadata field to be exactly boolean `true`. Runtime schema validation would be a stronger follow-up.
