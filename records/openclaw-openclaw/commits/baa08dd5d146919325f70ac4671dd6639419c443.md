# Commit baa08dd5d

## Summary

One low-severity compatibility bug remains on current `main`: the new Playwright cache detection does not fully mirror Playwright’s own browser registry path resolution.

## Findings

### Low: Playwright cache detection skips valid registry locations

- Kind: compatibility
- File: `extensions/browser/src/browser/chrome.executables.ts`
- Line: 507
- Evidence: `getPlaywrightBrowserCachePaths()` only checks a non-`0` `PLAYWRIGHT_BROWSERS_PATH` value and `~/.cache/ms-playwright`. The pinned `playwright-core@1.59.1` source resolves the default Linux cache to `$XDG_CACHE_HOME/ms-playwright` when `XDG_CACHE_HOME` is set, and resolves `PLAYWRIGHT_BROWSERS_PATH=0` to the package-local `.local-browsers` directory. Focused in-memory smokes with only those Playwright-valid locations populated both returned `actual: null`.
- Impact: Linux hosts that installed Chromium through Playwright using `XDG_CACHE_HOME` or `PLAYWRIGHT_BROWSERS_PATH=0` can still be reported as having no Chromium executable, so managed browser launch/status/doctor flows can fail even though Playwright’s installed browser is present.
- Suggested fix: Mirror Playwright’s registry resolution: include `$XDG_CACHE_HOME/ms-playwright` in the default Linux candidates, handle `PLAYWRIGHT_BROWSERS_PATH=0` as the package-local Playwright browser directory, and resolve relative configured paths the same way Playwright does. Keep the current `chrome-linux64` and `chrome-linux` executable directory checks.
- Confidence: high

## Details

- Do we have a high-confidence way to reproduce the issue? yes. I ran focused `pnpm exec tsx -e` smokes that monkeypatched `fs` in memory: one exposed only an executable under an XDG cache `ms-playwright` path, and one exposed only an executable under `node_modules/playwright-core/.local-browsers`; both returned `null`.
- Is this the best way to solve the issue? yes. The fix should align OpenClaw’s candidate path construction with Playwright’s own registry path contract instead of adding one-off cache guesses.
