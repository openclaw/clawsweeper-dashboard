## Summary

Found one concrete issue: the commit adds a Swift test for filler-prefixed trigger-only input, but the new helper still rejects that exact input when called with the tester’s trim function.

## Findings

### Medium: Trigger-only fallback test case cannot pass for `hey openclaw`

- Kind: bug
- File: `apps/macos/Sources/OpenClaw/VoiceWakeTextUtils.swift`
- Line: 162
- Evidence: `triggerOnlyFallbackMatch` delegates to `VoiceWakeTextUtils.isTriggerOnly`, which requires `trimWake(transcript, triggers).isEmpty`. The tester passes `WakeWordGate.stripWake` at `apps/macos/Sources/OpenClaw/VoiceWakeTester.swift:237` and `apps/macos/Sources/OpenClaw/VoiceWakeTester.swift:419`. `stripWake("hey openclaw", ["openclaw"])` removes only `openclaw`, leaving `hey`, so the new test at `apps/macos/Tests/OpenClawIPCTests/VoiceWakeTesterTests.swift:35` gets `nil`, not a match.
- Impact: The newly added `trigger only fallback accepts bare test trigger` test should fail on a macOS Swift test lane. It also means the tester fallback is still not aligned with the runtime trigger-only helper for filler-prefixed phrases like `hey openclaw`; runtime trims after the trigger via `VoiceWakeRuntime.trimmedAfterTrigger`, while the tester uses `WakeWordGate.stripWake`.
- Suggested fix: Use the same “after trigger” trimming semantics for the tester trigger-only fallback, or make `isTriggerOnly` evaluate “nothing after the matched trigger” directly instead of relying on `WakeWordGate.stripWake`.
- Confidence: high

## Details

- Do we have a high-confidence way to reproduce the issue? Yes. The focused Swift test command could not complete on this Linux runner because the macOS package dependency imports `CoreGraphics`, but source inspection plus an inline Swift check confirmed `WakeWordGate.stripWake` leaves `hey` for `hey openclaw`.
- Is this the best way to solve the issue? Yes. The narrow fix is to align tester trimming with runtime trigger-only trimming, preserving the existing rejection of ordinary trailing mentions like `tell me about openclaw`.
