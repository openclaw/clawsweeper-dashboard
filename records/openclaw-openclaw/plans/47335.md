## Summary

Current main still has the source-level compaction-retry delivery race: the runner drains block replies before the retry wait but not after it, while lifecycle code resolves the retry wait before the async block/channel flush settles. The related closed source PR is unmerged, and the open final-payload filtering PR does not replace this retry-specific delivery barrier.

## Plan
