## Summary

Keep open: the browser tool now exposes `evaluate` on current main and in the latest release, but the requested opt-in `force` click flag is still absent from the tool schema, normalized action type, route normalizer, CLI, docs, and Playwright click forwarding path. ## Plan

Fix https://github.com/openclaw/openclaw/issues/60381 only for the remaining browser click force flag. Keep default behavior unchanged (`force` false/omitted), do not make force an automatic retry, and do not rework `evaluate`.
