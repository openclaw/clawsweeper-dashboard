## Summary

Keep open. Current main still rewrites normal skill slash invocations into a model request and only handles `command-dispatch: tool` deterministically, so the requested automatic `SKILL.md` injection remains unimplemented.

## Plan

Implement deterministic `SKILL.md` injection for explicit normal skill slash commands. Limit the change to `/skill <name> [input]` and direct user-invocable skill commands that do not use `command-dispatch: tool`; leave natural-language skill matching and tool-dispatch skills unchanged. Reuse the eligible/visible skill snapshot or command metadata to locate the selected skill safely, inject the selected skill instructions into the agent run context before the user args using an existing transcript-safe mechanism, preserve sender authorization and skill filters, and update docs/changelog if the user-visible behavior changes. Add focused regression coverage for direct skill commands, `/skill` lookup, args preservation, tool-dispatch bypass, and bundle prompt commands.
