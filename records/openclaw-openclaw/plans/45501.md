## Plan

Implement configurable bare session reset prompt support for https://github.com/openclaw/openclaw/issues/45501. Add a typed `session.resetPrompt?: string` root config and a per-agent override shape that can replace the normal non-bootstrap `BARE_SESSION_RESET_PROMPT_BASE` for `/new` and `/reset`; preserve the current built-in prompt when unset. Apply precedence per-agent over root, keep bootstrap-pending and limited-bootstrap safety behavior explicit, and do not implement greeting-model selection, greeting suppression, reset archival, or WebChat acknowledgement rendering from related issues. Wire the resolver through the reply-producing reset path and Gateway agent reset path, update config schemas/types/help/docs, add focused regression tests for root, per-agent, fallback, and bootstrap precedence, and include a user-facing changelog entry if the repair lane owns changelog work.
