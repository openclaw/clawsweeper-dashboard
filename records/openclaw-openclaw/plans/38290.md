## Plan

In `src/gateway/origin-check.ts`, keep configured `allowedOrigins` as the current exact trimmed lowercase strings, and change only incoming parsed-origin normalization so `url.origin === "null"` with a non-empty protocol and host matches as `${url.protocol}//${url.host}`.
