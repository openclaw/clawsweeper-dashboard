## Plan

Implement the fix in the current plugin-owned replay-policy path, likely the shared passthrough-Gemini provider replay helper/family and OpenRouter coverage, while preserving existing Gemini thought-signature cleanup and non-Mistral OpenRouter behavior. Add focused regression coverage for OpenRouter Mistral/Codestral positive cases and non-Mistral negative cases, keep direct Mistral strict9 tests green, and add a changelog fix entry because this is user-visible provider behavior.
