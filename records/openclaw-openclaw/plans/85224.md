## Plan

Fix https://github.com/openclaw/openclaw/issues/85224. On macOS, Darwin with only `XPC_SERVICE_NAME` currently makes `detectRespawnSupervisor(...)` return `launchd`, and `respawnGatewayProcessForUpdate` / `restartGatewayProcessWithFreshPid` then exit as supervised; this can strand GUI-spawned gateways when `ai.openclaw.gateway` is not loaded.
