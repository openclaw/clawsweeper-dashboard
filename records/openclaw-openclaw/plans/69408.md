## Plan

Fix https://github.com/openclaw/openclaw/issues/69408 by threading typed isolated-cron schedule/run metadata from the cron executor into the embedded agent tool-hook context, then demote only non-critical bounded isolated-cron loop diagnostics from warning to info. Preserve critical blocks, unknown-tool protection, generic no-progress critical behavior, run-id isolation, and default loop-detection thresholds.
