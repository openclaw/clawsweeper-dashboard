## Plan

Fix the remaining `OPENCLAW_HOME` path-contract gaps for https://github.com/openclaw/openclaw/issues/54014. Scope the PR to internal OpenClaw defaults: make default git/update checkout and installer bootstrap workspace/config checks derive from the effective OpenClaw home when `OPENCLAW_GIT_DIR`, `OPENCLAW_STATE_DIR`, or `OPENCLAW_CONFIG_PATH` are not explicit; add a narrow guard so `OPENCLAW_HOME` values whose basename is an OpenClaw state dir do not become `.openclaw/.openclaw`; update docs to state intentional OS-home exceptions.
