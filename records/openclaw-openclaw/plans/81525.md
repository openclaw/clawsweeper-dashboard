Keep open: current main and the latest shipped release still accept configured image capability metadata before any catalog text-only check, and failed media-understanding summaries still drop the attempted provider/model detail and truncate the capability diagnostic.

## Plan

Fix https://github.com/openclaw/openclaw/issues/81525 in one focused PR. Improve failed media-understanding decision summaries so all-failed attempts still show the attempted provider/model and keep the useful tail of `Model does not support images: ...
