## Plan

Implement https://github.com/openclaw/openclaw/issues/70443 as a narrow Docker setup feature. Update `src/docker-setup.e2e.test.ts` to pass script args and cover successful preloaded-image use, missing main image failure, and missing optional sandbox warning. Update `docs/install/docker.md` with the offline rerun flow and add a concise `CHANGELOG.md` entry because this is user-facing setup behavior.
