## Summary

Keep open: current main has partial watcher hardening, but the source-visible fallback gap remains. If chokidar stops delivering events after the dirty flag has been cleared, the default interval is still disabled and even a configured interval calls `sync()` without marking memory dirty, so `syncMemoryFiles()` is skipped.

## Plan

Fix https://github.com/openclaw/openclaw/issues/40088 on current main.
