## Plan

Implement the `/usage` footer provider-quota enhancement for https://github.com/openclaw/openclaw/issues/7379. Current main builds the footer in `src/auto-reply/reply/agent-runner.ts` with `formatResponseUsageLine`, which only emits token/cache/cost/session data; `/status` already uses `loadProviderUsageSummary` plus `formatUsageWindowSummary`, and the GitHub Copilot plugin already exposes `Premium` and `Chat` windows.
