## Plan

Implement a focused OpenClaw logger enhancement for https://github.com/openclaw/openclaw/issues/32438. Add a human-readable companion log file derived from the resolved active `logging.file` path, for example `openclaw.log` -> `openclaw-formatted.log`, while preserving the existing JSONL file path, JSONL format, `logs.tail`, CLI, Control UI, and scripts. Add focused regression coverage for the writer, derived path, child logger/subsystem formatting, redaction parity, JSONL consumer compatibility, and size/date lifecycle. Update `docs/logging.md`, `docs/gateway/logging.md`, and the configuration reference if needed; add a changelog entry because this is user-visible.
