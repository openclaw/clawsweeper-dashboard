Add a derived human-readable sidecar file next to the active `logging.file` path, inserting `-formatted` before the extension, and write one redacted plain-text line per JSONL log event in the requested `YYYY-MM-DD HH:mm:ss [level] [subsystem] message` shape. Update docs at `docs/logging.md` and `docs/gateway/configuration-reference.md`, add a user-facing changelog entry if behavior changes, and add regression coverage for redaction, unchanged JSONL writes, derived path naming, rolling-day behavior, and size/rotation behavior.
