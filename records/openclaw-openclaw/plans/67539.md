## Plan

Implement provider-specific TTS prompt hints for https://github.com/openclaw/openclaw/issues/67539. Add a backwards-compatible optional hook on SpeechProviderPlugin, with typed context for the resolved speech provider config, resolved model/voice/target, and modelOverrides policy. Let the Google speech provider own Gemini square-bracket audio-tag guidance and avoid hard-coding provider-specific syntax in speech-core. Do not add a new config option, do not advertise inactive provider syntax, and stop for maintainer review if the scope expands into shared autoEmotion/product policy. Update SDK/API baseline/docs/contract tests as required for a public seam.
