## Plan

Implement the cron account lint feature for https://github.com/openclaw/openclaw/issues/69054. Add a shared helper that reports announce-mode cron jobs missing `delivery.accountId` only when the resolved explicit/prefixed channel has more than one configured account according to the channel plugin account-list contract.
