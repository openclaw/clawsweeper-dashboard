## Plan

Current main treats Xiaomi and custom `xiaomi-coding`/`xiaomimimo.com` OpenAI-completions endpoints as ordinary OpenAI-compatible providers, while Xiaomi’s docs require `reasoning_content` to be passed back on assistant tool-call history when thinking is enabled. Implement a narrow compatibility path that covers bundled `xiaomi` models and custom Xiaomi MiMo endpoints by provider id or baseUrl host, preserves/backfills `reasoning_content` for assistant tool-call replay when thinking is enabled, and strips or avoids it when thinking is off.
