## Summary

Keep open. Current main still has a source-level Telegram typing gap: `thinking` mode is documented to start typing on the first reasoning delta, but `signalReasoningDelta` still returns before any renderable text exists, so long pre-first-output reasoning can leave Telegram without a keepalive. Related broad and transport issues are either separate or point back to this narrower follow-up rather than superseding it.

## Plan
