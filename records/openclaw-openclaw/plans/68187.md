## Plan

Fix https://github.com/openclaw/openclaw/issues/68187. Current main caches bundled MCP SSE sessions and calls `session.client.callTool()` without stale-session recovery; after a downstream SSE MCP server restart, the pinned MCP SDK can surface `Error POSTing to endpoint ...
