Add concise guidance warning that `agents.defaults.memorySearch.experimental.sessionMemory` with `sources: ["memory", "sessions"]` overlaps with the bundled `session-memory` hook because the hook writes session-derived Markdown to `<workspace>/memory/`, which the `memory` source also indexes.
