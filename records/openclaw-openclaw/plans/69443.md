Keep open. Current main and the latest release still choose the Gateway `agent` run session from `sessionKey`, `sessionId`, or the agent default before `to` is treated as delivery metadata, so a canonical `agent:<id>:...` session key supplied only in `to` can still collapse to the main session.

## Plan
