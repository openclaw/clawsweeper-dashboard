## Plan

Implement a durable paired-device rename workflow for https://github.com/openclaw/openclaw/issues/13870. Add an operator-owned label field separate from reconnect-owned `displayName`, add additive Gateway protocol/schema/handler support with the existing paired-device self/admin authorization boundaries, expose `openclaw devices rename --device <id> --name <label>`, add a Control UI rename action, and document display precedence. Keep this separate from broader metadata display work in https://github.com/openclaw/openclaw/issues/23916 and timestamp-only work in https://github.com/openclaw/openclaw/pull/55973.
