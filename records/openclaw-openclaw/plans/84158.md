## Summary

Keep open: current main and the latest shipped tag still feed timeout recovery only from backlog-blocked lanes, so a lone active `.json.processing` claim can miss recovery when no later same-lane update exists. The closed follow-up PR had the right narrow shape but was closed unmerged because live Telegram proof was not available.

## Plan
