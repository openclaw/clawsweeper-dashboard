## Summary

Keep open: current main still has a real Slack-owned gap for interactive reply follow-up liveness. Ordinary Slack message turns can set Slack assistant thread status, and interactive reply clicks now enqueue/wake a thread-aware heartbeat, but the Slack channel plugin does not expose the heartbeat typing hooks that heartbeat runs use for channel-native busy indicators.
