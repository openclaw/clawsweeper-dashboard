## Summary

Keep open. Current main and the latest stable release have reduced the reported Anthropic-specific risk by replacing the Anthropic SDK stream iterator with OpenClaw's own parser that drains SSE bodies to EOF, but the shared managed-response wrapper still has no cleanup path for an abandoned body that reaches neither EOF nor explicit cancel.

## Plan

Preserve the current Anthropic custom SSE parser that drains to EOF. Add focused regression coverage in src/agents/provider-transport-fetch.test.ts for: natural EOF releases once, explicit cancel releases once, read error releases once, and an abandoned/GC or deterministic idle cleanup path releases once.
