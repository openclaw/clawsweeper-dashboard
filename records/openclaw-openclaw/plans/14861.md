## Plan

Implement a redacted Gateway startup outcome summary for https://github.com/openclaw/openclaw/issues/14861. Add focused regression coverage for the Gmail missing-account path and summary output, update docs/gateway/logging.md if operator-facing output changes, and add a CHANGELOG.md entry because startup logs are user-visible.
