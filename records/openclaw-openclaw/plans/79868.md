## Summary

Keep open. Current main still has a source-level path where an auto-sourced fallback model can be reused as the next embedded run's first candidate, so the primary is bypassed without the failed-candidate fallback log that operators expect.

## Plan

Do not add a config option and do not log raw provider error bodies, request IDs, secrets, or credentials.
