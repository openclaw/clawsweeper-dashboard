## Plan

Fix the WebUI `/new` dashboard-session auto-label gap from https://github.com/openclaw/openclaw/issues/76848. Add a default friendly label for generated dashboard sessions when no explicit label is supplied, preserve explicit labels, and expose stored labels through local `openclaw sessions --json` plus the human table while keeping the raw session key available for disambiguation.
