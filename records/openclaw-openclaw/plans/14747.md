## Plan

Implement the narrow lane-wait diagnostics config request from https://github.com/openclaw/openclaw/issues/14747. Add a documented positive-integer `diagnostics.laneWaitWarnMs` setting that provides the default `warnAfterMs` for future queue entries when callers do not pass an explicit override; keep the current 2,000 ms fallback when unset and preserve existing explicit overrides such as manual cron's `warnAfterMs: 5_000`.
