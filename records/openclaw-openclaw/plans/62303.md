## Summary

Keep open. Current main still uses the session-store `systemSent` flag as the Telegram DM-topic first-turn gate before `generateTopicLabel` and `editForumTopic`, so a newly-created topic whose session entry is already marked `systemSent: true` can skip the rename path and keep the fallback title; the later Codex label-generator fix does not remove this dispatcher gate.

## Plan
