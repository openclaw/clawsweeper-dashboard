Keep open. Current main and the latest stable release still gate Telegram DM-topic auto-renaming on `!entry?.systemSent`, so a topic whose session entry is already marked bootstrapped can skip the label generation and `editForumTopic` path.

## Plan

Fix https://github.com/openclaw/openclaw/issues/62303 in the Telegram plugin.
