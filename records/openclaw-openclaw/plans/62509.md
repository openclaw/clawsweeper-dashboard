## Plan

Fix https://github.com/openclaw/openclaw/issues/62509. In `scripts/install.sh`, add a portable bounded-probe helper for nonessential installer finalization and diagnostic probes so a hung `npm prefix -g`, `npm config get prefix`, or `openclaw daemon status --json` cannot block installation indefinitely.
