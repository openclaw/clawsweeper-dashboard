## Plan

Preserve existing reconnect/group metadata fallback behavior from https://github.com/openclaw/openclaw/pull/73732 and avoid core SDK or loader changes. Add focused regression coverage for socket options, outbound retry lookup, group metadata cache refresh, participant updates, and LID mapping updates.
