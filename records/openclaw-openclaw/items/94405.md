- **[P1] Avoid replaying the whole message after partial rich sends:** `extensions/telegram/src/send.ts:858`
  - body: The new catch wraps the entire rich chunk loop, but that loop records each successful rich chunk before continuing.   - confidence: 0.9

  - confidence: 0.86

  - confidence: 0.8
