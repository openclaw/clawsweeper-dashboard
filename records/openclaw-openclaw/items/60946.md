## Summary

Keep open: current main still only partially satisfies this feature; edit mismatch failures include only bounded current contents, while similar-match, duplicate-location, apply_patch mismatch, and exec command-not-found recovery guidance remain missing.

## What This Changes

Request richer contextual tool error diagnostics with troubleshooting hints and fix suggestions for edit, apply_patch, and exec failures.

## Best Possible Solution

Add a bounded shared diagnostic helper in the core agent tool runtime and wire it into edit mismatch, apply_patch mismatch, and exec exit-127 formatting without changing tool semantics or plugin trust boundaries.
