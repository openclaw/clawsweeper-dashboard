## Summary

Keep open: current main still only partially satisfies the requested richer model-facing tool diagnostics. Edit mismatch errors include bounded current file contents and apply_patch has tolerant matching, but similar-text candidates, duplicate match locations, structured retry guidance, and conservative exec command-not-found suggestions are still missing.

## What This Changes

Request richer tool error responses with contextual troubleshooting hints, similar match suggestions, duplicate match locations, and actionable fix steps.

## Best Possible Solution

Add a shared bounded diagnostic helper in the core agent tool runtime, then wire it into edit mismatch, apply_patch mismatch, and exec exit-127 formatting without changing tool semantics or broadening plugin trust boundaries.

## Solution Assessment

Yes. A bounded core-runtime helper is the narrowest maintainable direction because external plugins cannot currently rewrite these high-trust tool results before the model sees them.
