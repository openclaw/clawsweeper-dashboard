The issue reports high idle CPU on macOS Gateway 2026.5.22, suspected to come from config and skills chokidar watchers using awaitWriteFinish polling.

## Best Possible Solution

Reduce Gateway idle watcher overhead by avoiding unnecessary awaitWriteFinish file-size polling on config and skills watchers while preserving documented hot reload and skills refresh behavior.
