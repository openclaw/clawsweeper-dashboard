## Summary

Keep this PR open: the backpressure direction is plausible, but the branch still has a queued Codex RPC timeout bug, adds undocumented Discord queue-drop controls, lacks after-fix real runtime proof, and is currently conflicting with main.

## What This Changes

The PR adds Discord inbound queue caps, preview/media coalescing, transcript idempotency sidecars, Codex app-server write serialization/timeouts, async session-store reads, and related tests.

## Best Possible Solution

Make queued Codex RPC writes cancellable before they reach stdin, document the Discord queue controls, rebase over current main, and add redacted runtime proof before maintainer merge review.

## Reproduction Assessment

No high-confidence live workload reproduction is present. Source review is enough to reproduce the queued RPC flaw on the PR branch because a request can time out while waiting behind writeChain and still be written later.

## Solution Assessment

No, not in its current form. The overall direction is plausible, but the branch needs cancellable queued writes, public docs for the new Discord controls, a rebase, and real behavior proof before it is mergeable.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.87

Full review comments:

- **[P2] Avoid sending timed-out queued RPCs:** `extensions/codex/src/app-server/client.ts:229-230`
  - body: The timeout starts before the serialized stdin write has actually run. If an earlier write keeps the chain backpressured past the timeout, `rejectPending` removes the id, but this still sends the request later, so the app server can execute work the caller already saw fail.
  - confidence: 0.9
- **[P2] Document the new Discord queue controls:** `src/config/types.discord.ts:411-416`
  - body: The PR adds user-facing `inboundWorker.maxPendingPerSession` and `maxQueuedAgeMs` drop behavior, but the public Discord docs do not describe the new paths, defaults, or when messages are skipped. Users need that contract before this ships.
  - confidence: 0.88
