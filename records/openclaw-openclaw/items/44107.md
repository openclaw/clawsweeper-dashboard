Add a small pre-module bootstrap fallback in `ui/index.html` or the nearest established shell path that detects when `<openclaw-app>` has not mounted after a short timeout and renders plain HTML guidance about disabling browser extensions or using an Incognito/private window. Add a troubleshooting note to `docs/web/control-ui.md` and update `CHANGELOG.md` if the behavior is user-facing. Do not rewrite `chat.history` pagination/budgeting, markdown rendering, or large-session history behavior; current main already handles that half and #46707/#43857 are context only. Prefer existing style, keep the fallback dependency-free, and avoid generated locale edits unless the chosen implementation changes localized UI strings.
