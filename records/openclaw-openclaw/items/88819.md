## Best Possible Solution

Keep the task-backed Workboard flow, but route unassigned UI starts through the Gateway's configured default agent and only synthesize agent-scoped child keys when a card has an explicit agent assignment.

No: task-backed runs are the right owner boundary, but the UI start path must preserve configured default-agent routing for unassigned cards before merge.
