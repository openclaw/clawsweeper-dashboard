## Summary

Keep this PR open, but the current patch is not merge-ready: it treats complete Claude `assistant` records as streaming deltas without live Claude CLI JSONL proof, while the current Claude Agent SDK contract exposes partial assistant output as `stream_event` records.

## What This Changes

The PR adds a Claude CLI JSONL parser fallback that turns top-level `assistant` messages into computed streaming deltas and adds parser tests for that behavior.

## Best Possible Solution

Land a contract-safe fix only after redacted live Claude CLI JSONL proves the ignored record shape, then parse that exact shape and keep final assistant messages out of the delta path.
