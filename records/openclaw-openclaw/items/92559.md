Requested support for resolving session-aware template variables inside provider `extra_body` values at request time.

## Best Possible Solution

Define a narrow provider-extra-param template contract with an explicit allowlist of stable session fields, render it at the per-turn request boundary, and cover cache isolation plus docs before implementation.
