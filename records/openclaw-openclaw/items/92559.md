## What This Changes

The issue requests `extra_body` values to support per-request session/channel template variables such as `{{session.channel}}`.

## Best Possible Solution

Design a narrow request-context templating contract for provider request params, or extend the provider plugin hook with explicit session context, while preserving static `extra_body` values by default.
