Ensure successful QQBot replies, including very short replies, are written to the durable session transcript so WebChat history can show them.

Persist every successful normal QQBot assistant reply through Pi's durable transcript path, or add a narrowly guarded SessionManager fallback only when Pi did not already write the assistant turn.
