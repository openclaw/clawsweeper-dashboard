## Best Possible Solution

Land one runtime-catalog/native-menu fix, likely https://github.com/openclaw/openclaw/pull/94067, that passes discovered reasoning metadata to `/think` menus while preserving provider-owned off-only policies.

## Reproduction Assessment

## Solution Assessment

No. The visible behavior improves, but this PR changes the shared provider-profile fallback too broadly; the safer fix is to feed runtime catalog metadata into native `/think` menus or scope the policy to Ollama with regression coverage.
