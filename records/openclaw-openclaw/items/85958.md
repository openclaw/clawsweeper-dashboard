## What This Changes

This PR removes OpenClaw-owned automatic and fallback compaction for Codex runtime sessions, forwards explicit compaction as pending native Codex starts, and updates related docs, tests, Gateway/CLI/channel bookkeeping, and changelog text.

## Best Possible Solution

Land this only if maintainers explicitly accept Codex-owned compaction as the stable contract; otherwise preserve the old fallback as default and introduce a documented strict mode with upgrade tests.
