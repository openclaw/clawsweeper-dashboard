  - body: This makes `metadata` an arbitrary public `send` parameter and then forwards it to delivery hooks.   - confidence: 0.86

  - body: When async direct send is enabled, this returns after `maybeDeliverMediaGenerationResultDirectly` and never wakes the requester agent. Current OpenClaw policy and mainline behavior route generated-media completions through the requester agent first so visible-reply/message-tool config decides what the user sees; direct delivery is only fallback/recovery, so this can silently bypass that contract.
  - confidence: 0.9
- **[P2] Move Hermes literals out of core delivery code:** `src/agents/tools/media-generate-background-shared.ts:25-26`
  - body: Hardcoding `dev-iox` and `AHC_A8_bot` in the shared media completion path makes a deployment-specific Hermes route part of core behavior. Core should expose a generic hook/plugin seam, while the Hermes topic/bot policy should live in the owner plugin or configuration so other installations are not coupled to this environment.
  - confidence: 0.84
