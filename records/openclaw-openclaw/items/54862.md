## Best Possible Solution

Land a rebased provider-local implementation that injects `allowed_models` only for normalized `openrouter/auto` requests, composes with current OpenRouter stream wrappers, exposes any reusable helper through a public plugin seam or plugin-owned tests, documents the config, and includes redacted live OpenRouter proof.

## Solution Assessment

No. The feature direction is plausible, but this patch is not the best merge path until it is gated to Auto Router requests, rebased over current wrappers, moved through the proper plugin seam or plugin-owned tests, documented, and backed by live proof.
