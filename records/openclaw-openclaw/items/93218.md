- **[P2] Read stream overrides from the command target session:** `extensions/slack/src/monitor/message-handler/dispatch.ts:678-686`
  - body: Native slash commands can use a transport-specific `SessionKey` while `CommandTargetSessionKey` points at the chat session that command handlers mutate. Prefer `CommandTargetSessionKey ?? SessionKey ?? route.sessionKey` and cover native slash routing.
  - confidence: 0.88

  - body: `blockStreaming` is the legacy block-reply toggle, not preview streaming config. When Discord only has `blockStreaming` set, this status path takes the channel-config branch and reports `off` instead of the effective Discord default `progress`, so `/stream status` can mislead operators.
  - confidence: 0.78
