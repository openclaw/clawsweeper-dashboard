- **[P1] Exclude DMs from owner blocking:** `extensions/discord/src/monitor/message-handler.preflight.ts:442`
  - body: The new owner check runs for direct messages, but `bypassOwnerBecauseMentioned` is always false for DMs. If another account has a more specific direct-peer binding for the same sender, this handler drops a legitimate DM even though that other bot will not receive this DM event.
  - confidence: 0.91
- **[P2] Limit owner candidates to active Discord accounts:** `extensions/discord/src/monitor/route-owner.ts:40`
  - body: `resolveDiscordRouteOwner` includes account IDs found only in bindings. A stale or disabled Discord account binding can win ownership and make active accounts drop channel messages for a handler that is not running.
  - confidence: 0.88
- **[P2] Use plugin SDK seams for route owner logic:** `extensions/discord/src/monitor/route-owner.ts:1-8`
  - body: This new Discord production module imports config, routing, and session-key internals via `../../../../src/**` paths. Extension production code should use `openclaw/plugin-sdk/*` or a documented local/public seam so bundled plugins do not depend on private core layout.
  - confidence: 0.86
- **[P2] Require Discord metadata for owner bypass:** `extensions/discord/src/monitor/message-handler.preflight.ts:85-100`
  - body: The bot-name regex treats plain text like `@Bot Name` as a mention even when Discord did not resolve a user mention. That can let a non-owner account bypass the owner guard from incidental text, while the canonical Discord routing issue calls for native mention and reply metadata to drive cross-bot targeting.
  - confidence: 0.78
- **[P2] Rank the current wildcard peer route kind:** `extensions/discord/src/monitor/route-owner.ts:11-20`
  - body: Current main includes `binding.peer.wildcard` in `ResolvedAgentRoute["matchedBy"]`, but the new exhaustive `ROUTE_PRIORITY` map does not include it. After rebase this either fails the type check or leaves wildcard peer ownership without a defined specificity.
  - confidence: 0.82
