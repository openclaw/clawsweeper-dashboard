  - confidence: 0.88
- **[P2] Use plugin SDK seams for route owner logic:** `extensions/discord/src/monitor/route-owner.ts:1-8`
  - body: This new Discord production module imports config, routing, and session-key internals through `../../../../src/**` paths. Extension production code should use `openclaw/plugin-sdk/*` or a documented local/public seam so bundled plugins do not depend on private core layout.
  - confidence: 0.86
- **[P2] Require Discord metadata for owner bypass:** `extensions/discord/src/monitor/message-handler.preflight.ts:85-100`
  - body: The bot-name regex treats plain text like `@Bot Name` as a mention even when Discord did not resolve a user mention.   - confidence: 0.78
