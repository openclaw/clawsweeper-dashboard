Report that node-host exec in YOLO mode can return `SYSTEM_RUN_DENIED` after `exec-approvals.json` contains a persisted socket token.

Land or replace the linked narrow fix so full/off resolution bypasses approval sockets even when a token exists, while preserving `requireSocket` and non-YOLO socket behavior with regression coverage.
