## Summary

## Best Possible Solution

Land a narrow per-agent avatar storage fix that handles the legacy global key deterministically, keeps visible readers scoped to the active agent, and includes redacted real two-agent browser proof.

## Solution Assessment

No. Per-agent keys are the right layer, but this branch is not the best fix until legacy global fallback clear semantics and real two-agent Control UI proof are handled.
