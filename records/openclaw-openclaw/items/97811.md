The PR changes the Nextcloud Talk bot-response preflight non-OK response path to read at most 8 KiB with the shared bounded text reader and adds a cancellation regression test.

## Best Possible Solution

Land the plugin-local bounded-read fix after maintainer review, keeping startup diagnostics useful while preventing hostile or misbehaving Nextcloud endpoints from streaming unbounded error bodies into memory.
