## What This Changes

## Best Possible Solution

Key the cache on the resolved manifest registry or current metadata snapshot identity for omitted-registry calls, add snapshot-replacement coverage, then merge the focused optimization if maintainer review accepts the helper-level proof.

## Reproduction Assessment

## Solution Assessment
