## Summary

Keep open: this PR targets a real current-main ACP background-task terminal delivery bug, but the submitted fallback would expose existing silent control tokens in user-visible completion text and the required changelog entry is missing.

## Best Possible Solution

Keep the fallback-and-cap direction, but strip exact/trailing silent control tokens at the ACP terminal-result boundary, add focused regression coverage for normal, blocked, and silent-token progress, and add the changelog entry before landing against #74070.
