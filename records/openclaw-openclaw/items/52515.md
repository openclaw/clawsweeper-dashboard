  - confidence: 0.93

  - body: Filtering out every legacy service before rendering hints drops the only manual cleanup commands when legacy cleanup is skipped by policy, declined, fails, or is system-scoped. In those cases doctor still reports the legacy service, but the user no longer gets an actionable fallback.
