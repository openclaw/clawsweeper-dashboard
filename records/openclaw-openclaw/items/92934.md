## What This Changes

## Best Possible Solution

Land a fix that applies one consumed-or-fallback acknowledgement invariant to both completion active-wake paths, with regression coverage for direct-failure-to-steer-fallback and redacted real runtime proof.

## Reproduction Assessment

## Solution Assessment

No. The assistant-ack helper is a good fix shape, but applying it only to the direct wake is too narrow; the fallback completion steer needs the same boundary or should avoid active steering.
