  - body: Current main still exposes and tests localRoots: "any" for host-read callers with a readFile override. Removing the type and bypass here leaves existing outbound and plugin callers unmigrated.

  - confidence: 0.86

  - body: Local ! commands are currently approved operator shell commands. Parsing to argv with shell: false treats pipes, redirects, globs, and variable expansion as literal arguments, silently changing existing workflows.
  - confidence: 0.82

  - body: Current exec SecretRef resolution passes only configured passEnv plus provider env. Adding baseline HOME, USER, LANG, TERM, TMPDIR, TMP, and TEMP for every provider broadens subprocess context without an explicit contract or test.
