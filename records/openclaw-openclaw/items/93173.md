The issue asks for Skill Workshop `apply` approvals to show proposal metadata and for the lifecycle approval window to be longer or configurable.

## Best Possible Solution

Define a bounded approval-detail contract for Skill Workshop proposal context, then choose a longer-default or explicit-timeout policy that preserves pending approval safety without unnecessary config surface.
