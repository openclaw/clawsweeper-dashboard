## Summary

Keep open: current main and v2026.6.8 still build Skill Workshop lifecycle approvals from static action text, the UI only renders generic plugin approval fields, and Skill Workshop still has no approval-timeout policy beyond the generic 120s plugin approval default.

## What This Changes

The issue requests proposal-aware Skill Workshop apply approval popup details plus a longer or configurable approval timeout.

## Best Possible Solution

Define a bounded approval presentation contract for Skill Workshop lifecycle actions, enrich it with safe proposal context, and choose a timeout policy that improves review time without adding config unless maintainers approve the knob.
