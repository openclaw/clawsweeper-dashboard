This is current product/architecture design work with concrete cost observations and useful community input; current main has lower-level controls for isolated subagents, bounded recall, usage reporting, and fan-out limits, but it still does not define tiered conversational context delivery, shared decision-summary state, saliency routing, or pessimistic multi-agent budget reservation.

The issue requests design input on lowering token-cost growth in conversational multi-agent coordination while preserving emergent cross-agent context.
