## Summary

Keep this PR open for repair. The exact-session mirror addresses a reproduced channel bug and has strong live proof, but the current predicate can misclassify ordinary unphased assistant deltas as commentary, and the branch also removes an unrelated documentation navigation entry.

## What This Changes

The PR emits normalized assistant commentary progress, mirrors commentary and selected assistant deltas to exact hidden session-message subscribers, adds gateway and agent regression tests, and documents an agent-event/channel contract.

## Best Possible Solution

Normalize producer events to explicit commentary or final-answer phases before gateway fanout, mirror only verified commentary to exact hidden subscribers, restore unrelated documentation navigation, and narrow or split the broad contract document to behavior supported by current source and owner review.

## Solution Assessment

No, not as currently generalized. Exact-session mirroring is the correct boundary, but unphased deltas require reliable producer classification so normal final-answer streams cannot be duplicated as commentary.
