- **[P1] Wire the Teams manager in realtime startup:** `extensions/voice-call/src/runtime.ts:573`
  - body: In the realtime branch we call `setRealtimeRuntime` and skip `wireMsteamsRuntime`, so the provider never receives the `CallManager`.   - confidence: 0.9

  - body: The upgrade check accepts any valid `(timestamp, callId, signature)` until the timestamp window expires and never records used tuples.   - confidence: 0.78
