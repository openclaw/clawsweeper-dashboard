## Summary

Keep open. The feature direction is plausible and the PR includes useful gateway, SDK, UI, and proof-plugin context, but the new in-app iframe bridge can hand a privileged plugin fetch port to an externally navigated or redirected document and needs a security fix before merge.

## What This Changes

The branch adds a plugin SDK Control UI entry-point registration API plus gateway listing/launch-token methods, plugin route auth plumbing, generated protocol models, Control UI navigation/iframe rendering, docs, and focused gateway/UI contract tests.

## Best Possible Solution

Keep the generic entry-point seam only after the in-app bridge is constrained to plugin-owned same-origin documents or redesigned to avoid exposing a cookie-backed request port, then add visible real-browser proof and maintainer API/security review.

## Reproduction Assessment

Not applicable for the feature request itself. The blocking security concern is source-reproducible: the PR posts a bridge port to the iframe content window after each load without validating that the loaded document is still the intended plugin-owned page.

## Solution Assessment

No. The high-level plugin UI entry-point seam is plausible, but the current in-app iframe bridge is not the safest implementation because redirects or self-navigation can give an external document a plugin-scoped, cookie-backed fetch bridge.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P1] Verify the frame before handing out the bridge port:** `ui/src/ui/app.ts:840`
  - body: The load handler sends the plugin fetch MessagePort to whatever document is currently in the iframe. A plugin route can redirect or navigate itself to an external origin, and that page would receive a port that proxies same-origin, cookie-backed requests under this plugin path, exposing plugin data or actions. Gate the bridge on a proven same-origin plugin-owned load, or remove this port handoff until that can be enforced.
  - confidence: 0.86
