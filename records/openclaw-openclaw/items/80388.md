## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.82

Full review comments:

- **[P1] Stop handing the plugin bridge to navigated frames:** `ui/src/ui/app.ts:916`
  - body: The iframe bridge is reinstalled on every frame load and transfers a MessagePort with `targetOrigin: "*"`. A plugin page can navigate its sandboxed iframe away from the registered `/plugins/<id>` document, and the next load would hand that external document the same proxy that can make credentialed same-origin requests under the plugin root. Please only install the bridge for the intended plugin document or tear it down after navigation instead of reposting the port.
  - confidence: 0.86

## Security Review

Status: needs_attention
