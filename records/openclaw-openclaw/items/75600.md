## Summary

Keep open. Current main does not implement the bounded `before_agent_finalize` retry contract, so this PR still carries useful Plugin SDK work, but the latest head still has a merge-order bug where one exhausted retry candidate can suppress a later valid retry from another hook.

## What This Changes

The PR adds bounded `before_agent_finalize` retry metadata and budgeting, terminal run-context cleanup guards, bounded plugin cleanup timeouts, docs/changelog/API baseline updates, and a small Matrix DM live-config pinning fix.

## Best Possible Solution

Keep the focused Plugin SDK hardening direction, but preserve or evaluate all valid retry candidates until one with remaining budget is found. Add a regression test where the first valid retry key is exhausted and a later plugin retry is honored.

## Reproduction Assessment

Yes. Register two `before_agent_finalize` hooks for one run, have both return valid retry metadata with distinct keys, exhaust the first key's `maxAttempts`, and verify the second hook still triggers a `revise` instead of the merged result becoming `continue`.

## Solution Assessment

No, not as submitted. The seam placement is maintainable, but choosing one retry during hook-result merge before budget evaluation lets an exhausted plugin suppress another plugin's valid retry request; preserving candidates through merge or selecting after budget evaluation is safer.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Honor later retry requests after a spent retry:** `src/plugins/hooks.ts:341`
  - body: When multiple finalize hooks return retry metadata, this merge keeps only the first valid retry and drops later candidates before budget evaluation. If that first retry key has already exceeded `maxAttempts`, normalization returns `continue`, so a later plugin with unused retry budget never gets its requested extra pass. Preserve retry candidates through merge or choose a retry after budget evaluation.
  - confidence: 0.88
