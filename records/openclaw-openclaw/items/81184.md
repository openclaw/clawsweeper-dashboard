- **[P2] Prune expired announce dedupe entries:** `src/agents/subagent-announce-dedupe.ts:83`
  - body: Every successful unique announce key is inserted into `state.delivered`, but expired entries are only removed when that same key is looked up again.   - confidence: 0.9
- **[P3] Release inflight entries for synchronous throws:** `src/agents/subagent-announce-dedupe.ts:79-90`

  - confidence: 0.82
