## Best Possible Solution

Serve negotiated gzip/Brotli Control UI assets from cached, async, streamed, or precomputed variants while preserving current-main routing, CSP, base-path href rewriting, HEAD behavior, q-values, and binary passthrough.

## Reproduction Assessment

Yes, at source level. Current main and v2026.6.10 do not inspect Accept-Encoding or emit Content-Encoding for Control UI static assets, and the linked report plus PR proof give concrete curl checks for the missing header.

## Solution Assessment

No. The Control UI handler is the right owner boundary, but synchronous zlib on the request path is not the best implementation shape for large static bundles; cached, async, streamed, or precomputed compression is safer.
