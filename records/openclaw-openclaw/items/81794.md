Adds gzip/Brotli Accept-Encoding negotiation for Control UI static assets and tests gzip, q-values, wildcard fallback, HEAD responses, and binary passthrough.

## Best Possible Solution

Serve negotiated gzip/Brotli from the Control UI handler using cached, async, streamed, or precomputed compressed variants while preserving routing, CSP, base-path index rewriting, HEAD behavior, q-values, and binary passthrough.

## Reproduction Assessment

## Solution Assessment

No. The Control UI handler is the right owner boundary, but synchronous zlib on the request path is not the best implementation shape for megabyte-scale bundles; cached, async, streamed, or precomputed compression is safer.
