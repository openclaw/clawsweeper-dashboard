Fix ACPX Codex worker failures when configured Codex ACP `-c model` and `-c model_reasoning_effort` args overlap with runtime model/thinking overrides.

## Best Possible Solution

Review and land PR #74339, or an equivalent ACPX Codex command-normalization fix that appends only missing runtime config keys and covers configured command, partial-overlap, wrapper-command, and `sessions_spawn` paths.
