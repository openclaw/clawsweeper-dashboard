  - confidence: 0.9

  - body: After this PR adds document blocks to assistant content, the append-failure fallback still strips only managed image URLs. If append fails, the gateway can broadcast document links whose records remain transient and later 404.
  - confidence: 0.88

  - body: `safeAttachmentFilename` preserves Unicode and the handler inserts it directly into `filename="..."`; Node response headers reject non-Latin-1 characters. Build an ASCII fallback plus `filename*` for UTF-8 names so downloads do not fail for non-ASCII filenames.
  - confidence: 0.88
- **[P3] Guard the document route before importing it:** `src/gateway/server-http.ts:750-753`
  - body: Unlike the current image route, this stage imports the document module for every request path. Gate it with an `isManagedOutgoingDocumentPath` check so unrelated gateway requests do not pay for or fail on the document handler.
