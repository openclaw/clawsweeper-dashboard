  - confidence: 0.87

  - body: Unlike the existing image stage, this document stage is pushed for every request, so ordinary gateway requests import and run the new handler until it returns false. Add an `isManagedOutgoingDocumentPath(scopedRequestPath)` guard like the image route.
  - confidence: 0.84
