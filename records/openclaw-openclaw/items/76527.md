## What This Changes

The PR adds a `/steer` fallback that auto-selects the sole active subagent when the first token is missing or unresolvable and adds two unit tests for that path.

## Best Possible Solution

Settle targetless `/steer` semantics, then land the handler, command registry/docs/changelog, and regression tests as one coherent command contract.

## Reproduction Assessment

Yes, from source inspection: current main treats the first token after `/steer` as the target and resolves it before steering. `/steer please check logs` therefore fails target resolution instead of sending the whole text to the sole active subagent.

## Solution Assessment

Unclear. The code path is plausible for the one-active-subagent case, but it is not the best complete fix until the command contract and public docs agree with the chosen grammar.
