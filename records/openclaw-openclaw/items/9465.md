## Summary

Keep open. Current main still lacks persisted cron lifecycle hooks, and #45465 remains the active but conflicting implementation candidate that needs maintainer product/security review before this feature can land.

## What This Changes

Add a cron job hooks lifecycle for custom pre/post/error logic, including deterministic pre-run gates that can skip model calls.

## Best Possible Solution

Land or explicitly supersede #45465 with an additive cron hook lifecycle covering persisted job/config types, Gateway protocol, scheduled/startup/manual execution, CLI, Control UI, docs, tests, and a documented security model.

## Reproduction Assessment

Not applicable as a runtime bug reproduction. The missing feature is source-verifiable on current main: cron types, config, Gateway schemas, UI types, and execution paths do not expose or dispatch cron lifecycle hooks.

## Solution Assessment

Unclear for the exact proposed schema because this is active product and security design work. The maintainable path is to review, harden, and land or supersede #45465 rather than close this issue or start a separate automated repair branch.
