## What This Changes

Request to add cron job lifecycle hooks, including pre/post/error hooks and deterministic shell or webhook gating before scheduled execution.

## Best Possible Solution

Land or explicitly supersede #45465 with an additive cron hook lifecycle covering persisted job/config types, Gateway protocol, scheduled/startup/manual execution, CLI, Control UI, docs, tests, and a clear security model.

## Reproduction Assessment

Not applicable as a runtime bug reproduction. The missing feature is source-verifiable on current main: cron persisted types and Gateway schemas have no hooks field, and scheduled/manual runs call the core executor without a hook phase.
