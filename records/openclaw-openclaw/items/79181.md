## What This Changes

The PR adds a gateway startup throttle that records rapid starts in stateDir, backs off before sidecar loading, clears after stable uptime, wires it into post-attach startup, adds tests, and updates the changelog.

## Best Possible Solution

Land the narrow sentinel if maintainers accept the default backoff policy, while leaving full graduated recovery design to https://github.com/openclaw/openclaw/issues/79165.
