## Best Possible Solution

Land the shared title-keyed slug resolver after maintainer acceptance of the collision-case persisted identity contract, keeping reserved-index handling separate.

## Solution Assessment

Yes. A shared helper next to `slugifyWikiSegment` is the narrow maintainable boundary for both direct page writers; changing the global slug helper would be broader, and the older source-only approach would not fix synthesis writes.
