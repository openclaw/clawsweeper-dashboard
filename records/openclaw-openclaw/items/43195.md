## What This Changes

The PR moves unhandled-rejection handler registration and install-once state into a shared `globalThis` state so repeated installer calls avoid adding multiple process listeners.

## Best Possible Solution

Rebase a minimal install-once guard onto current main, preserve or safely migrate `openclaw.unhandledRejection.handlers`, add listener-count coverage, and attach redacted runtime output.

## Reproduction Assessment
