## Summary

## What This Changes

The PR makes unhandled rejection handler installation idempotent and moves the install flag plus handler registry into a shared globalThis state object.

## Best Possible Solution

Rebase a minimal install-once guard onto current main, preserve or safely migrate openclaw.unhandledRejection.handlers, add listener-count coverage, and attach redacted runtime proof.

## Reproduction Assessment

## Solution Assessment

No, not as currently branched. The install-once idea is narrow, but the PR must be rebased onto current main while preserving the shipped handler Set key and adding real runtime proof.
