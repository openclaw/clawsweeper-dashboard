Repair this PR or replace it with a narrow branch so explicit no-tool sessions omit tool-only skills/provider guidance and report only rendered prompt contents, while leaving payload stripping to https://github.com/openclaw/openclaw/pull/70790 and broader diagnostics to https://github.com/openclaw/openclaw/issues/44484.

  - confidence: 0.88
