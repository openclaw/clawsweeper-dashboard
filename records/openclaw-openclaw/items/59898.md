- **[P2] Suppress skills in explicit no-tool prompts:** `src/agents/system-prompt.ts:837-846`
  - body: When `toolNames: []` and `explicitEmptyToolListMeansNoTools` are active, this still builds and later inserts `skillsSection` from a non-empty `skillsPrompt`.   - confidence: 0.93

  - body: The explicit no-tools path still injects `providerStablePrefix` unconditionally. For GPT-5/OpenAI sessions that stable prefix includes tool-discipline instructions, so the final prompt can still combine `No tools are available` with provider-owned guidance to keep using tools.
  - confidence: 0.88
