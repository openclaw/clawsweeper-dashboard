  - body: The bundled entry declares `setChannelBrokerRuntime` from `./runtime-api.js`, but that module never exports the setter. `defineBundledChannelEntry` loads the named export during registration and throws when it is absent, so the new channel can fail to register before any broker request is sent.
  - confidence: 0.94
- **[P2] Infer broker-prefixed Slack DMs as direct:** `extensions/channel-broker/src/channel.ts:58-65`
  - body: The Slack-specific parser maps `broker:slack:user:U12345678` to a direct conversation for send/session routes, but `inferBrokerTargetChatType` still sees the first broker conversation segment as `slack` and returns `channel`.
