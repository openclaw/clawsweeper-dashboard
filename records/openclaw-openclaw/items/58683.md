- **[P1] Honor fallback intervals after schedule gaps:** `src/infra/heartbeat-runner.ts:1151-1157`
  - body: When a boundary moves from a scheduled window into a gap, `resolveScheduleIntervalMs(...)` returns `null`, so this guard skips recalculating `nextDueMs` from the fallback base interval.   - confidence: 0.91

  - body: `resolveNextWindowBoundaryMs` derives the next wake from minute-of-day arithmetic with a fixed 24-hour day. On DST transition days, a local schedule boundary can be scheduled about an hour early or late, so the interval transition is missed at the intended local time.
  - confidence: 0.86
