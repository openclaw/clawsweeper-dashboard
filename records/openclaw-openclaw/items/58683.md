- **[P1] Honor fallback intervals after schedule gaps:** `src/infra/heartbeat-runner.ts:1152-1155`
  - body: When a boundary moves from a scheduled window into an uncovered gap, `resolveScheduleIntervalMs(...)` returns `null`, so this guard skips recomputing `nextDueMs` from the fallback base interval. Compute the next local boundary as an epoch instant, or narrow the documented guarantee and tests to make that behavior explicit.
  - confidence: 0.84
