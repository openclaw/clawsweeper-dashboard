## Summary

Keep open. Current main has provider usage plumbing, CLI/status/docs surfaces, and an Overview model-auth usage hint, but it still does not provide the requested manual Control UI fuel gauge at the run/model decision point or the requested source/confidence/stale/unavailable metadata contract.

## What This Changes

Add a manual provider remaining-capacity fuel gauge in the Control UI near expensive run/model decisions, with normalized source, confidence, reset, stale, and unavailable metadata.

## Best Possible Solution
