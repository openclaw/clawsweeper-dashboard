An open PR already references this issue with closing syntax, with a broader live-catalog PR also open, so the issue should remain open until maintainers land or supersede those implementation candidates.

## What This Changes

Request to add `gpt-5.5` and `claude-opus-4.7-1m-internal` to the GitHub Copilot plugin model defaults/catalog and use accurate context metadata for 1M Copilot model IDs.
