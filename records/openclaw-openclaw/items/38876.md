Overall confidence: 0.86

Full review comments:

- **[P2] Bound array tool-output collection:** `ui/src/ui/chat/tool-cards.ts:49-60`
  - body: The added array path collects every string fragment and joins the full result before canvas preview parsing, sidebar formatting, and raw rendering. ## Security Review

Status: needs_attention

Summary: The PR diff still introduces broader unbounded client-side tool-output joining that can amplify memory and CPU use in the Control UI.
