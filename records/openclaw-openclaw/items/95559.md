  - body: `Client.postMessage` does not take a target-origin string; the second argument is a transfer list or options object. Passing `"*"` can throw as soon as a window client is found, so the update notification may never reach the stale tab.
  - confidence: 0.95

  - body: The new `matchAll()` promise is outside `event.waitUntil()` and races `clients.claim()`. Because `matchAll()` defaults to clients already controlled by this worker, existing tabs can be missed and the worker can finish activation without delivering the reload message.
  - confidence: 0.9
