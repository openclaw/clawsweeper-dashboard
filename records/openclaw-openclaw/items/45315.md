  - body: This only seeds the dispatcher context, but fast-abort delivery first tries `routeReplyToOriginating()`. `routeReply()` resolves `responsePrefix` and normalizes without `responsePrefixContext`, so routed abort replies can still send raw `{model}` or `{thinkingLevel}` tokens.
  - confidence: 0.87

Fast abort skips that callback, so this can replace raw placeholders with the wrong model or thinking level.
  - confidence: 0.84
