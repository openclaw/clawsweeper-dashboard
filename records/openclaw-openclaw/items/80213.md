## Security Review

Status: needs_attention

Summary: The feature is security-sensitive because it would introduce install/update-time execution of skill-supplied local commands.

  - body: Install/update-time setup scripts can run local commands from third-party skill packages, so the design needs explicit consent, path confinement, timeout ownership, logging/redaction, rollback behavior, and ClawHub review semantics.
