## Summary

Keep this PR open for normal merge review: current main still has the reported workflow template-injection pattern, and the PR's env-hoisting diff is narrow and matches the documented mitigation with no blocking review finding.

## What This Changes

The PR updates `.github/workflows/openclaw-performance.yml` to pass Kova report output paths through `REPORT_JSON` and `REPORT_MD` environment variables before the clawgrit publish shell script copies them.

## Best Possible Solution

Land this focused workflow hardening after the exact-head CI/security checks pass, while leaving the broader #68428 tracker open for the remaining workflow findings.

## Reproduction Assessment

## Solution Assessment
