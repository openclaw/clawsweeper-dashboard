## Summary

Keep open: the reported provider fallback bug is still present on current main and the latest release, but several open PRs now target it. This issue should remain the canonical tracker until one proven fix lands.

## What This Changes

## Best Possible Solution

Land one canonical narrow classifier fix that recognizes structured upstream provider failures without broadening format/schema failures, then close this issue and supersede overlapping PRs.
