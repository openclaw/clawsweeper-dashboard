## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Use a defined gateway error code:** `src/gateway/server-methods/sessions.ts:465`
  - body: `ErrorCodes.INTERNAL_ERROR` is not part of the current gateway protocol error enum, and the PR's `pnpm check` job fails `tsgo` at this line. Use an existing protocol code such as `UNAVAILABLE`, or add a new code consistently through the protocol contract before referencing it.
  - confidence: 0.96
- **[P1] Enforce session visibility before cancelling:** `src/agents/tools/sessions-cancel-tool.ts:32-34`
  - body: `sessions_cancel` forwards the caller-provided key directly to `sessions.cancel`, while existing session tools resolve the target and apply session visibility/A2A policy first. A model that learns another ACP session key could cancel a session it should not control, so resolve and guard the target before invoking the gateway RPC.
  - confidence: 0.93
- **[P2] Reject non-ACP targets before dispatch:** `src/agents/tools/acp-send-tool.ts:218-224`
  - body: `acp_send` accepts any visible session and starts an internal run with `deliver: false`, but only ACP sessions have the callback path this tool depends on. Validate the resolved target as ACP-only so normal or native subagent sessions fail fast instead of losing their replies.
  - confidence: 0.87
- **[P2] Regenerate protocol artifacts:** `src/gateway/protocol/schema/sessions.ts:119-123`
  - body: Adding `SessionsCancelParamsSchema` changes the gateway protocol contract, but the generated Swift protocol models are not committed and live `pnpm protocol:check` fails with those diffs. Regenerate and include the protocol outputs before merge.
  - confidence: 0.96
- **[P2] Use the current TypeBox package:** `src/agents/tools/acp-send-tool.ts:2`
  - body: The new tool imports `@sinclair/typebox`, but current main has migrated agent and protocol code to the direct `typebox` package. Rebase these imports so the branch compiles against current main.
  - confidence: 0.84
