## What This Changes

## Best Possible Solution

Ship or rework a native backup exclude-pattern design that has restore-safe protected defaults, docs, focused tests, and clear JSON/manifest semantics; keep the ClawHub plugin as a useful workaround, not a replacement for the core backup CLI gap.
