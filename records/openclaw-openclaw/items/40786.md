## Summary

Keep open. Current main and v2026.5.7 still do not expose user-supplied backup exclude patterns, while an open closing PR already owns the native CLI implementation path; Keep My Claw is useful prior art and workaround, but it does not close the first-party backup CLI gap.

## What This Changes

Add .gitignore-like exclude controls to `openclaw backup create`, including repeatable patterns, an exclude-file option, and possible default exclusions for bulky or sensitive files.

## Best Possible Solution

Resolve the native backup exclusion design through https://github.com/openclaw/openclaw/pull/44288 or a narrower replacement with CLI forwarding, archive filtering, restore-safe defaults, manifest/docs updates, and focused regression coverage.
