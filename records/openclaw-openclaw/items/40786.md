## Summary

Keep open. Current main and the latest release still do not provide user-supplied exclude patterns for native backup creation, and #44288 remains the active closing PR to review; Keep My Claw is useful prior art and a workaround, but it does not close the first-party CLI gap.

## What This Changes

Add .gitignore-like `--exclude` and `--exclude-file` support, optional defaults, and sensitive or bloated path omission for `openclaw backup create`.

## Best Possible Solution
