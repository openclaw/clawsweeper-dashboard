## What This Changes

The issue requests channel config that detects owner-authored outgoing DM messages and pauses auto-replies per conversation until cooldown expiry or explicit resume.

## Best Possible Solution

Define a stable WhatsApp-owned config and resume contract, implement per-account/per-conversation pause state where outgoing manual messages are observable, then document and test any shared helper before extending the behavior to other channels.
