## What This Changes

The PR moves four older markdown sidebar labels into new `markdownSidebar.*` locale keys across six locale files and wires `renderMarkdownSidebar` through `t(...)`.

## Best Possible Solution

Refresh against current main, define English source keys for the current markdown/canvas sidebar chrome, regenerate all locale bundles plus `.i18n` metadata, update focused tests, and attach redacted real UI proof.

## Reproduction Assessment

Yes. Source inspection on current main shows the markdown sidebar still renders current chrome labels directly in English, and the focused sidebar test asserts several of those English labels.

## Solution Assessment

No. Locale-backed sidebar chrome is the right direction, but this patch localizes an older sidebar shape and hand-edits generated locale files instead of using the current generated-locale workflow.
