## Summary

Keep open: the normal-compaction cleanup is a valid bug-fix direction, but this patch clears messaging-tool delivery evidence before the retry branch, which can erase state that current retry cleanup needs to preserve deterministic side effects; the external PR also still has only test/CI proof, not real behavior proof.

## What This Changes

The PR clears messaging-tool sent text, target, and media bookkeeping at every embedded Pi compaction end and updates the compaction handler test fixture.

## Best Possible Solution

Clear messaging-tool duplicate-detection state for completed non-retry compactions, but preserve or snapshot committed delivery evidence before retry cleanup records `hadDeterministicSideEffect`, with focused tests for both paths.

## Reproduction Assessment

Yes. Source inspection shows successful messaging sends populate normalized sent-text state, current normal compaction leaves it present, and `message_end` can skip a later final reply from that stale state; the PR-introduced retry regression is also source-reproducible from the clearing order.

## Solution Assessment

No, not as-is. Clearing stale state is the right direction, but doing it before the `willRetry` branch is not the narrow maintainable fix because retry cleanup currently needs that evidence first.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Preserve retry delivery evidence before clearing state:** `src/agents/pi-embedded-subscribe.handlers.compaction.ts:90-93`
  - body: The added clear runs before `willRetry` is evaluated. On overflow compaction retries, `resetForCompactionRetry()` currently reads `messagingToolSentTexts`, media URLs, and targets to set `hadDeterministicSideEffect` before it clears them; this patch empties those arrays first, so a run that already sent via a messaging tool can lose its side-effect marker and later be treated as replayable or abandoned. Move the clear to the non-retry path, or snapshot the evidence before clearing.
  - confidence: 0.9
