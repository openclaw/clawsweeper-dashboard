## What This Changes

The branch splits secrets runtime state and fast-path helpers, lazily loads the full secrets runtime in gateway startup/reload paths, and updates web-tool/config callers plus import-boundary regression tests.

## Best Possible Solution

Integrate the lightweight startup fast path with current main's prepared-snapshot activation hook so plain no-SecretRef startup avoids heavy imports while reusable gateway-auth SecretRef preflight still prepares once.

## Reproduction Assessment

## Solution Assessment
