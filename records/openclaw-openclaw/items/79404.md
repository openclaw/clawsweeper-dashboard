  - body: A plain `.trivyignore` entry suppresses `KSV-0125` for every scanned target, while the comment accepts it only for the Kubernetes Deployment manifest. Use an inline ignore or `.trivyignore.yaml` with `paths` and an enforced expiry so future occurrences still fail the gate.

  - confidence: 0.88
