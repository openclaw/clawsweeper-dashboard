Keep open. Current main and v2026.5.7 still carry media-understanding provider/model telemetry internally, but the formatter drops it before `BodyForAgent` is forced from the formatted body, and I found no open PR or canonical issue that owns the remaining fix.

## What This Changes

Expose the actual media-understanding provider/model in the assistant-visible inbound body so the assistant does not guess which media model analyzed the current turn.

## Best Possible Solution
