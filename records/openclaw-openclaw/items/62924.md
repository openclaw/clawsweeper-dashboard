Keep open. Current main and the latest release still carry media-understanding provider/model telemetry internally, but the assistant-visible inbound body is formatted from only title, user text, and transcript/description text, so the reporter’s guessed-model failure remains a narrow unresolved formatter/apply-path bug.

Expose the actual current-turn media-understanding provider/model in the assistant-visible inbound media body so the assistant does not guess which model analyzed media.
