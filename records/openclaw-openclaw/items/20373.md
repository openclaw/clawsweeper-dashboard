## Summary

Keep open. Current main already covers most of the security-header portion, but it does not implement the broader gateway-wide per-IP/per-session request limiting requested in #20373.

## Evidence

  - file: [src/gateway/control-ui.ts:142](https://github.com/openclaw/openclaw/blob/a958b6e72328d1d922a6d133b1a70e866faf4013/src/gateway/control-ui.ts#L142)
