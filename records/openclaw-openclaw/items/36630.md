  - confidence: 0.86

  - body: `effectiveReplyTo` can come from an explicit payload-level `replyToId`, but `markFirstReplyConsumed` marks the shared inherited state consumed for any valid effective target. A first explicit quote can therefore prevent later payloads in the same Signal turn from inheriting the current inbound reply, dropping expected quote/thread context; return the reply source from `resolveSignalReplyDelivery` and consume only when the inherited target was actually used.
  - confidence: 0.9
