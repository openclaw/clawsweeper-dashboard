Keep the availability-plan reuse, but make image planning use the same workspace-scoped snapshot as the other media tools and add a regression test for explicit `workspaceDir` image availability.

No. Reusing the plan is the right direction, but the image path should use the already loaded workspace-scoped snapshot or pass `params.workspaceDir` into `getCurrentCapabilityMetadataSnapshot`.
