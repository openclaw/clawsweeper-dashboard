## What This Changes

The branch adds a shared optional media factory plan for image, video, and music registration, threads an `availabilityResolved` fast path into generation tool factories, and expands media factory planning tests.

## Best Possible Solution

Keep the performance fast path, but make the shared plan use the same generation availability contract as the factories and cover auth-store, agent-dir, env/config, workspace snapshot, and manifest fallback cases before merge.

## Reproduction Assessment
