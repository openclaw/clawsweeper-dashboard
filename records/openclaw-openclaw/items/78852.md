## Best Possible Solution

Keep the fast path, but make `resolveOptionalMediaToolFactoryPlan` reuse the same provider-auth fallback as `hasGenerationToolAvailability` for image, video, and music generation, with focused config-apiKey-only regression coverage before merge.
