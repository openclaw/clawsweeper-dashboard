## What This Changes

## Best Possible Solution

Rebase onto current main, run backup/snapshot scanning before the active-config validity gate with invalid-active-config coverage, then land only if maintainers accept the stricter check-mode behavior or choose a softer rollout.

## Reproduction Assessment

## Solution Assessment
