## What This Changes

The issue requests graph-aware loop detection for sessions_send so 3+ agent cascades cannot bypass maxPingPongTurns.

## Best Possible Solution

Add one canonical sessions_send cycle guard that propagates normalized visited agent IDs in input provenance and blocks revisiting an agent before delivery, with focused protocol, runtime, and regression tests.
