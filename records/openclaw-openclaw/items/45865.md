## Summary

Keep open: current main still lacks the requested Plugin SDK LLM runtime helper, and the open #64294 PR is the active implementation candidate rather than a reason to close this issue early.

## What This Changes

Request adding an SDK-managed `api.runtime.llm()` or completion helper so plugins can make OpenClaw-routed LLM calls instead of direct provider calls.

## Best Possible Solution

Review and either land #64294 or agree a narrower SDK/runtime API that exposes single-shot completions through existing model resolution, auth, retry/rate-limit behavior, docs, and regression tests.

## Solution Assessment

Unclear pending maintainer API review. A host-mediated single-shot completion helper is the right maintainable direction, but budget, attribution, permission, streaming, and naming semantics should be settled before merging.
