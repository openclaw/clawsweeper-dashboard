## Summary

Keep open: current main still lacks the requested Plugin SDK LLM runtime helper, and #64294 remains open and unmerged as the active implementation candidate rather than a landed fix.

## What This Changes

Add a Plugin SDK runtime LLM helper so plugins can perform inference through OpenClaw's model routing and auth stack instead of calling providers directly.

## Best Possible Solution

Review and either land #64294 or agree a narrower SDK/runtime API that exposes single-shot completions through existing model resolution, auth, retry/rate-limit behavior, docs, and regression tests.
