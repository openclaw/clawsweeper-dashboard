Request asks for human-readable/truncated recent-session titles plus sidebar rename and individual session delete controls in the Control UI.

## Best Possible Solution

Keep this as the narrow Control UI session-management tracker, land or close the related chat-picker rename PR separately, and then add focused per-row delete/sidebar action UX after maintainers choose the interaction pattern.
