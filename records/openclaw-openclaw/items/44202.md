## Best Possible Solution

Keep this as the canonical Apple Silicon local-memory safety tracker until OpenClaw has maintainer-approved CPU/Metal guidance, explicit backend policy, crash containment or worker isolation, safe local batching semantics, QMD non-zero-exit handling where appropriate, and real Apple Silicon proof.
