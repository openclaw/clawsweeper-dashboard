  - confidence: 0.82
- **[P1] Remove the workspace-wide lodash-es override:** `package.json:1253`
  - body: A Control UI rendering feature should not add a root `lodash-es` override without visible maintainer approval. That override changes dependency resolution for the whole workspace and is especially risky on a branch already stale against current package-management layout.
