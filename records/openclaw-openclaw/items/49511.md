  - confidence: 0.86

  - body: The renderer only queries `.mermaid-block:not([data-mermaid-status])`, but both module-load and render failures set `data-mermaid-status="error"`. A transient chunk or render timing failure leaves the same block permanently skipped on later chat updates even though the loader promise is reset.
  - confidence: 0.82
