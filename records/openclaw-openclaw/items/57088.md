  - confidence: 0.88
- **[P2] Grow tail reads until keepLines are complete:** `src/cron/run-log.ts:166-167`
  - body: The prune window is capped at `maxBytes` or `keepLines * 2048`, so long JSONL entries can leave fewer than `keepLines` complete records before rewriting. Expand the tail in bounded chunks until enough complete lines are collected or the file start is reached.
  - confidence: 0.9
- **[P2] Drop the first streamed line only when partial:** `src/cron/run-log.ts:184-185`
  - body: The prune path drops the first streamed line whenever `startPos > 0`, but `startPos` can land just after a newline. In that case the line is complete and pruning loses a valid recent entry.
  - confidence: 0.87
