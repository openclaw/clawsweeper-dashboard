## What This Changes

Matrix inbound voice messages should be transcribed before mention gating so spoken bot mentions and voice-only messages reach the agent.

## Best Possible Solution

Land a Matrix-owned preflight implementation that downloads audio after access checks but before mention gating, feeds successful transcripts into mention checks and BodyForAgent, marks only surfaced transcripts as transcribed, and preserves Matrix media/E2EE behavior.
