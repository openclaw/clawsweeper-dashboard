## What This Changes

## Best Possible Solution

A Matrix plugin implementation should preflight audio attachments with the shared helper, feed successful transcripts into mention checks and `BodyForAgent`, set `MediaTranscribedIndexes`, preserve Matrix media/E2EE fallback behavior, and document the STT data flow.
