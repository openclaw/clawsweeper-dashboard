## Summary

## What This Changes

## Best Possible Solution

Define a Codex app-server incomplete-turn recovery contract that releases the lane, sends an explicit channel-visible timeout status, records replay-unsafe native tool side effects, and offers only a safe retry/resume path that cannot duplicate prior work.

## Solution Assessment

Unclear for the full requested auto-resume behavior. A narrow status and replay-state fix looks maintainable, but automatic continuation after native tool side effects needs a maintainer-owned recovery policy before implementation.
