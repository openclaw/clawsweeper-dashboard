## Summary

Keep open. Current `main` has newer shared rich-message presentation support, but it still does not implement the lane-oriented plugin/runtime contract, generic interaction registration, or lane/actor outbound helpers described in PR #59986, so this work is not already shipped and is not safely closable as obsolete.

## Evidence
