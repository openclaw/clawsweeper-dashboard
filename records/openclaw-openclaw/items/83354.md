## What This Changes

The issue asks OpenClaw to stop helper commands from silently restarting a stopped and disabled Linux user-level `openclaw-gateway` systemd unit.

## Best Possible Solution

Define the Linux gateway service lifecycle contract so diagnostic commands remain non-mutating, helper recovery respects disabled or masked operator intent, and only explicit lifecycle commands start disabled units with docs and regression coverage.
