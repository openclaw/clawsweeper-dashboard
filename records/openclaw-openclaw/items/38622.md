## Label Justifications

- `P1`: Affected users silently start every session without persona and operating instructions, breaking a core agent workflow on both macOS and Linux.
- `impact:session-state`: The rejected bootstrap reads remove identity, memory, and instruction context from the agent session.
- `impact:security`: The requested behavior changes the filesystem trust boundary for content injected into the agent’s privileged prompt context.

## Summary

Current main still rejects external-target symlinked bootstrap files and renders the failed reads as missing. Keep this issue open because https://github.com/openclaw/openclaw/pull/73401 is an open maintainer-authored implementation candidate for the same behavior, and the security-sensitive boundary decision should be resolved there rather than through another fix path.

## What This Changes

Allow fixed workspace bootstrap files such as `SOUL.md` and `AGENTS.md` to load through symlinks instead of being injected as missing.

## Best Possible Solution

Land a fixed-bootstrap-only external-symlink reader through https://github.com/openclaw/openclaw/pull/73401, preserving the shared `@openclaw/fs-safe` boundary for all other reads and retaining hardlink, file-type, size, dangling-link, and extra-bootstrap protections.

## Reproduction Assessment

Yes, at high confidence from source: create a fixed bootstrap symlink whose canonical target is outside the workspace, then call `loadWorkspaceBootstrapFiles`; the root-scoped guarded open fails and the loader marks the entry missing. The discussion independently reports this exact path on macOS and Linux, though this read-only review did not run a live session.

## Solution Assessment

Yes, the linked PR’s fixed-bootstrap-only layer is the best available direction because it solves the user-visible bug without weakening the shared filesystem boundary globally. Final acceptance still requires maintainer judgment on whether user-created fixed bootstrap symlinks are sufficiently trusted and current real-session proof of the preserved guards.
