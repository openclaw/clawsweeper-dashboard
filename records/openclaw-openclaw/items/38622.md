## What This Changes

The issue reports that symlinked workspace bootstrap files such as SOUL.md are injected as missing instead of following the symlink target and loading content.

## Best Possible Solution

Keep this issue open until maintainers land or replace the scoped bootstrap-reader fix with explicit security tests that preserve hardlink, dangling-link, non-regular, oversized, direct-external-path, and extra/glob bootstrap rejections.

## Solution Assessment
