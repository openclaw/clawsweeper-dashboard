## Summary

This issue should remain open. Current main has session labels, an admin-style Sessions table, and a chat header session selector, but it still lacks the requested ChatGPT/Claude-style left conversation list management: persistent manual ordering, folder/tag/custom grouping, move-between-group actions, and double-click/right-click rename in the sidebar. There are overlapping open session-management issues and an open partial PR, so this needs maintainer consolidation rather than an automatic close.

## What This Changes

Requests ChatGPT/Claude-style left conversation list management with drag ordering, folder/tag/custom grouping, rename, and move-between-group actions.

## Best Possible Solution

Consolidate this into the Control UI session-management roadmap: define additive Gateway/session-store organization metadata for folders or tags/categories, manual order, and move/update operations, then implement a left conversation sidebar with drag ordering, context-menu or double-click rename, and move/group actions. Reuse existing session labels where they fit, and add focused protocol, store, UI, and docs coverage once the product contract is settled.
