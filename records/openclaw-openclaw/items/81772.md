## Summary

Keep open: current main already preserves explicit Telegram inline links through the Markdown-to-HTML send path, but the requested automatic `@username` rewrite or Telegram-owned mention helper/alias contract is still missing and needs maintainer product judgment.

## What This Changes

Requests Telegram channel support for highlighted, clickable user mentions using `tg://user?id=` links or an equivalent mention helper.

## Best Possible Solution

Add a Telegram-owned explicit mention helper or deterministic numeric user-ID alias mapping, document the Telegram privacy constraints, and avoid rewriting raw `@username` text unless the mapping is explicit.

## Reproduction Assessment

Not applicable: this is a feature request, not a broken documented behavior. Source inspection shows explicit inline links can already be formatted, while the requested automatic rewrite or helper contract is absent.

## Solution Assessment

No: automatically rewriting every raw `@username` is not the best default because Telegram mention links require numeric IDs and are constrained by membership/privacy rules. An explicit helper or alias mapping would be the narrower maintainable direction.
