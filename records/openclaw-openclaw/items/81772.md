## Summary

Keep open: current main can send explicit Telegram inline links, but it still lacks the requested outbound `@username` to numeric-ID mention helper or Telegram mention-alias contract.

## What This Changes

Request support for highlighted Telegram user mentions by converting or helping agents emit `tg://user?id=` links instead of plain `@username` text.

## Best Possible Solution

Add a Telegram-owned outbound mention helper or `mentionAliases`-style config using numeric user IDs, document it, test it, and leave plain `@username` text unchanged unless explicitly mapped.

## Solution Assessment
