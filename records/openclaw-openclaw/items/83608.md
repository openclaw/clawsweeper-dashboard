## Summary

Keep open: the patch addresses the optional-file symptom by changing shared setup semantics, which conflicts with documented setup/onboard seeding and current main tests for existing workspaces.

## Best Possible Solution

Preserve setup/onboard/configure seeding semantics and fix spawned runs by resolving the correct agent workspace or adding a runtime-only no-optional-seed mode with focused regression coverage.
