- **[P2] Limit normalization to tag candidates:** `src/shared/text/reasoning-tags.ts:45-49`
  - body: This maps every U+300C-U+300F character in the assistant message before stripping.   - confidence: 0.93

  - body: The new cases do not contain valid closing reasoning tags after normalization: the first uses `/log`, and the mixed case reverses the closing delimiter. With the strict parser these inputs leave the thinking block open and cannot produce the expected trailing visible text, so the fixtures need to use valid reasoning close tags before they can prove the change.
  - confidence: 0.9
