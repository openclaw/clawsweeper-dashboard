## Best Possible Solution

Teach the parser to recognize fullwidth delimiters only as part of candidate reasoning/final tags, preserve ordinary CJK punctuation, update the fixtures, and add redacted real behavior proof while keeping the linked WebChat streaming fix separate.

## Reproduction Assessment
