## Best Possible Solution

## Solution Assessment

Yes, with merge-state caveats. Parsing an existing absolute entrypoint while keeping shell fallback disabled is the narrow maintainable fix; the branch still needs rebase/current-main CI proof and an intentional decision on the memory-host local copy.
