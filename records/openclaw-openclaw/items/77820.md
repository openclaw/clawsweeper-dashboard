## Summary

Keep open: current main still routes WhatsApp terminal login QR output through `qrcode` with `small: true`, and upstream `qrcode@1.5.4` small terminal rendering has a source-visible final odd-row path that emits color resets per cell. Related PR #72762 is useful WSL font guidance, but it is still open and addresses a narrower symptom rather than this ANSI-density root cause.
