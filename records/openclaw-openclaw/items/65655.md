## Solution Assessment

Mostly yes for the remaining transport gap: the guard belongs in the Mattermost registration path before token-bearing callbacks are registered over cleartext. Merge readiness still depends on docs alignment, live proof, and maintainer/security acceptance of the compatibility break.
