## Summary

Keep open. This security-sensitive external PR is partially superseded by the merged Mattermost command-validation refresh, still has concrete patch findings, is conflicting against current main, and lacks required real Mattermost or gateway behavior proof.

## What This Changes

The branch updates Mattermost slash-command docs, registration, state, HTTP callback auth, and tests to reject omitted non-HTTPS derived callback URLs and bind callback tokens to slash triggers.

## Best Possible Solution

Rework or replace the branch with a current-main Mattermost-plugin fix that rejects the agreed class of non-HTTPS slash callback URLs before registration while preserving current live command validation, docs, focused tests, and redacted real-environment proof.

## Reproduction Assessment

Yes, at source level. Enable Mattermost native slash commands without `channels.mattermost.commands.callbackUrl`; current main derives `http://...` and continues into registration after the warning path.

## Solution Assessment

No, not as submitted. The token-binding half is superseded by stronger current-main validation, and the remaining transport fix is too narrow because explicit `http://` callback URLs still bypass it.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Reject explicit non-HTTPS callback URLs:** `extensions/mattermost/src/mattermost/monitor-slash.ts:168-174`
  - body: The new guard only runs when `callbackUrl` is omitted. An operator who explicitly configures an `http://` callback still reaches registration over cleartext, despite the PR's docs and rationale requiring HTTPS for slash callbacks.
  - confidence: 0.91
- **[P2] Make trigger binding fail closed:** `extensions/mattermost/src/mattermost/slash-http.ts:55`
  - body: `commandTokenTriggers` is optional, so constructing the handler without it silently falls back to token-only validation. Since this patch relies on token-to-trigger binding for the security fix, make the binding required or reject callbacks when the map is missing.
  - confidence: 0.84
