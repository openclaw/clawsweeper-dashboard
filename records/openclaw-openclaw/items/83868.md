## Best Possible Solution

## Reproduction Assessment

Yes at source level: current main builds full combined reasoning/status snapshots and the proposed patch still routes non-prefix rewrites through the streaming delta update path. I did not run a live Feishu tenant session.

No: removing the extra merges addresses one source of poisoned state, but the safest maintainable fix also needs prefix detection before delivery and a replacement path for rewritten card bodies.
