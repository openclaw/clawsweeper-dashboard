## What This Changes

The PR prevents channel-surface tool-discovery loads with an empty channel registry from replacing the existing pinned channel registry, and adds regression coverage for that state transition.

## Best Possible Solution

Land the narrow guard and regression tests once the registry owner accepts the channel-less tool-discovery invariant.

## Solution Assessment

Yes. The guard is at the narrow install/pin boundary, preserves the active tool-discovery registry for tool lookup, and avoids broader fallback changes in channel delivery or tools.catalog routing.
