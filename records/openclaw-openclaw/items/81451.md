## Summary

Keep open: the warm-path optimization is useful and the PR body has sufficient live log proof, but PR head still introduces two blocking P2 issues: an unbounded process-wide resolvedSkills cache and unconditional perf logging on the normal reply path.

## What This Changes

The PR adds a module-level resolvedSkills cache, skips redundant skill watcher path scans when config roots are unchanged, and adds focused tests for watcher reuse and cache hits.

## Best Possible Solution

Keep the warm-path optimization, but bound or current-version-prune the resolvedSkills cache and remove or debug-gate the perf logs while leaving session storage and public APIs unchanged.

## Reproduction Assessment

Yes for the review findings: PR head source shows a module-level Map that only tests clear and unconditional `console.log` calls inside `ensureSkillSnapshot`. I did not execute heap-growth or runtime log-volume proof in this read-only pass.

## Solution Assessment

No: the optimization targets the right hot path, and the watcher reconfiguration fix is present, but the safest maintainable version needs bounded cache invalidation and debug-gated or removed timing logs.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Bound the resolvedSkills cache:** `src/auto-reply/reply/session-updates.ts:36`
  - body: This module-level Map retains full `resolvedSkills` arrays and is only cleared in tests. Because `resolvedSkills` includes parsed SKILL.md bodies and snapshot versions change on invalidation, a long-lived gateway can accumulate stale snapshots indefinitely; keep only current entries or add a small TTL/LRU bound.
  - confidence: 0.87
- **[P2] Gate the perf logging:** `src/auto-reply/reply/session-updates.ts:188-190`
  - body: `ensureSkillSnapshot` runs during normal reply dispatch, but this branch prints `[perf]` timing lines on every call. Remove these logs or guard them behind the existing debug/timing mechanism so normal gateway stdout and hot-path work stay quiet.
  - confidence: 0.9
