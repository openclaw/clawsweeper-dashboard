## What This Changes

Request adds configurable `agents.defaults.bootstrapTier` tiers to control how much workspace bootstrap context main, subagent, and cron runs receive.

## Best Possible Solution

Review and either land or deliberately replace #22439 with an additive, documented bootstrap-tier contract that preserves current defaults, updates schema/types/help/docs, wires hook extras deliberately, and adds focused coverage for main, subagent, cron, lightweight, and extra-file behavior.
