## What This Changes

The issue requests configurable tiered bootstrap-file loading so different session types can load minimal, standard, or full workspace context.

## Best Possible Solution

Review and either land or deliberately replace https://github.com/openclaw/openclaw/pull/22439 with an additive, documented bootstrap-tier contract that preserves current defaults, wires root and hook-loaded files consistently, updates schema/help/docs, and covers main, subagent, cron, lightweight, and extra-file behavior.
