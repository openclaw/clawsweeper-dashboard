## Security Review

Status: needs_attention

Summary: Security review needs attention because plugin-provided canonical IDs become direct-message session keys and can merge transcript isolation without a fully documented trust contract.

Concerns:

- **[medium] Identity plugins can collapse DM session isolation:** `src/routing/session-key.ts:146`
