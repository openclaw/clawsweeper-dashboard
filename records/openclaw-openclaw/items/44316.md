This PR adds Web UI filtering for tool/function/tool-result messages and strips EXTERNAL_UNTRUSTED_CONTENT and SECURITY NOTICE text during message extraction.

## Best Possible Solution

Rebuild the fix on current main in `ui/src/ui/chat/build-chat-items.ts` and `ui/src/ui/chat/message-extract.ts`, preserving `showToolCalls`, suppressing only unsafe/internal plumbing before history windowing, and adding focused tests plus redacted Web UI proof.

## Reproduction Assessment

No. Source inspection shows wrapped external content can contain the reported SECURITY NOTICE markers and current extraction does not strip them, but no high-confidence live Web UI reproduction or after-fix proof is attached.

## Solution Assessment

No. The direction is useful, but the current branch is not the best solution because it bypasses the current `showToolCalls` contract, applies marker redaction too broadly, and filters hidden plumbing after raw history windowing.
