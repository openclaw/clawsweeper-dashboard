## Summary

Keep open: current main still lacks the external-marker cleanup this PR is trying to add, but the branch is not merge-ready because it bypasses the current `showToolCalls` contract, redacts marker-like text globally, and applies filtering after raw history windowing.

## What This Changes

The PR adds Web UI filtering for tool/function/tool-result chat messages and strips SECURITY NOTICE plus EXTERNAL_UNTRUSTED_CONTENT text during message extraction.

## Best Possible Solution

Rebuild the fix on current main so unsafe/security marker plumbing is hidden without defeating `showToolCalls`, with focused extraction/history tests and redacted Web UI proof.

## Reproduction Assessment

No. Source inspection shows current main can receive wrapped external content with SECURITY NOTICE markers and does not strip them in `extractText`, but there is no live Web UI before/after reproduction or proof attached.

## Solution Assessment

No. The direction is useful, but the branch is stale and too broad; the safer fix belongs in `build-chat-items.ts` and `message-extract.ts` with scoped filtering that preserves the tool-call toggle.

## Review Findings

Overall correctness: patch is incorrect

Full review comments:

- **[P2] Preserve the tool-call visibility toggle:** `ui/src/ui/views/chat.ts:1440`
  - body: Current main has a default-on `showToolCalls` control for showing tool calls/results. This unconditional filter makes the enabled state ineffective after rebase, so preserve the toggle and only suppress plumbing that must never render.
  - confidence: 0.86
- **[P2] Scope marker stripping to wrapped tool payloads:** `ui/src/ui/chat/message-extract.ts:42`

  - confidence: 0.86
- **[P2] Filter hidden plumbing before history windowing:** `ui/src/ui/views/chat.ts:1440`
  - body: The branch computes the render window from raw history and only then drops tool messages. In tool-heavy tails this can show far fewer real chat turns than the limit allows, so hidden-message filtering needs to feed the windowing calculation.
  - confidence: 0.8
