## Summary

Keep open: current main still has the reported Control UI tool-plumbing leak, but this PR needs real behavior proof and has two blocking correctness issues in its current patch shape.

## What This Changes

The PR changes Control UI chat rendering to always filter tool/function/tool-result messages and strip external-content security markers from extracted chat text.

## Best Possible Solution

A refreshed fix should filter non-renderable tool plumbing before applying `CHAT_HISTORY_RENDER_LIMIT`, keep live tool cards, scope marker cleanup to confirmed external/tool payloads, and add focused regression coverage.

## Reproduction Assessment

Yes. Source inspection gives a high-confidence path: pass `buildChatItems` a history message that normalizes to `toolResult` with `showToolCalls: true`, and current main still does not skip it; `extractText` also lacks external marker stripping.

## Solution Assessment

No. The intended direction is right, but this patch filters after raw history limiting and applies marker stripping globally instead of limiting it to confirmed external/tool-result payloads.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Filter tool plumbing before applying the history limit:** `ui/src/ui/views/chat.ts:1439-1441`
  - body: `historyStart` is calculated from raw history before the new unconditional filter. In tool-heavy sessions, dropping tool/toolResult entries after slicing can leave the UI showing far fewer real chat turns than the configured limit even though older user/assistant turns would fit.
  - confidence: 0.88
- **[P2] Limit marker stripping to tool-result paths:** `ui/src/ui/chat/message-extract.ts:42`
  - body: `processMessageText` now strips security notices and external-content markers for every role. Normal user or assistant messages that discuss those strings can be silently changed or become empty, so the sanitizer needs a confirmed tool-result or wrapped-external-content boundary.
  - confidence: 0.86
