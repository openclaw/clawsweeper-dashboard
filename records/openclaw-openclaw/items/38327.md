## What This Changes

The issue reports a regression where `google-vertex/gemini-3.1-pro-preview` embedded-agent runs fail with `Cannot convert undefined or null to object` after upgrading to OpenClaw 2026.3.2.

## Best Possible Solution

Land a narrow Google-family guard widening with regression tests for `google-generative-ai`, `google-vertex`, and `google-gemini-cli`, then track any remaining gaxios/native-fetch failure separately.
