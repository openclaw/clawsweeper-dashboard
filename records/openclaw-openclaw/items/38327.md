## What This Changes

The issue reports embedded-agent crashes with `Cannot convert undefined or null to object` for `google-vertex/gemini-3.1-pro-preview` after upgrading to OpenClaw 2026.3.2.

## Best Possible Solution

Land a narrow Google-family wrapper guard widening with regression tests for `google-generative-ai`, `google-vertex`, and `google-gemini-cli`, and track any gaxios/native-fetch dependency behavior separately.

## Solution Assessment
