## What This Changes

Add native iOS Siri/AppIntent support so users can ask OpenClaw a message from Siri or Shortcuts and receive a spoken/dialog reply through the paired Gateway.

## Best Possible Solution

Add a focused iOS AskOpenClaw AppIntent/AppShortcutsProvider surface that accepts a message parameter, uses the maintainer-approved paired-gateway trust path, returns a dialog result, and includes Swift coverage plus Siri/Shortcuts docs.
