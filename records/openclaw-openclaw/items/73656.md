## Summary

Keep open. Current main still has the reported bootstrap-only first visible subagent message, but that shape is a deliberate shipped prompt/token contract, so changing it needs maintainer product judgment rather than conservative cleanup.

## What This Changes

Request to include delegated task text in the first visible subagent user message as an auditable diagnostic and fallback while keeping the system prompt as the canonical task source.

## Best Possible Solution

Define an explicit subagent task visibility contract, then implement the chosen diagnostic surface, sanitized fallback, or metadata path while preserving token deduplication and fixing #73624 directly.

Unclear. Passing task into buildSubagentInitialUserMessage is mechanically small, but the best fix depends on maintainer choice between verbatim visible duplication, sanitized transcript diagnostics, metadata, and the separate #73624 repair.
