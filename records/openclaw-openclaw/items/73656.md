## What This Changes

Make the first visible native subagent user message include the delegated task text as a transcript/audit fallback while retaining the task in the subagent system prompt.

## Best Possible Solution

Define the subagent task visibility contract explicitly, then update the prompt helper, spawn coverage, docs or changelog together: either accept visible task duplication, move the canonical task to the user turn, or add a transcript-only diagnostic that makes assignment auditing possible without duplicating model-visible input.
