## Summary

Keep open: current main attributes the current turn through untrusted sender/history context blocks, but the persisted transcript prompt remains clean and hidden runtime-context messages are stripped before LLM replay, so durable group sender attribution in the messages array is still missing.

## What This Changes

Request durable per-message sender attribution for group chat user turns in the persisted LLM messages array so continuation-skip replay preserves who said what.

## Best Possible Solution

Define a generic untrusted sender-attribution field on persisted user transcript messages, map it per provider at the LLM boundary, and add group continuation-skip regression coverage.
