## What This Changes

The PR adds an in-memory `hasIndexedContent()` cache in memory-core and invalidates it on chunk writes, deletes, resets, and safe reindex DB swaps.

## Best Possible Solution

Refresh or replace the branch with a manager-local cache invalidated on every indexed-table mutation and active DB handle replacement, plus focused cache tests and redacted runtime proof.

## Solution Assessment
