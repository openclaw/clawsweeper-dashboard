## What This Changes

## Best Possible Solution

Implement the maintainer-requested flow: classify the raw Anthropic long-context 429 as overflow, run bounded compaction, then hand off a failoverable terminal outcome to configured fallback if compaction cannot recover, with docs matching that contract.
