## What This Changes

The PR adds an Anthropic long-context extra-usage classifier, routes that exact 429 through context-overflow detection, exports and tests the helper, and updates troubleshooting docs for compact + retry.

## Best Possible Solution

Classify the raw Anthropic long-context 429 as overflow, run bounded compact + retry first, then make exhausted recovery failoverable so configured fallback models still run.
