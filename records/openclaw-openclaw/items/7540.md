## Best Possible Solution

Implement this inside the WhatsApp plugin as one opt-in sanitized call-activity path that applies existing account, DM/group, self-chat, allowlist, and privacy boundaries before exposing any activity.

## Reproduction Assessment

## Solution Assessment

Unclear as a final product shape: the WhatsApp plugin is the right owner boundary, but maintainers should choose one sanitized opt-in activity surface rather than adding ad hoc session metadata plus a parallel tool API.
