## Summary

Keep open: current main still suppresses ordinary finals under message-tool-only delivery without an audit signal, and this PR is a narrow privacy-preserving diagnostic change with no blocking correctness or security finding found. ## What This Changes

The PR adds an internal diagnostic warning when substantive normal final replies are suppressed by message-tool-only source delivery and expands reply-dispatch coverage for silent, denied, media-only, and aggregate cases.

## Best Possible Solution

Land one canonical privacy-preserving diagnostic path for suppressed message-tool-only finals, preserving the no-auto-forward contract and avoiding suppressed final text in logs or diagnostic payloads.

## Reproduction Assessment

## Solution Assessment
