## Best Possible Solution

Keep redacted config readback and runtime credential resolution; handle sandboxed task secrets through explicit operator-managed env, mounted files, SecretRefs, or external secret delivery rather than exposing all provider keys to agents by default.

## Solution Assessment

No, exposing raw provider keys to agents by default is not the best solution. The maintainable path is runtime credential resolution plus explicit operator-controlled secret delivery for sandboxed tasks that need credentials.
