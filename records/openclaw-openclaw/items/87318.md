Request support for Bedrock Haiku 4.5 inference-profile model identity through a Bedrock-owned alias, full ARN model ref, or `params.modelId` request override.

## Best Possible Solution

Maintainers should choose one Bedrock model-identity contract, preferably discovered inference profile IDs/full refs, and only add `params.modelId` request override if that is accepted as provider API.
