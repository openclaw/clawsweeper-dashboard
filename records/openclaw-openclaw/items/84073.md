The PR adds `agents.list[].experimental.localModelLean` with schema/docs/test coverage and applies lean tool filtering per resolved agent across normal and compacted tool surfaces.

Rebase or repair the branch so it preserves all current-main runtime plumbing, then layer per-agent lean filtering after existing normalization/policy stages and add focused regression coverage for the restored paths plus live proof using the new config key.
