## Summary

Keep open. Current main still has adjacent memory indexing, thresholded promotion, REM/backfill, and memory-wiki provenance surfaces, but it does not implement the requested durable write resolver with classification, source-required writes, near-duplicate/conflict search, merge/revision handling, and explicit review routing.
