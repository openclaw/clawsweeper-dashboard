## Summary

Keep open. Current main still has no Linux/Windows desktop companion app, and this PR is the active successor to earlier desktop attempts, but the branch is not merge-ready because it reimplements config/dashboard contracts, keeps a security-sensitive restart path on ambient shell/PATH lookup, and misses required docs/changelog/labeler follow-through.

## What This Changes

The PR adds a new Tauri v2 React/Rust desktop companion under `apps/desktop`, root `desktop:*` scripts, workspace and lockfile entries, and a small tool-display export.

## Best Possible Solution

Land a desktop MVP only after maintainers accept the Tauri/restart model and the branch reuses canonical config, dashboard URL, and restart contracts, updates public docs/changelog/labeler coverage, and has Linux/Windows build/runtime proof.

## Reproduction Assessment

Yes for the static defects: the PR head files can be compared directly against current main's config IO, dashboard link helper, platform docs, labeler, and changelog policy. Full desktop runtime reproduction is incomplete because the discussion lacks `cargo check`, frontend validation, and platform smoke proof.

## Solution Assessment

No. A Tauri companion may be a reasonable direction, but this branch should first delegate to the existing config/dashboard contracts and use a trusted restart surface or resolved executable path instead of reimplementing those behaviors in Rust.

## Security Review

Status: needs_attention

Summary: The new desktop restart action shells through `cmd`/`sh` and resolves `openclaw` from the ambient PATH.

Concerns:

- **[medium] Ambient shell restart execution:** `apps/desktop/src-tauri/src/main.rs:556`
  - body: `run_gateway_restart()` invokes `sh -lc "openclaw gateway restart"` on non-Windows and `cmd /C openclaw gateway restart` on Windows. A first-party desktop app should not execute a PATH/cwd-hijackable command for a privileged-feeling GUI action; it should resolve a trusted CLI path or use a controlled Gateway restart surface.
  - confidence: 0.84
