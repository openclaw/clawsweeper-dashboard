  - body: These tools are added to every OpenClaw tool inventory, while current `update_plan` is gated by tool policy, `tools.experimental.planTool`, or strict-agentic support. That changes existing users' tool surface and gives models two new mutating session-state tools by default.

  - body: Adding only the TypeScript field advertises `agents.defaults.cotPlanning`, but `AgentDefaultsSchema` remains strict and has no matching schema/help/docs entry. Users who configure the new key can still hit an unknown-key config failure instead of controlling the feature.
  - confidence: 0.94
- **[P1] Keep mutable step plans out of SessionGoal:** `src/config/sessions/types.ts:218-219`
  - body: Goals are documented as one durable objective and not a task queue, but this adds a model-mutable `planSnapshot` of ordered steps directly to persisted goal state. That needs an approved goal-state contract and upgrade story before merge.
  - confidence: 0.88
- **[P2] Avoid prompting for visible chain-of-thought:** `src/agents/cot-planning-prompt.ts:89`
  - body: The prompt asks the model to produce a Chain-of-Thought plan in assistant-visible `<cot_plan>` tags. OpenClaw should request a concise execution plan or structured tool payload, not visible internal reasoning.
  - confidence: 0.86
- **[P2] Keep goal progress out of the stable prompt prefix:** `src/agents/system-prompt.ts:1026-1027`
  - body: `cotPlanningSection` is derived from current goal and plan progress but is included in the stable prefix cache key and inserted before the cache boundary. Plan state changes during execution, so this can churn the stable prefix cache instead of preserving byte-identical cacheable text.
  - confidence: 0.87
- **[P2] Reject plan-step updates after the goal completes:** `src/agents/tools/cot-planning-tools.ts:248-251`
  - body: `update_goal_plan_step` mutates `planSnapshot` without checking whether the goal is already complete, even though completed goals are terminal. That allows post-completion plan state drift after the goal should be closed.
  - confidence: 0.86
