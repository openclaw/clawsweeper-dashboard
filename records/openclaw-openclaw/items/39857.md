The issue requests that effective session context tokens resolve to the smaller of a configured override and a known selected-model window.

## Best Possible Solution

Review, rebase, or replace https://github.com/openclaw/openclaw/pull/39859 with a narrow current-main fix that applies min(override, knownWindow) consistently while preserving override behavior when the model window is unknown.
