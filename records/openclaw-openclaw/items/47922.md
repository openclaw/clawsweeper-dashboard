## Best Possible Solution

## Solution Assessment

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.89

Full review comments:

- **[P2] Preserve user-authored trailing commas:** `src/infra/outbound/payloads.ts:59`
  - body: When comma mode is enabled, this helper always runs `.replace(/,\s*$/g, "")`, so `Hello,` with no em dash becomes `Hello` across every send path using the transform. Trim only commas introduced by a trailing em dash, or return early when the input has no em dash.
  - confidence: 0.93
- **[P2] Handle leading em dashes without adding a comma:** `src/infra/outbound/payloads.ts:57-59`
  - body: Replacing `\s*—\s*` with `, ` turns leading em dashes into visible leading commas, so `— hello` becomes `, hello`. Add a leading-edge case or cleanup so the opt-in transform does not emit malformed outbound text.
  - confidence: 0.91
