  - body: When comma mode is enabled, this helper runs `.replace(/,\s*$/g, "")` for every message, so `Hello,` with no em dash becomes `Hello` across all delivery paths. Trim only commas introduced by a trailing em dash, or return early when the input has no em dash.
  - confidence: 0.93

  - confidence: 0.91
