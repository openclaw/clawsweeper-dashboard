## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

No, this is not yet the best fix. Bounded retries are the right direction, but the unconditional admin preflight changes the existing least-privilege spawn path; the safer fix is to preserve the current scope boundary and retry the gateway calls at the scope actually needed.
