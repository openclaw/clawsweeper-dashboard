## Best Possible Solution

## Reproduction Assessment

Yes. Current main has a high-confidence static reproduction path because bundled LSP startup directly spawns command names while the linked Windows report shows npm-installed language servers exposed through `.cmd` shims and failing with `uv_spawn` ENOENT.
