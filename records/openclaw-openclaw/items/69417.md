  - body: `loadMergedBundleMcpConfig` filters `enabled: false` MCP servers before launch, but this collector still resolves refs from every saved server. A disabled server with an unresolved SecretRef can therefore block agent runtime preparation even though that server would not run.
  - confidence: 0.9

  - body: The launch path uses `toMcpEnvRecord`, whose MCP-specific predicate also drops inherited host config pivots such as `AWS_CONFIG_FILE` and `KUBECONFIG`. This collector only checks `isDangerousHostEnvVarName`, so refs for env keys the child will drop can still be resolved or required.
  - confidence: 0.88

  - body: MCP env/header strings are valid literals today.
