## Evidence

The explicit `promptDefaultModel(...)` plus `applyPrimaryModel(...)` path only runs for `authChoice === "skip"`, not after a provider auth choice as described in the PR.
  - file: [src/commands/configure.gateway-auth.ts:129](https://github.com/openclaw/openclaw/blob/3a64aa49a9014518fa1977fbc023f3e0b978ae5d/src/commands/configure.gateway-auth.ts#L129)

  - file: [src/plugins/provider-auth-input.ts:163](https://github.com/openclaw/openclaw/blob/3a64aa49a9014518fa1977fbc023f3e0b978ae5d/src/plugins/provider-auth-input.ts#L163)
