## Summary

Keep open: current main still does not expose Slack `channel-info` or `channel-list` as agent-facing message actions. The linked open PRs are implementation candidates, not a reason to close the issue before one lands or a narrower follow-up owns any missing half.

The issue requests Slack message tool actions that let agents resolve Slack channel IDs and list visible Slack conversations.

## Best Possible Solution
