## Summary

## What This Changes

The PR extends the Gateway `chat.inject` protocol and generated Swift model with metadata fields, persists and broadcasts them, updates docs, and adds focused Gateway tests.

## Best Possible Solution

Land the metadata contract only after moving idempotency detection into the transcript append critical section, adding concurrent replay coverage, and confirming the API surface with maintainers.
