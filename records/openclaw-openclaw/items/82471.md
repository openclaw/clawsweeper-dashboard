## Summary

The PR adds optional `idempotencyKey`, `command`, `interactive`, and `channelData` fields to operator-admin `chat.inject`, persists and broadcasts them, updates generated Swift protocol models, docs, and focused gateway tests.

## Best Possible Solution

Keep the generic Gateway metadata seam if maintainers accept it, but enforce idempotency inside the serialized transcript append boundary or an equivalent inflight lock, add concurrent replay coverage, and require redacted live Gateway proof before merge.
