## Summary

Keep open. Current main has adjacent prompt guidance, memory confidence metadata, input provenance, task summaries, and tool-error warnings, but it still does not define the requested structured per-claim confidence/source-attribution contract or autonomous unverified-claim reporting surface.

## What This Changes

The issue requests a structured truthfulness mechanism for agent output, including confidence tiers, source attribution, known-vs-inferred wording, and autonomous-mode reporting for unverified claims.

## Best Possible Solution

Define a versioned claim-truthfulness contract and thread the smallest maintainable slice through agent runtime events, gateway schemas, task records, user-facing completion reports, tests, and docs.
