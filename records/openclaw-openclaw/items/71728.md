## Summary

Keep open. Current main has adjacent guardrails for evidence use, input provenance, internal completion events, task summaries, and memory-wiki claim metadata, but it still does not implement the requested per-claim confidence tiers, per-claim source attribution, known-versus-inferred semantics, or autonomous unverified-claim reporting.

## What This Changes

The RFC requests a structured truthfulness and confidence-signaling contract for factual claims in agent output, including source attribution and autonomous-mode reporting of unverified claims.

## Best Possible Solution

Define a versioned truthfulness contract for assistant output claims, then implement the smallest enforceable slice through runtime events, protocol schemas, stored task/completion records, completion summaries, tests, and docs.
