  - body: The new hook can return `group` for a cached private `channel:<id>`, but heartbeat delivery uses `resolveHeartbeatDeliveryTargetWithSessionRoute`, which then calls Mattermost's session-route resolver and replaces `delivery.chatType` with `route.chatType`.   - confidence: 0.9
