## Repair Work Prompt

Implement closest-root suggestions for unknown OpenClaw CLI root commands. Add a small pure helper under `src/cli/program/` that keeps a conservative explicit alias map such as `upgrade -> update`, computes local edit distance without a new dependency, and returns only close matches from core command names plus sub-CLI roots. Wire it into `formatCliParseErrorOutput()` and `resolveUnownedCliPrimaryMessage()` without auto-running commands and without weakening existing plugin ownership or `plugins.allow` diagnostics. Add focused tests for `upate -> update`, `upgrade -> update`, weak matches producing no suggestion, Commander formatter output, and early unowned-root/plugin-policy preservation. Add a concise `CHANGELOG.md` entry because this is user-visible CLI behavior. Do not change command registration semantics, plugin allowlist policy, or add Commander aliases for typo terms.
