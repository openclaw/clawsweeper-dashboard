## Summary

Keep open. Current main already has a default bundled browser plugin with managed headless launches, persistent profiles, Playwright-backed actions, optional Docker Chromium install, and extra Chrome launch args, but the request still asks for product/security decisions around always-bundled Chromium, first-class proxy/user-agent policy, credentials, and browser egress boundaries.
