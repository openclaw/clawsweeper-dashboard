## What This Changes

The issue asks for Control UI webchat streaming replies to update within one assistant bubble instead of showing transient fragment bubbles that collapse after completion.

## Best Possible Solution

Keep this issue open and land a focused Control UI rendering change that represents one in-flight assistant run as one assistant group while preserving tool-card ordering and gateway stream semantics.
