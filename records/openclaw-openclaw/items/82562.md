## What This Changes

The branch adds descriptor-scoped retention for cold-loaded plugin tool registries, reuses compatible metadata snapshots after plugin auto-enable, and adds regression coverage for cached descriptor execution after active registry replacement.

## Best Possible Solution

Land the descriptor-scoped registry retention fix after plugin/runtime owner signoff and current required checks, while preserving request-local tool-discovery behavior.

## Reproduction Assessment

## Solution Assessment

Yes, conditionally. Descriptor-scoped retention is narrower than globally installing the recovery registry and preserves the request-local tool-discovery behavior from https://github.com/openclaw/openclaw/pull/93276, but owner signoff should settle the compatibility model.
