Keep open. Request a cross-channel config or hook that sends a short text acknowledgement before agent processing and then continues normal dispatch.

## Best Possible Solution

Define one supported cross-channel acknowledgement-text contract or non-terminal send-and-continue hook, then align config types, strict schema/help, public docs, channel behavior, and focused regression tests.
