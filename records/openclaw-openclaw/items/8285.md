## Summary

Keep open. Current main still lacks a supported cross-channel text acknowledgment before agent processing, and the related Feishu pull request is open, channel-specific, and not a replacement for the requested config or plugin-API contract.

## What This Changes

Add configurable pre-processing acknowledgment text for messaging channels before normal agent processing begins.

## Best Possible Solution

Design a supported acknowledgment-text contract with scope and channel/account overrides, or a documented non-terminal pre-processing hook, then update config types, strict schema/help, public docs, channel docs, and representative messaging tests.

## Solution Assessment
