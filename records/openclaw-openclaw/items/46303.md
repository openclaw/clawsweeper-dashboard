- **[P1] Count summary-only followup work before restart drains:** `src/auto-reply/reply/queue/drain-all.ts:21`
  - body: `waitForFollowupQueueDrain` only counts queued items and an active drain sentinel. Include `droppedCount` in the pending count and add a regression for that summary-only state.
  - confidence: 0.93
