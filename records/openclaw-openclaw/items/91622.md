The issue reports two provider/runtime cron failures: Copilot Claude 4.5 rejects eager tool streaming, and Copilot GPT-5.4-mini via Responses aborts after a tool call with an incomplete-turn error.

## Best Possible Solution

Keep this issue open for the remaining GPT-5.4-mini Responses cron path and land a proven continuation or terminal-fallback design that uses recorded tool output without duplicating mutating tools.

## Reproduction Assessment

No high-confidence live reproduction was established in this review. Source inspection matches the remaining failure class: a replay-unsafe post-tool empty turn skips retry and reaches the current incomplete-turn error path.

## Solution Assessment
