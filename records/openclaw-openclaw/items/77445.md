## Summary

Keep this PR open: current main lacks the requested transcript reconciliation, but the proposed patch introduces a usage-accounting regression by making the synthetic zero-usage transcript mirror become the attempt's last/current assistant.

## What This Changes

The PR appends delivered `assistantTexts` into the embedded-runner transcript after successful non-compacted attempts, with docs, changelog, and focused unit coverage.

## Best Possible Solution

Land a variant that appends the durable assistant-text mirror while preserving the original model-call assistant and usage metadata; keep the broader memory and artifact sanitizer requests in their separate issues.

## Reproduction Assessment

Yes from source inspection: main collects delivered `assistantTexts` and uses them for outbound payloads, but the successful embedded-attempt path only snapshots existing session messages and does not append missing delivered text to the `SessionManager`. I did not run tests because this review was required to stay read-only.

## Solution Assessment

No, not as written. Appending a transcript mirror is the right narrow direction, but it should not replace the real attempt assistant used for usage and prompt-cache accounting.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.85

Full review comments:

- **[P2] Preserve model-call usage after reconciliation:** `src/agents/pi-embedded-runner/run/attempt.ts:3242`
  - body: After appending the synthetic transcript mirror, this block makes it the attempt's `lastAssistant` and `currentAttemptAssistant`. That mirror is created with `makeZeroUsageSnapshot()`, while the outer runner derives `lastRunPromptUsage`, `lastTurnTotal`, prompt-cache metadata, and final `lastCallUsage` from the attempt assistant before falling back to `attemptUsage`; reconciled tool turns will therefore report zero prompt/last-call usage even when the model call had real totals. Keep the real model-call assistant for accounting and only append the mirror to the transcript, or carry the original usage through explicitly.
  - confidence: 0.87
