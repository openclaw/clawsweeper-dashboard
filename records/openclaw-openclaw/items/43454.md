Keep open. Current main still lacks the requested workspace/internal `HOOK.md` events for `subagent:ended`, `agent:turnComplete`, and `agent:toolCallThreshold`; adjacent Plugin SDK hooks exist, but they do not solve the workspace hook gap. ## Best Possible Solution
