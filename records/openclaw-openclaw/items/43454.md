## Summary

Keep open. Current main and v2026.5.19 still lack the requested workspace/internal lifecycle events; adjacent plugin hooks exist, but this remains a public lifecycle API and security design request rather than a cleanup-close candidate.

## What This Changes

Expose workspace/internal lifecycle hooks for subagent completion, agent turn completion, and tool-call threshold events.

## Best Possible Solution

Define one canonical lifecycle-hook contract with minimal typed payloads, explicit opt-in and security boundaries, ordering and failure semantics, focused tests, and docs before wiring subagent, turn-complete, or threshold events.

Not applicable as a bug reproduction; this is a feature/API request. Source, docs, tests, and v2026.5.19 show the requested workspace lifecycle events are not present while adjacent plugin hooks exist.

## Solution Assessment

Unclear as proposed. Bridging subagent_ended into internal hooks is mechanically plausible, but the full request needs maintainer decisions on surface ownership, payload limits, threshold config, ordering, timeout, and failure behavior.
