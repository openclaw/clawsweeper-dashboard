## Security Review

Status: needs_attention

Summary: The requested hook surface is security-sensitive because trusted workspace hooks would gain new lifecycle execution points and potentially sensitive run/session metadata.

Concerns:

- **[medium] Minimize trusted hook payloads:** `src/hooks/loader.ts:46`

  - confidence: 0.9

  - body: triggerInternalHook awaits handlers in order, so firing it from an active tool loop for threshold events could stall turns unless timeout, ordering, and failure behavior are designed first.
