Keep open. ## What This Changes

Request gateway/workspace lifecycle hooks for subagent completion, agent turn completion, and tool-call threshold events.

## Best Possible Solution

Define one minimal lifecycle-hook contract with typed payloads, explicit opt-in/security boundaries, ordering and failure semantics, focused tests, and docs before wiring the requested events.

## Reproduction Assessment

Not applicable as a bug reproduction; this is a feature/API request. Source, docs, release inspection, and related open work show the requested workspace lifecycle events are absent while adjacent plugin hooks exist.

## Solution Assessment

Unclear as proposed. Bridging subagent_ended into internal hooks is mechanically plausible, but the combined surface needs maintainer decisions on ownership, payload limits, threshold config, ordering, timeout, and failure behavior.
