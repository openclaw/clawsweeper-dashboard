## Summary

## What This Changes

This PR changes the Model Aliases system-prompt wording, resolves exact configured slash-form aliases before provider/model passthrough for subagent model overrides, and adds focused tests.

## Best Possible Solution

Land a narrow prompt and alias fix only after maintainers explicitly accept or adjust slash-form alias precedence so exact provider/model routing stays upgrade-safe.

## Solution Assessment
