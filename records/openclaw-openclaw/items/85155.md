This PR changes the model-alias system prompt sentence and its focused prompt test so exact `provider/model` overrides are preserved while aliases are described as shortcuts for unqualified model requests.

## Best Possible Solution

Land the prompt and test update if maintainers accept exact `provider/model` precedence for model-facing alias guidance.
