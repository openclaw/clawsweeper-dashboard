## Summary

Keep open. Current main has addressed several audit bullets, but the dashboard-wide Control UI request is still only partially implemented: Instances is still presence-list oriented, Sessions still has dense per-row advanced controls without an explicit key-copy affordance, Config still uses terse Save/Apply/Update labels, and global shortcuts/action feedback remain limited; the narrower related reports do not replace this umbrella.

The issue requests a dashboard-wide Control UI UX cleanup across Navigation, Overview, Instances, Sessions, Config, Usage, Agents, and general design-system behavior.
