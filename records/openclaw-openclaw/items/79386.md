## Summary

Keep this PR open, but it is not ready to merge: the bug is valid and the fix direction is close, yet the patch can still leave the original poisoned binding persisted if the internal cold retry fails with a different error, and the external PR only provides test proof rather than real behavior proof.

## What This Changes

This PR extends Claude CLI recovery so a resumed no-output timeout retries without `--resume`, clears stored bindings in `runAgentAttempt`, adds focused regression tests, and updates the changelog.

## Best Possible Solution

Land a focused fix that clears the stored resumed `claude-cli` binding as soon as the poisoned timeout is identified, then either retries cold or defers to the next turn per maintainer choice, with regression coverage for retry-failure cleanup and real runtime proof.

## Reproduction Assessment

Yes. A high-confidence source path is to seed a stored `claude-cli` binding with a present transcript, have the resumed attempt throw `FailoverError` reason `timeout`, and observe that current main only clears `session_expired` while the PR demonstrates the intended timeout branch in tests.

## Solution Assessment

No, not as written. The direction is narrow and maintainable, but the binding needs to be cleared before or independently from the cold retry so any retry failure cannot leave the poisoned session id persisted.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.84

Full review comments:

- **[P2] Clear poisoned bindings on every retry outcome:** `src/agents/command/attempt-execution.ts:548`
  - body: When `runPreparedCliAgent` handles the resumed timeout, it retries cold before this outer catch can clear the store. If that cold retry then throws a non-timeout `FailoverError` or a plain error, the error reaching this condition no longer satisfies the new timeout branch, so the original `activeCliSessionBinding` remains persisted and the next turn can resume the same poisoned transcript again. Clear the active binding before starting the cold retry, or carry the original poisoned-resume signal through retry failures.
  - confidence: 0.86
