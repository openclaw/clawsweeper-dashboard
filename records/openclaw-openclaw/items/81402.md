  - body: The auth-profile state importer coerces whatever `loadJsonFile` returns, saves that state, and unlinks the source. A malformed legacy `auth-state.json` can therefore become an empty SQLite import with no retry source left behind.
  - confidence: 0.87
- **[P1] Merge stale device-auth snapshots safely:** `src/commands/doctor/legacy/device-auth-store.ts:35-42`
