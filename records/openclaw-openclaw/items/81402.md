  - body: `importLegacyDeviceIdentityFileToSqlite` writes the legacy identity into SQLite unconditionally and then removes the source.   - confidence: 0.89
- **[P1] Reject empty APNS imports before clearing registrations:** `src/commands/doctor/legacy/push-apns.ts:40`
  - body: This importer accepts a normalized empty APNS snapshot and calls the writer, whose empty-state path deletes all `apns_registrations`. A stale or shape-drifted `push/apns-registrations.json` can therefore wipe newer push registrations during `doctor --fix`; reject empty/malformed legacy snapshots or merge without deleting existing rows.
  - confidence: 0.88
