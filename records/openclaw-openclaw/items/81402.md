- **[P1] Reject empty APNS snapshots before replacing registrations:** `src/commands/doctor/legacy/push-apns.ts:40-41`

  - confidence: 0.92
- **[P1] Do not clear plugin approvals from malformed legacy input:** `src/commands/doctor/legacy/plugin-conversation-binding.ts:42-45`
  - body: A malformed-but-parseable `plugin-binding-approvals.json` normalizes to an empty approval set, then `writePluginBindingApprovalsSnapshot` deletes all persisted approvals and the importer unlinks the source. That can revoke current plugin/channel approvals during upgrade; treat invalid or empty legacy snapshots as failed imports unless maintainers explicitly choose a destructive reset.
  - confidence: 0.93
- **[P1] Merge bootstrap tokens instead of replacing current rows:** `src/commands/doctor/legacy/device-bootstrap.ts:35-41`
  - body: The bootstrap importer passes the legacy snapshot to `writePairingStateRecord`, whose `devices/bootstrap` branch deletes SQLite token rows not present in the incoming file. A stale `devices/bootstrap.json` can roll back or wipe newer bootstrap/onboarding tokens, so the migration should merge or skip stale rows rather than treat the file as authoritative.
  - confidence: 0.91
- **[P1] Validate Voice Wake routing before replacing routes:** `src/commands/doctor/legacy/voicewake-routing.ts:40-42`
  - body: This import normalizes any object-shaped legacy routing payload and immediately writes it, while the writer deletes existing `voicewake_routing_routes` before inserting normalized routes. Shape-drifted or empty legacy input can reset current Voice Wake routing to defaults during `doctor --fix`; validate the snapshot or preserve existing rows on invalid input.
  - confidence: 0.89
- **[P2] Preserve ACP replay data when normalization drops entries:** `src/commands/doctor/legacy/acp-event-ledger.ts:57-59`
  - body: This path writes the normalized ACP ledger and removes the legacy file even when normalization filters malformed sessions or events. A partially corrupted replay ledger can therefore lose skipped data and the only repairable source artifact; keep the source or fail the import when rows are dropped.
