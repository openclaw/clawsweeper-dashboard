## What This Changes

Adds a shared hook-aware CLI session-id selector and regression tests so Claude SessionStart hook IDs do not replace real resumable session IDs in JSON, buffered JSONL, and streaming JSONL parser paths.

## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Yes, after rebase. A shared selector that lets non-hook records win while retaining hook IDs only as fallback is the narrowest maintainable fix because all three parser paths carry session ids forward.
