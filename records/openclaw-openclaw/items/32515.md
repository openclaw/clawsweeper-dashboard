## What This Changes

The issue requests replacing Anthropic auth setup with a one-click browser OAuth flow that auto-detects API-key versus subscription-token account type.

## Best Possible Solution

Ship an approved Anthropic provider-owned browser auth method only after Anthropic’s supported contract is clear, while preserving API-key, Claude CLI, and setup-token routes.

## Solution Assessment
