## Summary

Keep open. Current main still lacks any public hook result field, merge rule, or runner path that lets a plugin replace the submitted user prompt; the related implementation work is either closed unmerged or still awaiting maintainer review.

## What This Changes

This issue requests an opt-in before_agent_start hook result that can replace the current user prompt before model submission.

## Best Possible Solution

Define and document a modern before_prompt_build user-prompt replacement contract first, including legacy before_agent_start mapping, merge precedence, injection filtering, cross-runner behavior, and regression coverage.

## Reproduction Assessment

Not applicable as a bug reproduction: this is a requested plugin API capability. Source inspection on current main shows no result field, merge rule, or runner path for replacing the user prompt.

## Solution Assessment

No, not as a before_agent_start-only change because that hook is documented as legacy. The maintainable solution should start with the current before_prompt_build contract, then deliberately mirror compatibility behavior through before_agent_start if approved.
