## Best Possible Solution

Make local TUI help/autocomplete and dispatch agree so every advertised local slash command is handled, safely routed, or explicitly rejected before it can reach the embedded model path, then prove it with the local PTY lane.

## Solution Assessment

No. The two-command branch is a useful mitigation, but the best fix is the full local TUI command boundary plus local PTY proof rather than adding only two special cases.
