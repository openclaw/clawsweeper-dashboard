The issue reports that the documented `curl -fsSL https://openclaw.ai/install.sh | bash` installer path can hang and corrupt the streamed script when child processes read from stdin.

## Best Possible Solution

Review and land or close https://github.com/openclaw/openclaw/pull/87799; close this issue only after the stdin-isolation fix reaches current main.

## Reproduction Assessment
