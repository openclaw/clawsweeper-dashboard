## What This Changes

Request to add WhatsApp pre-built link-preview payload support, optional Baileys high-quality preview fetching, and a batch send RPC for bulk outbound messages.

## Best Possible Solution

Stage the work: first add a typed WhatsApp-owned pre-built linkPreview pass-through through the supported outbound/Gateway path, then decide automatic fetching and batch Gateway sending as separate security/protocol designs.

## Solution Assessment

No, not as one bundled change. The narrow preview pass-through is maintainable first, while automatic fetching and batch RPC need separate maintainer decisions for security and protocol semantics.
