## Best Possible Solution

## Reproduction Assessment

Yes. Source inspection shows current main strips trailing NO_REPLY and returns the remaining preamble, and the current normalizer test asserts that behavior; I did not run tests because this was a read-only review.

## Solution Assessment

No, not as-is. The own-line classifier may be a good policy direction, but this patch applies it to only one path and makes matching narrower than existing case-insensitive silent-token helpers.
