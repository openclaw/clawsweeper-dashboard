## Summary

Keep open. Current main still runs through a Node-hosted OpenClaw CLI/Gateway and in-process agent-harness/plugin seams, so the embedded/no-Node request remains a broad runtime/API product design item; MimiClaw is useful prior art, and closed #45496 is narrower browser/WASM work rather than a canonical tracker.

## What This Changes

The issue requests runtime-agnostic support for running OpenClaw-style agent loops on devices without Linux or Node.js, using MimiClaw as prior art.

## Best Possible Solution
