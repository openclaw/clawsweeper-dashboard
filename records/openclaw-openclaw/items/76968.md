## What This Changes

Persist orphaned trailing user-leaf removal so file-backed transcripts stay aligned with the active in-memory branch.

## Best Possible Solution

Ship a narrow fix that removes the orphan leaf from the file-backed transcript under the session write path, keeps the nonstandard manager fallback, and adds regression coverage for both simple text and tool-use/tool-result queued-turn cases.

## Solution Assessment
