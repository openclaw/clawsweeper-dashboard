## What This Changes

Ensure orphaned trailing user leaf repair updates the persisted transcript file instead of only moving the in-memory session branch.

## Best Possible Solution

Review and land PR #76969, or an equivalent narrow fix, so file-backed sessions delete the persisted orphan leaf and include regression coverage for the rewrite path.
