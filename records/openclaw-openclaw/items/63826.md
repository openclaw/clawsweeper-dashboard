  - body: Plugin docs currently define `priority` as descending handler ordering without a public cap, and terminal hooks can stop lower-priority handlers.   - confidence: 0.87
- **[P2] Match aliases after earlier import specifiers:** `src/security/skill-scanner.ts:203`
  - body: The new alias rule only matches when `exec`/`spawn` is the first name after `{`, so `import { readFile, exec as run } from "child_process"` or `{ readFile, exec: run }` still bypasses the added scanner rule.   - confidence: 0.86
