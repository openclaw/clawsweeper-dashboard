- **[P1] Scope no-compression to Unix endpoints:** `extensions/codex-supervisor/src/json-rpc-client.ts:288`
  - body: The shared `options` object is also passed to the normal URL branch, so configured `ws://` or `wss://` endpoints silently stop advertising `permessage-deflate`.   - body: This test calls `server.listen(socketPath)` with an `os.tmpdir()` filesystem path.   - confidence: 0.9
