- **[P1] Keep no-compression scoped to Unix endpoints:** `extensions/codex-supervisor/src/json-rpc-client.ts:288`
  - body: The shared `options` object now reaches the normal URL branch too. Current config accepts `ws://` and `wss://` endpoints, so existing configured endpoints would silently stop advertising `permessage-deflate` even though the live proof and new test only cover the Unix-socket app-server path. - **[P2] Guard the Unix socket test on Windows:** `extensions/codex-supervisor/src/supervisor.test.ts:811`
  - body: `server.listen(socketPath)` uses a POSIX-style filesystem socket path. On Windows, Node IPC sockets need named-pipe paths, so this test can fail or hang before assertions run in a Windows lane. Skip this regression on non-POSIX platforms or build a valid Windows pipe path.
  - confidence: 0.82
