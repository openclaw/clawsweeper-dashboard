- **[P1] Create the config before starting EasyRunner:** `docs/platforms/easyrunner.md:108-113`

  - body: The update flow pulls new source and immediately redeploys `localhost/openclaw:latest`, so EasyRunner can keep launching the previously built image. Add the same `podman build -t localhost/openclaw:latest .` step before redeploying updates.
  - confidence: 0.9
