## What This Changes

Request that heartbeat runs skip the model prompt when workspace `HEARTBEAT.md` is absent, preferably with no log output.

## Best Possible Solution

Maintainers should decide whether missing-file skip should replace the shipped default or become an opt-in heartbeat setting, then update preflight behavior, tests, docs, and changelog together.

## Reproduction Assessment

Yes for the observed current behavior: source inspection shows a missing `HEARTBEAT.md` returns runnable preflight, and the test matrix expects the missing-file case to run. This is a feature/policy request rather than a failing current contract.

## Solution Assessment

Unclear. The unconditional skip is mechanically small, but it would reverse documented and shipped behavior, so the maintainable fix depends on an owner decision about compatibility versus token savings.
