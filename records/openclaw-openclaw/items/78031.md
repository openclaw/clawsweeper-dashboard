## What This Changes

The branch adds compact-writing guidance and examples to the BOOTSTRAP workspace template and a changelog entry advertising shorter generated workspace files.

## Best Possible Solution

Ship compact guidance only after the template accurately reflects context-injection semantics, release notes describe conditional savings, and maintainers accept the style tradeoff for first-run workspace files.

## Reproduction Assessment

Yes for the review findings: the PR diff contains the `continuation-skip` startup exception and current main source/tests show first-turn injection still occurs. ## Solution Assessment

No. The narrow maintainable path is to remove `continuation-skip` from the startup exception, qualify the changelog savings as conditional on successful bootstrap rewriting, and then let maintainers decide whether this compact template guidance should ship.
