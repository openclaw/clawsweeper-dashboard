## Summary

Keep open: current main has provider catalogs, model-list auth/probe surfaces, and a separate plugin compatibility policy, but it still does not provide the requested provider/model-family/capability lifecycle matrix, deprecation windows, generated docs page, or doctor/models lifecycle output.
