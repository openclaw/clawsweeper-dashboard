## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Yes, with maintainer acceptance. The best fix is to cancel at the shared subprocess spawn helper and thread the same signal through direct, grouped, and mcporter QMD search routes; a narrower route-only patch would miss supported paths.
