## What This Changes

The issue asks for the documented `openclaw memory` commands to dispatch successfully, or for help and docs to stop advertising unavailable commands.

## Best Possible Solution

Make the `openclaw memory` contract and advertising agree for active memory plugins: either provide reliable generic dispatch/extensibility for the active memory slot, or make help/docs conditional on the plugin that can actually handle the command.
