## Summary

Keep open. Current main has useful adjacent work in scoped routing keys, Gateway session create/delete/compaction APIs, Control UI session management, and memory search, but the RFC’s first-class multi-session architecture, archive/restore/pin/tag lifecycle, and shared public knowledge contract are still not implemented.

The issue proposes shared model inference with isolated per-channel sessions, public knowledge/RAG, and WebUI session lifecycle controls.
