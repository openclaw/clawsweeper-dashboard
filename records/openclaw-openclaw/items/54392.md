## What This Changes

The PR clamps the compaction `reserveTokens` value passed to Pi `generateSummary` by the selected model output limit, adds reserve-token clamping tests plus a compaction mock alignment, and adds an Unreleased changelog fix entry.

## Best Possible Solution
