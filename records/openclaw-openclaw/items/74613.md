The PR updates `docs/gateway/cli-backends.md` so the quick-start command passes `--agent main` and briefly explains replacing `main` for configured agents.

## Best Possible Solution

Merge this docs-only correction, or an equivalent maintainer edit, so the public CLI Backends quick start uses a selector-valid `openclaw agent` command without changing runtime behavior.
