  - confidence: 0.99

  - confidence: 0.96

  - body: Detection can accept a mention through `union_id`, but normalization still strips only the stored `botOpenId`. In the mismatch case the bot token remains before `/model` or another slash command, preventing command recognition; resolve the event mention once and use it for both detection and stripping.
  - confidence: 0.97
