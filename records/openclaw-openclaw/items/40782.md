## Summary

Keep this PR open, but it is not merge-ready: the proposed `union_id` fallback depends on an identity field that the linked issue says is not available from the probed Feishu bot-info response, and the patch does not carry the fallback through stripping/debounce paths that still affect message delivery.

## Best Possible Solution
