## What This Changes

The PR adds `botUnionId` plumbing through Feishu probe, monitor state/startup, and message parsing so mention detection can fall back from `open_id` to `union_id` in multi-bot groups.

## Best Possible Solution

Rewrite the fix on current main around a proven stable Feishu bot identity source, then cover text, post, mention stripping, mention-forward, debounce, focused tests, and redacted live Feishu proof before merge.

  - confidence: 0.84
