  - confidence: 0.83
- **[P2] Cover post messages in fallback matching:** `extensions/feishu/src/bot.ts:477-479`
  - body: The new `botUnionId` parameter is only used for top-level `message.mentions`; the rich-text `post` branch still compares parsed mention IDs only with `botOpenId`. Post messages in the same app-scoped mismatch scenario can still be treated as unmentioned and skipped.
