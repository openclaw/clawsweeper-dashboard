## What This Changes

Adds HTTP(S) and generic SSRF preflight validation to QQBot direct media URL uploads, adds QQBot media regression tests, updates small adjacent test fixtures, and serializes the gateway-watch built-artifact CI check.

## Best Possible Solution

Have QQBot/security owners review the fail-closed direct-upload policy, then land this guarded sink with the focused tests if the compatibility trade-off is accepted.

## Solution Assessment

Yes, pending owner acceptance. Placing the guard at MediaApi.uploadMedia() protects every direct-upload caller while preserving public HTTP(S) and fake-IP DNS hostname behavior in tests.
