- **[P2] Cover runnable Node release lines:** `src/commands/doctor-node-runtime.ts:121-136`
  - body: `runtimeSatisfies()` accepts any Node version `>=22.16.0`, but this schedule table only tracks Node 22. A runnable line like Node 25 is already in maintenance and reaches EOL on 2026-06-01, so Doctor would stay silent where this feature is meant to provide advisory context.
  - confidence: 0.9

  - body: This formatter only shortens paths under `homeDir + "/"`, so Windows paths such as `C:\Users\Alice\...` are printed with the username intact in support output. `shortenHomePath()` already handles both `/` and `\` boundaries; use it or match that behavior here.
  - confidence: 0.9

  - body: Current main requires Node `>=22.16.0`, but this warning still tells users that `>=22.14.0` is required. If Doctor reaches this edge case, it gives stale remediation guidance; derive the displayed version from the runtime contract or update it with coverage.
  - confidence: 0.86
