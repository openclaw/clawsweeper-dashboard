## What This Changes

Request accepted Discord reaction notifications to wake the routed agent session immediately after their existing system event is successfully queued.

## Best Possible Solution

Review the linked PR and, if maintainers approve the reaction-wake policy, land a Discord-owned wake after successful enqueue with focused tests and redacted live Discord proof.
