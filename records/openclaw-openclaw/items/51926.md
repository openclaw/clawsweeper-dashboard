## Best Possible Solution

Define omitted-node `describe` semantics for all describable nodes, implement that path without the canvas-only default filter, add focused regression coverage, and provide redacted real behavior proof.

## Solution Assessment

No. Reusing the shared canvas-biased resolver is a plausible minimal patch, but the best fix should make `describe` defaulting or discovery explicit for all describable nodes and cover it with a focused test.
