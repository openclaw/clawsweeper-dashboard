## Security Review

Status: needs_attention

Summary: The requested feature expands provider-hosted execution, network fetch/search, data-retention, and tool-authority surfaces, so it needs security design before implementation.

  - body: Anthropic-hosted web_search and web_fetch would move outbound access and domain controls into provider request parameters, so OpenClaw needs explicit opt-in policy, domain controls, cost controls, and transcript handling before enabling them.

  - confidence: 0.88
