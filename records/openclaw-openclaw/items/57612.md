## Summary

Keep open. The feature is useful, but the latest PR still has a blocking resource-limit issue in DOCX parsing, lacks real behavior proof, and changes the public input_file contract without matching docs.

## What This Changes

The PR adds a JSZip-based DOCX text extractor, routes .docx through OpenResponses input_file and the read tool, expands the default MIME allowlist, and adds regression tests.

## Best Possible Solution

Land a bounded, documented, maintainer-approved DOCX extraction path that enforces uncompressed XML limits before materialization and fits the document-extract/plugin ownership model.

## Reproduction Assessment

Yes for the blocking review finding: PR-head source plus JSZip v3.10.1 source show documentXmlFile.async("uint8array") accumulates content before the byte-cap check. I did not run tests because this review is read-only.

## Solution Assessment

No, not as-is. The feature direction is useful, but the implementation should enforce limits before XML materialization, document the new input_file contract, and get maintainer agreement on core versus document-extract plugin ownership.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Enforce DOCX XML limits before inflation:** `src/media/docx-extract.ts:69`
  - body: documentXmlFile.async("uint8array") inflates and accumulates the whole word/document.xml entry before maxXmlBytes is checked. A compressed DOCX that passes the outer file-size cap can still expand into a large XML payload in input_file or read, so enforce an uncompressed entry or streaming byte limit before materializing the XML.
  - confidence: 0.9
- **[P3] Document the DOCX input_file contract:** `src/media/input-files.ts:103`
  - body: Adding DOCX to DEFAULT_INPUT_FILE_MIMES changes the public OpenResponses input_file allowlist, but the docs still list only text/markdown/html/csv/json/pdf and only describe PDF parsing. Please document the DOCX MIME type, extraction scope, and limits.
  - confidence: 0.84
