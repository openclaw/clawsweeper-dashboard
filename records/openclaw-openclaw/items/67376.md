  - body: `clearActiveMcpLoopbackRuntime()` clears the global runtime and token table unconditionally from the close callback.   - confidence: 0.9

  - confidence: 0.87
- **[P2] Revoke issued tokens on all prepare failures:** `src/agents/cli-runner/prepare.ts:173-176`
  - body: The prepare path only unregisters the token when `prepareCliBundleMcpConfig` itself rejects. Later work such as docs/skills/session preparation can still throw before the returned cleanup is installed, leaving the issued token live in the process registry.
  - confidence: 0.84
