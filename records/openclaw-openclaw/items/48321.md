Keep open. Current `main` and the latest shipped release still preserve Slack thread image/file metadata but do not add prompt-visible per-image timestamp, sender, or thread-position annotations; the only related implementation PR is closed unmerged.

## What This Changes

## Best Possible Solution

## Solution Assessment
