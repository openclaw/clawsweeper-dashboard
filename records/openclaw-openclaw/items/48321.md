## What This Changes

Request to add timestamp, sender, and thread-position metadata for prior images in threaded channel history.

## Best Possible Solution

Add a Slack-owned annotation at thread-history formatting using sanitized image metadata and window-qualified position wording; defer a shared cross-channel contract until more channels show the same need.

## Solution Assessment
