## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.94

Full review comments:

- **[P1] Route remote_trigger through the guarded fetch path:** `src/agents/tools/claw-compat-tools.ts:635-640`
  - body: `remote_trigger` performs a direct `fetch` on a model-supplied URL with caller-controlled method, headers, and body. This bypasses the existing SSRF-protected web fetch path, allowing calls to localhost, metadata endpoints, or private-network services.
  - confidence: 0.96
- **[P1] Redact or allowlist config_compat reads:** `src/agents/tools/claw-compat-tools.ts:577-583`
  - body: When no path is supplied, `config_compat` returns the entire `loadConfig()` object, and dotted paths can select sensitive fields. That can expose provider keys, auth tokens, webhook secrets, or auth settings to model-visible tool results.
  - confidence: 0.95
- **[P1] Gate config_compat writes behind owner authorization:** `src/agents/tools/claw-compat-tools.ts:551-555`
  - body: Passing `setting` plus `value` writes persisted config with `writeConfigFile`, but this compat tool is not owner-only. Any session that can call it can mutate runtime configuration, including provider or permission-related settings.
  - confidence: 0.94
- **[P1] Gate plan-mode writes behind owner authorization:** `src/agents/tools/claw-compat-tools.ts:238-239`
  - body: `enter_plan_mode` and `exit_plan_mode` persist `permissions.defaultMode` without owner-only metadata. In non-owner sessions this changes control-plane permission state through broadly registered compatibility tools.
  - confidence: 0.92
- **[P1] Parse LSP frames by byte offsets:** `src/agents/pi-bundle-lsp-runtime.ts:79-80`
  - body: `Content-Length` is measured in bytes, but the parser slices a UTF-16 JavaScript string using that byte count. Non-ASCII hover, diagnostic, or symbol text can truncate or desynchronize JSON frames and cause request failures.
  - confidence: 0.9
- **[P1] Dispose failed LSP startup sessions:** `src/agents/pi-bundle-lsp-runtime.ts:477-480`
  - body: If LSP initialization throws, the catch block only logs and continues; the spawned child is not added to `sessions` and is never disposed. Repeated failed startups can leave orphan LSP processes.
  - confidence: 0.87
- **[P2] Preserve LSP paths for URI conversion:** `src/agents/pi-embedded-runner/run/attempt.tool-call-normalization.ts:413-417`
  - body: The alias shim copies `path` into `uri`, but `createLspTool` treats supplied `uri` values as canonical and only converts `path` when `uri` is absent. A relative path like `src/main.ts` is sent as a non-URI document id.
  - confidence: 0.9
- **[P2] Resolve safe MCP server names before dispatch:** `src/agents/tools/mcp-tool.ts:151-153`
  - body: Qualified MCP names parse the prefix before `__` and pass it directly to `runtime.callTool`. Catalog entries expose sanitized safe server names while sessions are keyed by original server names, so catalog-driven calls can fail for sanitized servers.
  - confidence: 0.86
- **[P2] Do not claim message delivery without dispatch:** `src/agents/tools/claw-compat-tools.ts:192`
  - body: `send_user_message` returns `delivered: true` but only echoes JSON and never calls the message/channel delivery path. Models using it will believe a user-facing message was sent when it was silently dropped.
  - confidence: 0.91
- **[P2] Preserve empty team arrays on update:** `src/agents/team-registry.ts:104-106`
  - body: `updateTeam` only applies `members`, `labels`, and `taskIds` when the patch arrays are truthy, and the tool normalizers turn empty arrays into `undefined`. Callers cannot clear existing team members, labels, or task links.
  - confidence: 0.84
- **[P2] Use flat enum schemas for bridge actions:** `src/agents/tools/mcp-tool.ts:11`
  - body: The bridge schemas use `Type.Union` over literal action values, which emits `anyOf`. Agent tool schemas should use the repo's flat string enum helpers so provider schema ingestion remains compatible.
  - confidence: 0.88
