## What This Changes

The issue asks for iOS group chats to show new agent replies in real time without leaving and reopening the conversation.

## Best Possible Solution

Implement the existing Gateway session-message subscription/decoding path in the iOS transport, then merge active-session assistant transcript events in the shared chat view model with focused Swift/iOS regression coverage.
