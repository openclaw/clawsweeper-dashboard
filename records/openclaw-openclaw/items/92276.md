## What This Changes

The PR adds trusted provenance for active diagnostic trace context on logger records, consumes it in the diagnostics OTEL plugin, and strengthens unit and real-collector smoke assertions for OTLP trace/span correlation.

## Best Possible Solution

Approve the narrow additive diagnostic metadata provenance seam, document or contract-test its intended plugin meaning, then land the patch without broadening trust for explicit trace objects or changing default log-body capture.
