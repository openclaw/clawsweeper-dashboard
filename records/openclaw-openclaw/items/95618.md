## What This Changes

This PR changes Gateway session and task registry terminal reconciliation so later same-run successes can repair failed/killed/timeout projections, and adds regression tests for retry-success ordering.

## Best Possible Solution

Limit retry-success repair to explicitly provisional same-run lifecycle failures while preserving operator cancellations, lost records, and hard-timeout semantics through the shared terminal outcome contract.
