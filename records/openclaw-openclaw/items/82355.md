## What This Changes

This PR updates OpenAI-compatible chat-completion streaming to recover missing assistant text from cumulative payloads, defer SSE finalization by a microtask, and add gateway/agent regression coverage.

## Best Possible Solution

Preserve literal custom XML text and final SSE chunk ordering in the gateway while retaining the existing stripped `<final>/<think>` behavior.

## Reproduction Assessment

Yes, at source level: current main can buffer a bare `<`, then the gateway reads incremental `delta` before cumulative `text`, and synchronous `[DONE]` finalization can race the text-end microtask. I did not run a live HTTP repro in this read-only review.

## Solution Assessment

Yes. Recovering missing assistant content from cumulative text in the SSE writer and deferring finalization by one microtask is a narrow maintainable fix for the observed source-level failure; the remaining blocker is proof quality, not the code direction.
