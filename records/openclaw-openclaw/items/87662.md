## Summary

Keep open. Current main and v2026.6.8 still provide mouse/sidebar/picker session switching and Cmd/Ctrl+K, but not guarded positional or cyclic Control UI session/agent switching shortcuts; the open guarded-dispatcher work at https://github.com/openclaw/openclaw/pull/84827 is adjacent infrastructure for different actions, not this exact feature.

## What This Changes

The issue requests guarded Control UI keyboard shortcuts for positional or cyclic session/agent switching that do not fire while editable controls have focus.

## Best Possible Solution

Choose the shortcut contract, then extend the guarded dashboard dispatcher to switch active-agent-filtered sessions through switchChatSession with focused editable-target, ordering, Escape, and picker/composer regression coverage.

## Reproduction Assessment

Not applicable: this is a feature request rather than broken existing behavior. Source inspection gives a high-confidence current-main comparison showing picker/sidebar switching exists but not the requested guarded positional or cyclic keyboard shortcuts.

## Solution Assessment

Unclear until maintainers choose the shortcut contract. The maintainable implementation should extend the guarded dashboard dispatcher and call switchChatSession for an active-agent-filtered ordered session list, rather than add another document listener.
