## Reproduction Assessment

## Solution Assessment

Yes for the runtime shape, but not as this stale branch's current landing vehicle. The maintainable fix is to make the callback optional while preserving `canShowReasoning` and `silentExpected`, then land it on the current embedded-agent files with tests and docs.
