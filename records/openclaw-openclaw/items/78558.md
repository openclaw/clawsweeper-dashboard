  - body: `handlePluginUpgrade` is always called with `gatewayAuthSatisfied: false`, while `createGatewayPluginUpgradeHandler` rejects gateway-auth routes unless that flag is true and auth/scope context is present.   - body: The moved Canvas snapshot path returns `imageResultFromFile(...)` without passing the configured image sanitization limits. Current main forwards those limits, so users relying on `agents.defaults.imageMaxDimensionPx` can again produce oversized provider image payloads.
  - confidence: 0.94

  - confidence: 0.92
- **[P2] Isolate hosted media resolver failures:** `src/media/web-media.ts:78`
  - body: `resolveHostedPluginMediaUrl` awaits each plugin resolver without a guard, so an unexpected throw from one optional plugin resolver rejects `loadWebMedia` for unrelated inputs. Catch and log resolver faults, then continue, so one plugin cannot break all media loading.
  - confidence: 0.82
