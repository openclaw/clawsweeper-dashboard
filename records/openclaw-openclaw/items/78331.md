## What This Changes

The PR adds a DeepSeek DSML suppress-only text filter to OpenAI-compatible streaming, with focused replay tests and a changelog entry.

## Best Possible Solution

Keep the compat-gated suppress-only design, fix safe-text buffering so normal DeepSeek chunks stream promptly, keep native `delta.tool_calls` authoritative, and attach real OpenClaw/DeepSeek output that exercises DSML suppression.

## Reproduction Assessment

## Solution Assessment

No. Suppress-only filtering gated by `compat.thinkingFormat` is the right direction, but the filter should stream safe text immediately and the PR needs DSML-exercising real proof before merge.
