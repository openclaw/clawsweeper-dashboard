## Summary

Keep open. Current main still only suppresses the reminder note for same-turn cron.add or same-session cron jobs, so the central fix is not implemented, but this PR has blocking implementation defects and no after-fix real behavior proof.

## What This Changes

The PR adds a reminder-sweep-like cron heuristic to suppress the unscheduled-reminder note and adds a runReplyAgent regression test for sweep-backed reminders.

## Best Possible Solution

Preserve same-session cron suppression and add a narrow, documented isolated agentTurn reminder-sweep path with positive and negative coverage, coordinated with the overlapping reminder-guard PRs.

## Reproduction Assessment

Yes. Current-main behavior is source-visible in the guard, caller, and regression tests, and the PR-specific regex/session-target defects are visible directly in the patch.

## Solution Assessment

No, not as-is. The branch should use real word-boundary escapes, restrict the heuristic to isolated agentTurn cron jobs, add negative coverage, and be reconciled with the overlapping reminder-guard changes.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.92

Full review comments:

- **[P1] Use real word-boundary escapes for sweep detection:** `src/auto-reply/reply/agent-runner-reminder-guard.ts:36`
  - body: The new regex contains raw U+0008 control characters around the alternation instead of JavaScript `\b` word-boundary escapes. Normal cron names/messages such as `family-reminder sweep` will not match, so the new suppression path remains inactive for the intended sweep-backed reminder case.
  - confidence: 0.97
- **[P2] Restrict sweep suppression to isolated cron jobs:** `src/auto-reply/reply/agent-runner-reminder-guard.ts:24-26`
  - body: The helper accepts any enabled `agentTurn` job whose name/message mentions reminders or sweeps. Main, current, custom, or unrelated session cron jobs could suppress the guard for a different reply, which is broader than the PR's stated isolated-cron contract.
  - confidence: 0.88
