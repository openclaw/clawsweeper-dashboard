  - body: Token rows still build `scopes: ${formatList(token.scopes)}` directly, so paired devices with role tokens keep an English prefix even after the surrounding Nodes page is translated. Route this through the new Nodes i18n keys like the paired-device access summary does.

  - body: The exec approvals target select still renders the `Gateway` option as a hardcoded English label, and the PR's own after screenshot shows it remaining English in zh-CN. Add a target label key so the main target selector is fully localized with the rest of the panel.
  - confidence: 0.86
- **[P3] Localize the allowlist never value:** `ui/src/ui/views/nodes-exec-approvals.ts:571-581`
  - body: The allowlist table uses the literal `never` when an entry has no last-used timestamp, then interpolates that into a localized label. Add a translated fallback value so unused allowlist entries do not keep an English status in non-English locales.
  - confidence: 0.82
