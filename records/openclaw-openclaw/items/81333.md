## Summary

Keep open: this is useful localization work, but the patch still leaks English in visible zh-CN Nodes paths, so it should not merge as-is.

## What This Changes

The PR replaces hardcoded Nodes and exec approvals UI strings with i18n lookups, adds English and zh-CN keys, and regenerates locale metadata/caches.

## Best Possible Solution

Finish the remaining localized wrappers for dynamic labels and sentinel values, then rerun the Control UI i18n check so zh-CN Nodes pages do not expose raw English UI text.

## Reproduction Assessment

Yes. Source inspection shows current main has hardcoded English Nodes strings, and PR-head inspection shows the remaining English leaks without needing a live UI run.

## Solution Assessment

No. The i18n extraction direction is maintainable, but the patch should reuse existing translation keys/helpers for token scopes, default enum labels, and `common.never` before merge.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Localize the token scope label:** `ui/src/ui/views/nodes.ts:199-200`
  - body: `renderTokenRow` still builds `scopes: ...` as a raw English string, so paired devices with tokens will show an English prefix in zh-CN even after this localization pass. Use the existing `nodes.scopesLabel` key or add a token-specific translation key.
  - confidence: 0.9
- **[P2] Translate default policy values:** `ui/src/ui/views/nodes-exec-approvals.ts:395`
  - body: The non-default exec approval rows now translate the surrounding text, but the interpolated defaults still use raw enum values like `allowlist`, `deny`, and `on-miss`. Pass these through `getSecurityLabel()` / `getAskLabel()` before using them in `defaultValue` and `useDefault` strings.
  - confidence: 0.88
- **[P3] Use the localized never value:** `ui/src/ui/views/nodes-exec-approvals.ts:581`
  - body: When an allowlist entry has no `lastUsedAt`, `lastUsed` is still the literal English `never`, which becomes `上次使用: never` in zh-CN. Use `t("common.never")` for that fallback value.
  - confidence: 0.86
