  - body: The allowlist remove button now calls `t("common.remove")`, but the English source map does not define `common.remove`. Because missing keys render as the raw key string, this button will show `common.remove` in every locale; add a real key or reuse an existing defined key.

  - confidence: 0.9
- **[P2] Localize token scope labels too:** `ui/src/ui/views/nodes.ts:200`
  - body: `renderTokenRow` still builds `scopes: ...` with a hardcoded English prefix after the PR localizes the surrounding token status/actions. Paired devices with tokens will still show English under zh-CN; use the existing scope translation key or add a token-specific one.
  - confidence: 0.86
- **[P2] Localize the empty last-used timestamp:** `ui/src/ui/views/nodes-exec-approvals.ts:581`
  - body: The allowlist row now localizes `Last used`, but the empty timestamp still passes the literal `never` into that template. In zh-CN this becomes mixed-language text like `上次使用: never`; use `common.never` or a dedicated allowlist key before interpolation.
  - confidence: 0.84
