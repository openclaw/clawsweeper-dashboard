## Summary

Current main still has the reported direct Anthropic request-shape bug, and PR #76871 is already open with closing syntax for this issue, so this issue should stay open until that PR is reviewed and merged or closed. ## What This Changes

The issue requests filtering unsupported sampling parameters, especially `temperature`, from Anthropic reasoning/adaptive-thinking model requests.

## Best Possible Solution
