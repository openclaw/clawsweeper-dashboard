## What This Changes

Homebrew users report that gateway auto-update records a newer package version but never installs it after the configured stable delay.

## Best Possible Solution

Land or replace the linked Homebrew-aware updater fix with consistent runtime guidance, public docs, regression tests, and real macOS/Homebrew proof.

## Reproduction Assessment

## Solution Assessment

Unclear. Skipping the doomed npm-style updater for Homebrew is plausible, but maintainers should first settle whether the supported contract is automatic brew-based apply or a documented manual `brew upgrade openclaw` path.
