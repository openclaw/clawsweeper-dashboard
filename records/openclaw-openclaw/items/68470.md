The issue asks OpenClaw to stop MiniMax prompt-cache usage from inflating context accounting and prematurely triggering timeout compaction.

## Best Possible Solution

Keep this issue open until #68750 or an equivalent narrow fix lands: normalize MiniMax cache-hit fields at the shared usage boundary, preserve the additive internal usage contract, and add focused regression tests.
