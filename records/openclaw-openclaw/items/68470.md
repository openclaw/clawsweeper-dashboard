## What This Changes

The issue requests correct MiniMax prompt-cache usage normalization so cached prompt tokens are not counted twice in context and timeout-compaction accounting.

## Best Possible Solution

Keep this issue open until #68750 or an equivalent narrow fix lands, normalizing MiniMax cache-hit usage at `src/agents/usage.ts` and adding focused coverage in `src/agents/usage.test.ts`.
