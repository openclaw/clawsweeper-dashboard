Full review comments:

- **[P1] Revert the libsignal lockfile source swap:** `pnpm-lock.yaml:4436-4437`
  - body: This PR is about structured memory, but the lockfile changes `@whiskeysockets/libsignal-node` from the existing codeload tarball to a `git@github.com` source. That unrelated supply-chain and install change can break environments without GitHub SSH access and should be reverted before this branch is mergeable.
  - confidence: 0.92
- **[P2] Declare the memory record tools in the manifest:** `extensions/structured-memory/openclaw.plugin.json:3-5`
  - body: The runtime registers `memory_record_add`, `memory_record_find`, and `memory_record_archive`, but the manifest has `activation.onStartup: false` and no `contracts.tools`. OpenClaw requires runtime-registered tools in `contracts.tools` so the loader can discover the owning plugin without importing every runtime, so these tools may not be available on demand.
  - confidence: 0.88
