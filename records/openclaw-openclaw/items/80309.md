## Best Possible Solution

Grant `actions: read` under `jobs.live_and_openwebui_checks.permissions`, keep the reusable workflow least-privilege, run `pnpm check:workflows`, and prove a caller run creates jobs before escalating to admin-only secret or runner hypotheses.
