Keep this maintainer-labeled PR open for normal review and CI; the patch is a focused macOS approval-prompt fix and current main does not already hide the durable approval button for `ask: always`. ## What This Changes

The PR changes the macOS exec approval prompt to derive its buttons from the request ask policy, omitting `Always Allow` for `ask: always`, keeping it for `on-miss`, and adding Swift layout/decision tests.

## Best Possible Solution

Land the focused macOS prompt decision change after CI or the targeted Swift test confirms the ask-always and on-miss button sets.

## Reproduction Assessment
