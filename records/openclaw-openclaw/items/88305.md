## Summary

## What This Changes

The PR rewrites Chrome MCP cached pending-session handling to use per-waiter leases with an internal abort controller and adds regression tests for abort, stale transport, timeout, and cleanup races.

## Best Possible Solution

Land the browser-plugin cache fix once a browser owner accepts the session lifecycle risk and proof depth, keeping the focused regression tests around the per-waiter abort, stale pending, timeout, and cleanup paths.
