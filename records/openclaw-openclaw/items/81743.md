## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Bind the raw placeholder through Lit:** `ui/src/ui/views/config.ts:1862`
  - body: This changes the raw-config textarea to `placeholder=t("config.raw.label")`, which Lit treats as a literal attribute value. In raw mode users will see the untranslated helper expression; bind it as `placeholder=${t(...)}` or use the existing raw placeholder key.
  - confidence: 0.94
- **[P2] Compute the local user label at render time:** `ui/src/ui/views/config-quick.ts:155`
  - body: `LOCAL_USER_LABEL` is computed with `t()` when the module loads, but non-English locale loading and later locale changes happen after that through the i18n manager. The zh-CN proof still shows `You`; compute this label during render or pass it into the avatar preview helpers so it follows the active locale.
  - confidence: 0.9
- **[P2] Use the quick-card locale keys in the rendered cards:** `ui/src/ui/views/config-quick.ts:464`
  - body: The branch adds quick-card translation keys, but the rendered UI still hard-codes visible headers and controls such as `Channels`, `Security`, `Appearance`, `Advanced`, and `Text size`. Wire those keys through `config-quick.ts` before this fixes the reported mixed-language config page.
  - confidence: 0.86
