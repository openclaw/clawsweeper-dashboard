## What This Changes

The issue reports that agent-run `skill_workshop` lifecycle apply can hit the Codex dynamic-tool watchdog before the default Skill Workshop approval wait resolves.

## Best Possible Solution

Review and land the open timeout-alignment PR or an equivalent narrow fix that lets Skill Workshop lifecycle approval resolve or fail cleanly before the Codex dynamic-tool watchdog.

## Reproduction Assessment

Yes. Source inspection shows default pending Skill Workshop lifecycle approval can wait 120s plus 10s gateway grace while Codex dynamic tools time out at 90s; I did not run a live Gateway/Codex session in this read-only review.

## Solution Assessment

Yes. The best narrow fix is to align the `skill_workshop` Codex dynamic-tool watchdog with the approval ceiling while preserving the existing pending approval policy; adding new per-action config is not needed for this bug.
