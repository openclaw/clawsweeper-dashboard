  - body: The PR allows `/tabs/open` to create an inert local `file:` tab, but listing and focusing tabs still revalidate `tab.url` without `allowLocalFileNavigation`. Under the default navigation guard, the tab this PR just allowed can then be redacted from list output or rejected when focused.
  - confidence: 0.88

  - confidence: 0.85
