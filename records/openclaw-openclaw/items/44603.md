Request to add a global `tools.exec.pty` configuration default so exec commands can use PTY mode without each tool call passing `pty: true`.

## Best Possible Solution

Have maintainers decide whether a global PTY default is acceptable; if approved, implement it through the config schema/types/help/docs plus both gateway-approval and direct runtime paths with focused tests.

## Reproduction Assessment

## Solution Assessment

Unclear until maintainer product/security review. The maintainable implementation needs the full config contract and both gateway-approval and direct runtime behavior, not only one fallback expression.
