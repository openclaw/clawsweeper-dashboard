## Summary

Keep open. Current main still lacks a gateway WebSocket protocol ping path, and this PR remains a focused fix for the reported WebChat/Control UI idle-drop behavior; it should not be closed as implemented or superseded while #74783 is still open. ## What This Changes

The PR adds a 25-second per-connection `socket.ping()` interval in the gateway WebSocket handler and clears that interval during connection teardown.

## Best Possible Solution
