- **[P1] Preserve Request policy fields when flattening:** `src/infra/net/proxy-fetch.ts:122-145`
  - body: This helper turns a `Request` into URL plus init, but it only carries body, signal, headers, and method. `Request` also carries policy fields such as `redirect`, `referrer`/`referrerPolicy`, `mode`, `credentials`, `cache`, `integrity`, and `keepalive`; after this patch a proxied `fetch(request)` can fall back to Undici defaults instead of the caller's request policy.
