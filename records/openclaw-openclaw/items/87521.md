  - body: This helper turns a `Request` into URL plus init but only carries body, signal, headers, and method.   - body: The request-like guard uses `"url" in input`, so an inherited prototype url is enough to redirect the proxy fetch and have inherited method/header/body fields considered. In this plugin-facing network egress helper, require an own url property or remove the plain-object escape hatch.
  - confidence: 0.78
