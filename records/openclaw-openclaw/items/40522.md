## Best Possible Solution

## Solution Assessment

Mostly yes: the classifier-level redirect is a narrow fit and the proposed tests cover GET, query preservation, and POST fallthrough. It still needs maintainer acceptance of the root-path compatibility tradeoff plus a rebase against current routing precedence.
