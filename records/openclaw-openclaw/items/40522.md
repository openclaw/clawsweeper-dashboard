## Best Possible Solution

Merge the focused classifier-owned root-to-basePath redirect after the branch is refreshed if needed and redacted real gateway proof demonstrates the redirect on a running gateway.

## Reproduction Assessment

## Solution Assessment

Yes for the basePath-specific bug. The classifier is the narrow maintainable fix point because redirect responses are already centralized in `handleControlUiHttpRequest`, plugin routes run earlier, and non-read root requests keep falling through.
