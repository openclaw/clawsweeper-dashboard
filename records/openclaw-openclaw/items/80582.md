## What This Changes

Reject markdown skeleton placeholder snippets during short-term memory promotion so empty bullets and heading-only ranges cannot become durable `MEMORY.md` entries.

## Best Possible Solution

Review and land https://github.com/openclaw/openclaw/pull/82028 or an equivalent guard across recording, store load, ranking, relocation, apply, and rehydration, then close this issue after the fix reaches main.

## Solution Assessment

Yes. The narrow maintainable fix is a shared unpromotable-snippet predicate at every promotion boundary plus meaningful-content gating for relocation; the open closing PR follows that direction.
