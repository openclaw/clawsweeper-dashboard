## Summary

Keep open: current main still lacks placeholder-only filtering in short-term promotion, and the reporter added sanitized field evidence that this polluted MEMORY.md with 89 malformed promoted-placeholder blocks.

## What This Changes

The issue requests filtering markdown placeholder-only snippets out of short-term memory promotion recording, ranking, apply, and relocation.

## Best Possible Solution

Add a shared unpromotable-snippet predicate and enforce it consistently across store normalization, recall/grounded recording, ranking, apply/rehydration, and relocation while preserving ordinary non-empty bullet content.

## Reproduction Assessment

Yes. A '-' snippet is non-empty after normalization, passes the current empty/dreaming-contamination gates, can accumulate enough stored signals to rank, and can rehydrate through exact or substring relocation; the reporter also supplied a sanitized cleanup count from a real workspace.

## Solution Assessment

Yes. The narrow maintainable fix is to centralize placeholder filtering near the existing snippet filters and add a meaningful-content guard to relocation rather than relying on downstream MEMORY.md cleanup.

Fix https://github.com/openclaw/openclaw/issues/80582 in memory-core. Add a shared predicate, near the existing snippet normalization/contamination helpers, that rejects short-term promotion snippets with no durable semantic content: bare list markers, marker-only markdown fragments, fence/quote/table-marker-only fragments, and heading plus empty-bullet skeletons. Use it in normalizeStore/load filtering, recordShortTermRecalls, recordGroundedShortTermCandidates, rankShortTermPromotionCandidates, apply/rehydration, and relocation matching. Relocation must not exact-match or substring-match placeholder-only snippets, and substring matching should require meaningful content; preserve ordinary short bullets that contain letters or numbers. Add focused regression coverage in extensions/memory-core/src/short-term-promotion.test.ts for record, already-stored rank/apply, relocation, and preservation of real bullet text.
