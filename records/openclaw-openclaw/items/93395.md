## Best Possible Solution

Land the additive compact list path after real behavior proof shows `cron.list` and `cron(action: "list")` return compact summaries while the default full response remains unchanged.

No high-confidence live reproduction was established in this review. Source inspection does show current main always returns full cron jobs plus delivery previews and cannot accept `compact`, matching the linked issue's reported failure mode.

## Solution Assessment
