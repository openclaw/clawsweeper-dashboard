The issue requests side-effect-free dry-run/test and validation support for cron jobs and skills, including CLI commands, CI-friendly JSON output, and a mock/read-only tool layer.

Design an additive core implementation: cron test/validate RPC plus CLI, a skills test runner with explicit input and CI output semantics, and a shared side-effect interception contract coordinated with https://github.com/openclaw/openclaw/issues/41418.
