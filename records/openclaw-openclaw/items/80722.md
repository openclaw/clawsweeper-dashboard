## Best Possible Solution

Land a narrow CLI hint-selection fix that uses post-mutation config facts, emits a next-invocation hint only for positively hot-loadable agent mutations, and keeps the restart hint for non-agent, mixed, unknown, or runtime-structure-changing edits.
