  - body: `channel.ts` is a hot channel entrypoint, but this import pulls in `agent-tools-location.ts`, which imports `./send.js` at module load time. Current main keeps the full send runtime behind `send-runtime.ts`; move the send import behind tool execution or use the existing lazy loader so discovery does not eagerly load the delivery runtime.
  - confidence: 0.88
- **[P2] Validate horizontal accuracy before sending:** `extensions/telegram/src/send.ts:2065-2067`
  - body: `horizontalAccuracy` is advertised as 0-1500 and Telegram's Bot API uses that range, but the runtime forwards any parsed number after validating only latitude and longitude. Add the same runtime guard so CLI/tool calls fail locally with a clear error instead of sending an invalid Bot API request.
  - confidence: 0.82
- **[P2] Do not advertise unreadable local video-note paths:** `extensions/telegram/src/agent-tools-location.ts:105-113`

Either carry supported media access through this seam or narrow the tool schema/description to URLs only.
  - confidence: 0.78
