Full review comments:

- **[P2] Use the scheduler's stable stagger offset:** `ui/src/ui/helpers/cron-timeline.ts:384-388`
  - body: The timeline shifts every staggered cron marker by `staggerMs / 2`, but runtime scheduling uses a stable per-job hash offset modulo `staggerMs`.   - confidence: 0.88

  - confidence: 0.83
