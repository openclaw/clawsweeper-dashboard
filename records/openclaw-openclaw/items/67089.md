  - confidence: 0.96

  - body: Current main already authorizes gateway methods through the method-scope registry before handler dispatch, and config writes are admin-scoped there.   - confidence: 0.91

  - body: The signature write is swallowed, but read-side enforcement treats a missing signature as suspicious and can reject the config later. A config write that cannot persist its `.sig` must fail or surface a clear non-enforcing state instead of creating a latent lockout.
