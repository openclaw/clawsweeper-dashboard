Add a hostname segment to the OpenClaw TUI status bar so users can distinguish sessions across multiple hosts.

Add `host <hostname>` to the TUI footer for URL-backed gateway connections, handle local embedded mode deliberately, and update focused TUI tests, docs, and changelog without adding protocol or config surface.
