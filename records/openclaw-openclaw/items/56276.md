Add a hostname segment to the TUI status bar/footer so users can tell which OpenClaw host they are using across multiple machines.

## Best Possible Solution

Add a small TUI formatter that derives a compact host label from `TuiBackend.connection.url`, inserts it into the footer for URL-backed sessions, handles local embedded mode deliberately, and updates focused tests, docs, and changelog.
