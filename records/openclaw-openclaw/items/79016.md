  - confidence: 0.86

  - body: This key is built after `formatCliCommand()`, which can inject `--profile` or `--container` flags from env; then it no longer matches the locale-map entry and zh users see the English hint. Translate a stable template and interpolate the formatted command.
  - confidence: 0.85
- **[P2] Wrap the password secret-mode option copy:** `src/wizard/setup.gateway-config.ts:249-251`
  - body: Password auth translates the mode question but leaves the plaintext selector label and hint in English, so Chinese users still get mixed-language options in this setup path. Wrap both strings and add matching locale entries.
  - confidence: 0.92
