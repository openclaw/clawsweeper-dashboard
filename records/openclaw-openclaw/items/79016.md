Full review comments:

- **[P2] Handle script-tagged Traditional Chinese locales:** `src/wizard/i18n/index.ts:7-8`
  - body: The detector only treats exact `zh-TW`/`zh-HK`/`zh-MO` as Traditional Chinese, then sends any other `zh*` locale to zh-CN. Script-tagged locales such as `zh-Hant-TW` would therefore get Simplified Chinese despite the advertised zh-TW/HK/MO support; check script and region subtags before the generic zh fallback.
  - confidence: 0.86
- **[P2] Translate the quickstart hint from a stable template:** `src/wizard/setup.ts:237`
  - body: This calls `t()` after `formatCliCommand()` has produced the full command text. When the formatter injects `--profile` or `--container`, the resulting English string no longer matches the locale-map key, so zh users see the English hint; translate a stable template and interpolate the formatted command.
  - confidence: 0.88
- **[P2] Wrap password secret-mode labels and hints:** `src/wizard/setup.gateway-config.ts:249-251`
  - body: Password auth localizes the mode question but still passes English plaintext option copy into the secret-mode selector. Chinese users in this path still see mixed-language choices, so wrap these strings and add matching locale entries.
  - confidence: 0.92
- **[P3] Translate optional-plugin skip hints:** `src/wizard/setup.official-plugins.ts:106`
  - body: The optional plugin selector translates the skip label but leaves its visible hint in English. Wrap the hint and add locale entries so this setup menu is not mixed-language under zh locales.
  - confidence: 0.84
- **[P3] Translate plugin-config skip hints:** `src/wizard/setup.plugin-config.ts:340`
  - body: The plugin configuration selector has the same mixed-language skip option: the label is localized but the hint remains English. Wrap the hint and cover it in both locale maps.
  - confidence: 0.84

  - body: This PR adds a PR-evidence screenshot under `docs/reference`, but PR-only artifacts should be attached to the PR/comment or an external artifact store, not committed into the docs tree.
