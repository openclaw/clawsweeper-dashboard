Full review comments:

- **[P2] Handle script-tagged Traditional Chinese locales:** `src/wizard/i18n/index.ts:7-8`
  - body: The detector checks only exact `zh-TW`/`zh-HK`/`zh-MO` before falling through to `locale.startsWith("zh")`, so a resolved locale like `zh-Hant-TW` would select the zh-CN map even though the PR advertises Traditional Chinese coverage. Check script/region subtags before the generic zh fallback.
  - confidence: 0.86
- **[P2] Translate the quickstart hint with a stable template:** `src/wizard/setup.ts:237`
  - body: This key is built after `formatCliCommand()`, which can inject `--profile` or `--container` flags from env; then it no longer matches the locale-map entry and zh users see the English hint. Translate a stable template and interpolate the formatted command.
  - confidence: 0.83
- **[P2] Wrap the password secret-mode option copy:** `src/wizard/setup.gateway-config.ts:250-251`
  - body: Password auth translates the mode question but leaves the selector label and hint in English, so Chinese users still get mixed-language options in this setup path. Wrap `plaintextLabel` and `plaintextHint` and add matching locale entries.
  - confidence: 0.91
- **[P3] Translate the skip-option hints:** `src/wizard/setup.official-plugins.ts:105-106`
  - body: The PR wraps the skip labels but leaves their hints in English, so the plugin setup selectors remain mixed-language under zh locales. Wrap the hints in the localized files as well.
  - confidence: 0.84
