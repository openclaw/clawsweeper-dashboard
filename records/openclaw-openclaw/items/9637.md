## What This Changes

## Best Possible Solution

Ship a documented core TUI plain-text mode that is read once from config, validated in schema/generated docs, and consistently applied across tool summaries, selectors, settings cursors, waiting/status text, and relevant component glyphs.

## Solution Assessment
