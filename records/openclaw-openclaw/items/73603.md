## Summary

Keep open. Current main still has in-process plan events, subagent run/session tracking, trajectory sidecars, and plugin event subscriptions, but not the requested default-off durable `/var/log/openclaw-events.jsonl` plan/session audit stream with stable plan identifiers and spawn/yield/terminal correlation.

## What This Changes

Request a default-off `events.emit.plan` config that writes plan/session lifecycle records to `/var/log/openclaw-events.jsonl` for external task-completion auditing.

## Best Possible Solution

Define an additive experimental observability contract with stable plan/run identifiers, open/update/spawn/yield/end records, redaction guarantees, file permission and retention rules, config/docs coverage, and focused disabled/enabled tests.

## Reproduction Assessment

Yes. Static inspection of current main reproduces the gap: the requested config key and JSONL sink are absent, plan events are update-only, and subagent lifecycle records are run-scoped rather than plan-scoped.

## Solution Assessment

Unclear. A default-off JSONL stream is plausible, but the maintainable fix needs maintainer decisions on schema stability, privacy, sink ownership, and whether to build on trajectory capture or plugin event APIs.
