## Summary

Keep open. Current main still has in-process plan and lifecycle events, trajectory sidecars, and plugin event subscriptions, but not the requested default-off durable plan/session audit stream with stable plan identifiers, spawn/yield correlation, and terminal close/abort records.

## What This Changes

The issue requests a default-off `events.emit.plan` config flag that writes plan/session lifecycle audit events to a canonical JSONL sink.

## Best Possible Solution

Define an additive experimental observability contract with stable plan/run identifiers, open/update/spawn/yield/end records, redaction guarantees, retention/permission rules, config/docs coverage, and focused tests for disabled and enabled behavior.

## Reproduction Assessment

Yes. Static inspection of current main reproduces the gap: the requested config key and canonical JSONL sink are absent, plan events are update-only, and session/subagent lifecycle tracking is run-scoped rather than plan-scoped.

## Solution Assessment

Unclear. A default-off JSONL stream is plausible, but the maintainable fix needs maintainer decisions on schema stability, privacy, sink ownership, and whether to build on trajectory capture or plugin event APIs.
