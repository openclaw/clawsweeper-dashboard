Request caching memory-core indexed-content existence state so repeated memory searches avoid redundant SQLite `chunks` and FTS probes.

## Best Possible Solution

## Solution Assessment

Yes, with the linked PR refreshed or replaced. A memory-core-local cache is the narrow maintainable approach if it is invalidated across writes, deletes, resets, safe reindex, and readonly-recovery DB replacement.
