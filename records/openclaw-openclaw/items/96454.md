## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Detach attach signal handlers on child completion:** `src/cli/attach-cli.ts:175-176`
  - body: The attach command registers process-level SIGINT/SIGTERM handlers for every spawned harness but never removes them when the child exits or errors. OpenClaw's child-process guidance and `attachChildProcessBridge` detach signal listeners on child exit/error; leaving these handlers installed can swallow later SIGINTs or route SIGTERM to a stale child in in-process callers/tests. Use the shared bridge or equivalent detach-on-exit/error logic for this terminal path.
  - confidence: 0.84
