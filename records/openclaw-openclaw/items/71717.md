## What This Changes

The issue reports that on Windows 11, every agent exec-tool command fails immediately with `spawn EPERM` while the OpenClaw CLI itself still runs.

## Best Possible Solution

## Solution Assessment

Unclear. No patch is proposed, and treating all `EPERM` spawn failures as retryable would be risky because `EPERM` can be a real permission denial; native evidence should decide between a constrained fallback and a diagnostic improvement.
