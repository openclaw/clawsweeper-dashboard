## Summary

## What This Changes

The PR gates the Control UI chat In progress badge and Stop button behind `sending || stream !== null` and adds focused chat view regression tests for stale abortability.

## Best Possible Solution

Preserve the completed-state stale-session fix while deriving visible abort UI from an authoritative local active-run signal, not from stream text alone.
