## What This Changes

This PR adds an embedded-runner dynamic temperature heuristic for tool-capable agent runs, along with tests, changelog and lockfile updates, and file-secret allowInsecurePath plumbing.

## Best Possible Solution

If maintainers accept the sampling policy, land a focused agents-only implementation that uses actual provider thinking state, covers normal and compaction tool-capable runs, preserves cache semantics, drops secrets and lockfile churn, and includes real provider proof.
