- **[P2] Require a real group route before skipping the agent prefix:** `src/config/sessions/group.ts:75`
  - body: With this change, any agent-scoped main key whose third segment is a channel surface is parsed as a provider-prefixed group route. Only skip `agent:<id>:` when the remaining key has an explicit `group` or `channel` route segment.
