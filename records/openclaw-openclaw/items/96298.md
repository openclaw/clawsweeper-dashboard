## Summary

## What This Changes

This PR updates the Control UI `formatTokens` helper and its tests so near-million token counts that round to `1000k` fall through to the `M` formatter.

## Best Possible Solution

Land the narrow `formatTokens`/Overview fix with its regression test, and either clarify the Usage-tab scope or handle `usage-metrics.ts` in a follow-up if maintainers want all UI token formatters aligned.

## Reproduction Assessment

Yes, source-reproducible: current `formatTokens` checks the raw value before rounding thousands, so 999,500 through 999,999 can produce `1000k`. I did not run tests because this was a read-only review.

## Solution Assessment

Yes for the changed helper: falling through after the rounded kilo value reaches 1000 matches the existing sibling pattern without changing lower values. A shared formatter could reduce duplication later, but it is not required for this XS bug fix.
