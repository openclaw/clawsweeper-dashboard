## Summary

Keep open: this is a feature/design request, current main still lacks an explicit subagent final-summary ownership contract, and an open linked implementation PR with closing syntax is the right place to review the proposed fix before closing this issue.

## What This Changes

The issue requests a first-class final-summary ownership contract for detached or migrated long-running subagent work so only one surface publishes the full final result.

## Best Possible Solution

Review and either land or close https://github.com/openclaw/openclaw/pull/80544; the desired end state is a native subagent delivery contract that persists one full-final owner while allowing non-owner pointer/status notes.
