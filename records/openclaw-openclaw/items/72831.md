  - confidence: 0.85
- **[P3] Point setup at Settings > General:** `skills/obsidian/SKILL.md:30`
  - body: Official Obsidian CLI setup is under Settings > General, where users enable Command line interface and follow the registration prompt. This text points first-time users to a separate Command-line interface settings page, which does not match the current docs.
  - confidence: 0.87
- **[P3] Keep link updates conditional:** `skills/obsidian/SKILL.md:107`
  - body: Official Obsidian CLI docs say `move` and `rename` update internal links only when the vault's automatic link-update setting is enabled. This sentence guarantees link repair unconditionally, which can make users trust a refactor Obsidian may not apply.
  - confidence: 0.84
