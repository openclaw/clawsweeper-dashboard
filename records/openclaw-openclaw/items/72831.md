  - confidence: 0.86

  - body: Official Obsidian CLI setup is under Settings > General, where users enable Command line interface and follow the registration prompt.   - confidence: 0.84

  - body: Official docs say `move` and `rename` update internal links only when the vault's automatic link-update setting is enabled.
