## What This Changes

Prevent sub-agent timeout recovery from re-posting the same logical external/API side effect after a successful request loses its acknowledgement.

## Best Possible Solution

Define a maintainer-approved logical-action idempotency contract for OpenClaw-owned sub-agent side effects, then implement it in the narrow owner path with regression coverage and real retry/recovery proof.
