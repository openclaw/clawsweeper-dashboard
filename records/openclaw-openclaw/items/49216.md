## What This Changes

The PR changes enforced session maintenance to sweep expired deleted/reset session archive artifacts from the store directory and expands archive timestamp parsing for legacy filename formats.

## Best Possible Solution

Land this focused storage fix after adding redacted real behavior proof and tightening the regression test to exercise `resetArchiveRetention: false`.

## Reproduction Assessment

Yes, by source inspection. ## Solution Assessment

Yes for the code direction: sweeping the store directory for deleted archives during enforced maintenance and broadening only archive timestamp parsing is the narrow maintainable fix. The test should be tightened to cover the disabled reset-retention path before merge.
