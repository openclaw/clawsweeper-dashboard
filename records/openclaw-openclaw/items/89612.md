## What This Changes

## Best Possible Solution

Land the resolver-level persisted-aware lookup fix with the existing regression and real behavior proof, then close the linked issue after it reaches main.

## Reproduction Assessment

Yes. Source inspection on current main shows the standalone migrate path calls a resolver that disables persisted reads, and the linked issue includes Linux and macOS packaged-install reproductions.

## Solution Assessment

Yes. Removing the generic manifest-contract opt-out is the narrowest maintainable fix because the only production caller is migration-provider discovery and persisted-index freshness is already handled by the registry snapshot loader.
