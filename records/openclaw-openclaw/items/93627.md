## What This Changes

## Best Possible Solution

Repair the recall/emission boundary so unchanged stripped daily content cannot refresh light freshness across dreaming days, add a cross-day regression test, and require redacted repeated-cycle DREAMS.md proof before merge.

## Solution Assessment

No. The phase-signal filter is a plausible part of the fix, but the same-day-only stripped-hash skip leaves the linked two-night path unresolved; the durable fix should track a real emission boundary or prevent deduped daily ingestion from advancing light freshness.
