## Summary

Keep open. Current main still exposes sessions through session lists, pickers, reset/create operations, labels, search, and per-session overrides, but it does not provide the requested blind-user-focused linear persistent workspace with timeline, bookmark, jump, or task-section navigation.

## What This Changes

Add a blind-user-friendly linear persistent workspace mode that reduces context fragmentation across multiple chats and sessions.

## Best Possible Solution

Define a first-class accessibility workflow that preserves session isolation while adding a linear timeline, task sections, jump/bookmark navigation, and screen-reader-friendly session recovery across the primary clients.
