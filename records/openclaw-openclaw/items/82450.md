## Summary

Keep open. Current main has session creation, session pickers, session lists, compaction checkpoints, and session recovery affordances, but it still does not provide the requested first-class linear persistent workspace workflow for blind users; this needs accessibility and product design rather than conservative cleanup.

## What This Changes

The issue requests an accessibility-first linear persistent workspace/chat mode with task sections, jump navigation, a screen-reader-friendly session index, and easier recovery of older work without fragmented session switching.

## Best Possible Solution

Design an accessibility-first linear workspace workflow that preserves session isolation while adding screen-reader-friendly timeline, index, jump, task-section, and context recovery behavior in the primary clients.
