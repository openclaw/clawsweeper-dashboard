## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Yes. Removing the compat-only normalizer is the narrow best fix because native object conversion remains in `convertToOllamaMessages` and Ollama response parsing while OpenAI-compatible replay keeps the dependency-required string form.
