## Summary

## What This Changes

## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Mostly yes: applying the existing provider JSON reader at the plugin-owned response reads is the narrow maintainable code fix. It is not merge-ready until proof is live or overridden and maintainers accept the cap policy.
