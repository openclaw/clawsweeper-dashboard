## Summary

Keep this open: current main and v2026.6.8 still expose lower-level agent delivery, session messaging, A2A reply-loop, and transcript-inspection primitives, but not the requested meeting lifecycle, dial/pickup/send/wrap verbs, durable meeting artifact, visible meeting frames, or meeting transcript command. Related maintainer discussion treats this issue as the scoped lifecycle RFC rather than a stale duplicate, and the remaining work needs product/API, storage, and privacy decisions before implementation.

## What This Changes

The issue requests a scoped agent teleconference primitive with dial/pickup/send/wrap lifecycle verbs, visible channel frames, durable meeting artifacts, and transcripts.

## Best Possible Solution

Keep this as the scoped meeting-lifecycle RFC unless maintainers intentionally fold it into the broader multi-agent collaboration RFC; any implementation should first settle storage, transcript privacy, handoff layering, and config/API shape.
