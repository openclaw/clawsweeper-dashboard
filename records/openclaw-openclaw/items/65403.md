## Summary

Keep this open: current main and the latest release still have the lower-level `openclaw agent --deliver` and `sessions_send` agent-to-agent paths, but not the requested dial/pickup/send/wrap lifecycle, durable meeting artifact, visible meeting frames, or transcript command surface. ## What This Changes

The issue requests a new agent teleconference primitive with CLI verbs, meeting lifecycle state, visible channel frames, durable transcripts, and optional agent icon configuration.

## Best Possible Solution
