  - confidence: 0.9

Since `buildPluginToolDescriptorCacheKey` uses this string to reuse plugin tool descriptors, distinct plugin contexts can receive stale descriptors; mirror the type-tagged cache-key format and add a collision regression.
  - confidence: 0.88
