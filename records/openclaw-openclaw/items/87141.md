  - body: The new walker accepts any non-array object it can enumerate.   - confidence: 0.87
- **[P2] Type-tag descriptor context cache values:** `src/plugins/tool-descriptor-cache.ts:96-97`
  - body: `stableDescriptorCacheValueKey()` serializes bigint values as `JSON.stringify(value.toString())`, while strings are also serialized with `JSON.stringify(value)`, so `1n` and `"1"` produce the same descriptor context fragment. A plugin whose descriptors vary by context metadata can reuse the wrong cached descriptor after this hardening; include type tags and add a string-vs-bigint regression test.
  - confidence: 0.9
