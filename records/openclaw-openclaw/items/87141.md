- **[P1] Reject non-plain JSON values in the compatibility walker:** `src/shared/json-compatible.ts:32-34`

  - confidence: 0.9
- **[P1] Type-tag descriptor context cache values:** `src/plugins/tool-descriptor-cache.ts:96-98`
  - body: The descriptor context encoder still serializes `bigint` by `value.toString()` and returns the same fragment as a string with that text, so context metadata `{ route: "1" }` and `{ route: 1n }` can share cached tool descriptors. Use the type-tagged cache-key encoder or add scalar type prefixes here, with a regression mirroring the provider runtime cache-key test.
  - confidence: 0.9
