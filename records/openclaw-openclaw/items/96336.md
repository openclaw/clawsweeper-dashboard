Full review comments:

- **[P1] Preserve project settings directory permissions:** `src/agents/sessions/settings-manager.ts:217-222`
  - body: `FileSettingsStorage.withLock()` handles both global settings and project settings, and project settings live under `cwd/.openclaw/settings.json`. Adding 0o700 plus chmod here can owner-lock a shared workspace settings directory outside OpenClaw-managed private state.
  - confidence: 0.9
- **[P1] Preserve explicit export destination permissions:** `src/agents/sessions/agent-session.ts:3197`
  - body: `exportToJsonl(outputPath)` accepts an arbitrary caller-provided output path. Creating that parent as 0o700 changes user export directory permissions, despite the PR scope saying export targets stay preserved.
  - confidence: 0.9
- **[P1] Preserve custom session.store parent permissions:** `src/config/sessions/session-accessor.ts:1026-1030`
  - body: This transcript parent is derived from the configured `session.store` path, which is documented for custom disk layout, backup routing, or mounted volumes. The added chmod can change an existing custom store parent whenever a new transcript is created.
  - confidence: 0.88
