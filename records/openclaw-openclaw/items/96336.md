## Summary

## What This Changes

The PR adds `0o700` directory creation and best-effort chmod repair across session, settings, tool, gateway transcript, and plugin-skill directory paths.

## Best Possible Solution

Narrow private-mode hardening to OpenClaw-managed state directories, preserve caller-owned and custom paths, and add focused tests plus final-head umask proof.
