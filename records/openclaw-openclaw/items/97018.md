Full review comments:

- **[P2] Apply provider contextTokens before model contextWindow:** `extensions/lmstudio/src/embedding-provider.ts:81-83`
  - body: This returns `modelConfig.contextWindow` before checking `providerConfig.contextTokens`. For a provider config with `contextTokens: 4096` and a model entry that only has `contextWindow: 32768`, embedding preload will still request 32768 even though `contextTokens` is the effective runtime cap when the model entry omits it. Please prefer `model.contextTokens`, then `provider.contextTokens`, then `model.contextWindow`, then `provider.contextWindow`, and cover that combination in the test.
  - confidence: 0.86
