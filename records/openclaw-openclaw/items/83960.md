The issue reports that `openclaw models auth login --provider xai --method oauth` fails immediately with an xAI “Unknown scope” error before browser redirect or credential storage.

## Best Possible Solution

Validate the browser OAuth flow with an eligible xAI account, then update the xAI plugin-owned OAuth scope/client handling and regression tests if the provider rejects the shipped request.
