## What This Changes

Add a WhatsApp listen-only/hooks-only inbound mode that emits `message_received` hooks for selected accounts or routes without agent runs, model calls, replies, typing, reactions, read receipts, or pairing messages.

## Best Possible Solution

Design an explicit WhatsApp account/route ingestion mode in the WhatsApp plugin that admits configured DMs/groups to hook emission, bypasses agent dispatch and all outbound side effects, and documents validation/status semantics.
