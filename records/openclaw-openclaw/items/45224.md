The issue asks OpenClaw to prevent Playwright CDP assertion failures from crashing the Gateway and losing unrelated in-flight sessions.

## Best Possible Solution

Unclear. A targeted browser-owned CDP assertion boundary with reconnect/degrade behavior looks safer than broad global suppression, while subprocess isolation needs explicit maintainer architecture approval.
