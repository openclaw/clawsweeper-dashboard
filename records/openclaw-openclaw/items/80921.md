## What This Changes

## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Yes for the fix direction, but not for the current landing state. A bounded generic bind-probe timeout is narrower than a macOS-only skip because it preserves working IPv6 loopback binding, but the branch still needs rebase, proof, and focused coverage.
