Keep open. Current `main` still has the Feishu descendant payload bug in the `write`/`append` path, so the central runtime failure is not fully implemented on main even though the conversion scope is documented in the bundled Feishu doc skill.

Fix `feishu_doc` write and append so valid markdown content can be converted and inserted without Feishu 400 responses.
