## What This Changes

The branch adds an optional `probe` argument to `probeGatewayMemoryStatus`, forwards `doctor --deep` into `doctor.memory.status`, and suppresses the local-provider warning when the gateway intentionally skipped the memory probe.

## Best Possible Solution

Land one consolidated fix that forwards `--deep`, treats intentional local skipped probes as neutral, preserves timeout/not-ready warnings, and includes focused regression tests plus redacted CLI proof.
