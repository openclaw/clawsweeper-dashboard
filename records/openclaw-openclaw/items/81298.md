## Summary

Keep open. The PR is a focused fix for a real completion-profile bug, but the current resolver can still choose a file the shell will not read in override/no-profile cases, so it should not merge as-is.

## What This Changes

The PR centralizes shell completion profile-path resolution for zsh, bash, and fish, updates install/status helpers to share it, and adds profile-path regression tests.

## Best Possible Solution

Land the shared resolver after separating shell-loaded override targets from fallback candidates and preserving bash .bash_profile creation behavior, with focused regression tests for those cases.

## Reproduction Assessment

Yes. Source inspection gives a high-confidence path: current main uses HOME-backed zsh/fish profiles and a bash check/install mismatch, and the linked issue gives concrete ZDOTDIR, XDG_CONFIG_HOME, and bash fallback scenarios.

## Solution Assessment

No, not yet. A shared resolver is the right direction, but this implementation still picks the wrong target when override-specific files are absent and when bash has no existing profile file.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.89

Full review comments:

- **[P2] Honor override profile directories before HOME fallbacks:** `src/cli/completion-runtime.ts:199-201`
  - body: When `ZDOTDIR` or `XDG_CONFIG_HOME` is set but the override-specific file does not exist yet, this loop can return an existing HOME profile instead. zsh uses HOME only when ZDOTDIR is unset, and fish's user config is under XDG_CONFIG_HOME when set, so this can keep reporting completion installed in a stale file the shell will not read.
  - confidence: 0.86
- **[P2] Preserve bash_profile fallback when creating bash profiles:** `src/cli/completion-runtime.ts:204`
  - body: When no bash profile file exists, `getShellProfilePath()` returns `candidates[0]`, which is `~/.bashrc`. Current install behavior creates `~/.bash_profile` when `~/.bashrc` is absent, so this changes macOS login-bash installs into a file that may not be sourced.
  - confidence: 0.91
