## Summary

Keep open. This is a plausible focused sandbox path fix, but the current patch should not merge as-is because it bypasses the repo’s shared normalized OS-home helper and still has no after-fix real behavior proof from the reported systemd/sub-agent setup.

## What This Changes

The PR adds a local sandbox home resolver that prefers HOME/USERPROFILE over os.homedir() for tilde expansion and sandbox error path shortening.

## Best Possible Solution

Reuse the shared OS-home helper contract for sandbox path resolution and display shortening, add focused sandbox regression coverage, attach redacted real behavior proof, and leave broader OPENCLAW_HOME semantics to https://github.com/openclaw/openclaw/issues/54014.

## Reproduction Assessment

## Solution Assessment
