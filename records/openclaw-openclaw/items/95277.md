## Summary

## What This Changes

The PR changes the Amazon Bedrock plugin and tests so direct `amazon.nova-*` models bypass the no-cache wrapper and receive Converse cache points via `onPayload`.

## Best Possible Solution

Land provider-owned Nova prompt caching that normalizes supported direct, regional, and global Nova IDs, preserves explicit retention semantics including `none`, keeps unsupported Bedrock families no-cache, updates docs/tests, and includes redacted live AWS proof.
