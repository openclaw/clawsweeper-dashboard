## Best Possible Solution

Land provider-owned Nova prompt caching that normalizes direct, regional, and global Nova IDs, preserves explicit disable/retention semantics according to the AWS contract, keeps unsupported Bedrock families no-cache, updates docs/tests, and includes redacted live Bedrock proof.
