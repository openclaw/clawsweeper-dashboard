Request a machine-readable gateway diagnostic/stability event when stale exec approval-timeout follow-ups are intentionally suppressed.

## Best Possible Solution

Land one narrow, sanitized diagnostic event implementation covering both direct and gateway preflight suppression paths plus stability/readback, then close this issue.
