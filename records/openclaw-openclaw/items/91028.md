## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Strip gateway credentials from embedded env:** `extensions/lobster/src/lobster-runner.ts:286-288`
  - body: `createEmbeddedToolContext` still starts with a copy of `process.env`, so a gateway launched with `OPENCLAW_URL`, `OPENCLAW_TOKEN`, `CLAWD_URL`, or `CLAWD_TOKEN` will pass those values to Lobster shell steps. Since this PR's security model is to avoid exposing gateway auth, delete those keys from `ctx.env` and seed them in the regression test before asserting they are absent.
  - confidence: 0.93
- **[P1] Do not silently shadow the HTTP openclaw provider:** `extensions/lobster/src/lobster-tool.ts:308-310`
  - body: `@clawdbot/lobster` resolves direct adapters before its built-in providers, so registering `llmAdapters.openclaw` makes existing embedded workflows that intentionally configured `OPENCLAW_URL`/token use this new in-process path instead. Use a distinct provider id or add an explicit compatibility/upgrade path before changing `provider=openclaw` semantics.
  - confidence: 0.88
- **[P2] Preserve bare model resolution for llm.invoke:** `extensions/lobster/src/lobster-tool.ts:242-243`
  - body: The dependency passes the CLI `--provider` outside the payload and leaves `payload.model` as the bare `--model` value; examples like `llm.invoke --provider openclaw --model claude-3-sonnet` therefore reach this adapter with no slash in `payload.model`. This code treats that as an unresolved provider/model instead of using the selected/default provider and OpenClaw alias resolution like `llm-task` does.
  - confidence: 0.9
- **[P2] Clean up embedded LLM session files:** `extensions/lobster/src/lobster-tool.ts:261-264`
  - body: The adapter writes each embedded-agent session file directly under `os.tmpdir()` and never removes it. `runEmbeddedAgent` persists prompt/transcript data there, so use the same cleaned temp-workspace pattern as `llm-task` to avoid orphaned workflow inputs and model output after each run.
  - confidence: 0.82
