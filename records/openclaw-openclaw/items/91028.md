## Summary

This PR adds host-provided Lobster embedded runtime bridges, registers an in-process `openclaw` LLM adapter backed by `runEmbeddedAgent`, and adds runner tests for adapter injection and credential stripping.

## Best Possible Solution

Land a maintainer-approved in-process adapter that keeps Gateway credentials out of the final Lobster runtime env, preserves `llm-task` auth/model policy, chooses and documents the provider name, and includes redacted live embedded invocation proof.
