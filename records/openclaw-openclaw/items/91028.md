## Summary

Keep open. ## What This Changes

The PR adds Lobster embedded runtime bridges, registers an in-process `openclaw` `llm.invoke` adapter backed by `runEmbeddedAgent`, and adds unit tests for bridge injection and credential stripping.

## Best Possible Solution

Keep the bridge work, but land it only after the provider name/contract is maintainer-approved, the real runtime env is sanitized through Lobster's final context path, llm-task policy is preserved or deliberately replaced, and a live redacted model invocation proves the path.
