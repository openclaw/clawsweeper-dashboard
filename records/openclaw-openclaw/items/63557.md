Implement one focused gateway audit feature on current main that records redacted, outcome-aware allowed, denied, and failed gateway tool attempts from the shared direct-invoke helper and the embedded OpenResponses tool wrapper, while leaving query UI, retention, rotation, and signing for separate work.
