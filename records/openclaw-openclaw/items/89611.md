## Best Possible Solution

Land the focused guard and regression after maintainer review and draft resolution, keeping legacy `action` merging best-effort and non-fatal.

## Reproduction Assessment

## Solution Assessment

Yes. Guarding only the optional schema-inspection helper is the narrowest maintainable fix because unreadable schemas should be treated as not supporting the legacy `action` shortcut while normal action injection remains unchanged.
