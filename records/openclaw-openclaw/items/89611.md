## What This Changes

The PR wraps the Gateway `/tools/invoke` legacy action schema probe in a safe helper and adds an HTTP regression test for a throwing schema `properties` getter.

## Best Possible Solution

Land the focused guard and regression after draft resolution and maintainer review, keeping legacy HTTP `action` probing best-effort and non-fatal.

## Reproduction Assessment

## Solution Assessment

Yes. Guarding only the optional schema-inspection helper is the narrowest maintainable fix; changing the tool contract or swallowing all invoke errors would be broader and less precise.
