## What This Changes

The issue reports that MiniMax Token Plan media and office-generation workflows no longer work reliably in sub-agents after v5.22 limited sub-agent bootstrap context to `AGENTS.md` and `TOOLS.md`.

## Best Possible Solution

Split the work into a maintainer-approved MiniMax Token Plan endpoint/auth decision, a focused sub-agent media-completion fix if source proof holds, and a documented skill/ClawHub path for the office-generation bundle rather than broad bootstrap restoration.

## Reproduction Assessment

No high-confidence live reproduction was established. Source inspection gives a focused repro target: current main routes MiniMax image generation away from `api.minimax.chat`, and sub-agent media completion combines async media tasks with a disabled message tool.

## Solution Assessment
