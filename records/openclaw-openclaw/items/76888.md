## What This Changes

The issue asks OpenClaw to ensure queued/orphaned user-message repair cannot make an assistant answer an earlier unresolved user turn instead of the latest triggering instruction.

## Best Possible Solution

Update the embedded-runner orphan-trailing-user repair so older orphan content cannot outrank the latest trigger while preserving provider-safe role ordering, then refresh regression coverage and the changelog.
