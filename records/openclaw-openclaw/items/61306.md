## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Move mission state into approved storage:** `src/claw/service.ts:67-70`
  - body: This PR creates and writes `mission-state.json`, `control-state.json`, and `AUDIT_LOG.jsonl` as durable mission runtime state. Root policy requires OpenClaw-owned runtime state and queues to live in the approved SQLite/Kysely-backed stores unless maintainers approve a named artifact exception with migration/upgrade proof, so this sidecar data model should not land as the steady-state store.
  - confidence: 0.93
- **[P1] Reject unsafe mission IDs before resolving paths:** `src/claw/service.ts:1717-1722`
  - body: `resolveMissionDir()` joins the externally supplied `missionId` directly under the missions root, and read/mutate RPC paths reach this resolver after only checking for a non-empty string. A mission id containing `..` or separators can escape the missions directory and target arbitrary `mission-state.json` paths, so validate the generated id format or enforce normalized containment before any path resolution.
  - confidence: 0.91
- **[P1] Preserve HTTP/1-only pinned SSRF dispatchers:** `src/infra/net/ssrf.ts:426-432`
  - body: `createPinnedDispatcher()` now constructs raw Undici dispatchers from `loadUndiciRuntimeDeps()` instead of the `createHttp1*` helpers used on main. Those helpers enforce `allowH2: false` for guarded networking, so this can re-enable the HTTP/2 path on SSRF-guarded requests and weaken the pinned-lookup assumptions.
  - confidence: 0.9
- **[P1] Reload live mission state after agent calls:** `src/claw/service.ts:2922`
  - body: The helper, runner, and verifier cycles await long `runMissionAgent()` calls and then mutate the original in-memory mission state without checking whether an operator paused or cancelled the mission while the call was in flight. Reload or compare the live mission status before applying post-agent transitions so operator control actions cannot be overwritten.
  - confidence: 0.86
