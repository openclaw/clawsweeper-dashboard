## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Validate mission ids before resolving paths:** `src/claw/service.ts:1717-1722`
  - body: Gateway handlers pass raw `missionId` strings through to this resolver, which joins them under the missions root without rejecting path separators or enforcing resolved containment. A crafted id such as a parent-directory path can make later state reads/writes target files outside the intended mission directory if the expected filename exists, so the service should validate canonical mission ids or reject paths that escape the root before any filesystem access.
  - confidence: 0.9
- **[P1] Keep mission runtime state in SQLite:** `src/claw/service.ts:67-70`
  - body: The PR establishes `mission-state.json`, `control-state.json`, and `AUDIT_LOG.jsonl` as durable mission runtime state. That conflicts with the current SQLite-backed runtime-state direction and would create a new sidecar migration/upgrade surface before the Claw storage model is settled, so mission state should use the shared SQLite path or have an explicit maintainer-approved migration design before merge.
  - confidence: 0.88
- **[P1] Preserve HTTP/1-only pinned dispatchers:** `src/infra/net/ssrf.ts:430-447`
  - body: This pinned SSRF path now constructs Undici dispatchers directly instead of using the current `createHttp1*` helpers. That can drop the HTTP/1-only enforcement used by guarded pinned lookups, weakening the networking security assumptions for requests that rely on SSRF protection.
  - confidence: 0.86
- **[P1] Revalidate mission status after agent calls:** `src/claw/service.ts:3050-3079`
  - body: The helper, runner, and verifier cycles call `runMissionAgent` and then write lifecycle transitions using the pre-call mission state. If an operator pauses or cancels the mission while the agent call is in flight, these post-call writes can overwrite that control action, so each long-running path needs to reload and validate current status before applying outcomes.
  - confidence: 0.87
