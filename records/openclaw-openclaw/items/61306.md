Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Preserve HTTP/1-only dispatchers for pinned SSRF requests:** `src/infra/net/ssrf.ts:430-432`
  - body: `createPinnedDispatcher` now constructs raw Undici dispatchers instead of the current `createHttp1*` helpers that enforce `allowH2: false` for guarded fetch paths. That can re-enable HTTP/2 on SSRF-guarded requests and weaken the pinned-lookup assumptions already present on main; keep the helper constructors here.
  - confidence: 0.92
- **[P1] Let normal mission cycles run after recovery errors:** `src/claw/runtime.ts:76-80`
  - body: `recoverInterruptedMissions()` runs before every normal cycle until it succeeds, and any exception skips `runMissionCycles()` while leaving `state.recovered` false. One bad mission recovery can therefore stall all otherwise healthy queued or active missions; isolate recovery failures and continue the normal cycle pass.
  - confidence: 0.89
- **[P1] Re-read mission state after agent runs return:** `src/claw/service.ts:2956-2960`
  - body: After a runner/helper/verifier agent call returns, the code mutates the stale in-memory mission record without checking whether an operator paused or cancelled the mission during the call. Reload and validate the live status before persisting outcomes so operator controls are not overwritten.
  - confidence: 0.88
- **[P2] Avoid dashboard reloads when Claw tab is inactive:** `ui/src/ui/app-gateway.ts:424-426`
  - body: Every `claw.*` event reloads the full Claw dashboard for every connected UI client, even outside the Claw tab. Runtime cycles emit multiple events, so unrelated tabs can generate avoidable list/get/audit/artifact requests; gate the reload to the Claw view or apply targeted local updates.
  - confidence: 0.84
- **[P2] Skip malformed audit lines instead of failing the feed:** `src/claw/service.ts:3846-3848`
  - body: The audit reader parses every JSONL line with a bare `JSON.parse`, so one truncated line makes `claw.audit.get` fail and hides the rest of the mission history. Treat malformed lines like recoverable ledger corruption and skip or surface them as individual bad records.
  - confidence: 0.82
