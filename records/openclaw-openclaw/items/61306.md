Full review comments:

- **[P1] Preserve HTTP/1-only pinned dispatchers:** `src/infra/net/ssrf.ts:430-447`
  - body: `createPinnedDispatcher` now constructs raw Undici dispatchers, bypassing the shared `createHttp1*` helpers that enforce `allowH2: false` for guarded networking. Please keep the pinned SSRF path on the wrapper helpers so HTTP/2 cannot bypass assumptions those guards rely on.
  - confidence: 0.91
- **[P1] Use effective browser plugin activation:** `src/claw/service.ts:600-602`
  - body: `isBrowserPluginEnabled` only accepts an explicit `plugins.entries.browser.enabled === true`, but the browser plugin is bundled with `enabledByDefault: true`. Default configs can therefore block browser-required missions as if the plugin were disabled until users add redundant config.
  - confidence: 0.89
- **[P1] Revalidate mission state after runner returns:** `src/claw/service.ts:2956-2960`
  - body: The runner, verifier, and helper cycles await a long agent run and then persist the stale in-memory mission record. If an operator pauses or cancels the mission while the agent is in flight, this path can overwrite that control action when the agent returns; reload and gate on the live mission status before applying results.
  - confidence: 0.86
- **[P2] Return before reading limit after missionId validation fails:** `src/gateway/server-methods/claw.ts:372-374`
  - body: `claw.audit.get` calls both validators before checking whether `missionId` failed. When `missionId` is invalid and `limit` is also invalid, both helpers can call `respond(false, ...)` for one request; follow the other handlers and return after the required field failure.
  - confidence: 0.87
- **[P2] Gate Claw dashboard reloads to the Claw tab:** `ui/src/ui/app-gateway.ts:424-426`
  - body: Every `claw.*` event currently reloads the full Claw dashboard for every connected UI regardless of the active tab. Runtime mission cycles can emit several events, so unrelated tabs can fan out avoidable Gateway calls; only reload when the Claw view is active or update local state incrementally.
  - confidence: 0.78
