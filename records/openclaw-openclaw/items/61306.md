## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Preserve the HTTP/1-only pinned dispatcher wrappers:** `src/infra/net/ssrf.ts:430-447`
  - body: `createPinnedDispatcher` now instantiates Undici agents directly, bypassing the current `createHttp1*` helpers that enforce HTTP/1-only guarded dispatchers and proxy-safe policy. SSRF-pinned requests should keep using those shared wrappers rather than raw constructors.
  - confidence: 0.92
- **[P1] Use effective browser plugin activation for preflight:** `src/claw/service.ts:600-602`
  - body: This treats the browser plugin as enabled only when config explicitly sets `plugins.entries.browser.enabled === true`. The bundled browser plugin is enabled by default, so normal configs with no explicit entry can be blocked as missing browser support.
  - confidence: 0.91
- **[P1] Revalidate live mission state after verifier returns:** `src/claw/service.ts:3092-3101`
  - body: The verifier applies `done`, `blocked`, or `running` transitions after a long `runMissionAgent` call using the stale in-memory mission record. If an operator pauses or cancels while the verifier is running, this path can overwrite that control action; reload and guard the current status before persisting outcomes.
  - confidence: 0.88
- **[P1] Isolate recovery failures before running cycles:** `src/claw/runtime.ts:75-82`
  - body: A single exception from `recoverInterruptedMissions()` aborts the whole runtime iteration before `runMissionCycles()` and leaves `state.recovered` false, so the same bad recovery can repeatedly block other missions from progressing.
  - confidence: 0.87
- **[P2] Return after missionId validation fails before reading limit:** `src/gateway/server-methods/claw.ts:372-374`
  - body: `readRequiredString` can already send an invalid-request response, but this handler still reads `limit` before checking `missionId`. A request with both a missing missionId and invalid limit can call `respond(false)` twice for one RPC.
  - confidence: 0.9
