  - body: This calls `getReplyFromConfig` before `dispatchReplyWithBufferedBlockDispatcher`, so `/compact` can abort/compact/write session state and non-status native command continuations can perform side effects before reply-run admission and abort ownership are established.   - confidence: 0.9

Route marked status notices through the normal hook-aware delivery path instead of direct delivery.
