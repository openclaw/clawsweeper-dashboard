## Summary

Keep open. Current main still only partially satisfies the request: recursive file collection exists, but nested dated memory files are still treated as evergreen, dirty built-in searches still sync in the background, QMD watcher paths still include glob-shaped patterns under chokidar 5, and several user-facing strings still advertise flat `memory/*.md` files. The reporter also has paired open PR #72748 for the same remaining gaps, so this issue should stay open until that PR or an equivalent fix lands.

## What This Changes

Request recursive `memory/**/*.md` support for `memory_search`, including nested memory subdirectories such as monthly folders.

## Best Possible Solution

Land #72748 or an equivalent focused memory-core fix that treats nested daily memory paths as dated memory, synchronizes dirty search state before querying, watches QMD collection roots while filtering patterns safely under chokidar 5, and updates the remaining flat-path copy to `memory/**/*.md` with targeted regression coverage.
