## Summary

Keep open. Current main recursively collects memory files and has a fixed built-in watcher path, but nested dated memory files still miss temporal decay, dirty built-in searches still sync in the background, QMD watch targets still include glob-shaped paths, and several user-facing strings still advertise only `memory/*.md`; the discussion also points to open PR #72748 as the active paired fix path.

## What This Changes

The issue requests recursive `memory/**/*.md` support for memory search so nested memory notes remain searchable and correctly ranked.

## Best Possible Solution

Land #72748 or an equivalent focused memory-core fix that handles nested dated paths, awaits dirty search synchronization before querying, avoids QMD glob-shaped chokidar watch paths, updates flat-path copy, and adds targeted regression coverage.
