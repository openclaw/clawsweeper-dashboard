## What This Changes

Request a general host-owned structured `api.runtime.llm.completeStructured` runtime API for plugin text and image inputs with JSON/schema output, bounded auth/model/profile policy, and controlled errors.

## Best Possible Solution

Resolve the linked implementation PR or a revised SDK design that adds a versioned structured LLM runtime surface while keeping auth, provider routing, timeouts, and secrets host-owned.
