## What This Changes

The issue requests a general host-owned `api.runtime.llm.completeStructured(...)` plugin runtime API for typed text and image structured inference without exposing credentials.

## Best Possible Solution

Review and land a maintainer-approved structured `runtime.llm` seam, likely through the replacement PR, with explicit auth-profile policy, docs, tests, and real behavior proof; then close this issue after merge.
