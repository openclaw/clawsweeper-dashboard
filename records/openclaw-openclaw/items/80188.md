Keep open: current main has the narrower image-first media-understanding structured extraction seam, but it still does not expose the requested generic host-owned `api.runtime.llm.completeStructured(...)` API. ## What This Changes

Request a generic host-owned `api.runtime.llm.completeStructured(...)` SDK runtime API for typed text/image structured inference without exposing raw credentials to plugins.

## Best Possible Solution

Review and land a maintainer-approved structured `runtime.llm` seam, keeping the existing media-understanding API narrow and making auth/profile policy explicit.
