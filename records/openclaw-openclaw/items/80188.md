## What This Changes

Request a generic host-owned structured inference API for trusted plugins under `api.runtime.llm`, including text and image inputs, JSON/schema handling, host auth/model policy, bounded execution, and controlled result metadata.

## Best Possible Solution

Settle the SDK boundary in the linked implementation PR, preserving host ownership of auth/model policy while keeping plugin routes, prompts, schemas, and storage plugin-owned.
