## Summary

Keep open: current main has the narrower image-first media-understanding structured extraction lane, but it still does not expose the requested generic `api.runtime.llm.completeStructured(...)` host-owned inference API. Request a generic host-owned structured inference API under `api.runtime.llm` for text and image inputs without exposing plugin credentials.

## Best Possible Solution
