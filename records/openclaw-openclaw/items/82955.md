Adds `validate_downloaded_script` to reject empty or non-shebang Homebrew and NodeSource setup downloads before `bash` execution, plus a shell validation harness.

## Best Possible Solution

Land the validation after replacing the GNU-only harness assertion, with explicit maintainer acceptance that shebang-based structural validation is the intended fail-closed contract for these third-party installer scripts.

## Solution Assessment
