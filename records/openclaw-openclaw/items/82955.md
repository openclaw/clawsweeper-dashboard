## What This Changes

The PR adds `validate_downloaded_script()` to `scripts/install.sh`, validates Homebrew and NodeSource setup-script downloads before Bash execution, and adds a Bash validation harness.

## Best Possible Solution

Land the validator after making the harness portable and with maintainer acceptance that malformed or non-shebang Homebrew/NodeSource responses should fail closed instead of being executed.

## Reproduction Assessment

## Solution Assessment
