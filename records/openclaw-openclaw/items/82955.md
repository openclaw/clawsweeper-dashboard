## Summary

## What This Changes

The PR adds downloaded-script structural validation to Homebrew and NodeSource installer paths and adds a shell harness covering rejected and accepted payloads.

## Best Possible Solution

Ship the structural validation only after downloader failures short-circuit before any validator/executor and maintainers accept the fail-closed installer behavior.
