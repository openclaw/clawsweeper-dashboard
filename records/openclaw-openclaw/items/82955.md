## Summary

Keep open. The hardening is useful and the supplied terminal proof is strong, but the patch still lets untrusted non-printable bytes reach installer error output and it intentionally makes third-party setup-script shape a fail-closed install gate.

## What This Changes

The PR adds shared validation for downloaded Homebrew and NodeSource shell scripts before execution, plus a standalone shell validation harness.

## Best Possible Solution

Keep the shared validator, make the diagnostic preview printable-safe with deterministic coverage, and merge only after maintainers accept strict Homebrew and NodeSource shebang enforcement.

## Reproduction Assessment

Yes from source inspection: current main downloads Homebrew and NodeSource setup scripts to temp files and executes them without validating content first. I did not run the full installer in this read-only review, but the PR body includes after-fix terminal proof for the changed path.

## Solution Assessment

Not yet. The shared validator is the right boundary for this fix, but the diagnostic preview should be printable-safe and maintainers need to accept the strict third-party script-shape failure mode.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Escape every non-printable byte before logging:** `scripts/install.sh:87-88`
  - body: The validator removes C0 controls and DEL, then passes the remaining downloaded first line through `ui_error`. In the no-gum path `ui_error` uses `echo -e`, so raw C1/high-bit control bytes such as CSI can still be emitted from untrusted content; whitelist printable text or encode the preview before logging it.
  - confidence: 0.84
- **[P3] Make the binary fixture deterministic:** `scripts/test-install-validation.sh:85`
  - body: The binary fixture is sourced from `/dev/urandom`, so it has a small chance to begin with `#!` and be accepted by the shebang check. Use a fixed invalid byte sequence that cannot start with a shebang so the harness cannot flake.
  - confidence: 0.86
