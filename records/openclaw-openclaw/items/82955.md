- The first-line shebang requirement can intentionally stop installs if Homebrew or NodeSource ever serves a bash-compatible setup script with a BOM, blank preamble, comment preamble, or no shebang where current main would still hand it to bash.

- This is structural validation only; a trusted upstream response that starts with a shebang remains inside the existing third-party-script execution trust boundary.
