The PR adds `validate_downloaded_script()` to `scripts/install.sh`, calls it before Homebrew and NodeSource downloaded setup scripts are executed, and adds a standalone shell validation harness.

## Best Possible Solution

Keep the validation hardening if installer owners accept the fail-closed shebang policy; otherwise preserve compatibility by making stricter structural validation an explicit installer policy or mode.
