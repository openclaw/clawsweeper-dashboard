## Summary

## What This Changes

The PR adds a downloaded-script validator and download-failure guards to Homebrew and NodeSource installer execution paths, plus a standalone shell harness for bad-download cases.

## Best Possible Solution

Keep the centralized validator and checked download guards, make the binary fixture deterministic, and land only after maintainers accept structural fail-closed validation as the intended installer policy.
