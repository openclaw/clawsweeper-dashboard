## What This Changes

The PR adds a macOS known-bundle-ID executable fallback for Chromium browsers when osascript/defaults cannot resolve the default browser app path.

## Best Possible Solution

## Solution Assessment

Yes in direction, but not merge-ready. A known bundle-ID path fallback is the narrow maintainable fix for this failure mode, but it needs regression coverage, contributor real behavior proof, rebase, and fresh validation.
