Keep this PR open: current main and the latest release still default human-readable `openclaw logs` output to UTC, and this PR is the active closing implementation for https://github.com/openclaw/openclaw/issues/46748. The PR changes `openclaw logs` to use local timestamps by default, keeps `--local-time` accepted, adds `--utc`, and includes type-only OpenRouter/OpenAI-compatible image response cleanup.

## Best Possible Solution
