## What This Changes

This PR changes `openclaw logs` to use local timestamp rendering by default, keeps `--local-time` accepted, adds `--utc`, and removes redundant `unknown` unions from image parsing helpers.

## Best Possible Solution

Land or replace this with a focused logs CLI change that defaults human-readable output to local time only after maintainers accept the compatibility tradeoff, documents and tests `--utc`, and includes real log-stream proof for default-local plus UTC output.

## Solution Assessment
