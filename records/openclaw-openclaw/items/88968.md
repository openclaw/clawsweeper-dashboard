## Summary

## What This Changes

The PR changes auto-reply memory-flush failures from terminal reply errors into log-and-continue recovery with exhausted-session rotation, flush-state cleanup, tests, and expanded `notifyUser` docs.

## Best Possible Solution

Land this or an equivalent bounded recovery only after maintainers explicitly accept the fail-open reply behavior, session-rotation recovery, expanded `notifyUser` notice, and compatibility with https://github.com/openclaw/openclaw/pull/84792.

## Reproduction Assessment

## Solution Assessment

Mostly yes.
