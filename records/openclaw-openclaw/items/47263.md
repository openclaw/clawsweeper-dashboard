  - confidence: 0.98

  - confidence: 0.96

  - body: `localConfig()` can resolve LAN, tailnet, or arbitrary custom hosts and includes the gateway token/password. Automatic startup polling can therefore repeatedly send credentials to a non-loopback endpoint, including over `ws://` when TLS is disabled; use a loopback-only process probe or enforce a secure trusted host.
  - confidence: 0.93
