## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Keep health probes on a local-safe path:** `apps/macos/Sources/OpenClaw/GatewayProcessManager.swift:415-419`
  - body: `probeLocalGatewayHealth()` builds its URL and credentials from `GatewayEndpointStore.localConfig()`, whose local host can be custom/tailnet and whose token/password can come from env, config, or launchd. Because `GatewayChannelActor` sends those credentials in the connect frame, startup polling can leak gateway auth to a non-loopback endpoint; reuse the existing connection contract or force a loopback/no-secret probe.
  - confidence: 0.9
- **[P1] Preserve env-backed auth before blocking node mode:** `apps/macos/Sources/OpenClaw/NodeMode/MacNodeModeCoordinator.swift:147`
  - body: The new node-mode gate only treats `openclaw.json` as an existing local setup, but current endpoint resolution also supports `OPENCLAW_GATEWAY_TOKEN`, `OPENCLAW_GATEWAY_PASSWORD`, and launchd snapshots. Upgraded users with env/service-backed gateway auth can be disconnected until they rerun onboarding even though their gateway is configured.
  - confidence: 0.86
- **[P2] Block dot navigation until the notice is acknowledged:** `apps/macos/Sources/OpenClaw/OnboardingView+Actions.swift:62`
  - body: `finish()` now persists the security acknowledgement unconditionally, while the existing progress dots can still jump a first-run user past page 0. That lets onboarding complete without the checkbox ever being selected, so route all page changes through the same gate or only persist acknowledgement after explicit selection.
  - confidence: 0.9
- **[P2] Provide an in-page recheck for manual CLI installs:** `apps/macos/Sources/OpenClaw/OnboardingView+Pages.swift:748`
  - body: The manual install copy path disables `Next` until `cliInstalled` becomes true, but this page only refreshes on entry or after the in-app install action. A user who runs the copied command in Terminal stays blocked until they leave and re-enter the page; add a visible recheck action or polling for the manual path.
  - confidence: 0.82
- **[P2] Avoid implicit latest installs in debug builds:** `apps/macos/Sources/OpenClaw/GatewayEnvironment.swift:100-102`
  - body: Returning `nil` for `.debug` bundle IDs makes both the installer and displayed command fall back to `latest`, while the install path still executes the network shell installer. Keep debug installs pinned or require an explicit developer opt-in for moving latest.
  - confidence: 0.78
