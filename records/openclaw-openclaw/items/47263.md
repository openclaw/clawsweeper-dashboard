  - body: The manual install path only copies the command, but the CLI page disables Next until `cliInstalled` becomes true and only refreshes on page entry or after the in-app install flow.   - confidence: 0.85

  - body: The attach/start/readiness loops now allocate a fresh `GatewayChannelActor` for every poll instead of using `self.connection`.   - confidence: 0.78
