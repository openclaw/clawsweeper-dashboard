  - confidence: 0.92

  - confidence: 0.88
- **[P2] Keep debug CLI installs version-pinned by default:** `apps/macos/Sources/OpenClaw/GatewayEnvironment.swift:99-103`
  - body: Returning `nil` for `.debug` bundle identifiers flows into `CLIInstaller.install()` and `devLinkCommand` as `latest`, so debug builds start executing a moving remote installer target. Please keep a pinned version by default or require an explicit developer opt-in for latest installs.
  - confidence: 0.82
- **[P2] Preserve injected gateway health probes:** `apps/macos/Sources/OpenClaw/GatewayProcessManager.swift:380`
  - body: `waitForGatewayReady()` and the attach/start loops now call `probeLocalGatewayHealth()`, which ignores the existing `setTestingConnection` hook. That breaks the current fake-connection tests and makes these paths require a real socket instead of the injected gateway connection.
  - confidence: 0.86

  - body: The manual install path only copies the command, while `Next` stays disabled until `cliInstalled` refreshes. A user who runs the copied command in Terminal remains stuck on this page unless they leave and re-enter onboarding.
