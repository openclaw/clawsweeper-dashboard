Overall confidence: 0.97

Full review comments:

- **[P1] Enforce acknowledgement across page-dot navigation:** `apps/macos/Sources/OpenClaw/OnboardingView+Layout.swift:113-124`
  - body: The new acknowledgement only disables Next on page 0, but these dot buttons can jump directly to later pages. The user can then finish, and `finish()` records acknowledgement as true even though the toggle was never accepted; lock forward dot navigation until acknowledgement or validate it again before completion.
  - confidence: 0.99
- **[P1] Preserve all effective auth sources in the node-mode gate:** `apps/macos/Sources/OpenClaw/NodeMode/MacNodeModeCoordinator.swift:136-147`
  - body: This gate falls back to `hasExistingLocalSetup(root:)`, which only inspects `openclaw.json`. Existing local installs authenticated through `OPENCLAW_GATEWAY_TOKEN`/`PASSWORD` or the saved launchd snapshot are treated as unconfigured after an onboarding-version bump and node mode disconnects; derive this from the canonical effective endpoint/auth state instead.
  - confidence: 0.97
- **[P1] Keep readiness checks on the existing connection owner:** `apps/macos/Sources/OpenClaw/GatewayProcessManager.swift:200`
  - body: Replacing `self.connection.requestRaw` with a newly constructed channel bypasses `setTestingConnection`, so the existing readiness tests and injected runtime path no longer control these loops. It also duplicates endpoint/auth resolution and creates one asynchronously cleaned-up channel per poll; keep the checks on the injected connection or inject a single scoped probe owner.
  - confidence: 0.98
- **[P2] Keep debug CLI installs version-pinned:** `apps/macos/Sources/OpenClaw/GatewayEnvironment.swift:98-103`
  - body: Returning nil for `.debug` bundle identifiers makes `CLIInstaller.install()` and the manual command fall back to `openclaw@latest`. A debug app can therefore install a gateway version unrelated to the app under test; preserve an explicit app/build version or require a deliberate developer override rather than silently selecting latest.
  - confidence: 0.99
- **[P2] Add a same-page recheck for manual CLI installs:** `apps/macos/Sources/OpenClaw/OnboardingView+Pages.swift:205-220`
  - body: The CLI page disables Next until `cliInstalled` is true, but the manual-install path has no recheck action or polling after the user runs the copied command. Readiness refreshes only on page entry or after the built-in installer, so a successful manual install leaves the page stuck until navigation or restart; expose a recheck action on this page.
  - confidence: 0.96
