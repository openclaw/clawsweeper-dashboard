  - confidence: 0.9
- **[P2] Preserve env-backed node-mode auth:** `apps/macos/Sources/OpenClaw/NodeMode/MacNodeModeCoordinator.swift:145-147`
  - body: This fallback only checks `openclaw.json` through `hasExistingLocalSetup(root:)`. Existing local installs can keep gateway auth in `OPENCLAW_GATEWAY_TOKEN` / `OPENCLAW_GATEWAY_PASSWORD` or the launchd snapshot, so those upgraded users can have node mode disconnected until they rerun onboarding despite having valid local auth.
  - confidence: 0.86
