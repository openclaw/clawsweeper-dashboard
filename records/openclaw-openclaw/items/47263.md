- **[P1] Gate finish behind the security acknowledgement:** `apps/macos/Sources/OpenClaw/OnboardingView+Actions.swift:62`
  - body: `finish()` now persists the acknowledgement unconditionally, but the page dots can still jump off page 0 because they only check the wizard lock. A first-run user can skip the checkbox, finish onboarding, and permanently record the security acknowledgement.
  - confidence: 0.92
- **[P2] Keep gateway readiness on the injectable connection:** `apps/macos/Sources/OpenClaw/GatewayProcessManager.swift:202`
  - body: Switching readiness checks to `probeLocalGatewayHealth` bypasses `self.connection`, so DEBUG tests using `setTestingConnection` no longer drive attach/start/readiness paths and those loops open real per-probe channels instead of the shared configured connection.
  - confidence: 0.88
- **[P2] Let manual CLI installs refresh without leaving the page:** `apps/macos/Sources/OpenClaw/OnboardingView+Pages.swift:747-748`
  - body: The manual install path only copies the command, while `Next` stays disabled until `cliInstalled` changes and refresh only runs on page entry or app-triggered install. Users who run the copied command in Terminal can be stuck until they leave and re-enter onboarding.
  - confidence: 0.84
- **[P2] Preserve env and launchd auth in node-mode gating:** `apps/macos/Sources/OpenClaw/NodeMode/MacNodeModeCoordinator.swift:147`
  - body: This fallback only checks `openclaw.json`, but the app treats `OPENCLAW_GATEWAY_TOKEN`, `OPENCLAW_GATEWAY_PASSWORD`, and launchd snapshots as valid local auth sources. Upgraded installs using those sources can have node mode disconnected until onboarding is rerun.
  - confidence: 0.86
- **[P2] Avoid defaulting debug installs to unpinned latest:** `apps/macos/Sources/OpenClaw/GatewayEnvironment.swift:102`
  - body: Returning `nil` for `.debug` bundle identifiers makes the installer fall back to `latest`, which feeds a network `curl | bash` installer without a pinned version. Developer/debug builds should still use an explicit version or an intentional opt-in.
  - confidence: 0.81
