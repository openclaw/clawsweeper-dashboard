Overall confidence: 0.86

Full review comments:

- **[P1] Preserve env-backed auth before blocking node mode:** `apps/macos/Sources/OpenClaw/NodeMode/MacNodeModeCoordinator.swift:144`
  - body: This fallback only checks `openclaw.json` via `hasExistingLocalSetup(root:)`. The macOS gateway endpoint store also treats `OPENCLAW_GATEWAY_TOKEN`, `OPENCLAW_GATEWAY_PASSWORD`, and the saved launchd snapshot as valid local auth sources, so upgraded users with env/plist-backed auth and an older onboarding version will now be disconnected until they rerun onboarding despite having a configured gateway.
  - confidence: 0.9
- **[P2] Enforce acknowledgement outside the Next button:** `apps/macos/Sources/OpenClaw/OnboardingView+Actions.swift:61`
  - body: `finish()` now persists the security acknowledgement unconditionally, but the page dots still allow jumping away from page 0 without using `handleNext()`. A first-run user can click a later dot, finish onboarding, and store the acknowledgement without checking the box, so the new security gate is not enforced.
  - confidence: 0.88
- **[P2] Keep readiness probes on the injectable connection path:** `apps/macos/Sources/OpenClaw/GatewayProcessManager.swift:380`
  - body: Replacing `self.connection.requestRaw` with `probeLocalGatewayHealth` makes `waitForGatewayReady` and the start/attach loops ignore the DEBUG `setTestingConnection` hook. Existing `GatewayProcessManagerTests` drive these paths through a fake GatewayConnection, so the patched paths will depend on a real local socket instead of deterministic test input.
  - confidence: 0.86
- **[P2] Add a recheck path after manual CLI installs:** `apps/macos/Sources/OpenClaw/Onboarding.swift:158`
  - body: The new CLI page disables Next until `cliInstalled` is true, but users who copy and run the manual install command only get refreshes on page entry or after the in-app installer/CLT recheck path. If the manual command succeeds while they remain on this page, onboarding has no enabled recheck action and stays stuck with Next disabled.
  - confidence: 0.82
