- **[P1] Keep health probes on a local-safe path:** `apps/macos/Sources/OpenClaw/GatewayProcessManager.swift:415-419`
  - body: `probeLocalGatewayHealth()` builds its URL and credentials from `GatewayEndpointStore.localConfig()`, whose local host can be custom/tailnet and whose token/password can come from env, config, or launchd.   - confidence: 0.9

  - body: The new node-mode gate only treats `openclaw.json` as an existing local setup, but current endpoint resolution also supports `OPENCLAW_GATEWAY_TOKEN`, `OPENCLAW_GATEWAY_PASSWORD`, and launchd snapshots. Upgraded users with env/service-backed gateway auth can be disconnected until they rerun onboarding even though their gateway is configured.
  - confidence: 0.86

  - body: `finish()` now persists the security acknowledgement unconditionally, while the existing progress dots can still jump a first-run user past page 0.   - confidence: 0.9

  - confidence: 0.82
