Full review comments:

- **[P2] Enforce the advertised rerank score contract:** `extensions/memory-core/src/memory/manager.ts:790-802`
  - body: The new docs/API promise that core rejects dropped, duplicated, fabricated, or non-normalized rerank scores, but this validator only checks ref uniqueness and range. A provider can return a partial list, `NaN`, `2`, or `-1`, and the search will be marked `active` with invalid or partial `rerankScore` data. Please require exactly one score per candidate and a finite `[0,1]` score before calling `reorderByRerankScores`.
  - confidence: 0.9

The repository's API check treats intentional SDK surface changes as baseline drift until `pnpm plugin-sdk:api:gen` is run and the updated hash is committed, so this will fail the Plugin SDK API gate and leaves the new public contract unrecorded.
  - confidence: 0.86
