## Best Possible Solution

Port a narrow current-main fix that makes root logo/favicon URLs absolute, updates the current chat-avatar fallback if needed, scopes any CSP exception only to bundled static SVG/favicon responses, preserves CSP on HTML/avatar/media routes, adds focused tests, and records the user-facing fix in `CHANGELOG.md`.
