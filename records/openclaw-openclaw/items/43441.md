- **[P2] Isolate argv1 in the bundled fallback test:** `src/agents/workspace-templates.test.ts:74`
  - body: This test intends packageRoot to be null, but resolveWorkspaceTemplateDir() still defaults argv1 to process.argv[1]; in repo test runs that can let package-root lookup find the checkout and return repo templates before the synthetic bundled fallback.   - body: When package-root discovery is null in a bundled dist layout, ../../docs is the known wrong shape and is checked before the intended ../docs candidate. Reversing those fallback candidates avoids selecting an unrelated sibling docs tree and better matches the reported failure mode.
  - confidence: 0.7
