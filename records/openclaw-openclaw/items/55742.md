  - confidence: 0.88

  - confidence: 0.86

  - body: The new type is added only to internal plugin types, but plugin authors import supported contracts through `openclaw/plugin-sdk/*`. If this fifth argument is intended API, the SDK exports, docs, and contract tests need to expose it so external plugins do not depend on internals.
  - confidence: 0.82
