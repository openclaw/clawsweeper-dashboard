## Summary

Keep open: current main still has no on-demand Clear completed or per-entry dismiss affordance for completed subagent rows in the chat session selector. The broader orchestration and session-kind filter issues are related, but neither replaces this focused cleanup request.

## What This Changes

Add an on-demand Control UI action to clear or dismiss completed subagent sessions from the session dropdown.

## Best Possible Solution

Add a focused desktop and mobile Control UI action near the chat session selector that clears only terminal completed subagent rows, refreshes sessions.list, and preserves active, main, channel, cron, and broader filtering behavior.
