## Best Possible Solution

## Reproduction Assessment

Yes, from source inspection: current main suppresses default tool progress in Telegram `streamMode: "off"`, and the colocated regression test asserts that path. I did not run tests because this review was required to stay read-only.

## Solution Assessment

No. The production change is narrow, but it reverses the documented final-only contract without updating docs/changelog and leaves a weak optional-chain assertion in the updated test.
