- **[P1] Refresh persisted prompts after changing their path format:** `src/skills/loading/workspace.ts:83-85`

  - confidence: 0.9

That makes prompts longer without improving readability and can push large skill catalogs into compact/truncated output, so gate preservation on the config skills root not being equivalent to the read tool's home expansion.
  - confidence: 0.78
