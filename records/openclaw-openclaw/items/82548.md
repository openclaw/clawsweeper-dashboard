## Summary

Keep open. Current main still has operational diagnostics, OTel/Prometheus exporters, tool execution telemetry, and external-content hardening, but it does not have the requested AI safety and quality event taxonomy; the reporter also added a concrete schema proposal after the earlier review, so this remains active feature-design work rather than cleanup material.

## What This Changes

Request to add first-class AI safety and quality diagnostics events, exporter mappings, operator summaries, and optional feedback labels for prompt-injection, external-content, tool-policy, source-quality, and memory/context observability.

## Best Possible Solution

Define a small additive diagnostic schema with bounded metadata and exporter mappings first, then land emitters at external-content, tool-policy, context/memory, source-quality, and feedback boundaries in staged PRs.

## Solution Assessment
