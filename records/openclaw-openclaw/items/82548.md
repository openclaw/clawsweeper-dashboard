Request adding structured AI safety and quality observability events for prompt-injection signals, external-content consumption, tool-policy decisions, memory/context selection, source quality, and user feedback.

## Best Possible Solution

Define a small additive AI diagnostic schema with bounded metadata and exporter mappings first, then split implementation across core-owned boundaries and plugin/eval emitters according to maintainer ownership decisions.
