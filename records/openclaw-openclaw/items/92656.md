## Best Possible Solution

Decide the Teams/Planner ownership and permission model first, then implement a narrow opt-in plugin-owned tool with manifest metadata, explicit Graph consent setup, confirmation, DLP/audit behavior, docs, and live Teams-to-Planner proof.
