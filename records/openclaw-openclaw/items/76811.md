## Summary

## What This Changes

The PR broadens ClawHub artifact resolver metadata handling, fetches version metadata when summary-shaped legacy ZIP artifacts omit usable verification fields, adds regression coverage, and updates the changelog.

## Best Possible Solution

Land a synced version of this PR or equivalent that uses version metadata as the fallback verification source for summary-shaped legacy ZIP artifacts while preserving fail-closed archive and ClawPack verification.

## Solution Assessment
