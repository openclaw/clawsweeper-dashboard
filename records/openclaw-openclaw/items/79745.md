## What This Changes

## Solution Assessment

No, not yet. The agent-scoped config plus clean mcporter env direction is plausible, but the current implementation loses mcporter lifecycle metadata and needs safer handling for copied server secrets.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.87

Full review comments:

- **[P2] Preserve lifecycle when cloning mcporter servers:** `extensions/memory-core/src/memory/qmd-manager.ts:2079-2082`
  - body: This clones the configured server from `mcporter config get --json`, but mcporter v0.10.2's JSON serializer does not include `lifecycle`. A user config that correctly marks `qmd` as `keep-alive` will be written into the generated per-agent config without that field, and mcporter does not default qmd to keep-alive, so `startDaemon` can stop keeping the qmd server warm.
  - confidence: 0.9
- **[P2] Write generated mcporter configs without exposing secrets:** `extensions/memory-core/src/memory/qmd-manager.ts:2063`
  - body: The generated config can include serialized mcporter headers, including values derived from static headers or bearer tokens, and `fs.writeFile` creates this new agent-state file with default permissions. That can duplicate auth material more broadly than the source config; either avoid materializing static secrets or write the generated file with restrictive permissions and covered tests.
  - confidence: 0.78

## Security Review

Status: needs_attention

Summary: The diff can copy mcporter auth headers or token-derived headers into a new generated config file without restrictive permissions.

Concerns:

- **[medium] Generated config can duplicate auth material:** `extensions/memory-core/src/memory/qmd-manager.ts:2063`
  - body: mcporter supports static headers and bearer-token fields that serialize into HTTP headers; this PR copies the serialized server entry into an agent-scoped `mcporter.json` using default file permissions, so sensitive values may be duplicated outside the original config's protection.
  - confidence: 0.78
