## Summary

Keep this PR open. The central QMD/mcporter isolation fix is useful and the local real-behavior proof is solid, but the latest diff still has a merge-blocking auth-preservation gap for remote mcporter configs with token-like URL query auth or refresh-only auth metadata.

## What This Changes

## Best Possible Solution
