Full review comments:

- **[P1] Use the current pi package namespace:** `extensions/volcengine/cache.ts:1-3`
  - body: This new module imports `@mariozechner/pi-agent-core` and `@mariozechner/pi-ai`, but current main only declares the `@earendil-works/*` pi packages. As written, the added file will not resolve in this workspace after merge.
  - confidence: 0.97
- **[P1] Invalidate cached responses when input prompts change:** `extensions/volcengine/cache.ts:277`
  - body: Responses carries the system/developer prompt in `input`, but this hash only covers `tools` and `instructions`. After a prompt/profile change, the wrapper can reuse the old `previous_response_id` and trim the updated prompt from the continuation request, leaving the next turn under stale instructions.
  - confidence: 0.9
- **[P2] Return a fresh stream when retrying stale response IDs:** `extensions/volcengine/cache.ts:354`
  - body: The retry path calls `pipeStream(output, false)` with the same output stream. If the failed attempt emitted start or content events before the invalid-response error, those partial events are delivered before the retry output.
  - confidence: 0.82
