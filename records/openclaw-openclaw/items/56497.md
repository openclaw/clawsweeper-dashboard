Full review comments:

- **[P1] Use the current Pi package namespace:** `extensions/volcengine/cache.ts:1-3`
  - body: Current main declares `@earendil-works/pi-agent-core` and `@earendil-works/pi-ai`, but this new file imports the old `@mariozechner/*` packages. After rebasing onto current main those imports are undeclared, so the Volcengine plugin will fail to typecheck/build until the imports are updated.
  - confidence: 0.95
- **[P1] Preserve current Volcengine provider hooks:** `extensions/volcengine/index.ts:81`
  - body: The wrapper is added on an older provider entry that lacks current main's `hookAliases: ["volcengine-plan"]` and Volcengine compatibility hooks. Rebase the wrapper onto the current provider object so `volcengine-plan/*` settings still find the plugin and existing tool-schema compatibility is not lost.
  - confidence: 0.9
- **[P1] Include system input in cache invalidation:** `extensions/volcengine/cache.ts:277-278`
  - body: The invalidation hash only covers `target.tools` and `target.instructions`, but the Responses path carries the system/developer prompt as an `input` item. A prompt/profile change can reuse an old `previous_response_id`, trim the payload to continuation input, and run under stale instructions.
  - confidence: 0.9
- **[P2] Retry stale response IDs through isolated output:** `extensions/volcengine/cache.ts:354`
  - body: The stale-response branch retries with `pipeStream(output, false)` using the same output stream. If the failed attempt emitted a start or partial event before the retry decision, consumers can see failed-attempt events followed by retry events; buffer until retry is ruled out or retry through a fresh output path.
  - confidence: 0.86
- **[P2] Align the documented thinking default:** `extensions/volcengine/cache.ts:110-113`
  - body: The docs say omitted `cache.thinking` defaults to `true`, but `resolveCacheConfig` leaves it `undefined` and the wrapper only sends `thinking` when explicitly configured. Either make the code default true or document the actual omitted request shape.
  - confidence: 0.84
