  - confidence: 0.86
- **[P3] Align the documented thinking default:** `docs/providers/volcengine.md:72-73`
  - body: The docs say `thinking` defaults to `true`, but `resolveCacheConfig` leaves it undefined unless the user sets a boolean and the wrapper only writes `target.thinking` for explicit values. Either the code should set the default or the docs should not promise it.
  - confidence: 0.82
