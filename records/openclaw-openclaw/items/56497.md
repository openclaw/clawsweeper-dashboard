## Review Findings

Overall correctness: patch is incorrect

Full review comments:

- **[P1] Use the current pi package namespace:** `extensions/volcengine/cache.ts:1-3`
  - body: This file imports `@mariozechner/pi-agent-core` and `@mariozechner/pi-ai`, but current main declares `@earendil-works/pi-agent-core` and `@earendil-works/pi-ai` 0.74.0. The new module will not resolve in this workspace until these imports use the current namespace.
  - confidence: 0.97
- **[P1] Invalidate cached responses when input prompts change:** `extensions/volcengine/cache.ts:277`
  - body: Responses puts the system/developer prompt in `input`; this hash only includes `tools` and `instructions`. After a prompt/profile change, the wrapper can reuse the old `previous_response_id` and then trim the updated prompt from the continuation request, leaving the next turn under stale instructions.
  - confidence: 0.9
- **[P2] Return a fresh stream when retrying stale response IDs:** `extensions/volcengine/cache.ts:354`
  - body: The retry path calls `pipeStream(output, false)` with the same output stream. If the failed attempt emitted start or content events before the invalid-response error, those partial events are delivered before the retry output.
  - confidence: 0.82
