  - confidence: 0.9

  - confidence: 0.87

  - body: The docs say `thinking` defaults to `true`, but `resolveCacheConfig` leaves it undefined when omitted and the wrapper only sends a thinking body for explicit boolean values. Either set the default in code or document that thinking is unchanged unless configured.
