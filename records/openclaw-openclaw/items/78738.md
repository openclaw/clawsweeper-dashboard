## What This Changes

The issue asks OpenClaw to return an agent-visible error when exec approval prompt or followup dispatch fails instead of only logging and losing the command result.

## Best Possible Solution

Review and land https://github.com/openclaw/openclaw/pull/78766 or an equivalent narrow fix that makes failed approval prompt/followup delivery visible to the agent, with focused regression coverage for transient send failure and final failure behavior.
