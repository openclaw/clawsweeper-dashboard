On current main, `src/infra/exec-approval-forwarder.ts` resolves configured/session targets, fires `deliverToTargets(...)` in the background, logs per-target failures, and returns `true`, so `src/gateway/server-methods/approval-shared.ts` treats the approval as delivered and leaves the agent waiting.
