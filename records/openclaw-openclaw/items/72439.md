The PR changes gateway session-kill path parsing to return a 400 invalid_request_error for malformed encoded session keys, adds focused regression coverage, and records the fix in the changelog.

## Best Possible Solution

Land the narrow gateway handler, regression test, and changelog fix after the branch is rebased or otherwise made mergeable and exact-head CI is green or explicitly triaged.

## Solution Assessment
