## What This Changes

Add a stable unauthenticated gateway version field or endpoint for operators to identify running OpenClaw instances.

## Best Possible Solution

Add a documented additive version-only public surface after maintainers approve the unauthenticated field set, while preserving the cheap pre-config liveness behavior.
