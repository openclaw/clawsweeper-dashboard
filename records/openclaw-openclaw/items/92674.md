## Summary

Keep open: current main and v2026.6.6 still let an inherited/configured `medium` thinking level resolve to `adaptive` for MiniMax-M3's `[off, adaptive]` profile, and no merged or open canonical item already owns this exact cost-regression.

## What This Changes

The issue asks OpenClaw to stop silently falling back from a requested budget level such as `medium` to provider-managed `adaptive` on adaptive-only models.

## Best Possible Solution

Make unsupported non-adaptive budget levels fall back to a supported budget level or `off`, while preserving explicit `adaptive`, MiniMax-M3's provider default, and focused tests/docs for the fallback contract.

## Reproduction Assessment

Yes, from source inspection: `thinkingDefault: "medium"` is accepted as the configured default, MiniMax-M3 supports only `[off, adaptive]`, and the shared rank fallback selects `adaptive`. I did not run a live MiniMax cron job in this read-only review.

## Solution Assessment

Yes, the best fix is a narrow fallback-policy repair rather than a new config option: do not auto-select `adaptive` for non-adaptive budget requests, and cover explicit adaptive/default behavior with tests.

## Repair Work Prompt

Fix OpenClaw issue https://github.com/openclaw/openclaw/issues/92674. Current main maps configured/inherited `medium` thinking to `adaptive` for MiniMax-M3 because `adaptive` has the same rank as `medium` and the MiniMax-M3 profile exposes only `[off, adaptive]`. Make unsupported non-adaptive budget requests fall back to a supported non-adaptive level or `off`, never silently to `adaptive`; preserve explicit `adaptive` requests, MiniMax-M3's provider default, explicit `/think off`, and MiniMax M2.x budget levels. Add focused regression coverage for `resolveSupportedThinkingLevel` on `[off, adaptive]`, cron/config default behavior where practical, and existing MiniMax M3 adaptive/default behavior. Update thinking docs if the stored-level remap contract changes. Do not add a new config option or broaden MiniMax xhigh/max support.
