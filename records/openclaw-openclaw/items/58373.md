## Summary

## What This Changes

The PR bootstraps a missing non-main agent `models.json` from the main agent when model planning returns `skip`, and adds targeted regression tests for inheritance, retry, and concurrent creation.

## Best Possible Solution

Keep the narrow skip-branch bootstrap, but resolve the credential boundary by writing lookup-safe catalog data, reusing existing auth fallback, or recording explicit maintainer/security approval for raw same-user copying before landing.

## Reproduction Assessment

Yes. Source inspection gives a high-confidence path: `agents.create` can leave a non-main agent without `models.json`, an empty provider plan returns `skip`, `ensureOpenClawModelsJson` writes nothing on current main, and the embedded runner can then throw `Unknown model`.

## Solution Assessment

No, not as-is. The skip-branch bootstrap is the right narrow direction, but verbatim copying of the main `models.json` needs sanitization or explicit maintainer/security approval before merge.
