## Summary

Keep this PR open: current main still has the reported non-main skip-path gap, and this branch is a focused fix, but it should not merge until the raw main-agent models.json credential-copy boundary is resolved.

## What This Changes

The PR adds a one-time non-main agent models.json bootstrap from the main agent in the plan.action === "skip" path, plus regression tests for inheritance, retry, and concurrent creation.

## Best Possible Solution

Keep the narrow skip-branch bootstrap, but sanitize the inherited catalog to model lookup data or get explicit maintainer/security approval for raw same-user models.json propagation.

## Reproduction Assessment

Yes. Source inspection gives a high-confidence path: agents.create can leave a non-main agent without models.json, an empty provider plan returns skip, ensureOpenClawModelsJson writes nothing, and the embedded runner can then throw Unknown model.

## Solution Assessment

No, not as-is. The one-time bootstrap is the right narrow direction, but writing the full raw main models.json needs sanitization or explicit maintainer/security approval before merge.
