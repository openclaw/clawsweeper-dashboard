## Summary

Keep open. The bug path is valid and current main still lacks a non-main skip-branch bootstrap, but this PR should not merge until the raw `models.json` credential-copy boundary is sanitized or explicitly accepted by maintainers/security.

## What This Changes

This PR adds a skip-branch helper that one-time bootstraps a missing non-main agent `models.json` from the main agent and adds regression tests for inheritance, retry, and concurrent creation.

## Best Possible Solution

Keep the narrow one-time missing-catalog bootstrap, but copy only catalog-safe fields or obtain explicit maintainer/security acceptance for raw credential propagation.

## Reproduction Assessment

Yes, source-level reproduction is high confidence: current main can create a non-main agent without `models.json`, empty provider planning returns `skip`, and the skip branch leaves the registry missing before model resolution can emit `Unknown model`. I did not run the live Windows packaged flow in this read-only review.

## Solution Assessment

No, not as submitted. The atomic, retry, and concurrency guardrails are now present, but raw-copying the main agent `models.json` needs sanitization or explicit maintainer/security acceptance before merge.

## Review Findings

Overall correctness: patch is incorrect
