## Summary

Keep this PR open: it targets a real current-main non-main agent startup bug and supplies sufficient real behavior proof, but the patch is not merge-ready because it can copy raw credential-bearing provider configuration from the main agent models.json into secondary agent directories.

## What This Changes

The PR adds a skip-branch helper that bootstraps a missing non-main agent models.json from the main agent and adds focused tests for inheritance, retry, and concurrent creation.

## Best Possible Solution

Keep the narrow skip-branch bootstrap, but write only lookup-safe catalog data or obtain explicit maintainer/security approval for raw same-user models.json propagation.

## Reproduction Assessment

Yes, source inspection gives a high-confidence path: agents.create can leave a non-main agent without models.json, an empty provider plan returns skip, ensureOpenClawModelsJson writes nothing, and the embedded runner can then throw Unknown model. I did not run a live Windows packaged repro because this review is read-only.

## Solution Assessment

No, not as-is. The skip-branch bootstrap is the right narrow direction, but verbatim copying of the main models.json needs sanitization or explicit maintainer/security approval before merge.
