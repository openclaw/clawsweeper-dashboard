Keep open. Current main still has a source-level Telegram path where visible non-final/tool/block output can be delivered after an answer preview exists, then the final answer can be applied by editing that older short-lived preview, leaving the final text visually above later bubbles in Telegram.

The issue asks Telegram direct replies to preserve chronological order when assistant tool-call/intermediate bubbles and the final answer are streamed during one turn.
