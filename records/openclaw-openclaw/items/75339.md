## Best Possible Solution

Land one canonical current-main parser fix that shape-checks structured OpenAI-compatible content before appending, preserves string behavior, routes thinking blocks through existing thinking output, keeps regression tests and changelog credit, then use it to resolve #75268 and supersede overlapping parser PRs.
