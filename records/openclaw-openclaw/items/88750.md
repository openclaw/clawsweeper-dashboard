This PR adds a versioned `ContextEngineRuntimeSettings` contract, threads it through context-engine lifecycle helper paths, extends strict legacy stripping, exports the contract through the plugin SDK, and documents it.

## Best Possible Solution

Finish the runtime-settings contract before merge: pass the same versioned snapshot through every documented lifecycle hook, source requested/resolved/fallback/degraded facts from canonical runtime state, keep the strict legacy strip/retry compatibility, and align docs/tests/proof with the final behavior.
