  - body: NetBird upstream defaults to `utun100` on Darwin, but this helper only scans `wt*`. That makes `gateway.bind: "netbird"` fall back to loopback on macOS, including the macOS environment listed in the PR body.
  - confidence: 0.93

  - confidence: 0.9

  - body: Current main validates bind modes through the later `VALID_BIND_MODES` set and the effective bind union. Updating the older validator block will not keep `openclaw gateway run --bind netbird` valid after rebase unless the current set, union, and help text are updated.
  - confidence: 0.88

  - body: Current main uses `src/wizard/setup.gateway-config.ts`, `setup.ts`, and `setup.types.ts`; the `onboarding.*` files changed here are no longer the active setup surface. After rebase, users would not see the NetBird option in the wizard they actually run.
  - confidence: 0.94
- **[P2] Seed the advertised NetBird origin:** `src/config/gateway-control-ui-origins.ts:35-38`

  - confidence: 0.9
