  - body: NetBird upstream defaults to `utun100` on Darwin while this helper only scans `wt*`. That makes `gateway.bind: "netbird"` fall back to loopback on macOS, including the macOS environment listed in the PR body.
  - confidence: 0.93

  - confidence: 0.9

  - body: Current main validates bind modes through the later `VALID_BIND_MODES` set. Updating the old validator shape does not keep `openclaw gateway run --bind netbird` valid after rebase unless the current set and help text are updated too.
  - confidence: 0.88
- **[P2] Port the wizard option to the active setup files:** `src/wizard/onboarding.gateway-config.ts:81`
  - body: Current main uses `src/wizard/setup.gateway-config.ts`, `setup.ts`, and `setup.types.ts`; the `onboarding.*` files changed here are no longer the active setup surface. After rebase, users would not see the NetBird option in the wizard they actually run.
  - confidence: 0.94
- **[P2] Move link resolution into the active helper:** `src/commands/onboard-helpers.ts:459-465`
  - body: Current main re-exports Control UI link generation from `src/gateway/control-ui-links.ts`, whose API also carries `tlsEnabled`. Updating the old `onboard-helpers.ts` implementation will not make setup/dashboard/status links use the NetBird host or HTTPS after rebase.
  - confidence: 0.92
- **[P2] Preserve auto origin seeding:** `src/config/gateway-control-ui-origins.ts:4-7`
  - body: Current main includes `auto` in non-loopback origin seeding for container-default binds. Replacing that union while adding NetBird can stop seeding Control UI origins for container installs, causing startup/origin failures unrelated to NetBird.
  - confidence: 0.88
- **[P2] Seed the advertised NetBird origin:** `src/config/gateway-control-ui-origins.ts:35-38`
  - body: The PR advertises `https://<netbird-ip>:port`, but default origins only switch localhost and 127.0.0.1 to HTTPS. Because browser origin checks exact-match the scheme and host, the default NetBird UI path can still reject unless users manually add the NetBird allowed origin.
  - confidence: 0.9
