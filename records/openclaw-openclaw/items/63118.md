Keep open. Current main has lower-level session abort/delete/reset primitives, cleanup/export CLI support, and UI deletion/status metadata, but the requested public stop/kill/pause/resume lifecycle commands, Sessions UI lifecycle controls, and resource-limit enforcement are still not implemented.
