Keep open: current main and v2026.5.12 still block or skip unauthorized WhatsApp group traffic, but they do not add opt-in auto-leave, a group-join hook/event, or a WhatsApp leave-group message action. ## What This Changes

Add opt-in WhatsApp behavior to leave unauthorized groups, expose a group-join event or hook, and optionally add a WhatsApp leave-group message action.

## Best Possible Solution

Design this as an opt-in WhatsApp plugin feature with schema/docs, authorized group-leave behavior, join/participant event handling, minimal metadata exposure, focused tests, and coordination with https://github.com/openclaw/openclaw/issues/11489.
