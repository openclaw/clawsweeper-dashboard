## Summary

Keep open: current main and v2026.5.12 still only block or skip unauthorized WhatsApp group traffic and expose no auto-leave setting, group-join event, or WhatsApp leave-group action. Prior Codex/ClawSweeper reviews reached the same conclusion, and the broader related group approval request remains open at https://github.com/openclaw/openclaw/issues/11489.

## What This Changes

Request an opt-in WhatsApp auto-leave behavior for non-allowlisted groups, a group-join hook/event, and a manual WhatsApp leave-group message action.

## Best Possible Solution

Design this as an opt-in WhatsApp plugin feature with config/schema/docs, authorized Baileys-backed group-leave behavior, join/participant update detection or owner notification, and focused tests, coordinated with https://github.com/openclaw/openclaw/issues/11489.
