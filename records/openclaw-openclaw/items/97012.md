Full review comments:

- **[P1] Preserve redirect auth stripping for paginated fetches:** `src/plugin-sdk/provider-catalog-live-runtime.ts:290`
  - body: When the first catalog request reaches a different final origin after a permitted redirect, `fetchWithSsrFGuard` has already stripped Authorization for that redirect.
