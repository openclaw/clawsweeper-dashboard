## What This Changes

The PR changes Discord forum/media thread-name derivation to use `truncateUtf16Safe` instead of raw `slice(0, 100)` and adds focused tests for the thread-name helper.

## Best Possible Solution

Land the focused helper reuse once required CI is green, keeping Discord forum/media thread titles capped through the shared UTF-16-safe truncation helper.

## Solution Assessment
