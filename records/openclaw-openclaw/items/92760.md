## What This Changes

The issue reports that `openclaw status` resolves context windows through a local config-only copy and falls back to 200k for discovered models without configured `contextTokens`.

## Best Possible Solution

Keep this issue open while https://github.com/openclaw/openclaw/pull/92775 is reviewed; if that PR is rejected, land a narrower fix that delegates CLI status context-window resolution to the shared `src/agents/context.ts` resolver with focused regression coverage.
