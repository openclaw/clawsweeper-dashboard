Keep the fix Slack-owned unless a small generic seam is clearly required. Expected end state: normal replies, default message-tool sends from a Slack DM, and explicit user sends deliver successfully and mirror/attribute under the same direct Slack session in Control. Add regression tests for DM D-vs-U convergence, update `CHANGELOG.md` under Unreleased Fixes, and stop if the repair turns into a product/config policy change.
