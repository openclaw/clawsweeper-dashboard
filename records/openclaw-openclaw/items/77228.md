## What This Changes

The issue asks OpenClaw to prevent a malformed failed-turn transcript from causing GitHub Copilot Claude prefill errors, provider-wide auth-profile cooldown, and destructive session-file repair.

## Best Possible Solution

Land or replace the three focused fixes that exclude `format` from shared cooldown, drop trailing synthetic error assistant turns before replay, and make session-file repair drop structurally invalid message entries; then track any remaining fresh-session reset behavior separately.
