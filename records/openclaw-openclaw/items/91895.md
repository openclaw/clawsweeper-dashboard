## Reproduction Assessment

## Solution Assessment

Yes, this is a narrow maintainable fix: the runner marks only fallback-exhausted failures, while the gateway preserves retry grace for ordinary per-attempt errors. A more formal typed lifecycle event could be cleaner long term, but the patch matches the existing generic AgentEventPayload contract and adds focused tests.
