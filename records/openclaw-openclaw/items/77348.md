Review and land #77379, or a replacement, with a fix that removes the hot-path sync read while preserving installed-index freshness boundaries.

Yes. Source inspection on current main shows `resolvePluginTools()` can reach `loadOpenClawPlugins()`, whose cache context synchronously reads install records before checking the reusable runtime registry cache.
