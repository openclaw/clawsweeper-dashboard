## What This Changes

The issue requests that `memory_search` surface gateway-dispatched session transcript content when session memory and session sources are enabled, with QMD/builtin behavior and diagnostics made understandable.

## Best Possible Solution

Define the gateway/QMD session-recall contract explicitly: keep the archived-path fix, review or replace https://github.com/openclaw/openclaw/pull/80947 for diagnostics/docs, and only widen export or visibility behavior after maintainer privacy review.

## Reproduction Assessment

## Solution Assessment

Unclear as a single automatic fix. The narrow archived-path bug is already fixed, while the broader gateway-to-DM recall behavior needs maintainers to choose between stricter diagnostics/docs and intentionally broader transcript export or visibility.
