## Summary

The issue reports that `memory_search` does not surface gateway-dispatched session transcripts even when session memory and the `sessions` source are configured.

## Best Possible Solution

Define the supported session-recall contract first, keep privacy defaults conservative, then land focused diagnostics and regression/live proof for QMD export, visibility filtering, and gateway-to-DM recall.
