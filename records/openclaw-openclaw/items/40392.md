  - body: On the first config write after upgrade, existing `openclaw.json.bak.1` through `.bak.4` files enter this `else` branch and are unlinked as non-datetime orphans.   - confidence: 0.96

  - body: `formatBackupTimestamp()` has only millisecond precision, so two rotations in the same millisecond target the same path.   - confidence: 0.9
