- **[P1] Preserve legacy numeric backups during migration:** `src/config/backup-rotation.ts:147`
  - body: This branch deletes every non-datetime `.bak.*` file, including the shipped `.bak.1`-`.bak.4` rotation files.   - body: The destination name is only the millisecond timestamp, so two rotations with the same `now` value collide.   - confidence: 0.86
