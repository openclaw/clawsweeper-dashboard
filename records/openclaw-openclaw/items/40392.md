  - body: The PR classifies every non-datetime `.bak.*` as an orphan and unlinks it, so an existing user upgrading from the shipped numeric ring loses `.bak.1`-`.bak.4` on the first config write. Add a collision fallback or unique suffix before relying on timestamps for retention.
  - confidence: 0.87
