## Summary

Keep open: current main still lacks bounded EIO retry on the direct bootstrap paths, but this PR conflicts with the current kickstart-first restart lifecycle, adds PID-only SIGKILL escalation, and lacks after-fix macOS launchd proof.

## What This Changes

The PR adds launchd bootstrap EIO retries, PID-exit escalation, install/restart/repair call-site changes, and related unit coverage.

## Best Possible Solution

Rework the fix as bounded EIO-only retry on current bootstrap fallback, install, repair, and detached handoff paths while preserving kickstart-first restart behavior and avoiding force-kill unless the launchd job identity is verified.

## Reproduction Assessment

Yes for the blocking PR findings: source inspection shows the PR head still uses bootout/bootstrap restart and sends SIGKILL after PID-only liveness. The original EIO race is plausible from the PR's pre-fix trap test, but after-fix macOS launchd behavior has not been shown.

## Solution Assessment

No. The retry idea is useful, but the maintainable path is a current-main rework that keeps kickstart-first restart semantics and either removes force-kill escalation or verifies the launchd job before signaling.

## Review Findings

Overall correctness: patch is incorrect

Full review comments:

- **[P2] Preserve the kickstart restart lifecycle:** `src/daemon/launchd.ts:603-614`
  - body: Current main restarts normal LaunchAgents with `launchctl kickstart -k`, and tests assert the normal restart path does not call `bootout` or `bootstrap`. This branch still waits after `bootout` and then bootstraps in `restartLaunchAgent`, so rebasing it as-is would reintroduce the unloaded-service restart shape that main replaced; move the retry to actual bootstrap fallback, install, repair, or handoff paths while keeping kickstart restart.
  - confidence: 0.88

If the PID is reused between probes, this can kill an unrelated local process; drop force-kill or verify the process still belongs to the expected launchd job before signaling.
  - confidence: 0.86
