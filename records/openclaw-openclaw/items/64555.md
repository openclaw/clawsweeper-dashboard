## Summary

Keep open. Current main and the latest release still contain the WhatsApp auth-dir resolution patterns implicated by the report, and the target-repo fix PR is closed unmerged while the remaining fork candidate has not landed in openclaw/openclaw.

## What This Changes

The issue reports WhatsApp Baileys credentials resolving to the default credentials root instead of the active named profile state directory.

## Best Possible Solution

Review and land a focused WhatsApp plugin fix that lazy-resolves the default web auth dir, binds default/account/legacy auth-dir helpers to the active profile env or state dir, and adds a regression test for `OPENCLAW_STATE_DIR`.

## Reproduction Assessment

Yes. The issue provides concrete profile-gateway steps and logs, and current source inspection still shows default and account auth paths resolving from ambient OAuth state rather than an explicit active profile state dir.

## Solution Assessment

Yes. A plugin-owned path-resolution fix with focused regression coverage is the narrowest maintainable correction; documenting the `OPENCLAW_OAUTH_DIR` workaround alone would leave the isolation bug in place.

## Security Review

Status: needs_attention

Summary: Current main still contains credential path-resolution patterns that can break WhatsApp profile isolation.

Concerns:

- **[high] WhatsApp credentials can resolve outside the active profile:** `extensions/whatsapp/src/auth-store.ts:39`
  - body: The default WhatsApp auth export is computed from `resolveOAuthDir()` at module load, so an early light-runtime import can pin Baileys credentials under the default state dir instead of the named profile state dir.
  - confidence: 0.88
- **[medium] Account discovery lacks explicit profile state binding:** `extensions/whatsapp/src/accounts.ts:53`
  - body: Account discovery and default/legacy auth-dir helpers call `resolveOAuthDir()` without an explicit env or state dir, so discovery can drift from the profile boundary promised by the multiple-gateways docs.
  - confidence: 0.84
