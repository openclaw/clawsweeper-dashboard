- **[P2] Call the current heap snapshot MCP tools:** `extensions/browser/src/browser/chrome-mcp.ts:2327`

  - confidence: 0.94

The upstream `--categoryExtensions` contract says autoConnect/browserUrl/wsEndpoint are not supported for that category, so the new default can expose an extension route that fails for the very managed profile class it enables.
  - confidence: 0.82
