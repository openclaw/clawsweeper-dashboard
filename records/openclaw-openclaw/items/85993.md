- **[P1] Wrap Chrome MCP debug route results before model return:** `extensions/browser/src/browser-tool.ts:515`
  - body: The new debug-route actions all return through `executeBrowserRouteAction`, which currently does `jsonResult(result)` for outputs such as console-message, request-detail, third-party/WebMCP tool execution, traces, heap/Lighthouse data, and extension results.
