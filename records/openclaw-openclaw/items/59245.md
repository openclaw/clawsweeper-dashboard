## Summary

## What This Changes

Request an autonomous outbound Voice Call task workflow that can place phone calls, navigate IVR/hold/callback situations, escalate to the user, share only authorized information, and use natural speech.

## Best Possible Solution

Define a Voice Call plugin task-call layer on top of the existing initiate/continue/speak/DTMF/status/realtime primitives, with a task objective schema, call state machine, IVR/hold strategy, trusted user escalation, allowed-information policy, cost/duration caps, transcript reporting, and callback/reschedule semantics.
