## Summary

Keep this PR open, but it is not merge-ready: the proposed audit can miss enabled plugins, can warn on disabled plugins, and the new public capability/security contract needs maintainer direction plus real behavior proof.

## What This Changes

The PR adds a `PluginManifest.capabilities` declaration model, a `plugins.capabilities.legacy_unrestricted` security audit warning, and an RFC for phased runtime enforcement.

## Best Possible Solution

Define the capability schema through security design, implement the audit against resolved enabled plugin manifests, add focused tests/docs, and include real `openclaw security audit` output proof.

## Reproduction Assessment

Yes. Source inspection is enough for the PR-introduced audit defects: current helpers return extension basenames, the manifest loader expects a plugin root, and the PR scans all discovered directories instead of the enabled plugin set.

## Solution Assessment

No. A declaration-only phase may be reasonable, but this branch should first resolve enabled manifest records correctly and get maintainer security/product approval for the public capability contract.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Resolve plugin manifests from the extensions root:** `src/security/audit-extra.async.ts:833`
  - body: `pluginDirs` are directory basenames, but this passes each basename directly to `loadPluginManifest`. Since that loader resolves `openclaw.plugin.json` under the supplied root, installed plugins under the extensions state directory will usually be skipped and the new security warning will not fire.
  - confidence: 0.94
- **[P2] Scope legacy capability warnings to enabled plugins:** `src/security/audit-extra.async.ts:831-834`
  - body: This loop scans every discovered extension directory instead of the enabled plugin set. A globally disabled, denied, or per-entry disabled plugin can be reported as running in legacy unrestricted mode, producing false security audit output.
  - confidence: 0.88
