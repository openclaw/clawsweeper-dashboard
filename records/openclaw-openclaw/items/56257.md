## Summary

Keep open: current main does not implement this manifest capability model, and the PR still needs maintainer security/product approval plus audit correctness fixes before it can merge.

## What This Changes

The branch adds an optional plugin-manifest capabilities field with normalizers, a legacy-unrestricted plugin audit warning, and an RFC for phased capability enforcement.

## Best Possible Solution

Define the capability schema with security ownership, implement the audit in the current trust-audit module using resolved enabled plugin manifests, and add focused tests plus public docs before merging.

## Reproduction Assessment

Yes. Source inspection gives a high-confidence path: an installed plugin under `<stateDir>/extensions/demo/openclaw.plugin.json` can be skipped when the PR passes only `demo` to `loadPluginManifest`, and disabled installed plugins are still scanned from the raw directory list.

## Solution Assessment

No. A declaration-only phase may be reasonable, but this patch is not the best implementation until it fixes manifest path resolution, scopes findings to enabled plugins, rebases onto the current audit module, and gets maintainer security/product approval.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Resolve installed manifest paths before loading:** `src/security/audit-extra.async.ts:831`
  - body: `pluginDirs` are names from the extensions directory, not plugin roots. Passing a bare name to `loadPluginManifest` resolves `openclaw.plugin.json` under the process cwd, so real installed legacy plugins can be skipped and the new warning may never fire.
  - confidence: 0.94
- **[P2] Scope legacy capability warnings to enabled plugins:** `src/security/audit-extra.async.ts:829`
  - body: The loop scans every discovered extension directory, including globally disabled, denied, or per-entry disabled plugins. The finding says those plugins run without capabilities, so inactive plugins can produce false security warnings.
  - confidence: 0.88
