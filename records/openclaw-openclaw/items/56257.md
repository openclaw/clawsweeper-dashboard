## Summary

Keep this PR open, but do not merge it as-is: current main still lacks the proposed plugin-manifest runtime capability declaration, while the PR's audit implementation can miss active installed plugins, warn on inactive plugins, lacks real behavior proof, and needs security/product approval for a new public plugin security contract.

## What This Changes

This PR adds an optional plugin-manifest `capabilities` field with normalizers, a `plugins.capabilities.legacy_unrestricted` audit warning, and an RFC for phased runtime capability enforcement.

## Best Possible Solution

Define the capability schema through a maintainer-approved security design, implement the audit in the current trust-audit module using resolved enabled plugin manifests, add focused tests and public docs, then provide real `openclaw security audit` output proof.

## Reproduction Assessment

Yes. Source inspection gives a high-confidence reproduction path: current helpers return installed plugin basenames, while the PR passes those basenames directly to `loadPluginManifest`, and the PR loops all discovered plugin directories instead of the enabled set.

## Solution Assessment

No. A declaration-only phase may be reasonable, but this patch is not the best implementation until it resolves installed manifest paths, scopes the finding to enabled plugins, rebases onto the current audit split, and gets security/product approval.

## Review Findings

Overall confidence: 0.9

Full review comments:

- **[P1] Resolve plugin manifests from the extensions root:** `src/security/audit-extra.async.ts:833`
  - body: `pluginDirs` are basenames from the extensions directory, but this passes each basename directly to `loadPluginManifest`.   - confidence: 0.94

  - body: This scans every discovered plugin directory, including globally disabled, denied, or per-entry disabled plugins. The finding says those plugins run without capabilities, so inactive plugins can produce false security warnings and reduce operator trust in the audit output.
  - confidence: 0.88
