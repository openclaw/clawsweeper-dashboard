Overall correctness: patch is incorrect

Full review comments:

- **[P1] Derive the coordinator instead of hard-coding main:** `extensions/feishu/src/monitor.ts:38`
  - body: The arbitration only treats an account literally named `main` as the coordinator, but current Feishu config can choose a different default account or have no `main` account at all. In those setups multi-specialist mentions can skip every mentioned account or fail to route once.
  - confidence: 0.91
- **[P2] Avoid substring text mentions for gating:** `extensions/feishu/src/bot.ts:256-260`
  - body: The implicit fallback uses `normalized.includes(pattern)`, so `@main` can match text like `@mainbot` or `@maintenance` and mark an unaddressed group message as mentioned. That can bypass mention-required gating and cause unexpected dispatches.
  - confidence: 0.86
