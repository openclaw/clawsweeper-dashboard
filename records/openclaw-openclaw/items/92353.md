The issue requests cwd-aware automatic loading of nested AGENTS.md / CLAUDE.md files into session context.

## Best Possible Solution

Keep this open for maintainer product review of a bounded, deterministic workspace-root-to-cwd context-discovery contract that applies to the normal bootstrap path without duplicating or weakening existing workspace guards.
