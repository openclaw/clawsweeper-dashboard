  - body: The PR checks `preflight.isWakeReason`, but `HeartbeatPreflight` only carries `isWakePayload`, `isExecEventWake`, and `isCronWake`. This should fail strict TypeScript checking; use the existing wake flag instead.

  - confidence: 0.91

  - body: The refactor removes the `heartbeatPlugin.heartbeat.checkReady` gate before the main `deliverOutboundPayloads` call, so alert heartbeats can attempt delivery through a channel that explicitly reports it is not ready.
  - confidence: 0.84

  - body: Assigning `log.isEnabled = () => false` does not silence `log.info()`, because the subsystem logger methods call `emitLog()` directly. Keep the old no-op wrapper or guard the calls.
  - confidence: 0.86
