## Summary

This PR addresses a reproduced shipped regression at its source: current main terminates the agent loop after a qualifying message-tool-only source send, while the patch records delivery without terminating and suppresses only subsequent duplicate presentation. The PR changes message-tool-only source replies from terminal sends into attempt-scoped delivery evidence so follow-up tools continue while duplicate assistant, reasoning, block, and terminal replies are suppressed.

## Best Possible Solution
