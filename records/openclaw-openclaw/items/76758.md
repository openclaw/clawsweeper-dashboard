## Summary

Keep open. Current main has provider transcript replay hooks and live plugin tool execution, but no host-side Clock or frozen RPC snapshot contract; this is a new public SDK/runtime design tied to #76756, not conservative cleanup material.

## What This Changes

Request a public plugin-sdk replay determinism contract with injectable Clock support, frozen host RPC snapshots during replay, and reusable byte-diff replay testing helpers.

## Best Possible Solution

Keep this open for a maintainer design pass that defines a versioned plugin replay contract and coordinates it with the gateway RPC accessor work.

## Solution Assessment

Unclear; the direction is plausible, but the specific Clock plus frozen RPC shape should be chosen with public SDK versioning and #76756 rather than implemented as an isolated small fix.
