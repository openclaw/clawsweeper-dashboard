## Summary

Keep open. Current main still lacks the host-side Clock injection and frozen host-RPC replay contract requested; existing code executes plugin tools live and only provides provider transcript replay policy, so this is public SDK/runtime design rather than conservative cleanup.

## What This Changes

Request a public plugin-sdk/runtime contract for deterministic plugin tool-call replay, including Clock injection, frozen host-RPC snapshots, and reusable replay-output tests.

## Best Possible Solution

Keep this open for a maintainer-owned SDK/runtime RFC that coordinates Clock injection, frozen read-only gateway snapshots, and plugin replay test helpers with #76756 and #76759.
