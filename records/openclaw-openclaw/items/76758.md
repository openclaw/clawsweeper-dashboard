## What This Changes

Request a plugin SDK and host replay contract with injected clocks, frozen Gateway RPC snapshots during replay, and reusable record/replay byte-diff testing for plugin tool calls.

## Best Possible Solution

Design a narrow plugin SDK replay contract that adds a generic clock/runtime snapshot seam and integrates only allowlisted read-only Gateway access with deterministic recorded values, then document and test plugin record/replay behavior.
