## Summary

Keep open. Current main has provider transcript replay helpers and live plugin tool execution, but no host-side Clock injection or frozen RPC snapshot contract; this is new public SDK/runtime design work tied to #76756.

## What This Changes

Request a deterministic plugin tool-call replay contract with SDK Clock injection, frozen replay-time host RPC results, and reusable record/replay test helpers.

## Best Possible Solution

Keep this open for a maintainer design pass that defines a versioned plugin replay contract and coordinates it with gateway RPC access and public type exports.

## Reproduction Assessment

Not applicable; this is an RFC-shaped feature request, not broken existing behavior with an established contract. Source and docs inspection show the requested host-side replay contract is absent on current main.

## Solution Assessment
