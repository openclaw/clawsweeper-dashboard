## Summary

Keep open. Current main still lacks a plugin-facing Clock injection surface or frozen host-RPC snapshot replay contract, and the request is a public SDK/runtime design rather than conservative cleanup.

## What This Changes

Request a host-side plugin tool-call replay contract with standard Clock injection, frozen replay-time RPC results, and a reusable record/replay test helper.

## Best Possible Solution

Keep this open for a maintainer design pass that defines a versioned plugin replay contract and coordinates it with the gateway RPC accessor and SDK type-export work.

## Solution Assessment
