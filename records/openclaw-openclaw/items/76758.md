## Summary

Keep open. Current main still lacks a host-side Clock, frozen RPC snapshot, or plugin tool-call replay harness; this is a new public SDK/runtime design tied to the still-open gateway RPC accessor work, not conservative cleanup material.

## What This Changes

Request a plugin SDK/runtime replay contract with Clock injection, frozen host RPC snapshots, and a reusable plugin tool-output replay harness.

## Best Possible Solution

Keep this open for a maintainer design pass that defines a versioned plugin replay contract and coordinates it with the gateway RPC accessor/type export issues.
