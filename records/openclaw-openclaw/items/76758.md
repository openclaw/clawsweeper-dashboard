## Summary

Keep open. Current main still lacks a host-side Clock, frozen RPC snapshot, or plugin tool-call replay contract; this is new public SDK/runtime design tied to #76756 and #76759, not conservative cleanup material.

## What This Changes

The issue requests a deterministic plugin tool-call replay contract with SDK Clock injection, frozen host-RPC snapshots during replay, and reusable plugin replay tests.

## Best Possible Solution

Keep this open for a maintainer design pass that defines a versioned plugin replay contract and coordinates it with gateway RPC access and public type exports.
