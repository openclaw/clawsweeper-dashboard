Keep open. Current main and the latest release still make Feishu WebSocket inherit ambient proxy env, while the open closing PR remains unmerged and the later closed PR did not land on main.

## What This Changes

Make Feishu WebSocket bypass ambient proxy env by default and use proxying only through an explicit opt-in.

## Best Possible Solution
