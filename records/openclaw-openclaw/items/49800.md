## Review Findings

Overall correctness: patch is incorrect

Full review comments:

- **[P1] Normalize same-profile retry aborts:** `src/agents/pi-embedded-runner/run.ts:935`
  - body: This new retry helper awaits `sleepWithAbort(...)` directly. If the run is canceled during the backoff, `sleepWithAbort` throws a plain `Error("aborted")`, while the outer fallback runner only treats `name === "AbortError"` as user cancellation; this can turn a canceled run into extra fallback/API attempts. Wrap this sleep the same way `maybeBackoffBeforeOverloadFailover` does and add cancellation coverage.
  - confidence: 0.9
