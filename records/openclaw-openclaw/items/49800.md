## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Normalize retry sleep aborts to AbortError:** `src/agents/pi-embedded-runner/run.ts:935`
  - body: The new same-profile overload retry path awaits `sleepWithAbort` directly, which rejects cancellation as a plain `Error("aborted")`. The outer fallback runner only treats `name === "AbortError"` as user cancellation, so a stopped run during this backoff can be misclassified as a normal candidate failure and continue into fallback/API attempts; wrap this sleep like the existing overload-failover backoff.
  - confidence: 0.9
