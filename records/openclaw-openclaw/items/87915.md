- **[P1] Keep harness roles out of CustomAgentMessages:** `packages/agent-core/src/types.ts:332-337`
  - body: The PR turns the exported base CustomAgentMessages interface into a built-in map for bashExecution, custom, branchSummary, and compactionSummary.   - confidence: 0.88

  - body: Adding media-understanding-common to the pre-clean list makes the build wrapper remove packages/media-understanding-common/dist, but that package tracks those dist files and declares them as exported types. Flows such as run-node and Docker packaging set OPENCLAW_RUN_NODE_SKIP_DTS_BUILD=1, so a skipped or failed DTS rebuild can leave the declared files missing; exclude tracked outputs or preserve/regenerate them before skip-DTS paths can observe the cleaned tree.
  - confidence: 0.9

  - body: listTsdownOutputRoots returns POSIX-style tsdown outDir strings like packages/agent-core/dist, but path.join("packages", "agent-core", "dist") uses backslashes on Windows. This assertion can fail before it exercises cleanup behavior; compare against POSIX literals or normalize both sides.
