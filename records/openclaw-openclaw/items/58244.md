Full review comments:

- **[P2] Preserve suspended dynamic lanes:** `src/process/command-queue.ts:434`
  - body: Current main suspends session lanes with `setCommandLaneConcurrency(params.laneId, 0)`.   - confidence: 0.91
- **[P2] Notify active-task waiters on rejection:** `src/process/command-queue.ts:298`
  - body: The catch path removes the active task id but now calls only `pump(true)`. `waitForActiveTasks()` relies on `notifyActiveTaskWaiters()` to resolve restart-drain waiters, so rejected or timed-out lane work can leave drain callers waiting until timeout after the active set is already empty.
  - confidence: 0.88
