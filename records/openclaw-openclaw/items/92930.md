- **[P1] Keep catalog warmup off the startup critical path:** `src/gateway/server-startup-early.ts:131`
  - body: This awaits `ensureContextWindowCacheLoaded()` during `startGatewayEarlyRuntime`, but that promise loads the model catalog and all bundled provider static catalogs with no timeout.
