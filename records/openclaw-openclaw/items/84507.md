## Best Possible Solution

Use one strict positive-whole-integer parser for both skills and plugins search, add focused command-level regression coverage, and keep the linked issue open until both reported commands fail fast.

## Reproduction Assessment

Yes. Source inspection gives a high-confidence path: current main registers both affected `--limit` options with `Number.parseInt`, and plugins search still clamps bad parsed values instead of rejecting the original input.

## Solution Assessment
