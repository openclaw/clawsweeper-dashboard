## Summary

Keep open. Current main still has only one Feishu group mention gate, the requested `topicRequireMention` key is absent from source and docs, and open PR #76020 is already linked with closing syntax as the active implementation candidate.

## What This Changes

Request separate Feishu mention requirements so normal group messages can require @mention while topic/thread replies can continue without one.

## Best Possible Solution

Land or replace PR #76020 with a Feishu-owned additive config implementation that passes topic/thread context into mention-policy resolution and preserves current defaults when unset.
