## Summary

Keep open. Current main still has one Feishu group mention gate for both normal group messages and topic/thread replies, and open PR #76020 specifically targets this request with closing syntax, so this issue should wait for that PR to merge or be replaced.

Request support for a Feishu `topicRequireMention` setting so normal group messages can require @mentions while topic/thread replies can continue without them.
