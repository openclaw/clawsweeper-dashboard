## Summary

Keep open. Current `main` still uses one Feishu group mention policy for normal messages and thread replies, `topicRequireMention` is absent from schema/docs/tests, and open PR #76020 uses closing syntax for this issue, so the issue should wait for that PR to land or be replaced.

## What This Changes

The issue requests a Feishu configuration split so normal group messages can require an @mention while topic/thread replies can continue without one.

## Best Possible Solution

Review and land or replace PR #76020 with Feishu-owned additive `topicRequireMention` handling across schema, policy, bot gating, docs, tests, and changelog while preserving current defaults when unset.
