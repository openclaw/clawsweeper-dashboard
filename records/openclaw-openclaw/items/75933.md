## What This Changes

The PR changes Codex conversation image input conversion to decode usable local `file://` media paths with `safeFileURLToPath`, fall back to remote image URLs when local file URLs are unusable, and adds regression tests plus a changelog entry.

## Best Possible Solution

Merge a Codex-owned adapter fix that keeps the normalization local to the plugin and relies on the existing public SDK file-access helper.
