## What This Changes

The PR adds SHA-256 fingerprints to native ClawHub skill lockfiles, exposes verifier helpers, wires non-deep security audit findings, and adds a skill-integrity proposal document.

## Best Possible Solution

Rebase and redesign this as security-reviewed partial hardening aligned with https://github.com/openclaw/openclaw/issues/12507: trusted metadata, verifier-error findings, runtime-relevant content hashing, documented audit-only limits, and proof of install/audit behavior.
