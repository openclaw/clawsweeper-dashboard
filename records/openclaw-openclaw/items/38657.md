## What This Changes

The issue reports that Onboard appears to save provider edits but does not overwrite existing provider/auth/model configuration values.

## Best Possible Solution

Define explicit user-initiated Onboard update semantics that update canonical config and generated agent state consistently while preserving normal merge-mode and SecretRef behavior.
