## Best Possible Solution

Run a Windows packaged smoke that loads `dist/extensions/minimax/index.js`, `dist/extensions/google/index.js`, and `dist/extensions/talk-voice/index.js` under the default Node stack, then close with proof or land the narrow Gateway/plugin loader fix it exposes.

## Solution Assessment
