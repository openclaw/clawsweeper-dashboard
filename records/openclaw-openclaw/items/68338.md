The PR adds one-line continuation parsing for empty `MEDIA:` directives in `src/media/parse.ts` and focused parser tests in `src/media/parse.test.ts`.

## Best Possible Solution

Merge the continuation parser support after adding the changelog entry and validating against current main, while keeping #75182 as a separate parser-hardening track unless maintainers intentionally combine them.
