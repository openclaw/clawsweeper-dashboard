Keep open. Current main still lacks the requested dashboard-wide shortcut dispatcher and Control UI New Agent wizard; the implementation only covers Cmd/Ctrl+K, textarea-local chat keys, click-only new-message scrolling, lower-level agent RPCs, and a CLI agent helper.
