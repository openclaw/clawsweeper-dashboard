## Summary

Keep open. Current main still lacks the requested dashboard-wide shortcut dispatcher and Control UI New Agent wizard; existing support is limited to Cmd/Ctrl+K, textarea-local chat keys, click-only new-message scrolling, CLI agent creation, and lower-level Gateway RPCs.

## What This Changes

Add dashboard-level keyboard shortcuts and a Control UI workflow for creating agents from the dashboard.

## Best Possible Solution

Split or explicitly scope this into two Control UI enhancements: a guarded dashboard shortcut dispatcher for chat/navigation actions and a New Agent wizard that composes existing agent, file, config, model, tool, and session APIs.

## Reproduction Assessment

Not applicable as a bug reproduction; this is an enhancement request. Source inspection on current main gives high-confidence verification that the requested dashboard shortcuts and dashboard New Agent wizard are still absent.

## Solution Assessment

No current implementation solves this yet. The maintainable path is to split or scope the two UI features and reuse existing Gateway/CLI contracts instead of creating a parallel agent/session creation path.
