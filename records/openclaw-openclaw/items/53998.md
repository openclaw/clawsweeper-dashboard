## Summary

Keep open. Current main still matches the reported secrets-audit false-positive paths for bare `$VAR` strings and auth-profile refs, and the mixed OAuth/static auth-profile apply path still throws instead of gracefully skipping. ## What This Changes

The issue requests that secrets audit distinguish env-var references from plaintext, scrub migrated plaintext with rotation guidance, and handle mixed OAuth/static auth profiles without aborting.
