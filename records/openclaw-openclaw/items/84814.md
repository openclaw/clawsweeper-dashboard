## Reproduction Assessment

Yes. Current main is source-reproducible: an embedded non-GPT run with only an unclassified `isError` payload returns null from the classifier, and null classification is treated as a successful fallback candidate.

## Solution Assessment

Yes. The PR reuses the existing failover matcher at the embedded-result boundary, which is narrower than retrying every arbitrary `isError` payload and preserves the current no-fallback path for unclassified non-GPT error text.
