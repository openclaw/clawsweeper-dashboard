  - confidence: 0.9

  - confidence: 0.84
- **[P2] Keep structured cron payloads on direct delivery:** `src/cron/isolated-agent/delivery-dispatch.ts:499-503`
  - body: The new direct-delivery gate ignores `deliveryPayloadHasStructuredContent`, so structured payload jobs can fall through the announce/text path when cleanup, follow-up, or silent-token conditions are true. That path only forwards synthesized text, so media or channel-data cron outputs can be dropped.
