  - confidence: 0.84

  - body: The new gate no longer forces structured payloads through direct delivery. If `deleteAfterRun`, follow-up, or silent-token conditions are true, media/channel payloads fall into the announce path, which only forwards synthesized text and can drop the structured output.

  - body: When `runSubagentAnnounceFlow` reports no delivery, the non-best-effort path still returns `status: "ok"` with an error string. That makes failed required delivery look successful to cron scheduling and can delete one-shot jobs without delivering the message.
  - confidence: 0.86
