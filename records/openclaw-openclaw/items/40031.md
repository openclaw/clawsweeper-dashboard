## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Preserve durable direct cron delivery:** `src/cron/isolated-agent/delivery-dispatch.ts:220-232`
  - body: This branch sends direct cron payloads through plain `deliverOutboundPayloads`. Current main uses `sendDurableMessageBatch` with required durability, idempotency, transient retry, partial-failure handling, and cleanup, so merging this as-is would roll back scheduled delivery reliability.
  - confidence: 0.92
- **[P1] Fail cron runs when announce delivery fails:** `src/cron/isolated-agent/delivery-dispatch.ts:409-418`
  - body: When `runSubagentAnnounceFlow` returns false or throws, the non-best-effort path returns `status: "ok"` with `delivered: false`. One-shot jobs can then be recorded successful or deleted even though no message was delivered.
  - confidence: 0.88
- **[P2] Treat the default channel as non-explicit:** `src/cli/cron-cli/register.cron-add.ts:221-228`
  - body: `--channel` defaults to `last`, so `hasDeliveryTarget` is truthy even when the user did not request announce delivery. Main system-event jobs therefore become announce jobs by default, and `--no-deliver` can still be overridden by the default channel.
  - confidence: 0.9
- **[P2] Keep structured cron payloads on direct delivery:** `src/cron/isolated-agent/delivery-dispatch.ts:499-503`
  - body: The direct-delivery gate ignores whether the payload has structured content, so media or channelData outputs fall back to the announce/text path whenever cleanup or follow-up conditions apply. That path only forwards synthesized text, so structured cron outputs can be dropped.
  - confidence: 0.84
- **[P2] Preserve heartbeat defaults when fields are omitted:** `src/cron/service/timer.ts:1077-1083`
  - body: The override includes `to` and `accountId` even when they are undefined, and the runner later spreads it over configured heartbeat defaults. Main cron announce jobs that rely on configured recipients or accounts can lose those defaults and route to the wrong place or no target.
  - confidence: 0.82
