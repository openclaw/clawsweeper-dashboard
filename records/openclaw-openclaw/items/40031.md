  - body: When `runSubagentAnnounceFlow` returns false or throws, the non-best-effort path still returns `status: "ok"` with an error field.   - confidence: 0.88

  - body: The new `useDirectDelivery` gate can send jobs with structured payloads through the announce path whenever `deleteAfterRun` is true or descendant-followup heuristics fire. `deliverViaAnnounce` only propagates text, so image/file/channelData cron outputs can be dropped for jobs that previously used direct outbound delivery.
  - confidence: 0.86

  - body: The heartbeat override includes `to` and `accountId` even when they are undefined, and the wake runner later spreads that over the configured heartbeat.   - confidence: 0.82
