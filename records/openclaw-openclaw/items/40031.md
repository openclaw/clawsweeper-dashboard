  - body: `--channel` defaults to `last`, so `hasDeliveryTarget` is truthy even when the user did not request delivery. Main system-event jobs therefore get `delivery.mode="announce"` by default, and `--no-deliver` can still be overridden by the default channel.

  - confidence: 0.84

  - body: The new direct-delivery gate ignores `deliveryPayloadHasStructuredContent`, so structured payload jobs can fall through the announce/text path when delete-after-run, follow-up, or silent-token conditions are true.   - confidence: 0.88
