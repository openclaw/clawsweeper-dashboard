## What This Changes

## Best Possible Solution

## Solution Assessment

Yes, likely: updating both availability and submit eligibility keeps the Control UI paths aligned around generated `configRaw`; explicit `config.apply` generated-raw coverage would reduce the remaining symmetry risk.
