Keep open. Current main still lacks the Control UI composer safeguards and numeric-only regression coverage, and open PR #74369 is already the active closing implementation path.

## What This Changes

## Best Possible Solution

Land PR #74369, or an equivalent narrow composer-level fix with regression tests, without adding Gateway deformatting that could alter intentional hyphenated input.
