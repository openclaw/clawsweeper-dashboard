## Security Review

Status: needs_attention

Summary: The diff introduces low-severity path-disclosure and log-amplification concerns in the new diagnostics.

Concerns:

- **[low] Redact or downgrade path-heavy skill diagnostics:** `src/agents/skills/workspace.ts:440`
  - body: The new warning and summary logs include absolute `filePath`/`workspaceDir` values at warn/info level. In shared or collected logs those paths can expose usernames, repo names, and internal directory layout, so they should be compacted/redacted or moved to debug with a safer summary.
