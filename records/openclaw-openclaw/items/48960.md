- **[P2] Assert against formatted logger strings:** `src/agents/skills/workspace.test.ts:49-52`
  - body: `createSubsystemLogger` writes formatted console lines as strings, not objects with a `message` property. These assertions can fail while the diagnostic is actually emitted, so the added tests do not prove the intended behavior.
  - confidence: 0.91

  - body: `skillEntries` is derived from `merged.values()`, so `totalSkills` always equals `uniqueSkills` here. The summary still cannot show pre-deduplication versus post-deduplication counts, which is one of the diagnostics this PR is trying to add.
  - confidence: 0.9

## Security Review

Status: needs_attention

  - confidence: 0.78
