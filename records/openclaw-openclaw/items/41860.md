## Summary

Keep open. Current main and latest release still pass raw text into Feishu post `md` payloads for sends and edits, and #68514 is the active fix candidate rather than a reason to close before merge.

## What This Changes

Feishu outbound messages containing bare URLs with underscores should render the entire URL as one clickable link.

## Best Possible Solution

Land one Feishu plugin-local fix, likely #68514 or an equivalent narrow patch, that preserves normal Markdown while making bare underscore URLs fully clickable in send and edit post payloads.
