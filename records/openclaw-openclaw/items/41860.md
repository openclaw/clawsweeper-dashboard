## Summary

Keep open. Current main still sends Feishu outbound text and text edits through the raw post `md` payload with `messageText` unchanged, so the reported underscore URL rendering problem remains; #68514 is the active fix candidate to review.

## What This Changes

Feishu outbound text containing bare URLs with underscores should render the full URL as one clickable link.

## Best Possible Solution

Land one Feishu plugin-local fix, likely #68514 or an equivalent narrow patch, that preserves normal Markdown while making bare underscore URLs fully clickable in send and edit post payloads, then close this issue after merge.
