  - body: `agent` is not admin-only, so `baseScopes` is normally undefined here. With `allowModelOverride` true this leaves `scopes` undefined, then `callGateway` later negotiates only least-privilege write scope, so the gateway still rejects the forwarded provider/model on the out-of-process path.

  - body: Passing `allowSyntheticModelOverride` only helps the synthetic client.   - confidence: 0.88
- **[P2] Fix the test to match the mocked resolver:** `src/agents/subagent-spawn.test.ts:1130-1131`
