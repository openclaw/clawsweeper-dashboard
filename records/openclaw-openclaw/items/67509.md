The PR adds a CLI startup guard that exits when OpenClaw runs with uid/euid 0, plus an explicit root override, a forwarded-container bypass, legacy entrypoint coverage, and unit tests.

## Best Possible Solution

## Reproduction Assessment

## Solution Assessment
