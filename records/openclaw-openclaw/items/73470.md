Keep open. Current main still has advisory prompt guidance, read/web result truncation metadata, and incomplete-turn recovery, but it does not implement a mandatory completeness scan guardrail across read/search operations. The issue is also paired with open unmerged PR #73618 using closing syntax, so maintainers should review, land, replace, or close that PR before this issue is closed.

## What This Changes

## Best Possible Solution

Keep this issue open while maintainers review PR #73618 or a replacement. A durable implementation should define which read/search surfaces are covered, preserve partial or truncated result state where needed, and test that the agent either follows up or clearly reports blocked/partial evidence before finalizing.
