## Summary

Keep open: current main and the latest stable release still print the doctor cleanup preview without `--fix-missing`, while cleanup only counts missing transcript entries when that flag is set. The open linked PR https://github.com/openclaw/openclaw/pull/83630 is the right fix path, but it has not merged yet.

## What This Changes

The issue asks for the missing-transcript doctor warning to suggest a cleanup preview that actually includes missing-transcript pruning, or otherwise align cleanup dry-run behavior with the doctor warning.

## Best Possible Solution

Land https://github.com/openclaw/openclaw/pull/83630 or an equivalent focused change that adds `--fix-missing` only to the doctor preview hint and keeps cleanup pruning opt-in.

## Reproduction Assessment
