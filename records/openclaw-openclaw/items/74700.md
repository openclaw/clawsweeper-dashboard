- **[P2] Keep ACP run timeout enforcement:** `src/agents/acp-spawn.ts:953-957`
  - body: Removing `runTimeoutSeconds` from `runtimeOptions` means `AcpSessionManager.resolveTurnTimeoutMs` falls back to the configured agent timeout.
