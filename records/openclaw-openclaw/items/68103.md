  - confidence: 0.9

  - confidence: 0.86

  - confidence: 0.88

  - body: The branch reads `ctx.config?.models?.nvidiaDiscovery?.enabled`, but that key is not part of the current config contract or plugin schema. Either declare a typed plugin config/schema surface or remove the undeclared compatibility path.
