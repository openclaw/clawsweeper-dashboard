## Best Possible Solution

Land a repaired current-main patch that preserves surrogate pairs at the coarse chunk boundary and sanitizes both top-level and structured embedding text before provider calls, with focused regression coverage.

## Solution Assessment

Mostly yes. The chunker boundary guard is now aimed at the active runtime implementation, and the embed-boundary sanitizer is the right narrow direction, but the PR should also sanitize structured text parts before embedBatchInputs.
