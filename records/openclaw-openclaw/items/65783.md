## Summary

## What This Changes

The PR changes memory chunking and embedding input handling to preserve UTF-16 surrogate pairs, replace unpaired surrogates before embedding calls, and add focused regression tests.

## Best Possible Solution

Land a current-main patch that fixes the coarse chunk boundary, sanitizes every text-bearing embedding field including structured `parts[].text`, and keeps the broader retry/checkpointing reports separate.
