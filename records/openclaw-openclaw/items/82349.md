## Summary

## What This Changes

Adds a bundled `skills-setup` plugin that registers an admin-only `skills.setup` Gateway RPC, parses setup metadata from installed skills, resolves skill paths, applies env overlays, clamps timeout config, updates labeler metadata, and adds focused tests.

## Best Possible Solution

Keep the ClawHub plugin as the external experimentation path, and only merge core work after maintainers choose the setup boundary, the env/path behavior uses shared or purpose-built SDK security helpers, and real gateway proof is added.
