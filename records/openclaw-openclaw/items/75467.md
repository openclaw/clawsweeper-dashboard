## Best Possible Solution

Review and land or reject #75476; the desired end state is generic OpenAI tool-schema normalization honoring `ModelCompatConfig`, Fireworks unsupported-keyword metadata, and regression coverage for strict and non-strict paths.

## Solution Assessment
