  - body: `resolveGoogleInlineAudioPath` anchors trust at `process.cwd()`, while current embedded turns already pass `effectiveWorkspace` into `applyExtraParamsToAgent`. Google's audio docs frame this as total request size including files, prompt text, and system instructions, so a near-limit raw file can still produce an oversized request; leave headroom, check encoded total size, or use the Files API.   - confidence: 0.76
