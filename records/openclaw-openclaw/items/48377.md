## Summary

Keep open. Current main still lacks Feishu create-with-content support, while this PR is a focused plugin-boundary fix with no blocking functional or security finding from this sweep; the remaining action is maintainer review, mergeability/check confirmation, and landing.

## What This Changes

The PR adds optional `content` support to Feishu `feishu_doc` create, writes provided markdown after document creation including empty strings, omits `blocks_deleted` from create responses, and updates Feishu skill docs, regression tests, and the changelog.

## Best Possible Solution
