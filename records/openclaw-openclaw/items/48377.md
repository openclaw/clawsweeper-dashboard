## Best Possible Solution

Full review comments:

- **[P2] Preserve created metadata when content write fails:** `extensions/feishu/src/docx.ts:1478-1485`
  - body: After `createDoc` succeeds, any `writeDoc` conversion, insert, image, or media failure falls into the outer catch and returns only `{ error }`. Preserve the created metadata in the error response or roll back the new document before returning failure.
