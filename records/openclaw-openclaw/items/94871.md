  - body: The new async cached readers are added here, but the external CLI auth provider table still calls the sync Codex and Claude cached readers. That path can still reach the execSync keychain helpers, so the Gateway auth sync path named in the linked bug remains blocking after merge.

  - confidence: 0.84
