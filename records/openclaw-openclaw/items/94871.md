## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Remove unsupported exec option from async shell call:** `src/agents/sessions/resolve-config-value.ts:137-140`
  - body: `child_process.exec` options do not include `stdio` in the Node types used by this repo; this new object literal is likely to fail strict type checking. Drop the unsupported option or switch to an async API whose options support stdio control.
  - confidence: 0.86
- **[P1] Wire external CLI sync to async readers:** `src/agents/cli-credentials.ts:837-843`
  - body: The new async cached readers are added here, but the external CLI auth provider table still calls the sync Codex and Claude cached readers. That path can still reach the `execSync` keychain helpers, so the Gateway auth sync path named in the linked bug remains blocking after merge.
  - confidence: 0.9
- **[P2] Use async default deps in auth epoch:** `src/agents/cli-auth-epoch.ts:184-192`
  - body: With default deps, `hasAsyncDepOverride` is false, so this branch calls the sync cached reader and only awaits the async reader when tests override the async dependency. Invert the detection so production uses the async default and only sync-only test overrides fall back.
  - confidence: 0.92
- **[P2] Remove sync file fallbacks from async readers:** `src/agents/cli-credentials.ts:761-786`
  - body: `readCodexCliCredentialsAsync` falls back to `loadJsonFile` and `fs.statSync`, and the Claude async helper has the same sync file-read fallback. File-backed credentials still block the event loop when keychain is unavailable or disabled, undercutting the PR's async credential fix.
  - confidence: 0.84
- **[P2] Make Windows configured-shell execution async:** `src/agents/sessions/resolve-config-value.ts:147-156`
  - body: The async config resolver still calls `executeWithConfiguredShell` on Windows, and that helper uses `spawnSync`. A `!command` API key or header on Windows can therefore still block the event loop on the new async path.
  - confidence: 0.8
