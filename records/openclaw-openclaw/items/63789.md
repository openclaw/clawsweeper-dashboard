## What This Changes

Adds a docs-only concept proposal for a Phase 1 agent-to-agent handoff protocol with handoff IDs, structured acknowledgement states, control-token visibility, and an append-only ledger.

## Best Possible Solution

Keep the draft open as a focused RFC, or replace it with a narrow sessions_send implementation that adds the handoff id, structured ack states, control-token visibility, append-only ledger, docs, and regression tests without pulling in the meeting lifecycle.
