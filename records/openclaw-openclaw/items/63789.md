Keep open. This is a docs-only draft for a new agent-to-agent handoff contract; current main still lacks the proposed handoff id, structured acknowledgement states, and ledger, and live maintainer feedback says the lower-level primitive remains useful while staying separate from https://github.com/openclaw/openclaw/issues/65403.

## What This Changes

The PR adds a new docs concept page proposing a minimal `sessions_send`-style agent handoff protocol with handoff IDs, structured acknowledgements, control-token visibility, and an append-only ledger.
