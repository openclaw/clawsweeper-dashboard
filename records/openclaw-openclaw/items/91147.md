Full review comments:

- **[P1] Flush hook output before forcing exit:** `src/cli/hooks-cli.ts:563`
  - body: This new `process.exit(exitCode)` runs immediately after `runNativeHookRelayCli` returns, but that helper emits Codex hook decisions with plain `stream.write(...)` calls and never waits for drain/end. Please flush or await stdout/stderr delivery, or reserve forced exit for the hard deadline path, before terminating normal relay completion.
