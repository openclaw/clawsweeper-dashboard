This PR updates OpenAI PI Codex routing to resolve auth-profile ID prefixes through provider auth aliases and adds regression coverage for legacy Codex profile IDs.

## Best Possible Solution

## Solution Assessment

Yes. Reusing `resolveProviderIdForAuth` inside the narrow routing helper is the maintainable fix boundary and avoids auth storage migration, new discovery behavior, or a new auth method.
