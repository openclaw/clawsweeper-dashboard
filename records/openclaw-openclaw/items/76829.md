## What This Changes

The report asks OpenClaw to prevent malformed session transcript format errors from placing the shared provider auth profile into cooldown and blocking unrelated sessions.

## Best Possible Solution

Land one canonical narrow fix that filters session-local `format` failures before shared auth-profile health marking and update focused tests/docs if the contract changes.
