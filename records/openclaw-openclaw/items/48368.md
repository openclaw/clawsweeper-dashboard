  - body: The validator can only check missing pending items when `unresolvedItems` is present, and `compact.ts` passes `details.unresolvedItems`, but the safeguard details object only records the latest user goal. Live post-compaction diagnostics would skip the pending-item check the tests claim to cover.
  - confidence: 0.82
