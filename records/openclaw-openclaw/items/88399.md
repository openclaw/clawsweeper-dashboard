Full review comments:

- **[P1] Honor tool filters before projecting native MCP servers:** `src/config/mcp-config.ts:100-103`
  - body: `openclaw mcp tools` now persists `toolFilter` under the shared `mcp.servers` config, but only the embedded bundle-MCP runtime applies that selection.
