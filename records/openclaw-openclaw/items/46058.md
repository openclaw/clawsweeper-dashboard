Keep open. Current main has meaningful Android chat support, image attachments, assistant-launch routing into Chat, and Tailscale-aware connection guidance, but the official Android app still defaults and documents a Connect-first companion-node flow, so the remaining work is Android product direction and focused upstreaming scope rather than a completed core fix.
