## Best Possible Solution

Land a scoped heartbeat-runner fix that treats non-exception no-tool heartbeat final text as private in explicit `message_tool` mode while preserving explicit failure notices and relayable exec-completion delivery, then close this issue and retire duplicate repair PRs.
