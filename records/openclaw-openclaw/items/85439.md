## Best Possible Solution

Add a Feishu-owned opt-in final-card progress transcript only if maintainers accept the UX/privacy tradeoff, with focused coverage for live tool status, final card content, and the clean-final default.

## Solution Assessment

Unclear until maintainers decide the desired UX. A hard revert is not the best path; if accepted, a Feishu-scoped opt-in setting with regression tests is safer than changing the clean-final default for all users.
