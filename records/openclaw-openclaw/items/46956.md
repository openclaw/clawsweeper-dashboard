  - body: Current main builds core Gateway methods from `CORE_GATEWAY_METHOD_SPECS` and throws for handlers missing descriptors.   - confidence: 0.9

  - confidence: 0.86
- **[P1] Preserve hard-deny exec policies during trust:** `src/infra/exec-approvals.ts:624-629`
  - body: A trust window always carries `security: "full"` and the resolver substitutes it into the agent policy, so an agent configured with `security: "deny"` is silently promoted while trust is active. Reject trust for hard-deny agents, or make that override an explicit maintainer-approved contract with docs and tests.
  - confidence: 0.88
- **[P2] Audit commands that started under trust:** `src/agents/bash-tools.exec.ts:570-574`
