The issue requests mixed heartbeat precedence where agents without per-agent heartbeat config inherit `agents.defaults.heartbeat` while per-agent heartbeat config overrides or disables that default.

## Best Possible Solution

Adopt one heartbeat precedence contract and keep scheduler, summary, prompt, ACP spawn, doctor diagnostics, docs, and regression tests aligned before closing the canonical issue and duplicates.
