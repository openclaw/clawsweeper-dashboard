The PR changes shared model resolution to recover bundled-catalog image capabilities when configured model entries omit `input`, while attempting to preserve explicit input declarations.

## Best Possible Solution

## Solution Assessment

No. Shared model resolution is the correct ownership boundary, but explicitness should be retained by canonical resolution rather than rediscovered through a second parser with narrower identity rules.
