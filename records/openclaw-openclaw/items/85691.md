- **[P1] Route repairs through the session-store writer:** `src/commands/doctor-session-snapshots.ts:501`
  - body: This repair writes the parsed store back with `writeTextAtomic` after manually writing prompt blobs, bypassing the `saveSessionStore`/writer path that serializes prompt refs and refreshes session-store caches.
