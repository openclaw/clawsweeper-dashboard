Full review comments:

- **[P2] Handle paths that are both JSON- and XML-escaped:** `src/commands/doctor-session-snapshots.ts:356-358`
  - body: For a Windows prompt location containing an XML-escaped character, the raw `sessions.json` text contains doubled JSON backslashes plus entities like `&amp;`. This code tries the JSON-escaped form with a raw `&`, then the XML-escaped form with single backslashes, so neither form matches and `doctor --fix` leaves that stale path behind. Generate the stored form by XML-escaping and then JSON-escaping, and add a regression test for that combination.
