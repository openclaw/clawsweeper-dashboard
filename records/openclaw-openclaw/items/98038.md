## What This Changes

## Best Possible Solution

Review and prove the open LAN QR fix for the bootstrap half, then verify the manual-token reconnect path and update the shared Swift/iOS auth flow so approved device tokens are persisted and reused without weakening public plaintext auth boundaries.
