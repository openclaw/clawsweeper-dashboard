## Summary

Keep open. The bug still exists on current main, and this PR is still useful, but the branch is stale after completion install logic moved to `src/cli/completion-runtime.ts`, so the current diff would not fix the active installer path and still needs real Windows behavior proof.

## What This Changes

The branch adds PowerShell completion profile installation and reload syntax, switches Windows time-format detection to the PowerShell resolver, and adds PowerShell completion tests.
