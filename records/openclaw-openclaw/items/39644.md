## Summary

Keep open. The PR addresses a real supported Windows CLI bug that is still present on current main and in v2026.5.7, but the branch is stale against the current completion runtime split and still lacks real native Windows behavior proof.

## Best Possible Solution

Rebase the useful fix onto src/cli/completion-runtime.ts and src/agents/date-time.ts, keep focused regression coverage in current test files, and attach redacted native Windows proof for the profile write and pwsh selection.
