Keep open. ## What This Changes

The issue requests a standardized `skill.yaml` permission manifest for skills, with install-time permission review, audit/trust scoring, signing, attestation, and ClawHub trust badges.

## Best Possible Solution

Keep this open for a maintainer-designed, versioned skill permission or capability manifest integrated with SKILL.md skills, ClawHub install/update review, audit/trust reporting, signing or attestation expectations, and documented runtime-enforcement boundaries.
