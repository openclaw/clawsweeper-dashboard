Keep open. Current main still has SKILL.md/frontmatter skill metadata, ClawHub scan visibility, and generic install/tool-call hooks, but it does not implement the requested standardized permission manifest, install-time consent, audit/trust reporting, signing, or attestation contract.

## Best Possible Solution
