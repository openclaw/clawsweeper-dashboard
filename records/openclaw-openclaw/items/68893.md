## What This Changes

## Best Possible Solution

Land a canonical `darwin`/`linux` Homebrew prompt allowlist with unsupported-platform regression coverage through https://github.com/openclaw/openclaw/pull/69002 or an equivalent fix, then close this issue after merge.

## Solution Assessment

Yes. A `darwin`/`linux` allowlist before the Homebrew prompt is the narrow maintainable fix, and this issue should stay open until the linked PR or an equivalent fix merges.
