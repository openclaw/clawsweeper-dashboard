## Summary

Keep open. Current main only supports `exec.security` values `deny`, `allowlist`, and `full`; the requested denylist mode is not implemented. The report is recent, concrete, and in core scope, and current mitigations like `ask=on-miss`, `allow-always`, and `safeBins` do not provide the proposed `denyPatterns`/`denyAction` behavior.
