  - confidence: 0.98

  - body: The helper can return a peer URL with `gatewayToken: undefined`, but `callGateway` rejects explicit URL overrides without explicit credentials. That makes tokenless `gateway.peers` entries, including the PR's own `studio` example/test shape, fail before the remote request is attempted.
  - confidence: 0.9
