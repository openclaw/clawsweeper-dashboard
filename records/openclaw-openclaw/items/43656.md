  - body: The remote `sessions_spawn` branch calls `agent` without `idempotencyKey`, but the gateway protocol requires that field for `AgentParamsSchema`. As written, cross-gateway spawn requests are rejected before the remote run starts.
  - confidence: 0.98

  - body: Peer config marks `token` optional and the helper returns configured peers with `gatewayToken: undefined`, but `callGateway` rejects explicit URL overrides without explicit credentials. Tokenless peer entries therefore fail before any remote request unless this adds a tested auth-none or paired-auth path.
  - confidence: 0.9
- **[P2] Reject ambiguous remote sessions_send targets:** `src/agents/tools/sessions-send-tool.ts:70-96`
  - body: The remote branch accepts both `sessionKey` and `label` and silently prefers `sessionKey`, while the local tool rejects the same ambiguous input. Preserve the local invariant before routing remotely so cross-gateway sends cannot target the wrong session.
  - confidence: 0.86
