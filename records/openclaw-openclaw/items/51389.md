  - confidence: 0.91

  - confidence: 0.86
- **[P2] Stop image fallback on disallowed statuses:** `src/agents/model-fallback.ts:838`
  - body: This wires the allowlist into image attempts, but the outer image loop still records any returned error and advances to the next candidate. A custom allowlist such as `[429]` is therefore not enforced for image/PDF fallback.
  - confidence: 0.84

  - body: Once this is a persisted config key, `openclaw models scan` still rebuilds `agents.defaults.model` from only `primary` and `fallbacks`. Running that command would silently delete `fallbackOnErrors` and return users to the legacy retry policy.
  - confidence: 0.82
