## Summary

Keep open: this PR addresses a real Matrix reasoning-delivery bug, but the branch is still not merge-ready because it only changes the Matrix reply helper while the normal Matrix path still goes through generic dispatch that suppresses explicit reasoning payloads. ## What This Changes

The PR changes Matrix reply delivery and send options so reasoning-like replies are sent as `m.notice`, with Matrix reply tests updated to expect notice delivery instead of suppression.

## Best Possible Solution
