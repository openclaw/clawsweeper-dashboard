## What This Changes

The issue reports that preseeded profile files can make workspace bootstrap reconciliation mark onboarding complete and delete a user-provided BOOTSTRAP.md before first run.

Land a narrow agents workspace fix, likely via https://github.com/openclaw/openclaw/pull/91988 or a replacement, that keeps BOOTSTRAP.md pending while setupCompletedAt is unset and treats profile/skill preseeds separately from real onboarding completion evidence.
