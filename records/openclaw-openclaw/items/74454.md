## Summary

Keep open. Current main already blocks the workspace `.env` half through the Windows trust-root blocklist, but the registry helper still has env-derived candidates; this PR addresses that remaining surface while introducing a compatibility regression risk by hardcoding `C:\Windows` despite existing non-default-root support.

## What This Changes

This PR removes env-derived Windows `reg.exe` candidates, updates focused infra tests for fixed registry-client resolution and blocked workspace `.env` keys, and adds a changelog entry.

## Best Possible Solution

Land a security-reviewed version that prevents untrusted workspace env from controlling registry-client execution while preserving validated registry/install-root support for non-default Windows roots.

## Reproduction Assessment

Yes, at source level for the resolver under review: current main still builds `reg.exe` candidates from `SystemRoot` and `WINDIR` and executes the selected path. The full workspace `.env` injection route is already partly mitigated on current main by the `SYSTEMROOT`/`WINDIR` blocklist.

## Solution Assessment

No. Pinning to `C:\Windows` is a narrow hardening, but it drops the existing validated non-default Windows-root behavior; the safer maintainable direction is to separate untrusted workspace dotenv input from trusted host or registry-root discovery.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.82

Full review comments:

- **[P2] Preserve non-default Windows registry roots:** `src/infra/windows-install-roots.ts:96`
  - body: This hardcodes registry probing to `C:\Windows\System32\reg.exe`, dropping the validated `SystemRoot`/`WINDIR` candidates that current code and tests use for non-default Windows roots such as `D:\Windows`. On those hosts `locateWindowsRegExe()` will return null, registry roots will not be read, and callers lose the intended registry-backed install-root behavior; please keep the hardening scoped to untrusted workspace dotenv input or otherwise preserve non-default roots.
