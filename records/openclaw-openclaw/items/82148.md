## Summary

Keep open: this active feature PR is useful, but the ACP path still appears to record the new override only on the local session entry instead of applying it through ACP runtime controls before the first child turn. The supplied proof is focused tests/typechecks and source assertions, not inspectable after-fix real `sessions_spawn.fastMode` behavior.

## What This Changes

The PR adds a `sessions_spawn.fastMode` boolean to tool schema, docs, tests, prompt snapshots, and forwards it into native subagent and ACP spawn setup.

## Best Possible Solution

Keep the native subagent support, but route ACP fast-mode overrides through ACP runtime options or backendExtras so `session/set_config_option fast_mode` is applied before the first ACP child turn, with regression coverage for that control path.

## Reproduction Assessment

Yes for the review finding: source inspection shows the PR-added ACP patch stores `fastMode` on the child session entry while the ACP runtime-control path consumes `AcpSessionRuntimeOptions`. I did not execute the PR branch in this read-only review.

## Solution Assessment

No: persisting `SessionEntry.fastMode` is the right shape for native subagents, but ACP spawns need the override applied through the ACP runtime config-control path or the ACP-facing parameter/docs should be withheld.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Apply fastMode to ACP runtime options:** `src/agents/acp-spawn.ts:1324`
  - body: The ACP path accepts and documents `sessions_spawn.fastMode`, but this added line only stores it in OpenClaw's local session entry. ACP child turns are initialized through `initializeAcpSpawnRuntime` and dispatched through ACP runtime controls, which consume `AcpSessionRuntimeOptions`/config-option pairs rather than `SessionEntry.fastMode`, so the first ACP child turn can still use the backend default. Please pass the override through the ACP runtime config-option path before dispatch, or do not expose it for `runtime="acp"`.
  - confidence: 0.89
