## What This Changes

Make the docs formatting check used by `pnpm check:docs` run correctly from native Windows PowerShell without Unix-only shell utilities or overlong command lines.

## Best Possible Solution

Land https://github.com/openclaw/openclaw/pull/48887 or an equivalent maintained patch that batches Windows-sensitive `oxfmt` invocations, preserves tracked-doc semantics, adds focused regression coverage, and verifies `pnpm check:docs` in native PowerShell and a Unix shell.
