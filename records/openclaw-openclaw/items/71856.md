## Summary

Keep this PR open. Current main does not implement the dynamic TUI startup summary, and the latest provided PR patch still has merge-blocking issues: it reads recent session previews without current-agent scoping, appends on reconnects, and lacks the required changelog entry. The earlier undefined helper issue appears addressed by the second PR commit, but the remaining blockers need changes before merge.

## What This Changes

The PR adds a TUI startup-summary formatter and a non-local Gateway `sessions.list` call after history load to render a previous session title or last-message preview in the TUI startup log.

## Best Possible Solution

Revise this PR before merge so the startup summary is inserted only on the initial non-local connection, queries sessions for the current agent, preserves nonblocking best-effort failure behavior, adds focused TUI/session tests, and records the user-facing change in `CHANGELOG.md`.

## Reproduction Assessment

Yes for the PR regressions: trigger `client.onConnected` again after a disconnect to observe duplicate startup-summary lines, and mock `sessions.list` with a newer session from another agent to confirm the unscoped lookup can select it.

## Solution Assessment

No in its current form. Using the Gateway sessions preview data is a reasonable direction, but the maintainable fix needs current-agent scoping, one-time initial-connect behavior, tests, and changelog coverage before this PR is safe to merge.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Guard startup summaries on reconnect:** `src/tui/tui.ts:1105`
  - body: The new block runs inside `client.onConnected`, which current main also uses after disconnects via the `reconnected` flag. Because the condition is only `!isLocalMode`, every Gateway reconnect can fetch sessions again and append another copy of the startup-summary lines to the chat log; gate this to the initial connection or track a one-time insertion flag.

  - confidence: 0.9
