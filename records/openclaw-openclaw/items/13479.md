## Summary

Keep open. Current main still shows the cron prompt/task payload and supports editing stored descriptions, but the main Control UI cron job card does not render the saved description as the requested human-readable summary.

## What This Changes

The issue requests showing each cron job’s prompt/task and human-readable summary in the Web UI cron dashboard/detail flow.

## Best Possible Solution

Render saved CronJob.description as a visible Summary or Description field in the main Control UI cron job card or selected-job detail area for both systemEvent and agentTurn jobs, without changing the storage/RPC contract.

## Reproduction Assessment

Yes. A source-level reproduction is high confidence: render a CronJob with a non-empty description; current renderJob and renderJobPayload never read job.description, so the prompt appears but the summary does not.

## Solution Assessment

Yes. The narrowest maintainable solution is a Control UI presentation repair with focused view coverage because shared types, persistence, search copy, and edit-form plumbing already carry the field.

## Repair Work Prompt

Fix the Control UI cron dashboard so saved cron job descriptions are visible as the requested human-readable summary. In `ui/src/ui/views/cron.ts`, render `CronJob.description` in the main job card or selected-job detail area for both `systemEvent` and `agentTurn` jobs while preserving the existing prompt/system payload display and row-click behavior that loads job-scoped run history. Treat description as user-authored text: use Lit escaped text unless intentionally routing through the existing sanitized markdown pipeline. Add focused coverage in `ui/src/ui/views/cron.test.ts` for a job with `description` and a prompt/system payload. Update `ui/src/i18n/locales/en.ts` and run locale sync if a new label key is added; include a user-facing `CHANGELOG.md` entry for this issue. Do not change cron storage, RPC schemas, scheduler behavior, or unrelated controller logic unless the tests prove a small UI plumbing adjustment is needed.
