## Summary

Keep open. Current main and v2026.6.8 already have partial browser snapshot mitigations, but the reported default goal-shaped/context-budget browser interaction contract remains unresolved and needs maintainer product review rather than cleanup closure.

## What This Changes

The issue asks for browser interactions to avoid injecting excessive page content into model context during multi-step browser automation.

## Best Possible Solution

Define the browser read contract first: compact goal-shaped default reads, small action diffs after interactions, and an explicit raw/full snapshot escape hatch with focused tests across Playwright, CDP, and Chrome MCP.
