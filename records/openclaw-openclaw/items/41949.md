## What This Changes

The issue asks OpenClaw’s browser tool to avoid filling model context with large page snapshots during multi-step browser tasks.

## Best Possible Solution

Define browser-agent snapshot defaults, then make compact or task-scoped reads the normal path with explicit raw read escape hatches, compact action-diff responses, regression tests, docs, and live proof.

No high-confidence live reproduction was established; this review did not run a multi-step browser task on a noisy site. Source inspection does show current-main paths where browser snapshots remain bounded but not task-scoped by default.
