## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Accept nsSandbox in the strict config schema:** `src/config/types.tools.ts:243-258`
  - body: This adds `tools.exec.nsSandbox` to the public exec config type and documents it in the PR body, but the strict runtime config schema still omits the key. Users setting the documented config would hit an unrecognized-key validation error instead of enabling bwrap.
  - confidence: 0.93
- **[P1] Mount the exact planned executable paths:** `src/infra/exec-bwrap-sandbox.ts:232-235`
  - body: `buildBwrapArgs()` re-resolves configured safe-bin names from trusted dirs, but enforced execution runs the resolved paths from command analysis. On common `/bin` to `/usr/bin` layouts this can mount one path while the command executes another, making sandboxed safeBins fail with command-not-found.
  - confidence: 0.9
- **[P1] Mount the actual runtime shell path:** `src/agents/bash-tools.exec-host-gateway.ts:207-219`
  - body: `makeBwrapParams()` passes only the selected shell basename, while runtime execution uses the full shell path after `--`. Users with shells outside the trusted dirs, such as `/usr/local/bin/bash` or Nix shells, can have every sandboxed command fail before it starts.
  - confidence: 0.91
- **[P2] Preserve network for unclassified safe-bin profiles:** `src/infra/exec-bwrap-sandbox.ts:304-313`
  - body: The gateway always supplies `commandBins`, and `shouldShareNetwork()` disables networking whenever none of those names are hardcoded. Any custom profiled safeBin that legitimately needs network but is not listed will silently lose network under bwrap.
  - confidence: 0.78
- **[P2] Split out install lifecycle policy changes:** `package.json:435-439`
  - body: The package diff changes `onlyBuiltDependencies`, which controls install-time build-script execution and is unrelated to safeBins bwrap isolation. Drop this from the branch or split it into a separately reviewed package-policy change.
  - confidence: 0.86
