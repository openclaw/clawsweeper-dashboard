  - confidence: 0.88

  - confidence: 0.86

  - body: Only `sh` and `bash` are guaranteed in the namespace, while custom safe-bin scripts can use shebangs such as `/usr/bin/env node` or another interpreter. Mount declared interpreters safely or reject script safe bins that cannot run inside the namespace.
  - confidence: 0.8
