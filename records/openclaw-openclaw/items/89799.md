## Summary

Keep this PR open, but it is not merge-ready: the proposed guard is in the TypeScript dist entry helper, while npm-global users run the published `openclaw.mjs` launcher first and that launcher still enables compile cache for packaged installs.

## Best Possible Solution

## Solution Assessment
