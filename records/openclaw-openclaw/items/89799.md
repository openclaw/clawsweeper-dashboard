## Summary

Keep open: current main and the latest release still enable packaged compile cache on the reported Windows Node 24 startup path, while this PR supplies a focused mitigation with no concrete code-review defects found. ## What This Changes

The PR changes OpenClaw CLI startup so Windows packaged installs on Node 24.0-24.14 disable Node compile cache and respawn without inherited compile-cache state, with focused tests for affected and unaffected runtimes.
