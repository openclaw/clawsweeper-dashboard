## Summary

Keep open: the config-gated plain-language approval direction is useful, but PR-head `simple` mode can still hide security-relevant command context while suppressing the raw technical approval text.

## What This Changes

Adds `approvals.plugin.language` modes, a large plain-language plugin approval renderer, chat-channel wiring for Telegram and Matrix, docs/schema updates, and focused renderer tests.

## Best Possible Solution

Keep the `original` default and land the plain-language modes only after `simple` mode conservatively surfaces or fails closed on env prefixes and text-processor file operands with focused regression coverage.

## Reproduction Assessment

Yes. A high-confidence source reproduction is `LD_PRELOAD=./hook.so ls` or `jq . .env` with `language: "simple"`: PR-head source strips or ignores that context before returning without raw technical details.

## Solution Assessment

No, not as-is. The config-gated approach is maintainable, but the command summarizer must preserve a redacted preview or fail closed before it hides raw command details.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Preserve env prefixes before hiding commands:** `src/infra/plugin-approvals.ts:1796-1798`
  - body: `resolveCommandWords()` drops leading `KEY=value` tokens before classification, and `simple` mode then omits the raw technical block. Prefixes such as `PATH=./bin ls`, `LD_PRELOAD=./hook.so ls`, or credential-bearing env vars can materially change a known command while rendering as an ordinary low-risk action, so surface a redacted preview or fail closed for prefixed commands.
  - confidence: 0.94
- **[P1] Surface text-processor input files:** `src/infra/plugin-approvals.ts:2057-2062`
  - body: `getCommandTargetArgs()` returns no targets for `jq`, `awk`, `cut`, `sort`, `tr`, and `uniq`, and those cases summarize as low-risk formatting while `simple` hides the raw command. Commands like `jq . .env` or `sort .env` can read credential files without the file appearing in the prompt, so include operands and raise or fail closed for sensitive/system paths.
  - confidence: 0.9
