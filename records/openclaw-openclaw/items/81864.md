## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P1] Handle force-clobber redirections before summarizing:** `src/infra/plugin-approvals.ts:41-42`
  - body: `>|` is valid shell output redirection, but the new output-redirection matcher and leading-redirection parser only cover `>`, `>>`, and `&>` forms. In `simple` mode, `printf '%s\n' token >| .env` therefore does not enter `summarizeRedirectedWrite`; it can render as a low-risk formatted message with no command preview while writing a sensitive file. Please treat `>|` and descriptor-prefixed forms as redirected writes and add regressions for sensitive and leading uses.
  - confidence: 0.86
