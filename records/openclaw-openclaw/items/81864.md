Keep open: this is a substantial, active feature PR and the latest head still has one approval-safety bug in simple mode. The PR adds an `approvals.plugin.language` option for original, simple, and simple-technical plugin approval prompts, wires it through shared/Telegram/Matrix approval rendering, updates config docs/tests/baselines, aligns a grammY types dev dependency, and includes oc-path formatting churn.
