## Summary

Keep open. This is a useful opt-in feature PR with supplied real behavior proof, but the latest diff still has a blocking approval-safety bug: simple mode can hide sensitive input-redirection reads for text-filter commands after removing the raw technical command.

## What This Changes

The PR adds opt-in `approvals.plugin.language` modes for plain-language plugin approval prompts, wires them through shared fallback, Telegram, and Matrix approval rendering, and updates config/docs/tests plus the Telegram type dependency alignment.

## Best Possible Solution

Keep `original` as the default, but make the simple renderer fail closed or show a redacted preview for input redirections across non-network commands before hiding technical text.

## Reproduction Assessment

Not applicable for the original feature request; this is an opt-in UX/config feature. The blocker is source-reproducible on PR head by tracing `tr a b < .env` or `jq . < .env` through the simple renderer, where text filters ignore input redirection and render as low-risk formatting without a preview.

## Solution Assessment

No, not yet. A shared opt-in renderer is the right boundary, but the current implementation must surface or fail closed for shell input redirection before it is safe to hide the raw command in `simple` mode.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Surface input redirections before hiding commands:** `src/infra/plugin-approvals.ts:1229-1232`
  - body: In `simple` mode the raw command is omitted unless a preview is requested, but these text-filter branches treat `jq`/`awk`/`tr`/`sort`/`uniq` with no output redirection as low-risk formatting. Because input redirections are only used by the network upload path, commands like `tr a b < .env` or `jq . < .env` would show a low-risk “process/filter output” prompt without `.env` or any command preview, even though they read a sensitive file. Please surface input redirection targets, and fail closed or show a redacted preview when the source is sensitive or ambiguous.
  - confidence: 0.88
