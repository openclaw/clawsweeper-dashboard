## Summary

## What This Changes

Add an official linux/riscv64 image to OpenClaw’s multi-platform Docker release pipeline without blocking existing architecture releases.

## Best Possible Solution

Adopt riscv64 only through a release-owner-approved image strategy that proves all base and native dependencies, publishes the architecture atomically when available, preserves existing releases on failure, verifies attestations, documents support, and smoke-tests a pulled image.
