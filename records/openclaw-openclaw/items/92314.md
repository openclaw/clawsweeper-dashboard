## What This Changes

Request to expose Workboard card delete/remove support through the tool/API surface, with optional management UI support and a hard-delete or soft-delete policy.

## Best Possible Solution

Define the Workboard card deletion policy, then expose only the approved agent-tool or CLI surfaces by reusing `WorkboardStore.delete` with focused docs, tests, and real persisted-state proof.

## Solution Assessment
