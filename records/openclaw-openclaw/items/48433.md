## Summary

Keep https://github.com/openclaw/openclaw/pull/48433 open. Current main does not implement the requested per-host exec allowlist model, but this PR is not landable as-is because source review found a sandbox approval bypass and a node-host compatibility bug.

## What This Changes

This PR adds optional `tools.exec.allowedHosts`, per-host `exec-approvals.json` allowlist maps, doctor migration, sandbox approval plumbing, docs, and tests.

## Best Possible Solution

Define a host-aware exec approval contract that composes with current `host=auto`, preserves approval checks for wildcard sandbox buckets, and keeps node-host consumers reading the correct host bucket.

## Reproduction Assessment

Yes. Static source reproduction is high-confidence: PR head returns before approval checks for a sandbox `"*"` bucket, and node-host paths still read a flat allowlist that excludes host-specific `node` buckets.

## Solution Assessment

No. The feature direction may be useful, but this branch is not the best fix until it is rebased onto the current `host=auto` model and preserves approval policy plus node-host compatibility.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Preserve approval checks for sandbox wildcard allowlists:** `src/agents/bash-tools.exec-host-gateway.ts:95-99`
  - body: This early return exits before `requiresExecApproval`, heredoc handling, and obfuscation checks run. With a sandbox `"*"` bucket and `ask: "always"` or an obfuscated command, the command runs immediately instead of producing the required approval prompt.
  - confidence: 0.92
- **[P2] Include host-specific node buckets in the flat fallback:** `src/infra/exec-approvals.ts:589-599`
  - body: Node-host consumers still evaluate `resolved.allowlist`, but map-format agents only copy the `default` bucket into that flat list. A config such as `{ allowlist: { node: [...] } }` therefore appears empty to the node path and causes allowlist misses for commands that should be permitted.
  - confidence: 0.84
