Request deterministic `.skill` archive entry ordering in skill-creator by sorting packaged files before writing the zip.

## Best Possible Solution

Sort accepted entries by final POSIX archive path before writing, preserve the existing exclusion and symlink/root-escape guards, and add regression coverage for exact `ZipFile.namelist()` order.

## Solution Assessment

Yes. Sorting the already accepted entries by final POSIX archive path is the narrowest maintainable repair and avoids changing validation, exclusions, symlink handling, or packaging policy.
