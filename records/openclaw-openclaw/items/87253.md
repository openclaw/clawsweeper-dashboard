## Summary

## What This Changes

Request automatic skill-declared per-subagent workdir isolation, generated `WORKDIR` injection, and output/merge validation for parallel subagent skills.

## Best Possible Solution

Design a narrow opt-in skill/subagent output-isolation contract that creates unique per-spawn directories and exposes them to the parent without making core infer skill-specific merge success.
