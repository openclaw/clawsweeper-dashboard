  - confidence: 0.93

  - body: `capabilityConfig` is a `Record<string, unknown>`, so `capabilityConfig.dimensions` is `unknown`; passing it directly into `ComfyDimensionConfig` parameters breaks strict TypeScript and skips validation of the user-provided object. Parse or narrow the nested object before using it for base size and node injection.
  - confidence: 0.9
