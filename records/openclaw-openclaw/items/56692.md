## What This Changes

The issue requests clearer group-chat context targeting so agents can distinguish messages addressed to the current bot, another bot, or ambient group history.

## Best Possible Solution

Keep this issue as the canonical tracker until the linked implementation PR is rebased, reviewed for the sibling-bot identity source, proven in a real Telegram two-bot group, and landed or replaced by a narrower maintainer-approved fix.
