The PR adds a setup-only Podman bind mount from the host Claude CLI credentials directory into `/home/node/.claude`, gated on `.credentials.json` and overridable with `OPENCLAW_HOST_CLAUDE_DIR`.

## Best Possible Solution

## Solution Assessment
