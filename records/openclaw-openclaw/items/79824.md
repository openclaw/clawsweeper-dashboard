## Summary

Keep open: current main still emits Feishu interactive cards with deprecated `tag: "action"` button rows on the reported approval, quick-action, and shared presentation paths, and the active fix path is the open linked PR at https://github.com/openclaw/openclaw/pull/79832.

## What This Changes

The issue asks for Feishu button-bearing cards to stop using the deprecated V2 `action` container so exec/plugin approval and quick-action cards can be delivered.

## Best Possible Solution

Land the linked Feishu plugin fix or an equivalent owner-local helper that renders button rows as V2-compatible `column_set` across sanitizer, shared presentation buttons, approval cards, and launcher cards, with updated tests and Feishu/Lark validation.
