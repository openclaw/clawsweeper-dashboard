## Summary

Keep open: current `main` still emits deprecated Feishu `tag: "action"` button containers on the reported approval, launcher, native-card, and presentation-button paths, and the open linked fix PR should be reviewed before this issue is closed.

## What This Changes

The issue reports that Feishu schema V2 rejects bundled approval and button cards that still use deprecated `action` containers instead of V2-compatible button row elements.

## Best Possible Solution

Review and land https://github.com/openclaw/openclaw/pull/79832, or an equivalent Feishu-plugin fix that centralizes V2-compatible button-row rendering and updates the focused Feishu tests.

## Reproduction Assessment

Yes. ## Solution Assessment

Yes. The narrow maintainable solution is to fix the Feishu-owned card renderer/sanitizer to emit V2-compatible button rows and update the Feishu tests; the open linked PR is the right vehicle to review that implementation.
