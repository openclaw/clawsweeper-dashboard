## Summary

Keep open: current main still emits deprecated Feishu `tag: "action"` button containers on the reported card paths, and an open same-author PR with closing syntax targets the fix.

## What This Changes

Feishu approval, quick-action, and presentation button cards should use a Feishu V2-compatible button-row shape instead of deprecated `action` containers.

## Best Possible Solution

Land or close the linked Feishu PR after review, with a shared Feishu-owned `column_set` button-row helper, updated card tests, and focused live Feishu/Lark validation.

## Reproduction Assessment

Yes. Source inspection on current main shows any Feishu approval, quick-action launcher, or presentation button card serializes `tag: "action"` into an interactive Feishu card; I did not run a live Feishu send in this read-only review.

## Solution Assessment

Yes. The narrowest maintainable direction is an owner-local Feishu card helper that emits V2-compatible button rows and is reused by the sanitizer, presentation renderer, approval card, and launcher card; the open linked PR follows that shape and should be resolved before this issue closes.
