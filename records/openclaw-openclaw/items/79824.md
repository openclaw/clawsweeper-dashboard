Keep open: current main still emits Feishu schema-2.0 interactive cards with deprecated `tag: "action"` button containers, and the open fixing PR at https://github.com/openclaw/openclaw/pull/79832 has not merged. ## What This Changes

Replace Feishu button-bearing cards from deprecated `action` containers with V2-compatible `column_set` rows so approval, quick-action, and presentation button cards deliver.

## Best Possible Solution
