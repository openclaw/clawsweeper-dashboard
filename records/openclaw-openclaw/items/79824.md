## What This Changes

## Best Possible Solution

Land https://github.com/openclaw/openclaw/pull/79832 after refresh, or land an equivalent narrow Feishu plugin fix that emits only V2-supported button rows for approval, quick-action, and native-card paths with focused tests.

## Solution Assessment

Yes: replacing the remaining emitted `action` containers with V2-supported `column_set` rows inside the Feishu plugin is the narrow maintainable fix. The linked PR is the right implementation path to refresh or replace rather than closing the issue first.
