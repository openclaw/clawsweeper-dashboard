## Summary

Keep open: current main and the latest shipped tag still emit deprecated Feishu `tag: "action"` button containers on the reported card paths, and the linked closing PR is still open, unmerged, and conflicting.

## What This Changes

The issue requests replacing Feishu schema-2.0 button rows from deprecated `action` containers with `column_set` so approval and quick-action cards deliver successfully.

## Best Possible Solution

Land the paired Feishu-owned `column_set` migration, with updated card-shape tests and Feishu/Lark runtime validation when credentials are available.

## Reproduction Assessment

Yes. Source inspection on current main shows Feishu approval, quick-action, and presentation-button cards serialize `tag: "action"` into `msgType: "interactive"`; the paired PR also supplies redacted live Feishu 230099/200861 before/after logs.

## Solution Assessment

Yes, the narrow maintainable fix is to change the Feishu plugin's card builders and sanitizer to emit V2-compatible `column_set` rows. The issue should stay open until the linked PR or an equivalent fix merges.
