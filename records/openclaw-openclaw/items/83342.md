## Best Possible Solution

## Reproduction Assessment

Yes, source-level. Current `buildModelsProviderData()` adds canonical visible catalog rows and separately adds auth-gated CLI-runtime catalog rows, and existing tests assert both provider lanes can appear.

## Solution Assessment

Unclear. A unilateral hide-one-side fix would conflict with current tests, docs, and the same-author open dual-route design issue; maintainers should choose the route identity and picker policy first.
