## What This Changes

## Best Possible Solution

Define the shared `/models` display contract for canonical provider refs versus CLI runtime aliases, then implement one refreshed current-main dedupe rule in `buildModelsProviderData` while preserving explicit legacy CLI refs and explicit CLI-provider `all` browse.

## Solution Assessment
