## What This Changes

Request a Control UI Agents-tab avatar picker that uploads an image, stores it under the agent workspace, updates IDENTITY.md, and serves it through the existing avatar route.

## Best Possible Solution

Define a maintainer-approved admin-scoped upload contract that safely stores bounded avatar images, updates IDENTITY.md consistently, and reuses the existing authenticated avatar rendering policy.
