## Summary

Keep open. Current main still generates macOS service PATH entries from user/version-manager roots and still merges preserved PATH entries into gateway install plans, while open PR #75246 is an unmerged closing-syntax fix candidate that should be reviewed before this issue is resolved.

## What This Changes

The issue requests macOS Gateway LaunchAgent installs and doctor repairs to write a canonical minimal PATH instead of preserving interactive-shell or version-manager PATH entries.

## Best Possible Solution
