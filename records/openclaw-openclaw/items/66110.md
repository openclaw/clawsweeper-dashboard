Full review comments:

- **[P1] Apply the override before routed command dispatch:** `src/cli/program/preaction.ts:96-99`
  - body: `OPENCLAW_LOG_FILE` is set only by Commander's preaction hook, but `tryRouteCli()` returns before Commander parsing for routed commands. Invocations such as `openclaw --log-file /tmp/x status` therefore consume the option for routing but never install the override, so the advertised root flag is silently ignored.
  - confidence: 0.99
- **[P1] Resolve status from the managed service environment:** `src/logging/log-file-path.ts:38`
  - body: The new resolver reads the invoking process environment, while `gatherDaemonStatus()` already has the managed service's merged environment. A CLI-only override can be reported as the daemon path and a service-only override can be missed; accept an explicit environment and test differing CLI/service values.
  - confidence: 0.98
