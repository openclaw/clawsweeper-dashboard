No as-is: the fix addresses the central documented `imageModel` bug, but it is too broad for MiniMax because it drops the existing chat-model-to-VL fallback protection. The best fix should preserve that invariant while allowing trusted native-vision MiniMax M3 to skip redundant descriptions.
