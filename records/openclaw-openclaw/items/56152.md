## Best Possible Solution

Revise `assertLocalMediaAllowed()` so existing files remain contained by canonical target, tolerate only safe macOS root aliases, preserve the workspace sibling guard, add a symlink-escape regression test, and attach redacted Telegram local MP4 proof.

## Solution Assessment
