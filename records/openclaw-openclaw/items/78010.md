  - body: This example extracts the backup into `/tmp/openclaw-restore` and leaves it there. Backup archives can contain credentials, auth profiles, sessions, and workspace data, so public restore docs need a private per-run directory plus cleanup rather than a predictable shared path.

Readers should read `archiveRoot` or `assets[].archivePath` from `manifest.json`.
  - confidence: 0.89
