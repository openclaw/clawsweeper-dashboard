## Best Possible Solution

Land one reviewed JSON5-aware Raw config editor that preserves reveal-gated redaction, raw text round-trip/comment semantics, pending-change diffs, save/apply behavior, and has real behavior proof.

## Reproduction Assessment

Yes for the PR-introduced regressions: source inspection and CI logs show the missing reveal control, and the JSON5 parse/stringify path proves comments are dropped. This is not a current-main bug reproduction; it is a proposed feature patch review.

## Solution Assessment

No. CodeMirror is a plausible direction, but this patch is not the best mergeable form until it restores the raw reveal controls, avoids destructive comment loss, rebases cleanly, and includes real after-fix behavior proof.
