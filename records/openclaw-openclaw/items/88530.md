- **[P1] Keep explicit tokens ahead of cached device tokens:** `apps/shared/OpenClawKit/Sources/OpenClawKit/GatewayChannel.swift:559`
  - body: This makes a saved node device token win over the explicit token passed into connect. Current main and the sibling TypeScript gateway client choose the explicit token first, so existing users who rotate or configure a shared token can still authenticate with a stale stored token or fail in a new way.   - confidence: 0.88
- **[P2] Preserve no-target approval discovery:** `extensions/imessage/src/approval-reaction-poller.ts:230`

Gate idle scans on a narrower signal instead of deleting this discovery mode.
