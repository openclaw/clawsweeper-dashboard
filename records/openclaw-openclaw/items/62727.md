## Best Possible Solution

Preserve whole avatar URLs, data URIs, and workspace paths before applying a narrowly tested descriptive-tail heuristic, with regression coverage for descriptive tails and dash-containing filenames plus real behavior proof.

## Reproduction Assessment

## Solution Assessment

No. The useful direction is to preserve already-valid avatar values before applying any descriptive-tail heuristic, then prove both the new descriptive format and existing dash-containing filenames.
