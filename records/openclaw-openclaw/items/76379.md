## Summary

Keep open: the PR targets a valid current-main CLI bug, but the current branch is not mergeable as-is because the manifest-only ownership gate regresses documented runtime-owned plugin CLI commands.

## What This Changes

The PR adds an unknown-root preflight in `runCli`, changes plugin CLI primary-command planning to return an empty registry when no manifest owner matches, updates CLI/plugin tests, and adds a changelog entry.

## Best Possible Solution

Revise the fast-fail path so truly unknown roots fail before proxy/runtime loading while documented `api.registerCli(..., { commands })` plugin roots continue to work, or explicitly make and document a compatibility-breaking policy change.

## Reproduction Assessment

Yes for the CLI-scoped bug from the linked issue logs and current-main source path: `openclaw foo` can enter proxy startup and broad plugin CLI loading before command rejection. I did not execute the live command because this review was read-only and local package tooling was unavailable.

## Solution Assessment

No as currently written. The branch fixes `openclaw foo`, but it does so by treating manifest ownership as the only plugin CLI ownership signal, which conflicts with the documented runtime-owned `commands` compatibility path.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Preserve runtime-owned plugin CLI roots:** `src/cli/run-main.ts:408-410`
  - body: Current docs still define `commands` as explicit roots owned by a registrar and say using `commands` without descriptors remains an eager compatibility path. This preflight now treats any non-built-in primary with no manifest activation owner as unknown before plugin CLI registration runs, so a supported runtime-owned command such as a plugin registered with `api.registerCli(..., { commands: ["demo-plugin"] })` would throw instead of loading/registering the plugin. Please preserve that compatibility path or make the policy/docs change explicit.
  - confidence: 0.9
