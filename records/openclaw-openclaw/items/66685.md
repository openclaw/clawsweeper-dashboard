The branch imports `isApprovalNotFoundError` into `sendExecApprovalFollowupResult`, suppresses approval-not-found followup warnings, adds a regression test, and records a changelog entry.

## Best Possible Solution

Land this classifier guard with its regression test and changelog, while leaving the broader restart false-failure investigation to https://github.com/openclaw/openclaw/issues/66675.

## Solution Assessment

Yes. Reusing the existing approval-not-found classifier before formatting, dedupe, and warning is the narrowest maintainable fix for this log-noise class, and it keeps real transport failures on the existing warning path.
