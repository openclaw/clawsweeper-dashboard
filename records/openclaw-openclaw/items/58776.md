Keep open. Current main still lacks the requested sub-agent lifecycle phases and still maps aborted lifecycle ends to timeout, and an open implementation PR explicitly targets this issue with closing syntax.

Review and land a consolidated fix from https://github.com/openclaw/openclaw/pull/76356 or https://github.com/openclaw/openclaw/pull/77415 that emits and consumes additive startup lifecycle signals across embedded runs, sub-agent registry, gateway wait/session state, docs, and focused tests.
