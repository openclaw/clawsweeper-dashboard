- **[P1] Keep signal handling out of shared gateway RPC:** `src/gateway/call.ts:701-708`

  - body: Current outbound message gateway calls default to 10000 ms when no timeout is provided, but this line always sends 30000 ms for `message send`. That silently changes existing script failure timing unless maintainers deliberately approve the compatibility change and cover it in tests/docs.
  - confidence: 0.88

  - confidence: 0.86
