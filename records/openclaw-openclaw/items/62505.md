## Summary

## What This Changes

Fix a regression where coding/background agents can appear to stall or lose completion delivery after long-running tool work, especially in Codex-style runs with disabled heartbeat cadence.

## Best Possible Solution

Land a refreshed or replacement fix that lets targeted exec completion wakes run when heartbeat cadence is disabled while preserving explicit delivery suppression and retry ownership.
