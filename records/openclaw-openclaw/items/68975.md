## Summary

Keep open; current main still defaults local memory embeddings to embeddinggemma, but this PR is stale against the current memory-host-sdk layout and would not change the active runtime path as submitted. It also leaves public docs on the old default and changes the downloaded GGUF artifact without the maintainer product/security approval that a default model swap needs.

## What This Changes

The PR changes the old local memory embedding default to bge-m3 Q8_0 and adds a Vitest assertion for that exact model URI.

Decide explicitly whether bge-m3 Q8_0 should become OpenClaw's default local memory embedding model; if yes, implement the swap in packages/memory-host-sdk and the source mirror, update docs/config/changelog/tests, and verify dimension-change reindex behavior plus artifact-trust and footprint tradeoffs.
