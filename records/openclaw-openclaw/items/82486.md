Request asks OpenClaw to support bot-initiated Slack DMs, preconfigured team Slack ID aliases, and Claude Sonnet 4.6 as a default or available model.

## Best Possible Solution

Publish a team Slack routing skill/plugin on ClawHub that maps private names to Slack user IDs and calls the existing Slack message tool; keep Claude model selection in normal agent configuration.

## Reproduction Assessment
