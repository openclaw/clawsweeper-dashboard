The branch replaces two hardcoded Anthropic OAuth `claude-cli/2.1.75` user-agent constants with a shared runtime resolver that reads `@anthropic-ai/claude-code/package.json`, and adds resolver tests plus a repro script.

## Best Possible Solution

Resolve the Claude Code version lazily from a maintainer-approved supported source for both OAuth builders, or intentionally declare/package the dependency, then land one canonical PR with focused tests and redacted OpenClaw runtime/header proof.
