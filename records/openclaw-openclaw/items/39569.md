## Summary

Keep open. The PR targets a real Feishu bug, but it should not merge as-is: current main still lacks dropped-reply-mention handling, while this branch can bypass mention gating with any orphan placeholder and clears root routing that current topic-session logic intentionally preserves.

## What This Changes

The PR changes Feishu reply mention detection, clears `rootId` when the bot is treated as mentioned in a reply, and adds parser tests for orphaned mention placeholders.

## Best Possible Solution

Rework the Feishu plugin fix on current main in `bot-content.ts`, add focused regression coverage for dropped bot placeholders, and preserve the existing scope-aware reply routing instead of clearing `rootId` globally.

## Reproduction Assessment

Yes, at source level. A Feishu group reply with a non-empty `mentions` array that omits the bot but leaves its placeholder in text keeps `mentionedBot=false`, then the group mention gate drops the message.

## Solution Assessment

No. The direction is right, but this PR’s broad orphan-token detection and global root clearing are not the narrowest maintainable fix against current main.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.87

Full review comments:

- **[P1] Preserve topic roots when handling mention replies:** `extensions/feishu/src/bot.ts:823`
  - body: Clearing `rootId` for every mentioned reply removes the topic key before session resolution. In `group_topic` and `group_topic_sender` modes, current routing relies on that root to keep the session and reply target anchored to the topic instead of falling back toward chat-level behavior.
  - confidence: 0.91
- **[P2] Require bot-specific proof before accepting orphan mentions:** `extensions/feishu/src/bot.ts:489`
  - body: The orphan fallback treats any unmatched `@_user_N` token in a reply as a bot mention. A literal placeholder or a dropped non-bot mention can bypass `requireMention` and dispatch the bot even when it was not addressed.
  - confidence: 0.88

## Security Review
