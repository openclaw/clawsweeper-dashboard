## Summary

Keep open: current main and the latest release still miss the Feishu reply-mention shape, while this PR remains useful but cannot merge as-is because it lacks inspectable real Feishu proof, is conflicting, and introduces broader mention-gating and routing regressions.

## What This Changes

The PR changes Feishu mention parsing to infer dropped reply mentions from orphaned text placeholders, clears reply roots when addressed in replies, and adds parser tests.

## Best Possible Solution

Implement a Feishu-local fix that uses bot-specific dropped-mention evidence and chooses reply targets after group session scope is known, preserving normal and topic reply behavior.

## Reproduction Assessment

Yes, source-level. A Feishu group reply whose text retains an omitted bot placeholder but whose non-empty `mentions` array lacks `botOpenId` leaves `mentionedBot=false`, then mention-required group ingress can return before dispatch.

## Solution Assessment

No. The fix belongs in the Feishu plugin, but this PR's any-orphan placeholder heuristic and parse-time `rootId` clearing are broader than the narrow maintainable repair.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Preserve topic roots before clearing reply routing:** `extensions/feishu/src/bot.ts:823`
  - body: Clearing `rootId` inside `parseFeishuMessageEvent` runs before `groupSessionScope` is resolved. For `group_topic` and `group_topic_sender`, `rootId` is the topic/session key, so addressed replies with `root_id` but no `thread_id` can fall back to the chat-level session and merge unrelated topic context.
  - confidence: 0.91
- **[P2] Require bot-specific evidence for orphan placeholders:** `extensions/feishu/src/bot.ts:484`
  - body: This fallback marks any unmatched placeholder token in reply text as a bot mention. A literal token or dropped non-bot mention can bypass `requireMention` and trigger root clearing without evidence that the bot was addressed.
  - confidence: 0.88
