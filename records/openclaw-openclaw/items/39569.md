## Summary

Keep open. Current main still appears to miss the Feishu dropped-reply-mention path, and this PR is the linked implementation candidate, but it needs contributor proof and code rework before merge because the branch broadens mention gating and clears reply roots too early.

## What This Changes

The PR changes the Feishu plugin to treat unmatched reply mention placeholders as bot mentions, clear `rootId` for mentioned replies, and add parser tests.

## Best Possible Solution

Land a rebased Feishu plugin fix that detects the dropped bot mention with the strongest bot-specific evidence available, preserves topic and explicit-thread roots, and adds parser plus routing regression coverage.

## Reproduction Assessment

Yes, by source inspection. A Feishu group reply whose text contains the bot placeholder but whose non-empty `mentions` array omits `botOpenId` leaves `mentionedBot=false` and can be dropped by the `requireMention` gate.

## Solution Assessment

No. The PR targets the right bug, but the safe fix should live in the current `bot-content.ts` split, use bot-specific dropped-mention evidence where possible, and choose reply targets after session scope is known.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Preserve topic roots for mentioned replies:** `extensions/feishu/src/bot.ts:812-813`
  - body: Clearing `rootId` during parse happens before `groupSessionScope` and `replyInThread` are known. Topic and explicitly threaded groups use that root as the session/reply target, so addressed replies can collapse to the chat-level session or wrong target instead of staying in the topic.
  - confidence: 0.88
- **[P2] Require bot-specific evidence for orphan placeholders:** `extensions/feishu/src/bot.ts:481`
  - body: The fallback treats any unmatched `@_...` token in a reply as a bot mention. A literal placeholder or dropped non-bot mention can satisfy `requireMention` and trigger OpenClaw without evidence the bot was addressed.
  - confidence: 0.86
