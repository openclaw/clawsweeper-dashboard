  - confidence: 0.93

  - body: Bookmarks, canvases, and reminders are opt-in action groups, but the default app manifest now requests their read/write scopes for every new Slack install. Keep the base manifest least-privilege or gate these scopes behind explicit setup opt-in.
  - confidence: 0.91

  - confidence: 0.88
