  - body: Bookmarks, canvases, and reminders are runtime opt-in action groups, but the default setup manifest asks every new Slack install for their read/write bot scopes. Keep the base manifest least-privilege or move these scopes behind an explicit opt-in setup path.

- **[P2] Include user-token-only accounts in search discovery:** `extensions/slack/src/message-actions.ts:11-13`
  - body: Search discovery checks `userTokenSource` only after the account list has already been filtered to bot-token accounts, so a user-token-only account that can execute search is never advertised.   - body: `postAt` accepts strings, but digit-only Unix-second strings still fall through to `Date.parse`, so serialized payloads like `"1725000000"` throw even though Unix seconds are supported. Parse numeric strings before date parsing or narrow the schema/docs to numbers only.
