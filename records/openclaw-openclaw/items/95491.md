## What This Changes

This PR updates WebChat model switching to recache the dropdown override from the `sessions.patch.resolved` model and also adjusts a legacy real-behavior-proof parser regex.

## Best Possible Solution

Preserve `null`/Default cache semantics, reconcile only explicit non-empty dropdown selections with `sessions.patch.resolved`, add a Default-clearing regression, and split or test the proof-parser change before merge.
