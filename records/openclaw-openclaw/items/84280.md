Overall correctness: patch is incorrect

Full review comments:

- **[P1] Do not hand off when no Windows restart target is installed:** `src/cli/daemon-cli/lifecycle.ts:222`
  - body: `restartGatewayWithoutServiceManager` runs from the service `not loaded` path, which on Windows means the Scheduled Task/startup entry is already missing. Calling `triggerOpenClawRestart()` here first runs stale-process cleanup and then reports ok once a detached helper is spawned, even though the helper may later find no task and do nothing, so a raw gateway can be killed and the CLI times out instead of failing with the promised install hint. Guard for an installed restart target or fail before this handoff.
  - confidence: 0.89
