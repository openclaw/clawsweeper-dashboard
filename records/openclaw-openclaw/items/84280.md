## What This Changes

The PR adds a Windows SIGUSR1 guard and reroutes unmanaged gateway restart on Windows through the scheduled-task restart handoff with focused unit tests.

## Best Possible Solution

Keep the SIGUSR1 guard, but make Windows unmanaged restart either validate a supervised handoff before listener cleanup or fail clearly without stopping the running gateway, then prove the no-service-manager flow.
