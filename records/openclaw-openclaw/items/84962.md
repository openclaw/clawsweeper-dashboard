## What This Changes

The PR replaces Docker workspace-deps bind mounts with COPY steps, copies deploy assets into the runtime image, adds a bundled Railway gateway config, and documents loading it with `OPENCLAW_CONFIG_PATH`.

## Best Possible Solution
