## What This Changes

Request to add Telegram-specific handling that collapses soft mid-paragraph line breaks while preserving intentional paragraph breaks.

## Best Possible Solution

Add a Telegram-owned, Markdown-aware soft-line-break normalization path after maintainers choose default and config semantics, preserving blank lines, fences, blockquotes, lists, escaping, chunk limits, and plain fallback behavior.
