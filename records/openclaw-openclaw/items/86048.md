## What This Changes

The issue reports a WSL2 GPU-PV/NVIDIA driver lockup after a llama-server CUDA stall that leaves OpenClaw inference unavailable until WSL is restarted.

Keep OpenClaw’s core boundary at generic provider timeouts, readiness probes, and localService process management; handle WSL2/NVIDIA/llama-server driver recovery through external operations or a provider/plugin-specific design if a concrete missing OpenClaw API is identified.
