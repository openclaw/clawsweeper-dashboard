## Best Possible Solution

Land a Discord-code-aware classifier for code 10065 or explicit Unknown Voice State text, keep other 404s failing, and add redacted real Discord voice-status proof before merge.

## Reproduction Assessment

## Solution Assessment

No. Returning absence for explicit Unknown Voice State is the right direction, but matching every 404 is too broad when the local REST client preserves Discord's specific error code.
