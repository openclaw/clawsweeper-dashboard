## What This Changes

The PR adds warning diagnostics for bootstrap-named files found in a configured `agentDir` but absent from the workspace, adds tests for that behavior, and updates the agent workspace docs.

## Best Possible Solution

Land a warning-only diagnostic only after the check covers the shared bootstrap resolver path, docs match the exact trigger, real runtime proof is supplied, and maintainers confirm warning-only is the intended policy.
