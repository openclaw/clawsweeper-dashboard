## Summary

## What This Changes

The PR adds OpenRouter inference for selected `qwen/...` and `minimax/...` slash-style model refs, widens allowlist resolution for nested OpenRouter IDs, and adds model-selection regression tests.

## Best Possible Solution

## Solution Assessment
