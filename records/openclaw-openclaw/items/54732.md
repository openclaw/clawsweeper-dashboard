## Summary

Keep open. Current main still scopes the `message` action enum and plugin-owned schema fragments, but the shared core schema groups are still added unconditionally, so the focused token-reduction cleanup remains valid.

## What This Changes

Trim the shared `message` tool schema so core parameter groups are included only when the resolved enabled actions can use them.

## Best Possible Solution

Implement action-aware shared schema trimming in `src/agents/tools/message-tool.ts` using the resolved action set, with regression coverage for a send/react-only channel and for cross-channel actions whose parameters must remain visible.
