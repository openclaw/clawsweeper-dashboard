## Best Possible Solution

Fix the OpenCode built-in command in `openclaw/acpx`, publish a new acpx release, bump OpenClaw's ACPX dependency, refresh ACP router documentation, and validate with an OpenCode live ACP bind smoke.

## Reproduction Assessment
