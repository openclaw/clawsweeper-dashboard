  - body: For a configured `google-vertex` provider/model that explicitly keeps `api: "google-generative-ai"` with an AI Studio or proxy base URL, this predicate still chooses the Vertex transport solely because `model.provider === "google-vertex"`.   - confidence: 0.9

  - body: `response.output_item.added` for a reasoning item creates a thinking block before `response.completed`; then this guard sees `output.content.length > 0` and skips the final completed-output message or function call. Backfill should skip already-seen items, not disable itself after any prior reasoning/thinking block.
  - confidence: 0.91

  - body: Deleting `liveModelSwitchPending` here means an already-running session on a user-pinned model will not see that `sessions.patch` reset the persisted selection back to the default, because `shouldSwitchToLiveModel` only reconciles when that flag is present.   - confidence: 0.86
