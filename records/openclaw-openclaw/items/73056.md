## Summary

Keep open. Current main still has the raw-session outer compaction pre-check that can return before SDK preparation, and open PR #73092 is already the linked implementation candidate.

## What This Changes

Fix manual `/compact` so long tool-heavy sessions over the compaction threshold are not skipped as idle by the outer raw-message pre-check.

## Best Possible Solution

Review and land PR #73092 or an equivalent narrow patch, then close this as implemented once the fix is on main.

## Reproduction Assessment

Yes. A source-level reproduction is a history-limited `session.messages` tail where tool results have no meaningful user ask in the 20-message lookback, combined with over-threshold context usage; current main can still return the idle skip before compaction starts.

## Solution Assessment

Yes. Passing fresh observed token usage into manual compaction and bypassing only the outer idle skip when over threshold is narrower than weakening the shared real-conversation classifier.
