## Summary

Keep open. Current main still performs the raw-session outer real-conversation pre-check before calling `session.compact()`, manual `/compact` still does not pass observed token usage into that path, and open PR #73092 is already the linked implementation candidate.

## What This Changes

Manual `/compact` should not report an idle/no-real-conversation skip for over-threshold, tool-heavy Discord thread sessions.

## Best Possible Solution

Review and land PR #73092, or an equivalent targeted patch, so the empty-session skip remains intact while over-threshold manual compaction proceeds using observed token usage.

## Reproduction Assessment

Yes. A source-level reproduction is a limited `session.messages` slice made of heartbeat/tool-result entries, or tool results whose meaningful user ask is outside the 20-message lookback, combined with over-threshold context usage.

## Solution Assessment

Yes. The narrow maintainable direction is to propagate observed manual token usage and bypass only the over-threshold outer idle skip, rather than weakening the shared conversation classifier.
