## Summary

Keep open. Current main still has the raw-session outer compaction pre-check that can return `no real conversation messages` before compaction starts, and open PR #73092 already references this issue as the implementation candidate.

## What This Changes

Manual `/compact` should not classify over-threshold, tool-heavy sessions as idle before compaction can run.

## Best Possible Solution

Land PR #73092 or an equivalent patch that preserves the empty-session skip while allowing over-threshold manual compaction to proceed past the outer guard using fresh observed token usage.

## Reproduction Assessment

Yes. A high-confidence source-level reproduction is a limited `session.messages` slice of tool results whose meaningful user ask is outside the 20-message lookback, combined with over-threshold context usage and the manual `/compact` path.

## Solution Assessment

Yes. The narrow maintainable fix is to propagate manual token usage and bypass only the outer idle skip when the session is over the configured threshold, rather than weakening the shared real-conversation classifier.
