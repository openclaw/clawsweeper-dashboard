## Summary

Keep open. Current main still has the source-level failure path, and open PR #73092 is the tracked unmerged implementation candidate.

## What This Changes

Manual `/compact` should not skip a long tool-heavy session as idle when observed context usage is over the compaction threshold.

## Best Possible Solution

Land PR #73092 or an equivalent narrow patch that preserves empty-session skips while allowing over-threshold manual compaction to continue past the outer idle guard.

## Reproduction Assessment

Yes. Source inspection gives a high-confidence path: a limited `session.messages` slice with heartbeat/tool-result entries and no meaningful user message within the 20-message lookback still returns `no real conversation messages` before compaction, while manual `/compact` does not pass `currentTokenCount` to override that.

## Solution Assessment

Yes. The narrow maintainable fix is to carry fresh manual token usage into compaction and bypass only the outer idle guard when the session is over threshold, instead of weakening the shared real-conversation classifier.
