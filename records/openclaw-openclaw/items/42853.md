- **[P1] Generate only the managed heap flag:** `src/daemon/service-env.ts:340`
  - body: This writes resolveNodeOptions(env, ...) into durable service metadata, but resolveNodeOptions copies any ambient NODE_OPTIONS through unchanged or appends to it.
