## Summary

Keep open. Current main still has only the fixed Discord generated-thread-title prompt and does not expose the requested per-channel title-guidance config, schema entry, runtime plumbing, docs, or tests; the linked merged PR added generated titles, not this customization.

## What This Changes

The issue asks to add an optional per-channel Discord `threadPromptGuideline` setting for `autoThreadName: "generated"` so generated thread titles can follow project, language, and formatting guidance.

## Best Possible Solution

Add a backward-compatible, bounded Discord guild-channel title-guidance field that supplements the existing concise title-only prompt contract, then update schema, generated metadata, docs, tests, and changelog together if maintainers approve the config shape.

## Solution Assessment
