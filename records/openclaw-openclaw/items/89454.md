This PR adds an async Feishu helper that resolves unresolved `appId`/`appSecret` SecretRefs for selected outbound and message-tool send paths, plus regression tests around account resolution and a message-tool text send.

## Best Possible Solution

Resolve Feishu app credential SecretRefs before every user-visible send path that can build a Feishu client, preferably at a shared dispatcher or client-build boundary, with regression coverage for a final MEDIA payload.

## Reproduction Assessment

## Solution Assessment

No. The outbound/message-tool helper is a plausible mitigation, but the best pre-merge fix should cover the source-channel reply dispatcher path named by the report or centralize resolution at the Feishu client-build boundary.
