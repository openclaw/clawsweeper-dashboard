## What This Changes

## Best Possible Solution

Land a queue-owned persistence layer only after all queue-removal paths, including aborted drops, durably acknowledge state and redacted live Telegram proof confirms ordered exactly-once redelivery; keep startup-time cold-session dispatch scoped to https://github.com/openclaw/openclaw/pull/63330 unless maintainers deliberately expand this PR.
