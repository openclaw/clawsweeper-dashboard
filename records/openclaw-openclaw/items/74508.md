  - body: The adaptive throttle and chat send gate are process-global, and createTelegramDraftStream only receives chatId.   - confidence: 0.8

Keeping them will cause generated-doc drift or rebase churn; remove the hash change unless this branch actually changes generated config output.
  - confidence: 0.84
- **[P3] Use the remaining gate delay for retries:** `extensions/telegram/src/draft-stream.ts:471-475`
  - body: When the per-chat gate blocks a flush, the retry timer waits a full effectiveSendInterval instead of the remaining time until the chat is eligible. If no new deltas arrive to trigger another flush, preview text can wait roughly an extra interval before sending.
  - confidence: 0.72
