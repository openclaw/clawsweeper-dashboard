  - body: The adaptive throttle is process-global and the chat send gate is keyed only by chatId. In a supported multi-account Telegram setup, a 429 or recent send from one bot account can pause or delay another bot account even though their Telegram quotas are separate.
  - confidence: 0.86

  - body: When the per-chat gate blocks a flush, the retry timer waits a full effective interval instead of the remaining time until the chat is eligible. If no new deltas arrive, preview text can wait roughly an extra interval before sending.
  - confidence: 0.72
