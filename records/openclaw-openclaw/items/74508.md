## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Remove discarded streams from the chat gate queue:** `extensions/telegram/src/draft-stream.ts:541-550`
  - body: `discard()` can run for a superseded dispatch while this stream ID is sitting in `_perChatSendGate` after `acquireChatSendGate` returned false. Since discard only clears timers and stops the loop, that stale ID can remain at the head of the per-chat queue, making later streams for the same chat keep returning false indefinitely. Remove the stream ID from the gate queue when a stream is discarded or otherwise stopped without final gate acquisition.
  - confidence: 0.86
