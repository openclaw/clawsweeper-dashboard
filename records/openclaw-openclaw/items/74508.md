## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Remove stale streams from the send gate:** `extensions/telegram/src/draft-stream.ts:117-122`
  - body: The new per-chat gate queues stream IDs, but a stream that is cleared or discarded while waiting is never removed from `gate.queue`. If that dead stream remains at `gate.queue[0]`, every later draft stream for the same chat takes this branch, appends itself behind the stale ID, and can never send until process state is reset. Please unregister the stream ID on all teardown paths or prune stale queue entries before enforcing head-of-line ordering.
  - confidence: 0.86
- **[P2] Revert the unproved config baseline hashes:** `docs/.generated/config-baseline.sha256:1-4`
  - body: This PR changes only `docs/.generated/config-baseline.sha256`; it does not include config schema/source changes or generated config artifacts that would produce the new hashes. Since `src/config/doc-baseline.ts` computes this file from generated config baselines, merging these hash changes would either fail the config docs baseline check or mask unrelated drift. Please revert this file unless the branch intentionally changes config output and includes the corresponding generated proof.
  - confidence: 0.78
