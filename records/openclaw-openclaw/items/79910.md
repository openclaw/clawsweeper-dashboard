## Summary

Keep open: the recovery direction is useful, but the PR currently promotes 10-second-old temp artifacts even though current writer and cleanup code treat session-store temps as potentially in flight for 5 minutes, and the supplied terminal proof is for an older head after a later force-push.

## What This Changes

## Best Possible Solution
