## Summary

## What This Changes

Fix memory indexing so `chunkMarkdown` preserves UTF-16 surrogate pairs at coarse split boundaries and embedding inputs sanitize lone surrogates before provider calls.

## Best Possible Solution

Land or replace #65783 with the same narrow two-layer fix: preserve surrogate pairs at the coarse chunk boundary, sanitize lone surrogates before embedding provider calls, and keep focused regression tests for both paths.
