## What This Changes

Telegram reasoning streaming should update and clean up a single preview message instead of accumulating stale reasoning preview messages.

## Best Possible Solution

Repair the Telegram reasoning split path so the previous reasoning preview is stopped, preserved, or cleared before starting the next preview, then add a focused regression test and Telegram behavior proof.

## Solution Assessment
