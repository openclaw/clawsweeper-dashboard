## Summary

## What This Changes

This PR disables Codex app-server native execution surfaces during active OpenClaw sandboxed runs, exposes sandbox-backed `sandbox_exec`/`sandbox_process`, updates transient thread context projection, docs, tests, and the changelog.

## Best Possible Solution

Land a single maintainer-approved sandbox contract for Codex: active OpenClaw sandboxes should either run Codex-native execution inside the sandbox or fail closed to OpenClaw-owned sandbox tools with explicit docs and upgrade expectations.

## Solution Assessment
