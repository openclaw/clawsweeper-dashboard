## Summary

This PR is still necessary: current main and v2026.6.6 still promote activate:false tool-discovery registries into the global active registry, while the PR moves the guard to the standalone install boundary and keeps pinned tool-catalog metadata visible. ## What This Changes

The PR prevents activate:false tool-discovery standalone registry loads from replacing the active plugin registry, adds raw pinned channel registry reads for tools.catalog metadata, and adds focused regression tests.
