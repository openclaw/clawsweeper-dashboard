## Summary

Keep open. The underlying session-memory poisoning bug is still present on current main, and this PR is a useful fix direction, but the patch should not merge as-is because the sanitizer misses token families already modeled in the repo, leaves truncated tool-call XML intact, and can elide valid short replies after removing a leaked token. The overlapping #73768 is relevant but not a complete supersession because it covers broader special-token reuse while omitting this PR's tool-call and malformed-turn handling.

## What This Changes

The PR adds a session-memory sanitizer module, wires it into transcript reading and handler persistence, and adds tests for stripping leaked chat-template tokens, raw tool-call XML, orphan role labels, and malformed turns before writing memory files.

## Best Possible Solution

Consolidate the two open approaches into one focused session-memory fix: reuse or export the existing special-token sanitizer for broad token coverage, add memory-specific handling for raw and truncated tool-call XML plus NO_REPLY/orphan-role elision, preserve valid short content, add the missing changelog entry, and land that single implementation for #69943.

## Reproduction Assessment

Yes. A synthetic session JSONL message containing `<tool_call>...</tool_call><|im_end|>` or a special token is enough to reproduce the current-main persistence path because `getRecentSessionContent` returns raw text and the handler writes it under `## Conversation Summary`. The PR's own tests cover part of this path, and the review findings have small string-level repros such as `2+2=4<|im_end|>` and an unterminated `<tool_call>` block.

## Solution Assessment

No. Sanitizing at the transcript/read and handler/write boundaries is the right shape, but this implementation is not the best mergeable form because it duplicates narrower token logic than the existing helper and has avoidable false-positive and truncation gaps. The safer implementation is a shared broad token sanitizer plus session-memory-specific artifact elision tests.
