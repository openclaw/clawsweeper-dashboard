## Best Possible Solution

Capture a real Feishu custom-sticker payload, then update the Feishu parser to preserve verified sticker label text in agent context while keeping the existing media attachment behavior; only add Unicode mapping after the plugin owner confirms the mapping source and maintenance policy.
