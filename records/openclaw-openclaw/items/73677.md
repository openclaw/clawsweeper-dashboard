Preserve Feishu/Lark custom sticker meaning by converting bracket-style stickers to Unicode emoji or text visible to the agent.

## Best Possible Solution

Fix this in `extensions/feishu` by preserving verified text-like sticker labels in the inbound parser while keeping the media attachment path, then add focused parser/body tests.
