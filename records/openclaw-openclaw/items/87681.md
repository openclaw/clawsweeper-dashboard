- **[P2] Handle numeric SIGKILL exits too:** `src/agents/bash-tools.exec-runtime.ts:544`
  - body: The new hint only triggers for the string `SIGKILL`, but `RunExit.exitSignal` also allows numbers and the PTY adapter forwards `node-pty`'s numeric `event.signal` (`9` for SIGKILL).
