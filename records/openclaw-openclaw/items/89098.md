Full review comments:

- **[P1] Preserve plugin before-call hooks when materializing tools:** `src/plugins/tools.ts:557-600`
  - body: `materializePluginTool` rebuilds a plugin tool with only selected descriptor fields plus execute/prepareArguments/executionMode/displaySummary. `AnyAgentTool` also exposes `prepareBeforeToolCallParams` and `finalizeBeforeToolCallParams`, and the before-tool-call wrapper reads those fields before execution, so plugins using those hooks would silently lose argument preparation/finalization after this change.
  - confidence: 0.88
- **[P2] Preserve replaySafe metadata on resolved plugin tools:** `src/plugins/tools.ts:1489-1496`
  - body: The new `pluginToolMeta.set` call omits the `replaySafe` value that current main stores from manifest tool metadata. Agent retry/replay logic consults `getPluginToolMeta(tool).replaySafe`, so tools previously declared replay-safe would become unsafe after this patch and change recovery behavior.
  - confidence: 0.84
