## Summary

Keep open. The PR likely fixes the Flash-Lite case, but it also raises Pro and non-Lite Flash minimal budgets even though current Google AI and Vertex docs still list lower accepted minima for those model families; without live provider proof that the docs are stale, this needs maintainer/provider review before merge. 

## What This Changes

The PR raises Gemini 2.5 Pro and Flash/Flash-Lite `minimal` thinking budgets from 128 to 512 and adds a parameterized Google transport test matrix for Gemini 2.5 reasoning budgets.

## Best Possible Solution

Land a targeted Google plugin fix that aligns each Gemini 2.5 family with the verified provider contract: special-case Flash-Lite to a 512 minimal budget, keep Pro and non-Lite Flash at their documented lower minima unless live evidence proves the docs are stale, and retain focused transport tests for changed and unchanged tiers.

## Solution Assessment

No. The current PR is broader than the safest documented fix because only Flash-Lite has a documented 512 minimum; the narrower solution is to special-case Flash-Lite or attach live per-model evidence before raising Pro and non-Lite Flash.
