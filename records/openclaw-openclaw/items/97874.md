## Best Possible Solution

Yes, this is the best fix shape: subcommand-level gating preserves `/voice status` and `/voice list`, while `requiredScopes` on the whole command would over-block read-only behavior. The trusted bundled owner-status seam is already the adjacent phone-control pattern.
