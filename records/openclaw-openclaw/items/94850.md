## What This Changes

The issue requests neutral default Telegram progress draft labels instead of an automatic default pool that can show "Lobstering" without user opt-in.

## Best Possible Solution

If maintainers want neutral defaults, update the shared default label pool and progress-draft docs/tests while keeping custom `streaming.progress.label` and `streaming.progress.labels` as the opt-in path for themed copy.

## Reproduction Assessment

Yes, source-level reproduction is high confidence: leave progress labels unset or `auto`, then any seed selecting index 15 from the 20-label pool renders `Lobstering` as the progress draft header. I did not run a live Telegram turn in this read-only review.

Unclear until maintainer product direction is confirmed: the narrow code change is straightforward, but it reverses an explicitly intentional shipped default from the progress-drafts introduction. The safest path is maintainer-approved neutral default copy plus docs and regression updates.
