## What This Changes

This PR adds a bundle-MCP consent envelope that routes opted-in MCP tool calls through plugin approvals, adds `mcp.approvals` config/docs, plumbs agent/session routing, updates `/approve` handling, and covers the flow with focused tests.

## Best Possible Solution
