## What This Changes

Requests Simplified Chinese localization across the Gateway/Control UI, including human-readable Chinese labels for audit event names.

## Best Possible Solution

Add display-only localized aliases for an explicitly approved finite set of audit, event, tool, and action labels through the English-source/generated-locale pipeline, while retaining raw identifiers in protocol payloads and diagnostic details.

## Solution Assessment

Yes, a finite display-alias layer is the narrowest maintainable direction; translating protocol data, commands, or arbitrary debug JSON would conflate presentation with canonical identifiers.
