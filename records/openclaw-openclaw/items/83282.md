## Summary

Keep open: current `main` still has the reported source-level merge bug, and open pull requests already reference this issue with closing syntax, so the issue should remain open until one fix lands.

## What This Changes

The issue reports that an encoded trajectory export request can overwrite an explicitly supplied `sessionKey` with an empty decoded placeholder.

## Best Possible Solution

Land one of the focused fixes that omits absent or blank decoded fields without changing valid encoded override behavior, with regression coverage for direct options plus a partial encoded request.

## Reproduction Assessment

Yes. Source inspection gives a high-confidence reproduction path: call the export trajectory command with a direct `sessionKey` and an encoded request containing only another option such as `output`, and the decoded empty `sessionKey` can replace the direct value before the required-key check.

## Solution Assessment

Yes. The narrow maintainable fix is to stop materializing absent or blank decoded fields, or otherwise merge decoded values without overwriting already-defined direct options with placeholders.
