Adds `channels.discord.implicitReplyMentions` to Discord config, schema, docs, generated metadata, and preflight handling so bot-authored native reply-only activation can be disabled while preserving explicit mentions.

## Best Possible Solution

Land or replace this PR with an approved Discord-owned policy that disables bot-authored reply-only activation without regressing human native replies or textual explicit mentions.

## Solution Assessment
