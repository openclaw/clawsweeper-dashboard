## What This Changes

Add a run-scoped loop detector that blocks after a configurable threshold of consecutive completed tool errors across different tools.

## Best Possible Solution

Review and land #75924, or an equivalent narrow main patch, with detector logic, config/schema/help/docs, and focused unit plus hook-path coverage.
