The PR adds reset-archive lookup to bundled session-memory transcript recovery, adds tests for reset-only archive recovery, updates the session-logs skill examples for reset/deleted archives, and syncs a workspace lockfile importer.

Land this PR or an equivalent rebased patch after moving reset archive fallbacks behind every live `.jsonl` candidate, making the docs helper robust when `nullglob` is disabled under `set -e`, adding a single-line changelog entry with contributor credit, and leaving gateway/API archive visibility to the related PRs.
