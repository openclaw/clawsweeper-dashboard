Full review comments:

- **[P1] Keep reset fallback tied to the requested session:** `src/hooks/bundled/session-memory/transcript.ts:167-172`
  - body: After the requested session's active, canonical reset, and topic reset candidates fail, this picks the newest reset archive anywhere in the directory. A stale or missing pointer can summarize another session into the current session's memory; remove this fallback or require a matching session header/id.
  - confidence: 0.9
- **[P1] Gate deleted transcript recall separately:** `skills/session-logs/SKILL.md:78-80`
  - body: The skill now treats `.jsonl.deleted.*Z` as part of normal full recall. `sessions.delete` removes the session entry while archiving its transcript, so making deleted content a default search target changes the deletion/privacy boundary without an explicit maintainer decision.
  - confidence: 0.88
- **[P3] Preserve nullglob under set -e:** `skills/session-logs/SKILL.md:76`
  - body: `shopt -p nullglob` exits nonzero when the option is unset, so this helper can exit immediately under `set -e` before restoring state. Initialize the unset state explicitly or guard the assignment.
  - confidence: 0.78
