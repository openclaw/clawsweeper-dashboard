## Summary

Keep open. The reset-archive memory recovery is useful and the terminal proof is strong enough for the helper path, but the skill guidance should not make `.deleted.*` transcripts part of default recall without an explicit privacy/product decision.

## What This Changes

## Best Possible Solution

Land a narrower version that keeps `.reset.*` memory recovery and reset-archive search guidance, while making `.deleted.*` searches explicit opt-in with a privacy warning or leaving them out of the skill.
