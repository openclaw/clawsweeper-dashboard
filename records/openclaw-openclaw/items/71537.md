- **[P2] Keep reset fallback tied to the requested session:** `src/hooks/bundled/session-memory/transcript.ts:167-172`
  - body: When the specific reset path and sessionId do not match anything, this directory-wide fallback returns the newest `.jsonl.reset.*` file in the whole sessions directory.
