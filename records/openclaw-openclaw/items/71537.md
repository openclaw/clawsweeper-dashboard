Full review comments:

- **[P2] Keep reset fallback tied to the requested session:** `src/hooks/bundled/session-memory/transcript.ts:167-172`
  - body: When the specific reset path and sessionId do not match anything, this directory-wide fallback returns the newest `.jsonl.reset.*` file in the whole sessions directory. That can make the session-memory hook summarize another session's archive instead of writing no memory, so keep reset recovery tied to the current basename/sessionId or return `undefined`.
  - confidence: 0.78
- **[P3] Keep the helper working under set -e:** `skills/session-logs/SKILL.md:76`
  - body: In Bash, `shopt -p nullglob` exits nonzero when `nullglob` is unset, so this assignment can abort a copy-pasted helper in a `set -e` script before it searches anything. Use an unset-safe save/restore pattern such as appending `|| true` inside the command substitution.
  - confidence: 0.91
