Keep open: this is a fresh, plausibly source-backed Ollama/Qwen tool-call regression, and current code shows a path where model capability metadata can remove OpenClaw tools before the Ollama chat request. ## What This Changes

## Best Possible Solution

Keep this open until a live comparison captures /api/show capabilities and /api/chat payloads for v2026.5.7, v2026.5.12, and current main, then restore the compatible Ollama tool-call path or document the required model compat override.
