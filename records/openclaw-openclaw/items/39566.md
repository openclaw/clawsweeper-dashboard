## What This Changes

The PR adds `topics."*"` fallback semantics for Telegram group and DM topic config, with old-path tests/docs/type comments and `.secrets.baseline` line-number updates.

## Best Possible Solution

Land a refreshed PR that implements exact-topic-then-wildcard fallback in `extensions/telegram`, covers group and DM topic defaults, updates docs/types/changelog as needed, and leaves the removed secret baseline out.

## Reproduction Assessment
