## What This Changes

## Best Possible Solution

Define a reusable transient sensitive-input reveal component per platform, apply it only to approved credential/API-key fields, avoid persisted reveal state unless explicitly approved, and add focused UI or view tests for the touched surfaces.

## Solution Assessment
