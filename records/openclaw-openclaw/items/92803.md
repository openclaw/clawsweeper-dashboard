## Best Possible Solution

No. The shared resolver is the right choke point, but resolving MiniMax M3 fixed-budget defaults to `off` is not proven safe because the downstream payload path can become explicit disabled thinking; the better fix needs payload/live proof or a maintainer-approved diagnostic/provider-policy path.
