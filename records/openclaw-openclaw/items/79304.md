## Best Possible Solution

Keep the shipped Weixin catalog fix, then decide the Codex Computer Use path: either add a focused read-only bundled-marketplace remediation for status or fold Computer Use into reviewed native Codex plugin marketplace support without breaking the existing computerUse config.
