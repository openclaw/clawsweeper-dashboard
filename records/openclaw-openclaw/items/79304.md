## Summary

Keep open: the Weixin runtime and fetch regressions are fixed through the external package/catalog path, but the Codex Computer Use status path still returns generic `marketplace_missing` when the bundled marketplace exists but is not registered.

## What This Changes

## Best Possible Solution

Keep the shipped Weixin catalog/runtime fixes, and add a narrow Codex Computer Use status remediation that detects an existing bundled marketplace path without mutating status checks.

## Solution Assessment
