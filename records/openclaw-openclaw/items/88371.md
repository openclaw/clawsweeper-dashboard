The issue asks Windows QuickStart to avoid or clearly warn about paid Anthropic default models before the first terminal hatch chat.

## Best Possible Solution

Make the Anthropic first-run policy explicit before the first hatch through a safer default, a clear billing/credit warning or preflight, or documented intentional Opus behavior with better provider-failure messaging.
