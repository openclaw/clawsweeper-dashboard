## Summary

Keep open. Current main still forwards out-of-range read offsets into the upstream read tool without handling the EOF error, and the directly related fix PR was closed unmerged. The issue is narrow, reproducible from source, and a good focused repair candidate.

## What This Changes

Make the read tool handle offsets beyond the file length gracefully instead of surfacing the upstream `Offset X is beyond end of file` failure.

## Best Possible Solution

Add a narrow read-wrapper recovery path that handles out-of-range positive offsets consistently in both explicit-limit and adaptive paging flows, with regression coverage for one-past-EOF, far-past-EOF, explicit-limit, and empty-file cases.

## Reproduction Assessment

Yes. A small text file read through `createSandboxedReadTool` or `createOpenClawReadTool` with `offset` greater than the file's line count should reproduce because current main forwards that offset into the pinned upstream read implementation that throws on `startLine >= allLines.length`.

## Solution Assessment

Yes, a focused OpenClaw wrapper fix is the best direction. It should avoid dependency changes and cover both existing wrapper paths, while preserving current sandbox and image-normalization behavior.

## Repair Work Prompt

Fix openclaw/openclaw#62466. Current main's `createOpenClawReadTool` / `executeReadWithAdaptivePaging` forwards `offset` values beyond the file length into `@mariozechner/pi-coding-agent@0.70.6`, which throws `Offset X is beyond end of file`. Implement a narrow OpenClaw wrapper fix in `src/agents/pi-tools.read.ts` that covers both the explicit `limit` path and the adaptive paging loop. Prefer deterministic behavior for positive offsets beyond EOF: retry once with a safe offset for nonempty files and handle empty files gracefully, or return a clear EOF result if maintainers prefer that semantic. If EOF happens during later adaptive pagination, return the content already aggregated instead of failing. Do not change dependency versions, tool schemas, sandbox policy, path alias behavior, or unrelated read/image handling. Add colocated regression tests for offset one past EOF, far past EOF, explicit limit plus offset past EOF, and empty file plus offset past EOF.
