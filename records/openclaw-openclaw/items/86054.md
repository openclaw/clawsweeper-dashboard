## What This Changes

This PR rewrites `.github/pull_request_template.md` to shorten visible sections, collapse guidance in `<details>`, preserve real-behavior proof fields, and add a current review state section.

## Best Possible Solution

Keep the shorter template shape, but make the proof-limitations prompt parser-neutral or update the proof policy and docs so every visible proof field has a matching contract.
