  - body: The new `stripHiddenSystemEvents` default changes every existing `preprocess(markdown:)` caller. `ChatMarkdownRenderer` still uses the default, so user-pasted `System ... Exec ...` log quotes can disappear at render time even after the view-model preservation fix.

  - body: `detectInstallState` marks any symlinked active root as suspicious and emits a reinstall/relink recovery hint.   - confidence: 0.86

  - body: The mutating maintenance sweep creates one session-store cache before processing every task, then yields between batches. If session state changes after a yield, later `shouldMarkLost` checks can use a stale snapshot and mark active tasks as lost.
