- **[P1] List every approval decision in pending text:** `src/infra/exec-approval-reply.ts:260`
  - body: `buildExecApprovalPendingText` renders only `descriptors[0]`, so text-only clients and copied logs show only an allow command while `deny` and other allowed decisions remain hidden in interactive metadata.   - confidence: 0.93
- **[P1] Refresh session stores after maintenance yields:** `src/tasks/task-registry.maintenance.ts:381`
  - body: `runTaskRegistryMaintenance` creates one cached session-store context for the whole mutating sweep, then yields between batches. A session created or updated after a yield can be invisible to later `shouldMarkLost` checks using the stale cache, so active tasks can be marked `lost`; refresh or invalidate the cache across yield boundaries for the mutating path.
  - confidence: 0.86
- **[P2] Make hidden event stripping opt-in for rendering:** `apps/shared/OpenClawKit/Sources/OpenClawChatUI/ChatMarkdownPreprocessor.swift:49`
  - body: The preprocessor now defaults `stripHiddenSystemEvents` to true, and `ChatMarkdownRenderer` still calls it without role/source context. User-authored logs containing `System ... Exec completed` can therefore be stripped during normal rendering even though decode preserves them; make stripping opt-in or pass role context at the render call.
  - confidence: 0.88
- **[P2] Do not warn on symlinks by themselves:** `src/infra/update-check.ts:397-398`
  - body: `detectInstallState` treats any symlinked package root as suspicious and emits a reinstall/relink recovery hint even when the symlink resolves to a healthy install. Symlinked roots are common for linked installs and version managers, so warn only on a real mismatch or unsafe resolved path.
  - confidence: 0.82
