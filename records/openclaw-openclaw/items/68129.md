  - body: `buildExecApprovalPendingText` builds descriptors for every allowed decision but only renders `primaryAction.command`.   - body: The new `stripHiddenSystemEvents` default changes every existing `preprocess(markdown:)` caller. User-pasted `System ... Exec ...` log quotes can disappear at render time unless every caller explicitly knows whether the source is trusted injected metadata.
  - confidence: 0.9

  - body: `detectInstallState` marks any symlinked active root as suspicious and emits a reinstall/relink recovery hint. Symlinked package roots are common for linked installs or version-manager setups, so this can create persistent false drift warnings unless the resolved roots actually mismatch or land in an unsafe path.
  - confidence: 0.86
- **[P1] Refresh session lookups during mutating sweeps:** `src/tasks/task-registry.maintenance.ts:381-392`
  - body: The mutating maintenance sweep creates one session-store cache before processing every task, then yields between batches. If session state changes after a yield, later `shouldMarkLost` checks can use a stale snapshot and mark active tasks as lost.
