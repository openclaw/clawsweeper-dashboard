  - body: `buildExecApprovalPendingText` renders only `descriptors[0]` in the text body.   - body: `runTaskRegistryMaintenance()` creates one session-store cache for the whole mutating sweep while the loop yields between batches. If a session appears or updates after a yield, later `shouldMarkLost` checks still use the stale snapshot and can persistently mark an active task as `lost`.
  - confidence: 0.89
- **[P2] Make hidden exec stripping opt-in for generic Swift rendering:** `apps/shared/OpenClawKit/Sources/OpenClawChatUI/ChatMarkdownPreprocessor.swift:49`
  - body: The preprocessor defaults `stripHiddenSystemEvents` to true, and the generic `ChatMarkdownRenderer` still calls it without role context. That means user-authored log quotes starting with `System ... Exec ...` can disappear during normal chat display even after decode preserved the message.
  - confidence: 0.9
- **[P2] Stop warning for healthy symlinked installs:** `src/infra/update-check.ts:397-398`
  - body: `detectInstallState` marks every symlinked active root as suspicious and status-all turns that into an install-integrity warning plus reinstall/relink guidance. Healthy linked package roots are common, so this creates a persistent false operator warning unless there is an actual mismatch or unsafe resolved path.
