Full review comments:

- **[P1] Show every allowed approval action in pending text:** `src/infra/exec-approval-reply.ts:260-282`
  - body: `buildExecApprovalPendingText` renders only `descriptors[0]`, so text-only approval flows see one allow command and lose explicit deny or alternate allowed commands. Current main keeps secondary commands visible; this PR should list every allowed action in the plain text payload, not only interactive buttons.
  - confidence: 0.91
- **[P2] Do not strip hidden exec events without role context:** `apps/shared/OpenClawKit/Sources/OpenClawChatUI/ChatMarkdownPreprocessor.swift:49-55`
  - body: The preprocessor now strips hidden System/Exec blocks by default, but `ChatMarkdownRenderer` still calls it without passing whether the message is user-authored. A user who pastes those lines as a log quote can have real content removed at display time; make stripping opt-in for trusted system/assistant noise or pass role/source context through the renderer.
  - confidence: 0.86
- **[P1] Refresh session snapshots across maintenance yields:** `src/tasks/task-registry.maintenance.ts:381-392`
  - body: `runTaskRegistryMaintenance` creates one cached session-store context for the whole mutating sweep and then yields between batches. Session entries created or updated after a yield remain invisible to later `shouldMarkLost` calls, so active tasks can be incorrectly marked lost; refresh the cache after yields or limit pass-wide caching to read-only inspection.
  - confidence: 0.78
- **[P2] Do not warn on healthy symlinked installs:** `src/infra/update-check.ts:397-412`
  - body: `detectInstallState` marks every symlinked active root as suspicious and emits a reinstall/relink recovery hint. Symlinked package roots are common for linked installs and version-manager layouts, so this creates false install-drift warnings unless there is an actual resolved-root mismatch or unsafe path.
  - confidence: 0.82
