Overall confidence: 0.82

Full review comments:

- **[P1] Show every allowed approval action in text prompts:** `src/infra/exec-approval-reply.ts:260`
  - body: `buildExecApprovalPendingText` builds descriptors for all allowed decisions but renders only `descriptors[0]`. In text-only clients or copied logs, operators no longer see the deny or alternate approval commands, which weakens the approval safety path and can steer them toward an unintended allow.
  - confidence: 0.9
- **[P2] Preserve user-quoted exec lines in Swift rendering:** `apps/shared/OpenClawKit/Sources/OpenClawChatUI/ChatMarkdownPreprocessor.swift:49`
  - body: The preprocessor defaults hidden exec/system stripping on, and the Swift renderer calls it without passing user context. A user message that intentionally quotes `System ... Exec completed` lines can therefore be stripped at render time even though the decode path preserves it.
  - confidence: 0.88
- **[P1] Refresh session-store cache across maintenance yields:** `src/tasks/task-registry.maintenance.ts:381`
  - body: `runTaskRegistryMaintenance` creates one cached session-store context for the whole mutating sweep, then yields between batches. Session entries created after a yield remain invisible to later `shouldMarkLost` checks, so active tasks can be incorrectly marked lost from a stale snapshot.
  - confidence: 0.84
- **[P2] Do not warn for healthy symlinked installs:** `src/infra/update-check.ts:397-399`
  - body: `detectInstallState` marks every symlinked package root as suspicious and attaches a reinstall/relink recovery hint. Healthy installs through version managers or linked package roots can then show persistent drift warnings even when the resolved root is valid.
  - confidence: 0.81
