  - body: `buildExecApprovalPendingText` builds descriptors for all allowed decisions, but only renders `primaryAction.command`.   - body: The mutating maintenance sweep creates one session-store cache before processing every task, then yields between batches. If session state changes after a yield, later `shouldMarkLost` checks can use a stale snapshot and mark active tasks as lost.
  - confidence: 0.78

  - body: The new `stripHiddenSystemEvents` default changes every existing `preprocess(markdown:)` caller. `ChatMarkdownRenderer` still uses the default, so user-pasted `System ... Exec ...` log quotes can disappear at render time even after the view-model preservation fix.
  - confidence: 0.9
- **[P2] Do not warn on healthy symlinked installs:** `src/infra/update-check.ts:397-411`
  - body: `detectInstallState` marks any symlinked active root suspicious and emits a reinstall/relink recovery hint. Symlinked package roots are common for linked installs or version-manager setups, so this can create persistent false drift warnings unless the resolved roots actually mismatch or land in an unsafe path.
  - confidence: 0.86

  - body: The stricter gateway listener fallback accepts `openclaw-gateway` but not a listener whose only available command field is `openclaw`. lsof can provide only the command name, so a real `openclaw gateway run` listener can be misclassified as an unrelated busy port.
  - confidence: 0.72
