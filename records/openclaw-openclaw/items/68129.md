  - body: `buildExecApprovalPendingText` renders only the first descriptor in the text body.   - body: `runTaskRegistryMaintenance()` creates one session-store cache for the whole mutating sweep while the loop yields between batches. If a session appears or updates after a yield, later `shouldMarkLost` checks still use the stale snapshot and can persistently mark an active task as `lost`.
  - confidence: 0.89
- **[P2] Make hidden exec stripping opt-in for generic rendering:** `apps/shared/OpenClawKit/Sources/OpenClawChatUI/ChatMarkdownPreprocessor.swift:49`
  - body: The Swift preprocessor defaults `stripHiddenSystemEvents` to true, and generic markdown rendering has no role/source context. User-authored log quotes beginning with `System ... Exec ...` can disappear during normal chat display even when decode preserved the message.
  - confidence: 0.9
- **[P2] Do not flag every symlinked install as suspicious:** `src/infra/update-check.ts:397-398`
  - body: `detectInstallState` treats a symlinked active root as suspicious by itself and status diagnostics can turn that into reinstall/relink guidance. Healthy linked package roots are common, so this should warn only for an actual mismatch or unsafe resolved path.
  - confidence: 0.82
