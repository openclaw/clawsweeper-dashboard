## What This Changes

Request a provider-owned callback-settled or cancellable manual-prompt signal so remote OpenAI Codex OAuth can dismiss paste input after SSH-tunneled browser callback completion.

## Best Possible Solution

Add a provider-owned callback-settled or cancellable manual-input contract that keeps remote paste input immediate, avoids token-exchange inference, and covers paste-only plus callback-winning paths with focused tests or real-flow proof.

## Solution Assessment

Unclear until maintainers choose the contract shape. The requested direction is reasonable if the event remains provider-owned and any shared prompt-cancellation seam is added intentionally, rather than inferred from token-exchange timing or delays.
