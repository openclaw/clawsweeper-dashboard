## What This Changes

The issue requests an explicit callback-settled signal so remote OpenAI Codex OAuth can dismiss the manual paste prompt when a browser callback completes sign-in.

## Best Possible Solution

Keep this issue open until maintainers either land a provider-owned callback-settled or cancellation contract with paste-only and SSH-forwarded coverage, or explicitly reject that product/API direction.
