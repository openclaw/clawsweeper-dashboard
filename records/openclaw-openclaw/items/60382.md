## Summary

Keep open. Current main has incremental TUI polish, light/dark terminal readability support, waiting/status animation, and skippable channel setup, but it does not implement the broad visual and interaction redesign requested here. The related open theme PR is a useful partial overlap, not a canonical replacement for the remaining product/design decision.

The issue asks for a redesigned TUI experience with better visual design, smoother behavior, less channel-setup friction, and a Claude Code-like feel with animations.
