## What This Changes

`openclaw onboard` should install the bundled `model-usage` and `summarize` skill dependencies on macOS without failing on the CodexBar Homebrew cask/formula mismatch or the removed `summarize` formula.

## Best Possible Solution

Review and land PR #74408, or an equivalent narrow fix that preserves brew cask metadata, switches `summarize` to a maintained installer source, adds focused regression coverage, and records the user-facing fix in the changelog.

## Reproduction Assessment

Yes. On macOS with Homebrew and missing `codexbar` or `summarize`, selecting these bundled skills in `openclaw onboard` reaches the current formula-only brew command path shown in the reporter logs.

## Solution Assessment

Yes for the linked fix direction. Adding first-class cask handling and updating `summarize` to a maintained package source is narrower than changing onboarding broadly, but the issue should stay open until the fix is merged.
