- **[P2] Merge account bot-loop overrides with base defaults:** `extensions/telegram/src/bot-handlers.runtime.ts:1295-1299`

  - confidence: 0.86

In an `allowBots: true` group that still requires mentions, unmentioned bot chatter can consume the loop budget before a properly mentioned bot message arrives, so move guard recording after the normal group/mention acceptance decision.
