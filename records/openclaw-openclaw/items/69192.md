## What This Changes

The issue requests secret-safe warn-level gateway journal entries and stable internal reason codes for hook requests that return non-2xx validation errors.

## Best Possible Solution

Add centralized, secret-safe hook rejection diagnostics in the hook request handler with stable branch-specific reason codes and focused regression coverage for body-read, direct-hook, and mapped-hook rejects.
