## Best Possible Solution

Keep the local Vertex client cleanup, but satisfy plugin dependency ownership contracts, update the lockfile, rebase cleanly, and rerun exact-head checks before automerge.

Yes. The exact-head GitHub check annotations reproduce both blockers: the Anthropic Vertex manifest misses `@anthropic-ai/sdk`, and root `google-auth-library` violates the Google Chat plugin-local dependency contract.
