Request a way to register or auto-index local `SKILL.md` skill directories so they appear in `skill_search` results.

## Best Possible Solution

Define an additive source-aware local skill discovery contract that reuses the existing local loaders, respects per-agent visibility and trust boundaries, and keeps ClawHub registry discovery distinct.
