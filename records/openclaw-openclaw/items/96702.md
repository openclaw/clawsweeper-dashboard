## Summary

## What This Changes

The PR adds an argv-aware guard in `registerNodesCli` so built-in `nodes` subcommands skip plugin CLI/runtime registration while plugin-provided node subcommands still register lazily, with focused tests for both paths.

## Best Possible Solution
