Overall correctness: patch is incorrect

Full review comments:

- **[P2] Include call_id in transcript tool ID projections:** `src/config/sessions/transcript-store.sqlite.ts:176-177`
  - body: The new projection helpers promise OpenAI-style tool-call IDs, but they only read `id`, `toolCallId`, `tool_call_id`, `toolUseId`, and `tool_use_id`. Please include `call_id`/`callId` in both call and result extraction and add coverage for that shape.
