## What This Changes

The PR adds a fatal structured error flag to cron payload resolution, short-circuits isolated announce completion delivery before `dispatchCronDelivery`, and adds a regression test.

Land the narrow suppression of normal completion announce for fatal structured cron errors while preserving run diagnostics and separate failure notifications, after real cron/channel proof confirms the channel no longer receives raw provider JSON.
