## Summary

## What This Changes

The issue asks OpenClaw to stop silently migrating existing default models to paid Gemini preview variants and add explicit consent, warning, and cost-control policy.

## Best Possible Solution

A maintainer should choose a durable provider/default migration policy for paid preview models, then implement it with migration or warning behavior that preserves explicit user choices and updates docs/tests.
