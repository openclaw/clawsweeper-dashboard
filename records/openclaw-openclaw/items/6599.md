## What This Changes

The issue requests a `/models test-fallback` command that proactively exercises configured model fallbacks without waiting for a real provider failure.

## Best Possible Solution

Ship one shared, read-only fallback-chain probe that reuses real model/auth resolution, skips the primary, supports first-fallback and all-fallback modes, reports the answering model, and documents how it differs from auth-profile probes and broad per-model health checks.

## Solution Assessment
