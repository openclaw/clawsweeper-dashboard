Full review comments:

- **[P2] Stop the PowerShell smoke on the first failure:** `.github/workflows/ci.yml:1581-1582`
  - body: This `pwsh` step runs two native commands separately. GitHub Actions runs `pwsh` steps as a script and appends an exit using the last `$LASTEXITCODE`; the local probe showed that a failed first native command followed by a successful one exits 0, so a failing `pnpm check:docs` can be hidden by a successful build smoke. Chain with `&&` or explicitly exit after each command's `$LASTEXITCODE`. ([docs.github.com](https://docs.github.com/en/actions/reference/workflows-and-actions/workflow-syntax))
