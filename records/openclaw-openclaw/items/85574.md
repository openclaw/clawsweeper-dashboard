- The new trace API intentionally returns full prompts, tool schemas, invocation parameters, and raw response chunks once the source-checkout, env flag, admin scope, browser UI, and direct-loopback gates pass; maintainers need to accept that as the intended debugging boundary.

- Because the trace methods require `operator.admin`, existing browser sessions or devices with narrower stored operator scopes may need re-pairing or scope upgrade before the Traces tab can appear.
