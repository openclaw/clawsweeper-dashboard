## What This Changes

This PR classifies `ERR_HTTP2_INVALID_SESSION` as transient/benign network teardown, forces rebuilt global undici dispatchers to `allowH2: false`, and adds regression coverage for those paths.

## Best Possible Solution

Land this focused containment PR if maintainer merge gates remain green, and let the linked gateway crash issue at https://github.com/openclaw/openclaw/issues/81627 close after merge.
