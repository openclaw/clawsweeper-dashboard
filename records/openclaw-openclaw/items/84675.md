The issue already has an open closing implementation PR at https://github.com/openclaw/openclaw/pull/84677, so the right cleanup path is to review and land or reject that PR before closing this issue.

## What This Changes

The issue reports that the default seeded HEARTBEAT.md contains a docs-only Related link that prevents empty-heartbeat skipping and triggers recurring model calls.

## Best Possible Solution

Land one narrow fix that makes the current seeded HEARTBEAT.md non-actionable while preserving user-authored link/list tasks, with regression coverage for the actual shipped template.
