## Best Possible Solution

Keep the plugin-local non-empty schema repair, but make the flexible value schema include object items inside arrays and add regression coverage for user, attachment, and create-field option payloads.

## Solution Assessment

No. The Feishu-local schema repair is the right layer, but the proposed union is too narrow for existing Feishu bitable values; the best fix keeps the non-empty schema while preserving arrays of object values.
