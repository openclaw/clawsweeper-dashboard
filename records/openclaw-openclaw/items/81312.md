The PR propagates bundled capability manifest contracts into synthesized plugin records and adds regression coverage for the bundled capability runtime.

## Best Possible Solution

Land the narrow manifest-contract propagation fix after deduplicating contract-seeded provider id arrays, then let the merged PR close the linked bug and supersede the older overlapping PR at https://github.com/openclaw/openclaw/pull/78012.
