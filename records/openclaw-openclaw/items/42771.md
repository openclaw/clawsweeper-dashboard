The PR changes Docker to default Playwright browser downloads to `/opt/playwright`, precreate that directory, add a Chromium symlink, and update Docker/browser docs for that path.

## Best Possible Solution

I did not run a live Docker build in this read-only review.

No, not as-is. Current main already handles brittle Playwright cache discovery, while this broader path migration needs a refreshed implementation, upgrade decision, and real image proof before it is the best fix.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Keep existing cache users on a compatible path:** `Dockerfile:161`
  - body: This changes Docker's exported Playwright browser path from `/home/node/.cache/ms-playwright` to `/opt/playwright`. Users who already persist `/home/node` or mount the old cache path can lose the browser cache on upgrade unless the PR preserves compatibility or adds an explicit, tested migration that maintainers accept.
  - confidence: 0.88
- **[P1] Make `/opt/playwright` writable by `node`:** `Dockerfile:164-165`
  - body: This setup creates `/opt/playwright` as `root:root` and only changes mode to `775`. In the default browser-disabled image there is no later recursive `chown`, so runtime installs run as `node` and can fail with `EACCES` when they write into `/opt/playwright`.
  - confidence: 0.93
- **[P1] Point the Chromium symlink at the Linux x64 layout:** `Dockerfile:176`
  - body: Playwright v1.60 maps Chromium on Linux x64 to `chrome-linux64/chrome`, but this symlink targets `chrome-linux/chrome`. That can leave `/usr/local/bin/chromium` dangling on the common x64 Docker image even after Chromium installs successfully.
  - confidence: 0.91
- **[P2] Stop deleting the mounted apt lists cache:** `Dockerfile:174-175`
  - body: `/var/lib/apt` is mounted as a named BuildKit cache here, so removing `/var/lib/apt/lists/*` clears the reusable cache for later builds while providing no image-layer size reduction. Drop that cleanup from the cache-mounted RUN step.
  - confidence: 0.86
