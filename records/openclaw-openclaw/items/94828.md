- **[P2] Keep `--json` output free of the confirmation prompt:** `src/cli/skills-cli.ts:449-451`
  - body: When `--json` is passed without `--yes`, this branch still calls `promptYesNo()`, whose readline prompt writes to `process.stdout` before `writeJson()` runs. - **[P2] Treat origin-only ClawHub installs as removable:** `src/skills/lifecycle/clawhub.ts:1579`

Current status code treats that as invalid ClawHub provenance, so uninstall should remove the directory safely while leaving manual skills protected, with a regression test for the missing-lock case.
  - confidence: 0.86
