## What This Changes

The issue asks OpenClaw to automatically clean stale `.reset.*`, `.deleted.*`, `.tmp`, and `.bak.*` session artifacts to prevent disk bloat.

## Best Possible Solution

Review, land, or supersede https://github.com/openclaw/openclaw/pull/47635 with a narrow proactive cleanup that reuses existing retention contracts and safely handles orphaned `.tmp` and excess `.bak.*` artifacts.

## Reproduction Assessment

## Solution Assessment
