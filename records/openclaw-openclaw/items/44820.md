## What This Changes

The issue asks OpenClaw to automatically clean stale `.reset.*`, `.deleted.*`, `.tmp`, and `.bak.*` files from agent session directories to prevent disk bloat.

## Best Possible Solution

Land or replace the linked cleanup work with a focused implementation that respects session maintenance mode, covers configured session-store roots, and safely handles stale `.deleted.*`, `.reset.*`, orphaned `.tmp`, and excess `.bak.*` artifacts.
