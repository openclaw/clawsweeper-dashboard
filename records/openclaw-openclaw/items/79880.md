- **[P1] Keep invalid trusted-proxy exposure visible:** `src/commands/doctor-security.ts:241`
  - body: This treats any raw `gateway.auth.mode === "trusted-proxy"` as enough to skip the critical unauthenticated exposure warning. A lane with the override preset will not establish the default warning behavior and will leave later tests with a different environment, so snapshot and restore this key like the other state-integrity env vars.
