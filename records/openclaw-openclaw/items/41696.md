The PR changes legacy Discord resolver and test files so bare numeric allowlist entries try guild lookup before falling back to channel lookup.

## Best Possible Solution

## Solution Assessment

No. The intended behavior is the right narrow fix, but this PR applies it to inactive legacy files instead of the active Discord plugin resolver.
