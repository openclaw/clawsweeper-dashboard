- **[P1] Include terminal not-applicable rows in startup reconciliation:** `src/tasks/codex-native-subagent-reconciler.ts:231-234`
  - body: The startup sidecar is supposed to recover completions after a gateway restart, but this predicate only reconciles running/queued rows or rows already marked pending.
