Full review comments:

- **[P1] Keep task mutation helpers out of the public SDK:** `src/plugin-sdk/task-runtime.ts:26-31`
  - body: The new `openclaw/plugin-sdk/task-runtime` subpath exports `listTaskRecords` and `setDetachedTaskDeliveryStatusByRunId` through package exports. That gives plugin code a way to enumerate task rows and mark arbitrary runs delivered or pending across session boundaries; keep these helpers private/reserved or add a scoped, approved API before publishing the subpath.
  - confidence: 0.92
- **[P1] Reconcile terminal rows that still need delivery:** `src/tasks/codex-native-subagent-reconciler.ts:231-234`
  - body: The gateway-side reconciler only processes running/queued rows or rows already marked `pending`. If the mirror finalized a native Codex child as `succeeded` while delivery remained `not_applicable`, an app-server close or gateway restart leaves this sidecar skipping the row, so the parent reply can still never be delivered. Mirror the monitor's recent-terminal handling and cover that restart case.
  - confidence: 0.93
