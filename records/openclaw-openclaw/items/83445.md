- **[P1] Reconcile terminal native rows that still need delivery:** `src/plugin-sdk/codex-native-task-runtime.ts:310-313`
  - body: The task mirror creates Codex-native rows with `deliveryStatus: "not_applicable"` and finalizes them as `succeeded` when Codex reports `idle`.   - confidence: 0.9

During a slow parent handoff the sidecar can start a second delivery, and active requester queue delivery has no shared idempotency key, so the same completion can be queued twice.
  - confidence: 0.83
