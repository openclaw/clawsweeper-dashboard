## Summary

Keep open: the reported resource-loader performance problem still exists on current main and the latest release, but this branch is not merge-ready because it targets retired `pi-embedded-runner` files, bypasses the current loader reload contract through private members, is dirty, and has no real behavior proof.

## Best Possible Solution
