The PR adds a hosted Ollama Cloud guard that skips `/api/tags` discovery for `*.ollama.com` base URLs without explicit models, widens Ollama provider config input typing, adds discovery tests, and updates Knip dependency-ignore metadata.

## Best Possible Solution
