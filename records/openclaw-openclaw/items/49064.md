## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Parse custom range endpoints consistently:** `src/cli/usage-cli.ts:46-47`
  - body: `--from` parses a date-only string as UTC while `--to` appends a local end-of-day timestamp. In non-UTC environments custom ranges can include or exclude hours from the wrong local day, so parse both endpoints through the same local or gateway date helper.
  - confidence: 0.94
- **[P2] Aggregate sessions before applying display caps:** `src/cli/usage-cli.ts:321-323`
  - body: The sessions/by-model path hard-caps the most recently modified sessions before loading summaries. Older in-range expensive sessions can be omitted from top-session and per-model totals, so load/filter the full in-range data first and apply `--limit` only to rendered rows.
  - confidence: 0.91
- **[P3] Reuse the shared token formatter:** `src/cli/usage-cli.ts:102-109`
  - body: This command duplicates `formatTokenCount` and emits uppercase `K`/`M`, while the shared usage formatter emits the existing lowercase style and handles rounding consistently. Import the shared helper so usage output matches the rest of OpenClaw.
  - confidence: 0.84
