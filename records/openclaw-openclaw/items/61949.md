Ensure model alias prompt guidance does not cause agents to substitute providers when an explicit provider/model ref was requested.

## Best Possible Solution

Change model-alias prompt rendering so explicit provider/model refs are passed through unchanged and aliases are presented only as convenience names for unqualified model requests, with regression coverage for overlapping providers.
