## Best Possible Solution

Keep the Slack parity feature, but require resolved Slack bot-user-ID certainty or an equivalent Slack-ID-specific explicit mention signal before applying the other-mention drop, align the docs with that contract, add the degraded-identity regression test, and require redacted live Slack proof before merge.
