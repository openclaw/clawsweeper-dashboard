## Summary

Keep open: current main still lacks this hook and the linked DingTalk channel work shows a real plugin use case, but this PR is not ready to merge because the new raw-reply hook bypasses current conversation-access gating and the queued hook wait can still let the final announce overtake intermediate turns.

## What This Changes

The PR adds an `agent_to_agent_turn` plugin hook and emits A2A ping-pong turn events from `runSessionsSendA2AFlow` with flow, session, reply, and delivery-target metadata.

## Best Possible Solution

Ship a current-main plugin hook contract with conversation-access gating, tests for turn 0, ordering, routing metadata, and timeout behavior, plus plugin-hook docs and a changelog entry.

## Reproduction Assessment

Not applicable as a feature PR. Source inspection confirms current main has no `agent_to_agent_turn` hook and intermediate A2A turns stay inside `runSessionsSendA2AFlow` until the final announce path.

## Solution Assessment

No. A generic A2A turn hook is a plausible direction, but this branch needs to be rebased onto the current hook-type split, privacy-gated as conversation content, and given a chronology contract before it is the maintainable API.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P1] Gate A2A replies behind conversation access:** `src/plugins/types.ts:1869`
  - body: The event added here carries the raw A2A `reply`. On current main, raw conversation hooks are gated through `CONVERSATION_HOOK_NAMES` and `allowConversationAccess`; this hook is not added to that gated set, so a non-bundled plugin could receive conversation content without the opt-in required for `llm_output` and `agent_end`.
  - confidence: 0.9
- **[P2] Wait for the full hook queue before announcing:** `src/agents/tools/sessions-send-tool.a2a.ts:167`
  - body: This gives the whole serialized hook queue only one per-turn timeout budget. If several turn handlers each complete under the per-turn limit but take longer than that in total, the final announce can be delivered before later intermediate turns, breaking the transcript chronology this hook is meant to provide.
  - confidence: 0.86
