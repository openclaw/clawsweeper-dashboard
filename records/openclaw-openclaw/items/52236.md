  - body: Once this block runs for non-DM Mattermost turns, it still writes through the last-route key derived from the base route instead of the active thread `sessionKey`.   - confidence: 0.88

  - body: Broadening this write updates shared route context for channel/thread traffic, not only the thread-scoped session that needs cron recovery.   - confidence: 0.82
