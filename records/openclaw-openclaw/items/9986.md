## What This Changes

Request automatic model fallback to a configured larger-context model when the active model exhausts its context window.

## Best Possible Solution

Review and either land or close https://github.com/openclaw/openclaw/pull/82754; if approved, fallback should happen only after overflow recovery is exhausted and only to known larger-context candidates with focused regression coverage.

## Reproduction Assessment

## Solution Assessment

Yes, pending maintainer policy approval. The linked PR direction matches the narrow safe shape because it waits until overflow recovery is exhausted and only advances to known larger-context candidates.
