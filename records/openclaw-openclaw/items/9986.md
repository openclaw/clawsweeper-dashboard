## What This Changes

Request fallback to a configured larger-context model when the active model exceeds its context window after overflow recovery is exhausted.

## Best Possible Solution

Review and either land or close https://github.com/openclaw/openclaw/pull/82754; if approved, the final behavior should fallback only after overflow recovery is exhausted, select known larger-context candidates, preserve replay safety, log the event, and keep regression coverage.
