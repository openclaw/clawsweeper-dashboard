## What This Changes

Replace or fall back from hard-link-based session usage-cost cache locking so unsupported filesystems do not crash usage refresh.

## Best Possible Solution

Review and land the narrow usage-cost lock fallback with focused regression coverage and accepted real-environment or maintainer-run proof, then close this issue from that merge.

## Solution Assessment
