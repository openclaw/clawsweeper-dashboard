  - body: Current runtime tests document that plugin modelAuth wrappers strip `agentDir` and credential stores to prevent plugins from steering credentials.   - confidence: 0.9

  - body: The title generator takes `modelProvider` before `providerOverride` but only passes `modelOverride`, ignoring the runtime `model` field stored beside `modelProvider`. Existing sessions with `modelProvider`/`model` and no override can therefore resolve a provider from the session with the agent default model, causing the background title call to use the wrong model or auth path.
