Full review comments:

- **[P1] Keep plugin auth wrappers from forwarding credential selectors:** `src/plugins/runtime/index.ts:144-148`
  - body: The plugin runtime wrapper currently strips `agentDir`/store-style credential selectors so plugin callers cannot steer model-auth lookup outside workspace scope. This PR forwards `agentDir`, `profileId`, `preferredProfile`, and `lockedProfile` into `getRuntimeAuthForModel`; the session-title path can pass session-owned auth fields through the internal helper directly without exposing them on `runtime.modelAuth`.
  - confidence: 0.92
- **[P2] Cap transcript excerpts before title-model calls:** `src/auto-reply/reply/session-title-generator.ts:136-140`
  - body: `combinedMessages` is built from complete early user records and sent straight to the title model, while `maxChars` only caps the returned title. With `sessionTitle.enabled`, a huge first prompt or pasted file can be sent to a background summarization call, so bound the per-message or total excerpt before calling `generateConversationLabel`.
  - confidence: 0.86
