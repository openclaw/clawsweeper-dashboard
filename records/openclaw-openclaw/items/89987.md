## Best Possible Solution

## Solution Assessment

Yes, if maintainers accept the behavior change: cron delivery dispatch has both the resolved channel and fallback-summary context, while suppressing in the Matrix adapter would lose cron semantics and suppressing earlier would affect non-Matrix delivery.
