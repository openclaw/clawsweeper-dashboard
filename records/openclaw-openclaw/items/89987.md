## What This Changes

The PR adds Matrix-only suppression for leading `NO_SUBPROCESS_STARTED` cron diagnostics before outbound delivery, plus token and cron-dispatch regression tests.

## Best Possible Solution

Keep the Matrix-only suppression, but make the diagnostic contract exact or colon-delimited and require redacted live Matrix delivery proof before merge.

## Reproduction Assessment

## Solution Assessment
