## Summary

## Best Possible Solution

Implement one coordinated Discord active-target policy in the bundled Discord plugin, using bounded opening attention text plus native Discord mention/reference metadata and configured bot/account bindings, while leaving role-trigger and slash-command semantics aligned with their canonical trackers.
