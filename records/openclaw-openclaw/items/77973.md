## Best Possible Solution

Land the focused hard cap after gateway owners accept the 5,000-entry terminal-snapshot retention budget, and keep the linked issue open until the PR merges.

## Reproduction Assessment

## Solution Assessment

Yes. A local hard cap in the existing terminal snapshot cache is the narrowest maintainable fix; a new config knob or persistent store would add product and upgrade surface that this bug does not need.
