## What This Changes

Request a default-off approval-bound system.run setting that permits explicitly trusted symlink working directories while preserving canonical target display and inode revalidation.

## Best Possible Solution

Maintainers should either approve a narrow documented trust-knob contract that shows requested and canonical cwd while preserving inode revalidation, or explicitly keep and document the canonical-cwd-only policy.
