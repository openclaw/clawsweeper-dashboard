## Best Possible Solution

Preserve the existing supplement params while adding optional `engineCandidates`, add a regression assertion for `corpus`, then have maintainers approve the SDK shape before merge.

No as patched. The narrow API addition is plausible, but the implementation should forward `{ ...params, engineCandidates: params.engineCandidates }` or otherwise include `corpus` and cover that in tests.
