The issue reports that adding a new account in the channel setup wizard can reuse or overwrite existing account credentials for multi-account plugins such as WeCom.

## Best Possible Solution

Review and land or replace the open fix PR so shared setup promotion uses the contract-backed account-promotion path before selected-account writes, with WeCom-style regression coverage.
