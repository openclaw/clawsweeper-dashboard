Keep this PR open: it targets a plausible infra error-handling gap, but current main has already superseded much of the archive, queue, and device-identity work while the Tailscale JSON parse diagnostic remains a useful smaller fix. The PR wraps several JSON.parse calls with contextual errors and adds an atomic write helper for device identity key material.
