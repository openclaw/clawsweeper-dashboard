The issue reports a Linux npm-global regression where `openclaw status` hangs and exits 124 after upgrading from 2026.5.7 to 2026.5.18 or later.

## Best Possible Solution

Bound Linux service/process probes by the status command timeout and fail soft with degraded daemon rows, preserving the fast read-only status path.

## Solution Assessment
