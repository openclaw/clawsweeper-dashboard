- **[P2] Preserve deltas when `.done` omits arguments:** `src/agents/openai-transport-stream.ts:492-493`
  - body: This handler overwrites `partialJson` with `stringifyJsonLike(event.arguments)`. When a stream sends deltas and then a `.done` event without `arguments` or with an empty final value, this turns the accumulated JSON into `""`, so `toolcall_end` can lose the args that were already parsed. Mirror the current shared Responses parser: only replace the buffer when `.done.arguments` is a non-empty string, or when there was no prior partial JSON.
  - confidence: 0.82
