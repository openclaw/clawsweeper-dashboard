Full review comments:

- **[P1] Validate advisor model pairs before injection:** `extensions/anthropic/stream-wrappers.ts:240-242`
  - body: `advisor: true` defaults the advisor to `claude-sonnet-4-6`, and custom values are only checked for a Claude-looking id. Anthropic requires the advisor/executor to be a valid pair with the advisor at least as capable as the executor, so an Opus executor can be sent with a Sonnet advisor and fail the request with `400 invalid_request_error`. Validate the pair against the known Anthropic model catalog or reject unsupported combinations before adding the tool.
  - confidence: 0.89
- **[P1] Preserve server-tool blocks for Anthropic replay:** `src/agents/anthropic-transport-stream.ts:812-840`
  - body: This branch drops `server_tool_use` markers and stores `advisor_tool_result` as display-only text instead of preserving the raw assistant content. Anthropic's advisor contract requires result content, including encrypted content, to be round-tripped verbatim on later turns and uses `pause_turn` by resending the unchanged assistant message with the same advisor tool, so this can break multi-turn advisor sessions or resumptions. Store and replay the raw server-side blocks, with pruning support, rather than converting them only to local display text.
  - confidence: 0.92
