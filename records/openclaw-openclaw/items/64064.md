## Summary

Keep open. The feature is still useful and not implemented on current main, but this PR is not merge-ready because the advisor model default/config can produce Anthropic 400s and the stream handling drops raw server-side result blocks that Anthropic's beta contract exposes for replay.

## What This Changes

The PR adds Anthropic advisor beta configuration/header/tool injection, server-side tool result display handling, stop-reason hardening, docs, validation scripts, and focused tests.

## Best Possible Solution

Land a maintainer-reviewed advisor implementation that derives or requires valid executor/advisor pairs, preserves raw server-side tool result blocks through replay and pruning, and keeps the docs aligned with Anthropic's beta contract.

## Reproduction Assessment

Yes, source-reproducible. Current main lacks advisor config/tool injection and server-side result handling, and the PR-head blockers are visible in the advisor resolver, replay conversion, and Anthropic's documented beta contract.

## Solution Assessment

No. The direction is useful, but the implementation should use a pair-aware advisor model contract and preserve raw server-side blocks instead of replacing them with display-only text history.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Choose a compatible advisor default:** `extensions/anthropic/stream-wrappers.ts:225`
  - body: `advisor: true` resolves to `claude-sonnet-4-6`, and the new docs show that shorthand on a `claude-sonnet-4-6` executor. Anthropic documents that executor as pairing with `claude-opus-4-7` and returns a 400 for invalid pairs, so the documented common setup can fail before the stream handling runs.
  - confidence: 0.93
- **[P2] Preserve raw server-side tool blocks:** `src/agents/anthropic-transport-stream.ts:833-837`
  - body: The stream handler stores `advisor_tool_result` and encrypted/redacted server-tool results as synthetic display text and later skips or downgrades that data on Anthropic replay. The SDK exposes these as assistant content params, with encrypted advisor content required to be round-tripped verbatim, so replacing them with text-only history breaks multi-turn advisor/server-tool context.
  - confidence: 0.91
