## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P1] Validate advisor model pairs before injecting:** `extensions/anthropic/stream-wrappers.ts:225-230`
  - body: `advisor: true` currently resolves to `claude-sonnet-4-6`, and the custom-model guard accepts any `claude-*` string. Anthropic requires the executor and advisor models to be a supported pair and returns a 400 for invalid combinations, so the documented shorthand can fail the first enabled request and the custom path can silently generate bad payloads. Use a supported default for the selected executor and reject or warn on unsupported pairs.
  - confidence: 0.88
- **[P1] Preserve server-tool blocks for replay:** `src/agents/anthropic-transport-stream.ts:827-848`
  - body: This path converts readable `*_tool_result` blocks into display-only text and drops `server_tool_use` plus untranslatable result blocks. Anthropic's advisor and web-search server tools expect those native blocks, including encrypted content/citation state, to be passed back on later turns; dropping them can lose multi-turn provider context or break follow-up behavior. Store and prune the native blocks instead of replacing them with local display text only.
  - confidence: 0.84
