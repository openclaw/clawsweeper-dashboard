## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.92

Full review comments:

- **[P2] Preserve the thinking prefill wrapper:** `extensions/anthropic/stream-wrappers.ts:319`
  - body: Current main always composes `createAnthropicThinkingPrefillWrapper` so extended-thinking Anthropic requests do not end with an assistant prefill. This branch's wrapper chain ends at the new advisor wrapper and omits that guard, so rebasing or merging it would reintroduce rejected thinking-prefill payloads.
  - confidence: 0.94
- **[P2] Use a compatible advisor default:** `extensions/anthropic/stream-wrappers.ts:225-241`
  - body: `advisor: true` resolves to `claude-sonnet-4-6`, and the docs show that shorthand on a `claude-sonnet-4-6` executor. Anthropic requires the advisor to be at least as capable as the executor, so the documented common setup can fail with a 400 before streaming starts.
  - confidence: 0.93
- **[P2] Round-trip raw server-side tool blocks:** `src/agents/anthropic-transport-stream.ts:833-837`
  - body: The stream handler stores advisor/server-tool results as synthetic display text and later skips or downgrades them during Anthropic replay. The advisor contract expects full assistant content, including `advisor_tool_result` and encrypted content, to be passed back on later turns, so multi-turn advisor state is lost.
  - confidence: 0.92
