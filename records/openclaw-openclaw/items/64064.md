  - confidence: 0.93

  - body: The stream handler stores advisor/server-tool results as synthetic display text and skips or downgrades them during Anthropic replay. Anthropic expects full assistant content, including `advisor_tool_result` and encrypted content, on later turns, so multi-turn advisor state is lost.   - confidence: 0.92
