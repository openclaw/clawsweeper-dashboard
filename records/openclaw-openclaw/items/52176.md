## Summary

Keep this PR open, but it is not merge-ready: the central Docker feature is still unique on current main, while the patch adds an unpinned root installer path and lacks setup/runtime wiring plus real Docker proof.

## What This Changes

The PR adds an optional `OPENCLAW_INSTALL_UV` Docker build argument that downloads Astral's installer and moves `uv` and `uvx` into `/usr/local/bin`.

## Best Possible Solution

If OpenClaw accepts first-class uv Docker support, land it as a pinned and verified uv install path with setup/docs/tests, node-visible tool-bin configuration, and real Docker runtime proof; otherwise keep custom-image guidance as the supported path.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.93

Full review comments:

- **[P1] Pin and verify the uv installer before executing it:** `Dockerfile:202`
  - body: This build path executes whatever `https://astral.sh/uv/install.sh` serves at build time as root, so rebuilding the same OpenClaw commit can trust different installer code. Use a pinned uv release artifact or versioned installer with digest, signature, checksum, or equivalent verification before adding this production image knob.
  - confidence: 0.95
- **[P2] Add a node-visible uv tool bin path:** `Dockerfile:203-205`
  - body: Moving `uv` and `uvx` into `/usr/local/bin` does not make later `uv tool install` executables visible to the non-root runtime. Configure a node-writable `UV_TOOL_BIN_DIR` and put it on `PATH`, otherwise uv-backed skills can still fail PATH-based `requires.bins` checks after installation.
  - confidence: 0.9
- **[P2] Forward the uv arg through Docker setup:** `Dockerfile:198`
  - body: The supported `scripts/docker/setup.sh` local-build path persists and forwards a fixed build-arg list, and this PR only adds the Dockerfile arg. Operators setting `OPENCLAW_INSTALL_UV=1` around setup would still build an image without uv unless setup, docs, and tests are updated or the feature is explicitly raw `docker build` only.
  - confidence: 0.88
