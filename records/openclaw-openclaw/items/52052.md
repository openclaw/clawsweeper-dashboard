## Best Possible Solution

Expose the replayable Telegram sticker id in untrusted per-turn context using the existing `fileId`/`stickerId` contract, or add and document a supported alias, while keeping stable system metadata unchanged and proving a real Telegram sticker replay.

## Solution Assessment
