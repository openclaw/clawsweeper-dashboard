  - body: The new preflight gate is group-only and omits the current senderAllowedForAudioPreflight guard, then starts transcription before command authorization is computed. That can skip DM voice-note preflight and send unauthorized group audio through transcription before access is known.
  - confidence: 0.9

  - body: The timeout callback clears timer before awaiting tick(), but isRunning() still checks only timer !== undefined. TTL cleanup can observe the keepalive loop as stopped during an in-flight tick and skip cleanup for that run.

  - body: The no-fence fast path returns chunkText(), whose shared splitter trims chunk boundaries. Long markdown without fences can rely on leading indentation or blank-line context, so this can change rendered lists or indented code.
  - confidence: 0.81
