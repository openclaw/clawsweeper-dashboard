- **[P1] Gate audio transcription before starting STT:** `src/telegram/bot-message-context.ts:363-369`
  - body: Current main resolves command authorization before audio preflight and gates STT with senderAllowedForAudioPreflight.   - confidence: 0.93
- **[P2] Keep typing loops running during in-flight ticks:** `src/channels/typing-lifecycle.ts:72`
  - body: scheduleNext clears timer before awaiting tick(), but isRunning() still only checks timer. createTypingController uses isRunning() for TTL cleanup, so a TTL callback during an awaited onTick can treat the loop as stopped and skip cleanup.
  - confidence: 0.87
- **[P2] Preserve markdown spacing in the no-fence path:** `src/auto-reply/chunk.ts:321-323`
  - body: The new fast path sends all markdown without fences through chunkText(), whose shared splitter trims chunk ends and starts. That can strip indentation or newline context from long markdown replies such as indented code blocks or nested lists.
  - confidence: 0.78
- **[P2] Preserve max thinking as its own level:** `src/auto-reply/thinking.ts:72`
  - body: Current main treats max as a distinct ThinkLevel and has regression coverage for normalizeThinkLevel("max"). This map downgrades max to high, silently changing user/profile thinking settings if rebased as-is.
  - confidence: 0.76
