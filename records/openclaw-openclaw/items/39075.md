- **[P2] Preserve audio preflight eligibility gates:** `src/telegram/bot-message-context.ts:363-369`
  - body: The new preflight gate is group-only and omits the current senderAllowedForAudioPreflight guard, then starts transcription before command authorization is computed.   - confidence: 0.9
- **[P2] Preserve running state during awaited typing ticks:** `src/channels/typing-lifecycle.ts:42-43`
  - body: The timeout callback clears timer before awaiting tick(), but isRunning() still checks only timer !== undefined. TTL cleanup can observe the keepalive loop as stopped during an in-flight tick and skip cleanup for that run, so track active loop state independently or count tickInFlight as running.
  - confidence: 0.84
- **[P2] Preserve markdown whitespace on no-fence chunks:** `src/auto-reply/chunk.ts:323`
  - body: The no-fence fast path returns chunkText(), whose shared splitter trims chunk boundaries. Long markdown without fences can rely on leading indentation or blank-line context, so this can change rendered lists or indented code compared with the markdown splitter.
  - confidence: 0.81
