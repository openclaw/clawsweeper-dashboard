- **[P2] Preserve audio preflight eligibility gates:** `src/telegram/bot-message-context.ts:363-369`
  - body: The new preflight gate requires `isGroup` and runs before `commandGate` is computed, so direct-message voice notes no longer call `transcribeFirstAudio` and group audio can be transcribed before sender command authorization is known.   - confidence: 0.9
- **[P2] Preserve running state during awaited typing ticks:** `src/channels/typing-lifecycle.ts:42-43`
  - body: The timeout callback clears `timer` before awaiting `tick()`, but `isRunning()` still checks only `timer !== undefined`. During an awaited `onTick`, callers can see the loop as stopped and skip TTL cleanup for that run.
  - confidence: 0.84

  - body: The no-fence fast path delegates to `chunkText()`, whose shared splitter trims chunk boundaries. Long markdown without fences can rely on indentation or leading newlines, so this can change rendered lists or indented code at split points.
  - confidence: 0.81
