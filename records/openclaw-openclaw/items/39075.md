  - body: The new preflight gate starts with `isGroup`, so direct voice notes without text never call `transcribeFirstAudio` and fall back to `<media:audio>`. Current Telegram coverage expects DM voice notes to be transcribed, so keep the DM path while parallelizing group preflight.

  - confidence: 0.84

  - body: The no-fence fast path delegates to `chunkText()`, whose shared splitter trims chunk boundaries. Long markdown without fenced blocks can rely on leading indentation or newlines, so this can change rendered lists or indented code.
  - confidence: 0.81
