  - body: The timeout callback clears timer before awaiting tick(), but isRunning() still only checks timer. Callers such as the typing TTL cleanup can observe the loop as stopped during an in-flight sendChatAction call and skip the safety cleanup for that run.
  - confidence: 0.84

  - confidence: 0.82
