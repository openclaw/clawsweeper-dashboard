  - body: Current main gates group audio preflight with commandAuthorized before calling STT, but this PR starts transcribeFirstAudio in the early Promise.all before that authorization result exists.   - confidence: 0.89
- **[P2] Count in-flight typing ticks as running:** `src/channels/typing-lifecycle.ts:42`
  - body: The timeout callback clears timer before awaiting tick(), while isRunning() still only checks timer. During every in-flight sendChatAction, refreshTypingTtl can see the loop as stopped and skip cleanup, so typing state can outlive the configured TTL.
  - confidence: 0.86
- **[P2] Preserve markdown spacing in the no-fence fast path:** `src/auto-reply/chunk.ts:322-324`
  - body: chunkText trims chunk boundaries, but markdown without fences can still depend on leading indentation or blank-line context for lists and indented code. Routing all no-fence markdown through chunkText can change rendered replies compared with the markdown splitter.
  - confidence: 0.78
