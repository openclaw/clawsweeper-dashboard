Full review comments:

- **[P2] Preserve audio preflight eligibility gates:** `src/telegram/bot-message-context.ts:363-369`
  - body: The new condition requires `isGroup` and runs before command authorization is resolved, so direct-message voice notes skip `transcribeFirstAudio` and group audio can be preflighted without the current sender-authorization gate.   - confidence: 0.9
- **[P2] Preserve running state during awaited typing ticks:** `src/channels/typing-lifecycle.ts:72`
  - body: The timeout callback clears `timer` before awaiting `tick()`, but `isRunning()` still only checks `timer !== undefined`. During an awaited `onTick`, callers can observe the loop as stopped and skip TTL cleanup for that run.
  - confidence: 0.84
- **[P2] Preserve markdown whitespace on no-fence chunks:** `src/auto-reply/chunk.ts:323`
  - body: The no-fence fast path delegates to `chunkText()`, whose shared splitter trims chunk boundaries. Long markdown without fenced blocks can rely on indentation or leading newlines, so this can change rendered lists or indented code at split boundaries.
  - confidence: 0.81
