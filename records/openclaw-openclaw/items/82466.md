## What This Changes

This PR adds a CLI formatter guard and regression test to suppress the hook-pack fallback suffix when plugin install already reports missing compiled runtime output.

## Best Possible Solution

Narrow the formatter suppression to compiled-runtime plugin errors paired with a missing-hook-manifest fallback, preserve other hook validation details, add the regression coverage, then require redacted real CLI output before merge.

## Reproduction Assessment

Yes, source-level. Current main routes the compiled-runtime plugin failure through the hook fallback formatter, and current tests show hook packages may also be plugin-capable, which exposes the PR's broad suppression risk.

## Solution Assessment

No. The formatter is the right layer, but this implementation should key off the specific missing-hook-manifest fallback instead of suppressing every hook error when the plugin error mentions missing compiled runtime output.
