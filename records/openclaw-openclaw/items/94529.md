## What This Changes

The issue requests readable auto-derived session titles in the Control UI sidebar and session picker instead of repeated generic New Session labels.

## Best Possible Solution

Merge a small Control UI change that requests includeDerivedTitles for sidebar and picker session lists, adds UI derivedTitle typing and display fallback, preserves label/displayName precedence, and keeps broader AI summarization/title-quality work separate.

## Reproduction Assessment

## Solution Assessment
