## What This Changes

The PR persists yielded top-level session lifecycle endings as paused with a `sessions_yield` pause reason, propagates that state to session surfaces, skips restart recovery for paused entries, and adds regression tests plus a changelog entry.

## Best Possible Solution

Land this PR after maintainer approval and exact-head CI so main sessions share the paused-yield lifecycle contract already used by subagents.
