## What This Changes

This PR routes ACP task cancellation through Gateway `tasks.cancel`, fails ACP CLI cancellation when Gateway is unavailable, dedupes late-derived ACP run task rows, and cancels same-run ACP rows together.

## Best Possible Solution

Land a version that keeps Gateway as the ACP cancellation owner, adds focused Gateway `tasks.cancel` ACP regression coverage, includes redacted live Gateway cancellation and Gateway-down output, and has maintainer acceptance of the fail-closed behavior.

## Reproduction Assessment
