- **[P2] Return JSON for missing unset dry-run paths:** `src/cli/config-cli.ts:2030`
  - body: With `--dry-run --json`, a missing path still exits through the pre-existing text error before the new dry-run result writer runs. JSON callers receive stderr text instead of a structured failure, so emit a JSON payload for this preflight branch and cover it with a test.
  - confidence: 0.87

  - confidence: 0.82
