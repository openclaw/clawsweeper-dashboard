## What This Changes

The PR modifies `src/plugins/tools.ts` so cached plugin descriptor execution searches plugin tool candidates by the actual returned runtime tool name instead of treating plugin-level declared names as per-factory identity.

## Best Possible Solution

Dispatch cached descriptors by actual runtime tool name while preserving cold-load error reporting, add focused regression coverage for multiple unnamed factories under one manifest contract, add the required changelog entry, and require contributor-supplied real behavior proof before merge.
