## What This Changes

Issue requests that isolated cron `agentTurn` announcements prefer the final assistant reply over an intermediate tool error after the agent self-corrects.

## Best Possible Solution

Update `resolveCronPayloadOutcome` so recovered final assistant text wins for opted-in text-only cron announce paths, with regression coverage that still preserves true run-level failures and explicit fatal signals.

## Solution Assessment

Yes on direction. The narrow maintainable fix belongs in `resolveCronPayloadOutcome` and its focused tests, while preserving run-level errors and structured fatal failure signals.
