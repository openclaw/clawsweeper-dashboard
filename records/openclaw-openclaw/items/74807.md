## What This Changes

Cron isolated agent-turn run summaries should treat recovered terminal assistant output as the final outcome instead of reporting an earlier tool-error payload as the task result.

## Best Possible Solution

Land or close a rebased PR 84156-style fix that separates fatal run signals from recovered tool diagnostics while keeping useful diagnostics on successful cron runs.
