## Summary

Keep open. ## What This Changes

The issue reports Dashboard Stop corrupting session JSONL during in-flight tools and missing guardrails that let one agent delete another agent's state.

## Best Possible Solution

Keep this open for maintainer-owned hardening: prove the Stop/session corruption path, land a recorded cancel/drain/recovery invariant, and separately decide the default host-exec isolation/approval policy.
