- **[P1] Route wiki_get through configured page groups:** `extensions/memory-wiki/src/query.ts:1563-1568`

  - confidence: 0.95
- **[P1] Recurse through configured page-group directories:** `extensions/memory-wiki/src/query.ts:249-256`
  - body: The query scanner still maps each configured dir to `fs.readdir` and keeps only immediate `entry.isFile()` children. The compile scanner has the same shape, while status now walks recursively, so a page under `资料/project/alpha.md` can be counted by status but omitted from search, compile, indexes, and digests.
  - confidence: 0.94
- **[P1] Move generated reports into the configured report group:** `extensions/memory-wiki/src/compile.ts:83`
  - body: All dashboard definitions still have `relativePath: "reports/..."`, and `writeDashboardPage` writes that path directly. If users configure `report` to `报告`, defaults for the report kind are skipped, but generated dashboards are still written under `reports/` and then fall outside the configured report group.
  - confidence: 0.93
- **[P1] Keep source writers and default groups from splitting pages:** `extensions/memory-wiki/src/config.ts:303-305`
  - body: `mergePageGroups` skips the built-in default for any kind that has a user or vault group, but several existing source-producing paths still write to hardcoded `sources/`. A user who configures only `source -> 资料` will get imported/source pages written outside the configured source group, so they will not be indexed unless every writer is moved to the configured source dir or the default remains part of the canonical groups.
  - confidence: 0.9
- **[P2] Mirror runtime dir validation in the manifest schema:** `extensions/memory-wiki/openclaw.plugin.json:215-218`
  - body: Runtime `WikiPageGroupSchema` rejects absolute paths and traversal, but the manifest schema accepts any non-empty `dir`. That lets setup/control-plane validation treat a config as valid even though runtime validation rejects or drops it, which is the wrong contract for a filesystem path config surface.
  - confidence: 0.86
