## Summary

Keep this PR open. Current main still routes `openclaw gateway start` through the platform restart action even when the managed service runtime is already running, while the PR narrowly adds an idempotent already-running result, CLI handling, regression tests, and a changelog entry for the linked crash report. ## What This Changes

Adds an `already-running` gateway service start outcome, skips service restart when the runtime state is already running, updates CLI output, adjusts lifecycle tests, and adds a changelog fix entry for #60087.

## Best Possible Solution
