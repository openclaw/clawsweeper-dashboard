## What This Changes

The PR adds an `already-running` gateway service start outcome, skips service restart when runtime state is already running, updates CLI JSON/text output, adjusts lifecycle tests, and adds a changelog fix entry for #60087.

## Best Possible Solution

Land the service-layer idempotency behavior with the CLI result branch, focused regression tests, and changelog entry after maintainer review and Testbox changed-gate validation, while keeping `openclaw gateway restart` as the explicit forced-cycle command.
