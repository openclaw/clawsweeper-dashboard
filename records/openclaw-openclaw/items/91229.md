  - body: `SettingsHost` is declared locally in `app-settings.ts` and is not exported, so this new import breaks production type checking. Either export a deliberate type or define the narrow SSO host shape locally.
  - confidence: 0.98
- **[P1] Complete the UiSettings test fixture:** `ui/src/ui/gateway-sso-postmessage.test.ts:20-28`

  - confidence: 0.97
- **[P1] Reconnect when tokens arrive during startup:** `ui/src/ui/gateway-sso-postmessage.ts:95-97`
  - body: The listener only reconnects when `host.connected` is already true. If the parent posts the SSO token after `connectGateway` creates the client but before hello flips `connected`, the ack is sent while the active client still uses the stale token captured at connect start.
  - confidence: 0.91
- **[P1] Require HTTPS for production SSO origins:** `ui/src/ui/gateway-sso-postmessage.ts:26-33`
  - body: The allowlist accepts both HTTP and HTTPS before checking for `clusterclaw.ai`, so cleartext `http://clusterclaw.ai` or subdomains can inject a gateway token. Keep HTTP limited to loopback development origins.
  - confidence: 0.9
