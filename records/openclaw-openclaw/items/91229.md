Overall confidence: 0.91

Full review comments:

- **[P2] Use an exported or local host type:** `ui/src/ui/gateway-sso-postmessage.ts:1`
  - body: `SettingsHost` is private to `app-settings.ts` on current main, but this new module imports it. CI already reports the resulting error type, so define the narrow SSO host shape locally or deliberately export the type.
  - confidence: 0.96
- **[P2] Reconnect when the token changes during startup:** `ui/src/ui/gateway-sso-postmessage.ts:90-92`
  - body: The listener only reconnects when `host.connected` is already true. If the parent sends the token after the gateway client is created but before hello completes, the ack is sent while the active client still uses the stale token captured at connect start.
  - confidence: 0.9
- **[P2] Require HTTPS for production SSO origins:** `ui/src/ui/gateway-sso-postmessage.ts:25-32`
  - body: The allowlist accepts both HTTP and HTTPS before checking for `clusterclaw.ai`, so cleartext production ClusterClaw origins can inject a gateway token. Keep HTTP limited to loopback development origins.
  - confidence: 0.86
