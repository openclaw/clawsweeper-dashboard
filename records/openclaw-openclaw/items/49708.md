The issue requests preserving logical session transcript paths under symlinked OpenClaw state directories so doctor does not report referenced transcripts as orphaned.

Land or replace the linked PR with a current-main patch that preserves realpath containment checks, writes caller-visible logical session paths, normalizes legacy realpath entries, and proves the symlinked upgrade path.
