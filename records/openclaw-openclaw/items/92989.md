## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

No: the Windows-side reuse direction is plausible, but this PR is not the best fix while it adds a cross-platform first-success legacy fallback. The safer path is the existing active-codepage contract plus focused real proof.
