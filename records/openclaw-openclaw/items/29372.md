## What This Changes

Add an `agents.list[].subagents.workspace` option that assigns a separate default workspace to child subagents spawned without an explicit agent profile.

## Best Possible Solution

Define one documented precedence and security contract for inherited workspace, target profiles, explicit `cwd`, sandbox access, and any stable child-workspace override, then implement it with focused config, spawn, bootstrap, nested-spawn, and sandbox coverage.
