  - confidence: 0.92

  - confidence: 0.9

  - body: The importer writes the old identity snapshot into SQLite and then removes the source file. If runtime already generated or rotated identity state, doctor can restore stale keys and invalidate newer trust relationships.
  - confidence: 0.9
- **[P1] Merge channel pairing imports with live state:** `src/commands/doctor/legacy/channel-pairing.ts:231-247`
  - body: The pairing importer replaces request and allowlist entries from legacy files, then removes those files. If SQLite already contains newer approvals or revocations, migration can re-add revoked senders or drop newly approved ones.
  - confidence: 0.86
