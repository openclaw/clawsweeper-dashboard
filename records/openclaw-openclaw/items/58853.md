  - confidence: 0.9
- **[P2] Hide WSL2 resource fields in WSL1 summaries:** `src/commands/doctor-wsl.ts:379-383`
  - body: The summary appends .wslconfig memory and processor values even when diag.isWSL2 is false. Because those resource caps do not apply to WSL1, WSL1 users can see misleading environment context unless these fields are gated behind WSL2.
  - confidence: 0.84
