## Summary

Keep open: the reported Telegram plugin-bound topic bug is valid and the PR includes real runtime proof, but the current patch broadens Telegram group activation when a bound plugin cannot claim the turn because mention gating is disabled before fallback.

## What This Changes

The PR adds a plugin-owned runtime binding signal for Telegram topic routes, bypasses mention gating for those routes, allows plugin claims under `message_tool_only` source reply suppression, and adds focused routing/dispatch tests.

## Best Possible Solution

Keep the `message_tool_only` plugin-claim fix, but preserve Telegram mention gating for plugin fallback by carrying the original activation decision through dispatch or by stopping ordinary fallback processing for unmentioned plugin-bound group turns.

## Reproduction Assessment

Yes, source-reproducible. Current main skips plugin-owned claims when `message_tool_only` sets `suppressDelivery`, and PR head source-reproduces the fallback regression when `pluginOwnedRuntimeBinding` disables `requireMention` while `missing_plugin`/`no_handler` continues to normal dispatch.

## Solution Assessment

No. The dispatch fix is in the right area, but the Telegram mention bypass happens too early; the narrow maintainable solution is to bypass the gate only for a handled plugin claim or to block ordinary unmentioned fallback.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P1] Keep the mention gate on plugin fallback:** `extensions/telegram/src/bot-message-context.ts:431-433`
  - body: This sets `requireMention` to false for every plugin-owned runtime binding before dispatch knows whether the plugin can handle the message. When the plugin is missing or has no `inbound_claim` handler, dispatch falls through to the ordinary agent path, so unmentioned group/topic messages can be processed despite `requireMention=true`. Preserve the configured mention decision for fallback, or stop ordinary fallback for unmentioned plugin-bound group turns.
  - confidence: 0.88
