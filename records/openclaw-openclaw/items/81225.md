## Summary

Keep open: the PR addresses a real Telegram plugin-bound topic routing bug, but the latest patch still bypasses Telegram mention gating before the plugin claim succeeds, so fallback paths can process unmentioned group/topic traffic.

## What This Changes

The branch adds a plugin-owned Telegram runtime-binding signal, lets plugin claims run under `message_tool_only`, suppresses binding notices in that mode, and adds routing/dispatch regression tests.

## Best Possible Solution

Carry enough binding and mention-state through the turn so successful plugin-bound claims can bypass Telegram mention gating, while missing/no-handler plugin fallback either honors the configured group/topic mention policy or stops ordinary agent fallback for that unmentioned turn.

## Reproduction Assessment

Yes. Source inspection shows current main can skip a plugin-owned binding claim under `message_tool_only`, and the latest PR head source-reproduces the remaining fallback regression through `requireMention=false` plus `missing_plugin`/`no_handler` fallthrough.

## Solution Assessment

No. Allowing plugin claims under `message_tool_only` is in the right area, but bypassing the mention gate before claim success is broader than the narrow routing fix and should be constrained to handled plugin-bound turns or blocked fallback.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Keep mention-gated fallback blocked:** `extensions/telegram/src/bot-message-context.ts:431-433`
  - body: Setting `requireMention` to `false` for every grouped plugin-owned runtime binding happens before dispatch knows whether the plugin can claim the turn. When the plugin is missing or has no `inbound_claim` handler, dispatch breaks into the normal agent path, so unmentioned topic messages that were previously dropped under `requireMention` can now be processed. Preserve the configured mention decision for fallback, or stop ordinary fallback processing for unmentioned plugin-bound turns.
  - confidence: 0.9
