## Best Possible Solution

Keep the current fail-closed topic-safety boundary and route any remaining forum-topic delivery work through the canonical follow-up that preserves topic metadata instead of retrying into a different surface.

## Reproduction Assessment

## Solution Assessment

No. Broadening forum retry is not the best fix because it can misdeliver topic-scoped replies into the group/general context; the safer path is the open topic-preservation/fail-closed follow-up.
