## What This Changes

Request to allow `agents.defaults.models[provider/model].contextTokens` to override the global `agents.defaults.contextTokens` per selected model.

## Best Possible Solution

Maintainers should decide whether per-agent model entries may own context caps; if yes, add one canonical schema/resolver/docs path, otherwise steer users to the existing provider model `contextTokens` surface.

## Reproduction Assessment

## Solution Assessment
