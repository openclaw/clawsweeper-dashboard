The PR adds a gateway fallback that appends one display-safe assistant error transcript entry for started agent runs that return error payloads, plus a runtime-persistence signal to avoid duplicate error turns.

## Best Possible Solution

Land a bounded gateway fix after real behavior proof shows a started agent-run failure leaves exactly one durable assistant error in chat history while preserving the live error broadcast.

## Solution Assessment
