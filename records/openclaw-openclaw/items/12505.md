The existing discussion already treats this as the canonical sandbox/isolation architecture tracker rather than resolved work.

Request a unified, secure-by-default sandbox architecture with cross-platform process, filesystem, network, credential, and resource isolation plus tiered presets for plugins, skills, and tools.
