  - confidence: 0.93
- **[P2] Preserve ordered gitignore rules:** `src/infra/backup-exclude.ts:225-233`
  - body: This deduplication drops later duplicate patterns before passing rules to `ignore`, but gitignore-style rules are ordered and a repeated pattern after a negation can intentionally re-ignore a path. Preserve the full ordered rule list and track pattern stats without changing matcher semantics.
  - confidence: 0.87
- **[P2] Avoid rescanning large archives during create:** `src/infra/backup-create.ts:528-535`
  - body: The integrity check reads the completed archive into an array of every entry on every create path. For the large backups this feature targets, that adds a full second archive pass and O(entry count) memory, so make it streaming, opt-in via verify, or otherwise bounded.
  - confidence: 0.82
