  - body: The exclude branch creates and gzips a separate tar path from raw assets. Current main’s backup writer now does more than that: it snapshots SQLite databases, skips volatile files, retries tar EOF races, avoids temp roots inside sources, skips plugin dependency trees, and handles hardlinks.   - confidence: 0.88

  - body: Gitignore rules are order-sensitive, so duplicate patterns are not always redundant. A pattern list like `*.log`, `!important.log`, `*.log` should end with `important.log` excluded, but this deduplication drops the final rule before `ignore` evaluates it and flips the result. Keep the ordered rule list intact and track source attribution without collapsing later duplicates.
  - confidence: 0.9
