The PR refactors the DeepInfra plugin to discover surface-tagged live catalogs, register image/video catalog providers, add a DeepInfra Anthropic cache wrapper, update defaults/docs/changelog, and extend DeepInfra tests.

## Best Possible Solution

Thread resolved DeepInfra auth into the text discovery and capability paths, preserve existing implicit defaults unless maintainers explicitly approve a default migration, and add regression proof for env and non-env auth plus the cache wrapper.

## Solution Assessment

No, not yet. The catalog expansion and cache-wrapper direction are reasonable, but the fix needs to use the resolved provider-auth contract and avoid unrelated default changes unless maintainers deliberately approve them.
