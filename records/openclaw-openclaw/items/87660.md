Request a lifecycle-aware, likely LLM-assisted curator for root `MEMORY.md` that classifies entries, preserves provenance, budgets evidence, and produces reviewable updates.

## Best Possible Solution

Design one memory-core curator contract that classifies root-memory lifecycle state, preserves provenance, respects locked anchors, and emits a reviewable patch before any `MEMORY.md` write.
