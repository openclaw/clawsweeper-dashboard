## Summary

Add lifecycle-aware LLM-assisted long-term memory curation for MEMORY.md, separate from deterministic short-term promotion.

## Best Possible Solution

Design this as a memory-core curator or plugin seam that validates structured LLM output, preserves provenance, respects locked anchors, uses a bounded evidence budget, and emits a reviewable patch before any durable write.
