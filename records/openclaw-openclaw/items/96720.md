  - body: `resolveAgentDir()` falls back to the default per-agent state directory when no `agents.list[].agentDir` is configured.   - confidence: 0.92

  - body: This block runs after `applyBootstrapHookOverrides()` has produced the effective bootstrap list, then replaces same-named files with agentDir content.   - confidence: 0.9
- **[P2] Preserve BOOTSTRAP completion filtering:** `src/agents/bootstrap-files.ts:334-337`
  - body: `filterCompletedWorkspaceBootstrapFile()` compares `BOOTSTRAP.md` against the workspace root. After replacing a hook or workspace entry with an agentDir `BOOTSTRAP.md`, a stale first-run bootstrap file can remain injected after workspace setup completion.
  - confidence: 0.84
