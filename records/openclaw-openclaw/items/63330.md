## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.87

Full review comments:

- **[P1] Gate gateway_restart with owner and restart policy:** `extensions/gateway-restart/index.ts:89-95`
  - body: The new tool is registered as an ordinary plugin tool, then its execute path can run configured commands, write the marker, and call `process.exit(0)` without checking sender ownership or the existing `commands.restart=false` policy. The core `gateway` restart action has both of those protections, so enabling this plugin creates a second restart path that can bypass the operator's security boundary.
  - confidence: 0.89
- **[P2] Preserve persisted session context in cold followups:** `src/auto-reply/reply/queue/build-followup-run.ts:107-119`
  - body: This builder reconstructs the run from agent defaults instead of the persisted session context: it uses the default workspace and does not populate fields like account/group/cwd context that current followup runners forward into `runEmbeddedAgent`. Cold post-restart followups can therefore run in the wrong workspace or with the wrong routing/tool policy context for spawned, grouped, or multi-account sessions.
  - confidence: 0.86
- **[P2] Keep the marker until the callback is durably processed:** `extensions/gateway-restart/index.ts:221-223`
  - body: `enqueueFollowupTurn()` only reports that the followup was accepted into the process-local in-memory queue. Deleting the restart marker immediately after that means a second crash or restart before the queue drains loses the callback permanently, which breaks the resume-after-restart guarantee this plugin is adding.
  - confidence: 0.88
- **[P2] Declare the plugin runtime dependency in the plugin package:** `extensions/gateway-restart/index.ts:4-5`
  - body: The new plugin imports `@sinclair/typebox`, but its package manifest has no runtime dependency. Bundled plugin runtime dependencies need to be owned by the plugin package, otherwise isolated plugin installs or frozen workspace installs can fail or drift from the manifest.
  - confidence: 0.82
