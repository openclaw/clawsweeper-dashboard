## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Route restart continuation through the trusted gateway path:** `extensions/gateway-restart/index.ts:36-39`
  - body: This tool asks the model for `sessionKey` and later uses that value for an owner-trusted follow-up, while current `gateway` restart derives the session from trusted runtime context, checks `commands.restart`, and writes the canonical sentinel. A plugin-enabled model can target the wrong session and bypass the existing restart boundary; remove this parallel tool or rebuild it on the trusted gateway restart path.
  - confidence: 0.9
- **[P1] Preserve full follow-up run context:** `src/auto-reply/reply/queue/build-followup-run.ts:107-119`
  - body: The cold builder only reconstructs a subset of `FollowupRun.run` and hardcodes `senderIsOwner` plus `blockReplyBreak`, dropping account, group, sender, cwd, skills, elevated/bash, reply, and streaming context that current queued runs carry. Cold plugin follow-ups can resume in the wrong workspace or with broader/different policy than the original session.
  - confidence: 0.88
- **[P2] Use the canonical restart sentinel instead of a marker file:** `extensions/gateway-restart/index.ts:160-164`
  - body: This writes `restart-pending.json` and deletes it after an in-memory enqueue succeeds, but current restart continuation state is SQLite-backed and survives through the gateway sentinel/queued delivery flow. A crash after enqueue but before drain can lose the callback, and maintainers would have to support a second restart state format.
  - confidence: 0.86
- **[P2] Declare the TypeBox runtime dependency in the plugin package:** `extensions/gateway-restart/package.json:2-7`
  - body: The plugin imports `@sinclair/typebox`, but its package manifest declares no runtime dependency even though the `openclaw.install.npmSpec` metadata points at an installable plugin package. Installed outside the monorepo hoist, loading this plugin can fail before registration with a missing module error.
  - confidence: 0.78
