  - body: When an inference profile has no matching foundation model, this fallback reads `config?.defaultContextWindow`, but `config` is not defined inside `resolveInferenceProfiles`.   - confidence: 0.95
- **[P2] Include default provenance in discovery cache key:** `extensions/amazon-bedrock/discovery.ts:168-170`
  - body: The new flag depends on whether `defaultContextWindow` was omitted versus explicitly set to the same numeric value, but the cache key still only contains the resolved window number. With refresh caching enabled, one provenance mode can reuse the other's cached discovery result until expiry.
  - confidence: 0.89
