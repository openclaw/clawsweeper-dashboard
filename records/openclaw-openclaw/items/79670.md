## What This Changes

The issue asks Google onboarding to stop recommending Gemini preview models without quota guidance, by warning Tier 1 users, recommending stable defaults, or showing per-model quota limits.

## Best Possible Solution

Provider owners should choose a Google-plugin-owned policy: move first-run defaults to a stable model such as `google/gemini-2.5-flash`, add explicit preview quota guidance, or add a maintained quota display if Google exposes a reliable source for it.
