## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.84

Full review comments:

- **[P2] Use flat enums in the heartbeat tool schema:** `src/agents/tools/heartbeat-response-tool.ts:13-20`
  - body: `outcome` and `priority` are built with `Type.Union([...Type.Literal])`, which serializes to `anyOf`. This tool goes through the normal model tool path, and the existing enum helper avoids that shape because some providers reject it, so use `stringEnum`/`optionalStringEnum` here.
  - confidence: 0.87
- **[P2] Regenerate the Plugin SDK API baseline:** `src/plugin-sdk/agent-harness-runtime.ts:75-78`
  - body: This adds public exports from `openclaw/plugin-sdk/agent-harness-runtime`, but the PR does not update `docs/.generated/plugin-sdk-api-baseline.sha256`. Intentional SDK surface changes need the generated baseline refreshed so `pnpm plugin-sdk:api:check` stays green.
  - confidence: 0.78
