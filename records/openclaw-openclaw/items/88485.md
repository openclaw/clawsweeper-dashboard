- **[P1] Clear attestation markers during supported resets:** `src/agents/workspace.ts:350-351`
  - body: This stores a new `${workspace}.attested` file next to the workspace, but full reset/uninstall cleanup removes the workspace directories only.   - confidence: 0.88

Add coverage for that attested no-state/no-user-content case and reject before reseeding.
  - confidence: 0.9
