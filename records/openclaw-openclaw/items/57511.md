Full review comments:

- **[P1] Declare the gRPC runtime deps in the Teams plugin:** `extensions/msteams/src/voice/worker-bridge.ts:28-31`
  - body: `WorkerBridge` lazy-imports `@grpc/grpc-js` and `@grpc/proto-loader`, but the MS Teams plugin package does not declare either dependency. Plugin runtime deps must be plugin-local, so an installed `@openclaw/msteams` voice build can fail before connecting to the worker.
  - confidence: 0.91
- **[P1] Resolve the proto from the packaged Teams plugin path:** `extensions/msteams/src/voice/worker-bridge.ts:22`
  - body: From `extensions/msteams/src/voice`, `../../../media-worker/Protos/bridge.proto` resolves to `extensions/media-worker/...`, not `extensions/msteams/media-worker/...`. The first `connect()` will fail to load the gRPC service unless this path is fixed and packaged as an asset.
  - confidence: 0.94
- **[P1] Secure the worker channel before sending Teams secrets:** `extensions/msteams/src/voice/worker-bridge.ts:121`
  - body: The TS bridge uses `grpc.credentials.createInsecure()` while `joinMeeting` sends `tenantId`, `appId`, and `appSecret`, and the worker binds gRPC on all interfaces. Remote worker support needs TLS/auth or a loopback-only default before credentials are sent over this channel.
  - confidence: 0.93
- **[P1] Resolve SecretInput credentials before joining the worker:** `extensions/msteams/src/voice/manager.ts:147-150`
  - body: The voice manager only forwards `appPassword` when it is a literal string and silently sends an empty secret for the normal `SecretRef` path. Use the existing MS Teams credential resolver and fail early when a secret cannot be resolved.
  - confidence: 0.9
- **[P1] Convert TTS output to the worker PCM contract:** `extensions/msteams/src/voice/manager.ts:484-489`
  - body: `textToSpeech()` returns an audio file, not guaranteed raw 16 kHz mono signed PCM, but the code reads that file and sends its bytes directly to `PlayAudio`. Encoded WAV/MP3/Opus bytes would be injected as PCM noise unless the PR decodes and resamples or carries format metadata.
  - confidence: 0.86
- **[P1] Do not add the public SDK seam with the unproven worker contract:** `src/plugin-sdk/msteams.ts:74-75`
  - body: Exporting `MSTeamsVoiceConfig` and `MSTeamsVoicePermissionMode` creates a public plugin SDK contract before the worker protocol, permissions, transport security, and compatibility story are agreed. Keep this plugin-private or land the SDK contract separately with explicit review.
  - confidence: 0.88
