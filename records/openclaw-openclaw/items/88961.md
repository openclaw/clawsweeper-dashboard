- **[P1] Use effective plugin activation before suppressing warnings:** `src/agents/embedded-agent-runner/effective-tool-policy.ts:28-35`
  - body: This helper trusts raw `plugins.allow` and `plugins.entries.<id>.enabled` as known plugin IDs, but those are not the effective activation rules: `plugins.enabled: false`, `plugins.deny`, and `entries.<id>.enabled: false` all still block the plugin. Please derive the known IDs from effective activation and add disabled/deny/global-disable coverage.
