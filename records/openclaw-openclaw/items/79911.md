## Best Possible Solution

Use the canonical fix in https://github.com/openclaw/openclaw/pull/79468 or an equivalent patch that aligns onboarding with the documented 200k default, repairs generated low defaults, and preserves explicit small model limits.

## Reproduction Assessment

## Solution Assessment

No. The affected path is right, but this PR's blanket 32,768-token clamp can overwrite explicit small model limits; the canonical PR preserves those limits while fixing the generated default and proof gap.
