Overall confidence: 0.92

Full review comments:

- **[P1] Preserve taint dependencies in tool results:** `src/agents/camel/orchestrator.ts:153`
  - body: `wrapToolResult` only receives the callee name and raw result, so a tainted value passed through a trusted or read-only intermediate tool can be re-labeled clean before a later side-effect check. Carry `wrappedArgs` dependencies into the result, for example by deriving or merging capabilities from the input values.
  - confidence: 0.93
- **[P1] Disable tools for post-orchestration synthesis:** `src/agents/pi-embedded-runner/run/attempt.ts:2178-2182`
  - body: After CaMeL has already executed side-effect tools, this calls the normal tool-enabled session again and relies on prompt text saying not to re-run tools. A model can still call `exec`, `write`, or messaging tools again because the orchestrated calls are not represented as completed tool turns in the active transcript.
  - confidence: 0.91
- **[P1] Set the replay guard before executing side effects:** `src/agents/pi-embedded-runner/run/attempt.ts:2154`
  - body: `camelSideEffectsExecuted` is set only after `orchestrator.execute()` returns. If an earlier mutating step succeeds and a later step throws, the outer retry path can treat the attempt as replay-safe and run the completed side effects again.
  - confidence: 0.9
- **[P1] Merge defaults.camel into hook config:** `src/agents/pi-tools.before-tool-call.ts:187-196`
  - body: The schema and main runner accept `agents.defaults.camel`, but the before-tool-call safety path only reads `agents.camel` and per-agent overrides. Any fallback or direct tool loop running under defaults-only config will skip the configured CaMeL policy checks.
  - confidence: 0.88
- **[P1] Do not trust client source for approval forwarding:** `src/infra/exec-approval-forwarder.ts:71-75`
  - body: The new public `source` field flows from `exec.approval.request` into the forwarder, where `source === "camel"` bypasses the forwarding enabled gate. This needs a trusted server-side marker; otherwise any caller that can make an approval request can select the bypass path.
  - confidence: 0.9
