Fix https://github.com/openclaw/openclaw/issues/72952. The observable bug is that official Docker-image commands run as `node dist/index.js ...` take tens of seconds for root help, `channels status`, and Matrix verification/status commands while trivial `node -e` is fast. Keep package-root library exports intact. Start in `src/index.ts`, `src/entry.ts`, `src/cli/run-main.ts`, `src/cli/route.ts`, `src/cli/command-catalog.ts`, `src/cli/program/routed-command-definitions.ts`, `scripts/bench-cli-startup.ts`, Docker setup/docs/tests, and Matrix CLI registration. Do not broad-load bundled plugins for help/status fast paths. Add startup benchmark cases for `dist/index.js --help`, `channels status --json`, `channels status --probe --json`, `matrix --help`, and `matrix verify status --json`; add or update tests that prove the Docker-facing path is covered. Add a user-facing `CHANGELOG.md` fix entry, crediting the human reporter when appropriate.
