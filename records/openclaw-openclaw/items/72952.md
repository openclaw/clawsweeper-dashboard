Keep open. Current main still exposes and documents the official Docker `node dist/index.js` command surface, and the checked-in startup benchmark/coverage still does not prove the reported direct-dist help, channel, and Matrix latencies are fixed.

The issue reports 24s-49s startup latency for direct `node dist/index.js` Docker subcommands, including help, channel status, and Matrix verification commands.
