Keep open. Current main still exposes the official Docker direct `node dist/index.js` surface, and the checked-in startup coverage still does not prove the reported Docker `dist/index.js` channel and Matrix command latencies are fixed.

Keep Docker-facing direct CLI execution on the same lean startup budget as `openclaw.mjs`, or move first-party Docker workflows to the wrapper, then add startup-budget coverage for the reported channel and Matrix cases.
