## Best Possible Solution

## Reproduction Assessment

Yes, at source level: current main reaches retry-limit or expiry give-up, marks failed/blocked delivery, and suppresses non-cancelled subagent terminal task delivery before requester wake. I did not run the timing-sensitive live Signal/K8s scenario.

## Solution Assessment
