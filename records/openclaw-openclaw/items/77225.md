## Summary

Keep open: current main still does not implement AI-generated session titles, and this PR remains the open candidate for that feature. It is not merge-ready because the patch adds default-on transcript summarization, uses a raw provider HTTP call, can disclose subject-titled sessions with no visible benefit, misses config-baseline follow-through, is merge-conflicting with main, and lacks real behavior proof of title generation/persistence/rendering.
