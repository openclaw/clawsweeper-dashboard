## What This Changes

The issue reports that memory-core `memory_search` can block the Node.js event loop for 30-60+ seconds during Discord DMs, causing gateway disconnects and hung agents.

## Best Possible Solution

Keep this open until memory-core has a bounded, non-blocking `memory_search` execution contract and regression proof that slow search, embedding, or indexing cannot starve Discord or gateway I/O.
