- **[P2] Recompute runtime cwd before using TUI fallbacks:** `src/tui/tui.ts:523`
  - body: `fallbackLaunchCwd` is set once from the launch cwd. If TUI starts successfully and that directory is deleted later, the auth-login spawn still receives the now-missing path instead of falling back, so validate or recompute the cwd at the point of use.
  - confidence: 0.82
- **[P2] Avoid using the executable dir as OpenClaw home:** `src/infra/home-dir.ts:93`

  - confidence: 0.84
