Keep open: current main and the latest release still admit the reported Matrix channel path, and the linked fix PR is open rather than merged.

## Best Possible Solution

Review and land the linked fix PR if maintainers accept the shared policy: suppress default non-forum channel tool/progress text while keeping explicit opt-in, forum, media, and approval paths covered by tests.
