  - body: After checking out the PR branch, this passes a workspace `prompt-file` into the pinned Codex action.   - confidence: 0.91

  - body: The enforcement step only records that some change exists, and this commit step then stages everything. Prompt text is not a security boundary; reject untracked files, added/deleted files, and forbidden paths before any commit or push.
  - confidence: 0.93
- **[P2] Create an app token before ClawSweeper dispatch:** `.github/workflows/ci-fix-agent.yml:217-220`
  - body: This workflow checks for `CLAWSWEEPER_APP_PRIVATE_KEY` but never creates the ClawSweeper app token before calling the receiver repo dispatch endpoint. The repository `github.token` is scoped to `openclaw/openclaw`, so this best-effort notification can fail even when the app key exists.
  - confidence: 0.84
- **[P2] Resolve failed PRs with the configured token:** `.github/workflows/clawsweeper-dispatch.yml:317-319`
  - body: The step exports the ClawSweeper app token as `GH_TOKEN`, but this lookup overrides it with `${GITHUB_TOKEN}`, which is not set in the step environment. That can make the lookup fail or use the wrong credential and dispatch `pr_number: 0` instead of the failed PR.
  - confidence: 0.88
- **[P2] Pin third-party actions in the writer workflow:** `.github/workflows/ci-fix-codex.yml:42-49`
  - body: This secret-bearing workflow uses mutable `actions/checkout@v6` and `actions/setup-node@v6` refs. Current repo workflow patterns and GitHub secure-use guidance favor full-length commit SHAs so this branch-writing path is not exposed to moved tags.
  - confidence: 0.87
- **[P2] Allow only the intended workflow actor to run Codex:** `.github/workflows/ci-fix-codex.yml:78-90`
  - body: The upstream action checks `GITHUB_ACTOR` for write access and only bypasses trusted bots when `allow-bots` or `allow-bot-users` is set. Because this workflow is dispatched by Actions and does not configure either input, the Codex step can reject before the repair path runs; decide and encode the intended actor boundary.
  - confidence: 0.76
