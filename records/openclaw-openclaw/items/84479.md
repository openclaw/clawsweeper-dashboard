- **[P2] Use the ClawSweeper app token for dispatch:** `.github/workflows/ci-fix-agent.yml:217-220`
  - body: This step checks that the ClawSweeper private key exists but never creates or uses the app token before calling the cross-repo dispatch API. Using the source repo token, or no `GH_TOKEN`, can silently fail even when the app credential is configured.
  - confidence: 0.86
