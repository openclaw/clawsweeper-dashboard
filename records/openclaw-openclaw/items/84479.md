  - body: This step exports the ClawSweeper app token as `GH_TOKEN`, but the PR lookup overrides it with `${GITHUB_TOKEN}`, which is not provided in the step environment. That can make the lookup unauthenticated or empty and dispatch `pr_number: 0` for failed PR runs.
