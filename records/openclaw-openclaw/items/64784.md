## Best Possible Solution

Keep this issue as the canonical tracker, land a focused LINE-owned sticker send path plus any generic channelData-only coalescer fix maintainers approve, decide inbound metadata separately, and keep catalog/editor content ClawHub-first unless maintainers sponsor bundling.
