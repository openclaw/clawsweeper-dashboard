- **[P2] Enforce required files with folder rules:** `repotype.yaml:15-16`
  - body: These file globs only match files that already exist, so deleting `skills/foo/SKILL.md`, `packages/x/package.json`, or `extensions/y/openclaw.plugin.json` leaves no file for repotype to validate. Repotype’s missing-file diagnostics come from `folders[].requiredFiles`; add folder rules for skill, package, and plugin roots before this can enforce the structure promised by the PR.
  - confidence: 0.91
