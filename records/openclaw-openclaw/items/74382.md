## What This Changes

The issue asks for `openclaw onboard` to avoid prompting for OpenAI Whisper or ElevenLabs API keys after presenting a keyless `openai-whisper` dependency-install option.

## Best Possible Solution

Yes. Source inspection gives a high-confidence path: with missing local `openai-whisper` plus missing env-gated `openai-whisper-api` or `sag`, `openclaw onboard` can show the keyless install option and then prompt for the keyed missing skill.

## Solution Assessment
