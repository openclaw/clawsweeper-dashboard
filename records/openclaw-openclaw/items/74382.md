## What This Changes

The issue reports that `openclaw onboard` lists local keyless `openai-whisper` but later asks for OpenAI Whisper API or ElevenLabs API keys.

## Best Possible Solution

Keep this issue open until a linked PR lands. ## Solution Assessment
