## What This Changes

Correct the bundled Himalaya skill’s CLI examples and add accurate, version-scoped guidance for email composition and the upstream save-copy limitation.

## Best Possible Solution

Land one maintained, v1.2-scoped Himalaya documentation update that fixes every verified command shape, clearly labels MML and save-copy material as v1.x behavior, and can be cleanly replaced when the bundled installation moves to v2.

## Reproduction Assessment

## Solution Assessment

Partly. The syntax corrections are the narrow correct fix, but the broader proposal should explicitly target Himalaya v1.x rather than presenting MML and save-copy guidance as version-independent because upstream v2 removes those workflows.
