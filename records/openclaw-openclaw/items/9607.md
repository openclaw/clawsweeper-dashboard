## What This Changes

Correct the bundled Himalaya skill’s move/copy/export examples and add verified guidance for email formatting, HTML/MML composition, and the upstream save-copy limitation.

## Best Possible Solution

Publish one maintained Himalaya skill update that declares its supported CLI version, fixes every confirmed copy-paste command error, and documents only composition and save behavior verified against that version’s upstream source and migration guidance.

## Solution Assessment

No, the original broad proposal should not land unchanged. The command corrections are appropriate, but the formatting and save-copy sections need a maintainer-selected Himalaya version boundary and fresh upstream verification because v2 changed both areas substantially.
