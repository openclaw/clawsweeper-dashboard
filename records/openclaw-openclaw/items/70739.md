- **[P2] Validate the heartbeat interval before scheduling:** `src/gateway/http-common.ts:147`
  - body: `Math.max(1000, options?.intervalMs ?? 30_000)` leaves `NaN` and `Infinity` intact; Node normalizes those timer delays to 1 ms, so a future env-derived override could turn each stream into a tight heartbeat loop.
