The issue requests richer session derived titles using compaction summaries and meaningful recent user messages instead of low-information first user messages.

## Best Possible Solution

Land a bounded session-title implementation that keeps display metadata precedence, selects a vetted compaction summary or meaningful transcript candidate, adds regression coverage for greeting-first transcripts, and then closes this issue after the linked PR or a follow-up merges.
