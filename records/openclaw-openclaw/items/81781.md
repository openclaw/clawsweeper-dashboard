Request to improve derived session titles so session lists use compaction summaries or meaningful recent user content instead of low-information first user messages.

Resolve the linked implementation path with a bounded title-source design that preserves displayName/subject priority, uses a meaningful compaction summary or later transcript candidate when available, and keeps focused gateway/session-list coverage.
