Request better session derived-title generation by preferring compaction summaries or meaningful later user messages over low-information first user messages.

## Best Possible Solution

Review https://github.com/openclaw/openclaw/pull/81948 and either land an acceptable scoped greeting filter or revise it into one bounded session-title fix that covers compaction summaries and meaningful-message fallback.

## Reproduction Assessment

## Solution Assessment
