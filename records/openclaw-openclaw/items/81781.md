Improve session `derivedTitle` generation so session lists can use compaction summaries or later meaningful user messages instead of unhelpful first-message greetings.

## Best Possible Solution

Resolve https://github.com/openclaw/openclaw/pull/81948 by landing a scoped acceptable fix or revising it to cover bounded compaction-summary and meaningful-message title candidates before closing this issue.

## Reproduction Assessment

## Solution Assessment

Unclear. The requested direction is plausible, but maintainers still need to decide whether a small greeting filter is enough or whether the fix must include compaction-summary and bounded later-message candidates.
