## What This Changes

Prevent empty REM dreaming results from being written as candidate-like markdown and later promoted into long-term memory.

## Best Possible Solution

## Solution Assessment

Yes. The narrow maintainable fix is to stop producing candidate-like markdown for empty REM results and defensively filter already-written historical placeholders during promotion.
