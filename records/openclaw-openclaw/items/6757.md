## Summary

## What This Changes

Request an agent-callable way to compact its own active session, such as `session_status(action: "compact")`, without relying on an authorized sender slash command.

## Best Possible Solution

Review, repair, and land a narrow current-session-only self-compaction action such as `session_status(action: "compact")`, with explicit mutation tracking, visibility checks, audit semantics, and active-run recovery behavior.
