Land a narrow generic loader/catalog fix that preserves resolved `plugins.entries.<id>.config` for required tool plugins across startup, active registry reuse, `tools.catalog`, and post-ready plugin resolution, with regression coverage for valid config not being revalidated as `{}` or logged repeatedly.
