Add a Control UI chat composer Expand button that opens a large modal editor for long prompts, with Apply, Cancel, Send, and modal-specific keyboard behavior.

## Best Possible Solution

If accepted, implement this in the Control UI chat composer using the existing modal-dialog primitive, separate modal draft state, Apply/Cancel/Send semantics, modal-only keyboard handling, focus restoration, preserved attachment/queue behavior, and focused UI tests.
