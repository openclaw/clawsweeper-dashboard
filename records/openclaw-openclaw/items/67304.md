## Best Possible Solution

If maintainers accept the env surface, implement it as an opt-in override that omits `fsCache` when the OpenClaw variable is unset or blank, includes cache-key isolation for effective cache settings, rebases onto current Matrix/native loader code, and adds focused tests plus redacted real loader proof.

## Solution Assessment
