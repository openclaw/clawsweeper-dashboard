## Best Possible Solution

If maintainers accept the env var, make it an opt-in override that omits `fsCache` when unset or blank, isolates loader caching by the effective cache option, rebases onto the current Matrix wrapper, and adds focused tests plus redacted real loader proof.

## Reproduction Assessment

## Solution Assessment
