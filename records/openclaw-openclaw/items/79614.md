## Summary

Keep open. Current main has partial stale-final suppression and active-run steering, but it still does not provide the requested host-owned finalize-time latest-user snapshot, built-in complaint flag, or re-anchor/redraft contract.

The issue requests a host-owned stale-reply re-anchoring contract so an in-progress agent turn rechecks the latest user message before finalizing its response.

## Best Possible Solution
