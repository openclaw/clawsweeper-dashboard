## What This Changes

## Best Possible Solution

Land or replace https://github.com/openclaw/openclaw/pull/41067 with an additive gateway active-run snapshot contract and Control UI reconciliation, while keeping outbound queue/ACK loss tracked separately in https://github.com/openclaw/openclaw/issues/45952.
