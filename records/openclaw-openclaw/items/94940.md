  - body: `serviceAccountAdc` is a credential selector, but the patch only checks it during credential resolution.   - confidence: 0.92
- **[P1] Keep Compute ADC inside the auth boundary:** `extensions/googlechat/src/accounts.ts:159-160`
  - body: Returning `adc` lets `GoogleAuth` fall through to ADC when no key is provided. In `google-auth-library@10.6.2`, the Compute path calls `gcp-metadata` for availability/token fetches via direct `gaxios.request` to metadata hosts, bypassing the injected `createGoogleAuthFetch`/`fetchWithSsrFGuard`; guard that path or get explicit maintainer/security approval before merging.
  - confidence: 0.88
