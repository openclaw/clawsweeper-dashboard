Full review comments:

- **[P1] Treat serviceAccountAdc as credential-owned config:** `extensions/googlechat/src/accounts.ts:159-160`
  - body: `serviceAccountAdc` is a credential selector, but the patch only checks it during credential resolution.   - confidence: 0.92
- **[P1] Guard or approve metadata-server ADC requests:** `extensions/googlechat/src/accounts.ts:159-160`
  - body: Returning `adc` lets `GoogleAuth` choose the Compute ADC client when no credentials are provided. That dependency path calls `gcp-metadata`, whose metadata detection and token lookup use direct `gaxios.request` calls to metadata hosts instead of the injected guarded fetch, so this needs an OpenClaw-owned guard or explicit maintainer/security approval.
  - confidence: 0.88
