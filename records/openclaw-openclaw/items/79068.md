## What This Changes

The PR adds an OpenClaw wrapper around Pi `SessionManager` to prune in-memory post-compaction transcript state, routes gateway/CLI SessionManager usage through it, and also changes source-delivery matching, origin/port diagnostics, macOS channel config rendering, release checks, tests, and changelog text.

## Best Possible Solution

Keep the memory-bound fix, but make retained live context separate from the append-only persisted transcript contract, preserve plugin-owned target normalization, restore release-soak automation, and require current-head proof that disk history survives pruning and later rewrites.
