Full review comments:

- **[P1] Preserve disk history when pruning memory:** `src/agents/pi-session-manager-memory-bound.ts:201`
  - body: `pruneInMemoryEntriesIfNeeded` replaces `manager.fileEntries` with the pruned live branch. Pi's `_rewriteFile()` writes `this.fileEntries` back to the JSONL file, and OpenClaw still calls that rewrite path after appends, so a compacted long-running session can lose the historical pre-compaction lines the PR says remain append-only. Keep a separate bounded live view or patch rewrite behavior, and add a prune -> rewrite -> reopen regression test.
  - confidence: 0.91
- **[P1] Keep plugin target normalization in delivery matching:** `src/infra/outbound/source-delivery-plan.ts:104-112`
  - body: The PR replaces `normalizeTargetForProvider()` with a local partial parser for source-delivery matching. Existing channel normalizers handle supported forms such as Google Chat `gchat:user:User@Example.com` -> `users/user@example.com` and WhatsApp/Discord/Slack variants; bypassing them means a message-tool send can fail verification and trigger the wrong fallback or duplicate visible reply.
  - confidence: 0.86
- **[P2] Restore the macOS channel quick renderer:** `apps/macos/Sources/OpenClaw/ChannelConfigForm.swift:70-95`
  - body: The channel-quick path no longer uses `renderChannelQuickObject`, `SettingsCardGroup`, or the compact leaf rows for boolean/string/number/enum fields. That rolls back the existing macOS quick-settings experience in an unrelated memory PR, so the UI churn should be dropped or moved to a separately reviewed macOS change.
  - confidence: 0.82
- **[P1] Keep checkout credentials disabled in install smoke:** `.github/workflows/install-smoke.yml:109-111`
  - body: Several install-smoke checkouts now omit `persist-credentials: false`. Since actions/checkout v6 defaults `persist-credentials` to true, these jobs will leave the workflow token in local git config for later shell, Docker, and installer steps; restore the explicit false setting unless a security-reviewed workflow change needs it.
  - confidence: 0.87
