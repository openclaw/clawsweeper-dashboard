## Summary

Keep open. The SessionManager memory fix is plausible and the supplied real behavior proof is sufficient, but the current branch still regresses plugin config deprecation attribution by dropping scoped wrappers for the deprecated `loadConfig` and `writeConfigFile` helpers.

## What This Changes

The branch adds a pnpm patch for `@earendil-works/pi-coding-agent@0.74.0` to prune stale in-memory SessionManager entries after compaction, adds regression coverage, updates port diagnostics and several tests, and records the dependency patch in workspace metadata.

## Best Possible Solution

Restore plugin-scope wrapping for both current and deprecated config helpers, then merge only after maintainers approve this exact Pi patch or replace it with an upstream dependency release.

## Reproduction Assessment

Yes. Source inspection of published Pi 0.74.0 gives a high-confidence path: create or open a persisted SessionManager, append many entries through compaction, and observe all loaded/appended entries retained in `fileEntries` and `byId`.

## Solution Assessment

No, not as submitted. Bounding live memory at the SessionManager compaction boundary is a narrow and plausible fix, but the PR should preserve deprecated runtime config helper attribution and still receive explicit dependency-patch approval.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.87

Full review comments:

- **[P2] Keep deprecated config helpers scoped:** `src/plugins/registry.ts:2402-2405`
  - body: This config proxy now scopes only `current`, `mutateConfigFile`, and `replaceConfigFile`, while `loadConfig` and `writeConfigFile` fall through from the spread `config` object unwrapped. Those deprecated helpers are still part of `PluginRuntime`, and their warnings use the active plugin scope to include plugin id/source and de-dupe per plugin; external/global plugins calling them would regress to anonymous warnings. Keep the old wrappers while adding the new helper wrappers, and cover both paths in the registry test.
  - confidence: 0.87
