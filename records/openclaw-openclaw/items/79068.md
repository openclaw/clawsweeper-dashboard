## What This Changes

This PR patches `@earendil-works/pi-coding-agent@0.74.0` to prune live SessionManager entries after compaction, updates dependency patch metadata, adds memory and port tests, adjusts port diagnostics, and carries related test import updates.

## Best Possible Solution

Preserve runtime config scope for both current and deprecated helpers, then land bounded SessionManager pruning only after dependency-patch or upstream-release approval.

## Reproduction Assessment

Yes. Source inspection of `@earendil-works/pi-coding-agent@0.74.0` shows persisted sessions load and append all entries into `fileEntries`/`byId` without pruning, so a long persisted session with compaction can reproduce the unbounded live-memory path.

## Solution Assessment
