## What This Changes

The issue requests native per-call cost reporting attributed to the resolved auth profile, including visible fallback usage.

## Best Possible Solution

Define a privacy-safe stable profile attribution at the completed-call boundary, persist it with canonical assistant usage, aggregate cost by that identifier, and emit an explicit fallback or rotation event with controlled exposure.
