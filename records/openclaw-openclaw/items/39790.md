## Best Possible Solution

Land a rebased or replacement Gateway title fix that strips inbound metadata before title extraction, threads computed row displayName into both sync and async derived-title paths, and includes focused regression coverage plus redacted TUI/Gateway proof.

## Reproduction Assessment

Yes. Source inspection shows the TUI requests derived titles, renders derivedTitle before displayName, current transcript extraction omits stripInboundMetadata, and current sync/async title hydration omits the computed row displayName.

## Solution Assessment
