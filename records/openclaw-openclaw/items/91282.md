Settle the estimated-usage contract, then review and land the linked implementation PR with SDK exports, docs, baseline coverage, session/status labeling, and focused runner/parser tests before closing this issue.

## Solution Assessment

Unclear until maintainers decide the public SDK and UI semantics. A backend-owned hook is the right owner boundary if heuristic counts stay clearly marked and survive through session/status/cost consumers.
