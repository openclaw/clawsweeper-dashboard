  - body: `onWorkspaceNavigate` only compares the requested path before applying a list response.   - body: The broad `startsWith("..")` check rejects valid relative filenames such as `...` and `..notes`, even though only an exact `..` path segment is unsafe. The PR's own `...` test expects this path to succeed, so rely on absolute-path and segment validation instead.
  - confidence: 0.95
- **[P2] Default workspace writes to create parents:** `src/gateway/server-methods/agents-workspace.ts:527`
  - body: Omitted `createDirs` is converted to `false`, but the advertised API contract says `agents.workspace.set` creates parent directories by default. API clients sending a nested new path without the optional flag will fail despite that contract; use `params.createDirs !== false` or update the contract and tests together.
  - confidence: 0.89
- **[P2] Keep mkdir recursive by default:** `src/gateway/server-methods/agents-workspace.ts:617`

  - confidence: 0.93
