## What This Changes

Add a local or self-hosted STT provider path for Talk Mode realtime voice, including a proposed `baseUrl` override for realtime provider config.

## Best Possible Solution

Publish a ClawHub realtime transcription or realtime voice provider plugin for the local ASR engine, and open a narrow core SDK issue only if that implementation exposes a missing generic hook.

## Solution Assessment
