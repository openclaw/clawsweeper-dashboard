Full review comments:

- **[P2] Document the hosted Control UI security model:** `docs/install/xcloud.mdx:101-103`
  - body: The guide tells users to collect a hosted Control UI URL and Basic Auth password, but it never states how xCloud configures Gateway auth, `gateway.controlUi.allowedOrigins`, browser device identity, proxy/TLS trust, or recovery.   - confidence: 0.74
