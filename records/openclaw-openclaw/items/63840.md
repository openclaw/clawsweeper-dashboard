## What This Changes

The PR passes normalized Slack DM classification into thread-context resolution so assistant DM self-thread roots preserve `MessageThreadId`, adds Slack regression tests, and updates the changelog.

## Best Possible Solution

Land this PR or an equivalent Slack-owned patch that preserves `MessageThreadId` only for normalized DM assistant roots while keeping non-DM self-thread roots unset.

## Solution Assessment

Yes. Passing the already-normalized direct-message classification into the existing thread resolver is the narrowest maintainable fix, and the PR preserves the non-DM self-thread guard required by downstream Slack tool threading.
