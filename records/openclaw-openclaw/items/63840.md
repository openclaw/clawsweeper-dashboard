## Summary

Keep open. Current main still drops `MessageThreadId` for normalized Slack DM self-thread roots when direct-message reply mode is `off`, `first`, or `batched`, while the latest PR patch uses the existing normalized DM classification and preserves non-DM self-thread behavior.

## Best Possible Solution

## Solution Assessment
