## Best Possible Solution

Resolve Claude CLI command discovery on Windows with the existing shell-free wrapper/entrypoint resolver, make empty-result turns deliver or fail visibly when text deltas exist, and document a complete Windows-safe `claude-cli` config.

## Solution Assessment
