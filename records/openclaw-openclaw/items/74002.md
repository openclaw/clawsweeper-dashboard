  - body: When OPENCLAW_STATE_DIR points to a new directory, the PR creates a new approvals file instead of migrating or warning about an existing stricter ~/.openclaw/exec-approvals.json.   - confidence: 0.72

  - body: The default approval socket moves under OPENCLAW_STATE_DIR. If an operator sets that root to a shared or writable directory, another local process may be able to pre-bind the socket path and receive the client token before returning an approval decision; maintainers should decide whether ownership/mode checks or a private socket subdirectory are required.
  - confidence: 0.64
