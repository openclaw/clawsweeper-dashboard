## What This Changes

The issue requests a TUI compose mode that opens `$VISUAL` or `$EDITOR`, round-trips prompt text through a temporary markdown file, and restores it without auto-submitting.

## Best Possible Solution

Implement this as TUI-owned composer behavior using the existing suspend/resume boundary, focused regression tests, and public TUI documentation; keep #55836 scoped to in-place vi/vim editing.

## Solution Assessment
