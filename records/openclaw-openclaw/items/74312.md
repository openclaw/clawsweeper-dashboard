## Summary

Keep open. Current main still hashes rotating token material for Claude token-shaped credentials and token auth-profile credentials, and #74493 remains the focused open closing PR for the fix.

## What This Changes

Reported Claude CLI OAuth token rotation can flip auth epochs through token-shaped credential parsing, resetting active sessions mid-conversation.

## Best Possible Solution

Review and land #74493, or an equivalent narrow replacement, with identity-only token hashing, an auth-epoch version bump, regression tests, and changelog coverage.

## Reproduction Assessment

Yes. Source inspection on current main shows the parser can produce token-shaped Claude credentials, token encoders include `credential.token`, and same-version auth-epoch mismatches reset reusable CLI sessions; no live macOS keychain race was run.

## Solution Assessment

Yes. Identity-only hashing at the auth-epoch layer plus a version bump is the narrowest maintainable fix, and parser hardening can remain a follow-up if maintainers want extra defense in depth.
