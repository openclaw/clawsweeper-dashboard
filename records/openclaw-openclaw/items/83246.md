## Summary

Keep open: the PR targets a real source-backed installer bug, but the current patch is not mergeable because it makes Homebrew setup conditional only on a failed Node check while the macOS Git installer path still uses `brew`. The submitted proof is also mock-only Linux call-order output, not after-fix macOS installer behavior.

## What This Changes

The PR moves `install_homebrew` into the `if ! check_node` branch in `scripts/install.sh` and adds shell-order regression tests in `test/scripts/install-sh.test.ts`.

## Best Possible Solution

Make Homebrew setup conditional on actual macOS brew consumers, covering both `install_node` and `install_git`, then prove the supported-Node/no-Homebrew path and the missing-Node/Git paths with focused tests and real macOS or equivalent installer output.

## Reproduction Assessment

Yes, at source level: current main calls `install_homebrew` before `check_node`, and that function can exit for a missing-brew non-admin macOS account before an adequate Node on PATH is accepted. I did not run a live macOS non-admin installer.

## Solution Assessment

No. Moving `install_homebrew` only into the Node-missing branch misses the macOS Git install path, so the maintainable fix should put the Homebrew prerequisite next to every macOS brew consumer or gate both Node and Git installation paths.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Keep Homebrew setup available for macOS Git installs:** `scripts/install.sh:2831`
  - body: This moves the only `install_homebrew` call under the Node-missing branch, but the default npm path still calls `install_git`, and on macOS that runs `brew install git`. A user who already has a supported Node but lacks both Git and Homebrew now reaches the Git step with no brew available, so the installer can fail in a fresh macOS setup that current main handled. Put the Homebrew prerequisite next to each macOS brew consumer, or gate it before the Git install too.
  - confidence: 0.92
