## Best Possible Solution

Define a maintainer-approved, default-off SSRF policy model that composes global defaults with subsystem overrides, explicitly covers Teams/provider auth paths, preserves host-scoped fake-IP safeguards, and lands with docs plus focused security tests.

## Solution Assessment

Unclear until maintainer/security review: a gateway default may be the cleanest answer for recurring fake-IP gaps, but narrower per-surface hooks may be safer if broad inheritance is too risky.
