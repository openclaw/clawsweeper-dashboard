## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Restore existing Gateway and protocol surfaces:** `src/gateway/server-methods.ts:77`
  - body: Adding `memoryHandlers` should be additive, but this diff also drops existing plugin/session/node protocol surfaces such as `plugins.uiDescriptors`, `sessions.pluginPatch`, and node presence/result schemas. Those exist on current main and are used by Control UI/plugin/node clients, so keep the memory RPC additions without removing unrelated methods or generated models.
  - confidence: 0.9
- **[P1] Guard pathless search hits before sorting:** `src/gateway/server-methods/memory.ts:442-444`
  - body: `decorateSearchHit` can leave `path` undefined, but equal-score sorting calls `left.path.localeCompare(right.path)`. A pathless hit will throw and fail `memory.search.debug`; normalize both sides before comparing.
  - confidence: 0.93
- **[P2] Bound the index job registry:** `src/gateway/server-methods/memory.ts:52`
  - body: `indexJobs` is process-global and receives entries from the new index RPC without any TTL, pruning, or size cap. A long-running Gateway can retain completed or failed jobs indefinitely, so evict old jobs or cap the map like the `sourceRefs` TTL path.
  - confidence: 0.88
- **[P2] Keep the memory host SDK self-contained:** `packages/memory-host-sdk/src/host/memory-schema.ts:2`
  - body: The package-local memory schema now imports `../../../../src/infra/errors.js`, which makes published package consumers depend on the repository root `src` tree. Keep the local helper or expose a supported package boundary instead.
  - confidence: 0.84
- **[P2] Preserve confirmation for new sessions:** `ui/src/ui/app-render.ts:2369`
  - body: The chat New Session button now calls `handleSendChat('/new', { restoreDraft: true })`, omitting `confirmReset`. Because the confirmation guard only runs when that option is set, clicking the button can reset the current chat without the existing confirmation prompt.
  - confidence: 0.86
