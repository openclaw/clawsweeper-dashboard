  - confidence: 0.95

  - confidence: 0.9

  - body: This table uses lower numeric priority as the winner, but the PR adds `system: 4`, below config/workspace/global/bundled. A system channel plugin that is supposed to be highest precedence can therefore lose catalog ownership to a user or bundled entry, leaving the new origin's precedence inconsistent across discovery, channel catalog, and schema metadata.
  - confidence: 0.86
