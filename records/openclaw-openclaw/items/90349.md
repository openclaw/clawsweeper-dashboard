## Summary

Keep open. Current main has context-window detection, pre-prompt compaction, skill prompt limits, tool profiles, context-limit knobs, and opt-in localModelLean/Tool Search mitigations, but it still does not provide the requested automatic relevance-based prompt, skill, memory, and tool compactor for small-context models.

## What This Changes

The issue requests an intelligent context budget/compactor that detects model limits and automatically slims prompts, skills, memory, and tools for smaller-context models.

## Best Possible Solution

Design a maintainer-approved context-budget policy that composes the existing budget knobs and lean-mode/tool-search work before adding any new compactness levels or defaults.
