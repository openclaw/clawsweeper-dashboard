## Summary

Keep open. Current main and v2026.6.8 have several partial mitigations for small-context and prompt-pressure problems, but they still do not provide the requested automatic relevance-based prompt, skill, memory, and tool compactor; the open related regression report at https://github.com/openclaw/openclaw/issues/92451 also shows the remaining problem needs maintainer product direction rather than cleanup closure.

## What This Changes

Requests a context-budget compactor that detects model context limits, allocates prompt budget, summarizes or trims context, narrows skills and memory, and exposes compactness levels for small-context models.

## Best Possible Solution

Design a maintainer-approved context-budget policy that composes existing context-window, compaction, context-engine, Tool Search, skills, memory, and localModelLean surfaces before adding compactness modes or defaults.
