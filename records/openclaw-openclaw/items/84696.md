## Best Possible Solution

Review and land a focused fix such as https://github.com/openclaw/openclaw/pull/84911, or an equivalent narrow patch, that canonicalizes in-root symlink parents before host `write`/`edit` fs-safe writes while preserving outside-root rejection and regression coverage.

## Solution Assessment
