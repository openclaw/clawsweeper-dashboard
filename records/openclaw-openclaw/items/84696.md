## What This Changes

The issue reports intermittent host `write` failures when the target path traverses an in-workspace directory symlink.

## Best Possible Solution

Land a narrow host workspace-only write/edit fix that canonicalizes existing in-workspace symlink parents before pinned writes, preserves outside-workspace symlink rejection, and adds focused regressions for both cases.
