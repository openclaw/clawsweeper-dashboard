## Summary

Keep open: the requested UI feature is not implemented on current main, and this branch still has concrete correctness blockers plus missing real behavior proof before merge.

## What This Changes

This PR adds a Control UI Agents overview global Active Primary Model selector wired to `agents.defaults.model`, extracts model-state/update helpers, and adds focused helper tests.

## Best Possible Solution

Decide whether OpenClaw wants a separate global default selector beyond the default-agent editor fix, then rebase it into `agents-panels-overview.ts` with a defaults-only handler that preserves the complete `AgentModelSchema` object and leaves per-agent overrides intact.

## Reproduction Assessment

Yes for the review findings. Source inspection on current main plus the PR diff shows the selector is added to the obsolete overview path, and the `timeoutMs` loss follows directly from `AgentModelSchema` versus `buildPrimaryModelConfig`.

## Solution Assessment

No. The feature direction may be useful, but this branch should be rebased into the current split overview, reconciled with https://github.com/openclaw/openclaw/pull/40052, preserve the complete model object contract, and include real UI proof before merge.

Overall correctness: patch is incorrect

- **[P2] Move the selector into the split overview component:** `ui/src/ui/views/agents.ts:470-483`
  - body: Current main renders the visible Agents overview from `agents-panels-overview.ts`, but this PR adds the global selector to the older in-file overview body in `agents.ts`. After rebasing, the displayed overview would still only call `onModelChange(agent.id, ...)`, so the defaults-only selector would not appear on the current UI path.
  - confidence: 0.92

  - body: `AgentModelSchema` accepts model objects with `timeoutMs`, but this helper rebuilds object configs as only a string or `{ primary, fallbacks }`. Using it for the new defaults writer would silently drop configured model timeouts when the user changes the global primary model.
  - confidence: 0.9
