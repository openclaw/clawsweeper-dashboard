  - body: The PR converts HTTPS_PROXY/HTTP_PROXY into explicit Discord REST and gateway proxies.   - body: The OAuth helper enables trusted_env_proxy whenever any proxy env var exists, including ALL_PROXY-only environments, and it does not encode NO_PROXY/target matching at the call site.   - confidence: 0.88

  - body: normalizePubkey now returns lowercased input for malformed non-npub values. Current callers and tests rely on this helper rejecting invalid sender/allowlist IDs early, so bad keys can reach cryptographic and routing paths before failing unpredictably.
  - confidence: 0.92
