Correct the first-class Ollama Cloud provider hostname so direct cloud models can reach the provider API.

## Best Possible Solution

Keep the shipped canonical `https://ollama.com` endpoint and direct affected v2026.6.5 users to upgrade to v2026.6.6 or newer.
