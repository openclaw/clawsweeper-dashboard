## Best Possible Solution

Review and land https://github.com/openclaw/openclaw/pull/92594 if maintainers accept the bounded doctor migration, keeping runtime canonical and repairing retired endpoint config through the Ollama plugin doctor path.

## Solution Assessment

Yes. A plugin-owned doctor migration plus provider-id doctor discovery is the narrow maintainable fix under repository policy; adding a runtime fallback would hide stale config instead of repairing it.
