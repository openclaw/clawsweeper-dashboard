## Best Possible Solution

Verify an expired eligible xAI OAuth profile live; if xAI supports a non-challenged refresh contract, keep the fix in extensions/xai/xai-oauth.ts with focused refresh tests, otherwise document and surface xAI-specific reauth/API-key guidance.
