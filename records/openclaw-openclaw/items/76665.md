Keep open: the report is recent, post-2026.5.2, and includes concrete backup evidence that the session transcript itself lost prior visible turns. The issue reports that consecutive Feishu DM turns using a z.ai-style GLM provider drop prior user and assistant messages from the session file after the second media/DOCX turn.

## Best Possible Solution
