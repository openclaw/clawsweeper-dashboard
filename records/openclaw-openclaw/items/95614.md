## Summary

## What This Changes

The PR adds a memory-wiki helper and regression test so local source re-ingest carries forward an existing human Notes block before rewriting the generated source page.

## Best Possible Solution

Keep the PR open and tighten preservation to the actual `## Notes` human block, with regression coverage for marker comments inside source content.

## Reproduction Assessment

## Solution Assessment

No: the fix belongs in `ingestMemoryWikiSource`, but the new helper is too broad because it matches marker comments inside source content before the Notes section. The best implementation should preserve only the actual `## Notes` human block.
