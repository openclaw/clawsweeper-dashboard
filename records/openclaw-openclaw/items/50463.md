Overall confidence: 0.94

Full review comments:

- **[P1] Use the current Pi Agent API:** `src/agents/pi-embedded-runner/run/attempt.ts:8-14`
  - body: Current main pins `@earendil-works/pi-agent-core@0.74.0`, whose `Agent` exposes `afterToolCall` as an option/property and has no `setAfterToolCall` method. This branch imports the old `@mariozechner` packages and calls the old setter, so it will not compile or compose correctly after rebasing.
  - confidence: 0.96
- **[P2] Route prepare-time missing tools through the hint:** `src/agents/pi-embedded-runner/run/attempt.ts:1940`
  - body: In the current pinned Pi loop, a missing tool returns an immediate `Tool X not found` result from `prepareToolCall`, and `afterToolCall` only runs for prepared/executed tools. Installing the rewrite only through `setAfterToolCall` leaves the main skill-called-as-tool case unchanged on current main.
  - confidence: 0.94
- **[P2] Match only skills visible in this run:** `src/agents/pi-embedded-runner/run/attempt.ts:1956`
  - body: Current main clears the skill snapshot for sandboxed non-rw runs before building prompt/runtime state, but this rewrite checks `params.skillsSnapshot`. A guessed hidden skill name could receive a confirmation hint even though it was not visible to the model in that run.
  - confidence: 0.9
- **[P2] Handle snapshots without resolvedSkills:** `src/agents/pi-embedded-helpers/errors.ts:573-577`
  - body: `resolvedSkills` is optional, and the embedded runner reloads skill entries when it is absent. Returning false when this field is missing can suppress the hint for sessions that still have prompt-visible skills from the lightweight snapshot or reloaded entries.
  - confidence: 0.86
- **[P3] Parse prompt-facing skill names:** `src/agents/pi-embedded-helpers/errors.ts:560-561`
  - body: The unknown-tool parser captures only `[a-z0-9_-]+`, while skills can be rendered with raw names containing spaces or punctuation and users may spell skills with a leading `$`. Those visible skills will miss the new hint even when the tool error names them.
  - confidence: 0.82
