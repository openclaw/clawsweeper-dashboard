## Summary

Keep open. Current main still lacks the session attachment API, and this PR is the active implementation slice, but the latest head still has a normalized session-route lookup bug that can hide an older routable alias behind a newer non-routable alias.

## What This Changes

The PR adds `OpenClawPluginApi.sendSessionAttachment`, host-side bundled-plugin file validation and routing, delivery hints for attachments, Telegram formatting exports, gateway send option plumbing, tests, an API baseline update, and a changelog entry.

## Best Possible Solution

Keep the PR open, repair normalized delivery lookup so routable aliases outrank freshness in every fallback path, add the inverse regression, then re-run the focused session attachment, delivery-info, Telegram, SDK API, lint, and diff checks before normal maintainer review.

## Reproduction Assessment

Yes. Use a session store where the queried/canonical normalized key is absent or newer but non-routable, while an older mixed-case key with the same normalized value has `deliveryContext`; the current normalized index can return the non-routable record and make attachment delivery report no active route.

## Solution Assessment

No. The host-mediated attachment seam is the right narrow direction, but this implementation is not ready until the normalized fallback index preserves routable entries before comparing `updatedAt`.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Prefer routable normalized aliases:** `src/config/sessions/delivery-info.ts:137-138`
  - body: `buildFreshestSessionEntryIndex` keeps only the newest entry for each normalized key. If a newer lowercase/canonical alias lacks delivery metadata while an older mixed-case alias with the same normalized key is routable, the fallback returns the non-routable record and `sendSessionAttachment` reports no active route even though one exists. Rank routable entries before freshness and add that inverse alias regression.
  - confidence: 0.88
