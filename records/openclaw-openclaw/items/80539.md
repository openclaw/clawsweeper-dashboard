## What This Changes

The PR makes managed npm plugin installs retry once without managed `npm:` alias override leaves when npm reports `Invalid comparator: npm:`.

## Best Possible Solution

Land a focused managed npm-root retry after normal PR gates stay green, keeping alias overrides on the default path and using the filtered manifest only for the known npm comparator failure.

## Solution Assessment
