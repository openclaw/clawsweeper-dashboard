- **[P2] Probe trusted-proxy gateways through a trusted source:** `src/cli/gateway-cli/run.ts:562`
  - body: The probe only injects trusted-proxy headers, but gateway auth also checks that the socket remote address is in `gateway.trustedProxies`.   - confidence: 0.86
- **[P2] Bracket IPv6 custom bind hosts before probing:** `src/cli/gateway-cli/run.ts:169`
  - body: For `gateway.bind=custom` with an IPv6 literal such as `::1`, the bind URL helper produces an invalid `ws://::1:<port>` URL. That candidate is filtered out, so an IPv6-only healthy listener falls back to the wrong loopback probe and can still exit non-zero on `EADDRINUSE`.
  - confidence: 0.82
