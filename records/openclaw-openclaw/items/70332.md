- **[P2] Preserve italic diary body lines:** `extensions/memory-core/src/dreaming-narrative.ts:390-392`
  - body: The new fingerprint skips every standalone `*...*` line, not just the generated diary timestamp. If a generated entry contains an italic standalone sentence, that text disappears from the fingerprint, so distinct diary entries can be skipped or removed as duplicates; only exclude the timestamp line and add a regression.
  - confidence: 0.93
- **[P3] Normalize REM theme fragments:** `extensions/memory-core/src/dreaming-narrative.ts:153-155`
  - body: `runRemDreaming` passes reflection strings such as `- Theme: ...` as `themes`, but this helper only strips the dash before writing the fallback diary line. Request-scoped REM fallbacks can still expose raw `Theme:` and backticked metadata; pass plain tags or strip that wrapper and cover the real caller shape.
  - confidence: 0.82
- **[P3] Do not classify user heartbeat text as system automation:** `src/memory-host-sdk/host/session-files.ts:428-433`
  - body: `isHeartbeatPollRecord` treats any user message exactly equal to `[OpenClaw heartbeat poll]` as proof that the whole transcript is system-managed, after which later normal content is skipped. A real user can quote that sentinel; use session-store/runtime-context provenance for whole-transcript classification or drop only the generated heartbeat line.
  - confidence: 0.76
