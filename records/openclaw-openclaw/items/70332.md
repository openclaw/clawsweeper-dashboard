- **[P1] Keep fallback diary output free of raw snippets:** `extensions/memory-core/src/dreaming-narrative.ts:149-150`

- **[P1] Do not let transcript text suppress the whole session:** `src/memory-host-sdk/host/session-files.ts:431-435`
  - body: Setting `generatedBySystemAutomation` from a heartbeat-looking record causes the rest of the transcript to be skipped. A user can type the exact heartbeat prompt/token shape in an ordinary session and suppress later real assistant/user content from memory ingestion; current main avoids this cross-message coupling and sanitizes generated prompt/ack messages individually.
  - confidence: 0.88

  - body: The fingerprint loop now skips every line wrapped in `*...*`, not only the date line. A real diary entry with an italic standalone sentence can collapse to the same fingerprint as another entry, so append/dedupe can silently skip or delete valid diary content.
  - confidence: 0.82
