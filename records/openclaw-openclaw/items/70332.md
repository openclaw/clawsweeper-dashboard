## Summary

Keep this PR open. Current main still lacks the proposed Dream Diary hardening, and the branch contains useful regression coverage, but the diff introduces a real fingerprinting bug that can collapse distinct diary entries when they contain standalone italic prose; the fallback portion also overlaps the still-open #70523 and needs a clean landing plan.

## What This Changes

The PR changes memory-core dreaming narrative/phases code and tests to strip nested or legacy managed daily blocks, render request-scoped fallback snippets as diary prose, dedupe DREAMS.md entries by body, and skip duplicate diary appends.

## Best Possible Solution

Land a revised branch or narrow replacement that keeps the useful diary hardening, computes body-only duplicate fingerprints while excluding only the timestamp line, preserves standalone italic body prose, adds a regression for italic body lines, coordinates fallback behavior with #70523, and validates the memory-core touched surface before merge.

## Reproduction Assessment

Yes. Construct two diary blocks with the same non-italic body but different standalone italic body lines, then run the patched fingerprint/dedupe path; those italic lines are removed from the fingerprint, so the entries can be treated as duplicates.

## Solution Assessment

No, not as currently written. Body-only dedupe and prose fallback are reasonable directions, but the fingerprint implementation should ignore only the timestamp line and keep all diary body content; the fallback overlap with #70523 also needs maintainer coordination.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Preserve italic diary body lines in fingerprints:** `extensions/memory-core/src/dreaming-narrative.ts:392`
  - body: The patch skips every standalone `*...*` line when computing duplicate fingerprints. Dream Diary output is free prose and can include italicized standalone sentences, so two entries that differ only in such a line can produce the same fingerprint and be skipped or deleted by append/dedupe. Exclude only the timestamp line and add a regression for italic body prose.
  - confidence: 0.93
