## What This Changes

The issue requests classifying broad Windows ACL principals such as Anonymous Logon, Guests, Interactive, Local, and Network as world-equivalent in security audits.

## Best Possible Solution

Review and land PR #74383, or an equivalent narrow security patch, after confirming the final broad-principal policy and adding classifier plus filesystem-audit regressions.

Unclear until a security maintainer confirms the final SID/principal set. The narrow classifier update plus regression tests is the right fix boundary, and PR #74383 is already the direct implementation candidate.
