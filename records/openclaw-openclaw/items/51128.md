Full review comments:

- **[P2] Restore mouse modes through shared cleanup:** `src/tui/tui.ts:1111`
  - body: The PR enables `?1000/?1002/?1006` after `tui.start()`, but the reset only runs through `requestExit()`.   - confidence: 0.87
- **[P2] Preserve anchors during in-place transcript growth:** `src/tui/components/chat-log.ts:51-62`
  - body: Scroll compensation only wraps `append()`, while streaming/final assistant text, BTW result updates, and tool result updates mutate existing children. When a user is paged up, those line-count increases keep `scrollOffset` fixed and move the viewport away from the text being read.
  - confidence: 0.85
- **[P3] Document the new transcript controls:** `src/tui/components/custom-editor.ts:42-47`
  - body: Page Up/Page Down and mouse-wheel scrolling become user-facing TUI controls, but the TUI guide still lists only the existing shortcuts. Add docs so users can discover the new transcript navigation.
  - confidence: 0.82
