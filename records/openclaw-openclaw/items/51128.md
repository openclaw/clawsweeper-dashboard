## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Restore mouse modes through shared cleanup:** `src/tui/tui.ts:1111`
  - body: This enables SGR mouse reporting at startup, but the reset only runs inside requestExit. If client.start() or another path exits after the enable without requestExit, the terminal can stay in mouse-reporting mode and leak click/scroll escape sequences after the TUI exits.
  - confidence: 0.86
- **[P2] Preserve anchors during in-place transcript growth:** `src/tui/components/chat-log.ts:51-62`
  - body: Scroll compensation only wraps append(), so streaming updates, final text replacement, BTW/tool result updates, and tool expansion can grow existing components without adjusting scrollOffset. A user paged up can still be pushed away from the text they were reading.
  - confidence: 0.85
- **[P3] Document the new transcript controls:** `src/tui/components/custom-editor.ts:42-47`
  - body: The PR adds Page Up/Page Down and mouse-wheel transcript controls, but the TUI docs still list only the existing shortcuts. Add the user-facing docs entry so the controls are discoverable.
  - confidence: 0.8
