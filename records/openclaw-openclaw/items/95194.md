## What This Changes

The branch adds `resolveOsRuntimeName()` in `src/infra/os-summary.ts`, routes CLI/embedded/compaction runtime prompt OS metadata through it, and adds focused OS-summary and CLI prompt tests.

## Best Possible Solution

Use a shared cached helper that returns macOS product versions only on Darwin while preserving the previous `${os.type()} ${os.release()}` shape elsewhere, then add real macOS/Tahoe prompt proof.
