## Best Possible Solution

Land one cached Darwin product-version helper covering CLI, embedded attempt, and compaction runtime prompt metadata after real macOS/Tahoe proof and duplicate-PR consolidation.

## Reproduction Assessment

## Solution Assessment

Yes on code shape: centralizing the Darwin product-version lookup in `src/infra/os-summary.ts`, preserving non-Darwin strings, and leaving architecture to `buildRuntimeLine()` is the narrow maintainable fix. The remaining blockers are proof and canonical branch selection, not a concrete code defect found in this pass.
