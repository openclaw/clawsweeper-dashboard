## Best Possible Solution

Land a refreshed IRC-owned implementation that keeps the draft/multiline direction, preserves existing IRC-safe command sanitization, splits every outgoing IRC frame within the transport limit, uses concat semantics for long logical multiline lines, and adds focused regression tests.

## Solution Assessment
