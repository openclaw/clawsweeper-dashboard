  - body: The public SDK type uses nested `delivery`, `live`, `receive`, and `native` capability groups, but the config schema only accepts a strict flat shape such as `inbound`, `outbound`, and `threads`.   - confidence: 0.88
- **[P2] Guard broker webhooks before parsing bodies:** `extensions/channel-broker/src/http-routes.ts:80-85`
  - body: This route is registered with plugin auth, so gateway auth is not enforced, but the handler reads up to the post-auth body limit and parses JSON before verifying the broker signature. Use the pre-auth webhook guard/rate-limit pattern before parsing untrusted bodies, then verify the HMAC before dispatch.
  - confidence: 0.78
