- **[P1] Reuse the shared run-wait helpers:** `src/agents/acp-spawn-parent-stream.ts:282-297`
  - body: Current main already centralizes `chat.history` snapshots, assistant fingerprints, and `agent.wait` normalization in `src/agents/run-wait.ts`. Adding a parallel raw gateway parser here creates conflicting semantics for reply filtering, duplicate detection, and terminal statuses after rebase.
  - confidence: 0.86
- **[P1] Keep polling history before terminal completion:** `src/agents/acp-spawn-parent-stream.ts:381-383`
  - body: `maybeFinalizeFromGateway` does one forced history read and then emits completion/disposes. If `agent.wait` observes `ok` before `chat.history` has caught up, the parent still receives `run completed` without the child assistant text, preserving the missing-output case this fallback is meant to repair.
  - confidence: 0.84
- **[P1] Guard local deltas after history relay:** `src/agents/acp-spawn-parent-stream.ts:453-459`
  - body: If history fallback relays the full assistant reply before a late local assistant event arrives, the local delta path can append overlapping content again. Track history-relayed text or suppress overlapping local deltas so the parent does not receive duplicate child progress.
  - confidence: 0.82
- **[P2] Set local-assistant flag only after relaying text:** `src/agents/acp-spawn-parent-stream.ts:453-459`
  - body: This marks `sawLocalAssistantEvent` before validating that the assistant payload produced relayable text. A whitespace or otherwise empty local assistant event can suppress both polling and forced history backfill even though the parent has not received child output.
  - confidence: 0.88
- **[P2] Recheck the history baseline after awaiting:** `src/agents/acp-spawn-parent-stream.ts:304-309`
  - body: `primeHistoryBaseline` can race the first fallback poll: the poll may initialize and relay a newer history reply, then the slower priming read can overwrite `lastHistoryAssistant` with an older value. Rechecking `historyBaselineReady` after the await prevents duplicate replay on the next poll.
  - confidence: 0.9
