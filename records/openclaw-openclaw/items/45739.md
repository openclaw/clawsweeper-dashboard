  - confidence: 0.87
- **[P2] Deduplicate delayed local deltas after history backfill:** `src/agents/acp-spawn-parent-stream.ts:459`
  - body: If history fallback emits the full assistant text before delayed local assistant deltas arrive, the local path can append overlapping content again because it does not account for already-relayed history output.
  - confidence: 0.78

  - body: After `agent.wait` returns ok, this performs one `chat.history` read and then disposes; if history persistence lags run completion, the fallback can still emit completion without the child output it was meant to recover.
  - confidence: 0.72
