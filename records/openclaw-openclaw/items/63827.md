Keep this PR open, but do not merge it as-is: the dmPolicy bug is still present on current main, while this branch is stale against the refactored channel setup flow, has a compile-time scoping bug, and bundles unrelated security advisory and supply-chain changes.

## What This Changes

The branch tries to preserve dmPolicy as the initial wizard selection and also bundles gateway, channel, dependency, test, changelog, and security-advisory changes.

## Best Possible Solution
