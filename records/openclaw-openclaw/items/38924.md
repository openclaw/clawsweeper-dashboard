## Summary

Keep open. Current main does not contain the AAMA implementation, and the PR head still has blocking security/gateway protocol defects, red protocol/secrets checks, and no real behavior proof for the new enforcement path.

This PR adds AAMA Phase 1 approval-token enforcement across outbound sends, gateway send/poll requests, session-memory writes, durable SQLite state/audit logging, protocol schema fields, fixtures, and tests.

## Best Possible Solution

  - body: A missing approvalToken or approvalNonce now calls suspendAutonomy and persists the global suspended flag.   - confidence: 0.88
