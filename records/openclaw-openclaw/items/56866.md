  - body: The new helper allowlists the raw path string and then reads it directly, so a path under an allowed root that traverses a symlinked directory can still reach files outside that root. Current main routes ACP attachments through the shared media cache/root policy instead; this branch should not reintroduce direct reads here.
  - confidence: 0.86

  - body: This serializes `entry.mediaPath` into the prompt, exposing local filesystem layout to the ACP backend and prompt hooks. Use attachment metadata or a redacted media note instead of raw local paths.
