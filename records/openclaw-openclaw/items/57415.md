## What This Changes

The PR adds an `openclaw agent --cwd <dir>` CLI option, forwards it through local and gateway agent runs as `workspaceDir`, updates the agent protocol surface and generated Swift model, and adds focused tests.

## Best Possible Solution

Rebase the branch, keep the CLI `--cwd` path, normalize or reject gateway `workspaceDir` at ingress with an explicit absolute-path contract, update current generated clients and docs, and add redacted real behavior proof for both local and gateway runs.

## Solution Assessment
