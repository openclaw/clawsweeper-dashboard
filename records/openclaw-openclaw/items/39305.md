## Summary

Keep open. ## What This Changes

The issue requests configurable two-stage subagent stall recovery based on tool-call inactivity, with a nudge before forced kill and per-spawn overrides.

## Best Possible Solution

Settle the shared session/subagent activity model first, then add an accepted opt-in subagent nudge/kill policy through existing steer/kill and parent-notification paths with schema, docs, and focused tests.
