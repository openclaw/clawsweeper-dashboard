Keep open. ## What This Changes

The issue requests optional two-stage sub-agent stall recovery that nudges a child agent after tool-call inactivity and kills it after a later timeout, with defaults and per-spawn overrides.

## Best Possible Solution

Settle the shared session/subagent activity model first, then add an accepted opt-in sub-agent nudge/kill policy through existing steer/kill and parent-notification paths with schema, docs, and focused tests.
