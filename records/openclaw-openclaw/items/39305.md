Keep open. Feature request for per-subagent inactivity-based stall recovery with a nudge phase followed by auto-kill and parent notification.

## Best Possible Solution

Settle the shared session/subagent activity model first, then add an accepted opt-in subagent nudge/kill policy through existing steer/kill and parent-notification paths with schema, docs, and focused tests.
