  - confidence: 0.88

  - body: The fingerprint is saved only after the runner and post-health probe finish, so another invocation during a long rescue can start a duplicate capture, notification, and remediation attempt for the same outage.
  - confidence: 0.89

  - body: load_state sources state.env from an environment-configurable state directory without regular-file, owner, permission, or symlink checks, so replacing that file could execute shell code under the operator account on the next watchdog run.
  - confidence: 0.84
