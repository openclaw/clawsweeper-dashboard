## Security Review

Status: needs_attention

Summary: The diff adds command execution and diagnostic capture, and the service default plus state-file and incident-bundle handling need hardening before merge.

Concerns:

- **[medium] Keep Claude handoff disabled by default:** `scripts/systemd/openclaw-rescue-watchdog.service:9`
  - body: The systemd unit enables OPENCLAW_RESCUE_RUNNER by default, so enabling the sample timer can run Claude Code in acceptEdits mode on captured incidents without a separate explicit runner opt-in.
