  - body: `notifyReloadFailure` reads `sessionKey` and delivery context from the cached `lastWriteEvent` for every invalid snapshot.   - confidence: 0.91

  - confidence: 0.88

  - body: `docs/PR-PROPOSAL.md` is a draft design note with non-English content and a rebase-before-submit note. It should live in the PR discussion or be replaced with final English user documentation for a shipped command.
  - confidence: 0.86
