  - body: Current main documents invalid direct edits as failing startup or being skipped by hot reload, with whole-file recovery owned by `openclaw doctor --fix`.   - body: When `--from` selects an invalid candidate, this branch still returns `selectedSource` and `restoredPath`. `runConfigRecover` uses `selectedSource` as the dry-run success signal, so it can report that it would recover from an unusable backup and exit successfully.
