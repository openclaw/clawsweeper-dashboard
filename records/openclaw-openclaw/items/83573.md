  - body: When `GET /channels/<id>` is forbidden, this fallback treats any ordinary `channel_name` as public `O`. Existing Mattermost auth maps private `P` channels to `group`, so `/oc_model` dialog submit can write the model override into a different route/session than messages in the same private channel; carry a reliable channel kind, preserve the private-channel snapshot, or keep failing closed until the channel type is known.
  - confidence: 0.84

  - body: The new slash path opens the native dialog whenever `trigger_id` exists, but the provider select depends on field refresh through `refresh: true` and `source_url`. Mattermost documents dynamic field refresh as Server 11.1+, so older servers can lose the working message-picker fallback and see stale model/runtime fields; gate this by capability/version or keep the old picker for unsupported servers.
