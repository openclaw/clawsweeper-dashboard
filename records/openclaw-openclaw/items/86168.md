## Summary

This is a useful focused edit-tool improvement, but it is not merge-ready: the proposed matcher misses common near-match cases and reports the context window as the match line, and the external PR still lacks real after-fix behavior proof.

## What This Changes

The PR changes host edit mismatch errors to show a best-matching line-numbered region instead of always dumping the first 800 characters of the current file.

## Best Possible Solution

Keep the PR open, repair the matcher to score near/partial line matches and report the true matched line, add focused regression coverage, and require a redacted real edit-mismatch proof before merge.

## Reproduction Assessment

Yes. Source inspection of the PR head shows a single-line near miss or first-line divergence scores zero because `countSharedLines` requires exact leading line equality, and the heading line is derived from context start rather than match start.

## Solution Assessment

No. The affected wrapper is the right place, but the implementation needs a more tolerant region score and a correct match anchor before it is the maintainable solution.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.87

Full review comments:

- **[P2] Score partial line matches before falling back:** `src/agents/pi-tools.host-edit.ts:175`
  - body: The new matcher only counts exact shared leading whole lines. For a common one-line edit typo, or any oldText whose first line diverges from the current file, every window scores zero and the code falls back to `Current file contents`, so the PR does not show the best matching region for the central mismatch case it is meant to fix.
  - confidence: 0.89
- **[P2] Report the matched line instead of the context line:** `src/agents/pi-tools.host-edit.ts:197`
  - body: `startLine` is returned as `ctxBefore + 1`, but `ctxBefore` intentionally includes two lines of context above the match. When the best match is below the top of the file, the heading points to the context start rather than the actual matched region, which can misdirect the next edit attempt.
  - confidence: 0.92
