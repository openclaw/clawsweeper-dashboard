## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Report the matched line instead of the context line:** `src/agents/pi-tools.host-edit.ts:197`
  - body: `startLine` is currently derived from `ctxBefore`, so whenever the two context lines are present the hint says the best match starts above the actual matched oldText. That makes the new recovery hint point the model at the wrong anchor; keep the context lines in the snippet, but return `bestStart + 1` for the headline.
  - confidence: 0.93
- **[P2] Score a partial match when oldText is longer than the file:** `src/agents/pi-tools.host-edit.ts:173`
  - body: When `oldText` has more lines than the file, `contentLines.length - windowSize` is negative and the loop never scores any candidate. A file that shares the beginning of the requested block then falls back to the raw file-head hint, so the new recovery behavior disappears for extra-line mismatches.
  - confidence: 0.86
