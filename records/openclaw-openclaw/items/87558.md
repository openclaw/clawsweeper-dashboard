## Summary

Keep open for maintainer review. The patch appears technically coherent, but this PR is maintainer-owned/protected and intentionally changes public stream semantics, so the remaining compatibility and exact platform-proof decisions should stay with maintainers rather than cleanup automation.

## What This Changes

The branch adds a shared assistant stream accumulator, changes Ollama/gateway/proxy/subscriber streams toward delta-first lightweight partials, propagates replacement deltas through SDK/UI, documents the SDK contract, and adds tests plus a live dense-stream latency repro script.

## Best Possible Solution

Land only after maintainers explicitly accept the lightweight-partial stream contract and decide whether the non-Windows proof is enough; otherwise preserve full partials as the default or collect exact Windows/WSL2 dense-Ollama gateway proof first.

## Solution Assessment
