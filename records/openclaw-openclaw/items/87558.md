Full review comments:

- **[P1] Preserve media-only stream emissions:** `src/agents/embedded-agent-subscribe.handlers.messages.ts:595`
  - body: The new unphased path now builds `cleanedText` from the parsed delta text, so a finalized MEDIA/audio directive can have `parsedStreamDirectives` with media but `next === ""`. Because the emission logic is still inside `if (next)`, those media-only assistant updates never reach `hasMedia`/`hasAudio` and can be dropped before channel delivery.
  - confidence: 0.88
- **[P2] Stop the ping loop on every harness exit:** `scripts/repro-gateway-dense-stream-latency.ts:510-529`
  - body: `pingHealthLoop` starts before `chat.send` and `agent.wait`, but `stopFlag.done` is only set after both calls succeed. If either request throws or times out, cleanup disconnects the clients while the ping loop keeps scheduling work, so the opt-in proof command can hang instead of exiting with the useful failure.
  - confidence: 0.86
- **[P2] Update the Plugin SDK API baseline:** `src/plugin-sdk/provider-stream-shared.ts:10-15`
  - body: This re-export adds a new public helper and types under `openclaw/plugin-sdk/provider-stream-shared`, but the tracked Plugin SDK API baseline hash is unchanged and the subpath docs still list the old helper set. Either keep the helper internal or update the SDK baseline/docs/contract proof so the public API change is intentional and checkable.
  - confidence: 0.82
