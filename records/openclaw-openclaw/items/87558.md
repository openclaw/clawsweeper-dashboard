This draft PR adds an opt-in dense stream gateway latency harness, cooperative Ollama NDJSON parse yields, and an incremental unphased assistant text streaming path.

## Best Possible Solution

## Solution Assessment

No as written. The cooperative yield and incremental-delta direction is reasonable, but the subscriber path must preserve metadata-only delivery and the harness must stop its ping loop on all exits before this is the best maintainable fix.
