## Review Findings

Full review comments:

- **[P2] Preserve constraints beside resolved $ref:** `src/agents/pi-tools-parameter-schema.ts:361-363`
  - body: When a local `$ref` resolves, this branch replaces the referring schema with the referenced object and only copies `title`, `description`, and `default`. Merge those siblings into the resolved schema, or compose with `allOf`, and add a regression for validation siblings.
