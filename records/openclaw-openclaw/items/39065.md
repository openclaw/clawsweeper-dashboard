  - body: IRC and BlueBubbles add nullable-reply guards, but these call sites still call `buildPairingReply` without the configured mode.   - confidence: 0.96

  - body: The Discord component path applies `unpairedResponse` only when a pairing row is newly created, then falls back to the hardcoded “Pairing already requested” reply for pending users.   - confidence: 0.95
