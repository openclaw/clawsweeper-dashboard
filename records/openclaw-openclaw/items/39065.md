- **[P1] Preserve the string-returning pairing formatter:** `src/pairing/pairing-messages.ts:10-11`
  - body: `buildPairingReply` is exported through plugin runtime and SDK surfaces, and this change makes it return `null` for silent mode.   - confidence: 0.9
- **[P1] Wire advertised config through every owning schema:** `src/config/types.discord.ts:224-225`
  - body: This PR advertises `unpairedResponse` in public type/help surfaces, but the owning strict schemas and runtime paths are not aligned for all advertised channels. For example Discord and Slack config schemas do not accept the new key, LINE and plugin-local schemas have separate ownership, and IRC/BlueBubbles paths still do not pass a mode, so operators can hit startup validation failures or unchanged branded replies.
  - confidence: 0.88
- **[P1] Refresh Discord changes onto plugin-owned paths:** `src/discord/monitor/native-command.ts:1369-1396`
  - body: Current main no longer has `src/discord`; Discord runtime is owned under `extensions/discord`. This branch is also live-conflicting, so the changed `src/discord/monitor/*` paths cannot provide the intended Discord behavior after a real rebase and need to be moved to the current plugin files.
  - confidence: 0.93
- **[P2] Suppress repeat component replies in silent mode:** `src/discord/monitor/agent-components.ts:533-550`
  - body: The component path only calls `buildPairingReply(..., mode)` when a pairing request is newly created, then sends the hardcoded `Pairing already requested` response for existing pending requests. With `unpairedResponse: "silent"`, repeat attempts still receive a reply, which breaks the promised silent posture.
  - confidence: 0.86
- **[P3] Record outbound activity only after a send:** `extensions/nextcloud-talk/src/inbound.ts:187-193`
  - body: In Nextcloud Talk, `statusSink` is updated even when `replyText` is falsy and no message was sent. Silent mode should not move `lastOutboundAt`, or health/status tracking will report outbound activity that did not happen.
  - confidence: 0.78
