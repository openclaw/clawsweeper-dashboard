## Summary

Keep open: the reported doctor false-positive is valid, but this patch is too broad because it treats a slot name as active ownership even when the plugin entry/config is missing, and the PR body still lacks concrete after-fix real behavior proof.

## What This Changes

The branch suppresses the doctor missing-memory-plugin note for non-default memory slot owners, adds two doctor-memory unit tests, and adds a changelog entry.

## Best Possible Solution

Recognize a valid configured memory-lancedb slot without masking invalid config, preferably through runtime/capability registration or an explicit active-ownership check, with focused regression coverage and redacted real doctor output.

## Reproduction Assessment

Yes, at source level: current main warns when resolveActiveMemoryBackendConfig returns null, while memory-lancedb is documented as an active slot plugin and current main does not register the memory runtime this doctor path expects.

## Solution Assessment

No. The suppression target is valid, but this implementation treats slot text as enough proof of an active alternate memory plugin; require explicit configured or registry-confirmed active ownership, or pursue the runtime-registration approach in https://github.com/openclaw/openclaw/pull/80211.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Require active ownership before suppressing:** `src/commands/doctor-memory-search.ts:323`
  - body: This treats a slot-only alternate memory selection as healthy because `plugins.entries[memorySlot]?.enabled` is `undefined` when the entry is missing, so `undefined !== false` returns true. With `plugins.slots.memory = "memory-lancedb"` but no valid configured entry, memory-lancedb registers only its warning service and no memory tools/hooks, and doctor would now hide the missing-runtime note for an unavailable memory plugin. Require an explicit configured entry or registry-confirmed active ownership before returning here.
  - confidence: 0.87
