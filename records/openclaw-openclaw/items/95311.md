## Summary

## What This Changes

The PR adds a `disableBoundaryAwareCache` OpenAI-completions compat option, auto-enables it for DeepSeek/Xiaomi, preserves the cache boundary in the transport path, and moves the dynamic suffix into the first user message in the converter.

## Best Possible Solution

## Solution Assessment
