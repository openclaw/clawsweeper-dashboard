## Summary

Keep this PR open: the keepalive part is dependency-supported and the related Telegram reliability context is useful, but the status change can hide a hung reconnect after startup grace and the external PR still lacks artifact-backed real behavior proof.

## What This Changes

The PR enables Telegram undici TCP keepalive, changes polling-start status patches, updates Telegram polling tests, and raises the shared gateway channel connect grace from 120s to 300s.

## Best Possible Solution

Keep the TCP keepalive direction, but preserve an explicit startup/reconnect liveness signal so a polling session that hangs before a fresh `getUpdates` success becomes unhealthy after the intended grace window.

## Reproduction Assessment

Yes for the PR-introduced regression path: source inspection shows patch-merged status can retain `connected:true`, clear transport activity, and then pass health evaluation as healthy after grace. The original production NAT-stall failure was not independently reproduced in this read-only review.

## Solution Assessment

No as written. The keepalive delay is supported by undici, but removing `connected:false` from polling start needs a replacement lifecycle signal or health-policy path for startup/reconnects that never reach successful `getUpdates`.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P1] Preserve startup liveness after reconnects:** `extensions/telegram/src/polling-status.ts:18-23`
  - body: `setRuntime` patch-merges account status, so after a previously connected Telegram account restarts this patch preserves `connected:true` while clearing `lastTransportActivityAt`. Once startup grace expires, `evaluateChannelHealth` skips stale-socket checks when that timestamp is null and returns healthy, so a reconnect that hangs before the next successful `getUpdates` can stay ready forever. Keep an explicit connecting/disconnected/lifecycle liveness signal before dropping `connected:false` here.
  - confidence: 0.91
