Adds an ANSI-aware `truncateToVisibleWidth` helper, uses it when padding terminal table cells, and adds ANSI/table regression tests for narrow cells containing wide graphemes.

## Solution Assessment

Yes. Clamping in `padCell` is the narrowest maintainable fix because it is the last table-rendering step before borders are emitted, and the helper preserves ANSI resets while enforcing the visible-width invariant.
