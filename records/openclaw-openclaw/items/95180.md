## Summary

## What This Changes

The PR adds an `ingest: true` bypass before Telegram's unaddressed `requireMention` media skip and adds one regression test for successful unmentioned media download.

## Best Possible Solution

Land a narrow Telegram fix that mirrors text-path ingest precedence, carries an ingest-bypass signal into both media failure paths, and includes focused tests plus redacted live Telegram proof.

## Reproduction Assessment

Yes, at source level. Current main can return from the Telegram media pre-download skip before any ingest check, while the sibling text path already honors topic, group, and default ingest.

## Solution Assessment

No. The bypass is in the right helper, but the best fix also needs to carry an ingest-bypass signal into downstream media error handling so passive unaddressed media failures do not produce visible group replies.
