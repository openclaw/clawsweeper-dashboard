This PR adds `agents.list[].contextInjection`, passes agent ids into embedded run and compaction context-injection resolution, and updates schema help, tests, and changelog.

## Best Possible Solution

Land a complete per-agent `contextInjection` implementation that aligns the exported config type, Zod schema, generated config schema, docs, runtime call sites, tests, and changelog while preserving the defaults fallback.

Yes.
