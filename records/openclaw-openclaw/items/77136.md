Keep open: current main and v2026.6.8 still have the source-proven WebChat bounded-history gap, and the likely fix remains the open maintainer-labeled pagination PR rather than merged code. ## What This Changes

The issue requests that WebChat render assistant history reliably for long sessions where transcript data is intact and TUI still shows the messages.
