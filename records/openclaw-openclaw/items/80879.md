## What This Changes

The issue reports that blocked Atlas subagent terminal states with context-overflow metadata are announced to the parent as successful completions with no useful output or error.

## Best Possible Solution

Review and land https://github.com/openclaw/openclaw/pull/80886, or an equivalent narrow patch, so blocked terminal metadata is normalized before parent completion synthesis and covered by regressions.
