  - body: The replacement helper scans the whole transcript branch for any matching assistant MEDIA directive. Emitting a message-bearing update immediately afterward can make SSE history clients refresh to the rewritten branch and then append the same replacement again, so rely on the refresh or add sequencing that prevents a second append.
