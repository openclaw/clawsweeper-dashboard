The PR adds save-time rewriting of persisted `sessionFile` and `transcriptPath` values, types `transcriptPath` on `SessionEntry`, and adds a pruning integration test.

Land a corrected save-time canonicalization that derives persisted transcript paths from the active sessions directory plus a validated session id, with structural old-root regression coverage.
