## Best Possible Solution

Land a corrected save-time canonicalization patch that derives saved paths from the active `sessionsDir` and a validated session id, with structural old-root regression coverage.

## Reproduction Assessment

## Solution Assessment

No as written. Save-time canonicalization is the right boundary, but it should build the saved target from the active `sessionsDir` and validated `sessionId`, not from the compatibility resolver.
