  - body: When `includeFamily` is true, this loop pushes ancestor reset archives and active files until `MAX_CHAT_HISTORY_FAMILY_READ_TARGETS` is reached, then returns immediately. Reserve a slot for the current session target before capping ancestors, and cover the long-lineage case.
  - confidence: 0.88
