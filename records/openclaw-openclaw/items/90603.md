## Summary

The PR updates secrets auth-profile and models.json path discovery to use the configured default agent id and adds regression coverage for non-main default agents.

## Best Possible Solution

## Solution Assessment

No. The helper-level direction is right, but the default seed should share the custom-dir-aware path decision instead of adding a parallel stateDir path for a custom default agent.
