## What This Changes

## Best Possible Solution

Review and land a refreshed current-main fix, or a narrow replacement for https://github.com/openclaw/openclaw/pull/91015, that bounds Docker control-plane waits, preserves sandbox workspace behavior, documents any config/default surface, and proves wedged-Docker behavior with redacted real evidence.

## Solution Assessment

No final solution is on main yet. The linked PR is a plausible fix direction, but it still needs current-main review/refresh or replacement plus maintainer judgment on the timeout and fail-soft boundary.
