## Repair Work Prompt

Fix the in-place git update/source build failure class from this issue. Current main lets ignored `vendor/a2ui/renderers/lit/dist/` output survive the updater clean check, then `pnpm build` starts `canvas:a2ui:bundle` and runs renderer `tsgo` in the same checkout. Add a narrow cleanup or fail-fast guard before the renderer compile in `scripts/bundle-a2ui.mjs`, add regression coverage in `test/scripts/bundle-a2ui.test.ts`, and touch updater docs or diagnostics only if the implementation changes user-facing recovery behavior. Do not treat all ignored files as dirty, do not rewrite the global package-manager update flow, and do not edit vendor sources beyond the generated-output boundary. Add a single-line `CHANGELOG.md` fix entry if behavior changes.
