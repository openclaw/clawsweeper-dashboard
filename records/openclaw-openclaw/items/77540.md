  - confidence: 0.94

  - confidence: 0.92

  - body: Plugin metadata snapshots are treated as process-stable until an explicit lifecycle event, but this adds a 60-second expiry to manifest model-id policy caching. Keep reuse fingerprint/lifecycle-based so hot-path normalization does not periodically recollect plugin policy.
  - confidence: 0.86
