## What This Changes

Request per-Telegram-group session reset configuration so selected groups can use short idle resets while other groups keep normal session continuity.

## Best Possible Solution

Define an additive, account-aware scoped reset policy with explicit precedence, then align Telegram config types, strict schemas, generated metadata/help, public docs, and focused runtime tests.

## Solution Assessment
