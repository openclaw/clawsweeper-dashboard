## Best Possible Solution

Land the narrow alias and fingerprint fix only after attaching redacted exact-head runtime proof that `code_mode_exec` PreToolUse reaches OpenClaw `before_tool_call`; keep the fail-closed delivery guard as the separate path if Codex hook delivery can still be absent.

## Solution Assessment

It is not a substitute for the missing-delivery fail-closed work tracked by https://github.com/openclaw/openclaw/pull/90805.
