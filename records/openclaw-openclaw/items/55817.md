## Summary

Keep open. The PR targets a real session durability gap on current main, but the branch is not merge-ready because it writes the new header non-exclusively, still imports the old pi-coding-agent package name, is conflicting, and lacks after-fix real behavior proof.

## What This Changes

The PR adds an optional transcript-header bootstrap to session-file resolution, enables it from session initialization, and adds reset/rollover regression tests.
