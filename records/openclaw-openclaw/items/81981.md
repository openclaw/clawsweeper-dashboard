- **[P1] Align omitted bind evidence with the runtime default:** `extensions/policy/src/policy-state.ts:339-344`
  - body: `gateway.bind` is optional, and current startup resolves an omitted bind through `defaultGatewayBindMode()` while the docs call loopback the normal default.   - confidence: 0.88

  - body: Current gateway calls only use `gateway.remote.url` when `gateway.mode === "remote"`, but this branch reports any stored URL as `gateway-remote-enabled`. Local-mode configs that keep a remote URL for setup or client workflows will fail `gateway.remote.allow=false` even though the local Gateway is not in remote mode.
  - confidence: 0.82
