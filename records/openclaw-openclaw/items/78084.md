## Summary

Keep this PR open: current main still has the exact-match plugin tool contract limitation, and the PR is a focused attempt at the linked dynamic-tool problem, but it needs a correctness fix for explicit allowlist filtering plus external-contributor real behavior proof before merge.

## What This Changes

The PR adds terminal wildcard matching for `contracts.tools`, wires it into plugin registration and runtime tool resolution, adds regression tests, and updates manifest docs.

## Best Possible Solution

Land a maintainer-approved wildcard ownership contract that uses the wildcard to find the owning plugin while preserving exact explicit allowlist filtering, with real plugin or Gateway proof; that can then close https://github.com/openclaw/openclaw/issues/77800 and likely supersede https://github.com/openclaw/openclaw/issues/77797.

## Reproduction Assessment

Yes, at source level. Current main requires exact `contracts.tools` ownership and the resolver only exact-matches manifest tool names against explicit allowlists, so wildcard-owned dynamic tools remain unsupported without this kind of change.

## Solution Assessment

No, not yet. The direction is plausible, but the current patch should preserve exact allowlist filtering for factory-returned tools and still needs maintainer acceptance of the new manifest semantics plus real behavior proof.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Filter wildcard results by the requested tool name:** `src/plugins/tools.ts:1114-1116`
  - body: When an explicit allowlist requests `dynamic_alpha`, the patch selects the `dynamic_*` contract so the plugin can load, but this condition then lets any factory-returned tool matching `dynamic_*` through, such as `dynamic_beta`, even though it was not allowlisted. Keep the wildcard for ownership/plugin selection, but filter returned tools against the original requested tool names unless the whole plugin/group/default set was selected.
  - confidence: 0.9
