## What This Changes

Add a native Plugin SDK `registerMcpServer` API so code plugins can register managed MCP servers and have OpenClaw wire them into embedded OpenAI/Pi-backed runs.

## Best Possible Solution

Keep this open for a maintainer-sponsored design or PR that defines a narrow additive SDK seam reusing existing MCP validation, precedence, workspace scoping, tool policy, docs, and tests, or explicitly decides that bundle/user MCP paths remain the only supported route.
