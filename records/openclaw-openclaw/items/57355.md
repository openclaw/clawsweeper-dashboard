Add a native Plugin SDK `registerMcpServer` API so plugins can register managed MCP servers and project dynamic MCP-backed capabilities into embedded OpenAI/Pi runs.

## Best Possible Solution

Land a maintainer-approved additive SDK/runtime contract that validates managed stdio registrations, records plugin ownership, preserves operator config precedence and workspace scoping, documents the public API, and adds focused registry/runtime/status coverage.
