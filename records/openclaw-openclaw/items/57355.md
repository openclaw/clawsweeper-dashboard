## What This Changes

Add a native Plugin SDK API for plugins to register managed MCP servers and have OpenClaw wire them into embedded OpenAI/Pi-backed runs.

## Best Possible Solution

Land a maintainer-approved additive SDK/runtime contract that validates managed stdio-only registrations, records plugin ownership, preserves operator config precedence and workspace scoping, documents the public API, and adds focused registry/runtime/status coverage.
