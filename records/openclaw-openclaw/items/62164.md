Keep open. The PR is still useful because current main has no `sessions.list` result cache or `lastHash` contract, but the branch still has three merge blockers: scoped lists are invalidated by unrelated agent stores, the public Gateway protocol docs are not updated for the new wire contract, and the changelog includes an unrelated alpha-release note.

## Best Possible Solution
