## Best Possible Solution

Yes. Current main can be checked by calling `sessions.list` and observing a full response with no `lastHash` schema support; the PR behavior is reproducible by calling once for `hash`, calling again with `lastHash`, then mutating config/session/transcript/subagent-visible state.
