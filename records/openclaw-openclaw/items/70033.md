Keep open. Current main still leaves the reported Amazon Bedrock Converse stream path outside the malformed tool-call argument repair gate, while the pinned pi-ai Bedrock source can expose `{}` during incomplete or unparseable streamed tool arguments and OpenClaw still validates `write`/`exec` as required-argument tools.
