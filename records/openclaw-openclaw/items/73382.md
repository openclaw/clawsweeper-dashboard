## Summary

Keep this PR open. The underlying Bedrock context-window bug is still present on current main, but the proposed fallback has a concrete correctness issue for the Claude 4 ID shapes it claims to cover, and an open alternative PR is tracking the same report through configurable overrides.

## What This Changes

The PR adds missing Amazon Bedrock Claude context-window entries, a Claude family fallback, regression tests, and a changelog entry for the Bedrock discovery default-window bug.

## Best Possible Solution

Resolve the Bedrock discovery bug in the amazon-bedrock plugin by either repairing this PR's matcher to cover the documented Claude 4 ID shapes while preserving explicit-row precedence, or landing the configurable override approach from #73959 with enough tests and docs to let operators correct model limits without waiting for source updates.

## Reproduction Assessment

Yes. A high-confidence static reproduction exists on current main: an unlisted ID such as `anthropic.claude-3-5-sonnet-20241022-v2:0` is absent from `KNOWN_CONTEXT_WINDOWS`, so `resolveKnownContextWindow` returns undefined and discovery falls through to the 32000 default.

## Solution Assessment

No, not as-is. The explicit Claude 3.x backfill is a narrow useful fix, but the family fallback should either match real Claude 4 Bedrock ID shapes or be scoped to Claude 3.x while the maintainer evaluates the configurable override alternative.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Match real Claude 4 Bedrock IDs in the fallback:** `extensions/amazon-bedrock/discovery.ts:134`
  - body: The new fallback claims to cover Claude 4 variants, but Bedrock Claude 4 IDs use names like `anthropic.claude-sonnet-4...`, `anthropic.claude-opus-4...`, and `anthropic.claude-haiku-4...`; `^anthropic\.claude-(?:3|4)` only matches `anthropic.claude-4...`. An unlisted Claude 4 profile would still fall through to 32K, so widen the pattern and add a Claude 4 fallback test, or scope the fallback to Claude 3.x only.
  - confidence: 0.89
