## Best Possible Solution

Add a focused regression check for a 2026.5.2-style upgraded config with `messages.tts.provider` set to Microsoft or legacy `edge`, stale `plugins.entries.microsoft-speech`, and Telegram voice delivery, then fix the owner module or generic capability loader only where that proof fails.
