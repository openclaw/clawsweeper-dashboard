## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

No; the seam is plausible, but this implementation needs to keep the Pi threshold event enabled for `ownsCompaction && interceptsCompaction` engines and document the public plugin contract before merge.

- **[P2] Keep auto-compaction enabled for intercepting owners:** `src/agents/pi-embedded-runner/extensions.ts:196-201`

Plugin authors need the opt-in flag, request/result shape, fallback behavior, and interaction with `ownsCompaction` documented with this SDK change.
