- **[P2] Document non-staged gateway installer hooks:** `docs/gateway/protocol.md:620-623`
  - body: The new docs say `allowSetupHooks` runs setup after a staged install and before publishing, but the same parameter is added to gateway installer mode and forwarded to `installSkill`, where the hook runs against the already installed `entry.skill.baseDir` after the dependency command.
