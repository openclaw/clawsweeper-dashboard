  - body: `baseEnv: {}` is passed to `runCommandWithTimeout`, but `CommandOptions` does not include that property and the helper still calls `resolveCommandEnv({ argv, env })`. This both fails strict typechecking and still merges ambient `process.env` into setup scripts.
