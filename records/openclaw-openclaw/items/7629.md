Add configurable hybrid BM25 plus vector search to the bundled `memory-lancedb` plugin for exact keyword and semantic recall parity.

## Best Possible Solution

Implement this inside `extensions/memory-lancedb` with typed hybrid config, FTS index maintenance on `text`, query-text propagation, vector-only fallback, docs/changelog updates, and targeted tests for exact keyword recall and fallback behavior.
