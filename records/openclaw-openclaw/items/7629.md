Add hybrid BM25 + vector retrieval to the official `memory-lancedb` plugin, including FTS setup, query-text propagation, and weighted hybrid configuration.

## Best Possible Solution

Keep this as a focused `memory-lancedb` parity enhancement: add typed hybrid config and manifest/docs entries, maintain an FTS index on `text`, pass query text through recall/forget/CLI/hooks, use verified LanceDB hybrid search or a tested local fusion fallback, preserve vector-only fallback, and add targeted regression coverage.
