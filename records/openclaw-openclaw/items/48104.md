Requests clearer handling and observability when model safety behavior blocks explicitly authorized read-only SSH or operational diagnostics.

## Best Possible Solution

Define a maintainer-approved operator diagnostics contract that preserves hard authorization, then add narrow refusal-versus-authorization-versus-tool-failure observability with live provider proof.

Unclear. A broad trusted-admin mode is not the safest fix by itself; the maintainable path is a narrow operator diagnostics contract plus refusal/tool/auth observability that keeps hard exec authorization intact.
