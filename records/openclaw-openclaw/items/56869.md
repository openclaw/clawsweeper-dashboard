## What This Changes

The PR adds commented Docker Compose proxy environment examples and a Docker install docs subsection for configuring gateway outbound HTTP(S) proxy access.

## Best Possible Solution

Land the Docker docs/comment update after aligning the examples with OpenClaw and Undici `NO_PROXY` semantics, removing duplicate Compose guidance, and keeping the text clearly scoped to ambient Docker env proxy configuration rather than the managed `proxy.enabled` security feature.
