## Summary

Keep open: current main and the latest tagged release still support outbound WhatsApp `react`, but not WhatsApp `reactions` listing/query, stored historical reaction retrieval, inbound reaction exposure, or a reaction hook. The related PRs are useful implementation input, but they remain open or closed unmerged rather than shipped fixes.

## What This Changes

The issue requests WhatsApp support for querying, surfacing, and historically retrieving message reactions instead of only sending outbound reactions.

## Best Possible Solution

Keep this broad tracker open until the official WhatsApp plugin supports generic reaction list/query behavior, feasible historical retrieval or persistence, and privacy-gated inbound reaction exposure with focused tests, docs, and redacted live WhatsApp proof.

## Solution Assessment
