## Summary

Keep open: current main and the latest release still support outbound WhatsApp `react`, but not the generic WhatsApp `reactions` query, historical reaction retrieval, or privacy-gated inbound reaction exposure. ## What This Changes

Requests WhatsApp support for querying message reactions, retrieving or storing historical reactions, and exposing inbound reaction feedback to agents or plugins.

## Best Possible Solution

Keep this broad tracker open until the official WhatsApp plugin supports generic reaction query/list behavior, feasible historical retrieval or persistence, and privacy-gated inbound reaction exposure with tests, docs, and redacted live WhatsApp proof.

## Solution Assessment

Unclear until maintainers choose the API, storage, and privacy model. The related PRs are useful input, but the maintainable direction needs to reconcile generic reaction querying, historical storage or retrieval, inbound event delivery, and plugin hook boundaries.
