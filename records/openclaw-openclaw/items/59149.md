Add per-agent configuration to scope session visibility and agent-to-agent invocation or receipt instead of relying only on global `tools.sessions.visibility` and `tools.agentToAgent` settings.

## Best Possible Solution

Review and land or replace the linked implementation with one fail-closed, requester-aware per-agent access-control contract spanning schema/types, every session tool, outbound and any chosen inbound A2A semantics, docs, and tests.
