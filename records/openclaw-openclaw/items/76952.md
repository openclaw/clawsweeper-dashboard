Keep open. Current main has partial docs and implementation for browser realtime Talk, OpenAI realtime setup, Voice Call, and mobile nodes, but it still lacks the requested consolidated user-facing Talk guide, and subscription entitlement plus mobile/phone bridge behavior need product scoping.
