## Summary

Keep open: the interrupt-mode stale replay bug is still source-reproducible and this PR targets the right surface, but the proposed aborted-run early exit is too broad and would regress timeout delivery and compaction bookkeeping.

## What This Changes

The PR suppresses `lastAssistant` fallback text for aborted empty embedded attempts and skips auto-reply flush plus payload delivery for any aborted run.

## Best Possible Solution

Keep the narrow stale `lastAssistant` suppression, but make any silent aborted-run exit specific to superseded interrupt runs with no new output while preserving timeout/error delivery, compaction accounting, and focused regression coverage.

## Reproduction Assessment

Yes. Source inspection shows `interrupt` aborts active work, an aborted empty attempt can retain a prior `lastAssistant` from the full session snapshot, and current main can still turn that fallback into a delivered payload.

## Solution Assessment

No. The `lastAssistant` guard is the right direction, but treating every `meta.aborted` result as undeliverable is too broad because timeout and compaction paths intentionally carry useful outcomes on aborted metadata.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Preserve timeout payload delivery:** `src/auto-reply/reply/agent-runner.ts:483-490`
  - body: Timeout aborts intentionally return a user-facing timeout payload with `meta.aborted: true`. This early return drops all aborted payloads before `buildReplyPayloads`, so an auto-reply turn that hits `timeoutMs` becomes silent instead of sending the timeout notice.
  - confidence: 0.92
- **[P2] Keep compaction bookkeeping before exiting:** `src/auto-reply/reply/agent-runner.ts:483-490`
  - body: This return runs before `autoCompactionCount` handling later in `runReplyAgent`. An aborted run that already completed compaction would skip count persistence, queued-session refresh, and post-compaction context injection for the next turn.
  - confidence: 0.88
