## What This Changes

The issue requests a way to prevent idle heartbeat credit burn, including a `PAUSED` `HEARTBEAT.md` sentinel, persistent `0m` disable behavior, clearer documentation/onboarding, and a UI toggle.

## Best Possible Solution

Resolve the linked PR by choosing one heartbeat pause contract: land documented runtime support for `PAUSED` if accepted, or clarify `0m`/CLI disable as the canonical pause path if not.

## Solution Assessment
