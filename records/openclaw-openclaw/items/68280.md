## What This Changes

## Best Possible Solution

Land the local-loopback auth diagnostic only after daemon status honors the same no-anonymous-probe boundary and maintainers either accept the documented fail-closed behavior for upgrades or the PR preserves shipped probe compatibility by default.

## Solution Assessment
