Issue reports intermittent Linux exec SIGKILLs for broad discovery commands without actionable OOM or supervisor diagnostics.

## Best Possible Solution

Add backward-compatible termination metadata to foreground exec and process.poll details using existing supervisor/runtime fields, with focused regression coverage; keep signal escalation and OS-origin telemetry separate.
