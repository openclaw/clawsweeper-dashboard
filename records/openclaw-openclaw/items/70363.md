## What This Changes

The PR adds a per-session run-generation registry, stamps reply operations with currentness, bumps generation on abort, and gates one block-reply delivery path behind an optional stale-run predicate.

## Best Possible Solution

Keep this active only as a partial implementation path if maintainers accept the generation-fence contract, require real behavior proof, strengthen the delivery-path test, and coordinate follow-up work through https://github.com/openclaw/openclaw/issues/70319.

## Solution Assessment
