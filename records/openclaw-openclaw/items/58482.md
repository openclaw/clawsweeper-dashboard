  - body: Each polling sleep adds an abort listener but never removes it when the timer resolves normally, and it misses signals that were already aborted before the listener is attached. Long-running batch polls can leak handlers and late aborts can run stale callbacks; use a shared abortable sleep or remove the listener in a finally path across the providers.
  - confidence: 0.88
