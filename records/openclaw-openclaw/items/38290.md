## What This Changes

The PR normalizes opaque browser origins for gateway allowlist matching and adds unit plus WebSocket handshake tests for exact Chrome extension origins.

## Best Possible Solution

Land this PR or an equivalent consolidated gateway patch that preserves hosted opaque origins for exact allowlist matching, keeps wildcard policy separate, and retains the handshake regression test.
