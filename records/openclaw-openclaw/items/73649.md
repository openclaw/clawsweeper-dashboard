## What This Changes

The PR renames the bug report `logs` textarea to logs-only while retaining `render: shell` and adds a separate unrendered `screenshots_evidence` textarea for screenshots, recordings, and other evidence.

## Best Possible Solution

Land this focused issue-template fix, or switch the new screenshots/evidence control to `type: upload` if maintainers want an explicit file picker, while keeping shell rendering limited to logs.
