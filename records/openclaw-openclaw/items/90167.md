Full review comments:

- **[P1] Keep resolved plugin secrets out of loader cache keys:** `src/plugins/runtime/load-context.ts:102`
  - body: This resolves `plugins.entries.*.config` before handing the config to the normal cached runtime loader path, while leaving `resolveRawConfigEnvVars` unset. Please add a cache-safe path or use the loader's existing raw-env resolution mode so runtime loads do not retain resolved plugin secrets in cache keys.
