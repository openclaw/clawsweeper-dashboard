## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Bound collapsed markdown rendering:** `ui/src/ui/chat/grouped-render.ts:650`
  - body: Completed messages over the disclosure threshold still call `toSanitizedMarkdownHtml(markdown)` and insert the full rendered result before CSS collapses it. A 150k-500k payload can still force the parse, sanitize, and DOM work this feature is intended to avoid; render a capped preview or cheap fallback until expansion.
  - confidence: 0.92
- **[P2] Keep the streaming fallback cheap:** `ui/src/ui/markdown.ts:60`
  - body: Raising `MARKDOWN_PARSE_LIMIT` to 100k makes long in-progress replies keep parsing and sanitizing every growing chunk from 40k-100k characters, and the cache misses because each chunk differs. Preserve the existing cheap fallback for streaming content unless the user explicitly expands a completed message.
  - confidence: 0.88
- **[P2] Read expansion state when the button is clicked:** `ui/src/ui/chat/grouped-render.ts:659-673`
  - body: The click handler closes over the render-time `isExpanded` value. After the first click, later clicks on the same button keep taking the original branch until an unrelated render replaces the DOM, so Show less can fail to collapse the message.
  - confidence: 0.9
- **[P2] Keep the expanded button anchored to its message:** `ui/src/styles/chat/text.css:153-157`
  - body: `.chat-text__show-more` is absolutely positioned, but `.chat-text--expanded` drops the `position: relative` context from `.chat-text--long`. When expanded, the Show less button can be positioned against an ancestor rather than the message.
  - confidence: 0.86
- **[P3] Render the control outside the fade mask:** `ui/src/styles/chat/text.css:149-150`
  - body: The collapsed fade mask is applied to the same `.chat-text--long` element that contains the button, so the Show more control is faded with the message tail. Mask the content body or render the control as an unmasked sibling.
  - confidence: 0.78
