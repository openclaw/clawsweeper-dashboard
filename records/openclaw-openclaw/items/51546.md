## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Render a bounded preview before parsing full long messages:** `ui/src/ui/chat/grouped-render.ts:650`
  - body: For completed messages above the threshold, this still computes and inserts toSanitizedMarkdownHtml(markdown) for the entire payload before CSS hides the tail. Since the same PR raises the markdown hard and parse limits, the large messages this feature targets can still freeze the chat view; render only a capped preview until expansion and keep the cheap fallback for oversized content.
  - confidence: 0.93
- **[P2] Preserve the cheap streaming fallback for oversized replies:** `ui/src/ui/markdown.ts:60-61`
  - body: Raising MARKDOWN_PARSE_LIMIT from 40k to 100k makes in-progress long replies keep going through markdown parsing for an extra 60k characters, and cache hits do not help while the stream changes on every chunk. Preserve the current streaming/oversized fallback unless expansion is explicitly requested.
  - confidence: 0.88
- **[P2] Drive the toggle from live state instead of the render closure:** `ui/src/ui/chat/grouped-render.ts:659-674`
  - body: The click handler closes over the render-time isExpanded value, so after the first click it can keep taking the same branch and the button label remains bound to stale template state until an unrelated render. Store the state in reactive owner state or read the current state and request a re-render/text update on each toggle.
  - confidence: 0.9
- **[P2] Keep disclosure controls anchored outside the faded content:** `ui/src/styles/chat/text.css:153-160`
  - body: .chat-text__show-more is absolutely positioned, but .chat-text--expanded does not provide a positioned containing block, so Show less can escape the message when expanded. The collapsed control also sits inside the masked element, which can fade the affordance with the message tail; keep the control outside the masked content or give both states a stable wrapper.
  - confidence: 0.86
