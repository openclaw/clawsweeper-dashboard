## What This Changes

The PR adds a 131,072-token Gemma context-window fallback in `extensions/google/provider-models.ts` and a regression test for the Gemma template-clone path.

## Best Possible Solution

Keep the fix in the Google provider boundary, but use authoritative exact or family metadata for Gemma fallbacks while preserving configured/catalog metadata precedence, then add focused regression coverage and redacted CLI proof.
