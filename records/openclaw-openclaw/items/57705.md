## Best Possible Solution

Not applicable as a bug reproduction because this is a feature request. Source and docs inspection give a high-confidence current-behavior check: OpenClaw exposes status text and subagent lists, but not global remaining sub-agent capacity or a preflight capacity contract.

## Solution Assessment
