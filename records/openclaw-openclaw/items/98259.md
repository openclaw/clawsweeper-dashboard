## Summary

The PR auto-detects Azure OpenAI Chat Completions endpoints as `prompt_cache_key` capable, disables Azure long prompt-cache retention by default, and adds two payload regression tests.

## Best Possible Solution

Centralize Azure Chat Completions prompt-cache capability detection in the shared completions compat path used by both builders, preserve or deliberately model-gate long retention, cover the Foundry host forms, and require redacted real Azure proof before merge.
