## What This Changes

## Best Possible Solution

Land the focused Telegram dispatch fix if maintainers accept the fresh-resume split as the right exception to normal single-preview streaming, with CI and optional native visual proof as final gates.

## Reproduction Assessment

## Solution Assessment

Yes: this is the narrowest maintainable fix because Telegram dispatch is where tool-boundary callbacks and draft-lane state meet. Moving this into the shared reply pipeline or lane deliverer would be broader and would lack the Telegram-specific send/edit context.
