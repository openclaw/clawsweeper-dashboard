  - confidence: 0.86

Requests through the OpenAI-compatible embeddings endpoint will drop the configured Jina task; pass both fields through `createConfiguredEmbeddingProvider` and add endpoint coverage.
  - confidence: 0.9

  - body: The new section is labeled for Jina v5, but it names `jina-embeddings-v3`/`jina-embeddings-v4` and the sample config uses `jina-embeddings-v3`. Update the OpenAI provider and memory-config examples to v5 text model IDs, such as `jina-embeddings-v5-text-small` or `jina-embeddings-v5-text-nano`, so readers exercise the documented task-adapter setup.
