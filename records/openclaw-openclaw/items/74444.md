Land a current-main implementation that keeps the feature provider-owned and narrow: add typed task config plus labels/help/generated config metadata, resolve and forward query/document task values through memory-core and the OpenAI embedding paths, include task in cache identity and batch requests, preserve the package-owned SDK bridge layout, correct the v5 docs, update changelog, and add focused regression coverage.
