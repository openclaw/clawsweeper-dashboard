  - body: When stripping leaves an empty string, this falls back to the original text, so an emoji-only TTS request can still be synthesized as emoji names. Return a no-op or error after the policy is settled instead of reintroducing the content.
  - confidence: 0.85

  - body: Providers can synthesize inputText, but downstream payloads still record spokenText from the original request because the prepared text is not returned through TtsResult. Audio metadata and transcripts can disagree with what was actually sent to the provider.
