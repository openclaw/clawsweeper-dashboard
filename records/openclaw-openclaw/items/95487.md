  - body: This unconditionally rewrites text for every shared speech synthesis target.   - confidence: 0.88

  - body: When stripping leaves an empty string, this falls back to the original text, so an emoji-only TTS request can still be synthesized as emoji names. Return a no-op or error after the policy is settled instead of reintroducing the content.
  - confidence: 0.86

  - body: The provider can synthesize `inputText`, but downstream payloads still record the original `textForAudio` because the prepared text is not returned through `TtsResult`. Audio metadata and transcripts can disagree with what was actually sent to the provider.
