## Best Possible Solution

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.82

Full review comments:

- **[P2] Pass the Codex auth context into extraction clients:** `extensions/codex/media-understanding-provider.ts:198-205`
  - body: `extractCodexStructured` starts the Codex app-server without forwarding `req.agentDir` or `req.cfg` to either the injected client factory or the isolated shared-client path. Please pass the request context through this new path and add coverage for that selection.
