## Summary

Keep open. Current main has adjacent cron timeout, retry/backoff, retention, run-log, failure-alert, and `cron_changed` plugin-hook primitives, but it still lacks the requested per-job `lifecycle` API for timeout, retry/cleanup policy, and job-configured start/complete/failure hooks; the open related cleanup/circuit-breaker issue does not cover the full hook/config request.
