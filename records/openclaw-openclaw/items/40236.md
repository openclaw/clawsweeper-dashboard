## Summary

## What This Changes

Request to add job-level cron lifecycle configuration for timeouts, retry limits, automatic cleanup, and start/complete/failure hooks.

## Best Possible Solution

Define one additive cron lifecycle contract that coordinates existing timeout, retry/backoff, failure-alert, cron_changed observation, retention, run-history, and delete-time cleanup primitives with explicit security boundaries and regression coverage.
