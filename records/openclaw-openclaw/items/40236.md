- The exact public API must reconcile `payload.timeoutSeconds`, recurring backoff, one-shot retry, `deleteAfterRun`, `failureAlert`, retention, run logs, and `cron_changed` without duplicating semantics.

- Hook execution can become an unattended host-code or outbound-webhook surface, so path, permission, timeout, logging, and secret-handling policy need maintainer/security review.
