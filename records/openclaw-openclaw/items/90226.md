- **[P2] Make closed markers stale before provider-owned skips:** `src/auto-reply/reply/session.ts:532-533`
  - body: The new `entryForcesRollover` path forces closed rows to roll over even when `skipImplicitExpiry` made `entryFreshness` `{ fresh: true }` for provider-owned CLI sessions.   - confidence: 0.82

Pass `entry.sessionClosedAt` through this helper and add focused coverage.
  - confidence: 0.86
