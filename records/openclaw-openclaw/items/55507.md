Keep this PR open, but it needs changes before merge: the underlying OpenAI `outputDimensionality` bug still exists on current `main`, while this branch updates the removed `src/plugins/memory-host` implementation instead of the active `extensions/openai` provider path.

## What This Changes

The PR adds OpenAI embedding output dimensionality validation, request-body `dimensions` plumbing, and tests under the old `src/plugins/memory-host` memory embedding provider files.
