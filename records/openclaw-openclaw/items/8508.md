## Summary

Keep open: current main still exposes a single-value ack reaction path, and the requested configured list or contextual chooser remains unimplemented. This is a feature/API design item rather than a stale cleanup candidate.

## What This Changes

## Best Possible Solution

Preserve the existing `ackReaction` string contract and design an additive ack-selection contract covering config shape, selection timing, channel validation, docs, schemas, and bundled-channel tests.

## Solution Assessment
