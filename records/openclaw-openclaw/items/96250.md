## Summary

Keep this PR open but do not merge it as-is: the heap default belongs at the managed service boundary, but the patch also preserves arbitrary caller-provided NODE_OPTIONS even though current install paths intentionally filter that key as a dangerous startup override.

## Best Possible Solution

## Solution Assessment
