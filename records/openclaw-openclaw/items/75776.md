## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.92

Full review comments:

- **[P2] Persist the terminal lifecycle fallback:** `src/agents/command/session-store.ts:173-181`
  - body: The new fallback sets `status`, `endedAt`, and `runtimeMs`, but the function still passes `next` through `removeLifecycleStateFromMetadataPatch()` before merging.   - confidence: 0.96

  - body: The skip regex treats any prompt containing `exactly` as an output-only request, so ordinary prompts like "explain exactly why the UI froze" get no preflight acknowledgement. That undercuts the PR's main reliability goal for normal WebChat requests.
  - confidence: 0.84
