This PR adds Control UI/WebChat preflight acknowledgement and pending-turn continuity, delayed disconnect cleanup, reset transcript path handling, earlier compaction thresholds, tool allow-list plumbing, prompt acknowledgment copy, focused tests, and a changelog entry.

## Best Possible Solution
