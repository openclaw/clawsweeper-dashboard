Full review comments:

- **[P2] Strip the wrapper script on SEA re-entry:** `scripts/build-sea.mjs:243`
  - body: When `NODE_COMPILE_CACHE` is set, `openclaw.mjs` can respawn `process.execPath` with its own `openclaw.mjs` path as the first argument. Treat the packaged wrapper path as executable-like, or skip this respawn path under `OPENCLAW_SEA`.
