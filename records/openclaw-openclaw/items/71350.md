## Summary

Keep open. Current main implements the P0 slices through per-agent/global `thinkingDefault` and flat per-agent `params`, but the adaptive `thinkingStrategy` rules/escalation/AGENTS.md policy is still absent, and `agents.list[].models[*].params` is not a documented runtime parameter-inheritance contract.

## What This Changes

The issue requests per-agent thinking defaults, per-agent model parameter overrides, and adaptive thinking strategies.

## Best Possible Solution

Keep this as the canonical design tracker, preserve the shipped `thinkingDefault` and flat `params` contracts, and split any approved adaptive strategy or model-map params work into focused follow-up PRs after API review.
