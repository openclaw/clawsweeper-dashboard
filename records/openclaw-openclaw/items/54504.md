Add persistent per-role Markdown instruction files for ephemeral native `sessions_spawn` subagents, with optional frontmatter for model and tool overrides.

## Best Possible Solution

Design a native role-profile resolver for `sessions_spawn` if maintainers accept the feature, with workspace-before-user precedence, explicit parameter precedence over frontmatter, safe model validation, tool-policy narrowing by default, docs, and focused tests.
