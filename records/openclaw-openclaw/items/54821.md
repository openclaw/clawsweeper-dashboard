## What This Changes

The PR adds a `thinking-events` gateway client capability, cap-gated thinking-event routing, callback-independent reasoning streaming, and related gateway/agent tests and baselines.

## Best Possible Solution

Add the `thinking-events` capability on top of current main's raw reasoning stream contract, preserving `thinkingLevel` and `silentExpected` suppression and covering cap-only delivery in gateway tests.

## Reproduction Assessment

Yes. Source inspection shows current main still requires `onReasoningStream` before emitting thinking events and has no `thinking-events` cap, while the PR screenshots show the intended WebSocket cap/no-cap behavior.

## Solution Assessment

No. The cap-gated routing direction is right, but the branch should be rebased onto current main and preserve raw reasoning payloads plus the existing visibility gates before merge.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.84

Full review comments:

- **[P1] Preserve the current reasoning visibility gates:** `src/agents/pi-embedded-subscribe.ts:77`
  - body: Current main suppresses reasoning streaming when reasoning is not showable or a silent reply is expected. This branch decouples streaming from `onReasoningStream` by setting `streamReasoning` solely from `reasoningMode`, and its PR-head `emitReasoningStream` lacks the current `silentExpected` guard, so a rebased version would risk sending hidden reasoning tokens to any client that advertises the new cap. Please rebase and keep those gates while removing only the callback dependency.
  - confidence: 0.86
- **[P2] Keep thinking-event payloads raw:** `src/agents/pi-embedded-subscribe.ts:664-688`
  - body: Current main's merged raw-reasoning refactor sends unformatted reasoning text from the core stream and leaves channel formatting to the edges. This PR formats `rawAccumulated` with `formatReasoningMessage` and sends that formatted value in both the agent event and `onReasoningStream`, which would regress clients and channel dispatchers that now expect raw reasoning deltas.
  - confidence: 0.88
