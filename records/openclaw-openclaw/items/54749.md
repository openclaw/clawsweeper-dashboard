Keep open: current main still has the report-card content navigation gap. Report-kind Memory Palace pages can appear in the Reports cluster, but the primary card click still toggles inline page details instead of opening the existing wiki content preview.

## What This Changes

The issue reports that mobile report-list selection opens Command Overview instead of the selected report content.

## Best Possible Solution
