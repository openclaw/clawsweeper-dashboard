## Best Possible Solution

Yes for the stated direct memory-lancedb scope, with maintainer acceptance of the fallback policy. The implementation preserves server-side `dimensions` for accepting providers, retries only after clear field-rejection signals, and avoids turning this PR into a broader provider-policy change.
