## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.93

Full review comments:

- **[P1] Reject symlinked memory directories during export:** `extensions/memory-core/src/encrypted-backup.ts:33-43`
  - body: `listFilesRecursive()` calls `fs.readdir(rootDir)` on `workspace/memory` before checking whether that directory is a symlink. If `memory` points outside the workspace, export walks outside files and serializes them as `memory/...`; lstat/realpath-contain the root and each directory before descending, and add a root-symlink regression.
  - confidence: 0.93
- **[P1] Reject symlink leaves before overwrite restores:** `extensions/memory-core/src/encrypted-backup.ts:183-185`
  - body: With `--overwrite`, restore checks only parent directories and then writes the final path with `fs.writeFile(..., flag: "w")`. An existing `target/MEMORY.md` symlink can redirect the write outside `--target`; reject a final symlink leaf or use a no-follow safe-open path before overwriting.
  - confidence: 0.91

## Security Review
