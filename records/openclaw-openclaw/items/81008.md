## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.93

Full review comments:

- **[P1] Reject symlinked memory roots before walking:** `extensions/memory-core/src/encrypted-backup.ts:33-43`
  - body: `listFilesRecursive()` calls `fs.readdir(rootDir)` on `workspace/memory` before proving that directory is not a symlink. If `memory` points outside the workspace, export walks the target and serializes outside files as `memory/...`; lstat or realpath-contain the root before descending and add a root-symlink regression.
  - confidence: 0.93
- **[P1] Reject final symlink leaves on overwrite:** `extensions/memory-core/src/encrypted-backup.ts:183-185`
  - body: With `--overwrite`, restore skips the final `lstat(outputPath)` check and writes with `flag: "w"`, which follows an existing `target/MEMORY.md` symlink outside `--target`. Reject a symlink leaf, or use a no-follow safe-open/write path before allowing overwrite.
  - confidence: 0.91
