Full review comments:

- **[P1] Reject symlinked memory roots before export:** `extensions/memory-core/src/encrypted-backup.ts:33`
  - body: `listFilesRecursive()` calls `fs.readdir(rootDir)` before proving the `workspace/memory` root is not a symlink. A symlinked memory directory can be walked and serialized as `memory/...`; lstat or realpath-contain the root before descending and add a root-symlink regression.
  - confidence: 0.94

  - body: With `--overwrite`, restore checks parent directories but then writes the final output path with `flag: "w"`, which follows an existing `target/MEMORY.md` symlink outside `--target`. Reject symlink leaves or use a no-follow safe-open/write path before allowing overwrite.
  - confidence: 0.92

  - body: This helper hard-codes `/Users/anandsagar/.openclaw` and is allowlisted as an unused file, so the repo gains machine-specific safety policy that does not protect other contributors or CI. Replace it with generic temp-dir isolation in tests or remove it.
