## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.93

Full review comments:

- **[P1] Reject symlinked memory directories during export:** `extensions/memory-core/src/encrypted-backup.ts:33-43`
  - body: `collectMemoryBackupArchive()` calls the recursive walker on `workspace/memory`, and the walker starts with `fs.readdir(rootDir)` before lstat/realpath containment. If `memory` is a symlink, export can enumerate files outside the workspace and serialize them under `memory/...`; lstat or realpath-contain the root before descending and add a regression for a symlinked `workspace/memory`.
  - confidence: 0.92
- **[P1] Refuse final symlink leaves before overwrite restores:** `extensions/memory-core/src/encrypted-backup.ts:183-185`
  - body: With `--overwrite`, restore checks only parent directories and then writes the final path with normal `fs.writeFile(..., flag: "w")`. An existing symlink at `target/MEMORY.md` can redirect the write outside `--target`; reject a final symlink leaf or use a no-follow safe open path before overwriting.
  - confidence: 0.9
