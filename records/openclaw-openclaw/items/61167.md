Full review comments:

- **[P1] Validate UID and GID before root chown:** `scripts/docker/setup.sh:508-509`
  - body: `OPENCLAW_UID` and `OPENCLAW_GID` are caller-controlled environment values, but this builds them into a string executed by `sh -c` as root in the prestart container.
