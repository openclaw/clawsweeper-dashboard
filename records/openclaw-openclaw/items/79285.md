## Summary

Current main should not be closed: source inspection still shows a path where paused or abandoned subagent lifecycle ends can be surfaced through `agent.wait` as `ok`, then announced as `completed successfully` with the latest assistant progress text.

## What This Changes

The issue asks OpenClaw to report interrupted native subagent work as failed, timed out, or incomplete instead of completed successfully.

## Best Possible Solution

Use runtime completion facts, not text heuristics, so non-yielded paused, abandoned, replay-invalid, aborted, or timeout terminal states cannot flow to the parent as successful subagent completion.

## Reproduction Assessment

Do we have a high-confidence way to reproduce the issue? No live reproduction was run, but current source provides a credible reproduction path: compaction or incomplete terminal state can produce paused or abandoned lifecycle metadata that is later mapped to `ok`.

## Solution Assessment

Is this the best way to solve the issue? Yes: the narrow maintainable fix is to carry and honor existing runtime liveness, replay, abort, timeout, and yield facts in the wait and subagent outcome mapping rather than guessing from progress-like result text.

## Repair Work Prompt

Fix native subagent completion status so interrupted or incomplete child runs cannot announce `completed successfully`. Start from the runtime facts already present in lifecycle and wait snapshots: `livenessState`, `replayInvalid`, `aborted`, `stopReason`, `yielded`, and explicit wait timeout/error. Treat non-yielded paused, abandoned, replay-invalid, aborted, or timeout terminal states as timeout/error/unknown as appropriate, and do not use progress-text heuristics as the primary fix. Preserve intentional `sessions_yield` paused behavior and ordinary successful final-answer completions. Add regression coverage around lifecycle-to-wait mapping and subagent outcome/announcement behavior, and add a user-facing `CHANGELOG.md` fix entry crediting @LittleRollo if the final fix changes shipped behavior.
