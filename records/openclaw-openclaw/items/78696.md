## Summary

## What This Changes

The PR adds an exec-side parser, focused tests, and a changelog entry to reject raw global npm/pnpm/bun/yarn installs or updates of the `openclaw` package and direct self-updates toward `update.run` or `openclaw update`.

## Best Possible Solution

Port the detector into the shared exec-control guard, cover both exec and exec-review consumers, drop the direct changelog edit, and require real exec/Gateway proof plus maintainer acceptance of the hard-block policy before merge.
