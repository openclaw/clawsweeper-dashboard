## Summary

## Best Possible Solution

If maintainers want this taxonomy, land a rebased implementation that derives tiers from the normalized effective plugin format, defines the public CLI/JSON/status/docs contract, and proves native, bundle, missing-format, list, and info/inspect behavior.

## Solution Assessment

No. The direction may be useful, but this branch is not the best merge path until it computes from the effective format, rebases onto current main, and maintainers accept the security-facing taxonomy.
