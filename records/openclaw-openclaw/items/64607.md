## Summary

Keep open. The issue requests inline chat previews for generated or received images, audio, video, and web links.

## Best Possible Solution

Define the supported media classes and fetch/render trust policy first, then land staged Control UI and Gateway changes while keeping narrower image bugs linked to their canonical issues or PRs.

## Solution Assessment

No as one broad implementation; the maintainable path is a staged product and security decision that preserves the existing structured media path and explicitly approves any remote fetch behavior.
