Keep open. Current main has partial inline media support for structured Control UI attachments, but this umbrella still covers unresolved remote Markdown image policy, automatic web/link previews, filtered video uploads, and a linked remote-image issue/PR that still needs product and security review.

Split this umbrella into focused maintainer-owned slices for remote-image policy, secure automatic link previews, durable audio/video handling, and upload/render behavior while preserving the shipped structured media paths.
