## Security Review

Status: needs_attention

Summary: Remote media and web preview support is security-sensitive because it changes Control UI and possibly Gateway network-fetch policy.

Concerns:

- **[medium] Design remote preview policy before enabling loads:** `src/gateway/control-ui-csp.ts:47`
  - body: Current Control UI policy intentionally blocks arbitrary remote image/media loads; enabling remote media or web previews would expand browser and possibly server-side network exposure, so the implementation needs an approved proxy, CSP, SSRF, and allowlist design.
  - confidence: 0.9
- **[medium] Avoid default-on third-party web previews:**
  - body: Automatic link previews would need to fetch or embed third-party content from chat/model text, which can leak viewer metadata or create SSRF exposure unless the fetcher, cache, allowlist, and privacy behavior are designed first.
  - confidence: 0.84
