## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.94

Full review comments:

- **[P1] Register curator commands through the CLI contract:** `extensions/skill-curator/src/cli.ts:98-106`
  - body: `api.registerCommand` requires a flat command with a `handler`; it does not support `subcommands`, `run`, `options`, `display`, or `ctx.options`. As written, the curator shell subcommands will fail the current command contract instead of registering.
  - confidence: 0.96
- **[P1] Use a valid runtime lifecycle registration:** `extensions/skill-curator/src/index.ts:19-26`
  - body: `registerRuntimeLifecycle` requires an `id` and optional `cleanup`; `activate` and `deactivate` are not lifecycle callbacks in the current SDK. This registration is rejected by the host contract and should be removed or rewritten to the supported API.
  - confidence: 0.94
- **[P1] Validate skill names before filesystem moves:** `extensions/skill-curator/src/run.ts:86-100`
  - body: `moveSkillToArchive` joins the sidecar skill key directly into source and destination paths before `fs.rename`. A crafted `.usage.json` key such as `../../outside` can move directories outside `skills/`; normalize the name and verify resolved paths stay inside the skills root first.
  - confidence: 0.9
- **[P2] Keep skill-workshop off curator private internals:** `extensions/skill-workshop/src/tool.ts:24-30`
  - body: This lazy import reaches from `skill-workshop` into another plugin's private `src/telemetry.js`, which breaks the bundled plugin boundary and makes externalizing curator difficult. Expose a public API seam or keep the integration inside curator-owned code.
  - confidence: 0.87
- **[P2] Register long-lived hooks with live-config guardrails:** `extensions/skill-curator/src/index.ts:30-35`
  - body: Adding a bundled `agent_end` handler needs the hook allowlist/live-config guard path used by existing bundled hooks. Reading startup `api.pluginConfig` here means runtime config changes may not stop automatic curator mutations.
  - confidence: 0.83
