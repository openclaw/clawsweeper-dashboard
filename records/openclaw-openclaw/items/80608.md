  - body: This import was changed to `applyProposalToWorkshop`, but `skills.ts` exports `applyProposalToWorkspace` and this file still calls `applyProposalToWorkspace` below. The modified plugin will fail to typecheck before any curator behavior can run.

  - body: The current SDK does not accept `activate`/`deactivate` callbacks here, `api.on` is not part of `OpenClawPluginApi`, and `registerSessionSchedulerJob` is cleanup metadata that requires `sessionKey`/`kind` rather than an hourly runner. This will fail the contract and the periodic curator tick will not execute.
  - confidence: 0.94
- **[P1] Constrain usage skill names before moving directories:** `extensions/skill-curator/src/run.ts:88-89`
  - body: `determineAllTransitions` returns names from workspace `.usage.json`, but `moveSkillToArchive` joins that unvalidated name directly into paths. A crafted sidecar key such as `../../x` can make a curator run rename paths outside `skills/`; validate and resolve within the skills root before moving anything.
  - confidence: 0.9
- **[P1] Do not expose unchecked skill deletion:** `extensions/skill-workshop/src/tool.ts:301-305`
  - body: The new `delete` action removes any normalized workspace skill directory unless it is pinned, including hand-authored or hub/bundled skills that the PR says should be immune. Either drop this action or route it through explicit ownership and approval guards before deleting.
  - confidence: 0.86
- **[P2] Keep snapshot output outside the archived tree:** `extensions/skill-curator/src/snapshot.ts:59-67`
  - body: The snapshot tarball is created under `skills/.curator_backups` while `tar` archives the whole `skills` directory. That can include previous backups and the file currently being written, causing failures or rapidly growing archives before any mutation is applied.
  - confidence: 0.9
- **[P2] Wire the reviewer into curator runs:** `extensions/skill-curator/src/run.ts:189-212`
  - body: The PR adds reviewer parsing and prompt code, but `curatorRun` only computes age-based transitions and applies them; it never builds a review manifest, calls an auxiliary model, or applies parsed review decisions. The advertised LLM patch/consolidate/archive pass is currently a no-op.
  - confidence: 0.84
- **[P3] Use Mintlify-safe docs links:** `docs/cli/curator.md:161-163`
  - body: These internal docs links include `/docs/` and `.md`, which the scoped docs guide says should be root-relative without markdown suffixes. On the published docs site, these See Also links should use paths like `/automation/curator` and `/plugins/skill-workshop`.
  - confidence: 0.82
