## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.94

Full review comments:

- **[P1] Register curator commands through the CLI contract:** `extensions/skill-curator/src/cli.ts:98-106`
  - body: `api.registerCommand` only accepts a flat command with a `handler`; this object uses unsupported `subcommands`, per-subcommand `run`, `options`, and `ctx.options`, so the plugin will not satisfy the current command contract and the advertised CLI commands will not register.
  - confidence: 0.96
- **[P1] Use the runtime lifecycle contract:** `extensions/skill-curator/src/index.ts:19-26`
  - body: `registerRuntimeLifecycle` requires an `id` and optional `cleanup`; `activate` and `deactivate` are not host lifecycle callbacks, so this registration does not typecheck or produce the intended lifecycle behavior.
  - confidence: 0.94
- **[P1] Validate skill names before archive moves:** `extensions/skill-curator/src/run.ts:88-100`
  - body: `moveSkillToArchive` joins the sidecar skill key directly into source and destination paths before `fs.rename`. A crafted `.usage.json` key such as `../../outside` can move directories outside `skills/`; normalize the name and guard resolved paths within the skills root before moving directories.
  - confidence: 0.9
- **[P2] Expose a public curator/workshop seam:** `extensions/skill-workshop/src/tool.ts:27-30`
  - body: This lazy import reaches from `skill-workshop` into another plugin's private `src/telemetry.js`, breaking the bundled plugin boundary and making an external curator package harder to support. Move this through a public API seam or keep the integration inside the curator plugin.
  - confidence: 0.87
- **[P2] Resolve live config before automatic mutations:** `extensions/skill-curator/src/index.ts:30-35`
  - body: The long-lived `agent_end` hook runs workspace filesystem mutations using `api.pluginConfig` captured at plugin registration. If runtime config changes disable or retune the curator, this handler can keep applying the old policy until restart; resolve live config or add a live guard before mutating.
  - confidence: 0.83

## Security Review
