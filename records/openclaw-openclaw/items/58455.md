## Best Possible Solution

Rebase onto current main, keep only the compaction pieces that fit the current safeguard/provider contracts, fix the parser and package import, and remove or explicitly approve a safer transcript-recovery design before merge.

Full review comments:

- **[P2] Match the final summary block when stripping analysis:** `src/agents/compaction-analysis-strip.ts:42-44`
  - body: `text.match(/<summary>.../)` returns the first summary block anywhere in the response.   - confidence: 0.9
