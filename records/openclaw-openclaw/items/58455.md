Full review comments:

- **[P1] Parse the intended final summary block:** `src/agents/compaction-analysis-strip.ts:42`
  - body: `stripAnalysisBlock` takes the first `<summary>...</summary>` match in the whole response. Because the prompt itself asks the model to reason about `<summary>` tags inside `<analysis>`, a normal response can mention those literal tags before the real final summary, causing compaction to keep an analysis fragment and drop the actual session summary. Parse the summary after the closing analysis block or select the final well-formed summary section.
  - confidence: 0.88

  - body: This appends the full session JSONL path to the model-visible compaction summary and tells future turns where to recover details. OpenClaw documents transcripts as sensitive local files that can contain private messages, file contents, command output, and links, so this needs a maintainer-approved bounded/redacted recovery mechanism or should be removed.
  - confidence: 0.84
