- **[P1] Match the final summary block after analysis:** `src/agents/compaction-analysis-strip.ts:39-41`
  - body: `stripAnalysisBlock` takes the first `<summary>...</summary>` anywhere in the model output. If the analysis text mentions a literal summary tag pair before the real final summary, compaction will keep that analysis fragment and drop the actual summary, silently losing session context.
