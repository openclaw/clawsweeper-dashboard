Full review comments:

- **[P1] Match the final summary block after analysis:** `src/agents/compaction-analysis-strip.ts:42`
  - body: `text.match(/<summary>.../)` grabs the first summary tag anywhere in the response. Because the prompt itself asks the model to reason in `<analysis>`, analysis text can mention literal `<summary>` tags and cause OpenClaw to persist that fragment instead of the final summary. Parse after `</analysis>` or select the intended final complete summary block.
  - confidence: 0.87
- **[P1] Gate transcript path recovery behind a privacy boundary:** `src/agents/pi-hooks/compaction-safeguard.ts:943-945`
  - body: Appending the raw session JSONL path to every compaction summary makes that local path model-visible and tells the next agent to read the full transcript. That expands post-compaction access to old tool output and possibly sensitive transcript data without an approved redaction or access policy. Drop it or gate it behind a maintainer-approved recovery contract.
  - confidence: 0.82
