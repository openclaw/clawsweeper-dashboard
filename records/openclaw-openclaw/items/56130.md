The PR adds a native visionOS companion app and documentation for pairing Apple Vision Pro as an OpenClaw Gateway node.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Decode node invoke paramsJSON before dispatch:** `apps/visionos/Sources/NodeManager.swift:257-263`
  - body: Current Gateway invokes send command arguments as `paramsJSON`, but this handler only reads `payload["params"]`. Commands that need input will receive empty params, so the node will not honor caller-provided arguments.
  - confidence: 0.94
- **[P1] Return error.message in node.invoke.result:** `apps/visionos/Sources/NodeManager.swift:158-162`
  - body: `node.invoke.result` validates `error` with only `code` and `message`; adding `detail` makes failed command results invalid. Any error path using `sendError(..., detail:)` can be rejected by the gateway instead of completing the pending invoke.
  - confidence: 0.91
- **[P2] Register APNs through node.event:** `apps/visionos/Sources/APNsManager.swift:59-62`
  - body: The gateway does not expose a `node.apns` method; APNs registration is handled as `node.event` with `event: "push.apns.register"`. Sending `node.apns` means the token publish path is rejected and APNs wake support never gets configured.
  - confidence: 0.93
- **[P1] Add a build manifest for the visionOS app:** `apps/visionos/README.md:17`
  - body: The branch adds sources and tells users to open `visionOS-node/visionOS-node.xcodeproj`, but no Xcode project, `project.yml`, `Package.swift`, or equivalent build surface is included under `apps/visionos`. As submitted, the app cannot be built from the repository checkout.
  - confidence: 0.88
- **[P2] Nest allowCommands under gateway.nodes:** `docs/nodes/visionos.md:97-110`
  - body: The docs show a root-level `allowCommands` object, but the current config schema and docs use `gateway.nodes.allowCommands`. Users following this example will not enable the dangerous/spatial node commands they are trying to allow.
  - confidence: 0.9
