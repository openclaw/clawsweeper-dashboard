Keep open. ## What This Changes

Request to ship version-matched bundled docs and expose local-first docs search/get/schema/API/version retrieval to agents for onboarding.

## Best Possible Solution

Define a versioned docs manifest plus local search/get index shipped with releases, expose a maintained local-first agent or Gateway retrieval contract, and document installed-version versus latest-doc fallback semantics.

## Solution Assessment
