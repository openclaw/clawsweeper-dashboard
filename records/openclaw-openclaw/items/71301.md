## Summary

Keep open. Current main and v2026.6.10 ship bundled docs and agent prompt guidance, but the requested first-class local-first docs retrieval surface for agents is still missing and needs maintainer API/product direction.

## What This Changes

## Best Possible Solution

Define a maintainer-approved local-first docs retrieval contract that reuses shipped docs and config.schema.lookup, then choose whether it lives as a core tool, bundled plugin/skill, or MCP surface with explicit live fallback.

## Solution Assessment
