## Best Possible Solution

Define an additive opt-in hook lifecycle policy covering direct /hooks/agent, mapped hooks, Gmail presets, generated hook:* keys, delayed cleanup, archive/delete behavior, transcript retention, and completed-session visibility.

## Solution Assessment
