## What This Changes

The issue requests configurable cleanup, archival, transcript-retention, and completed-session visibility behavior for webhook/hook-created sessions such as Gmail hooks.

## Best Possible Solution

Define an explicit opt-in hook lifecycle policy in core hooks config, with clear archive/delete/no-op modes, transcript-retention semantics, completed-session visibility behavior, and tests covering direct hooks, mappings, Gmail presets, and agent-scoped hook keys.

## Solution Assessment
