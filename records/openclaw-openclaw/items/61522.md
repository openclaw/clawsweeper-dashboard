- **[P1] Avoid logging config reads through the config-backed logger:** `src/config/io.ts:775`
  - body: On this head, `configLogger.debug/warn/error` checks file log settings through `readLoggingConfig()`, which loads `loadConfig()`.   - confidence: 0.94
