## Summary

Keep this issue open: current main and v2026.6.8 still have adjacent source buckets, citations, Active Memory untrusted framing, and Memory Wiki provenance, but not durable memory trust tiers, review/expiration gates, or trust-weighted retrieval.

## What This Changes

The issue requests source-based trust metadata and trust-policy enforcement for durable agent memory to reduce memory poisoning from untrusted inputs.

## Best Possible Solution

Define one maintainer-approved, host-authored durable memory provenance/trust contract and enforce it consistently at ingestion, storage, retrieval, review, expiration, UI/docs, and plugin SDK boundaries.
