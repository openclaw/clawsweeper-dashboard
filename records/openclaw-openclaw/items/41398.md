  - body: Current main sends the Gemini PDF API key in the `x-goog-api-key` header and keeps the URL credential-free. This branch appends `?key=...`, so request, proxy, and access logs can capture the secret; preserve header auth while normalizing the model id.

  - confidence: 0.86
