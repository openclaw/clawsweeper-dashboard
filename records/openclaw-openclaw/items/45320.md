## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Ignore stale send failures after preview rotation:** `src/discord/draft-stream.ts:103`
  - body: After `forceNewMessage()` bumps `generation`, only successful superseded sends are ignored. If the old in-flight `rest.post` rejects, the catch still sets `streamState.stopped = true`, so the new generation can lose preview updates for the rest of the reply. Treat failures from stale generations as superseded before stopping the stream, and add coverage for that path.
  - confidence: 0.86
