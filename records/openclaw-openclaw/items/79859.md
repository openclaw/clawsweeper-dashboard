The issue requests a recording-friendly TUI mode that suppresses elapsed seconds, spinner/shimmer animation, and repaint loops while a request is running.

## Best Possible Solution

Define the supported TUI quiet-status surface, then add a narrow TUI rendering path that uses static text, omits elapsed time, skips shimmer/timer updates, and documents/tests the behavior.
