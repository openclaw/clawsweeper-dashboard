## Summary

Keep open: current main still animates the TUI busy status with elapsed time, spinner, and waiting shimmer, and exposes no quiet status flag, config key, or slash command. ## What This Changes

Request a TUI mode that suppresses elapsed seconds, spinner animation, and waiting shimmer during active requests for cleaner terminal recordings.

Add a maintainer-approved quiet status surface in the TUI renderer that keeps the animated default, switches busy states to stable text when enabled, and documents the chosen CLI/config behavior.
