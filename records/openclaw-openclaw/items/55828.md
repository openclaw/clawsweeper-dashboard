## Summary

Keep open. Current `main` does not contain this Teams parity work or an equivalent implementation: Teams `message/submitAction` handling still only routes feedback invokes, plugin runtime does not expose a `runtime.channel.msteams` surface, Teams allowlist/target parsing is still the older behavior, and the Graph/send helpers on `main` are still missing the broader compatibility changes described in the PR.
