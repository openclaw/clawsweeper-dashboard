## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Keep target parsing behind channel contracts:** `src/channels/plugins/target-parsing.ts:5-7`
  - body: This core helper now imports Discord, Telegram, and Teams private `extensions/*/src/**` parsers at module load. Current main uses loaded channel plugin contracts for this path; keeping direct extension imports here can eagerly load disabled channel internals and optional deps before the registry is available, so the rebase should keep parsing behind the channel contract or a lightweight public artifact.
  - confidence: 0.91
- **[P1] Migrate channel metadata before requiring graphChannelId:** `extensions/msteams/src/graph-messages.ts:140-143`
  - body: `graphChannelId` is newly introduced by this PR, but this branch immediately throws for stored channel conversations that lack it. Existing `msteams-conversations.json` entries can therefore break `getMessage`/pin/react/search Graph operations after upgrade until a fresh inbound message happens; add a doctor/backfill path or a safe fallback before requiring the field.
  - confidence: 0.93
- **[P1] Preserve graphChannelId on sparse activities:** `extensions/msteams/src/monitor-handler/message-handler.ts:288`
  - body: This writes `graphChannelId` on every upsert, including activities where `activity.channelData?.channel?.id` is absent. The merge helper preserves timezone only and then spreads incoming fields, so an undefined `graphChannelId` can erase previously cached metadata and make later Graph channel operations fail.
  - confidence: 0.9
- **[P1] Do not publish Teams internals through the SDK facade:** `src/plugin-sdk/msteams.ts:111-118`
  - body: These exports make Teams send/edit/delete/channel-edit internals part of `openclaw/plugin-sdk/msteams` and route the core runtime back through extension implementation files. That turns a host-owned Teams runtime operation into public SDK surface and creates a back-edge; keep this behind the approved runtime/channel contract or a narrow internal runtime module.
  - confidence: 0.86
