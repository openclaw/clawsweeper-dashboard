## What This Changes

## Best Possible Solution

Land a narrow Feishu/message-send patch that propagates `filename` through the existing payload and adapter contracts while preserving guarded media loading and UTF-8-safe upload sanitization.

## Solution Assessment

No. The filename plumbing is directionally right, but this branch is not the best fix because it bypasses media validation, drops upload sanitization, and carries unrelated security-sensitive files; a narrow patch on the existing loader/sanitizer path is safer.
