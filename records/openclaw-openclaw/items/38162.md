- **[medium] Relative symlink redirect rollback can be destructive:** `src/hooks/bundled/session-memory/handler.ts:563`
  - body: For relative `sessionSaveRedirectPath` values that point at an in-workspace symlink, the branch snapshots the lexical symlink path instead of the canonical write target, then writes through the symlink and may later unlink the real target on `blockSessionSave`.
