## Summary

Keep open. Current `main` still does not implement a `session:prune` lifecycle hook, so this is not `implemented_on_main` or obsolete. The PR still contains unique feature work for open issue `#8606`, but it also has unresolved review concerns around session identity propagation and hook behavior, so conservative cleanup should leave it open rather than close it.
