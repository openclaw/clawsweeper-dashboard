- **[P2] Parse authored config with the OpenClaw config contract:** `scripts/lib/agent-role-evals.mjs:1108`

  - confidence: 0.91

  - body: The static check errors when an agent's primary model is not present in `models.providers` or `agents.defaults.models`, but current docs and runtime selection allow `agents.defaults.model.primary` as the model choice while `agents.defaults.models` is a picker/catalog allowlist. This will make the harness fail valid configs, so it should delegate to the existing model-resolution contract or only flag explicit contradictions.
  - confidence: 0.88
