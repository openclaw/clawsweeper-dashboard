## What This Changes

Request Web UI assistant avatar changes to automatically sync back to the Telegram bot profile photo, while preserving Telegram-to-Web UI avatar sync.

## Best Possible Solution

Define a Telegram-plugin-owned, default-off bot profile-photo update flow using the correct Bot API profile-photo methods, account scoping, docs, focused tests, and live Telegram proof before deciding whether Control UI should call it.

## Reproduction Assessment

## Solution Assessment

No. The proposed file watcher plus setChatPhoto is not the best settled path because the UI override is browser-local and bot profile changes should be a Telegram-owned, opt-in action using the correct profile-photo API.
