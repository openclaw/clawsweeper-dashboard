## Summary

Keep open. Current main has ja-JP support for the browser Control UI and a separate Japanese docs translation pipeline, but the requested first-class core UX localization for CLI onboarding, help, workspace templates, and shared labels is still not implemented.

## What This Changes

Add key-based Japanese localization for OpenClaw core UX with ja-JP resources, English fallback, and initial coverage for onboarding, help, templates, and common labels.

## Best Possible Solution
