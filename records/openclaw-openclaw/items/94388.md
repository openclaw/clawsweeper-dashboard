- **[P2] Label bounded history positions accurately:** `src/auto-reply/reply/history-media.ts:129`
  - body: `historyPosition` and `historyTotal` are derived from `ctx.InboundHistory`, but that array is already bounded by the history window. Please either label this as recent-history/window position or carry true native thread counters before saying `in thread`.
