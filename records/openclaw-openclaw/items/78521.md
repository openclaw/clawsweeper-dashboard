## Summary

## Best Possible Solution

Keep the shared transport-boundary hardening, but always treat text `toolResult` content as untrusted, sanitize injected markers inside one fresh wrapper, add regression coverage for raw-marker injection, rebase, and require redacted real payload proof before merge.
