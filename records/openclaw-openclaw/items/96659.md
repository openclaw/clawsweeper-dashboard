## Summary

Keep open: current main and the latest v2026.6.10 release still have the source-level path that leaves a 2,048-token prompt budget on a 4,096-token model and routes positive overflow with no reducible tool content to `compact_only`; the cluster now has an open proof-positive candidate PR, but it has not merged yet.

## Best Possible Solution
