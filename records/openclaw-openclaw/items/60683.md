## Summary

Keep open. Current main still lacks the requested `memorySearch.local.gpu`, `gpuLayers`, and `flashAttention` config surface, so this PR is not obsolete, but the submitted patch is not mergeable as-is because it is stale against current local `contextSize` behavior, documents or wires an unsupported embedding-context option, misses generated schema/changelog work, and edits localized docs that this repo does not maintain.

This PR adds `memorySearch.local` tuning fields for local `node-llama-cpp` embeddings, wires them through config resolution into the local embedding provider, updates docs/schema/help labels, and adds focused tests.

Land a rebased or replacement implementation that exposes only supported node-llama-cpp local embedding options, preserves current `contextSize` defaults and `"auto"` support, updates generated schema/help and English docs, adds the required changelog entry, and leaves localized docs to the publish/i18n pipeline.
