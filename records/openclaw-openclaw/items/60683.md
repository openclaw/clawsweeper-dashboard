## Review Findings

Overall confidence: 0.9

Full review comments:

- **[P2] Remove unsupported flashAttention passthrough:** `packages/memory-host-sdk/src/host/embeddings.ts:152`
  - body: `node-llama-cpp` 3.18.1 does not define `flashAttention` on `LlamaEmbeddingContextOptions`; upstream exposes it on normal contexts and as `defaultContextFlashAttention` for model load defaults. Passing it to `createEmbeddingContext` creates a public config key that the embedding path does not honor, so remove it or wire it through a supported upstream option with runtime proof.
  - confidence: 0.91
- **[P2] Preserve contextSize default and auto support:** `packages/memory-host-sdk/src/host/embeddings.ts:116`
  - body: Current main defaults local embeddings to `contextSize: 4096` and accepts explicit `"auto"`, with tests covering both. This branch makes `contextSize` number-only and omits the default when unset, so rebasing as-is would restore node-llama-cpp's auto default and the VRAM spike the current config was added to avoid.
  - confidence: 0.9
- **[P3] Remove the localized docs edit:** `docs/zh-CN/concepts/memory.md:209`
  - body: `docs/AGENTS.md` says localized docs under `docs/<locale>/**` are not maintained here. Keep the English docs/config reference changes in this repo and let the docs publish/i18n pipeline regenerate the Chinese page.
  - confidence: 0.96
