## What This Changes

The issue reports Gemini 3.x subagent/cross-provider tool-call replay hangs because `thoughtSignature` is dropped or never synthesized for Google requests.

## Best Possible Solution

Land one focused repair that preserves real Gemini signatures through replay boundaries, uses Google's sentinel only for unsigned transferred function-call history, and adds regression coverage for native Google plus OpenAI-compatible Google replay.
