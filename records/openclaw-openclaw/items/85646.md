## Summary

## What This Changes

The PR moves the Codex app-server skills snapshot out of `turn/start.input` and into `collaborationMode.settings.developer_instructions`, while renaming the remaining user-input wrapper to workspace context and updating tests/snapshots.

## Best Possible Solution
