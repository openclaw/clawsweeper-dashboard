## What This Changes

The PR replaces Discord’s separate early typing cue and dispatcher typing owner with a carried reply typing feedback owner, starts it after accepted preflight for eligible Discord messages, reuses it through queue/dispatch, adds skipped-run cleanup, updates shared typing refresh behavior, and adds focused tests plus docs/changelog updates.

## Solution Assessment
