## What This Changes

The PR changes session-file repair to drop trailing errored assistant entries, adds a dropped-trailing counter and log segment, updates regression tests, and documents the Anthropic transcript-hygiene rule.

## Best Possible Solution

## Reproduction Assessment

Yes. The central bug reproduces with a session JSONL ending in `session -> user -> assistant` without replay tool calls; current main's repair tests exercise that shape and trim it before replay. I did not run tests because this review was required to stay read-only.

## Solution Assessment
