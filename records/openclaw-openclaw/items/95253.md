Keep open: the warning is a useful mitigation, but the patch is not merge-ready because it only probes the default gateway port while OpenClaw supports configured gateway ports, and the new install-time probe/env surface needs compatibility review.

The PR adds a postinstall-time loopback TCP probe and restart warning when a listener is detected on the default gateway port, plus an opt-out environment variable.
