## Best Possible Solution

Keep this PR open for maintainer review and either land or adapt the shared sanitizer cache while preserving full-payload keying, no-op MIME preservation, and byte-bounded eviction coverage.

## Reproduction Assessment

## Solution Assessment

Yes, likely. The shared sanitizer boundary is the narrowest maintainable fix because session replay, tool-result sanitization, and image block sanitization already route through it; per-channel or per-session fixes would duplicate the invariant.
