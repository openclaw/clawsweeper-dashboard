- **[P2] Preserve plugin warnings when suppressing the parent warning:** `src/commands/doctor/shared/empty-allowlist-scan.ts:126`

  - confidence: 0.89

  - body: The new `every(Object.values(channelAccounts))` check considers disabled accounts before the scanner's later disabled-account skip. If enabled accounts all have effective group allowlists but a disabled account lacks one, this keeps the false parent warning even though the disabled account is not scanned and cannot make the parent fallback relevant.
  - confidence: 0.84
