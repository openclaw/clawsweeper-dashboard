## Best Possible Solution

Land a maintainer-sponsored doctor/upgrade design that detects legacy/rebrand sibling workspaces, reports divergent data safely, offers a backup-preserving recovery path, and keeps memory writes anchored to the active resolved workspace.

Unclear as a complete fix plan. A detector plus regression coverage is a clear first step, but choosing warning/refuse/migrate behavior for divergent directories is a maintainer product and data-preservation decision.
