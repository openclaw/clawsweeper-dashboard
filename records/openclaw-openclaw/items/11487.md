## What This Changes

Request adding an optional ignored string metadata field, such as `name` or `label`, to group/channel config entries so numeric room IDs are easier to identify.

## Best Possible Solution

Choose one documented optional string metadata key and add it consistently to relevant group/channel entry types, strict schemas, generated metadata, docs, and validation coverage while ignoring it at runtime.
