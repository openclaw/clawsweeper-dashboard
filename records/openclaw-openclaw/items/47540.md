## Best Possible Solution

Rebase or replace the branch on current main, implement the threshold split in the current `ui/src/ui/chat/context-notice.ts` model, preserve bootstrap auth and fresh-token semantics, settle the pricing-threshold source of truth, and update only the current config baseline surfaces.
