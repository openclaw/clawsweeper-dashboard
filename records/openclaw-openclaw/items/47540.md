Rebase or replace the branch on current main, extend the current `ui/src/ui/chat/context-notice.ts` fresh-context model, preserve the authenticated bootstrap response path, settle the threshold source of truth, update only the current SHA baseline artifacts, and add redacted runtime proof of the visible Control UI notice.
