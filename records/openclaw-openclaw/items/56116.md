Full review comments:

- **[P1] Gate scratchpad filtering away from final replies:** `extensions/feishu/src/reply-dispatcher.ts:519-520`
  - body: At this call site, `hasText` becomes false for any openai-codex text matched by the broad heuristic, including `info.kind === "final"`. The added test expects such a final message to send no Feishu reply, so normal English final answers can silently disappear; restrict filtering to non-final/internal chunks or otherwise guarantee final replies are delivered.
  - confidence: 0.94
- **[P1] Preserve reasoning-preview gating during rebase:** `extensions/feishu/src/reply-dispatcher.ts:659-668`
  - body: PR head exposes `onReasoningStream` whenever Feishu streaming is enabled, but current main gates reasoning previews behind `allowReasoningPreview`. Rebase this branch so ordinary Feishu chat cards do not receive hidden reasoning streams by default.
  - confidence: 0.88
- **[P2] Stop the heartbeat on failed startup and cleanup:** `extensions/feishu/src/reply-dispatcher.ts:434`
  - body: The new heartbeat starts after launching the async streaming start, but a failed start only nulls stream state and `onCleanup` only clears typing. That leaves the interval alive after failed startup or dispatcher cleanup; stop it in those paths.
  - confidence: 0.86
- **[P2] Check completion ping before resetting stream state:** `extensions/feishu/src/reply-dispatcher.ts:561-563`
  - body: `closeStreaming()` resets `streamingStartedAt` to 0 before this check, so `shouldSendCompletionPing()` is always false on the final path and the advertised long-task completion ping never sends. Capture the elapsed/should-ping state before closing or reset after the ping.
  - confidence: 0.91
- **[P2] Expose statusLanguage in public config surfaces:** `extensions/feishu/src/config-schema.ts:186`
  - body: The PR adds a user-facing `statusLanguage` schema key, but the Feishu plugin manifest and channel docs still omit it. Keep schema, manifest metadata, and docs aligned so setup/doctor/docs expose the same config contract.
  - confidence: 0.82
