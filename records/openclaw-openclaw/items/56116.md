## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Do not filter final replies as scratchpad text:** `extensions/feishu/src/reply-dispatcher.ts:519-520`
  - body: hasText is cleared whenever the broad Codex scratchpad heuristic matches, before checking info.kind. A normal final English answer containing words like could, need, or update can therefore return without sending anything; restrict this filter to non-final/internal chunks or require a stronger scratchpad signal.
  - confidence: 0.91
- **[P2] Stop heartbeat timers during cleanup:** `extensions/feishu/src/reply-dispatcher.ts:434`
  - body: startHeartbeat() creates an interval, but onCleanup does not stop it or close streaming. If the dispatcher is torn down before onIdle/onError, the interval can survive and keep dispatcher state reachable; stop the heartbeat in cleanup as well.
  - confidence: 0.87
- **[P2] Check completion pings before closing streaming:** `extensions/feishu/src/reply-dispatcher.ts:561-563`
  - body: The final path calls closeStreaming() before shouldSendCompletionPing(), and closeStreaming() resets streamingStartedAt to zero. Long streaming tasks will never emit the advertised completion ping; capture the duration or decision before closing.
  - confidence: 0.9
- **[P3] Add the required docs and changelog entry:** `extensions/feishu/src/config-schema.ts:186`
  - body: statusLanguage is a new user-facing Feishu account config and the PR also changes visible streaming/completion behavior, but the Feishu docs and CHANGELOG.md are not updated. Add the option and behavior notes before merge.
  - confidence: 0.78
