## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.89

Full review comments:

- **[P1] Do not filter final replies with the scratchpad heuristic:** `extensions/feishu/src/reply-dispatcher.ts:117-125`
  - body: This broad English heuristic matches normal final replies such as “I could not find the file you requested. Please try again.” The deliver path applies it before checking `info.kind`, then returns when no text/media remains, so Feishu users can receive no answer at all. Restrict filtering to typed non-final/internal progress or keep final replies out of this filter.
  - confidence: 0.91
- **[P2] Expose statusLanguage in plugin metadata and docs:** `extensions/feishu/src/config-schema.ts:186`
  - body: The runtime schema now accepts `statusLanguage`, but the Feishu plugin manifest and channel docs do not expose it. That makes setup, discovery, and documented config drift from the accepted runtime surface; either add the matching metadata/docs or remove the option from this PR.
  - confidence: 0.84
- **[P2] Stop the streaming heartbeat on cleanup:** `extensions/feishu/src/reply-dispatcher.ts:434`
  - body: `startHeartbeat()` runs even while streaming start can still fail, and `onCleanup` only clears typing state. If the dispatcher is torn down before idle/error cleanup, the interval can keep firing note updates after the session is gone; stop it on failure and cleanup.
  - confidence: 0.82
