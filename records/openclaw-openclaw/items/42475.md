The issue requests optional per-agent daily/monthly model spend budgets enforced before gateway provider dispatch.

## Best Possible Solution

Define a native gateway spend-admission contract covering config, authoritative accounting or reservation, fallback attribution, structured over-budget receipts/errors, and operator visibility before wiring enforcement into provider dispatch.
