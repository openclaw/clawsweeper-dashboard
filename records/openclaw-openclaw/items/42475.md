Keep open. Request optional per-agent cost budgets enforced before gateway model dispatch, with daily/monthly caps, configurable over-budget actions, and status/doctor visibility.

## Best Possible Solution

Approve a maintainer-owned design for an opt-in budget policy that reuses existing cost accounting and pre-run hooks where appropriate, then implement schema, admission/reservation, persistence, status, doctor, docs, and targeted upgrade tests together.

## Solution Assessment
