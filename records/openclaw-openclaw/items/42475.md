## Summary

Keep open. Request optional global and per-agent daily/monthly cost budgets enforced before gateway model dispatch, with status/doctor visibility and block/warn/pause actions.

## Best Possible Solution

Define one native gateway spend-admission contract covering config, accounting/reservation, fallback attribution, structured over-budget errors, and operator visibility, then implement it against provider dispatch with regression coverage.

## Solution Assessment

Unclear as a final implementation. Gateway admission before provider dispatch is the right boundary, but schema, accounting, reservation, reset, fallback, and action semantics need maintainer product decisions first.
