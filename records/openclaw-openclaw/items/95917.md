## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P1] Validate xcode_path outside the shell template:** `.github/workflows/build-ios-unsigned.yml:50`
  - body: `xcode_path` is expanded directly into the generated shell script. A dispatch value containing shell substitution can run before `xcode-select`, so pass it through `env`, validate the allowed Xcode path in bash, and then quote the variable.
  - confidence: 0.86
- **[P1] Do not interpolate build_number into the run command:** `.github/workflows/build-ios-unsigned.yml:68`
  - body: The `build_number` input is inserted unquoted into the shell command, so a crafted manual-dispatch value can append commands before `ios-write-version-xcconfig.sh` gets to enforce digits-only validation. Pass the raw input through `env` and build the argument after validation.
  - confidence: 0.88
