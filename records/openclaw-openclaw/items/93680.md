  - body: `startCtx` is passed to every registered node-host plugin startup hook, and it contains `emitNodeGatewayEvent`. Omitting that field from the public TypeScript type is not a security boundary because plugin code can read runtime object properties, so any plugin with this hook can still originate node-attributed gateway events. Keep this emitter on a host-owned or bundled-browser-only path instead of passing it to all plugin hooks.
