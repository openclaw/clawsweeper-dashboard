Full review comments:

- **[P1] Write a schema-valid My Browser profile:** `apps/macos/Sources/OpenClaw/ConfigStore.swift:178`
  - body: The macOS toggle saves `profiles.mybrowser = ["driver": "extension"]`, but the PR-head browser profile schema still requires `color`. Because `ConfigStore.save` validates the submitted config, enabling My Browser can fail instead of persisting the new mode.
  - confidence: 0.92
- **[P1] Authenticate the extension bridge socket:** `extensions/browser/src/browser/extension-bridge-server.ts:268`
  - body: `background-utils.js` derives a relay HMAC for `/extension`, but this upgrade path accepts any `/extension` WebSocket and reaches `handleExtension` without validating that token. That socket can send `agent.request` frames into the node-host emitter, so a local client can impersonate the extension path and originate node-attributed turns or CDP frames.
  - confidence: 0.91
- **[P1] Keep node event emission off the broad SDK surface:** `src/plugin-sdk/gateway-runtime.ts:36-40`
  - body: `plugin-sdk/gateway-runtime` is a public package export, but this exposes node-event emission and process-global emitter registration to plugins. The browser bridge already has an injected `onAgentRequest` callback, so keep registration internal or introduce a narrow documented contract with maintainer approval and compatibility tests.
  - confidence: 0.84
- **[P2] Do not send a fake loopback gateway token:** `extensions/chrome-extension/sidepanel.js:189`
  - body: When no gateway token is configured, the side panel substitutes `test-token-for-container` for loopback gateways and sends it as `auth.token`. Token-auth local gateways can treat this as a mismatch and fail or rate-limit instead of prompting for the real credential.
  - confidence: 0.87
- **[P2] Forward the browser copilot prompt to the agent run:** `extensions/browser/src/browser/server-lifecycle.ts:50`
  - body: This adds `extraSystemPrompt` to the emitted `agent.request` payload, but the `server-node-events` parser and dispatch path do not read or forward that field. The advertised browser-copilot persona will be silently dropped unless the node-event contract and tests are updated or the prompt is injected at a supported call site.
  - confidence: 0.9
