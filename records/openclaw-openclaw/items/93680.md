  - body: `openclaw/plugin-sdk/gateway-runtime` is a published public SDK subpath, and this export lets plugin code originate authenticated node events such as `agent.request`. Keep this host-internal or add a typed, maintainer-approved SDK contract and permission model before exposing it.

  - confidence: 0.92

  - body: When `relayPort` is configured, the service worker derives `relayHost` from `gatewayUrl`; the bundled example uses a remote gateway URL, while the bridge listens on `127.0.0.1:18792`.
