## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.89

Full review comments:

- **[P1] Authenticate the extension bridge socket:** `extensions/browser/src/browser/extension-bridge-server.ts:268`
  - body: `background-utils.js` derives a relay HMAC for `/extension`, but the bridge upgrade path accepts any `/extension` WebSocket and reaches `handleExtension` without validating that token or an equivalent capability. Because that socket can send `agent.request` frames into the node-host emitter, a local client can impersonate the extension path and originate node-attributed turns or CDP frames.
  - confidence: 0.91
- **[P2] Do not send a fake loopback gateway token:** `extensions/chrome-extension/sidepanel.js:189`
  - body: When no gateway token is configured, the side panel substitutes `test-token-for-container` for loopback gateways and sends it as `auth.token`. Current gateway auth treats a wrong token as a token mismatch and records a failure, so token-auth local gateways can fail or rate-limit instead of prompting for the real credential.
  - confidence: 0.87
- **[P1] Keep node event emission off the broad SDK surface:** `src/plugin-sdk/gateway-runtime.ts:36-40`
  - body: `plugin-sdk/gateway-runtime` is a public package export, but this exposes both node-event emission and process-global emitter registration to plugins. The browser bridge already has an injected `onAgentRequest` callback, so keep registration internal or introduce a narrow documented contract with maintainer approval and compatibility tests.
  - confidence: 0.84
- **[P2] Forward the browser copilot prompt to the agent run:** `extensions/browser/src/browser/server-lifecycle.ts:50`
  - body: This adds `extraSystemPrompt` to the emitted `agent.request` payload, but the `server-node-events` parser and dispatch path do not read or forward that field. The advertised browser-copilot persona will be silently dropped unless the node-event contract and tests are updated or the prompt is injected at a supported call site.
  - confidence: 0.9
