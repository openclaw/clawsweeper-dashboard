## What This Changes

The issue requests mid-stream soft steering that can pause or abort an active generation, inject the user's new message into context, and resume without waiting for the next runtime boundary.

## Best Possible Solution

Keep this as the soft-steer tracker and land https://github.com/openclaw/openclaw/pull/73365 or a clean successor after defining streaming, tool-execution, abort/resume, fallback, backend-failure, and regression-test semantics.
