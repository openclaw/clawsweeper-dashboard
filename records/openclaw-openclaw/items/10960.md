## What This Changes

The issue requests mid-stream user-message injection that can pause or abort an active generation, add the new user input to context, and resume with that steering input.

## Best Possible Solution

Keep this issue as the canonical runtime tracker, then land https://github.com/openclaw/openclaw/pull/73365 or a maintainer-approved successor with documented lifecycle semantics and regression coverage.

## Solution Assessment

Unclear until maintainer review completes. The maintainable solution is a lifecycle-aware active-run steering and interruption contract with explicit stopped-state, fallback, backend compatibility, bounds, docs, and regression coverage rather than a queue-policy-only tweak.
