## What This Changes

The issue requests a soft-steer capability that can interrupt an active generation, inject a new user message into the same session context, and resume without waiting for the next model or tool boundary.

## Best Possible Solution

Keep this as the product/runtime tracker and review or land https://github.com/openclaw/openclaw/pull/73365, https://github.com/BingqingLyu/openclaw/pull/2307, or a narrower successor after maintainers decide default versus opt-in soft-steer semantics and document the lifecycle.
