Full review comments:

- **[P1] Export the helper from the facade you re-export:** `packages/memory-host-sdk/src/engine-foundation.ts:11`
  - body: `computeSharedScopeHash` is added to the re-export list from `./host/openclaw-runtime-agent.js`, but that facade does not export the helper. This will fail the package build until the helper is exported through the facade or re-exported from the module that actually defines it.
  - confidence: 0.94
- **[P1] Write sharedStorePath on the store object:** `src/agents/memory-search.ts:492-498`
  - body: The new type declares `store.sharedStorePath`, but this code writes `resolved.sharedStorePath` and `mergeConfig` never initializes the required store field. Strict TypeScript will reject the patch and consumers would not receive the field at the declared location.
  - confidence: 0.92
- **[P1] Keep session chunks out of shared stores:** `src/agents/memory-search.ts:493-498`
  - body: This override also runs for configs that include both `memory` and `sessions`. Current session sync writes per-agent transcript chunks into the active DB, while search/schema only separate by source, so same-workspace agents can retrieve each other's session chunks unless the shared store is memory-only or agent membership filtering is added.
  - confidence: 0.9
- **[P1] Add an upgrade path before moving the memory DB:** `src/agents/memory-search.ts:497-498`
  - body: Changing the resolved store from each agent's existing SQLite DB to `state/memory/shared-*.sqlite` makes upgraded users miss their current index and can trigger a full embedding rebuild. This needs an explicit doctor migration, rebuild/warning path, or other maintainer-approved upgrade plan before the default path changes.
  - confidence: 0.88
- **[P2] Do not publish registry helpers before the contract exists:** `packages/memory-host-sdk/src/engine-storage.ts:52-56`
  - body: The new registry helpers are exported through the memory-host SDK even though the shared-store lifecycle, migration, and access-boundary contract is not accepted or wired into callers. Keep this helper private or defer it until maintainers approve the persistent shared-store API.
  - confidence: 0.8
