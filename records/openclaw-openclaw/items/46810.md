## Summary

Keep this issue open. Current main has the Control UI exec approval modal, countdown, execution-context rows, queue indicator, scrollable command block, and allow-once/allow-always/deny actions, but the requested command syntax highlighting is still missing because the exec command is rendered as one plain Lit text interpolation.

## What This Changes

Requests a Control UI exec approval modal with highlighted command rendering, execution context labels, one-click approval actions, and timeout state.

## Best Possible Solution

Finish the remaining Control UI polish with a UI-only highlighter for the already-sanitized command string, using escaped Lit template parts and CSS token classes while preserving the existing modal behavior and gateway protocol.
