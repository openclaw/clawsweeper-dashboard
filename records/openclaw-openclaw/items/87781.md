## Best Possible Solution

Land a scoped recovery fix that only counts turn-owned native deltas, preserves unrelated or intentionally preserved bindings on timeout, and keeps runtime, docs, and manifest timeout defaults aligned with focused Codex app-server recovery tests and live Telegram proof.
