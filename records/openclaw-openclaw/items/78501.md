## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Honor the parent oxlint skip lock:** `scripts/run-oxlint-shards.mjs:8-12`
  - body: `check:changed` already holds the shared heavy-check lock and runs children with `OPENCLAW_OXLINT_SKIP_LOCK=1`. This wrapper ignores that flag and immediately acquires the same lock, so a changed-check lane that reaches sharded lint can wait behind its own parent lock until timeout.
  - confidence: 0.9
- **[P2] Gate the install refusal to source checkouts:** `scripts/preinstall-package-manager-warning.mjs:159-165`
  - body: `git rev-parse --is-inside-work-tree` is true for `node_modules/openclaw` installed inside a consumer Git repository, so this can make the packaged `preinstall` lifecycle fail on low-memory or high-load hosts. Use an actual OpenClaw source-checkout marker or keep packaged install warnings non-fatal.
  - confidence: 0.88
