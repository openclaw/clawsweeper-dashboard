  - body: The new cache is keyed only by channel id even though the Mattermost lookup is account/client-specific. Seed the cache or change the implementation to use a real authoritative route value so the test covers the source the route actually reads.
  - confidence: 0.94

  - body: The route contract requires `resolvedTarget` to include `to`, `kind`, and `source`; this fixture supplies an `id` field instead and omits the required fields. Use a production-shaped resolved target so test typechecking and coverage catch contract drift.
