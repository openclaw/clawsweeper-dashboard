## What This Changes

The issue asks for Telegram exec approval messages to be dismissed after the user clicks approve.

## Best Possible Solution

Land or replace the linked PR with a focused change in the active Telegram approval runtime that clears actions immediately, deletes resolved approval messages through the existing Telegram delete helper/seam, and covers approve and deny flows.
