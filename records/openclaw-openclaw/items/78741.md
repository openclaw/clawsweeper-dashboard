## Solution Assessment

Yes, this is the best bounded fix shape if maintainers accept the policy: it changes the legacy UI storage restore boundary without touching gateway routing, heartbeat targeting, or live session switching. If maintainers reject that policy, the safer alternative is to narrow the classifier or move the rule into a shared session-key contract.
