  - confidence: 0.92
- **[P2] Sanitize rename labels before gateway logging:** `src/gateway/server-methods/devices.ts:685`
  - body: `displayName` is validated only as non-empty text, then `trimmed` is interpolated directly into a Gateway log line. Labels containing newlines or terminal/control sequences can forge or corrupt log output; omit the label from that log or pass it through the existing safe terminal/log sanitizer first.
  - confidence: 0.87
