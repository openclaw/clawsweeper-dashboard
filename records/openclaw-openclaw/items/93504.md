This PR routes device-pairing role normalization through the shared guarded string-list normalizer and adds regression tests for malformed pending, approved, and token role values.

## Best Possible Solution

Land the narrow shared-normalizer change and tests so malformed persisted role values are dropped consistently with scopes and sibling pairing merges instead of crashing device approval/list rendering.
