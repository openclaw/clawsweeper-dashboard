## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.92

Full review comments:

- **[P1] Preserve secret-file validation errors:** `src/infra/secret-file.ts:29-30`
  - body: This catch-all wrapper turns every readSecretFileSync failure into `undefined`, including `rejectSymlink: true` credential files. Current main deliberately re-exports the upstream helper so blank/missing paths can be optional while symlink, oversized, non-regular, and other validation failures fail closed; returning `undefined` here would silently skip configured secrets instead of surfacing unsafe files.
  - confidence: 0.95
- **[P1] Avoid Object.assign on raw extra params:** `extensions/openrouter/provider-routing.ts:72-76`
  - body: `ctx.extraParams` is caller-controlled transport data, and Object.assign invokes the `__proto__` setter on the target object for an own `__proto__` key. Current main uses object spread plus explicit provider sanitization; this change can mutate the returned patch object's prototype before it reaches transport code.
  - confidence: 0.93
- **[P1] Keep source-delivery matches on provider normalizers:** `src/infra/outbound/source-delivery-plan.ts:115`
  - body: Replacing `normalizeTargetForProvider` with local prefix stripping drops provider-owned normalization for valid target forms. For example, Discord's normalizer maps a bare numeric channel ID to `channel:<id>`; with this local comparison, a message-tool send to the same channel can be treated as unverified, which can trigger fallback delivery or delivery-target errors.
  - confidence: 0.88
- **[P1] Do not change the fs-safe Python default in this refresh:** `src/infra/fs-safe-defaults.ts:13-15`
  - body: This changes the documented OpenClaw default from Python helper `off` to `auto`, which can start an optional Python helper in existing installs unless operators opted out. That is an upgrade-sensitive runtime/default change and needs a focused PR with explicit maintainer approval, docs alignment, and fresh-install plus upgrade proof.
  - confidence: 0.9
