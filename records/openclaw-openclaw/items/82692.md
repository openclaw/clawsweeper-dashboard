  - body: This flips OpenClaw's fs-safe Python helper from the documented `off` default to `auto` whenever no operator override is set.   - confidence: 0.91

  - body: `ctx.extraParams` is raw caller/config data, and `Object.assign({}, ctx.extraParams)` invokes the `__proto__` setter instead of preserving it as an own data property like object spread. That cuts against the blocked-key guardrails in this file and can return a patch object with a polluted prototype.
