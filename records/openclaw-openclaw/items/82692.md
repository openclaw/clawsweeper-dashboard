  - body: `ctx.extraParams` is caller-controlled transport data, and `Object.assign` invokes the `__proto__` setter for an own `__proto__` key. Current main uses object spread, which preserves a normal prototype; keep that construction or sanitize into a null-prototype object before merging.
  - confidence: 0.95

  - body: This local prefix stripping drops provider-owned normalization for valid target forms.   - confidence: 0.88
