## What This Changes

The PR makes isolated cron executor `sourceDelivery` optional, reconstructs a cron source-delivery plan from job delivery config when it is missing, moves the normal cron planner into a shared helper, and adds focused guard tests.

## Best Possible Solution

Land this shared fallback only if maintainers intentionally want runtime tolerance for stale executor artifacts with defaults tied to current cron delivery semantics; otherwise choose rebuild or reinstall remediation instead of adding this compatibility path.
