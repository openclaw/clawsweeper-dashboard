## Summary

## What This Changes

The branch makes isolated cron sourceDelivery optional at the executor boundary, moves cron source-delivery planning into a shared helper, and adds fallback/parity tests for missing sourceDelivery.

## Best Possible Solution

Land the shared planner guard only after redacted terminal or log proof from an installed or packaged OpenClaw cron run shows the missing-sourceDelivery path completes without crashing and without duplicate final delivery.

## Solution Assessment

Yes, the shared helper is the best code shape among the obvious options because it prevents normal and fallback cron delivery policy from drifting. The remaining blocker is proof quality, not a specific code defect found in this pass.
