## Summary

Keep open: this is an active fix PR with closing syntax for an open P1 isolated-cron crash report, and current main still has the unguarded executor dereference when a stale caller omits `sourceDelivery`. ## What This Changes

The branch makes isolated cron executor `sourceDelivery` optional, rebuilds a `SourceDeliveryPlan` from cron delivery config when absent, shares that planner with the normal path, and adds focused regression tests.

## Best Possible Solution

Land a single canonical cron source-delivery planner and guarded executor fallback after maintainers accept the version-skew contract, then let the merge close the linked crash reports.
