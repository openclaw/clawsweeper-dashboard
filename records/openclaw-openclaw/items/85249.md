## Summary

## What This Changes

The PR makes cron executor source-delivery input optional, reconstructs it from job delivery state when absent, centralizes cron source-delivery planning, and adds focused regression coverage.

## Best Possible Solution

Treat the shared reconstruction as a named, warning-backed shipped-upgrade compatibility boundary with parity and duplicate-delivery coverage, or fix the packaging/restart boundary instead and retain the required executor contract.
