## What This Changes

The PR makes isolated cron executor `sourceDelivery` optional, reconstructs a fallback source-delivery plan from cron job delivery config, shares that planner with the normal path, and adds guard/parity tests.

## Best Possible Solution

If maintainers accept the compatibility contract, land the shared fallback after required checks pass and let it close the canonical cron crash issue; otherwise reject the runtime fallback and document rebuild or upgrade remediation.
