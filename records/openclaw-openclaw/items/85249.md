## What This Changes

The PR makes isolated cron execution tolerate missing `sourceDelivery` by rebuilding the cron source-delivery plan from job delivery config and resolved delivery, shares that planner with the normal path, and adds regression coverage.

## Best Possible Solution

Land this guard if maintainers accept runtime compatibility for stale executor callers; otherwise replace it with an explicit rebuild or doctor-owned remediation path while keeping one canonical cron source-delivery planner.
