## Summary

## What This Changes

The PR makes `sourceDelivery` optional at the isolated cron executor boundary, adds a shared cron source-delivery planner/fallback helper, rewires the normal path to that helper, and adds focused guard/parity tests.

## Best Possible Solution

Land after maintainers explicitly accept the stale-artifact compatibility behavior, with optional live announce-transport proof if they want visible delivery confirmation before merge.
