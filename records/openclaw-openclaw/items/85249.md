This PR makes isolated cron executor `sourceDelivery` optional at the lazy execution boundary, reconstructs missing plans through a shared cron source-delivery helper, and adds guard/parity tests.

## Best Possible Solution

Merge the shared-planner fallback after maintainers accept current-job-config recovery semantics; a live Telegram cron smoke would reduce the remaining delivery-path uncertainty.

## Solution Assessment
