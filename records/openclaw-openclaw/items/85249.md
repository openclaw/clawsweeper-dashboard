## Summary

## What This Changes

The PR makes isolated cron source-delivery optional, reconstructs a fallback plan from cron job delivery state, shares that planner with the normal cron path, and adds regression/parity tests.

## Best Possible Solution

Merge the shared guarded planner once maintainers accept the runtime-skew fallback contract and required checks are clean; otherwise choose packaging, restart, or doctor remediation instead of steady-state runtime tolerance.

## Solution Assessment
