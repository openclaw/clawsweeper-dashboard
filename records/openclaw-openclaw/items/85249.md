## Summary

## What This Changes

The branch makes `sourceDelivery` optional at the isolated cron executor boundary, adds a shared cron source-delivery fallback/planner helper, rewires the normal planner to use it, and adds focused guard/parity tests.

## Best Possible Solution

## Solution Assessment
