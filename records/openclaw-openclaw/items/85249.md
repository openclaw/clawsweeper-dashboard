## What This Changes

The PR makes isolated cron executor `sourceDelivery` optional at the lazy executor boundary, reconstructs a cron `SourceDeliveryPlan` from job delivery config when absent, shares that planner with the normal path, and adds guard tests.

## Best Possible Solution

Land the shared planner and guarded executor fallback after maintainers accept the current-semantics version-skew fallback contract, with the branch refreshed against current main if needed.
