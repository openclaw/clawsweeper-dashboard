## Summary

## What This Changes

The branch makes isolated cron executor `sourceDelivery` optional, rebuilds missing plans through a shared cron source-delivery planner, and adds guard/parity tests.

## Best Possible Solution

Land the shared guarded fallback if maintainers accept current cron job delivery config as the recovery policy for stale executor callers; otherwise request one packaged skew smoke before merge.
