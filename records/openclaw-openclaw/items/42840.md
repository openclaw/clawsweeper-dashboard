## What This Changes

The issue requests MathJax/LaTeX rendering support in the Control UI so inline and display formulas render in chat instead of appearing as raw source text.

## Best Possible Solution

Review, land, or replace the open implementation with a maintainer-approved shared Control UI markdown renderer change that safely supports inline and display math while preserving existing markdown, sanitizer, cache, dependency, and fallback behavior.

## Reproduction Assessment

## Solution Assessment
