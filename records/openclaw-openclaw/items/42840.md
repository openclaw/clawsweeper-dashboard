## Summary

Keep open. Current main still lacks LaTeX/MathJax/KaTeX rendering in the Control UI markdown path, and #67103 remains an open implementation candidate rather than a merged fix.

## What This Changes

Add MathJax/LaTeX-style formula rendering support to Control UI chat messages, preferably with configurable delimiter behavior.

## Best Possible Solution

Land a reworked shared Control UI markdown renderer implementation with sanitizer-aware tests for inline math, display math, invalid LaTeX, code spans/fences, currency prose, and normal markdown compatibility.

## Solution Assessment
