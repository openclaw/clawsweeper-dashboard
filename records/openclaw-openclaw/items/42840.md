## Summary

Keep open. Current main still lacks MathJax/KaTeX rendering in the Control UI markdown path, and the related open PR #67103 is an implementation candidate rather than a merged fix or a reason to queue duplicate automation.

## What This Changes

Add MathJax/LaTeX rendering support to Control UI chat so inline and display formulas render as equations instead of raw source text.

## Best Possible Solution

Implement a sanitizer-aware math-rendering layer in `ui/src/ui/markdown.ts`, with focused tests for inline math, display math, invalid formulas, code spans/fences, ordinary dollar amounts, sanitizer output, and normal markdown compatibility; close after PR #67103 or an equivalent implementation lands.
