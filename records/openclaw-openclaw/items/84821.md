## What This Changes

The issue asks OpenClaw to avoid sending duplicate live tool websocket frames to clients that receive both run-scoped `agent` events and session-scoped `session.tool` mirrors while preserving mirrors for session-only subscribers.

## Best Possible Solution

Keep the shipped recipient-dedupe implementation, resolve or close https://github.com/openclaw/openclaw/pull/84823 as superseded, and then let a maintainer close this protected issue if no live regression evidence remains.

## Solution Assessment
