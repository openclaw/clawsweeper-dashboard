  - confidence: 0.9

  - body: `sharedPaths` entries are mapped without trimming/filtering empty strings, and the later shared read check uses prefix matching. A blank path can become an empty shared root, which makes absolute Markdown paths match the shared-path guard on POSIX and broadens file-read access outside the configured roots.
  - confidence: 0.88
- **[P2] Do not create directories for file-valued shared paths:** `src/memory/manager.ts:248-251`
  - body: The constructor calls `ensureDir` for every configured shared path, but the same paths are treated like `extraPaths` and may be individual Markdown files. A not-yet-created `policy.md` shared path would be created as a directory and block the intended file until manual cleanup.
  - confidence: 0.84
