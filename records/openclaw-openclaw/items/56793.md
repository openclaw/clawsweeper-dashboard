## What This Changes

The issue requests an explicit one-shot command or API to import an existing-session browser profile into a managed OpenClaw browser profile.

## Best Possible Solution

If maintainers accept the feature, implement it in the browser plugin as an explicit opt-in import with validation, destination stop/reset, documented exclusions, managed-profile restoration, focused tests, and docs.
