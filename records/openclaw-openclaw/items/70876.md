  - confidence: 0.88

  - body: The new `peerDependencies.openclaw` removal sits after early returns that require workspace devDependencies to be rewritten. Packages with only an `openclaw` peer, or no workspace devDependency rewrite, still keep that peer.
  - confidence: 0.84

  - body: The test now expects `--omit=peer`, but the production npm command still omits only dev dependencies and the nearby comment says preserving other peers is intentional. As written, the test fails or pressures production toward dropping legitimate runtime peers.
  - confidence: 0.94
