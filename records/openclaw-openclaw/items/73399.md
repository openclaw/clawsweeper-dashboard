  - body: When `chatMembers` returns one member whose `member_id` does not match the event sender, this fallback still accepts that member as the sender.   - confidence: 0.9

  - body: The new direct-member cache, plus the widened sender cache keys, can grow with account/chat/sender identifiers and PR-head code never removes expired entries or caps size. A long-running Feishu process can accumulate stale entries until memory pressure affects availability, and the branch also predates current-main stale deletion.
  - confidence: 0.88
- **[P2] Validate Feishu grant URLs before agent guidance:** `extensions/feishu/src/bot-sender-name.ts:52-65`
  - body: The widened permission-error parser accepts raw response objects and forwards the first `https://.../app/...` URL into the agent-visible permission notice without checking the host. Only official Feishu/Lark grant hosts should be surfaced, or the notice can carry an untrusted phishing/prompt-format link.
