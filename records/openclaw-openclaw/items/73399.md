Full review comments:

- **[P2] Require an exact member match before using the name:** `extensions/feishu/src/bot-sender-name.ts:129-131`
  - body: When `chatMembers.get` returns exactly one member whose `member_id` does not match the sender id, this fallback still uses that member's name. A short, permission-filtered, or otherwise incomplete member response can therefore attribute the inbound DM to the wrong human; fall through to the contact fallback unless the member id matches.
  - confidence: 0.86
- **[P2] Bound the new direct-member cache:** `extensions/feishu/src/bot-sender-name.ts:29`
  - body: `directMemberNameCache` is keyed by account, chat, and sender and only expires entries logically; old keys are never pruned and there is no max size. A long-running Feishu bot that sees many unique DMs can grow this map indefinitely, so reuse the current-main stale-entry cleanup pattern and add a small cap or pruning path.
  - confidence: 0.82
