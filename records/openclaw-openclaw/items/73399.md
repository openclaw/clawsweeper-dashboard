- **[P1] Require an exact member-id match for DM names:** `extensions/feishu/src/bot-sender-name.ts:141-143`
  - body: When no returned `member_id` matches the inbound sender id, this falls back to the sole returned member and then stamps that name into the agent-visible body and session metadata. A partial page, unexpected Feishu response, or id-type mismatch can therefore attribute a DM to the wrong person; fall through to contact/id fallback unless the member id matches the sender.
  - confidence: 0.9

  - body: `directMemberNameCache` is process-wide and keyed by account, chat, and sender, but entries are only checked for one exact key and are never capped or swept. A busy or hostile Feishu deployment can grow this map indefinitely; use the bounded/expiry pattern from the group-name cache or a shared bounded TTL helper.
  - confidence: 0.86
