  - body: `chatMembers.get` is called for a specific sender, but this fallback accepts the only returned member even when its `member_id` does not match the sender.   - body: The newly added direct-member cache, and the widened sender cache key space, use process-wide Maps with only expiry timestamps and no maximum size or active pruning.   - confidence: 0.9

  - body: This widens permission-error parsing to raw response objects, then extracts any HTTPS `/app/` URL from remote error text for agent-visible permission guidance. Please only accept known Feishu/Lark open-platform hosts so a crafted upstream error cannot present an attacker-controlled app URL as an admin authorization link.
  - confidence: 0.86
