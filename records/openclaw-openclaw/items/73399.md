  - body: The fallback accepts `members[0]` whenever the API returns one item, even when that member id does not match the sender. A long-running Feishu process can retain one entry per observed account/chat/sender combination, so the cache needs bounded stale pruning with focused tests.
  - confidence: 0.9
- **[P2] Validate Feishu grant URLs before agent guidance:** `extensions/feishu/src/bot-sender-name.ts:52-65`
  - body: By accepting the raw error object as Feishu error data, this path can carry a remote `msg`/`message` URL into `permissionErrorForAgent`, which is later inserted into agent-visible system guidance. Validate the URL against trusted Feishu/Lark hosts or drop it before surfacing permission guidance.
