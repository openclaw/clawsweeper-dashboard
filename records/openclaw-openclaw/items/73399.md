- **[P1] Require exact DM member matches:** `extensions/feishu/src/bot-sender-name.ts:141-143`
  - body: The new DM lookup falls back to `members[0]` whenever the chatMembers response has one item, even if that item's `member_id` does not equal the sender id. - **[P1] Bound sender-name caches:** `extensions/feishu/src/bot-sender-name.ts:31`
  - body: This adds another process-wide Map and widens the existing sender cache keys to account/chat/sender-derived values without a max size or general pruning. Long-running Feishu gateways can accumulate entries from distinct users and chats, so mirror the bounded group-name cache pattern or another capped TTL cache.
  - confidence: 0.9
- **[P2] Validate Feishu grant URLs before agent guidance:** `extensions/feishu/src/bot-sender-name.ts:52-65`
  - body: The widened permission-error path accepts any raw object with the Feishu permission code and extracts the first `https://.../app/...` URL, which is later shown in agent-visible permission guidance.
