  - confidence: 0.9

  - body: This widens permission-error parsing to raw response objects, then extracts an HTTPS `/app/` URL from remote error text for agent-visible permission guidance. Please only accept known Feishu/Lark open-platform hosts so a crafted upstream error cannot present an attacker-controlled app URL as an admin authorization link.
  - confidence: 0.86
