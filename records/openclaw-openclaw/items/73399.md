  - body: This adds a process-wide direct-member cache and widens the existing sender cache key space, but expired entries are only ignored and never removed, and neither map has a size cap. In a long-running Feishu process, distinct account/chat/sender values can grow these maps until availability suffers; keep current-main stale deletion and add a bounded eviction policy.
  - confidence: 0.88
- **[P2] Validate Feishu grant URLs before agent guidance:** `extensions/feishu/src/bot-sender-name.ts:52-65`
  - body: The parser now accepts raw API response objects and extracts the first `https://.../app/...` value from remote error text.
