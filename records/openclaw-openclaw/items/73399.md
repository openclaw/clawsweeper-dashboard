  - body: When chatMembers returns one item whose `member_id` does not match `senderId`, this fallback still uses that member's name and skips the contact fallback.   - confidence: 0.88

Restrict this to official Feishu/Lark hosts and safe formatting before surfacing it in agent-visible system text.
