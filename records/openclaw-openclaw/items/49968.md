  - body: Current main and the hook docs make model-generated session-memory slugs opt-in with `llmSlug: true`, but this change calls the model unless users set `llmSlug: false`.   - body: This replaces the current suffix allocation with a deterministic `${dateStr}-${slug}.md` path. Repeated captures in the same minute or repeated LLM slugs no longer get distinct filenames, so one session-memory artifact can overwrite or fail another instead of preserving both.
  - confidence: 0.88
- **[P2] Require config for daily freshness timezone calculations:** `src/config/sessions/reset-policy.ts:81`
  - body: Because `cfg` remains optional, any caller missed during rebase silently falls back to the host timezone; current main already has a requested-session gateway freshness path that does not pass cfg. The daily-mode API should require cfg or every caller must be updated so the same session cannot be fresh in one path and stale in another.
  - confidence: 0.84
