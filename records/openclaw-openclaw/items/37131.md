## Security Review

Status: needs_attention

Summary: The requested capability is security-sensitive because it can expose host command execution, filesystem mutation, session orchestration, cross-session messaging, and scheduled automation without a model turn.

  - confidence: 0.92

  - body: Cron schemas currently allow only systemEvent and agentTurn; adding direct toolCall or exec payloads would schedule unattended host actions and needs explicit sandbox, approval, timeout, retry, output retention, cancellation, and delivery rules.
