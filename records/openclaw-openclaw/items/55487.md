  - body: The linked crash reports show an uncoded `TypeError: Cannot read properties of null (reading 'setSession')` in `TLSSocket.setSession`, but this classifier only checks codes, selected names, message-code regexes, and generic TLS/socket snippets.   - confidence: 0.89
- **[P1] Keep EPIPE out of uncaught network suppression:** `src/infra/unhandled-rejections.ts:193-194`
  - body: `isStrictTransientNetworkError()` returns true for any code in `TRANSIENT_NETWORK_CODES`, which includes `EPIPE`. Broken pipes from stdout or downstream stream consumers are not safe gateway network transients; suppressing them can leave CLI output or gateway state invalid instead of failing cleanly.
  - confidence: 0.86
- **[P1] Preserve fatal precedence before transient suppression:** `src/cli/run-main.ts:164-169`
  - body: The new uncaughtException handler returns when the strict network classifier matches, and that classifier walks nested causes. A fatal wrapper with a transient network cause can be logged as non-fatal and keep the process running after an uncaught fatal error, so check fatal wrappers before transient suppression or make the strict classifier ignore them.
  - confidence: 0.82
