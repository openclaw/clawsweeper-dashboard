- **[P1] Preserve aliases after pruning:** `src/gateway/session-utils.ts:924-929`
  - body: For a raw external key, `target.storeKeys` contains both the canonical key and the raw requested key, and the existing `pruneLegacyStoreKeys` block deletes every candidate that is not `primaryKey`. Preserve the alias after pruning or exclude preserved aliases from the prune set, with a regression case for a structured key like `conversation:pair:user_a::user_b:space:space_123`.
