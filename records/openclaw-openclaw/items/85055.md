- **[P1] Route CLI refresh through the Gateway in bridge mode:** `extensions/memory-wiki/src/cli.ts:549-558`
  - body: The migration contract tells callers that used status as a refresh heartbeat to call `wiki refresh`, but this implementation always runs local sync/compile. That can silently remove compiled memory from prompts for existing users until refresh/compile succeeds, so the PR needs a migration/default-compat path or explicit maintainer-approved upgrade behavior with docs.
