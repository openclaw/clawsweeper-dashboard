## Summary

Keep open: current main still has the childless Codex-native task mirror gap, and this draft PR is a focused registry-side fix with adequate live proof and no blocking code or security findings.

## What This Changes

The PR adds Codex-native childless task detection, gives those registry-only mirrors a 30-minute stale reconciliation grace period, and lets cancellation mark them cancelled without subagent teardown.

## Best Possible Solution

Land a ready, maintainer-approved version that makes Codex-native registry-only mirrors recoverable and cancellable without broadening the public plugin SDK or changing normal ACP, subagent, CLI, and cron task behavior.

## Reproduction Assessment

Yes, source-reproducible without executing tests: current main treats childless non-CLI tasks as backed forever and rejects cancellation before reaching any Codex-native special case. The PR body also gives live after-fix proof for the stale-to-lost path.

## Solution Assessment

Yes. The fix is narrowly scoped to childless `codex-native` mirrors with `codex-thread:` identity, preserves ordinary task reconciliation, and makes cancellation registry-only when there is no OpenClaw child session to kill.
