## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.92

Full review comments:

- **[P2] Populate models in the cache summary path:** `src/infra/session-cost-usage.types.ts:68`
  - body: The PR makes `CostUsageSummary.models` required, but the cache-backed summary path on current main still returns no model aggregates. After rebase, cache-served usage summaries will either violate the type contract or leave `/total_usage` without the data it assumes, so add provider-aware model aggregation to the durable cache path too.
  - confidence: 0.9
- **[P2] Keep bounded usage scans efficient:** `src/infra/session-cost-usage.ts:399-402`
  - body: Removing the `mtimeMs < sinceTime` prefilter makes narrow ranges like `/total_usage 5h` enumerate and scan every transcript before discarding out-of-range entries. Route the command through the cache-backed summary path or keep a safe bounded prefilter for non-all ranges.
  - confidence: 0.88
- **[P2] Filter discovered sessions before applying the limit:** `src/infra/session-cost-usage.ts:497-502`
  - body: Dropping the `startMs` filter from `discoverAllSessions` means `sessions.usage` can sort and `slice(0, limit)` out-of-range sessions before per-session range filtering runs. In large histories, older sessions can consume the limit and hide in-range sessions from the response.
  - confidence: 0.86
- **[P2] Partition model buckets by provider:** `src/infra/session-cost-usage.ts:426-431`
  - body: The aggregate key is only `entry.model` while the bucket stores the first provider seen. Same-named models from different providers will be merged and mislabeled in the model breakdown, so key by provider and model.
  - confidence: 0.91
- **[P2] Use native Telegram callback payloads:** `src/auto-reply/reply/commands-session.ts:425-434`
  - body: The quick-select buttons send raw `/total_usage ...` callback text. Telegram native-command callbacks are expected to stay on the structured `tgcmd:` path, so these buttons can lose native routing when text commands are disabled.
  - confidence: 0.86
