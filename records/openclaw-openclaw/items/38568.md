Implement the canonical Runtime-line context usage feature for https://github.com/openclaw/openclaw/issues/38568. Add a fresh, optional context-usage value such as `context=49%` to the system prompt Runtime line for embedded runs by deriving it from reliable session token/window snapshots only.
