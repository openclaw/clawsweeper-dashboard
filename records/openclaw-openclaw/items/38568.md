## What This Changes

Add current context-window usage, such as `context=49%`, to the system prompt Runtime section so agents can see context pressure without calling `session_status`.

## Best Possible Solution

Add a typed optional context-usage snapshot to prompt assembly and render a compact Runtime `context=...` field only when the value is reliable, with focused tests, docs, and a changelog entry.

## Solution Assessment
