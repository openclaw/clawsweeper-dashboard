## Best Possible Solution

Keep this issue as the canonical tracker, refresh and review https://github.com/openclaw/openclaw/pull/73390, preserve Enter-to-send as the default unless maintainers choose otherwise, route new UI copy through Control UI i18n, and add focused coverage for plain Enter, Shift+Enter, IME composition, and Cmd/Ctrl+Enter.
