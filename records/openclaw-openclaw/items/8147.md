Request a configurable WebChat send shortcut so users can choose Cmd/Ctrl+Enter instead of plain Enter, mainly to avoid accidental sends during IME or multiline composition.

## Best Possible Solution

Keep this as the canonical tracker, then refresh and review https://github.com/openclaw/openclaw/pull/73390 with persistence scope, UI copy, IME/Shift+Enter/Cmd-Ctrl+Enter coverage, and any docs/config updates settled.
