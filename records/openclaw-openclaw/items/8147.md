Keep this issue open. Current main still sends from the Control UI/WebChat composer on plain Enter, has no persisted shortcut preference or gateway WebChat config option, and the explicitly linked implementation PR #73390 is still open and unmerged.

Request a configurable WebChat/Control UI send shortcut so users can choose Cmd/Ctrl+Enter instead of plain Enter while composing with IME or multiline text.
