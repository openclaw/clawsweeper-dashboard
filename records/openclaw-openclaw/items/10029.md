## Summary

Keep open. Current main still lacks syntax highlighting on the live WebChat/Control UI code-block path; the existing highlighter is export-only, and canonical search did not find an open implementation PR.

## What This Changes

Request syntax highlighting for fenced and indented code blocks in the WebChat/Control UI markdown renderer.

## Best Possible Solution

Add renderer-level, theme-aware highlighting in ui/src/ui/markdown.ts or a small helper, with sanitizer-safe token markup and coverage for copy buttons, language labels, JSON collapse, caching, and oversized fallback.
