## Summary

Keep open. Current main still lacks syntax highlighting on the live WebChat code-block path, the existing highlighter is limited to session-export HTML, and canonical search did not find an open implementation PR or better owner item for this exact enhancement.

## What This Changes

Request syntax highlighting for WebChat/Control UI Markdown code blocks with matching theme styles.

## Best Possible Solution

Add renderer-level highlighting in the Control UI markdown layer with theme-aware token styles and regression coverage for code-block chrome, copy text, JSON collapse, sanitization, caching, and oversized fallback.

## Solution Assessment
