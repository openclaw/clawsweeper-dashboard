Keep open. Current main still lacks live WebChat code-block syntax highlighting; code blocks are escaped plain text in the Control UI renderer, and the only syntax-highlighting path found is transcript export.

## What This Changes

The issue requests syntax highlighting for fenced and indented code blocks in the WebChat/Control UI markdown renderer.

## Best Possible Solution

Add renderer-owned, theme-aware highlighting in `ui/src/ui/markdown.ts`, with token markup kept inside sanitization and tests covering copy buttons, language labels, JSON collapse, cache behavior, and oversized fallback.

## Reproduction Assessment

## Solution Assessment
