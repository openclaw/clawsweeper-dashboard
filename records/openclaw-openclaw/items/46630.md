## Summary

Keep open. Current main has related Control UI sidebar/session shortcuts, slash-command discovery, a markdown/canvas chat side panel, and narrow plugin Control UI descriptors, but it still lacks the requested user-defined shortcut storage/config contract, keyword sidebar renderer, and click-to-insert composer behavior.

## What This Changes

Add a collapsible WebChat/Control UI sidebar listing user-defined shortcut keywords with labels, descriptions, optional grouping/icons, and click-to-insert prompt text.

## Best Possible Solution

Define a versioned shortcut discovery/storage contract first, then add a host-rendered Control UI panel that inserts prompt text into the composer without bypassing command or plugin contribution rules.
