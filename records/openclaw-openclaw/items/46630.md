## Summary

Keep open. Current main has adjacent slash-command discovery and a markdown/canvas chat side panel, but it still lacks user-defined shortcut storage, a `gateway.webchat.shortcuts` config contract, and a persistent shortcut browser that inserts arbitrary prompt snippets into the composer.

## What This Changes

Add a collapsible WebChat sidebar or drawer that lists user-defined shortcut keywords with labels, descriptions, optional icons/categories, and click-to-insert prompt text.

## Best Possible Solution

Define a versioned shortcut storage/discovery contract, then render a responsive host-owned shortcut panel or drawer that inserts text into the composer without auto-sending.
