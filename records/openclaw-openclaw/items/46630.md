Keep open. Current main and v2026.6.6 still do not provide a user-defined WebChat shortcut browser or prompt-insert sidebar; the nearest slash-command and sidebar surfaces are adjacent but do not solve arbitrary shortcut snippets, and the remaining storage/API choice needs maintainer product direction.

## Best Possible Solution

Define a maintainer-approved shortcut ownership and storage contract, then add a responsive Control UI panel or drawer that inserts prompt text without auto-sending and coexists with the broader plugin contribution-slot direction.
