## Best Possible Solution

Merge the scoped endpoint switch after required checks and maintainer acceptance of the account-host OAuth flow, while keeping post-login API resource URLs unchanged and leaving release-note generation to the release workflow.

## Solution Assessment

Yes. Calling the MiniMax account OAuth endpoints directly is narrower than enabling unsafe cross-origin redirect replay and keeps the vendor-specific fix inside the MiniMax plugin.
