## Summary

This PR is an active, narrow fix for a source-reproducible subagent completion delivery bug: current main still returns before the requester-agent `callGateway` handoff when an active requester cannot be woken, and the latest diff removes that dead-end while preserving the no-raw-send delivery contract. I found no blocking code-review finding in the two-file patch; the remaining merge work is normal CI and maintainer handling.

## What This Changes

The PR removes the active-requester early return in `sendSubagentAnnounceDirectly` and updates the focused regression expectation so delivery falls through to `callGateway` before queue fallback.
