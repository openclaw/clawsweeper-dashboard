Requests that agent-visible file path responses use full absolute paths instead of `~/` shorthand.

## Best Possible Solution

## Solution Assessment

Unclear. A blanket prompt rule or output post-processor is not clearly the best fix because current docs, tests, and skill prompt compaction intentionally use `~/`; maintainers should choose the surfaces first.
