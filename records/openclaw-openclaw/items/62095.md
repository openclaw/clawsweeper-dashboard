Keep open. ## What This Changes

The issue requests investigation and fixes for v2026.4.5 post-upgrade gateway stability regressions spanning doctor migrations, subagent announce timeouts, private WS behavior, Slack stale-socket restarts, and gateway memory growth.

## Best Possible Solution

Keep this as an umbrella until maintainers split or explicitly route the remaining symptoms: land or replace the Slack socket teardown fix, decide whether completion announce attempts should fail faster by default, and close the already-covered doctor/private-WS parts only after the remaining runtime stability paths have separate canonical tracking.

## Solution Assessment
