## Summary

## What This Changes

The PR changes memory-core embedding-cache writes from JSON text to packed Float32 BLOBs, keeps legacy JSON text reads, exports the existing vector BLOB helper, and adds cache round-trip tests.

## Best Possible Solution

Land the cache-only encoding change if maintainers accept the in-place mixed JSON/BLOB cache boundary; otherwise require an explicit BLOB column or fold this into the broader memory storage sequence.

## Reproduction Assessment

Not applicable as a bug reproduction; this is a performance/storage-format cleanup PR. Current main and v2026.6.8 source confirm the prior JSON cache behavior, and the PR proof demonstrates the after-fix mixed JSON/BLOB path.

## Solution Assessment
