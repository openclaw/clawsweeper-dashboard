## What This Changes

The PR changes memory-core to write embedding-cache rows as packed Float32 BLOBs via `vectorToBlob`, decode both BLOB and legacy JSON TEXT rows, and add cache round-trip and legacy-row tests.

## Best Possible Solution

Land the cache-only BLOB change only after maintainer acceptance of the mixed-format cache contract and exact-head CI; keep durable embedding storage, cache_size tuning, and hybrid rerank under the open umbrella issue.
