  - body: This unrelated override forces workspace `openclaw` ranges through a published package version; the lockfile then changes local workspace packages from `workspace:*` links to npm `openclaw@2026.3.11`, which changes supply-chain and local package resolution inside a TUI feature branch.

  - confidence: 0.91
- **[P2] Keep mounted theme objects live on theme switches:** `src/tui/theme/theme.ts:224-235`
  - body: `applyPalette()` replaces exported theme objects, but mounted markdown and selector components hold the object they received at construction. A runtime `/theme` switch will therefore leave existing chat, markdown, and selector UI partially on the old palette.
  - confidence: 0.88
- **[P3] Normalize and warn for env theme values:** `src/tui/theme/detect.ts:11-13`
  - body: `OPENCLAW_THEME` is advertised as the persistent theme setting, but the detector returns its raw trimmed value, so mixed-case or unknown values silently fall back through `resolveAndApply()` without the startup warning added for the CLI flag.
  - confidence: 0.78
