## Label Justifications

- `P1`: The request targets gateway-wide freezes and restart churn from one plugin, which can break real multi-agent and channel workflows.
- `impact:session-state`: The reported failure mode is plugin work holding session locks until sessions freeze or become stale.
- `impact:message-loss`: The issue and production comment describe inbound messages not being processed while sessions are locked or the gateway is degraded.
- `impact:crash-loop`: The requested breaker is meant to stop repeated plugin failures from cascading into gateway restart churn or frozen runtime availability.

## Summary

Keep open. Current main has partial safeguards, including hook timeouts, session-lock cleanup, channel/plugin-local circuit breakers, and an Active Memory-only timeout breaker, but it still does not provide the requested framework-level plugin circuit breaker with plugin/hook lock attribution, persisted breaker state, operator status/reset controls, half-open/forced-open behavior, and per-plugin degradation policy.

## What This Changes

The issue requests a framework-level plugin circuit breaker that can degrade only misbehaving plugin hooks or tools while keeping the rest of the gateway running.

## Best Possible Solution

Design this as a staged core plugin-runtime resilience feature: passive timing/error telemetry and plugin/hook lock attribution first, then persisted breaker state plus status/reset controls, then active skip/fail-request/half-open/forced-open behavior with explicit per-plugin fail modes.

## Reproduction Assessment

Not applicable as a strict bug reproduction path: this is a framework feature/design request. Source inspection gives a high-confidence current-main gap check for the missing generic breaker, attribution, persistence, and operator surfaces.

## Solution Assessment

Unclear as a final implementation. The circuit-breaker direction is plausible, but budgets, fail modes, persisted state, lock attribution, reset semantics, and safe degradation behavior need maintainer runtime/product design before code work.
