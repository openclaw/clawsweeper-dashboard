## Summary

## Best Possible Solution

Rebase a narrow compatibility shim onto current main, extract legacy `.to` values through the same string cleanup path, keep the SDK contract clear, and prove the installed-plugin send path with redacted real output.

No high-confidence live reproduction was established in this read-only review. Source inspection shows current main only accepts string normalizer output and ignores object-shaped `.to` values, but the PR does not include a real installed-plugin run.

## Solution Assessment
