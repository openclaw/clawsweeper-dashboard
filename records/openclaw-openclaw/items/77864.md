## What This Changes

The issue reports Active Memory returning `status=empty ... summaryChars=0` for every recall run across multiple models despite the feature being enabled.

Keep this issue open until https://github.com/openclaw/openclaw/pull/78083 or a replacement lands; route the broader normal no-useful-memory status cleanup through https://github.com/openclaw/openclaw/issues/79812 and one canonical PR.
