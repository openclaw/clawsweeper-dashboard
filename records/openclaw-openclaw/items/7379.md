The issue requests adding GitHub Copilot Premium/Chat provider quota percentages to the per-response `/usage` footer.

## Best Possible Solution

Add a generic active-provider usage-window append to `/usage tokens` and `/usage full` reply footers using the existing provider usage loader/formatter, while keeping Copilot-specific fetching inside the plugin and updating focused tests, docs, and changelog.

## Solution Assessment
