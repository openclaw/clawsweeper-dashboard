Keep open. ## What This Changes

Add active-provider Premium/Chat quota-window percentages to the per-response `/usage` footer when available.

## Best Possible Solution

Append a compact active-provider `formatUsageWindowSummary` result through the generic provider-usage contract when `/usage tokens` or `/usage full` is enabled, with bounded loading, focused regression coverage, and token-use docs updated.

## Solution Assessment
