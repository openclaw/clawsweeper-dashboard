Add GitHub Copilot premium/chat provider quota percentages to the per-response `/usage` footer when provider usage windows are available.

Add a generic provider-usage-window append to the existing `/usage` footer path, reusing plugin-owned `loadProviderUsageSummary` and `formatUsageWindowSummary` without adding Copilot-specific policy in core.
