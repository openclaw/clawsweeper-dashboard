- **[P3] Track only active ancestors as circular:** `src/agents/stable-stringify.ts:19`
  - body: The shared WeakSet tracks every object seen during the whole traversal. For an acyclic payload such as `const x = {a: 1}; {first: x, second: x}`, the second branch serializes as `"[Circular]"`, colliding with a literal string and changing hashes/fingerprints for non-cyclic data now that models/tool-loop callers use this helper. Track only the active recursion stack, for example by deleting on unwind, so only real cycles get the marker.
