## Summary

Keep this PR open. Current main still only falls back to the generic skipped-add warning when QMD reports a path/pattern conflict without structured list metadata, while this PR adds a focused, non-destructive manual repair hint; the remaining concrete issue is the missing changelog entry for a user-visible memory diagnostic fix.

## Solution Assessment
