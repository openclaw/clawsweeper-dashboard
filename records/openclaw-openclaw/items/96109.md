## Solution Assessment

Yes, the resolver is the best code layer for the reported empty-array regression because it preserves explicit object-format opt-out and legacy `["inlineButtons"]` behavior while restoring the documented default. Broader non-empty legacy arrays should stay out of this PR unless maintainer direction or reporter config proves they are the same bug.
