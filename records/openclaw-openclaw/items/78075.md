## What This Changes

The PR bypasses cached plugin tool descriptors when active/channel live registries already own both hooks and tools for the same plugin, and adds a regression test for shared closure state.

## Best Possible Solution

Align cached tool resolution with current live hook-owning registry sources, preserve descriptor caching for unaffected tool-only paths, and require redacted real behavior proof before merge.
