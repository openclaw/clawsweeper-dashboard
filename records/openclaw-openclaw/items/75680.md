The issue requests CJK/i18n-aware `memory-lancedb` auto-capture triggers or configurable custom trigger patterns so non-English memory-worthy phrases are captured.

## Best Possible Solution

Review and land #76121, or an equivalent narrow plugin-owned change, so CJK defaults and safe literal custom triggers are covered in source, config schema, manifest/docs, and tests.
