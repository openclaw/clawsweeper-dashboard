Full review comments:

- **[P2] Preserve discovery suppression during permission request:** `apps/ios/Sources/Gateway/GatewayConnectionController.swift:223-224`
  - body: When `localNetworkAccessRequested` is false, `startDiscovery(allowAutoConnect: false)` delegates to `requestLocalNetworkAccess`, which unconditionally calls `attemptAutoReconnectIfNeeded()`. Because `discoveryAutoConnectEnabled` is false, that path calls `maybeAutoConnectLastKnownWithoutDiscovery()`, so an onboarding nearby-discovery session can reconnect to the saved gateway before the user selects a discovered gateway. Carry the suppression through the permission request path or skip reconnect for explicit discovery sessions.
  - confidence: 0.87
