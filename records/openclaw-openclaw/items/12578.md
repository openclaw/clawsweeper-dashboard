The issue requests a persistent `agents.defaults` usage-footer config option so new sessions can inherit `/usage off|tokens|full` behavior.

## Best Possible Solution

Land, narrow, or replace one accepted implementation that adds a validated inherited usage-footer default, preserves session override semantics, updates schema/help/docs, and adds focused regression coverage.

## Solution Assessment
