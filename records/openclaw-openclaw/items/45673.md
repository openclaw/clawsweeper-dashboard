## Best Possible Solution

Land a rebased Feishu-plugin fix that preserves current runtime account resolution, gates inherited account precedence to Feishu-origin tool contexts, falls back to configured `defaultAccount`, and records the user-facing fix in the changelog.

## Solution Assessment
