## Best Possible Solution

Align the media skip with the text ingest precedence, cover wildcard-default ingest with a focused media regression test, and attach redacted live Telegram proof for the media-to-hook path before merge.

No. The PR is a plausible narrow fix, but the best fix needs the same ingest precedence as the text path, including wildcard/default group config, plus live Telegram proof.
