## Best Possible Solution

Land this PR or an equivalent focused patch only after rebasing, preserving explicit binding and topic override precedence, regenerating config artifacts, updating Telegram docs, adding release-owned changelog coverage, and attaching real behavior proof; keep broader Telegram bot-to-bot work on the canonical issue.

## Solution Assessment
