## Summary

## What This Changes

The PR changes the skills watcher, workspace skill loader, and loader/watcher tests to discover repo-style nested skill bundles under configured skill roots.

## Best Possible Solution

Land a rebased loader-focused fix that discovers repo-style nested bundles under workspace, project-agent, personal-agent, and managed skill roots before direct-child fallback, while preserving bounded discovery and containment checks.
