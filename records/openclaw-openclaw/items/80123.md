The PR adds registry-backed known-plugin detection to CLI missing-plugin diagnostics, wires it into `runCli`, updates focused tests, and adds a changelog entry.

Land a narrow CLI classification fix that shows `plugins.allow` remediation only for known plugin IDs or plugin-owned aliases and lets arbitrary unknown roots use unknown-command handling with regression coverage.
