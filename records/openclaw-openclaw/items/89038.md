## Summary

Keep open for maintainer review: the patch appears to address the linked QQBot reconnect outbound-loss regression and has sufficient redacted live proof plus focused regression tests, but it intentionally changes plugin-resolution ordering and QQBot reconnect replay timing.

## Best Possible Solution
