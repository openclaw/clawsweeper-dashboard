## Summary

## What This Changes

The PR adds operation-aware outbound delivery plugin resolution, routes message/direct delivery through it, and drains matching QQBot pending deliveries on gateway ready/resumed.

## Best Possible Solution

The remaining question is the compatibility tradeoff, not a concrete patch defect.
