## Summary

Keep open: current main still leaves GitHub Copilot `openai-responses` embedded runs on the `session-custom` route when the cached stream is the Pi default wrapper, so the #71120 guarded fetch can still be bypassed.

## What This Changes

Ensure GitHub Copilot `openai-responses` embedded runs apply guarded HTTP 429 fail-fast handling even when the session uses a custom/default stream closure.

## Best Possible Solution

Route embedded `openai-responses` Copilot turns through OpenClaw's guarded Responses transport, or an equivalent provider-owned guarded path, while keeping Copilot's header and payload wrapper layered on top.

## Reproduction Assessment

Yes. The report provides live `v2026.4.29` telemetry plus direct upstream 429 evidence, and current main plus the locked Pi SDK reproduce the routing precondition that keeps the run on `session-custom`.

## Solution Assessment

Yes. The requested invariant is the right target; the narrow maintainable fix is in embedded/provider stream resolution or guarded transport selection, not a broad retry-policy change.

## Repair Work Prompt

Fix the Copilot `openai-responses` 429 fail-fast bypass reported in this issue. Current flow: `attempt.ts` caches the Pi session default stream, `registerProviderStreamForModel` only sees provider `createStreamFn` or transport overrides, `resolveEmbeddedAgentStreamFn` only selects the boundary-aware OpenClaw transport for undefined or exact `streamSimple`, and Copilot's `wrapStreamFn` later wraps whatever base stream was selected. Make embedded GitHub Copilot `openai-responses` calls use `buildGuardedModelFetch` / `createOpenAIResponsesTransportStreamFn` or an equivalent guarded provider-owned path even when the cached base stream is Pi's default async wrapper. Preserve Copilot dynamic headers and Responses payload ID rewriting, avoid replacing arbitrary genuine custom session streams, do not change Pi dependency versions or broad SDK retry policy, and add a user-facing `CHANGELOG.md` fix entry crediting the reporter if appropriate. Add regression coverage for a non-referential Pi default stream wrapper, the reported `session-custom` classification gap, and Copilot wrapping around the guarded base transport.
