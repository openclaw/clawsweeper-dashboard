## What This Changes

This PR changes `llm_input` and `llm_output` plugin hooks from observe-only hooks into sequential modifying hooks that can block, rewrite model input, and replace assistant output.

## Best Possible Solution

Land a rebased, maintainer-approved writable LLM hook contract that preserves current hook timeouts, policy gates, payload isolation, `tools` context, Google cache ordering, docs, and focused tests.
