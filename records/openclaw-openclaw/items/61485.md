## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.87

Full review comments:

- **[P1] Add timeout budgets for awaited LLM hooks:** `src/plugins/hooks.ts:967`
  - body: `llm_input` and `llm_output` move onto `runModifyingHook`, but the default modifying-hook timeout table does not include either hook. A plugin handler that never resolves can now hang the model-call path indefinitely, while the current fire-and-forget hooks cannot block the run this way.
  - confidence: 0.89
- **[P1] Apply system prompt overrides to the model call:** `src/agents/embedded-agent-runner/run/attempt.ts:4148`
  - body: Assigning `systemPromptForHook` only updates the local precheck/diagnostic value; the active session system prompt and Google prompt-cache stream wrapper were already built from `systemPromptText`. A plugin returning `systemPrompt` will therefore not reliably change the request sent to the model.
  - confidence: 0.9
- **[P1] Re-detect images after prompt overrides:** `src/agents/embedded-agent-runner/run/attempt.ts:4145`
  - body: `imageResult` is computed before the new `llm_input` result is applied, then reused for `promptActiveSession`. If a plugin removes image references, the old images can still be sent; if it adds image references, they are omitted, which breaks the prompt rewrite contract and can leak redacted content.
  - confidence: 0.88
- **[P1] Preserve deliberate empty output overrides:** `src/agents/embedded-agent-runner/run/attempt.ts:5141`
  - body: An `llm_output` plugin can intentionally return `assistantTexts: []`, but the returned attempt then looks like a model produced no output. `run.ts` uses that empty array to permit live-model-switch restarts, so a deliberate suppression/redaction can be discarded by a retry path.
  - confidence: 0.86

## Security Review

Status: needs_attention

Summary: The diff introduces security-sensitive plugin mutation of model traffic, and current defects can bypass intended prompt/response controls.

Concerns:

- **[high] Stale image payloads after prompt rewrite:** `src/agents/embedded-agent-runner/run/attempt.ts:4145`
  - body: Prompt overrides are applied after image detection, so redacting image references from the prompt can still leave the original image payload attached to the model call.
  - confidence: 0.88
- **[medium] System prompt rewrite does not reach the model path:** `src/agents/embedded-agent-runner/run/attempt.ts:4148`
  - body: The system-prompt result updates only a local hook/precheck variable, so a compliance plugin may believe it rewrote policy instructions while the old system prompt is still used.
  - confidence: 0.9
- **[medium] Empty response suppression can be retried away:** `src/agents/embedded-agent-runner/run/attempt.ts:5141`
  - body: Returning an empty assistant-text override can be treated as no model output by the live-switch restart guard, defeating a plugin's deliberate response suppression.
  - confidence: 0.86
