## Summary

Keep this PR open: the feature idea is useful, but the current branch is not merge-ready because it changes an external plugin hook contract and has blocking availability/security correctness issues in the awaited hook, system-prompt override, and output-redaction paths.

## What This Changes

The PR changes `llm_input` and `llm_output` plugin hooks from fire-and-forget observation hooks into awaited modifying hooks that can block calls, rewrite prompt/system prompt text, and replace assistant response text.

## Best Possible Solution
