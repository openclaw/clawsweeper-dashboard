Adds a root `patchedDependencies` entry and a compiled patch for `@mariozechner/pi-agent-core@0.55.3` that strips a leading `mcp_` before dependency-level tool lookup.

## Best Possible Solution

Add a guarded first-party normalizer in `src/agents/pi-embedded-runner/run/attempt.tool-call-normalization.ts` that preserves exact registered names first, maps only non-empty single-underscore `mcp_` aliases when they uniquely match allowed canonical tools, and adds focused regression coverage.
