## Summary

## What This Changes

The PR changes the iMessage self-chat cache to match reflected self-chat rows with same normalized text and scope when `created_at` differs by less than one second, with focused cache and inbound-decision regression tests.

## Best Possible Solution

Land the bounded cache-level tolerance after required checks if maintainers accept the sub-second same-text suppression tradeoff; request live macOS watcher proof only if transport-level confidence is needed.
