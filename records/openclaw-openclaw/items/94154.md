## What This Changes

This PR changes `handleGatewayRequest` to use the caller-attached gateway method registry only when it owns the requested method, otherwise rebuilding from the live plugin registry, and adds gateway dispatch regression tests.

## Best Possible Solution

Land this branch or an equivalent attached-registry-first live-fallback fix after maintainers accept the chokepoint proof and required checks finish, then let https://github.com/openclaw/openclaw/issues/94127 close through the linked PR.
