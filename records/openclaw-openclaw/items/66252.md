## What This Changes

The issue requests per-agent TTS and STT overrides so different agents can use different speech providers, voices, and transcription languages in one OpenClaw instance.

## Best Possible Solution

Land a maintainer-approved per-agent STT config contract that merges into agent-aware audio transcription entry points, with schema/types/docs/tests and unchanged global defaults unless explicitly configured.
