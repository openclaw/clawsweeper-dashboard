## Summary

Keep open. Current main and the latest release cover the per-agent TTS half, but the requested per-agent STT language/provider override is still missing from the agent schema and transcription runtime path.

## What This Changes

Request per-agent TTS and STT configuration overrides so different configured agents can use different speech providers, voices, and transcription languages.

## Best Possible Solution

Land a maintainer-approved per-agent STT contract that merges into audio transcription at agent-aware media-understanding entry points, with schema/help/docs/tests and unchanged global defaults unless explicitly configured.
