## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Yes for the reported `turn.run` break: an object-identical alias avoids a second implementation path and fixes old compiled plugins without requiring every plugin to ship immediately. The open maintainer question is whether the compatibility surface should also restore the full older `turn` helper method set.
