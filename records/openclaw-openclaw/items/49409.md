Full review comments:

- **[P2] Make the first daily header append atomic:** `src/hooks/bundled/session-memory/handler.ts:382-393`
  - body: The branch probes the daily file with an empty append, closes it, then appends the real content based on that stale result. Two concurrent reset hooks can both decide the file needs `# YYYY-MM-DD`, so the canonical daily file can get duplicate top-level headers.
  - confidence: 0.94
- **[P2] Preserve local-time daily filenames:** `src/hooks/bundled/session-memory/handler.ts:248-250`
  - body: The branch derives the canonical daily filename from `toISOString()`, while current behavior and tests use the user's local date. Users west of UTC can write tomorrow's memory file during local evening resets.
  - confidence: 0.9
