## What This Changes

The issue requests replacing the Agents page dropdown-first navigation with a visible agent list and making the Default action persist reliably.

Restore visible agent navigation on normal-width Agents pages, keep a compact mobile fallback, and make Set Default persist or communicate the pending-save state unmistakably through the existing `agents.list[].default` contract.
