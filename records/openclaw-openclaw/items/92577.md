This PR adds a session-memory transcript guard that skips adjacent assistant messages with identical extracted text and adds focused transcript unit tests.

## Best Possible Solution

Constrain the hook-layer mitigation to the raw thinking assistant entry and its cleaned child using transcript lineage, then add a regression for unrelated repeated assistant text and real memory-file proof.

## Solution Assessment

No. Hook-layer dedupe can be an acceptable mitigation, but this implementation should identify the raw/cleaned parent-child transcript pair instead of collapsing any adjacent identical assistant text.
