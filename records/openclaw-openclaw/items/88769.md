## What This Changes

## Best Possible Solution

Keep the parser-boundary filter approach, but make the streaming filter match the existing delivery sanitizer contract for malformed boundaries and cover those cases in both filter and OpenAI-compatible parser tests before merge.

  - body: `flush()` now emits `droppedThinking` for any unclosed reasoning span.   - confidence: 0.93
