## What This Changes

This PR forwards selected OpenClaw OAuth profiles into `google-gemini-cli` runs by refreshing credentials, staging a temporary `GEMINI_CLI_HOME/.gemini/oauth_creds.json`, adding CLI backend SDK flags/types, and covering the routing and bridge paths with tests.

## Best Possible Solution

Land a narrow Gemini-owned bridge only after the credential-forwarding contract is explicit and approved, and after redacted real Gemini CLI proof shows the selected OpenClaw profile is consumed from the staged `GEMINI_CLI_HOME`.
