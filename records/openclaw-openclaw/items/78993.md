## What This Changes

The PR adds `defaultMarkdownTableMode: "bullets"` to the Matrix channel plugin, adds a Matrix adapter regression assertion, and adds an Unreleased changelog entry.

## Best Possible Solution

Land the Matrix-owned `bullets` default through the existing channel messaging contract, with the markdown formatting docs updated to list Matrix among bullet-default channels.

## Reproduction Assessment

Yes, with one docs gap: adding the Matrix default in plugin metadata is the narrowest maintainable fix because the shared resolver already consumes that field. The safer completion is to update `docs/concepts/markdown-formatting.md` alongside the behavior change.
