## Summary

PR #78441 is still necessary because current main has the embedded/runtime tool allowlist machinery but lacks the public `sessions_spawn` and Gateway plumbing, durable child-session state, and explicit unsupported-runtime handling this branch adds. ## What This Changes

The PR adds `toolsAllow` to native subagent spawning through `sessions_spawn` and Gateway `agent` requests, persists it on child sessions as `runtimeToolsAllow`, forwards it into embedded attempts, rejects unsupported ACP/CLI-backed paths, and updates docs, generated protocol bindings, prompt snapshots, and tests.
