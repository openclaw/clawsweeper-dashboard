Keep open for maintainer review: current main still lacks a public `sessions_spawn.toolsAllow` path, and this PR is a plausible scoped implementation with no blocking diff defect found. The PR adds a `toolsAllow` array to one-shot native `sessions_spawn` calls, forwards it through the Gateway `agent` path into embedded-runner tool filtering, rejects unsupported runtimes, and updates docs, tests, protocol models, changelog, and prompt snapshots.

## Best Possible Solution
