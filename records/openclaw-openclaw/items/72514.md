Keep open. Current main has useful session/subagent primitives, but the requested cross-channel and parent-child continuity model is only partially covered and remains privacy-sensitive architecture work.

## What This Changes

## Best Possible Solution

Define an owner-approved continuity model that preserves default isolation while specifying explicit identity/session linking, parent-child observation/control, failed child retry/resume behavior, and WebChat/QClaw plus WeChat-plugin UX.
