## Summary

## What This Changes

The PR adds Qwen3, Nomic, and Mixedbread query-instruction prefixing for bundled OpenAI and generic OpenAI-compatible memory query embeddings, leaves document embeddings raw, and updates tests and OpenAI provider docs.

## Best Possible Solution

Merge after maintainers explicitly accept scoped automatic query-prefixing for known instruction-aware OpenAI-compatible embedding families, or ask for a shared/opt-in prefix policy if silent matching is not the desired upgrade contract.

## Reproduction Assessment

## Solution Assessment

Mostly yes. Query-only prefixing matches upstream model guidance and existing Ollama behavior while leaving documents raw, but the silent upgrade behavior needs maintainer acceptance before merge.
