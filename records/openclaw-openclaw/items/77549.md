  - body: As submitted, this Compose env block changes HOME but does not preserve current main's OPENCLAW_STATE_DIR, OPENCLAW_CONFIG_PATH, OPENCLAW_CONFIG_DIR, and OPENCLAW_WORKSPACE_DIR overrides on either service.   - confidence: 0.9

  - body: The image sets OPENCLAW_LENIENT_CHANNEL_CONFIG=1 by default, and validation then downgrades invalid channel schemas to warnings and { enabled: false }.   - confidence: 0.88
- **[P2] Keep outbound loading on the bootstrap registry path:** `src/channels/plugins/outbound/load.ts:19`
  - body: This loader is documented as the cheap outbound send-primitive loader, and deliver.ts already bootstraps runtime plugins after a registry miss. Adding getBundledChannelPlugin(id)?.outbound here pulls full bundled channel runtime into the lightweight path and bypasses bootstrap ownership; repair registry/bootstrap seeding or add an outbound-only facade instead.
  - confidence: 0.86
