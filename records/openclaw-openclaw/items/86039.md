## What This Changes

Disabled bundled channel/plugin config should suppress bundled channel setup fallback loading so missing generated setup modules do not emit doctor warnings.

## Best Possible Solution

Review and land https://github.com/openclaw/openclaw/pull/86166, or an equivalent repair that passes config into bundled setup fallback discovery before generated setup modules are loaded.

## Reproduction Assessment

## Solution Assessment
