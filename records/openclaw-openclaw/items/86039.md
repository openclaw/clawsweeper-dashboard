## Summary

## What This Changes

The issue requests that bundled channel setup discovery skip explicitly disabled bundled channels before attempting to load missing generated setup modules.

## Best Possible Solution

Create a process-stable setup discovery snapshot that separates loaded, installable, bundled-display, and status ownership, then make doctor/status honor explicit disabled config before generated setup module loading.
