Request a per-system-event audience classification so internal runtime-context events can be kept out of user-visible consumer paths.

## Best Possible Solution

## Solution Assessment

Unclear pending maintainer design. The open PR is a plausible narrow wrap-on-drain implementation for hidden runtime context, but it explicitly does not settle the wider delivery-routing question raised by this tracker.
