## Summary

Keep open for maintainer architecture, product, and security review: current main and the latest release still use HTTP/WebSocket Gateway transport and isolated Gateway trust domains, while this request proposes a new Iroh P2P transport plus cross-device CRDT state sync.

## What This Changes

The issue requests embedding Iroh into OpenClaw Gateway and companion nodes for zero-config P2P connectivity, stable device identity, and cross-device workspace/session/memory sync.

## Best Possible Solution
