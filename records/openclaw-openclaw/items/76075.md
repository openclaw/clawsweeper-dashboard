## Summary

This PR targets a real claude-cli streaming gap, but it should not merge as-is because the proposed event payload bypasses the existing tool-event redaction path and uses phases that current gateway/UI consumers do not understand.

## What This Changes

The PR adds a Claude CLI JSONL tool-use callback, threads it through one-shot and live CLI runners, emits tool events, and adds parser tests plus a changelog entry.

## Best Possible Solution

Keep the parser callback seam, but emit the existing sanitized `stream: "tool"` contract and cover the parser-to-runner-to-gateway/UI path with focused regression tests.

## Reproduction Assessment

Yes. A Claude stream-json fixture with `content_block_start`, `input_json_delta`, and `content_block_stop` proves current main drops tool activity, and source inspection shows the proposed payload would not match downstream consumers.

## Solution Assessment

No. The callback wiring is the right boundary, but the event data should use the established sanitized `start`/`update`/`result` shape rather than introducing raw `delta`/`end` payloads.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Sanitize tool arguments before emitting them:** `src/agents/cli-output.ts:468`
  - body: This emits the raw parsed Claude tool input as `args`. The existing pi path redacts tool arguments before `emitAgentEvent`, while the gateway only strips `result`/`partialResult`, so sensitive argument fields can be broadcast to WS/session subscribers. Redact or omit args before forwarding.
  - confidence: 0.86
- **[P2] Emit the existing tool stream phases:** `src/agents/cli-output.ts:447-468`
  - body: The new parser emits `delta` and `end`, but current consumers handle `start`, `update`, and `result`; the Control UI reads args only from `start` and output only from `update`/`result`. With this shape, downstream tool cards and verbose channel events will not match the pi-embedded contract this is trying to mirror.
  - confidence: 0.83
