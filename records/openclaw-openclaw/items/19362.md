## Summary

Keep this PR open: the underlying Discord deleted-channel session cleanup is still useful and not implemented on current main, but the branch cannot merge as-is because it targets the removed `src/discord` layout and misses current transcript archive confinement semantics.

## What This Changes

The PR adds a Discord channel-delete listener, listener registration/export changes, tests, and a changelog entry to remove and archive channel-keyed Discord sessions after deleted-channel events.

## Best Possible Solution

## Solution Assessment
