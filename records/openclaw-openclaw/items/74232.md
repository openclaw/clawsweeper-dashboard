The PR adds warning logs, structured drop reports, and synthetic error tool results when the session transcript guard removes unknown or malformed assistant tool calls.

## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

No. Persisting feedback for a removed assistant turn is not the safest fix; diagnostics-only logging or a paired feedback design that respects `allowSyntheticToolResults` would be narrower and safer.
