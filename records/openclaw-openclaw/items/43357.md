## What This Changes

The branch updates Discord mention parsing/tests for Unicode and CJK handles and also changes official Yuanbao external catalog metadata and matching tests.

## Best Possible Solution

Consolidate a Discord-only Unicode/CJK mention rewrite that preserves alias, email, URL, code-span, reserved-mention, and account-scope safeguards while leaving official external catalog pins untouched.

## Reproduction Assessment

Yes, at source level: current main's Discord mention regex only matches ASCII handles, so a cached Unicode handle such as `@张三` will not be rewritten. I did not run a live Discord gateway proof in this read-only review.

## Solution Assessment

No. Unicode-aware parsing is the right direction, but this branch must first remove unrelated catalog changes, preserve current alias behavior during rebase, and provide real Discord proof.
