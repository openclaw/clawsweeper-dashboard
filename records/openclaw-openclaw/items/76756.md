## Summary

Keep open. Current main still lacks a plugin runtime gateway accessor while the host RPCs exist; adding one is a new public SDK seam with redaction, replay, and API-shape choices that need maintainer review.

## What This Changes

Request to add a plugin-sdk runtime gateway surface that lets plugins read selected host Gateway RPCs such as `status` and `models.list`.

## Best Possible Solution

Define a narrow documented read-only `api.runtime.gateway` facade with explicit method allowlisting, redacted status semantics, public response types, and replay snapshot behavior aligned with #76758 and #76759.

## Reproduction Assessment

Not applicable; this is a feature request rather than a broken existing contract. Source inspection confirms `PluginRuntime` and `createPluginRuntime` still expose subagent/nodes/channel but no gateway namespace while host `status` and `models.list` handlers exist.

## Solution Assessment

Unclear until maintainer API review. A curated facade is safer than exposing `GatewayClient`, but maintainers need to settle method allowlist, redaction, exported types, and replay semantics first.

## Security Review

Status: needs_attention

Summary: The requested API would expose host runtime state to plugin code, so implementation needs an explicit read-only allowlist plus redaction and replay policy.

Concerns:

- **[medium] Preserve Gateway status redaction for plugin access:** `src/gateway/server-methods/health.ts:129`
  - body: The `status` handler currently enables sensitive fields only when the Gateway client has `operator.admin`; an in-process plugin runtime accessor would need an equivalent policy or a deliberately redacted response so plugin code cannot bypass that scope check.
  - confidence: 0.86
- **[medium] Keep runtime Gateway calls read-only and allowlisted:** `src/gateway/server-methods-list.ts:31`
  - body: A generic plugin `request()` surface could reach mutating Gateway methods unless it is constrained to approved read-only methods such as `status` and `models.list`.
  - confidence: 0.78
