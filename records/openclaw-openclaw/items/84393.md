Keep open: current main and the latest release still route eligible OpenAI auto-mode agent turns through the Codex app-server by default, and OpenClaw still starts Codex threads with developer instructions rather than an OpenClaw-owned base-instructions override. ## What This Changes

The issue asks OpenClaw to prevent operational agents from silently receiving Codex coding-agent base instructions through Codex runtime routing, and to add diagnostics, warnings, quarantine tooling, and regression coverage.
