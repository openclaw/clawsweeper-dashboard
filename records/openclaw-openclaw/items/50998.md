## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Make Contacts permission waits cancellation-aware:** `apps/ios/Sources/Contacts/ContactsService.swift:105-109`
  - body: The new `.notDetermined` branch awaits `CNContactStore.requestAccess` through a bare continuation. When an invoke timeout cancels the task while the system prompt is pending, this async call does not resume until the OS callback arrives, so unattended first-use contacts invokes can strand work; use a cancellation-aware permission gate and skip starting the prompt once cancellation wins.
  - confidence: 0.9
- **[P2] Start Calendar prompts atomically with cancellation:** `apps/ios/Sources/Calendar/CalendarService.swift:183-184`
  - body: `awaitPermissionRequest` installs the continuation, checks `box.isResolved`, and then calls `start` as separate operations. If cancellation fires after the guard but before `start`, the caller already returns `false` while the EventKit prompt can still launch after timeout; make install-and-start a single locked or tokened operation.
  - confidence: 0.86
