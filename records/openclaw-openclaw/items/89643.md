- **[P2] Merge plugin namespaces when preserving extension state:** `src/config/sessions/store.ts:587-590`
  - body: `pluginExtensions` is shaped as `pluginExtensions[pluginId][namespace]`, and existing contract tests cover one plugin owning multiple namespaces. Please merge per namespace for both `pluginExtensions` and `pluginExtensionSlotKeys` in the preservation helpers.
