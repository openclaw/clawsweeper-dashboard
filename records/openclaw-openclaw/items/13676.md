## Best Possible Solution

Design a staged core capability: first add canonical Project identity, SQLite persistence, additive Gateway methods, and Control UI session/workspace navigation; then reuse or explicitly extend existing agent, tool, and sandbox policy owners for approved enforcement rather than creating parallel Project policy.
