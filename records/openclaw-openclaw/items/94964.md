- **[P1] Signal cancellation before the old lifecycle shuts down:** `src/infra/restart.ts:113-115`
  - body: `abortPendingChannelReloads()` is called from the next in-process restart iteration reset.   - confidence: 0.9

  - body: The new abort check only runs inside the polling loop. If active work drains, times out, or a restart appears after the loop returns, `restartChannel` still proceeds to stop/start channels; carry a cancelled result or recheck before starting each channel.
  - confidence: 0.88
