Keep open. Current main has fixed adjacent prefill/cooldown fallout, but the active-stream default still steers TUI follow-ups into the current run, GitHub Copilot Claude still has a narrow replay-policy gap, and the recovery-loop part remains covered by unresolved recovery work.
