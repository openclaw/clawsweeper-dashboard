- **[P1] Guard dashboard scheduling before requestContext exists:** `src/gateway/server.impl.ts:421`
  - body: The cron and agent-event callbacks are registered before requestContext is assigned. If cron startup or an agent event schedules a dashboard delta during that window, scheduleDashboardDelta receives undefined and reaches the WeakMap lookup, throwing instead of delivering the event.
  - confidence: 0.91

  - body: Tool result events can mark failure with data.isError while result is a plain string or object. Finalizing status from data.result alone shows those failed tool calls as completed, so Mission Control underreports failedRecent and recent tool statuses.
  - confidence: 0.9
