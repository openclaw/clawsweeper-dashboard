- **[P1] Preserve hidden-run redaction before recording dashboard tools:** `src/gateway/server.impl.ts:516`
  - body: The registry records every global agent tool event before any dashboard visibility/redaction boundary, then dashboard.summary exposes tool names plus args/output previews. Hidden channel-routed runs can leak private tool details; filter or redact before storing them.
  - confidence: 0.9

  - confidence: 0.9

  - body: Tool result events can signal failure through data.isError while result is plain text or another object without ok:false/error. Finalizing from data.result alone marks those failed calls completed and underreports failedRecent.
  - confidence: 0.88

  - body: SOUL.md, TOOLS.md, and HEARTBEAT.md are local workspace bootstrap notes, not Mission Control product files. Adding them at repo root changes local agent behavior and adds unrelated setup content to this feature branch.
