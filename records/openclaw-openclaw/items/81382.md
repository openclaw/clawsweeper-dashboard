Full review comments:

- **[P2] Cache empty heartbeat drains across retries:** `src/agents/pi-embedded-runner/run/attempt.prompt-helpers.ts:115-118`
  - body: The cache only stores `heartbeatPendingQuestionContext` when it is truthy, then uses `??` to recompute when the cached value is absent.
