## What This Changes

## Best Possible Solution

Land this narrow classifier/test change, then keep any longer-term runner-side `model_call_started` telemetry improvements separate from this unblocker.

## Reproduction Assessment

## Solution Assessment

Yes. For the reported regression, adding the existing `attempt_dispatch` phase to the private execution-start classifier is the narrowest maintainable fix; richer runner-side model-start telemetry can be pursued separately.
