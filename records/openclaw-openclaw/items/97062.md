## Best Possible Solution

Review and land one narrow shared media-loader fix, preferably the validated-text approach in https://github.com/openclaw/openclaw/pull/97143, instead of adding a broad configurable MIME allowlist.

## Solution Assessment

Yes for a narrow default allowlist addition with full-buffer text validation; a new operator-configurable MIME extension point would be broader than this user problem requires and would add config surface.
