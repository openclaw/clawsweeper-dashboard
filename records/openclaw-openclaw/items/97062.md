## Best Possible Solution

If maintainers accept vCard as a safe default, add `.vcf`/`text/vcard` through the existing validated-text host-read path with focused MIME, binary-disguise, docs, and delivery regression coverage; keep a broad MIME override as separate security/product design.

## Solution Assessment

Unclear until maintainers decide the security boundary. A narrow default vCard addition is likely the cleanest technical path if approved, while a configurable MIME override is broader than this single workflow needs.
