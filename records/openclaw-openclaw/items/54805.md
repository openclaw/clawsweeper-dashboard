## Summary

## What This Changes

Adds a typed `plugin_updated` lifecycle hook for `openclaw plugins update`, including updater callback wiring, CLI hook dispatch, hook runner methods, tests, docs, and a formatter buffer tweak.

## Best Possible Solution

Rebase the lifecycle hook onto `hook-types.ts` and the managed install-record update flow, dispatch only scoped updated-plugin runtimes, cover npm/marketplace/ClawHub/git update events, and merge only after API/security review plus real behavior proof.

## Reproduction Assessment

Not applicable as a bug report, but the review findings are source-reproducible: the PR diff broad-loads `scope: "all"`, edits the old hook contract file, and omits git from the event source while current main supports git plugin updates.

## Solution Assessment

No; the feature direction is reasonable, but this patch is not the best current solution until it is rebased onto `hook-types.ts`, scoped to updated plugin runtimes, and aligned with all current update sources.
