Keep open. The issue requests a scoped agent/default option to omit the human-readable Tooling list from the system prompt while preserving callable tool schemas.

## Best Possible Solution

Keep this as the narrow tracker until maintainers choose a scoped prompt/config contract that suppresses only the human-readable Tooling list, preserves callable schemas, and updates prompt construction, config schema/help, docs, context reporting, and focused tests together.
