## What This Changes

Requests an agent-level option to omit the human-readable tool list from the system prompt while retaining callable tool schemas.

## Best Possible Solution

Define a supported normal-session tool-guidance mode that can omit redundant bullets while preserving callable schemas, accurate context reporting, provider-safe defaults, and focused model/runtime validation.

## Reproduction Assessment
