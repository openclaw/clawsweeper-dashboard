## What This Changes

Request an opt-in config option to omit the human-readable Tooling list from the system prompt while preserving structured callable tool schemas.

## Best Possible Solution

Add an opt-in scoped agent/default setting that suppresses only prompt-side Tooling list text while preserving callable schemas, with aligned config schema, docs, context reporting, and focused prompt/runtime tests.

## Reproduction Assessment

## Solution Assessment

Yes in direction, but it needs maintainer product review before implementation. A scoped opt-in option is narrower than systemPromptOverride and distinct from broader Tool Search or schema-reduction work.
