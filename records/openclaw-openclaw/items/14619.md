## Summary

Keep open. Current main still emits summarized tool bullets in the normal system prompt while separately passing callable tool definitions, and no scoped user-facing option exists to suppress only that prompt text.

## What This Changes

The issue requests a scoped option to omit the human-readable tool list from the system prompt while keeping callable tool schemas available.

## Best Possible Solution

Define a supported prompt/tool-guidance mode that can omit redundant tool bullets while preserving callable schemas, accurate context reporting, provider-safe defaults, and focused validation.

## Reproduction Assessment

Yes. Current source shows normal sessions rendering summarized tool bullets in the system prompt while separately supplying full callable tool definitions.

## Solution Assessment

Unclear. Suppressing the text is plausible, but the permanent config shape and provider/tool-guidance policy need maintainer decision before implementation.
