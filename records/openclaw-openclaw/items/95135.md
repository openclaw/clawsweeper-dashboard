## What This Changes

The issue asks OpenClaw to allow Nix-mode auth-profile and session-state writes that do not mutate `openclaw.json`, especially `doctor --fix` repairs and model auth credential persistence.

## Best Possible Solution

Use this issue as the canonical tracker until maintainers land or replace the linked fix with a scoped Nix boundary: actual `openclaw.json` writes stay blocked, while approved non-config session/auth repairs are allowed and covered by docs, tests, and real CLI proof.

## Solution Assessment
