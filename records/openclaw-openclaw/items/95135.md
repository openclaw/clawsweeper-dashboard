## What This Changes

Nix mode currently blocks doctor repair and model auth flows even when the intended writes target auth-profile or session-state stores instead of openclaw.json.

## Best Possible Solution

Review and land or replace the linked PR with a scoped policy-confirmed fix that permits non-config auth/session-state writes while preserving strict openclaw.json immutability.
