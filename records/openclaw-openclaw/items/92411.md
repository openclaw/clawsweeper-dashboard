- **[P1] Validate group arrays before accepting availability metadata:** `src/tools/availability.ts:80`
  - body: This guard returns true for any object with an `allOf` or `anyOf` key, so `capturePluginToolDescriptor` can copy arbitrary plugin metadata into descriptors. Please require arrays here, or make the evaluator return `unsupported-signal` for non-array groups before capture.
