## Summary

Keep open: the PR addresses a real `/v1/chat/completions` tool-protocol gap and now includes usable real gateway proof, but it still advertises forced `tool_choice` support using only a system-prompt hint, so clients can still receive a normal assistant response instead of the required tool call.

## What This Changes

The PR updates the OpenAI-compatible Chat Completions gateway code, docs, tests, and changelog to parse client function tools/tool_choice, return tool_calls in streaming and non-streaming responses, and render tool-role follow-up messages.

## Best Possible Solution

Keep the adapter changes for client tools and tool_calls, but make the documented Chat Completions contract honest by either enforcing forced tool_choice through the runtime path or rejecting forced variants until that seam exists.

## Reproduction Assessment

Yes. Current main is source-reproducible because the Chat Completions adapter ignores tools/tool_choice and always maps non-stream results to finish_reason "stop"; I did not run tests because this review is read-only.

## Solution Assessment

No, not yet. The response mapping and follow-up loop are the right area, but forced tool_choice modes should be hard-enforced or explicitly rejected rather than represented as supported via an extra prompt.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Reject forced tool_choice until it is enforced:** `src/gateway/openai-http.ts:206-213`
  - body: This accepts `tool_choice: "required"` as supported but only adds an `extraSystemPrompt`, so a model that ignores the hint can still return a normal assistant message with `finish_reason: "stop"` and no `tool_calls`. OpenAI's Chat contract defines `required` as forcing one or more tool calls, so either wire a real forced-tool constraint through the run or fail fast for forced modes. ([platform.openai.com](https://platform.openai.com/docs/api-reference/chat/create-chat-completion?utm_source=openai))
  - confidence: 0.86
