## Best Possible Solution

Land a narrow version that derives sessionId and lifecycle sessionKey from the same resolved session, covers both ordinary interactive and native command-target dispatches, and includes redacted real interactive dispatch proof.

## Solution Assessment

No. Passing sessionId from the target-aware lookup fixes the ordinary path but is not the best merge shape until the lifecycle sessionKey is aligned with the same resolved session and covered for native targets.
