  - body: `knownAgentIds` is optional, but this condition throws for every explicit header/model selector when callers omit it. Current main routes `openclaw` and `openclaw/default` through the configured default agent, so configs whose default agent is not literal `main` would reject or route incorrectly.
  - confidence: 0.86
