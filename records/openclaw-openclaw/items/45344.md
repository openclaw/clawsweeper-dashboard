Full review comments:

- **[P2] Skip validation when the allowlist is absent:** `src/gateway/http-utils.ts:84-85`
  - body: `knownAgentIds` is optional, but this condition throws for every explicit header/model selector when callers omit it. Current compat callers such as embeddings still use `resolveAgentIdForRequest` without a list, so a rebase would turn valid explicit selectors into unexpected `InvalidGatewayAgentIdError`s; either guard only when the list is supplied or require every caller to pass one.
  - confidence: 0.88
- **[P2] Preserve configured default agent selectors:** `src/gateway/http-utils.ts:100-105`
  - body: This resolver treats `openclaw/default` as the literal agent id `default` and falls back to hard-coded `main`. Current main routes `openclaw` and `openclaw/default` through `resolveDefaultAgentId(cfg)` and advertises those aliases in `/v1/models`, so configs whose default agent is not literal `main` would reject or route incorrectly.
  - confidence: 0.86
