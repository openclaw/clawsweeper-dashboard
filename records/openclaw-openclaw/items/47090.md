## Summary

Keep open, but this PR is not merge-ready: PR-head source still has a strict TypeScript contract break, a full synchronous transcript read in the archive-summary path, missing real behavior proof, merge conflicts with current main, and unresolved retention/security policy questions.

## What This Changes

Adds archivedAt-based session archive/unarchive state across gateway, Control UI, Swift protocol, archive summary files under memory, inbound auto-unarchive behavior, and a Telegram /set_topic_name command.
