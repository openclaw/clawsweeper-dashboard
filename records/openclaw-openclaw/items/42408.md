Request adds memory indexing docs, exclusion/glob configuration, and stronger diagnostics for `extraPaths` drift and retrieval-corpus contamination.

## Best Possible Solution

No high-confidence live reproduction was established. Source inspection confirms the inclusion-only `extraPaths` path and missing exclusion/docs surfaces, but the reported ranking degradation requires a real local+hybrid setup with contaminated benchmark files.

## Solution Assessment

Unclear. `excludePaths` or glob support is plausible, but the best maintainable fix needs a decided public config shape and warning policy across builtin and QMD indexing rather than an ad hoc backend-only patch.
