Request support for PostgreSQL as an optional or replacement backend for OpenClaw-owned internal storage currently backed by SQLite.

Keep this issue open as the canonical storage-backend design tracker until maintainers choose SQLite-only core storage, snapshot-based durability, or a staged backend-neutral architecture with explicit config, migration, credential, transaction, plugin, and memory-search contracts.
