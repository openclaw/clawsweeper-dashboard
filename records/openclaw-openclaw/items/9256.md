## Summary

Keep open. Current main has dangerous-code scanning for gateway-backed skill dependency installs, but the ClawHub skill install path this issue is about still downloads and installs skill folders without that security gateway. Repo docs and the threat model explicitly describe this as a remaining gap rather than an implemented fix.

## Evidence
