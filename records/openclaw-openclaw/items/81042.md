## What This Changes

The issue requests a bounded retry and negative diagnostics for the Claude CLI transcript-content probe when a session JSONL exists before an assistant record is flushed.

## Best Possible Solution

Review and land the already-linked bounded retry and diagnostics PR if it remains correct and green, then let the merge close this issue.

## Solution Assessment

Yes, the proposed file-exists-gated one-shot retry with negative diagnostics is the narrow maintainable fix because it preserves the fast missing-session path and keeps Claude CLI resume policy local to the transcript probe.
