Show the active OpenClaw config file path in Control Overview or a prominent Settings location so users can identify which instance is running.

Add a visible, copyable config path row to the existing Control UI using state.configSnapshot?.path or admin-scoped snapshot data, while keeping low-privilege gateway snapshots unchanged.
