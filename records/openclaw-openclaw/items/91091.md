## Risks / Open Questions

- A missing or inaccessible agent/session parent can still be treated as authoritative empty, so the new destructive consumers may prune session rows or exported markdown as if all sessions were deleted.

- The pinned `@openclaw/fs-safe` dependency still cannot report subtree readdir failures, so this PR only guards probed roots and extra paths.
