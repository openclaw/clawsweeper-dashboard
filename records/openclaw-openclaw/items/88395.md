## Best Possible Solution

Land the default-on `Talk` trait only if maintainers accept the explicit trait-off API contract and keep CI proof for both the default-on app path and the chat-only trait-off package path.

## Solution Assessment

Yes, if maintainers accept the compatibility contract. The implementation uses SwiftPM's native trait mechanism and gates the only ElevenLabs symbol exports; the safer alternative is to keep the dependency unconditional until OpenClawKit traits are an approved public API surface.
