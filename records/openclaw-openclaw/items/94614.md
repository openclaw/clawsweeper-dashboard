- **[P2] Route the interval through every Feishu send path:** `extensions/feishu/src/reply-dispatcher.ts:241`
  - body: `sendIntervalMs` is added as outbound rate limiting, but this PR only reads it in the reply/comment dispatcher loops. Add an upper bound or clamp through the existing timer helper before arming the delay.
  - confidence: 0.82
