  - confidence: 0.9

  - body: The reset/model-switch cleanup only removes the singular `modelHealth*` fields added here, leaving `exhaustedModels` behind. Since the fallback loop skips candidates based on that map, a `/reset` or explicit model switch can continue skipping the user's selected model until the old cooldown expires.
  - confidence: 0.86
