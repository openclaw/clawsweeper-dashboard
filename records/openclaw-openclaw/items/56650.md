  - body: The PAT branch returns only `apiKey`, `baseUrl`, and `expiresAt`, while the new editor-header constant is never passed through the current request/header override seam.   - body: The new help text advertises `--token <value>` without warning that inline secrets can be captured by shell history, command echoing, CI logs, or process listings. Add CLI/docs guidance that prefers stdin or env/ref-backed storage before exposing this as a documented automation path.
  - confidence: 0.78

  - body: `isGitHubPAT` routes both fine-grained `github_pat_` and classic `ghp_` tokens directly to Copilot, but current GitHub Copilot CLI docs list classic PATs as unsupported. Either remove `ghp_` from the supported path or add live proof and docs explaining why OpenClaw can safely support it.
  - confidence: 0.74
