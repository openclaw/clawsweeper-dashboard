## Summary

Keep open: the PR is directionally focused on the linked compaction-loop bug, but the latest head still lets marker-proven stale output and pre-marker tail pressure through unless the prompt-disparity heuristic also trips, and the real behavior proof is for an older head.

## What This Changes

The branch changes the preflight compaction transcript estimator and focused tests to detect post-usage compaction markers, bound stale usage snapshots, and preserve fresh post-usage tail pressure.

## Best Possible Solution

Land a marker-aware estimator that treats a post-usage compaction marker as sufficient stale evidence for output and pre-marker tail removal, while keeping no-marker latest usage plus tail pressure conservative.

## Reproduction Assessment

Yes. A source-level reproduction is clear: use a transcript where the last usage record is followed by a compaction marker, moderate stale prompt usage, large stale output or pre-marker tail, and a small active replay; the PR head still retains stale output/tail unless the prompt-disparity heuristic trips.

## Solution Assessment

No. The marker-gated direction is right, but marker-proven stale output and pre-marker tail should be dropped independently of the prompt-disparity threshold.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Drop marker-proven stale output and tail pressure:** `src/auto-reply/reply/agent-runner-memory.ts:610-619`
  - body: When a compaction marker follows the latest usage record, that usage snapshot predates compaction even if `rawUsagePromptTokens` is not greater than `estimatedMessageTokens * 2 + 10000`. Because `hasStaleUsageSnapshot` gates both post-marker tail narrowing and output-token removal, a moderate stale prompt with large stale output or pre-marker tail can still trigger another early preflight compaction after compaction. Treat the post-usage marker as stale evidence for output/pre-marker tail pressure and add that regression.
  - confidence: 0.87
