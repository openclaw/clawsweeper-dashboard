## What This Changes

The PR removes provider/model filtering from keyword FTS memory search and adds a regression test for cross-model keyword results.

## Best Possible Solution

Retarget or replace this PR so active `extensions/memory-core` keyword search drops provider/model filtering for MATCH, MATCH-failure LIKE fallback, and substring-only paths while preserving model-bound vector search, with focused regression coverage and redacted real `memory_search` proof.

## Solution Assessment
