## What This Changes

## Best Possible Solution

Land a narrow transport fix that treats explicit IPv6-capable config/env overrides as an opt-out from sticky IPv4 fallback, preserves default and `ipv4first` recovery, corrects the linked issue refs, and includes live or diagnostic proof.

## Reproduction Assessment

## Solution Assessment
