## What This Changes

The issue requests an opt-in hold window for forwarded exec approval prompts plus an agent-accessible retraction action that cancels held approvals before human delivery.

## Best Possible Solution

Define a default-off hold/retract lifecycle that preserves fail-open human delivery, fail-closed execution, audit logging, and focused Gateway, forwarder, agent, and Codex bridge tests.
