This PR adds `memorySearch.excludeDirs` with default excluded directory basenames, threads it through legacy memory scanner/status/schema/help/types/tests, and adds an unrelated root `ipv4-fix.cjs` helper.

## Best Possible Solution

Define one backend-consistent memory traversal exclusion contract in the active memory-host SDK and QMD collection/update path, document compatibility semantics, remove unrelated network behavior, and require a real indexing proof before merge.
