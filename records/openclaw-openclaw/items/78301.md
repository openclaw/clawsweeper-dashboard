## Best Possible Solution

Keep this as the generic diagnostics/docs tracker: review or land the setup-runtime fix, then decide a plugin SDK fetch diagnostic strategy and document the 2026.4.x to 2026.5.x runtime migration without hard-coding Weixin behavior in core.
