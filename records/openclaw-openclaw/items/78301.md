## Best Possible Solution

Keep this as the plugin diagnostics umbrella until maintainers define a security-aware diagnostic contract and land focused changes that preserve guarded-fetch safety while adding plugin identity, contract field, resolved entry paths, package/version, and remediation details.

## Solution Assessment
