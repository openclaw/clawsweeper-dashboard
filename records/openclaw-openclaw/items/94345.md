## Summary

## What This Changes

The PR adds a Slack-local map from tool names to human-readable assistant thread status labels and calls `setSlackThreadStatus` from Slack dispatch `onToolStart`.

## Best Possible Solution

Repair this branch by deriving assistant status from the shared progress display model, writing only when the status changes, covering the Slack dispatch behavior in focused tests, and adding redacted live Slack proof.

## Solution Assessment
