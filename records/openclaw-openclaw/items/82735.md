Request stable, greppable diagnostic error codes for Codex managed app-server binary and native-hook CLI resolver failures, with eventual doctor/docs lookup support.

## Best Possible Solution

Define a small public OpenClaw startup diagnostic-code convention for these Codex/native-hook resolver failures, expose it through typed errors, and add focused doctor/docs lookup coverage.

## Solution Assessment
