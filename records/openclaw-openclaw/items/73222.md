## Summary

Keep open because current main still has only a cron-session hide toggle in the Control UI chat header, not a general session-kind filter; the linked implementation PR #73311 remains open and unmerged with closing syntax for this issue.

## What This Changes

The issue requests a Control UI session dropdown filter for main, subagent, dreaming, cron, isolated, and other session kinds.

## Best Possible Solution

Review and either land or close PR #73311; the desired end state is a chat-header session-kind filter that preserves the active session when hidden and keeps any protocol change explicit.

## Reproduction Assessment
