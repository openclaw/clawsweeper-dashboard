## Summary

Keep open. This is a fresh current-stable Windows report, and current main still has source-visible health/probe differences around `auth.mode: "none"`; the open auth-none thread at https://github.com/openclaw/openclaw/issues/75780 overlaps but does not fully cover the `gateway probe` and `doctor` diagnostic contradiction.

## Best Possible Solution
