## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Yes with a maintainer acceptance caveat: reusing the existing trust-list helper and threading config through all apply entry points is narrower than adding a new config surface. The unresolved decision is whether the existing load trust list should also authorize apply-time writes.
