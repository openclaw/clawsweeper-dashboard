## Summary

This PR addresses a real multimodal recall bug, but the implementation disables the text signal for every media candidate rather than only vector-only media, creating a concrete regression for media returned solely by keyword search. The supplied direct-function output is useful arithmetic evidence, but an external PR still needs after-fix proof through a real indexed multimodal search before merge.

## What This Changes

The PR adds media-path-aware hybrid scoring that removes the text weight from image and audio candidates and adds focused fusion regression tests.

## Best Possible Solution

Renormalize only classified media candidates that actually have a vector signal and no keyword hit, preserve keyword-only media scoring, add that candidate-shape regression test, and provide redacted live multimodal search output showing the intended ranking after the patch.

## Reproduction Assessment

Yes at source level: current main assigns zero to the absent media text signal and multiplies the vector score by only the vector weight, matching the linked issue and workaround. The PR's keyword-only regression is also directly reproducible from the merge candidate shapes.

## Solution Assessment

No, not as written. Reusing the existing classifier is the correct boundary, but the score adjustment must also distinguish vector-only media from keyword-only or dual-signal media candidates.
