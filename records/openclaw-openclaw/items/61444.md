- **[P1] Import helpers from modules that export them:** `src/commands/agents.commands.import.ts:7`
  - body: `../infra/archive.js` on current main does not export `fileExists` or `readJsonFile`, so this command will fail to typecheck or load after rebasing. Import JSON and existence helpers from their actual modules instead.
  - confidence: 0.94
- **[P2] Keep the current lazy agents command loading:** `src/cli/program/register.agent.ts:3-12`
  - body: Current main lazy-loads agent action modules and no longer has the same `src/commands/agents.ts` barrel path. Add this subcommand through the existing lazy-loader pattern instead of restoring static agent command imports.
  - confidence: 0.9
- **[P1] Constrain archive-selected workspace destinations:** `src/commands/agents.commands.import.ts:115-117`
  - body: `agent.json.workspace` is archive-controlled, resolved as a user path, and then used as the destination for copied archive contents. Ignore this field by default or constrain it to an approved agent workspace root before writing files.
  - confidence: 0.91
- **[P2] Apply config before copying workspace files:** `src/commands/agents.commands.import.ts:173`
  - body: The command copies extracted workspace files before `replaceConfigFile` runs. If the config write fails because of a hash conflict or I/O error, the import has already modified the filesystem without registering the agent.
  - confidence: 0.87
- **[P2] Validate all imported manifest fields:** `src/commands/agents.commands.import.ts:180-183`
  - body: `agent.json` is untrusted JSON, but `name` and `model` are passed through without runtime validation. A non-string `name` can crash in `applyAgentConfig`, and malformed model metadata can be persisted unexpectedly.
  - confidence: 0.87
- **[P3] Avoid extracting the archive twice:** `src/commands/agents.commands.import.ts:163-168`
  - body: The importer extracts once to read `agent.json` and again to copy `workspace/`. Keep the staged extraction available through validation and final copy so large imports do not pay the decompression cost twice.
  - confidence: 0.8
