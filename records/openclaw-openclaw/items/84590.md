## Best Possible Solution

Land the sidecar backup residue scan only after preserving fs-safe's fail-closed helper contract, scanning safe default sidecar backups even when active config is invalid, and keeping config artifact scanning aligned with https://github.com/openclaw/openclaw/pull/83275.
