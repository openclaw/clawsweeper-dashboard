## Summary

Keep open. Current main still has the deferred final-event reload drop and role-less final-payload suppression that this PR repairs, and the PR diff is focused with no discrete correctness or security finding from this review.

## What This Changes

This PR changes Control UI final chat-event reload handling to require an explicit renderable assistant role before suppressing final reloads, replay deferred same-session `session.message` reloads after final events, and update focused tests plus the changelog.

## Best Possible Solution

Land this PR or an equivalent focused Control UI patch that preserves local final assistant rendering while replaying deferred same-session transcript reloads after terminal events; keep #70348 separate for Gateway history sanitization.
