  - confidence: 0.93

  - confidence: 0.91

  - body: Runtime registration alone does not populate manifest-backed media discovery/defaults. Add `contracts.mediaUnderstandingProviders` and `mediaUnderstandingProviderMetadata` with Xiaomi default models and priority so pre-runtime auto selection can find the provider.
  - confidence: 0.86
