Ship Xiaomi media understanding as Xiaomi-plugin-owned code using the current shared OpenAI-compatible video helpers, provider-http request-policy plumbing, MiMo V2 Omni defaults, manifest metadata, focused tests, docs, and changelog coverage.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Import HTTP helpers from provider-http:** `extensions/xiaomi/media-understanding-provider.ts:7-9`
  - body: This new provider imports `assertOkOrThrowHttpError`, `normalizeBaseUrl`, and `postJsonRequest` from `openclaw/plugin-sdk/media-understanding`, but current main exports those helpers from `openclaw/plugin-sdk/provider-http`. As written, the new file will not typecheck against the current SDK surface.
  - confidence: 0.94
- **[P1] Forward request policy to Xiaomi video calls:** `extensions/xiaomi/media-understanding-provider.ts:90-97`
  - body: The video request posts user video bytes with provider credentials but never resolves or forwards `params.request`, `allowPrivateNetwork`, or `dispatcherPolicy`. Existing video providers use `resolveProviderHttpRequestConfig` so operator proxy/private-network policy is honored for provider egress.
  - confidence: 0.91
- **[P1] Keep Xiaomi video off the text-only active model:** `extensions/xiaomi/media-understanding-provider.ts:65`
  - body: `describeXiaomiVideo` prefers `params.model`, and the current video runner passes the active Xiaomi reply model through. A normal Xiaomi install uses `mimo-v2-flash`, which the catalog declares as text-only, so default video understanding can call the wrong model instead of MiMo V2 Omni.
  - confidence: 0.88
- **[P2] Declare Xiaomi media metadata in the manifest:** `extensions/xiaomi/media-understanding-provider.ts:117-122`
  - body: Runtime registration alone does not update manifest-backed media discovery, default models, or auto-priority. Add `contracts.mediaUnderstandingProviders` and `mediaUnderstandingProviderMetadata` for Xiaomi so discovery and defaults work before plugin runtime loading.
  - confidence: 0.85
