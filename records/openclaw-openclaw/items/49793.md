  - confidence: 0.93
- **[P1] Keep video requests on MiMo V2 Omni:** `extensions/xiaomi/media-understanding-provider.ts:60`
  - body: Video auto selection can pass Xiaomi's active default model, mimo-v2-flash, into describeVideo. This line prefers that text-only model over the MiMo V2 Omni fallback, so normal Xiaomi installs can send video requests to the wrong model.
  - confidence: 0.9
- **[P2] Declare Xiaomi media metadata in the manifest:** `extensions/xiaomi/index.ts:62`
  - body: Runtime registration alone does not populate manifest-backed media discovery/defaults before plugin runtime is active. Add contracts.mediaUnderstandingProviders and mediaUnderstandingProviderMetadata for Xiaomi so capability loading and auto-selection can see the provider.
  - confidence: 0.86
