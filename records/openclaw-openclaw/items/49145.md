## Best Possible Solution

Yes. ## Solution Assessment

No. The feature direction belongs in the Signal plugin, but this PR is not the best mergeable solution until quote author metadata is preserved for groups, authorNumber participates in visibility decisions, failing checks are resolved, and the user-facing change is documented.
