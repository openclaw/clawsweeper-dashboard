Full review comments:

- **[P1] Keep cached legacy gateways from loading default data:** `ui/src/ui/controllers/usage.ts:285`
  - body: After the first older-gateway rejection, `rememberLegacyUsageAgentId` makes `shouldSendLegacyUsageAgentId` false. On the next refresh or date change with the same selected agent, this line makes `includeAgentId` false, so `sessions.usage` is sent without `agentId` and the UI can show the backend default agent instead of preserving the unsupported-agent error. Short-circuit selected-agent loads when this cache is set.
  - confidence: 0.94
- **[P2] Compute selected-day totals from scoped sessions:** `ui/src/ui/views/usage.ts:287-310`
  - body: With a selected usage agent, this new daily chart path is rebuilt from scoped `activeAggregates`, but day-selected header totals still come from `computeDailyTotals`, which reads unscoped `data.costDaily` from `usage.cost`. Selecting a day can therefore display cross-agent totals while the session table and chart are scoped to one agent.
  - confidence: 0.87
