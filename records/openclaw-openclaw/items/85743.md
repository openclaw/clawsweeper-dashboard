- `impact:session-state`: The root condition is orphaned `pendingFinalDelivery*` session metadata with no terminal retry or expiry state.

## Summary

Keep open: current main and the latest release still have the unbounded heartbeat pending-final-delivery replay path, and https://github.com/openclaw/openclaw/pull/85770 is the open closing PR that now owns the requested retry/TTL fail-safe.

## What This Changes

The issue requests a heartbeat fail-safe that clears orphaned `pendingFinalDelivery*` state after an attempt cap or TTL so stale text cannot replay forever.

## Best Possible Solution

Review and land https://github.com/openclaw/openclaw/pull/85770, or replace it with an equivalent bounded heartbeat replay cleanup that preserves under-bound recovery and adds focused regression coverage.
