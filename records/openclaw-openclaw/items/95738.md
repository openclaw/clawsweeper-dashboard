- **[P3] Resolve alias chains when listing directories:** `extensions/signal/src/aliases.ts:146`
  - body: `resolveSignalAliasTarget` supports alias-to-alias resolution and only rejects recursive chains, but directory listing normalizes each raw value directly. Use the same resolver here or reject chained aliases consistently.
