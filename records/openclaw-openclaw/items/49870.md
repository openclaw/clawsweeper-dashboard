  - confidence: 0.88

  - confidence: 0.9
- **[P1] Run suppression before block replies can stream:** `src/auto-reply/reply/agent-runner-hooks.ts:45-52`
  - body: The wrapper invokes the hook only after `runAgentTurnWithFallback()` returns, but block streaming can enqueue or send reply chunks inside that call. A later `{ suppress: true }` cannot reliably stop already-visible output, and `{ reinject: true }` can duplicate partial responses.
  - confidence: 0.86

  - confidence: 0.91
