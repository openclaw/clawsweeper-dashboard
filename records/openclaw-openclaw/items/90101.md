- **[P2] Do not fall back to current runtime cost for targets:** `src/agents/tools/runtime-tool.ts:213`
  - body: For `cost_estimate`, an existing offload target with no `target.cost` currently returns `context.cost`, but that describes the current runtime.   - confidence: 0.9

  - body: `prompt_summary` counts every configured offload target as available even when the target's new `availability.state` is `unavailable`, `stopping`, or `error`.
