- **[P2] Include BMP/TIFF in the Photon artifact preflight:** `extensions/media-understanding-core/image-ops.ts:161-167`
  - body: `loadOrientedPhotonImage()` calls `assertHeaderPixelBudget()` before Photon decodes, but this local header reader only recognizes PNG/GIF/WebP/JPEG.   - confidence: 0.91
- **[P2] Preserve the Sharp guard for pinned installs:** `scripts/install.sh:845`
  - body: The installer still supports `--version` and `OPENCLAW_VERSION`, and the latest shipped release still installs Sharp. Dropping `SHARP_IGNORE_GLOBAL_LIBVIPS=1` from the npm install path can make rollback or pinned-version installs fail again on hosts with a global libvips setup; keep the guard for versioned installs even if current latest no longer needs it.
  - confidence: 0.9
