Keep open. ## What This Changes

The issue requests Feishu quoted or parent message content be included in the agent-visible inbound context when a user quotes a message and mentions the bot.

## Best Possible Solution

Review and land or revise https://github.com/openclaw/openclaw/pull/84588 as the Feishu-owned fix, then close this issue once the accepted quoted-reply semantics are on main.

## Solution Assessment

Unclear and product-dependent. A Feishu-plugin-local fix that chooses the correct quoted-message id and preserves the context-visibility gate is plausible, but the active PR also changes reply-routing semantics and needs maintainer acceptance.
