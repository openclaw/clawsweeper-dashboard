## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Yes, as a patch direction: guarding the client fd and each accepted server-side client fd is the narrow maintainable fix for this IPC failure mode. The extra listening-fd guard is harmless but not essential, and merge still needs real macOS proof plus conflict resolution.
