## What This Changes

The branch adds Azure `text` and `response.text.delta` handling to `src/llm/providers/openai-responses-shared.ts` and adds a conversion test for Azure assistant text replay.

## Best Possible Solution

Move the Azure `text` normalization into the active Azure Responses transport parser, keep the shared helper aligned only if that path is still supported, model the provider-specific event shape at a typed boundary, add stream-path regression coverage, and attach redacted live Azure proof.
