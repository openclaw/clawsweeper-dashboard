  - body: The new `remote-url` branch calls `loadWebMedia` with only `maxBytes`, unlike the existing image tool remote path that forwards the configured remote-media SSRF policy.   - confidence: 0.91

  - body: The URL regex can capture a closing `)` from parenthesized or Markdown-style text, and the trailing-punctuation sanitizer does not remove it. `new URL()` accepts `https://example.com/image.png)`, so the preload fetches the wrong path and can drop a common prompt image URL shape.
  - confidence: 0.84
