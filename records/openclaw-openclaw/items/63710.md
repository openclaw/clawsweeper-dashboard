## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.92

Full review comments:

- **[P1] Restrict remote-image detection before fetching:** `src/agents/pi-embedded-runner/run/images.ts:42`
  - body: `HTTP_URL_REGEX_SOURCE` matches every HTTP(S) URL, so a normal prompt link can trigger a gateway fetch before `media.kind` discards non-images. Limit native preload to explicit image refs, or another maintainer-approved signal, so ordinary links do not create surprise outbound requests.
  - confidence: 0.91
- **[P1] Forward the configured SSRF policy:** `src/agents/pi-embedded-runner/run/images.ts:458-474`
  - body: The new `remote-url` branch calls `loadWebMedia` with only `maxBytes`, unlike existing remote media paths that pass the configured SSRF policy. Thread the policy through `detectAndLoadPromptImages` and `loadImageFromRef` before adding gateway-side URL preloads.
  - confidence: 0.9
- **[P2] Keep remote URL dedupe case-sensitive:** `src/agents/pi-embedded-runner/run/images.ts:292`
  - body: `addRemoteUrlRef` reuses `normalizeRefForDedupe`, which lowercases on Windows for filesystem paths. HTTP URL paths can be case-sensitive, so Windows deployments can collapse two distinct image URLs and drop one from the native image payload.
  - confidence: 0.84
- **[P2] Only mark emitted remote refs as seen:** `src/agents/pi-embedded-runner/run/images.ts:347`
  - body: When a `[media attached: path | url]` entry has a local path, this branch marks the URL as seen even though it does not emit a `remote-url` ref. If the same URL is explicitly repeated later, the full-prompt scan skips it and the image can be lost when the local path cannot load.
  - confidence: 0.82
- **[P2] Trim closing delimiters from URL refs:** `src/agents/pi-embedded-runner/run/images.ts:96`
  - body: The sanitizer trims commas and sentence punctuation but not closing delimiters, while the URL regex can include `)`. Parenthesized or Markdown-style image URLs can be fetched with the trailing `)` in the path, causing avoidable image-load failures.
  - confidence: 0.8
