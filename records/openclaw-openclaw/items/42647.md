## What This Changes

The issue requests a durable memory provenance model with required source refs, multi-source support, merge/edit provenance retention, retrieval attribution, and promotion blocking for source-less memories.

## Best Possible Solution

Design and implement one memory-core durable memory contract with required source refs, revision lineage, retrieval/review attribution, and promotion enforcement, coordinated with the linked Memory MVP schema/write/review work.
