- **[P2] Preserve config write-stability before reload:** `src/gateway/config-reload.ts:414-416`
  - body: By deleting `awaitWriteFinish` here, a slow external edit can fire a change before `openclaw.json` is complete. `runReload()` then treats the invalid snapshot as a skipped reload and does not retry, so the final valid config may never apply if no later event arrives; keep a change-triggered stability wait or retry around config reads while removing idle overhead.
  - confidence: 0.87

  - body: In polling mode, `shouldIgnoreSkillsWatchPath()` allows SKILL.md through Chokidar and the raw-event stability path returns early, so deleting `awaitWriteFinish` makes polling SKILL.md changes bump snapshots immediately while the file may still be mid-write. Preserve a per-change stability check for this path or keep equivalent stability only when polling is enabled.
  - confidence: 0.78
