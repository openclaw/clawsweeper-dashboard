- **[P1] Avoid refreshing canvas by closing the websocket:** `ui/src/ui/gateway.ts:406`
  - body: The timer closes the active Gateway websocket every refresh interval. Current close handling rejects pending RPCs, and the app's reconnect hello path marks orphaned chat runs interrupted and clears local stream/tool state, so a long chat or request crossing this boundary can lose terminal state even when no canvas is visible. Refresh the canvas surface token without tearing down the active transport, or gate it to a proven safe idle state.
  - confidence: 0.9
