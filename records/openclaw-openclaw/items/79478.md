## Summary

Keep open. The force-pushed PR appears to fix the earlier comment-only network false positive and the PR body has sufficient real CLI proof, but the new matcher still scans YAML frontmatter as parameter text, which can produce critical false positives for ordinary skill descriptions that mention secrets or tokens.

## What This Changes

The PR adds a deep security-audit check that correlates sensitive-looking `SKILL.md` parameter guidance with outbound network patterns in installed skill code, plus tests, docs, and a changelog entry.

## Best Possible Solution

Skip YAML frontmatter before collecting sensitive parameter evidence, keep the comment-stripping reuse, and land the focused audit check after maintainer security review.

## Reproduction Assessment

Yes, source-level reproduction is clear: a `SKILL.md` with `description: ... token ...` in frontmatter plus any executable network call will satisfy the new matcher even without a sensitive argument line. The intended positive and comment-only negative cases also have terminal JSON proof in the PR body.

## Solution Assessment

No as written. The direction is narrow and useful, but the implementation should strip or skip YAML frontmatter before treating key/value lines as model-facing parameter guidance.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Skip SKILL.md frontmatter when matching parameters:** `src/security/audit-skill-markdown-context.ts:69-83`
  - body: The markdown scan runs over the entire `SKILL.md`, and `isLikelySkillParameterLine` accepts any `key: value` shape. That means YAML frontmatter such as `description: ... token ...` is treated as sensitive parameter guidance; since `description` is required metadata and shown to the agent, a normal networked skill with a secret-related description can now get a critical `skills.markdown_sensitive_network` finding even when no argument asks for a secret. Strip or skip frontmatter before collecting parameter evidence.
  - confidence: 0.87
