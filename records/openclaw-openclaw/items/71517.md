## Summary

This PR adds a shared media filename decoder, wires remote media fetch through it, normalizes Feishu inbound filenames, and updates related tests plus SDK exports.

## Best Possible Solution

Land a rebased focused decoder in src/media, keep Feishu-specific plumbing plugin-local unless an approved SDK seam is fully documented and baselined, and leave https://github.com/openclaw/openclaw/issues/48788 open for remaining channel migrations.
