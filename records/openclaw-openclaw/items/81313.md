## What This Changes

The branch adds a heartbeat-only pending-final replay suppression option, passes it from the heartbeat runner, skips heartbeats when base or isolated reply runs are active, and adds focused regression tests.

## Best Possible Solution

Land this focused suppression after redacted live Telegram or equivalent runtime proof shows stale pending final text is no longer emitted to the wrong heartbeat target while ack cleanup and durable recovery still work.
