## Summary

Keep open. Current main still rejects the documented Feishu `channels.feishu.blockStreaming` key because the strict Feishu source and generated schemas omit it, and #75573 is an open closing PR for the same gap.

## Best Possible Solution

Review and land or replace #75573 with a Feishu-owned source-schema change, regenerated metadata, focused validation tests, and an explicit runtime block-streaming decision.

## Solution Assessment
