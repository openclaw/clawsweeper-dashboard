## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Rollback registry-stored interactive handlers:** `src/plugins/registry.ts:2850-2856`
  - body: This stores successful `registerInteractiveHandler` calls in `registry.interactiveHandlers`, but the loader's snapshot/restore rollback path does not include that new array. If a plugin registers an interactive handler and then throws, rollback clears the global handler map but leaves the failed plugin's handler in the live registry; the new dispatch recovery can later restore and invoke it. Please include this array in snapshot/restore and extend the failing side-effects test to assert it stays empty.
  - confidence: 0.92
