Full review comments:

- **[P1] Rollback registry-stored interactive handlers:** `src/plugins/registry.ts:2850-2856`
  - body: This stores successful `registerInteractiveHandler` calls in `registry.interactiveHandlers`, but the loader snapshot/restore rollback path does not include that new array. Please snapshot/restore this array and extend the failed side-effects test to assert it stays empty.
