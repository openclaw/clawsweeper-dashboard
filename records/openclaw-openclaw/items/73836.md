Keep open. Current main has mitigations for the refresh, heartbeat, typing, and control-plane latency parts, but the outbound delivery mirror still collapses media URLs to filename text before writing session history, so the reported Control UI media-preview defect remains source-confirmed.

## What This Changes

## Best Possible Solution

Keep the landed responsiveness mitigations and fix outbound delivery mirroring so successful media sends append previewable managed media or attachment content to session history, then reassess only if new current-main evidence shows the reconnect or typing symptoms still fail.
