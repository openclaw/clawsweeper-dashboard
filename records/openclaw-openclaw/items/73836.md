## Summary

Keep open. Current main mitigates the Control UI refresh, heartbeat visibility, and Telegram typing critical-path portions, but it still mirrors outbound media into the transcript as filename text instead of previewable media, and the Docker/WSL2 responsiveness symptoms are still tied to the open #73428 latency tracker.

## What This Changes

The issue reports an umbrella regression across Control UI reconnect/refresh behavior, Telegram typing presence, heartbeat transcript noise, and Control UI mirroring of outbound Telegram media during active runs.

## Solution Assessment
