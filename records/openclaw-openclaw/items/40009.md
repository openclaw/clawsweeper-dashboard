Replace the Control UI raw configuration textarea with a structured JSON5-capable editor supporting highlighting, folding, validation, formatting, and unchanged save behavior.

## Best Possible Solution

Land one focused JSON5-aware editor after it preserves the existing reveal, round-trip, dirty-state, and save contracts, avoids destructive formatting surprises, and demonstrates the real Control UI flow with an accepted dependency cost.
