## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Treat terminal timeout snapshots as finished:** `ui/src/ui/app-chat.ts:84`
  - body: `agent.wait` can return `status: "timeout"` for both an active wait expiry and a terminal timeout snapshot carrying metadata such as `endedAt`, `stopReason`, or `livenessState`. This branch treats every timeout as still active, so a completed timed-out run can keep retrying forever and leave the queued message blocked.
  - confidence: 0.9
- **[P3] Suppress transient watchdog probe errors:** `ui/src/ui/app-chat.ts:97-99`
  - body: The watchdog is a silent 50 ms background recovery probe that retries automatically. Writing a thrown probe error into `host.lastError` can show a visible error banner even when the main gateway connection and run are otherwise healthy.
  - confidence: 0.82
