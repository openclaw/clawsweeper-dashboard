## Summary

Keep open: the linked Control UI bug is still source-reproducible on current main and this PR is a focused implementation candidate, but the current patch mishandles terminal timeout snapshots and lacks after-fix real behavior proof.

## What This Changes

The PR adds a Control UI chat-run watchdog that probes `agent.wait` after chat inactivity, clears stale run state, reloads history, flushes queued messages, and adds regression tests.

## Best Possible Solution

Land a rebased watchdog or equivalent Control UI recovery path that treats terminal timeout snapshots as finished, keeps transient watchdog probe failures out of visible error banners, preserves focused regression coverage, and includes redacted real browser/gateway proof.

## Reproduction Assessment

Yes, at source level. On current main, `sendChatMessage` sets `chatRunId`; if the terminal chat event is missed, `isChatBusy` stays true and `flushChatQueue` returns before sending queued work.

## Solution Assessment

No, not as currently patched. A UI-side `agent.wait` watchdog is a narrow maintainable approach, but it must distinguish wait-only deadlines from terminal timeout snapshots and avoid surfacing transient background probe failures as user-visible errors.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.87

Full review comments:

- **[P2] Recover terminal timeout wait results:** `ui/src/ui/app-chat.ts:84`
  - body: `agent.wait` can return `status: "timeout"` for already-ended timeout or cancellation snapshots when fields like `endedAt`, `error`, or `stopReason` are present. Retrying every timeout response keeps the UI stuck for those terminal runs and leaves queued messages blocked.
  - confidence: 0.87
- **[P3] Suppress transient watchdog probe errors:** `ui/src/ui/app-chat.ts:97-99`
  - body: The watchdog probe runs in the background, so assigning `host.lastError` when the short `agent.wait` request throws can show a visible error banner even though the main connection may be healthy and the watchdog will retry.
  - confidence: 0.82
