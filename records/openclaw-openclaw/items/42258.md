## Summary

Keep open: current main still has a source-provable stale `chatRunId` queue-block path, and this PR is a plausible fix direction but needs real behavior proof plus fixes for terminal `agent.wait` snapshots before merge.

## What This Changes

The PR adds a Control UI chat-run watchdog that probes `agent.wait` after inactivity, clears stale local run state, reloads history, flushes queued messages, and adds focused tests.

## Best Possible Solution

Land a rebased watchdog or equivalent Control UI recovery path that uses authoritative `agent.wait` terminal metadata, stays aligned with the current run-lifecycle helper, preserves regression coverage, and includes redacted live browser/gateway proof.

## Reproduction Assessment

Yes, at source level. Leave `chatRunId` non-null after a missed terminal chat event, submit a normal message, and current main queues the message while `flushChatQueue` exits because `isChatBusy` remains true.

## Solution Assessment

No, not as currently patched. A UI-side `agent.wait` watchdog is a narrow direction, but this branch must treat terminal timeout/cancel snapshots as finished, avoid visible transient probe errors, and rebase onto the newer run-lifecycle code.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Treat terminal timeout snapshots as finished:** `ui/src/ui/app-chat.ts:84`
  - body: `agent.wait` can return an already-ended run as `status: "timeout"` with terminal metadata such as `endedAt`, `stopReason`, `livenessState`, or `yielded`. This watchdog treats every timeout as still active, so a timed-out or cancelled run can keep looping and leave the queued message blocked.
  - confidence: 0.87
- **[P3] Suppress transient watchdog probe errors:** `ui/src/ui/app-chat.ts:97-99`
  - body: The probe is a silent 50 ms background recovery check that retries automatically. Writing a thrown probe error into `host.lastError` can show a visible error banner even when the main gateway connection and run are otherwise healthy.
  - confidence: 0.82
