## Summary

Keep open: this is a plausible narrow bug fix, but the current patch still has a blocking timeout-snapshot bug and lacks required real browser/gateway after-fix proof.

## What This Changes

The PR adds a Control UI chat-run watchdog that probes `agent.wait` after inactivity, clears stale run state, reloads chat history, flushes queued messages, and adds focused regression tests.

## Best Possible Solution

Land a rebased watchdog or replacement that treats terminal timeout snapshots as finished, keeps transient watchdog probe failures out of visible error banners, retains focused regression coverage, and includes redacted real browser/gateway proof.

## Reproduction Assessment

Yes, at source level. On current main, a missed terminal chat event can leave `chatRunId` set, `isChatBusy` remains true, and `flushChatQueue` returns before dequeuing the next message.

## Solution Assessment

Mostly, but not as currently patched. A UI-side `agent.wait` watchdog is a narrow maintainable fix, but this patch must distinguish wait-only deadlines from terminal timeout snapshots and avoid surfacing transient background probe failures as user-visible errors.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Recover terminal timeout wait results:** `ui/src/ui/app-chat.ts:84`
  - body: `agent.wait` can return `status: "timeout"` for terminal timeout or cancelled snapshots when fields like `endedAt`, `error`, `stopReason`, or `livenessState` are present. Retrying every timeout response means an already-ended timed-out run can keep the watchdog looping forever and leave queued messages blocked.
  - confidence: 0.86
- **[P3] Suppress transient watchdog probe errors:** `ui/src/ui/app-chat.ts:97-99`
  - body: The watchdog probe runs in the background, so assigning `host.lastError` when the short `agent.wait` request throws can show a visible error banner even though the main connection may still be healthy and the watchdog can retry.
  - confidence: 0.82
