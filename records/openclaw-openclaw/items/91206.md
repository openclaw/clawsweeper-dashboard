## What This Changes

## Best Possible Solution

Forward the resolved model through schema-valid `agent` params, carry override trust through gateway dispatch/client internal state, and add focused accepted-run coverage plus redacted real-run proof.

## Reproduction Assessment

## Solution Assessment

No. Passing `model` is necessary, but the best fix is to keep the `agent` request schema-valid and carry override authorization through gateway client or dispatch state instead of a request-body flag.
