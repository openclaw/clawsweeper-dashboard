## What This Changes

## Best Possible Solution

Review and land https://github.com/openclaw/openclaw/pull/83081, or a narrower replacement with the same unloaded-channel restart and stale-write protections, then close this issue after the fix merges.

## Reproduction Assessment

## Solution Assessment
