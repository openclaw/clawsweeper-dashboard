## Summary

## Best Possible Solution

Merge the helper-level fix only if maintainers confirm explicit selection is the intended allowlist trust gesture across setup, onboarding, and CLI enablement; otherwise narrow the mutation to the channel setup path or add an explicit prompt.

## Solution Assessment
