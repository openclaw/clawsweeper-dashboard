## Summary

Keep open. Current main still ranks Dreaming deep promotion candidates by hard gates plus composite-score thresholding, and adding randomized wildcard promotion for durable memory remains a product decision rather than a stale or already-implemented cleanup item.

## What This Changes

The issue requests adding a wildcard sampling lane so Dreaming deep-phase promotion can occasionally include sub-threshold short-term memories that pass hard gates but miss the composite score cutoff.

## Best Possible Solution

Keep the issue open for maintainer product review, then implement any approved wildcard lane with deterministic test controls and explicit coverage for default threshold behavior versus opt-in randomness.
