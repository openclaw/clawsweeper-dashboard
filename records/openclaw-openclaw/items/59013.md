## Summary

## What This Changes

The branch adds Windows-only EPERM retry handling and regression tests for session write-lock acquisition in the older inline lock loop.

## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

No as submitted. The mitigation idea is reasonable, but the patch needs to move to the current fs-safe-backed acquisition boundary and be proven in a real Windows setup.
