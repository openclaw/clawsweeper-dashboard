## What This Changes

## Best Possible Solution

Thread the trusted sender fields through `ChannelAgentToolFactory` while preserving the existing `includeChannelTools` and `includeOpenClawTools` gates, add focused regression coverage, and land only after redacted real channel-plugin proof is provided.
