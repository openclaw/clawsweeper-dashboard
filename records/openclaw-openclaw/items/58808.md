## Best Possible Solution

Not applicable as a strict bug reproduction because this is a feature/API-extension PR. Source inspection does confirm the current-main gap: channel agent-tool factories receive only `cfg` while nearby OpenClaw tool construction already receives `requesterSenderId` and `senderIsOwner`.
