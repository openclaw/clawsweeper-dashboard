## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Stop marking Cursor runs replay-safe:** `extensions/cursor-sdk/src/run-attempt.ts:249`
  - body: Cursor SDK can emit tool calls for file writes, shell commands, MCP, and subagents, but this wrapper drops those events and reports every success/error as `hadPotentialSideEffects: false`. OpenClaw retries empty/error turns based on this flag, so a Cursor run that already changed the workspace can be replayed and duplicate side effects.
  - confidence: 0.9
- **[P2] Catch Agent.create failures inside the harness result:** `extensions/cursor-sdk/src/run-attempt.ts:115`
  - body: `Agent.create(...)` is awaited before the guarded `try/finally`, so auth, config, or network failures reject out of `runCursorSdkAttempt` instead of returning an `AgentHarnessAttemptResult`, and the timeout/upstream abort listener cleanup is skipped. Move creation inside guarded handling and only dispose once an agent exists.
  - confidence: 0.87
