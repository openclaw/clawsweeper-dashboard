## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Stop marking Cursor runs replay-safe:** `extensions/cursor-sdk/src/run-attempt.ts:249`
  - body: Both result builders report Cursor attempts as replay-safe, but Cursor SDK can execute file, shell, MCP, and subagent tools while this wrapper drops those events. OpenClaw uses this metadata to decide whether it can retry empty/compaction-interrupted turns, so a run that already mutated the workspace can be replayed and duplicate side effects.
  - confidence: 0.9
- **[P2] Return harness results for Agent.create failures:** `extensions/cursor-sdk/src/run-attempt.ts:115`
  - body: `Agent.create(...)` is awaited before the guarded `try/finally`, so auth, config, or network failures reject out of `runCursorSdkAttempt` instead of returning an `AgentHarnessAttemptResult`, and the timeout/upstream abort cleanup is skipped. Move creation into the guarded block and only dispose when an agent exists.
  - confidence: 0.87
