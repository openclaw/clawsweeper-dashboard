## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P1] Mark Cursor SDK attempts as replay-unsafe after send:** `extensions/cursor-sdk/src/run-attempt.ts:249`
  - body: The harness always reports `hadPotentialSideEffects: false` and `replaySafe: true`, but Cursor owns file edit, terminal, and subagent tools while this wrapper filters tool events out. OpenClaw uses this metadata to decide whether empty/error retries can safely replay a turn, so a Cursor run that changed files but produced no visible assistant text can be submitted again and duplicate side effects. Mark runs unsafe once sent, or derive the flag from Cursor tool events before claiming replay safety.
  - confidence: 0.87
- **[P2] Handle Agent.create failures inside the attempt result:** `extensions/cursor-sdk/src/run-attempt.ts:115`
  - body: `Agent.create(...)` is awaited before the `try/catch/finally`, so auth, config, or network failures during agent creation reject out of `runCursorSdkAttempt` instead of returning an `AgentHarnessAttemptResult`. That path also skips the timeout cleanup and upstream abort listener removal; move creation into a guarded block and dispose conditionally after an agent exists.
  - confidence: 0.86

## Security Review

Status: needs_attention

Summary: The patch introduces a trusted native agent runtime plus dependency/native-build policy changes that need security review before merge.

Concerns:

- **[high] Cursor side effects are reported replay-safe:** `extensions/cursor-sdk/src/run-attempt.ts:249`
  - body: Cursor agents can execute their own file-edit, terminal, and subagent actions, but the wrapper reports every attempt as having no potential side effects. That can let OpenClaw retry/replay a turn after Cursor already mutated the workspace, which is a safety regression for a high-power runtime.
  - confidence: 0.86
- **[medium] sqlite3 build scripts are enabled workspace-wide:** `pnpm-workspace.yaml:89`
  - body: Adding `sqlite3: true` to `allowBuilds` expands install-time native/script execution for a new transitive dependency path. Combined with Socket's platform-package publisher-change warnings in the discussion, this needs explicit maintainer supply-chain acceptance or a different distribution boundary such as ClawHub.
  - confidence: 0.78
