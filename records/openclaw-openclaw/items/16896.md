Request adds a right-click/contextual Reply action in Dashboard WebChat that quotes a selected message into the composer.

## Best Possible Solution

Land a narrow Control UI quote-reply flow with accessible message Reply controls, a right-click menu, selected quote preview, composer focus, and safe quoted-Markdown insertion while leaving durable reply metadata to the separate threading work.
