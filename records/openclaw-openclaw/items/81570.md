## What This Changes

The PR adds a single-entry process-local memo for plugin metadata snapshots, clone-isolates cached results, memoizes only persisted/provided and policy-stale derived snapshots, and ignores vanished recovered install records on the persisted fast path.

## Best Possible Solution

Land the narrow memoization hotfix after the PR is marked ready and plugin/core validation passes, while keeping the broader freshness matrix in https://github.com/openclaw/openclaw/pull/81064 or a follow-up.
