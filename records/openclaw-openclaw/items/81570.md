## Summary

Keep open: the PR has sufficient real beta6 timing proof and the narrow memoization approach is directionally sound, but the current head has blocking typecheck, lint, and test failures introduced by the patch.

## What This Changes

This PR adds a single-entry process-local memo around plugin metadata snapshots, clone-isolates cached snapshots, memoizes persisted/provided and policy-stale derived snapshots, and ignores vanished recovered install records on the persisted fast path.
