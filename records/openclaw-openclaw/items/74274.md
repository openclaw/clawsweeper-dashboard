  - body: `buildAssistantAttachmentDownloadUrl` appends `download=1` to every non-image attachment URL.   - confidence: 0.86

  - body: The new header interpolates `path.basename(localPath)` directly after stripping only quotes and CR/LF. Non-ASCII filenames can make Node reject the header and fall into the route's 404 catch path, so use an ASCII fallback plus `filename*` and cover it with a header-validation test.
  - confidence: 0.82
