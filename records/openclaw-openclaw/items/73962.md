## Best Possible Solution

Land the narrow assistant-media guard after rebase and gateway validation, keeping the behavior limited to the agent-run media transcript supplement and closing the linked duplicate-message bug after merge.

## Solution Assessment

Yes. Guarding the supplement on persisted non-text media content is the narrowest maintainable fix because the normal model transcript owns text-only replies while successful audio/image supplements still persist media content.
