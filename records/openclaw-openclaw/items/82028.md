- **[P2] Reject normalized heading-marker-heading skeletons:** `extensions/memory-core/src/short-term-promotion.ts:354-360`
  - body: `normalizeStore` sees already-persisted snippets after whitespace has been collapsed, so a range such as `## Tagesnotizen\n-\n\n## Entscheidungen` can be stored as `## Tagesnotizen - ## Entscheidungen`. Please detect marker separators between headings before accepting the snippet.
