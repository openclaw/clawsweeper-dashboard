## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.93

Full review comments:

- **[P1] Wire the directory store into runtime operations:** `src/cron/store.ts:196`
  - body: The new directory-store helpers are exported but not called by Gateway startup or cron add/update/remove paths, which still use a single configured store path through `loadCronStore` and `saveCronStore`. As written, creating `cron/jobs/` will not activate the advertised store.
  - confidence: 0.94
- **[P1] Key directory files by immutable job ids:** `src/cron/store.ts:246-247`
  - body: `saveCronJobFile` builds the filename from `job.name`, which is mutable and not unique. Two jobs with the same name can overwrite each other, and a rename writes a second file for the same id; use a stable contained id-based filename or remove the old id-matched file before writing.
  - confidence: 0.97
- **[P1] Delete job files by persisted ids:** `src/cron/store.ts:271-272`
  - body: `removeCronJobFile` compares the requested id to the filename-derived slug, but files saved from human names commonly have a slug different from the stable job id. Removal then returns false and leaves the job file behind; load the candidate file and match `job.id`, or make filenames id-keyed.
  - confidence: 0.96
- **[P1] Preserve the split runtime state sidecar:** `src/cron/store.ts:248`
  - body: Current main strips runtime fields from `jobs.json` and stores `updatedAtMs` plus `state` in `jobs-state.json`, but this writer serializes the whole job into each per-job file and the directory loader never merges the sidecar. That can regress pending slots, running markers, and last-run metadata after upgrade.
  - confidence: 0.9
- **[P2] Reject unsafe raw filename fallback:** `src/cron/store.ts:151`
  - body: When slugification produces an empty string, this returns the raw name or id, so values containing path separators can escape the intended `jobs/` directory once used in `path.join`. Reject unsafe values or fall back to a sanitized stable id instead.
  - confidence: 0.88
