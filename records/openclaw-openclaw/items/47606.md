## What This Changes

Add enforceable execution anti-drift runtime guards for artifact-backed progress updates, status-loop escalation, timer-based stalls, post-green idle handling, persisted run state, telemetry, and a rollout flag.

## Best Possible Solution

Define a shared progress-fact and guard-state contract at the task/runtime boundary, persist it with task/flow state, and enforce artifact-gated status, timers, no-idle-after-green, and telemetry behind an explicit rollout flag across all execution paths.
