## Summary

Keep open: current main has adjacent task persistence, Task Flow, diagnostic stall timers, and loop detection, but it still lacks the requested cross-runtime artifact-gated status contract, persisted guard/escalation state, escalation timers, and no-idle-after-green enforcement.

## What This Changes

Request to add enforceable execution anti-drift guards for multi-batch agent runs, including persisted run state, artifact-gated status updates, escalation timers, a status-loop breaker, no-idle-after-green behavior, and rollout telemetry.

## Best Possible Solution

Define a shared progress-fact and guard-state contract at the task/runtime boundary, persist it with task/flow state, and enforce status gating and escalation across CLI, cron, ACP/subagent, and embedded-runner paths behind an explicit rollout flag with telemetry.
