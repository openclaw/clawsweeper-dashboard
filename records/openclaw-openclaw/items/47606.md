## Summary

Keep open: current main has adjacent task persistence, session-liveness diagnostics, and tool-loop circuit breakers, but it still lacks the requested cross-runtime artifact-gated status contract, persisted guard/escalation state, escalation timers, and no-idle-after-green enforcement.

## What This Changes

The issue requests enforceable runtime guardrails that persist per-run state, require artifacts for progress status, escalate stalled status loops, and prevent idle gaps after green verification.

## Best Possible Solution

Define a shared progress-fact and guard-state contract at the task/runtime boundary, persist it with task or flow state, and enforce artifact-gated status, timers, no-idle-after-green, and telemetry behind an explicit rollout flag.
