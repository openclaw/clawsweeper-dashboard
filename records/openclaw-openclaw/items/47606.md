## Summary

## What This Changes

The issue requests runtime anti-drift enforcement for agent execution status, including artifact-gated progress updates, persisted guard state, escalation timers, and no-idle-after-green handling.

## Best Possible Solution

Define a shared progress-fact and guard-state contract at the task/runtime boundary, persist it with task or TaskFlow state, then enforce artifact-gated status, timers, no-idle-after-green, and telemetry behind an explicit maintainer-approved rollout flag.
