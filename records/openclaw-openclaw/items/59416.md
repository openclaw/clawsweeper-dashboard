## Best Possible Solution

Review or replace the canonical Copilot provider-owned capability fix so static defaults, live catalog metadata, active runtime thinking profiles, bundled provider-policy profiles, and UI/session validation agree, enabling `xhigh` only for models with confirmed Copilot support.
