  - body: This selection path now writes only `memory.recall`, but the PR also keeps the legacy `plugins.slots.memory` key for plugin compatibility. Update the legacy key in the non-conflicting selection path or make the supported migration behavior explicit and tested.

  - body: These errors now include the full agent output and request log. The current main code only reports bounded tails/summaries because these proof logs can contain prompts, private endpoints, tokens, or other sensitive runtime data that then lands in CI artifacts or review comments. Restore the bounded tail/summary helpers and the coverage that prevented old output from being dumped.
  - confidence: 0.94

That lets generated npm shrinkwraps drift from the pinned dependency graph, and this branch already changes lock entries away from the current pinned lru-cache version. Restore the scoped override logic and tests before touching shrinkwrap output.
  - confidence: 0.9
