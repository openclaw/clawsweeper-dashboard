## Summary

The PR adds a pre-send OpenAI-compatible chat payload guard, changes blank prompt replay classification for system/tool-only history, and adds focused regressions for transport, prompt helper, and compaction branch recovery.

## Best Possible Solution

Land this after maintainers explicitly accept the global user/assistant-turn invariant, or scope the guard to an existing provider/model compatibility signal with matching default and scoped tests.
