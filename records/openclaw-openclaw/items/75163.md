The issue requests that Control UI/TUI model switches canonicalize configured aliases such as provider/Alias before persisting session model overrides.

## Solution Assessment

Yes. The maintainable fix belongs in shared model-resolution canonicalization before sessions.patch persists the override; the current issue should stay open until the linked fix is refreshed and merged.
