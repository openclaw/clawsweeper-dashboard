The issue requests built-in or officially supported content-based prompt-injection scanning for tool results before they reach the model.

## Best Possible Solution

Keep this issue as the canonical maintainer security-design item, define the tool-result scanning contract, then implement the chosen bundled middleware or vetted plugin/API path with focused regression coverage.
