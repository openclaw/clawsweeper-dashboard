Land a focused target-repo fix that centralizes completion profile-path resolution for check/install/update/doctor/setup, with regression tests for zsh, fish, and bash fallback while keeping PowerShell-specific follow-up aligned with https://github.com/openclaw/openclaw/issues/44296.
