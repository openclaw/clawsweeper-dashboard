## Summary

Keep open. Current main still resolves shell-completion profiles from hardcoded home-backed paths for zsh and fish, still has the bash check/install mismatch, and the same-author fix PR #63082 remains open with closing syntax.

## What This Changes

Shell completion install/status should respect `ZDOTDIR`, `XDG_CONFIG_HOME`, and consistent bash `.bashrc`/`.bash_profile` fallback behavior.

## Best Possible Solution

Land #63082 or an equivalent focused PR that centralizes profile-path resolution for completion status, install, update, doctor, and setup messaging with unit coverage for zsh, fish, and bash fallback cases.

## Reproduction Assessment

Yes. Source inspection gives a high-confidence path: set `ZDOTDIR` or `XDG_CONFIG_HOME`, run completion install/update, and current main still resolves zsh/fish profiles under `$HOME`; bash status can also miss a `.bash_profile` install.

## Solution Assessment

Yes. A shared resolver used by both check and install paths is the narrowest maintainable fix; reviewing or replacing #63082 is better than starting duplicate repair work.
