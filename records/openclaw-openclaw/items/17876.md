## What This Changes

## Best Possible Solution

Define and implement a maintainer-designed `openclaw models` migration workflow that stages any provider/model candidate, verifies intended agent surfaces before promotion, preserves the old primary in the fallback chain, persists rollback state, and documents/tests the config transitions.
