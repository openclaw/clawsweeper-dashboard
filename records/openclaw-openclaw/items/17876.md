## Summary

## What This Changes

Request a staged model migration workflow that tests a new model as fallback, promotes it only after verification, preserves the old primary, and supports rollback.

## Best Possible Solution

Design one model migration workflow that stages an arbitrary candidate, verifies it before promotion, preserves the old primary as fallback or rollback state, documents the transitions, and reconciles with the existing probe/fallback issues.
