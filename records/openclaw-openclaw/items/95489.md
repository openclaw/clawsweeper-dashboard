## What This Changes

The issue requests that `claude-cli` out-of-credits generic failure output advance the configured model fallback chain instead of being delivered as the final assistant response.

## Best Possible Solution

Land https://github.com/openclaw/openclaw/pull/95508 or an equivalent narrow fix that classifies only the exact centralized generic external-runner failure copy before the visible-output success guard while preserving no-retry guards.

## Solution Assessment
