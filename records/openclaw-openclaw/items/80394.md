## Summary

Keep open. ## Best Possible Solution

Rebase and land a single `modelAllowlist` contract that preserves `models` runtime metadata while enforcing the same effective per-agent allowlist in `models.list`, Control UI/TUI pickers, `/model`, and `sessions.patch`, with updated docs, regression tests, and fresh real UI proof.
