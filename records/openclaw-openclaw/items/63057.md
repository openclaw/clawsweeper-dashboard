## What This Changes

The PR updates Anthropic request construction to use a nullish thinking-budget fallback and skip the legacy manual `thinking` block when the resolved budget is zero.

## Best Possible Solution

Land a narrow Anthropic transport fix that emits legacy manual thinking only for provider-valid budgets, preserves adaptive thinking and the undefined-budget fallback, omits invalid budgets, and covers the zero, sub-minimum, and valid paths with tests plus redacted real proof.

## Solution Assessment

No as submitted. Nullish coalescing and omitting zero are the right direction, but the guard also needs to enforce Anthropic's valid manual-budget range and the PR needs focused regression tests plus real behavior proof.
