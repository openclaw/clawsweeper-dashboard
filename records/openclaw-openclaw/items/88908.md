- **[P2] Keep shutdown state out of the close runtime:** `src/gateway/server.impl.ts:548-549`
  - body: This startup reset loads `server-close.runtime.js`, which re-exports `server-close.ts`; that module statically imports shutdown-only agent, channel, and plugin cleanup code. Move this state into a small lifecycle-state module used by startup, HTTP, and close code.

Reset it when a new shutdown starts or when startup clears the shutdown state.
