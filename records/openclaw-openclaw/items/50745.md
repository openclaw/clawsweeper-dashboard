This PR adds a WhatsApp `lookupPnLidEntry` active listener API/type, implements LID and phone lookup in the inbound monitor, updates mocks/tests, and edits the unreleased changelog.

## Best Possible Solution

## Solution Assessment

No. The requested API may be useful, but this patch is not the best merge shape until it reuses existing resolver semantics, preserves the changelog, rebases cleanly, and gets maintainer approval for exposing phone-number lookup to plugins.
