## What This Changes

Requested support for attaching cost metadata to auto-discovered Amazon Bedrock models without duplicating an explicit provider model block.

## Best Possible Solution

Decide the supported pricing-overlay API, then implement it in the Bedrock plugin or a generic model-pricing overlay with matching schema, docs, cache behavior, and tests.

## Reproduction Assessment

## Solution Assessment
