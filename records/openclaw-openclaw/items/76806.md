## Summary

Keep this PR open: the circuit-breaker direction is useful and the later commits address the prior resettable-kind concern, but latest head still misclassifies recoverable history-only overflows as irreducible and keeps user guidance that omits oversized current input.

## What This Changes

The PR adds an `irreducible_overflow` precheck route, fail-fast runner handling, non-resettable overflow metadata, auto-reply reset skips, tests, docs, and a changelog entry for context-overflow compaction loops.

## Best Possible Solution

Keep the distinct non-resettable fail-fast path only when the zero-history baseline exceeds budget, preserve normal `compact_only` recovery for history overflow, and align all user-facing guidance/tests with system prompt plus current message causes.

## Reproduction Assessment

Yes, source inspection gives a high-confidence reproduction path: current main routes over-budget zero-history baselines through `compact_only`, then the runner can spend three overflow compaction attempts and auto-reply can reset generic `context_overflow` results. I did not run a live gateway scenario in this read-only review.

## Solution Assessment

No, not yet. The baseline detector and non-resettable metadata are the right shape, but the PR must not turn recoverable history-only overflow into `irreducible_overflow`, and the guidance should mention oversized current input.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.94

Full review comments:

- **[P1] Preserve compaction for history-only overflow:** `src/agents/pi-embedded-runner/run/preemptive-compaction.ts:119-120`
  - body: When the zero-history baseline fits but session history pushes the prompt over budget, `toolResultReducibleChars` is often `0` because there are no tool-result messages. This new branch now returns `irreducible_overflow`, so ordinary long-history sessions skip compaction and surface a terminal system-prompt error even though history compaction is the recoverable path. Keep this branch as `compact_only` unless the baseline itself exceeds budget.
  - confidence: 0.95
- **[P2] Mention current input in irreducible-overflow guidance:** `src/agents/pi-embedded-runner/run.ts:1447-1449`
  - body: The route is triggered by `systemPrompt + prompt` with no history, but this payload blames only the system prompt and skills/plugins. A very large current message can also trigger this path, so users need to be told to shorten the current message as well as reduce active skills/plugins or choose a larger-context model.
  - confidence: 0.87
