## What This Changes

Add a runtime-side subagent announce dedupe that skips delivery when the requester session's latest assistant message after child completion fingerprint-matches the child findings.

## Best Possible Solution

Add a conservative, fail-open parent-assistant transcript check before announce delivery, with regression coverage for duplicate, unrelated, stale, and transcript-read-failure cases.
