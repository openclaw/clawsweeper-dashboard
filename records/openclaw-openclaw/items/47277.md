- **[P1] Honor isolation when the optional config block is absent:** `extensions/memory-core/src/isolation-identity.ts:25-28`
  - body: The schema makes `memory.isolation` optional, but this branch returns `null` when it is omitted, so `resolveIsolationIdentity` supplies no `userId`.   - confidence: 0.99
- **[P1] Scope compaction memory writes to the current user:** `packages/memory-host-sdk/src/host/internal.ts:147-163`
  - body: The patch includes root-level `memory/*.md` in each isolated user's corpus, while the existing compaction flow still writes durable notes to shared `memory/YYYY-MM-DD.md`. A note flushed from one user's session can therefore be searched or read by every isolated user; the write target must receive the same canonical tenant scope as retrieval.
  - confidence: 0.97
