  - confidence: 0.92
- **[P1] Do not share flush-written daily memory across users:** `packages/memory-host-sdk/src/host/internal.ts:262-263`
  - body: In isolated mode this still adds root-level `memory/*.md` for every user, but the current memory-flush path writes private compaction notes to root daily files like `memory/YYYY-MM-DD.md`. Those notes would be indexed for every isolated user, so either route flush writes into the scoped directory or stop treating root daily files as shared user memory.
