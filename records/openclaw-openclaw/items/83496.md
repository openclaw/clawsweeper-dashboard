## What This Changes

The issue requests an explicit Codex auth-profile selection policy for stable-primary plus backup-account setups.

## Best Possible Solution

Add a documented auth-profile selection policy in the auth-routing layer, preserving current behavior unless maintainers intentionally choose a migration path, with schema/help/docs and focused tests for primary-first plus fallback recovery.

## Solution Assessment
