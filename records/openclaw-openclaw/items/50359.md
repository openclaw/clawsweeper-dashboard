## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.78

Full review comments:

- **[P2] Gate start-arg media fallback to read:** `src/agents/pi-embedded-subscribe.handlers.tools.ts:287-288`
  - body: The new helper is named for read args, but `emitToolResultOutput` passes `startArgs` for every tool. Because local media trust applies to many built-ins, a pathless image result from another trusted tool could promote its input `path`/`filePath` as outbound media even when the result did not declare that file. Scope this fallback to the read tool, or pass the tool name into the extractor before using start args.
  - confidence: 0.78
