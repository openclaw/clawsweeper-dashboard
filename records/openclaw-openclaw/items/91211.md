  - body: This only checks an exact `agents.defaults.models[provider/model].agentRuntime` entry.   - confidence: 0.91
- **[P2] Thread model API into auth labels:** `src/gateway/server-methods/models-list-result.ts:228-233`
  - body: The label helper only receives provider and model id, so every model under a provider gets the same OAuth/API label. `models.list` availability already filters auth by `entry.api` and treats OpenAI platform vs ChatGPT Responses differently; with both OpenAI OAuth and API-key profiles, platform-only entries can be shown as `OAuth` even when they require API-key auth.
  - confidence: 0.88
- **[P2] Default the new Swift initializer parameter:** `apps/shared/OpenClawKit/Sources/OpenClawProtocol/GatewayModels.swift:4705-4706`
  - body: `ModelChoice` is public, and adding `runtimelabel` as a required argument to its public initializer is a source break for existing Swift callers that construct this type. Since the JSON field is optional, give the initializer parameter a `nil` default or add a compatibility overload.
  - confidence: 0.85
