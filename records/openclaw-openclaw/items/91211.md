  - body: The label helper only receives provider and model id, so every model under a provider can get the same OAuth/API label. `models.list` availability already filters by `entry.api` and has a separate OpenAI ChatGPT Responses fallback; pass the entry API through and reuse that compatibility logic for the label.
  - confidence: 0.88
