- **[P1] Treat denied mirror DMs as revocation:** `extensions/telegram/src/bot-message.ts:327-341`
  - body: When `buildTelegramMessageContext` returns `null` because `enforceTelegramDmAccess` denies a mirrored DM, `revoked` is still false, so this branch throws and `launchMirrorDispatch` unmarks the target.   - confidence: 0.94

  - body: These public SDK exports expose global register/unregister functions keyed only by caller-provided `channel` and `accountId`, so any plugin can replace or remove another channel/account's mirror dispatcher. The new public seam should be owner-scoped or kept internal until it can enforce that a plugin only manages its own dispatcher.
  - confidence: 0.88
