## Reproduction Assessment

## Solution Assessment

Yes in direction, pending maintainer acceptance. A narrow additive admin-scoped RPC is the right owner-boundary if maintainers approve the API and security contract; raw `config.patch` remains a workaround that leaks config shape to clients.
