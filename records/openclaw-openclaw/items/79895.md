Full review comments:

- **[P1] Keep hostname allowlists from bypassing private DNS checks:** `src/infra/net/ssrf.ts:165`
  - body: `hostnameAllowlist` is used as a target-host restriction for public and fake-IP requests, not as private-network trust. Returning true here skips the resolved-address check in `resolvePinnedHostnameWithPolicy`, so an allowlisted provider host can DNS-rebind to RFC1918, link-local, or metadata addresses without `allowPrivateNetwork`; keep that skip gated by explicit private-network policy and add a blocked-IP regression.
  - confidence: 0.89
- **[P2] Keep this branch scoped to the SSRF fix:** `extensions/active-memory/index.ts:1403`
  - body: This PR is presented as a single-file SSRF fix, but it also changes active-memory caching and doctor migration behavior. Merging it as-is would ship unrelated behavior without the review and proof for those surfaces, so those commits should be removed or handled in their own PRs.
  - confidence: 0.86
