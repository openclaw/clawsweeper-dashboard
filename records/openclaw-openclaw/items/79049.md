## What This Changes

## Best Possible Solution

Keep the narrow `t()`-based localization approach, also localize the side-panel markdown headings, regenerate Control UI i18n artifacts from current main, and add redacted browser proof showing the localized labels on the chat/tool-card surface.

## Solution Assessment
