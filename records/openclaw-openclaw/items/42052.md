## What This Changes

The PR changes backup output-path resolution so a missing extensionless `--output` path is treated as a destination directory while existing or suffixed paths remain explicit archive filenames.

## Best Possible Solution

Move the missing-extensionless-output heuristic into `src/infra/backup-create.ts`, preserve explicit suffix filenames and non-ENOENT stat errors, add focused regression coverage, and require redacted real backup-run proof before merge.
