## What This Changes

This PR snapshots streamed fallback text, clears Control UI chat stream/run state before appending final or aborted assistant messages, adds an order-sensitive controller regression test, and adds a changelog entry.

## Best Possible Solution

Land this PR, or an equivalent narrow controller patch, so active-run terminal events clear live stream/run state before committing final or aborted assistant messages while preserving streamed fallback behavior and regression coverage.
