  - confidence: 0.9

  - body: Closing the session sidebar also forces `sidebarOpen = false`, so a user who had markdown or tool-output content open loses that panel after briefly opening the session list. The session overlay should restore or decouple the previous sidebar state.
  - confidence: 0.82
