  - body: This makes light/rem phase hits count toward signal count and context diversity, so one daily or grounded signal can satisfy deep gates after phase review. Keep phase hits as scoring boosts unless maintainers explicitly choose this gate policy.
  - confidence: 0.86

  - body: Every selected REM truth writes `signalCount: 3`, and the grounded writer adds that count into `groundedCount` and `totalScore`. A barely eligible truth gets the same promotion weight as high-confidence evidence, so derive or clamp this count from REM confidence instead.
  - confidence: 0.82

  - confidence: 0.87
