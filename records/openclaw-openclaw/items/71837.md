## Summary

Keep open. The PR is useful and addresses real memory-core dreaming noise, but the current branch is not safe to merge because the REM-to-grounded path can inflate diversity with day-varying synthetic queries, gives borderline REM truths the same grounded weight as high-confidence truths, and reintroduces `system` as a positive REM signal even though current main treats it as meta vocabulary.

## What This Changes

The PR changes memory-core dreaming and memory-host session extraction to strip transcript scaffolding, denoise REM themes/truths, feed selected REM truths into grounded short-term candidates, adjust promotion scoring, and add focused tests.

## Best Possible Solution

Land a rebased version that keeps the scaffold denoise and REM-to-grounded promotion direction, but uses stable synthetic REM identity, confidence-scaled grounded signal strength, complete meta stopword handling, current-main session-ingestion structure, a changelog entry, and targeted tests before the changed gate.

## Reproduction Assessment

Yes. The PR diff is enough to reproduce the review issues: repeat the same REM candidate across two day buckets to get distinct synthetic query hashes, inspect the fixed `signalCount: 3`, and compare the PR's positive `system` tag against current-main meta filtering.

## Solution Assessment

No. The direction is plausible, but the branch is not the safest merge shape until the identity, weighting, meta-tag, changelog, and rebase issues are corrected.

- **[P2] Use a stable REM query id:** `extensions/memory-core/src/dreaming-phases.ts:1856-1862`
  - body: The REM grounding path writes candidates with a day-suffixed query key, and grounded candidate recording hashes that query into `queryHashes`.   - body: Each selected REM truth records `signalCount: 3` even though `score` is computed per entry. That gives a barely passing REM truth the same grounded promotion weight as a high-confidence one; derive the count from the candidate confidence or lower the floor for borderline candidates.
  - confidence: 0.86

  - confidence: 0.84
