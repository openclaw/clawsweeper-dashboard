- **[P2] Keep phase hits out of recall/diversity gates:** `extensions/memory-core/src/short-term-promotion.ts:402-429`
  - body: This makes light/rem phase hits count toward signal count and context diversity, so one daily recall can satisfy deep gates after phase review.   - confidence: 0.86

  - body: Every selected REM truth writes `signalCount: 3`, and the grounded writer adds that count into `groundedCount` and `totalScore`. A barely eligible truth gets the same promotion weight as high-confidence evidence, so derive or clamp this count from REM confidence instead.
  - confidence: 0.82
