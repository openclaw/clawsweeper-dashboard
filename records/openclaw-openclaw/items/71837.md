- **[P1] Keep phase hits out of promotion gates:** `extensions/memory-core/src/short-term-promotion.ts:402-426`
  - body: This adds light/rem phase hits to both `signalCount` and context diversity, then uses those values to pass `minRecallCount` and `minUniqueQueries`.   - confidence: 0.86

  - body: Every REM-selected truth is persisted with `signalCount: 3` and a date-suffixed query. `recordGroundedShortTermCandidates` stores that query hash and grounded count, so repeated REM selection over multiple days inflates diversity and signal strength regardless of candidate confidence. Use a stable REM query and scale or clamp the signal count from the calculated confidence.
  - confidence: 0.84
