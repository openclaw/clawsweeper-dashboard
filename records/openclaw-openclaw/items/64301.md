The PR adds reverse lookup in embedded-runner tool-call normalization for underscore-stripped, dot-prefixed, dropped-leading-character, and digit-bearing hex-suffixed malformed tool names.

## Best Possible Solution

Land a narrow normalizer fix that recovers only structurally signaled malformed names, fails closed on ambiguous or tail-only collisions, and adds focused regression tests plus redacted real-provider proof.

## Solution Assessment

No. Prefix plus strict hash-suffix recovery is a reasonable direction, but dropped-leading-character exact recovery is too broad for dispatch-sensitive code and should be removed or structurally gated before merge.
