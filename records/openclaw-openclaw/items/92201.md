## Label Justifications

- `P1`: Multiple users report live Anthropic embedded-runner sessions becoming permanently unusable until manual transcript repair.
- `impact:session-state`: The failure poisons persisted session history by replaying invalid Anthropic thinking blocks on every later turn.
- `impact:message-loss`: Affected Slack/Telegram turns fail before producing a usable assistant reply, and retries keep failing on the same session history.
- `impact:auth-provider`: The issue is tied to Anthropic provider replay, thinking signatures, provider error bodies, and fallback behavior.

## Summary

Keep open. Current main has the merged raw-error recovery work and signature-delta concatenation, but it still has a source-visible recovery gap for pre-content stream `start` events, and the merged recovery PR explicitly did not fix the create-side invalid-signature path reported here; recent native-Anthropic reports in https://github.com/openclaw/openclaw/issues/94228 show the remaining failure class is still active.

## What This Changes

The issue requests fixing Anthropic embedded-runner thinking-signature corruption during streaming/replay and making recovery see raw provider errors so poisoned sessions self-heal.

## Best Possible Solution

Keep the merged raw-error and signature-delta repairs, add a focused recovery fix for pre-content `start`/control events, and continue the native Anthropic create-side/replay-policy investigation with live replay proof before closing this issue.

## Reproduction Assessment

Yes, from source and field reports: current main still marks a pre-content `start` event as yielded output, so a subsequent thinking-signature error skips recovery. I did not run a fresh live Anthropic replay in this read-only pass.

## Solution Assessment

No, current main is only a partial solution: raw-error matching and signature-delta append are present, but recovery can still miss pre-content stream errors and the create-side invalid-signature source remains unproven. A safer path is a narrow start-event recovery fix plus a provider-contract review for historical native Anthropic thinking replay.
