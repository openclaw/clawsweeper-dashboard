## What This Changes

This PR updates shared MEDIA parsing to skip indented code-block lines, preserve code indentation during media cleanup, validate closing fence lines more strictly, and add parser/fence regression tests.

## Best Possible Solution

Land a narrow parser fix that treats indented-code MEDIA lines as text, reuses code-aware leading normalization after audio tag removal, preserves lowercase directive support for valid payloads, and includes focused tests plus redacted OpenClaw or Telegram proof.
