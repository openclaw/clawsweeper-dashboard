  - confidence: 0.9

A streamed code example ending with `    MEDIA:/tmp/code.png` can therefore be withheld or dropped on reset before the protected parser path gets a chance to preserve it; share the new classification with the streaming accumulator and update reply-utils coverage.
  - confidence: 0.86
