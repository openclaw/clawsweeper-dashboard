## What This Changes

The issue requests that agent sessions maintain an internal work queue or state machine so multi-step tasks continue without manual prompts after each completed sub-task.

## Best Possible Solution

Define a scoped autonomy contract, likely through Task Flow, standing orders, commitments, or an explicit session mode, and file narrower runtime bugs only when a concrete channel/config/log transcript shows an existing queue or completion path failing.
