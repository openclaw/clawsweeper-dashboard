## What This Changes

Request instruction isolation and enforceable role, memory, tool, and monitoring boundaries for model-consumed SKILL.md content.

## Best Possible Solution

Define a maintainer-approved skill trust and activation model, represent role, memory, and tool authority with typed metadata, enforce those permissions at runtime rather than through prompt wording alone, and add malicious-skill regression coverage plus clear operator documentation.
