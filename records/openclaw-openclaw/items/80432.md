The issue requests Matrix outbound auto-replies to render sibling account `@name` mentions as matrix.to pills with `m.mentions.user_ids` metadata.

## Best Possible Solution

Define one Matrix-owned sibling roster/alias contract, then feed it into outbound mention rendering with regression coverage for auto-replies and coordinate it with the inbound sibling-mention work.
