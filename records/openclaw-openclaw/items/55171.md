## What This Changes

The branch adds Feishu logic that unbinds a live conversation binding when the bound session agent differs from the resolved route agent and updates Feishu routing tests.

## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

No. Agent mismatch is not a safe stale signal because current main intentionally preserves explicit cross-agent ACP and focus bindings; cleanup should use a narrower invalid-target, dispatch-failure, or explicit rebind signal.
