Status: needs_attention

Summary: The requested feature is security-sensitive because it would expand automatic persistence and possible model summarization of private transcripts.

- **[medium] Define retention and redaction before automatic persistence:** `src/hooks/bundled/session-memory/handler.ts:273`
  - body: Expanding session-memory from explicit command saves to idle, shutdown, and periodic automatic saves can retain private or prompt-hostile conversation content without a user action; the design needs explicit opt-in/defaults, redaction, retention, and workspace trust-boundary policy.
