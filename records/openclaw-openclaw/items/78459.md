## What This Changes

## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

No. Removing forced `true` is directionally right, but the best fix should use the shared fast-mode resolver so default-off MiniMax plans and explicit configured fast-mode opt-ins both behave as documented.
