## Best Possible Solution

## Reproduction Assessment

Yes, source-level: current main passes `fastMode: true` for hidden commitment extraction, and the MiniMax wrapper rewrites `MiniMax-M2.7` to `MiniMax-M2.7-highspeed` when that flag is true. I did not run a live MiniMax OAuth account in this read-only review.

## Solution Assessment

No, not as currently shaped. Removing the hardcode is the right direction, but the PR should forward `resolveFastModeState(...).enabled` so configured opt-ins remain intact.
