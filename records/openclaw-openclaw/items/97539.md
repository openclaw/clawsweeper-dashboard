## What This Changes

The PR changes Signal container REST success and error body reads to use bounded SDK readers and adds focused tests for oversized success and error responses.

## Best Possible Solution

Merge the central bounded-read fix after contributor-supplied redacted real behavior proof or maintainer override, keeping the implementation in the Signal container REST helper and reusing the existing SDK readers.

## Solution Assessment
