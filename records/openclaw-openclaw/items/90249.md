- **[P1] Fail closed for skipped pinned Google tool choices:** `extensions/google/transport-stream.ts:295-297`
  - body: When the pinned function's declaration is filtered but another tool survives, this returns `undefined`, and the caller omits `toolConfig`; that silently turns a forced Google transport tool request into an unconstrained request where other tools may be called. Preserve the requested restriction by throwing or otherwise failing closed when a function-specific choice has no surviving declaration.
  - confidence: 0.87
- **[P2] Preserve reserved schema property names while snapshotting:** `src/llm/providers/google-shared.ts:392`
  - body: Assigning into a normal `{}` with `snapshot[key] = ...` does not preserve an own `__proto__` property, so a valid JSON Schema property named `__proto__` is dropped or treated as prototype mutation. Use a null-prototype object or `Object.defineProperty` in both Google snapshot helpers and add coverage for reserved property names.
  - confidence: 0.81
