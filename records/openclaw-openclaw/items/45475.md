Request to add a Slack `channels.slack.ignoreSubtypes` config key that can suppress selected Slack message subtype events before they wake an agent session.

## Best Possible Solution

Decide the Slack-owned suppression policy; if accepted, add a typed account/channel-scoped subtype suppression list at the event layer with schema, docs, and tests proving ignored subtypes enqueue no system events.
