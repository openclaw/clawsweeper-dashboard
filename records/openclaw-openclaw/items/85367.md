  - body: The string branch splits and truncates labels but skips the 40-character guard and de-duplication that the array branch applies.   - confidence: 0.91
- **[P2] Require a session key before marking cards running:** `ui/src/ui/controllers/workboard.ts:645-652`
  - body: `startWorkboardCard` treats a missing `sessions.create` key as nullable but still updates the card to `running`. Existing session creation code treats a missing key as an error, so this can leave a card running with no linked session and no navigable handoff target.
  - confidence: 0.9
- **[P2] Respect zero positions when creating cards:** `extensions/workboard/src/store.ts:177-180`
  - body: Using `normalizePosition(input.position, 0) || tailPosition` discards an explicit `position: 0` and moves the new card to the end of the column. Any caller importing or capturing cards with deterministic top ordering cannot persist that order.
  - confidence: 0.86
