The branch changes Control UI cron validation and payload construction so finite nonnegative `timeoutSeconds`, including `0`, is accepted and sent for agent-turn cron jobs.

## Best Possible Solution

Land the focused UI fix after aligning controller tests, keeping malformed and negative input rejected, and adding redacted real behavior proof for a cron job saved with `timeoutSeconds: 0`.

## Solution Assessment
