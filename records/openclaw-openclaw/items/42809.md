  - confidence: 0.86

  - body: The splitter appends one native table component for every Markdown table, but Feishu documents a five-table-per-card limit and the related issue shows table-limit card rejection can drop delivery. Add a count fallback or split strategy before switching card bodies to native tables.
  - confidence: 0.88
- **[P2] Extract native table cards when quoting replies:** `extensions/feishu/src/markdown-card.ts:185-186`
  - body: After this change, outgoing cards can contain tag: "table" elements, but current quoted interactive-card extraction only reads text-like elements. Replies to a table-only card can lose the table context and fall back to generic interactive-card text.
  - confidence: 0.82
