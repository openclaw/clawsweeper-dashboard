## What This Changes

Request WhatsApp `channels.whatsapp.groups` entries to match an exact current group subject/name as an alternative to the stable group JID while preserving JID and wildcard behavior.

## Best Possible Solution

Land a WhatsApp-local, default-safe implementation that keeps stable JID matching first and allows exact subject matching only under maintainer-approved guardrails, preferably through https://github.com/openclaw/openclaw/pull/91180 if it can be updated and proven.

## Solution Assessment
