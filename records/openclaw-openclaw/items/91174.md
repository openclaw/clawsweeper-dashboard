## Summary

Keep open. ## What This Changes

Support exact WhatsApp group subject/name keys in `channels.whatsapp.groups` while preserving JID-keyed and wildcard behavior.

## Best Possible Solution

Add a WhatsApp-local resolver that prefers the JID entry, then uses an exact subject match only under maintainer-approved ambiguity semantics, with wildcard behavior, docs, diagnostics, and focused tests updated together.
