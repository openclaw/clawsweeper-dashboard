## Summary

The branch adds a shared `tryChmodSync` helper and routes exec approvals, task/task-flow registry, and plugin-state chmod hardening through it with focused chmod fallback tests.

## Best Possible Solution

Land the helper after maintainers accept the fsGroup permission boundary, or split the store-sidecar behavior into a narrower follow-up if file-mode policy needs separate review.
