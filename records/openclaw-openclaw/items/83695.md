## What This Changes

The PR adds a shared `tryChmodSync` helper and routes exec approvals, task registry, task-flow registry, and plugin-state SQLite permission hardening through it for K8s fsGroup-owned paths.

## Best Possible Solution

Land the shared helper once maintainers accept the fsGroup permission boundary, preserving fail-closed behavior for inaccessible paths, other-writable directories, and private files with unsafe other bits.
