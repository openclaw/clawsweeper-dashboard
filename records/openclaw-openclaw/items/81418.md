## Summary

Keep open. The bug-fix direction is plausible and current main lacks a parent-liveness guard, but this PR has blocking review/CI issues and the required external after-fix real behavior proof is missing.

## What This Changes

The PR adds a parent-PID polling watchdog to `serveOpenClawChannelMcp` plus a new watchdog contract test file to reap orphaned stdio MCP workers.

## Best Possible Solution

Keep the parent-liveness fix in the MCP channel server, but make the shutdown timer safe, fix the lint failures, and require real orphan-process proof before landing.

## Reproduction Assessment

No, I do not have a high-confidence live reproduction from this read-only review. Source inspection shows current main only exits on stdio close/end, transport close, or delivered signals, so the abrupt-parent-death path is plausible but still needs real behavior proof.

## Solution Assessment

Unclear as written. A parent-PID watchdog is a reasonable narrow fix, but this patch should not merge until the force-exit timer is made conditional or cancelable, lint passes, and after-fix real behavior proof is added.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Cancel the watchdog force-exit after close completes:** `src/mcp/channel-server.ts:159-162`
  - body: After detecting a dead parent, the watchdog calls `shutdown()` and always schedules `process.exit(0)` two seconds later. If `close()` resolves cleanly but any other handle keeps the process alive, a successful cleanup still hard-exits the process; keep the timeout handle and clear it when shutdown completes, or only fire it while close is still pending.
  - confidence: 0.84
- **[P2] Apply repo lint rules to the watchdog block:** `src/mcp/channel-server.ts:153-160`
  - body: The PR currently fails `check-lint`: the repo requires braces for `if` bodies, so the added one-line returns here and in the copied probe test trip `curly`, and the no-process-exit suppression is reported unused. Add braces and remove the unused suppression so required lint can pass.
  - confidence: 0.95
