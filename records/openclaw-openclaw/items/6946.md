## What This Changes

Request a Telegram processing indicator that sends a visible placeholder message while a turn runs, then deletes or updates it when the reply or failure is delivered.

## Best Possible Solution

Choose the intended Telegram feedback UX, then land an opt-in Telegram plugin change with typed config/schema/help/docs, dispatch wiring that tracks a placeholder message id, deletes it after successful final delivery, and edits or replaces it on failure with focused regression coverage.
