## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.89

Full review comments:

- **[P1] Store ACP runtime state in SQLite:** `src/acp/store/store.ts:111-112`
  - body: This creates persistent ACP sessions, runs, events, checkpoints, leases, and idempotency in a JSON sidecar. OpenClaw-owned runtime state and checkpoints belong in SQLite with a clear doctor/migration owner, otherwise upgrades, backups, and recovery can drift from the canonical state store.
  - confidence: 0.94
- **[P1] Return the existing projection promise:** `src/auto-reply/reply/dispatch-acp-replay.ts:456`
  - body: `ensureProjection` calls `existing.start()`, but `start()` launches another `run()` loop every time. A second ensure for the same target can therefore run parallel projection loops and duplicate outbound sends or race checkpoints.
  - confidence: 0.91
- **[P1] Await reconnect reconciliation before worker frames:** `src/gateway/server/ws-connection/message-handler.ts:1024-1038`
  - body: The connect handler fires `handleNodeConnected` in the background, so incoming ACP worker events can hit the store while the lease is still `suspect`. That can reject valid terminal/progress frames and leave a completed node run stuck in gateway recovery.
  - confidence: 0.88
- **[P1] Gate ACP node commands explicitly:** `src/gateway/node-command-policy.ts:54-57`
  - body: Adding the ACP session/turn commands to `SYSTEM_COMMANDS` makes them part of the default desktop node allowlist. This expands a command-execution/control surface without an ACP-specific policy decision; use an explicit node invoke policy or opt-in gate instead.
  - confidence: 0.9
- **[P1] Reject cross-session run id reuse:** `src/acp/store/store.ts:532-535`
  - body: `startRun` returns any existing `store.runs[runId]` without checking that it belongs to the same session and request. If two sessions reuse a provider/request id, the second session aliases the first run and later lease/session validation can corrupt or strand delivery.
  - confidence: 0.86
