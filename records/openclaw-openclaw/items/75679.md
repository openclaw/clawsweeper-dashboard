## Summary

## What This Changes

The PR canonicalizes main-session aliases at subagent list/control requester resolution and adds tool and slash-command regression tests.

## Best Possible Solution

Normalize requester/controller scope symmetrically at registry write and read boundaries, preserve existing raw-main records, add reciprocal regression tests, and include one single-line changelog fix entry.
