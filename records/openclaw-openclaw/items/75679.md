## What This Changes

The PR canonicalizes main-session aliases in subagent list/control requester-key resolution and adds tool plus slash-command regression tests.

## Best Possible Solution

Normalize requester/controller keys symmetrically at writes and reads, or compare equivalent main-session aliases as one scope, with reciprocal coverage for canonical-stored and raw-main-stored runs plus real spawn/list proof.
