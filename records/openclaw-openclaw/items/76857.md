## Best Possible Solution

Review, validate, and land PR #76870 or an equivalent generic transport-layer fix that propagates Node/undici `autoSelectFamily` to per-request pinned dispatchers with regression coverage.

## Reproduction Assessment

I did not run a live chatgpt.com OAuth reproduction on an affected network.
