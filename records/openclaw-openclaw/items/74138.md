## What This Changes

## Best Possible Solution

Yes. A source-level reproduction is enough: a legacy Feishu package importing `openclaw/plugin-sdk/feishu` cannot resolve the missing facade/export on current main, and the user log shows the resulting missing `createScopedPairingAccess` call.

## Solution Assessment

Yes. A compatibility facade plus export/contract coverage is the narrowest maintainable fix; closing this issue before the linked PR merges would drop the regression tracker.
