## Summary

Keep open. ## What This Changes

Request configurable rate limiting, throttling, budgets, alerts, metrics, and CLI controls for OpenClaw external API calls.

## Best Possible Solution

Keep this as the canonical design tracker for a maintainer-scoped, opt-in outbound policy layer that starts with model-provider dispatch and explicitly defines storage, CLI/status, backoff, and budget semantics.
