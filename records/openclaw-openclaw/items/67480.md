## What This Changes

The PR adds a pre-redirect root-skill probe in skills discovery and a regression test for a configured root containing both `my-skill/SKILL.md` and `skills/decoy/SKILL.md`.

## Best Possible Solution

Land a rebased version that validates the configured root with current direct and grouped loadable-skill semantics before applying the repo-style nested redirect, with regression coverage and redacted real command proof.

## Solution Assessment
