Keep open. Current main has a narrower Control UI theming model with Claw, Knot, Dash, and one browser-local tweakcn import slot; it does not implement the requested preset gallery, hue generator, gradient controls, grouped editor, or per-variable override/reset studio. Add a Control UI theming system with six preset themes and a Custom Theme Studio for hue-generated palettes, light/dark mode, gradient accents, and per-variable overrides.
