## Summary

Keep open. Current main has Claw, Knot, Dash, and one browser-local tweakcn import slot, but it still lacks the requested preset gallery and built-in Theme Studio; the remaining scope needs maintainer product/design direction.

## What This Changes

Request adds six built-in Control UI theme presets plus a Custom Theme Studio with hue generation, gradient controls, per-variable overrides, local persistence, and locale strings.

## Best Possible Solution

Choose the intended Control UI theming model, then implement the selected scope consistently across theme types, CSS tokens, Appearance and Quick Settings, browser-local storage, docs, i18n, and focused UI tests.

## Reproduction Assessment

Not applicable as a runtime bug reproduction; this is a feature request. Current-main source, docs, tests, and GitHub search give high-confidence verification that the requested preset gallery and Theme Studio are absent.

## Solution Assessment

Unclear until maintainers choose product direction. A full built-in studio may be valid, but the current approved implementation deliberately chose one tweakcn import slot and treated a theme library/editor as non-goals.
