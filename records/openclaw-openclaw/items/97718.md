- **[P1] Gate reconciled retries behind the same delivery bounds:** `extensions/codex/src/app-server/native-subagent-monitor.ts:644-648`
  - body: With `codexHome` set, the default task-row reconciler can process a `pending` native-subagent row and call `processCompletion()` → `deliverPendingCompletion()` while a retry timer is already armed. Add a guard so reconciliation does not deliver while a retry is scheduled, or enforce the same cap/deadline before any delivery attempt.
