## Summary

## What This Changes

This PR adds a stale-run liveness gate to subagent orphan recovery so restart-aborted child runs past the liveness window are finalized instead of resumed, with regression tests and docs.

## Best Possible Solution

Land one canonical subagent recovery fix that applies the shared liveness policy at the right recovery/restore boundary, accounts for finalization failures, preserves fresh and explicit-timeout recovery, and includes focused tests, docs, and restart-path proof.
