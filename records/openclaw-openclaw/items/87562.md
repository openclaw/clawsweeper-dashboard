Full review comments:

- **[P1] Skip reconciliation for custom OpenRouter routes:** `extensions/openrouter/cost-reconciliation.ts:158-162`
  - body: OpenRouter provider config can preserve a custom `baseUrl`, and the existing payload wrappers skip custom routes before applying OpenRouter-specific behavior.   - confidence: 0.86

  - body: The wrapper only calls `reconcileOpenRouterUsageCost` while iterating to the done event. Current callers such as `complete()` and compaction summarization can call `stream.result()` directly, so those OpenRouter calls would keep the streamed undercount unless `result()` shares the same reconciled final-message path.
