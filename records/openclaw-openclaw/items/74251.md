Keep open. Current main improves fresh missing-Node installs by defaulting the installer to Node 24, but it still accepts existing Node 22.22.x runtimes, still documents Node 22.16+ and WSL2/npm global installs, and has no proven WSL2/V8-fatal mitigation for the reported crash.

The issue asks OpenClaw to stop crashing during the official WSL2 Ubuntu 24.04 installer path when `npm install -g openclaw@latest` runs under Node 22.22.x.
