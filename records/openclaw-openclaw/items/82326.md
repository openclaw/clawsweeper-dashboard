- **[P2] Keep Node trust separate from resolver script checks:** `src/secrets/provider-integrations.ts:105-119`
  - body: For `${node}` presets, `allowInsecurePath` may be needed because `process.execPath` can be root-owned, but this same flag also skips group/world-writable and owner checks for the plugin resolver script here.
