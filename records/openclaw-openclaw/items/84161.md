Classic Telnyx TTS calls can therefore store duplicate bot transcript entries.
  - confidence: 0.9
- **[P2] Emit one final Google transcript per utterance:** `extensions/google/realtime-voice-provider.ts:770-771`

The voice-call realtime handler persists every final assistant callback as `call.speaking`, so this can record both the last chunk and the aggregate as separate bot turns.
  - confidence: 0.88
