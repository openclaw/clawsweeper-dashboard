## Best Possible Solution

Land the focused agent-tool-side scoping behavior after normal maintainer validation while keeping the gateway RPC `config.get` contract unchanged.

## Solution Assessment

Yes. Slicing the already-redacted snapshot in the agent tool is the narrowest maintainable fix because it avoids changing the gateway RPC schema and preserves full-snapshot behavior when no path is supplied.
