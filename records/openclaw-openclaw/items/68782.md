Keep open. Current main has useful plugin-management and runtime allowlist foundations, but the requested install-time composition, `openclaw setup --channels`-style selection, strict discovery semantics, and skill/channel dependency policy are still not implemented as a complete product surface.

## What This Changes

Request adds install-time and runtime selection for plugin, skill, and channel surfaces to reduce package size, discovery work, startup overhead, and unused dependency surface.

## Best Possible Solution

Keep the existing plugin install and allowlist foundations, then design a narrow install composition/profile contract that reduces packaged and discovered surfaces without unexpectedly changing upgrade defaults.

## Solution Assessment
