The issue requests selective installation and loading for plugins, skills, and channels to reduce install size and startup overhead.

## Best Possible Solution

Define a maintainer-approved selective install and activation design that reuses the existing plugin install index, channel setup, `plugins.allow`, skill config, and doctor repair paths instead of adding a parallel strict mode or installer policy ad hoc.

The safest direction is to extend existing plugin install, channel setup, allowlist, skills config, and doctor seams deliberately.
