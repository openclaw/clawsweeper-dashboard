- **[P1] Keep null out of ToolStreamEntry.output:** `ui/src/ui/app-tool-stream.ts:521-534`

Normalize nullish formatted output back to `undefined` while still preserving empty strings.
  - confidence: 0.94

Build the synthesized message from the stored string instead.
