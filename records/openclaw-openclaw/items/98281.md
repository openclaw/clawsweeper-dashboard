Full review comments:

- **[P1] Preserve source-owned provider fields:** `src/gateway/server-methods/config.ts:448-455`
  - body: `stripBundledProviderRuntimeDefaults` is now shared by `config.set`, `config.apply`, and this new `config.patch` path, but this helper treats a source-owned empty string or empty array as if the source did not own the field. A valid authored overlay like `models.providers.openai.models: []` will be deleted on an unrelated write, and an authored empty `baseUrl` is sanitized instead of being validated. Keep the guard based on field ownership, and only strip fields that are truly absent from the source or patch intent.
