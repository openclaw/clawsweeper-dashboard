## Summary

Keep open. Current main still has the MCP loopback propagation gap: cron can preserve a mixed-case Matrix target when `currentDeliveryContext` is populated, but the loopback HTTP path still does not carry a current channel target into tool resolution, and the related fix PRs #75303 and #75493 are closed unmerged.
