## Summary

Keep open: current main still leaves the shared agent-scope helper and cron-facing tests with the agent primary model ahead of the global subagent default, while this PR is the narrow open fix for the linked regression. The PR reorders shared subagent model selection so per-agent subagent config wins first, then the global subagent default, then the agent primary model, with updated agent/cron tests and a terminal proof script.
