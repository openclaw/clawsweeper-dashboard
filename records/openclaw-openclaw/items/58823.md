## Summary

Keep this PR open for maintainer selection of the canonical fix. ## What This Changes

The branch reorders subagent model selection helpers and tests so per-agent `subagents.model` wins first, then `agents.defaults.subagents.model`, then the agent model, while also adding a proof script and unrelated formatting-only edits.
