## Summary

Keep open: current main still leaves the shared agent-scope helper, and therefore the cron path, using the agent primary before the global subagent default. This PR reorders shared subagent model selection so global subagent defaults beat agent primary models, updates cron/fallback tests, and adds a resolver proof script.

## Best Possible Solution

Land this narrow helper and cron precedence fix if maintainers accept the documented global-subagent-default-first contract; otherwise preserve the shipped agent-primary order by updating docs/tests and closing the linked issue and this PR with that product decision.

## Solution Assessment
