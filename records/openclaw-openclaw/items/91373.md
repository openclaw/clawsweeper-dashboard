## Best Possible Solution

Thread a raw-config cron-local inheritance signal and inherit defaults only for the default-primary string shorthand, with strict tests for differing string and object-style agent primaries.

## Solution Assessment

No. The PR is a plausible cron-scoped fix, but the best fix needs the narrower raw-config/default-primary guard used by the safer companion implementation, not inheritance for every string agent model.
