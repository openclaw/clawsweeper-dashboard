The focused patch appears correct and preserves the existing wrapped-response path, but the external-PR merge gate remains open because the supplied run uses a local replay rather than a live fal queue result.

## Best Possible Solution

Land the narrow completed-result parser after a redacted live fal queue run confirms that a raw model result is accepted, downloaded, and returned without regressing the wrapped-response path.
