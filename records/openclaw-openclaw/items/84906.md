Keep this PR open: the Gradium realtime STT provider is useful, but the current patch still has a merge-blocking compatibility and sensitive-data routing issue because an existing Gradium TTS key can make Gradium eligible for Voice Call streaming STT auto-selection without an explicit streaming opt-in. ## Best Possible Solution
