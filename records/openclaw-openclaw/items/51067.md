The PR adds an optional `gateway.controlUi.title` field and passes it through the gateway bootstrap response so the browser sets `document.title`, with jsdom tests for configured and unset cases.

## Best Possible Solution

Land the bootstrap-based title support with complete config metadata and docs so `gateway.controlUi.title` is accepted, discoverable in schema-driven UI surfaces, documented, and covered by focused tests.
