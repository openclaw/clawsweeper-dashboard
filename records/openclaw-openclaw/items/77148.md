## Reproduction Assessment

Yes for the narrow source-level regression: current main's default Claude CLI `resumeArgs` omit `--fork-session`, and upstream docs define that flag as the resume-time way to create a fresh session id. I did not live-reproduce the Slack transcript collision in this read-only pass.

Custom override behavior should remain operator-owned rather than migrated silently.
