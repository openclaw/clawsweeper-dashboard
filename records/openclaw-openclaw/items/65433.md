## Summary

Keep open. The PR contains useful inbound NUL hardening, but the current patch has a concrete P2 context-window regression, is currently merge-conflicted, and its Gemini CLI template direction overlaps with the still-open #67758 discussion.

## What This Changes

The PR strips NUL bytes from untrusted inbound metadata strings, changes Google Gemini CLI forward-compatible template lookup to prefer 3.1 preview IDs, and adds default 1M context handling/tests for `anthropic/claude-sonnet-4-6`.

## Best Possible Solution

Land the remaining generic NUL sanitizer gap as a narrow hardening change, rework Sonnet 4.6 1M handling behind an explicit-provider or otherwise unambiguous provider source, and resolve the Gemini CLI model-ID direction together with #67758 before merging that portion.

## Reproduction Assessment

Yes. The review path is static and high-confidence: a model-only call such as `resolveContextTokensForModel({ model: "anthropic/claude-sonnet-4-6", fallbackContextTokens: 200_000 })` would enter the PR's new resolved-ref fallback and return 1M despite current-main tests preserving model-only inferred-provider behavior.

## Solution Assessment

No. The NUL sanitizer line is the right shape, but the combined patch is not the best landing path because the Sonnet fallback needs an explicit-provider guard and the Gemini portion needs current catalog/provider validation.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.87

Full review comments:

- **[P2] Gate the Sonnet fallback on explicit provider:** `src/agents/context.ts:432-434`
  - body: The new fallback runs after `resolveProviderModelRef` even when the provider was inferred from a slash-containing model string. Model-only status/lookup paths deliberately avoid provider-specific assumptions, so a routed `anthropic/claude-sonnet-4-6` transcript can be reported as 1M instead of using its discovered or fallback window. Restrict this branch to an explicit provider, matching the Opus 4.7 guard.
  - confidence: 0.89
