  - confidence: 0.95

  - body: `reconcileOrphans` always force-cancels active runs, but this exported type lets callers pass natural terminal reasons like `exit`, `signal`, or `spawn-error`. That would record a forced orphan cleanup as a natural exit and make process diagnostics misleading; the type should only allow forced-cancellation reasons.
  - confidence: 0.78
