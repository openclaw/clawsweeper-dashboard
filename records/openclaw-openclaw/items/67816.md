- **[P1] Keep logout ownership checks on timeout:** `extensions/whatsapp/src/auth-store.ts:304-305`
  - body: On a barrier timeout this skips `shouldClearOnLogout`, and the non-legacy path then recursively deletes `resolvedAuthDir`.   - confidence: 0.91

  - confidence: 0.88
