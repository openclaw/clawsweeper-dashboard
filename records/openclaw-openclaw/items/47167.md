## Summary

Keep open: current main still lacks a public early raw-inbound Plugin SDK hook, and representative mention-gated channel paths still return before downstream plugin message hooks can observe skipped group messages.

## What This Changes

Add a passive, read-only hook or equivalent lifecycle contract so plugins can archive group messages skipped before mention-gated dispatch.

## Best Possible Solution

Define a channel-neutral, opt-in, read-only early inbound observation contract with minimized payloads, privacy documentation, and channel tests that preserve existing routing and no-reply behavior.

## Reproduction Assessment

Yes, as a source-level reproduction. The public hook catalog has no early raw inbound hook, and representative requireMention skip branches return before the shared message_received and before_dispatch plugin paths.

## Solution Assessment

Unclear for the exact API shape, but the direction is valid. The safest solution is an explicit opt-in early observation contract, not making all skipped traffic visible to existing hooks by default.
