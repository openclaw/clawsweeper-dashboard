## What This Changes

The issue requests a passive read-only inbound observation hook so archive plugins can record group messages skipped by requireMention without creating sessions or replies.

## Best Possible Solution

Keep this issue as the canonical tracker, then review or replace PR 83101 with a rebased implementation that adds an access-gated read-only Plugin SDK hook and leaves any non-Telegram rollout clearly tracked.
