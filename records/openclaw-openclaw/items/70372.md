## Summary

Keep open: current main still has the underlying unphased-provider leak path, but this PR should not merge as-is because its synthetic phase leaves unphased text extractable and no real after-fix provider/channel proof is present.

## What This Changes

The PR adds a `hadThinkingInMessage` state flag and uses it to synthesize `final_answer` delivery for unphased non-OpenAI Responses text after thinking events.

## Best Possible Solution

Attach reliable phase metadata at the Anthropic/Bedrock transport boundary or structurally suppress thinking-adjacent/tool-use prelude before channel delivery, with regression coverage and real provider/channel proof.

## Reproduction Assessment

Yes for a source-level path: Anthropic emits unphased thinking and text events, and current delivery only phase-filters explicit commentary/final-answer metadata. I did not run a live Bedrock or channel replay in this read-only review.

## Solution Assessment

No. This PR only marks the delivery path as `final_answer`; because the assistant text remains unphased, `extractAssistantVisibleText()` can still return the same narration prelude.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Classify unphased text before delivering it:** `src/agents/pi-embedded-subscribe.handlers.messages.ts:497-501`
  - body: The fallback only labels the delivery path as `final_answer`; the Anthropic text block itself remains unphased. On `text_end`, `extractAssistantVisibleText()` falls back to unphased text when no explicit final-answer block exists, so a thinking event followed by `Let me check...` style prose can still be sent to channel delivery.
  - confidence: 0.88
- **[P2] Preserve thinking state across stream item resets:** `src/agents/pi-embedded-subscribe.ts:271`
  - body: `resetAssistantMessageState()` is also called when the handler switches phase-aware stream items inside one assistant message. Clearing `hadThinkingInMessage` there means a later unphased text block in the same thinking message can bypass the synthetic phase path again.
  - confidence: 0.76
