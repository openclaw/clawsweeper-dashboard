  - body: `replyPayload` can be `undefined` when the heartbeat response tool is enabled but the model returns no usable payload.   - confidence: 0.94
- **[P1] Do not fire pre-send hooks before the final send decision:** `src/infra/outbound/deliver.ts:1639-1644`
  - body: The new internal `message:sending` notification is fire-and-forget and runs before `applyMessageSendingHook`, so async user hooks can execute after the platform send and can observe content that plugin hooks later rewrite or cancel. Emit it only after the plugin cancel/rewrite decision and await it, or document this as observation-only before adding the surface.
  - confidence: 0.88
