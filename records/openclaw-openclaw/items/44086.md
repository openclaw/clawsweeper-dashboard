Adds a Chat Completions payload wrapper and tests that convert assistant messages with `tool_calls` from `content: null` to `content: ""` before provider submission.

## Solution Assessment

Yes in shape, but not as submitted. A scoped payload wrapper is the narrow maintainable fix, but it should be ported onto the current post-plugin wrapper chain and include the required changelog entry.
