## Best Possible Solution

Land a refreshed fix that passes the existing `InputProvenance` envelope into the child `agent` call from `spawnSubagentDirect` with focused regression coverage; if https://github.com/openclaw/openclaw/pull/47087 cannot be revived, use it as source for a narrow replacement.

## Solution Assessment
