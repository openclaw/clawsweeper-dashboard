## Best Possible Solution

Keep this as the canonical maintainer decision point for the managed Gateway credential contract, then align service-env rendering, startup config-env application, model-status wording/JSON, Kimi docs, and focused systemd regression proof around that contract.

## Reproduction Assessment

## Solution Assessment

Unclear until maintainers choose the intended managed-service credential contract.
