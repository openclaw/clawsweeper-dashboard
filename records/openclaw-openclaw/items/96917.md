## What This Changes

The branch adds Anthropic OAuth loopback host validation, derives the redirect URI from the resolved host, and adds helper tests for callback host and URI formatting.

## Best Possible Solution

Keep the loopback restriction, preserve the shipped localhost default redirect unless live Anthropic proof confirms alternatives, and land with focused tests plus real OAuth-flow evidence.

## Reproduction Assessment

## Solution Assessment

No. The loopback allowlist is the right direction, but deriving the default redirect from 127.0.0.1 changes the shipped provider-facing URI without proof; the safer fix keeps localhost as the default or proves Anthropic accepts the new URI.
