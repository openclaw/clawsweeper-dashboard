The PR restricts Anthropic OAuth callback hosts to loopback values, derives the redirect URI from the resolved host, threads that URI through authorization and token exchange, and adds focused regression tests.

## Solution Assessment

Yes, with maintainer acceptance of the compatibility tradeoff. The loopback allowlist plus derived redirect URI is the narrowest in-repo repair and matches the sibling OpenAI OAuth pattern better than preserving arbitrary callback hosts.
