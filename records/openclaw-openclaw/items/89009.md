## Best Possible Solution

Land one Feishu-owned dispatcher fix that records the first committed streaming final, prevents later final-shaped diagnostics/status updates from mutating that card, skips exact duplicates, and routes distinct diagnostics separately with focused tests and visible Feishu proof.

## Solution Assessment

Yes, the Feishu dispatcher is the right owner boundary. The current main mitigation is incomplete; the best fix is a single committed-final guard that preserves the first card and sends distinct diagnostics outside that card.
