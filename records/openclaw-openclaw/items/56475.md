## Summary

Keep open. Current `main` still implements `openclaw logs --follow` as a polling loop over one-shot `logs.tail` RPCs, and `main` does not contain the additive `logs.subscribe` / `logs.unsubscribe` / `logs.appended` gateway capability described by PR #56475.
