## Summary

Keep open. Current main still lacks the requested channel-scoped hard tool-call budget and the `warn`/`block`/`auto-delegate` overflow behaviors, while related discussion keeps this as the canonical per-channel design tracker rather than a duplicate of the global budget request.

## What This Changes

Add channel-scoped tool-call budgets with warn, block, and auto-delegate overflow behavior for agent runs.

## Best Possible Solution

Define an additive scoped budget contract that composes with existing channel/group tool policy and loop detection; implement warn/block first in trusted before-tool enforcement, then add auto-delegation only through existing `sessions_spawn` guardrails.

## Reproduction Assessment

Not applicable as a bug reproduction. This is a feature/config request, and source, schema, docs, related-issue, and prior-comment inspection give high confidence that the requested channel budget and overflow behavior are absent on current main.

## Solution Assessment

Unclear. A channel-scoped hard budget is plausible, but automatic subagent creation is not the narrowest safe first step until maintainers define precedence, warning injection, block result shape, and delegation authority.
