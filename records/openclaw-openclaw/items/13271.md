## Summary

Keep open. Current main and the latest shipped release still lack a channel-scoped max-per-turn tool-call budget and the requested warn, block, or auto-delegate overflow behavior; this issue remains the canonical per-channel request rather than a duplicate of the open global budget request.

## What This Changes

This issue requests per-channel tool-call limits with warn, block, and auto-delegate overflow modes for multi-channel agents.

## Best Possible Solution

Define an additive scoped budget contract, enforce deterministic warn/block behavior in the trusted before-tool path first, then design auto-delegation around sessions_spawn guardrails and child-run handoff.
