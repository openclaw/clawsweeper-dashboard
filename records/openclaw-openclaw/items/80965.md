## Summary

Keep this PR open, but do not merge it as-is: current main still has the raw resolver bug, yet the patch breaks the promised backwards-compatibility path for WhatsApp configs whose `groups` map is populated while `groupPolicy` is omitted.

## Best Possible Solution
