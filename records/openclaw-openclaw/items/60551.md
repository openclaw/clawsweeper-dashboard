## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Gate schema-envelope stripping on leak markers:** `src/shared/text/assistant-visible-text.ts:765-766`
  - body: `CONTAM_ENVELOPE_RE` matches any fenced `json` block containing a `schema` key, and `stripStructuralContamination` removes those matches unconditionally. A normal answer showing an OpenClaw plugin manifest or config schema in fenced JSON will lose the whole example before delivery; require leak-specific metadata or a stricter envelope marker instead of every schema block.
  - confidence: 0.91
- **[P2] Narrow ordinary first-person preamble cues:** `src/shared/text/assistant-visible-text.ts:633-634`
  - body: The scorer treats normal openings like `I need to ...`, `I should ...`, and `Need to ...` as leaked-reasoning evidence. Two such opening lines plus the multi-line bonus reach the strip threshold, so instructional replies can lose their opening paragraph before send; require a stronger leak marker or remove these ordinary cues from scoring.
  - confidence: 0.88
- **[P2] Reject tilde fences as preamble-strip suffixes:** `src/shared/text/assistant-visible-text.ts:688`
  - body: `looksUserFacingStart` rejects backtick fences but treats `~~~` fences as user-facing starts. After two heuristic preamble lines, a legitimate tilde-fenced example can become the split point and lose the lead-in; reject tilde fences the same way as backtick fences.
  - confidence: 0.82
