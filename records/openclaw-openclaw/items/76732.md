Full review comments:

- **[P1] Keep fresh-image pruning state out of module scope:** `src/agents/pi-embedded-runner/run/history-image-prune.ts:176-180`
  - body: `activeTurnHasFreshImages` is shared by every run in this Node process, but it is set from one attempt's params and later read by provider replay.
