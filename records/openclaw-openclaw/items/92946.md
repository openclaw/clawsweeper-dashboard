  - body: When rich previews fall back to legacy delivery, this helper caps the actual `sendMessage`/`editMessageText` payload to 4096 characters, but a successful send later records the full requested text as delivered.   - confidence: 0.9

Long rich edits can still fail unless the PR scopes edits to short text, truncates intentionally, or switches to a follow-up send path.
  - confidence: 0.86
