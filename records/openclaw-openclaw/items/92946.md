  - body: When rich previews fall back, `capTelegramLegacyTransportPreview` sends at most the first legacy-sized chunk, but the caller records the full requested text as delivered after success.   - body: This regex arm matches `Call to 'sendRichMessage' failed! Restrict fallback to actual missing/unknown rich-method responses and add negative tests for non-method 400s.
  - confidence: 0.9
