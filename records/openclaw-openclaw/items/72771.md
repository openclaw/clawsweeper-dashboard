## What This Changes

## Best Possible Solution

If maintainers want this event, define it as an explicit public Gateway contract that preserves selected-agent ownership, includes `runId`, documents and tests the SDK/Gateway relationship, and is proven with a real client run; otherwise keep the broader status design in https://github.com/openclaw/openclaw/issues/52640.
