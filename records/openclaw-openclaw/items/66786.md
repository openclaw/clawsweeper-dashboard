The issue asks for default-agent and system-prompt workspace resolution to honor the existing `OPENCLAW_WORKSPACE_DIR` environment variable instead of home/profile fallbacks.

## Best Possible Solution

Review and land, rebase, or replace https://github.com/openclaw/openclaw/pull/81447 so the shared default workspace resolver honors `OPENCLAW_WORKSPACE_DIR` while keeping explicit workspace config precedence and regression coverage.
