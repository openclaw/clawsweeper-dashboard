## What This Changes

The PR installs console capture in the Gateway foreground fast path before command parsing, always suppresses known libsignal session-dump prefixes, and adds regression tests plus a changelog entry.

## Best Possible Solution

Land the scoped logging-order and suppression fix after required exact-head checks, keeping it limited to console capture timing and known sensitive dependency prefixes.

## Solution Assessment
