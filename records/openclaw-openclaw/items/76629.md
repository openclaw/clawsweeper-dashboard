## Summary

Keep open. Current main has partial runtime-context protection, but provider-bound replay still lacks the broader transient replay filter requested here for exact silent assistant payloads, stream-error placeholder text, and copied runtime-context assistant scaffolding.

## What This Changes

The issue asks for provider-bound replay to remove OpenClaw internal runtime/control artifacts and no-op assistant sentinels before constructing model context.

## Best Possible Solution

Add a generic replay-hygiene helper in the embedded runner that removes only OpenClaw-owned nonsemantic artifacts before provider conversion, with focused regression coverage and optional doctor repair for persisted polluted transcripts.
