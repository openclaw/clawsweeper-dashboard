Keep open. Current main has partial runtime-context and metadata-only replay hygiene, but it still lacks the requested provider-bound pruning for standalone silent assistant payloads, stream-error fallback text, and copied OpenClaw runtime-context assistant scaffolding.

## Best Possible Solution

Add a generic embedded-runner replay-hygiene helper before provider hooks/conversion, with separate doctor/repair cleanup only for already-persisted polluted transcripts if needed.
