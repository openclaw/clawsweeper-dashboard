## Reproduction Assessment

Yes. ## Solution Assessment

Mostly yes. Settling parent aborts in the embedding timeout helper is the narrow runtime fix, but the named public MemorySearchOptions export should either be accepted with the generated API baseline updated or removed to avoid unnecessary SDK expansion.
