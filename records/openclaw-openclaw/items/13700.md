## What This Changes

Request adds first-class session snapshot commands/API to save, load, restore, fork, and reuse conversation state checkpoints.

## Best Possible Solution

Define one native session snapshot contract over session storage, compaction checkpoints, reset/deleted archives, and subagent fork primitives, then expose agreed save/list/load/branch/restore semantics through commands, protocol, UI/CLI, docs, and tests.
