## Best Possible Solution

Define one native session snapshot design that composes SessionManager transcript lineage, compaction checkpoints, reset/deleted archive discovery, Gateway protocol, slash commands, Control UI, docs, and focused tests.

Not applicable as a bug reproduction; this is a feature request. Source, docs, protocol, UI, tests, latest-release inspection, and live issue context give high-confidence evidence that the requested named save/load snapshot workflow is absent.

## Solution Assessment
