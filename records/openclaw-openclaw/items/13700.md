## Summary

## What This Changes

Request to add native session snapshots so users can explicitly save, load, fork, and restore conversation context checkpoints.

## Best Possible Solution

Define one native session snapshot design that composes SessionManager transcript lineage, compaction checkpoints, reset/deleted archive discovery, Gateway protocol, slash commands, Control UI, docs, and focused tests.
