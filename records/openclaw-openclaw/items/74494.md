Full review comments:

- **[P2] Preserve Lit-owned mobile menu state:** `ui/src/ui/app-render.helpers.ts:508`
  - body: Current main keeps the mobile settings menu open state in `chatMobileControlsOpen` so ARIA, Escape/outside dismissal, tab changes, and focus restoration stay synchronized. This close button removes `.open` directly, so the overlay can visually close while Lit state still says it is open.
  - confidence: 0.92
- **[P2] Keep mobile refresh disabled during active runs:** `ui/src/ui/app-render.helpers.ts:559`
  - body: The desktop refresh button is disabled while chat is sending, has an active run, or is streaming. The new mobile refresh button only checks loading/disconnected, so it can trigger a manual refresh during an active run and reintroduce the race current main avoids.
  - confidence: 0.86
- **[P2] Scope expanded-card scrolling to mobile:** `ui/src/styles/chat/tool-cards.css:21-23`
  - body: This selector is global, so desktop expanded tool cards that currently grow with `max-height: none` and `overflow: visible` become capped nested scrollers. Scope the height/overflow containment to the mobile overflow case or another mobile-only selector.
  - confidence: 0.9
- **[P2] Do not cap mobile page zoom:** `ui/index.html:5`
  - body: Adding `maximum-scale=1.0` to the global viewport meta can prevent pinch zoom throughout the Control UI on mobile browsers and webviews. Keep the safe-area/keyboard viewport behavior if needed, but remove the zoom restriction.
  - confidence: 0.84
- **[P3] Localize the new settings copy:** `ui/src/ui/app-render.helpers.ts:499-528`
  - body: The overlay adds visible English-only strings such as the settings title, close label, and select labels without adding Control UI locale keys. Route the new text through the existing locale source so non-English users do not get mixed-language chat controls.
  - confidence: 0.78

  - body: This is a user-facing Control UI bug fix, but the patch does not include `CHANGELOG.md`. Add a single-line entry under the active `### Fixes` section before merge.
  - confidence: 0.86
