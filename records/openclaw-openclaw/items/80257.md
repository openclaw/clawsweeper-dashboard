## Best Possible Solution

Resolve the effective doctor env before snapshotting, snapshot exactly the config path the update doctor will mutate, keep the update-time strip/write guards, fix the type-gate failure, and require redacted packaged upgrade proof preserving config and credentials.

## Solution Assessment
