The branch changes update-time doctor config preservation by adding unknown-key strip guards, `plugins.installs` protection, pre-update snapshots, size-drop guard changes, and focused tests.

## Best Possible Solution

Land a corrected preservation fix that uses the exact config path doctor will mutate, keeps bounded or refreshed recovery snapshots, fixes the typecheck failure, and includes redacted real upgrade proof.

  - body: Package-update doctor receives env from `resolvePostInstallDoctorEnv`, which can replace `OPENCLAW_CONFIG_PATH` from the managed service profile.   - confidence: 0.88
