## Summary

Keep open. Current `main` still permits host-path `read` access outside an agent workspace unless `tools.fs.workspaceOnly=true` or sandboxing is enabled, and the repo’s tests/docs explicitly preserve that behavior. This is not implemented on `main`, not stale, and not safe to auto-close.
