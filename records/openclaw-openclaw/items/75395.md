Request that OpenClaw strip trailing assistant prefill turns for Anthropic Claude models routed through OpenRouter’s OpenAI-compatible completions adapter when reasoning/thinking is enabled.

## Best Possible Solution

Add an OpenRouter-owned payload patch that detects Anthropic model references on the OpenAI-compatible route and strips trailing non-tool assistant turns only when reasoning/thinking is enabled, reusing the shared Anthropic sanitizer and adding focused tests plus a user-facing changelog entry.
