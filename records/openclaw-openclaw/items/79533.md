## Summary

Keep this PR open: it targets a real native sub-agent skill prompt gap, but the current patch has a blocking read-before-initialization error and needs focused regression coverage before merge.

## What This Changes

The PR imports workspace skill helpers and appends the resolved target-agent workspace skills prompt to the native sub-agent system prompt.

## Best Possible Solution

Move skill prompt resolution after normalized spawned metadata is available, keep using the existing target-agent filtered skill loader, and add sub-agent prompt regression tests.
