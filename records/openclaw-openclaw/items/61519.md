  - body: The new PR-triggered job uses `actions/checkout@v6` and `actions/upload-artifact@v7`, while current workflows pin these Actions to immutable SHAs. Floating major tags let upstream changes alter code executed in CI without a repo review.
  - confidence: 0.95
- **[P1] Escape report labels before rendering summaries:** `scripts/depcruise-circular-report.mjs:212`
  - body: Repository-derived boundary and directory names are written directly into Mermaid and Markdown tables, but the script only escapes `|`. A PR can add crafted path names that break or spoof the GitHub Actions step summary unless labels are escaped for Markdown and Mermaid or rendered as plain text.
  - confidence: 0.88
- **[P2] Bound dependency-cruiser output before parsing:** `scripts/depcruise-circular-report.mjs:300-311`
  - body: The PR-triggered job lets contributors change the analyzed graph, but the script buffers up to 64 MB of dependency-cruiser stdout and then parses the whole JSON string in memory. A graph-heavy PR can make this report job reliably fail or consume excessive memory; stream to a file and enforce a size limit before parsing.
  - confidence: 0.86
