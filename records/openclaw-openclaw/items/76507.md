The issue reports a v2026.5.2 regression where the browser plugin tool is missing from agent-visible tools under `tools.profile: "full"` despite browser being enabled and healthy.

A narrow regression fix should make the declared non-optional browser plugin tool model-visible under `tools.profile: "full"` whenever plugin and browser config allow it, while preserving the documented `coding` profile exclusion.
