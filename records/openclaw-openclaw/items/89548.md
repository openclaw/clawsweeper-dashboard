The PR changes agent tool mutation classification so simple read-only `exec`/`bash` commands are non-mutating while ambiguous or mutating shell commands remain mutating.

## Best Possible Solution

Land the narrow shared classifier once normal CI/reviewer gates are satisfied, keeping ambiguous shell commands fail-closed and preserving claimed-success warnings for mutating exec failures.

## Solution Assessment
