## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Reject gh short web flags before marking commands read-only:** `src/agents/tool-mutation.ts:249`
  - body: `gh` documents `-w` as an alias for `--web` on view/list/search commands, but this helper only rejects the exact `--web` token. That means `gh pr view 123 -w` or `gh search prs bug -w` pass as non-mutating even though they can open a browser, which can suppress the final tool warning and avoid replay side-effect invalidation. Reject `-w` and add tests alongside the `--web` cases.
  - confidence: 0.91
- **[P2] Reject attached git pager flags before allowing git grep:** `src/agents/tool-mutation.ts:230`
  - body: `git grep` accepts `-O, --open-files-in-pager[=<pager>]`; this check catches exact `-O` and long forms but not attached short forms like `-Ovim`. Those still pass as read-only because `grep` is an allowed git subcommand, even though they can launch a pager/editor. Treat `-O*` as unsafe and cover it.
  - confidence: 0.84
