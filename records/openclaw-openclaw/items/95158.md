## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

No as submitted. Using the macOS product version for Darwin runtime metadata is the right direction, but the implementation should cache or reuse the existing cached `sw_vers` lookup and add focused helper coverage before merge.
