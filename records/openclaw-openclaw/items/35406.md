## What This Changes

Request that the generic `read` tool avoid returning raw binary document bytes for PDF/Office files, either by warning or by extracting readable text.

## Solution Assessment

Unclear for the final implementation. A lightweight fallback is plausible, but parser-backed extraction should coordinate with the existing document-extractor contract and the open PRs before this is considered solved.
