Add path-scoped read, write, and execute permissions enforced consistently across file and exec tools.

## Best Possible Solution

Keep this as the canonical design thread and define one security-owner-approved policy contract that composes with existing exec approvals and sandbox backends, clearly distinguishing enforceable OS containment from best-effort tool checks before accepting a replacement PR.
