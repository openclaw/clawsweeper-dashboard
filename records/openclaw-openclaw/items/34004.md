Keep open: current main still has Discord presence/status updates only, while the requested default-off self-profile action for bot nickname/avatar/status with self-only and multi-account guardrails is not implemented. Add a default-off Discord self-profile message action for bot-only nickname, avatar, status, and activity updates with self-only, multi-account, and avatar-validation guardrails.
