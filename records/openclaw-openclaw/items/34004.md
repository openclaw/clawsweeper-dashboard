Keep open: current main still lacks the requested Discord `self-profile` action and `selfProfile` gate, and the related implementation PRs are closed and unmerged. ## What This Changes

Request a Discord-only `self-profile` message action for bot self-only nickname, avatar, status, and activity updates behind a default-off `channels.discord.actions.selfProfile` gate.

## Best Possible Solution

Maintainers should decide whether this belongs in the bundled Discord plugin; if accepted, implement a plugin-local default-off action with strict self-only checks, avatar validation, explicit account routing, docs, focused tests, and live Discord proof.
