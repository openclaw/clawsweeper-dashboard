  - confidence: 0.94

  - body: This inserts `applySafetyPosture` into the old inline load chain, but current main now materializes config through `materializeRuntimeConfig`. The branch needs to rebase and hook the preset into the current materialization path so load, missing-config, and snapshot modes cannot drift.
  - confidence: 0.86
