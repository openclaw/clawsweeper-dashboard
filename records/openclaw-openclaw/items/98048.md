Wraps MiniMax music-generation SSE frame JSON parsing in a try/catch to skip malformed data frames and adds a regression test for a malformed frame followed by valid audio.

## Best Possible Solution

Keep the provider-local parser guard and regression coverage, then merge after redacted MiniMax/OpenClaw runtime proof is added or a maintainer explicitly overrides the proof gate.

## Reproduction Assessment

## Solution Assessment

Yes: the failure is provider-local per-frame SSE parsing, while the shared JSON helper covers whole-response JSON and would not address this loop without a broader new abstraction.
