## Best Possible Solution

Add a read-only on-demand Telegram plugin message action that returns the current chat-level pinned message as a zero-or-one result, without automatic session injection or a new config surface.

## Solution Assessment

Yes. The narrow maintainable path is an on-demand Telegram plugin action; automatic session-start injection would add avoidable prompt/state policy and configuration pressure.
