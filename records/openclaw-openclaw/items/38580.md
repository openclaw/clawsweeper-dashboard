Request Telegram support for reading the current pinned chat message through the bot API on demand, with automatic session-start context left as a possible later behavior.

Add a Telegram plugin-owned on-demand `list-pins` action backed by `getChat`, return zero or one current pinned message, gate it with `channels.telegram.actions.pins`, document it, and keep automatic session-start context injection for a separate product decision.
