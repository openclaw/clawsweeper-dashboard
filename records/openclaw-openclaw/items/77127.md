## What This Changes

The PR adds optional `append?: boolean` support to the shared agent write tool, host/sandbox write operations, public write input/event contracts, prompt metadata, append recovery behavior, and focused regression tests.

## Best Possible Solution

Land the append-capable write contract only after maintainer API acceptance, preserving the overwrite default, workspace/sandbox guards, same-file mutation serialization, append recovery behavior, and the existing memory-flush append-only wrapper.
