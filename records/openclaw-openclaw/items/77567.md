## Summary

Keep open: this is a broad new product capability, not an obsolete cleanup item. Current main has isolated-agent routing, heartbeat, commitments, and task primitives, but it still lacks the requested family-agent supervision queue, supervisor routing contract, daily digest, escalation states, and `openclaw agents queue` CLI surface.

## What This Changes

Requests a first-class family-assistant mode that captures isolated-agent requests/errors into a structured queue and surfaces them to a nominated supervisor during syncs.

## Best Possible Solution
