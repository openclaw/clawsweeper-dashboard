## Summary

Keep open: current main has adjacent isolated-agent, heartbeat, commitments, task-ledger, system-event, and plugin primitives, but it does not implement the requested first-class family-agent supervision queue, supervisor routing contract, morning-sync digest, escalation model, or queue CLI.

## What This Changes

Add a first-class family-agent supervision mode with structured operating queues, supervisor routing, daily digest delivery, escalation states, and CLI inspection/resolution commands.

## Best Possible Solution

Define a reviewed supervision contract that composes with `agents.list[]`, durable queue storage, heartbeat or scheduled digest delivery, plugin/runtime hooks where appropriate, CLI/docs, and explicit cross-agent privacy boundaries.
