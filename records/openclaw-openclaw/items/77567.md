## Summary

## What This Changes

The issue requests a first-class supervised family-agent mode with agent-local queueing, supervisor routing, morning digests, escalation levels, and CLI inspection/resolution commands.

## Best Possible Solution

Define a generic supervised agent-state or digest contract only if maintainers accept it as core; otherwise route the capability through a plugin or API-extension design instead of hard-coding a family-specific mode.
