  - body: Config validation and tests accept `https://controlplane.tailscale.com` with `serve` or `funnel`, but runtime still throws for any non-empty `controlUrl`. Use the same official-control-server distinction here so explicitly spelling the official control plane does not fail at startup.
  - confidence: 0.93
