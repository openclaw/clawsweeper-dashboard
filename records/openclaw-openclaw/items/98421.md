- **[P2] Key poll-vote echo suppression to the resolved chat:** `src/agents/tools/message-tool.ts:1536`
  - body: `recordPollVoteForEchoGuard` records `pollVoteEchoTargetKey(params)`, but that helper only reads `target`/`to`/`channelId` while iMessage poll-vote accepts `chatGuid`/`chatIdentifier`/`chatId` and resolves the real chat later.
