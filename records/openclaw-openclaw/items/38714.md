Request to expose Discord reaction add/remove events through the plugin Hooks system for reaction-driven automation.

## Best Possible Solution

Keep this open for maintainer API review, choose a default-passive observation hook contract, then implement Discord reaction add/remove emission with docs and focused hook/Discord tests.

## Solution Assessment
