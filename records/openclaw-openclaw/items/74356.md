## Best Possible Solution

## Solution Assessment

Yes, the requested direction is the narrow maintainable fix, but it is not implemented on `main`. Replacing the remote shell installer with a maintainer-approved pinned or verified Bun source is safer than leaving a build-time network script execution path.
