## Summary

Keep open. Current main and the latest release still do not make unmentioned Telegram group images available to a later mention; main now skips those media downloads before history recording, and the skipped-history path that remains is text-only.

## What This Changes

Make Telegram group history preserve media from unmentioned group messages so a later bot mention can review recent images.

## Best Possible Solution

Have a Telegram area reviewer approve the unmentioned-media retention policy, then wire Telegram into the existing HistoryEntry.media/recordWithMedia or preflight-media contract with bounded local image retention, stale-file filtering, path-safe prompt behavior, and live Telegram proof.

## Reproduction Assessment

Yes, at source level. Current main skips unmentioned requireMention Telegram group media before download, and the skipped-history branch records text-only entries without HistoryEntry.media.

## Solution Assessment

Unclear until Telegram retention policy is decided. The old mediaPaths/temp-file touch approach is not the best shape now that main has a generic bounded history-media contract, but Telegram still needs owner approval before using it for unmentioned group media.
