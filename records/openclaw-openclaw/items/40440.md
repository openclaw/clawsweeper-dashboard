## Summary

Keep open: current main and the latest release still do not preserve Telegram unmentioned-group media for a later mentioned turn; current main actually skips ordinary unaddressed group media before download, while the later history-recording path remains text-only.

## What This Changes

Preserve media attachments from unmentioned Telegram group messages in recent chat history so a later @mention can inspect them.

## Best Possible Solution

Use the existing `HistoryEntry.media` and `recordWithMedia` plumbing for Telegram only after maintainers approve bounded retention, redaction, and safe-default behavior for unmentioned group media.

## Reproduction Assessment

Yes at source level: current Telegram group media can return from `shouldSkipMediaDownloadForUnaddressedMentionGroup` before download, and the later no-mention history record path is text-only. I did not run a live Telegram group repro in this read-only review.

## Solution Assessment

Unclear until maintainers decide the privacy and retention policy. The existing shared history-media API is the right technical seam, but Telegram should not start retaining unmentioned group media without explicit bounded behavior and tests.
