## Summary

Keep open: current main and the latest shipped release still drop Telegram media from unmentioned group-message history, and the related fix attempts were closed unmerged.

## What This Changes

Preserve media paths and types for unmentioned Telegram group messages so later mentions can access recent history media.

## Best Possible Solution

Implement bounded Telegram history-media preservation that carries surviving media refs through HistoryEntry/InboundHistory and into later mentioned-turn MediaPaths without adding config or weakening cleanup.

## Reproduction Assessment

Yes, at source level. The no-mention group path receives downloaded allMedia but records only text history, and later mentioned-turn context builds MediaPaths from current and reply media only.

## Solution Assessment

Yes, the right fix direction is to thread mediaPaths/mediaTypes through the history contract and replay only surviving refs. A safe implementation should bound retention and avoid stale file refs or sticker MediaPath regressions.
