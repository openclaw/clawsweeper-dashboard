  - body: Current main records result.meta.agentMeta.contextTokens before falling back to model lookup. This branch only calls resolveContextTokensForModel, so providers that report the actual context window can lose that value from sessions.json and status when lookup is unresolved.

  - confidence: 0.9
- **[P2] Keep session cost as a snapshot:** `src/agents/command/session-store.ts:133-135`
  - body: Current main assigns runEstimatedCostUsd directly because the usage object is already cumulative for the run. Adding it to entry.estimatedCostUsd reintroduces over-counting when multiple persist paths write the same run.
  - confidence: 0.92

  - body: Current main persists agentMeta.compactionTokensAfter when provider usage is unavailable. This branch never reads that field and only preserves an older total as stale, so a fresh compaction-only update can lose the token snapshot used by status.
  - confidence: 0.88
