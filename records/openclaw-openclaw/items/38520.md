## Summary

Keep open. Current main has adjacent compaction start/end notices, before/after hooks, silent memory flush, checkpoints, and context-pressure checks, but it still does not provide configurable pre-compaction warning thresholds, an agent-visible structured handoff turn, or a bounded audited deferral contract.

## What This Changes

The issue requests pre-compaction warning events, a structured handoff turn before compaction, and a bounded compaction deferral mechanism.

## Best Possible Solution

Define a single compaction lifecycle contract that reuses existing memory flush, hooks, notifier, checkpoint, and context-engine seams while adding explicit threshold events, handoff semantics, and audited bounded deferral.
