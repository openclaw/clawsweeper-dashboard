## Best Possible Solution

Define one maintainer-approved compaction lifecycle contract that reuses existing memoryFlush, notifyUser, hooks, and post-compaction context where possible, then add only narrow opt-in warning, handoff, or deferral surfaces with explicit policy and audit behavior.
