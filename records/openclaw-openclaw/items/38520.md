## Summary

Keep open. Current main has adjacent compaction start/end notices, silent memory flush, context-pressure checks, and before/after hooks, but it still lacks configurable pre-compaction warning thresholds, an agent-visible structured handoff turn, and a bounded audited deferral contract.

The issue requests configurable pre-compaction warning thresholds, a structured agent handoff turn before compaction, and a bounded audited deferral mechanism.

## Best Possible Solution

Define one compaction lifecycle contract that reuses existing memory flush, hooks, notifier, checkpoint, and context-pressure seams while adding explicit threshold events, handoff semantics, and audited bounded deferral.
