## Summary

Keep open. Current main has adjacent compaction notices, hooks, and silent memory flush behavior, but it still lacks configurable pre-compaction warning thresholds, a structured agent handoff turn, and a bounded audited deferral contract.

## What This Changes

The issue requests configurable pre-compaction warning events, a structured handoff turn before compaction, and a bounded audited deferral mechanism.

## Best Possible Solution

Define a single first-class compaction lifecycle contract that reuses the existing memory flush, hook, notifier, context-engine, and checkpoint seams while adding explicit threshold events, handoff semantics, and audited bounded deferral.

## Solution Assessment
