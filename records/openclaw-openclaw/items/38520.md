## Risks / Open Questions

- Implementing the proposal literally would create a second handoff configuration and execution path alongside memoryFlush.
- An agent-requested deferral can delay context-budget enforcement; maximum duration, request frequency, audit behavior, timeout handling, and behavior across legacy and plugin context engines need explicit decisions.
- Threshold warnings based on estimated session counters may not precisely match provider-owned or server-side compaction, so the notification contract must define which compaction authority and token measurement it represents.
