The remaining work is a product/API design item rather than conservative cleanup.

## What This Changes

The issue requests configurable pre-compaction warning events, a structured pre-compaction handoff turn, and a bounded deferral mechanism for long stateful agent workflows.

## Best Possible Solution

Define one versioned compaction lifecycle contract that adds staged agent warnings, one bounded handoff opportunity, and audited deferral while preserving existing memoryFlush, hooks, checkpoints, transcript rotation, post-compaction sections, and user-notice behavior.
