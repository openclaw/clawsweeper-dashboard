- **[P2] Avoid false offline failures when saving language:** `apps/android/app/src/main/java/ai/openclaw/app/NodeRuntime.kt:640`
  - body: `setLanguageOverride()` saves a local preference and then calls `refreshGatewayConnection()` even when there is no cached endpoint.   - confidence: 0.9

After rebasing, the override needs to be applied to that active engine or local fallback TTS can keep using the system/directive locale.
  - confidence: 0.84
