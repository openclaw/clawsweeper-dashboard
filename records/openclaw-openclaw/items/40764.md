## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Avoid failing offline language changes:** `apps/android/app/src/main/java/ai/openclaw/app/NodeRuntime.kt:640`
  - body: `setLanguageOverride()` triggers `refreshGatewayConnection()` after changing a local setting, but current main's refresh path reports `Failed: no cached gateway endpoint` when there is no cached endpoint. Changing the language while offline or before the first connection can therefore surface a misleading failure even though no reconnect is possible.
  - confidence: 0.9
- **[P2] Apply the override to the active TTS path:** `apps/android/app/src/main/java/ai/openclaw/app/voice/TalkModeManager.kt:1198-1201`
  - body: The branch updates `ensureSystemTts()`, but current main's local fallback speech goes through `speakWithSystemTts()` and `ensureTextToSpeech()`. After rebase, the selected locale needs to be applied to that active TTS flow or fallback TTS will still ignore the new setting.
  - confidence: 0.84
