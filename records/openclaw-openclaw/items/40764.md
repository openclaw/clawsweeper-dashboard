## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Avoid false offline failures when saving language:** `apps/android/app/src/main/java/ai/openclaw/app/NodeRuntime.kt:640`
  - body: `setLanguageOverride()` saves a local preference and then always calls `refreshGatewayConnection()`. When no endpoint is cached, that method reports `Failed: no cached gateway endpoint`, so changing a local setting while offline can show a failure even though the save succeeded.
  - confidence: 0.9
- **[P2] Wire the override into the active fallback TTS path:** `apps/android/app/src/main/java/ai/openclaw/app/voice/TalkModeManager.kt:1201-1218`
  - body: The branch applies the override to the older `systemTts`/`ensureSystemTts()` path, but current main speaks fallback replies through `speakWithSystemTts()` and `ensureTextToSpeech()`. After rebase, the override must feed that active engine or local fallback replies can keep using the system/directive locale.
  - confidence: 0.86
