## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Avoid offline failure states on language changes:** `apps/android/app/src/main/java/ai/openclaw/app/NodeRuntime.kt:640`
  - body: `setLanguageOverride()` saves a local preference and then unconditionally calls `refreshGatewayConnection()`. On this branch, that refresh path reports `Failed: no cached gateway endpoint` when no endpoint has been connected yet, so changing a settings dropdown while offline can surface a misleading failure for a successful preference update.
  - confidence: 0.9
- **[P2] Apply the override on the current TTS path:** `apps/android/app/src/main/java/ai/openclaw/app/voice/TalkModeManager.kt:1201-1218`
  - body: This applies the selected locale to the older `systemTts` / `ensureSystemTts()` path, but current main now runs fallback speech through `speakWithSystemTts()` and `ensureTextToSpeech()`. After rebasing, the override needs to reach that active `TextToSpeech` engine or local fallback TTS can keep ignoring the new setting.
  - confidence: 0.84
