Rebase the Feishu localization onto current main, keep the single-dispatch `[System: ...]` permission-notice contract, preserve `loadWebMedia(..., { localRoots })` for docx local uploads, and add focused regression coverage for localized copy plus workspace-only sandbox behavior.
