Fix idle Gateway CPU saturation when `plugins.bundledDiscovery` is `compat` by avoiding repeated bundled plugin manifest inventory scans.

## Best Possible Solution

Land the narrow snapshot `inventoryFingerprint` handoff after maintainer validation, preserving explicit snapshot lifecycle invalidation instead of adding a broad persistent discovery cache.
