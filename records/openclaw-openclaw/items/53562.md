## Summary

Keep open. Current main and the latest release still lack per-autoJoin `sessionChannelId` routing and text mirroring; the existing `voice.agentSession.target` is only a global shared-session route, and the remaining Discord voice bridge behavior needs maintainer product direction.

## Best Possible Solution
