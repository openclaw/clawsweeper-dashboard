## Summary

Keep open. Current main still has the hardcoded 20000 overflow hint, and the PR direction is useful, but the latest patch still sizes the new hint only from catalog-resolved primary model metadata and can fall back to the old 20000 advice even when the session already has a 200k context snapshot.

## What This Changes

The PR replaces the static context-overflow `reserveTokensFloor` hint with tiered context-window-aware values and adds an Unreleased changelog entry for plain numeric output.

## Best Possible Solution

Keep the context-window-aware hint direction if maintainers accept the tiers, but compute it from the same session/default fallback context source used for runtime accounting and add focused assertions for the tier boundaries and fallback path.

## Reproduction Assessment

Yes. A source-level reproduction is `buildContextOverflowRecoveryText()` on the PR head with empty model metadata and an active session carrying `contextTokens: 200000`; the new resolver-only path leaves `primaryContextWindow` undefined and emits the 20000 default instead of the 50k tier.

## Solution Assessment

No. The force-push fixed parseable numeric output and changelog coverage, but the mergeable solution should seed the hint from `activeSessionEntry.contextTokens` or an explicit fallback before dropping to the 20k default.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Use session context when sizing the reset hint:** `src/auto-reply/reply/agent-runner-execution.ts:651-657`
  - body: The new hint resolves only the configured primary provider/model and passes no fallback. When the model is not in the synchronous context cache, even though the active session already stores `contextTokens` or OpenClaw would use the 200k default, `primaryContextWindow` is undefined and the helper still recommends `20000`, leaving the reported large-context path unfixed. Use the active session/default context fallback, or reuse the existing context-window helper, before defaulting to 20k.
  - confidence: 0.86
