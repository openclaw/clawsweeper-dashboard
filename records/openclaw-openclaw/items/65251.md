## Best Possible Solution

Keep this as a maintainer design item for a scoped LSP detection and lifecycle API with explicit install authority, documented overrides, diagnostics/status events, and clear plugin or bundle ownership for language defaults.

## Solution Assessment

No, a direct core auto-installer should not be assumed as the best fix. The maintainable path needs product/API/security design for detection, install permission, overrides, diagnostics, and plugin-boundary ownership before implementation.
