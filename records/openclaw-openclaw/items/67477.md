Overall confidence: 0.88

Full review comments:

- **[P1] Report cancellations to the dispatcher:** `extensions/whatsapp/src/auto-reply/deliver-reply.ts:93`
  - body: When `message_sending` returns `cancel: true`, `deliverWebReply` returns without telling `dispatchWhatsAppBufferedReply` that delivery was skipped. The caller then treats the awaited call as sent and records echo/history text for a reply a plugin intentionally cancelled.
  - confidence: 0.91
- **[P2] Return hook-rewritten content to the dispatcher:** `extensions/whatsapp/src/auto-reply/deliver-reply.ts:96`
  - body: Hook rewrites only update the local `replyResult` copy. `dispatchWhatsAppBufferedReply` still records the original payload text after delivery, so self-echo suppression/history can track text that was not actually sent.
  - confidence: 0.88
- **[P2] Emit success only after an accepted provider send:** `extensions/whatsapp/src/auto-reply/deliver-reply.ts:309`
  - body: This emits successful sent hooks after the media/text section even when the effective payload has no sendable text or media, such as a hook rewrite to an empty string. That reports `message_sent` success for a WhatsApp message that never reached the provider.
  - confidence: 0.9
- **[P2] Keep sending hook failures in the failure path:** `extensions/whatsapp/src/auto-reply/deliver-reply.ts:77`
  - body: `runMessageSending` executes before the delivery `try` block, so a plugin hook exception exits before delivery and before failed sent-hook emission. The generic outbound path catches hook failures in the same delivery accounting path; this new path should preserve that contract.
  - confidence: 0.84
- **[P2] Update SDK contract artifacts or avoid the export:** `src/plugin-sdk/hook-runtime.ts:3`
  - body: Adding `getGlobalHookRunner` to `openclaw/plugin-sdk/hook-runtime` changes a public SDK subpath, but the branch does not update the aligned docs/baseline contract artifacts required for SDK surface expansion. Either avoid the new export or include the contract follow-through.
  - confidence: 0.76
