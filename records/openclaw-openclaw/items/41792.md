## Best Possible Solution

Land only a maintainer-approved, documented, classified backup SDK contract with real plugin import proof, or replace it with a narrower helper if full plan access is unnecessary.

## Reproduction Assessment

## Solution Assessment

No, not as-is. The direction may be useful, but the maintainable solution needs maintainer API/security approval, SDK docs/API metadata and baseline alignment, a clean rebase, and real plugin import proof; a narrower helper may be safer than exposing the full plan resolver.
