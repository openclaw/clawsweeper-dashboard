## What This Changes

The PR adds an `openclaw/plugin-sdk/backup` package export and facade that re-exports backup archive/plan helpers and backup-related types for plugin use.

## Best Possible Solution

Publish a documented, metadata-backed backup SDK facade only after maintainers approve the trusted-plugin data scope; otherwise expose a narrower helper or keep the behavior plugin-local.

## Reproduction Assessment

## Solution Assessment
