- **[P2] Preserve configured local model idle timeouts:** `src/agents/pi-embedded-runner/run/llm-idle-timeout.ts:102-111`

  - confidence: 0.9

  - body: This branch forces think: false whenever model.reasoning is not exactly true, so explicit params.think or params.thinking on a configured native Ollama model can no longer force thinking as documented, and non-off runtime levels on models without discovered metadata are silently downgraded.
  - confidence: 0.84
- **[P2] Wire agent model compat overrides into resolution:** `src/config/zod-schema.agent-defaults.ts:74-76`
  - body: Adding compat to agents.defaults.models validation makes that config parse, but runtime resolution currently only reads .params from those entries and gets compat from models.providers.*.models[] or discovered metadata. This new override is silently ignored unless it is propagated into the resolved model compat.
  - confidence: 0.87
