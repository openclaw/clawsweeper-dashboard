## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Keep native think wrapping scoped to Ollama transport:** `extensions/ollama/src/stream.ts:405-406`
  - body: The non-reasoning branch runs even when `model.api` is not native `ollama`: for OpenAI-compatible Ollama models `runtimeThinkValue` is `undefined`, so `undefined !== false` wraps the payload with native `think:false`. Current main intentionally gates runtime thinking on `isNativeOllamaTransport`; this would send a native `/api/chat` field into `/v1/chat/completions` payloads for custom Ollama OpenAI-compatible providers.
  - confidence: 0.91
- **[P2] Preserve the Ollama Cloud watchdog exception:** `src/agents/pi-embedded-runner/run/llm-idle-timeout.ts:109-111`
  - body: Moving the local-baseUrl return to the top bypasses current main's `!isOllamaCloudModel` check. A `glm-5.1:cloud` or other `*:cloud` request routed through local Ollama would now return `0` instead of the default idle watchdog, dropping the remote-service guard covered by the existing test.
  - confidence: 0.88
- **[P2] Wire the agent-default compat override into model resolution:** `src/config/zod-schema.agent-defaults.ts:76`
  - body: The schema/type additions accept `agents.defaults.models.*.compat`, but model resolution still only reads `params` from those entries and assigns `compat` from provider model metadata. As written, the advertised static override path is parsed and then ignored.
  - confidence: 0.86
