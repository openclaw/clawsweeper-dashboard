  - body: Matrix passwords are opaque credentials. This callback trims the configured password and then submits the trimmed value in the m.login.password auth dict, so a valid password with leading or trailing whitespace would be changed before authentication.
  - confidence: 0.92

  - body: Synapse has stabilized the cross-signing reset UIA flow as m.oauth and deprecated org.matrix.cross_signing_reset. This helper only reads params.org.matrix.cross_signing_reset.url and the router only matches that legacy stage, so a stable m.oauth challenge can still lose the approval URL.
  - confidence: 0.9

  - body: This test defines bootstrapCrossSigning as a zero-argument mock and then reads mock.calls[0]?.[0], which is unsafe for test type checking. Give the mock the bootstrap-options signature or reuse a typed helper for call extraction.
  - confidence: 0.94
