## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.94

Full review comments:

- **[P2] Preserve the submitted Matrix password:** `extensions/matrix/src/matrix/sdk/crypto-bootstrap.ts:217-228`
  - body: The callback stores `params.password?.trim()` and sends that trimmed value in the `m.login.password` auth dict. Matrix passwords are opaque credentials; current main only trims to test configuredness while sending the original value, so accounts with intentional leading or trailing whitespace would fail UIA after this change.
  - confidence: 0.92
- **[P2] Handle stable MAS OAuth UIA:** `extensions/matrix/src/matrix/sdk/crypto-bootstrap.ts:232-237`
  - body: The reset branch only recognizes `org.matrix.cross_signing_reset`, but the Matrix spec uses `m.oauth` for OAuth-authenticated cross-signing UIA and Synapse advertises the reset URL under both params keys. Treat `m.oauth` as the same operator reset flow so stable-only servers do not fall through to unsupported-stage handling.
  - confidence: 0.9
- **[P2] Remove or use the mismatch helper:** `extensions/matrix/src/matrix/sdk/crypto-bootstrap.ts:68`
  - body: `isCrossSigningKeyMismatchError` is declared but never called. Delete it or wire it into the intended fallback path so lint does not fail on the unused function.
  - confidence: 0.99
- **[P2] Type the bootstrap mock before reading its first argument:** `extensions/matrix/src/matrix/sdk/crypto-bootstrap.test.ts:607`
  - body: This test defines `bootstrapCrossSigning` as `vi.fn(async () => {})`, so TypeScript infers a zero-argument mock and `bootstrapCrossSigning.mock.calls[0]?.[0]` fails test type checking. Give the mock an explicit bootstrap-options signature or reuse the typed helper.
  - confidence: 0.97
- **[P3] Document MAS reset recovery:** `extensions/matrix/src/matrix/sdk/crypto-bootstrap.ts:232-248`
  - body: The new reset-stage path changes the operator recovery workflow, but the Matrix docs still describe only the old no-auth, dummy, and password UIA sequence. Update the Matrix channel docs to explain opening the MAS reset URL, approving the reset, and rerunning bootstrap inside the approval window.
  - confidence: 0.82
