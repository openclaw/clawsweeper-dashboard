  - confidence: 0.9
- **[P2] Submit Matrix passwords unchanged:** `extensions/matrix/src/matrix/sdk/crypto-bootstrap.ts:218-230`
  - body: Matrix passwords are opaque credentials. The existing code only trimmed to decide whether a password was configured but submitted `params.password`; this patch submits the trimmed value, which breaks any account whose valid password includes leading or trailing whitespace.
  - confidence: 0.88

  - body: The new test reads `bootstrapCrossSigning.mock.calls[0]?.[0]`, but the mock is inferred as a zero-argument tuple, so the PR's test-type lane fails with TS2493. Type the mock or test harness so the asserted options argument is part of the call signature.
  - confidence: 0.9
