## What This Changes

The PR adds optional `skillKey` fields to skill/session snapshots, writes the resolved key for newly built workspace snapshots, uses it for snapshot env override lookup, and adds focused regression tests.

## Best Possible Solution

Land one canonical fix branch that writes `skillKey` for new snapshots, refreshes or backfills persisted snapshots that predate the field, keeps the prompt format version stable, and includes redacted runtime proof.
