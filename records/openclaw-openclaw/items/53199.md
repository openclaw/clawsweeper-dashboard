Full review comments:

- **[P2] Return Slack search hits in the results envelope:** `extensions/slack/src/action-runtime.ts:494`
  - body: The CLI search formatter reads rows from `payload.results.messages`, but this branch returns top-level `messages` and `total`. Successful Slack searches will skip the search-results table and fall back to generic output unless users request raw JSON.
  - confidence: 0.86
- **[P2] Report or paginate channel-list truncation:** `extensions/slack/src/actions.ts:283-284`
  - body: The new channel-list helper stops at the default 500-channel cap but returns only `channels`. Large workspaces will silently drop valid conversations, so callers cannot tell discovery was incomplete or request the next page.
  - confidence: 0.82
- **[P2] Scope search discovery to the invokable account:** `extensions/slack/src/message-actions.ts:62-63`
  - body: Discovery advertises `search` when any enabled account has both a user token and search enabled, but invocation without `accountId` resolves the default account. In multi-account setups, the action can be advertised even though the default call will fail with the user-token error.
  - confidence: 0.75
