## What This Changes

The issue requests that Google Gemini CLI OAuth complete successfully from onboarding/model auth on macOS instead of aborting during the localhost callback/token exchange flow.

## Best Possible Solution

Update the Google plugin’s Gemini CLI OAuth flow to handle the macOS callback failure with an upstream-compatible loopback/manual fallback, add focused regression coverage, and verify with a real macOS OAuth run when credentials are available.

## Reproduction Assessment

## Solution Assessment
