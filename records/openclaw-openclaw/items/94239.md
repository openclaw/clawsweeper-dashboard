## Best Possible Solution

Merge the helper-based denominator alignment after the PR body accurately describes archive semantics and real `openclaw memory status` proof covers representative session artifacts.

## Reproduction Assessment

## Solution Assessment

Yes for the code shape: sharing the indexer's existing filename predicate is the narrow maintainable fix, while hand-coded exclusions would drift and excluding reset/deleted archives would conflict with current tests. It is not merge-ready until the PR body is corrected and real CLI behavior proof is supplied.
