## What This Changes

## Best Possible Solution

Land this predicate-sharing fix after redacted real CLI proof shows the status denominator matching the indexer on representative session artifacts, and keep https://github.com/openclaw/openclaw/issues/77338 open until this or an equivalent fix reaches main.

## Solution Assessment
