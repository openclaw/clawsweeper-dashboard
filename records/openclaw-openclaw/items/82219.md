## Best Possible Solution

If maintainers accept the wider first-party Codex marketplace boundary, land a shared typed allowlist with runtime, schema, migration, tests, and public docs kept in sync.

Yes. Current main has a source-level reproduction path: `openai-bundled` or `openai-primary-runtime` fails the manifest/schema boundary and is dropped before runtime policy resolution, though I did not execute the gateway in this read-only review.
