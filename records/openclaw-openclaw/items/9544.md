## Summary

## What This Changes

The issue requests replacing Baileys `useMultiFileAuthState` in the bundled WhatsApp plugin with a production SQL-backed auth store for credentials and signal keys.

## Best Possible Solution

Keep this issue open until the WhatsApp plugin has a maintainer-approved database-backed auth store with safe migration, security review, docs, and focused fresh-install and upgrade coverage.
