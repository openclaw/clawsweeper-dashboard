The PR adds an `openclaw-managed` non-secret auth marker, sanitizes generated `models.json` provider `apiKey` values, and adds model config plus registry regression tests.

## Best Possible Solution

Land one canonical, rebased Layer 1 fix that removes source-managed plaintext from generated `models.json`, preserves current plugin catalog and agent registry behavior, and keeps models.json-only credentials unless maintainers choose a stricter migration.

## Solution Assessment
