## What This Changes

Request a Discord per-parent-channel thread `requireMention` override so parent channels can require mentions while their threads can allow natural unmentioned replies.

## Best Possible Solution

Review, land, or replace https://github.com/openclaw/openclaw/pull/76058 with an additive Discord-owned thread mention policy that preserves existing defaults and coordinates with the separate autoThread follow-up gate.
