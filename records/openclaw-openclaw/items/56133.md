## What This Changes

The PR changes session, mirrored, and chat-created transcript header helpers to derive `cwd` from the agent workspace and adds focused gateway regression coverage.

Land a current-main-compatible fix that threads an explicit resolved agent workspace through every transcript header creation helper and removes `process.cwd()` fallbacks, with regression coverage for sessions.create, mirrored appends, chat-created transcripts, and abort-created transcripts.

No, not as submitted. The direction is narrow and maintainable, but the patch must require or derive agent workspace context before every create-if-missing transcript header write.
