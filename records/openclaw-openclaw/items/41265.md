## Best Possible Solution

Land a rebased current-main fix that preserves detected MIME types across all Discord upload paths, preserves WebP/AVIF/APNG and animated media without weakening fetch or size caps, records the SDK surface change, and includes focused tests plus redacted Discord proof.
