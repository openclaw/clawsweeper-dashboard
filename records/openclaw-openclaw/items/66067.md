- **[P2] Normalize CR line endings before replacing newlines:** `src/tui/tui-formatters.ts:250`
  - body: This helper only replaces `\n`, but `sanitizeRenderableText()` currently preserves `\r`. Normalize `\r\n?` to `\n` before this replacement and add the CRLF regression suggested in review.
