  - body: `generateConversationLabel()` resolves the agent default model, but dashboard sessions can carry their own `model`/provider selection. Use the existing derived-title contract for the UI symptom, or introduce a maintainer-approved generated-title field/policy instead of silently taking over the user label slot.
