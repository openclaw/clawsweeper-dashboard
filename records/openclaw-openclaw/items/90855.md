## Summary

Keep this PR open, but not merge-ready: the pending-final replay direction is plausible, while the heartbeat auto-lightweight heuristic currently ignores the session token freshness contract and can change existing heartbeat behavior from stale data.

## What This Changes

## Best Possible Solution
