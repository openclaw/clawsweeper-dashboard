## Summary

Keep open: current main still lacks gateway-status SSRF URL validation, and this PR is not mergeable as-is because the proposed guard has a mapped-IPv6 metadata bypass, conflicts with documented gateway hostnames, and only provides test output rather than real behavior proof.

## What This Changes

The PR adds hostname checks in gateway-status WebSocket URL normalization plus a new security test file for metadata, wildcard, blocked-hostname, localhost, private-LAN, and public-host cases.

## Best Possible Solution

Implement gateway-specific URL validation that reuses the shared IP-literal classifier for metadata, link-local, wildcard, and malformed IPs while preserving legitimate Bonjour, LAN, and tailnet gateway hostnames.

## Reproduction Assessment

Yes. Source inspection shows resolveTargets() accepts normalized ws(s) explicit/config URLs and probe-run passes target.url to probeGateway(); a focused Node URL check shows mapped-IPv6 metadata URLs canonicalize to a hostname that bypasses the PR's string checks.

## Solution Assessment

No. The PR targets the right gateway-status surface, but the current implementation should be replaced with a gateway-specific policy built on the shared IP classifier rather than broad hostname denylist reuse.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.92

- **[P1] Reject mapped IPv6 metadata addresses:** `src/commands/gateway-status/helpers.ts:103-106`
  - body: The guard only blocks 0.0.0.0 and hostnames starting with 169.254. Node normalizes ws://[::ffff:169.254.169.254]:80 to [::ffff:a9fe:a9fe], so the same metadata endpoint is still accepted.
  - confidence: 0.94
- **[P2] Preserve documented gateway hostnames:** `src/commands/gateway-status/helpers.ts:110-112`
  - body: Calling isBlockedHostname() rejects .local and .internal, but OpenClaw documents Bonjour .local hosts and wide-area domains such as openclaw.internal for gateway endpoints, so this can drop legitimate explicit or configured gateway-status probes.
  - confidence: 0.86
