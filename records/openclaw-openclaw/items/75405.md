Keep open. Current main still skips runtime conversation binding lookup for top-level Telegram groups, and open PR #75558 explicitly targets this regression with closing syntax, so the issue should stay open until that PR or an equivalent fix lands.

## Best Possible Solution

Review and land PR #75558, or an equivalent narrow patch, so Telegram native commands consult top-level group conversation bindings while preserving current DM and forum-topic behavior with regression coverage.
