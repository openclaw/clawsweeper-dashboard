## Best Possible Solution

Keep this as a consolidation tracker or split it into maintained narrow trackers: mark discovery, watcher, and #75983 diagnostics as solved; route pure ClawHub publish/service bugs to ClawHub; and define OpenClaw API/core work for direct install scanning, permission/provenance manifests, lifecycle hooks, and trigger/lint schema.
