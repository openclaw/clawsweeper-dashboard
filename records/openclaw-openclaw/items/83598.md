Keep the core bug open, define the supported no-prompt Claude CLI refresh policy, add diagnostics for skipped bootstrap decisions, and implement a narrow auth-provider fix with macOS Keychain-only and Linux file-credential regression proof; leave proactive keepalive/canary behavior to ClawHub.
