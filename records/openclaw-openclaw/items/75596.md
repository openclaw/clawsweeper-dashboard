## Summary

Close as ClawHub/plugin-owned. The request is for WeChat-specific outbound streaming behavior in the external `@tencent-weixin/openclaw-weixin` channel plugin; OpenClaw core already provides generic channel plugin, outbound, and block-streaming seams, and no missing core SDK API is identified.

## What This Changes

Request WeChat/openclaw-weixin to send partial AI responses during generation instead of one completed message after the model finishes.

## Best Possible Solution

Track WeChat real-time/preview behavior in the external `@tencent-weixin/openclaw-weixin` plugin or its ClawHub listing, using the existing OpenClaw channel outbound and block-streaming contracts; only open a narrow OpenClaw core issue if plugin implementation proves a specific missing SDK hook.
