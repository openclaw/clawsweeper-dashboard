## What This Changes

The issue asks for the Control UI WebChat code-block Copy button to copy only the fenced code block content instead of the whole assistant message.

## Best Possible Solution

Keep the issue open and consolidate the linked Control UI fixes into one canonical change that copies rendered code-block text only, prevents parent/message-level copy actions from winning the click, and adds focused regression tests for multiple blocks, special characters, inner button targets, propagation, and the bubble action spacing contract.
