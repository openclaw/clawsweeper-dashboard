## What This Changes

The PR changes Xiaomi TTS auth to Bearer, adds fallback from the Xiaomi model provider base URL when no speech base URL is set, updates Xiaomi/TTS docs and tests, and adds a changelog entry.

## Best Possible Solution

Rebase onto current Xiaomi TTS code, make endpoint and credential resolution consume the supported `xiaomi-token-plan` path while preserving explicit speech overrides, remove the release-owned changelog edit, and add redacted live proof for that final path.
