## Summary

This should close as a duplicate of the already-open macOS config writer work. The report adds useful 2026.4.27/dashboard confirmation, but #64973 already tracks the same `config.set`/token mismatch/`gateway.auth` loss failure, and #74890 tracks the broader dashboard/macOS direct-write clobber path that remains unfixed on current main.

## What This Changes

Prevent macOS app/dashboard config writes from dropping `gateway.auth` or unrelated config fields and from causing unnecessary Gateway restarts during Talk/UI settings changes.

## Best Possible Solution

Keep #64973 and #74890 as the canonical repair threads, and fix the macOS/dashboard config writer there by moving these settings writes to conflict-aware partial mutation or guarded merge behavior that preserves `gateway.auth` and unknown fields before any full-file replacement.
