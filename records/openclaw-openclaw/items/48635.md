  - confidence: 0.94

  - confidence: 0.89

  - body: The proposed config docs say `tokens` shows token counts while `full` adds estimated cost, model, and thinking. Passing model and thinking unconditionally means `/usage tokens` also shows the extra metadata, breaking the documented mode contract.
  - confidence: 0.88

  - body: The existing footer labels cost as estimated, and the PR's own example still uses `est`. Dropping that prefix makes a local estimate read like an exact charge.
  - confidence: 0.78
