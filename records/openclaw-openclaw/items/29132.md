## What This Changes

The issue requests a new synchronous `tool_result_before_model` plugin hook that transforms tool results after execution and before same-turn model visibility, distinct from `tool_result_persist`.

## Best Possible Solution

Keep this as the canonical synchronous pre-model tool-result hook design thread until maintainers either approve and land a public contract through https://github.com/openclaw/openclaw/pull/64907 or document that pre-model result rewriting remains bundled-only.
