## Summary

Keep open: current main and the latest release still lack a public pre-model tool-result sanitization hook, while the existing awaited rewrite seam remains bundled-only. ## What This Changes

Request to add a synchronous `tool_result_before_model` Plugin SDK hook that transforms tool results after execution and before any model step can consume them.

## Best Possible Solution

Keep this as the canonical sync/pre-model design thread, then land a maintainer-approved Plugin SDK contract or document that model-visible result rewriting remains bundled-only.
