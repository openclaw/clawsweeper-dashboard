## What This Changes

This issue requests a new synchronous `tool_result_before_model` plugin hook that can transform tool results before any model step consumes them.

## Best Possible Solution

Keep this as the canonical synchronous pre-model tool-result hook design thread until maintainers either land an approved public contract through https://github.com/openclaw/openclaw/pull/64907 or document that pre-model result rewriting remains bundled-only, with async behavior aligned through https://github.com/openclaw/openclaw/issues/62123.
