## What This Changes

The issue requests a synchronous public plugin hook that can sanitize tool results after tool execution and before model consumption.

## Best Possible Solution

Keep this as the canonical synchronous pre-model tool-result hook design thread until maintainers either land an approved public contract through https://github.com/openclaw/openclaw/pull/64907 or document that the rewrite boundary remains bundled-only, with async behavior tracked separately.
