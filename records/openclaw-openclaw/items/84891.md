## What This Changes

The issue reports that Control UI hides older external-channel sessions for non-main agents after restart or upgrade because the agent-scoped session picker only loads recent rows.

## Best Possible Solution

Keep https://github.com/openclaw/openclaw/issues/84537 and https://github.com/openclaw/openclaw/pull/84683 as the canonical fix path, while preserving this issue’s external-channel multi-agent evidence there.
