## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.89

Full review comments:

- **[P2] Flush trajectories even when cleanup throws:** `src/agents/cli-runner/execute.ts:818-825`
  - body: The cleanup awaits run before the trajectory fallback and `flush`. If `claudeSkillsPlugin.cleanup()`, `systemPromptFile.cleanup()`, or `cleanupImages()` throws, execution leaves this `finally` before flushing the recorder, so a sidecar can still miss the terminal write this PR is meant to guarantee. Put cleanup inside its own `try` with trajectory terminal/flush in a nested `finally`, or otherwise make recorder flush impossible to bypass.
  - confidence: 0.91
