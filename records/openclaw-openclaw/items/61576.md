- **[P1] Advertise the current gateway protocol:** `apps/linux-gtk/crates/gateway-client/src/protocol.rs:218`
  - body: Current main requires gateway protocol 4, but this new Rust client sends protocol 3 as both min and max during `connect`, so the gateway closes the app with `protocol mismatch` before any UI workflow can run.
  - confidence: 0.96
- **[P1] Use current config RPC payloads:** `apps/linux-gtk/crates/app/src/views/config.rs:222-225`
  - body: `config.get` accepts `{}` and `config.set` requires `{ raw, baseHash? }`; sending `path`/`value` here makes load/save fail validation, leaving the Config view unable to persist edits.
  - confidence: 0.92

  - body: This button only mutates process environment variables and explicitly does not rebuild or persist the active `GatewayClient`, so the success message tells users settings are saved even though they are lost on restart and may not affect the running connection.
  - confidence: 0.9
- **[P2] Send valid skills RPC payloads:** `apps/linux-gtk/crates/app/src/views/skills.rs:123-127`
  - body: `skills.update` requires `skillKey` and `skills.install` requires an install id, ClawHub slug, or upload source; sending only `skillId` makes these visible skill enable/install controls fail validation.
  - confidence: 0.88
- **[P2] Wrap cron updates in a patch object:** `apps/linux-gtk/crates/app/src/views/cron.rs:327-331`
  - body: `cron.update` requires the changed fields under `patch`; sending top-level `enabled` means the UI flips its chip optimistically while the gateway rejects the update.
  - confidence: 0.88
- **[P2] Call channel status with supported parameters:** `apps/linux-gtk/crates/app/src/views/channels.rs:126-128`
  - body: `channels.status` accepts `probe`, `timeoutMs`, and `channel`; `channelId` is an additional property, so the Probe button returns invalid-request errors instead of checking channel health.
  - confidence: 0.86
- **[P2] Read pending node pair requests from `pending`:** `apps/linux-gtk/crates/app/src/views/instances.rs:96-100`

  - confidence: 0.85
