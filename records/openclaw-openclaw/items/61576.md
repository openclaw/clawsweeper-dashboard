- **[P1] Advertise protocol 4 for gateway connections:** `apps/linux-gtk/crates/gateway-client/src/protocol.rs:218`

  - confidence: 0.96
- **[P1] Send raw config text to config.set:** `apps/linux-gtk/crates/app/src/views/config.rs:222-225`
  - body: `config.set` no longer accepts `{ path, value }`; the current schema requires `{ raw, baseHash? }`. As written, the Save button will always fail validation instead of persisting the edited config.
  - confidence: 0.95
- **[P1] Persist settings before claiming they are saved:** `apps/linux-gtk/crates/app/src/views/settings.rs:219-236`
  - body: The settings flow only mutates process environment variables, so the running `GatewayClient` keeps its captured config and the values are lost after restart. Users can be told settings were saved while the endpoint/token change never becomes durable.
  - confidence: 0.92
- **[P2] Use current skill RPC parameter names:** `apps/linux-gtk/crates/app/src/views/skills.rs:123-163`
  - body: `skills.update` requires `skillKey`, and `skills.install` requires a supported install shape such as `{ name, installId }` or ClawHub source fields. Sending `skillId` makes both toggle and install actions fail request validation.
  - confidence: 0.91
- **[P2] Wrap cron update fields in patch:** `apps/linux-gtk/crates/app/src/views/cron.rs:327-330`
  - body: `cron.update` expects mutable fields under `patch`, but this sends `enabled` at the top level. The optimistic chip update can therefore diverge from the gateway because the backend rejects the toggle.
  - confidence: 0.91
- **[P2] Send supported channel probe params:** `apps/linux-gtk/crates/app/src/views/channels.rs:126-128`
  - body: `channels.status` accepts `probe`, `timeoutMs`, and `channel`, not `channelId`. The Probe button will get an invalid-request response instead of checking the selected channel.
  - confidence: 0.89

  - body: The gateway advertises `gateway.restart.request` and `gateway.restart.preflight`; there is no `gateway.restart` method. The primary Restart button will always fail with an unknown-method error.
  - confidence: 0.9
