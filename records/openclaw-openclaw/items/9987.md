## Summary

Keep this open. Current main has backend pending exec approval listing support, but the requested user-facing `openclaw approvals pending` command, CLI coverage, and approvals CLI docs are still missing.

## What This Changes

Request to add an `openclaw approvals pending` CLI subcommand with table, JSON, and optional watch output for currently pending exec approvals.

## Best Possible Solution

Add `openclaw approvals pending` in `src/cli/exec-approvals-cli.ts` using existing `exec.approval.list`, with table output, `--json`, focused CLI tests, docs, and changelog coverage.

## Reproduction Assessment

## Solution Assessment
