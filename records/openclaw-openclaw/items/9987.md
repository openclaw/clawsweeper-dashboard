## Summary

Keep open: current main still has the backend pending-approval RPC, but the requested `openclaw approvals pending` CLI command, CLI tests, and approvals CLI docs are still missing.

## What This Changes

Request adds an `openclaw approvals pending` command to list pending exec approval requests, with table, JSON, and watch-oriented scripting support.

## Best Possible Solution

Add a focused one-shot `openclaw approvals pending` backed by `exec.approval.list`, with table output, `--json`, docs, changelog, and CLI regression coverage; leave `--watch` as a follow-up unless maintainers explicitly scope it in.

## Solution Assessment
