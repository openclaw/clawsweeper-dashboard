## Summary

Keep open: current main still has the backend `exec.approval.list` RPC, but it does not expose the requested `openclaw approvals pending` CLI command, CLI coverage, or approvals CLI documentation; the direct implementation PRs found in the discussion are closed unmerged.

## What This Changes

Request adds `openclaw approvals pending` to list live pending exec approval requests with command details, table output, JSON output, and optional watch behavior.

## Best Possible Solution

Add a focused one-shot `openclaw approvals pending` backed by `exec.approval.list`, with table output, `--json`, docs, changelog, and CLI regression coverage; leave `--watch` as a follow-up unless maintainers scope it in.

## Reproduction Assessment

## Solution Assessment
