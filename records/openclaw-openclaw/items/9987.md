## Summary

Keep open: current main still exposes the backend `exec.approval.list` RPC, but it does not expose the requested `openclaw approvals pending` CLI command, CLI coverage, or approvals CLI documentation; the direct implementation PRs in the discussion are closed unmerged.

## What This Changes

Add an `openclaw approvals pending` CLI command to list active exec approval requests with command details, JSON output, and optional watch mode.

Add a focused one-shot `openclaw approvals pending` backed by `exec.approval.list`, with table output, `--json`, docs, changelog, and CLI regression coverage; leave `--watch` as a follow-up unless maintainers explicitly scope it in.
