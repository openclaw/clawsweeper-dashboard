## Summary

Keep open: current main still has the backend and SDK path for pending exec approvals, but it does not expose the requested `openclaw approvals pending` CLI command or matching CLI docs/tests; the two related implementation PRs were closed unmerged.

## What This Changes

Add a CLI command such as `openclaw approvals pending` to list pending exec approval requests with command details, including table, JSON, and possibly watch output.

## Best Possible Solution

Add `openclaw approvals pending` in the existing approvals CLI using `exec.approval.list`, with table output, `--json`, focused tests, docs, and a changelog entry; defer `--watch` unless maintainers explicitly include it.

## Solution Assessment
