## Summary

Keep open. Current main and the latest release still provide the backend pending-approval listing API, SDK/MCP surfaces, and tests, but not the requested `openclaw approvals pending` CLI command, CLI tests, or CLI docs; the closed implementation PRs and broader Web dashboard request do not supersede this narrower CLI feature.

## What This Changes

Add an `openclaw approvals pending` CLI command with table and JSON output for currently pending exec approval requests.

## Best Possible Solution

Add a focused one-shot `openclaw approvals pending` command backed by `exec.approval.list`, with table and JSON output, CLI tests, and public docs; defer live watch behavior until maintainers explicitly scope it.

## Reproduction Assessment

Not applicable as a bug reproduction; this is a feature request. Source and release inspection show the backend listing RPC exists while the requested CLI/docs/tests surface is still absent on current main and `v2026.5.12`.

## Solution Assessment

Partly: the requested CLI is the right user-facing shape, but the first implementation should reuse `exec.approval.list` in the existing approvals CLI and leave `--watch` out unless maintainers define live-update behavior.
