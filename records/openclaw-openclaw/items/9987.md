## What This Changes

Request adds an `openclaw approvals pending` CLI command to list pending exec approval requests with command details, JSON output, and optional watch behavior.

## Best Possible Solution

Land a reviewed implementation, likely through https://github.com/openclaw/openclaw/pull/83440 or a successor, that adds one-shot pending/list output backed by `exec.approval.list`, focused CLI tests, and public docs; keep resolve/watch behavior explicit maintainer scope.

## Reproduction Assessment

Not applicable as a bug reproduction; this is a feature request. Source and release inspection show the backend RPC exists while the requested CLI/docs/tests surface is absent from current main and `v2026.5.19`.

## Solution Assessment

Partly: a one-shot `openclaw approvals pending` command using the existing `exec.approval.list` RPC is the narrow maintainable path. Resolve and watch behavior need explicit maintainer/security scope before they become part of the first public CLI surface.
