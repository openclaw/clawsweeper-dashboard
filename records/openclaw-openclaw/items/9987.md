## Summary

Keep open. Current main still has the backend pending approval list RPC, but not the requested `openclaw approvals pending` CLI surface, CLI coverage, or docs; the related implementation PRs were closed unmerged.

## What This Changes

Request a CLI command to list pending exec approval requests, with command details and scriptable output for headless monitoring/debugging.

## Best Possible Solution

Add a focused one-shot `openclaw approvals pending` table/JSON command backed by `exec.approval.list`; keep `--watch` and the broader Web dashboard as separately scoped follow-ups.

## Reproduction Assessment

Not applicable as a bug reproduction; this is a feature request. Source and release inspection give high confidence that backend listing exists while the requested CLI/docs/tests surface is still absent.

## Solution Assessment

Yes for the narrow first pass: reuse the existing `exec.approval.list` RPC from `src/cli/exec-approvals-cli.ts` instead of adding another protocol method or pending store. Defer `--watch` unless maintainers explicitly include live updates in scope.
