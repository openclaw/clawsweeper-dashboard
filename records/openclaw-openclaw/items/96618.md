## Summary

Keep this PR open because the hardening goal is valid and current main still has unbounded success-body reads in several touched areas, but the branch is not merge-ready: it duplicates the shared JSON parser path, conflicts with current main, overlaps merged Copilot fixes, lowers a current-main embeddings cap, leaves a Meet create-space read unbounded, and has only test/mock proof.

## Best Possible Solution
