## Summary

Keep open: the config-validation idea is still useful, but this PR is not mergeable as-is because it mixes an unrelated TTS behavior regression with stale Slack changes and needs rebasing onto current plugin metadata boundaries.

## What This Changes

The PR adds a config-validation helper for known plugin IDs, parameterized plugin validation tests, a Slack Bolt `ignoreSelf: false` change, and TTS directive parsing/test changes.

## Best Possible Solution

Ship a focused config-validation patch against current metadata snapshot contracts, preserve inline TTS directive parsing, and omit the Slack change already present on main.

## Reproduction Assessment

Yes for the blocking regression: `parseTtsDirectives("Hello [[tts:provider=edge]] world", policyWithAllowProvider)` is parsed by current main but ignored by the PR head's newline-bounded regex. The config-validation false-positive is also source-reproducible from `ensureKnownIds` using only manifest registry IDs when bundled manifests are unavailable, though I did not run tests under the read-only review constraint.

## Solution Assessment

No. Centralizing known plugin ID resolution is a reasonable narrow fix, but this branch is not the best vehicle because it mixes unrelated TTS behavior changes and stale Slack hunks and targets outdated metadata exports.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Preserve inline TTS directives:** `src/tts/directives.ts:78`
  - body: The new `directiveRegex` only matches `[[tts:...]]` markers on standalone lines. Current main and contract tests support inline directives such as `Hello [[tts:provider=edge]] world`; with this patch those overrides are left in the reply and ignored, so tagged TTS can silently use the wrong provider or voice.
  - confidence: 0.92
- **[P2] Rebase known-id helper onto current metadata:** `src/config/validation.ts:4-8`

Rebase this onto the current plugin metadata snapshot boundary before merging.
  - confidence: 0.86
