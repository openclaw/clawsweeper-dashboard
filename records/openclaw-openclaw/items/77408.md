## Best Possible Solution

Land a narrow Control UI CSS fix that gives the composer and mobile controls a defensive bottom clearance floor, adds responsive/PWA regression coverage, and changes service-worker caching only if validation proves stale cached HTML/CSS blocks the layout fix.
