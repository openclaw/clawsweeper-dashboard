## Summary

Keep open: current main has generic reset/delete lifecycle hooks and pre-compaction memory flush, but the bundled session-memory hook still only handles command new/reset events and maintenance pruning removes stale sessions without a memory-finalization hook. Related open work covers overlapping reset behavior, but not the prune-before-delete requirement here.

## What This Changes

The issue requests firing session-memory finalization when sessions reset or are pruned, not only when compaction or explicit reset commands discard context.

## Best Possible Solution

Define one maintainer-approved memory-finalization contract for destructive session endings that covers explicit reset, idle/daily rollover, explicit delete, shutdown where feasible, and maintenance prune before deletion, with bounded async behavior, privacy/security rules, dedupe against compaction flushes, and focused regression tests.

## Reproduction Assessment

Yes at source level: current main shows the bundled session-memory handler only accepts command new/reset events, while maintenance prune deletes stale entries and archives transcripts without invoking memory finalization. I did not run a live destructive prune because this is a read-only review.

## Solution Assessment

No single implementation path is proven best yet. The safest direction is a canonical lifecycle/memory contract rather than ad hoc wiring of the old command hook into prune/reset paths.

## Risks / Open Questions

- A direct implementation could duplicate or conflict with the newer typed session_end/before_reset plugin lifecycle unless maintainers choose one canonical memory-finalization contract.
- Running an agentic memory flush or bundled hook automatically before prune may persist sensitive transcript content, spend model tokens, or delay destructive maintenance without a clear timeout and privacy policy.
- The prune path needs careful async ordering: firing after the store entry is deleted is too late, while blocking pruning indefinitely would risk maintenance stalls.
