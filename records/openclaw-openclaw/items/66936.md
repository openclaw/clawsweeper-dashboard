Keep open. Current main fixes the unresolved SecretRef crash and the `agents list`/bare `agents` plugin-preload hang path, but `agents add` still has no exact or argv-sensitive startup policy and can inherit the broad parent `agents` plugin load on the reported non-interactive script path.

## What This Changes

Fix `openclaw agents` CLI commands so `agents list` handles unresolved Telegram SecretRefs gracefully and completed `agents` subcommands exit instead of hanging after plugin preload.

## Best Possible Solution

Keep the shipped SecretRef and list/bare-agent fixes, then add a narrow startup-policy regression so non-interactive `agents add` avoids plugin runtime preload while interactive setup still has the plugin surfaces it needs.

## Solution Assessment
