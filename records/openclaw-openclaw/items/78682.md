This PR collapses the doctor-side hardcoded Gateway runtime "not found" note into the `renderSystemNodeWarning` fallback, adds focused doctor service tests, and updates the changelog.

## Best Possible Solution

## Solution Assessment

No. The PR removes the contradictory hardcoded not-found panel, but it leaves the separate install-plan warning plus a doctor-side warning, so the requested one-panel behavior is still not solved.
