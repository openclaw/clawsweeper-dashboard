Keep open: current main has the later Telegram forum-topic last-route fix that superseded the linked PR, but that does not prove the reported non-forum group restart/session-history loss is fixed. ## What This Changes

The issue asks OpenClaw to reliably persist Telegram group conversation sessions and transcript history across multi-day group chats and gateway restarts.

## Best Possible Solution
