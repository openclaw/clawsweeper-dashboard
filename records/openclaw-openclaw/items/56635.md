## Best Possible Solution

Refresh, land, or replace https://github.com/openclaw/openclaw/pull/56720 so current main restores `SPAM,TRASH,DRAFT,SENT` exclusions through the shared Gmail watcher argument builder and keeps any config/docs/tests aligned.

## Solution Assessment
