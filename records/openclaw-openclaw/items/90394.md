## Solution Assessment

Unclear until maintainer policy is accepted: the implementation is the narrow owner-boundary change for this TTL policy, but it must also update the remaining public config reference. If maintainers reject the broader stale-action window, the safer path is to keep the old default or choose a smaller cap/default and align all docs/tests/metadata.
