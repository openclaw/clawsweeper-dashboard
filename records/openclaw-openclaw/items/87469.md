Full review comments:

- **[P2] Pass the selected agent into local goal lookups:** `src/tui/embedded-backend.ts:537`
  - body: In local TUI mode, the current session can be the literal `global` key while the operator has selected a non-default agent.   - confidence: 0.9
- **[P2] Freeze terminal goal usage after completion:** `src/config/sessions/goals.ts:110-113`
  - body: `accountGoalUsage` recalculates `tokensUsed` for every goal status, including `complete`. If a user keeps chatting in the same session after completing a goal, later unrelated tokens will increase the achieved goal's final usage even though `complete` is treated as terminal elsewhere.
  - confidence: 0.86
