## What This Changes

The issue reports `openclaw backup create` stalling or silently exiting with a tiny temp archive on a 4.1 GB local state directory and asks for completion, clear failure, or large-state controls.

## Best Possible Solution

Land a reviewed version of https://github.com/openclaw/openclaw/pull/44288 or replace it with a narrower core backup fix that gives large-state backups bounded progress, clear failure, and safe exclusion behavior without weakening credential/restore safety.
