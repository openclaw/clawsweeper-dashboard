Improve the skill workshop proposal description limit or make an over-limit rejection clearer and more actionable.

## Best Possible Solution

Keep the documented 160-byte limit for this fix, remove or relax the schema-level character cap so the byte-aware workshop service provides the single actionable error, and consider a larger or configurable cap only as a separate product decision.
