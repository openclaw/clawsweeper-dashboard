Full review comments:

- **[P2] Retarget half-block detection to markdown-it renderers:** `ui/src/ui/markdown.ts:216-222`
  - body: This adds the half-block wrapper inside the old htmlEscapeRenderer.code path, but current main renders fenced and indented code through md.renderer.rules.fence and md.renderer.rules.code_block.   - confidence: 0.92

  - confidence: 0.88
- **[P3] Inherit the sidebar half-block code font size:** `ui/src/styles/chat/sidebar.css:128-134`
  - body: The sidebar half-block pre code rule inherits color and line-height but not font-size, leaving the existing sidebar code font-size rule active for the glyphs. Add font-size: inherit so the intended QR sizing applies consistently.
  - confidence: 0.78
