- **[P2] Retarget half-block detection to markdown-it renderers:** `ui/src/ui/markdown.ts:215-222`
  - body: This adds `.half-block-art` in the old code-rendering path, but current main renders fenced and indented code through `md.renderer.rules.fence` and `md.renderer.rules.code_block`.   - confidence: 0.91

  - confidence: 0.88

  - body: The sidebar override inherits color and line-height but leaves the existing `.sidebar-markdown pre code` font-size active on the inner `<code>`, shrinking the glyphs below the intended half-block `<pre>` size. Add `font-size: inherit` to keep sidebar rendering consistent.
  - confidence: 0.78
