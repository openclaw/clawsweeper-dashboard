Full review comments:

- **[P2] Move half-block detection into markdown-it renderers:** `ui/src/ui/markdown.ts:215-219`
  - body: This adds `.half-block-art` from the old `htmlEscapeRenderer.code()` path, but current main renders fenced and indented code through markdown-it `fence` and `code_block` rules.   - confidence: 0.9

  - confidence: 0.88

  - body: The sidebar override inherits color and line-height but leaves the existing `pre code` font-size rule in effect, so the glyphs do not use the larger half-block `<pre>` size. Add `font-size: inherit` for consistent QR tiling.
  - confidence: 0.78
