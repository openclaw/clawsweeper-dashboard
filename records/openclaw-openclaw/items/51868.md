- **[P2] Move half-block detection into the active markdown renderers:** `ui/src/ui/markdown.ts:215-219`
  - body: This adds `.half-block-art` from the old `htmlEscapeRenderer.code()` path, but current main renders fenced and indented code through markdown-it `fence` and `code_block` rules.   - body: The sidebar override inherits color and line-height but leaves the existing `pre code` font-size rule in effect, so the glyphs do not use the `1.3em` size set on the half-block `<pre>`. Add `font-size: inherit` for consistent QR tiling.
  - confidence: 0.78
