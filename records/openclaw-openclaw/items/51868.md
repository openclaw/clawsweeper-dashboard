  - confidence: 0.88

  - body: The sidebar override inherits color and line-height but leaves `.sidebar-markdown code` free to shrink the inner `<code>` relative to the half-block `<pre>`. Add `font-size: inherit` so sidebar QR output uses the intended size.
  - confidence: 0.78
