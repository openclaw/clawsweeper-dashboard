Keep this issue open. Current main still has the reported config-contract mismatch: per-agent entries are validated by strict `AgentEntrySchema`, that schema still omits `verboseDefault` and `elevatedDefault`, and the active closing implementation is open PR #74643 rather than merged code.

## Best Possible Solution
