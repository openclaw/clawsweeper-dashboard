## Summary

Keep this PR open: current main and v2026.6.9 still have the fixed 15s memory_search wrapper, and this branch is the strongest active candidate with real QMD proof, but the current head still budgets active-memory QMD searches from the manager's configured mode rather than the effective per-call override.
