  - body: `activeMemoryDeadline` is created before `qmdSearchModeOverride` is resolved, while `QmdMemoryManager.getSearchTimeoutMs()` budgets from the manager's configured mode. Active-memory QMD searches can therefore get a deadline for a different command lifecycle than the one actually executed.
