## What This Changes

The PR changes the GitHub Copilot provider to merge live `/models` rows ahead of bounded static fallback rows, add a static catalog hook, and add regression coverage for disabled, failed, and duplicate discovery cases.

## Best Possible Solution

Land the live-first catalog merge if maintainers accept fallback-only visibility; otherwise restrict successful live responses to live rows and reserve static rows for disabled, unauthenticated, failed, or offline discovery.
