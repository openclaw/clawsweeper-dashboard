## Summary

Keep this canonical tracker open. Current main and v2026.6.5 now cover a meaningful portion of basic conversation management—new sessions, a recent-session sidebar, switching, labels, search, derived titles/previews, lineage, goals, and checkpoint branching—but still lack project/topic trees, folders/tags/pins, user-controlled archival, recommendations, side-by-side multi-model responses, collaboration rules, committees, meeting workflows, task extraction, and project tracking. The discussion explicitly designated this as the canonical umbrella after https://github.com/openclaw/openclaw/issues/27603 was consolidated, while narrower open issues cover only slices of the remaining work.

## What This Changes

Requests structured WebChat conversation management, parallel multi-model collaboration, AI committee workflows, and project tracking.

## Best Possible Solution

Keep this as the umbrella tracker, record the shipped sidebar/session primitives as partial progress, and define separate canonical milestones for organization metadata and storage, Control UI project/topic navigation, and collaboration UX/protocol built on existing agent and subagent infrastructure.
