## Summary

Keep open. Current main has useful flat session listing/switching, labels, search, derived titles/previews, New Chat session creation, and sub-agent primitives, but it still lacks the requested project/topic directory, folders/tags/persistent pins/recommendations, side-by-side multi-model collaboration, consensus/meeting/committee workflows, task extraction, and project tracking; maintainer discussion also designates this as the canonical tracker after #27603 was consolidated here.

## What This Changes

Request broad WebChat conversation organization and multi-AI collaboration features, including project folders, search/tagging/pinning/archive/recommendations, parallel model comparison, role/committee workflows, meeting minutes, tasks, and project tracking.

## Best Possible Solution

Keep this as the canonical product tracker and split it into milestones for session organization metadata/storage/protocol, WebChat directory/navigation UI, and a separate multi-agent collaboration design that reuses existing sub-agent primitives where appropriate.
