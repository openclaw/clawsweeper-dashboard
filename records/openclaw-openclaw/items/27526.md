## Summary

Keep open. Current main has useful flat session listing/switching, labels/search, derived titles/previews, sub-agent primitives, and multi-agent routing, but it still lacks the requested project/topic directory, multi-tag/archive/persistent-pin/recommendation workflow, side-by-side multi-model collaboration, consensus/meeting flows, committee templates, task extraction, and project tracking; maintainer discussion already designates this as the canonical tracker after #27603 was consolidated here.

## What This Changes

The issue requests WebChat conversation organization with folders/tags/search/pinning/archiving plus parallel multi-AI roles, comparison, committee meetings, minutes, tasks, and project tracking.

## Best Possible Solution

Keep this as the canonical product tracker and split it into milestones for session organization metadata/storage/protocol, WebChat directory navigation, and a separate multi-agent collaboration design that reuses existing agents/sub-agents where appropriate.

Not applicable. This is a feature request, and current-source inspection confirms only partial primitives exist rather than a broken existing documented behavior.
