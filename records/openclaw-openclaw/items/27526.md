## Summary

Keep open. Current main has useful session primitives, labels/search, archive filtering, New Chat session creation, a header session picker, and sub-agent metadata, but it still does not implement the requested project/topic directory, durable folders/tags/pins/recommendations, side-by-side multi-model collaboration, committee workflows, task extraction, or project tracking; the discussion also keeps this as the canonical tracker.
