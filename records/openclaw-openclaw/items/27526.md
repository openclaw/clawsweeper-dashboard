## Summary

Keep open. Current main has flat session listing/switching, labels, search, derived titles/previews, New Chat session creation, local message pins, and sub-agent primitives, but it still lacks the requested project/topic directory, persistent folder/tag/archive/pin metadata, recommendations, side-by-side multi-model collaboration, consensus/meeting/committee workflows, task extraction, and project tracking; maintainer discussion also designates this as the canonical tracker.

## What This Changes

The issue requests first-class WebChat conversation organization plus multi-AI collaboration and committee workflows across storage, protocol, UI, and agent orchestration.

## Best Possible Solution

Keep this as the canonical product tracker and split it into milestones for durable session organization metadata, WebChat directory/navigation UI, and a separate multi-agent collaboration design that reuses sub-agent primitives where appropriate.
