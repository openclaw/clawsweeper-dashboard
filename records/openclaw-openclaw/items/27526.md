## Summary

Keep open. Current main has useful flat session listing, search/filtering, labels, create/switch support, compaction checkpoints, model/thinking overrides, and sub-agent primitives, but the requested project/topic directory, durable tags/archive/pins/recommendations, side-by-side multi-model collaboration, role/consensus/meeting workflows, task extraction, and project tracking are still not implemented; maintainer discussion also marks this as the canonical tracker.

## What This Changes

The issue requests broad WebChat conversation organization plus multi-AI collaboration and committee/project-management workflows.

## Best Possible Solution

Keep this as the canonical product tracker and split it into milestones for session metadata/storage/protocol, Control UI directory organization, and a separate collaboration design that reuses existing sub-agent primitives without making heavy orchestration the default architecture.

## Reproduction Assessment

Not applicable. This is a feature tracker, and source/docs inspection shows the missing capabilities are not a failing current contract but new product surface beyond the flat session and sub-agent primitives already present.
