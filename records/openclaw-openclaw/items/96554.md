  - body: This adds `workboard_move` to the tool factory, but Workboard exposes agent tools through the explicit names list in `extensions/workboard/index.ts` and the manifest `contracts.tools` list. A card claimed between that precheck and the queued update can still be moved by the stale caller; pass the scope into `store.move` and assert it inside the mutation like the sibling lifecycle methods.
  - confidence: 0.86
