## Summary

Keep open: the Feishu topic-affinity bug is plausible, but this PR needs changes because its ACP sender-scoped fallback binds to a bare topic id that current Feishu runtime binding lookup will not match, and the PR still lacks real behavior proof.

## What This Changes

The PR changes Feishu topic/session parsing and reply routing across same-turn context, subagent/ACP spawns, outbound sends, and related tests, with incidental formatting churn in UI/Discord/Telegram files.

## Best Possible Solution

Rebase and narrow the PR on current main, reuse Feishu conversation-id helpers so ACP bindings store the exact live sender-scoped topic conversation id or fail closed, drop unrelated churn, and add regression plus redacted live Feishu proof.

## Reproduction Assessment

Yes, from source inspection: the PR recovers only the bare Feishu topic id for ACP binding, while current Feishu ingress and the binding adapter resolve runtime bindings by exact live conversation id. I did not run a live Feishu tenant reproduction.

## Solution Assessment

No. The direction is useful, but the current ACP fallback would create an unreachable sender-scoped Feishu binding, and the branch should be narrowed around current-main Feishu binding semantics before merge.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Bind Feishu ACP sessions to the live conversation id:** `src/agents/acp-spawn.ts:120`
  - body: For a sender-scoped Feishu topic session this helper returns only the topic id, for example `om_x...`. `prepareAcpThreadBinding` then stores that as the binding conversation, but Feishu ingress resolves runtime bindings by exact live conversation ids such as `oc_chat:topic:om_x:sender:ou_user`, and the adapter looks up that exact id. A thread-bound ACP session created from this path will not receive follow-up messages from the same sender-scoped topic; resolve the exact Feishu conversation id or fail closed when it cannot be recovered.
  - confidence: 0.87
