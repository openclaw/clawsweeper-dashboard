Add a first-class manual-only session reset mode, such as `mode: "never"`, for base, type, and channel reset policies.

Keep this as the narrow config-contract tracker; after maintainers choose the literal and semantics, update schema, public types, runtime policy, docs/help, and focused regression tests together.
