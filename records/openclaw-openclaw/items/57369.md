The issue requests a true manual-only session reset mode, such as `mode: "never"`, for base, type, and channel reset policies.

Define one additive manual-only reset mode and apply it consistently across exported types, Zod validation, runtime freshness evaluation, config help/docs, and focused reset-policy tests.
