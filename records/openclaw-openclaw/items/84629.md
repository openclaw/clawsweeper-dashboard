Confirm the actual Talk transport, browser console, gateway Talk events, and whether `talk.session.cancelOutput` fires at the cutoff; if gateway-relay self-echo is confirmed, harden Browser Talk barge-in with an echo/noise guard or minimum playback duration while preserving real user interruption.
