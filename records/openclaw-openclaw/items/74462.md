## Best Possible Solution

Decide the product contract first. If `contextSafety` is intended, implement it end to end across runtime, schema, exported config types, generated schema/help metadata, docs, and focused validation/runtime tests; if it is not intended, document the supported `agents.defaults.compaction` path and investigate why the reporter saw a divergent dist artifact.
