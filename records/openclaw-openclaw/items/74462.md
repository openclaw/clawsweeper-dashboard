## What This Changes

## Best Possible Solution

Decide the public config contract first. If `contextSafety` is intended, implement it end to end across runtime consumers, strict schemas, exported config types, schema labels/help, docs, and focused validation/runtime tests; if it is not intended, document the supported `agents.defaults.compaction` controls and close the loop with an explanation of the supported path.

## Solution Assessment
