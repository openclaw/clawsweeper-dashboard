## What This Changes

The issue reports that Claude CLI active-tool heartbeat progress can keep wedged live-session subprocesses looking active and delay stuck-session recovery.

## Best Possible Solution

Review and either land or replace the linked fix PR with a diagnostic policy that lets truly wedged Claude tools age without regressing valid long-running Claude-native tools.
