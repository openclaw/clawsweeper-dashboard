## What This Changes

Demote or suppress warn-level stuck-session diagnostics when recovery skips because a healthy embedded run is active.

## Best Possible Solution

Land a narrow diagnostic recovery change that demotes the `active_embedded_run` observe-only skip logs to debug while preserving warnings for real stuck, stalled, aborted, or failed recovery states.
