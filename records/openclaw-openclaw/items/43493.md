Full review comments:

- **[P1] Preserve all existing input modalities:** `src/commands/model-metadata.ts:140-149`
  - body: The editor only offers `text` and `image`, then always writes `result.input`. If an existing custom model has another supported modality such as `audio` or `video`, editing an unrelated field like context window will save the model back without that modality. Preserve unoffered existing values or expose every supported modality before writing this field.
  - confidence: 0.86
- **[P2] Clear model-level API overrides for Auto:** `src/commands/model-metadata.ts:131-133`
  - body: Selecting `Auto (inherit from provider)` leaves `result.api` unset, but `promptModelMetadataForPrimary` later merges over `entry.model`, so any previous model-level `api` value survives. That makes the Auto option a no-op for existing overrides instead of returning the model to provider-level routing.
  - confidence: 0.91
