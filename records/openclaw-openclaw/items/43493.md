## What This Changes

The PR adds a custom-provider-only metadata wizard to `openclaw configure --section model`, plus helper functions and tests for editing display name, API adapter, reasoning, input modalities, token limits, and pricing.

## Best Possible Solution

Keep the custom-provider metadata wizard direction, but make the API adapter Auto option truly clear model-level overrides or relabel it to preserve the current adapter intentionally, then cover that behavior with a focused regression test.
