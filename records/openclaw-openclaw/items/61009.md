## What This Changes

The issue reports a docs/runtime mismatch for per-call `host=node` overrides when `tools.exec.host=auto`.

## Best Possible Solution

Land a narrow docs-only correction across the exec, exec approvals, and nodes pages so both `host=node` and `host=gateway` overrides from `auto` are documented as allowed only when no sandbox runtime is active, then redeploy the public docs.
