Keep open. Current main partially addresses readable session names through stored `label`/`displayName` projection and picker rendering, but the requested mutable `displayName` field and Control UI derived-title fallback are still missing or need a maintainer API/UX decision.

## Best Possible Solution

Define one session title contract for Control UI, then update the Gateway schema/store handling and picker to use that contract with focused Gateway and UI coverage.
