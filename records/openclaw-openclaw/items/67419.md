## Summary

Keep open. Current `origin/main` still defaults missing `agents.defaults.contextInjection` to per-turn bootstrap injection, still injects full `MEMORY.md` when present, keeps only `HEARTBEAT.md` after the cache boundary, and has an open draft PR for only one continuation-skip subcase.

## What This Changes

The issue requests making lower-cost bootstrap injection behavior the default, reducing `MEMORY.md` bootstrap load, and adding dynamic bootstrap-file cache-boundary controls.

## Best Possible Solution

Decide the default bootstrap reinjection policy first, then land coordinated runtime, config, docs, UI, and regression-test changes; keep `MEMORY.md` lazy loading and dynamic-file ordering as explicit scoped follow-ups or under their canonical trackers.

## Reproduction Assessment
