## Summary

Keep open. ## What This Changes

The issue requests lower-token bootstrap/session context behavior by making `continuation-skip` the default, stubbing or lazy-loading `MEMORY.md`, and adding dynamic bootstrap-file cache-boundary configuration.

## Best Possible Solution

Keep this as the bootstrap/context-cost tracker, review or replace the narrow continuation-skip fix, and split default-policy, `MEMORY.md` lazy-loading, and dynamic bootstrap ordering into explicit maintainer decisions.
