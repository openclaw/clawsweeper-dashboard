## What This Changes

The PR rewrites favicon and apple-touch-icon hrefs in served Control UI index HTML when `gateway.controlUi.basePath` is set and adds focused gateway tests.

## Best Possible Solution

Port the favicon rewrite onto current main, emit only escaped or strictly validated icon hrefs, and keep current Control UI CSP hash behavior intact.

## Reproduction Assessment

Yes, source-level. Current main and v2026.5.12 emit root icon hrefs while the basePath classifier only serves paths under the configured mount, so `/favicon.svg` bypasses the Control UI handler.

## Solution Assessment

Yes, with changes. Rewriting only icon hrefs at index-serving time is narrow, but the implementation needs escaped or validated output and a rebase that preserves CSP hash handling.
