## Best Possible Solution

## Reproduction Assessment

No high-confidence live reproduction is present. Source inspection shows the current automatic group prompt broadly forbids same-group message tool sends while Telegram presentation buttons are supported, but this PR does not include before/after Telegram runtime proof.

## Solution Assessment

No. The direction is narrow, but the current wording is not the best fix because it unconditionally asks for a silent final turn even when group silent replies are disabled.
