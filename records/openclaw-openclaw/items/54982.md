## Summary

Keep open: current main still lacks the default-off TUI preservation feature, and this PR is the canonical implementation path for the closed related feature request, but the branch has source-reproducible preserve-mode defects that should be fixed before merge.

## What This Changes

The PR adds a `--preserve-intermediate` TUI flag, threads it into `runTui`, makes `ChatLog` append preserved streaming assistant snapshots, and adds focused docs plus CLI/ChatLog tests.

## Best Possible Solution

Land a rebased default-off implementation that preserves intermediate snapshots while removing all chunks for suppressed runs, skipping unchanged final duplicates, and documenting the flag in the CLI/web references with its fixed log-budget trade-off.

## Reproduction Assessment

Yes for the PR-head defects by source inspection: enable preserve mode, append multiple deltas for one run, then either drop a suppressed final or finalize with text equal to the latest delta. I did not run tests because this review is read-only.

## Solution Assessment

No. The CLI/TUI option boundary is reasonable, but the branch should track all preserved chunks for a run, drop them together when output is suppressed, skip unchanged final snapshots, and update the reference docs before merge.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Drop every preserved chunk for suppressed runs:** `src/tui/components/chat-log.ts:81-84`
  - body: Preserve mode overwrites the run's tracked component on every delta, so `dropAssistant` only knows about the newest chunk. Existing final handlers call `dropAssistant` for missing finals and suppressed `(no output)`, leaving earlier chunks visible even though that run was intentionally discarded.
  - confidence: 0.91

  - body: The PR adds the flag only to an example. Current main has a CLI options table and web TUI options section that should include the supported option and the per-delta log-budget trade-off so users scanning references can find it.
  - confidence: 0.8
