## Summary

## Best Possible Solution

Fix inbound image download/decrypt and media-fact injection in the external @tencent-weixin/openclaw-weixin plugin or ClawHub package; open a narrow OpenClaw SDK issue only if that work exposes a missing generic channel media seam.
