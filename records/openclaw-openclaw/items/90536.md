## What This Changes

Request to add the currently required OpenAI ChatGPT model-access OAuth scope to the bundled OpenAI login path and avoid silently rotating past the requested primary model on auth failure.

## Best Possible Solution

Validate the current OpenAI ChatGPT OAuth scope contract live or with maintainer-approved upstream evidence, update the login scope and focused tests accordingly, and ensure primary auth-scope failures are visible instead of quietly rotating to a fallback model.
