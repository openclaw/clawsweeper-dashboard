Overall confidence: 0.92

Full review comments:

- **[P1] Restore inbound-reply-dispatch aliases:** `src/plugin-sdk/inbound-reply-dispatch.ts:84-106`
  - body: This public SDK subpath still ships in package exports, but the patch renames `runPreparedInboundReplyTurn`, `runInboundReplyTurn`, `recordDroppedChannelTurnHistory`, and the `ChannelTurn*` types without deprecated aliases. Existing plugins importing those names will fail immediately after upgrade, so keep the old exports as wrappers/type aliases while introducing the new inbound names.
  - confidence: 0.94
- **[P1] Keep channel-turn SDK aliases importable:** `src/plugin-sdk/channel-inbound.ts:73-119`
  - body: `openclaw/plugin-sdk/channel-inbound` used to export `buildChannelTurnContext`, `filterChannelTurnSupplementalContext`, `buildChannelTurnMediaPayload`, and related `*Turn*` type aliases, but the new export block drops them. Because this is public plugin API and the PR frames the old names as deprecated compatibility, those names need to remain exported as deprecated aliases until a breaking release.
  - confidence: 0.95
- **[P1] Re-export legacy turn dispatch helpers:** `src/plugin-sdk/channel-message.ts:154-158`
  - body: The `channel-message` entrypoint no longer exports `hasFinalChannelTurnDispatch`, `hasVisibleChannelTurnDispatch`, or `resolveChannelTurnDispatchCounts`; it only exports the renamed `ChannelMessageReply` names. Plugin code importing the old helpers from this public subpath will break, so keep the legacy names as deprecated aliases alongside the new names.
  - confidence: 0.93
- **[P2] Preserve Signal quote metadata for empty replies:** `extensions/signal/src/monitor/event-handler.ts:204-210`
  - body: The new Signal path only creates `supplemental.quote` when `entry.replyToBody` is truthy, so replies to media-only or otherwise empty-text messages drop `replyToSender` and `replyToIsQuote`. The old path forwarded those fields independently of body text, so the refactor should construct quote metadata whenever any quote facts are present.
  - confidence: 0.92
