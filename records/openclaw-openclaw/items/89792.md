## Solution Assessment

Yes, the revised split is the best code shape I found: it avoids process.exit(), preserves awaited shutdown, and limits the private ws dependency behind a guarded helper. The remaining blocker is real behavior proof for the revised awaited path, not an identified line-level code defect.
