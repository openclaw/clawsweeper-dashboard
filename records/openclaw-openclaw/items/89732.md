## Summary

The PR routes MCP HTTP, SSE, and Streamable HTTP fetches through the shared SSRF guard, adds per-hop dispatcher policy handling for MCP TLS/proxy behavior, removes the local Streamable HTTP redirect loop, and adds focused redirect tests.

## Best Possible Solution

## Reproduction Assessment
