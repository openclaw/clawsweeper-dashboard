Full review comments:

- **[P1] Read dropped media from the normalized payload:** `src/infra/outbound/deliver.ts:1921`
  - body: `normalizedPayloads` contains `{ index, payload }` entries, so `p.droppedMedia` does not exist. This currently fails `check-prod-types` with TS2339 and would also make transcript mirroring collect no dropped-media notices; read `p.payload.droppedMedia` instead.
  - confidence: 0.99
- **[P1] Remove the unused delivery imports:** `src/infra/outbound/deliver.ts:1-4`
  - body: The new imports for `sendMediaWithLeadingCaption`, `chunkByParagraph`, and `chunkMarkdownTextWithMode` are unused, and the repo's no-unused-locals gate fails on them. Drop these imports so the production type check can pass.
  - confidence: 0.97
- **[P1] Remove the unused config-dir import:** `src/auto-reply/reply/reply-media-paths.ts:18`
  - body: `resolveConfigDir` is imported but never used in the updated reply media normalizer, which is another TS6133 failure in `check-prod-types`. Removing it is required before this branch can build.
  - confidence: 0.97
- **[P2] Preserve silent replies when media normalization fails:** `src/auto-reply/reply/reply-payloads-base.ts:83-86`
  - body: By making dropped-media-only payloads renderable globally, an exact `NO_REPLY\nMEDIA: ./missing.png` payload now survives after media normalization strips the media. The existing agent-runner test expects zero reply payloads for that silent case and CI is failing with one payload returned, so dropped-media notices need to remain suppressed for exact silent replies.
  - confidence: 0.93
- **[P2] Drop the legacy generic media warning:** `src/auto-reply/reply/reply-media-paths.ts:240-247`
  - body: When every media item fails, this branch still adds the old `Media failed` warning while also carrying `droppedMedia`; delivery then appends the new detailed `Attachment not sent` notice. That produces duplicate warnings for the all-dropped path, so the legacy warning should be removed or skipped when `droppedMedia` is emitted.
  - confidence: 0.86
