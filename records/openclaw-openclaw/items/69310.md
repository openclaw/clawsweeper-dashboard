  - body: `normalizedPayloads` contains `{ index, payload }` entries, so `p.droppedMedia` is not a valid property. This breaks typecheck now and would prevent transcript mirroring from seeing dropped-media notices; read from `p.payload.droppedMedia` instead.

  - confidence: 0.94

  - body: When every media item fails, this branch keeps the legacy `Media failed` text while also carrying `droppedMedia`. Delivery later formats `droppedMedia` into an `Attachment not sent` notice, so users can receive two warnings for one failed attachment.
  - confidence: 0.9
