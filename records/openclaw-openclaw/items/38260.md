## Best Possible Solution

Keep this as the native libvips/SIGILL crash tracker until maintainers reproduce on latest packaged Linux x64, collect coredump/backtrace plus CPU and binary details, then choose a dependency, backend, packaging, or isolation fix with focused WebChat image-flow coverage.
