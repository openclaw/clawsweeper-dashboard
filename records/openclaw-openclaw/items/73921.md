## Summary

Keep open. Current main still sends local `infer model run --model` through the agent-aware simple-completion path, and open PR #74028 is linked with closing syntax, so this issue should remain open until that PR lands or is replaced.

## What This Changes

The issue asks for `openclaw infer model run --model <provider/model>` to honor explicit Copilot model overrides instead of falling back to the configured/default model.

## Best Possible Solution

Review and land or replace #74028 with a narrow CLI fix for fully qualified local overrides while preserving unqualified/default selection behavior.

## Reproduction Assessment

Yes. The issue gives exact CLI commands and response-metadata observations, and current main still exposes the matching local dispatch seam in `runModelRun`.

## Solution Assessment
