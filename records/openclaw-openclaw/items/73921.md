## Summary

Keep open. Current main still routes local `model.run` through the agent-aware simple-completion preparation path, and open PR #74028 explicitly closes this issue, so the issue should remain open until that PR lands or is replaced.

## What This Changes

The issue reports that local `infer model run --model <provider/model>` ignores provider-qualified Copilot overrides and runs the configured Anthropic default instead.

## Best Possible Solution

Review and land or replace #74028 with a narrow CLI fix for provider-qualified local overrides while preserving unqualified/default model selection behavior.
