## What This Changes

The issue requests that `openclaw infer model run --model <provider/model>` honor fully qualified Copilot model overrides instead of routing through the default Anthropic model.

## Best Possible Solution

Review and land PR #74028, or replace it with the same narrow fix: fully qualified local model-run overrides should prepare that exact provider/model directly, while unqualified model refs continue through existing agent-aware default selection.
