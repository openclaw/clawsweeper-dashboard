## Summary

Keep open. Current main still routes local `infer model run --model` through the agent-aware preparation path, and #74028 is an open closing PR for the provider-qualified local override behavior, so this issue should remain open until that PR lands or is replaced.

## What This Changes

The issue requests that local `openclaw infer model run --model <provider/model>` honor the requested Copilot model instead of falling back to the configured Anthropic default.

## Best Possible Solution

Review and land or replace #74028 with a narrow CLI fix that prepares provider-qualified local model overrides directly while preserving existing unqualified/default selection behavior.

## Reproduction Assessment

Yes, with medium confidence from source and reporter metadata rather than a live provider call. The issue gives exact CLI commands and current main still exposes the local dispatch seam targeted by the open closing PR.

## Solution Assessment

Yes. A CLI-layer fix that handles fully qualified local overrides before the agent-aware default-selection path is the narrowest maintainable solution; changing the generic runtime resolver would be broader than needed.
