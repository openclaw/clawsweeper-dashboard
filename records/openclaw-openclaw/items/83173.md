## What This Changes

The issue requests a user-visible annotation when memory retrieval relies on low-confidence, stale, or otherwise uncertain memory metadata.

## Best Possible Solution

Define a small memory confidence/provenance display contract, then surface authoritative low-confidence or stale metadata in the memory_search/wiki_search retrieval surface with focused tests and docs.

## Solution Assessment

Unclear until maintainers choose the UX and metadata contract. The narrow maintainable path is to extend the retrieval/result surfaces rather than relying on the model to self-assess confidence after retrieval.
