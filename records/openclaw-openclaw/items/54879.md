## What This Changes

The issue requests a CDP-backed browser action for IME/native-style text insertion into contenteditable editors.

## Best Possible Solution

Add a narrow browser-plugin act action that focuses a ref/selector or current editable target and sends native text through the appropriate CDP or existing-session path, with docs and regression coverage for rich contenteditable IME text.

## Reproduction Assessment

## Solution Assessment
