- **[P1] Preserve private mode when recreating transcripts:** `src/agents/pi-embedded-runner/session-manager-init.ts:56`
  - body: After `archiveFileOnDisk` renames the existing transcript, this write recreates the original path without the transcript file's `0o600` mode contract.   - body: The new tests hardcode `/tmp` and concatenate the session path, so the regression coverage can fail before exercising the bug on Windows or custom runners. Use `os.tmpdir()` with `path.join()`.
  - confidence: 0.82

  - body: This helper invokes raw `pnpm vitest run`, bypassing the repo's test routing and process handling guidance. Use `pnpm test src/agents/pi-embedded-runner/session-manager-init.test.ts` or remove the wrapper.
  - confidence: 0.8
