## What This Changes

Display per-assistant-message response latency, optional token count, and throughput in the Control UI chat transcript.

## Best Possible Solution

Add a canonical assistant-turn timing contract that can feed both live chat events and history reloads, then render it through the existing compact message metadata UI.

## Solution Assessment
