## Summary

Keep open. Current main has compaction-only AGENTS.md section reinjection and a separate daily-memory startup prelude, but no configured AGENTS.md section injector for fresh parentless sessions.

## What This Changes

Request a fresh-session lifecycle/config hook that injects selected AGENTS.md sections on first turns, analogous to postCompactionSections.

## Best Possible Solution

Add a documented fresh-session AGENTS.md section injection contract, sharing the existing section extraction behavior while keeping startupContext, compaction, and hook semantics clear.

## Reproduction Assessment

Yes, source-reproducible. Current main wires readPostCompactionContext only through compaction paths, while startupContext loads daily memory for reset/startup model runs and no onSessionStart-style AGENTS.md section schema is present.

## Solution Assessment

Unclear. A parallel key may work, but the narrowest maintainable solution needs maintainer choice on config placement, default sections, trust level, and extraSystemPrompt versus system-event injection.
