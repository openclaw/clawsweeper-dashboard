## What This Changes

The issue requests live Signal tool-call progress using an edit-free send/delete pattern instead of editable preview messages.

## Best Possible Solution

If maintainers accept the UX, implement a Signal-owned opt-in rolling progress tombstone lifecycle with schema, docs, tests, and changelog aligned.

## Solution Assessment
