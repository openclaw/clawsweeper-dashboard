## What This Changes

The issue requests bounded handling for very large session JSONL transcripts so gateway history, usage, and session-event paths do not block the event loop while reading and splitting full transcript files.

## Best Possible Solution

Replace the gateway transcript readers with bounded, decoder-safe line or entry iteration that preserves active-branch traversal, compaction markers, sequence metadata, malformed-line tolerance, cross-agent transcript resolution, and usage aggregation. Treat rotation, doctor warnings, and health-probe wording as follow-up hardening after the event-loop blocking read path is removed.
