## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.92

Full review comments:

- **[P2] Redact activity tool arguments before emitting them:** `src/agents/pi-embedded-subscribe.handlers.tools.ts:411`
  - body: This new activity event serializes tool args into metadata. Tool args can contain API keys, Authorization headers, command-embedded secrets, or private paths, and the same metadata is later saved locally by the controller.
  - confidence: 0.9
- **[P2] Keep activity events ephemeral by default:** `ui/src/ui/activity/activity-controller.ts:247`
  - body: handleEvent schedules a localStorage save for every activity event. Since activity metadata can include tool inputs, tool outputs, and subagent result previews, opening the Activity tab would persist sensitive execution data despite the stated ephemeral default.
  - confidence: 0.93
- **[P2] Preserve ancestors when filtering the activity tree:** `ui/src/ui/activity/activity-tree.ts:348-349`
  - body: filterTree drops nodes whose own kind is not selected and then rebuilds roots only from surviving original roots. A tool-only or subagent-only filter can render an empty tree even when matching descendants exist under a run.
  - confidence: 0.9
- **[P2] Emit a terminal event when subagent startup fails:** `src/agents/subagent-spawn.ts:725-738`
  - body: The PR emits subagent.start before the child gateway run request, but the catch path returns an error without any matching activity end or error event. Failed spawns leave a running subagent node until reload or pruning.
  - confidence: 0.84
- **[P2] Regenerate every shipped locale bundle:** `ui/src/i18n/locales/en.ts:32`
  - body: The PR adds English Activity tab keys but updates only a subset of locale bundles. Current main asserts each shipped locale has the same flattened key shape as English, so locales such as ar, fa, fr, and ja-JP would be missing these keys after rebase.
  - confidence: 0.92
