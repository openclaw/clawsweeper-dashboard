- **[P1] Quote provider IDs before rendering re-login commands:** `src/agents/failover-error.ts:517`
  - body: This constructs a copy-paste shell command with `provider` inserted verbatim, and the skip-cache path does the same with `candidate.provider`.   - confidence: 0.82
