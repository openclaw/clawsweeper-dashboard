## Summary

Keep open. ## What This Changes

The issue requests an opt-in Telegram trusted-owner mode that can bypass approval prompts for selected owner/group/tool-class actions.

## Best Possible Solution

Maintainers should either document `/elevated full`/YOLO policy as the intended break-glass route or approve a narrow audited Telegram policy scoped by numeric owner ID, allowlisted group/topic, tool class, and destructive-operation rules.
