## What This Changes

Add collapsible/foldable assistant messages in Control UI chat, with long messages collapsed by default and Show more/Show less controls.

## Best Possible Solution

Define the Control UI long-message folding semantics, then land a revised bounded implementation or replacement for https://github.com/openclaw/openclaw/pull/51546 with focused tests and browser proof while preserving oversized markdown fallback behavior.
