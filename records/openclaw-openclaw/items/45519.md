## Summary

Add a Control UI chat affordance to collapse long assistant responses with show-more and expand/collapse controls.

## Best Possible Solution

Add accessible Control UI-owned folding state for long assistant markdown after maintainers choose the UX policy, reusing existing chat state/action patterns without changing tool-output or duplicate-message behavior.

## Solution Assessment

Unclear until maintainers choose the UX policy. The fold/show-more direction is plausible, but the implementation should first define threshold, latest/streaming behavior, code-block handling, and accessibility semantics.
