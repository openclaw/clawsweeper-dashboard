The issue requests collapsible or foldable long assistant messages in the Control UI chat to reduce scrolling in long conversations.

Define a Control UI-native progressive-disclosure design, then add accessible per-message folding in the chat renderer with streaming-safe behavior, mobile coverage, focused renderer/browser tests, and the existing side reader preserved.
