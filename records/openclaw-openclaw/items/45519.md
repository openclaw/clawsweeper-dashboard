## What This Changes

The issue requests collapsible or foldable long assistant responses in the Control UI chat, with per-message expansion and optional bulk controls.

## Best Possible Solution

Define the Control UI long-message folding semantics, then land a revised bounded implementation or replacement for https://github.com/openclaw/openclaw/pull/51546 with focused tests and browser proof while preserving oversized markdown fallback behavior.

## Reproduction Assessment

## Solution Assessment

Unclear until maintainers choose the UX policy. Progressive disclosure is plausible, but the implementation should bound parse/render work, preserve oversized fallback behavior, and handle streaming/latest-message behavior deliberately.
