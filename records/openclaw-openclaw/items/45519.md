## What This Changes

The issue requests collapsible or foldable assistant messages in the Control UI chat to reduce scrolling in long conversations.

## Best Possible Solution

Define a Control UI-native progressive-disclosure design, then implement accessible per-message folding in the chat renderer with streaming-safe behavior, mobile coverage, and focused renderer/browser tests while preserving the existing side reader.
