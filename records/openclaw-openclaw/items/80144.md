## Review Findings

Overall correctness: patch is incorrect

Full review comments:

- **[P2] Expire old query cache entries before storing more:** `extensions/memory-core/src/memory/manager.ts:375-378`
  - body: With `cacheTtlMs` enabled, expired entries are only removed when the same key is read or when a sync/provider/close clears the whole map.
