## Best Possible Solution

Land a secops-reviewed sanitizer update with regression coverage for long alphanumeric, hex-like, JWT-like, and valid encoded-secret values while preserving null-byte blocking, maximum-length handling, explicit credential-name blocking, and useful warning coverage.
