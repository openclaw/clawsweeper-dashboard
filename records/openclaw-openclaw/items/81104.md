## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.87

Full review comments:

- **[P2] Keep policy check read-only:** `extensions/policy/src/cli.ts:142`
  - body: The policy docs describe `policy check` and `policy watch` as read-only, but this path calls `readConfigFileSnapshot()` with observation enabled. In this PR, default snapshot reads still write config observation state, while `observe: false` is the explicit no-write path, so running either policy command can mutate `.openclaw/logs/config-health.json`. Use `readConfigFileSnapshot({ observe: false })` here and cover the policy command path.
  - confidence: 0.91
