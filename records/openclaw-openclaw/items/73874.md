Keep open. ## What This Changes

The issue reports Windows Docker Desktop Gateway event-loop starvation that leaves HTTP probes, Gateway WebSocket RPC, and Slack dispatch unresponsive after v2026.4.24 upgrades.

## Best Possible Solution

Keep this as the focused Docker/Slack tracker until maintainers capture a current-main or latest-release Windows Docker profile during an event-loop-delay spike, then fix the identified blocking startup, periodic, or Slack lifecycle operation.
