  - body: The slim runtime install list omits `ca-certificates`, `python3`, `tini`, and `update-ca-certificates`; paired with the removed named-volume state creation, this can break HTTPS egress, skills, process handling, and first-run persistence in Docker.
  - confidence: 0.93

  - body: The Docker CLI block only compares the first fingerprint returned by `gpg`; current main first requires exactly one primary `pub` key so a multi-key file cannot pass fingerprint comparison while apt trusts extra keys.
