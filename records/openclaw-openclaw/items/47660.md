  - body: This reintroduces `curl | bash` for Bun and bypasses current main's digest-pinned `OPENCLAW_BUN_IMAGE` source stage, moving image builds back to executing a remote installer as root.

  - confidence: 0.94

  - body: The slim runtime installs only `procps hostname curl git lsof openssl`, dropping `ca-certificates`, `python3`, `tini`, and `update-ca-certificates`; that breaks Node HTTPS egress and removes current Python-backed skill and init-process support.
  - confidence: 0.96
- **[P1] Keep the Docker apt key primary-key guard:** `Dockerfile:171-177`
  - body: The Docker CLI install path loses the current `docker_gpg_pub_count` check, so a multi-key GPG file could pass the first fingerprint comparison while extra keys are still imported into apt trust.
