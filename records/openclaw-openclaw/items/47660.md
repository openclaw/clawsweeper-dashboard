Overall confidence: 0.96

Full review comments:

- **[P1] Keep Bun on the pinned source image:** `Dockerfile:33-34`
  - body: This reintroduces `curl | bash` for Bun and drops the digest-pinned `OPENCLAW_BUN_IMAGE` source stage that current main uses. That moves Docker builds back to executing a remote installer as root instead of consuming a reviewed image digest.
  - confidence: 0.95
- **[P1] Restore runtime CA, Python, and tini packages:** `Dockerfile:106-108`
  - body: The runtime apt set installs only `procps hostname curl git lsof openssl`, dropping `ca-certificates`, `python3`, `tini`, and `update-ca-certificates`. On the slim runtime this breaks Node HTTPS egress and removes current Python-backed skill and init-process support.
  - confidence: 0.96
- **[P1] Preserve the runtime-assets validation stage:** `Dockerfile:77-78`
  - body: Collapsing `runtime-assets` means the runtime image no longer runs bundled-plugin postinstall, Docker plugin-dist pruning, package export validation, QA build/copy, or the Matrix native-addon guard before files are copied into the final image.
  - confidence: 0.94
- **[P1] Preserve selective plugin dependency staging:** `Dockerfile:43`
  - body: Copying the whole `extensions` tree before `pnpm install` makes every plugin affect the install layer and ignores the documented `OPENCLAW_EXTENSIONS` selective dependency contract that setup and release workflows still use.
  - confidence: 0.91
- **[P1] Keep the Docker apt key primary-key guard:** `Dockerfile:171-177`
  - body: The Docker CLI install path loses the current `docker_gpg_pub_count` check, so a multi-key GPG file could pass the first fingerprint comparison while extra keys are still imported into apt trust.
  - confidence: 0.93
- **[P1] Pre-create the node-owned state directory:** `Dockerfile:197`
  - body: The branch drops the `install -d -m 0700 -o node -g node /home/node/.openclaw` guard before switching to `USER node`. First-run named volumes can then be root-owned or permissive, causing write failures or weakening credential-state permissions.
  - confidence: 0.9
