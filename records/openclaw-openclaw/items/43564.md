## What This Changes

## Best Possible Solution

Define a narrow opt-in ACP skill-context contract that reuses existing skill snapshots and filters, preserves ACP isolation, and documents redaction/token behavior before adding auto/default injection.

## Solution Assessment

Unclear until maintainer product and security scoping is done. An explicit opt-in skill list looks like the safest starting point; automatic relevance and default injection should wait for a broader policy decision.
