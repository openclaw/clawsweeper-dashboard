Add optional skill context injection for ACP `sessions_spawn`, including explicit, automatic, or default skill selection for spawned external harness sessions.

## Best Possible Solution

Define a narrow opt-in ACP spawn skill-context contract that reuses existing skill snapshot/filtering, preserves ACP isolation, documents redaction and token behavior, and starts with explicit skill lists before broader auto/default injection.
