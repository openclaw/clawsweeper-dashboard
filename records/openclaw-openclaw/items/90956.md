- **[P1] Filter quoted context before logging it:** `extensions/whatsapp/src/inbound/monitor.ts:1011-1012`
  - body: This logs `enriched.replyContext` directly into the persistent inbound log, but WhatsApp later runs quoted context through `contextVisibility`/allowlist filtering before building agent and plugin context.
