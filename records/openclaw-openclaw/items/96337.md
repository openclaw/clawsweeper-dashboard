## What This Changes

The issue reports that OpenClaw v2026.6.10 changed `anthropic-vertex` from proxy-like to native routing and now drops pure text Claude replies as non-visible-output while tool-call-first replies can recover.

## Best Possible Solution

Fix and test native Anthropic Vertex text streaming in the owning provider/parser path, preserving tool-call behavior; use a temporary route fallback only if live/dependency proof shows the native path cannot be made reliable for a patch release.
