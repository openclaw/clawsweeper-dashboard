- **[P2] Reject objectives that cannot reach realtime instructions:** `extensions/voice-call/src/manager/outbound.ts:200`
  - body: The new `objective` is documented as private context for realtime task calls, but `initiateCall` now stores it for every outbound call before proving that the selected path can use realtime instructions.
