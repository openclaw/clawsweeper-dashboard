Full review comments:

- **[P1] Reject disabled Feishu accounts before client creation:** `extensions/feishu/src/chat.ts:145-156`
  - body: The explicit/default account path can resolve a configured account with `enabled: false`, but this code checks only the account's `tools.chat` flag before creating the client. That bypasses the account-level disable flag for chat actions.
  - confidence: 0.94
- **[P2] Preserve the member_info chat action:** `extensions/feishu/src/chat-schema.ts:3`
  - body: Current main exposes `member_info` in the schema, handler, and tests, but this branch's action enum keeps only `members`, `info`, and `list`. Adding `list` should not remove the existing member lookup action.
  - confidence: 0.91
- **[P2] Keep the current TypeBox package import:** `extensions/feishu/src/chat-schema.ts:1`
  - body: The Feishu package depends on `typebox`, while this branch imports `@sinclair/typebox`. Keep the current import after rebase so the plugin does not rely on an undeclared package.
  - confidence: 0.88
- **[P2] Keep the local Feishu runtime API import:** `extensions/feishu/src/chat.ts:2`
  - body: Current Feishu production files use the plugin-local `../runtime-api.js` barrel, and current SDK entrypoints do not export `openclaw/plugin-sdk/feishu`. This stale import would break the packaged plugin contract after rebase.
  - confidence: 0.87
- **[P2] Preserve Feishu diagnostic error formatting:** `extensions/feishu/src/chat.ts:183-184`
  - body: Current main formats chat errors with `formatFeishuApiError(...includeNestedErrorLogId...)`; this branch returns only `err.message`, dropping Feishu log IDs and troubleshooters operators need.
  - confidence: 0.88
- **[P2] Align chat list wording with Feishu group listing:** `extensions/feishu/src/chat.ts:139-140`
  - body: The PR describes recent chats covering groups and DMs, but the implementation uses the same `im.chat.list` API current code treats as group-directory listing. Restrict the contract to groups or add a real DM path before exposing this broadly.
  - confidence: 0.83
