  - confidence: 0.91

  - confidence: 0.87

  - body: The PR describes listing recent chats including groups and DMs, but `im.chat.list` is the Feishu group-list API and excludes P2P single chats. Either restrict the contract and wording to groups or implement the real message-history/DM path.
