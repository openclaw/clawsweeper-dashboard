## What This Changes

Requests fixing Anthropic setup-token/paste-token onboarding so the token flow is clearer, persists reliably, and works across multi-agent installs without manual auth-profile copying.

## Best Possible Solution

Keep the issue open while maintainers review PR #52912 against the current auth inheritance contract.
