The report asks OpenClaw to prevent launchd-managed Gateway startup failures from entering rapid restart loops so macOS users can inspect the failure.

## Best Possible Solution

Adopt a reviewed macOS launchd failure policy across plist generation, service audit, tests, docs, and changelog that prevents rapid startup-failure loops while preserving deliberate managed restarts.
