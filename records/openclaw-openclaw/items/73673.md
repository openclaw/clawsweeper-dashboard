## What This Changes

The issue requests a macOS LaunchAgent supervision policy that avoids rapid Gateway restart loops after startup crashes so failures are easier to debug.

## Best Possible Solution

A coordinated macOS launchd failure policy should update plist generation, service audit, restart and stop semantics, tests, docs, and changelog while preserving intentional managed Gateway restarts.
