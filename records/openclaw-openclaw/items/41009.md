This PR should stay open because current main still lacks the Tailscale Serve/Funnel bind migration, but the branch is not merge-ready: it targets the retired migration surface, weakens the explicit `auto` bind validation invariant, and still lacks inspectable real behavior proof.

## Best Possible Solution
