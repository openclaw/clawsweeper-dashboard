Keep open. Current main and the latest release still reject the reported Feishu config before runtime: Feishu accepts `streaming` only as a boolean, omits `contextVisibility` from the strict config contract, and has no Feishu doctor compatibility normalizer to repair the legacy shape.

## What This Changes

## Best Possible Solution

Add Feishu-owned config/schema/manifest/docs support for runtime-supported `contextVisibility`, plus a doctor compatibility migration for legacy scalar Feishu streaming values, with focused regression tests.

## Solution Assessment
