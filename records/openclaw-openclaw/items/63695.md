  - confidence: 0.86
- **[P2] Finalize retained active previews before returning:** `extensions/telegram/src/lane-delivery-text-deliverer.ts:581-584`
  - body: When the active preview path returns `preview-retained`, it sets the retain flag but does not mark the active preview complete. That leaves the lane lifecycle transient, so later streaming can still treat a retained final as editable or clearable.
  - confidence: 0.8
