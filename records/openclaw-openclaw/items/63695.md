An earlier `deleteIfUnused: true` preview with score 0 is then neither tracked nor deleted, leaving a stale Telegram preview visible indefinitely.
  - confidence: 0.88
- **[P2] Finalize the lane when retaining the active preview:** `extensions/telegram/src/lane-delivery-text-deliverer.ts:581-584`
  - body: When the active preview is selected and `tryUpdatePreviewForLane` returns `retained`, this branch sets the retain flag but never marks the active preview complete. The lane stays transient, so later streaming can treat the retained final as still editable or clearable instead of finalized.
  - confidence: 0.78
