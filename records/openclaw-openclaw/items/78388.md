## What This Changes

The branch adds a changelog entry, CLI install regressions, and a helper check intended to suppress non-actionable hook-pack fallback text for missing `openclaw.hooks`.

## Best Possible Solution

Normalize the hook fallback manifest error against the actual `String(Error)` output, add a regression using `Error: package.json missing openclaw.hooks`, and keep detailed hook validation failures such as missing `HOOK.md`.
