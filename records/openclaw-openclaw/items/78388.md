## Best Possible Solution

Refresh the branch and keep the formatter-layer fix, but normalize the missing-hook-manifest fallback shape, preserve other hook validation details, and remove the release-owned changelog edit before merge.

## Solution Assessment

No. The formatter is the right layer, but this patch should match the real Error-prefixed missing-manifest shape, keep actionable hook validation failures visible, and avoid editing release-owned changelog content.
