Full review comments:

- **[P1] Keep managed skills inside the configured root:** `src/agents/skills/workspace.ts:216-224`
  - body: This returns `candidateRealPath` for every `openclaw-managed` escape after `isPathInside()` fails. A symlinked managed skill directory or `SKILL.md` under the managed root would now be loaded from outside that root, so the WSL tolerance needs to distinguish the WSL alias false positive from a real escape.
  - confidence: 0.93
- **[P2] Only ignore missing targets on forced remove:** `src/agents/sandbox/fs-bridge-mutation-helper.ts:236`
  - body: Changing the remove handler from `FileNotFoundError` to `OSError` means the default forced remove path can report success for failures like `ENOTEMPTY`, `EACCES`, or `EPERM` while leaving the entry behind. Force should ignore missing targets only and propagate other removal failures.
  - confidence: 0.9
- **[P2] Preserve later final text during concurrent close:** `extensions/feishu/src/streaming-card.ts:326-329`
  - body: When `closePromise` already exists, a concurrent `close(finalText)` returns the first promise and drops the later final text. If the first close was started without the final response, the card never receives that content; merge or retain the latest non-empty final text before closing.
  - confidence: 0.78
