  - body: This returns `candidateRealPath` whenever an `openclaw-managed` path fails containment. A symlinked managed skill directory or `SKILL.md` can then load from outside the managed root, so the WSL tolerance needs to prove an intended source-scoped exception instead of bypassing the guard.
  - confidence: 0.93
- **[P2] Propagate non-missing forced remove failures:** `src/agents/sandbox/fs-bridge-mutation-helper.ts:236`
  - body: Changing the remove handler to catch `OSError` makes force mode report success for `ENOTEMPTY`, `EACCES`, `EPERM`, and other failures while leaving the entry behind. Force semantics should ignore missing targets only and raise real removal failures.
  - confidence: 0.9
- **[P2] Preserve final text across concurrent close calls:** `extensions/feishu/src/streaming-card.ts:326-329`
  - body: When `closePromise` already exists, a later `close(finalText)` returns the first promise and drops its own `finalText`. If the first close had no final response, the card never receives the later user-facing text.
  - confidence: 0.78
