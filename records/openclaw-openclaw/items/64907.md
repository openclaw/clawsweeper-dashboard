## What This Changes

This PR adds a synchronous text-only `tool_result_before_model` Plugin SDK hook, wires it into embedded Pi tool-result handling, updates docs/API baselines, and adds hook/runner tests.

## Best Possible Solution

Land a rebased, maintainer-approved Plugin SDK contract that preserves bundled middleware and explicitly documents trust, ordering, bounds, failure behavior, and how async/full-result mutation remains tracked.

## Solution Assessment

No for this branch as-is. A narrow synchronous text-only hook may be viable, but this PR must be rebased, must preserve current bundled middleware, and needs Plugin SDK/security trust-boundary approval before merge.
