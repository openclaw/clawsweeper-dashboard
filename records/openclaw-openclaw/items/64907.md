This PR adds a public synchronous text-only `tool_result_before_model` Plugin SDK hook, wires it into Pi embedded tool-result processing, updates docs/API baselines, and adds hook/runner coverage.

Land only a rebased, maintainer-approved Plugin SDK contract that preserves bundled middleware and explicitly defines trust, ordering, failure, bounds, transcript, and async-follow-up semantics.
