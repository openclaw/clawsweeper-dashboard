## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Register real governance hooks before startup activation:** `extensions/openclaw-governance/src/index.ts:65-70`
  - body: `register` currently does no work while the manifest startup-loads the plugin and the PR advertises AIBOM, DLP, and cost-ledger behavior. Since `activation` is only planner metadata, an enabled plugin will import successfully but never record real agent turns, scan outbound payloads, or write ledger rows. Wire supported runtime hooks or remove the startup/plugin claims until the hook surface is approved.
  - confidence: 0.95
- **[P1] Avoid persisting raw DLP matches by default:** `extensions/openclaw-governance/src/store/sqlite.ts:96-108`
  - body: The DLP table requires a plaintext `matched_snippet`, so once the scanner is wired the database can retain SSNs, card numbers, emails, or other detected secrets even for log/warn/redact modes. Store only a redacted preview, category, offsets, and/or hashes by default, with any raw capture behind an explicit trusted content-capture setting.
  - confidence: 0.92
- **[P2] Allow cache-token rate overrides through config:** `extensions/openclaw-governance/openclaw.plugin.json:81-84`
  - body: The ledger now prices `cacheReadUsd` and `cacheWriteUsd`, but the strict manifest schema only allows `inputUsd` and `outputUsd`, and `normalizeGovernanceConfig` drops cache-specific fields. Operators cannot set their real cache rate card through `openclaw config.patch`, so cached calls will be mispriced whenever defaults do not match their provider terms.
  - confidence: 0.87
- **[P1] Remove or authorize the dependency graph change:** `extensions/openclaw-governance/package.json:12-14`
  - body: Adding this package manifest and lockfile importer triggers the repository dependency graph guard, which blocks external PR dependency graph changes until an admin/security approval covers the exact head SHA. Either avoid the workspace graph change by externalizing the plugin, or get the required approval before this can merge.
  - confidence: 0.9
