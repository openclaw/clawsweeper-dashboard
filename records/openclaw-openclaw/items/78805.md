## What This Changes

Issue requests replacing synchronous gateway-adjacent keychain and file I/O with async paths to prevent macOS event-loop stalls and Telegram/WebSocket timeouts.

## Best Possible Solution

Land a focused, profiled gateway/auth fix that moves confirmed hot macOS keychain/file reads off the main event loop while preserving credential cache, expiry, and auth-store semantics.
