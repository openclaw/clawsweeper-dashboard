## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Preserve non-cron schedules when saving edits:** `apps/linux/src/section_cron.c:274-275`
  - body: The edit save path always sends `schedule_kind = "cron"`, so saving an existing `every` or `at` job rewrites its schedule shape and can fail validation or change when the job runs. Preserve the existing schedule variant unless the user explicitly changes it.
  - confidence: 0.91
- **[P2] Resolve session logs by session ID:** `apps/linux/src/section_sessions.c:266`
  - body: This local transcript path uses `<storeDir>/<session-key>/transcript.jsonl`, but the gateway resolves transcripts by `sessionId` and optional `sessionFile` under the sessions directory. The Log action will miss real local transcripts and can fail when dashboard fallback is unavailable.
  - confidence: 0.86
- **[P2] Bracket IPv6 hosts before composing URLs:** `apps/linux/src/gateway_config.c:1153-1155`
  - body: Interpolating `config->host` directly into `scheme://host:port` produces malformed URLs for IPv6 literals such as `::1`. Bracket IPv6 literals before formatting dashboard and transcript URLs.
  - confidence: 0.85
