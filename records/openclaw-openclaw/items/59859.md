## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Preserve existing cron schedule variants:** `apps/linux/src/section_cron.c:274-275`
  - body: The edit path always sends `schedule.kind: "cron"` with an `expr`. Current cron jobs may use `kind: "at"` or `kind: "every"`, so saving an existing non-cron job after changing unrelated fields can rewrite the schedule shape or fail validation. Carry the original schedule variant through the dialog or omit schedule from the patch unless the user changes it.
  - confidence: 0.93
- **[P2] Keep session-scoped cron targets on edit:** `apps/linux/src/test_seams.c:47`
  - body: `session:<id>` is a valid gateway cron target, but the edit UI maps unknown targets to the default combo entry and later saves that as `isolated`. Editing a session-scoped job can silently retarget it away from the intended session even when the user changes only metadata.
  - confidence: 0.89
- **[P2] Resolve local session logs from session IDs:** `apps/linux/src/section_sessions.c:265-266`
  - body: This builds local transcript paths as `<storeDir>/<session-key>/transcript.jsonl`, while the gateway stores and resolves transcript files by `sessionId` and optional `sessionFile`. Normal local transcripts will be missed and the UI will fall back to the dashboard even when the file exists locally.
  - confidence: 0.88
- **[P2] Bracket IPv6 hosts before composing URLs:** `apps/linux/src/gateway_config.c:1153-1155`
  - body: Interpolating an IPv6 literal directly into `scheme://host:port` produces invalid URLs such as `http://::1:18789/`. Bracket IPv6 literals before building health, dashboard, and handoff URLs so valid custom-bind setups keep working.
  - confidence: 0.86
- **[P2] Allow clearing skill secrets and env values:** `apps/linux/src/section_skills.c:133-134`
  - body: The dialog returns early on an empty value, but `skills.update` treats empty `apiKey` or env strings as the deletion/cleanup signal. As written, users cannot remove stale skill secrets or env overrides from the Linux app once set.
  - confidence: 0.84
