- **[P1] Register Linux in the canonical protocol package:** `src/gateway/protocol/client-info.ts:13`
  - body: The Linux app sends `client.id = "openclaw-linux"`, but this PR only adds that id under `src/gateway/protocol`.   - body: The verbose combo sends `none`/`low`/`high` through `sessions.patch`, but the server parser accepts `off`/`on`/`full`. Users changing verbosity from the Linux app will get a validation failure instead of updating the session.
  - confidence: 0.93
- **[P1] Preserve existing cron schedule and target variants:** `apps/linux/src/section_cron.c:270-276`

  - confidence: 0.92
- **[P3] Add Linux app labeler routing:** `apps/linux/README.md:1`
