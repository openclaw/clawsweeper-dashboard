## What This Changes

The issue reports that Anthropic Vertex OAuth token fetches fail on Node 25+ because the bundled Google auth dependency chain still uses google-auth-library 9.x and gaxios 6.x.

## Best Possible Solution

Ship a maintainer-reviewed Anthropic Vertex provider fix that removes or bypasses the google-auth-library 9/gaxios 6 token transport path, refreshes provider package metadata and shrinkwrap, preserves current stream-runtime behavior, and proves a Node 25+/26 ADC Vertex turn when feasible.

## Solution Assessment
