## Summary

Keep this PR open. Current main still has a real gateway-to-memory incremental indexing gap for reset/deleted transcript archives, and the PR’s diff is a narrow implementation with no discrete correctness finding from this review.

## What This Changes

The PR emits transcript update events when session files are archived, exposes existing archive classifiers through the memory host SDK, bypasses session delta thresholds for reset/deleted archives, and adds regression tests plus a changelog entry.

## Best Possible Solution
