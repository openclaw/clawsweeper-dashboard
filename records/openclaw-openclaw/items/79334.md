## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Validate structured output against jsonSchema:** `extensions/codex/media-understanding-provider.ts:374`
  - body: `jsonSchema` is currently only inserted into the prompt, and successful responses are accepted after `JSON.parse` alone. That means a provider can return valid JSON missing required fields or with the wrong types and plugins still get `contentType: "json"` plus `parsed` as if the schema was honored. Please validate `parsed` against `jsonSchema` with the existing JSON Schema helper, or remove/rename the field so it is not a validation contract.
  - confidence: 0.88
