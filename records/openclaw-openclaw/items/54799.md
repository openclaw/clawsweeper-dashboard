## What This Changes

Request to add declarative scheduled/event trigger metadata in SKILL.md frontmatter, register schedule-triggered skills with cron, emit lifecycle events, and expose last-triggered or never-triggered skill audit status.

## Best Possible Solution

Define the permissioned automation contract first, then implement one coordinated path for metadata parsing, scheduler registration, event/audit state, CLI/docs output, and regression tests.

## Reproduction Assessment

## Solution Assessment
