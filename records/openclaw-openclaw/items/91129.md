## What This Changes

## Best Possible Solution

Keep the issue open and route it with the WeChat startup cluster; the durable fix should make the external plugin's account discovery or repair path reliable and make OpenClaw surface a clear diagnostic when a configured channel plugin exposes no runnable accounts.
