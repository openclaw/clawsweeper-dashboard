## What This Changes

The issue requests a Control UI Linked Apps hub that can discover installed channels, manage connections, and run QR-based login or pairing flows starting with WeChat.

## Best Possible Solution

Design a generic, plugin-owned Linked Apps flow that reuses channel status/catalog metadata, adds only the missing Control UI/gateway/SDK seams, and keeps channel-specific QR login behavior inside each plugin with explicit operator consent.
