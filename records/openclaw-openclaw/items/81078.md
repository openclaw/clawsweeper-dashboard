## What This Changes

The issue reports that WhatsApp block streaming can drop the complete final assistant reply after a partial streamed payload was delivered.

## Best Possible Solution

Change final-payload filtering so successful block streaming suppresses only payloads proven delivered by the block pipeline or direct block keys, while preserving the existing media-only follow-up behavior.
