Add cron observability for execution attempts, skipped/failure warnings, cron health probing, and `openclaw cron list` warnings.

## Best Possible Solution

Define the cron observability contract, then add bounded scheduled/manual outcome logs, skipped-run warnings, cron-aware probe/readiness reporting, and list warnings while preserving existing counters, failure alerts, task records, diagnostics, and run history.

## Solution Assessment
