## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

No for current head. A narrow usage-only stream is a strong direction, but provider-owner scoping no longer supports the standalone ledger/report plugin use case described in the PR discussion unless maintainers explicitly choose that narrower scope.
