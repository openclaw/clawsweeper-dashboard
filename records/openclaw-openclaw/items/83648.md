- **[P1] Honor the existing provider auth contract:** `src/cron/isolated-agent/model-preflight.runtime.ts:161-202`
  - body: This new resolver treats any string `apiKey` as a bearer token and bypasses the existing provider auth/request-auth helpers. Please build the preflight headers from the same sanitized provider auth/request config contract as the model request.
