## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.92

Full review comments:

- **[P1] Reject symlinked transcripts before opening:** `src/agents/cli-runner/session-history.ts:50-54`
  - body: This path uses `statSync()` and then `SessionManager.open()` on the transcript, which follows symlinks. A symlink planted at the expected session file can redirect hook history loading to arbitrary local files or blocking special files; current main fixes this with `lstat` plus realpath containment before parsing.
  - confidence: 0.93
- **[P1] Gate raw conversation hooks:** `src/agents/cli-runner.ts:64-72`
  - body: The new CLI hook event includes the system prompt, user prompt, and transcript history, but the branch does not add the current-main `allowConversationAccess` gate for non-bundled plugins. That makes raw conversation data available to any registered lifecycle hook and crosses the plugin trust boundary.
  - confidence: 0.91
- **[P2] Honor configured session stores:** `src/agents/cli-runner/session-history.ts:29-35`
  - body: `resolveSessionFilePathOptions` is called without `storePath: params.config?.session?.store`, so deployments using a custom session store resolve hook history against the default state directory and silently lose prior transcript context.
  - confidence: 0.9
- **[P2] Load history only for active hooks:** `src/agents/cli-runner.ts:57-63`
  - body: `runPreparedCliAgent` loads and parses session history before checking whether any lifecycle hook needs it. That adds synchronous transcript I/O to the default CLI path for users without those hooks; current main avoids this by checking hook presence first.
  - confidence: 0.86
- **[P2] Skip empty llm_output events:** `src/agents/cli-runner.ts:133-144`
  - body: `runAgentHarnessLlmOutputHook` fires even when the CLI returned only empty output, producing an llm_output event with no assistant text. The embedded runner and current main only emit this hook when there is a real assistant/source-reply payload.
  - confidence: 0.84
