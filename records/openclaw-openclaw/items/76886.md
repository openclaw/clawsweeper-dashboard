The issue reports that Codex app-server final assistant replies can be mirrored to transcript JSONL without becoming the live branch leaf, so a later user turn may orphan the prior reply.

Add a focused Codex app-server two-turn regression for the active transcript parent chain, then make final reply persistence advance the branch through the app-server transcript path without reintroducing WebChat agent-run mirroring.
