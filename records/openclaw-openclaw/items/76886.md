## What This Changes

The issue reports that Codex app-server mirrored final assistant replies can be orphaned from the active transcript branch when a later message is appended.

## Best Possible Solution

Add a focused consecutive-turn Codex app-server regression that asserts active-branch parent links, then make final reply persistence advance the branch before later user appends.
