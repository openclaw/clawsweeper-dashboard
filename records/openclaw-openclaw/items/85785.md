## What This Changes

The PR wraps plugin-registered gateway RPC handlers so non-undefined return values are sent as successful responses, updates descriptors to use the wrapped handler, adds loader regressions, and adds a changelog entry.

## Best Possible Solution

Land only after maintainers accept return-value completion as plugin API behavior, align the SDK type/docs/contract tests, and add redacted running-gateway proof for openclaw gateway call.
