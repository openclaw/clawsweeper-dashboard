This PR adds a `captionedFinalText` channel TTS capability, opts Telegram into it, and updates ACP/non-ACP final-mode TTS dispatch, tests, docs, and SDK baseline so text can be deferred into voice-note captions with fallback text.

## Best Possible Solution
