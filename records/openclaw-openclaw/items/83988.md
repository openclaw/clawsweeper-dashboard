## What This Changes

The PR adds a `captionedFinalText` channel TTS capability, opts Telegram into it, and changes ACP and non-ACP final-mode TTS dispatch to defer or suppress visible text so Telegram can send one captioned voice note with fallback coverage and docs.

## Best Possible Solution

Land only after maintainers accept the `captionedFinalText` channel capability contract and current-head Telegram proof covers always mode, tagged directive mode, media preservation, and fallback behavior.
