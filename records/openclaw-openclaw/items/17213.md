## Summary

Keep open. Current main has adjacent strict-agentic and liveness safeguards, but the public agent/Gateway contract still exposes legacy lifecycle start/end/error snapshots rather than the requested atomic begin/exec/commit-or-abort transaction with structured abort blockers.

## What This Changes

## Best Possible Solution
