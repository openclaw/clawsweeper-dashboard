## Summary

## What This Changes

## Best Possible Solution

Define an additive Gateway/runtime lifecycle contract that carries begin intent, execution evidence, mandatory commit or abort, and structured blockers through schemas, AgentEvent types, runtime emitters, wait results, client/plugin-visible surfaces, docs, and regression tests.
