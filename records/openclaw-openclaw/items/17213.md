## Summary

Keep open. Current main has adjacent strict-agentic, liveness, and terminal-snapshot safeguards, but it still lacks the requested atomic begin/exec/commit-or-abort lifecycle contract with structured abort blockers across the event bus, Gateway protocol, runtime lifecycle, session state, and wait path.
