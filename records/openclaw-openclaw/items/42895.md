## Best Possible Solution

Land a rebased Termux `PREFIX` fallback in `src/infra/home-dir.ts` with the existing regression tests and docs after redacted real Termux startup proof is added.

## Solution Assessment

Yes, the PR direction is the narrow maintainable fix: add a guarded Termux fallback below `HOME` and `USERPROFILE`, and share it with tilde expansion. It still needs real Termux proof and fresh checks before merge.
