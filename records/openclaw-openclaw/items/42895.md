The PR adds a Termux-only `PREFIX` fallback to home-directory resolution, regression tests for precedence and tilde expansion, environment docs, and a changelog entry.

## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Yes, the proposed direction is the narrow maintainable fix: add a Termux-only fallback below `HOME`/`USERPROFILE` and share it with tilde expansion. The branch still needs real Termux proof, rebase, and fresh checks before merge.
