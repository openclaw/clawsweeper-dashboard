## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Expand home-relative skill paths before filtering:** `src/commands/doctor-session-snapshots.ts:172-174`
  - body: Persisted `skillsSnapshot.prompt` is built from prompt-facing skill paths that are compacted to `~/...` for home roots. This guard drops every non-absolute cached path before classifying it, so an old user-scoped bundled install such as `~/.openclaw/.../skills/github/SKILL.md` is invisible even though install-method changes are part of the linked acceptance criteria. Expand home-relative paths before deciding whether the cached path is stale bundled metadata.
  - confidence: 0.88
- **[P2] Scan resolvedSkills before normalizing the store:** `src/commands/doctor-session-snapshots.ts:273`
  - body: `noteSessionSnapshotHealth()` loads each sessions.json through `loadSessionStore()`, whose normalization strips `skillsSnapshot.resolvedSkills` from legacy persisted entries. That means the real doctor path cannot detect stale paths that exist only in the claimed `resolvedSkills` source. Read the raw JSON for this diagnostic or add a preserve option before scanning that field.
  - confidence: 0.91
