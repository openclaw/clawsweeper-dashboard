## Summary

Keep open. The PR is a focused and useful doctor diagnostic with sufficient fixture-backed proof, but the current patch still has two source-proven false negatives in the scanner and the protected `maintainer` label means it should not be cleanup-closed.

## What This Changes

The PR adds a read-only `openclaw doctor` health contribution that scans session stores for stale bundled-skill paths in cached session snapshot metadata.

## Best Possible Solution

Keep the read-only doctor contribution shape, but fix the scanner to expand home-compacted paths and inspect raw legacy `resolvedSkills` metadata before merge.

## Reproduction Assessment

Yes. Source inspection and the linked proof artifacts show current main lacks this doctor diagnostic, and the remaining false negatives are reproducible from the scanner and session-store contracts without running the suite.

## Solution Assessment

No. A read-only doctor contribution is the right maintainable direction, but the current implementation still misses home-compacted prompt paths and legacy `resolvedSkills`-only stale metadata.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Expand home-relative snapshot paths before classifying them:** `src/commands/doctor-session-snapshots.ts:172-174`
  - body: Persisted `skillsSnapshot.prompt` can contain prompt-facing skill paths compacted to `~/...`. This guard drops every non-absolute cached path before classification, so an old user-scoped bundled install path under the home directory is invisible even though install-method changes are part of the linked acceptance criteria.
  - confidence: 0.88
- **[P2] Scan resolvedSkills before normalizing the store:** `src/commands/doctor-session-snapshots.ts:273`
  - body: `noteSessionSnapshotHealth()` loads each `sessions.json` through `loadSessionStore()`, whose normalization strips `skillsSnapshot.resolvedSkills` from legacy persisted entries. That makes the real doctor path unable to detect stale paths that exist only in the claimed `resolvedSkills` source; read raw JSON for this diagnostic or add a preserve option before scanning that field.
  - confidence: 0.91
