## What This Changes

The PR changes the agent runner callback gate so an explicit run-level `reasoningLevel: "stream"` passes an `onReasoningStream` hook, and adds a focused regression test.

## Best Possible Solution

Close this PR as superseded and route the remaining no-callback reasoning-stream fix through https://github.com/openclaw/openclaw/pull/79687, including its docs cleanup before merge.

## Reproduction Assessment

Yes. Source inspection gives a high-confidence path: current main forwards run-level `reasoningLevel` but suppresses stream handling when no `onReasoningStream` callback is supplied, and gateway clients do not supply that callback.

## Solution Assessment

No. This PR can make the runner pass a sentinel callback, but the narrower maintainable fix is the subscriber-boundary callback decoupling already tracked in https://github.com/openclaw/openclaw/pull/79687.
