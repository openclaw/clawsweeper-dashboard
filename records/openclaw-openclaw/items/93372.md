- **[P1] Preserve Telegram chunk mode:** `extensions/telegram/src/send.ts:722-725`
  - body: The new legacy text planner calls `markdownToTelegramChunks` without resolving or passing `channels.telegram.chunkMode`, while current main feeds the resolved mode into the Telegram splitter.
