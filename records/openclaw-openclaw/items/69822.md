## What This Changes

The PR adds Gateway `socket.drain` teardown broadcasts for reset/delete/new flows, routes them to session-event and per-session-message subscribers, registers the event for read-scope filtering and discovery, and expands focused Gateway tests.

## Best Possible Solution

Land a maintainer-approved Gateway protocol update that documents `socket.drain`, keeps teardown routing centered in session mutation handling, and includes redacted live WebSocket proof for reset/delete delivery.

## Reproduction Assessment

## Solution Assessment
