  - body: `createGatewayBroadcaster` closes slow sockets whenever `dropIfSlow` is false or omitted. That compatibility change needs either a different implementation or explicit maintainer-approved contract/docs before merge.
  - confidence: 0.94

  - body: For selected global sessions, the PR targets subscribers through `agent:<id>:global`, but the emitted drain payload only contains `sessionKey: "global"`. A WebSocket client can subscribe to multiple global-agent streams on the same connection, so without `agentId` it cannot know which global session to evict and may flush the default or wrong agent view.
  - confidence: 0.88

Please document the event before making it discoverable so downstream clients have a stable contract.
