## Summary

## What This Changes

The PR adds shared gateway TLS fingerprint resolution, forwards the resolved fingerprint into ACP, MCP, and operator approval GatewayClient construction, and adds targeted regression coverage plus a changelog entry.

## Best Possible Solution

Land the focused bootstrap propagation after normal maintainer review, then use the merged fix as the resolution path for https://github.com/openclaw/openclaw/issues/41740 if maintainers agree it covers that report.

## Solution Assessment
