## Summary

Keep open. The prior persistence blocker was addressed, but the branch still changes Homebrew auto-update from automatic apply to manual `brew upgrade` while the surrounding user guidance can still point users at the normal `openclaw update` path.

## What This Changes

The PR adds Homebrew package-root detection, skips gateway auto-update for detected Homebrew installs with a `brew upgrade openclaw` hint, persists skip notification state, and adds tests plus a changelog entry.

## Best Possible Solution
