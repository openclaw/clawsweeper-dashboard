## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.78

Full review comments:

- **[P2] Preserve the bounded binary probe:** `src/infra/tailscale.ts:40`
  - body: `execFile`'s timeout sends the kill signal and settles from the child `close` handler. If the candidate `tailscale` binary traps `SIGTERM` or never closes, this direct await may never settle, while current main returns `false` from the outer timeout after 3 seconds. Keep a wall-clock bound while also owning child cleanup so the setup probe cannot hang on the slow or stuck binary path this PR is hardening.
