## Summary

Keep open: the underlying stop/reset gap is still present on current main, but this external PR is not merge-ready because it lacks real behavior proof and has blocking correctness findings on the submitted head.

## What This Changes

The PR makes subagent stop propagation async, cancels ACP-backed child sessions during stop/reset, persists child aborted state, updates `/stop` requester fallback behavior, and adds abort regression coverage.

## Best Possible Solution

Land a rebased implementation that propagates stop/reset to active ACP-backed children through a bounded cancellation path, preserves child abort state, and includes focused tests plus redacted runtime proof.

## Reproduction Assessment

Yes, at source level. Current main stops subagent queues and reply/embedded runs but does not cancel ACP-backed child sessions, and gateway reset calls that helper before bounded target-session ACP cleanup.

## Solution Assessment

No, not as submitted. The direction is right, but directly awaiting child ACP cancellation can hang reset/delete and the submitted test misses the latest-run mock needed on the PR head.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Bound ACP child cancellation during stop cascade:** `src/auto-reply/reply/abort.ts:199-203`
  - body: The new child-stop path awaits `acpManager.cancelSession` directly. On active turns, `cancelSession` awaits `activeTurn.cancelPromise` without a local timeout, and reset now awaits this cascade, so a stalled ACP backend can hang reset/delete instead of using the existing bounded cleanup flow.
  - confidence: 0.9
- **[P2] Mock the latest child run in the ACP cascade test:** `src/auto-reply/reply/abort.test.ts:600-613`
  - body: On PR head, `stopSubagentsForRequester` drops listed runs when `getLatestSubagentRunByChildSessionKey` returns null, but this test only mocks `listSubagentRunsForRequester`. `cancelSession` is never reached and `stoppedSubagents` stays 0.
  - confidence: 0.86
