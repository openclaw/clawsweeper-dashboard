- **[P2] Preserve timestamped OAuth refresh failures:** `src/commands/status-all/gateway.ts:75-76`

  - body: Returning `null` here suppresses every unrecognized gateway stderr/stdout tail line, not only wrapper noise. `status --all` and daemon status use this helper as a local failure hint, so a new startup failure outside `GATEWAY_LOG_ERROR_PATTERNS` would disappear unless the PR preserves a tested fallback policy.
  - confidence: 0.78
