## What This Changes

Fix Feishu P2P direct-message thread replies so bot responses stay inside the thread instead of being sent as standalone DM messages.

## Best Possible Solution

Land one Feishu plugin patch, preferably https://github.com/openclaw/openclaw/pull/73972 if refreshed and review-clean, that treats direct events with `thread_id` as thread replies while preserving root-id-only P2P quote replies and existing group behavior.
