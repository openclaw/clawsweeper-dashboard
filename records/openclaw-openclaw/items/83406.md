## Best Possible Solution

Normalize MiniMax Anthropic-compatible `reasoning_content` deltas only when MiniMax thinking is explicitly enabled, preserving the default disabled-thinking mitigation and adding focused regression coverage for both streamed reasoning and final text.

No live high-confidence reproduction was established. The source path is reproducible: MiniMax explicit thinking can emit OpenAI-style `reasoning_content`/`content` deltas, while current main only consumes that shape behind the Xiaomi-native replay gate.

## Solution Assessment

Yes, a narrow parser/transport fix is the right direction. The safer solution is to handle MiniMax explicit-thinking deltas without changing generic Anthropic behavior or the default MiniMax thinking-disabled safety path.
