  - body: `resolveAgentWorkspaceDir` sends agents with `multipleWorkspaces` to `workspace-composite-<id>`, but agent command and isolated cron setup still call `ensureAgentWorkspace` without passing the workspace list.   - confidence: 0.92

  - confidence: 0.86
