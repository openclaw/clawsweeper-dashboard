## Best Possible Solution

## Solution Assessment

Yes in direction, but not merge-ready as submitted. The maintainable fix is to detect user-configured heartbeat prompt sources and skip only the generated workspace-path hint for those prompts while preserving default/internal prompt behavior on current main.
