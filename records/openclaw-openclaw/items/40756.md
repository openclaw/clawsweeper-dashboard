## Summary

## What This Changes

The PR exports the heartbeat hint helper, adds a custom-prompt guard to skip workspace-path hint injection for user-authored heartbeat prompts, and adds focused helper-level coverage.

## Best Possible Solution

Preserve user-authored heartbeat prompts verbatim while retaining workspace HEARTBEAT.md guidance for default and internally generated prompts, with current-main regression coverage and a changelog entry.

## Solution Assessment

Yes, with a rebase. An explicit user-custom-prompt guard is the narrow maintainable fix, but it should be carried through current main's normal and due-commitment append paths and released with changelog coverage.
