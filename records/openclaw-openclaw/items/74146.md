## What This Changes

This PR changes Feishu streaming-card block delivery from delta append to snapshot merging, adds dispatcher regression tests, and adds a changelog entry.

## Best Possible Solution

Land this narrow Feishu plugin fix after normal maintainer review and current CI, while keeping broader final/stale streaming-card bugs on the separate tracker.

## Solution Assessment

Yes. Switching only the Feishu block fallback update to snapshot mode reuses the existing `mergeStreamingText` path, preserves block-only fallback behavior through non-overlap coverage, and stays inside the Feishu plugin boundary.
