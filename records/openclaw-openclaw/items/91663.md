## Best Possible Solution

Keep the cleanup idea, but make deletion stale- or owner-safe, add real backup-create integration and overlapping-run regression coverage, and clarify whether this PR is a narrow pre-run mitigation or the full lifecycle fix for https://github.com/openclaw/openclaw/issues/50442.
