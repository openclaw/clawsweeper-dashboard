Keep open: current main and the latest stable release still ship Matrix runtime code without declaring the Matrix SDK runtime packages on the root npm package, and an open closing PR already tracks the same fix.

## What This Changes

Matrix should continue to start after a host npm update by resolving its Matrix SDK runtime dependencies from the published OpenClaw package.

## Best Possible Solution
