## What This Changes

The issue asks OpenClaw to keep the bundled Matrix channel working after host npm updates by making its runtime packages resolvable from the installed root package.

## Best Possible Solution

Review and land or update https://github.com/openclaw/openclaw/pull/77988 so the root package, lockfile, verifier, and guardrail encode bundled Matrix runtime ownership, including a clear decision on the `sharp` media dependency follow-up.

Yes. Source inspection on current main shows bundled Matrix code is packaged without plugin `node_modules` while the root manifest lacks the runtime packages Matrix imports, and multiple reporter logs confirm the same host-install failure through 2026.5.7.

## Solution Assessment

Yes. While Matrix remains bundled in the root npm package, declaring its runtime packages at the root and tightening the verifier/guardrail is the narrow maintainable fix; the issue should stay open until that PR or an equivalent fix merges.
