## Summary

Keep open: current main and v2026.5.7 still publish bundled Matrix code without root-declared Matrix runtime packages, and https://github.com/openclaw/openclaw/pull/77988 is still open rather than merged.

## What This Changes

Matrix should continue starting after host npm updates by resolving its bundled runtime dependencies from the published OpenClaw package root.

## Best Possible Solution

Review and land or replace https://github.com/openclaw/openclaw/pull/77988 so root dependency ownership, verifier behavior, and guardrails match the bundled Matrix runtime before the next release.

## Reproduction Assessment

Yes. Source inspection gives a high-confidence path: the root package ships Matrix dist output, excludes plugin node_modules, omits the Matrix runtime deps, and Matrix imports those packages; reporters reproduced the same missing matrix-js-sdk failure on 2026.5.4 and 2026.5.6.

## Solution Assessment

Yes. While Matrix remains bundled into the root npm package, declaring its runtime dependencies at the root and updating the verifier/guardrail is the narrow maintainable fix; this issue should stay open until the linked PR or equivalent lands.
