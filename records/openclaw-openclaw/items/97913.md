## What This Changes

The issue requests hardening the Codex dynamic-tool bridge so raw plugin tool payloads are normalized into `AgentToolResult` envelopes before middleware/extensions and Codex response conversion.

## Best Possible Solution

Review and land one canonical Codex dynamic-tool bridge normalizer that envelopes raw plugin payloads before middleware/extensions and content conversion, then close this issue and resolve duplicate candidate PRs.
