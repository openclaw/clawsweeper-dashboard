This PR adds a secrets-audit suppression for known non-secret API-key markers in `openclaw.json` apiKey fields and adds regression tests around marker, all-caps, real-secret, and non-apiKey token behavior.

## Best Possible Solution

Restrict the marker skip to `target.entry.id === "models.providers.*.apiKey"` and add a regression proving a non-model `.apiKey` target still emits `PLAINTEXT_FOUND`.
