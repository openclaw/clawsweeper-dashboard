## Summary

Keep open. The PR still targets a real remaining Instances localization gap, but the branch is conflicting/unrebased, lacks external real behavior proof, and has two P2 i18n correctness issues that should be fixed before merge.

## What This Changes

The branch adds `instances.*` locale keys in English and pt-BR and rewires the Control UI Instances view to call `t(...)` for labels, fallbacks, scopes, last-input, and reason metadata.

## Best Possible Solution

Land a rebased Instances i18n cleanup that localizes the remaining metadata through the English source of truth and generated locale pipeline while preserving shared common helpers.

## Reproduction Assessment

Yes. Source inspection on current main shows the remaining hardcoded Instances metadata path, and the PR diff shows the rebase-sensitive locale and seconds-formatting regressions.

## Solution Assessment

No. The direction is right, but the branch should be rebased onto current i18n keys, preserve `common.secondsAgo` and shared common labels, regenerate every shipped locale bundle plus `.i18n` metadata, and include real UI proof.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Regenerate shipped locales for the new Instances keys:** `ui/src/i18n/locales/en.ts:155-170`
  - body: Adding new `instances.*` keys and changing placeholder names only in English and pt-BR leaves the generated shipped locales behind. The locale parity test expects every shipped locale to match English, so a rebased branch that adds `instances.unknownHost`, `instances.scopesCount`, and `instances.scopesList` needs the generated locale bundles and metadata refreshed too.
  - confidence: 0.91
- **[P2] Keep last input age on common.secondsAgo:** `ui/src/ui/views/instances.ts:60`
  - body: This branch computes `${entry.lastInputSeconds}s ago` before passing it into the localized label. That still embeds English text in non-English Instances metadata; preserve the current `common.secondsAgo` formatting and only localize the surrounding label.
  - confidence: 0.86
