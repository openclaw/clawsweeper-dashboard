  - body: The fallback reaches into `../../memory-core/src/public-artifacts.js` whenever the active helper returns no artifacts and the slot string is `memory-core`.   - confidence: 0.9

Make `cfg` required here or return before provider calls so strict TypeScript and runtime callers cannot pass undefined config through the provider path.
