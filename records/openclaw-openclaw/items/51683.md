This PR adds local ISO/UTC current-time text to `session_status` output, adds a local ISO formatter, and adds pinned-clock regression coverage.

## Best Possible Solution

Expose machine-readable local and UTC current time through the shared status renderer so `/status` and `session_status` keep one ordering/format contract, then refresh the test and include redacted real output proof.
