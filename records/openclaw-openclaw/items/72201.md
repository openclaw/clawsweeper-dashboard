## Summary

Keep open: the latest head still has a P2 queue-stranding regression for internal events whose text matches exec-completion syntax, and merge still needs exact-head real behavior proof for the final shape.

## What This Changes

The PR adds `SystemEvent.audience`, wraps internal system events as runtime context during reply drain, marks isolated-cron awareness internal, updates heartbeat/owner-auth consumers, and adds tests plus a changelog entry.

## Best Possible Solution

Keep the audience primitive only if internal events always reach the wrapped drain regardless of text shape, while user-facing exec completions continue using the heartbeat delivery path.

## Reproduction Assessment

Yes, at source level: enqueue an `audience: "internal"` system event whose text matches `isExecCompletionEvent`, and the PR head's generic drain and heartbeat selectors both skip it. I did not run a live cron/Matrix setup in this read-only review.

## Solution Assessment

No, not as currently patched: the event-level audience direction is plausible, but the implementation must first handle exec-shaped internal awareness without stranding it in the queue. The narrow fix is to exclude exec completions from the generic drain only for user-facing events, or otherwise sanitize/classify the cron-awareness producer before queuing.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.92

Full review comments:

- **[P2] Let internal exec-shaped events reach the wrapped drain:** `src/cron/isolated-agent/delivery-dispatch.ts:420`
  - body: The new producer queues arbitrary delivered cron text as `audience: "internal"`. If that text starts with `Exec finished` or matches the structured `exec completed/failed (...)` form, `selectGenericSystemEvents` still filters it out before `drainFormattedSystemEvents` can wrap it, while the heartbeat selectors now also skip internal events. That entry stays in `peekSystemEventEntries` indefinitely instead of becoming hidden runtime context, so please route internal exec-shaped events through the wrapped drain or prevent this producer from queuing that shape as internal.
  - confidence: 0.94
