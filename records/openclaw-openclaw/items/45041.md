## Best Possible Solution

Decide a guarded product contract: land or replace conservative sanitizer work for high-confidence leak shapes, then decide whether opt-in per-agent filters belong in core config or a `message_sending` plugin.

## Solution Assessment

Unclear. A raw regex filter is flexible but privacy-sensitive and over-stripping-prone; a conservative sanitizer plus an explicit core-vs-plugin config decision is the safer maintainable path.
