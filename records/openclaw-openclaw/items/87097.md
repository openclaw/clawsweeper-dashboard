- **[P1] Drain output before forcing CLI exit:** `src/cli/local-agent-lifetime.ts:75`
  - body: Please do not call `process.exit()` directly here. Node documents that `process.exit()` can terminate before pending stdout/stderr writes complete, and this helper is called after successful CLI commands, so a command that just wrote JSON or log output to a pipe can be cut off. Use a bounded drain of `process.stdout`/`process.stderr` before forcing the process down.
