## Summary

Keep open. Current main does not include the proposed local-agent force-exit/hard-timeout behavior, and the patch is a plausible workaround for the reported CLI hang class, but it is still draft and needs current-main real command proof plus maintainer acceptance of the forced-exit lifecycle tradeoff.

## What This Changes

The PR adds a local-agent lifetime helper plus tests, wiring `src/entry.ts` and `src/index.ts` to arm an explicit `agent --local --timeout` hard timer and force process exit after successful local-agent completion.

## Best Possible Solution

## Reproduction Assessment
