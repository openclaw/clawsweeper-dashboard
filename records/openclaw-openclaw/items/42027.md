  - body: Current main serves browser status from extensions/browser, while this PR adds the diagnostics payload under the removed src/browser route.   - body: When an HTML error page has no title or h1, the fallback strips tags and returns the full remaining body. Large proxy pages can still produce multi-kilobyte browser errors, so the fallback should sanitize entities and cap the text before throwing it.
  - confidence: 0.78
