## What This Changes

Add a validated `sessions_spawn.lane` field defaulting to `subagent`, forward it through native and ACP dispatch, and support compatible per-lane subagent concurrency config with docs and regression tests.

## Solution Assessment

Yes. The narrow maintainable path is an additive lane parameter backed by the existing string-lane queue, with validation, default fallback, and compatible per-lane config.
