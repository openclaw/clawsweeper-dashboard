## What This Changes

The issue requests failure diagnostics that log the last completed Gateway WebSocket handshake phase when a connection times out or closes before completing setup.

## Best Possible Solution

Land a narrow Gateway WS diagnostics change that records a redacted last-completed phase enum on failure without changing protocol behavior or adding config.
