- **[P2] Gate delivery warnings on unprocessed traffic:** `src/commands/status-all/diagnosis.ts:320-327`
  - body: The warning treats any `received > 0 && dispatchStarted === 0` snapshot as broken, but terminal plugin-bound, `before_dispatch`, and `reply_dispatch` paths run before `recordAgentDispatchStarted()` and can return after `recordProcessed(...)`.
