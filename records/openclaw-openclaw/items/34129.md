## What This Changes

The issue asks OpenClaw to stop malformed blank tool-call names from reaching dispatch as empty names that loop with `Tool  not found` for NVIDIA Kimi K2.5.

## Best Possible Solution

Land a revised generic runner-normalization fix that treats blank emitted tool names as invalid and non-dispatchable while preserving id-based recovery and replay pruning.

## Solution Assessment
