## What This Changes

The issue requests official subscription- or OAuth-backed model-provider support in OpenClaw when providers expose sanctioned account or subscription billing paths.

## Best Possible Solution

Keep the existing provider plugin/auth-choice boundary and split concrete official subscription paths into provider-owned issues or PRs with docs, setup labels, entitlement diagnostics, and live proof where available.

## Solution Assessment
