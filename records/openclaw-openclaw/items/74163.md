  - confidence: 0.88

  - body: This PR commits generated Markdown, but the script only writes the file and optionally edits a PR body; there is no check path to fail when `MICROSOFT_TRACKER.md` drifts from the generator. Add a check mode and wire it through the repo's normal script/check surface before merging this as maintainer automation.
  - confidence: 0.84

  - confidence: 0.86
