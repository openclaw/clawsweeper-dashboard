Full review comments:

- **[P1] Honor explicit priority labels before keyword inference:** `scripts/generate-microsoft-tracker.mjs:151-153`
  - body: `priorityFor` scans title plus all labels for broad words like `security` and `auth` before honoring explicit P0-P3 labels, so lower-priority items can be promoted into the P0/P1 queue. The generated tracker already renders a P3-labeled item as P0 because it carries `needs-security-review`; read exact priority labels first and infer only when none exists.
  - confidence: 0.99
- **[P1] Add a stale-output check for the generated tracker:** `scripts/generate-microsoft-tracker.mjs:421-423`
  - body: The script writes `MICROSOFT_TRACKER.md`, but the branch adds no package-level generate/check command that regenerates and compares the committed artifact. Since the source is live GitHub state, the file can become stale immediately and CI has no way to detect it.
  - confidence: 0.95
- **[P2] Emit valid Markdown table delimiters:** `scripts/generate-microsoft-tracker.mjs:362`
  - body: The generator emits `-` as the number-column delimiter cell in issue, PR, and appendix tables. GitHub-flavored Markdown delimiter cells should use at least three hyphens, so a raw generator run can produce tables that do not render consistently unless another formatter rewrites them.
  - confidence: 0.98
- **[P2] Generate a location-neutral update instruction:** `scripts/generate-microsoft-tracker.mjs:280`
  - body: The generated tracker tells readers to edit `this PR body`, but the same text is committed as `MICROSOFT_TRACKER.md`. Use wording that works in both places, or generate different wording for file output versus PR-body output, so maintainers do not update the wrong surface.
  - confidence: 0.96
- **[P2] Remove the release-owned changelog entry:** `CHANGELOG.md:9`
  - body: Normal PRs should not edit `CHANGELOG.md`; release generation owns that file. Keep the release-note context in the PR body or eventual squash message instead of adding this line to the changelog.
  - confidence: 0.99
