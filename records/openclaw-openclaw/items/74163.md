  - confidence: 0.89

  - body: The generator still emits a single `-` separator for the number column, while the committed tracker has expanded separators and the PR body shows the single-hyphen form. Regenerating from this script can produce fragile GitHub Markdown tables and drift from the checked-in artifact.
  - confidence: 0.82

  - body: The generator tells maintainers to edit the PR body even though the PR adds `MICROSOFT_TRACKER.md` as a repository file. That sends future updates to the wrong surface unless the intended workflow is explicitly body-only.
  - confidence: 0.9
