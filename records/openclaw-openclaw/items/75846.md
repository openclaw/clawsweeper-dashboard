Keep open. Current main still drops Claude CLI `tool_use` stream-json records before they can become `stream: "tool"` agent events, and the directly related implementation PR #76075 is closed without merge.

## What This Changes

## Best Possible Solution

Add a Claude stream-json tool-event bridge in the CLI parser and thread it through both live-session and one-shot runner paths while preserving the existing `stream: "tool"` contract.
