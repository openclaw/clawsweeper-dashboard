  - body: injectRestartNoticeIntoSession can throw from ensureGatewayTranscriptFile, but callers only enqueue the fallback system event when the helper returns false.   - confidence: 0.9

When a session has more than one active run entry, reconnect can restore an older run id and then ignore deltas or finals for the newer run due to run-id mismatch; keep the current run when still active or choose the newest startedAtMs.
