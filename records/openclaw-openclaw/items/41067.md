- **[P1] Preserve restart fallback when transcript setup fails:** `src/gateway/server-restart-sentinel.ts:32-33`
  - body: `ensureGatewayTranscriptFile` can reject before the helper reaches the `{ ok: false }` append result.   - confidence: 0.96

  - body: `activeChatRuns` is built from a run-id map and can contain more than one run for the same session, but this loop returns the first matching entry. Restoring an older run ID makes later events for the real active run look unrelated and can leave the composer stuck; keep the existing tracked run when present or choose the newest valid `startedAtMs`.
  - confidence: 0.93
