- **[P1] Return false when transcript setup fails:** `src/gateway/server-restart-sentinel.ts:32`
  - body: `injectRestartNoticeIntoSession` is used as a boolean guard for the system-event fallback, but this await can throw before the helper returns. Because the sentinel has already been consumed, a filesystem/path failure in transcript setup skips both transcript injection and `enqueueSystemEvent`, dropping the restart notice instead of falling back.
  - confidence: 0.88

  - body: The client returns the first matching `activeChatRuns` entry, while the server snapshot includes every `chatAbortControllers` entry for that session. If an older run remains in the map, reconnect can adopt the stale runId and then ignore deltas/finals from the newer run; choose the newest matching entry, and preserve the agent-scope invariant for global sessions.
  - confidence: 0.82
