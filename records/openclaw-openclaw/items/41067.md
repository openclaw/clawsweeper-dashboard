## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Select recovered chat runs deterministically:** `ui/src/ui/app-gateway.ts:126-136`
  - body: `activeChatRuns` is built from every `chatAbortControllers` entry, so a session can have multiple active run IDs. Returning the first matching entry can restore an older run and cause later events for the newer run to be ignored or leave stale busy state; keep the current run when present or choose the newest `startedAtMs`.
  - confidence: 0.9
- **[P2] Preserve restart fallback on injection errors:** `src/gateway/server-restart-sentinel.ts:22-39`
  - body: `injectRestartNoticeIntoSession` can throw before it returns `false`, but the callers only enqueue the fallback system event when it returns `false`. A filesystem, path, or transcript append setup error can consume the sentinel and drop the visible restart notice; catch helper errors and return `false`.
  - confidence: 0.92
