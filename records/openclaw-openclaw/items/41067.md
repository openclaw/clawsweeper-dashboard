Full review comments:

- **[P2] Select the actual active run during reconnect recovery:** `ui/src/ui/app-gateway.ts:126-136`
  - body: `activeChatRuns` can contain multiple entries for one session, but this returns the first map entry. If an older controller remains alongside a newer run, the UI restores the stale ID and rejects later events for the real run, leaving the composer stuck. Preserve the matching tracked run or select the newest valid `startedAtMs`.
  - confidence: 0.94
- **[P1] Preserve the restart-sentinel fallback on transcript errors:** `src/gateway/server-restart-sentinel.ts:34`
  - body: `ensureGatewayTranscriptFile` can reject on path or filesystem errors, but this helper only converts the later append result to a boolean. The rejection escapes after the sentinel is consumed, so the caller never enqueues its fallback system event and the recovery notice is lost. Catch the full injection operation and return `false`.
  - confidence: 0.96
