## What This Changes

The PR adds active-memory chitchat pattern detection, rejects matching summaries in `normalizeActiveSummary()`, and adds regression tests for observed chatter plus false-positive controls.

## Best Possible Solution

Land this bounded normalizer-layer hotfix after final-head checks pass, while leaving a structured active-memory recall protocol as separate future hardening.

## Solution Assessment

Yes for the immediate hotfix. The shared `normalizeActiveSummary()` boundary covers completed recall and timeout partial recovery, while JSON/structured recall output is a broader follow-up rather than a prerequisite for this patch.
