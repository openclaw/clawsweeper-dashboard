Implement prompt inspection through a shared runtime prompt-preview helper that reuses attempt prompt assembly and plugin prompt APIs, with explicit documentation for any live-only state that cannot be rendered offline.

Overall correctness: patch is incorrect

Overall confidence: 0.94

Full review comments:

- **[P1] Route channel prompt helpers through plugin APIs:** `src/commands/agents.commands.view-system-prompt.ts:31-33`
  - body: These imports target `src/signal` and `src/telegram` modules that no longer exist on current main, and PR CI fails with TS2307 for all three. Use the channel plugin prompt APIs, such as `resolveChannelReactionGuidance` and capability collection helpers, instead of hard-coding removed bundled channel internals from the core CLI command.
  - confidence: 0.98
- **[P2] Use runtime capability collection for channel previews:** `src/commands/agents.commands.view-system-prompt.ts:78-99`
  - body: This path combines raw channel capabilities with a Telegram special case, so it misses plugin prompt capabilities and thread-bound spawn capabilities that real runs include through `collectRuntimeChannelCapabilities`. `--channel` can therefore print guidance that does not match an actual agent turn.
  - confidence: 0.92
- **[P2] Build prompt params from the selected workspace:** `src/commands/agents.commands.view-system-prompt.ts:103`
  - body: Passing `process.cwd()` makes repo/source resolution depend on the shell launch directory rather than the inspected agent workspace. Embedded runs build prompt params with the effective workspace as `cwd`, so this can render a different Runtime section for the selected agent.
  - confidence: 0.9
- **[P2] Share attempt prompt assembly before printing:** `src/commands/agents.commands.view-system-prompt.ts:164-187`
  - body: Calling `buildEmbeddedSystemPrompt` directly skips attempt-level behavior such as system prompt overrides, provider contributions/transforms, source path, native command guidance, sandbox info, and effective tool filtering. Configurations using those hooks will see a preview that is not the submitted prompt.
  - confidence: 0.88
