Full review comments:

- **[P1] Use current channel prompt helpers:** `src/commands/agents.commands.view-system-prompt.ts:32-34`
  - body: These imports point at `src/signal` and `src/telegram`, but those modules do not exist on current main; the helpers now live behind channel plugins. As written, the branch cannot build and should route channel prompt details through the generic channel/plugin helpers.
  - confidence: 0.97
- **[P2] Collect runtime channel capabilities:** `src/commands/agents.commands.view-system-prompt.ts:79-100`
  - body: This uses config-only capabilities plus hardcoded Telegram handling, while real runs call `collectRuntimeChannelCapabilities` and plugin prompt helpers. `--channel` previews can therefore miss native approval UI, plugin prompt capabilities, and thread-bound spawn capabilities that the agent actually receives.
  - confidence: 0.91
- **[P2] Use the agent workspace for prompt params:** `src/commands/agents.commands.view-system-prompt.ts:102-106`
  - body: Passing `process.cwd()` makes repo/source resolution depend on where the operator launches the CLI. Embedded runs build prompt params with the effective agent workspace, so this can render a different Runtime section for the selected agent.
  - confidence: 0.88
- **[P2] Gate heartbeat text through runtime policy:** `src/commands/agents.commands.view-system-prompt.ts:154-157`
  - body: Resolving heartbeat text for every default-agent preview bypasses the current runtime policy for per-agent heartbeat config, cadence, and `includeSystemPromptSection`. Disabled heartbeat configs can show prompt text the agent would not receive.
  - confidence: 0.9
- **[P2] Build through the attempt prompt path:** `src/commands/agents.commands.view-system-prompt.ts:165-188`
  - body: Calling `buildEmbeddedSystemPrompt` directly skips current attempt-level handling for prompt overrides, provider transforms, source path, native command guidance, sandbox info, and provider prompt contributions. Configs/providers using those hooks will see a preview that is not the submitted prompt.
  - confidence: 0.87
