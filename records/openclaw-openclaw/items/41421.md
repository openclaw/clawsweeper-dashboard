Full review comments:

- **[P1] Use current channel helper imports:** `src/commands/agents.commands.view-system-prompt.ts:31-33`
  - body: The new command imports `../signal/reaction-level.js` and `../telegram/*`, but those modules do not exist under current core `src`, so the branch cannot build. Route through the generic channel/plugin prompt helpers instead of importing channel internals from core.
  - confidence: 0.95
- **[P2] Honor heartbeat prompt gating:** `src/commands/agents.commands.view-system-prompt.ts:153-156`
  - body: This resolves heartbeat text for every default-agent preview. Runtime only includes heartbeat prompt text after policy, cadence, trigger, and `includeSystemPromptSection` checks, so disabled heartbeat configs would show prompt text the agent does not actually receive.
  - confidence: 0.9
- **[P2] Use the agent workspace as prompt cwd:** `src/commands/agents.commands.view-system-prompt.ts:101-105`
  - body: Passing `process.cwd()` makes repo and docs/source resolution depend on where the operator launches the CLI. Embedded runs build prompt params with the effective agent workspace as `cwd`, so this can render a different Runtime section for the selected agent.
  - confidence: 0.88
- **[P2] Apply runtime prompt wrappers before output:** `src/commands/agents.commands.view-system-prompt.ts:164-187`
  - body: Calling `buildEmbeddedSystemPrompt` directly skips the current attempt-level wrapper for system-prompt overrides, provider prompt contributions, provider transforms, source path, native command guidance, and sandbox info. Configs/providers using those hooks will see a preview that is not the submitted prompt.
  - confidence: 0.86
- **[P2] Normalize the channel before channel-specific checks:** `src/commands/agents.commands.view-system-prompt.ts:81-82`
  - body: The Telegram/Signal branches compare `opts.channel` directly, while the embedded runtime normalizes channel IDs first. Mixed-case or alias inputs can resolve generic channel hints but skip inline-button or reaction guidance, making the preview inconsistent.
  - confidence: 0.82
