## Summary

The Codex warning fix is narrow and supported by source-level reasoning, regression tests, and supplied terminal proof, but the branch also adds `meetingUrl` to user-visible transcripts tool-display details. That URL detail should be removed or explicitly redacted before merge because tool-display details feed UI/progress summaries and meeting links can be private join URLs.

## What This Changes

The PR narrows Codex app-server `turn/completed` mismatch warnings, adds regression tests for cross-thread versus same-thread mismatches, and adds `transcripts` tool-display metadata to core and OpenClawKit resources.

## Best Possible Solution

Keep the Codex warning predicate and regression tests, but remove `meetingUrl` from transcripts display details or add explicit safe redaction before merging the metadata change.
