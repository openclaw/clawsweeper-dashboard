## Summary

Keep open: current main has coarse session lifecycle fields in the Gateway and Control UI state, but the normal Sessions table still does not show an active/completed/timed-out badge at a glance. The older overlapping issue is closed as a duplicate of this one, while the newer heartbeat issue is broader and does not replace this narrow UI tracker.

## What This Changes

Add a visible active/completed/timed-out status indicator and prominent last-activity information to each row in the Control UI sessions list.

## Best Possible Solution

Add a compact localized Status badge or column in the Sessions table from existing GatewaySessionRow lifecycle fields, and leave heartbeat/stall semantics to the broader health-tracking issue.

## Reproduction Assessment

## Solution Assessment
