## What This Changes

Add a visible lifecycle status indicator, and possibly status-oriented sorting or filtering, to the Control UI Sessions list.

## Best Possible Solution

Ship a compact Status column or badge in the Sessions table that uses the existing GatewaySessionRow lifecycle fields without changing gateway lifecycle semantics.
