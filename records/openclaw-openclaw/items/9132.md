Keep open: current main exposes session lifecycle fields through the Gateway and Control UI state, but the Sessions table still lacks an always-visible active/completed/timed-out status badge or column. ## What This Changes

Add a visible Control UI session-row status indicator for active, completed, timed-out/aborted sessions, with recent activity easier to scan.

## Best Possible Solution

Add a compact localized Status badge or column on every Sessions row using existing GatewaySessionRow lifecycle fields, with focused render coverage and a user-facing changelog entry.
