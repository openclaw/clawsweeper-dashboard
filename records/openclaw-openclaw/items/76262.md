## Best Possible Solution

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.83

Full review comments:

- **[P2] Parse comma-separated extension lists before skipping Matrix checks:** `Dockerfile:84`
  - body: `OPENCLAW_EXTENSIONS` is documented as space- or comma-separated, and the manifest-copy stage converts commas to spaces. This new skip guard only splits on spaces, so `diagnostics-otel,matrix` installs Matrix but exits before verifying `matrix-sdk-crypto`; split on commas here too or reuse the same parser.
  - confidence: 0.93
- **[P3] Preserve the OpenClaw User-Agent fragment:** `extensions/msteams/src/sdk.ts:282`
  - body: The Teams SDK constructor spreads `options.client.headers` and then overwrites `User-Agent` with `teams.ts[apps]/<version>`, so the object passed here drops the `OpenClaw/<version>` fragment. Use a client path the SDK actually merges, or add a regression test that inspects the constructed client headers.
  - confidence: 0.9
