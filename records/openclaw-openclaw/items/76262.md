## Best Possible Solution

Fix the remaining Docker parser and User-Agent preservation issues on this branch, then land after targeted Microsoft Teams and Docker validation stays green.

## Reproduction Assessment

I did not rerun live Teams validation in this read-only review.

## Solution Assessment

Is this the best way to solve the issue? Not yet. The SDK migration direction is plausible and current main does not already implement it, but the branch should first fix the Docker parser mismatch and use a SDK client path that demonstrably preserves the OpenClaw User-Agent fragment.
