## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Parse comma-separated extension lists before skipping Matrix checks:** `Dockerfile:84`
  - body: `OPENCLAW_EXTENSIONS` is documented as space- or comma-separated, and the manifest-copy stage converts commas to spaces. This new skip guard only splits on spaces, so `OPENCLAW_EXTENSIONS=diagnostics-otel,matrix` installs Matrix but exits before verifying `matrix-sdk-crypto`. Reuse the same comma-aware parser here before deciding Matrix is absent.
  - confidence: 0.94
- **[P3] Preserve the OpenClaw User-Agent fragment:** `extensions/msteams/src/sdk.ts:282`
  - body: `createMSTeamsApp` passes `OpenClaw/<version>` through plain App `client` options, but `@microsoft/teams.apps@2.0.10` overwrites object-client `User-Agent` headers with `teams.ts[apps]/<version>` before downstream clones. Use a client path the SDK actually merges, or add a regression test that proves the constructed SDK client keeps both fragments.
  - confidence: 0.9
