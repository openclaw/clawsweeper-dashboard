## What This Changes

The PR adds docs and runtime/test changes to prioritize `SOUL.md` and `HARD_EXECUTION_RULES.md` ahead of `AGENTS.md` during bootstrap budget selection, recognize `HARD_EXECUTION_RULES.md`, and warn when files are skipped.

## Best Possible Solution

Maintainers should choose one approved bootstrap contract; if SOUL/HARD/AGENTS is accepted, align discovery, budget selection, final prompt sorting, docs, diagnostics, and tests around it, otherwise leave this PR in favor of the existing warning model and the broader #39120/#74735 extra-file path.
