## What This Changes

Request to add `openclaw channels start|stop|restart --channel <name> [--account <id>]` so linked channel accounts can recover without QR re-pairing or container restart.

## Best Possible Solution

Review and land #75560 or an equivalent focused implementation that adds admin-scoped lifecycle CLI/Gateway coverage while leaving automatic retry policy for a follow-up.

## Solution Assessment
