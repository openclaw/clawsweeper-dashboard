  - body: When `sessionId` is present, this branch clears `requestedSessionKey`, but the PR head does not preserve the validated `agentId` for the later ingress dispatch.   - body: `Promise.allSettled` consumes failures from channel maintenance and session migration, but the callback ignores rejected results. Operators get no warning when deferred startup work fails, so inspect the settled results and log rejected reasons.
  - confidence: 0.88
- **[P2] Clean up Bonjour rejection handlers on startup errors:** `src/infra/bonjour.ts:329-333`
  - body: The unhandled-rejection handler is registered before cycle creation, but the PR error path only restores console logging. If cycle creation or listener setup throws before an advertiser is returned, the process-level handler can leak for the rest of the process.
  - confidence: 0.84

  - body: This test sets `OPENCLAW_GATEWAY_FAST_CONFIG` and deletes it only on the happy path. If an import or assertion throws first, later tests inherit fast gateway mode; save the previous value and restore it in a `finally`.
  - confidence: 0.83

  - body: Several trace calls pass the bootstrap start time into a helper whose `stage_ms` is computed from the supplied start time, so those labels report elapsed-since-bootstrap rather than per-stage duration. Capture a local timestamp for each measured stage.
  - confidence: 0.78
