## What This Changes

Telegram direct-message PPTX attachments are reported as missing while PDFs work, with a follow-up report that all Telegram file downloads fail after 2026.5.3 with an `EPERM` fsync log.

## Best Possible Solution

Keep the issue open until the reporter confirms the PPTX size and current-main or post-fix release logs; then either close against the media fsync fix or add explicit Telegram oversized-attachment handling/docs if the Bot API limit is the real cause.
