Rebase on current main and implement `pre_route` against the active `hook-types.ts` and `get-reply-options.types.ts` contracts, preserving `claimInboundDedupe` semantics and making route/session/account/agent overrides consistent across ACP and non-ACP dispatch before updating public hook docs and changelog credit.
