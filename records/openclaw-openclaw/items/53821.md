## What This Changes

## Best Possible Solution

## Solution Assessment

Unclear as a permanent product shape. The implementation is a coherent narrow fix, but maintainers should decide whether this separate sandbox override should land independently or be consolidated with the broader per-agent session/A2A policy work.
