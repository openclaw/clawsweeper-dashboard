## Summary

Keep open. The latest head is useful and exact-head CI is green, but `session_status` still has a requester-derived path that misses the per-agent sandbox clamp and can over-restrict valid same-agent status lookups.

## What This Changes

The PR adds per-agent `agents.list[].sandbox.sessionToolsVisibility` precedence to session visibility resolution, wires requester identity into session tools, updates docs/changelog, and adds targeted tests.

## Best Possible Solution

Land the sandbox-clamp override after `session_status` uses one resolved requester identity for visibility and fallback behavior, leaving broader per-agent sessions/A2A policy to #59149/#59170.

## Reproduction Assessment

Yes. Static reproduction is clear: create a sandboxed `session_status` tool for `agent:tony:main` with `tools.sessions.visibility: "all"`, default sandbox visibility `"spawned"`, and `agents.list[{id:"tony"}].sandbox.sessionToolsVisibility: "all"`; same-agent non-tree or sessionId status lookups still take the tree clamp because the resolver receives no agent id.

## Solution Assessment

No. The PR's narrow sandbox-clamp direction is maintainable, but the status path should pass the computed `requesterAgentId` and avoid forcing spawned fallback solely because the explicit override option is absent.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.89

Full review comments:

- **[P2] Pass resolved requester into status visibility:** `src/agents/tools/session-status-tool.ts:349-354`
  - body: `session_status` still passes only `opts?.requesterAgentIdOverride` into `resolveEffectiveSessionToolsVisibility` and also forces session-id fallback to spawned when no explicit override is present. A sandboxed `agent:tony:main` with per-agent `sessionToolsVisibility: "all"` therefore remains clamped to `tree` for same-agent status lookups unless the caller supplies the override separately. Pass the computed `requesterAgentId` through this path and base fallback restriction on the effective clamp.
