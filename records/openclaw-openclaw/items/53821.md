## What This Changes

The PR adds requester-aware per-agent `agents.list[].sandbox.sessionToolsVisibility` handling for session list/history/send/status, with tests, docs, changelog, and Plugin SDK API checksum updates.

## Best Possible Solution

The end state should be a coherent per-agent session access contract where sandbox clamps, session visibility, A2A policy, requester identity, tests, and docs agree across all session tools.
