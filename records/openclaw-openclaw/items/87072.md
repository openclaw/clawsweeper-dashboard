  - confidence: 0.9

  - body: The tail-as-prefix loop strips any matching prefix down to one character before the minimum-overlap guard runs. For ordinary streaming deltas, if the previous body happens to end with the next delta's first character, that leading character is dropped, corrupting the visible interleaved text; require a full duplicate or a safe word-boundary/minimum overlap before slicing.
  - confidence: 0.88
