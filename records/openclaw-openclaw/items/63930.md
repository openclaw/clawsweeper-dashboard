Add Anthropic advisor tool support plus generic Anthropic server-side tool block handling, including safer handling for unknown stop reasons.

## Best Possible Solution

Review and land #64064, or a maintainer-approved replacement, with the boundary kept clear: core Anthropic transport should handle generic server-side stream/display/replay semantics, while the bundled Anthropic plugin owns advisor config parsing, beta-header/tool injection, model validation, docs, and focused tests.
