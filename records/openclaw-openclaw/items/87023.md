This issue requests a first-class path for MCP binary Content blocks to survive tool execution, become channel attachments, and be replaced by bounded transcript text or receipts.

## Best Possible Solution

Define and land a narrow host-owned MCP media artifact relay that reuses existing outbound delivery, records bounded transcript receipts, and keeps unrestricted byte-sending APIs behind maintainer-approved security boundaries.
