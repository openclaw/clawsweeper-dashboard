This PR adds global and per-agent per-sender tool policy config, channel-scoped sender keys, docs/baseline updates, and sender-policy filtering in core and bundled tool pipelines.

## Best Possible Solution

Keep the PR open, propagate canonical channel ids through the plugin-owned group sender policy resolvers, add focused plugin regression tests, and then let maintainers review the final security semantics.
