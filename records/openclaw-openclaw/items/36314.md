## Summary

Keep open: current main has adjacent Gateway resilience mechanisms, but it still lacks the requested coherent Gateway process watchdog, operator-configurable recovery policies, richer HTTP health contract, and generalized degraded startup mode. ## What This Changes

Add built-in Gateway robustness mechanisms covering watchdog supervision, global error handling and recovery strategies, richer health checks, and degraded startup behavior.

## Best Possible Solution
