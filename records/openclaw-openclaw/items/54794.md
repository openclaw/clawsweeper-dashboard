## What This Changes

Add Telegram inline-query support so users can invoke an OpenClaw Telegram bot from other chats with `@botname query` and receive inline results.

## Best Possible Solution

Keep this as the canonical Telegram inline-query tracker and implement it inside the Telegram plugin with opt-in config/schema/docs, conservative auth and session semantics, bounded `InlineQueryResultArticle` responses, focused tests, and live Telegram proof while coordinating with the separate Guest Bot work.
