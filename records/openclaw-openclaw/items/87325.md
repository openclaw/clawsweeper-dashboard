## Summary

Keep open: current main has useful OpenAI realtime and Gateway relay plumbing, plus partial Azure deployment bridge support, but it still does not provide the requested first-class Azure Foundry GPT Realtime Talk path with current GA endpoint/auth behavior, Azure-specific diagnostics, docs, and live proof.

## What This Changes

## Best Possible Solution

Design and implement the Azure Foundry Realtime path in the bundled OpenAI provider and Talk Gateway relay with server-side API-key and Entra handling, Azure-specific diagnostics, docs, regression tests, and live Azure proof when quota is available.
