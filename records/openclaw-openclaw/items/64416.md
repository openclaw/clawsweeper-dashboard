## What This Changes

This PR changes ACP/subagent completion delivery by adding task-level parent lineage, silent parent wake routing, live-output preference, farewell suppression, ACP turn prompt guidance, and related tests.

## Best Possible Solution

Consolidate on one durable completion-delivery path that keeps structured events authoritative, sanitizes child output before prompt interpolation, persists or derives parent lineage across restart, and reconciles this branch with https://github.com/openclaw/openclaw/pull/65242 before merge.
