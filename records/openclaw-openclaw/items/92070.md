## What This Changes

The PR adds `--safe-mode` to the default Claude CLI backend launch arguments and normalizes user overrides so fresh and resumed `claude-cli` runs keep that flag.

## Best Possible Solution

Keep the isolation goal, but land it only with a maintainer-approved minimum-version policy or a narrow version/doctor/setup gate, focused regression coverage, a current-main rebase, and real fresh/resume launch proof.

## Solution Assessment
