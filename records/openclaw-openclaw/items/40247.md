## Best Possible Solution

## Solution Assessment

Unclear. Listener-side dual-bind is plausible and matches the loopback caller contract, but this branch is not the best form until it preserves the Windows guard, updates locality/auth docs, reconciles the overlapping client-side PR, and includes real setup proof.
