## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Yes. Passing targetAgentId through the existing registration seam plus using childSessionKey as a fallback in task creation is the narrowest maintainable fix; reader-side label or key parsing should remain only a workaround.
