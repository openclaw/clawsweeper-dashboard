Unclear as one issue: the security direction is valid, but the maintainable solution needs staged decisions around Codex runtime enforcement, sandbox backend contracts, per-tool permissions, session state, and compatibility. The linked macOS sandbox-exec PR is not sufficient as the complete fix.
