Keep the terminal channel signal narrow to non-recoverable Slack auth failures, preserve restart or unlimited retry behavior for recoverable Slack network failures, move supervisor coverage to a core/public seam, update SDK baseline/docs only if the signal is intentionally public, and require redacted real gateway proof.
