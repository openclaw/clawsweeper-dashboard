Add a browser-owned doctor diagnostic that warns on confirmed unused legacy `clawd` profile residue and moves it to a safe Trash/archive location only during explicit repair.

Yes. A doctor/browser diagnostic with opt-in Trash/archive repair is the narrow maintainable direction; startup migration or unconditional deletion would be riskier.
