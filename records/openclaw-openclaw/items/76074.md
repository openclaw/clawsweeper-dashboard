## What This Changes

Define a docs-level workspace convention for durable capability descriptors such as `capabilities/*.md` so agents can discover local integrations before declaring tasks unsupported.

## Best Possible Solution

Review and either land or revise PR #76082 so the docs define the convention, cross-link the bootstrap surfaces, and keep runtime auto-discovery as a separate follow-up.
