## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.84

Full review comments:

- **[P2] Respect quoted attributes when injecting the CSP meta tag:** `ui/src/ui/chat/html-preview.ts:75`
  - body: `findFirstHtmlStartTag` treats the first `>` as the end of a start tag, even when that character appears inside a quoted attribute. A valid document such as `<html data-x=\">\"><body><img src=\"https://example.test/leak.png\"></body></html>` would have the CSP meta inserted into the attribute text instead of the parsed head, so the fallback CSP meant to block navigation/network egress may not apply. Please make the scanner quote-aware or use a parser-backed injection path and add a regression test for this shape.
  - confidence: 0.88
