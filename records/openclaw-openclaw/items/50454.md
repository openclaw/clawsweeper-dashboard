## Best Possible Solution

Refresh or replace this branch against current main so transcript header creation receives one explicit cwd derived from the active runtime config/session context, covers both ordinary and spawned mirrored sessions, and keeps mirror failures non-fatal after delivery.

  - body: This new `loadConfig()` runs inside the transcript mirror append path. Use one resolved agent id for both decisions.
