## What This Changes

The issue requests `/model <name> --turns N`, warning near expiry, automatic revert to the session default after N turns, and per-agent per-model turn-limit config.

## Best Possible Solution

Define one session-owned, opt-in model lease contract that extends the existing `/model` and session override path, preserves explicit user selections and auth-profile pins, and covers warnings, expiry, docs, config schema, and regression tests.
