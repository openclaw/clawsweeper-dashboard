## Summary

Keep open: current main still supports persistent session model selection plus narrower one-turn/image-related model override work, but it does not implement an N-turn model lease with countdown warnings, per-model/per-agent limits, or automatic default reversion.

## What This Changes

Add `/model <name> --turns N` and per-agent model turn-limit config so temporary premium model use warns and automatically reverts to the default model.

## Best Possible Solution

Define a canonical model-lease contract first; if accepted, implement additive parser/config/session lease state with warning and revert notifications, docs, and focused tests around model override expiration.
