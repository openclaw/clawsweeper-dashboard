## Summary

Keep open: current main still implements persistent `/model` session switching, not a turn-limited model lease with warning, countdown state, automatic default reversion, or a public per-model limits contract. Related open items cover one-turn overrides and image-analysis reversion, but neither supersedes this broader per-model turn-budget request.

## What This Changes

Add `/model <name> --turns N` support that temporarily switches models, warns near expiry, and reverts to the default model after the configured turn budget.

## Best Possible Solution

Define the model-governance contract first; if accepted, implement parser support, durable countdown state, warning/revert notifications, config/docs, and focused regression coverage around session model overrides.
