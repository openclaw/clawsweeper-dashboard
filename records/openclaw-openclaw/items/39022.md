## Summary

Keep open: current main and the latest release still do not support `/model --turns`, a model-lease counter, warning/revert state, or per-model turn-limit config. The request is useful but needs maintainer product/API direction because it changes session model state, provider routing, and config semantics.

## What This Changes

The issue requests `/model <name> --turns N` plus per-agent/per-model turn limits that warn before reverting to the default model.

## Best Possible Solution

Define one maintainer-approved model lease contract that composes with persistent user model selections, provider/auth routing, session reset behavior, and the narrower one-shot override design.
