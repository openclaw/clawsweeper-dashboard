Implement safe URL linkification for tool output cards on current main. Start with Control UI: add or reuse a shared helper/renderer that escapes arbitrary tool output before creating anchors, only links http/https URLs, handles trailing punctuation and whitespace, preserves JSON/plain-text formatting, and does not route untrusted tool output through broad markdown/HTML injection. Apply it consistently to expanded output, raw details, collapsed previews, inline short cards, grouped tool-message paths, and sidebar/raw views.
