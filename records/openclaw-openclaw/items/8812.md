Render URLs inside Control UI and native WebChat tool output cards as clickable links instead of plain text.

## Best Possible Solution

Add a shared safe URL-aware rendering path for tool output across expanded, preview, inline, raw, grouped, and sidebar surfaces, with native WebChat URL rendering if maintainers keep that scope.

## Solution Assessment

Yes. The maintainable path is a safe URL-aware renderer that escapes arbitrary text and only creates safe HTTP(S) links, rather than broad regex replacement on untrusted HTML.
