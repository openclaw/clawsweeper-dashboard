## Summary

Keep open. Current main still has the skipped-reasoning buffered-answer path, so this PR is not obsolete, but the patch still misses preview-finalized hook emission and lacks live Telegram proof.

## What This Changes

The PR adds a Telegram dispatcher flush for buffered final-answer text, a skipped-reasoning regression test, and a changelog fix entry.

## Best Possible Solution

Update this PR or an equivalent Telegram-owned patch so the skipped-reasoning flush drains the buffered final answer, preserves preview-finalized hook emission, adds focused regression coverage, and includes live Telegram proof.

## Reproduction Assessment

Yes, by source inspection. Current main can buffer an answer while reasoning remains hinted, then return from the segmented final branch without flushing when reasoning delivery is skipped; I did not run tests because this review is read-only.

## Solution Assessment

No, not as currently proposed. Adding the missing final flush is the right narrow direction, but the new flush path must preserve the LaneDeliveryResult hook path before the patch is safe to merge.

- **[P2] Emit the hook when flushing buffered final answers:** `extensions/telegram/src/bot-message-dispatch.ts:703`
