The PR updates `docs/providers/openai.md` and `CHANGELOG.md` to clarify OpenAI Realtime voice billing and API-key requirements for the confusion reported in #76498.

## Best Possible Solution

Land a qualified docs clarification that distinguishes public OpenAI Platform Realtime from the existing Azure OpenAI Voice Call realtime path.
