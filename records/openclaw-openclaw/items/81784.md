- **[P1] Fail closed when the inbound-turn file cannot refresh:** `src/agents/cli-runner/prepare.ts:252-257`
  - body: This catch only logs and then continues the CLI run. For a reused live-session subprocess, `OPENCLAW_INBOUND_TURN_FILE` is already fixed in the process environment and the old file remains at that path, so a wrapper can read the previous turn's message/chat identifiers and send to the wrong target. Surface the failure or replace the file with a safe current-turn tombstone before allowing the run to continue.
  - confidence: 0.88

  - confidence: 0.84
