  - body: `group-policy-allowlist-empty` is not a confirmed unknown chat.   - confidence: 0.9
- **[P2] Reuse thread param helper for warning sends:** `extensions/telegram/src/bot-handlers.runtime.ts:1660`
  - body: `resolvedThreadId` can be `1` for a forum's General topic, and current Telegram sends omit `message_thread_id=1` because Telegram rejects it. This new direct warning path can 400 before sending the configured warning or before a leave action.
  - confidence: 0.88
- **[P2] Apply unknown-group policy before native commands too:** `extensions/telegram/src/bot-handlers.runtime.ts:1645`
  - body: Native command middleware is registered before the regular message handlers, and command auth replies directly for group-policy failures. A first `/help` or plugin command in an unknown group bypasses the new warn/leave action, leaving the bot in place with the old denial behavior.
  - confidence: 0.87
