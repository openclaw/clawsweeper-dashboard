- **[P1] Gate sensitive env admission to selected skills:** `src/agents/skills/env-overrides.ts:163-165`
  - body: The added allowlist loop admits every staged `skills.entries.*.env` key before no-snapshot runs narrow the catalog to eligible or prompted skills.   - confidence: 0.86

  - body: `acquireActiveSkillEnvKey()` keeps the first active value for a repeated key and only increments a count. After this PR admits arbitrary configured sensitive names, two skills with the same token/password env name can silently share the first skill's credential instead of failing or isolating the conflict.
  - confidence: 0.78
