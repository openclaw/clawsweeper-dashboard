## What This Changes

## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

No, not as submitted. Deferring the marker until verified success is the right narrow direction, but this head must be rebased over current snapshot hydration and fixed for the optional session-entry mutation before merge.
