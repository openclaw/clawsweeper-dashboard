## What This Changes

The branch moves `systemSent` disk persistence out of `ensureSkillSnapshot()` into post-success reply/followup accounting and adds regression tests.

## Best Possible Solution

Land a rebased patch that preserves snapshot hydration, persists `systemSent` only after successful reply/followup completion, fixes the optional entry assignment, and includes redacted retry proof.
