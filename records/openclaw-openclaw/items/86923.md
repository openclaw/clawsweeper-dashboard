  - confidence: 0.9

  - confidence: 0.86

  - body: When `probe()` returns `null`, this branch still runs an auto-orient encode but discards the encoded output and returns the original buffer. If Rastermill can decode the image even though probing missed metadata, the caller pays the encode cost but receives unnormalized bytes.
  - confidence: 0.82

  - body: This changes `rastermill` from the published npm version to a GitHub codeload tarball while keeping it in the root dependency set.   - confidence: 0.9
