## Best Possible Solution

Pick one canonical branch for `https://github.com/openclaw/openclaw/issues/81089`, then land a tested fallback that handles known unsupported-link errors while preserving `EEXIST` busy and stale-lock semantics with real affected-filesystem or maintainer-run proof.

## Solution Assessment
