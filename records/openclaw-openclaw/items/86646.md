## Best Possible Solution

  - body: When `isTailDuplicate(...)` is true for a final or aborted payload, the branch falls into `else` and calls `materializeVisibleAssistantStreamMessages` without the terminal message as a replacement.   - confidence: 0.96

Require an assistant tail or include role in the signature, and add a user-echo regression test.
