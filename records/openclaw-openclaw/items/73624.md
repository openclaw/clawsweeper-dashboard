## Summary

Keep open. Current `main` still drops the delegated subagent task when `systemPromptOverride` is configured, and the related fix PR #73637 was closed unmerged rather than shipped.

## What This Changes

Preserve task-specific subagent instructions from `extraSystemPrompt` when an agent-level `systemPromptOverride` is active.

## Best Possible Solution

Define and document the override composition contract, then preserve task-bearing runtime context across embedded attempts, compaction retries, and CLI preparation with focused regression tests.

## Reproduction Assessment

Yes. A high-confidence static reproduction exists: spawn a native subagent under an agent with `systemPromptOverride`; the delegated task is only in `extraSystemPrompt`, and all three prompt assembly paths skip the builders that append it.

## Solution Assessment

No. Current main does not solve the issue, and the reporter's append approach is only a candidate until maintainers choose between system-prompt composition and a separate lower-priority task message contract.
