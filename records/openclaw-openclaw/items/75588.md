## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Preserve exact tag boundaries during cleanup:** `src/plugins/host-hook-workflow.ts:469`
  - body: `unscheduleSessionTurnsByTag` matches cron jobs with `name.startsWith(namePrefix)`, but tag values are inserted into the colon-delimited name without escaping. A job tagged `foo:agent:main:main:x` in the same session starts with the cleanup prefix for tag `foo`, so unscheduling `foo` can remove the longer-tag job too. Encode or length-prefix the components, or reject delimiter-bearing tags, so tag cleanup only removes the exact tag requested.
  - confidence: 0.87
