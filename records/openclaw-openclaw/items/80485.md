Keep open: this PR addresses a valid current-main systemd/update bug, but it is a draft policy proposal, the contributor proof is unit-test-only, and maintainers still need to choose the intended ownership model for update-time service repair.

The PR defers noninteractive update-mode rewrites of live Linux systemd gateway units while preserving non-systemd staging behavior and adding focused doctor tests.
