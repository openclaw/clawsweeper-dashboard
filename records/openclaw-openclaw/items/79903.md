## What This Changes

Expose a public Gateway or SDK seam for canonical session-key resolution and lineage/family discovery across sessionId rotations.

## Best Possible Solution

Keep this issue as the focused session-lineage tracker until maintainers land a narrow Gateway/SDK family-discovery seam or explicitly make https://github.com/openclaw/openclaw/issues/79902 and https://github.com/openclaw/openclaw/pull/79962 the canonical path.
