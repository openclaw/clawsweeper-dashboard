## Summary

Keep open: current main still lacks inline provenance metadata for injected prompt context, and the same-author Phase 1 PR #54830 remains open with closing syntax, so this issue should stay open for maintainer review rather than be closed or queued separately.

## What This Changes

The RFC asks OpenClaw to annotate injected prompt context segments with source, injection timing, and volatility metadata.

## Best Possible Solution

Keep this RFC open while maintainers review PR #54830 as the immediate Phase 1 candidate, then split memory timestamps, prompt rules, and deterministic refresh into narrower follow-ups if the tag format is accepted.

## Reproduction Assessment

## Solution Assessment
