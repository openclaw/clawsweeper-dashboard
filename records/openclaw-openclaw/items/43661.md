## Label Justifications

- `P1`: Multiple deployments report duplicated outbound messages and sessions remaining unusable until restart or manual session rotation.
- `impact:message-loss`: The retry path can duplicate already-delivered text or media and can leave later replies undelivered while the lane is wedged.
- `impact:session-state`: The affected session can retain stuck or oversized state and re-enter the failure after restart.
- `impact:crash-loop`: The gateway can remain alive while the affected lane repeatedly times out and requires operator restart or rotation to restore availability.

## Summary

Keep open: current main and the latest stable release still allow timeout-triggered compaction to retry the prompt without a side-effect/replay-safety gate, while successful identical message-tool sends intentionally receive fresh autogenerated idempotency keys. Adjacent open issues cover generic idempotency and compaction timeout progress, but neither replaces this source-backed compaction replay plus poisoned-session report.

## What This Changes

The issue asks OpenClaw to recover cleanly from compaction timeouts without replaying already delivered outbound messages or leaving the session wedged until manual restart or rotation.

## Best Possible Solution

Persist a logical outbound side-effect identity across compaction retries, return or reuse the prior committed result instead of sending again, and make timeout/poisoned-session paths terminate or rotate safely without replaying completed tools.

## Reproduction Assessment

Yes, at source level: current main retries the prompt after successful timeout compaction, and message-tool tests explicitly allow a fresh idempotency key after a successful identical send. I did not run a live current-main end-to-end reproduction in this read-only review.

## Solution Assessment

No, the current partial mitigations are not the complete fix because unresolved-send retry dedupe does not cover successful side effects replayed after compaction, and it does not define recovery for a poisoned session.
