## Summary

Keep open: current main still uses and documents daily reset as the implicit default, and this PR is not mergeable as-is because the submitted guard suppresses the new group/thread idle default whenever any unrelated `resetByType` entry exists.

## What This Changes

The PR changes implicit group/thread session reset defaults to idle with a 10080-minute timeout while leaving direct sessions daily, and updates reset-policy tests.

## Best Possible Solution

Land a refreshed, maintainer-approved implementation in `reset-policy.ts` with exact group/thread 10080-minute coverage, docs and changelog updates, and explicit preservation of base, type, and channel overrides.

## Reproduction Assessment

Yes. Source inspection shows current main still resolves implicit group/thread reset policy through the daily default, and tracing the PR diff shows the proposed guard fails when only an unrelated direct/DM override exists.

## Solution Assessment

No. The direction may be acceptable, but this branch is not the best fix as submitted because it scopes the implied default too broadly and needs maintainer approval for the longer shared-channel retention default.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Scope the idle default to the current reset type:** `src/config/sessions/reset.ts:114-115`
  - body: The new `impliedDefaultMode` uses `!hasExplicitReset`, but `hasExplicitReset` becomes true when any `resetByType` entry exists. A config with only a direct/DM override therefore keeps group/thread sessions on `daily`, so the primary behavior change does not apply for a common override shape.
  - confidence: 0.93
- **[P3] Pin the group idle timeout assertion:** `src/config/sessions/sessions.test.ts:151`
  - body: The updated assertion only checks that `idleMinutes` is not `45`, which would also pass for `undefined`, `0`, or another wrong value. Pin the expected `10080`-minute default so this behavior is actually covered.
  - confidence: 0.84
