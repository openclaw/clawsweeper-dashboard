- **[P1] Scope tool activity to the active run:** `src/shared/tool-activity-heartbeat.ts:1-5`
  - body: `notifyToolActivity()` broadcasts through one process-global listener set, while every idle-timeout wrapper subscribes to the same bus. Keep the heartbeat keyed to the run/session/signal lifecycle instead of broadcasting globally.
