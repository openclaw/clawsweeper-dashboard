## Summary

## What This Changes

The branch adds Claude/Gemini ACP launch-time model injection, suppresses selected Claude/Gemini runtime config replays, and changes `sessions_yield` to report success when session context exists without an `onYield` callback.

## Best Possible Solution

Rebase onto current main, keep only dependency-backed launch-time model support, preserve visible apply-or-reject semantics for runtime controls, keep `sessions_yield` success tied to an actual yield callback, and add redacted ACP/Foreman proof.
