  - confidence: 0.93

  - body: This creates a service-key-backed Supabase client from raw environment variables and logs every received/sent message payload to an external database. A bundled core plugin needs explicit config/SecretRef handling, operator opt-in, data policy, and real runtime proof before it can introduce this security boundary.
  - confidence: 0.9
