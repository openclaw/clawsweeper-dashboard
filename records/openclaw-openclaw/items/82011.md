## Summary

Keep open. This is a new Control UI/WebChat composer proofreading capability, not broken existing behavior; current main still has only slash-command/static suggestions in the composer, and the existing plugin/control-UI contracts do not expose an in-composer diagnostics or highlighting seam.

Requests configurable typo and grammar checking in the chat input, with highlighted suggestions for common Chinese misspellings and grammar issues without automatic edits.

## Best Possible Solution
