- **[P2] Bound task metadata before returning Gateway summaries:** `src/gateway/server-methods/tasks.ts:94`

The ACP create-collapse path already adopts preferred label/task fields, so a caller can create or update a run with metadata and get back the old row without the task-ledger fields this PR advertises. Thread metadata through that merge path and add a regression test.
  - confidence: 0.9
