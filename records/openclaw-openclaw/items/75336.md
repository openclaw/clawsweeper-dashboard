Full review comments:

- **[P1] Parse string arguments before identifier scanning:** `src/agents/agent-hooks/compaction-safeguard-quality.ts:265`
  - body: Object tool arguments are sanitized with sanitizeDiagnosticPayload, but string arguments are appended raw. A JSON string like {"apiKey":"abcdef123456"} bypasses field-name redaction, letting the hex-looking value be extracted and later injected into retry feedback or a recovered summary. Parse JSON strings and sanitize them before falling back to raw text.
  - confidence: 0.92

An identifier already present in the preserved suffix can still produce missing_identifiers and spend a retry, so split the identifier sets or audit against the full summary consistently.
  - confidence: 0.87
