Keep open: current main and the latest release still leave the cache-trace and Anthropic payload diagnostic JSONL sinks without a rotation/archive policy, and the linked implementation PR is still open rather than merged.

Land or replace the canonical diagnostic-retention fix so cache-trace and Anthropic payload diagnostics are bounded through a symlink-safe writer path, with schema/docs/tests and real OpenClaw proof; longer term, align these diagnostics with the SQLite-backed direction already documented for runtime diagnostics.
