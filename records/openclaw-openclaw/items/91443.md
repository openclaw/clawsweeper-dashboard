## Best Possible Solution

Resolve hosted Canvas/A2UI targets from the node-session current/refresh capability source, preserve local and external loads, redact capability-bearing diagnostics, and add redacted real macOS proof.

## Solution Assessment

No. Capability URL rewriting is the right direction, but this implementation bypasses the existing node-session capability source/refresh contract and adds public capability URL logging.
