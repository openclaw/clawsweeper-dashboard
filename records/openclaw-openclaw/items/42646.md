## Best Possible Solution

Settle the durable-memory contract across memory-core, memory-host SDK, Memory Wiki/Markdown compatibility, and migration semantics, then implement schema, write/revision/provenance/review flows, docs, and tests as one coordinated Memory MVP slice.

## Solution Assessment
