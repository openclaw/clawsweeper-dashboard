## What This Changes

Define a v1 SQLite-backed durable memory schema with typed records, canonical current state, revision history, status transitions, validity windows, and normalized subject/entity fields.

## Best Possible Solution

Have maintainers choose the memory-plugin durable record contract and migration plan across the Memory MVP cluster, then implement it at the storage boundary with focused tests and docs while keeping embeddings retrieval-only.

## Solution Assessment
