  - confidence: 0.93
- **[P2] Handle aborted Slack body reads:** `src/slack/monitor/provider.ts:100-107`
  - body: The new body reader only waits for data/end/error. If a client disconnects before end, the promise can remain unresolved and leave request handlers stuck under repeated aborted uploads.
  - confidence: 0.88

  - confidence: 0.96
