## Summary

Keep open: current main has multi-agent workspaces, per-agent tool policies, and channel `toolsBySender`, and the discussion links narrower RBAC work in #53530/#53496, but this umbrella still needs new CLI/template, manifest, docs, and product/security decisions around tenant-grade RBAC that OpenClaw does not currently promise.

## What This Changes

Request first-class org/team deployment support: workspace scaffolding CLI, per-session/per-sender tool policies, declarative deployment manifests, and official org deployment docs.

## Best Possible Solution

Split this into scoped design and implementation tracks: finish or close the per-session policy work in #49183/#53530, then separately decide the manifest/workspace CLI and document supported same-trust multi-agent deployments.
