Keep open. Current main has manual multi-agent workspaces, config includes, and partial per-agent/per-sender tool-policy pieces, but it still lacks the requested first-class team scaffolding, declarative deployment manifests, and reviewed session/RBAC contract; no single related item fully supersedes the umbrella.
