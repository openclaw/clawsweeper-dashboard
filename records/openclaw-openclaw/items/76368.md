## Summary

## What This Changes

The PR wires `memory-core`'s runtime plugin entry to reuse its manifest config schema, adds entry-level config validation tests, and records a changelog fix.

## Best Possible Solution

## Solution Assessment

Yes. Reusing the existing manifest schema through `buildJsonPluginConfigSchema` is the narrowest maintainable fix because it avoids a second schema and preserves strict unknown-key rejection.
