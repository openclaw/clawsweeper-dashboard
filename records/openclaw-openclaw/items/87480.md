## Best Possible Solution

Implement a capped same-key retry for clearly transient pre-response socket or fetch failures inside the guarded Anthropic model transport, with focused fallback coverage and redacted live or staging proof.

## Solution Assessment

No. The SDK dispatcher is a plausible mitigation, but the narrower and safer fix is retrying transient pre-response failures inside guarded transport before model fallback.
