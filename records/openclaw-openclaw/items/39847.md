## What This Changes

The issue reports echoed inbound metadata leaking through outbound channel delivery because reply normalization does not sanitize payload text before directive parsing.

## Best Possible Solution

Land one canonical source-level fix that strips inbound metadata at the shared outbound normalization boundary and adds regression coverage for metadata plus directives/media.
