Overall correctness: patch is incorrect

Full review comments:

- **[P2] Use the worker session key for the hint:** `src/agents/embedded-agent-runner/context-engine-maintenance.ts:712`
  - body: The deferred worker is scheduled with `params.sessionKey ?? params.sessionId`, but the new hint receives only `params.sessionKey`. For callers that omit `sessionKey`, an engine that keys debt by the same session key passed to `maintain()` can see `undefined`, return false, and skip real maintenance. Pass the same normalized fallback key to the hint and cover that case.
  - confidence: 0.87
- **[P2] Export the new hint params from the SDK:** `src/context-engine/types.ts:190`
  - body: This introduces a named callback parameter type for a plugin-facing `ContextEngineInfo` field, but `openclaw/plugin-sdk` still does not export it. External engines implementing the companion change should not need to deep-import host internals or duplicate the shape; add the type to the SDK barrel and the relevant API/export coverage.
  - confidence: 0.8
