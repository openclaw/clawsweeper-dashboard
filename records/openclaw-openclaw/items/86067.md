## Summary

Keep open for maintainer review: the patch targets the active cross-lane session-file race behind dropped embedded replies, current main does not have the file-scoped prompt queue, and I did not find a concrete line-level defect, but the new hot-path queue is availability-sensitive.

## What This Changes

## Best Possible Solution
