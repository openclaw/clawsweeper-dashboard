## What This Changes

The PR adds a resolved-session-file prompt-turn queue to the embedded session-lock controller, separates compaction idle release from prompt release, and expands session-lock regression coverage.

## Best Possible Solution

Land the file-scoped prompt-window serialization once required CI is green and maintainers accept the bounded availability tradeoff for this hot embedded-runner path.

## Solution Assessment
