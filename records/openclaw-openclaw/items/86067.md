## Summary

## What This Changes

The PR adds a process-global, file-scoped prompt-window queue to the embedded pi runner session-lock controller, splits compaction idle waits into `releaseForSessionIdleWait()`, and adds regression coverage for same-file serialization and cleanup paths.

## Best Possible Solution

Land the file-scoped guard only after a maintainer explicitly accepts the same-file availability tradeoff and verifies the exact head with focused session-lock tests plus the current changed-check/CI gate.
