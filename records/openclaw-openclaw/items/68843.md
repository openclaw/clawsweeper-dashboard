## Best Possible Solution

Have an ACP/session owner confirm the cwd-based stale trigger and target-session-key-wide unbind semantics, then land with the focused regression tests if that behavior is intended.

## Solution Assessment

Mostly yes, because filesystem-confirmed missing cwd is a narrow signal for a dead ACP runtime, but the target-session-key-wide unbind needs maintainer acceptance before merge.
