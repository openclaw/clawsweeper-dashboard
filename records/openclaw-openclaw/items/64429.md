## Summary

Keep this PR open. Current main still lacks the PR's intended structured Slack file ID propagation into `SlackMediaResult` and finalized message context; the shipped placeholder/schema path lets agents read and pass Slack file IDs, but it does not implement the proposed structured context contract.
