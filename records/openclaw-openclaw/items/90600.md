## What This Changes

The issue requests an agent-callable tool that renders SVG or simple HTML as inline chat media across OpenClaw channels.

## Best Possible Solution

Resolve the product/API boundary first, then implement this as a plugin-owned render capability or a narrow plugin API extension rather than a one-off core tool.
