## What This Changes

Report a 2026.5.12 CLI startup regression where short-lived subcommands spend most time in Jiti/Babel traversal and repeatedly read plugin manifests even for disabled plugins.

Profile the packaged CLI path, then remove avoidable Jiti transforms and duplicate manifest-registry reads by carrying explicit metadata/load-context objects through hot paths instead of adding hidden persistent registry caches.
