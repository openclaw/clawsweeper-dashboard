## Best Possible Solution

Add an owner-approved upgrade or `openclaw doctor --fix` repair that detects legacy Claude CLI runtime use, imports or validates the existing Claude CLI credential/profile safely, records what was migrated, and emits a clear actionable failure when the Keychain state cannot be used.
