Define a reviewed typed browser SSRF policy contract with hard-deny precedence for metadata/link-local targets, explicit approval routing, timeout/rate-limit behavior, audit semantics, config schema/help/docs updates, and regression coverage across the shared SSRF and browser navigation layers.
