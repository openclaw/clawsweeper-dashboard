## Best Possible Solution

Land or replace the linked PR with a focused fix that prefers the phase-filtered final answer whenever explicit phase metadata exists while preserving existing raw media-directive handling.

Yes. Source inspection gives a high-confidence helper-level reproduction: one non-empty raw streamed `assistantTexts` entry can still bypass the phase-filtered `lastAssistant` fallback on current main.

## Solution Assessment

Yes, the narrow payload-helper fix direction is maintainable. The implementation should use the existing phase-filtered final-answer extraction without bypassing the media-aware raw final-answer path.
