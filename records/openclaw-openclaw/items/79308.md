## What This Changes

## Best Possible Solution

## Solution Assessment

Unclear: sending replies to the originating group is the correct existing behavior, but the narrow maintainable fix depends on whether suppression, durable outbound support, target normalization, account binding, or deployment state is dropping the group target.
