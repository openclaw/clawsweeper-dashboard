Full review comments:

- **[P1] Heal using the original requested key aliases:** `src/gateway/server-methods/sessions.ts:1900`
  - body: This branch resolves the store target from `key`, but `key` has already gone through `resolveScopedAbortKey` when `agentId` is supplied. Resolve the heal target from the original requested key or loaded session target, and add the raw-alias regression.
