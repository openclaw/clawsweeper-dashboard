## What This Changes

The issue requests a CLI runner repair so fallback retries against resumed CLI sessions do not append duplicate user turns.

## Best Possible Solution

Implement a CLI-owned prompt de-duplication or transcript repair path for reusable CLI session fallback retries, preserving fresh-session reseed behavior and avoiding a new config switch.

## Solution Assessment
