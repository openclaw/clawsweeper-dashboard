## Summary

## What This Changes

## Best Possible Solution

Keep the implemented validation and env-preservation work, then make a maintainer-approved root CLI semantics and secret-writing policy change with matching docs and regression tests.

## Solution Assessment

Unclear until maintainers decide the product and security policy. Showing help for the root command and tightening literal-secret writes are plausible, but both change documented CLI behavior or credential-writing semantics.
