## What This Changes

Fix intermittent raw JSON parse errors surfacing as Telegram/Discord/Codex chat messages during long tool-call conversations on Raspberry Pi/Linux/container runs.

## Best Possible Solution

Add focused cross-transport regression coverage for malformed streaming parse failures through external chat delivery, then implement the chosen behavior consistently: retry/fail over when recoverable, or render one stable friendly malformed-fragment message without masking provider request-validation diagnostics.
