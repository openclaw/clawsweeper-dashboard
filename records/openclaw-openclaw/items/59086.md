Full review comments:

- **[P2] Preserve the reasoning visibility gate:** `src/agents/pi-embedded-subscribe.ts:50`
  - body: Current main gates streamed reasoning on `canShowReasoning`, so `thinkingLevel: "off"` suppresses thinking. This branch sets `streamReasoning` from `reasoningMode` alone; a rebase that keeps that shape would emit thinking events even when visibility is disabled.
  - confidence: 0.87
- **[P2] Preserve the loading indicator when reasoning is hidden:** `ui/src/ui/views/chat.ts:1549-1555`
  - body: This adds a stream item whenever `props.streamMessage` exists. If that payload only contains thinking blocks and `showReasoning` is false, `renderGroupedMessage` suppresses the content, so the existing reading-indicator fallback is skipped and an empty assistant row/footer renders until visible text arrives.
  - confidence: 0.93
- **[P3] Avoid stripping double Markdown markers:** `ui/src/ui/chat/message-extract.ts:125-127`
  - body: The single-marker checks only test the first and last character, so `**bold**` and `__text__` lose one marker on each side before the reasoning formatter re-wraps them. Exclude double-marker prefixes and suffixes before slicing.
  - confidence: 0.82
