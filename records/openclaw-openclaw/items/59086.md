## Summary

Keep open. Current main still lacks the end-to-end Gateway chat and Control UI support this PR is trying to add, but the PR should not merge as-is because it introduces a hidden-reasoning streaming regression and overlaps active follow-up work in #68982.

## What This Changes

The PR streams thinking events through Gateway chat payloads, adds structured Control UI streaming/rendering for interleaved thinking/text/tool blocks, reorders agent-event callback fan-out, and updates focused Gateway, agent, and UI tests.

## Best Possible Solution

Land one rebased, consolidated implementation that carries structured thinking/text payloads from agent events through Gateway chat delta/final payloads and Control UI rendering while preserving `/reasoning` visibility and hidden-reasoning loading behavior.

## Reproduction Assessment

Yes. A high-confidence static reproduction exists at PR head: a reasoning-only structured stream with Show Thinking off creates a stream row but renders no visible content, while current main confirms the underlying feature gap remains text-only.

## Solution Assessment

No, not as-is. The direction is maintainable, but this branch needs the hidden-reasoning fix, markdown normalizer fix, and consolidation with #68982 before it is the best merge path.
