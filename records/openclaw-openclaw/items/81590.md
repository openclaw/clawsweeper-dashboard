That bypasses `requestSafeGatewayRestart`/`scheduleGatewaySigusr1Restart` deferral, cooldown, coalescing, and session-continuation handling, so enabling the watchdog can interrupt active gateway work outside the managed lifecycle.

  - body: The watchdog persists cooldown/history state under the OpenClaw state tree as a JSON sidecar. Repository policy keeps OpenClaw-owned runtime state in SQLite or a named product artifact, so this should use an existing state store or be made transient before becoming a core script.
