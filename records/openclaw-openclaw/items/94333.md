## What This Changes

The PR reads `allowModelOverride` and `allowAgentIdOverride` from a context engine plugin's `llm` config and passes those booleans into `resolveContextEngineCapabilities` runtime LLM authority.

## Best Possible Solution

## Reproduction Assessment

Unclear for the original end-to-end LCM failure because no gateway or Lossless compaction run was performed. The PR-introduced allowlist regression is source-reproducible by configuring `llm.allowModelOverride: true` with a narrow `llm.allowedModels` list, then requesting a different model.
