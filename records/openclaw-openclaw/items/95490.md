## What This Changes

The issue asks OpenClaw to prevent a Slack/CLI follow-up race from making a persisted CLI assistant final disappear from Slack without delivery or suppression metadata.

## Best Possible Solution

Maintain one invariant for normal CLI assistant finals: a transcript row that looks visible must either be delivered to Slack or carry explicit suppressed-delivery metadata, while preserving stale-reply suppression for genuinely superseded turns.
