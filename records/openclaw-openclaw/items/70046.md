## What This Changes

The PR adds `HH:MM[:SS]` parsing for `cron --at`, applies `--tz` to those time-only values, and updates cron CLI help, docs, and tests.

## Best Possible Solution

Leave current main's ISO/duration `--at` contract in place unless a maintainer sponsors time-only syntax; if sponsored, this scoped parser/docs approach is the right branch to revive or land.

## Reproduction Assessment

## Solution Assessment
