## What This Changes

The issue reports that Dreaming Advanced UI surfaces labelled “From Daily Log” are backed by `groundedCount` instead of `dailyCount`.

## Best Possible Solution

Align the Daily Log Review summary, staged list, origin labels, section action, and regression tests around `dailyCount`-backed entries, or intentionally rename the UI if grounded replay is the desired concept.
