## Best Possible Solution

## Reproduction Assessment

Yes. Source inspection gives a high-confidence path: emit an assistant `text_delta` before phase/tool metadata is known while partial replies or `text_end` block delivery are enabled, then classify the same assistant turn as commentary/tool-use later.

## Solution Assessment

No. The handler is the right surface, but clearing buffers only after classification is too late for already emitted chunks; the safer fix is to quarantine phase-pending text before visible callbacks and align tool-call detection with the current structured-tool contract.
