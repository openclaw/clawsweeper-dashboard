## What This Changes

The issue asks OpenClaw's stdio MCP server to answer `tools/list` requests with matching JSON-RPC responses so Hermes and other standard MCP clients can discover tools.

## Best Possible Solution

Route MCP serve stdout isolation before CLI config/bootstrap preflight, prove clean stdio plus a normal `tools/list` response, and land or replace the linked PR.
