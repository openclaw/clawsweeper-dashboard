- **[P1] Make feed builds deterministic by default:** `extensions/feeds/src/cli.ts:388`
  - body: `feeds build` is advertised as producing deterministic curated feed artifacts, but `generatedAt` defaults to `new Date().toISOString()`.   - confidence: 0.91

  - body: The PR body advertises `feeds validate <path-or-url>` and `feeds hash <path-or-url>`, but validation reads the argument with `readFeedFile`, so `https://...` and `file://...` inputs fail as local filesystem paths even though feed sources support those URL schemes. Route these commands through URL-aware loading or make the contract path-only consistently.
  - confidence: 0.84
