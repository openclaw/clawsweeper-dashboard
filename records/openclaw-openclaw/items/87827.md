- **[P1] Make feed builds deterministic by default:** `extensions/feeds/src/cli.ts:388`
  - body: `feeds build` is advertised as producing deterministic curated feed artifacts, but the default output includes `new Date().toISOString()`.   - confidence: 0.92
- **[P2] Align validate/hash inputs with the public contract:** `extensions/feeds/src/cli.ts:849-851`
  - body: The PR describes `feeds validate` and `feeds hash` as accepting a path-or-URL, and the sibling feed loader supports `file:` and `https:` sources. These commands call `readFeedFile`, which passes the raw string to `fs.readFile`, so URL inputs fail as path strings; share the feed URL loader or narrow the public contract to local paths.
  - confidence: 0.9
