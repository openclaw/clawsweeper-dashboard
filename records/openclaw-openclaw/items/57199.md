  - body: The handler starts passing `agentId` into `context.cron.wake`, but current main's `CronServiceContract.wake` only accepts `mode`, `text`, and optional `sessionKey`. A refreshed patch needs to widen the contract and generated/adapted surfaces before sending this field across the boundary.

  - confidence: 0.88

  - confidence: 0.84
