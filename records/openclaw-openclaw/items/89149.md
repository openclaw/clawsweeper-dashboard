## Summary

Keep open for maintainer handling. ## What This Changes

The branch validates session extension tool names before registry insertion and adds regression coverage for non-string and throwing `name` accessors.

## Best Possible Solution

## Solution Assessment

Is this the best way to solve the issue? Unclear until maintainers choose the contract: `registerTool` is the right boundary for validation, but the newer related PR offers a less disruptive per-tool quarantine approach.
