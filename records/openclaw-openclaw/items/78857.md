- **[P1] Preserve configured-agent child sessions:** `src/gateway/server-methods/sessions.ts:124-129`
  - body: With the UI opting into configuredAgentsOnly, this filter only checks the row's own key. Current main also keeps rows whose spawnedBy or parentSessionKey belongs to a configured agent, so rebasing this logic as-is would hide valid spawned or ACP child sessions.
  - confidence: 0.91

  - body: This rewrite emits every native name as a plain command spec and drops isAlias. Telegram command-menu budgeting uses that flag to reserve slots for canonical commands before aliases, so aliases can displace canonical slash commands near the menu limit.
  - confidence: 0.89

  - body: The indexed queue rewrite is based on older resolver logic and omits duplicateToolCallIdStyle: "openai" plus the call_ allocation path. Merging it would stop preserving the first OpenAI-style duplicate ID and generating provider-style IDs for later repeats.
  - confidence: 0.88

  - body: Current main no longer has this local OpenAI WebSocket stream; active provider and plugin streams use the shared EventStream, which still shifts queued events and waiters. Put the index compaction there or split this part so the hot-path optimization reaches the live path.
  - confidence: 0.82
