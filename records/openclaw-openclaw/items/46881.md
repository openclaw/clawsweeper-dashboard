- **[P1] Read the reaction actor from Baileys reaction.key:** `extensions/whatsapp/src/inbound/monitor.ts:453-459`

  - confidence: 0.91

  - body: Calling `onMessage` for every emoji reaction makes a reaction create a normal agent turn or reply path. The remaining design needs an explicit hook/agent-turn policy and tests before changing default WhatsApp behavior.
  - confidence: 0.82
- **[P2] Clear reaction dedupe when a reaction is removed:** `extensions/whatsapp/src/inbound/monitor.ts:443-459`
  - body: The empty-emoji removal path returns before the dedupe key is computed, so removing and re-applying the same emoji within the TTL can leave the second real reaction hidden from the agent.
  - confidence: 0.84
