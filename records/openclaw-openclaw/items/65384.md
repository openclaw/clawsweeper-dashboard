## What This Changes

## Best Possible Solution

Land this PR or an equivalent Feishu-owned doctor normalizer after the regression test exercises the current contract path and the contributor adds redacted real doctor/config validation proof.

## Reproduction Assessment

## Solution Assessment

Yes for the implementation direction, but no for the current patch as merge-ready. A Feishu plugin doctor adapter exposed through the existing contract is the narrow maintainable fix, but the test and real behavior proof still need follow-up.
