## Summary

Keep open. Current main still lacks the requested public `reactionGuidance` config/schema/docs surface and incoming soft-CTA prompt guidance, and prior maintainer-side review in the discussion already identified the same remaining gaps.

## What This Changes

Add configurable Telegram reaction prompt guidance, gated by Telegram reaction notifications, with outbound sparing-use guidance and inbound reaction interpretation.

## Best Possible Solution

Define the public Telegram reaction-guidance contract, then update Telegram config types, strict schema, generated metadata, docs, prompt composition, notification gating, and focused tests for `off`, `minimal`, `full`, and inbound reaction interpretation.

## Reproduction Assessment

Not applicable as a bug reproduction. Source, docs, generated metadata search, and related-item context show the requested config key and inbound soft-CTA prompt guidance are absent on current main.

## Solution Assessment

Unclear until maintainers choose the public config shape.
