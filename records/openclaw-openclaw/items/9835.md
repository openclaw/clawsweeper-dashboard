## Summary

## What This Changes

The issue requests bidirectional Telegram reaction prompt guidance plus a configurable `reactionGuidance` option for Telegram.

## Best Possible Solution

Define one Telegram reaction-guidance contract that composes `reactionLevel`, `reactionNotifications`, and wake policy, then update the Telegram plugin prompt adapter, shared prompt text, config schema/metadata/docs, and focused tests.

## Reproduction Assessment
