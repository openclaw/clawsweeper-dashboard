Define the public Telegram reaction guidance contract, then implement it across the bundled Telegram plugin prompt adapter, shared prompt builder, strict schema, generated metadata, docs, and focused tests while coordinating with https://github.com/openclaw/openclaw/issues/70043 and the open wake-policy PRs.
