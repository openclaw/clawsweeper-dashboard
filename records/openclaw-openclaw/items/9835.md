## Summary

Keep open. Current main still has partial Telegram reaction support, but the requested `reactionGuidance` config, notification-gated prompt guidance, and incoming reaction soft-CTA guidance are still absent.

## What This Changes

Add Telegram reaction prompt guidance for sending and interpreting reactions, gated by `reactionNotifications`, with a configurable `reactionGuidance` option.

## Best Possible Solution

Define the public Telegram reaction-guidance semantics first, then implement the chosen behavior through the bundled Telegram plugin prompt resolver, shared system prompt text, strict schema, generated metadata, docs/changelog, and focused tests.

## Reproduction Assessment

Not applicable as a bug reproduction path. Source and docs inspection on current main give high confidence that the requested public config key, notification-gated prompt behavior, and incoming reaction guidance are still missing.

## Solution Assessment

Unclear until maintainers choose the public config shape. The narrow maintainable path is to settle how `reactionGuidance` composes with `reactionLevel` and `reactionNotifications`, then update Telegram and prompt composition consistently.
