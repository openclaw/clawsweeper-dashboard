## Summary

Keep open: current main still has Telegram reaction notifications and outbound reaction prompt guidance, but it does not implement the requested public `reactionGuidance` config/schema/docs contract or incoming-reaction soft-CTA prompt guidance.

## What This Changes

Request to add Telegram two-way reaction prompt guidance, gated by reaction notification settings, plus a new `reactionGuidance: off|minimal|full` config option.

## Best Possible Solution

Define the public Telegram reaction guidance contract, then implement it across the bundled Telegram plugin prompt adapter, shared prompt builder, strict schema, generated metadata, docs, and focused tests while coordinating with https://github.com/openclaw/openclaw/issues/70043 and the open wake-policy PRs.
