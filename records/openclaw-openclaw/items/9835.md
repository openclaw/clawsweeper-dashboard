## Summary

Keep open: current main has partial Telegram reaction support, but the requested `reactionGuidance` config, notification-gated prompt behavior, and incoming reaction soft-CTA guidance are still missing.

## What This Changes

Add configurable Telegram reaction prompt guidance with outbound reaction advice and incoming reaction interpretation when Telegram reaction notifications are enabled.

## Best Possible Solution

Define how `reactionGuidance` composes with `reactionLevel` and `reactionNotifications`, then update the Telegram plugin, shared prompt builder, schema/metadata/docs, and focused tests together.
