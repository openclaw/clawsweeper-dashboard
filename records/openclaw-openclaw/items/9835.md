## Summary

Keep open. Current main still has partial Telegram reaction support, but the requested public `reactionGuidance` option, notification-gated prompt behavior, and incoming reaction soft-CTA guidance are still absent.

## Best Possible Solution

Define the public Telegram reaction-guidance contract, then implement it through the bundled Telegram plugin prompt resolver, shared system prompt, strict schema, generated metadata, docs/changelog, and focused tests.

## Solution Assessment
