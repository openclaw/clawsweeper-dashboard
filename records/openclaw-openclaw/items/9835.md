## Summary

Keep open: current main still only partially covers Telegram reactions. The requested public `reactionGuidance` config/schema/docs surface, prompt gating by `reactionNotifications`, and incoming reaction soft-CTA guidance are still missing, and the related open issue/PR address adjacent defaulting and wake behavior rather than superseding this request.
