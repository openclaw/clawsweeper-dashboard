Keep open. Current main only has partial Telegram reaction support; the requested public `reactionGuidance` option, notification-gated prompt behavior, and incoming reaction soft-CTA guidance are still absent.

## Best Possible Solution

Define the public Telegram reaction-guidance contract first, then implement it consistently through the Telegram plugin prompt resolver, shared system prompt, strict schema, generated metadata, docs, changelog, and focused tests.
