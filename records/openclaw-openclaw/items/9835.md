Keep open: current main and v2026.6.8 still provide Telegram reaction events and outbound prompt guidance through reactionLevel, but the requested reactionGuidance config surface, reactionNotifications-gated prompt policy, and incoming soft-CTA guidance are still missing.

## What This Changes

Request adds Telegram reaction prompt guidance for sending and receiving reactions, gated by reactionNotifications and configurable through a new reactionGuidance option.

## Best Possible Solution

Define one canonical Telegram reaction policy, then update the bundled Telegram plugin prompt resolver, config schema, generated metadata, docs, and focused tests so notification gating and prompt guidance compose predictably.
