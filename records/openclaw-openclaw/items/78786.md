## What This Changes

Heartbeat poll transcript files are reported to accumulate under agent `sessions/` without matching `sessions.json` entries.

## Best Possible Solution

Pin the heartbeat path that writes a primary transcript, then ensure that transcript is either registered in `sessions.json` or intentionally written outside the primary transcript scan with matching doctor behavior.

## Solution Assessment

Unclear until the failing path is pinned. The narrow maintainable fix is to keep heartbeat transcript creation on the existing session registration path rather than relying on external archival cron cleanup.
