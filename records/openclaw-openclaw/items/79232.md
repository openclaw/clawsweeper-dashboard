## Summary

Keep open: current main and v2026.5.7 still generate the macOS Gateway LaunchAgent service PATH without Apple Silicon Homebrew dirs, and two open PRs already reference this issue as their closing fix.

## What This Changes

Add `/opt/homebrew/bin` and `/opt/homebrew/sbin` to the generated macOS Gateway LaunchAgent service PATH so Homebrew-managed runtimes survive install and restart rewrites.

## Best Possible Solution

Review and land one linked PR that adds the Homebrew dirs to the canonical Darwin service PATH, updates the service-env/install-plan tests and docs, and includes redacted macOS LaunchAgent restart proof.
