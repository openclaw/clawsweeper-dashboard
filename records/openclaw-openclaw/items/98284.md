## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Clear stale cookie snapshots on successful empty saves:** `extensions/browser/src/browser/cookie-persistence.ts:51-52`
  - body: A successful empty cookie list is a real browser state after logout or clearing site data. Returning here leaves the previous `cookies.json` intact, so the next launch can restore old live cookies and undo the user's cleared session; distinguish command failure from an empty successful snapshot and delete or overwrite the sidecar.
  - confidence: 0.93
- **[P2] Stop cookie flushing when managed Chrome exits:** `extensions/browser/src/browser/chrome.ts:1124-1126`
  - body: This starts a periodic flush timer, but the current process-exit handler only clears `ProfileRuntimeState.running`. If Chrome exits or crashes outside `stopOpenClawChrome`, the old timer keeps firing against a stale WebSocket and each later launch adds another interval, so wire timer cleanup into the managed Chrome exit path.
  - confidence: 0.86
