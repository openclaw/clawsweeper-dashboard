This PR adds an opt-in `agents.defaults.bootstrapTier` minimal mode, loopback local-model doctor guidance, config schema/generated metadata/docs, changelog, and focused tests.

## Best Possible Solution

Land one maintainer-approved bootstrap-context control that preserves `standard` behavior, keeps delegated and hook paths tier-aware, and aligns doctor guidance, schema, docs, tests, and real proof with the constrained-local-model issue.

## Reproduction Assessment

No, not as-is. The config knob and doctor warning are plausible, but the implementation must preserve delegated/background semantics, make hook-added files tier-aware, and settle the public API overlap before merge.
