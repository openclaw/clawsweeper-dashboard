## What This Changes

The issue requests configurable font size or font scaling for the OpenClaw web UI/chat interface.

## Best Possible Solution

Add a bounded browser-local Control UI text-size or font-scale setting in UiSettings, expose it in Appearance and Quick Settings, apply it through CSS variables across chat and dense UI surfaces, and document any gateway config only if maintainers approve that public contract.
