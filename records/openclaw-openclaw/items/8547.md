## Summary

Keep open. Current main still lacks a Control UI/WebChat font-size or font-scale setting in browser-local settings, Appearance/Quick Settings, gateway `ui` config, exported config types, and public docs; the closed related issues do not supersede this accessibility feature request.

## What This Changes

Request to add configurable font size or font scale controls for the WebChat/Control UI, possibly through gateway `ui` config.

## Best Possible Solution

Implement a bounded browser-local text-size or font-scale setting applied through CSS variables across chat, compose input, and surrounding Control UI; extend gateway `ui` schema/types/docs only if maintainers choose shared server-side sizing.

## Reproduction Assessment
