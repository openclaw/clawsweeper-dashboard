## Summary

Keep open. Current main still lacks a user-adjustable Control UI/WebChat font-size or text-scale setting, and the latest MEMBER comment explicitly says this remains a valid accessibility/customization enhancement rather than a stale or duplicate item.

## What This Changes

The issue requests configurable font size or font scaling for the OpenClaw web chat interface.

## Best Possible Solution

Implement a bounded browser-local Control UI text-scale setting first, expose it in Appearance and Quick Settings, apply it through shared CSS custom properties across chat and dense UI surfaces, and add a public gateway config contract only after explicit maintainer approval.
