## Summary

Keep open: current main has a Markdown download for the loaded Control UI chat, but the requested full session transcript export, plain-text option, and optional scriptable export endpoint are not implemented. The earlier combined report was intentionally split into focused search and export issues, so this remains a valid feature request rather than stale cleanup.

## What This Changes

Add a Control UI way to download a full session conversation as Markdown or plain text, with an optional export endpoint for automation.

## Best Possible Solution

Define one scoped Control UI transcript export path that reuses existing chat/session history projection where appropriate, adds explicit Markdown/plain-text behavior, and settles auth, size, and privacy rules before any optional API endpoint is added.

## Reproduction Assessment

Not applicable; this is a feature request rather than broken existing behavior. Source and docs inspection show the current UI can export loaded chat messages as Markdown, but that path is bounded and not a full transcript export/API.

## Solution Assessment

Unclear pending maintainer direction; the current Markdown button is a useful partial implementation, but the requested full-session export needs a deliberate UI/API/privacy design before implementation.
