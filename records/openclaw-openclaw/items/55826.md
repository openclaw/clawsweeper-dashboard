  - body: The audit line uses `step.label`, but a step labeled `tools.allow` can include both `allow` and `deny`. A deny removal can therefore be reported as an allow-list rule instead of the matched deny pattern, so policy evaluation needs to carry matched allow/deny rule metadata before logging.
  - confidence: 0.91
- **[P2] Move sandbox audit logging out of the formatter:** `src/agents/sandbox/runtime-status.ts:160-162`

  - confidence: 0.87
