## Summary

Keep open. Current main and v2026.6.5 provide searchable session selection, single-session sending, parent/child session metadata, and checkpoint branching, but not project trees, multi-target composer routing, arbitrary selected-message split/move, or cross-project context seeding; adjacent open items cover only parts of this product surface.

## What This Changes

Add project-organized session trees, routed or fanned-out chat entry points, message split/move triage, and cross-project context seeding to Control UI.

## Best Possible Solution

Adopt one canonical Projects/session hierarchy, then define additive storage and Gateway contracts for branch-copy or move operations and multi-target delivery before implementing the Control UI tree and triage actions.
