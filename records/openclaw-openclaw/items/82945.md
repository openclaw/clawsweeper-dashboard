## Best Possible Solution

Land a conservative raw-update redactor that covers every default allowed Telegram update family, with explicit regression cases for users, chats, callbacks, business/join/reaction, payment/shipping, and passport payloads.

## Solution Assessment

No. Extracting a formatter is the right shape, but the current key coverage is still too narrow for Telegram's default payment/shipping and passport payload surfaces.
