Overall confidence: 0.86

Full review comments:

- **[P1] Honor explicit OpenClaw runtime pins before CLI inference:** `src/auto-reply/reply/agent-runner-execution.ts:1687-1696`
  - body: `resolveSessionRuntimeOverrideForProvider` now returns `openclaw` for persisted built-in runtime pins, but this branch still falls through to `resolveCliRuntimeExecutionProvider` unless the override is itself a CLI provider. That lets auth-profile inference reroute a session explicitly pinned to OpenClaw onto a CLI backend such as `claude-cli`; short-circuit the built-in runtime to the original provider before CLI inference.
  - confidence: 0.9
- **[P2] Preserve casing for custom harness ids:** `src/agents/agent-runtime-id.ts:30-31`
  - body: `normalizeOptionalAgentRuntimeId` lowercases every unknown runtime id before selection, but harness registration and lookup are case-sensitive on current main. Only normalize known aliases such as `pi` and `codex-app-server`; otherwise mixed-case plugin harness ids that previously worked via `agentRuntime.id` or overrides will fail as missing harnesses.
  - confidence: 0.86
- **[P1] Use graceful process-tree termination:** `packages/agent-core/src/harness/env/nodejs.ts:226-241`
  - body: The new agent-core Node execution environment adds a local `killProcessTree` that sends immediate `SIGKILL` or `taskkill /F` for timeouts and aborts. OpenClaw already has a shared graceful kill-tree path that sends SIGTERM before escalation; using the immediate helper here can skip cleanup and repeats the security-sensitive termination pattern flagged by code scanning.
  - confidence: 0.82
- **[P2] Normalize the legacy QA runtime override:** `src/agents/model-runtime-policy.ts:184-186`
  - body: This gate now accepts only `OPENCLAW_QA_FORCE_RUNTIME=openclaw|codex`, so existing private QA lanes that still export the shipped `pi` value silently lose the forced built-in runtime policy. Since the migration keeps `pi` as a deprecated alias elsewhere, normalize it here too before enforcing the allowed values.
  - confidence: 0.8
