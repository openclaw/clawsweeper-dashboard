- **[P1] Normalize legacy `pi` to the built-in runtime:** `src/agents/agent-runtime-id.ts:20`

  - body: `isPathInside(candidatePath, trackedPath)` lets a config/global candidate inside a recorded official install path inherit trusted official install status. Keep this exact or only allow the recorded install to be inside the candidate root.
  - confidence: 0.9
- **[P1] Restore manifest-backed env auth lookup:** `src/llm/env-api-keys.ts:121-164`
  - body: The fixed map misses provider env vars still declared by plugin manifests, so `findEnvKeys` and `getEnvApiKey` can report env-only providers as unauthenticated. Keep a manifest/setup-provider fallback instead of hardcoding every provider id here.
  - confidence: 0.92

  - body: `openclaw/plugin-sdk/provider-usage` no longer exports `resolveLegacyPiAgentAccessToken`, which current main ships. Third-party plugins importing the public symbol can fail immediately on upgrade; keep a deprecated alias until a documented removal window.
