Full review comments:

- **[P1] Derive env auth keys from plugin manifests:** `src/llm/env-api-keys.ts:131-160`
  - body: Hardcoding this provider map drops manifest-declared env credentials for bundled providers that are not listed, for example `arcee` declares `ARCEEAI_API_KEY`. `AuthStorage.hasAuth`/`getAuthStatus` and OpenAI-compatible transports call `getEnvApiKey(model.provider)`, so env-only installs can be reported unauthenticated or miss request credentials; keep a manifest/setup-provider fallback instead of relying on this fixed table.
  - confidence: 0.92
- **[P1] Restore Mistral proxy transport compat:** `extensions/mistral/index.ts:46-47`
  - body: Removing the Mistral provider's cross-provider compat contribution also deletes the test that covered `openrouter`/custom Mistral-family routes.   - confidence: 0.9
