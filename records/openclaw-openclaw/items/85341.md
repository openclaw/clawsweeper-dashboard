- **[P1] Use plugin-declared env credential aliases:** `src/llm/env-api-keys.ts:148-180`

  - body: `ProviderPlugin` is still exported through the plugin SDK, but this shape removes the shipped `contributeResolvedModelCompat` hook rather than keeping a deprecated alias/application path.   - confidence: 0.9

  - body: This resolver now honors only `OPENCLAW_CACHE_RETENTION`, while the latest release still used `PI_CACHE_RETENTION` for implicit long cache retention.
