  - body: The added reverse containment check lets a config-scoped candidate inside an installed official plugin directory match the install record, so `trustedOfficialInstall` can be applied to an unverified subtree.   - confidence: 0.87

  - body: Returning `undefined` for any provider ID missing from this hardcoded map drops manifest-declared env credentials for bundled and third-party providers, which makes env-only installs look unauthenticated and can leave request fallbacks without the expected key.   - confidence: 0.9

  - body: This public SDK subpath no longer exports `resolveLegacyPiAgentAccessToken`, so existing provider plugins that still import the shipped symbol fail to build or load immediately after upgrade. Re-export it as a deprecated alias until the documented removal window passes.
  - confidence: 0.9

  - body: The plugin now only normalizes native Mistral models and no longer contributes Mistral request-shape compat for hinted OpenAI-compatible routes such as OpenRouter Mistral models.
