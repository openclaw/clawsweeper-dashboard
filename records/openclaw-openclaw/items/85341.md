  - body: The added `isPathInside(candidatePath, trackedPath)` branch treats a parent candidate path as matching a recorded official install. That broadens `trustedOfficialInstall` beyond the recorded package root and can apply official-only behavior to an unverified subtree or parent path.

  - body: `getApiKeyEnvVars` is now a fixed map, while auth status depends on `findEnvKeys` and plugins still declare provider env vars in manifests.   - body: `normalizeEmbeddedAgentRuntime` returns `pi` unchanged, so old persisted `agentHarnessId` or runtime overrides are treated as a missing plugin runtime instead of the built-in OpenClaw runtime. Map `pi` to `openclaw` in this compatibility normalizer.
