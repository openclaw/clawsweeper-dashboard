  - body: The new `envMap` is a competing hard-coded source of truth and omits shipped provider IDs whose manifests declare env auth, including `moonshot`, `qianfan`, `nvidia`, `stepfun`, `deepinfra`, and `microsoft-foundry`.   - confidence: 0.9
- **[P2] Keep legacy agent-dir env fallback:** `src/plugin-sdk/agent-dir-compat.ts:9`
  - body: This helper is explicitly kept for third-party plugin SDK compatibility, but the PR now ignores `PI_CODING_AGENT_DIR`.
