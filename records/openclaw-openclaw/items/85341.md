Keep `pi` as a deprecated alias for `openclaw` in this normalizer and its public type surface.
  - confidence: 0.91

  - body: `OPENCLAW_QA_FORCE_RUNTIME=pi` is accepted on current main, but this branch ignores it and falls back to normal model runtime selection. Private QA lanes using the shipped legacy value can silently stop exercising the intended built-in runtime.
  - confidence: 0.88

  - confidence: 0.86

  - confidence: 0.87
