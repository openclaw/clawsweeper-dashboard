  - body: `isPathInside(candidatePath, trackedPath)` treats a candidate path that is an ancestor of the recorded official install as trusted. A config-scoped plugin path should not inherit official-install trust unless it is the recorded package root or lives inside that recorded root.
  - confidence: 0.91
