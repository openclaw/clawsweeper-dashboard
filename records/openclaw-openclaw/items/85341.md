Full review comments:

- **[P1] Preserve plugin runtime ID casing:** `src/agents/agent-runtime-id.ts:30-31`
  - body: `normalizeOptionalAgentRuntimeId` lowercases every runtime id before selection, but plugin harness ids are registered and looked up case-sensitively. A previously working mixed-case custom harness id will be changed to lowercase and then fail as a missing harness.
  - confidence: 0.9
- **[P1] Honor OpenClaw runtime pins before CLI inference:** `src/auto-reply/reply/agent-runner-execution.ts:1678-1689`
  - body: An explicit persisted `agentRuntimeOverride: "openclaw"` now falls through to `resolveCliRuntimeExecutionProvider`. If auth/profile inference picks a CLI backend, the turn can run on the wrong runtime despite the session pin; keep the embedded-runtime short-circuit that existed for `pi`.
  - confidence: 0.9
- **[P1] Honor follow-up runtime pins before CLI inference:** `src/auto-reply/reply/followup-runner.ts:579-590`
  - body: The queued follow-up runner has the same CLI fall-through, so follow-up turns from a session pinned to the built-in runtime can be rerouted to a CLI backend. This needs the same explicit OpenClaw short-circuit as the main agent-runner path.
  - confidence: 0.88
- **[P1] Reuse graceful process-tree termination:** `packages/agent-core/src/harness/env/nodejs.ts:226-245`
  - body: The new agent-core helper sends immediate SIGKILL/taskkill `/F` for timeouts and aborts. OpenClaw already has a shared helper that sends SIGTERM first and only force-kills after a grace period; duplicating the hard-kill path regresses cleanup and process-boundary behavior.
  - confidence: 0.87
- **[P2] Keep legacy Pi provider embedded in error context:** `src/agents/embedded-agent-runner/run/helpers.ts:104-106`
  - body: `isEmbeddedHarnessProvider` now recognizes only `openclaw`, but pre-rename session history can still carry assistant messages tagged with provider `pi`. Failure paths that inspect the last assistant can misclassify failover/cooldown context unless `pi` remains a deprecated embedded-runtime alias here.
  - confidence: 0.84
