Full review comments:

- **[P1] Keep official install matching one-way:** `src/plugins/manifest-registry.ts:785`
  - body: Adding `isPathInside(candidatePath, trackedPath)` makes a candidate parent directory match the recorded official install path. That result is used before `trustedOfficialInstall` is assigned, so a config/global candidate at a parent path can be treated as the installed official package root and receive official-only capabilities. Keep the match to exact paths or tracked install/source paths containing the candidate package path, not the reverse.
  - confidence: 0.88

  - body: This removes the public `ProviderPlugin.contributeResolvedModelCompat` hook and its runtime application path.
