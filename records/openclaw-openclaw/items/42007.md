## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Propagate keyword-triggered state into broadcast dispatch:** `extensions/feishu/src/bot.ts:1053-1057`
  - body: When this gate lets a keyword-only group message through, `ctx.mentionedBot` remains false. The broadcast path later computes `activeAgentId` and `WasMentioned` from `ctx.mentionedBot`, so broadcast groups dispatch every agent as an observer and create no Feishu reply dispatcher for the triggered turn.
  - confidence: 0.91
- **[P2] Reject empty mention triggers before matching:** `extensions/feishu/src/config-schema.ts:160`
  - body: The schema accepts `""`, and the runtime uses `ctx.content.includes(t)`. A blank trigger therefore matches every group message and bypasses `requireMention`, so empty or whitespace-only entries need to be rejected or normalized before matching.
  - confidence: 0.92
