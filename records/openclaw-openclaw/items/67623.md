The PR changes `src/agents/sandbox/backend.ts` so sandbox backend factories are stored in a `Symbol.for` process-global map via `resolveGlobalMap` instead of a module-local `Map`.

## Best Possible Solution

## Solution Assessment

Yes on code direction, but not yet merge-ready on evidence. A process-global registry is the narrow maintainable fix; it should be paired with split-module/Jiti regression coverage and real external backend proof.
