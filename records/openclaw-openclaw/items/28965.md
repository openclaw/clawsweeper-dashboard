## Summary

Keep open: current main still lacks the requested `openclaw models reconcile` and bulk `openclaw sessions model-reset` operator surfaces. Existing docs, `models status`, session reset behavior, and `doctor --fix` cover adjacent pieces, but they do not settle the public model-state contract or supersede this umbrella proposal.

## What This Changes

The issue proposes a single-source model configuration reconcile command, an explicit session model override reset command, and clearer model-source observability.

## Best Possible Solution
