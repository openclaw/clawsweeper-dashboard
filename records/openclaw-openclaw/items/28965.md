Request to add model configuration reconciliation, bulk session model/auth override reset, and clearer model-source observability.

## Best Possible Solution

## Solution Assessment

No, not as one broad automatic reset/reconcile change. The safer solution is a source-aware resolver plus staged CLI, doctor, docs, and status behavior after maintainers decide which selections should be preserved, warned about, or reset.
