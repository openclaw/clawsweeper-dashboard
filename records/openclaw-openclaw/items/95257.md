  - body: This reads `getPluginRegistryState()?.activeRegistry` and appends its routes to this gateway server's route registry.   - confidence: 0.9

  - confidence: 0.88
- **[P2] Avoid duplicating startup routes:** `src/gateway/server-runtime-state.ts:145-146`
  - body: On normal startup the process-global active registry is the same object as the base gateway registry, so concatenating `base.httpRoutes` with `activeHttpRoutes` duplicates every route. Fallthrough or prefix handlers that return `false` can then be considered twice, and route matching no longer reflects the registry's conflict checks.
  - confidence: 0.82
