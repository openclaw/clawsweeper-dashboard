## Label Justifications

- `P1`: Multiple independent deployments report compaction failing, looping, or replacing real session history with empty summaries across active channel workflows.
- `impact:data-loss`: The reported empty fallback path can replace usable conversation history with a placeholder summary, losing session context.
- `impact:session-state`: The issue centers on session transcript, compaction preparation, and cached token state disagreeing about the same conversation.

## Summary

Keep open: current main and the latest shipped release still contain the direct empty-session skip, the safeguard empty-boundary fallback, and positive-only token accounting that match the reported compaction failure class; related fixes only partially mitigate it and open PRs still cover remaining work.

## What This Changes

The issue asks OpenClaw to compact real persisted conversation history instead of skipping or writing empty fallback summaries when compaction preparation/session message arrays appear empty.

## Best Possible Solution

Preserve the empty-session guard for genuine idle sessions, but make compaction use trusted persisted transcript/token-pressure state so real over-threshold histories summarize and zero-token compaction results clear stale session accounting.

## Reproduction Assessment

Yes, source-backed but not live: current main still has a path where a high-token session whose limited `session.messages` fails the real-conversation classifier returns before compaction, plus a safeguard path that writes the reported empty boundary when preparation and branch fallback both fail. The issue comments provide concrete shipped-runtime repros, but I did not replay a live session in this review.

## Solution Assessment

No to the report's workaround of deleting the guards, because true empty sessions still need a no-provider-dispatch path. The maintainable fix is an invariant and accounting repair, likely by landing or replacing the focused related PRs with regression coverage for manual and automatic compaction.
