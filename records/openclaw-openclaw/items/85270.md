## What This Changes

The PR tags native alias command specs, budgets Telegram menu entries canonical-command-first with aliases only in leftover slots, strips alias metadata from registered menu payloads, and adds focused regression tests.

## Best Possible Solution

Land the focused canonical-first menu-budgeting fix, with an explicit maintainer decision on whether alias metadata belongs in the public NativeCommandSpec contract or an internal Telegram-only helper.

## Solution Assessment
