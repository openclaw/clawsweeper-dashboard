- Inbound SMS would expose private message content to agents, so permission prompts, sender allowlists, phone-number normalization, logging/redaction, and retention need explicit security review.

- The ownership boundary is unresolved: Android receiver support must live in the node app, while gateway routing may belong in a bundled channel, plugin-backed channel, or a new generic node-event seam.
