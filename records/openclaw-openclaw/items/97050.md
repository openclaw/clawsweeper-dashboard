Keep `.npmrc` out of the runtime COPY. ([docs.docker.com](https://docs.docker.com/reference/dockerfile/?utm_source=openai))
  - confidence: 0.94

  - body: Dropping the separate `FROM base-runtime` makes the named base stage become the full runtime image and changes a tested stage/target boundary. Preserve the final runtime stage unless maintainers approve the compatibility change with matching tests/docs.
  - confidence: 0.92
- **[P2] Align the browser install assertion:** `src/dockerfile.test.ts:97`
