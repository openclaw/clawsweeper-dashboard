## Summary

Keep open: current main still sends the final Telegram `audioAsVoice` TTS payload separately after a streamed preview, while this branch applies a narrow Telegram dispatch fix with inspectable screenshot proof and no remaining diff-level correctness or security findings.

## Best Possible Solution

## Solution Assessment
