Fix the Feishu streaming-card delivery bugs tracked by https://github.com/openclaw/openclaw/issues/77685. Work in TypeScript source, not built `node_modules` output. Keep the repair inside the Feishu plugin boundary: `extensions/feishu/src/reply-dispatcher.ts` and `extensions/feishu/src/streaming-card.ts`, with focused tests in the colocated test files.
