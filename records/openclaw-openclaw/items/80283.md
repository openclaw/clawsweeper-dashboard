## Summary

Keep open: this is an active, focused bug-fix PR and current main does not yet cover the central watchdog gap. Current main clears the cron pre-model watchdog only after `model_call_started` progress, but the Codex app-server `turn/start` dispatch path and embedded attempt/tool-start paths do not consistently emit or forward that progress; the PR addresses those paths and includes convincing live after-fix proof.
