  - body: When `activityStatus` remains plain `"error"`, `lastActivityStatus` is also plain `"error"`, which does not start with `"error:"`; a connection-status rerender then clears `errorElapsed` and drops the suffix. Treat plain error as an error state when preserving cached elapsed time.
  - confidence: 0.89

  - confidence: 0.82
