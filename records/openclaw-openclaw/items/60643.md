## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Gate The Angle out of minimal prompts:** `src/agents/system-prompt.ts:426`
  - body: The docs say `promptMode: "minimal"` omits `The Angle`, but this section is inserted into the unconditional prompt prefix before `## Tooling`. Minimal sub-agent prompts will still carry the full-mode framing unless the section is gated on `!isMinimal` or the docs/tests intentionally change that contract.
  - confidence: 0.91
- **[P2] Make the MEMORY.md template match the line-1 rule:** `extensions/memory-core/src/prompt-section.ts:26-30`
  - body: The prompt says line 1 of `MEMORY.md` is always the recovery pointer, but the template starts with `## recovery` and puts `[last-session: ...]` on the next line. Agents following the template can produce a file shape that violates the stated recovery contract.
  - confidence: 0.89
- **[P2] Do not classify package installs as reversible:** `src/agents/system-prompt.ts:499`
  - body: This global prompt tells agents to execute installs without hesitation as reversible operations. Package installs can run lifecycle scripts and change dependency resolution, so the prompt should keep installs in the sensitive/verify-first bucket instead of treating them like ordinary reversible edits.
  - confidence: 0.78
