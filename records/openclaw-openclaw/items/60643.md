Full review comments:

- **[P2] Gate The Angle out of minimal prompts:** `src/agents/system-prompt.ts:426`
  - body: The docs say minimal prompts omit The Angle, but the section is inserted unconditionally before Tooling. Minimal sub-agent prompts will still carry the full-mode framing unless the section is gated on `!isMinimal` or the docs/tests intentionally change that contract.
  - confidence: 0.91
- **[P2] Align the MEMORY.md template with the line-1 rule:** `extensions/memory-core/src/prompt-section.ts:29-30`
  - body: The prompt says line 1 of `MEMORY.md` is always the recovery pointer, but the template starts with `## recovery` and puts `[last-session: ...]` on the next line. Agents following the template can produce a file shape that violates the stated recovery contract.
  - confidence: 0.89
- **[P2] Keep installs out of the reversible fast path:** `src/agents/system-prompt.ts:502`
  - body: This global prompt tells agents to execute installs without hesitation as reversible operations. Package installs can run lifecycle scripts and change dependency resolution, so installs should remain in the sensitive or verify-first bucket instead of ordinary reversible edits.
  - confidence: 0.8
