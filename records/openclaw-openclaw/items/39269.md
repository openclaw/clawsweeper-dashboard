Full review comments:

- **[P1] Honor explicit completion routes from internal requesters:** `src/agents/subagent-announce.ts:780`
  - body: `directOriginIsInternal` is derived from the original requester route, not the effective completion route. When a webchat requester has a valid external `completionDirectOrigin`, this forces `deliver=false` and drops that explicit completion delivery.
  - confidence: 0.9
- **[P2] Filter mirror entries before applying the history limit:** `src/gateway/server-methods/chat.ts:749-750`
  - body: The PR slices `rawMessages` before removing `delivery-mirror` entries. If the newest entries are mirrors, `chat.history` can return fewer visible messages than requested even though older visible messages exist.
  - confidence: 0.88
- **[P2] Require install intent before silent plugin shadowing:** `src/plugins/manifest-registry.ts:240-245`
  - body: The origin-rank branch suppresses duplicate diagnostics and lets a different physical plugin copy win whenever origins differ. Current main only suppresses this for intentional installed-bundled duplicates, so this can silently change which plugin code executes.
  - confidence: 0.84
