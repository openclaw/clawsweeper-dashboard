## What This Changes

## Best Possible Solution

Keep this issue as the narrow canonical sanitizer item until maintainers either land a conservative current-path fix for glued final-tag repeats and explicitly supported adjacent duplicate shapes, or port and approve the broader visible-output normalization work as covering this exact case.
