Keep open. Current main still lacks bundled state-dir Node discovery, while this PR is a narrow, green implementation with a focused regression test and the requested changelog entry.

## What This Changes

The branch makes daemon runtime path selection scan for a validated bundled Node under the active state directory before current/system Node fallback, adds a regression test, and records a changelog entry.

## Best Possible Solution

Land the narrow bundled-Node preference after maintainer review while keeping the broader Homebrew runtime/package/cache drift from #75250 tracked separately.

## Solution Assessment

Yes. This is the narrowest maintainable fix for the service runtime path because it uses the documented state-dir tools layout, validates candidates before selection, and preserves current/system Node fallbacks.
