## What This Changes

The PR stores timeout handles and clears them in finally blocks for Synology Chat, Twitch, and Zalouser Promise.race timeout paths.

## Best Possible Solution

Refresh the PR to keep the Zalouser cleanup, leave Twitch unchanged, tighten or remove the Synology watchdog in coordination with the open configured-timeout issue, and add focused regression coverage.

## Reproduction Assessment

## Solution Assessment

Partially. The handle-plus-finally cleanup is the right narrow shape for Zalouser, but Synology should clear the race timer as soon as the race settles and should be coordinated with the hardcoded-timeout behavior.
