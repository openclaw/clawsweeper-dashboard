  - body: `just` recipe names are `NAME` tokens and cannot contain `:`; the colon starts the dependency list. The first `build:strict:` style recipe means `just --list` will not work as described until these recipes are renamed, for example to `build-strict` while still running `pnpm build:strict-smoke`.
  - confidence: 0.93
- **[P2] Point template choices at compare URLs:** `.github/pull_request_template.md:8-12`
  - body: GitHub documents multiple PR templates through `.github/PULL_REQUEST_TEMPLATE/` plus the `template` query parameter. These source-file links do not switch the active PR body, and the default root template will still be shown unless contributors use a `?template=quick_fix.md` or `?template=full_change.md` compare URL.
  - confidence: 0.91
- **[P2] Make the scripts index complete and current:** `scripts/README.md:3`
  - body: The README claims 318 script files, but current main has hundreds more tracked paths under `scripts`, and the index uses placeholder rows such as “other 45” instead of a complete lookup. As a contributor index, it is immediately misleading unless generated or manually completed from the current tree.
  - confidence: 0.9
- **[P3] Translate the scripts README to English:** `scripts/README.md:3-21`
  - body: This new README is contributor-facing documentation, but its headings and descriptions are mostly Chinese while the repository contributor policy requires American English in docs. Translate the index text before merging so it matches the rest of the project documentation.
  - confidence: 0.86
