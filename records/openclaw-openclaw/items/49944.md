The issue asks for Control UI tool cards to expand by default when the effective `agents.defaults.verboseDefault` is `full`.

## Best Possible Solution

Review, rebase, and land or replace https://github.com/openclaw/openclaw/pull/74398 so Control UI preserves `chat.history.verboseLevel`, derives tool auto-expansion from `full`, and keeps manual disclosure state behavior intact.

## Solution Assessment
