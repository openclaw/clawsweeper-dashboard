Keep open. ## Best Possible Solution

Maintainers should choose a provider-owned UX: either add guarded interactive LiteLLM URL capture and proxy model discovery aligned with the broader dynamic catalog design, or explicitly keep remote proxy setup on `--custom-base-url`, Custom Provider, and manual config paths.
