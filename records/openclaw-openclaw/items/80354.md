- **[P2] Keep native Ollama think on the request field:** `extensions/ollama/src/stream.ts:239-242`
  - body: Ollama's native chat API reads thinking control from top-level `ChatRequest.Think`, and the server validates thinking support through `req.Think`; `options` is only the model-parameter map. Moving runtime thinking to `options.think` can make `/think off` or effort levels ignored for thinking-capable Ollama models, so keep the documented top-level field and guard the non-thinking path instead.
  - confidence: 0.9

Merging that commit as-is would close the linked request while leaving part of the supported CLI JSON surface unavailable.
  - confidence: 0.88
