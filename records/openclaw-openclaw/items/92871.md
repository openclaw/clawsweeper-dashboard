Full review comments:

- **[P1] Derive status labels from the progress renderer:** `extensions/slack/src/monitor/message-handler/dispatch.ts:129-150`
  - body: This adds a Slack-only tool display table even though this handler already builds a `ChannelProgressDraftLine` via `buildChannelProgressDraftLineForEntry`, which resolves labels and fallback display through `resolveToolDisplay`.   - confidence: 0.9

  - confidence: 0.84
