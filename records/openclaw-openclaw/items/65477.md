## What This Changes

The request asks OpenClaw to prepend or append the active model identifier to the per-response `/usage` footer in messaging channels.

## Best Possible Solution

Consolidate the open implementations into one narrow footer change that renders the actual resolved provider/model while preserving token, cache, cost, and session output semantics.

## Reproduction Assessment
