## What This Changes

This PR suppresses generated exec approval command spans for non-POSIX shell wrappers and Windows gateway approval commands, preserves explicit `commandSpans`, and adds focused tests plus a changelog entry.

## Best Possible Solution

Resolve the effective shell-wrapper transport argv before deciding highlight support, then cover direct and transparent-carrier PowerShell/cmd cases in formatter and host approval tests.
