This PR updates exec approval command span formatting to omit generated spans for non-POSIX top-level shell wrappers, with coverage for PowerShell, cmd.exe, and host approval payload behavior.

## Best Possible Solution

Finish maintainer review and CI, then land the localized formatter suppression once the draft is ready so approval decisions remain unchanged while unsupported wrapper commands render as plain text.

## Solution Assessment
