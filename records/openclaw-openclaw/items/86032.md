The branch adds DSML-to-tool-call parsing in the OpenAI completions stream, adds regression tests, and adjusts DeepSeek V4 extra-params handling when compat thinking format is OpenAI-style.

## Best Possible Solution

Refactor recovery into a bounded DSML span parser that runs for the explicit DeepSeek/Foundry completions compat cases, preserves normal filtered text exactly once, rejects malformed blocks, and includes Foundry/OpenAI-compat plus plain-text regression tests with after-fix runtime proof.
