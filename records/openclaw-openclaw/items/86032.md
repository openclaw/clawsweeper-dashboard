Full review comments:

- **[P1] Skip DSML recovery when native tool calls are present:** `src/agents/openai-transport-stream.ts:2587`
  - body: This flush runs while handling `delta.content`, before the chunk's native `choiceDelta.tool_calls` are processed below.
