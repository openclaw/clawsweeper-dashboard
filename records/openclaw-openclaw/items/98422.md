- **[P2] Preserve config freshness before records-only commits:** `src/cli/plugins-update-command.ts:350-353`
  - body: In this new branch, an install-record-only update no longer calls `replaceConfigFile(...)`, so it also no longer enforces the `baseHash` check that used to reject and roll back when `openclaw.json` changed while the npm update was running. Please re-check the current config hash or refresh from a fresh config snapshot before committing the records-only index.
