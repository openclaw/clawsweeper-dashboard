## Best Possible Solution

## Solution Assessment

Unclear until the linked PR is reviewed in its latest state. A typed query/document task surface looks more maintainable than unrestricted embedding `extraBody` because query embeddings, document indexing, batch requests, and cache identity need distinct values.
