## What This Changes

Add configurable Jina/OpenAI-compatible embedding task parameters, likely separated for query and document memory embeddings.

## Best Possible Solution

Review and land PR #74444, or a narrow replacement, that carries typed query/document task settings through schema, resolved config, provider bodies, batch requests, cache identity, docs, and changelog.

## Solution Assessment

Unclear until PR #74444 is reviewed. A typed query/document task surface appears safer than unrestricted `extraBody` because direct query embeddings, document indexing, batch bodies, and cache identity need separate values.
