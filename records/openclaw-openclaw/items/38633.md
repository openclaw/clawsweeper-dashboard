Keep open: current main still documents `pidsLimit: 256` as a Docker sandbox config example while the resolver, schema, and Docker argv path leave PID limits unset unless explicitly configured. The issue asks OpenClaw to either add a sandbox Docker `pidsLimit` default or clarify the docs so the `256` example is not mistaken for a default.
