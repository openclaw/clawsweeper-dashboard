## Summary

Keep open because current main still lacks configurable per-job retry controls for recurring cron jobs, and the linked implementation candidate at https://github.com/openclaw/openclaw/pull/51001 is still open and unmerged.

## What This Changes

The issue requests opt-in retry controls such as `--retry-count`, `--retry-delay`, and optional backoff for failed recurring cron jobs.

## Best Possible Solution
