  - confidence: 0.91

  - body: When the config does not select a configured image provider, this falls through to `providers[0]?.id`, so a completely unconfigured setup can report the first visible provider as active. Return `null` unless the selected or configured provider is actually usable, matching the existing CLI/provider readiness semantics.
