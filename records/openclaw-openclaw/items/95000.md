## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Keep configured checks on a read-only auth path:** `src/gateway/server-methods/image.ts:72`
  - body: `image.providers` is registered as `operator.read`, but this line still calls each provider's `isConfigured` with only `cfg` and `agentDir`. Current image providers route that through helpers such as `isProviderApiKeyConfigured`, which call `ensureAuthProfileStore(...)` without `readOnly: true`, so a read-only inventory call can sync or persist auth profile state. Load/provide a read-only auth profile view, or use a non-mutating projection instead of write-capable provider hooks.
  - confidence: 0.9
- **[P1] Stop reporting the first provider as active:** `src/gateway/server-methods/image.ts:97-98`
  - body: When no image provider is configured, the handler still falls back to `providers[0]` and reports it as `active`. That publishes a provider choice to Console clients even though no explicit/configured provider exists, contradicting the PR's stated no-fallback behavior. Return `null` in this branch and add a regression case where every provider is unconfigured.
  - confidence: 0.88
- **[P2] Include array output capabilities in the exported type:** `packages/gateway-protocol/src/schema/image-generation.ts:82-88`
  - body: The schema accepts `output` as `boolean`, `string[]`, or an object, and the handler defaults missing output to `[]`, but the exported `ImageProviderCapabilityOutput` type only allows `boolean` or the object form. TypeScript clients importing the new protocol type will see a different contract than the validator and server actually allow.
  - confidence: 0.86
- **[P2] Remove the root PR body artifact:** `pr-body-final.md:1`
  - body: `pr-body-final.md` is a scratch copy of PR prose and proof, including stale commit and validation wording. It is not product source, public docs, or generated protocol output, so landing it would add misleading repository clutter.
  - confidence: 0.95
