## What This Changes

The PR moves native hook relay registry, bridge, invocation, and permission-window state from module-local collections to a `globalThis` symbol and adds a regression test for shared relay/bridge storage.

## Best Possible Solution

## Solution Assessment

Yes for the code direction: sharing relays, bridges, invocations, and permission state through one process-global symbol is the narrowest in-process fix for duplicate module copies. The patch still needs the required changelog entry before merge.
