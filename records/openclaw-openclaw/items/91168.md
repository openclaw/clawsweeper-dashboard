## Best Possible Solution

## Solution Assessment

No: the UI cue is useful, but this implementation adds a parallel narrow runtime label that can disagree with execution policy and can break Swift source callers. The safer solution is to consume canonical runtime metadata or policy and keep generated clients compatible.
