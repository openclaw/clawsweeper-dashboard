## What This Changes

The branch changes the plugin module loader to try native loading for compiled JavaScript plugin entries with `openclaw/plugin-sdk/*` imports before falling back to alias-aware jiti transformation, with regression tests for native hit and fallback behavior.

## Best Possible Solution

Land the native-first loader ordering once packaged install or upgrade proof shows Codex and at least one compiled SDK-import plugin resolve through the intended native SDK path while preserving alias-aware jiti fallback.
