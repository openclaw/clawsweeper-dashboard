## Summary

Keep open: this PR targets a real auth bug, but the current head still leaves tokenless remote node hosts able to write local auth profiles, duplicates narrower loopback handling, and lacks after-fix real behavior proof.

## What This Changes

The branch adds a `models auth login` guard, regression test, and changelog entry to block credential writes on node hosts configured for non-loopback gateways.

## Best Possible Solution

Land a guard that blocks non-loopback `nodeHostConfig.gateway.host` independent of token, reuses shared gateway loopback handling, covers tokenless and local-loopback cases, and includes redacted real remote-node auth-login proof.

## Reproduction Assessment

Yes, as a source-level reproduction. Current main resolves a local agent auth store and writes provider profiles there, while node-host config can point at a remote gateway whose credentials come from env/config rather than `node.json.token`; I did not run a live OAuth flow.

## Solution Assessment

No. Blocking remote node-host auth writes is the right narrow direction, but this patch should not require `token` and should reuse the existing shared loopback classifier.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Remove the token precondition from the guard:** `src/commands/models/auth.ts:620`
  - body: `NodeHostConfig.token` is optional, and node-host gateway auth can come from env/config while `nodeHostConfig.gateway.host` still points at a remote gateway. Returning when `token` is absent leaves the local auth-write path reachable for those remote node hosts.
  - confidence: 0.93
- **[P2] Reuse the shared loopback classifier:** `src/commands/models/auth.ts:611-615`
  - body: This helper only treats empty, `localhost`, `127.0.0.1`, and `::1` as local, while gateway code already accepts broader loopback forms like `127/8`, `[::1]`, IPv4-mapped loopback, and `localhost.`. Valid local node-host configs could be blocked as remote.
  - confidence: 0.88
