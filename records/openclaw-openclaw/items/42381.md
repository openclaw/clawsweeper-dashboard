## Summary

Keep open: current main still has the local auth-profile write path, and this PR is relevant but not mergeable as-is because the guard is token-gated, duplicates a narrower loopback classifier, and lacks real behavior proof.

## What This Changes

The PR adds a remote node-host guard before `modelsAuthLoginCommand`, a regression test, and a changelog entry for blocking local auth-login credential writes.

## Best Possible Solution

Update this PR or a narrow replacement to block based on non-loopback `nodeHostConfig.gateway.host` regardless of token, reuse the shared loopback helper, cover tokenless and loopback cases, and add redacted real remote-node proof.

## Reproduction Assessment

Yes. Source inspection shows current main resolves a local agent directory and writes provider auth profiles there, while node-host config can point at a remote gateway whose credentials come from env/config.

## Solution Assessment

No. Blocking remote node-host auth writes is the right narrow direction, but this implementation should key off the non-loopback gateway host without requiring `token` and should reuse the existing gateway loopback classifier.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Remove the token precondition from the guard:** `src/commands/models/auth.ts:620`
  - body: `NodeHostConfig.token` is optional, and node-host gateway auth can come from environment or config while `nodeHostConfig.gateway.host` still points at a remote gateway. Returning when `token` is absent leaves the local auth-write path reachable for those remote node hosts.
  - confidence: 0.93
- **[P2] Reuse the shared loopback classifier:** `src/commands/models/auth.ts:611-615`
  - body: This helper only treats empty, `localhost`, `127.0.0.1`, and `::1` as local, while gateway code already accepts broader loopback forms like `127/8`, `[::1]`, IPv4-mapped loopback, and `localhost.`. Valid local node-host configs could be blocked as remote.
  - confidence: 0.88
