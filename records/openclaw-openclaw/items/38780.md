## Security Review

Status: needs_attention

Summary: The diff introduces autonomous continuation/delegation paths with concrete drain-control, replay, and build-boundary concerns.

  - body: The embedded tool path does not forward the delegate-drain capability consistently, which can broaden the autonomous scheduling surface beyond runs expected to drain delegate work.
