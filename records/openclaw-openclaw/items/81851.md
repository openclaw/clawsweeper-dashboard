Status: needs_attention

Summary: The diff intentionally introduces local TLS interception of Anthropic CLI traffic, which should not merge without explicit maintainer security/product approval.

  - body: The wrapper injects `HTTPS_PROXY` and `NODE_EXTRA_CA_CERTS` into the spawned Claude CLI so OpenClaw can decrypt Anthropic API traffic locally. That is the core feature, but it changes the trust boundary and needs explicit opt-in policy, docs, and maintainer approval before shipping.
