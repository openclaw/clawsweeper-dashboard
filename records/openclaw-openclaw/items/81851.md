- **[P2] Normalize the CONNECT host before routing it:** `extensions/anthropic/interactive-proxy/mitm-server.ts:334-336`
  - body: `isAllowedConnectHost` trims and lowercases the CONNECT host, but the routing check below still compares the original `host` to `api.anthropic.com`. A valid case-insensitive target such as `API.Anthropic.Com:443` is allowed but routed as an undecrypted pass-through tunnel, so the wrapper will not see SSE events and can lose streamed reasoning/tool progress. Add the interactive runtime/session ownership and focused doctor coverage with the manifest change.
