## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Keep the default backend pointed at claude:** `extensions/anthropic/cli-backend-interactive.ts:139-147`
  - body: `normalizeConfig` always copies `merged.command` into `OPENCLAW_INTERACTIVE_CLAUDE_BINARY`. For the default config, `merged.command` is the backend's own `bun`, and `wrapper.ts` prefers that env var over its `claude` fallback, so selecting `claude-cli-interactive` without a user override spawns Bun as if it were Claude and the backend cannot start.
  - confidence: 0.94
- **[P1] Ship the TTY preload with the proxy assets:** `extensions/anthropic/package.json:18-30`
  - body: `wrapper.ts` passes `interactive-proxy/tty-spoof.cjs` through `NODE_OPTIONS=--require`, but the package metadata only declares wrapper, mitm-server, and cert-manager as static assets. The postbuild copier ships only declared assets, so packaged installs will miss the preload file and the Claude child will fail before exercising the proxy.
  - confidence: 0.91
