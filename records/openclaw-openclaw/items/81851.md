The PR adds an opt-in Anthropic `claude-cli-interactive` backend that runs Claude Code through a Bun loopback TLS proxy, reconstructs Claude stream-json reasoning output, registers the backend across Anthropic/runtime gates, and adds tests/static assets.

## Best Possible Solution

Rebase onto current main, preserve auth-profile-aware runtime selection, keep overflow prompt recovery inside a maintainer-approved narrow permission model, and land only after explicit security/provider approval for the local TLS proxy design.
