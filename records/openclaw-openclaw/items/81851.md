## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Constrain the local CONNECT proxy:** `extensions/anthropic/interactive-proxy/mitm-server.ts:284-288`
  - body: The CONNECT listener only special-cases `api.anthropic.com` and otherwise tunnels to the requested host and port. Because this is an unauthenticated localhost proxy while each wrapper run is active, another local process that discovers the port can use OpenClaw as an arbitrary TCP tunnel; reject non-Anthropic CONNECTs or require an explicit allowlist/security boundary.
  - confidence: 0.89
- **[P2] Document the backend inheritance hook:** `src/plugins/cli-backend.types.ts:199-202`
  - body: This adds `inheritUserConfigFrom` to `CliBackendPlugin`, which is exported through `openclaw/plugin-sdk/cli-backend`, but the CLI-backend plugin docs still omit the hook from the public advanced-hook surface. Plugin SDK changes are compatibility-sensitive, so third-party backend authors need the inheritance and `filterArgs` semantics documented, or the hook should stay internal.
  - confidence: 0.84
- **[P2] Keep the proxy runtime in checked lanes:** `tsconfig.extensions.json:19`
  - body: The proxy `.ts` files are shipped as static runtime assets, but this exclusion removes the whole `interactive-proxy` subtree from the normal extensions typecheck. The added local tsconfig is not referenced by the Anthropic package tsconfig or normal package scripts, so CI can package broken Bun/TypeScript runtime code without noticing.
  - confidence: 0.88
- **[P2] Update both postbuild asset expectations:** `test/scripts/runtime-postbuild.test.ts:42-45`
  - body: The first postbuild assertion now expects the Anthropic proxy assets, but the no-fs-scan assertion below still expects the old output list. `listStaticExtensionAssetOutputs()` discovers package metadata without counted `existsSync` or `readdirSync` scans, so that focused test should now return these new outputs and fail until both expectations are aligned.
  - confidence: 0.9
