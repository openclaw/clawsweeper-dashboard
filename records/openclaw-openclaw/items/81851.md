  - confidence: 0.84

  - body: Adding `inheritUserConfigFrom` changes the public `CliBackendPlugin` contract, including sibling override and arg-filter semantics. Please add the plugin docs and focused contract coverage in the same PR so external CLI backend authors have a supported API rather than an undocumented bundled-plugin-only behavior.
  - confidence: 0.82
- **[P1] Constrain or classify CONNECT proxy egress:** `extensions/anthropic/interactive-proxy/mitm-server.ts:288`
  - body: The loopback proxy accepts CONNECT targets and forwards non-Anthropic hosts with `net.connect`. Even though it binds to 127.0.0.1, this is a new unauthenticated local raw-socket egress path during each run; it needs a destination constraint or an explicit managed/local-only outbound classification with tests/docs before merge.
  - confidence: 0.78
