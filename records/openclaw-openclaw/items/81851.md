- **[P1] Preserve Claude CLI answer previews:** `src/auto-reply/reply/agent-runner-execution.ts:2075-2079`
  - body: This branch now returns before `onPartialReply` whenever `shouldBridgeCliAssistantTextToReasoning(cliExecutionProvider)` is true and no thinking event has arrived. That includes the existing `claude-cli` backend, whose assistant text was already bridged to reasoning but still delivered to the answer preview on current main, so normal Claude CLI users can lose live answer-lane previews until the final result.
  - confidence: 0.92
- **[P1] Override every proxy env variant before spawning Claude:** `extensions/anthropic/interactive-proxy/wrapper.ts:488-492`
  - body: The child env copies the whole parent process and only overrides uppercase `HTTPS_PROXY`. If the service has `https_proxy`, `ALL_PROXY`, or `NO_PROXY`/`no_proxy` set, the spawned Claude CLI can bypass this local MITM proxy or route through the wrong proxy, silently losing SSE capture and weakening the documented proxy boundary.
  - confidence: 0.86
- **[P1] Route pass-through tunnels through managed egress:** `extensions/anthropic/interactive-proxy/mitm-server.ts:343-347`
  - body: For allowed non-API Anthropic hosts, the CONNECT proxy opens a raw `net.connect` directly to the destination. OpenClaw's managed-proxy contract treats raw sockets as bypass-prone and requires classification before landing, so this path needs to use the managed CONNECT/egress path or fail closed when managed proxy mode is active.
  - confidence: 0.88
- **[P1] Remove or authorize the dependency graph change:** `extensions/anthropic/package.json:9`
  - body: This external PR adds `bun-types` to the Anthropic plugin manifest and lockfile.
