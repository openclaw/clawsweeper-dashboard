## Security Review

Status: needs_attention

Summary: The diff intentionally decrypts Claude CLI Anthropic API traffic through a generated local CA/TLS proxy, so it needs explicit maintainer security review before merge.

Concerns:

- **[medium] Review the TLS interception trust boundary:** `extensions/anthropic/interactive-proxy/wrapper.ts:85`
  - body: The wrapper injects `HTTPS_PROXY` and `NODE_EXTRA_CA_CERTS` into the Claude CLI child so OpenClaw can decrypt Anthropic API traffic. That should ship only with explicit opt-in policy, docs, failure behavior, and maintainer approval.
  - confidence: 0.93
- **[medium] Constrain the generated CA lifecycle:** `extensions/anthropic/interactive-proxy/cert-manager.ts:54`
  - body: The proxy creates and reuses a local CA under the OS temp directory and gives the CA a long lifetime. The implementation should define permissions, rotation, cleanup, and what can trust that CA before merge.
  - confidence: 0.86
