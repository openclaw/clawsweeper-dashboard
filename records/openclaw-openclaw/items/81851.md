## Security Review

Status: needs_attention

Summary: The diff still has a prompt debug leak and introduces a local TLS interception trust boundary that needs maintainer/security acceptance.

Concerns:

- **[medium] Dash-prefixed prompts can leak to debug logs:** `extensions/anthropic/interactive-proxy/wrapper.ts:148`
  - body: `redactArgsForLog` preserves every dash-prefixed arg, but the prompt is a positional arg and can itself begin with `-`; debug mode can still write private prompt text to stderr and gateway logs.
