## What This Changes

The PR adds an opt-in Anthropic `claude-cli-interactive` backend that runs Claude Code through a Bun loopback TLS proxy, reconstructs stream-json reasoning output, wires the backend through Claude CLI routing/auth/capability paths, and adds package assets plus tests.

## Best Possible Solution

Land this only after a rebase that preserves current auth-profile routing, keeps overflow recovery within a maintainer-approved narrow permission model, and explicitly accepts the local TLS interception boundary while coordinating parser/rendering surfaces with https://github.com/openclaw/openclaw/pull/80046 and https://github.com/openclaw/openclaw/pull/85381.
