  - confidence: 0.89

  - confidence: 0.9

  - body: The Escape handler is attached to the overlay, so it only receives key events while focus remains inside a descendant. If focus leaves the dialog, Escape no longer closes or guards it; use a document-level listener or focus trap while the modal is open.
  - confidence: 0.78
