## Best Possible Solution

Keep the compatible opt-in timeout behavior on main until maintainers approve a semantic contract, then land a focused change with regression coverage for pre-tool release, post-tool default budget, stuck-turn latency, and subagent handoff behavior backed by real workload proof.
