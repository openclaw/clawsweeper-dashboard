## What This Changes

The PR adds a `memorySearch.local.gpu` config field, runtime validation, node-llama-cpp type updates, and a `getLlama` GPU option passthrough for local embeddings.

## Best Possible Solution

Rebase into the current local embedding config contract: add only upstream-supported `gpu` wiring through type/schema/help/labels/docs/resolver/provider/wrapper surfaces, preserve existing `contextSize` behavior, add focused tests, and attach redacted runtime proof.
