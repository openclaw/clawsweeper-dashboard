## What This Changes

The PR adds two embedded-runner regression tests for NVIDIA/Nemotron empty terminal stop turns that assert a visible terminal error and a single run attempt.

## Best Possible Solution

Keep the regression coverage test-only, but model NVIDIA/Nemotron with `api: "openai-completions"` and assert the current retry, continuation, or exhaustion behavior unless maintainers deliberately add a production exception.
