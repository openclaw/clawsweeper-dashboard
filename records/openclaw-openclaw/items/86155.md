  - body: `npm install @github/copilot-sdk@1.0.0-beta.4` pins only the top-level SDK package; that package declares range dependencies such as `@github/copilot: ^1.0.46`, so user machines can install a different CLI/transitive graph than the reviewed lockfile and QA inventory.
