## Summary

Keep this PR open: current `origin/main` still emits and documents the retired Anthropic 1M beta header, so the central change is still useful. It is not merge-ready because the docs added by this PR still describe 1M context as an explicit opt-in even though current context sizing now defaults Anthropic Opus/Sonnet 4 to 1M, and the external PR has only test/CI proof rather than after-fix real behavior proof.

## What This Changes

The PR removes Anthropic 1M beta-header emission, strips legacy configured 1M beta values, preserves OAuth/default Anthropic beta headers, and updates related docs, tests, and changelog.

## Best Possible Solution

Land a rebased version that removes the retired 1M beta header, updates docs to match current default 1M context sizing on `origin/main`, includes redacted real behavior proof, and keeps the broader OAuth wrapper work in https://github.com/openclaw/openclaw/pull/41461 separate.

## Reproduction Assessment

Yes, source-backed. On `origin/main`, the Anthropic wrapper still adds `context-1m-2025-08-07` for `context1m`, and docs/tests still assert the beta-gated behavior.

## Solution Assessment

Not yet. The header-removal approach is narrow and appropriate, but the PR needs a docs correction for current context-sizing behavior plus real behavior proof and exact-head validation before merge.
