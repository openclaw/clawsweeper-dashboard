Keep open. Request to add a Control UI chat token/quota widget with per-message usage, estimated cost, provider credit states for OpenRouter/LLMAPI/Ollama, and optional low-credit alerts.

## Best Possible Solution

Scope a single chat usage/quota UX contract that reuses provider-owned usage hooks and existing session usage data, then add OpenRouter/LLMAPI/Ollama provider semantics before expanding alerts or cost comparison UI.

## Solution Assessment
