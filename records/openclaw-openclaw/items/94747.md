## Best Possible Solution

Review and land https://github.com/openclaw/openclaw/pull/84485, or replace it with an equivalent maintainer-approved Discord guard that keeps private final text hidden while preventing directly addressed Discord turns from ending with no visible reply.

## Solution Assessment
