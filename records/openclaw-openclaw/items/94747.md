## Summary

## What This Changes

## Best Possible Solution

Review and land https://github.com/openclaw/openclaw/pull/84485, or replace it with an equally narrow Discord source-channel guard that preserves the private-final boundary while emitting a safe diagnostic when no source-visible delivery happened.
