## What This Changes

Request adding a bundled `openai-compatible` memory embedding provider for self-hosted OpenAI-compatible HTTP embedding servers without LM Studio warmup or OpenAI global config inheritance.

## Best Possible Solution

Resolve the implementation and scope through https://github.com/openclaw/openclaw/pull/80479; if accepted, merge the provider with docs and tests, otherwise publish it as an external plugin while preserving the documented OpenAI baseUrl workaround.

## Solution Assessment
