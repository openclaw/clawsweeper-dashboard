## Summary

Keep open. ## What This Changes

The PR adds default-agent generated plugin model catalog read-through for secondary agents, threads that option through cached model discovery, updates the cache fingerprint, and adds ModelRegistry regression tests.

## Best Possible Solution

Land the local-first, per-provider inherited catalog read-through if maintainers accept default-agent catalog inheritance for secondary agents; keep broader lazy per-agent discovery as a follow-up rather than blocking this narrow fix.

## Reproduction Assessment

## Solution Assessment

Yes, this appears to be the best bounded repair for the reported runtime path. Running discovery for every agent would address the deeper root cause but is broader, costlier, and better left as follow-up work.
