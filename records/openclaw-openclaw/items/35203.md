Keep open: current main has adjacent subagent, session messaging, memory, usage, and hook-context surfaces, but it still does not implement this RFC’s learned capability profiles, shared blackboard, layered private/team/global memory, or enforceable multi-agent token/cost budgets. This RFC requests a four-layer multi-agent collaboration roadmap: capability profiling, a shared blackboard, layered memory visibility, and token/cost governance.
