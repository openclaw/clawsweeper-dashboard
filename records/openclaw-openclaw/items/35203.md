Keep open: current main still has isolated subagent delegation, adjacent usage reporting, memory/search hooks, and context-engine extension points, but not the RFC's learned capability profiles, shared blackboard/query layer, private/team/global memory boundaries, or enforceable multi-agent token/cost budgets. ## What This Changes

The issue requests a four-layer multi-agent collaboration roadmap covering capability profiling, shared blackboard state, layered memory visibility, and token cost governance.

## Best Possible Solution
