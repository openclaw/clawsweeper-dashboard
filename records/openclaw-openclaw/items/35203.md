## Summary

Keep open: current main has isolated multi-agent routing, subagent spawning, bounded agent-to-agent follow-up, memory search/promotion, and usage reporting, but it does not implement the RFC's full capability profiling, shared blackboard, layered memory visibility, or budget enforcement stack. The related discussion and linked prototypes are useful staged on-ramps, not complete superseding work.

## What This Changes

The RFC requests a staged multi-agent collaboration stack with capability profiling, a shared blackboard, layered memory visibility, and token/cost governance.

## Best Possible Solution
