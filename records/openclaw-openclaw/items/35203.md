## Summary

Keep open for maintainer product and security direction: current main has isolated subagent spawning, scoped inter-session messaging, plugin-owned memory, and usage reporting, but not the requested learned routing, authorized shared blackboard, layered cross-agent memory, or pre-dispatch budget governance.

## What This Changes

The issue requests a four-layer multi-agent collaboration architecture covering capability profiling, shared blackboard storage, layered memory visibility, and token cost governance.

## Best Possible Solution

Keep this as the umbrella design record while maintainers split it into separately approved contracts for routing, memory visibility, blackboard storage, and budget governance.
