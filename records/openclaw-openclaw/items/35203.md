## Summary

Keep open for maintainer product and security direction: current main has isolated subagent spawning, scoped agent-to-agent messaging, plugin-owned memory, and usage reporting, but not learned capability routing, an authorized shared blackboard, layered cross-agent memory visibility, or pre-dispatch budget admission.

## What This Changes

The issue requests a multi-agent collaboration stack covering capability profiling, shared blackboard state, layered memory visibility, and token-cost governance.

## Best Possible Solution

Keep #35203 as the umbrella design record while maintainers split approved contracts into narrower routing, memory-visibility, blackboard-storage, and budget-governance tracks.
