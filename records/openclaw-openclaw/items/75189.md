## Summary

Keep open: current main and v2026.6.10 still use the 20,000/60,000 bootstrap budgets, have no `bootstrapTier` config surface, and doctor still warns only on configured budget pressure rather than small-local-model prompt pressure. ## What This Changes

Reduce or warn about default workspace bootstrap prompt pressure so small/mid local models reliably dispatch tools instead of hallucinating tool use.

## Best Possible Solution

Choose an upgrade-safe prompt-pressure mitigation, likely combining tier/default or template trimming with doctor guidance, then validate it with focused source tests and small-model behavior proof.
