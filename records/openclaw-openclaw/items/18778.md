## Security Review

Status: needs_attention

Summary: The diff introduces Activity auth, token transport, and WebSocket fanout concerns that need security and owner review before merge.

Concerns:

- **[high] Activity mode can bypass Gateway auth without a token:** `src/gateway/a2ui-activity.ts:35`
  - body: `isA2uiActivityAccessAllowed` returns true when Activity mode is enabled and no token is configured, and the HTTP/WS authorization path accepts that before normal Gateway auth.
