## Security Review

Status: needs_attention

Summary: The diff introduces concrete Activity auth, token transport, and unbounded broadcast concerns that need owner/security review before merge.

Concerns:

- **[high] Activity mode can bypass Gateway auth without a token:** `src/gateway/a2ui-activity.ts:35`
  - body: `isA2uiActivityAccessAllowed` returns true for enabled Activity config with no token, and A2UI HTTP/WS authorization consults it before normal bearer or capability auth.
