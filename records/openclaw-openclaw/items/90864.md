## What This Changes

This PR adds a result-marker guard and focused tests for task completion summaries that start with progress narration but later contain result or verification markers.

## Best Possible Solution

Land a classifier-only fix that accepts explicit result/report/verification evidence while keeping future and conditional progress-only summaries blocked.

## Solution Assessment

No as submitted: the classifier is the right layer, but the marker expression is too broad. The safer fix is to accept explicit result/report evidence while keeping conditional and future-tense progress summaries blocked.
