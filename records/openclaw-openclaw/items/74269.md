A WhatsApp direct-message auto-reply starts typing after QR setup but stops without sending a visible reply, while Telegram works on the same provider route.

## Best Possible Solution

Keep this issue open until maintainers capture a failed WhatsApp direct-message turn on current main or the latest release and compare it with a successful Telegram turn on the same provider route.
