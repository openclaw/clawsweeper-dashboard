Keep open: current main still lacks Feishu `reactionLevel` schema, metadata, prompt guidance, and action gating; the related Feishu reaction PRs address API reliability rather than this auto-reaction policy.

Request Feishu plugin support for `channels.feishu.reactionLevel` so the agent can auto-add emoji reactions according to `off`/`ack`/`minimal`/`extensive` levels.
