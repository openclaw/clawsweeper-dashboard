## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Parse XML attributes in any order:** `src/agents/pi-embedded-utils.ts:41`
  - body: `parseMinimaxToolCallXml` only recognizes `<invoke>` and `<parameter>` when `name` is the first attribute. Valid XML such as `<invoke id="x" name="Bash">...` is skipped, so the intended tool call can still be stripped instead of executed.
  - confidence: 0.91
- **[P2] Decode XML entities before passing tool inputs:** `src/agents/pi-embedded-utils.ts:49-51`
  - body: Parameter bodies are copied directly into `input`, so XML encodings such as `&amp;`, `&lt;`, `&quot;`, and `&apos;` reach tools literally and can corrupt commands or file contents.
  - confidence: 0.88
