## What This Changes

The issue reports a 2026.5.7 Discord regression where the bot connects but never reaches READY, leaving guild channel messages unprocessed.

## Best Possible Solution

Land a narrow Discord plugin fix for the null-socket IDENTIFY wait path, add regression coverage for that branch, and require redacted live Discord proof that READY and guild inbound delivery recover on a 2026.5.7-style setup.
