## What This Changes

The issue reports that wedged main-session restart recovery can retry indefinitely across gateway reboots because the retry budget is only in-process.

## Best Possible Solution

Review and land one canonical durable retry-budget/quarantine fix, likely the strongest current candidate at https://github.com/openclaw/openclaw/pull/96230, then close this issue and supersede duplicate PRs.
