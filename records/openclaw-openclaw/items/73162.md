## What This Changes

The PR changes Slack Socket Mode reconnect policy from 12 attempts to unlimited retries and adds a changelog entry for the linked Slack outage bug.

## Best Possible Solution

## Solution Assessment

Yes. Setting `maxAttempts` to `0` is the narrowest maintainable fix because the existing guard and log formatting already define non-positive attempts as unlimited, while permanent Slack auth failures still fail fast.
