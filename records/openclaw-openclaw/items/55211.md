## Best Possible Solution

Land the ALS guard if hook owners accept same-key async-lineage suppression; otherwise adjust the guard semantics and add delayed-child redispatch coverage before merge.

## Solution Assessment

Mostly yes. AsyncLocalStorage is a clean fit for blocking same-chain recursion while preserving independent concurrent calls, but maintainers need to accept or refine the delayed-child async suppression semantics.
