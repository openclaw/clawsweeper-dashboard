## What This Changes

The PR adds an AsyncLocalStorage guard around internal hook dispatch and regression tests for re-entrant, sequential, concurrent, cross-session, and cross-action hook delivery.

## Best Possible Solution

Land the AsyncLocalStorage guard and regression tests after contributor-supplied real behavior proof is added and a hook owner accepts the same-key re-entry suppression semantics.

## Reproduction Assessment

## Solution Assessment

Yes for the code direction, but not merge-ready yet. AsyncLocalStorage is a narrow fit for same-chain suppression while preserving independent dispatch, and the remaining question is proof plus maintainer acceptance of the compatibility semantics.
