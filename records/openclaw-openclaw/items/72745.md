## Summary

Keep open. Current main has adjacent diagnostics, trajectory export, Codex feedback upload, plugin-scope guidance, and optional sandboxing, but it does not implement local AI-fix capture/upload, business-impact review, or sandboxed upstream PR generation; the remaining request is broad and security-sensitive.

## What This Changes

Request to add an opt-in local AI-fix feedback/upload loop, upstream business-impact code review, and sandboxed regression/PR-generation pipeline for AI-generated fixes.

## Best Possible Solution

A maintainer-owned RFC should split this into privacy/consent/redaction, upstream intake/provenance, untrusted-code sandbox/regression policy, and business-impact review UX before any implementation work.
