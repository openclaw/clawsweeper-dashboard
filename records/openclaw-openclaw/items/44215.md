  - body: This puts `%LOCALAPPDATA%\pnpm`, WindowsApps, and `%APPDATA%\npm` in `prepend`, so OpenClaw subprocesses can resolve profile-owned shims before existing PATH and system roots.   - body: The helper now joins PATH with `;` for `platform: "win32"`, but this assertion splits using the host delimiter. On Linux/macOS CI, fake `C:/...` entries are split at the drive colon, so the regression test fails for reasons unrelated to the intended behavior.
  - confidence: 0.94

  - confidence: 0.86
