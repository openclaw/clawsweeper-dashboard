  - body: This adds `%LOCALAPPDATA%\pnpm`, `WindowsApps`, and `%APPDATA%\npm` to `prepend`, so they resolve before the existing PATH and trusted roots. Current main deliberately appends comparable user-writable package-manager paths to avoid binary shadowing.
  - confidence: 0.92

  - body: The helper joins PATH with `;` for `platform: "win32"`, but this assertion splits with the host delimiter. On Linux/macOS CI, fake `C:/...` entries get split at the drive colon, so the test can fail for reasons unrelated to the intended behavior.
  - confidence: 0.94

  - body: This PATH bootstrap PR also adds gateway push labels near the auth labels, but current main already defines those keys later in `FIELD_LABELS`. The unrelated stale hunk would create duplicate object keys when rebased or merged.
  - confidence: 0.86
