## What This Changes

The PR adds a process-local session write epoch to accept prompt-window co-tenant session writes and extends session-lock tests.

## Best Possible Solution

Use a fence design that permits legitimate lock-mediated co-tenant progress while preserving detection of external or malformed same-window mutation, then prove it in a real shared-session runtime path.

## Solution Assessment

No. The epoch shortcut fixes the false positive by weakening the fence whenever any co-tenant lock acquisition occurred; a safer solution should validate or attribute the accepted write while still rejecting same-window external mutations.
