## Best Possible Solution

Land a focused Telegram plugin fix that carries media-resolution failure state through resolver, handler, debounce/media-group, and body rendering paths, preserves useful unavailable-media metadata without pretending an attachment exists, and sends accurate rate-limited user-visible feedback.
