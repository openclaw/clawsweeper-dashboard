## Best Possible Solution

Add a focused regression with a stub MCP server that spawns a worker subagent and asserts the child provider schema and system prompt contain the MCP tools allowed by subagent policy and the target agent policy; then either fix inherited allow propagation or document the parent-surface requirement if that is intentional.
