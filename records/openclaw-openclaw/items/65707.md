## Best Possible Solution

Land an updated version that passes the effective prompt mode into pruning setup, preserves explicit `keepLastAssistants: 0`, covers subagent/cron and `toolsAllow` minimal paths, and includes redacted real behavior proof.

## Solution Assessment

No. The optimization direction is narrow, but the run path should pass `effectivePromptMode` or compute the same effective signal before building extension factories, with regression coverage for `toolsAllow`.
