Full review comments:

- **[P2] Prevent late connects from recaching timed-out browsers:** `extensions/browser/src/browser/pw-session.ts:1625-1629`
  - body: If the operation timeout fires before `connectBrowser()` has populated `cachedByCdpUrl`, this cleanup only removes the current map entry. Guard the pending connect/cache population with a timeout generation or otherwise ensure timed-out work cannot repopulate the scoped cache.
  - confidence: 0.84
