Support Xiaomi MiMo's provider-native `web_search` tool by adding opt-in Xiaomi provider configuration and request payload handling.

## Best Possible Solution

Implement this in the bundled Xiaomi plugin with validated opt-in config, provider-owned payload injection that preserves ordinary function tools, docs/changelog coverage, and focused tests; avoid a core Xiaomi hardcode.
