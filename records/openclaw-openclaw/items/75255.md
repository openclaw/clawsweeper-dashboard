## What This Changes

Add opt-in Xiaomi MiMo native `web_search` support that appends Xiaomi's provider-native tool to chat-completions requests.

## Best Possible Solution

Implement this in the bundled Xiaomi plugin as a validated opt-in provider feature that appends the native tool only for Xiaomi model calls and preserves normal function tools and managed web-search behavior elsewhere.
