## Summary

Keep open. Current main and the latest release still reject the reported MSYS/MINGW `install.sh` route and still provide bash/zsh-only PATH persistence and warning guidance, so the request remains a Windows platform support decision rather than implemented cleanup.

## What This Changes

Support running OpenClaw's Unix installer path in Windows 11 MSYS2/MINGW environments and add fish shell PATH configuration or guidance.

## Best Possible Solution

Maintainers should either explicitly support MSYS2/MINGW with tested OS detection, Windows-safe install behavior, fish-aware PATH handling, docs, and smoke coverage, or document PowerShell/WSL as the supported Windows routes and MSYS2 as unsupported.

## Solution Assessment
