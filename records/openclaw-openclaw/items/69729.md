## Summary

Keep open. Current main still lacks the requested Qwen catalog updates, but this PR needs changes before merge because it reintroduces a deprecated runtime suppression hook, changes the documented Global Standard endpoint host, and drops Coding Plan host normalization.

## What This Changes

## Best Possible Solution

Keep the model catalog additions and qwen3.6-plus reasoning correction, but express endpoint-specific suppression through the Qwen plugin manifest, preserve existing endpoint host text and host normalization, and update focused Qwen docs/tests.

This is not a user bug reproduction path; it is source-reproducible review evidence.
