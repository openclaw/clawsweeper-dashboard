## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Yes. Preserving small structured accounting fields inside the existing bounded truncation helpers is the narrowest maintainable fix; raising the cap or reconstructing usage downstream would leave the root recorder behavior wrong.
