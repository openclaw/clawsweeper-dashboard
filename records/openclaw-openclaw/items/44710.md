## What This Changes

The PR adds a public `before_tools_resolve` plugin hook, a helper that filters agent tools before model exposure, attempt/compaction wiring, and allow/deny merge tests.

## Best Possible Solution

If maintainers want this API, land a rebased implementation that defines the hook in the current hook contract, makes failures fail closed, filters before denied names can affect reservations/history, and includes focused tests plus real runtime proof.
