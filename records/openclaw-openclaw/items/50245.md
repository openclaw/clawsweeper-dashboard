## What This Changes

Request to display the existing `HealthSummary.durationMs` gateway probe duration in human-readable `openclaw health` output without changing JSON mode.

## Best Possible Solution

Refresh and land https://github.com/openclaw/openclaw/pull/50250, or an equivalent narrow patch, so `openclaw health` text output prints `summary.durationMs` with focused regression coverage while preserving `--json`.

## Solution Assessment

Yes. Printing the already-required `summary.durationMs` in the non-JSON health renderer is the narrow maintainable fix; no protocol, schema, gateway RPC, or config change is needed.
