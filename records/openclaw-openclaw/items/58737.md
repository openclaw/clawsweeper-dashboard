Report that Slack message edits after tool/progress updates lose the configured agent display name and avatar and fall back to the Slack app identity.

## Best Possible Solution

Land or replace https://github.com/openclaw/openclaw/pull/58800 with a Slack-plugin fix that threads `SlackSendIdentity` through every edit path, keeps missing-scope fallback parity with sends, and includes focused tests plus redacted live Slack proof.

## Solution Assessment
