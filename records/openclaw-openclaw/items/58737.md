The issue requests that edited Slack messages preserve the configured agent display name and avatar instead of reverting to the Slack app default.

## Solution Assessment

Yes on direction, but it is not complete on main. The maintainable boundary is Slack-plugin identity threading through edit paths, plus send-path fallback parity and live/API proof for Slack edit authorship behavior.
