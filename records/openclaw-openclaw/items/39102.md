Keep open: current main still has only the global A2A policy, and this PR contains useful unique work plus sufficient live-output proof. The PR adds `agents.list[].tools.agentToAgent.allow`, structured A2A access decisions/errors, docs/schema updates, config pruning, and tests for per-agent outbound cross-agent session access.

## Best Possible Solution

Preserve Plugin SDK compatibility in the narrow PR, then decide whether to land this outbound-only A2A allowlist or fold it into the broader per-agent session and A2A scoping contract.

## Solution Assessment
