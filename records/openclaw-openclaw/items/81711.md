## Summary

Keep open. The state/report separation is a reasonable direction, but this diff drops once-mode dedupe for CLI-backed runs because CLI results do not emit the new metadata field that the changed callers now rely on; the external PR also still lacks required after-fix real behavior proof.

## What This Changes

The PR moves bootstrap truncation warning dedupe history out of `systemPromptReport` into `SessionEntry.bootstrapPromptWarningState`, updates run/reset call sites, and adjusts tests.

## Best Possible Solution

Keep the state/report separation, but carry the new warning dedupe state through CLI prepared context and CLI result metadata, then add a focused CLI once-mode dedupe regression test before merge.

## Reproduction Assessment

Yes, by source inspection: the changed post-run resolver reads `result.meta.bootstrapPromptWarningSignaturesSeen`, while current CLI run results only expose `systemPromptReport` and the PR removes the report history field. That is enough to trace how CLI once-mode dedupe state becomes empty after a CLI run.

## Solution Assessment

No. Separating runtime dedupe state from current-run report metadata is the maintainable direction, but the patch must first make the CLI runner emit the separate state just like the embedded runner does.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.92

Full review comments:

- **[P2] Preserve CLI warning state before removing it from reports:** `src/auto-reply/reply/agent-runner-execution.ts:1594-1596`
  - body: This path runs after `runCliAgent()`, but CLI results currently only put the dedupe history in `systemPromptReport`; they do not populate `meta.bootstrapPromptWarningSignaturesSeen`. Because this PR removes `warningSignaturesSeen` from the report, resolving from the new meta field returns `[]` after every CLI run, so once-mode bootstrap truncation warnings repeat instead of deduping. Carry `bootstrapPromptWarning.warningSignaturesSeen` through the CLI prepared context and result meta, or keep a fallback until that exists.
  - confidence: 0.92
