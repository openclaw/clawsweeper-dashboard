## Summary

Keep open: the Cron zh-CN localization is useful and has real screenshot proof, but current PR head still introduces source-proven UI regressions for custom cron session target labels and Traditional Chinese relative-time output.

## What This Changes

The PR extracts Cron Control UI strings into i18n keys, adds zh-CN/generated locale updates, localizes Cron schedule labels, and passes the active UI locale into relative-time formatting.

## Best Possible Solution

Keep the Cron i18n extraction, but preserve specific session target values and the zh-CN/zh-TW locale boundary with focused regression coverage.

## Reproduction Assessment

Yes. Source inspection of PR head shows custom Cron session targets collapse to the isolated label and `zh-TW` flows into a Simplified Chinese relative-time branch.

## Solution Assessment

No, not yet. The extraction is the right direction, but the formatter and Cron chip labels need narrower locale/target handling before this is mergeable.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P2] Preserve custom cron session target labels:** `ui/src/ui/views/cron.ts:1537`
  - body: This now renders every non-`main` cron job as the localized isolated label. `CronSessionTarget` still includes `current` and `session:<id>`, and the CLI docs/normalizer preserve persistent session targets, so jobs created through CLI/plugins with `session:...` will appear as isolated even though they are pinned to an existing session. Preserve those specific labels or fall back to the raw target when it is not `main`/`isolated`.
  - confidence: 0.92
- **[P2] Preserve the zh-TW relative-time locale:** `src/infra/format-time/format-relative.ts:84`
  - body: The UI wrapper now passes the active Control UI locale into this formatter, so selecting `zh-TW` also enters this Simplified Chinese branch. Since `zh-CN` and `zh-TW` are separate supported bundles, Traditional Chinese users will see Simplified strings such as `刚刚` and `分钟前`. Gate this on `zh-CN` or add a Traditional Chinese/translation-backed branch before merging.
  - confidence: 0.9
