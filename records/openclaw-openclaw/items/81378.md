  - confidence: 0.93

  - body: Using `locale.startsWith("zh")` makes `zh-TW` render Simplified Chinese strings such as `刚刚` and `分钟后` across shared UI timestamps. Restrict this path to `zh-CN` or make the relative-time copy table-driven per supported locale.
  - confidence: 0.91

  - confidence: 0.94
