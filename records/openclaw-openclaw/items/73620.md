## Summary

Keep open. Current main still has the linked wizard-default bug, but this PR is not merge-ready because its helper changes the prompt default for all custom main/shared workspace shapes and can bypass the existing #59789 shared-base behavior when the user accepts the prompt.

## What This Changes

The PR adds and exports a peer-workspace suggestion helper, uses it for new interactive `openclaw agents add` workspace prompt defaults, and adds helper-level tests.

## Best Possible Solution

Land one canonical fix for #71889 that suggests the documented peer-level workspace only for the setup/default main workspace case, preserves intentional custom shared workspace roots and existing-agent update prompts, adds wizard-level regression coverage, and includes the required changelog entry.

## Reproduction Assessment

Yes. Source inspection gives a high-confidence reproduction path: setup writes `agents.defaults.workspace`, the wizard currently uses `resolveAgentWorkspaceDir` for the prompt default, and the resolver appends non-default ids under that fallback.

## Solution Assessment

No. The wizard-only direction is maintainable, but this implementation is too broad; a safer fix should gate the peer-level suggestion to the default setup workspace case or defer to `resolveAgentWorkspaceDir` for custom shared roots.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Preserve custom shared workspace defaults:** `src/agents/agent-scope-config.ts:204`
  - body: When `agents.defaults.workspace` is an intentional shared base, current main resolves non-default agents underneath it and has #59789 regression coverage for that behavior. This helper instead builds a sibling of the resolved default agent workspace for every new agent, and the wizard persists the accepted prompt, so accepting the default can silently bypass the configured shared root. Gate the peer-level suggestion to the setup/default workspace case or defer to the resolver for custom shared roots.
  - confidence: 0.86
