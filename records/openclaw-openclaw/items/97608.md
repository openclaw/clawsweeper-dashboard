  - body: This branch releases the local-service lease before retrying and then manages a successful retried response with the released lease reference.   - confidence: 0.88

If the caller passed a `Request` with a stream body, the first fetch can disturb or lock that body, so the retry can fail locally instead of making a fresh provider request; restrict this retry to replay-safe bodies or rebuild a buffered body before dispatch.
