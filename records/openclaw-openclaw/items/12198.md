Define a threat-model-backed shared trust and diagnostic contract for model-visible Markdown, then apply it consistently to prompt ingestion, skill loading, Skill Workshop, ClawHub install/update, and publish-side verdicts while rejecting broad workspace-write blocking unless maintainers explicitly choose it.

## Solution Assessment
