## What This Changes

## Best Possible Solution

Either retain opt-in failure alerts, or land opt-out semantics only with explicit maintainer approval, docs and upgrade alignment, compatibility acceptance, and real cron alert proof.

## Reproduction Assessment

## Solution Assessment
