## What This Changes

The issue asks for session memory sync to stop reindexing whole growing JSONL transcripts and preserve/index only changed or appended session chunks.

## Best Possible Solution

Land a rebased memory-core fix that preserves unchanged session chunks across direct and source-wide batch sync without unintentionally expanding public SDK or storage contracts, then close this issue after merge.
