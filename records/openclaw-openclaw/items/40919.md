## What This Changes

Request incremental session memory indexing so growing session JSONL files do not require deleting and re-embedding all chunks on each dirty sync.

## Best Possible Solution

Land a rebased memory-core fix that preserves existing session chunks for appended transcript content across direct and source-wide session sync, with storage and API compatibility settled before merge.
