The issue requests a Control UI toggle for viewing channel conversations either separated by channel/session or merged into one unified timeline.

## Best Possible Solution

Design an additive Control UI and Gateway protocol surface that keeps the existing per-session picker while adding authorized channel-separated views and a merged timeline with stable channel, account, session, source, and timestamp metadata.
