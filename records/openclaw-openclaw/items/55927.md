## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Honor input/output tokens past the stale guard:** `ui/src/ui/views/chat.ts:439-442`
  - body: The fallback sum is still unreachable for rows where no fresh total exists, because this helper returns when `totalTokensFresh === false` before reading `inputTokens` or `outputTokens`.   - confidence: 0.91

  - body: The new duration label rounds at each unit boundary, so a 91-minute completed session renders as `2h 31m`. Use floor or truncation when converting seconds to minutes and minutes to hours so elapsed time is not overstated.
  - confidence: 0.88
