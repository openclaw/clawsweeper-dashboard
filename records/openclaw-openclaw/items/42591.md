The issue requests splitting the Unix installer into smaller shell modules for UI, download, platform detection, npm/git install, and build-tool logic.

## Best Possible Solution

Define the installer module or generated-artifact contract first, then refactor while preserving `https://openclaw.ai/install.sh` behavior and updating installer sync, smoke, and focused tests.
