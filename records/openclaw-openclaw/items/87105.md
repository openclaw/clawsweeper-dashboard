- **[P1] Preserve fallback-local approval runtime auth:** `src/agents/tools/gateway.ts:120`
  - body: `resolveDefaultGatewayTarget` treats every `gateway.mode === "remote"` default as remote, but `buildGatewayConnectionDetails` falls back to local loopback when `gateway.remote.url` is missing and labels that source `missing gateway.remote.url (fallback local)`.
