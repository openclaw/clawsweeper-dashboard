  - body: This registers completion context before `scheduler(scheduleRequest)` accepts the job, while completion lookup later starts from `sessionKey`.   - confidence: 0.9

  - body: Dynamic subscriptions include generated HMAC secrets and operator-managed runtime state, but this writes them to `webhooks/subscriptions.json`. Use the existing plugin-state/SQLite owner instead so secrets and runtime state follow the approved persistence path.
  - confidence: 0.87
