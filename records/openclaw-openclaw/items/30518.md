## What This Changes

Request standalone Telegram message-tool pin and unpin actions so agents can maintain pinned Telegram status or dashboard messages programmatically.

## Best Possible Solution

Add Telegram plugin-owned `pin` and `unpin` discovery plus runtime dispatch behind a maintainer-approved action gate, reuse the existing helpers, document permission expectations, and require focused tests plus live Telegram proof.

## Solution Assessment
