## What This Changes

The issue reports that image, video, and music generation tool registration eagerly lists capability providers before complete explicit model configs can short-circuit, causing repeated plugin registry reload latency on hosted gateways.

## Best Possible Solution

Reconcile #74097 against current main: land or adapt the narrow lazy-provider thunk cleanup if maintainers still want to eliminate execution-time eager provider listing, or close that PR as superseded if the current registration gate and provider-runtime cache changes are sufficient.
