## What This Changes

The branch carries Telegram URL buttons through shared interactive mapping and inline-keyboard serialization, updates prompt/schema/tests, and adds a changelog fix entry.

## Best Possible Solution

Update the Telegram channel docs to show URL buttons and clarify callback versus link behavior, then land the focused runtime fix with the existing targeted tests and changed gate.

## Solution Assessment

No as-is. The runtime approach is the narrow maintainable fix and matches Telegram's exactly-one action-field contract, but the public Telegram message-action docs should be aligned before merge.
