Keep open. Request a compact Control UI run-progress indicator for long or multi-agent tasks, including phase, latest action, last activity, elapsed time, and reconnect context.

## Best Possible Solution

Define an additive active-run activity/progress contract, then render a compact Control UI status strip from explicit stream, tool, subagent, delivery, and reconnect signals without turning it into a raw event log.
