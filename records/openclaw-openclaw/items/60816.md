## What This Changes

The requested change is to make identical concurrent edit followers return a distinct noop result when the file is already at the requested final state, while preserving success for the writer and errors for true mismatches.

## Best Possible Solution

Review and land https://github.com/openclaw/openclaw/pull/77638, or replace it with an equivalent narrow wrapper fix, then close this issue after the fix reaches main.
