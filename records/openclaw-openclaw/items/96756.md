Overall correctness: patch is incorrect

Full review comments:

- **[P1] Keep Feishu command bodies slash-clean:** `extensions/feishu/src/bot-content.ts:284-285`
  - body: This changes a self-mention before a slash command from `/model` to `@Bot /model`, but the Feishu runtime still assigns `agentFacingContent` directly to `message.commandBody`. Downstream command parsing only treats slash-leading command bodies as commands, so mention-prefixed slash commands can fall through as normal chat text. Preserve the agent-visible self-mention, but feed command-facing text from the slash-clean probe/effective body too.
  - confidence: 0.9
- **[P2] Avoid bare @name self-mentions for command text:** `extensions/feishu/src/bot-content.ts:285`
  - body: The replacement uses the display name directly, so a bot named `OpenClaw Bot` produces `@OpenClaw Bot /model`. The command-probe regex only strips the first non-space `@...` token, leaving `Bot /model` instead of a slash-leading command. Keep structured mention text for the agent or carry a separate cleaned command body instead of relying on a lossy `@name` form.
  - confidence: 0.86
