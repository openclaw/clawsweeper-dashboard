## What This Changes

The issue requests fixing Signal image captions so an unbounded Markdown chunk limit does not reduce captions to their first character.

## Best Possible Solution

Land a narrow shared Markdown chunker fix that treats positive Infinity as an explicit unbounded sentinel while preserving NaN fallback and finite splitting behavior, with focused regression coverage and Signal caption proof.
