## Summary

## What This Changes

Expose the effective agent runtime for configured model choices so canonical `openai/gpt-5.5` can visibly show Codex runtime routing without reintroducing `openai-codex/*` as the selected model route.

## Best Possible Solution

Review and land https://github.com/openclaw/openclaw/pull/90328 or an equivalent additive Gateway/WebUI runtime-metadata change, then close this issue as implemented while keeping the broader routing-ambiguity discussion linked to https://github.com/openclaw/openclaw/issues/84032.

## Solution Assessment
