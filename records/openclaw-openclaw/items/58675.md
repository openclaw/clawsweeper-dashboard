Full review comments:

- **[P2] Read max_context_window_tokens from Copilot limits:** `extensions/github-copilot/discovery.ts:69`
  - body: The mapper only reads `max_prompt_tokens` for `contextWindow`, but current Copilot `/models` payloads use `capabilities.limits.max_context_window_tokens` for the actual context window. Models without `max_prompt_tokens` will fall back to 128k and lose the API-accurate catalog data this feature is meant to surface.
  - confidence: 0.87
- **[P2] Filter Copilot router records before mapping models:** `extensions/github-copilot/discovery.ts:129-150`
  - body: The `/models` payload can include internal router IDs such as `accounts/...`; this filter only rejects missing IDs, disabled rows, known IDs, and non-chat types, so a chat-type router can be registered as a selectable model. Current main rejects non-`model` objects and the router prefix before mapping.
  - confidence: 0.83
