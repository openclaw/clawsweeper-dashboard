## What This Changes

## Best Possible Solution

A maintainer should choose and document the Slack reporting boundary in the Slack plugin/message-presentation docs: either bless fallback-text-plus-Block-Kit for trusted producers or keep `presentation` as the only supported shared producer API and track richer report layouts separately.
