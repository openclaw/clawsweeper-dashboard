## What This Changes

The PR makes `sessions_spawn` use `runSessionKey` as requester ownership when present, prefers requester bindings before child bindings for subagent completion origin resolution, and adds focused regression tests.

## Best Possible Solution

Land this narrow routing fix after redacted live Telegram or equivalent runtime proof shows the originating parent session receives the completion announce, while leaving mediated-delivery policy to https://github.com/openclaw/openclaw/pull/80223.

## Solution Assessment
