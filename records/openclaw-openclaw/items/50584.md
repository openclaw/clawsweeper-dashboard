  - body: Current main starts session-memory capture in the background so `/new` and `/reset` acknowledgements are not delayed by transcript reads, file writes, slugging, or model work.   - confidence: 0.95

  - confidence: 0.9

  - body: The branch writes `${dateStr}-${slug}.md` directly. Current main allocates a suffix when the same timestamp or generated slug already exists, so this would reintroduce silent overwrites for repeated same-minute resets or repeated LLM slugs.
  - confidence: 0.9
