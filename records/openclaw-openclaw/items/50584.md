  - body: Current main resolves an available filename before writing so repeated resets with the same timestamp or slug do not overwrite each other. This PR returns to a direct `${dateStr}-${slug}.md` write, which can replace an earlier per-session artifact when two saves resolve the same slug.
  - confidence: 0.9
