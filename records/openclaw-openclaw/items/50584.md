- **[P1] Keep reset capture off the command path:** `src/hooks/bundled/session-memory/handler.ts:97`
  - body: Current main starts session-memory capture in a background promise so `/new` and `/reset` can acknowledge immediately. This PR makes the hook itself `async` and awaits transcript reads plus optional model work before returning, so a slow provider can delay reset handling by up to the synthesis timeout.
  - confidence: 0.95
- **[P1] Keep model-backed slugging explicitly opt-in:** `src/hooks/bundled/session-memory/handler.ts:193`
  - body: Current main only calls the model provider for filename slugs when `llmSlug === true`. This branch changes the default to `hookConfig?.llmSlug !== false`, so existing users with the hook enabled would start sending reset transcript excerpts to their configured provider without opting in.
  - confidence: 0.96
- **[P1] Do not duplicate startup memory inputs:** `src/hooks/bundled/session-memory/handler.ts:280-286`
  - body: Current main already loads date-prefixed slugged memory files during the reset startup prelude alongside the date-only file. Appending the same entry to `YYYY-MM-DD.md` while also writing `YYYY-MM-DD-<slug>.md` makes the first post-reset turn eligible to receive duplicate copies of the same session memory.
  - confidence: 0.94
- **[P1] Preserve collision-safe filenames:** `src/hooks/bundled/session-memory/handler.ts:210`
  - body: The direct `${dateStr}-${slug}.md` write can overwrite an earlier same-day same-slug or fallback timestamp capture. Current main uses a suffix search before writing so repeated `/new` or `/reset` captures do not lose the earlier session archive.
  - confidence: 0.9
- **[P1] Use an exported fs-safe append path:** `src/hooks/bundled/session-memory/handler.ts:26`
  - body: This imports `appendFileWithinRoot`, but current main's `src/infra/fs-safe.ts` does not export that helper. Rebasing this branch onto current main leaves the hook unable to typecheck or build until it uses the current fs-safe root/append surface or removes the canonical append path.
  - confidence: 0.92
