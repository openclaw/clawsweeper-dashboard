## Summary

Keep open: current main and the latest release still only support remote embedding config plus local/QMD/MMR reranking, while https://github.com/openclaw/openclaw/pull/92725 is an open closing implementation candidate that needs maintainer API/security/product review before this issue can close.

## What This Changes

Request support for Cohere-style remote reranker endpoints for OpenClaw memory search instead of local-only reranking.

## Best Possible Solution

Use this issue as the user-facing anchor while maintainers review the open implementation, choose one plugin/API/config/security direction, then close the issue only after the chosen PR lands or the direction is explicitly rejected.

The existing code only supports remote embeddings, local MMR diversity, and QMD's own rerank/no-rerank behavior.

## Solution Assessment
