## What This Changes

The PR makes memory-core skip managed-dreaming cron reconciliation warnings for non-managed heartbeat hook contexts without a gateway cron handle, while preserving gateway and managed cron failure warnings.

## Best Possible Solution

Land the focused memory-core fix after normal maintainer review and CI, keeping warning behavior only for contexts that can actually reconcile the managed dreaming cron.
