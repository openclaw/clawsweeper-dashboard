## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

No, not as currently shaped. A non-transcript progress event is safer than fake assistant deltas, and sibling Codex source tolerates unknown Responses SSE events, but the opt-in should not land under a namespace that current docs and doctor remove.
