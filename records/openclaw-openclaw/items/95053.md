## What This Changes

The PR adds repeatable `--file` support to `image.generate` capability metadata and command registration, forwards `opts.file` into `runImageGenerate`, and updates the capability-inspect test expectation.

## Best Possible Solution

Land the narrow CLI change after adding user-facing docs, direct forwarding coverage, and redacted real CLI proof that `image generate --file` reaches runtime dispatch while no-file generation still works.
