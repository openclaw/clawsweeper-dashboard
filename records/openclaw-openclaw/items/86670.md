- **[P2] Pass Eden AI auth into live model discovery:** `extensions/edenai/provider-catalog.ts:271-275`
  - body: `discoverEdenaiModels()` calls Eden AI's `/v3/models` endpoint without an `Authorization` header.   - confidence: 0.93

  - body: Current main skips manifest `providerEndpoints` whose `endpointClass` is not in its known `ProviderEndpointClass` set, and `edenai` is not in that set. This makes the `edenai.run` metadata ineffective and leaves the route classified as a generic custom endpoint; add core endpoint-class support with tests or remove the misleading metadata.
  - confidence: 0.91
