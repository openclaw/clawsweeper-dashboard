Full review comments:

- **[P2] Replace the raw session deletion workaround:** `docs/providers/edenai.md:131-133`
  - body: The troubleshooting page tells users to run `rm -rf ~/.openclaw/agents/main/sessions/*` for a stale transcript.   - confidence: 0.86

  - body: The live `/v3/models` response exposes cache prices as `cache_read_input_token_cost` and `cache_creation_input_token_cost`, but this normalizer reads `input_cache_read_cost_per_token` and `input_cache_write_cost_per_token`. Models with live cache pricing will therefore show zero cache read/write costs instead of Eden AI's returned metadata.
  - confidence: 0.9
