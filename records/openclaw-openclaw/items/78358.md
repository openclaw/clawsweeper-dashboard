## Summary

Keep this PR open: current main still lacks iMessage per-group `systemPrompt` support, and the PR is a focused implementation with schema, inbound-context, docs, changelog, and regression-test coverage. The PR adds iMessage `groups.*.systemPrompt` schema/type/docs support, resolves group keys from `chat_id`, `chat_guid`, `chat_identifier`, and wildcard entries, threads matched prompts into inbound group contexts, and adds regression tests plus a changelog entry.
