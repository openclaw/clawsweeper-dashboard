## What This Changes

The branch makes `openclaw gateway run` skip the doctor state-migration preflight unless lightweight legacy-state detection finds known migration inputs, with expanded config-guard coverage.

## Best Possible Solution

Keep the Gateway skip path only after detector errors conservatively fall back to the doctor preflight and current-head proof covers both no-legacy startup and at least one legacy-state case.
