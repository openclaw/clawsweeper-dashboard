## What This Changes

Request optional auto-scrolling display behavior for long status or info text in constrained OpenClaw UI/status surfaces.

Define a small accessible opt-in overflow-status helper, wire it into one maintainer-selected surface first, and coordinate with https://github.com/openclaw/openclaw/issues/13670 if the first target is the topbar model/token display.

## Solution Assessment
