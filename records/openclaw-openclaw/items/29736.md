## Best Possible Solution

Review and land https://github.com/openclaw/openclaw/pull/74002 or a maintainer-approved equivalent that derives approvals JSON, default socket, and effective-policy host-source paths from the active state-dir/profile contract while preserving the unset fallback and adding focused regression coverage plus docs updates.
