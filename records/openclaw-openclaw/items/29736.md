## Best Possible Solution

Review and land https://github.com/openclaw/openclaw/pull/74002, or a maintainer-approved replacement, so approvals JSON, default socket path, effective-policy source text, docs, and focused regression tests follow the active state-dir/profile contract while preserving the unset fallback.
