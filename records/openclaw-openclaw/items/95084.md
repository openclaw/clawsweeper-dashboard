## What This Changes

The PR composes `sanitizeAssistantVisibleText()` before `sanitizeForPlainText()` for Google Chat outbound text and adds adapter tests for stripping failed-tool trace banners while preserving ordinary prose.

## Best Possible Solution

Land this focused Google Chat sanitizer fix after ordinary maintainer review and CI gating, while keeping the broader cross-channel sanitizer and Codex-native status taxonomy work on their canonical items.
