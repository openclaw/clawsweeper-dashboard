  - confidence: 0.86

  - body: The workspace mirror flow creates `backup.target/workspace/mirrors` and `.staging` before rejecting symlinked target-side directories, and later copies/renames/removes below those paths. If a synced backup target contains `workspace`, `mirrors`, or `.staging` as a symlink, backup can write or delete outside the selected backup folder.
  - confidence: 0.84
