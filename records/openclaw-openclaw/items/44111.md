Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Reject target-side symlinks before snapshot writes:** `src/backup/snapshot-store/provider-folder.ts:85-95`
  - body: `copyFileAtomic` and `writeFileAtomic` create parent directories and then write/rename under `backup.target/snapshots/<installationId>` without checking existing intermediate segments inside the target. A symlinked `snapshots` or installation directory can redirect snapshot payload/envelope writes outside the configured backup target, so the store needs a no-symlink directory walk before mkdir/write/rename.
  - confidence: 0.92
- **[P1] Reject target-side symlinks before mirroring workspaces:** `src/commands/workspace-backup.ts:350-351`
  - body: This creates `backup.target/workspace/mirrors` and `.staging` before effective target-side symlink rejection, and the later containment check canonicalizes both child and root through the same symlink. If `backup.target/workspace` or a child directory is a symlink, `cp`, `rename`, and stale mirror cleanup can operate outside the backup target.
  - confidence: 0.91
- **[P2] Compare in-state restore paths before skipping assets:** `src/commands/backup-restore.ts:710-748`
  - body: Full-host restore skips standalone config/OAuth assets whenever the archive source path and current target are both somewhere under `stateDir`. If a backup was made with `state/openclaw.json` or `state/credentials` but the restore host points to a different in-state path, the active config/OAuth directory remains missing or stale after a reported successful restore.
  - confidence: 0.86
- **[P2] Remove or implement backup.retention:** `src/config/zod-schema.backup.ts:25`
  - body: The schema accepts `backup.retention`, but the diff does not add pruning behavior or docs for it. Users can configure retention expecting snapshots to be deleted while `backup run/list` will keep accumulating them, so either implement retention with tests/docs or remove the public config surface.
  - confidence: 0.88
