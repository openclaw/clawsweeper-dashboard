## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.82

Full review comments:

- **[P1] Reject symlinked snapshot store directories before writes:** `src/backup/snapshot-store/provider-folder.ts:67`
  - body: `backup.target` is meant to be a user-selected cloud/synced folder, so contents inside it are not a trusted directory tree. `uploadSnapshot` reaches `writeFileAtomic`/`copyFileAtomic`, which call recursive `mkdir` and then write/rename through `targetDir/snapshots/<installationId>` without rejecting symlinked intermediate directories. A symlink planted at `snapshots` or the installation directory can redirect encrypted payload/envelope writes outside `backup.target`; validate each target-side directory segment with `lstat` before any mkdir/copy/rename.
  - confidence: 0.9
- **[P1] Reject symlinked workspace mirror directories before writes:** `src/commands/workspace-backup.ts:350-351`
  - body: `workspaceBackupRunCommand` creates `backup.target/workspace/mirrors` and `.staging` before checking containment, and the later containment check follows symlinks on both the child and root. If a synced backup folder contains `workspace -> $HOME` or another directory symlink, those `mkdir`/mirror writes can occur outside the backup target while the realpath-based assertion still passes. Check `backup.target`, `workspace`, `mirrors`, and `.staging` with a no-symlink segment walk before creating or replacing anything there.
  - confidence: 0.88
- **[P2] Remove or implement retention config:** `src/config/zod-schema.backup.ts:25`
  - body: This adds public `backup.retention` keys to the OpenClaw config schema, but the PR does not read those values in the snapshot store, list/status, run, or cleanup paths. Accepting unused retention settings makes users think snapshot pruning is active when it is not, and it creates a config contract that will need compatibility support later. Either implement retention behavior and tests now or leave these keys out until the behavior exists.
  - confidence: 0.86
