  - confidence: 0.89
- **[P1] Reject symlinked workspace target dirs before mirroring:** `src/commands/workspace-backup.ts:350-351`
  - body: `workspaceBackupRunCommand` creates `mirrors` and `.staging` before proving those target-side paths are real directories.   - confidence: 0.88

  - body: The schema accepts `backup.retention.keepDaily`, `keepWeekly`, `keepMonthly`, and `maxSnapshots`, but the inspected diff does not implement snapshot pruning. Exposing inert public config will make users think retention is enforced when snapshots keep accumulating.
  - confidence: 0.8
