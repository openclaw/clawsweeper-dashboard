  - body: `workspaceBackupRunCommand` creates `backup.target/workspace/mirrors` and `.staging` before any target-side no-symlink validation, and the later containment check canonicalizes through existing symlinks. A symlinked `backup.target/workspace` can make mirror writes, renames, or stale cleanup operate outside the backup directory.
  - confidence: 0.88
- **[P1] Validate restore targets before rollback:** `src/commands/backup-restore.ts:442-443`
  - body: The restore loop moves an existing target into rollback before `copySourceToTarget` validates the target parent with `ensureDirectoryPathWithoutSymlinks`.
