## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Reject symlinked backup target roots before writes:** `src/commands/workspace-backup.ts:350-352`
  - body: `workspaceBackupRunCommand` creates `backup.target/workspace/mirrors` and `.staging` before proving those target-side directories are not symlinks, and the later containment checks canonicalize through symlinks. If a synced/shared backup folder already contains `workspace` or `mirrors` as a symlink, the mirror writes and stale cleanup can operate outside the configured backup target.
  - confidence: 0.91
- **[P1] Reject symlinked snapshot store directories before writing:** `src/backup/snapshot-store/provider-folder.ts:85`
  - body: The folder store uses recursive `mkdir` followed by copy/rename under `targetDir/snapshots/...`, but it never rejects symlinked intermediate directories inside the backup target. A modified cloud/shared backup folder can redirect payload or envelope writes outside the intended target, so writes should walk/create store directories without following symlinks before copying.
  - confidence: 0.88
- **[P2] Restore config when the in-state target path differs:** `src/commands/backup-restore.ts:715-716`
  - body: This condition treats any current config path under `stateDir` as covered by the state asset. If the backup's config was under state at one relative path but the restore host uses a different in-state config path, full-host restore skips the config operation and leaves the active config stale or missing.
  - confidence: 0.84
- **[P2] Restore OAuth when the in-state target path differs:** `src/commands/backup-restore.ts:741-742`
  - body: The credentials check has the same issue: any current OAuth dir under `stateDir` is considered covered without comparing the source and target relative paths. Restoring onto a host with a different in-state OAuth location can report success while leaving provider auth data absent from the active directory.
  - confidence: 0.84
