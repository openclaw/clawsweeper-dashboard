- **[P1] Preserve the Teams channel-data tenant:** `extensions/msteams/src/monitor-handler/message-handler.ts:792`
  - body: Use the same tenant source as current Teams conversation-reference capture: prefer `activity.channelData.tenant.id` before `conversation.tenantId`.   - body: This new cleanup path performs raw `fetch()` calls to Microsoft OAuth and Graph endpoints with credentials.   - confidence: 0.86
