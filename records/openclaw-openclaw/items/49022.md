- **[P1] Constrain MEDIA image reads to trusted sources:** `src/tui/components/tool-execution.ts:97`
  - body: `processResult` passes every parsed `MEDIA:` path to `readMediaImageAsBase64`, and that helper accepts any absolute local image path.   - confidence: 0.88
- **[P2] Scope rendered-image dedupe to live components:** `src/tui/components/tool-execution.ts:92-96`
  - body: The shared ChatLog set is append-only while `setResult`/`setPartialResult` can detach cached images and pruning can remove components. A retry or updated final result for the same image can then skip rendering and leave the `MEDIA:` line or no image; track ownership reversibly or keep dedupe local to one processed result.
  - confidence: 0.92
