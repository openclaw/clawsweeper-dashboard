  - body: splitMediaFromOutput() accepts syntactically valid local paths from tool text, and this line then reads that path in the host TUI process.   - body: Current main depends on @earendil-works/pi-tui, but this new helper imports @mariozechner/pi-tui. After rebasing, this will not match the repository's current dependency surface and will break the TUI build unless the new files use the current package name and API.
  - confidence: 0.9
- **[P2] Make image dedupe reversible:** `src/tui/components/chat-log.ts:17`
  - body: This shared set is append-only for the lifetime of ChatLog, but cached image components can be detached, replaced, or pruned. Once a path or fingerprint is added, a retry or later result with the same image can be skipped even though no live component owns it anymore, so the dedupe entries need to be released with the image lifecycle or scoped to one processed result.
  - confidence: 0.86
