## Review Findings

Overall confidence: 0.86

Full review comments:

- **[P1] Constrain MEDIA reads to trusted image sources:** `src/tui/components/inline-image.ts:44-61`
  - body: `readMediaImageAsBase64` accepts any absolute path with an image extension, then reads and renders it in the local TUI. A gateway/tool result can emit `MEDIA:/Users/.../private.png` and make the client display a local file that OpenClaw did not generate. Please root file reads in a trusted media/workspace contract or prefer validated image content blocks instead of arbitrary absolute paths.
  - confidence: 0.82

  - confidence: 0.92
- **[P2] Release dedupe claims when images detach:** `src/tui/components/tool-execution.ts:92-96`
  - body: `globalRendered` is shared for the `ChatLog` lifetime, but keys are added while processing a result and never removed when the component clears, replaces, detaches, or gets pruned. A retry or updated final result with the same path/fingerprint can then skip the image that should be visible now.
  - confidence: 0.88
