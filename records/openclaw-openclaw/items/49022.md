  - confidence: 0.92
- **[P1] Restrict MEDIA reads to trusted image artifacts:** `src/tui/components/inline-image.ts:61-69`
  - body: `processResult()` feeds every extracted `MEDIA:` path into this helper, and the helper only checks absolute path, extension, file type, and size before reading. That lets tool/model text cause the local TUI process to read any absolute image-looking file on the user's machine; please tie rendering to a trusted media artifact/root or structured image block instead of arbitrary text paths.
  - confidence: 0.86

  - body: The shared `globalRendered` set is append-only, but `setResult()`, `setPartialResult()`, and chat-log pruning can detach or replace the component that owned the cached image. Once a path/fingerprint is added here, a later retry or refreshed result can skip the image even though it should render again.
  - confidence: 0.9
