After that, a retry or updated final result with the same path/data will be skipped even though the current tool result should render it; make the dedupe local to one processed result or remove/ref-count keys when cached images are detached/pruned.

Because `textStripped` is used whenever any image renders, a mixed result can hide non-image media lines instead of falling back to visible text; track the rendered image paths or parsed segments and only remove those lines.
