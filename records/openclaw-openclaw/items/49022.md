  - confidence: 0.82

  - body: Current main depends on `@earendil-works/pi-tui`, but the PR imports the new image helpers from `@mariozechner/pi-tui`. Rebase and update the imports before merge, otherwise this branch will fail module resolution on current main.
  - confidence: 0.92
- **[P2] Release dedupe claims when images detach:** `src/tui/components/tool-execution.ts:96-103`
  - body: `globalRendered` is shared for the `ChatLog` lifetime, but keys are added while processing a result and never removed when the component clears, replaces, detaches, or gets pruned. A retry or updated final result with the same path or fingerprint can then skip the image that should be visible now.
  - confidence: 0.88
