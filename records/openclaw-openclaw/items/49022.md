  - body: `splitMediaFromOutput()` accepts legacy MEDIA directives from tool text, and this call then reads the referenced absolute path in the host TUI process.   - confidence: 0.89

A later retry or updated result with the same path/fingerprint can be skipped even when no live image remains, so the dedupe entry needs to be released with the cached image lifecycle or scoped to one processed result.
  - confidence: 0.87
