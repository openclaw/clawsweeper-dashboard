## What This Changes

The issue asks OpenClaw to stop subagent or session startup from recreating root bootstrap markdown files in repository workspaces that already have completed agent workspace state.

## Best Possible Solution

Keep this issue as the canonical report, then land a narrow linked fix that prevents runtime root seeding while preserving documented bootstrap repair behavior and adding focused regression coverage for subagent and nested-worktree starts.

## Solution Assessment
