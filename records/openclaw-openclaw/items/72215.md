  - body: The gateway tool guard is the model-to-operator trust boundary, and `cron.enabled` currently controls whether the scheduler starts or is restarted. Keep it protected unless the PR adds a directionally safe API and tests that it cannot disable safeguards.
  - confidence: 0.86
