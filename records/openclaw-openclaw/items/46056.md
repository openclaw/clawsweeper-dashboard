## What This Changes

The issue asks for shell builtins such as `cd` to stop causing unexpected exec approval prompts in allowlist mode, or at least for the allowlist docs to explain the limitation and workaround.

## Best Possible Solution

Review and either land or close the linked opt-in safeBuiltins PR; if the runtime policy is not accepted, land a focused docs clarification so users understand the current allowlist limitation.
