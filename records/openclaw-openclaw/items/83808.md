## What This Changes

Harden macOS LaunchAgent managed update handoff so a deleted cwd or package root cannot crash the updater with `uv_cwd` after the service is stopped.

## Solution Assessment

Yes: the linked PR's direction of hardening the managed helper cwd, early CLI bootstrap, update runner, and npm env cwd paths is the narrow maintainable fix. The issue should remain open until that PR lands or maintainers choose a different fix.
