## Best Possible Solution

Add a narrowly scoped, tested guard for high-confidence internal-deliberation prefixes in the provider or delivery sanitizer while preserving normal prose and existing structured reasoning behavior.

## Solution Assessment

Unclear as proposed: provider-agnostic semantic stripping is risky because the text has no structured marker. A narrow, regression-tested pattern set or provider-owned handling would be more maintainable.
