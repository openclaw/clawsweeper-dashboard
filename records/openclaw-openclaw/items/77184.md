## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Unclear until maintainers accept the API contract. Re-exporting from the existing focused `command-status` subpath is a reasonable owner-boundary shape, but direct `StatusSummary` exposure may also publish nested host-internal type references.
