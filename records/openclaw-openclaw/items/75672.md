  - body: `mode` can still be `warn` when cleanup runs without `--enforce`, and current cleanup avoids destructive artifact pruning in that mode.   - body: This branch removes any checkpoint whose session id is absent from the index immediately. Existing unreferenced-artifact cleanup only removes compaction checkpoints after the `pruneAfter` mtime cutoff, so fresh or in-flight checkpoint files can be deleted here; reuse that sweep or apply the same cutoff.
  - confidence: 0.9
