## Best Possible Solution

Rebase onto current main and use one Windows suffixless load helper for both bundled `sqliteVec.load(...)` and platform-variant `db.loadExtension(...)` paths, while preserving explicit path pass-through and the returned effective path.

## Reproduction Assessment
