- **[P1] Preserve omitted agentId as the default agent:** `src/cron/store/row-codec.ts:134`
  - body: `job.agentId` being absent is not equivalent to the literal `main`: `listPage` resolves omitted ids through `state.deps.defaultAgentId`, and `assertMainSessionAgentId` allows main-session jobs with no agent id for whatever agent is configured as default.
