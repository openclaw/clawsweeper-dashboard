## Summary

Keep open: the branch is on the right sanitizer path and includes tests/changelog, but the current parser still rejects generated single-newline exec-state blocks and leaves some generated metadata wrappers visible.

## What This Changes

The PR adds modern runtime-context stripping in `src/agents/internal-runtime-context.ts`, sanitizer regression tests, and a changelog fix for #75726.

## Best Possible Solution

Make `stripInternalRuntimeContext` consume generator-shaped runtime sections line-by-line or from shared sentinels, with tests built from current generator outputs while preserving standalone JSON and fenced replies.

## Reproduction Assessment

Yes. Feed the PR sanitizer a modern runtime header followed by current `buildExecOverridePromptHint` output joined with single newlines, or a generated location/structured metadata block followed by visible reply text; source inspection shows those sections survive.

## Solution Assessment

No. Section-aware stripping is the right direction, but the implementation should align with the actual generator output instead of fixture-shaped blank-line paragraphs and a partial metadata heading allowlist.

## Review Findings

Full review comments:

- **[P2] Consume generated exec-state blocks:** `src/agents/internal-runtime-context.ts:259-264`
  - body: `buildExecOverridePromptHint` emits the heading and state lines joined with single newlines, but this parser reads one blank-line-delimited paragraph and only accepts an isolated heading.   - confidence: 0.94

If a model echoes one of those wrappers, the location or structured payload remains visible before the real reply.
  - confidence: 0.9
- **[P2] Handle missing-output async completions:** `src/agents/internal-runtime-context.ts:245-248`
  - body: The async matcher only accepts the normal `has completed` intro, but the current async-event generator also emits a missing-output variant starting `An async command you ran earlier completed without captured stdout/stderr`. Echoes of that generated wrapper will stop stripping at the intro and leak the completion scaffold.
  - confidence: 0.84
