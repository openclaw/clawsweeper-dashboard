The issue requests an OpenAI/Grok-style live reasoning stream in the OpenClaw TUI, ideally rewriting a chat or footer line while a run is active.

## Best Possible Solution

Land one coordinated reasoning-stream UX that preserves the thinking visibility gate, makes gateway thinking events reliably available, renders the chosen TUI transient surface, and updates public docs to match the final behavior.

## Solution Assessment

No for an isolated automated patch. The maintainable solution needs maintainer UX/privacy direction and coordination with the open gateway/Web reasoning-stream PRs while preserving visibility gating.
