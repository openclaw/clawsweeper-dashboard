## What This Changes

The issue reports a WhatsApp channel regression where HTTP 428 Precondition Required / Connection Terminated exits the channel after upgrading to 2026.4.29.

## Best Possible Solution

Review and land or replace PR #75773 with WhatsApp-owned 428 handling for both opening and post-open close paths, plus regression tests that preserve existing 401 and 440 behavior.

## Solution Assessment
