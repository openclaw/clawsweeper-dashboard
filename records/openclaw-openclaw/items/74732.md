Keep this open because current main still lacks oMLX-specific memorySearch documentation, and open PR #74761 is the linked, unmerged implementation candidate from the same author. Request documentation, and possibly first-class support, for using oMLX as a memorySearch embedding provider on Apple Silicon through an OpenAI-compatible local endpoint.

## Solution Assessment
