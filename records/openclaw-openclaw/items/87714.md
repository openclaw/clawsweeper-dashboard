The issue proposes tagging instructions or actions by source and denying tool calls derived from external content at the tool execution gate.

## Best Possible Solution

Design a core provenance and tool-policy model that tags external content at ingress, carries that source through transcripts/runtime/tool gates, and enforces high-risk tool behavior with focused tests, docs, and an operator-controlled exception path.
