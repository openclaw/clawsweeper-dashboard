## What This Changes

Add a Memory Dreaming session-corpus hygiene filter that rejects assistant process chatter while preserving durable decisions, verification results, PR refs, and commit refs.

## Best Possible Solution

Add a conservative process-chatter rejection helper at the session-corpus ingestion boundary, with regression coverage proving the reported chatter is dropped and durable decision/evidence lines still flow through.
