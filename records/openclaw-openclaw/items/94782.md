## Summary

## What This Changes

Adds two `fs.mkdir` calls in `ensureAgentWorkspace()` to create `skills` and `.agents/skills` during bootstrap-enabled workspace setup.

## Best Possible Solution

Create the host skill directories in the writable sandbox layout before backend, Docker, and browser mount resolution, cover that path with focused tests, and include redacted real first-launch sandbox proof.

## Solution Assessment
