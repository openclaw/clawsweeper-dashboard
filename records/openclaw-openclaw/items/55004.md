- **[P1] Stop passing tokens into SSH hints:** `src/commands/dashboard.ts:187-190`

  - confidence: 0.89

  - body: This call omits tlsEnabled, so gateway.tls.enabled would still produce plaintext dashboard links. Current main passes TLS state and the dashboard docs require https/wss in that mode.
  - confidence: 0.87
