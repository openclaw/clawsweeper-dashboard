That means sending a chat after enabling the sidebar toggle can widen table/session state even though the PR says All agents is sidebar-only.   - confidence: 0.89

That leaks the sidebar mode into non-sidebar Sessions-tab state. Keep these callers on the scoped helpers, or explicitly make and prove the broader Sessions-table behavior.
  - confidence: 0.88
