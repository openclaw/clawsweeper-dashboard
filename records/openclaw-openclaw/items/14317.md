Add a Slack `groupPolicy: "private"` mode that allows invited private channels while blocking public channels unless explicitly allowlisted.

## Best Possible Solution

Approve or reject Slack private-channel membership as a trust boundary; if approved, implement a Slack-scoped private policy across config, schema, docs, security audit, normal routing, slash-command routing, and focused tests while preserving current defaults.
