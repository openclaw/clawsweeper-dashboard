## Summary

Keep open. Current main and the latest release still lose image/audio modality before hybrid ranking, so media vector hits can be merged with `textScore: 0` and diluted by the default BM25 weight.

## What This Changes

The issue asks for hybrid memory search to avoid BM25 dilution for image and audio chunks so multimodal vector hits can rank under default weights.

## Best Possible Solution

Persist media modality through the memory schema and search-result pipeline, then score image/audio vector hits without BM25 dilution while preserving current hybrid weighting for text chunks.

## Reproduction Assessment

Yes, at source level. Current main classifies image/audio entries before indexing, drops modality from persisted/search rows, and gives vector-only hits `textScore: 0` in the default weighted hybrid formula.

## Solution Assessment

Yes. Carrying persisted modality into the ranker is the narrow maintainable fix; changing global hybrid weights would weaken text search, and inferring modality from file paths would be brittle.
