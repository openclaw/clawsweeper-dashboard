## Best Possible Solution

Refresh this PR or replace it with a narrow current-main patch that keeps the three title attributes, places the scoped `.context-breakdown-item .mono` ellipsis rule in `ui/src/styles/usage.css`, and adds focused render/style coverage with updated UI proof.
