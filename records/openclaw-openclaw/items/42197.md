## What This Changes

Adds native title tooltips and ellipsis styling for skill, tool, and file names in the Usage panel context breakdown.

## Best Possible Solution

Port the intended behavior onto current main by adding `title` attributes in `usage-render-details.ts` and a scoped `.context-breakdown-item .mono` ellipsis rule in `ui/src/styles/usage.css`, then visually verify the Usage panel.

## Reproduction Assessment

Yes. Current source renders long context names as unbounded `.mono` spans and lacks a scoped truncation rule, while the PR screenshots show the before overflow and after truncation path.

## Solution Assessment

Yes in direction, but not as currently patched. Native `title` attributes plus a scoped ellipsis rule are narrow and maintainable, but the CSS must move from the deleted `usage-styles` path into `ui/src/styles/usage.css`.
