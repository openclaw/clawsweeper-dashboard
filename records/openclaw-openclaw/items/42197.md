The PR adds native title tooltips to Usage panel skill/tool/file names and scoped ellipsis CSS for context breakdown rows.

## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Partly. Native title attributes plus scoped ellipsis are the right narrow fix, but the CSS must live in the active `ui/src/styles/usage.css` stylesheet instead of the removed `usageStyles-part3.ts` module.
