## Summary

Keep open. Current main still lacks a native `openclaw doctor --dry-run` mode, and https://github.com/openclaw/openclaw/pull/79734 is an open closing PR with review blockers rather than a merged fix.

## What This Changes

Request a native doctor dry-run/diff mode that previews `openclaw doctor --fix` changes without applying them.

## Best Possible Solution

Land or replace https://github.com/openclaw/openclaw/pull/79734 with a whole-doctor read-only planning mode that previews the same repairs as `--fix`, suppresses writes and side effects, defines exit behavior, and updates docs/tests.
