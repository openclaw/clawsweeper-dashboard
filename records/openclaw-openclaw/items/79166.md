## Summary

Keep open. Current main still lacks a native doctor dry-run mode, and https://github.com/openclaw/openclaw/pull/79734 is an open same-author implementation PR with closing syntax, so this issue should remain paired with that PR until a correct implementation lands or is replaced.

## What This Changes

The issue requests a `--dry-run` or preview mode for `openclaw doctor --fix` so proposed config mutations can be inspected before writing.

## Best Possible Solution

Land a corrected doctor-scoped dry-run implementation through the linked PR or a narrow replacement, aligning it with https://github.com/openclaw/openclaw/issues/13362 and keeping all doctor repair paths read-only while previewing the same plan as `--fix`.
