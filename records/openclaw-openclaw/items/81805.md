## Summary

Keep open. The PR targets a plausible startup performance problem, but the patch places the lazy shortcut too low in shared channel contract loaders, which can drop channel secret targets and can still force full plugin fallback in doctor scans.

## What This Changes

Adds a changelog entry and an opt-in `OPENCLAW_LAZY_BUNDLED_CHANNEL_ARTIFACTS=1` env gate that makes bundled channel doctor and secrets contract artifact loaders return `undefined` before loading contract barrels.

## Best Possible Solution

Keep `openclaw status --json` on a metadata/read-only path while preserving complete channel secret and doctor contracts for audit, configure, doctor, and runtime flows.

## Reproduction Assessment

Yes for the introduced regression by source inspection: with the proposed env var set, bundled channel contract APIs return `undefined` and registry/status paths lose channel secret entries. The original 4.27 timing failure was not independently reproduced here.

## Solution Assessment

No. The shortcut is placed in a shared contract loader; the safer solution is a status-scoped lazy path or metadata fallback that does not change secret audit/configure and doctor behavior.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Preserve channel secret targets in lazy mode:** `src/secrets/channel-contract-api.ts:53-55`
  - body: Returning `undefined` here also affects `getSecretTargetRegistry()` and configured status target discovery: those paths only push `contractApi?.secretTargetRegistryEntries`, so bundled channel targets such as Telegram or Google Chat disappear when the env var is enabled. Scope the shortcut to the status cold path or add a metadata/read-only fallback before skipping artifact loading.
  - confidence: 0.9
- **[P3] Avoid full plugin fallback in doctor lazy mode:** `src/channels/plugins/doctor-contract-api.ts:24-26`
  - body: When the doctor loader returns `undefined`, `collectChannelLegacyConfigRules` falls back to `getBootstrapChannelPlugin`, which can load the full bundled channel plugin. That defeats the lazy-mode guarantee for doctor/legacy-config scans; leave doctor contracts loadable or add a lightweight fallback there too.
  - confidence: 0.82
