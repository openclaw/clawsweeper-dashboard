After an operator turns the knob off, `/sessions <n>` can switch to stale retained sessions and the zero-limit eviction loop can archive the session the user was just in.
  - confidence: 0.9

  - body: This keeps previous transcripts live by assigning them to `sessionHistory`, but current disk-budget cleanup protects only active store entries and compaction checkpoints.
