  - body: `/sessions` builds list and target candidates from persisted `sessionHistory` before checking `session.historyLimit`. If an operator changes the limit to `0` while old history exists, stale sessions remain visible and switchable despite the documented disabled mode.
  - confidence: 0.88

  - body: Adding `sessionHistory` keeps old transcripts live while they are no longer the active `entry.sessionId`.   - confidence: 0.86

  - confidence: 0.78
