  - body: This switch rewrites the active `SessionEntry` to the target session and only reapplies the small `SessionHistoryItem.metadata` subset.   - confidence: 0.84

  - body: When `session.historyLimit` is changed to `0`, this handler still builds `/sessions` output and switch targets from persisted `sessionHistory` before checking the limit.
