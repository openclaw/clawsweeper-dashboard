## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Yes, with a maintainer threshold decision. The latest shape applies the SSE cap only to the unterminated tail after draining valid events, caps JSON synthesis by bytes, and cancels upstream readers; a shared helper is a cleanup alternative, not required for correctness.
