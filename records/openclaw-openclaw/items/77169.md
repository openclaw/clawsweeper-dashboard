## What This Changes

Request native video analysis input for multimodal models, especially Kimi K2.6 `video_url`, accepting local paths or URLs without manual ffmpeg frame extraction.

## Best Possible Solution

## Solution Assessment

Yes: the shared media-understanding provider contract is the maintainable solution here because it covers provider APIs, fallback, local/URL attachment handling, and CLI access without adding a provider-specific core tool.
