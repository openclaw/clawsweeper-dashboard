## Summary

## What This Changes

The PR adds a Browserbase browser profile driver that fetches a fresh Browserbase connectUrl per CDP attach, wires several browser call sites, updates profile types/capabilities/docs/tests, and raises remote CDP timeout defaults.

## Best Possible Solution

Land a Browserbase driver only after runtime schema/help/docs/tests, centralized CDP URL resolution, redacted error handling, credential-surface approval, and redacted real Browserbase proof are aligned; otherwise continue the broader contract discussion in https://github.com/openclaw/openclaw/issues/43455.

## Solution Assessment
