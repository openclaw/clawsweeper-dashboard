- **[P1] Skip cached agent responses before registering:** `src/gateway/server-plugins.ts:594-600`
  - body: The Gateway `agent` method can return cached in-flight payloads for repeated `idempotencyKey` calls, but this code registers every response that contains a `runId`.   - confidence: 0.9

  - confidence: 0.88
