The PR adds `models auth paste-token --token <value>`, validates non-interactive token input, includes `validTypes` in invalid auth-profile warnings, and updates docs, changelog, and focused tests.

## Best Possible Solution

Land this PR or an equivalent maintainer patch if the explicit argv-token path is acceptable; otherwise choose a safer non-interactive credential input design such as stdin or a secret reference while preserving the `validTypes` diagnostic and regression coverage.

## Solution Assessment
