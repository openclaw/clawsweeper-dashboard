Keep this PR open. The PR adds a documented `models auth paste-token --token <value>` CLI path, validates non-interactive token input, surfaces `validTypes` in invalid auth-profile warnings, and updates docs, changelog, and tests.

## Best Possible Solution

Land this PR or an equivalent maintainer patch if argv-token input is acceptable; otherwise choose a safer non-interactive credential design while preserving the `validTypes` diagnostic and regression coverage.
