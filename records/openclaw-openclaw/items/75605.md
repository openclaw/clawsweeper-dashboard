## Best Possible Solution

Land the additive `derivedPaths` hook seam after the remaining exact-head checks complete successfully and maintainers accept the best-effort public contract; keep path enforcement authoritative in the existing executor/policy paths rather than treating derived hints as a security boundary.

## Solution Assessment
