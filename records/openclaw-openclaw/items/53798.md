Keep open. Current main and v2026.5.18 still send Feishu native Ogg/Opus voice uploads through `uploadFileFeishu` without a duration, and the related fix PR was closed unmerged as superseded by this issue, so the user-visible bug still needs a narrow Feishu plugin fix.

## What This Changes

The issue asks for Feishu message-tool voice media sends to display the normal audio duration indicator in the Feishu UI.

## Best Possible Solution

Land a narrow Feishu plugin fix that computes a finite millisecond duration from the actual native or successfully transcoded upload bytes and passes it through `uploadFileFeishu` while preserving current routing, metadata, and fallback behavior.
