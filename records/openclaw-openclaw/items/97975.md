## What This Changes

The PR adds optional chaining before two `.trim()` calls in the embedded run auth controller and image tool sandbox setup.

## Best Possible Solution

Preserve required-state invariants for valid callers, fail closed when sandbox input is malformed, and add focused coverage only for a real reachable bad state.

## Solution Assessment

No. The sandbox fix should fail closed instead of falling back to unsandboxed media loading, and the auth fix should either preserve the required `sourceApiKey` invariant or update the whole refresh path with coverage.
