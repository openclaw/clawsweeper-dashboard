## What This Changes

The issue requests Telegram custom emoji support for messages, captions, and replies, including custom emoji identifiers and outgoing rendering wherever ordinary emoji formatting is supported.

## Best Possible Solution

Choose one canonical product/API representation for Telegram custom emoji, then implement it through the shared message tool and Telegram send/edit/caption delivery paths while preserving the existing raw HTML fallback behavior.

## Reproduction Assessment

## Solution Assessment
