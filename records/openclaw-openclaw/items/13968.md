## Best Possible Solution

Define and document run-kind-aware bootstrap load/cache semantics that skip HEARTBEAT.md before disk reads and cache snapshots for ordinary non-heartbeat runs, or explicitly fold that narrow requirement into https://github.com/openclaw/openclaw/issues/61347.

## Solution Assessment
