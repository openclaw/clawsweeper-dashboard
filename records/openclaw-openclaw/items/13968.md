## Best Possible Solution

Define a maintainer-approved run-kind-aware bootstrap loader/cache contract that skips HEARTBEAT.md for normal default runs before raw file reads, snapshot caching, hooks, and prompt injection, while preserving heartbeat guidance for real heartbeat runs.
