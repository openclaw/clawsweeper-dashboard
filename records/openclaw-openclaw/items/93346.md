## What This Changes

## Best Possible Solution

Review and repair https://github.com/openclaw/openclaw/pull/95491 or a narrow replacement so dropdown reconciliation uses the server-resolved model without breaking Default/null semantics, while leaving the broader backend metadata contract to https://github.com/openclaw/openclaw/issues/51441.
