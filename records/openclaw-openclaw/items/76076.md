## Summary

## What This Changes

Request support or a workaround for automatic replies in WeChat group chats from the wechat-access/OpenClaw Weixin channel plugin.

## Best Possible Solution

Keep OpenClaw core channel-agnostic and route WeChat group-chat support to the external @tencent-weixin/openclaw-weixin/ClawHub plugin, using existing group/channel SDK seams unless plugin work identifies a concrete missing core contract.

## Solution Assessment
