## Summary

## Best Possible Solution

Define the Feishu streaming policy for reasoning and tool status, then implement a Feishu-owned fix that keeps the card body focused on user-facing answer text while preserving any intentional progress signal in a safer footer or separate diagnostic path.

## Solution Assessment
