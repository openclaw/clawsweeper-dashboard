Full review comments:

- **[P1] Scope limit windows to the active credential:** `src/auto-reply/reply/agent-runner.ts:1784-1786`
  - body: `getProviderUsageLimitsCached` only receives the provider and auth mechanism, but the usage loader then resolves whatever configured auth it finds for that provider. In multi-profile/fallback cases, especially Codex/OpenAI auth-profile runs, the custom footer can show quota windows for a different account than the one that produced the reply. Please pass the selected profile/account through the snapshot and cache key, or omit `limits` when the active credential cannot be proven.
  - confidence: 0.87
- **[P2] Do not reuse usage snapshots by session key:** `src/auto-reply/reply/reply-usage-state.ts:53-55`
  - body: This returns a stored snapshot without consuming or clearing it, and the fallback key is `sk:<sessionKey>`. Durable/recovered delivery reruns `reply_payload_sending` as a stateless queued hook with session correlation, so a replay or later delivery within the TTL can receive stale usage numbers despite the new event comment promising `usageState` is absent off the live dispatcher path.
  - confidence: 0.82
- **[P2] Move template loading out of the reply hot path:** `src/auto-reply/usage-bar/template.ts:47`
  - body: Every configured `/usage full` render calls `statSync` before the cache can be used, and a changed mtime then performs a synchronous JSON read/parse. That puts filesystem freshness polling on visible reply delivery; a slow or remote path can stall replies. Please load from process-stable config/startup state or an explicit reload/doctor flow instead of polling during each reply.
  - confidence: 0.9
- **[P2] Document the new config and hook contract:** `src/config/zod-schema.session.ts:160`
  - body: This adds `messages.usageTemplate` and a sizeable `reply_payload_sending.usageState` plugin contract, but current docs/help have no `usageTemplate`, `openclaw.usageBar`, or usage-line contract entries. Operators and plugin authors need the config reference/help text and hook docs updated before this public surface lands.
  - confidence: 0.78
