## What This Changes

## Best Possible Solution

Preserve static Telegram sticker files as normal inbound media through current-message, reply-chain, prompt-context, and sandbox-staging paths, while keeping animated/video sticker policy unchanged unless separately approved.

## Reproduction Assessment

## Solution Assessment

Yes, the best fix is in the Telegram inbound media/context boundary, not in the agent image tool. The fix should ensure the existing saved sticker media path survives into `MediaPaths` and sandbox `media/inbound/*` wherever the sticker is shown to the agent.
