Overall correctness: patch is incorrect

Overall confidence: 0.89

Full review comments:

- **[P2] Remove resolved router config from tool results:** `src/agents/subagent-model-router.ts:191`
  - body: `routerRecommendation` is returned through `sessions_spawn` and serialized into the tool result, so including `resolvedConfig` exposes `command`, `args`, `policyPath`, and `telemetryPath` to the agent transcript. Return only sanitized recommendation metadata.
  - confidence: 0.93
- **[P2] Avoid core fallback model ids:** `src/agents/subagent-model-router.ts:114-120`
  - body: When no external router command returns a model, core hard-codes provider-specific recommendations such as OpenAI Codex and xAI refs. That can emit unavailable or misleading model ids and belongs in configured catalog/plugin policy instead of core.
  - confidence: 0.9
