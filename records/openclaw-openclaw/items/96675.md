## What This Changes

## Best Possible Solution

Define a maintainer-approved responsibility/trust contract that coordinates existing approval hooks, Skill Workshop proposal gates, and memory provenance with durable owner-confirmed reuse state, then split implementation into narrower compatibility-tested PRs.
