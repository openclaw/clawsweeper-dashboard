Keep open. Add optional owner-confirmation gates for assistant memory, actions, skills, and reusable evidence before they become durable trusted context.

## Best Possible Solution

Define a maintainer-approved responsibility/trust contract that coordinates existing approval hooks, external-content handling, memory/wiki provenance, and plugin APIs before adding durable owner-confirmed reuse state.
