Fix https://github.com/openclaw/openclaw/issues/87045. Implement a stable, generic boundary for plugin-emitted system-context sections at the shared composition boundary where feasible; do not hardcode the reporter's plugin, file names, or example strings. Check Codex, Pi embedded, and CLI harness composition so sibling surfaces either share the fix or have a documented reason they are unaffected.
