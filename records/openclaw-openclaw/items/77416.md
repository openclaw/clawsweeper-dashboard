## Summary

Keep open: current main still omits the dist/ sidecar lookup for external channel secret contracts, and the linked open PR targets this exact remaining Discord SecretRef regression.

## What This Changes

Fix Discord external channel SecretRef startup by loading compiled secret-contract API sidecars from plugin dist/ directories.

## Best Possible Solution

Review and land or close #77421, keeping the fix limited to external channel contract discovery and a dist-layout regression test.

## Reproduction Assessment

Yes. Source inspection gives a high-confidence reproduction path: a plugin record whose contract file exists only at rootDir/dist/secret-contract-api.js makes loadExternalChannelSecretContractFromRecord return undefined on current main.

## Solution Assessment

Yes. The narrow maintainable solution is to make resolvePluginContractApiPath recognize compiled dist/ contract sidecars while preserving the existing boundary-file check and flat-layout compatibility.
