- **[P2] Preserve current embedded steer routing:** `src/auto-reply/reply/agent-runner.ts:262`
  - body: This queues the wrapped prompt with `followupRun.run.sessionId` and no queue options, reverting current main's active-session lookup plus steeringMode/debounce handling. After a reset, that can steer into a stale session id or ignore the configured queue behavior.
  - confidence: 0.86

  - confidence: 0.88
