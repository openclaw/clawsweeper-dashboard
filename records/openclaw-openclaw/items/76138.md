## Summary

Keep open. Current main preserves the old transcript by creating and switching to a new dashboard session, but it still displays the same plus-labelled toolbar action next to Send without confirmation, clearer placement/icon, or a hide setting; #61423 is open overlap, not a shipped fix.

Request that the Control UI New Session toolbar button add confirmation, use a clearer icon or location, or expose a hide option to prevent accidental session changes.

## Best Possible Solution
