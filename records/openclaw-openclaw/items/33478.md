The issue requests structured `sessions_send` callback actions plus inherited or explicit announce routing context for agent-to-agent handoffs.

## Best Possible Solution

Define one backward-compatible A2A handoff contract that carries structured control intent and routing context, preserves shipped sentinel compatibility where required, and lands with focused sessions_send, announce, docs, and regression coverage.
