## What This Changes

The PR adds Gateway public-network hardening for IP allow/block lists, per-IP request limiting, TLS and credential warnings, auth audit logging, HSTS behavior, and bind fallback handling.

## Best Possible Solution

Rebase or replace this with one coherent Gateway hardening design that validates the public config surface, uses proxy-aware client-IP contracts, completes audit logging, documents the operator workflow, and includes redacted real-runtime proof.

## Solution Assessment
