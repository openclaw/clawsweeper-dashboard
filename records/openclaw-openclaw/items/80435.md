## What This Changes

The issue requests that clean `openclaw doctor` runs without `--fix` suppress the `Run "openclaw doctor --fix" to apply changes.` trailer when no config repair is pending.

## Best Possible Solution

Land one canonical doctor-flow fix so clean no-write doctor runs end at `Doctor complete.` while pending repair guidance remains actionable and covered by regression tests.
