## Summary

Keep open for review changes: the feature is useful and has real gateway-mode proof, but the patch still leaks the node-host sentinel as a session marker in the macOS node-mode path and needs maintainer approval for the trusted routing-marker contract.

## What This Changes

The PR injects `OPENCLAW_AGENT_ID` and `OPENCLAW_SESSION_KEY` into exec child-process environments across gateway/node-host and macOS execution paths, with docs, changelog, and tests.

## Best Possible Solution

Ship one trusted exec env-marker contract that clears missing context to empty strings, prevents user-env spoofing, documents routing-not-secret semantics, and reconciles the broader ACP/TUI scope separately.

## Reproduction Assessment

Yes for source-level verification: current main lacks the requested child env markers, and the PR head shows the macOS node-mode path exporting `sessionKey` directly while other paths clear the `node` sentinel. I did not run live exec commands in this read-only pass.

## Solution Assessment

No as-is. The exec-boundary approach is maintainable, but the macOS node-mode sentinel handling should be fixed and maintainers should approve the routing-marker and canonical-scope contract before merge.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Clear the node sentinel before exporting session markers:** `apps/macos/Sources/OpenClaw/NodeMode/MacNodeRuntime.swift:945`
  - body: The TypeScript and companion paths treat `sessionKey === "node"` as missing and export an empty string, but MacNodeRuntime writes that sentinel directly to `OPENCLAW_SESSION_KEY`. A host=node run without a real session can expose `OPENCLAW_SESSION_KEY=node`, contradicting the docs and giving callback scripts an invalid routing key; normalize the child env marker separately from the event routing key.
  - confidence: 0.87

## Security Review

This is the requested capability, but it should be documented as routing context rather than a standalone authorization secret.
