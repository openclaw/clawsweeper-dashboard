## Summary

Current main and v2026.6.5 still do not provide an enforceable private/no-persist session mode. This remains the canonical privacy-mode tracker and requires maintainer product and security design before implementation.

## What This Changes

Requests an enforceable per-session private mode that suppresses workspace and memory context, avoids durable session history, uses temporary state, and exposes a Control UI toggle.

## Best Possible Solution

Design one additive private/ephemeral session capability with a documented threat model, then enforce it across Gateway protocol, in-memory session lifecycle, bootstrap and memory access, every durable writer, channel activation, and Control UI, with negative persistence tests.

## Reproduction Assessment

Yes at source level: current protocol and session-state types lack a privacy mode, and durable session paths have no enforcement bit that could suppress persistence. A write-producing runtime test was not run because this review is read-only.

## Solution Assessment

No, a Mission Control toggle alone is not the best solution. The maintainable approach is one runtime-enforced session capability whose UI toggle is only an activation and status surface.

## Security Review

Status: needs_attention

Summary: The requested mode is itself a security boundary and needs an explicit threat model before its UI or API can promise privacy.

Concerns:

- **[medium] Define the complete non-persistence boundary:**
  - body: The design must state whether transcripts, session metadata, memory and compaction output, tools, media, logs, plugin state, channel events, and crash artifacts are prohibited, retained, or redacted; otherwise the private-mode promise is not enforceable.
  - confidence: 0.96
- **[medium] Avoid a UI-only privacy guarantee:**
  - body: A private indicator must only activate after the runtime has accepted and enforced the session capability; otherwise users could reasonably believe data is protected while durable writers remain active.
  - confidence: 0.94
