## Summary

Keep open: current main and v2026.6.8 still lack an enforceable Private/Secret/Incognito session contract, while session creation and transcript storage remain durable by default.

## What This Changes

Request a Mission Control Private Mode that suppresses workspace context and prevents private sessions from reading/writing workspace memory or durable history.

## Best Possible Solution

Keep this as the canonical tracker and design one additive private/ephemeral session contract across Gateway protocol, session storage/transcripts, agent context/memory, plugin enforcement, channel triggers, and Control UI.
