Implement IPv6 support for OpenClaw Gateway LAN/listen behavior for https://github.com/openclaw/openclaw/issues/50253, preserving existing non-loopback auth and Control UI origin safeguards. Treat https://github.com/openclaw/openclaw/issues/63743 as related duplicate context. Update the gateway bind helpers so LAN mode can listen on IPv6 as well as IPv4 where supported, and allow appropriate IPv6 literals for `gateway.bind="custom"`/`gateway.customBindHost` without accepting malformed host:port strings.
