## What This Changes

The branch extracts shared subagent cleanup finalization in `src/agents/subagent-registry.ts` and adds a steer-restart test for delete-mode termination idempotency.

## Best Possible Solution

No, not as-is. The cleanup invariant is the right direction, but the patch targets stale monolithic registry code and must be ported to the active split modules before merge.
