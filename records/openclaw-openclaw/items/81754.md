## What This Changes

Request gzip or Brotli compression for gateway-served Control UI text assets when clients send Accept-Encoding.

## Best Possible Solution

Review and land https://github.com/openclaw/openclaw/pull/81794, or an equivalent narrow gateway static-asset compression change, then close this issue after merge.
