  - body: The UI seq comes from current main’s active transcript index, but this handler recomputes seq by walking every physical JSONL message/compaction line and then slices by that line index. Deleting from that view can therefore resolve the default/global transcript instead of the selected agent’s transcript.
  - confidence: 0.86
