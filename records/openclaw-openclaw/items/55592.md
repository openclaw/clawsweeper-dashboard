Current reset/history paths add `scopedAgentParamsForSession(...)` so selected-global sessions target the chosen agent; without the same `agentId`, deleting while viewing a non-default global agent can truncate the default/global transcript instead of the displayed one.
  - confidence: 0.91

  - body: The server maps `seq` by counting raw JSONL records and then slices `allLines`, but current chat-history seq values come from the active transcript index, which follows the leaf/parent branch and tracks byte ranges for visible entries. On branched or compacted transcripts this can remove or preserve the wrong records, so the mutation needs to use the accessor/index seam rather than physical line counting.
  - confidence: 0.88
