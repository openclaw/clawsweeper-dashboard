- **[P1] Carry selected agent scope into truncation:** `ui/src/ui/app-render.ts:1508-1510`
  - body: The truncate request only sends `{ key, seq }`, but selected-global session mutations on current main carry `agentId` through the protocol and server target resolution.   - confidence: 0.9

  - body: The UI sequence comes from the active transcript index, while this handler counts physical JSONL records and slices raw lines. With side branches, leaf controls, or compaction records, the requested seq can map to a different physical row, causing the backend to preserve deleted content or remove unrelated transcript data.
  - confidence: 0.91
