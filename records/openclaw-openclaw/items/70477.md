## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Bind bootstrap bypass to server-trusted local provenance:** `src/gateway/control-ui.ts:690-692`
  - body: This branch skips normal Control UI auth and can include `gatewayToken` based on a loopback socket plus client-supplied browser headers. A same-host reverse proxy or raw client that reaches the backend over loopback can spoof `Host`/`Origin`/`Sec-Fetch` metadata and receive the token-bearing response; keep this route authenticated or prove the request is direct using server-owned provenance.
  - confidence: 0.9
- **[P2] Swallow terminal events from pending-abort runs:** `ui/src/ui/controllers/chat.ts:443-445`
  - body: When a terminal event arrives for `chatAbortPendingRunId`, this branch clears the marker but returns the terminal state. `handleChatGatewayEvent` then runs terminal cleanup and can reset tool streams or reload history while a replacement run is active; return `null` after clearing the marker so the stale abort event is ignored.
  - confidence: 0.93
