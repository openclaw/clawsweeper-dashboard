## What This Changes

Issue requests native Discord typing indicator support through the message tool, either as a Discord action or a more generic channel typing/busy action.

Define an additive channel-owned typing or busy-control contract, wire Discord execution to the existing typing primitive, and cover discovery, lifecycle cleanup, docs, focused tests, and real Discord proof.
