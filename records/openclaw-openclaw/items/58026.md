## Best Possible Solution

Expand a leading home prefix in the canonical builtin `normalizeExtraMemoryPaths()` path before the absolute/workspace-relative branch, add regression coverage for list/read/status behavior, and keep doctor diagnostics aligned with that shared normalizer.
