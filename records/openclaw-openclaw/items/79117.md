## What This Changes

## Best Possible Solution

Keep the doctor migration path for legacy runtime keys, then decide a narrow safe policy for invalid-config `sessions` behavior and either document heartbeat timeout as operator-only or deliberately add it to the agent-facing allowlist with focused tests.
