  - body: `resolveSandboxContext` runs `maybePruneSandboxes` before backend creation, and prune can call Docker removal with no timeout. With an expired registry entry and a wedged Docker engine, a run can still hang before it reaches this newly bounded container-state call.

  - confidence: 0.78
