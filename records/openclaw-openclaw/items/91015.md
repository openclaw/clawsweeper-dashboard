## Summary

## Best Possible Solution

## Solution Assessment

Partly. A per-call Docker exec deadline is a plausible mitigation, but the config/default surface, overlap with the broader timeout PR, missing docs, and missing real proof mean this is not merge-ready as the best complete fix yet.
