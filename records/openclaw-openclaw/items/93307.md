  - body: This samples page-global downloads after the action without owning an action-scoped waiter. A download from another page action can be attributed to this `/act` call, while a click-triggered download that starts after this drain/sample window is still omitted from the response.
  - confidence: 0.84

  - body: `drainPendingDownloadSaves` awaits the current global array and then clears it. An older unrelated download can stall this action, and a new save pushed while `Promise.all` is already waiting can be cleared before a later caller awaits it; track only the downloads for this action window.
  - confidence: 0.87
