The branch scopes plugin-tool runtime loads to active plus enabled plugin IDs, stores core Gateway method names on plugin registries and snapshots, relaxes active-registry cache compatibility when requests omit those method names, adds regression tests, and updates CHANGELOG.

## Solution Assessment
