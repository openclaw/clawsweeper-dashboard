- **[P1] Decode detected encodings with a supported decoder:** `src/agents/sessions/tools/read.ts:347`
  - body: The detector can return `shift-jis`, `gbk`, or `utf-16be`, but this call passes those names directly to `Buffer#toString`. Node Buffer does not support those encodings, so a legacy-encoded file can throw `ERR_UNKNOWN_ENCODING` instead of returning text; route the result through `decodeBuffer`, `TextDecoder`, or an existing decoder helper before using these detections.
  - confidence: 0.94
- **[P1] Make the GBK detector fix the linked GBK case:** `src/agents/sessions/tools/encoding-detection.ts:218-219`

That leaves the linked GBK bug unfixed even after the call site uses a real decoder, so the detector needs focused GBK coverage or the branch should avoid claiming GBK auto-detection.
  - confidence: 0.9
