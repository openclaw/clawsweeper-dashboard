  - body: The new config field accepts any string, then the gateway forwards it to the UI where the helper applies it as Intl's `timeZone` option. A typo such as `Not/AZone` throws `RangeError` on first timestamp render, so reject or safely omit invalid values before bootstrapping them.

  - confidence: 0.84
