The issue asks to tighten voice-call realtime WebSocket upgrade routing so configured stream paths match exactly or as slash-delimited child paths, not sibling prefixes.

## Best Possible Solution

Review and land https://github.com/openclaw/openclaw/pull/79919 or an equivalent focused fix with regression coverage for sibling and root stream paths.

## Solution Assessment
