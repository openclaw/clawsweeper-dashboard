The issue requests a reliable way for cron isolated sessions to spawn multiple subagents, wait for all results, and aggregate them before cron delivery.

## Best Possible Solution

Keep the shipped run-scoped descendant drain and delivery fallback; broader explicit delegation APIs can be evaluated separately from this now-fixed cron delivery failure.
