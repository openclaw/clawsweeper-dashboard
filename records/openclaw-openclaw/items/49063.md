That enables slash commands in any readable Telegram channel, contradicting the PR's explicitly allowed chat boundary.
  - confidence: 0.88
- **[P1] Preserve channel context for command callbacks:** `extensions/telegram/src/bot-native-commands.ts:602`
  - body: Changing native/plugin commands to read `ctx.msg` makes channel commands that emit inline keyboards reachable, but the callback handler still treats `chat.type === "channel"` as non-group. Button follow-ups from commands such as `/models` can then be authorized and routed with DM callback/session rules instead of the channel context.
  - confidence: 0.86
