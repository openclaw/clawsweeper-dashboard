## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P1] Use sender_chat when authorizing channel-post commands:** `extensions/telegram/src/bot-native-commands.ts:227-228`
  - body: This branch enables `ctx.msg` channel posts, but authorization still derives `senderId` only from `msg.from`. Telegram channel posts normally identify the posting channel in `sender_chat`, so `commands.allowFrom.telegram` or owner allowlists that name the channel never match and explicitly allowed channel commands are rejected.
  - confidence: 0.89
- **[P1] Do not allow all channel posts when access groups are off:** `extensions/telegram/src/bot-native-commands.ts:333-344`
  - body: For `isChannelPost` with `useAccessGroups: false`, this branch supplies an empty authorizer list. With no `commands.allowFrom` configured, `modeWhenAccessGroupsOff: "configured"` treats that as allowed, so any Telegram channel the bot can read may execute native/plugin commands instead of only explicitly allowed chats.
  - confidence: 0.88
- **[P1] Keep channel command callbacks in the channel session:** `extensions/telegram/src/bot-native-commands.ts:602`
  - body: Switching handlers to `ctx.msg` makes interactive channel commands such as `/models` reachable, but the callback runtime still treats `chat.type === "channel"` as non-group and authorizes/routes through the direct session. Button follow-ups from a channel command can therefore be dropped or mutate the clicker's direct session instead of the channel session.
  - confidence: 0.86
- **[P2] Honor per-channel groupPolicy for channel posts:** `extensions/telegram/src/bot-native-commands.ts:298-299`
  - body: The channel-post path calls group policy access without topic/group overrides and then treats `channelPolicy.groupConfig` as enough to authorize the command. A channel entry with `groupPolicy: "disabled"` or narrower per-channel policy can still pass this explicit-entry check, unlike the normal inbound channel_post path that applies group overrides.
  - confidence: 0.8
