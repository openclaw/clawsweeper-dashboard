  - body: For channel posts, this branch builds an empty authorizer list when useAccessGroups is false and commands.allowFrom is unset. configured-mode command gating treats that as allowed, so slash commands can run from any Telegram channel the bot can read, contradicting the stated explicitly allowed chat boundary.

  - confidence: 0.86

  - body: Reading ctx.msg enables channel_post native commands that can emit inline buttons, but callback handling still routes through the clicking user's direct-chat auth/session path.   - confidence: 0.82
