## Summary

Keep open: the performance goal is useful and the PR body includes real benchmark evidence, but the patch still breaks compatibility for runtime-only plugin help and incomplete pairing help by skipping the plugin/channel registration path too broadly.

## What This Changes

The PR adds early CLI help fast paths for built-in and plugin subcommands, a help-only metadata plugin registration path, and focused CLI/plugin tests.

## Best Possible Solution

Rebuild the help performance work on current main using manifest/descriptor-owned command lookup while preserving runtime plugin fallback and pairing channel help semantics.

## Reproduction Assessment

Yes for the review findings by source inspection. The diff routes fallback plugin help registration through metadata-only loading and registers pairing help before plugin-backed channel data can be populated.

## Solution Assessment

No. The broad helpOnly switch and pairing short-circuit are not the safest implementation; the repair should use current owner resolution and only take metadata-only paths when compatibility is known.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Preserve runtime plugin help fallback:** `src/cli/run-main.ts:345`
  - body: Passing helpOnly: hasHelpOrVersion(parseArgv) makes every non-fast-path plugin help invocation use the CLI metadata collector. Third-party or legacy plugins that only register CLI commands from their runtime entry can stop exposing openclaw <plugin> --help, so keep the runtime fallback unless the command owner is proven metadata-backed.
  - confidence: 0.88
- **[P2] Keep pairing help channel data:** `src/cli/program/register.subclis.ts:235-237`
  - body: Returning before plugin registration means registerPairingCli() snapshots listPairingChannels() before plugin-backed channels are registered. The --channel option and argument help can render empty or incomplete channel lists, so preserve a channel metadata source before registering pairing help.
  - confidence: 0.8
