## Summary

Keep open. The performance goal is plausible and the PR includes benchmark-style behavior proof, but the branch is currently conflicting and the proposed implementation still has compatibility blockers around metadata-only plugin help and pairing channel help.

## What This Changes

The PR adds an early CLI help path for built-in and plugin subcommands, switches help-only plugin CLI loading toward metadata registration, and adds focused CLI/plugin registration tests.

## Best Possible Solution

Rebuild the help fast path on current main using existing descriptor- and activation-aware plugin CLI loaders while retaining runtime fallback and pairing channel help semantics.

## Reproduction Assessment

Yes for the review findings. Source inspection of the PR diff and current contracts shows runtime-only plugin CLI help can be skipped, and pairing help derives channel choices at registration time.

## Solution Assessment

No. The fast-path direction is reasonable, but this patch should be replaced or repaired on current main so it does not route all help fallback through metadata-only loading or drop channel data needed by pairing help.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Preserve runtime plugin help fallback:** `src/cli/run-main.ts:345`
  - body: Passing `helpOnly` for every help invocation routes fallback plugin command registration through metadata-only entries. Runtime-only third-party plugin CLIs that do not expose `cli-metadata` can lose `openclaw <plugin> --help`, so keep the full runtime fallback unless the primary command is known descriptor-backed.
  - confidence: 0.88
- **[P2] Keep pairing help channel data:** `src/cli/program/register.subclis.ts:232-235`
  - body: This short-circuits `pairing --help` before plugin CLI registration, but `registerPairingCli()` snapshots `listPairingChannels()` into option and argument help. Plugin-backed channels can disappear from help text, so preserve the registration or metadata source needed for pairing help.
  - confidence: 0.84
