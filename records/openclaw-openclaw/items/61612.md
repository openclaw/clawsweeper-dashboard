## Summary

Keep open: current main has not absorbed this subcommand help fast path, but the PR still needs compatibility fixes before merge because its metadata-only fallback can drop runtime-registered plugin help and its pairing shortcut can omit plugin-backed channel data.

## What This Changes

The PR adds CLI subcommand help fast-path helpers, a metadata-only plugin help registration option, pairing/plugins help shortcuts, and focused CLI/plugin tests.

## Best Possible Solution

Keep the performance direction, but implement a descriptor-aware fast path on current main that preserves runtime plugin CLI compatibility and plugin-populated pairing help.

## Reproduction Assessment

Yes for the review findings: source inspection shows the metadata-only fallback can skip runtime-registered plugin CLI commands, and pairing help snapshots channel names after plugin registration. I did not run live CLI commands because this review was read-only.

## Solution Assessment

No. The fast-path idea is useful, but the current diff is broader than the compatibility contract; a conservative descriptor-aware implementation with full-loader fallback and preserved pairing channel registration is safer.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Preserve runtime plugin help fallback:** `src/cli/run-main.ts:342`
  - body: Passing `helpOnly` for every help invocation sends fallback plugin command registration through metadata-only entries. Plugins that expose CLI commands only from their runtime entry can lose `openclaw <plugin> --help`, so keep the full runtime fallback unless the command is known descriptor-backed.
  - confidence: 0.88
- **[P2] Keep pairing help channel data:** `src/cli/program/register.subclis.ts:232-235`
  - body: This short-circuits `pairing --help` before plugin CLI registration, but `registerPairingCli()` snapshots `listPairingChannels()` into option and argument help. Plugin-backed channels can disappear from the help text, so preserve the plugin registration needed for pairing help.
  - confidence: 0.86
