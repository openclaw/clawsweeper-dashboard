## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Preserve runtime plugin help fallback:** `src/cli/run-main.ts:342`
  - body: Passing `helpOnly` for every help invocation sends fallback plugin command registration through metadata-only entries. Plugins that expose CLI commands only from their runtime entry can lose `openclaw <plugin> --help`, so keep the full runtime fallback unless the command is known descriptor-backed.
  - confidence: 0.88
- **[P2] Keep pairing help channel data:** `src/cli/program/register.subclis.ts:231-235`
  - body: This short-circuits `pairing --help` before plugin CLI registration, but `registerPairingCli()` snapshots `listPairingChannels()` into option and argument help. Plugin-backed channels can disappear from help text, so preserve the registration needed for pairing help.
  - confidence: 0.86
