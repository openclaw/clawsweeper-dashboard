## Summary

Keep open: the performance goal is useful, but the branch is conflicting and still drops runtime-backed plugin help fallback plus pairing channel help data, so it needs repair rather than cleanup closure.

## What This Changes

The PR adds early CLI help fast paths and a help-only plugin CLI registration mode so selected built-in and plugin subcommand help can avoid full CLI bootstrap.

## Best Possible Solution

Rebuild the fast path on current main using manifest/descriptor ownership and activation-planning loaders while preserving runtime fallback and pairing channel help semantics.

## Reproduction Assessment

Yes for the review findings by source inspection. The PR routes fallback plugin help registration through metadata-only loading, and pairing help registers before channel-providing plugins can populate listPairingChannels().

## Solution Assessment

No. The optimization should be rebuilt on current main around existing manifest/descriptor ownership and targeted runtime fallback instead of global helpOnly metadata loading and a hardcoded core map.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Preserve runtime plugin help fallback:** `src/cli/run-main.ts:345`
  - body: Passing `helpOnly` for every help invocation routes fallback plugin command registration through metadata-only entries. Runtime-only third-party plugin CLIs, or plugins with incomplete `cli-metadata`, can lose their `openclaw <plugin> --help` surface, so keep the full runtime fallback unless the primary command is known to be safely materialized from metadata.
  - confidence: 0.88
- **[P2] Keep pairing help channel data:** `src/cli/program/register.subclis.ts:232-237`
  - body: This early return registers `pairing` help before plugin CLI registration, but `registerPairingCli()` snapshots `listPairingChannels()` into option and argument help. Plugin-backed channels can disappear from fresh help output, so preserve the registration or metadata source needed before rendering pairing help.
  - confidence: 0.84
