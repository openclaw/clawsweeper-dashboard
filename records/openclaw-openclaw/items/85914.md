## What This Changes

Request native, configurable tool-call failure recovery inside the active OpenClaw agent run loop.

## Best Possible Solution

Define an opt-in core recovery contract that classifies structured failed tool results, bounds same-run continuations, preserves side-effect metadata, and documents embedded/Codex capability limits.

## Solution Assessment
