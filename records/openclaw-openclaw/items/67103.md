Rework this PR or a successor against current main so math rendering is added in the shared Control UI markdown renderer while preserving cache and DOMPurify link hardening, using robust delimiter rules, scoping KaTeX-specific sanitizer allowances, updating the lockfile, and proving the visible UI behavior.
