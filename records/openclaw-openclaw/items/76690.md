Fix the OpenAI plugin Codex OAuth image-generation failure reported at https://github.com/openclaw/openclaw/issues/76690. Do not add a new config option, do not move provider-owned behavior into core, and do not silently fall back to an API key when a configured Codex OAuth profile was selected.
