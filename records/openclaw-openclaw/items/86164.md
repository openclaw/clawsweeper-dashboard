Full review comments:

- **[P1] Export the broker runtime setter:** `extensions/channel-broker/runtime-api.ts:1`
  - body: `index.ts` declares `runtime.exportName: "setChannelBrokerRuntime"`, and the bundled entry contract loads that named export during registration.   - confidence: 0.98

  - body: This handler reads the unsigned broker webhook body with the post-auth profile before checking the HMAC signature. That gives unauthenticated requests the 1 MiB/30s body budget instead of the shared 64 KiB/5s pre-auth budget, so invalid webhook traffic can consume more memory/time before authentication.
  - confidence: 0.95
