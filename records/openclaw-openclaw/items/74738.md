The branch adds a `supportsModelTools(model)` guard to OpenAI Completions payload construction, adds two regression tests, and adds a changelog line.

## Best Possible Solution

Land the builder-level guard and regression tests while keeping the dispatch wrapper as defense in depth, then consolidate the duplicate changelog text during landing.
