## What This Changes

The PR adds a `supportsModelTools(model)` guard to the OpenAI Completions payload builder, adds regression tests for non-tool models, and adds a changelog entry.

## Best Possible Solution

Land the builder-level guard and focused regression tests while keeping the existing wrapper as defense in depth, with the duplicate changelog entry consolidated during landing.

## Solution Assessment
