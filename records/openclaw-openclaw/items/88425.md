## What This Changes

The branch resolves a context engine before ACP dispatch, passes assembled prompt text to `acpManager.runTurn`, keeps transcript persistence on the original prompt text, and adds unit coverage for injection and assemble-failure fallback.

## Best Possible Solution

Define ACP's context-engine host support or an explicitly enforced text-only subset, reject or handle unsupported assemble outputs, resolve conflicts, and prove the behavior in a real bound ACP session.

## Reproduction Assessment

Yes, source-level: current main sends ACP `turnPromptText` directly to `runTurn`, while the context-engine contract has an `assemble()` pre-prompt hook. I did not run a live bound ACP/context-engine session in this read-only review.
