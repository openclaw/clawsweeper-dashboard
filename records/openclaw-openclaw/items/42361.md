- **[P1] Regenerate the lockfile after removing overrides:** `package.json:427`
  - body: Removing the `node-domexception` override from `package.json` without the matching `pnpm-lock.yaml` update leaves frozen installs out of sync. CI and fresh checkouts can fail before tests run until the lockfile is regenerated and reviewed.
  - confidence: 0.95
- **[P2] Keep rejecting Feishu key separators:** `extensions/feishu/src/external-keys.ts:18-19`
  - body: This check only rejects path segments equal to `.` or `..`, so values like `a/b` and `a\b` pass normalization. Those keys are later used as `image_key` and `file_key` path parameters, regressing the current separator guard.
  - confidence: 0.9
- **[P2] Keep group chat IDs out of sender allowlists:** `src/telegram/bot-access.ts:47-51`
  - body: Allowing negative IDs makes Telegram group chat IDs look valid in `allowFrom` and `groupAllowFrom`, but runtime authorization compares those lists with `from.id`. Users still will not authorize the group, and the invalid-config warning disappears.
  - confidence: 0.88
