- **[P1] Regenerate the lockfile with override changes:** `package.json:427`
  - body: Removing the node-domexception override from package.json without the matching pnpm-lock.yaml update leaves frozen installs/config out of sync, so CI and fresh checkouts can fail before tests run.

- **[P2] Keep Feishu key separators rejected:** `extensions/feishu/src/external-keys.ts:18-19`
  - body: The new segment check lets values like `a/b` and `a\\b` pass normalization, and those values later feed Feishu `image_key`/`file_key` path parameters. Keep slash and backslash rejected while relaxing only the intended embedded `..` case.
  - confidence: 0.9
- **[P2] Keep group chat IDs out of sender allowlists:** `src/telegram/bot-access.ts:47-51`
  - body: Negative Telegram values are chat IDs, but this allowlist authorizes `from.id` sender IDs. Accepting negative entries hides the invalid-config warning without authorizing the group; group chat IDs belong in the group configuration path.
  - confidence: 0.88
