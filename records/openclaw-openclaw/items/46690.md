## What This Changes

The issue requests guardrails and diagnostics for skill repositories created outside OpenClaw’s canonical skill search paths.

## Best Possible Solution

Define a bounded orphan-skill diagnostic for openclaw skills check or doctor, decide scan roots/redaction/duplicate rules, update skill-creator guardrails, and keep normal loading limited to configured roots.
