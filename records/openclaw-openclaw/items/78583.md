## Summary

Keep open: the feature is active and proof-backed, but the patch exposes operator.admin gateway helpers through the public plugin SDK, which undercuts the claimed proof-only approval boundary and needs maintainer security review before merge.

## What This Changes

Adds an opt-in bundled AgentKit plugin with World ID HITL approvals, proof-backed plugin approval resolution, TUI/chat/channel approval action rendering, docs, dependencies, and focused tests.

## Best Possible Solution

Keep the AgentKit flow only if maintainers accept bundling it, and replace the public admin Gateway client exposure with a narrow plugin-bound proof-resolution/injection seam before merge.

## Reproduction Assessment

Not applicable: this PR adds a new opt-in plugin/protocol/channel capability rather than fixing a broken current-main behavior. The PR body, copied live TUI/gateway output, downloaded MP4 artifact, and successful Real behavior proof check give after-fix evidence for the intended TUI flow.

## Solution Assessment

No as written: the opt-in AgentKit shape is coherent, but exporting a general operator.admin Gateway client through the public plugin SDK is broader than the proof-backed approval problem requires. A narrower plugin-bound resolver or internal-only admin helper would preserve the HITL boundary more safely.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.82

Full review comments:

- **[P1] Keep admin Gateway clients out of the plugin SDK:** `src/plugin-sdk/gateway-runtime.ts:38-40`
  - body: These new SDK exports let any plugin bootstrap an `operator.admin` Gateway client and then invoke every admin-scoped method, including `chat.inject`, `secrets.resolve`, `skills.install`, and `plugin.approval.resolveVerified`. Because `resolveVerified` trusts a caller-supplied `pluginId`, another plugin could resolve AgentKit approvals without going through the World proof path. Keep this helper internal or expose a narrow plugin-bound resolver/injection seam instead.
  - confidence: 0.87

## Security Review
