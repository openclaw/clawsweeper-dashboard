## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Keep normal Stop on chat.abort:** `ui/src/ui/controllers/chat.ts:337`
  - body: The normal WebChat stop path now calls `sessions.abort`, whose new handler clears queues, stops subagents, cancels ACP sessions, and disables session-bound cron jobs. Stopping one in-flight answer can therefore pause unrelated reminders or background work; keep ordinary generation aborts on `chat.abort` and use an explicit autonomy/session-stop action for broader cleanup.
  - confidence: 0.9
- **[P2] Replace no-delta snapshots instead of appending them:** `ui/src/ui/controllers/chat.ts:208`
  - body: When a non-prefix assistant snapshot arrives without `delta`, this fallback appends `nextText` to the existing stream. Gateway snapshots are authoritative in that case, so replayed or rewritten text can be duplicated or corrupted in the UI and in locally committed abort partials.
  - confidence: 0.86
- **[P2] Count only pending followups:** `src/auto-reply/reply/queue/state.ts:43`
  - body: `droppedCount` records discarded queue entries, not pending work. Including it makes `followupQueued > 0` and `canStop: true` after the queue is empty, so the UI can show a stop control for work that cannot be stopped.
  - confidence: 0.91
- **[P2] Return cleanup state after timeout errors:** `src/gateway/server-methods/sessions.ts:1030-1040`
  - body: By this point queues, subagents, ACP sessions, and cron jobs may already have been cleaned up. If the embedded-run wait times out, returning only `UNAVAILABLE` leaves clients unable to reconcile stopped work and can keep stale UI state or trigger redundant retries.
  - confidence: 0.75
- **[P3] Refresh heartbeat state when disabled elsewhere:** `ui/src/ui/app-gateway.ts:413-419`
  - body: Incoming heartbeat events only set `heartbeatsEnabled` to true, and no connected event path clears it when another client disables heartbeats. The heart-pulse button can remain visible until reconnect even though heartbeats are off.
  - confidence: 0.78
