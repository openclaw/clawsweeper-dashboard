  - body: The PR replaces the repository root package.json with a standalone alphabet-harvester manifest. Because the workspace includes the repo root, this removes OpenClaw's CLI bin, scripts, files list, and dependencies, breaking pnpm tooling and packaging for the whole project.

  - body: The pg Pool is configured with rejectUnauthorized: false whenever sslmode is not disabled, so the default cloud path uses encrypted transport without certificate validation. That weakens database TLS and should require an explicit insecure/local opt-in instead.
  - confidence: 0.88

  - body: The added Dockerfiles FROM local unqualified tags such as alpacoreengine-alpacore:latest and alpacore-service:latest. Azure/CI builds will fail unless those private local tags already exist, so the ACA flow is not reproducible from this repository.
  - confidence: 0.93
