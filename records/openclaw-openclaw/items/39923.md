## Summary

Keep open: current main still uses numbered config backup rotation, the focused tests still assert `.bak.N` slots, and two open PRs already target this exact request with closing references.

## What This Changes

Use datetime-stamped config backup suffixes instead of the current numbered `.bak.N` rotation ring.

## Best Possible Solution

Review and land one chosen timestamp-suffix implementation, or fold it into https://github.com/openclaw/openclaw/issues/20460, with explicit legacy migration behavior, collision-safe naming, bounded pruning, permission hardening, and focused backup rotation tests.

## Solution Assessment
