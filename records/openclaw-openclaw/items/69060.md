## What This Changes

The PR adds an `x-openclaw-session` Gateway HTTP header, docs, changelog text, and Chat Completions tests for stable agent-scoped session reuse when callers cannot set the OpenAI `user` field.

## Best Possible Solution

Make the public header contract explicit first, then either keep the hint Chat Completions-only or update Responses scoping, docs, and tests consistently; also fix the Mintlify heading and proof-gate parsing before merge.

## Solution Assessment
