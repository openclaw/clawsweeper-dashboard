Drive the first version from existing local chat lifecycle signals: chat send/run start, terminal final/aborted/error events, abort, disconnect/reconnect orphan cleanup, queued/finalizing state where available, and fail closed to Interrupted or non-running when reconnect ownership is uncertain.
