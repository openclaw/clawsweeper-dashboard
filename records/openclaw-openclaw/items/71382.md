  - confidence: 0.88

  - body: An empty text/no-media send returns a success-shaped result before loading config and calling `assertWhatsAppWritable`.   - confidence: 0.82
- **[P2] Document the new readOnly config contract:** `src/config/types.whatsapp.ts:56-57`

  - confidence: 0.78
- **[P3] Align composing with the read-only send error:** `extensions/whatsapp/src/inbound/monitor.ts:546-548`
  - body: `sendComposing` silently returns in read-only mode while `sendTrackedMessage` and `sendPresenceUpdate` throw the read-only error. Callers that rely on the blocked-send signal will see different behavior for typing indicators than for other outbound paths.
  - confidence: 0.75
