## Summary

Keep open: current main still has the Telegram polling watchdog, session-store save path, transcript rewrite/rotation path, and adjacent plugin hook infrastructure, but it does not expose the requested plugin-observable polling or session-persistence hook families. The issue requests new plugin-observable hook families for polling watchdog lifecycle events and session persistence/transcript rewrite or rotation events.
