- **[P2] Propagate callback silence to select submit paths:** `extensions/telegram/src/bot-handlers.runtime.ts:2764`

  - confidence: 0.9
- **[P2] Keep callback silence from masking delivery failures:** `extensions/telegram/src/bot-message-dispatch.ts:2622`
  - body: `silentCallbackSuppressed` is true whenever the callback flag is set and no message/fallback was delivered, even if `deliverySummary.skippedNonSilent` or `failedNonSilent` was incremented. That turns a failed visible callback delivery into `hasFinalResponse`, bypassing the existing retry/error path. Gate this on zero non-silent skips/failures or compute it after `deliveryFailureWithoutFinalResponse`.
  - confidence: 0.78
