## What This Changes

## Best Possible Solution

Carry the callback-only silent fallback suppression through every agent-routed Telegram callback submit path, with focused tests for generic, OC_MULTI submit, and OC_SELECT silent and visible replies.

## Reproduction Assessment

Yes, source-reproducible. The PR body proves the generic callback path in live Telegram, and PR-head source shows managed select callbacks still omit the suppression flag while dispatch only suppresses when that flag reaches context.

## Solution Assessment

No. The flag and dispatch shape are narrow and maintainable, but applying it only to generic callback data is incomplete because managed select submit callbacks share the same synthetic button-turn behavior.
