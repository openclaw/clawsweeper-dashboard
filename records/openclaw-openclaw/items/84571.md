## Summary

Keep open for maintainer design review: current main has partial compaction visibility, manual compaction, token estimates, and context-engine seams, but it does not implement the requested agent-facing budget/status tool or preserve/exempt steering contract. Related open issues cover budget awareness and user-visible compaction, but exact `compaction_guide` agent steering is not fully owned elsewhere.

## What This Changes

The issue requests opt-in agent context-awareness and steering for compaction, including imminent warnings, preserve/exempt hints, a `compaction_guide` tool, and configurable thresholds.

## Best Possible Solution

Define a maintainer-approved compaction guidance contract that extends the existing context-engine, manual-compaction, and hook surfaces, with explicit estimated-vs-reported status semantics and best-effort hint guarantees before exposing any model-facing tool.
