## Summary

Keep open: current main and the latest release still ship the bundled summarize skill with only the deleted Homebrew formula, while the Control UI list toggle still does not make the missing-binary requirement visible at the point the user clicks it. The remaining fix needs a maintainer choice among a verified installer source, platform gating, de-bundling/removal, and UI/status feedback.

## What This Changes

The issue requests a valid or clearly platform-gated installer for the bundled summarize skill and visible feedback when enabling the skill fails due to a missing binary.

## Best Possible Solution

Make the bundled summarize entry truthful by using a verified maintained installer, explicit platform gating, or removal/de-bundling, with matching Skills UI/status/install error feedback and focused regression coverage.
