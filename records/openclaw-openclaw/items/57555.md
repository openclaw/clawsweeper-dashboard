Keep open: current main and v2026.6.8 still ship the bundled summarize skill with only the removed Homebrew formula, while the durable fix needs maintainer direction on installer source, platform gating, de-bundling, or removal.

## What This Changes

The issue asks OpenClaw to stop exposing the bundled summarize skill as a brew-only/deleted installer in Linux/Docker and to show clear feedback when enabling cannot make the skill usable.

## Best Possible Solution

Make the bundled summarize entry truthful by using a verified maintained installer, gating or marking unsupported platforms clearly, or removing/de-bundling it with matching Skills UI/status/docs coverage.

Yes at source level: current main and v2026.6.8 still require the summarize binary and list only the removed steipete/tap/summarize installer. I did not live-run the Docker Control UI toggle path in this read-only review.
