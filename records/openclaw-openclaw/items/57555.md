## Best Possible Solution

Land or replace https://github.com/openclaw/openclaw/pull/74408 for the installer-source failure after confirming the supported package source, then add requirement-aware Control UI enable feedback that names the missing binary and available install action.

## Solution Assessment

Unclear. Replacing the deleted brew formula is necessary, but the exact installer source or platform gating needs maintainer confirmation, and the UI feedback fix should be a narrow requirement-aware message instead of a competing broad install policy change.
