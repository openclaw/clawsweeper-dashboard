## Summary

Issue remains valid on current main: the bundled summarize skill still advertises a deleted Homebrew formula, and the current UI/install flow only partially improves feedback rather than resolving the broken install source. A closed unmerged related PR attempted a fix, so there is not a safe canonical merged replacement to close against.

## What This Changes

The issue asks OpenClaw to make the bundled summarize skill installable outside brew/macOS or clearly report/disable unsupported installs when enabling fails.

## Best Possible Solution

Maintain one honest bundled-skill path: either point summarize at a maintained cross-platform install source or stop advertising it as enableable where its required binary cannot be installed, with clear UI/install feedback.

## Reproduction Assessment

Yes, from source: current main still declares only the deleted Homebrew formula for the summarize binary, and the installer uses that formula directly. I did not run a live Homebrew install in this read-only review, so this is source-reproducible rather than live-reproduced.

## Solution Assessment

Unclear as an implementation choice: the issue identifies a real broken path, but maintainers need to choose whether bundled summarize should be removed, marked unsupported, or moved to a maintained installer source.
