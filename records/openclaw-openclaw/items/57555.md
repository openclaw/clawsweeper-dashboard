## Summary

Keep open: current main and v2026.6.6 still advertise the deleted `steipete/tap/summarize` Homebrew formula, and the UI/install feedback improvements do not resolve the broken installer source. The related fix PR is closed unmerged, so this issue is now the canonical remaining tracker.

## What This Changes

The issue reports that the bundled `summarize` skill depends on a brew-only/deleted installer and that the Control UI gives confusing feedback when enablement fails.

## Best Possible Solution

Choose one honest bundled-skill path: either use a verified maintained installer source for `summarize`, or stop presenting the bundled skill as installable where its required binary cannot be provided, with clear UI/install output for the chosen behavior.

## Reproduction Assessment

Yes. Current main and v2026.6.6 still declare only `steipete/tap/summarize`, and upstream removed that formula; I did not rerun a live brew install.

## Solution Assessment

Unclear. The issue identifies a real broken path, but the best fix depends on whether maintainers want to keep this bundled skill with a maintained installer, mark it unsupported, or remove/de-bundle it.
