  - body: `buildMcpAppCanvasJson` serializes view metadata together with raw `toolInput` and full `toolResult`, and that metadata already includes `sessionKey`.   - body: This path accepts MCP resource text or base64 blobs and immediately decodes/writes the HTML canvas document without raw or decoded byte limits. A configured MCP server can exhaust memory or disk when MCP Apps are enabled; add explicit caps before base64 decode and canvas persistence.
  - confidence: 0.9

  - confidence: 0.86
