## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Stop persisting MCP session data in canvas metadata:** `src/agents/mcp-ui-resource.ts:96-99`
  - body: `params.view.mcpApp` can include `sessionKey`, and this code also serializes raw `toolInput` and `toolResult` into the canvas JSON that is stored and rendered from chat history. Keep the session capability in live UI context and persist only a minimal redacted view descriptor so history, logs, exports, and previews do not carry bearer/session or user payload data.
  - confidence: 0.93
- **[P1] Bound MCP App resources before decoding or writing:** `src/agents/mcp-ui-resource.ts:49-53`
  - body: A configured MCP server can return arbitrarily large `text` or base64 `blob` content here, and the branch decodes it and writes the resulting HTML into a canvas document with no raw or decoded size cap. Add explicit limits before base64 decode and before canvas persistence so an enabled but buggy server cannot exhaust memory or disk.
  - confidence: 0.9
- **[P1] Enforce MCP Apps resource CSP metadata:** `ui/src/ui/components/mcp-app-view.ts:249-250`
  - body: The MCP Apps spec requires hosts to enforce declared `ui.csp` domains and restrictive defaults, but the branch discards the resource `_meta.ui.csp` and only sets an iframe CSP for navigation/forms. Carry the resource CSP through to the rendered app document or sandbox path and enforce the default `default-src 'none'` policy when metadata is absent.
  - confidence: 0.88
- **[P2] Avoid orphaned synthetic runtimes for restored views:** `src/gateway/server-methods/mcp-app-proxy.ts:31-37`
  - body: When a restored app view has no session-key mapping, this creates a runtime under `mcp-app:${sessionKey}`; later chat turns can create the real session runtime for the same key, leaving a duplicate MCP runtime and server processes alive until idle sweep. Reuse or bind the canonical session identity, or make restored app runtimes explicitly owned and retired when the real session resumes.
  - confidence: 0.82
