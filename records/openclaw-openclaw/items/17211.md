## Summary

Keep open. Current main still has compaction-scoped checkpoint snapshot branch/restore, not the requested native checkpoint engine with create/list/resume/diff APIs, deterministic replay packets, task-boundary contracts, diffing, or validation hooks; recent discussion also explicitly keeps this umbrella open.

Request a native task-boundary checkpoint engine with deterministic resume, replay packets, checkpoint diffing, and structured validation contracts.

## Best Possible Solution
