## Summary

Keep open: current main and v2026.5.22 still provide compaction-scoped snapshot list/get/branch/restore plus context-engine compaction hooks, not the requested native checkpoint create/list/resume/diff API, deterministic replay packet, task-boundary contract, diffing, or missing-field validation. ## What This Changes

The issue requests a native checkpoint engine with create/list/resume/diff APIs, deterministic replay packets, task-boundary contracts, and validation hooks to reduce session/context drift after compaction.

## Best Possible Solution

Design a versioned checkpoint/replay/diff contract across Gateway/session protocol, persistence, runtime/context-engine hooks, UI, docs, and tests while preserving the current compaction branch/restore recovery primitive.
