## Summary

Keep open. Current main only has compaction-scoped checkpoint inspection and snapshot branch/restore, not the requested native checkpoint engine with create/list/resume/diff APIs, deterministic replay packets, structured task contracts, diffing, or validation hooks; the live discussion and related searches also show adjacent work rather than a merged replacement.

Request to add native task-boundary checkpoint APIs plus deterministic resume and diff semantics for reliable long-running task continuation.

## Best Possible Solution
