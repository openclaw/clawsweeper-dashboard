## Summary

Keep open. Current main has compaction checkpoint snapshot branch/restore and context-engine compaction hooks, but it still lacks the requested native checkpoint.create/list/resume/diff APIs, deterministic replay packets, task-boundary contracts, diffing, and missing-field validation hooks; existing maintainer-facing comments also kept this umbrella open.

## What This Changes

Request adds first-class task-boundary checkpoint APIs, deterministic replay packets, diffing, and schema validation for reliable long-running task resume.

## Best Possible Solution

Keep this as the native checkpoint/replay umbrella and design a versioned Gateway/session/context-engine contract that builds on compaction checkpoint branch/restore without treating it as deterministic replay.
