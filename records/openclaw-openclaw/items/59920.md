## Summary

Keep open for maintainer review: current main still lacks the terminal-field preference, while this PR implements it with focused buffered and streaming coverage plus sufficient terminal proof. ## What This Changes

This PR updates CLI JSONL parsing and regression tests so `task_complete.last_agent_message` and `phase:"final_answer"` records win over commentary or generic fallback text in buffered and streaming parser paths.

## Best Possible Solution

Merge the focused parser fix after maintainers accept terminal JSONL fields as the final-reply contract, while leaving the broader inter-tool leakage work tracked separately at https://github.com/openclaw/openclaw/issues/25592.
