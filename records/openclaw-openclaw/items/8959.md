Request to add cross-platform multi-select support to the message tool so users can choose multiple options visually and submit one selected-values callback.

## Best Possible Solution

Add an additive portable batch-selection contract to the message presentation/message-tool surface, export and document it through the plugin SDK, and implement channel render/callback handling plus fallback behavior after maintainer API review.
