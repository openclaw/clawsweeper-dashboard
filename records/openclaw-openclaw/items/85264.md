- **[P1] Keep realpath checks tied to live filesystem state:** `src/infra/path-safety.ts:13-27`
  - body: `safeRealpathSync` now returns process-wide cached realpaths before hitting `@openclaw/fs-safe`.   - confidence: 0.88
- **[P1] Do not cache ancestor resolution across filesystem mutations:** `src/infra/boundary-path.ts:9-14`
  - body: `resolvePathViaExistingAncestorSync` caches the resolved output for a target path globally. Sandbox host path normalization depends on this helper to honor the deepest existing symlink parent when the final leaf is missing; if that ancestor or symlink target changes, future calls keep using the previous target until a test-only invalidator is called. Scope this cache to an owned operation/snapshot or add production invalidation and stale-ancestor coverage.
  - confidence: 0.84
