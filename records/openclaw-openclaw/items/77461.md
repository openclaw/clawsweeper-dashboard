Ensure the generated plugin runtime dependency sandbox includes `json5` so packaged `memory_search` can import shared redaction/config chunks without module-resolution failures.

Fix the packaged runtime-deps manifest or sandbox materialization so every external package imported by copied shared runtime chunks is installed, then add package-level coverage that would catch a missing `json5` import for `memory_search`.
