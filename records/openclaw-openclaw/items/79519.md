## Summary

Keep open: current main still lacks provider-scoped dynamic model visibility, and this PR is the active implementation tied to the open user report, but the proposed patch has one blocking mixed allowlist correctness issue.

## What This Changes

The PR adds literal `provider/*` entries to `agents.defaults.models` across shared model selection, Gateway `models.list`, `/models`, Discord picker startup, docs, changelog, generated config baseline hashes, and focused tests.

## Best Possible Solution

Tighten mixed wildcard/exact allowlists so only wildcard providers plus explicit model refs are exposed, then rerun the focused model-selection, Gateway, command, Discord picker, docs, and changed-lane checks before maintainer approval.

## Reproduction Assessment

Yes, by source inspection: current main treats `agents.defaults.models` entries as exact model refs and has no selected-provider dynamic catalog mode. I did not run live current-main Gateway reproduction because this review is read-only and tests would create artifacts.

## Solution Assessment

No as submitted: extending the existing model map with literal `provider/*` is a narrow direction, but the current patch still re-adds a default model outside mixed wildcard/exact policies. The safer fix is to preserve legacy default inclusion only for exact-only allowlists, or when the default is explicitly covered.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Do not re-add defaults outside wildcard providers:** `src/agents/model-selection-shared.ts:790-791`
  - body: When an exact ref is mixed with a provider wildcard, this condition still adds `defaultKey` even if the default provider is neither wildcarded nor exact-listed. For example, `{"openai-codex/*": {}, "vllm/qwen-local": {}}` with default `anthropic/claude` will still allow/select `anthropic/claude`, which leaks the default provider back into a provider-scoped policy. Preserve the legacy default add only for exact-only allowlists, or when the default is explicitly covered.
  - confidence: 0.87
