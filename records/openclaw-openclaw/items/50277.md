Keep open. The issue asks to keep the Control UI chat delete confirmation popover within the visible page boundary when deleting messages, with a separate request for fuller Chinese localization.

Yes. A narrow placement fix in the existing grouped chat render/CSS path is the maintainable approach; removing absolute positioning may avoid one case but risks changing message layout flow.
