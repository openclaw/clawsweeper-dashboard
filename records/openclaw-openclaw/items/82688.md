## What This Changes

The issue requests a launcher fix so packaged OpenClaw does not respawn on every run when Node reports a versioned compile-cache child under the desired cache base.

## Best Possible Solution

Keep this issue open and use the linked PR as the implementation path after revising the launcher guard and adding packaged-launcher regression proof.

## Reproduction Assessment

## Solution Assessment
