## Summary

## What This Changes

## Best Possible Solution

Review and land https://github.com/openclaw/openclaw/pull/90586 after resolving its reload-classification fallback concern, or replace it with an equivalent narrow fix that preserves transport-affecting restarts while making runtime-only Telegram config edits non-restarting.
