## Summary

Close as ClawHub/plugin-owned: the request is useful, but current main documents personal WeChat and Enterprise WeChat as external channel plugins and the SDK already exposes the generic group, ingress, mention-gating, and outbound seams needed by a channel plugin.

## What This Changes

The issue requests WeChat group chat support for the Weixin channel, including group message receive/reply, @mention or keyword triggering, reply context, and support for both Enterprise WeChat and personal WeChat groups.

## Best Possible Solution

Keep OpenClaw core channel-agnostic and have the external Weixin/WeCom plugin or ClawHub package implement group capability metadata, @mention parsing, allowlist/trigger policy, and outbound group delivery; open a narrow SDK issue only if that work exposes a missing core seam.
