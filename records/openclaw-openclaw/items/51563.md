## Summary

Keep open: current main still has the reported status mis-detection and this PR remains the canonical fix attempt, but the branch is not merge-ready because it has blocking correctness defects and only unit-test proof.

## What This Changes

The PR adds a path-heuristic global-install package-manager detector, uses it for package-install update status, and adds detector tests.

## Best Possible Solution

Repair this PR or replace it with a narrow fix that uses the existing global-install-manager contract for package installs, preserves git-install behavior, covers npm Homebrew, pnpm, and configured Bun roots, and includes real global-install proof.

## Reproduction Assessment

Yes. Source inspection of current main shows a non-git package root flows through `checkUpdateStatus()` to `detectPackageManager()`, then `openclaw update status` renders that value as `Install`; I did not run a live global install in this read-only review.

## Solution Assessment

No. The direction is right, but the current patch should import `node:path`, align with `detectGlobalInstallManagerForRoot()` or equivalent behavior for configured Bun roots, and be rebased without losing current symlinked-git handling.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.95

Full review comments:

- **[P1] Import node:path before using path.sep:** `src/infra/detect-package-manager.ts:42`
  - body: `detectGlobalInstallManager()` calls `path.sep`, but this module does not import `node:path`. Since `checkUpdateStatus()` calls this for package installs, `openclaw update status` would fail before it can report the install manager.
  - confidence: 0.96
- **[P2] Match configured Bun global roots:** `src/infra/detect-package-manager.ts:51-52`
  - body: The detector only returns `bun` when the package path contains `/.bun/`, then falls back to `npm`. Existing updater code resolves Bun's global root from `BUN_INSTALL`, so configured Bun installs outside `~/.bun` would still be reported as npm.
  - confidence: 0.9
