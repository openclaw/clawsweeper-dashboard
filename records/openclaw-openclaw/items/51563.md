## Summary

Keep open: current main still has the reported status mis-detection and the closed linked issue explicitly routes the remaining fix through this PR, but the PR is not merge-ready because it has blocking correctness findings and only unit-test proof.

## What This Changes

The branch adds a path-heuristic global-install package-manager detector, uses it for package-install update status, and adds detector tests.

## Best Possible Solution

Repair this PR or replace it with a narrow fix that uses the existing global-install-manager contract for package installs, preserves git-install behavior, covers npm Homebrew, pnpm, and configured Bun roots, and includes real global-install proof.

## Reproduction Assessment

Yes. Source inspection of current main shows a non-git package root flows through `checkUpdateStatus()` to `detectPackageManager()`, then `openclaw update status` renders that value as `Install`; I did not run a live global install in this read-only review.

## Solution Assessment

No. The direction is right, but the current patch should first import `node:path` and align with `detectGlobalInstallManagerForRoot()` or equivalent behavior so supported Bun and custom global-root layouts are not misreported.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.95

Full review comments:

- **[P1] Import `node:path` before using `path.sep`:** `src/infra/detect-package-manager.ts:42`
  - body: `detectGlobalInstallManager()` calls `path.sep`, but this module does not import `node:path`. Since `checkUpdateStatus()` calls this for package installs, `openclaw update status` would throw before it can report the install manager.
  - confidence: 0.96
- **[P2] Match configured Bun global roots:** `src/infra/detect-package-manager.ts:51-52`
  - body: The detector only returns `bun` when the package path contains `/.bun/`, then falls back to `npm`. Existing updater code resolves Bun's global root, so a configured `BUN_INSTALL` under another directory would still be reported as npm.
  - confidence: 0.9
