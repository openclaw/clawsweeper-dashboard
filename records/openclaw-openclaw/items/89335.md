## What This Changes

The issue asks OpenClaw to prevent or recover from direct package-manager self-updates that leave core, managed plugins, and Codex route metadata version-skewed.

## Best Possible Solution

Keep this open for a maintainer-owned update UX decision: route agent self-updates through `update.run` or `openclaw update`, surface finalization after manual package-manager upgrades, and add a clear startup/readiness diagnosis for active managed plugin or Codex route skew.
