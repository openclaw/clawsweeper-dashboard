The PR changes sandbox skill-sync fallback resolution to use the existing agent workspace resolver and adds regression coverage for omitted and blank sandbox workspaceDir inputs.

## Best Possible Solution

## Solution Assessment

Yes. Using the existing agent workspace resolver at the sandbox fallback point is the narrowest maintainable fix; passing explicit workspaces everywhere would leave the exported helper inconsistent with configured-workspace behavior.
