## Best Possible Solution

Keep this issue open until `message` tool schemas are byte-stable across delivery modes, OpenAI/Codex prompt assembly keeps volatile runtime guidance out of the reusable request prefix, and focused cache-stability plus live cache proof cover the change.

## Solution Assessment
