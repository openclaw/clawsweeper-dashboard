## Summary

## Best Possible Solution

Preserve streamed assistant text only when it is final assistant output, keep pre-tool commentary and tool-only empty turns under the existing empty-response or silent policy, and add a regression for the live-session boundary.

## Reproduction Assessment

## Solution Assessment

No. The parser-layer idea is plausible, but the current complete-parser fallback can treat Claude pre-tool commentary as final assistant text in live sessions; the safer fix preserves the streaming commentary boundary or carries equivalent boundary state.
