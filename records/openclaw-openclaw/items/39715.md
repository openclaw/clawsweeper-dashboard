- **[P2] Keep completion state until session publish:** `src/process/supervisor/supervisor.ts:228-233`
  - body: The new pre-yield `registry.finalize()` makes `supervisor.getRecord()` report `exited` while the background session is still in `runningSessions` and `managedRun.wait()` has not resolved. Keep the run in a completing state or make process actions treat it as in-flight until final publication finishes.
