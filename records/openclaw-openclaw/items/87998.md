## Best Possible Solution

Keep the minimal internal-routing approach, but make marker-free hook `systemPrompt` overrides get a stable cache boundary in both media-active and media-idle turns before merge.

## Reproduction Assessment

## Solution Assessment

No, not as submitted. Moving media hints below the cache boundary is the right fix shape, but marker-free `systemPrompt` overrides need the boundary on idle turns too so active and idle cacheable prefixes remain stable.
