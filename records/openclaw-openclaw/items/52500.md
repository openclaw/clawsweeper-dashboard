## What This Changes

Request that config validation errors for schema-enforced numeric ceilings or floors state the applicable maximum or minimum value.

## Best Possible Solution

Add a central config validation mapper enhancement that appends accurate numeric-origin max/min hints, handles inclusive and exclusive bounds, preserves allowed-values behavior, and adds focused regression coverage.

## Reproduction Assessment

## Solution Assessment

Yes. The central mapper is the right surface because it fixes schema-generated config diagnostics once while avoiding per-field custom messages.
