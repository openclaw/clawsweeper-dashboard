## What This Changes

The issue reports that keeping the macOS WebChat chat window visible on an Intel Retina iMac drives WindowServer CPU to roughly 330-380%, while hiding OpenClaw drops WindowServer CPU sharply.

## Best Possible Solution

Profile current main on affected Intel Retina hardware, isolate the expensive compositor feature, and ship the narrowest measured low-compositor fallback while preserving normal macOS visuals elsewhere.
