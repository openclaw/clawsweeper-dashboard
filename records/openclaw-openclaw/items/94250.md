The PR adds a Feishu reply-dispatcher branch that sends block payloads through `sendMessageFeishu` when `blockStreaming` is enabled and no streaming card is active, plus dispatcher tests for independent block sends and duplicate-final suppression.

## Best Possible Solution

## Solution Assessment
