## What This Changes

Request adding an Anthropic-native `web_search_20260209` / `web_search_20250305` provider option that passes Anthropic server tools through the Messages API instead of using managed external search providers.

## Best Possible Solution

## Solution Assessment

No, not as a bare `tools.web.search.provider: "anthropic"` addition. The best solution needs a provider-owned opt-in contract covering config, auth, hosted web policy, request injection, result preservation, docs, and upgrade behavior.
