## Best Possible Solution

Fix the manual finalization boundary so manual runs preserve the scheduled at-slot through the final recompute, then prove the full `cron run` to `cron show/list` path for a successful `deleteAfterRun` due job while keeping scheduled-run deletion behavior unchanged.
