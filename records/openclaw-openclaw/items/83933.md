## Summary

## Best Possible Solution

## Solution Assessment

Yes, pending maintainer acceptance. Threading the manual discriminator through the shared result helper and both manual callers is the narrowest maintainable fix; the alternative reject/override flag would add more CLI surface and still leave scheduled-state side effects to police elsewhere.
