## Summary

## What This Changes

The PR adds an `isManual` result-application path for cron manual runs, records `lastRunWasManual`, updates the gateway cron state schema, and expands cron regression coverage around manual-run isolation.

## Best Possible Solution

Merge the focused cron fix only after current-head Gateway CLI proof demonstrates manual `deleteAfterRun` preservation and maintainers accept the additive state field and manual-run semantics.
