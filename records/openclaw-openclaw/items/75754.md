## What This Changes

The PR updates `scripts/clawdock/clawdock-helpers.sh` so `clawdock-dashboard` maps loopback dashboard URLs from the Docker CLI container to the published Compose gateway port, appends and redacts token fragments, and avoids fragile URL extraction dependencies on the dashboard path.

## Best Possible Solution

Keep the fix in the ClawDock shell helper, because the mismatch is caused by the helper launching a host browser from a URL produced inside the Compose network namespace. Before merge, add the required changelog entry and let the normal PR checks or a targeted Docker smoke validate the exact head.

## Solution Assessment
