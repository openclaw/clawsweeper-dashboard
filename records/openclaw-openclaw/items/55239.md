Keep open. Current main has Telegram runtime support for plugin-provided localized command descriptions, but it still does not provide the config-backed Chinese descriptions for built-in Telegram command menus requested here.

## What This Changes

Request support for configuring localized Telegram bot command-menu descriptions, especially Chinese labels for commands such as `/help` and `/tts`.

## Best Possible Solution

Define one additive locale-map contract that keeps command names stable, feeds Telegram `setMyCommands` `language_code`, and coordinates built-in command metadata with the broader slash-command i18n tracker.

## Reproduction Assessment

Yes for the source-level feature gap: current main still lacks a locale-keyed Telegram command-description config path, and built-in command declarations remain English-only. A live Telegram run is not needed to prove the missing config surface.

## Solution Assessment
