## Summary

Keep open: the feature is not on current main, but this PR still needs changes because the skip trusts generic PATH-based version resolution, exits before shared installer finalization, and lacks after-fix real behavior proof.

## What This Changes

This PR adds an `install.sh` version check that skips npm install when the detected local version matches the npm target, plus `--force` and `OPENCLAW_FORCE_INSTALL` overrides.

## Best Possible Solution

Implement the optimization only after verifying npm-managed provenance, skip only the npm install step, preserve finalization and `--verify`, add focused installer coverage, and require redacted real install proof before merge.

## Reproduction Assessment

Yes for the PR-head blockers: the diff shows the skip uses generic version resolution, and a MEMBER review reproduced a fake-PATH false skip. This is not a current-main bug reproduction because main has no skip path.

## Solution Assessment

No. The feature should verify the npm-managed package or npm-global binary before skipping, then continue the normal finalization path instead of returning early.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P2] Verify the npm-managed install before skipping:** `scripts/install.sh:1990`
  - body: The skip compares the target version with `resolve_openclaw_version()`, which can return any `openclaw` earlier on PATH before checking npm's global package tree. A wrapper or different install with the same semver can suppress the intended npm install, so base the skip on npm package metadata or a verified npm-global binary.
  - confidence: 0.93
- **[P2] Keep finalization after skipped installs:** `scripts/install.sh:2340-2342`
  - body: Returning from `main` when `INSTALL_SKIPPED=1` bypasses the shared finalization path. The npm install step can no-op, but the installer should still run duplicate-install diagnostics, gateway refresh/restart handling, doctor/onboarding follow-up, and explicit `--verify` behavior.
  - confidence: 0.88
