## Summary

Keep open: the optimization is useful, but this branch is not mergeable yet because the skip can trust a PATH-shadowed binary, exits before shared installer finalization, and still lacks after-fix real installer proof.

## What This Changes

The PR adds an `install.sh` npm-version skip with `--force` and `OPENCLAW_FORCE_INSTALL` overrides when local and target versions match.

## Best Possible Solution

Implement the optimization only after verifying npm-managed provenance, skip only the npm install step, preserve shared finalization, add focused installer coverage, and require redacted real install proof before merge.

## Reproduction Assessment

Yes for the PR-head blockers: the diff shows the skip trusting generic version resolution, and a MEMBER review reports a fake-PATH reproduction. This is not a current-main bug reproduction because main does not have the skip path.

## Solution Assessment

No. The skip should be based on npm package metadata or a verified npm-global binary and should continue through shared finalization instead of returning from `main`.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P2] Verify npm provenance before skipping:** `scripts/install.sh:1990`
  - body: The skip compares the target version with `resolve_openclaw_version()`, which can return any `openclaw` earlier on `PATH` before checking npm's global package tree. A wrapper or different install with the same semver can suppress the intended npm install, so base the skip on npm package metadata or a verified npm-global binary.
  - confidence: 0.93
- **[P2] Keep finalization after skipped installs:** `scripts/install.sh:2340-2342`
  - body: Returning from `main` when `INSTALL_SKIPPED=1` bypasses the shared finalization path. The npm install step can no-op, but the installer should still run duplicate-install diagnostics, gateway refresh/restart handling, doctor/onboarding follow-up, and explicit `--verify` behavior.
  - confidence: 0.88

## Security Review
