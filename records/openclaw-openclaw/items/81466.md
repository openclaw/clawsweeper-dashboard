## Summary

Keep open: the branch still blocks a valid current OpenClaw route by treating canonical `openai/*` agent model refs as direct API routes under a legacy `openai-codex/*` subagent default, even though current policy and docs route official OpenAI agent refs through the Codex harness by default.

## What This Changes

The branch adds a forbidden subagent model-plan state that rejects `gpt` and `openai/*` overrides under an `openai-codex/*` subagent default and adds regression tests for those paths.

## Best Possible Solution

Keep a narrow guard for actual direct/custom OpenAI runtime selections while preserving `openai/*` as the canonical Codex-backed agent route, with regression coverage for both allowed and rejected cases.
