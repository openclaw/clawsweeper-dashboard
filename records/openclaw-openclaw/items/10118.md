## Summary

Keep open because current main only partially satisfies the request: the pinned TUI editor provides default Shift+Enter newline behavior, but OpenClaw still lacks user-facing docs/help for multiline composition and has no public keybinding preference surface for the configurable behavior requested in discussion.
