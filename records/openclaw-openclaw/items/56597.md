## Summary

## Best Possible Solution

Maintainers should decide the intended lifetime and precedence of session-level exec overrides, then align source, docs, migration/reset behavior, and regression coverage so operator global exec policy changes are not silently shadowed unless that durable override contract is explicitly kept.

## Reproduction Assessment

## Solution Assessment

Unclear. The likely code fixes are mechanically bounded, but choosing non-persistence, different precedence, reset/migration behavior, or a deliberate durable override contract is a security and compatibility decision.
