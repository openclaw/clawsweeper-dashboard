## What This Changes

The branch adds a Control UI document-title helper that uses the active session display name, synchronizes it on connection and session metadata changes, and adds focused resolver and lifecycle tests.

## Best Possible Solution

Adopt one Control UI-owned title resolver with explicit precedence—prefer an operator-configured static title when present, otherwise use the active session display name if maintainers accept its browser-chrome visibility—and prove session switching, metadata refresh, and static fallback in a real browser.
