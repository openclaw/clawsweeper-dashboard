## Summary

## What This Changes

The PR adds a Control UI document-title resolver, lifecycle title sync, an active-session title-row cache, and focused UI tests so browser tabs can show safe active session labels while falling back to `OpenClaw Control`.

## Best Possible Solution

Land one maintainer-approved Control UI title policy that keeps this session-name source if desired, defines precedence with static configured titles and active-agent titles, and documents the privacy expectation for title-visible metadata.
