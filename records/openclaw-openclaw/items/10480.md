## Best Possible Solution

Add a provider-owned Workers AI endpoint mode or model selector in `extensions/cloudflare-ai-gateway`, with docs and focused tests for the generated OpenAI-compatible `/workers-ai/v1` provider config; if maintainers prefer the existing custom-provider path, document the exact supported command before closing.
