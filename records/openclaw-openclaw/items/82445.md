## What This Changes

The PR suppresses truthy native Ollama `think` values for models marked `reasoning: false` and adds regression tests for configured and runtime thinking paths.

## Best Possible Solution

## Solution Assessment

Yes, the PR is the narrow maintainable fix because it preserves upstream-compatible top-level `think` for reasoning-capable or unknown models, preserves explicit `think:false`, and only suppresses the known-invalid truthy case.
