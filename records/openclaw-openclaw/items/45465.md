## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.92

Full review comments:

- **[P1] Normalize per-job hook paths before validation:** `src/cron/hooks.ts:201-212`
  - body: `isValidJobHookPath` checks the raw persisted value, but hook loading later resolves it through `resolveUserPath`, which trims first. A value like `" /etc/passwd"` or spaced traversal can pass this guard and then resolve outside the intended relative per-job scope.
  - confidence: 0.94
- **[P2] Validate per-job hook entry types before path checks:** `src/cron/hooks.ts:67-71`
  - body: `job.hooks` comes from persisted job data, but this loop assumes the selected hook point is a string array. Malformed values such as `[123]` can throw inside the path helper before normal hook isolation and cron finalization run.
  - confidence: 0.88
- **[P2] Reuse stable hook import specifiers:** `src/cron/hooks.ts:192`
  - body: Passing `cacheBust: true` appends a fresh timestamp query to every file import. Node keeps each ESM specifier cached, so a long-lived cron hook creates unbounded module instances across repeated runs instead of reusing one stable hook module.
  - confidence: 0.86

## Security Review

Status: needs_attention

Summary: The diff introduces unattended cron-triggered local module execution and still has a concrete per-job path restriction bypass.

Concerns:

- **[high] Per-job hook path restrictions can be bypassed:** `src/cron/hooks.ts:201`
  - body: The guard checks untrimmed script paths, while the loader trims and resolves them later. That lets whitespace-prefixed absolute or traversal paths escape the intended relative per-job hook boundary.
  - confidence: 0.94
- **[medium] Define the scheduled host-execution boundary:** `src/cron/hooks.ts:181`
  - body: Global and per-job hooks run local modules outside an interactive turn. Before merge, maintainers should settle cwd/base path, allowed specifiers, environment and secret exposure, reload behavior, timeout/cancellation, and logging policy.
  - confidence: 0.86
