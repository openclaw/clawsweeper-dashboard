Full review comments:

- **[P1] Normalize per-job hook paths before validation:** `src/cron/hooks.ts:201-212`
  - body: Validate the same normalized string that `loadHookModule` imports. As written, values such as `" /absolute/hook.cjs"` or `" ../outside.cjs"` can pass this raw-string guard and then be trimmed into an escaped path by `resolveUserPath`, defeating the intended relative-only boundary.
