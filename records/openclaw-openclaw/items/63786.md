## Best Possible Solution

Refresh the branch to carry an explicit normalized Copilot replay-sanitizer strategy through history replay, tool-continuation replay, and pruning estimates; preserve all-thinking turns with the existing non-empty omitted-reasoning placeholder; remove or split the guard-script allowlist changes; and add the required changelog entry.
