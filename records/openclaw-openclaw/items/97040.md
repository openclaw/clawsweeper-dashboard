## Best Possible Solution

- **[P1] Gate the Herdr sidecar through plugin enablement:** `src/tui/tui.ts:526-531`
  - body: This PR adds `herdr-state` as a bundled plugin, but `runTui()` starts the sidecar from `HERDR_PANE_ID` before config is loaded and never checks the plugin activation decision.
