Keep open. Current main still lacks the requested WhatsApp first-use guidance, and the related closing PR #75082 is open rather than merged, so this issue should remain open until that PR or an equivalent patch lands.

## What This Changes

The issue requests WhatsApp onboarding and docs to show how to send or receive a first hello-world message after setup.

## Best Possible Solution

Land #75082 or an equivalent narrow change that adds manual WhatsApp first-message guidance to the channel docs and setup completion flow while keeping outbound sends user-initiated.

## Reproduction Assessment

Yes. Source and docs inspection against current main reproduces the gap: the WhatsApp docs stop after setup/pairing, and the setup finalizer returns after DM access prompts without a successful post-link hello-world or next-step note.

## Solution Assessment

Yes for the current paired implementation direction. Manual next-step guidance in docs and onboarding is the narrow maintainable solution; automatic WhatsApp sends would be broader and could surprise users or select the wrong recipient.
