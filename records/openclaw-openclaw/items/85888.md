## What This Changes

Investigate scheduled cron runs failing with MiniMax 503 overload while later manual runs of the same jobs succeed.

## Best Possible Solution

Cron should honor documented recurring retry backoff for transient provider failures, retrying the failed run in the configured backoff window while returning to the normal cron schedule after success or exhausted retry policy.
