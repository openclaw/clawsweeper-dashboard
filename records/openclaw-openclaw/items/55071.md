## What This Changes

Add a SKILL.md `metadata.openclaw.activates.plugins` contract so matched skills can activate plugin tools and bundled skills per turn instead of exposing all plugin tools globally.

## Best Possible Solution

Define an additive public skill activation contract that threads parsed metadata through skill snapshots into request-time plugin tool planning, with docs and tests for allow/deny precedence, optional tools, plugin disablement, prompt-cache stability, and bundled plugin skills.

## Solution Assessment
