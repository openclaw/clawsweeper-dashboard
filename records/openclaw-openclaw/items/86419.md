- **[P1] Keep number and boolean helpers out of this string subpath:** `src/plugin-sdk/string-coerce-runtime.ts:16-21`
  - body: This public Plugin SDK subpath is documented as narrow record/string coercion, but the PR now exports number and boolean helpers from it and updates the API baseline.   - confidence: 0.86

  - body: The imported `asRecord` accepts arrays, unlike the removed local helper. `resolveProvidersConfig()` then iterates `Object.entries(raw)`, so a malformed array under provider config can create numeric provider IDs instead of being ignored; use an array-rejecting record helper here.
  - confidence: 0.9

  - body: `sortUniqueStrings()` uses default `localeCompare`, while several replaced call sites used plain `.toSorted()` or deterministic ID ordering. Default collation can vary by host locale/ICU data, so shared normalized plugin IDs, config keys, and cache inputs can drift across machines; use code-unit comparison or an explicit fixed collation.
  - confidence: 0.82
