## What This Changes

The PR routes concurrent Codex app-server run attempts to per-run isolated clients when `agents.defaults.maxConcurrent > 1`, closes those clients on cleanup/error paths, and adds regression coverage plus a changelog entry.

## Best Possible Solution

## Reproduction Assessment

## Solution Assessment
