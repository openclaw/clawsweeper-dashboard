- **[P2] Normalize names before checking duplicates:** `src/cron/service/ops.ts:322`
  - body: `createJob` and `applyJobPatch` trim names via `normalizeRequiredName`, but this lookup compares the raw incoming value. Normalize the candidate name before calling `findJobByName` and cover whitespace add/edit cases.
  - confidence: 0.92

Use an error type/contract the handler maps to `INVALID_REQUEST`, or handle the collision in the Gateway path with tests.
