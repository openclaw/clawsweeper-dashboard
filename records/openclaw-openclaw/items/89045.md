The PR recovers visible inbound turns from failed/timeout/killed terminal sessions by clearing stale lifecycle fields, clearing same-session stale reply operations with concurrency guards, mirroring the behavior in the Gateway agent path, and adding zero-count dispatch diagnostics and tests.
