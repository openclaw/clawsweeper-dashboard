## Summary

## What This Changes

The branch changes the Codex plugin app-server resolver, tests, docs, and changelog so implicit local stdio defaults read Codex requirements and choose allowed guardian approval, reviewer, and sandbox values instead of always using YOLO.

## Best Possible Solution

Make the requirements-policy resolver apply upstream-compatible hostname-scoped remote sandbox rules before selecting implicit defaults, then land one canonical branch for this Codex requirements behavior.
