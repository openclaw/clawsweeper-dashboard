## Summary

Keep this open. Current main still drops trusted turn-source routing metadata before `before_tool_call` plugin approvals call `plugin.approval.request`, so chat-originated approvals can still hit the reported `Plugin approval unavailable (no approval route)` path. The reporter's correction is important: the fix should propagate trusted runtime context, not model/tool params, and the linked repair PR #74150 is closed and unmerged.

## What This Changes

The issue reports that `before_tool_call` plugin approval requests do not forward originating channel route metadata to `plugin.approval.request`, causing no-route approval failures on Telegram and other chat channels.

## Best Possible Solution

Extend the internal hook/tool wrapper context with trusted `turnSourceChannel`, `turnSourceTo`, `turnSourceAccountId`, and `turnSourceThreadId` from the agent runtime, forward those fields into `plugin.approval.request` for both trusted-policy and ordinary `before_tool_call` approval paths, and add regression coverage proving routed requests are accepted while missing routes still fail closed.

## Reproduction Assessment

Yes. A high-confidence source-level path exists: a channel-originated run passes route data into `createOpenClawCodingTools`, the hook wrapper drops it, `requestPluginToolApproval` sends no `turnSource*` fields, and the gateway returns `decision: null` when no other approval route exists.

## Solution Assessment

Yes, with the corrected trust boundary. The narrow maintainable fix is to propagate trusted runtime route context through `HookContext` into `plugin.approval.request`; reading route fields from tool params or hook-returned params would be less safe because those values are model/plugin controlled.

Fix the plugin approval route propagation bug reported in https://github.com/openclaw/openclaw/issues/74003. Current `before_tool_call` approvals call `plugin.approval.request` without trusted turn-source route metadata, so channel-originated runs can fail with `Plugin approval unavailable (no approval route)`.
