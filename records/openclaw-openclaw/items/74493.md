## Summary

Keep open. The identity-only token hashing and version bump address the central reported reset path, but the new Claude null-read fallback conflates true credential absence with a valid identity-less OAuth read, so a same-version reusable session can survive logout or local credential removal.

## What This Changes

The PR bumps CLI auth-epoch version to 5, changes Claude CLI and token auth-profile epoch hashing to identity-only, adds a Claude null-read fallback, expands auth-epoch regression tests, and updates the changelog.

## Best Possible Solution

Keep the identity-only token hashing and epoch version bump, but leave true Claude credential absence as `undefined` or distinguish transient parse failures from logout/removal, with regression coverage for both paths.

## Reproduction Assessment

Yes from source inspection. Current main can parse partial Claude credentials as token-shaped, token encoders include rotating token material, and same-version auth-epoch mismatches reset reusable CLI sessions; I did not run the live macOS keychain race.

## Solution Assessment

No, not yet. The identity-only encoder change is the narrow maintainable fix, but the synthetic null fallback should not make true missing Claude credentials indistinguishable from a valid identity-less local credential.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.82

Full review comments:

- **[P2] Preserve invalidation for missing Claude credentials:** `src/agents/cli-auth-epoch.ts:167-173`
  - body: This fallback turns any null Claude credential read into the same provider-only fingerprint as a valid identity-less OAuth read. Because `readClaudeCliCredentialsCached` also returns null for true absence after logout, credential removal, or no local keychain/file credential, a version-5 binding can keep reusing the old session instead of getting the current `auth-epoch` reset. Please keep true absence as `undefined` or distinguish it from transient parse failure, and cover that behavior.
  - confidence: 0.78
