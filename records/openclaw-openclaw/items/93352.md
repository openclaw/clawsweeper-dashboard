## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

No. The static-profile lazy import is plausible, but importing every coerced legacy profile crosses doctor-owned OAuth sidecar repair; the safer fix is to filter unsafe sidecar-backed profiles or move that recovery into doctor.
