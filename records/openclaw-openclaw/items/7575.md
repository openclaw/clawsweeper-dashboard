Request host-aware Sysbox Docker runtime support so OpenClaw can run sandbox containers without mounting the host Docker socket.

## Best Possible Solution

Keep this as the canonical sandbox runtime architecture tracker until owners choose either a constrained core Docker runtime selector or a plugin-backed Sysbox backend with host prerequisites, threat model, config semantics, docs, and tests.
