Keep open. Current main still lacks Sysbox/sysbox-runc support or a Docker runtime selector, and the issue discussion explicitly keeps this as the remaining sandbox runtime architecture and security decision rather than completed by Docker-socket sandboxing, sandbox-common, or OpenShell backend work.
