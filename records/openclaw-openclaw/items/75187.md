The issue asks OpenClaw to preserve load-bearing `AGENTS.md` safety and tool-use guidance when `agents.defaults.bootstrapMaxChars` trims generated workspace bootstrap content.

## Best Possible Solution

Review and land PR #75248, or an equivalent narrow template-order fix, with regression coverage proving `Red Lines`, `External vs Internal`, and `Tools` survive a 1500-character `AGENTS.md` bootstrap budget; broader tiered or priority-aware bootstrap loading can remain separate follow-up work.

## Solution Assessment
