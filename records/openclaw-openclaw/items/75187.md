## Best Possible Solution

Review and land PR #75248, or an equivalent narrow template-order fix, with regression coverage proving `Red Lines`, `External vs Internal`, and `Tools` survive a 1500-character `AGENTS.md` bootstrap budget; broader tiered or priority-aware loading can stay as separate follow-up work.
