## What This Changes

The issue requests a first-class, channel-aware rendering convention for shell/bash command blocks in chat outputs so operators can copy commands safely.

## Best Possible Solution

Define a portable command-block presentation or Markdown metadata contract, then implement channel-specific renderers and fixtures that prove exact copy text plus safe plain-text fallback behavior.

## Solution Assessment
