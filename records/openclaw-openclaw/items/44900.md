## What This Changes

The branch adds a `skipDmThreadSuffix` flag to the old Telegram route result and uses it in old inbound, native-command, and handler code with regression tests.

## Best Possible Solution

Port the bound-route authority signal into `extensions/telegram/src`, guard inbound, native-command, and session-state DM-topic suffixing there, add active-tree regression coverage, and attach redacted live Telegram proof.
