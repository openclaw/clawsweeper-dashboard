  - body: The stats/list collection path opens an existing sidecar with openSidecarDatabase, which runs ensureSidecarSchema and writes meta/schema updates.   - confidence: 0.89

  - body: Invalid sidecar write inputs only log a warning and return, and unresolved or ambiguous ref prefixes are rendered without setting a non-zero exit code.   - confidence: 0.9
- **[P2] Touch only results that were shown:** `extensions/memory-core/src/memory-v2/rerank/wrapper.ts:90-100`
