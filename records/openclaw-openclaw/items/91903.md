- **[P2] Use a native value for hook chatId:** `src/agents/embedded-agent-runner/run/attempt.ts:3257`
  - body: The new docs define `ctx.chatId` and `ctx.channel.chat.id` as transport-native conversation IDs, but the embedded runner fills them from `params.currentChannelId`.   - confidence: 0.88

Either gate these fields to user-originated runs or document and test system-run chat context explicitly.
