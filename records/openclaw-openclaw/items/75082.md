- **[P2] Keep Codex location fields optional:** `src/config/zod-schema.agent-runtime.ts:286-295`
  - body: This changes `TrimmedOptionalConfigStringSchema` from the current top-level `.optional()` string transform back to `z.preprocess(..., z.string().optional())`. Keep current main's helper or cover it with a behavioral parse test.
