Keep open. Requested a WebChat mode that groups multiple tool executions into one collapsible summary with an optional detailed-versus-summary preference.

## Best Possible Solution

Design a turn-level Control UI tool-call summary disclosure that groups related live and history tool calls while preserving the existing show/hide toggle and per-message expansion state, then decide the default and preference semantics before implementation.
