Request stable `openai-responses` request normalization/configuration for stricter Responses-compatible proxy endpoints.

Add opt-in model-compat and payload-policy controls for fallback `instructions` and `max_output_tokens` omission, with schema, catalog normalization, docs, and tests while preserving native defaults.
