- **[P1] Validate preHook.file before execFile:** `src/cron/pre-hook.ts:66-70`
  - body: The docs and tool copy describe `preHook.file` as an absolute executable path, but the schemas/normalizer accept any non-empty string and this call passes it straight to `execFile`.
