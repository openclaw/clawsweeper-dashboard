- **[P1] Restore the legacy OpenAI Codex capability aliases:** `extensions/openai/openclaw.plugin.json:287-289`
  - body: Removing `openai-codex` from the OpenAI capability contracts and metadata leaves explicit legacy refs such as `openai-codex/gpt-image-1`, `openai-codex/sora-2`, or `tools.media.*` provider entries with no registered provider alias.
