## Summary

This PR is still useful because current main does not gate Feishu Bitable tools, but it should not merge yet: the new global any-account registration still leaves a disabled selected account able to execute Bitable tools, and the contributor proof does not show the patched disabled-config runtime path.

## Best Possible Solution
