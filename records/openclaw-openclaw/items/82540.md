- **[P2] Clear stale handoff accounts when plugin reload disables a channel:** `src/gateway/server-channels.ts:919`
  - body: When a plugin reload disables or removes a channel, `reloadAttachedGatewayPlugins` returns it outside `activeChannels`, so the restart loop skips `startChannel`.
