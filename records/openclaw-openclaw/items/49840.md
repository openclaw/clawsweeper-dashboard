  - confidence: 0.93

  - body: When no explicit heartbeat model is set, the estimate uses a local default-model helper, but the real run can still apply stored session, channel, or runtime model overrides later. That mismatch can under-price an expensive run or over-skip a cheap one.
  - confidence: 0.86
