## What This Changes

The issue asks Matrix per-room DM handling to discover and route newer strict DM rooms even when older `m.direct` mappings exist, and to make direct repair less fragile around stale mappings and startup readiness.

## Best Possible Solution

Land one Matrix plugin fix that keeps joined strict rooms discoverable under stale `m.direct` mappings and preserves the existing direct-room safety gates, then close this issue through the merged PR.
