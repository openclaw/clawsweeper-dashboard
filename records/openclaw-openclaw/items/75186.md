## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Classify missing path installs as invalid requests:** `src/plugins/management-core.ts:75-78`
  - body: When `plugins.install` receives `source: "path"` for a missing Gateway-host path, the underlying path install returns uncoded `directory not found` or `file not found` failures.
