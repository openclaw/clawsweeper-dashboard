## What This Changes

This PR makes `agents.defaults.skipBootstrap` skip existing workspace bootstrap files during runtime bootstrap resolution while preserving `agent:bootstrap` hook-injected files, and updates docs, JSDoc, and resolver tests.

## Best Possible Solution

Align resolver behavior, docs/JSDoc, generated schema help/labels, and redacted prompt-report proof, then land only if maintainers accept runtime suppression as the intended `skipBootstrap` contract.
