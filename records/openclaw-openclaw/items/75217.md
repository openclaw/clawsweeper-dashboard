The PR adds a runtime skipBootstrap guard to bootstrap-file resolution, documents the new runtime-injection scope, and adds regression tests for existing workspace bootstrap files.

Keep the shared resolver as the fix boundary, but have skipBootstrap suppress only default workspace-file loading while preserving explicitly configured bootstrap hooks, schema/help/generated docs, regression tests, and changelog coverage.
