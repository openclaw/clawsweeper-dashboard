## Summary

## What This Changes

The PR makes `agents.defaults.skipBootstrap` skip workspace bootstrap-file loading in `resolveBootstrapFilesForRun` while still running `agent:bootstrap` hooks, and updates docs/JSDoc plus resolver tests.

## Best Possible Solution

Land the resolver-level skip with hook preservation after syncing config schema/help metadata and adding real `openclaw agent --json` proof that `systemPromptReport.injectedWorkspaceFiles` drops workspace files under `skipBootstrap: true`.

## Solution Assessment
