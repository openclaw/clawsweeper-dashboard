## Best Possible Solution

Repair the PR as a warning-only diagnostic: warn only for recognized bootstrap filenames present in agentDir when the corresponding effective workspace file is absent, preserve the existing workspace-only loading contract, update tests to account for missing placeholders, and add a single-line user-facing changelog entry.
