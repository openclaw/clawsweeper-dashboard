## What This Changes

The issue reports that long TUI assistant messages disappear because the bundled pi-tui renderer can clear terminal scrollback during a content-driven full redraw.

## Best Possible Solution

Keep this issue open until a maintainer-approved repair prevents content-driven redraws from erasing terminal scrollback and proves that behavior in a PTY or real terminal.
