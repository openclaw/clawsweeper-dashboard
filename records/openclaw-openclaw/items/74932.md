## Best Possible Solution

Persist durable auto-fallback origin metadata, or an equivalent changed-default marker, when the auto override is written so heartbeat can clear only stale direct auto overrides after the configured default changes.

## Solution Assessment

No. The PR changes the right surface, but relying on fallback notice fields is not durable because current runtime clears and persists those fields while leaving the auto override in place.
