## Summary

Keep open. Add a Control UI pinned-context panel for editing user-defined key-value facts that are always injected into the agent context.

## Best Possible Solution

Define a versioned pinned-facts storage and injection contract, then implement scoped Gateway RPCs and a Control UI surface aligned with the memory-panel and contribution-slot decisions.
