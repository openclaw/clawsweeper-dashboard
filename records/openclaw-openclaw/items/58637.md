The branch adds a Model column to the Control UI Sessions table and changes `buildGatewaySessionRow` to prefer transcript model metadata when stored runtime and override model fields are absent.

## Best Possible Solution

Rebase onto current main, add an exact localized Model column, and load transcript model metadata only when model fallback is needed while preserving bounded list behavior, with focused regression coverage and real dashboard or gateway proof.

## Solution Assessment

No. The direction is useful, but the patch should decouple transcript model recovery from usage fallback, preserve exact model IDs, and rebase onto current localized Sessions UI conventions.
