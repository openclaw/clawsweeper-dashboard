## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Constrain cross-session continuation fanout:** `src/agents/tools/continue-delegate-tool.ts:49-67`
  - body: `continue_delegate` exposes explicit target session keys and `fanoutMode: "all"` directly to the model. The return path resolves every known host session and enqueues silent system events/wakeups to those targets, so a prompt-controlled session with continuation enabled can inject context into unrelated conversations. Restrict returns to the requester/ancestor set or require an explicit operator-approved targeting policy before exposing host-wide fanout.
  - confidence: 0.91
- **[P2] Keep readLocalFileFromRoots on the SDK barrel:** `src/plugin-sdk/file-access-runtime.ts:3-8`
  - body: `openclaw/plugin-sdk/file-access-runtime` is a documented plugin SDK subpath, and current main exports `readLocalFileFromRoots` from it. Dropping that export while adding `appendFileWithinRoot` breaks existing plugins that import the media-source helper. Add the new helper without removing the existing export.
  - confidence: 0.9
- **[P2] Redact session keys in fanout spans:** `src/infra/continuation-tracer.ts:762`
  - body: `emitContinuationFanoutSpan` records full target session keys in span attributes. Session keys can contain channel, account, or conversation identifiers, and diagnostics-otel can export spans outside the local process, so fanout telemetry should use counts, stable hashes, or redacted labels instead of raw keys.
  - confidence: 0.86

## Security Review

Status: needs_attention

Summary: The diff introduces model-controlled cross-session return targeting and raw session-key telemetry attributes.

Concerns:

- **[medium] Unrestricted host-wide session injection:** `src/agents/tools/continue-delegate-tool.ts:49`
  - body: `fanoutMode: "all"` and explicit target session keys let a continuation delegate return enqueue internal system events or wakeups for sessions beyond the requester lineage. Because this is exposed through model tool calls and fallback syntax, prompt injection or a compromised session could inject context across unrelated conversations when continuation is enabled.
  - confidence: 0.91
- **[medium] Raw session keys exported to telemetry:** `src/infra/continuation-tracer.ts:762`
  - body: Fanout spans include full target session keys as attributes. These keys may include channel, account, or conversation identifiers and can leave the host through configured OTEL exporters, so they should be redacted or hashed before export.
  - confidence: 0.86
