Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Thread format retry into assistant failover decisions:** `src/agents/pi-embedded-runner/run.ts:2276-2288`
  - body: This assistant failover decision no longer passes cloudCodeAssistFormatError as allowFormatRetry. Because format failures remain terminal when allowFormatRetry is absent, Cloud Code Assist format rejections that should sanitize and retry will surface immediately; thread the flag through both assistant decision call sites.
  - confidence: 0.9
- **[P1] Restore hook wrapping opt-outs for shared tool paths:** `src/agents/pi-tools.ts:752-753`
  - body: Dropping wrapBeforeToolCallHook: false lets createOpenClawTools wrap tools at construction time even though the embedded/gateway execution paths also run before_tool_call handling at their shared boundary. Restore the opt-out here and in gateway tool resolution, or remove the later wrapper intentionally.
  - confidence: 0.88
- **[P1] Constrain cross-session continuation fanout:** `src/agents/tools/continue-delegate-tool.ts:63-67`
  - body: continue_delegate exposes targetSessionKey(s) and fanoutMode="all" to the model. The return path can then enqueue internal system events and wakes for every known host session, so a prompt-injected session can push context into unrelated conversations once continuation is enabled; restrict recipients or require an approved policy.
  - confidence: 0.88
- **[P2] Guard partial continuation tool registration:** `src/agents/openclaw-tools.ts:474-476`
  - body: continue_delegate registers whenever drainsContinuationDelegateQueue is not false, while continue_work and request_compaction require explicit callbacks. A caller with continuation enabled but incomplete callbacks can expose only the delegate tool without the sibling runtime surface; gate the tool on an explicit complete registration path or warn and skip partial registration.
  - confidence: 0.82
- **[P3] Forward traceparent into spawned delegates:** `src/auto-reply/continuation/delegate-dispatch.ts:255-268`
  - body: The tool schema accepts traceparent, but dispatchToolDelegates never passes delegate.traceparent into spawnSubagentDirect. That drops the advertised cross-boundary trace correlation for normal tool delegates; include the validated traceparent when spawning.
  - confidence: 0.8
