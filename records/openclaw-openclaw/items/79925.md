## What This Changes

The PR adds opt-in continuation tools/runtime for `continue_work`, `continue_delegate`, and `request_compaction`, with delegate fanout, context-pressure handling, status/tracing/docs/tests, Mantis workflow routing, and a `uuid` dependency.

## Best Possible Solution

## Solution Assessment

## Review Findings

Overall correctness: patch is incorrect

Full review comments:

- **[P2] Honor the configured default delay when omitted:** `src/agents/tools/continue-work-tool.ts:75-80`
  - body: `createOpenClawTools` now always passes config into `continue_work`, but the tool converts an omitted `delaySeconds` to `0` before clamping.
