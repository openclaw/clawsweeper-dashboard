## Security Review

Status: needs_attention

Summary: The diff introduces model-controlled cross-session context injection and trusted delegate-return insertion without an approved policy gate.

Concerns:

- **[high] Model-controlled cross-session context injection:** `src/agents/tools/continue-delegate-tool.ts:49`
  - body: The continuation tool schema exposes explicit target sessions and host-wide fanout to the model, and the delivery path writes/wakes the resolved target sessions. Without an operator policy or lineage restriction, a compromised continuation-enabled session can affect unrelated sessions on the same host.
  - confidence: 0.92
