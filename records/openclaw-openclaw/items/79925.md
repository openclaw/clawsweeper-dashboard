## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Constrain cross-session continuation fanout:** `src/agents/tools/continue-delegate-tool.ts:63-67`
  - body: `continue_delegate` exposes `targetSessionKey`, `targetSessionKeys`, and `fanoutMode: "all"` directly to the model. The added return path can enqueue silent system events and wakes into other sessions on the host, so a prompt-controlled session can push context into unrelated conversations once continuation is enabled. Restrict returns to the requester/ancestor set or require an explicit owner-approved policy before exposing host-wide targeting.
  - confidence: 0.9
- **[P1] Preserve format-error retry failover:** `src/agents/pi-embedded-runner/run/assistant-failover.ts:182-190`
  - body: The PR removes `allowFormatRetry: params.cloudCodeAssistFormatError` from the assistant failover decision and drops the corresponding decision field. CloudCodeAssist format errors that previously retried under the format-retry gate will now surface instead of taking the intended retry path.
  - confidence: 0.88
- **[P2] Keep hook wrapping disabled for shared tool paths:** `src/agents/pi-tools.ts:750-753`
  - body: `createOpenClawTools` still wraps normalized tools unless `wrapBeforeToolCallHook` is explicitly false, and later shared/gateway paths also expect raw tools here. Removing the opt-out means OpenClaw-owned tools can run `before_tool_call` twice, duplicating approvals, diagnostics, or loop-accounting side effects.
  - confidence: 0.87
- **[P3] Warn or gate partial continuation registration:** `src/agents/openclaw-tools.ts:430-438`
  - body: `continue_delegate` registers when `drainsContinuationDelegateQueue !== false`, so a continuation-enabled caller with incomplete callbacks can expose delegate continuation without the sibling `continue_work` or `request_compaction` tools. Add the acknowledged guard/warning or gate this tool on explicit runtime options so operators do not get a silent partial continuation surface.
  - confidence: 0.82
- **[P3] Forward traceparent when spawning delegates:** `src/auto-reply/continuation/delegate-dispatch.ts:261-268`
  - body: The tool and delegate descriptor accept a validated `traceparent`, but normal delegate dispatch never passes it to `spawnSubagentDirect`. The advertised trace stitching will be broken for regular continuation delegates unless the spawn params include `delegate.traceparent` when present.
  - confidence: 0.82

## Security Review

Status: needs_attention

Summary: The diff introduces model-controlled cross-session continuation return and host-wide fanout without an evident authorization policy.

Concerns:

- **[medium] Unrestricted host-wide session injection:** `src/agents/tools/continue-delegate-tool.ts:63`

Because this is exposed to model output/tool calls, prompt injection or a compromised session could leak or inject context across unrelated conversations when continuation is enabled.
