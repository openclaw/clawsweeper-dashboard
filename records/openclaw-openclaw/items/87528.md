## Summary

Keep open: the WebChat download bug is real, but this branch is not merge-ready because it applies `attachment` to every assistant-media response, including image/audio/video preview URLs, while newer related work tracks the safer document-only fix.

## What This Changes

The PR adds a basename-derived `Content-Disposition: attachment` header with `filename` and `filename*` to all successful Control UI assistant-media responses.

## Best Possible Solution

Land one canonical fix that sets safe attachment filenames only for non-inline assistant media, keeps previewable media inline, and covers both gateway response headers and WebChat rendering behavior with focused tests.

## Reproduction Assessment

Yes, at source level: current `main` and `v2026.6.10` stream assistant-media files without `Content-Disposition`, while the WebChat UI uses that route for document links. I did not run a browser save-dialog reproduction in this read-only review.

## Solution Assessment

No. The submitted route-wide attachment header is only a plausible symptom fix; the maintainable fix should gate disposition by MIME/media kind, use RFC 5987-safe filename encoding, and preserve inline previews.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Keep preview media inline:** `src/gateway/control-ui.ts:606-610`
  - body: This shared endpoint also backs image, audio, and video `src` URLs in WebChat. Setting `attachment` for every successful response can make existing previews download instead of render, so gate the header to document/non-inline media or a dedicated download path.
  - confidence: 0.92
- **[P2] Encode filename* with RFC 5987-safe characters:** `src/gateway/control-ui.ts:610`
  - body: `encodeURIComponent` leaves characters such as apostrophes, parentheses, and `*` unescaped, and apostrophes are delimiter-sensitive after `UTF-8''`. Use a small RFC 5987-safe encoder plus a clean ASCII fallback before interpolating the header.
  - confidence: 0.84
