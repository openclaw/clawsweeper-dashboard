Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Keep previewable media inline:** `src/gateway/control-ui.ts:606-610`
  - body: This endpoint also feeds local assistant images, audio, and video in WebChat via the same assistant-media URL. Setting every successful response to `attachment` can turn existing previews into downloads, so gate this header to document/non-inline MIME responses or another explicit download path.
  - confidence: 0.9
- **[P2] Encode filename* with RFC 5987-safe characters:** `src/gateway/control-ui.ts:610`
  - body: `encodeURIComponent` leaves characters such as apostrophes, parentheses, and `*` unescaped, but those are not safe in the RFC 5987 extended value grammar. Use a small header builder that tightens percent-encoding and supplies an ASCII fallback before interpolating the header.
  - confidence: 0.82
