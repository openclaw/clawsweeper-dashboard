## Summary

The PR adds a boot-volume-aware macOS LaunchAgent home resolver and routes daemon lifecycle, restart/recovery, inspect, update helper, managed update handoff, macOS app snapshot/log lookup, and regression tests through it for external APFS HOME setups.

## Best Possible Solution
