## Summary

Keep this PR open for maintainer review: current main and v2026.6.9 still use HOME-derived macOS LaunchAgent paths, and this PR is the active fix candidate for the linked external-APFS-home install failure. ## What This Changes

The PR adds a boot-volume-aware macOS LaunchAgent home resolver and routes daemon lifecycle, update restart/recovery helpers, macOS app lookup, and regression tests through it for external APFS homes.

## Best Possible Solution

Land one canonical boot-volume LaunchAgent path across macOS install, restart, uninstall, repair, inspection, update recovery, and app lookup, with cleanup hints either routed through the same resolver or explicitly deferred as follow-up.
