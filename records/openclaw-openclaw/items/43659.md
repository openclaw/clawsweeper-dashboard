## Best Possible Solution

Port the fingerprint/cache-ownership check into src/plugin-sdk/provider-auth.ts, keep both wrappers as re-exports, preserve Copilot integration identity behavior, add same-profile hit/cross-profile miss/legacy-cache coverage, and require redacted real rotation proof before merge.

## Solution Assessment
