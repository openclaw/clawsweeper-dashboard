## Summary

## What This Changes

The PR adds a GitHub-token fingerprint to cached Copilot API tokens and regression tests so cache reuse is skipped for different GitHub PATs or legacy unbound entries.

## Best Possible Solution

Land a rebased or replacement PR that applies the owner-aware cache check in src/plugin-sdk/provider-auth.ts, exports any helper needed by the current barrels/tests, treats legacy cache entries as misses, and includes redacted real two-profile Copilot proof.

## Solution Assessment
