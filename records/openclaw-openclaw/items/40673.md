## Summary

Keep open. Current main still only documents enabling raw-stream logging, while the LaunchAgent restart path still has an unchanged-plist `kickstart -k` path that does not prove stale raw-stream environment state is cleared after `unsetenv` or `0`.

The issue requests clear disable semantics and reliable LaunchAgent behavior for `OPENCLAW_RAW_STREAM` so raw-stream logging stops after unset or `0`.

## Best Possible Solution
