## What This Changes

The PR adds cron load-time UUID backfill and persistence for jobs missing both `id` and `jobId`, guards invalid cron outcome IDs, and updates cron tests, docs, and changelog.

## Best Possible Solution

Land the focused cron fix after maintainer review and exact-head validation, keeping ID repair in the store load path and the malformed-outcome guard in the timer path.

## Solution Assessment
