## Summary

Keep open: current main still has the reported drift between Anthropic's full plugin runtime and the lightweight bundled provider-policy artifact, so no-runtime policy paths can miss Opus 4.7 thinking levels.

## What This Changes

Request to add Anthropic `resolveThinkingProfile` to the bundled provider-policy artifact so Opus 4.7 `/think max`, `xhigh`, and `adaptive` are honored in no-runtime policy paths.

## Best Possible Solution

Expose Anthropic's Claude thinking profile through the lightweight bundled provider-policy artifact and lock it with regression coverage for Opus 4.7 and adaptive-only Claude models.
