## Summary

Keep open. This is a protected maintainer draft that makes a real crash-safety change with no blocking source-level finding found, but the default local embedding runtime moves from in-process to a forked worker and still needs maintainer review plus packaged fresh-install/upgrade proof before merge.

## What This Changes

The branch moves local memory embeddings behind a forked worker by default, serializes in-process local batches, degrades to configured fallback or FTS-only search after worker failure, and adds focused memory/config/build-entry coverage.

## Best Possible Solution

Land after maintainer review confirms worker isolation is the intended short-term local-memory contract and packaged fresh-install plus upgrade proof shows the bundled worker starts, degrades, falls back, and searches correctly.

## Solution Assessment
