## Summary

Keep open. Current main already covers the GatewayClient timeout portion, but the broader issue is not fully implemented: auth profiles still default to plaintext JSON, session transcripts are still plain JSONL, current skill/plugin defenses are scanner/moderation-based rather than signature verification, and the security docs still describe prompt-injection hardening plus encryption-at-rest as outstanding recommendations.
