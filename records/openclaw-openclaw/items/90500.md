- **[P1] Validate the model used by reply execution:** `src/gateway/server-methods/chat.ts:3030-3032`
  - body: The PR resolves a catalog-aware `resolvedSessionModel` here, but that value is not passed into `dispatchInboundMessage` or the auto-reply model-selection path.
