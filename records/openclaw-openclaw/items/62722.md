AbortController().signal` creates a
    signal from an anonymous, unreferenced controller that can never be
    aborted. If stopAccount implementations use the signal for cleanup
    coordination, the never-aborting signal can cause hangs during graceful
    shutdown.
    
    Replace with `AbortSignal.abort()` which returns an already-aborted
    signal, correctly communicating that the account is being stopped.
