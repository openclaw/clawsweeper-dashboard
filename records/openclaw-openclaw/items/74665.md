## Solution Assessment

Yes. A Responses-specific replay-boundary normalizer with focused regression coverage is the narrow maintainable fix because generic OpenAI tool-call ID sanitization is disabled for Responses and canonical `call_*|fc_*` reasoning pairs must be preserved.
