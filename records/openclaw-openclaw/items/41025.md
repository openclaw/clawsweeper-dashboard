## Summary

Keep open. The PR addresses a real Control UI feature request, but it cannot merge as-is because current main intentionally blocks remote chat image fetches in the renderer, CSP, tests, and docs, and the PR lacks real browser/CSP proof.

## What This Changes

The PR changes Control UI markdown rendering/tests and the changelog so http/https markdown image URLs emit inline `<img>` tags with `referrerpolicy` instead of falling back to alt text.

## Best Possible Solution

Decide the Control UI remote-media policy first; if approved, port the renderer change to current main and align CSP, docs, tests, changelog, privacy mitigations, and real browser proof.

## Reproduction Assessment

Yes, at source level. Current main returns alt text for non-data markdown image sources, tests assert that behavior, and the served CSP would block remote loads even if an `<img>` were emitted.

## Solution Assessment

No. A renderer-only http/https allowance is not the best way to solve this because current main documents and enforces a no-remote-fetch policy across renderer, CSP, tests, and docs.

## Security Review

Status: needs_attention

Summary: The PR enables untrusted remote image fetches from rendered chat content without an approved remote-media policy or matching CSP change.

Concerns:

- **[medium] Remote image fetches leak viewer metadata:** `ui/src/ui/markdown.ts:148`
  - body: Rendered chat/model content could include tracking pixels that make the browser fetch a third-party URL, exposing viewer IP address, user agent, and read timing; `referrerpolicy` does not prevent those requests.
  - confidence: 0.91
- **[low] Plain HTTP image sources weaken the policy:** `ui/src/ui/markdown.ts:45`
  - body: The proposed regex allows `http://` as well as `https://`; on HTTPS Control UI deployments this can be mixed-content blocked and is a weaker policy than a secure remote-media allowlist.
  - confidence: 0.82

  - body: The changelog states that `https:` is allowed in the Control UI CSP, but the PR does not change the CSP helper, which can mislead reviewers and users about actual browser behavior.
