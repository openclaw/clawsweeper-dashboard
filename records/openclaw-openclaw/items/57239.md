The PR adds an automatic fast-lane path for short active-session follow-ups, including a queue-policy override, transient run/lane wiring, cleanup, and unit coverage.

## Best Possible Solution

Land a maintainer-approved quick-reply or concurrency design that isolates reply operations and session state, preserves ordering, enforces hard tool controls, and proves the real active-run flow.
