## Best Possible Solution

If maintainers accept opt-out empty-success notifications, land a rebased update or replacement that flips the default consistently while preserving the current `exec-event` completion wake contract, updates generated schema/public docs, adds the required changelog entry, and validates the focused exec/config lanes.
