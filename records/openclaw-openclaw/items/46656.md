Add WebChat/Control UI inline button support, including capability config, final message payload/rendering, callback RPC, and optional webhook routing.

## Best Possible Solution

Implement an additive WebChat interactive-control path using the shared presentation contract, add a scoped chat.callback RPC that initially routes selected values back into the same session, and keep webhook routing as a separately hardened design.
