- **[P1] Preserve fallback send context:** `extensions/telegram/src/send.ts:877-879`
  - body: The rich send uses `acceptedParams` for topic, reply, and final-chunk button context, but the fallback plain send calls `sendTelegramTextChunk` without those params.   - confidence: 0.93

  - confidence: 0.88

  - body: The `not supported on .* telegram` branch can match future or unrelated Telegram errors and convert them into plain-text retries. Keep this scoped to the known Telegram Web or version-unsupported wording so unrelated API failures still propagate.
  - confidence: 0.78
