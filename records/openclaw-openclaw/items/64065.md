Fix the Race C plugin-binding approval persistence bug from https://github.com/openclaw/openclaw/issues/64065. Keep the change narrowly scoped to `src/plugins/conversation-binding.ts`: serialize the persistent approval read/merge/write path for allow-always approvals so concurrent saves cannot leave `plugin-binding-approvals.json` with stale data while memory has newer data.
