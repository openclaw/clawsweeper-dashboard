## Best Possible Solution

Repair the predecessor lookup to use the filtered `segment` being rendered, add a keyed-preamble-plus-tool regression, and keep this as the focused causal-ordering fix unless maintainers fold the same rule into broader WebChat stream work.

## Solution Assessment
