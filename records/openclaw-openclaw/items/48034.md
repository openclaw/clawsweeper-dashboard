## Summary

## What This Changes

The branch removes the rw guard around sandbox skill sync, passes managed and bundled skill directories, and changes the Overview locale selector to prefer persisted settings.

## Best Possible Solution

Land a fresh narrow sandbox fix that materializes or remaps managed and bundled skills into the actual rw sandbox-readable workspace path, preserves `agentId`, remote eligibility and bootstrap contracts, keeps strict path boundaries, adds rw regression coverage, and drops the locale change already shipped on main.

## Solution Assessment

No. Mirroring or remapping skill files into a sandbox-readable path is the right direction, but this patch keeps the sync unreachable for normal rw mode, imports the bundled resolver from the wrong module, and should drop the already-shipped locale change.
