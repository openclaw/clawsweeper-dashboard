## Summary

Keep open: the timeout hook bug is still present on current main, but this PR is not merge-ready because the guard uses session-level `lastAssistant`, the branch mixes in an unsafe unrelated parser change, and no after-fix real timeout proof is attached.

## What This Changes

The PR guards embedded-runner `llm_output` emission after timeouts and adds JSON-stringified chat-content parsing with tests.

## Best Possible Solution

Land a focused fix that gates embedded-runner `llm_output` on current-attempt assistant output with regression coverage, and leave JSON-stringified chat-content parsing to the separate parser PR.

## Reproduction Assessment

Yes, at source level: current main computes `currentAttemptAssistant` but emits `llm_output` without requiring current-attempt assistant output. I did not run a live timeout scenario in this read-only review.

## Solution Assessment

No. The hook fix should gate on `currentAttemptAssistant` or current-attempt assistant text, and the unrelated JSON parser work should stay in https://github.com/openclaw/openclaw/pull/39583.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Gate llm_output on current-attempt output:** `src/agents/pi-embedded-runner/run/attempt.ts:2009`
  - body: A timeout after an earlier assistant message can leave session-level `lastAssistant` truthy, so this condition can still emit `llm_output` even when the current model attempt produced no output. Gate on `currentAttemptAssistant` or current-attempt assistant text instead.
  - confidence: 0.88
- **[P2] Preserve unsupported serialized blocks as text:** `src/shared/chat-content.ts:18-24`
  - body: The stringified-array path recurses for items with `type` and `text`, but the array extractor only accepts `type === "text"`. Payloads like `[{"type":"output_text","text":"hello"}]` can now collapse to `null` instead of preserving the original string, so only parse supported text blocks or leave unsupported serialized arrays unchanged.
  - confidence: 0.88
