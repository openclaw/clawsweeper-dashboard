## Summary

Keep open: the route-activation goal is still useful and not implemented on current main, but this maintainer-labeled PR has source-backed lifecycle, security-boundary, availability, and rebase blockers before merge.

## What This Changes

The PR adds manifest `activation.onRoutes` route-owner planning, annotates bundled route-capable plugin manifests, and makes gateway plugin HTTP miss handling try a scoped route registry.

## Best Possible Solution

Keep the route-activation goal, but move scoped registry ownership into a gateway lifecycle path or non-activating memoized snapshot, make auth and dispatch share route candidates, and rebase over current main before merge.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Keep scoped route loads out of request dispatch:** `src/plugins/runtime/http-route-registry-loader.ts:30-38`
  - body: This helper is wired into the plugin HTTP miss path, but it calls `loadOpenClawPlugins` without `activate: false`. The loader default activates global plugin state, so an ordinary HTTP miss can execute route-owner runtime code and replace process-wide registry state outside a controlled startup or reload lifecycle.
  - confidence: 0.92
- **[P1] Do not fail all scoped routes on one owner error:** `src/plugins/runtime/http-route-registry-loader.ts:32`
  - body: `throwOnLoadError: true` makes any failing plugin in the scoped `gateway-plugin-http` owner set throw before route matching can choose a healthy route. Because plugin HTTP is a `continueOnError` stage, that exception skips the whole plugin-route stage for the request.
  - confidence: 0.88
- **[P2] Run auth against scoped route candidates:** `src/gateway/server/plugins-http.ts:85-90`
  - body: Scoped routes are resolved only after the earlier `plugin-auth` stage decides whether to authenticate using the startup registry. A scoped-only route with `auth: "gateway"` can therefore reject valid gateway credentials because the auth stage never saw the scoped candidate set.
  - confidence: 0.82
