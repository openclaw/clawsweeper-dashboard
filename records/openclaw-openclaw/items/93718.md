- **[P1] Preserve owned root dependencies:** `src/infra/npm-managed-root.ts:813`
  - body: The new reconciliation loop rewrites any direct dependency that shares a name with a managed override, but managed roots also store explicitly installed plugin packages as direct dependencies.
