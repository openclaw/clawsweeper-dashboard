  - body: When a lower-priority hook returns `{ block: false }`, this branch clears the soft block fields but leaves `acc.requireApproval` in the merged result. The consumer can still prompt for approval even though the soft block was explicitly cleared, so clearing a soft block is incomplete.

  - confidence: 0.74
