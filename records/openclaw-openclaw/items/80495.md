## What This Changes

The issue asks OpenClaw to expand LM Studio API-key environment variables and adapt LM Studio inference endpoint selection away from failing `/v1/chat/completions`.

## Best Possible Solution

Land or replace the focused auth-resolution fix, then split any native `/api/v1/chat` or endpoint fallback work into a separate LM Studio provider design issue backed by live logs.

## Solution Assessment
