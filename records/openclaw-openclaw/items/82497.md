## What This Changes

The PR changes auth-profile merging so selected-agent provider profiles are inserted and ordered before inherited main-agent profiles while preserving inherited profiles as fallback.

## Best Possible Solution

Review and land this PR or an equivalent shared auth-profile precedence fix once CI is understood, then close the linked issue after merge.
