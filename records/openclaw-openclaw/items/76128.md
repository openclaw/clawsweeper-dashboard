## Summary

Keep open. ## What This Changes

The issue asks OpenClaw to avoid silently leaving the main/default agent on an old explicit `agents.list[].model` after a default-model change.

## Best Possible Solution

Keep the current explicit-agent precedence until maintainers choose a non-destructive source-aware contract, likely `models set` warning plus status or doctor guidance before any explicit opt-in agent mutation.
