## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Yes, if maintainers want this route in the shared-secret compatible bucket. Using resolveOpenAiCompatibleHttpOperatorScopes at both call sites is the narrowest maintainable implementation for that policy.
