## What This Changes

Prevent Telegram agent replies from claiming file or messaging actions were completed when no corresponding tool execution occurred.

## Best Possible Solution

Keep https://github.com/openclaw/openclaw/issues/60955 as the single tracker, link this Claude/ACP evidence there, and design a structured cross-provider execution contract or validation seam that preserves legitimate text-only answers instead of matching natural-language phrases.
