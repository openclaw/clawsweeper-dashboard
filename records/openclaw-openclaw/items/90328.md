## Summary

## What This Changes

The PR adds optional `agentRuntime` metadata to Gateway `models.list` model choices, propagates it through protocol/Swift/UI types, and appends runtime labels in the WebUI model picker while preserving canonical model values.

## Best Possible Solution

Land the additive Gateway/UI protocol update after normal maintainer review and merge gates, keeping canonical model refs stable and runtime metadata optional.

## Solution Assessment

Yes. The PR uses an optional protocol field resolved through the existing runtime-policy path and renders it as a display suffix, which is narrower and safer than rewriting model refs or generating aliases.
