## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.84

Full review comments:

- **[P1] Make the new models.list field upgrade-safe:** `packages/gateway-protocol/src/schema/agents-models-skills.ts:24`
  - body: `ModelChoiceSchema` is part of the public `models.list` contract, and existing generated Swift clients reject unknown keys for schemas marked `additionalProperties: false`. Emitting `agentRuntime` unconditionally can make older native clients fail to decode model lists after a Gateway upgrade, so this needs gating/versioning or explicit maintainer acceptance before merge.
  - confidence: 0.86
- **[P1] Preserve configured model aliases:** `ui/src/ui/chat-model-ref.ts:237`
  - body: `resolveCatalogDisplayName(entry)` returns `entry.alias` before the catalog name, so this path appends the runtime suffix to labels users already configured to control picker text. Existing aliases like `GPT-5.5 · OpenAI` will become `GPT-5.5 · OpenAI · OpenAI Codex` after upgrade; keep aliases exact or only decorate generated catalog labels.
  - confidence: 0.82
