## Best Possible Solution

Keep the additive gateway metadata and UI label path, but merge only after redacted real Gateway or WebUI proof shows the runtime label and unchanged `openai/gpt-5.5` value.

## Reproduction Assessment

## Solution Assessment

Yes, this is the best code layer: the gateway owns `models.list` metadata and can reuse existing runtime-policy helpers, while the UI renders metadata instead of inferring runtime from provider strings.
