Keep open. Current main covers the normal delivered-message Google Chat thread reply path, but the receive-all/non-mention space behavior is still missing and requires OAuth, event-delivery, credential, and Workspace security decisions rather than cleanup closure.

The issue requests Google Chat threaded replies and an optional mode to receive all messages in spaces without requiring @mentions.

## Best Possible Solution

Keep this open as Google Chat plugin-owned work for an opt-in all-message space delivery mode, coordinated with user OAuth or Workspace Events as appropriate while preserving service-account and mention-gated defaults.
