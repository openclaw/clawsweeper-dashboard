## Summary

Keep this open. The Google Chat thread-reply primitive is present on current main and v2026.6.6, but the distinct non-mention space-message request still lacks a delegated-auth or event-subscription design and needs maintainer security/product review before implementation.

## What This Changes

The issue requests Google Chat threaded replies plus optional delivery of all space messages instead of only bot-invoking or mentioned messages.

## Best Possible Solution

Keep this issue open until maintainers split or approve a Google Chat delegated-auth/event-subscription design for non-mention space messages, while letting the narrower thread PR proceed separately if it remains the preferred fix.
