## Summary

Keep open: this is a fresh, multi-reporter runtime failure with concrete logs across Anthropic, Anthropic Vertex, and MiniMax on v2026.5.7, and current `main` still contains the matching embedded-runner empty-response retry/exhaustion behavior without proof that the macOS/Node 26 empty-payload root cause is fixed. Related open issues suggest a broader transport/runtime problem rather than an obsolete single-model report.

## What This Changes

The issue requests a fix for embedded agent turns returning `payloads=0` incomplete terminal responses for Anthropic-family models while lower-level direct inference and provider API calls succeed.

## Best Possible Solution

Keep this issue as the canonical embedded-agent empty-payload investigation until a maintainer captures redacted live request/stream evidence on an affected macOS/Node 26 setup and fixes the shared runtime or transport boundary with regression coverage.
