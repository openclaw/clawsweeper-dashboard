## Summary

Keep open: current main and the latest release still resolve bootstrap context from the active workspace only, with no merged warning or agentDir merge path, and all directly linked fix PRs are closed unmerged. ## What This Changes

The issue asks OpenClaw to warn about, document, or support bootstrap-named files placed in a configured per-agent agentDir.

## Best Possible Solution
