## Summary

Keep this open. Current main still reads bootstrap files from the workspace only, has no agentDir warning path, and an open closing PR (#72871) should be reviewed or superseded before cleanup.

## What This Changes

The issue asks OpenClaw to either load per-agent agentDir bootstrap files or warn when bootstrap-named files there will be ignored.

## Best Possible Solution

Keep this issue open until PR #72871 lands or maintainers choose a documented agentDir/workspace merge contract; the minimum end state is a warning for ignored bootstrap-named files in agentDir.

## Reproduction Assessment

Yes. Source inspection on current main shows the bootstrap resolver loads only workspaceDir and never receives agentDir, so an agentDir-only SOUL.md or AGENTS.md will not be injected or warned about.

## Solution Assessment

Unclear for the preferred merge/override behavior because that changes the agent workspace contract. The narrow warning approach in PR #72871 is the smallest maintainable fix for the confirmed silent-failure path if maintainers accept Option B.
