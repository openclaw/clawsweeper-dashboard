The issue reports a regression where isolated lightweight `sessions_spawn` subagents using local Ollama runtimes inherit excessive context through context-engine preparation.

## Best Possible Solution

Land or replace the linked focused fix so isolated lightweight native subagent spawns avoid inherited payload bloat while preserving the intended context-engine lifecycle for fork/default modes.
