Isolated `sessions_spawn` subagent runs with `lightContext=true` should stay lightweight for local Ollama subagents instead of inheriting context-engine payload bloat.

## Best Possible Solution

Land or replace the linked fix with a contract-aware change that prevents inherited payload bloat for isolated `lightContext` native subagent spawns and includes focused tests plus live or redacted Ollama proof.

## Reproduction Assessment

## Solution Assessment
