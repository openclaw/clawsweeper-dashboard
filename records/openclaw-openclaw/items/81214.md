## Best Possible Solution

Land a reviewed fix that keeps isolated `lightContext` runs lightweight while preserving or deliberately revising the context-engine `prepareSubagentSpawn` contract, with targeted regression tests and redacted Ollama proof.

The safer fix should stop heavyweight inherited payloads for isolated lightweight runs without silently breaking the context-engine hook contract.
