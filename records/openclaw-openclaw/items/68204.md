Keep open. ## Best Possible Solution

Keep this as the architecture tracker until maintainers define a versioned, redacted, authorized run-trace contract with explicit ACP correlation and clock semantics, then land focused adapters and tests for the existing producers.

## Risks / Open Questions

- The ACP handoff still needs an explicit wire-level correlation carrier and a timestamp/clock-source policy before parent-child timelines can be reconstructed reliably across processes.
