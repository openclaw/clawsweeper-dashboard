## Summary

## What This Changes

## Best Possible Solution

Apply message-tool-delivered metadata only after evaluating the whole heartbeat result and only when exactly one non-reasoning plain visible fallback remains eligible; otherwise keep normal fallback delivery intact.

## Reproduction Assessment
