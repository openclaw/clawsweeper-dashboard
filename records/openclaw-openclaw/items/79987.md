## Summary

Keep open: the latest branch fixes the earlier validation/comparator problems, but the cached-skip path still returns requested npm resolution metadata for a different installed artifact, which can persist stale install records and leave a later update path able to downgrade. The PR also still lacks after-fix real CLI or wizard behavior proof.

## What This Changes

The PR adds a managed npm install-mode fast path that validates and skips reinstalling an already-installed plugin when its package version satisfies the requested version, with docs, changelog, and focused tests.

## Best Possible Solution

Keep the validated cache-hit fast path, but record metadata from the installed package or avoid persisting stale requested resolution fields so future update paths cannot target the older request, then add real CLI or wizard proof.

## Reproduction Assessment

Yes from source inspection: current main resolves npm metadata, then rejects an existing managed npm package directory before checking whether the installed version satisfies the request. I did not run a live Docker wizard repro in this read-only review.

## Solution Assessment

No. The validated skip direction is right, but returning requested npmResolution for an installed newer package can persist stale resolution and integrity metadata, leaving a later update path able to downgrade.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Record the cached package's actual npm resolution:** `src/plugins/install.ts:515`
  - body: On a cache hit with an installed newer package, this returns the installed version but keeps `npmResolution` from the older requested spec. The onboarding and CLI record writers persist those resolution fields, so a later npm update can resolve the old exact spec, miss the unchanged check, and downgrade the package this PR is meant to preserve. Derive the recordable metadata from the cached package, or avoid persisting stale requested resolution fields for this path.
  - confidence: 0.84
