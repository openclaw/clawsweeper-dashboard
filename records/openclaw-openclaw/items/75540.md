## Summary

Keep open. The PR targets a real Slack capabilities bug and the `auth.test` fallback direction is useful, but the current head still treats `response_metadata.acceptedScopes` as granted token scopes, which can make `openclaw channels capabilities` falsely report method-accepted scopes as `Bot scopes` or `User scopes`.

## What This Changes

This PR adds an `auth.test` fallback to Slack scope discovery, extends extraction to Slack response metadata, adds focused Slack scope tests, and records a changelog fix for modern Slack bot tokens.

## Best Possible Solution

Keep the Slack plugin fix direction, but return granted scopes only from `response_metadata.scopes` for the `auth.test` fallback. Remove `acceptedScopes` from the grant list, update the focused accepted-scopes test and changelog wording, and leave core channel APIs unchanged.

## Reproduction Assessment

Yes. The linked bug has concrete `openclaw channels capabilities` steps, a contributor reproduced the same two-method `unknown_method` path on upstream main, and current main still contains that exact attempt list and joined-error behavior.

## Solution Assessment

No. The `auth.test` fallback plus `response_metadata.scopes` extraction is the narrow maintainable solution, but including `acceptedScopes` in the granted-scope list is not correct for capability diagnostics.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Do not treat accepted scopes as granted scopes:** `extensions/slack/src/scopes.ts:75-77`
  - body: The Slack SDK fills `response_metadata.acceptedScopes` from `x-accepted-oauth-scopes`, which is the set accepted by the Web API method rather than scopes granted to this token. Since capability diagnostics print these values as `Bot scopes` or `User scopes`, an acceptedScopes-only response can falsely report method requirements as token grants; use only `response_metadata.scopes` for granted-scope discovery or label accepted scopes separately.
  - confidence: 0.88
