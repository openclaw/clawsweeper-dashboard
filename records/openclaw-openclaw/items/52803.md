## Summary

Keep open. Add a scalable Control UI orchestration view for agent/main-session/subagent hierarchy, active-first filtering, and bulk kill/steer/retry/cleanup operations.

## Best Possible Solution

Keep this as the umbrella runtime subagent-orchestration request and split implementation into reviewed slices: read-only scalable hierarchy first, then privileged bulk actions after semantics are chosen.

## Reproduction Assessment
