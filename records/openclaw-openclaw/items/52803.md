## Summary

Keep open. Current main has made partial progress by exposing session status/runtime/model metadata and chat-side kill/steer controls, but the requested browser Control UI orchestration surface is still missing: no orchestration route, no session/subagent hierarchy or virtualization, and selected-row bulk actions only delete sessions.

The issue requests a scalable Control UI orchestration surface with agent/session/subagent hierarchy, active-first filters, bulk kill/steer/retry/cleanup, per-row metadata, and large-list rendering.
