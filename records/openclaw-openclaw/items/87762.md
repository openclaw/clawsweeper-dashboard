  - body: This change removes the shared `profileIds` fan-out from both OpenCode auth methods.   - confidence: 0.93

  - confidence: 0.9

  - body: The new `opencode-go` precedence test still runs against the hoisted mock candidate map that only lists `OPENCODE_API_KEY` and `OPENCODE_ZEN_API_KEY`.   - confidence: 0.91
