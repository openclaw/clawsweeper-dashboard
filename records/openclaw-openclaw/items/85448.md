Keep this PR open. ## What This Changes

The PR adds a guarded direct managed-proxy bypass for exact configured loopback Ollama memory embedding requests, updates SSRF policy use, adds focused regression tests, and documents the proxy exception.

## Best Possible Solution

## Solution Assessment
