- **[P1] Cache MCP metadata after collecting all pages:** `src/agents/agent-bundle-mcp-runtime.ts:327`
  - body: `cacheToolMetadata()` clears the SDK's cached validators and task-tool sets every time it runs, but this call runs once per `tools/list` page while `tools` keeps every previous page.
