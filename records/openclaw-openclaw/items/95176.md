## Summary

Keep open: the PR moved the mitigation to the right guarded model-fetch layer, but it is not merge-ready because the retry only covers `UND_ERR_SOCKET`, can replay a consumed `Request` body on shared transport callers, and still lacks after-fix Anthropic gateway proof.

## Best Possible Solution

## Solution Assessment
