  - body: This branch imports `OpenClawConfig` from `src/config/config.js`, but that module does not export the type at PR head; the exported type is in the config type module. The changed files will fail type checking before the behavior can be exercised.

  - body: The branch makes both slugging and continuity extraction opt-out, then awaits extraction inside the hook. `/new` and `/reset` await `triggerInternalHook`, so default resets can send transcript text to the configured provider and wait on the embedded run; preserve the current local/background behavior unless this is explicitly enabled.
  - confidence: 0.95
- **[P2] Register the compact event for session-memory:** `src/hooks/bundled/session-memory/handler.ts:131`
  - body: The new `session:compact:before` branch will not run when the bundled hook is loaded normally because `HOOK.md` still lists only `command:new` and `command:reset`, and the loader registers only metadata-listed events. Add the compact event metadata/docs or remove the advertised compact path.
  - confidence: 0.96
- **[P2] Preserve local-time archive naming:** `src/hooks/bundled/session-memory/handler.ts:161`
  - body: Current main intentionally names session-memory archive files and headings in the host local timezone. This branch derives the archive date, fallback slug, and header time from `toISOString()`, so `/new` and `/reset` would return to UTC-dated artifacts after rebase.
  - confidence: 0.93
- **[P2] Compare conversation summaries before skipping writes:** `packages/memory-host-sdk/src/host/continuity.ts:417-438`
  - body: When LLM extraction is disabled or fails, `conversationSummary` can be the only meaningful recent state. `hasMaterialContinuityChange()` omits it from both canonical objects, so later sessions with different conversation text but the same fallback fields can be skipped and leave `latest.md` stale.
  - confidence: 0.91
- **[P3] Honor fallback list section headings:** `packages/memory-host-sdk/src/host/continuity.ts:361-366`
  - body: `sectionToList()` always returns an array, so the nullish coalescing here never reaches the `Blockers` or singular `Next Step` fallback. User-authored continuity docs using those headings parse as empty lists; choose the first non-empty parsed list instead.
  - confidence: 0.82
