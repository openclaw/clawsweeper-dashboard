Keep this open. Current main restores conversation history from the gateway, but queued messages and drafts are still in browser memory only, and the issue is paired with open PR #73154 using closing syntax.

## What This Changes

The issue requests WebChat to restore queued messages, conversation history, and draft input after a browser refresh.

## Best Possible Solution
