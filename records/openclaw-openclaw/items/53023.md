## What This Changes

Request configurable same-session lane concurrency, automatic yielding during long tool execution, and configurable lane-wait warning thresholds.

## Best Possible Solution

Keep this as the umbrella tracker while maintainers decide auto-yield semantics, review or replace https://github.com/openclaw/openclaw/pull/63864 for same-session concurrency, and leave https://github.com/openclaw/openclaw/issues/14747 as the warning-threshold slice unless intentionally folded into one design.

Unclear as one implementation. Session concurrency, lane warning thresholds, and auto-yield are separable changes, and auto-yield needs maintainer decisions around cancellation, transcript ordering, and session mutation safety.
