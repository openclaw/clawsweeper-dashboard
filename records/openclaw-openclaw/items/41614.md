  - body: `loadCronStore()` now merges runtime fields from the `jobs-state.json` sidecar, and `saveCronStore(..., { stateOnly: true })` can update that sidecar without touching `jobs.json`.   - confidence: 0.92

Capture freshness after reading the relevant files, or return the loaded store with the mtimes that correspond to it, so future checks stay conservative.
  - confidence: 0.74
