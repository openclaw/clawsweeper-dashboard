Keep open: current main still matches the three requested gaps, and the related canon-guardian plugin plus open PRs only cover parts of the requested native contract. Request native fallback-transition behavior that can notify users, re-inject workspace bootstrap context after a model fallback, and expose a plugin hook for fallback transitions.
