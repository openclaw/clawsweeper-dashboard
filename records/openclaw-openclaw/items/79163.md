Keep open. ## Best Possible Solution

Define one maintainer-approved fallback-transition contract: keep or adjust shipped notices through the visibility-policy issue, add a narrow plugin hook if transition events should be public, and decide whether continuation-skip re-injects bootstrap automatically or via an explicit context profile.
