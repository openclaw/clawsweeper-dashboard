## Summary

Keep open: current main still supports BlueBubbles `systemPrompt` only for groups, and #77086 is an open unmerged implementation candidate for the remaining DM-scoped config gap.

## What This Changes

Request BlueBubbles `direct.<chatGuidOrHandle>.systemPrompt`, with optional account-level prompt support, so DMs can receive channel-scoped prompt instructions.

## Best Possible Solution

Land or replace #77086 with BlueBubbles-owned `direct.*.systemPrompt` support, generated config metadata, docs, and regression tests; treat account-wide prompts as a deliberate follow-up unless maintainers include precedence rules now.

## Reproduction Assessment

Yes at source level: current main only reads `account.config.groups` for `GroupSystemPrompt` when `isGroup` is true, and the existing BlueBubbles test asserts DMs omit the prompt. I did not run tests because this review is read-only.

## Solution Assessment

Yes for the direct-DM option: mirroring WhatsApp and Telegram `direct.*.systemPrompt` is the narrowest maintainable fix. Account-wide prompt support should be included only with clear precedence rules or split into a separate follow-up.
