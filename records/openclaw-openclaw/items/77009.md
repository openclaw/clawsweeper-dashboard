## Summary

Keep open: current main still lacks BlueBubbles per-DM prompt config, and open PR #77086 is already the active implementation candidate for this issue.

## What This Changes

Add BlueBubbles `direct.<handle>.systemPrompt` and wildcard per-DM prompt support, with optional account-wide prompt support as a possible complement.

## Best Possible Solution

Review and land #77086, or an equivalent BlueBubbles-owned direct prompt implementation, while treating account-wide prompt support as a deliberate follow-up unless maintainers choose to include it now.

## Reproduction Assessment

Yes at source level: current main only resolves BlueBubbles `GroupSystemPrompt` from `groups` when `isGroup` is true, and the existing test suite asserts the DM omission. I did not run tests in this read-only review.

## Solution Assessment

Yes for the direct-DM path: mirroring the existing WhatsApp/Telegram `direct.*.systemPrompt` pattern is the narrowest maintainable fix. Account-wide prompts should be scoped separately unless maintainers want the broader config surface in the same change.
