Keep open: current main still only supports BlueBubbles `systemPrompt` for groups, while the requested direct-message prompt surface remains missing and is a narrow bundled-plugin feature gap.

Add BlueBubbles per-DM `systemPrompt` support, with optional account-wide prompt support as a follow-up or deliberate extension.

Implement BlueBubbles-owned `direct.<id>.systemPrompt` with `"*"` fallback, update docs/generated config metadata and regression tests, and treat account-wide prompts as a deliberate follow-up unless maintainers expand scope.
