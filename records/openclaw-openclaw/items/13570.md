## What This Changes

Add optional `agents.defaults.contextPruning.forcePruneRatio` so high estimated context usage can bypass the cache TTL, with schema and TTL-bypass test coverage.

## Best Possible Solution

Maintainers should choose the public pruning trigger API, then implement the selected additive surface across active hook settings/factory, schema/types/docs, changelog, and focused tests.
