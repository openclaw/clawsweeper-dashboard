Request adds an optional `agents.defaults.contextPruning.forcePruneRatio` threshold that lets high estimated context usage bypass the cache TTL, with schema and regression coverage.

## Best Possible Solution

Maintainers should choose the public pruning trigger API, then implement the selected additive config surface across settings, schema/generated docs, runtime TTL gating, docs, changelog, and focused tests.

## Solution Assessment
