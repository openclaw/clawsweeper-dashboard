## Summary

Keep open. Add a per-target subagent concurrency cap so a specific spawned agent can serialize while other subagents remain parallel.

## Best Possible Solution

Add a documented per-target subagent concurrency setting, likely under `agents.list[].subagents` if maintainers keep the existing namespace pattern, with schema/type/help/docs updates and native plus ACP enforcement alongside the global cap.

## Reproduction Assessment

## Solution Assessment
