## Summary

Keep open. Current main still hard-codes the 30 second completion-cache spawn timeout in both update and doctor paths, while this PR provides a focused configurable override with strict parsing, docs, tests, and sufficient after-fix terminal proof; no blocking code or security finding surfaced. ## Best Possible Solution
