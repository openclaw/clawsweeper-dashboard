## Summary

Keep open. Current main does not yet contain the Skill Workshop path, TOCTOU, and prompt-budget hardening, and the PR is still useful, but the latest head has two merge-blocking defects in adjacent proxy and bundled-facade code that should be fixed before landing.

## What This Changes

The PR hardens Skill Workshop writes with realpath containment, write-boundary revalidation, prompt-budget preview/enforcement, SDK exports, docs/tests, and related QA/plugin/proxy stabilization changes.

## Best Possible Solution

Land the Skill Workshop hardening after proxy activation fails closed when required Node HTTP hooks cannot load and bundled override comparisons use the same user-path normalization as bundled-dir resolution; then let normal exact-head checks decide whether the broad QA/CI stabilization can stay in this PR.

## Reproduction Assessment

Yes for the PR findings. A focused check can simulate unavailable global-agent hooks and a home-relative OPENCLAW_BUNDLED_PLUGINS_DIR override to exercise the two blocking paths.

## Solution Assessment

No, not yet. The plugin-owned Skill Workshop hardening direction fits the architecture, but the current patch should restore proxy fail-closed behavior and normalize bundled override paths before it is the narrowest maintainable solution.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.87

Full review comments:

- **[P2] Fail closed when proxy hooks are unavailable:** `src/infra/net/proxy/proxy-lifecycle.ts:255-258`
  - body: When global-agent cannot be resolved, bootstrapNodeHttpStack only logs and returns, but startProxy still pushes an active registration. That leaves Node http/https callers able to bypass the operator-managed proxy while the process reports proxy mode as active.
  - confidence: 0.86
- **[P2] Normalize bundled override paths before comparing:** `src/plugin-sdk/facade-loader.ts:149-156`
  - body: resolveBundledPluginsDir expands OPENCLAW_BUNDLED_PLUGINS_DIR through resolveUserPath, but this comparison uses path.resolve on the raw env value. A valid override such as ~/bundled will be treated as outside the override and skip runtime dependency staging for bundled facade loads.
  - confidence: 0.84
