## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.87

Full review comments:

- **[P2] Fail closed when proxy hooks are unavailable:** `src/infra/net/proxy/proxy-lifecycle.ts:255-258`
  - body: When global-agent cannot be resolved, `bootstrapNodeHttpStack` only logs and returns, but `startProxy` still registers the proxy as active. That can leave Node `http`/`https` callers bypassing the configured operator proxy while the process reports proxy mode as active; treat the missing bootstrap as an activation failure and restore env.
  - confidence: 0.87
- **[P2] Normalize bundled override paths before comparing:** `src/plugin-sdk/facade-loader.ts:149-156`
  - body: `resolveBundledPluginsDir` expands `OPENCLAW_BUNDLED_PLUGINS_DIR` through `resolveUserPath`, but this comparison uses `path.resolve` on the raw env value. A valid override such as `~/bundled` can be treated as outside the override and skip runtime dependency staging for bundled facade loads.
  - confidence: 0.84
- **[P3] Add the required changelog entry:** `docs/plugins/skill-workshop.md:114-120`
  - body: This PR changes user-visible Skill Workshop behavior and plugin SDK/runtime behavior, but `CHANGELOG.md` is not updated with a new entry under Unreleased. Repo policy requires a changelog entry for user-facing fix/feat/perf changes before merge.
  - confidence: 0.78
