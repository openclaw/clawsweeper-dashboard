## Best Possible Solution

Land a narrow macOS permission fix after signed-app diagnostics prove lower TCCAccessRequest counts and unchanged grant, revoke, and Settings refresh behavior; if freshness fails, use a delegate-owned shared location manager instead.

## Reproduction Assessment

## Solution Assessment
