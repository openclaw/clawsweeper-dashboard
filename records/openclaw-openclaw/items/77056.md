Request to reduce Control UI chat-turn sessions.list cost for large session stores by caching CLI-provider classification and bounding the chat sidebar refresh window.

## Best Possible Solution

Review and land #77057 or an equivalent narrow change that caps only the chat-turn session refresh and avoids repeated CLI-provider lookup while preserving explicit Sessions page and search behavior.
