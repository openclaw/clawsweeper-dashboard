## What This Changes

The issue requests that Mattermost `post_edited` WebSocket events be evaluated so @mentions added by editing a message can wake the agent.

Review, rebase, prove, and land https://github.com/openclaw/openclaw/pull/72513 or an equivalent Mattermost-only fix that routes edited posts through mention handling and uses edit-aware replay dedupe.
