  - body: `vi.stubGlobal("process", ...)` is not undone by `vi.restoreAllMocks()`, so later tests in the same worker can keep the mocked ARM process. Add `vi.unstubAllGlobals()` in teardown or avoid replacing `process`.

  - body: The low-profile grid still includes `sideStart` first, so a 2000px input starts with a 2000px encode before reaching 1200/1000/800. Clamp the first low-profile value to the intended ceiling so the profile actually reduces the expensive first pass.
  - confidence: 0.82
- **[P3] Validate systemd memory limit values before rendering:** `src/daemon/systemd-unit.ts:53-58`
  - body: The new `resourceLimits` strings are appended to the unit without the CR/LF checks used for description, args, env, and EnvironmentFile values. Keep these values closed over or validate them before rendering so future callers cannot inject extra unit lines.
  - confidence: 0.74
