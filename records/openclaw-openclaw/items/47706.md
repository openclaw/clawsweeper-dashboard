  - confidence: 0.82

  - body: The new `resourceLimits` strings are appended to the unit without the CR/LF checks used for description, args, env, and EnvironmentFile values. Keep these values as a closed set or validate them before rendering so future callers cannot inject extra unit lines.
  - confidence: 0.74
