## What This Changes

This PR records when a subagent completion announce last hit a transient parent-busy handoff and exempts that case from the cleanup and resume retry-count cap while preserving the hard expiry and non-pending failure cap.

## Best Possible Solution

Land the targeted retry-accounting fix after required checks are green and maintainers accept the bounded retry-semantics change, keeping the existing hard expiry and the retry cap for non-pending failures.

## Solution Assessment
