## What This Changes

The branch splits image-description setup and request timeout handling, preserves the full configured timeout for provider requests after setup, and updates docs/tests for phase-specific timeout behavior.

## Best Possible Solution

Land the focused fix if maintainers accept phase-level timeout semantics; otherwise keep the full provider-request timeout but add an explicit outer operation watchdog.
