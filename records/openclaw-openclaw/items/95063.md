## Best Possible Solution

Align the media gate with the text-path ingest hierarchy, carry an ingest-bypass mode through media failure handling, and prove it with focused tests plus redacted live Telegram evidence.

No. The PR touches the right helper, but the best fix should mirror the full text-path ingest hierarchy and preserve silent media failure handling for ingest-bypassed unaddressed media.
