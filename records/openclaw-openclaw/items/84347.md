Full review comments:

- **[P2] Remove the unsafe socat wrapper example:** `docs/channels/imessage.md:130`
  - body: This line labels `socat - EXEC:"ssh host imsg",pty,stderr` as safe, but socat's `stderr` option duplicates child stderr onto the output channel and `pty` enables terminal behavior. ([man7.org](https://man7.org/linux/man-pages/man1/socat.1.html))
