## Summary

Keep open. Current `main` has a real generic command-arg menu layer and uses it across Discord, Slack, and Telegram, but the specific `/model` flow this issue calls out still goes through a Discord-only picker path instead of the generic arg-menu pipeline. That means the request is only partially implemented, not fully satisfied.
