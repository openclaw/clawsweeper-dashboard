## Summary

Keep open. Current main still excludes `.md` from the built-in skill scanner, so the PR targets a real gap, but the proposed critical Markdown download-and-execute rule still misses common wrapper and interpreter forms. The existing review discussion flags the same security-sensitive weakness, so this should not merge as-is.

## What This Changes

The PR adds `.md`/`SKILL.md` scanning to `src/security/skill-scanner.ts` with Markdown-specific hidden Unicode, executable data URI, download-and-execute, base64, and hex payload rules plus scanner tests.

## Best Possible Solution

Keep this PR open as the active Markdown skill-scanning hardening candidate, but require a rebase and a stronger download-and-execute heuristic with regression tests for wrapped, absolute-path, versioned, and line-wrapped interpreter forms. Treat direct ClawHub skill-install scanning as a separate product/security decision rather than folding it into this patch.

## Reproduction Assessment

Yes. Current main reproduces the missing coverage by inspection because `.md` is absent from `SCANNABLE_EXTENSIONS` and the tests still expect Markdown to be non-scannable; the PR-specific false negative reproduces with a direct Node check against the proposed regex.

## Solution Assessment

No, not as-is. Adding Markdown-specific rules inside the existing scanner is the right narrow boundary, but the critical rule needs stronger matching or parsing plus focused regression tests before this is the best fix.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.89

Full review comments:

- **[P1] Match wrapped download-exec interpreters:** `src/security/skill-scanner.ts:239`
  - body: The new `markdown-download-exec` rule only recognizes a bare interpreter token immediately after the pipe. Common equivalent payloads such as `curl ... | sudo bash`, `curl ... | /bin/bash`, `curl ... | env bash`, `curl ... | python3`, or a backslash-newline before `| bash` return no finding, leaving an easy evasion path for the Markdown skill content this PR is meant to harden.
  - confidence: 0.91
