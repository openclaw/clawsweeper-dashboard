## What This Changes

The PR appends downgrade remediation hints to the newer-config warning emitted by `warnIfConfigFromFuture` in `src/config/io.ts`.

## Best Possible Solution

Revise or replace the PR with a shared downgrade-remediation hint used by valid newer-config warnings and invalid-config/snapshot diagnostics, while keeping doctor guidance precise about unknown-key stripping versus manual value-format repair.

## Reproduction Assessment

Yes by source inspection. A future-stamped config with unknown keys or changed value shapes fails validation in `loadConfig` or `readConfigSnapshot` before `warnIfConfigFromFuture` runs, matching the linked downgrade report.

## Solution Assessment

No. Updating only `warnIfConfigFromFuture` is too narrow because the affected downgrade cases can fail before that function runs; the maintainable fix should share remediation with invalid-config and snapshot diagnostics.
