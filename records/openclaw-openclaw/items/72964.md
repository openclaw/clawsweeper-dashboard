Keep the repair narrow: separate trusted active-context/prompt snapshots from provider billing/cache totals so anomalous cacheRead or usage.total cannot prematurely trigger destructive compaction, while preserving the #66520 requirement that real cached prompt context can still drive compaction when it represents the active prompt.
