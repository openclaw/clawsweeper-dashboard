The branch fans out heartbeat broadcasts, scopes `skipWhenBusy` to agent-owned lanes, appends `HEARTBEAT.md` directives across dispatch paths, adds LLM stream connect-timeout cleanup, and adds doctor/docs/tests/changelog coverage for missing heartbeat sessions.

## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Mostly yes. The PR changes the implicated heartbeat runner, prompt assembly, timeout wrapper, doctor warning, docs, and tests directly, but the new doctor warning should treat an omitted target as the documented `none` default before merge.
