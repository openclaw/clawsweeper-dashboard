## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Normalize new diary entries before joining capped blocks:** `extensions/memory-core/src/dreaming-narrative.ts:697-699`
  - body: `buildDiaryEntry()` already returns a block that starts with `---`, but the new capped append path pushes `entry.trim()` into `joinDiaryBlocks()`, which also prepends `---` to every block. After the first append to an existing diary this writes a doubled separator before the new entry, corrupting the diary block shape the splitter/dedupe helpers expect. Strip the separator or build the new block body before joining.
  - confidence: 0.92
