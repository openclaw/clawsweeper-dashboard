  - confidence: 0.88
- **[P2] Forward overrides through static catalog fallback:** `src/agents/embedded-agent-runner/model.ts:1567-1577`
  - body: `resolveModelAsync` forwards the new override fields through dynamic resolution, but the bundled static catalog fallback calls `applyConfiguredProviderOverrides` without `agentId` or `jobProviderRequest`. Runs resolved by that fallback can silently drop the new headers.
  - confidence: 0.82
