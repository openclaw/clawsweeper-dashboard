Full review comments:

- **[P1] Pass live usage tokens, not context windows:** `src/auto-reply/reply/commands-compact.ts:169`
  - body: `observedContextTokens` comes from `params.contextTokens` or persisted `contextTokens`, which are model/window budgets, but `currentTokenCount` is caller-observed live prompt usage and is reused as `tokensBefore` in compact diagnostics/checkpoints. Passing the window can report or persist model capacity as session usage.
  - confidence: 0.93

  - body: `Math.max(storedTotal, fileEstimate)` lets a rough JSONL bytes/4 estimate replace a valid stored token count whenever transcript JSON is bulky. Use an estimate only when the stored total is missing or explicitly proven stale.
  - confidence: 0.86
