- **[P1] Pass live usage tokens, not context windows:** `src/auto-reply/reply/commands-compact.ts:169`
  - body: `observedContextTokens` comes from `params.contextTokens` or persisted `contextTokens`, which are context-window budget values, but `currentTokenCount` is the caller-observed live prompt-token usage.   - confidence: 0.93

Use this only when the stored total is missing or known stale, or keep the canonical stored/session result so the context summary does not overstate usage.
  - confidence: 0.86
