- **[P1] Pass usage tokens, not context windows:** `src/auto-reply/reply/commands-compact.ts:169`
  - body: `observedContextTokens` is derived from `params.contextTokens` or `targetSessionEntry.contextTokens`, which are context-window/budget values, but `currentTokenCount` is the live prompt-token usage contract.   - confidence: 0.88
