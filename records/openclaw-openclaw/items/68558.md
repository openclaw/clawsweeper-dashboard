- **[P1] Pass live usage tokens, not context windows:** `src/auto-reply/reply/commands-compact.ts:169`
  - body: `observedContextTokens` is derived from `params.contextTokens` or persisted `contextTokens`, which are model context-window budgets. `currentTokenCount` is caller-observed live prompt-token usage and is reused as `tokensBefore`, so this can make small sessions report or checkpoint a huge before value.
