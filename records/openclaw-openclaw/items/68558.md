- **[P1] Pass usage tokens, not context windows:** `src/auto-reply/reply/commands-compact.ts:169`
  - body: `observedContextTokens` comes from `params.contextTokens` or `targetSessionEntry.contextTokens`, which are model context-window/budget values, but `currentTokenCount` is the live prompt-token usage contract.   - confidence: 0.91
