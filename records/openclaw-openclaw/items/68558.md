- **[P1] Pass real usage tokens to currentTokenCount:** `src/auto-reply/reply/commands-compact.ts:169`
  - body: `observedContextTokens` is derived from `params.contextTokens` or the session context budget, but `currentTokenCount` is defined as observed prompt usage and is reused as `tokensBefore` in diagnostics and checkpoints. Passing the model window here can make manual compaction replies and metadata report the budget as pre-compaction usage, so omit this field unless the command has a real live usage count.
  - confidence: 0.93

Use the file-size estimate only when no fresh stored total exists, otherwise transcripts with metadata or tool JSON can inflate the context line.
  - confidence: 0.88
