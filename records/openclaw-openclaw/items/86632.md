## Best Possible Solution

Review and either land or close the linked direct-exec visibility PR, then handle any `cmd` aliasing or automatic live-data-to-shell routing as separate security/product decisions with live Ollama/Qwen proof.

## Reproduction Assessment

No high-confidence live reproduction was run in this read-only review. Source inspection confirms the narrow mechanism: lean mode defaults to structured Tool Search, `exec` remains catalogable rather than directly visible in that path, and `cmd` is not accepted as an exec argument.

## Solution Assessment

Unclear as a complete solution. Keeping `exec` directly visible is a plausible narrow first fix because it exposes an existing capability, but `cmd` aliasing and automatic live-data-to-shell routing need separate maintainer security/product approval.
