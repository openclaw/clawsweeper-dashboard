## What This Changes

The issue reports that a local embedded Ollama/Qwen TUI session searches for a weather tool and stops instead of finding or using an available shell/curl execution path for live-data requests.

## Best Possible Solution

Keep this open for maintainer direction on the intended local embedded execution contract, then either make `exec` discoverable for this Tool Search class with focused regression coverage or improve the failure message when execution is unavailable or schema-incompatible.

## Reproduction Assessment

## Solution Assessment
