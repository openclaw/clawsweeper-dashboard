## Best Possible Solution

Land one canonical Matrix repair that forwards only explicit reasoning as `m.notice`, preserves raw and off-mode suppression, keeps reasoning out of TTS/audio accumulation, and includes live Matrix/Element proof or redacted runtime logs.

## Solution Assessment
