## Summary

Keep open. Current main still has no `whatsapp_create_group` tool or WhatsApp group-creation runtime path, and the discussion raises enough WhatsApp account-enforcement and outbound-invite risk that this needs maintainer product/security review before implementation.

## What This Changes

Add an agent-callable WhatsApp tool that creates a WhatsApp group from a group name and participant phone numbers.

## Best Possible Solution

If approved, add this inside the WhatsApp plugin as an explicitly enabled owner-controlled capability with participant validation, redacted audit logging, account-safety docs, focused tests, and a changelog entry; otherwise document agent-created WhatsApp groups as unsupported.

## Reproduction Assessment

Not applicable as a feature request. Current-main verification is high-confidence because source and docs searches show no `whatsapp_create_group`, no `groupCreate` wrapper, and WhatsApp agent-visible actions currently stop at login, polls, and reactions.

## Solution Assessment

Unclear. A direct ungated tool is not the narrowest maintainable solution until maintainers approve the product/security posture and required operator controls for creating groups and inviting phone numbers.
