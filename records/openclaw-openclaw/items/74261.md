Keep open. Current main still has no WhatsApp group-create tool or runtime seam, and the requested agent-controlled group/invite operation is a new security/product capability rather than a small fix; the follow-up discussion strengthens the account-safety concern.

Request to add a `whatsapp_create_group` agent tool with `groupName` and `participants` parameters for WhatsApp property-listing workflows.
