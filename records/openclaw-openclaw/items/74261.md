## Summary

Keep open. Current main still has no `whatsapp_create_group` tool or group-creation runtime path, and the discussion flags this as a high-risk WhatsApp identity operation that needs maintainer product/security review before implementation.

## What This Changes

Add an agent-callable WhatsApp tool that creates a WhatsApp group from a group name and participant phone numbers.

## Best Possible Solution

If approved, implement this inside the WhatsApp plugin as an explicitly enabled, owner-controlled capability with participant normalization, redacted logging, abuse/account-safety docs, focused tests, and a changelog entry; otherwise document agent-created WhatsApp groups as unsupported.

## Reproduction Assessment

Not applicable as a feature request. Current-main source and docs verification shows no `whatsapp_create_group`, no `groupCreate` wrapper, and no agent-visible WhatsApp group-creation action.

## Solution Assessment

No, not as proposed. A direct always-available Baileys-backed tool would skip the existing owner-only/action-gate pattern and the account-safety review needed for inviting phone numbers through a linked WhatsApp identity.
