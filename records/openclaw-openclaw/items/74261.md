## Summary

Keep open. Current main still lacks a WhatsApp group-creation tool or runtime method, and the issue discussion raises enough WhatsApp account-safety risk that this needs maintainer product/security judgment before implementation.

## What This Changes

Add a WhatsApp agent tool that creates a group with a name and participant phone numbers.

## Best Possible Solution

If approved, implement this inside the WhatsApp plugin behind an explicit operator-controlled capability with participant normalization, identifier redaction, outbound activity logging, focused tests, docs, and a changelog entry; otherwise document that agent-created WhatsApp groups are unsupported.

## Reproduction Assessment

Not applicable as a feature request. Current-main verification is high-confidence because source, config types, docs, and symbol searches show no `whatsapp_create_group` tool, `groupCreate` call, or active-listener group-creation method.

## Solution Assessment

Unclear. A Baileys-backed implementation belongs in the WhatsApp plugin if accepted, but the discussion makes account-safety policy and gating the first decision rather than a safe automated patch.

## Security Review

Status: needs_attention

Summary: The requested feature is security-sensitive because it would let an agent create WhatsApp groups and invite phone numbers through the operator’s linked account.

Concerns:

- **[medium] Gate agent-created WhatsApp group invites:**
  - body: Any implementation should require explicit operator enablement, validate and normalize participant numbers, redact group and phone identifiers in logs, and record outbound activity so one agent turn cannot silently invite people through the linked WhatsApp account.
  - confidence: 0.92
- **[medium] Review WhatsApp account enforcement risk:**
  - body: The discussion reports that repeated Baileys group creation can look abusive to WhatsApp, especially for fixed repeated listing groups, so maintainers should approve the production posture before exposing this as an agent tool.
  - confidence: 0.84
