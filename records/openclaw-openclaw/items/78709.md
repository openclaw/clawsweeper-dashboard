  - body: `dryRun` only guards the episodic delete, but the loop has already written semantic entries and the function still inserts a dream record. A dry run of the dream cycle will mutate `.openclaw/log-memory.db`, so preview callers cannot inspect consolidation safely.

  - body: The FTS query takes the top N matches across all layers/tags and only then filters in JavaScript. If those first text hits belong to another layer or tag, a filtered search can return no BM25 hit even though matching entries exist.
  - confidence: 0.86

  - body: The installed skill tells the agent to call existing `memory_search` and write `MEMORY.md`, but this PR stores log patterns in the new SQLite store and `KNOWLEDGE.md` without wiring those into `memory_search`. A real log-analyst run would not reliably retrieve the patterns this subsystem stores.
  - confidence: 0.78
