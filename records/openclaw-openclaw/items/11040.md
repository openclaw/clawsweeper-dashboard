## Summary

Keep open: current main still exposes only partial spawned-session lineage and does not implement the requested durable rootSessionKey/traceId session-chain contract. The direct implementation PR is closed unmerged, and the merged ACP lineage PR only serializes existing Gateway lineage fields.
