## What This Changes

The PR changes LINE auto-reply delivery and tests to batch rich/media/text into one Reply API call up to five messages, then push overflow.

## Best Possible Solution

Land the batching fix after real LINE bot/account proof or maintainer override, preserving the Reply API limit, overflow behavior, and quick-reply ordering coverage.

## Solution Assessment
