## What This Changes

The PR modifies `.github/workflows/ci.yml` to force Android CI for all non-docs runs and upload APK artifacts from the `build-play` Android matrix task.

## Best Possible Solution

Official release APK distribution should stay on #9443 and be implemented through release tooling after product/release approval.
