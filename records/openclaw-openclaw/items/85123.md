## Summary

## What This Changes

This PR adds Discord SDK facade send exports, installs an exact-specifier native resolver for `openclaw/plugin-sdk/*` and `@openclaw/plugin-sdk/*`, and adds regression coverage for external plugin `createRequire()` resolution without package-tree writes.

## Best Possible Solution

Land the narrow exact-SDK resolver and Discord facade exports only after plugin SDK/runtime owners accept the process-wide resolver contract and the release proof level for external plugins.
