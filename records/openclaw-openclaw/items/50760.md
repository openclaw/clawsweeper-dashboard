## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Use the task ledger as the board source:** `ui/src/ui/task-board/service.ts:23-45`
  - body: Current main exposes `tasks.list`/`TaskSummary` as the durable background-task contract, but this service only reads sessions and cron APIs. A general Task Board built from those secondary sources will miss ACP, subagent, CLI, and media task records and create a competing source of truth.
  - confidence: 0.91
- **[P2] Restrict waiting duration to waiting sessions:** `ui/src/ui/task-board/adapters/session-adapter.ts:137-141`
  - body: Failed, killed, and timed-out rows map to `status === "error"`, but this condition sets `waitingForSec` for every non-running status. Those cards would render an error blocker plus a misleading waiting duration; compute it only for actual waiting rows.
  - confidence: 0.94
- **[P2] Use next run time for never-run cron health:** `ui/src/ui/task-board/adapters/cron-adapter.ts:86-98`
  - body: For an enabled job with no recorded run, this falls back to `job.updatedAtMs`, so a daily or weekly job created more than an hour ago can show warning or stale even if its first `nextRunAtMs` is still in the future.
  - confidence: 0.88
- **[P2] Move Task Board strings into i18n:** `ui/src/ui/components/task-board/page.ts:43-47`
  - body: The new page and toolbar hard-code user-visible Chinese and English strings directly in components, bypassing the Control UI locale source of truth. Add source strings through the i18n catalog so other locales do not see mixed-language UI.
  - confidence: 0.9
- **[P2] Exclude session-targeted cron runs from active cards:** `ui/src/ui/task-board/adapters/session-adapter.ts:129`
  - body: This filter only removes cron-namespace session keys. Cron jobs can also target `current` or `session:<id>` sessions that reuse ordinary session keys, so the same scheduled work can appear in both lanes and be mislabeled as an independent active session.
  - confidence: 0.8
