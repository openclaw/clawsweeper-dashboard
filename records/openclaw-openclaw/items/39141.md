## What This Changes

The issue requests an opt-in gateway-side session activity watchdog and alerting layer built on the proposed `sessions.activity` contract.

## Best Possible Solution

Land the shared `sessions.activity` contract first, then add a gateway-owned opt-in watchdog that consumes it with privacy-safe alerts, configurable thresholds, dedupe/cooldowns, docs, and focused tests.
