## Summary

Keep open. Current `main` still lacks any security-audit acknowledgement/suppression config, still uses a hard `<=300B` heuristic for `models.small_params`, still warns on `hooks.allowRequestSessionKey=true` even when prefixes are configured, and still emits the loopback `gateway.trusted_proxies_missing` warning exactly as described.
