The branch adds managed npm install rollback snapshots for package metadata, lockfiles, and node_modules, preserves managed-root npm config scope, and expands installer regression coverage.

## Best Possible Solution

Merge the exact snapshot rollback if maintainers accept the availability cost; otherwise require a lighter rollback design that still atomically restores package metadata, lockfile, plugin files, and hoisted dependencies.

## Solution Assessment

Yes, with a maintainer tradeoff. Exact snapshot restore is the most reliable narrow fix for npm mutations across package metadata, lockfile, plugin files, and hoisted dependencies; a lighter design is only preferable if maintainers reject the I/O cost.
