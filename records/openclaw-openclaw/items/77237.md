## What This Changes

The PR snapshots the managed npm plugin root before npm-backed installs and restores package.json, package-lock.json, and node_modules when install validation rejects a candidate.

## Best Possible Solution

Land the snapshot-based rollback after maintainer review so rejected npm candidates restore the prior managed root without losing existing plugins or hoisted dependencies.

## Solution Assessment
