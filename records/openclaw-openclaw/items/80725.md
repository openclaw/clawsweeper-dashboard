The PR adds optional `deltaText` to Gateway chat delta payloads, carries the per-run state through cleanup paths, updates SDK projection to use it safely, and documents/tests the additive protocol field.

## Best Possible Solution

## Solution Assessment

Yes. The optional `deltaText` field preserves the cumulative `message` contract, and the latest SDK guard avoids using suffix-only deltas when replay has no previous snapshot.
