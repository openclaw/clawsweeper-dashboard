## What This Changes

The PR changes agent compaction so a non-abort mid-chain summarization failure preserves a marked partial summary, with focused Vitest coverage for the failure branches.

## Best Possible Solution

Land the focused compaction fallback after exact-head CI and maintainer review, keeping the incomplete-summary marker and the existing first-chunk, abort, timeout, and oversized-retry behavior.

## Reproduction Assessment

Yes. Source inspection shows current main can lose the accumulated rolling summary by making generateSummary succeed for an earlier chunk and reject after retries on a later chunk, after which summarizeWithFallback returns the generic fallback when no oversized retry succeeds.

## Solution Assessment

Yes. The patch fixes the implicated compaction loop directly, carries the partial summary through the existing fallback order, and keeps first-chunk, abort, and timeout cases out of partial-summary recovery.
