Keep open. Current main still lacks the requested Mission Control/Control UI File Gallery left-nav entry and workspace docs/images gallery, while the existing file surfaces are chat attachments and an agent Core Files editor scoped to bootstrap files.

## Best Possible Solution

Design a scoped workspace File Gallery surface with authenticated Gateway APIs constrained to the configured workspace, a Control UI tab/view, refresh and browser-open actions, docs/images filtering, unsafe-path tests, and user-facing docs/changelog if shipped.
