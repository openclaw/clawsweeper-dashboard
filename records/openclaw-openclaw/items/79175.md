Full review comments:

- **[P1] Keep discovered gateway selections usable:** `src/commands/onboard-remote.ts:91`
  - body: This changes discovered gateway option values from string indexes to beacon objects, but the handler still only assigns `selectedBeacon` for non-manual string selections. Because the prompter returns option values unchanged, choosing a discovered gateway now skips the beacon and falls through to manual URL entry.
  - confidence: 0.98
- **[P1] Remove duplicate locale keys:** `ui/src/i18n/locales/en.ts:1052`
  - body: `chat.welcome.startTalk` is defined twice in the new locale object, and the same duplicate exists in the changed zh-CN map. TypeScript rejects duplicate object-literal properties, so keep one key in each locale map before this can pass UI type/build checks.
  - confidence: 0.96
- **[P2] Bind translated Lit attributes as expressions:** `ui/src/ui/chat/run-controls.ts:28-29`
  - body: In Lit templates, `title={t(...)}` and `aria-label={t(...)}` are literal attribute text, not expression bindings. Use `${...}` bindings here and in the matching changed chat toolbar attributes so tooltips and assistive labels localize correctly.
  - confidence: 0.95
- **[P2] Keep welcome suggestions locale-neutral:** `ui/src/ui/chat/chat-welcome.ts:20-23`
  - body: `WELCOME_SUGGESTIONS` is now a Chinese literal array in shared source, so English and every other locale would receive Chinese quick prompts. Keep the source strings English or move the suggestions into the locale map before translating them.
  - confidence: 0.97
- **[P2] Preserve assistant-specific chat placeholders:** `ui/src/ui/views/chat.ts:811`
  - body: The current placeholder includes the active assistant name, but this replacement hard-codes the generic locale string to OpenClaw. Keep the assistant name as a translation parameter so multi-agent chats do not lose their context.
  - confidence: 0.92
- **[P2] Preserve current changelog entries:** `CHANGELOG.md:5-9`
  - body: The PR branch replaces current main's released changelog section with an `Unreleased` section and drops unrelated entries. Rebase and add this i18n entry without deleting existing release notes.
  - confidence: 0.94
