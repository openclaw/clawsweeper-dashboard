  - confidence: 0.99
- **[P1] Keep the Control UI locale source parseable:** `ui/src/i18n/locales/en.ts:10-11`
  - body: The English locale source now contains corrupted string literals such as `Loading鈥?` without closing quotes. That breaks the Control UI TypeScript source and blocks the locale generator/runtime from building.
  - confidence: 0.99
- **[P2] Preserve Bonjour gateway selection:** `src/commands/onboard-remote.ts:92-101`
  - body: The select options now return beacon objects, but the handler only assigns `selectedBeacon` for non-`manual` strings. Choosing a discovered gateway therefore skips the direct/SSH branch and falls through to manual URL entry.
  - confidence: 0.96
