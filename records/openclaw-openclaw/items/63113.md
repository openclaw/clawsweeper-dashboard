Full review comments:

- **[P1] Write config backups with owner-only permissions:** `scripts/dev/bootstrap-local-coding-agents.mjs:70`
  - body: openclaw.json may contain gateway tokens, provider refs, or other sensitive local settings, and current config backups are explicitly hardened to 0o600. This raw backup write has no mode/chmod, so permissive umasks can leave a secret-bearing backup readable by other local users.
  - confidence: 0.9
- **[P2] Parse OpenClaw config with JSON5-compatible IO:** `scripts/dev/bootstrap-local-coding-agents.mjs:60-62`
  - body: OpenClaw config accepts JSON5 comments and trailing commas, but this script uses JSON.parse. Existing valid configs will hard-fail even though current runtime and docs support them.
  - confidence: 0.94
- **[P2] Initialize missing config before reading:** `scripts/dev/bootstrap-local-coding-agents.mjs:19-20`
  - body: The documented bootstrap command first requires the config directory and then unconditionally reads openclaw.json. A clean state directory or first-run machine fails before bootstrapping anything, while current OpenClaw config loading treats a missing config as safe defaults.
  - confidence: 0.92
- **[P2] Preserve existing agent overrides:** `scripts/dev/bootstrap-local-coding-agents.mjs:132-135`
  - body: On rerun, the upsert spread replaces existing fields such as workspace, model, and skills with bootstrap defaults. That silently discards local customizations for the same agent IDs instead of only filling missing fields or warning.
  - confidence: 0.88

  - body: The bootstrap creates agents with openai-codex/gpt-5.3-codex-spark, but current main defaults OpenAI agent refs to openai/gpt-5.5 and doctor repairs legacy openai-codex/* refs. Freshly bootstrapped agents can fail until users manually change the model.
  - confidence: 0.9

  - body: The selftest creates .local-agent-selftest.* under the repo root. If the process is killed before the cleanup trap runs, untracked scratch files remain in the working tree and can be accidentally staged.
  - confidence: 0.88
