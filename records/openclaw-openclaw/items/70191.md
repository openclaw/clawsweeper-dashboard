## Summary

Keep open. Current main still lacks a native no-inference model-contract HTTP probe, and the downstream overlay is useful prior art rather than an OpenClaw implementation; the remaining work needs Gateway API and disclosure-policy decisions before code should land.

## What This Changes

Add a native Gateway `/health/deep`-style endpoint that reports active model, provider, fallbacks, uptime, and routing/error state without invoking inference.

## Best Possible Solution

Keep this as a maintainer Gateway design item and, if accepted, land an additive diagnostics/readiness contract with explicit auth, redaction, no-inference behavior, telemetry semantics, tests, and docs.
