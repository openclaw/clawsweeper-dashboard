Rename Chinese locale identifiers from `zh-CN`/`zh-TW` to `zh-Hans`/`zh-Hant` across UI, wizard, docs metadata, detection, and persisted locale values.

If maintainers approve the contract change, make `zh-Hans` and `zh-Hant` canonical across Control UI, wizard, docs publish metadata, docs, and tests while preserving `zh-CN`/`zh-TW` aliases, storage migration, and URL compatibility.
