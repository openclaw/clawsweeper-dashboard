Full review comments:

- **[P1] Preserve current exec preflight hardening on rebase:** `src/agents/bash-tools.exec.ts:64`
  - body: This PR routes script detection through the new `extractSandboxExecutionTargetFromCommand` helper. Current main has since grown splitShellArgs-based parsing, env-prefix stripping, Node preload handling, fail-closed complex invocation checks, yolo-host skipping, Windows path handling, and fs-safe pinned reads; compose the import-template check into that path instead of replacing it.
  - confidence: 0.9
- **[P1] Parse Node imports across multiline statements:** `src/agents/sandbox-execution-templates.ts:442-457`
  - body: `validateNodeImports` scans one trimmed line at a time, so standard forms like `import { execSync }` split across lines with `from "node:child_process"` are never matched. That lets denied runtime modules through the new sandbox preflight.
  - confidence: 0.93
- **[P1] Reject spaced dynamic require and import calls:** `src/agents/sandbox-execution-templates.ts:473-484`
  - body: The dynamic-call guards only look for `require(` and `import(` with no whitespace. Valid JavaScript such as `require (name)` or `import (name)` bypasses both literal matching and the dynamic import rejection, so denied modules can still reach execution.
  - confidence: 0.89
- **[P1] Validate allowed local helpers before trusting them:** `src/agents/sandbox-execution-templates.ts:409-416`
  - body: A relative local import returns as allowed once the target exists, but the helper file is not scanned. A sandboxed script can import `./helper.js` while that helper imports `node:child_process`, bypassing the advertised import boundary through transitive local code.
  - confidence: 0.84
- **[P2] Check bare relative Python import names:** `src/agents/sandbox-execution-templates.ts:287-290`
  - body: For `from . import subprocess`, the source specifier becomes empty after dot stripping and returns before checking denied import names. If a local wrapper with that name exists, the text-level deny policy is bypassed.
  - confidence: 0.78
