## Best Possible Solution

## Solution Assessment

Unclear until maintainer confirmation: the field-level `.pipe(z.string())` repairs are narrow, but switching the public base config schema to input mode is a compatibility-sensitive contract choice. The safer alternative is to keep output-mode stable unless maintainers explicitly want accepted-input public schemas.
