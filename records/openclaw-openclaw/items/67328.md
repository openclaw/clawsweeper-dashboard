## Reproduction Assessment

Yes. The current-main source path is clear: unpiped transforms feed `toJSONSchema({ unrepresentable: "any" })`, and Zod v4.4.3 turns unrepresentable transforms into empty schema branches while pipes resolve through the output schema.
