## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Yes, conditionally: using input-mode public schema plus typed transform outputs is a maintainable fix if maintainers confirm accepted config input is the public schema contract. The safer alternative is a narrower field-specific schema repair if output-mode compatibility must be preserved.
