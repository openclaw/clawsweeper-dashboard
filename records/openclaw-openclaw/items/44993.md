Keep open because current main and the latest release still return prompt text unchanged when it already contains `Current time:`, and an open closing PR already owns the narrow helper/test fix path.

## What This Changes

Heartbeat and cron prompt timestamps should refresh on every helper invocation instead of preserving an earlier `Current time` line.

## Best Possible Solution
