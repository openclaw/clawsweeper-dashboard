## Best Possible Solution

Keep the Matrix-local warning, but gate it on true accumulated populated roots, pass fields as logger metadata, cover both single-root no-warning and multi-root warning cases, and require contributor real behavior proof before merge.

## Solution Assessment

No. A Matrix-local warning is the right layer, but this patch should distinguish true multi-root accumulation from normal single-root reuse and should pass warning fields as logger metadata.
