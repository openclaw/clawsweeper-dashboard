Fix Docker-hosted Gateway sandboxing so `workspaceAccess` bind mounts use host-resolvable workspace paths instead of container-internal paths.

## Best Possible Solution

Keep this canonical issue open until maintainers choose and implement one secure contract: automatic host-source remapping, or Docker setup/doctor behavior that makes host-path parity explicit and verifiable.
