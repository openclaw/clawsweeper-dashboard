## Best Possible Solution

## Solution Assessment

Unclear as submitted. SQLite `snippet()` is a plausible narrow fix for exact FTS MATCH results, but the best mergeable solution must be ported to current `extensions/memory-core` code and either preserve or explicitly defer vector and fallback snippet behavior.
