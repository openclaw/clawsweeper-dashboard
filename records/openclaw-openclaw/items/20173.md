## Summary

Keep open. Request opt-in Discord MESSAGE_UPDATE handling so edited user messages can re-trigger agent processing with debounce and edit-aware behavior.

## Best Possible Solution

Implement this in the bundled Discord plugin with typed MESSAGE_UPDATE dispatch, content-change filtering, opt-in config/docs, per-message debounce, edit-aware replay/session handling, mention-gating coverage, and focused monitor tests while coordinating delete-to-cancel separately.

## Solution Assessment
