## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

No, not quite. The hook point is the narrow maintainable place to solve this, but the durable-signal bypass should not treat arbitrary lowercase `pass` as explicit PASS evidence because it overrides the new `Assistant: Need` rejection.
