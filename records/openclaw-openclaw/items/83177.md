## What This Changes

The issue asks for memory flush to estimate transcript tokens when providers such as MiniMax omit usage data so session totalTokens can recover.

## Best Possible Solution

Review and land or replace the linked PR with a guarded estimator fallback in runMemoryFlushIfNeeded plus focused regression coverage, then close this issue after merge.

## Reproduction Assessment

## Solution Assessment
