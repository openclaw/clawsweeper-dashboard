## Summary

## What This Changes

The issue requests a shared, session-agnostic Skill Workshop proposal index so proposals created by one agent session can be found from another.

## Best Possible Solution

Decide the canonical Skill Workshop visibility and authorization model, ideally alongside https://github.com/openclaw/openclaw/issues/74601, then update service filtering, Gateway RPCs, model-tool behavior, docs, and tests around that model.
