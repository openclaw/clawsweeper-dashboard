  - confidence: 0.88

  - confidence: 0.84

## Security Review

Status: needs_attention

Summary: Needs attention: the wrapper starts `bw` via PATH while passing Bitwarden env, bypassing OpenClaw's configured exec command path validation.

Concerns:

- **[medium] PATH-resolved Bitwarden CLI bypasses exec hardening:** `scripts/bw-exec-resolver.mjs:29`
