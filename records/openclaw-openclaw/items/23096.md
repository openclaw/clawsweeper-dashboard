## Security Review

Status: needs_attention

Summary: Needs attention: the wrapper passes Bitwarden environment variables to a PATH-resolved nested `bw` binary outside OpenClaw's exec command path validation.

Concerns:

- **[medium] PATH-resolved Bitwarden CLI bypasses exec hardening:** `scripts/bw-exec-resolver.mjs:29`
