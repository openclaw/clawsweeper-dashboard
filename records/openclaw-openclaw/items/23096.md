  - body: OpenClaw validates the configured exec resolver path before launch, but this wrapper then runs bare `bw` from PATH while the docs pass `BW_SESSION`. ## Security Review

Status: needs_attention

Summary: The PR is security-sensitive and needs command-path and credential-env hardening before merge.

- **[medium] Ambient PATH can choose the bw executable:** `scripts/bw-exec-resolver.mjs:29`
