## Summary

No current-main failing reproduction was established. Source, tests, the release tag, and the npm package show the reported off-only metadata path is fixed for the named OpenAI/OpenAI Codex models.

## Solution Assessment

Yes. Provider-owned thinking profiles plus reasoning-capable catalog rows are the maintainable fix, and the subagent path already carries the requested thinking level into the child session for shared validation.
