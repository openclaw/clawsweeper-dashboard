- **[P2] Fail on mid-stream OpenRouter errors:** `extensions/openrouter/music-generation-provider.ts:147-148`
  - body: OpenRouter can send a post-200 SSE event with a top-level error and choices[0].finish_reason set to "error". ([openrouter.ai](https://openrouter.ai/docs/api-reference/errors?utm_source=openai))
  - confidence: 0.88
