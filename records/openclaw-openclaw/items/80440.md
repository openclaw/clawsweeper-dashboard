Full review comments:

- **[P2] Fail on mid-stream OpenRouter errors:** `extensions/openrouter/music-generation-provider.ts:147-148`
  - body: OpenRouter documents mid-stream failures as SSE events with a top-level `error` and `finish_reason: "error"`. This loop only collects `delta.audio`, so a failure after some audio chunks can be returned as a successful partial track; check those error fields and throw the provider message before returning audio.
  - confidence: 0.88
