Full review comments:

- **[P2] Decode the joined audio payload once:** `extensions/openrouter/music-generation-provider.ts:176-177`
  - body: OpenRouter's streaming audio docs collect `delta.audio.data` chunks, join them into one base64 string, then decode. Decoding each SSE fragment independently corrupts valid streams whenever OpenRouter splits the base64 payload on a non-quad boundary, so accumulate the strings and call `Buffer.from(audioDataStrings.join(""), "base64")` once.
  - confidence: 0.95

  - confidence: 0.9

  - body: The intro now says OpenRouter is supported, but the same page's supported-provider table and capability matrix still list only ComfyUI, Google, and MiniMax. Add OpenRouter rows so users get consistent setup and capability guidance.
  - confidence: 0.86
