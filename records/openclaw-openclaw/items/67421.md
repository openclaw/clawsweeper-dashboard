  - body: This branch emits the named-agent private-network warning whenever the effective policy allows private-network access, even if that access came entirely from the global baseline.   - body: The cache key serializes `allowedHostnames` and `hostnameAllowlist` in input order, but enforcement treats those lists as normalized sets.
