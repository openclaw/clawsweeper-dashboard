## Summary

## What This Changes

The PR relaxes the Control UI tweakcn theme ID regex in `ui/src/ui/custom-theme.ts` from an 8-128 character total length to a 1-128 character total length.

## Best Possible Solution

Land the regex fix with an Unreleased/Fixes changelog entry and a focused custom-theme test for short tweakcn IDs, while leaving CSP/connect-src handling to the separate gateway CSP work.

## Reproduction Assessment

Yes. Current main’s regex rejects short IDs such as `twitter`, the importer routes raw IDs and tweakcn URLs through that regex, and the live tweakcn registry serves `twitter` as a valid theme payload.

## Solution Assessment

Yes for the code path. Relaxing the existing allowlisted ID validator is the narrowest maintainable fix, but the PR should add the required changelog entry and focused regression coverage before merge.
