## What This Changes

The issue requests that reasoning-only no-text assistant turns retry in group chats instead of being absorbed by the default silent-reply policy.

## Best Possible Solution

Maintainers should decide the group silent-reply contract, then either update the helper, tests, and group docs to retry reasoning-only turns visibly or keep the documented `NO_REPLY` behavior and close this as not planned.
