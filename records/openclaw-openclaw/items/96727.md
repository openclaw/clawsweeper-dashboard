The PR changes voice-call manager lookups to fall back to persisted call records and adds restore tests for completed-call lookup after eviction.

## Best Possible Solution

Keep active manager lookups active-only, add a newest-first persisted lookup only to the status/read surfaces that need historical calls, and require redacted after-fix status proof before merge.

## Solution Assessment
