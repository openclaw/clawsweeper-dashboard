## Summary

This PR adds `--include-archived` to `openclaw workboard list`, filters cards with `metadata.archivedAt` before status filtering for text and JSON output, and adds CLI tests for the default, opt-out, and JSON paths.

No. Filtering compact text output is the right boundary, but this PR also filters default JSON output; the safer candidate preserves the full-card JSON contract and updates docs.
