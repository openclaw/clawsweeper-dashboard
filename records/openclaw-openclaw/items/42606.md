## Summary

This PR should stay open because the hardening goal is still valid on current main, but the branch patches retired `src/browser/**` files instead of the active browser plugin route. The shipped route still lacks the requested document CSP, `nosniff`, and frame-deny headers, so the useful fix needs to be ported rather than closed.

## What This Changes

The PR adds a nonce-based CSP, `X-Content-Type-Options: nosniff`, and `X-Frame-Options: DENY` assertions for the noVNC bootstrap response under the old `src/browser/**` path.

## Best Possible Solution
