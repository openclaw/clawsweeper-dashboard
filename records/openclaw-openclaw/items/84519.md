The PR gates `SUB_CLI_DESCRIPTORS` through the private-QA visibility filter, centralizes the `qa` filter helper, and adds descriptor regression tests.

## Best Possible Solution

Land this focused descriptor-gating fix after normal CI so exported descriptors and helper surfaces share one private-QA visibility contract.
