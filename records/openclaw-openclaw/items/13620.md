## Summary

Keep open. Current main and the latest release still lack outbound Telegram `sendMediaGroup` album support, and there is an open closing fork PR plus a narrower related PR that maintainers should review or supersede instead of closing this feature request.

## What This Changes

Add Telegram album/media-group sending for the message tool so 2-10 related media items can be delivered as one grouped Telegram album with shared caption semantics.

## Best Possible Solution

Implement album support in the bundled Telegram plugin by using `sendMediaGroup` for eligible media groups while preserving existing sequential fallbacks and documenting the chosen message-tool API.

## Reproduction Assessment

Not applicable as a feature request. Source inspection on current main and v2026.5.20 gives high confidence that outbound Telegram media is still singular or sequential rather than grouped through `sendMediaGroup`.

## Solution Assessment

Yes on the requested direction, but the exact API shape and fallback policy need maintainer judgment. The maintainable solution belongs in the bundled Telegram plugin send/delivery surfaces using Telegram album semantics, not a core-only workaround.
