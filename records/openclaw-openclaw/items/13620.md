## What This Changes

Add outbound Telegram album sending for grouped photos and videos through the message and reply delivery paths.

## Best Possible Solution

Use the existing portable multi-media payload and add a grammY `sendMediaGroup` path inside the Telegram plugin for eligible 2–10 photo/video groups across all outbound entry points, preserving current single-media and unsupported-media fallbacks.

## Reproduction Assessment

Not applicable as a bug reproduction: this is a feature request, and source inspection conclusively shows that current outbound Telegram paths send media individually rather than as albums.

## Solution Assessment

Not yet. Telegram-native grouping belongs in the bundled plugin, but automatic grouping of existing `mediaUrls` appears narrower and more consistent than introducing a separate Telegram-specific message-tool action.
