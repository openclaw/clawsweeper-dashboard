## What This Changes

The issue requests Telegram media-group album support so multiple outbound photos/videos can be sent through `sendMediaGroup` with shared caption semantics.

## Best Possible Solution

Review, port, or supersede a current-main Telegram implementation that uses `sendMediaGroup` for eligible 2-10 photo/video groups while preserving existing single-media and non-groupable fallbacks.

## Reproduction Assessment

## Solution Assessment
