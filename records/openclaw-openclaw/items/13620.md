## Summary

Current main still does not expose outbound Telegram album/media-group sending, so this remains a valid feature request. The thread also has an open external implementation with closing syntax, which should be reviewed, ported, landed, or explicitly superseded before a separate repair lane is queued.

## What This Changes

Add Telegram outbound media-group album sending so the message tool can send multiple related photos or videos as one Telegram album with a shared caption.

## Best Possible Solution

Review the open external implementation and either land, port, or explicitly supersede Telegram plugin support for sendMediaGroup with focused tests and message-tool/Telegram docs.

## Reproduction Assessment

Not applicable as a feature request. Current-main source inspection gives high confidence that the message-tool, direct Telegram send, reply delivery, and outbound payload paths still send one media item at a time or sequentially.

## Solution Assessment

Yes on the requested direction, but the exact public API shape is still a maintainer decision. The maintainable fix belongs in the bundled Telegram plugin plus shared message-action schema, using sendMediaGroup only for eligible groups while preserving existing fallbacks.
