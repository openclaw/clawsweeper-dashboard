## Summary

Keep open. Current main and the latest release still lack outbound Telegram sendMediaGroup album support, and the open related PR only fixes sequential URL-media fan-out while explicitly leaving album semantics out of scope.

## What This Changes

Request support for sending Telegram media groups/albums from the message tool so 2-10 related photos or videos can be delivered with album semantics and a shared caption.

## Best Possible Solution

Implement album support in the bundled Telegram plugin using Telegram sendMediaGroup for eligible media groups, while preserving existing sequential fallbacks and documenting the chosen message-tool API.

## Reproduction Assessment

Not applicable as a feature request. Source inspection on current main and v2026.5.20 gives high confidence that outbound Telegram media is still singular or sequential rather than grouped through sendMediaGroup.

## Solution Assessment

Yes on the requested direction, but the exact API shape and fallback policy need maintainer judgment. The maintainable solution belongs in the bundled Telegram plugin send/delivery surfaces using Telegram album semantics, not a core-only workaround.
