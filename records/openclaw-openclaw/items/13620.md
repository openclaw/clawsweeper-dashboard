Keep open. ## What This Changes

Add Telegram album/media-group support so multiple related outbound media items can be sent via Telegram `sendMediaGroup` with a shared caption instead of separate messages.

## Best Possible Solution

Review the linked draft PR and either land, port, or supersede a current-main Telegram implementation that uses `sendMediaGroup` for eligible 2-10 media groups while preserving existing single-media and fallback behavior.

## Reproduction Assessment

Not applicable as a feature request. Source inspection on current main and `v2026.5.12` gives high confidence that outbound Telegram media is still sent sequentially rather than through a media-group send path.

## Solution Assessment

Yes on the requested direction, but the exact public API and fallback semantics need maintainer judgment. The maintainable path belongs in the bundled Telegram plugin delivery/send surfaces using grammY `sendMediaGroup`, not a core-only workaround.
