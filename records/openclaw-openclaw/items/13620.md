## Summary

Keep open: current main still lacks outbound Telegram media-group sending, and an open external PR with closing syntax is already the implementation candidate, so this should wait for maintainer review, porting, landing, or explicit superseding.

## What This Changes

Add a Telegram album/media-group send surface so the message tool and outbound Telegram delivery can group eligible 2-10 media items via `sendMediaGroup`.

## Best Possible Solution

Review the open external implementation and either land, port, or explicitly supersede Telegram plugin support for `sendMediaGroup` with focused tests and Telegram/message-tool documentation.

## Reproduction Assessment

Not applicable as a feature request. Source inspection on current main gives high confidence that message-tool, direct Telegram send, reply delivery, and outbound payload paths still send one media item at a time or sequentially.

## Solution Assessment

Yes on the requested direction, but the exact message-tool API shape is not settled. The narrow maintainable path is in the bundled Telegram plugin plus shared message-action schema, using `sendMediaGroup` only for eligible groups while preserving existing fallbacks.
