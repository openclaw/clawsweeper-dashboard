## What This Changes

Fix local JSON agent turns that emit their final response but fail to terminate the CLI and related workers promptly.

## Best Possible Solution

Propagate one-shot CLI lifecycle intent through local agent memory-tool construction, skip or cancel nonessential QMD synchronization for that invocation, and deterministically close embedding workers without forcing process exit.
