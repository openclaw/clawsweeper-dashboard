Keep open. Current main still lacks WhatsApp delete/revoke support, and https://github.com/openclaw/openclaw/pull/70813 is still open with closing syntax for this issue, so closing now would race the active implementation path.

## What This Changes

Add a WhatsApp message delete/revoke action so agents can retract their own sent messages through the message tool.

## Best Possible Solution
