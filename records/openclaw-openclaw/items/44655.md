## Best Possible Solution

Add a Z.AI-owned live catalog provider through the existing `registerModelCatalogProvider`/`liveCatalog` seam, preserving the static manifest fallback and covering endpoint selection, parsing, caching, and failure behavior with focused tests/docs.
