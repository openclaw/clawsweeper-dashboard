## What This Changes

Add authenticated dynamic Z.AI/Zhipu model discovery from the provider `/models` endpoint, with static fallback and custom-model handling for Coding Plan users.

## Best Possible Solution

Add a Z.AI-owned live catalog path that uses the authenticated provider endpoint for the selected region/plan, parses responses defensively, falls back to the static manifest catalog, preserves GLM-5 forward compatibility, and updates provider docs plus focused catalog tests.
