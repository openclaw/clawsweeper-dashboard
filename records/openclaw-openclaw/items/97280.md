## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

No as-is. The shared auth predicate is a plausible owner boundary, but the PR is too broad because it changes shipped mixed-auth API-key precedence instead of limiting OAuth restoration to explicit or OAuth-only audio cases.
