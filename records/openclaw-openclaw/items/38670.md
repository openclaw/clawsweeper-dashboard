## Best Possible Solution

Rebase onto current main, apply agent overlays only when hook dispatch has confirmed agent scope, keep lifecycle hooks governed by global policy, clarify or extend event matching semantics, regenerate config baseline artifacts, and add redacted real-run proof.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Gate agent policies on real agent scope:** `src/hooks/internal-hooks.ts:307-308`
  - body: `resolveEffectiveInternalHookPolicy` always reads `agents.defaults.hooks` before a real agent scope is confirmed.   - confidence: 0.91
