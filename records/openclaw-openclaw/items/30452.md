## Summary

Keep open: current main and v2026.5.20 implement the compaction and memory-flush model override pieces, but there is still no `resetAfterFlush` config contract or runtime path. The remaining work is a session lifecycle policy decision, not conservative cleanup.

## What This Changes

## Best Possible Solution
