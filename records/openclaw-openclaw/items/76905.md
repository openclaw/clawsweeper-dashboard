## What This Changes

The issue reports stale WhatsApp foreground final replies from an older tool-backed run being delivered after a newer inbound message in the same chat.

## Best Possible Solution

No high-confidence live reproduction was established in this read-only review. Source inspection shows a credible current-main path: newer inbound messages can steer an active run while the older buffered dispatcher still has no freshness check before final delivery.

## Solution Assessment
