## What This Changes

The issue reports stale WhatsApp foreground final replies from older tool-backed runs being delivered after newer inbound messages in the same chat.

## Best Possible Solution

Land or replace #76970 with a shared auto-reply freshness guard and regression coverage, then let the merged fix close this issue.
