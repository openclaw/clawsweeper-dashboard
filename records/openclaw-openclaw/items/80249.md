## What This Changes

The issue requests a backward-compatible Feishu TTS option to send visible text alongside native voice messages.

## Best Possible Solution

## Solution Assessment

Unclear. A Feishu-only boolean is plausible, but the open PR already shows fallback edge cases, and maintainers still need to choose Feishu-specific versus shared TTS text-delivery semantics.
