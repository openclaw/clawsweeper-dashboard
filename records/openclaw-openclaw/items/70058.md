## Summary

## What This Changes

The branch adds env-controlled chokidar polling knobs for the skills watcher with tests/docs, plus an experimental search plugin, labeler/lockfile entries, and a QA scenario assertion.

## Best Possible Solution

Land a narrowed polling fallback on top of current main while preserving the existing watcher root/filter/unlinkDir semantics, focused tests/docs, and a changelog entry; split or remove all unrelated plugin and QA changes.

Yes for the code-level regression: compare current main's watcher root/filter/unlinkDir implementation with the PR head's glob targets, DEFAULT_SKILLS_WATCH_IGNORED, and missing unlinkDir handler. The live filesystem mode that misses native events was not reproduced in this read-only review.
