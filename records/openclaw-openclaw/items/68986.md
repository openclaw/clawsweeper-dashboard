## Summary

Keep open: the reported leak class is still real on current main, but this draft PR is not merge-ready because it wires runtime behavior through stale `pi-embedded-*` paths while current main delivers through `embedded-agent-*`, and its shared sanitizer heuristics need owner review before they can rewrite final user-visible text.

## What This Changes

The PR adds `<channel|>` delimiter cleanup, repeated visible-suffix normalization, delivery-vs-stream sanitizer profiles, and regression tests around legacy pi embedded-agent delivery paths.

## Best Possible Solution

Port the useful leak regressions to current `embedded-agent-*` final-delivery paths, narrow shared output rewrites to confirmed leak evidence, and keep broader cross-channel sanitizer policy tied to https://github.com/openclaw/openclaw/issues/90684.

## Reproduction Assessment

## Solution Assessment
