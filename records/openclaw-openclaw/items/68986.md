The PR adds shared delivery-time assistant visible-text normalization for leaked `<channel|>` scaffolding and repeated visible output, then wires it into embedded agent stream/block delivery paths with regression tests.

## Best Possible Solution

Rebase and land a version that applies delivery-mode normalization to every final flush, including stream-item rollover, with focused coverage for literal marker text, intentional repeats, and rollover delivery.
