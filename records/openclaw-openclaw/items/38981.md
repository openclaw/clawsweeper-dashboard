Rebase the PR onto current main, preserve current bootstrap refresh semantics while validating cached entries against allowlist state, reject symlink aliases by actual alias status rather than lowercased path text, regenerate config metadata/help/labels, add the changelog entry, and land only after focused file/bootstrap/config checks plus maintainer security review.
