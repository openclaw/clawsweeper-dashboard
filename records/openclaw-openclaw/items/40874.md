## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Preserve newer working tasks when HUD timers expire:** `apps/ios/Sources/Model/NodeAppModel.swift:1753`
  - body: The auto-hide camera HUD task always calls `handleWorking(task: nil)` after its sleep. If screen recording starts before a prior camera HUD timer expires, the old timer can revert the Dynamic Island or lock-screen status while recording is still running; make cleanup conditional on the task it started.
  - confidence: 0.88
- **[P2] Refresh identity after gateway metadata changes:** `apps/ios/Sources/Model/NodeAppModel.swift:689-694`
  - body: `refreshIdentity` is only called from `setSelectedAgentId`, but `refreshAgentsFromGateway()` can also change the default agent, resolved agent name, and main session key after the activity starts. Since the widget renders activity attributes captured at request time, the label/session can stay on a fallback identity until a manual switch or reconnect.
  - confidence: 0.84
