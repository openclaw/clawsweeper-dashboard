## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Preserve newer working tasks when HUD timers expire:** `apps/ios/Sources/Model/NodeAppModel.swift:1753`
  - body: The auto-hide HUD task always calls `handleWorking(task: nil)` after its sleep, even if screen recording has since set a newer long-lived working task. Because `handleScreenRecordInvoke` does not cancel or own this camera HUD timer, a prior camera HUD can revert Dynamic Island or lock-screen status while recording is still running.
  - confidence: 0.88
- **[P2] Refresh identity after gateway metadata changes:** `apps/ios/Sources/Model/NodeAppModel.swift:689-694`
  - body: This refresh path only runs when the selected agent changes, but `refreshAgentsFromGateway()` can also change the default agent, resolved agent name, and main session key after the activity starts. Since ActivityKit attributes are immutable, the Live Activity label can stay on the fallback identity until a manual switch or reconnect.
  - confidence: 0.84
