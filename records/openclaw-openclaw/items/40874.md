## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Preserve newer working tasks when HUD timers expire:** `apps/ios/Sources/Model/NodeAppModel.swift:1753`
  - body: The auto-hide task clears the Live Activity with `handleWorking(task: nil)` after its sleep, regardless of which task is now active. Because screen recording sets its own long-lived working state without canceling this camera HUD timer, a prior camera HUD can revert Dynamic Island or lock-screen status while recording is still running.
  - confidence: 0.88
- **[P2] Refresh Live Activity identity after metadata loads:** `apps/ios/Sources/Model/NodeAppModel.swift:689-694`
  - body: `refreshAgentsFromGateway()` can change `gatewayDefaultAgentId`, `gatewayAgents`, and `mainSessionKey` after the activity is started with a fallback `activeAgentName`, but `refreshIdentity` is only called from `setSelectedAgentId`. Since ActivityKit attributes are immutable, the lock-screen/Dynamic Island agent label can stay on `Main` or a raw id until manual agent switch or reconnect.
  - confidence: 0.84
