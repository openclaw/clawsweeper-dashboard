## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Preserve newer Live Activity work when HUD timers expire:** `apps/ios/Sources/Model/NodeAppModel.swift:1753`
  - body: The camera HUD auto-hide task always calls `handleWorking(task: nil)` after its sleep. If screen recording or another operation starts before that old timer fires, the old HUD timer clears the newer Dynamic Island working state even though work is still active.
  - confidence: 0.88
- **[P2] Refresh Live Activity identity after gateway metadata loads:** `apps/ios/Sources/Model/NodeAppModel.swift:689-694`
  - body: This refresh path only runs when `setSelectedAgentId` changes user selection, but `refreshAgentsFromGateway()` can later change `gatewayDefaultAgentId`, agent names, and `mainSessionKey`. Because ActivityKit attributes are immutable, the Dynamic Island/lock-screen label can remain on the fallback identity after metadata loads.
  - confidence: 0.84
