## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.84

Full review comments:

- **[P2] Guard Live Activity cleanup by task ownership:** `apps/ios/Sources/Model/NodeAppModel.swift:1753`
  - body: The camera HUD auto-hide task unconditionally calls `handleWorking(task: nil)` after its sleep. If screen recording or another Live Activity task starts before that timer fires, the older HUD timer clears the newer Dynamic Island/lock-screen working state; clear only when the current task still matches the HUD task, or move ownership/token checks into `LiveActivityManager`.
  - confidence: 0.86
- **[P2] Refresh Live Activity labels after agent metadata updates:** `apps/ios/ActivityWidget/OpenClawLiveActivity.swift:56`
  - body: The widget displays immutable `context.attributes.agentName`, but the activity can be started before gateway agent metadata is refreshed and the PR only refreshes identity from explicit agent selection. A normal connection can leave the lock-screen/Dynamic Island label stuck at `Main` or a raw id until reconnect; refresh the activity when `refreshAgentsFromGateway` changes `activeAgentName` or `mainSessionKey`.
  - confidence: 0.8
