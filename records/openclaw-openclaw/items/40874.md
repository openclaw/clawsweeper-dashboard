## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Preserve active working tasks when old HUD timers expire:** `apps/ios/Sources/Model/NodeAppModel.swift:1753`
  - body: The new auto-hide cleanup always calls `handleWorking(task: nil)` after a camera HUD timeout. A screen recording or other task can start while that older timer is still pending, and when it fires it resets the Live Activity to idle even though the recording is still running. Cancel or guard the cleanup against the currently active task before clearing.
  - confidence: 0.88
- **[P2] Refresh the Live Activity after agent metadata loads:** `apps/ios/Sources/Model/NodeAppModel.swift:692-694`
  - body: `refreshIdentity` only runs from `setSelectedAgentId`, but `refreshAgentsFromGateway()` can change `gatewayDefaultAgentId`, `gatewayAgents`, `activeAgentName`, and `mainSessionKey` after the activity was started in the reconnect loop. Without a refresh there, the Dynamic Island can keep a fallback label such as `Main` or a raw id until the user switches agents.
  - confidence: 0.84
