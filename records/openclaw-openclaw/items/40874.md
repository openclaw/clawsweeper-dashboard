## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Refresh Live Activity after agent metadata loads:** `apps/ios/Sources/Model/NodeAppModel.swift:2212-2213`
  - body: This starts the Live Activity with `activeAgentName` before the operator connection refreshes branding and agents, and ActivityKit attributes cannot be changed in place. Since `refreshIdentity` is only called from `setSelectedAgentId`, the first connection can keep showing `Main` or a raw fallback even after gateway default-agent metadata arrives.
  - confidence: 0.87
- **[P2] Keep stale HUD timers from clearing active work:** `apps/ios/Sources/Model/NodeAppModel.swift:1753`
  - body: The auto-hide timer still clears Live Activity working state unconditionally. If a camera HUD timer is pending and screen recording starts before it fires, this line will reset `Recording screen...` back to the connection state while recording is still in progress.
  - confidence: 0.88
