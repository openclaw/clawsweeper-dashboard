## Summary

Keep this PR open: current main does not contain the Action Button/App Intent or Liquid Glass parts of the feature, and the PR still has two source-level Live Activity correctness blockers before it is mergeable.

## What This Changes

The PR adds iOS 26-gated Liquid Glass styling, an Open Talk Mode App Intent/shortcut, and richer ActivityKit Live Activity/Dynamic Island status UI to the iOS app.

## Best Possible Solution

Fix the deterministic Live Activity working-state and identity-refresh paths, then have the iOS owners decide whether the iOS 26 platform UX direction should land.

## Reproduction Assessment

Yes for the review findings by source inspection: a stale camera HUD timer can clear a newer screen-recording working state, and gateway metadata refresh can change the effective ActivityKit identity without recreating attributes. I did not run the iOS app.

## Solution Assessment

No, not as submitted. The Action Button path is now wired through RootCanvas, but the Live Activity task-clear and identity-refresh transitions need deterministic fixes before merge.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Preserve newer working tasks when HUD timers expire:** `apps/ios/Sources/Model/NodeAppModel.swift:1753`
  - body: The auto-hide task clears the Live Activity via `handleWorking(task: nil)` after its sleep, regardless of which task is now active. Because screen recording sets its own long-lived working state without canceling the camera HUD timer, a prior camera HUD can revert Dynamic Island or lock-screen status while recording is still running.
  - confidence: 0.88
- **[P2] Refresh Live Activity identity after gateway metadata loads:** `apps/ios/Sources/Model/NodeAppModel.swift:689-694`
  - body: `refreshAgentsFromGateway()` can change `gatewayDefaultAgentId`, `gatewayAgents`, and `mainSessionKey` after the activity is started with a fallback `activeAgentName`, but `refreshIdentity` is only called from `setSelectedAgentId`. Since ActivityKit attributes are immutable, the lock-screen/Dynamic Island agent label can stay on `Main` or a raw id until manual agent switch or reconnect.
  - confidence: 0.84
