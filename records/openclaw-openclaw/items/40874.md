## Review Findings

Full review comments:

- **[P2] Preserve newer working tasks when HUD timers expire:** `apps/ios/Sources/Model/NodeAppModel.swift:1753`
  - body: The camera HUD auto-hide task always calls `handleWorking(task: nil)` after its sleep. If screen recording or another operation starts before that old timer fires, the timer clears the Live Activity back to the connection state while the newer task is still active.
  - confidence: 0.88
- **[P2] Refresh Live Activity identity after metadata loads:** `apps/ios/Sources/Model/NodeAppModel.swift:689-694`
  - body: `refreshIdentity` only runs from `setSelectedAgentId`, but `refreshAgentsFromGateway()` can update the default agent, resolved name, and main session key after the Live Activity has started. The lock-screen/Dynamic Island label can therefore stay on a fallback name or raw id until a manual switch or reconnect.
  - confidence: 0.84
