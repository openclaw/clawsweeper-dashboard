## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Preserve active working tasks when old HUD timers expire:** `apps/ios/Sources/Model/NodeAppModel.swift:1753`
  - body: The camera HUD auto-hide task always calls `handleWorking(task: nil)` after its sleep. A screen recording or other long-lived task can start while that older timer is still pending, and when it fires the Live Activity is reset even though the newer task is active.
  - confidence: 0.88
- **[P2] Refresh Live Activity identity after agent metadata loads:** `apps/ios/Sources/Model/NodeAppModel.swift:692-694`
  - body: `refreshIdentity` only runs when the user changes the selected agent, but `refreshAgentsFromGateway()` can update the default agent, agent list, active name, and main session key after the activity has started. The lock-screen/Dynamic Island label can therefore stay on a fallback name or raw id until a manual agent switch.
  - confidence: 0.84
