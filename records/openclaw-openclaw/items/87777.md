## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.93

Full review comments:

- **[P1] Do not run workspace Python from Telegram taps:** `extensions/telegram/src/speakeasy-voice.ts:458-459`
  - body: When Speakeasy is enabled, this resolves the workspace and executes `scripts/tts_elevenlabs_v2.py` from that mutable workspace. Since the agent workspace is writable by normal OpenClaw work, an authorized Telegram tap can become gateway-side code execution with the OpenClaw process credentials; route this through a trusted TTS/provider seam or an explicitly approved trusted command outside the workspace.
  - confidence: 0.94
- **[P1] Write the Speakeasy cache with private permissions:** `extensions/telegram/src/speakeasy-voice.ts:153`
  - body: The cache stores exact assistant reply text for each visible voice button, but this `writeFileSync(..., "utf8")` creates the file with the process default mode, which was `0644` under a normal `022` umask in the local probe. Use a private mode like the Telegram ingress spool does so private DM content is not readable by other local users.
  - confidence: 0.91
- **[P1] Handle stdin pipe errors from failed TTS scripts:** `extensions/telegram/src/speakeasy-voice.ts:536`
  - body: `child.stdin.end(params.text)` has no `error` handler. If the Python sidecar exits before consuming a large reply, Node emits `EPIPE` on the stdin stream; the local Node 24 probe showed that without a listener this is an unhandled error that terminates the process instead of rejecting this voice generation.
  - confidence: 0.92
- **[P2] Recover stale Speakeasy cache locks:** `extensions/telegram/src/speakeasy-voice.ts:167`
  - body: The lock is a plain sidecar opened with `wx` and removed only by the returned release function. If the gateway exits while holding it, later cache updates hit the stale `.lock` until manual cleanup, suppressing new buttons and breaking existing taps after the timeout; use an existing file-lock helper or add safe stale-lock recovery.
  - confidence: 0.86
