## Solution Assessment

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.95

Full review comments:

- **[P1] Keep the checkout ref before the path separator:** `src/plugins/git-install.ts:290`
  - body: `git checkout --detach -- <ref>` does not detach at the ref; Git treats `<ref>` as a pathspec and rejects normal tags/branches with `--detach does not take a path argument`. This would make every `git:` plugin install or update that specifies a ref fail before npm install runs, so the separator approach needs to be replaced or split out.
  - confidence: 0.96

## Security Review
