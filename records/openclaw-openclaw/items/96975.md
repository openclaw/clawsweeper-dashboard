The issue requests a bounded, metadata-first subagent completion contract that keeps verbose child output out of parent context by default and leaves detailed inspection to explicit history or artifact reads.

## Best Possible Solution

Define a bounded subagent completion envelope in the agent/gateway contract, update docs and regression tests, and keep full child details available only through explicit bounded sessions_history or artifact reads.
