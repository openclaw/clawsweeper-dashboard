## What This Changes

The issue reports that Docker setup rejects `OPENCLAW_CONFIG_DIR` values containing whitespace on Windows/Git Bash.

## Best Possible Solution

Allow ordinary spaces in Docker config/workspace bind-mount source paths, safely serialize generated Compose mounts, preserve injection guards, and add focused regression coverage plus real setup proof.

## Solution Assessment
