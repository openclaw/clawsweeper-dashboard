Full review comments:

- **[P2] Disable rename on missing pinned chats:** `ui/src/ui/app-render.helpers.ts:674-682`
  - body: `resolvePinnedChatEntries` marks stale saved keys as `missing`, but the row still exposes Rename. Saving that rename calls `sessions.patch`, and the gateway patch path creates a new `SessionEntry` when no entry exists, so renaming a deleted/stale pin can resurrect a ghost session instead of leaving it missing or unpin-only.
  - confidence: 0.83
- **[P2] Keep rename drafts separate from disambiguated labels:** `ui/src/ui/app-render.helpers.ts:679-682`
  - body: Duplicate pinned labels are disambiguated by appending ` · <scope>` for display, but the Rename button seeds the edit draft with that rendered label. Saving or blurring without cleaning it up persists the UI-only suffix as the real session label through `sessions.patch`; keep a raw label for editing.
  - confidence: 0.81
