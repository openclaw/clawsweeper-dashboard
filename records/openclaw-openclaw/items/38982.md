- **[P2] Seed runtime theme from current detection:** `src/tui/theme/theme.ts:191-192`
  - body: Current main selects the light palette when `OPENCLAW_THEME=light` or `COLORFGBG` indicates a light background, but this branch initializes mutable runtime state from hard-coded `default`. Light-background users would be forced back to the dark palette until they manually switch themes.

  - confidence: 0.88

  - confidence: 0.86
