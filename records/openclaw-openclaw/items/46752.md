Rebase onto current main, move browser schema changes to the active plugin path, keep only contract-backed bounds, add focused regression coverage, and include redacted real gateway/model-call proof.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Allow node timeouts to cover media durations:** `src/agents/tools/nodes-tool.ts:110-116`
  - body: The schema caps `timeoutMs` at 120000 ms while the same tool permits media `durationMs` up to 300000 ms. Valid five-minute `camera_clip` or `screen_record` calls would be unable to request a gateway timeout long enough to finish.
  - confidence: 0.92
- **[P2] Keep sessions_list compatible with 200-row lookups:** `src/agents/tools/sessions-list-tool.ts:33-39`
  - body: Current main still calls `sessions.list` with `limit: 200` when resolving announce targets, but this patch adds `maximum: 50` to the public tool schema. Larger installs could fail to find older or less-active sessions that remain valid today.
  - confidence: 0.9
- **[P2] Preserve long-running session spawns:** `src/agents/tools/sessions-spawn-tool.ts:41-55`
  - body: The runtime accepts non-negative `runTimeoutSeconds` and `timeoutSeconds`, with `0` preserving no-timeout behavior. Capping both schema fields at one hour rejects legitimate longer subagent jobs.
  - confidence: 0.9
- **[P2] Restore browser snapshot limits to active backend capacity:** `src/agents/tools/browser-tool.schema.ts:98`
  - body: Current main's active browser tool schema lives under `extensions/browser`, and the inspected aria snapshot backends still clamp at 2000 nodes. A 1000-node maximum on the stale core schema path both misses the current source of truth and would reduce supported large snapshots after rebase.
  - confidence: 0.86
- **[P2] Permit Discord media channel type 16:** `src/agents/tools/message-tool.ts:329`
  - body: Discord runtime paths recognize `GuildMedia` as forum-like behavior, while this patch caps the numeric channel type at 15. That would reject supported channel-management calls for Discord media channels.
  - confidence: 0.84
