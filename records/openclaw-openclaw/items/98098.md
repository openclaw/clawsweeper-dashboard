Full review comments:

- **[P2] Use an overflow-signaling reader for OpenRouter success JSON:** `extensions/openrouter/oauth.ts:78-80`
  - body: This switches the OK OAuth response from `response.text()` to `readResponseTextLimited()`, but that helper is a diagnostic snippet reader: it truncates and returns text instead of reporting overflow.
