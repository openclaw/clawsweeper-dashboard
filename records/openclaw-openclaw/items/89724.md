- **[P1] Bind the Teams media socket to an explicit safe host:** `extensions/voice-call/src/msteams-media-stream.ts:149`

  - confidence: 0.9
- **[P1] Move Teams media startup and shutdown into runtime lifecycle:** `extensions/voice-call/src/providers/msteams.ts:125-127`
  - body: Starting the WebSocket server from the provider constructor and catching bind failures in the logger lets Gateway startup continue even when the Teams listener failed.   - confidence: 0.89

  - body: `port`, `path`, and `sharedSecret` are all optional and there is no msteams validation block, so `provider: "msteams"` can pass validation while the constructor only logs that no server started. Make the active provider fail validation until the required transport settings and SecretInput are configured.
