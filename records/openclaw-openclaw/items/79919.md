## What This Changes

The PR changes voice-call realtime WebSocket routing from raw prefix matching to exact-or-slash-child path matching and adds regression tests for sibling and root stream paths.

## Best Possible Solution

## Solution Assessment

Yes for the code direction: exact-or-slash-child matching is the narrow maintainable fix, and the second commit preserves the root stream-path child route. It is not merge-ready until real runtime proof is supplied.
