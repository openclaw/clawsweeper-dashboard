## Summary

Keep open. ## What This Changes

Request configurable WebChat/Control UI chat-input suggestions for project-specific terms to help typed and dictated input.

## Best Possible Solution

Define one accessible composer suggestion contract first, then implement custom-term matching with the chosen persistence, schema/docs, keyboard behavior, and coordination with @mention and proofreading work.
