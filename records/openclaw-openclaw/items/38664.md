  - body: `STAGED_FILES=$(git diff --cached --name-only -z ...)` uses command substitution, which Bash cannot preserve NUL bytes; the later `read -d ''` loop receives no records, so staged blobs are never copied and TruffleHog can scan an empty directory while secrets pass.

  - body: The Linux path downloads `install.sh` from `main` and the Go fallback uses `@latest`, so a security guide asks readers to execute moving code. Use a fixed release artifact with an expected checksum or a package-manager path that provides provenance.
  - confidence: 0.9

  - confidence: 0.93
