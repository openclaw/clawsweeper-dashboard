## What This Changes

The PR adds `--disallowedTools ScheduleWakeup,CronCreate` to the bundled Anthropic `claude-cli` fresh and resume argv defaults.

## Best Possible Solution

Treat Claude scheduler tools as unsupported for OpenClaw `claude-cli` print sessions unless maintainers choose to design a scheduler bridge; keep the default deny-list narrow and add coverage or docs only if maintainers want that guardrail locked down.
