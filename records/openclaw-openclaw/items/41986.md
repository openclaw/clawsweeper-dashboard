## Best Possible Solution

## Reproduction Assessment

Yes for the current-main aggregation mismatch: source inspection shows `sessions.usage` scans configured agents while `usage.cost` reads the unscoped/default cost cache. The exact MiniMax/dashscope dollar magnitude was not live-reproduced.

## Solution Assessment

Yes. The narrow maintainable fix is to make `usage.cost` resolve and merge the same configured-agent scope as `sessions.usage`; a frontend-only formatting change would not align the endpoints.
