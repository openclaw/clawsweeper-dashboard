## Summary

Keep open: current main does not implement WhatsApp forwarding, and the request is a new WhatsApp plugin capability with privacy, authorization, and action-contract choices.

## What This Changes

Add a WhatsApp message-tool `forward` action that forwards an existing message or media item to another WhatsApp chat by message id and target.

## Best Possible Solution

Design and implement a gated WhatsApp-owned forward action with authorization, target resolution, docs, config schema, and tests; update shared action typing only if maintainers want `forward` exposed as a general message action.
