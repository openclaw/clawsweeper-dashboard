## Summary

Keep open: current main does not implement WhatsApp forwarding, and adding cross-chat message/media forwarding is a new WhatsApp plugin capability with safety and contract choices.

## What This Changes

Request to add a WhatsApp `forward` action that forwards an existing message to another chat.

## Best Possible Solution

Design a gated `@openclaw/whatsapp` forward action, update shared action typing only if maintainers want it public across channels, and add docs plus regression coverage for authorization, target resolution, and media forwarding behavior.

## Solution Assessment
