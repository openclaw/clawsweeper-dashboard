## Summary

Keep open: current main still does not implement WhatsApp message forwarding, and the missing `forward` action requires WhatsApp plugin/API contract and privacy decisions rather than a repair to broken existing behavior.

## What This Changes

Add a WhatsApp channel `forward` action that forwards an existing message by `message_id` to a destination chat.

## Best Possible Solution

Design a gated WhatsApp-owned forward action, update the shared action contract only if maintainers want it public across channels, then document and test authorization, target resolution, message lookup, and media forwarding.
