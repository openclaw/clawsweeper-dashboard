## Summary

Keep open. Current main still matches the reported browser-tool timeout class: navigate exposes caller controls in the schema but drops them before the client and route layers, snapshot still lacks end-to-end timeout propagation, and the related open PR #75702 covers only the #72934 snapshot subset rather than this broader Linux CDP report.
