  - body: The new continuation config is advertised as hot-reloadable, and the cross-session gate/cost caps read it at enforcement time, but `agents.defaults.continuation.*` still falls under the no-op `agents` reload rule and `startGatewayConfigReloader` returns before `onHotReload`.   - confidence: 0.9

  - body: When more than one text payload contains bracket continuation syntax, this loop strips only the latest marker and leaves earlier `[[CONTINUE_DELEGATE: ...]]` text in the reply payload array. The winning signal may run a different delegate while the stale control marker remains visible to the channel, so all non-winning markers should be stripped or the reply payloads should be rebuilt before delivery.
  - confidence: 0.78
