  - body: markPendingWorkDelivered returns false when the TaskFlow row is missing or the expected-revision update loses the race, but this path still returns ran.   - confidence: 0.92
- **[P1] Keep legacy openai-codex out of live routing:** `extensions/codex/harness.ts:74-75`
  - body: The tokenized provider match accepts openai-codex/openai:codex at priority 100.   - confidence: 0.9
- **[P1] Do not trust cross-session returns by default:** `src/auto-reply/continuation/targeting.ts:113-116`
  - body: Cross-session delegate text is model-produced and can target other sessions when the gate is enabled, but enqueue marks it trusted.
