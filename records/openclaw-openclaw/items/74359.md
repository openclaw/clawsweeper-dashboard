## Review Findings

Overall correctness: patch is correct

Overall confidence: 0.88

Full review comments:

- **[P3] Limit ARG defaults to Docker global scope:** `src/docker-image-digests.test.ts:38-45`
  - body: `resolveArgDefaults` now collects ARG defaults from the entire Dockerfile, but Docker only lets `FROM` use ARGs declared before the first `FROM`. A future stage-local `ARG FOO=...` plus `FROM $FOO` would make this test resolve a value Docker would not resolve, so keep the resolver scoped to global ARGs or model Docker ARG scope explicitly.
  - confidence: 0.74
