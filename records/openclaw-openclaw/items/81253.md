## What This Changes

Request to add localized or operator-customizable templates for exec approval prompts, especially Telegram/native chat approval messages.

## Best Possible Solution

Define a shared typed approval presentation/localization contract that preserves mandatory safety fields, then let native channels render localized labels or approved layout presets from that contract.

## Reproduction Assessment

Yes for the feature gap: source inspection shows fixed English exec approval rendering and no Telegram/native locale or template config fields. This is not a broken-existing-behavior bug reproduction.

## Solution Assessment

No for the raw template sketch as-is: a free-form template could hide safety-critical fields. A typed shared approval presentation/i18n contract with mandatory fields is the safer maintainable direction.
