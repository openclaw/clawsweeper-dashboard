The issue requests localized or customizable exec approval prompt text, especially for Telegram/native chat approval messages.

## Best Possible Solution

Define a shared typed approval presentation/i18n contract that preserves mandatory safety fields, then let native channel renderers apply localized labels or approved layout presets.

## Solution Assessment
