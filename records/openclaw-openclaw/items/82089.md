## What This Changes

Request to add temporary Telegram tool-progress previews for selected native quote-reply turns while preserving final native quote replies and cleaning up the temporary message.

## Best Possible Solution

Keep this as the canonical Telegram product request until maintainers decide the temporary progress-lane UX, then implement it inside the Telegram plugin with focused cleanup/error tests and live Telegram proof.
