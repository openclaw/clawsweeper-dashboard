Add native JSONL ingestion to builtin memory search extra paths so raw message logs can be searched without Markdown conversion.

## Best Possible Solution

Adopt one canonical contract: either implement schema-aware JSONL admission, text extraction, line mapping, watching, retrieval, documentation, and focused tests across the builtin memory surface, or document session memory, QMD, or an external converter/plugin as the supported workflow.
