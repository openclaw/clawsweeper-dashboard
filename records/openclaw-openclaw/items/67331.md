Keep open: current main does not implement the standardized permission-denied model or payload guards, so the PR is not obsolete, but the branch should not merge as-is because source review still shows blocking behavior defects and missing contributor real-behavior proof.

## What This Changes

The PR adds a filesystem permission-denied helper, wraps apply_patch and sandbox filesystem boundary errors with structured reason metadata, and adds write/edit text payload size and NUL-byte guards.

## Best Possible Solution
