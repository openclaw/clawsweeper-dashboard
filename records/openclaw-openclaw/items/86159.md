## Summary

Keep this open: current main and the latest release still only observe, release lanes, or abort-drain stuck diagnostic sessions, while the requested steer/nudge/abort ladder would add new core recovery policy and configuration that needs maintainer design review.

## What This Changes

Request to add a configurable diagnostics recovery ladder that steers a stalled session, nudges it if still stalled, and abort-drains only as the final fallback.

## Best Possible Solution

Design an explicit opt-in diagnostics recovery policy that preserves today’s abort-drain default, reuses the existing session RPC semantics internally, and adds config, telemetry, docs, and tests for ladder ordering and delivery safety.
