## What This Changes

The branch adds content-sniffed Gateway image attachment detection, per-turn `imageModel` overrides and fallback candidates, auth-profile handling for temporary provider switches, and focused auto-reply/Gateway tests.

## Reproduction Assessment

Yes. Current main source and docs show a high-confidence source reproduction: `chat.send` routes image attachments based on session model capability without applying the documented `agents.defaults.imageModel` override for text-only primary models.

## Solution Assessment
