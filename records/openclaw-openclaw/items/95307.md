- **[P1] Use the existing unscoped sessions.list contract:** `packages/gateway-protocol/src/schema/sessions.ts:180`
  - body: Current `sessions.list` already returns all configured-agent rows when the caller omits `agentId` and sets `configuredAgentsOnly`, so adding `allAgents` reimplements existing behavior as a new gateway/Swift/TUI API.   - confidence: 0.88
- **[P2] Respect explicit agentId refreshes:** `ui/src/ui/controllers/sessions.ts:1134-1147`
  - body: When `state.sessionsAllAgents` is true, any later `loadSessions(state, { agentId })` call that does not also pass `allAgents: false` still sets `allAgents` from state and drops the explicit `agentId`. Existing compaction, patch, and delete refreshes pass scoped agent overrides this way, so enabling the toggle can silently widen refresh results that intended to stay agent-scoped.
  - confidence: 0.84
