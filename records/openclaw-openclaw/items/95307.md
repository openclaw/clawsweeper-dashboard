- **[P1] Keep checkpoint refreshes scoped:** `ui/src/ui/app-render.ts:3879`
  - body: This changes the checkpoint-entry refresh to use `createSidebarRecentSessionsLoadOverrides`, which returns no `agentId` when the sidebar All agents toggle is on.   - body: The gateway handler avoids forwarding `configuredAgentsOnly` into `loadCombinedSessionStoreForGateway` because that can skip unconfigured on-disk stores before configured-parent child rows are retained. Forwarding it here makes embedded TUI `sessions.list` diverge from gateway behavior for the same params.
  - confidence: 0.8
