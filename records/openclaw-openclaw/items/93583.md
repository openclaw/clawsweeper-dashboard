## Summary

## What This Changes

The branch changes the unmigrated legacy exec approvals fallback to read `~/.openclaw/exec-approvals.json` when `$OPENCLAW_STATE_DIR/exec-approvals.json` is absent, and adds a focused `resolveExecApprovals()` regression test.

## Best Possible Solution

Keep runtime resolution fail-closed until a canonical approvals file exists, and repair the startup/doctor migration path so valid legacy approvals are copied and archived before exec policy is evaluated.
