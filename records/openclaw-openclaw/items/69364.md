## Summary

Keep open. Current main still lacks the requested joined team topology/debugging surface, #52803 is adjacent rather than a duplicate, and the linked openclaw-studio prototype is useful external evidence but does not remove the core product/API/security decision.

## What This Changes

The issue requests a Control UI team/topology surface for multi-agent relationships across delegation, shared-workspace dataflow, task sequencing, config editing, tools/skills overrides, and per-agent chat.

## Best Possible Solution

Keep this open for maintainer product review; if accepted, start with a read-only topology MVP backed by an allowlisted Gateway contract, then split editing, tools/skills mutation, and chat controls into later security-reviewed follow-ups.

## Reproduction Assessment
