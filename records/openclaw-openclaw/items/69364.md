## Summary

Keep open. Current main still lacks the requested joined topology/debugging surface, and the follow-up discussion plus related #52803 make this a product/API/security direction question rather than a stale duplicate or narrow widget fix.

## What This Changes

Request a Control UI team/topology surface that joins agent delegation, dataflow, task sequence, status, and per-agent editing for multi-agent workflows.

## Best Possible Solution

Keep this open for maintainer product review; if accepted for core, start with a read-only, allowlisted topology contract and split editing, tools/skills mutation, and chat controls into later security-reviewed follow-ups.
