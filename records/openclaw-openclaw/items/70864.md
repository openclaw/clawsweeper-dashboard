## Summary

Keep open: the feature is not present on current main, and the latest PR head still has a blocking mention-gating regression where policy-denied regexes also make pattern-only channels think mention detection is unavailable.

## What This Changes

The PR adds scoped mention-pattern policy config and SDK helpers, wires the policy through bundled channel runtimes, updates schemas/docs/generated metadata, and adds regression coverage plus a changelog entry.

## Best Possible Solution

Keep the shared resolver and config surface, but separate policy-filtered regex evidence from mention-detection availability so deny policies cannot turn `requireMention` groups into process-all groups.

## Reproduction Assessment

Yes. Source-reproducible: configure a Signal, iMessage, or BlueBubbles group with `requireMention`, configured `mentionPatterns`, and a provider/global deny policy; the PR empties regexes and then lets `canDetectMention` become false, so unmentioned messages are not dropped.

## Solution Assessment

No, not yet. The shared policy resolver is a reasonable direction, but pattern-only callers need a detection-capability signal that is independent from the regex list allowed to count as mention evidence.

  - confidence: 0.9

  - body: This filters the configured regexes to `[]` before the handler derives `canDetectMention` from the filtered list. With `requireMention` enabled, an iMessage group whose patterns are denied now dispatches unmentioned messages; keep the policy-filtered match result separate from whether mention gating is configured.
  - confidence: 0.87
- **[P2] Preserve BlueBubbles mention gating when patterns are denied:** `extensions/bluebubbles/src/monitor-processing.ts:1090-1098`
  - body: The policy-disabled branch clears `mentionRegexes`, and the existing gate treats that as no mention detection. For BlueBubbles groups with `requireMention` and configured patterns, deny mode should prevent regex matches from satisfying the mention, not make every unmentioned message eligible for processing.
