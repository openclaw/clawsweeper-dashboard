## Summary

Keep open. Current main has useful hardening primitives, but it still lacks the requested native data-centric Security Profile v1.1; the live discussion has an explicit keep-open review, a same-author open filesystem-only implementation candidate, and ClawMoat as useful external prior art rather than a completed OpenClaw implementation.
