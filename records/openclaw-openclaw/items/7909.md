Keep open: current main and v2026.6.10 still expose only the Markdown copy action for Control UI assistant messages, and the remaining work needs a maintainer UX decision for the selector/preference shape and plain-text conversion contract.

## What This Changes

Add a Control UI option to copy assistant message content as plain text instead of only Markdown.

## Best Possible Solution

Keep Markdown copy as the default and add a compact secondary plain-text action with a narrowly defined, tested conversion contract; avoid a persisted preference unless product review chooses it.
