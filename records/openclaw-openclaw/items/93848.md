  - body: This branch sends absolute `MediaPath` values through `resolveUsableLocalCandidate`, which calls `realpathSync` and `statSync` before `ensureLocalStat` applies the inbound-root allowlist.   - confidence: 0.91

Current runner coverage expects partial batches to fall back instead, because prompts that refer to multiple images would otherwise receive an unlabeled subset with lost positions.
