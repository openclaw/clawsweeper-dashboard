  - body: This new absolute-path branch sends channel-supplied `MediaPath` values to `resolveUsableLocalCandidate`, whose helper calls `realpathSync` and `statSync` before the path has passed the existing inbound-root allowlist and safe-open flow.   - confidence: 0.91

Keep the all-or-nothing fallback or carry attachment indexes before forwarding partial batches.
