## What This Changes

The issue reports that `openclaw doctor --non-interactive` and `doctor --fix` can misclassify live `*.trajectory.jsonl` sidecar files as orphan primary transcripts.

## Best Possible Solution

Resolve #70812 first, then close this issue as implemented by the shipped trajectory-sidecar exclusion if no separate doctor coverage remains needed.
