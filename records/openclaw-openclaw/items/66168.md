  - body: This branch validates a single token and then trusts `x-openclaw-sender-is-owner`. Current main derives owner status from separate owner/non-owner bearer tokens, so a loopback caller must not be able to self-declare owner and unlock owner-only tools.

  - confidence: 0.91

  - body: The new `mcp.tools.call` WebSocket handler also calls `tool.execute` directly. A new gateway tool-execution surface needs the same before-tool-call protections and abort behavior as the HTTP loopback and `tools.invoke` paths.
  - confidence: 0.9
- **[P1] Scope resources to the requesting session:** `src/gateway/mcp-http.runtime.ts:77-78`
  - body: The cache syncs all active tool entries into one process-global registry, and list/read serve that registry without caller or session filtering. With file-backed resources, one session can enumerate or read another session's synced `ui://` resource.
  - confidence: 0.84
