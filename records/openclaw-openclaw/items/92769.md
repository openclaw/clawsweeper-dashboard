## What This Changes

The issue reports that MiniMax M3 via a custom OpenRouter provider drops returned `reasoning` and `reasoning_details` from subsequent assistant-message history for both plain-text and tool-call turns.

## Best Possible Solution

Review and land a narrow candidate PR that preserves the actual MiniMax `reasoning` and/or `reasoning_details` fields through both transcript policy and transport sanitization, with focused regression tests and redacted OpenRouter payload proof.
