## Summary

Keep open. Current main has task summaries, cancellation, artifact RPCs, subagent list/kill/steer controls, and internal flow resume helpers, but it still lacks the requested durable per-subagent lifecycle timeline, artifact-linked terminal payload, first-failure next-action diagnostic, public resume action, and audited idempotent supervision contract.

## What This Changes

Request deterministic observability and supervision controls for async subagent workflows, including lifecycle events, run summaries, artifact pointers, and cancel/steer/resume actions.

## Best Possible Solution

Design a versioned subagent/task observability contract that unifies durable lifecycle events, artifact metadata, terminal diagnostics, and audited idempotent cancel/steer/resume controls with the related run-trace and activity-timeline work.
