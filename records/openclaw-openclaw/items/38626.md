## Summary

Keep open. Current main has useful task-ledger, artifact lookup, subagent listing, and partial control helpers, but it still does not provide the requested durable per-subagent lifecycle timeline, artifact-linked terminal diagnostics, first-failure next action, public resume control, or audited idempotent supervision contract.

## What This Changes

Request to add deterministic observability and async supervision for subagent runs, including lifecycle events, summaries with artifacts, cancel/steer/resume controls, first-failure diagnostics, and audit logging.

## Best Possible Solution

Design one versioned observability and supervision contract that connects subagent lifecycle events, task/run trace storage, artifact metadata, terminal diagnostics, and audited cancel/steer/resume behavior across Gateway, SDK, CLI/chat, and operator surfaces.
