## Summary

Keep open. Current main has task records, task cancellation, artifact RPCs, and subagent list/kill/steer controls, but it still does not provide the requested durable per-subagent lifecycle timeline, artifact-linked completion payload, first-failure next-action diagnostic, public resume action, or audited idempotent supervision contract.

## What This Changes

The issue requests deterministic subagent lifecycle observability and async supervision controls, including event timelines, artifact pointers, failure diagnostics, and cancel/steer/resume actions.

## Best Possible Solution

Design a versioned subagent/task observability contract with durable event history, artifact-reference metadata, terminal diagnostics, and audited idempotent cancel/steer/resume operations, coordinated with the open run-trace and activity-timeline trackers before implementation.
