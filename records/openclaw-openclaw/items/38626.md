Keep open. Current main has partial task and subagent status surfaces, plus artifact lookup by task/run, but it still lacks the requested durable per-subagent event timeline, completion payload artifact pointers, first-failure next-action diagnostics, resume supervision action, and audited idempotent supervision contract.
