## Summary

Keep open. Current main has a useful task ledger, artifact lookup, state-change delivery, and subagent list/kill/steer controls, but it still lacks the requested durable per-subagent event timeline, artifact-pointer completion payload, first-failure diagnostics, public resume action, and audited idempotent supervision contract.

## What This Changes

The issue requests durable subagent lifecycle observability and deterministic async supervision controls for async subagent workflows.

## Best Possible Solution

Design a versioned subagent/task observability and supervision contract with durable event history, artifact-reference metadata, terminal diagnostics, and audited idempotent cancel/steer/resume operations.
