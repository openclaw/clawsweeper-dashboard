## Summary

Keep open: current main has useful subagent records, task summaries, artifact lookup, task cancellation, and internal kill/steer controls, but it still does not provide the requested durable per-subagent lifecycle receipt, failure/artifact timeline, public idempotent cancel/steer/resume contract, or supervision audit model.

Request deterministic subagent lifecycle event receipts, persisted run summaries with artifact and failure pointers, and idempotent supervision actions for cancel, steer, and resume.

## Best Possible Solution

Maintainers should define one core lifecycle receipt and supervision contract across the subagent registry, task ledger, artifact APIs, gateway authorization, and redaction rules before implementation.
