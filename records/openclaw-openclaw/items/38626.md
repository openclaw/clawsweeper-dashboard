## Summary

Keep open. Current main has partial task ledger, artifact lookup, subagent list/kill/steer, task state-change delivery, and task-flow internals, but it still lacks the requested durable per-subagent timeline, artifact-pointer completion payload, first-failure diagnostics, public resume supervision action, and audited idempotent supervision contract.

## What This Changes

The issue requests deterministic subagent lifecycle observability and async supervision controls for async child-run workflows.

## Best Possible Solution

Design a versioned subagent/task observability and supervision contract with durable event history, artifact-reference metadata, terminal diagnostics, and audited idempotent cancel/steer/resume operations.
