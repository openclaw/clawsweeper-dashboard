## Best Possible Solution

Put the suspicious-extension check in the shared path-param normalization/read wrapper so read/write/edit cover workspaceOnly true, false, and unset, then add focused integration tests for those paths and document any intentional fail-closed compatibility choice.

## Reproduction Assessment

## Solution Assessment
