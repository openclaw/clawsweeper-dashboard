## Best Possible Solution

Land the shared helper hardening after the draft/remote-gate path is complete, keeping unreadable schema handling centralized in src/shared/json-schema-defaults.ts rather than duplicating catches at each caller.

## Reproduction Assessment

## Solution Assessment
