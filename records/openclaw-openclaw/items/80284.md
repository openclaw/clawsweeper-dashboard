## What This Changes

## Best Possible Solution

Yes, source-reproducible: enable diagnostics-otel with an OTLP endpoint that returns a non-retryable response such as HTTP 410, trigger export, and current main has no plugin handler before the global fatal fallback. I did not run a live collector crash repro.

## Solution Assessment
