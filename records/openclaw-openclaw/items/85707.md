## Summary

## Best Possible Solution

## Solution Assessment

Likely yes if maintainers accept the policy exception. Exact-origin, loopback-only, DNS-pinned direct routing is narrower than a general proxy bypass; the safer alternative is preserving proxy routing by default and making direct local embeddings an explicit opt-in.
