Full review comments:

- **[P1] Preserve agent ownership for UUID store rows:** `src/gateway/session-utils.ts:643-644`
  - body: Returning a bare UUID here drops the owning agent while per-agent stores are merged, so later get/patch/delete/preview calls for that visible bare key can resolve against the current default agent instead of the store that owned the row. Keep enough agent ownership in the target or use a tested alias contract that round-trips the owning store.
  - confidence: 0.9
- **[P1] Use a header-capable WebSocket client:** `src/agents/pi-embedded-runner/codex-proxy-transport.ts:809`
  - body: This constructs globalThis.WebSocket with a headers option, but the OpenClaw Node runtime's global WebSocket does not expose the ws-style ClientOptions contract used elsewhere. The Authorization and chatgpt-account-id headers can be dropped, causing proxy Codex requests to fail auth or tenant routing.
  - confidence: 0.86
- **[P2] Resolve main-key aliases before UUID normalization:** `src/gateway/session-utils.ts:668-669`
  - body: The new UUID early return runs before configured main-key alias handling, so a valid session.mainKey that is UUID-shaped now routes to the bare UUID instead of the canonical main session key. That silently changes existing main-session routing for stored config.
  - confidence: 0.82
- **[P2] Normalize UUID links for any default agent:** `ui/src/ui/app-settings.ts:51-52`
  - body: This strips the prefix only for agent:main:<uuid>, but legacy Control UI links were generated with the configured default agent. Installations whose default agent was not main can keep prefixed UUID links and continue alias drift instead of converging on the shared UUID session.
  - confidence: 0.86
- **[P2] Avoid duplicating the system prompt in Codex payloads:** `src/agents/pi-embedded-runner/codex-proxy-transport.ts:722-723`
  - body: buildRequestBody sends instructions: context.systemPrompt while convertResponsesMessages already inserts the same prompt into the input messages. That duplicates prompt tokens and can change behavior or exhaust context earlier on long prompts.
  - confidence: 0.78
