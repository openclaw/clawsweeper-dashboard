## Best Possible Solution

Have maintainers explicitly choose the default routing policy: land the provider-local wrapper and docs if hosted Copilot search should be the default, or adjust the PR to preserve managed auto-detection ahead of native search or require an explicit opt-in.

## Solution Assessment
