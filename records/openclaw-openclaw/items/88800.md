## Best Possible Solution

## Solution Assessment

Yes, with maintainer acceptance. A string marker is a maintainable fit for the generated `models.json` runtime path because object SecretRefs are still treated as unresolved generated-file state, while the new marker can be resolved at request time and ignored by audit.
