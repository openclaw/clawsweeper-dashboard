## What This Changes

The PR changes `src/llm/providers/anthropic.ts` to compute the OAuth `claude-cli/*` user-agent from `claude --version` with a cached fallback instead of the hardcoded `2.1.75`.

## Best Possible Solution

Consolidate Claude Code user-agent version resolution across both Anthropic HTTP transports, avoid request-path process spawning where possible, and add focused coverage for the embedded gateway transport path before requiring redacted live OAuth proof.
