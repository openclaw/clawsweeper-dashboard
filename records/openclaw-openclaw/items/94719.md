## Summary

The PR replaces two hardcoded Anthropic OAuth `claude-cli/2.1.75` user-agent constants with a shared runtime resolver that reads `@anthropic-ai/claude-code/package.json`, then uses it in both legacy and managed Anthropic paths.

Land one canonical fix that updates both Anthropic OAuth header builders from a declared, cached or otherwise maintainer-approved version source, preserves supported setup-token behavior, and includes focused tests plus redacted OpenClaw runtime header proof.
