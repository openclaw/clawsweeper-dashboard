## Best Possible Solution

Land one complete selected reminder-guard UX: either a fully typed, documented, generated config opt-out with runtime coverage and live proof, or a preferred no-leak/matcher/cron-signal fix that prevents confusing user-visible note behavior without adding an unnecessary config knob.

## Solution Assessment
