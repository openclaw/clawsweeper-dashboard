## Best Possible Solution

Merge the minimal message-id handoff with a single-line changelog entry and a focused regression test proving followup delivery forwards `currentMessageId` under `replyToMode: "all"`, while keeping broader session-reset followup work in #52234 separate.

## Solution Assessment
