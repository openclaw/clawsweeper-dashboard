## What This Changes

The issue reports that long-response TTS summarization can speak the summary model's prompt echo and reasoning text instead of only the cleaned summary.

## Best Possible Solution

Review and land one narrow TTS sanitization PR that strips assistant-internal scaffolding before summary text reaches speech synthesis, adds focused regression coverage, and then close this issue with merged commit and release provenance.
