## Summary

Keep open. Current `main` still implements native plugins as unsandboxed, in-process code loaded via jiti into a shared registry, so the reported lateral-movement risk is not obsolete or already fixed. The related manifest-capabilities hardening work referenced from the issue is still not on `main`.

## Evidence
