- **[P1] Use the guarded live catalog fetch path:** `extensions/openai/discovery.ts:168-176`
  - body: This exported helper fetches an arbitrary `baseUrl` through the provided/global `fetchFn`, bypassing the existing live-catalog SSRF policy, timeout, abort, cleanup, and secret-marker handling.   - confidence: 0.9

  - body: The cache key only includes `baseUrl` and the last 8 characters of the API key, so two credentials or tenants with the same suffix can receive each other's cached model list for an hour.   - confidence: 0.86

  - body: `api.ts` is the OpenAI extension public contract, so exporting `resetOpenAIDiscoveryCacheForTest` makes a test-only implementation detail consumable by downstream code. Keep the reset local to tests or expose it from a test-support-only module instead.
