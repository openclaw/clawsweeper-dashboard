  - body: This helper defaults to raw `fetch` and sends `Authorization: Bearer ${apiKey}` to an endpoint derived from caller-controlled `baseUrl`.   - body: Caching by `apiKey.slice(-8)` lets different credentials for the same base URL share a catalog entry when their suffix matches.   - confidence: 0.88
