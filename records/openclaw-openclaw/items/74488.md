- **[P1] Route discovery through the guarded live catalog helper:** `extensions/openai/discovery.ts:168-181`
  - body: This helper defaults to raw `fetch` and sends `Authorization: Bearer ${apiKey}` to an arbitrary endpoint derived from `baseUrl`.   - body: Caching by only `apiKey.slice(-8)` means two different credentials for the same base URL can share a catalog entry when their suffix matches.   - confidence: 0.88
