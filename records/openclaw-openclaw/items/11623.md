## Summary

Keep open. Current main still has adjacent macOS menu-bar HUD, WebChat, Talk overlay, Canvas, work-activity, and session-menu surfaces, but not persistent configurable per-agent floating bubbles, and prior issue discussion already reached the same keep-open conclusion.
