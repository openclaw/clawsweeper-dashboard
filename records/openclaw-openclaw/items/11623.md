## Summary

Keep open. Current main still has adjacent macOS HUD, WebChat, Canvas, Talk overlay, and session/status indicator work, but it does not implement configurable persistent per-agent floating bubbles with drag/anchor or mouse-follow behavior and per-agent chat routing; the existing issue discussion reached the same conclusion.
