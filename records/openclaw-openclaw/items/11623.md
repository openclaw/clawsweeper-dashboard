## Summary

Keep open. Current main has adjacent macOS menu-bar HUD, WebChat, Canvas, Talk orb, activity, and session-menu surfaces, but not configurable persistent per-agent floating bubbles; the prior issue discussion reached the same keep-open conclusion.

## What This Changes

Request to ship macOS floating per-agent bubbles with configurable labels, persisted positions, mouse-follow or anchor behavior, and click-to-chat routing.

## Best Possible Solution

Keep this open for a maintainer-designed native macOS feature or a generic native UI/window plugin API that maps agent/session state to persistent configurable per-agent bubbles and routes each bubble to the selected agent chat.
