## What This Changes

Add configurable always-on-top macOS bubbles for running agents, including persisted positioning, mouse-follow or anchoring, and per-agent chat access.

## Best Possible Solution

Design the feature as a native macOS surface backed by canonical agent/session activity and canonical persisted settings, with one explicit routing model for selecting an agent; if maintainers do not want the UX bundled, first define a safe native UI/window extension API that makes an external implementation viable.
