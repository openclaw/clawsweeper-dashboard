The issue requests first-class config fields for appending agent-level system prompt rules from `openclaw.json`, including default and per-agent variants plus optional include-file support.

Settle the canonical prompt-injection boundary first; if core config is accepted, implement schema, types, docs, metadata, gateway mutation policy, and session-creation resolution across main, cron, subagent, and channel-triggered runs.
