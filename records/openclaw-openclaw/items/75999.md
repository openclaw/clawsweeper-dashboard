Keep open. ## What This Changes

The issue reports a v2026.4.29 regression where embedded agent dispatch prep performs tens of seconds of synchronous CPU work before the model starts streaming.

## Best Possible Solution

Add measured amortization for stable embedded-run prep inputs, especially tool/plugin surfaces, system-prompt stable sections, and provider stream setup, while keeping channel/session/provider-dynamic context below the cache boundary and validating with timing traces.
