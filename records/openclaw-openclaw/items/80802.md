- **[P2] Avoid restarting playback during bridge teardown:** `apps/shared/OpenClawKit/Sources/OpenClawKit/CanvasRealtimeTalkBridge.swift:643`

  - confidence: 0.87

  - body: `INSTALLED_APP` comes from `OPENCLAW_APP_BUNDLE`, but the script deletes it with `rm -rf` before copying the staged app. A typo or poisoned environment can recursively remove an unrelated directory through the new Run action; reject unsafe targets before deletion.
  - confidence: 0.86
