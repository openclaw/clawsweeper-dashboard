- **[P2] Avoid restarting playback during bridge teardown:** `apps/shared/OpenClawKit/Sources/OpenClawKit/CanvasRealtimeTalkBridge.swift:643`
  - body: `stopInternally` calls `clearOutput()` before clearing `self.session`, and `clearOutput()` starts a new playback stream whenever `session.outputSampleRateHz` is still present.   - body: This adds `node.canvas.capability.refresh` to advertised gateway methods, but the PR head still has no handler for that method, so a client that discovers it will get `unknown method` when calling it.   - confidence: 0.87

  - body: `INSTALLED_APP` comes from `OPENCLAW_APP_BUNDLE`, but the script deletes it with `rm -rf` before copying the staged app. A typo or poisoned environment can recursively remove an unrelated directory through the new Run action; reject unsafe targets before deletion.
  - confidence: 0.86
