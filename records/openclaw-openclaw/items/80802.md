Overall confidence: 0.88

Full review comments:

- **[P2] Avoid restarting playback during bridge teardown:** `apps/shared/OpenClawKit/Sources/OpenClawKit/CanvasRealtimeTalkBridge.swift:643`
  - body: `stopInternally` calls `clearOutput()` while `self.session` is still non-nil, and `clearOutput` starts a new playback stream whenever `session.outputSampleRateHz` exists. Stopping or erroring the bridge can therefore leave a fresh playback handle/continuation alive after teardown; clear the session first or add a stop-only clear path.
  - confidence: 0.89
- **[P2] Stop advertising the removed Canvas capability method:** `src/gateway/server-methods-list.ts:163`
  - body: This re-adds `node.canvas.capability.refresh` to the advertised gateway methods, but the PR head still has no core handler for that method, so a client that discovers it gets `unknown method` when calling it. Current protocol docs say this deprecated path is unsupported; use `node.pluginSurface.refresh` or add a complete compatible handler and contract update.
  - confidence: 0.84
- **[P2] Validate the app install path before deleting it:** `script/build_and_run.sh:47`
  - body: The run script deletes `$OPENCLAW_APP_BUNDLE` before copying the staged app, but that env override can point anywhere. A typo or poisoned environment can recursively remove an unrelated directory; guard that the target is an expected `OpenClaw.app` bundle path before `rm -rf`.
  - confidence: 0.82
