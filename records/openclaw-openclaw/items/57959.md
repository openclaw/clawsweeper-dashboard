  - body: `runWithModelFallback` now forwards `described.partialExecution`, but the PR does not populate that field at any production `FailoverError` throw site.   - confidence: 0.9

  - body: `resolveRetryImages` strips explicit images for same-provider `format` failures, but `suppressPromptImageDetection` is only true for cross-provider retries. A format retry can still re-load local image references from the prompt and send them to the fallback model, defeating the strip path that is supposed to recover from image-format failures.
  - confidence: 0.86
