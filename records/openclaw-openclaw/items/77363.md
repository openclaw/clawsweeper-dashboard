## Summary

Keep open: current main still sends the workspace bootstrap only through Codex `config.instructions`, while `developerInstructions` is finalized before the bootstrap block is built. The open closing PR #77372 appears to target the right narrow fix, so this issue should stay open until that PR lands or is closed.

## What This Changes

Workspace bootstrap persona files should be forwarded to Codex app-server `developerInstructions` in addition to `config.instructions`.

## Best Possible Solution

Land a narrow fix like #77372 that builds the workspace bootstrap before finalizing app-server developer instructions, preserves the existing config merge and AGENTS.md exclusion, and adds baseline plus context-engine regression coverage.

## Reproduction Assessment

Yes. Source inspection on current main shows `promptBuild.developerInstructions` is computed before `workspaceBootstrapInstructions` is loaded, and the existing SOUL.md regression test only verifies `config.instructions`.

## Solution Assessment

Yes. The narrow maintainable fix is to add the already-rendered workspace bootstrap block to Codex `developerInstructions` in both baseline and context-engine paths without changing bootstrap selection, AGENTS.md native handling, or the existing config path.
