## Summary

Keep open. Current main still lacks a public mutating per-LLM-call context hook, and loop-time `assemble()` remains limited to `ownsCompaction` engines; the related additive hook PR was closed unmerged, so the requested API/semantic change still needs maintainer design review.

## What This Changes

Request that context engines or plugins be able to rewrite or assemble the full message array before each LLM call inside the agentic loop.

## Best Possible Solution

Define a documented per-model-call context rewrite contract that covers hook ordering, message validation, raw conversation access, compaction ownership, and regression coverage for embedded-runner plugin hooks.

## Reproduction Assessment

Yes. Source inspection on current main shows setup-time `assembleAttemptContextEngine()`, the `ownsCompaction`-gated private loop wrapper, and no public mutating `transform_context` hook.

## Solution Assessment

Unclear. An additive hook like the closed related PR appears safer than changing `assemble()` timing for all engines, but the API shape and trust model need maintainer review first.
