## What This Changes

The issue requests that ContextEngine.assemble participate before each LLM call rather than only once per user prompt.

Close this issue as fixed by the shipped owning-context-engine loop hook; keep adjacent metadata contract work in issue 82137.

Yes: the shipped hook keeps the API centered on ContextEngine ownership of compaction instead of adding the raw transform_context hook requested in older related discussion.
