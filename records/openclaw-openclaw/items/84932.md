The PR removes the `?? 512` fallback from media-understanding image descriptions so the shared 4096-token helper default applies, and updates/adds tests for the default and model-cap behavior.

## Best Possible Solution

Land the focused fix after maintainers accept the higher default budget tradeoff, and track any future operator-facing token-budget control as a separate typed config/docs change.
