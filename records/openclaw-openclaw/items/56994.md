Do not refactor the already-fixed `saveJsonFile` or restart-sentinel paths except to adjust tests if needed. For exec approvals, preserve `assertSafeExecApprovalsDestination`, symlink-parent rejection, hard-link replacement behavior, Windows EPERM/EEXIST fallback protections, JSON formatting with trailing newline, `0o600` file mode, `0o700` directory mode, and temp cleanup; prefer `replaceFileAtomicSync` or an equally durable sync helper with `syncTempFile` and `syncParentDir`.
