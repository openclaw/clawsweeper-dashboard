## What This Changes

Fix Claude CLI command spawning on Windows where npm-installed command shims fail with ENOENT.

## Best Possible Solution

Keep https://github.com/openclaw/openclaw/issues/91489 as the single implementation tracker for a shared, safely tested Windows shim-resolution and batch-launch correction, incorporating this report's Windows 11 reproduction details.
