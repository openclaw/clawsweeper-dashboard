## What This Changes

The issue requests a global `openclaw agent --dry-run` mode that lets the model see tools while returning preview results instead of executing tool calls.

## Best Possible Solution

Define a request-scoped agent execution-mode contract, then implement dry-run at every canonical tool execution/delegation boundary with tests for built-in, plugin/channel, client-hosted, Gateway, hook-order, and latest-release upgrade behavior.

## Solution Assessment
