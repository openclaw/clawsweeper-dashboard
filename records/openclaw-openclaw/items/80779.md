## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Yes, with a rebase. Improving the CLI fallback diagnostic is the narrow maintainable fix for this symptom; changing request-id reconciliation or auto-approving a different pending id would alter approval semantics and carry more operator-safety risk.
