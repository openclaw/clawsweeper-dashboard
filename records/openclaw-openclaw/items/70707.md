## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P1] Recheck invalidation before handler dispatch:** `src/gateway/server/ws-connection/message-handler.ts:1799-1807`
  - body: This guard runs before the async IIFE that imports and calls handleGatewayRequest. A request can pass the early check, then another pipelined request can rotate or revoke the token and mark the same client invalidated before the first request resumes, so the first request can still reach the handler. Add the same invalidation close path immediately before handleGatewayRequest and cover that interleaving.
  - confidence: 0.84

As merged with current main, the test helper will fail type checking until it matches the exported param type.
