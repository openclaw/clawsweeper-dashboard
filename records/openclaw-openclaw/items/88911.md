- **[P1] Preserve DM pairing sends that carry chat ids:** `extensions/imessage/src/send.ts:1065-1070`
  - body: The pairing challenge still calls `sendMessageIMessage(sender, ..., { chatId })` for direct-message notifications that include a chat id, but this new assertion parses that `chatId` as a group target and applies `groupPolicy`/`groupAllowFrom` before sending.   - confidence: 0.9

  - confidence: 0.88
