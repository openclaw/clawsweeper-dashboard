The PR updates xAI OAuth token and callback handling, xAI OAuth-aware sidecar auth for TTS/STT/realtime/video/search, xAI video submit/poll parsing, xAI User-Agent attribution, endpoint host classification, and related tests.

## Best Possible Solution

Keep the focused xAI fixes, but make the callback server itself allowlist-aware, add a live server denial regression test, and require redacted after-fix OAuth/video/proxy evidence before merge.
