## Summary

Keep this PR open, but it is not merge-ready: current main already uses a narrower history contract, while this branch would hide rows that current tests intentionally preserve and would change the raw `sessions.get` API without an approved audit/raw escape hatch.

## What This Changes

This PR adds a predicate for OpenClaw `delivery-mirror` and `gateway-injected` assistant rows, filters them from `chat.history` and `sessions.get` after over-reading transcript tails, and updates gateway tests, protocol docs, and the changelog.

## Best Possible Solution

Rebase onto current main, reuse the shared helper, preserve current visible standalone/non-duplicate rows, filter only proven duplicate/internal artifacts, keep `sessions.get` raw unless maintainers approve a documented audit/include-internal contract, remove the changelog edit, and add redacted real gateway-call proof.
