## Summary

Keep open. The diagnostic request is valid and current main still lacks the new `[context-overflow-hint]`, but the PR currently keys the hint to a cumulative run-scoped counter, so a later independent overflow in the same run can get a misleading "persisted after compaction" hint.

## What This Changes

The PR adds a `[context-overflow-hint]` warning for post-compaction context overflow, documents the troubleshooting path, adds two overflow-loop regression tests, and updates the changelog.

## Best Possible Solution

Keep the additive operator hint, docs, changelog, and positive/first-overflow tests, but gate the runtime warning on an immediate post-compaction retry state that is reset after any attempt that does not immediately overflow again.

## Reproduction Assessment

Yes. A focused harness test can reproduce the review bug by compacting after an overflow, taking at least one non-overflow continuation in the same run, then causing a later independent overflow and asserting `[context-overflow-hint]` does not fire.

## Solution Assessment

No. The user-facing hint is the right direction, but the current `overflowCompactionAttempts > 0` gate is broader than the stated symptom; an explicit immediate-post-compaction flag is the narrower maintainable fix.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Gate the hint to the immediate post-compaction retry:** `src/agents/pi-embedded-runner/run.ts:1171`
  - body: `overflowCompactionAttempts` is cumulative for the whole `runEmbeddedPiAgent` call, so once any earlier overflow compacted, a later unrelated overflow in the same run will still emit `[context-overflow-hint]`. That mislabels the later failure as having persisted directly after compaction and can send operators toward `toolResultMaxChars` when the latest overflow may have a different cause. Track an immediate post-compaction retry state and clear it after any non-overflow continuation before logging this hint.
  - confidence: 0.88
