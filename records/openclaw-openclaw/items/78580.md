## What This Changes

## Best Possible Solution

Keep the PR open, add canonical root validation and symlink-parent regression coverage for local iMessage attachment staging, preserve the remote raw-path behavior, and track the separate remote SCP portability report at https://github.com/openclaw/openclaw/issues/78677 independently.
