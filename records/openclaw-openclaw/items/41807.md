## Best Possible Solution

Define one maintainer-approved additive agent-level prompt contract with file/include semantics, explicit trust-boundary rules, session-reuse hashing, docs/schema/help metadata, and runtime coverage across CLI and embedded session creation.

## Solution Assessment

No. The proposed hook points are plausible, but this is not the best merge shape until the public config contract, docs/metadata, session invalidation, trust boundary, and real behavior proof are settled.
