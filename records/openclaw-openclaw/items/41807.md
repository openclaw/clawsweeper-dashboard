The PR adds an optional `agents.defaults.systemPromptFile` config key, reads the configured local file into CLI and embedded runner system prompts, and adds helper coverage for file-reading behavior.

## Best Possible Solution

Define a maintainer-approved agent-level prompt-injection contract that covers file/include semantics, trust boundary, session reuse invalidation, docs/schema/help metadata, and both CLI and embedded session creation paths.
