## Summary

## What This Changes

The PR makes the bundled Codex plugin prefer the signed macOS Codex.app app-server for managed stdio startup, retry plugin-local managed candidates when that app-server is below the version floor, and updates Computer Use docs and tests.

## Best Possible Solution

Land after a maintainer accepts desktop-first managed startup as the intended macOS Computer Use default; otherwise keep plugin-local managed startup as default and require an explicit command override.
