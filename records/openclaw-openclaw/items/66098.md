- **[P2] Include the global reasoning default:** `src/gateway/server-methods/chat.ts:1667-1668`
  - body: The new effective reasoning value only considers `entry?.reasoningLevel` and `agentConfig?.reasoningDefault`, but current docs/schema also support `agents.defaults.reasoningDefault`.   - confidence: 0.9

  - body: `props.reasoningLevel` is only refreshed by `chat.history`, while session rows can update via `sessions.patch` or `sessions.changed`. Prioritizing this cache keeps reasoning blocks shown or hidden from old history after a live session change until history reloads.
  - confidence: 0.86
