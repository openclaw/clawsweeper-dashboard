- **[P2] Honor global reasoning defaults in chat.history:** `src/gateway/server-methods/chat.ts:1662-1663`
  - body: This new effective reasoning calculation skips `agents.defaults.reasoningDefault`. Runtime and docs resolve reasoning from session override, then per-agent default, then global default, then fallback, so a fresh session with only `agents.defaults.reasoningDefault: "stream"` would still get the wrong history/UI state here.
  - confidence: 0.91
- **[P2] Prefer fresh session reasoning after patches:** `ui/src/ui/views/chat.ts:1123`
  - body: `props.reasoningLevel` is only refreshed by `chat.history`, while session patches and `sessions.changed` update the active session row. By reading the cached history value first, the chat view can keep showing or hiding reasoning after a user changes the session reasoning setting until another history reload happens.
  - confidence: 0.88

  - body: The default label now prefers `state.chatThinkingLevel`, but model changes refresh session rows without reloading history. That lets the header keep the previous model's effective default label even after the selected model changes.
