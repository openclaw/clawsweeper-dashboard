## Best Possible Solution

Keep the generic Docker extra-package design in https://github.com/openclaw/openclaw/issues/85332 and avoid adding iproute2 to the default image unless maintainers explicitly change official-image policy.

Not applicable as a bug reproduction: this is a feature request. Source inspection confirms the current and latest-release Docker image do not preinstall iproute2 and only support extra APT packages during image build.
