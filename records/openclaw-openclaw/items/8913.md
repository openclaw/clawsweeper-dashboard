## What This Changes

Request a configurable per-channel or per-agent display format for visible thinking/reasoning blocks, especially for Telegram readability.

## Best Possible Solution

Define a documented reasoning display-format contract with italic as the compatibility default, choose precedence, then thread it through core reasoning payloads and Telegram with regression tests and docs.

## Solution Assessment
