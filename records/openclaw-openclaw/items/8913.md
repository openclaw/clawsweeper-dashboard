The issue requests a configurable format for visible thinking/reasoning blocks per channel or agent, especially for Telegram readability.

## Best Possible Solution

Define a documented reasoning display-format contract with compatibility defaults and explicit precedence, thread it through channel-edge delivery paths, and add focused tests plus real Telegram proof.

## Solution Assessment
