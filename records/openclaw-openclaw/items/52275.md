## What This Changes

The PR adds legacy delivery-queue normalization/tests for recovered outbound entries and also adds gateway agents.create readiness/runtime config refresh changes.

## Best Possible Solution

Land a current-main outbound-only delivery-queue normalizer that preserves every QueuedDelivery field with focused storage/recovery regression tests, and split gateway readiness or runtime refresh work into a separate review.
