## Summary

Keep open. Current `main` has strong config-clobber safeguards, but this issue targets the shipped stable v2026.5.7 updater, includes unexplained credentials-directory deletion, and still lacks packaged macOS update proof or a newer stable shipped fix.

## What This Changes

The issue reports that updating from v2026.5.6 to v2026.5.7 reset `openclaw.json` and removed the credentials directory, losing agents, channels, plugins, bindings, and tokens.

## Best Possible Solution

Keep this open until maintainers prove the packaged updater preserves both config and credentials, then ship or backport the validated fix path before cleanup closure.

## Reproduction Assessment

No, not with high confidence. Source and tests show current-main protection for the update-channel-only config clobber shape, but there is no live packaged macOS v2026.5.6 to v2026.5.7 reproduction and the credential deletion path remains unisolated.

## Solution Assessment

Unclear as a complete fix. The current-main write rejection, backup-before-replace path, and backup recovery are the right direction for config loss, but the best resolution also needs packaged-update validation and a credential-preservation explanation.
