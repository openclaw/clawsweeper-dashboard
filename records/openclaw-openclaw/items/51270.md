## Summary

Keep open: the TUI bug is still source-reproducible on current main, but this PR is not mergeable as-is because it shortcuts `/model list` and `/model status` away from the shared context-aware model-command contract.

## What This Changes

The PR changes TUI `/model` command handling so `/model list` opens the model selector and `/model status` prints cached session model information, with related tests.

## Best Possible Solution

Route TUI `/model list` and `/model status` through the shared model directive behavior, or match it exactly while preserving active agent/workspace/session context and full status diagnostics.

## Reproduction Assessment

Yes. Source inspection shows `/model list` and `/model status` parse as `model` with non-empty args, and current main's TUI handler sends those args to `patchSession` as model refs.

## Solution Assessment

No. The PR avoids the literal `patchSession` fallthrough, but `/model list` uses a context-poor picker path and `/model status` replaces the documented shared diagnostics with one cached line.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.92

Full review comments:

- **[P2] Route `/model list` through the session-aware path:** `src/tui/tui-command-handlers.ts:295-297`
  - body: The new `args === "list"` branch calls `openModelSelector()`, whose gateway backend sends unparameterized `models.list`. The shared `/model list` behavior carries active agent/workspace/session context, so workspace-scoped auth or agent-specific catalogs can show the wrong model list here.
  - confidence: 0.88
- **[P2] Preserve full `/model status` diagnostics:** `src/tui/tui-command-handlers.ts:298-305`
  - body: The new `status` branch reports only cached `state.sessionInfo` or `unknown`. The shared status response includes selected versus runtime model, default model, agent, auth file, provider auth, endpoint, and API details, so this loses troubleshooting output and can say unknown when the shared command can still answer.
  - confidence: 0.9
