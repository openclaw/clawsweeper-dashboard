## Summary

## Best Possible Solution

## Reproduction Assessment

Yes. At source level, current main passes useTopicAndGroupOverrides: false for native commands while regular group messages pass true, so topic/group groupPolicy overrides are skipped only on the native-command path.

## Solution Assessment

No, not as submitted. The flag flip is pointed at the right root cause, but it reuses the visible disabled-command reply branch and does not yet prove or implement the canonical issue's silent non-owner behavior.
