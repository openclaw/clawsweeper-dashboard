## What This Changes

Report that gateway startup can hang on macOS Big Sur because the IPv6 loopback bind probe has no timeout.

## Best Possible Solution

Review and land the linked timeout fix, or an equivalent narrow patch, with redacted macOS 11 proof and focused coverage for a never-settling probe.

Yes. A bounded timeout around the existing temporary bind probe is the narrowest maintainable fix, and this issue should stay open until the linked PR or an equivalent patch lands.
