Keep open: current main and the latest release still have an unbounded IPv6 loopback bind probe in gateway startup, and the linked fix pull request is open but unmerged. Because that PR uses closing syntax for this issue, the issue should remain open until the PR is reviewed, landed, or closed.

## What This Changes

The issue requests a bounded timeout for the gateway IPv6 loopback bind probe so macOS Big Sur startup cannot hang indefinitely.

## Best Possible Solution

Review and land a narrow fix that bounds the IPv6 loopback bind probe and adds regression coverage for a never-settling probe, without changing gateway bind policy.
