## Best Possible Solution

Fix this inside the macOS VoiceWakeOverlay owner files by making height measurement side-effect-free or guarded/coalesced, deferring attributed-text frame updates out of the SwiftUI update cycle, and adding focused coverage plus manual overlay stress proof.
