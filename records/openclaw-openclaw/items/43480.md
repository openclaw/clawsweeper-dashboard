## What This Changes

The issue reports intermittent macOS pinwheels when VoiceWakeOverlay mutates observed overflow state during SwiftUI layout updates.

## Best Possible Solution

Make overlay height measurement pure or safely coalesced, move overflow-state updates outside SwiftUI's synchronous view-update path with equality guards, and verify the real macOS overlay no longer pinwheels or emits update-loop warnings.

## Solution Assessment
