## What This Changes

This PR updates gateway session-key migration to preserve structured raw external session keys as aliases of canonical agent-scoped keys and adds a regression test for the helper.

## Best Possible Solution

Keep raw external aliases tied to the canonical entry through migrate, patch, agent writeback, reset, and delete, with regression tests proving stale aliases cannot revive old session state.

## Reproduction Assessment

## Solution Assessment

No, not as currently patched. Preserving the raw alias is the right direction, but the maintainable fix needs one alias lifecycle contract instead of a one-time copy inside migration.
