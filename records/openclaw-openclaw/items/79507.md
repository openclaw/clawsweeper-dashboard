## What This Changes

Dreaming should stop treating runtime-generated cron delivery reports and one-way system announcements as short-term promotion evidence for durable memory.

## Best Possible Solution

Rebase and review https://github.com/openclaw/openclaw/pull/79547, then land the provenance-backed session export filter so this issue closes through the merge path.

## Solution Assessment
