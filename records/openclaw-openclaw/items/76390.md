Current main already implements this PR's intended CLI behavior with the missing catalog identity and integrity propagation included, so this PR should close instead of being merged over the safer main implementation.

## What This Changes

The PR branch changes CLI plugin install resolution so bare official plugin ids consult the official external catalog before generic npm fallback, adds CLI/helper tests, and updates CHANGELOG.md.

## Best Possible Solution

Keep the current main implementation that resolves official ids before npm while preserving catalog identity and integrity checks; close this older PR rather than merge it.

## Reproduction Assessment

Yes. Source inspection shows the reported packaged-install failure path and the PR-head safety gap, while current main has targeted tests for `brave` and an integrity-pinned catalog entry.

## Solution Assessment

Yes for current main, no for the PR branch as-is. The main implementation is the narrow maintainable fix because it preserves bundled precedence, explicit npm semantics, and catalog safety metadata before generic npm fallback.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Pass catalog identity and integrity into npm installs:** `src/cli/plugins-install-command.ts:766`
  - body: The new official-catalog branch can resolve entries that carry `expectedIntegrity`, but this PR head calls the npm installer with only the resolved `spec`. A bare catalog-id install would bypass the expected plugin-id and integrity checks used by the existing catalog repair paths, so those fields need to be threaded through before merging this branch.
  - confidence: 0.87

## Security Review

Status: needs_attention

Summary: The PR diff introduces a catalog-driven npm install route without carrying catalog identity and integrity pins, though current main already fixes that concern.

Concerns:

- **[medium] Catalog integrity is not enforced on the PR branch:** `src/cli/plugins-install-command.ts:766`

  - confidence: 0.87
