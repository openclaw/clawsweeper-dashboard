## Best Possible Solution

Resolve this through the linked PR or an equivalent maintainer-owned policy update: either consistently default-allow macOS `screen.record` across gateway policy, setup defaults, audit/docs/tests, and real macOS proof, or keep explicit opt-in and make the capability messaging unmistakable.

## Solution Assessment
