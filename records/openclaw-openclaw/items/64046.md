## Summary

Keep open. Current main has meaningful partial coverage for log redaction, SecretRefs, and Control UI masking, but the umbrella request is not complete because normal config writes/backups can still persist plaintext secret-bearing JSON and some opt-in diagnostic/raw outputs still lack a uniform redaction policy.

The issue requests encryption or redaction for sensitive values in config files, logs, gateway logs, and Control UI secret fields.
