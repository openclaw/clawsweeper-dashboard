## Summary

Keep open for maintainer review: current main still preserves the documented explicit `audioAsVoice`/Ogg-Opus policy, while this PR changes the default for ordinary common audio attachments and still needs package, docs, changelog, and lockfile alignment before it is mergeable.

## What This Changes

The PR changes Feishu outbound media routing so common audio extensions and audio MIME types upload as `file_type: "opus"`/`msg_type: "audio"`, adds `music-metadata` duration parsing, and updates Feishu media tests/package metadata.

## Best Possible Solution

Decide the Feishu audio contract explicitly; if direct playable bubbles for common formats are desired, land a rebased narrow change that updates routing, duration handling, tests, docs, package metadata, lockfile, and changelog together.

## Reproduction Assessment
