  - confidence: 0.88
- **[P2] Resolve the QMD export freshness contract:** `extensions/memory-core/src/memory/qmd-manager.ts:2588-2591`

Current main fully reads and hashes on each export pass; this PR needs a stronger invalidation contract or explicit owner acceptance of the stale-export tradeoff.
