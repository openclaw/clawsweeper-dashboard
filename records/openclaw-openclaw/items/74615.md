## Summary

Keep open: current main and the latest release still provide static mention/always group activation plus always-on ambient room events, but not a timed post-mention attention window. The related contextual activation and passive-observation threads overlap, but neither safely supersedes this narrower product request.

## What This Changes

Request a configurable post-mention group-chat attention window that temporarily processes follow-up messages without repeated mentions.

## Best Possible Solution

Choose a consent-aware group activation design that can express timed post-mention attention centrally, then document and test it across the supported channels without duplicating per-channel policy.
