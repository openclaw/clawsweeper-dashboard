## Summary

Keep open: current main and v2026.6.10 still only support static mention/always activation plus always-on ambient room events, not a configurable timed post-mention attention window. Request a configurable cross-channel group-chat attention window that temporarily processes unmentioned follow-up messages after the agent is mentioned.

## Best Possible Solution

Choose a maintainer-sponsored, default-off attention-window design that reuses shared mention-gating and ambient room-event boundaries, defines privacy/security behavior, and then implements it consistently across supported group channels with docs and focused tests.
