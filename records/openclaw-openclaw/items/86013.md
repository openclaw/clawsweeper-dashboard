  - confidence: 0.92

  - body: `stripMarkdown` is shared by TTS and channel fallback text. Removing every literal `|` here corrupts ordinary non-LINE text such as `A | B`, so table-remnant cleanup needs to stay in a LINE-owned fallback path.
  - confidence: 0.95

  - body: LINE's current Flex spec caps bubble contents below this threshold, but the markdown guard allows contents up to `32768 - 200` bytes before fallback. Generated table/code bubbles between those values can still be sent and rejected instead of using the safe fallback.
  - confidence: 0.86
