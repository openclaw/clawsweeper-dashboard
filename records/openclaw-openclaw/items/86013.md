  - body: The LINE FlexBox schema includes `maxHeight` but no `overflow` field, and this casted object is sent as Flex JSON. Code-block replies can therefore produce a non-contract Flex payload instead of a renderable LINE message; remove it or use a LINE-supported layout.

  - confidence: 0.92

  - body: This guard only runs for directive-built `lineData.flexMessage`; markdown-generated Flex messages and direct `pushFlexMessage` sends bypass it. Put validation in the shared LINE Flex create/send path and align it with LINE's documented Flex limits.
  - confidence: 0.88
