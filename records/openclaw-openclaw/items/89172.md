The PR adds Feishu audio duration probing with ffprobe for prepared outgoing voice buffers, passes the millisecond duration to Feishu file upload, and covers success and probe-failure paths.

## Best Possible Solution

Land this Feishu-local duration propagation fix, or an equivalent patch that probes the final outgoing voice buffer and preserves send behavior when probing fails.
