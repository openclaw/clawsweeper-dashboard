## Best Possible Solution

Merge the small UI controller fix after live managed-service proof or a maintainer proof override confirms the Dashboard click-through no longer shows the skipped-update banner.

## Solution Assessment

Yes for the code shape: using `res.ok === true` is the narrowest UI-side alignment with the Gateway response contract, and `updateAvailable.latestVersion` is already available on the Dashboard state path.
