- **[P2] Guard backend-unavailable behind truly silent prompt exits:** `extensions/acpx/src/runtime.ts:676`
  - body: This maps any non-zero prompt exit without a structured ACP error to `ACP_BACKEND_UNAVAILABLE`, even when the process produced stderr explaining a normal turn/input failure.   - confidence: 0.87

  - confidence: 0.84

Preserve the recovery path unless a current-main test proves this fail-closed behavior is intentional.
  - confidence: 0.82
