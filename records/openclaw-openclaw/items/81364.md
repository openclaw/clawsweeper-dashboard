## What This Changes

The branch adds ClawHub exact-release trust checks and `--acknowledge-clawhub-risk` across plugin install/update/repair/onboarding/chat paths, with docs, tests, fixtures, and small adjacent runtime cleanups.

## Best Possible Solution
