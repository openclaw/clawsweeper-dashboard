## Best Possible Solution

Define a generic direct-human-intent provenance contract, sensitive-action taxonomy, fallback approval/challenge semantics, config/schema/docs, and regression coverage across exec, gateway mutation, messaging, cron/heartbeat, autonomous, and subagent flows.
