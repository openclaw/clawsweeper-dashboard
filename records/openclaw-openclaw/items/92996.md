Preserve the intended behavior: missing `--timeout` value returns `null` to defer to Commander, absent `--timeout` returns `timeoutMs: undefined`, invalid present values such as `0`, `-5`, `nope`, and `5s` return `null` from the route parser, and valid positive integers return `timeoutMs: number`.
