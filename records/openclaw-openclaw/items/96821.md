## Summary

Keep open: current main has OpenClaw code mode, Tool Search, internal code-mode namespaces, and MCP code-mode namespacing, but not the requested public external catalog-provider bridge for provider-owned search, describe, and call delegation. ## What This Changes

The issue requests a first-class code-mode external catalog-provider bridge so configured providers can expose search, describe, and call without nesting a second executor runtime.
