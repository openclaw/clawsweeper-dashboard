This PR adds a configurable cron ghost-run warning for fast successful main-session `systemEvent` jobs using `wakeMode="next-heartbeat"`, persists the warning in cron run logs, and updates config, protocol, docs, changelog, and focused tests.

## Best Possible Solution

## Solution Assessment
