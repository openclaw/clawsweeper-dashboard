- **[P2] Replace existing session headers case-insensitively:** `extensions/zai/stream.ts:39-42`
  - body: This merge leaves an existing `x-session-id` header in place and adds `X-Session-Id` beside it. Header normalization is case-insensitive; a local Fetch proof sends those as `old, new`, which can break ZAI stickiness or be rejected. Remove any existing session-id header key case-insensitively before setting the canonical one, and add a regression test.
  - confidence: 0.93
