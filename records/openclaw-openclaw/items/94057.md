## What This Changes

Adds `sandbox.docker.hostMountPrefixMap` to translate Docker bind source prefixes for DooD sandboxes and rewrites shell sandbox workspace, skill, and custom bind sources before `docker create`.

## Best Possible Solution

Repair this as a schema-backed, documented, security-validated host-path mapping contract across shell and browser sandbox containers, resolve conflicts, then require redacted live DooD proof before merge.
