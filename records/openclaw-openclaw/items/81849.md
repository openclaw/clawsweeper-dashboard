## Summary

Keep open: this is an active Telegram bug-fix PR with a source-provable regression on current main, but it still needs real Telegram behavior proof and the diff leaves a backlog case where unrelated lanes can remain hidden behind the first active-lane window.

## What This Changes

The PR changes Telegram isolated polling to dispatch spooled updates by Telegram sequential lanes, tracks active spooled handlers during shutdown, and adds polling-session regression tests.

## Best Possible Solution

Keep the lane-based isolated-ingress fix direction, repair the backlog scan hole, then require live Telegram proof showing topic/DM concurrency and control-command responsiveness before merge.

## Reproduction Assessment

Yes at source level: current main awaits each spooled bot.handleUpdate inside the isolated-ingress drain loop before reading later files, so one long handler blocks all other spooled updates before the normal sequentializer sees them. I did not run live Telegram traffic in this read-only review.

## Solution Assessment

Partly: reusing getTelegramSequentialKey and waiting active spooled handlers is the right maintainable boundary, but the current PR still only inspects the first 100 spool files and can miss unrelated lanes behind an active-lane prefix.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.84

Full review comments:

- **[P2] Scan beyond active-lane prefixes:** `extensions/telegram/src/polling-session.ts:297-299`
  - body: Because listTelegramSpooledUpdates slices to the first 100 files before this skip, a busy lane with 100 queued files at the front makes every later lane invisible while the first handler is active. In that backlog shape, an unrelated topic or /status update can still wait behind the active turn, so the isolated-lane fix does not cover the high-spool case it is meant to unblock. List enough files before skipping active lanes, page past the active prefix, or otherwise keep scanning until available lanes are found.
  - confidence: 0.86
