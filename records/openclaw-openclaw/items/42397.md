## Summary

Keep open: current main has a run-level heartbeat model override and structured due-task filtering, but no per-task HEARTBEAT.md model selection. This is a user-facing feature/config design change, not a stale or already-implemented bug fix.

The issue requests a per-task model hint in HEARTBEAT.md so selected heartbeat tasks can use a stronger model than the run-level heartbeat model.
