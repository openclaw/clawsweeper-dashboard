## Summary

Keep open. Current main still has the live Control UI path where cumulative gateway chat snapshots can be rendered beside already-committed tool-boundary stream segments, and the linked closing PR #74389 remains an open implementation candidate rather than a shipped fix.

## What This Changes

Fix Control UI WebChat live rendering so assistant text after tool-call boundaries appears as fresh non-cumulative cards instead of duplicating prior text or mirroring tool bubbles.

## Best Possible Solution

Review and land #74389 or an equivalent narrow Control UI fix that accounts for cumulative gateway snapshots at tool boundaries, clears terminal live artifacts safely, and adds regression coverage for multi-tool text -> tool -> text turns plus the mirrored tool-bubble variant. A user-facing changelog entry should accompany the fix.
