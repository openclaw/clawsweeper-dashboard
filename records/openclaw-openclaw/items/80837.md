## Summary

Keep open. The PR is a plausible plugin-local fix for a real PDF extraction bug, but the current head still fails real behavior proof and CI on temp-root policy, nullable test results, and redundant import assertions.

## What This Changes

The PR adds a plugin-local `pdftoppm` PDF page-rendering fallback when `@napi-rs/canvas` is unavailable, with tests, docs, a changelog entry, and several unrelated formatting-only diffs.

## Best Possible Solution

Keep the fallback plugin-local, switch temporary PDF staging to the SDK/OpenClaw temp root, fix the test type/lint blockers, trim unrelated churn, and add redacted real OpenClaw proof from a canvas-absent Poppler setup.

## Reproduction Assessment

Yes, source-reproducible. Current main’s low-text PDF path catches a missing canvas module and returns empty images, and the linked Ubuntu report gives matching `pdftoppm`-available evidence.

## Solution Assessment

Not yet. A plugin-local Poppler fallback is the right owner-boundary direction, but this branch needs temp-root, type/lint, and real-proof fixes before it is the best mergeable solution.

## Review Findings

Overall correctness: patch is incorrect

  - body: `os.tmpdir()` is guarded in extension/runtime sources, and CI is failing this line. Stage the PDF under `resolvePreferredOpenClawTmpDir()` or the plugin-sdk temp helper so user PDF bytes stay in the managed OpenClaw temp area and the boundary check passes.

  - body: `extract()` is typed as `DocumentExtractionResult | null`, but the new tests read `result.images` directly at the image assertions. Add an explicit non-null guard before accessing images in these new cases so `check-test-types` can pass.
  - confidence: 0.94
- **[P2] Remove redundant dynamic import assertions:** `extensions/document-extract/document-extractor.test.ts:157`
  - body: The dynamic imports already infer the module type, so the `as typeof import(...)` assertions are unnecessary and oxlint flags both new assertions. Remove the assertions and keep the assigned module values unchanged.
  - confidence: 0.9
