## What This Changes

The issue reports that isolated scheduled cron `agentTurn` jobs can recover from an early tool error, produce final output, and still skip delivery because cron finalization treats the earlier tool error as fatal.

## Best Possible Solution

Land one reviewed canonical cron payload-outcome fix that treats recovered tool-warning runs with final output as non-fatal for delivery while preserving true fatal structured-error suppression.
