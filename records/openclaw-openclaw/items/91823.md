## What This Changes

Repair Linux systemd Gateway auto-update detection so it uses the existing detached managed-service handoff instead of spawning an updater inside the Gateway process tree.

## Solution Assessment

Yes. Correcting supervisor classification at the existing boundary is the narrowest maintainable fix because it reuses the established detached handoff rather than weakening the safety guard or adding another updater path.
