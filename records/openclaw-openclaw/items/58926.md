## What This Changes

The branch adds a `FIELD_HELP`-based Levenshtein suggestion helper, unit tests, and missing-path hint output for `openclaw config get` and `openclaw config unset`.

## Best Possible Solution

Build suggestions from the existing runtime config schema/UI-hints inventory, filter non-copyable wildcard and array placeholders there, and refresh real CLI not-found proof for both `config get` and `config unset`.

## Solution Assessment
