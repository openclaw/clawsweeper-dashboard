Full review comments:

- **[P3] Avoid suggesting unchanged paths as corrections:** `src/config/suggest-path.ts:117-118`
  - body: Phase 2 accepts distance 0, so a known key that is absent from the current config can print `Config path not found: gateway.port` followed by `Did you mean: gateway.port?`. Skip exact matches in typo corrections or use wording that says the key is valid but unset.
  - confidence: 0.78
- **[P3] Include runtime config paths or constrain the scope:** `src/config/suggest-path.ts:76-78`
  - body: The helper only reads static `FIELD_HELP`, while current config schema metadata merges plugin and channel UI hints. Channel examples such as Telegram streaming will not be suggested unless they are in `FIELD_HELP`, so either source candidates from runtime schema/UI hints or make the static-only behavior explicit.
  - confidence: 0.84
