## What This Changes

Requested adding a Grok/xAI-backed Live voice loop to the `/chat` web UI for browser microphone input, spoken agent output, transcripts, and follow-up chat messages.

## Best Possible Solution

Keep this as the tracker for Grok/xAI Live Voice Mode in `/chat`.
