Request a Grok/xAI-powered Live voice loop in the `/chat` UI that speaks agent summaries, captures browser microphone input, transcribes the user, and continues the conversation without a phone call.

## Best Possible Solution

Add a maintainer-approved xAI `RealtimeVoiceProviderPlugin` registered from `extensions/xai`, wire it through `/chat` Talk provider selection and docs, and close only after focused tests plus redacted live xAI browser or relay proof.

## Solution Assessment
