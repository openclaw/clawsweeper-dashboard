## Summary

Keep open. The PR targets a valid Feishu quoted-media bug, but it still needs real Feishu behavior proof and the diff appears incomplete for quote replies where the triggering message becomes empty after the bot mention is stripped.

## What This Changes

The PR preserves raw Feishu fetched-message JSON, resolves media from allowed quoted Feishu parent messages, rebuilds the agent media payload, and adds focused Feishu regression coverage.

## Best Possible Solution

Land one current Feishu fix that preserves raw fetched parent content, resolves quoted media before empty quote-only messages are skipped, keeps existing visibility checks, includes focused regression coverage, and is backed by redacted real Feishu proof.

## Reproduction Assessment

Yes, from source inspection: on current main a Feishu reply to a file/image parent fetches only parsed placeholder text because raw media keys are not retained and MediaPaths is built before the quote fetch. I did not run a live Feishu account, so this is source-reproducible rather than live reproduced.

## Solution Assessment

No, not quite yet. Preserving rawContent and reusing the existing media resolver is the right direction, but the quoted-media resolution needs to happen early enough that @bot-only replies are not dropped by the empty-message guard, and merge should wait for redacted real Feishu proof.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.82

Full review comments:

- **[P2] Handle empty quote-only replies before skipping:** `extensions/feishu/src/bot.ts:1015-1027`
  - body: The new quoted-media resolution runs after the existing empty-message guard. In the common group flow where a user replies to a file/image with only the bot mention, `normalizeMentions` strips that mention, `ctx.content.trim()` is empty, and the current-message `mediaList` is empty, so the handler returns before this new block can fetch the parent media. Move the quoted fetch/media merge before the empty skip, or make the skip account for quoted media.
