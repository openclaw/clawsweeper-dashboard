## Summary

Keep open. The PR points at a plausible user-abort bug, but the proposed patch also gates timeout payload synthesis on `externalAbort`, which can make the stopped timeout path return without the explicit timeout error payload, and the draft has no real behavior proof.

## What This Changes

The PR changes PI embedded runner timeout handling to skip timeout-triggered compaction when `externalAbort` is true, adds a regression test, and adds a changelog entry.

## Best Possible Solution

Split the predicates: suppress timeout-triggered compaction for `externalAbort`, but keep prompt-timeout payload synthesis independent so stopped timeout runs still end with a clear terminal payload when needed.

## Reproduction Assessment

Yes. Source inspection gives a high-confidence focused reproduction for the PR regression: use the added harness setup with `timedOut=true`, `aborted=true`, `externalAbort=true`, high prompt usage, and no messaging-tool delivery; the patched `timedOutDuringPrompt` becomes false and skips timeout payload synthesis.

## Solution Assessment

No. Skipping timeout compaction for user aborts is a reasonable narrow direction, but applying the same `externalAbort` guard to `timedOutDuringPrompt` mixes compaction policy with terminal response synthesis.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Do not gate timeout payload synthesis on externalAbort:** `src/agents/pi-embedded-runner/run.ts:2514`
  - body: This boolean also controls the explicit timeout error return. With `timedOut=true`, `aborted=true`, and `externalAbort=true`, the patch skips compaction and then makes `timedOutDuringPrompt` false, so the timeout payload branch is bypassed. Since aborted runs suppress assistant output and incomplete-turn fallback returns null for aborted/timed-out attempts, this can return with no error payload instead of the timeout text your new test expects. Keep this predicate unchanged and add `!externalAbort` only to the compaction trigger, or split the two predicates.
  - confidence: 0.92
