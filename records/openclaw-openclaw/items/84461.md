  - body: `enforceTelegramDmAccess` returns false after sending/upserting a pairing challenge, and allowlist denials also return false.   - confidence: 0.94

  - body: The limiter keeps every new sender/scope key in `buckets` for the account lifetime. A rotating-sender flood can therefore grow memory without a hard cap or sweep, which leaves an availability gap in the same threat model this PR is trying to address.
  - confidence: 0.89

  - confidence: 0.9
