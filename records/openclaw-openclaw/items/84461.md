Full review comments:

- **[P1] Bound limiter buckets for unique senders:** `extensions/telegram/src/rate-limiter.ts:128`
  - body: This map creates a bucket for every new numeric sender, and stale buckets are only cleared by `reset()` or by seeing that same key again.
