## Best Possible Solution

Keep this as an active QMD traversal robustness item: add the controlled symlink-loop repro as regression coverage, then define one maintained QMD indexing contract through version enforcement, upstream traversal, or pre-update exclusion plumbing before removing the docs workaround.
