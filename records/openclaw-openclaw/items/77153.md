- **[P2] Preserve the existing changelog section:** `CHANGELOG.md:25`
  - body: The added `### Fixes` heading splits the current Unreleased `### Changes` list and leaves the original `### Fixes` heading later in the same section.   - confidence: 0.91
- **[P2] Avoid highlighting placeholder xargs risk text:** `src/infra/command-explainer/format.ts:55-60`
  - body: For xargs positional execution, the existing detector reports the synthetic flag `<command>`, but this formatter searches for the literal text `xargs <command>`. That string is not present in real approval commands, so the danger mark falls back to the start of the xargs segment and can highlight options instead of the carried command.
  - confidence: 0.87

  - confidence: 0.83
