## Review Findings

Overall confidence: 0.9

Full review comments:

- **[P2] Resolve filtering with channel context:** `extensions/speech-core/src/tts.ts:1106-1109`
  - body: This resolves `skipEmojiSymbols` with only agent and account IDs, while the current TTS config contract also applies `channels.<id>.tts` and channel account overrides through `channelId`. Channel policy for the new option would be ignored on automatic and command TTS paths.
