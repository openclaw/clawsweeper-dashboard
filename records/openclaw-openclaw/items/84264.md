Full review comments:

- **[P2] Preserve another followed user before leaving:** `extensions/discord/src/voice/manager.ts:828-833`
  - body: After deleting the current followed user, this branch only checks whether any remaining followed user is still in the existing channel. If another configured user is in a different allowed voice channel, `leave()` clears the guild follow state and the bot stops following instead of handing off, so select a remaining followed target before leaving or preserve state and join it.
  - confidence: 0.87
- **[P2] Bound follow-user reconciliation polling:** `extensions/discord/src/voice/manager.ts:944-947`
  - body: The interval reconciliation walks every resolved guild and every followed user every 10 seconds, issuing a REST voice-state lookup for each pair. A larger config can keep generating Discord REST traffic with no cap or backoff, so add a bounded strategy and coverage for the limit.
  - confidence: 0.82
