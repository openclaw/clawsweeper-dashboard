## What This Changes

The branch changes gateway reset cleanup, followup queue draining, reply delivery, and agent-runner paths to preserve queued followups across session resets and suppress stale replies.

## Best Possible Solution

Land a smaller rebased fix that preserves followups during sessions.reset without replaying them for sessions.new, rebinds queued work to canonical replacement sessions, suppresses stale streamed/tool/final output, preserves reset-stable metadata, and adds focused race tests plus live Telegram proof.
