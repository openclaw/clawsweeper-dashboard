- **[P2] Mark successful bootstraps as loaded:** `apps/shared/OpenClawKit/Sources/OpenClawChatUI/ChatViewModel.swift:254`
  - body: If the first `load()` bootstrap fails, `hasLoadedInitialState` is reset to false, but a later successful `refresh()` runs `bootstrap()` without ever setting it true. The next view `load()` can start another bootstrap and clear `pendingRuns`/`streamingAssistantText`, so navigating back during a recovered in-flight reply can drop live UI state.
