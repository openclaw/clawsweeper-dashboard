## Reproduction Assessment

Yes. Source inspection shows current main reads `plugin.qaRunners` during filtering and `rootDir` during sorting, so throwing accessors can abort QA runner discovery before healthy plugins are reached.

## Solution Assessment

Yes. The PR guards only the manifest metadata slice used by discovery and preserves the existing runtime contract errors for duplicate commands, missing registrations, and undeclared runtime exports.
