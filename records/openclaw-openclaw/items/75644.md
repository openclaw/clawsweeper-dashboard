## What This Changes

Adds a `plugins.bundledMode` config/schema/type surface and routes bundled web-search, web-fetch, document-extractor, and LLM provider runtime compatibility flags through a helper so `respect-allow` can honor plugin allow/entry policy, with web provider loader caching enabled in that mode.

## Best Possible Solution

Land a default-preserving `plugins.bundledMode: "respect-allow"` implementation that keeps compatibility as the default, gates bundled runtime provider/extractor paths through normal plugin allow/entry policy when opted in, documents the new public config key, and leaves the larger manifest-only runtime architecture cleanup for separate follow-up work.
