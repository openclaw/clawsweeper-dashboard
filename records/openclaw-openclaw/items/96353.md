  - body: The embedded helper caches the first drain result for a run even when it is empty. This guard only remembers non-empty drains, so a same-run retry after an empty first prompt build can consume context queued for the next turn instead of preserving exactly-once semantics.
  - confidence: 0.89

  - body: The PR exports a cleanup function but does not call it from Codex or Copilot harness finalizers, unlike the embedded runner. Non-empty queued injection text can stay in the process cache after the run ends until FIFO eviction.
  - confidence: 0.86
