## What This Changes

Add Telegram Business support so `business_connection` and `business_message` updates are handled by the Telegram plugin and outbound replies include the correct Business connection context.

## Best Possible Solution

Ship a reviewed Telegram plugin implementation that registers Business connect/message handlers, routes Business messages through existing DM authorization, keys/sequences Business updates, persists account-scoped Business connection state, applies it only to supported outbound API calls, and documents/tests Telegram Business behavior.

I did not run a live Telegram Business account reproduction in this read-only review.

## Solution Assessment

No for an allowed-updates-only fix; the best solution belongs in the Telegram plugin but must include handler wiring, durable account-scoped connection state, outbound reply mapping, tests, docs, and owner-approved access semantics.
