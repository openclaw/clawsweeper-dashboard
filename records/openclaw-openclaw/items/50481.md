## Summary

Keep open. Current main still has Slack Assistant status support but no suggested-prompt implementation: there is no Slack `setSuggestedPrompts` helper, no static config schema/type, no dynamic message-action name or dispatch branch, and no Slack docs for prompt suggestions. The related Slack Agents umbrella remains open and explicitly treats suggested prompts as separately tracked here rather than superseding this issue.

## What This Changes

The issue requests Slack Assistant suggested prompt support through static Slack configuration and, optionally, a dynamic agent message action.

## Best Possible Solution

Implement static suggested prompts inside the Slack plugin with validated config, post-reply dispatch for eligible Assistant threads, tests, and docs; treat the dynamic action as a separate additive message-action contract decision.

## Reproduction Assessment

Not applicable as a bug reproduction. Source inspection gives a high-confidence feature-gap check: current main has adjacent Slack Assistant status support but no suggested-prompt config, dispatch, action contract, or docs surface.

## Solution Assessment

Yes for the static Slack-plugin direction, provided it stays inside Slack config and reply handling. The dynamic agent action is not automatically part of a bug fix and should be approved as an additive shared action-contract feature.
