## What This Changes

The issue reports that indexed memory markdown files do not appear in `openclaw memory search` results while session transcript results do.

## Best Possible Solution

Fix memory-core hybrid selection so keyword-backed memory-file hits can surface even when strict session/vector results exist, with regression coverage for mixed `memory` and `sessions` sources under default scoring.

Yes, at source level. With hybrid enabled and both sources indexed, any strict session/vector result can trigger the early return before keyword-backed memory-file rows below the weighted threshold are considered.
