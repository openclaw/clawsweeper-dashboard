## Summary

保持打开：当前 `main` 仍会在恢复/复用会话时优先使用已持久化的 session/parent 模型覆盖项，配置 schema、文档和测试中还没有可选的 resume-time 模型策略。相关的更宽泛模型收敛提案仍是开放状态，但没有替代这个更窄的配置/API 决策。

## What This Changes

该 issue 提议新增一个恢复会话时的模型选择策略配置，让 resumed session 可选择沿用保存的会话模型或改用当前默认模型。

## Best Possible Solution
