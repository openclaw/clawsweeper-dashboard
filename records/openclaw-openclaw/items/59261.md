## Summary

Keep open. This docs-only PR is not already implemented on main, but the proposed guide still has live-contract blockers: the recommended shortcut returns 404, the documented npm install does not provide a `computer` bin, and the documented port-publish flags are not in Agent Computer's public port schema. The PR also lacks after-fix real setup proof.

## What This Changes

Adds a new Agent Computer hosting guide and links it from OpenClaw install, VPS, platform, FAQ, Mintlify navigation, redirects, and zh-CN glossary surfaces.

## Best Possible Solution

Keep the guide only after it matches Agent Computer's current public install and port-publishing contract and includes redacted evidence of OpenClaw running there.

## Reproduction Assessment

Yes for the PR defects: the added lines are visible at the PR head, the shortcut returns 404, npm metadata has no CLI bin, and Agent Computer's public llms/openapi contract does not match the documented install and port flags. This is not a current-main bug because the guide is not on main.

## Solution Assessment

No. A first-party hosting guide could be appropriate, but this version is not the narrow maintainable solution until it is rewritten against the live Agent Computer contract and proven in a real setup.
