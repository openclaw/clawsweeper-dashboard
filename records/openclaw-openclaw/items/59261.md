Adds an Agent Computer hosting guide and links it from OpenClaw install, VPS/platform, FAQ, Mintlify navigation/redirects, and zh-CN glossary docs.

## Best Possible Solution

Revise the guide against Agent Computer's current public CLI/API contract, remove or provider-verify the shortcut, document a supported port command, and include the required OpenClaw remote-origin/auth setup before maintainer docs review.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.94

Full review comments:

- **[P2] Point the shortcut at a live Agent Computer page:** `docs/install/agentcomputer.md:16`
  - body: The recommended `https://agentcomputer.ai/openclaw` shortcut redirects to the www host and returns HTTP 404, so new users hit a dead page before they can create a machine. Remove this path or replace it with a provider-verified live template URL.
  - confidence: 0.96
- **[P2] Install Agent Computer's documented CLI:** `docs/install/agentcomputer.md:26`
  - body: The guide tells users to run `npm install -g aicomputer`, but Agent Computer's current public docs install the CLI with their install script and the latest npm package has no `bin`. Fresh users will not get the documented `computer` command from this step.
  - confidence: 0.95
- **[P2] Use supported Agent Computer port flags:** `docs/install/agentcomputer.md:33`
  - body: The guide passes `--subdomain` and `--protocol`, but the live API schema exposes only `port`, optional `name`, and optional `visibility`, and the public docs show publish with optional `--private`. Document a supported command or include provider proof that these flags exist.
  - confidence: 0.92
- **[P2] Configure the published Control UI origin:** `docs/install/agentcomputer.md:108`
  - body: The guide switches the Gateway to `lan` and then opens the published HTTPS host, but current OpenClaw requires remote browser origins to be added to `gateway.controlUi.allowedOrigins`. Without that published origin, the browser WebSocket path can fail with `origin not allowed`.
  - confidence: 0.88
