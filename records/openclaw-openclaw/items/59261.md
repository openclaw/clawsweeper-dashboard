## What This Changes

This PR adds an Agent Computer hosting guide and links it from install, VPS, platforms, FAQ, Mintlify redirect/navigation, and zh-CN glossary docs surfaces.

## Best Possible Solution

Keep the guide idea, but amend it to match Agent Computer's public contract, remove or replace the dead shortcut until live, and include redacted real setup proof.

Yes. The docs defects reproduce with read-only live checks: the shortcut returns 404, the `aicomputer` npm metadata has no CLI bin, and the public docs/API do not expose the PR's port flags.

## Solution Assessment

No. A first-party hosting guide is the right surface, but this version should be corrected against Agent Computer's live public contract before it is maintainable.
