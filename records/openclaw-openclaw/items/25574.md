## What This Changes

## Best Possible Solution

Land a focused fix, likely through https://github.com/openclaw/openclaw/pull/38295 or a split replacement, that fingerprints warning/error payloads per config path, resets on clean state transitions, and adds regression coverage for load, write, and reload behavior.
