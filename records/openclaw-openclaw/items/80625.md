## Best Possible Solution

Wire a production Anthropic/Bedrock one-shot recovery wrapper that preserves normal signed-thinking replay, strips thinking blocks only on the exact provider rejection retry, and handles pre-content transport error events without duplicate output.

## Solution Assessment
