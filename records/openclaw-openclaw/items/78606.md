## Summary

Keep open: the PR addresses a real approval-routing concern and has sufficient terminal proof for the planner behavior, but the patch still has a correctness blocker because an empty origin-only native plan can suppress both approver DMs and same-chat manual approval instructions.

## What This Changes

The PR changes native approval planning so origin-preferred Discord channel delivery does not fall back to approver DMs when no origin target resolves, with tests, Discord docs, and a changelog entry.

## Best Possible Solution

Keep channel/origin-only approvals fail-closed to DMs, but ensure a same-chat manual approval notice or unsuppressed local prompt remains whenever native planning resolves no delivery target.

## Reproduction Assessment

Yes, at source level: current main maps target channel to preferredSurface origin and then falls back to approver DMs when originTarget is null. I did not run the targeted tests in this read-only review.

## Solution Assessment

No: the one-line planner guard prevents DM fallback, but because Discord suppresses the local prompt and the route notice only fires when a plan had targets, an empty plan can hide approval instructions. The safer fix keeps the no-DM behavior while preserving same-chat/manual fallback visibility.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Preserve a manual approval path for origin misses:** `src/infra/approval-native-delivery.ts:113`
  - body: With this guard, an origin-only Discord approval whose origin cannot be resolved returns an empty delivery plan. Discord still suppresses the local /approve prompt whenever the native route runtime is active, and the route reporter only sends the manual failed-delivery notice when at least one native target was planned, so this case can leave the originating chat with no approval instructions at all. Keep the no-DM behavior, but also make this path emit a same-chat manual notice when possible or avoid suppressing the local prompt for empty origin-only plans.
  - confidence: 0.87
