## Summary

Keep open: the no-DM routing intent is useful and the PR supplies real planner proof, but the current diff can make origin-only approvals invisible when origin resolution misses because the local prompt remains suppressed and the failed-delivery notice path requires a planned native target.

## What This Changes

The PR changes Discord native approval routing so channel-targeted approvals do not fall back to approver DMs when origin resolution fails, with docs, tests, and changelog updates.

## Best Possible Solution

Keep the no-DM behavior for channel/origin-only routing, but preserve a same-chat manual approval notice or avoid local prompt suppression when an origin-only native plan resolves no target.

## Reproduction Assessment

Yes, source-reproducible: current main maps Discord `target: "channel"` to origin, falls back to approver DMs when `originTarget` is null, and has a unit test asserting that fallback. I did not run tests in this read-only review.

## Solution Assessment

No: the planner guard is the right narrow place to prevent DM fallback, but returning an empty plan without adjusting prompt suppression or route notices can hide the approval entirely. The safer solution should also cover the same-chat/manual fallback path with regression tests.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.84

Full review comments:

- **[P2] Preserve approval visibility when origin resolution misses:** `src/infra/approval-native-delivery.ts:113`
  - body: With this guard, an origin-only Discord approval whose origin cannot be resolved returns an empty delivery plan. Discord still suppresses the local `/approve` prompt whenever the native route runtime is active, and the route reporter only sends the manual failed-delivery notice when at least one native target was planned, so this case can leave the originating chat with no approval instructions. Keep the no-DM behavior, but also make this path emit a same-chat manual notice when possible or avoid suppressing the local prompt for empty origin-only plans.
  - confidence: 0.87
