## Summary

Keep open: the no-DM fallback direction is sound and the PR includes real behavior proof, but the current patch can leave origin-only approvals with no visible manual approval path because Discord suppresses the local prompt while the route reporter only emits a failed-delivery notice when at least one native target was planned.

## What This Changes

The PR changes shared native approval delivery so origin-preferred/channel-only plans no longer fall back to approver DMs when no origin target resolves, and updates Discord docs, tests, and the changelog.

## Best Possible Solution

Keep the fail-closed-to-DMs behavior, but preserve a same-chat/manual approval path or avoid local prompt suppression when origin-only native planning resolves no target.

## Reproduction Assessment

Yes, at source level: current main maps target "channel" to preferredSurface "origin", then falls back to approver DMs when originTarget is null, and the existing unit test asserts that fallback. I did not run tests in this read-only review.

## Solution Assessment

No: the one-line guard is the right place to prevent DM fallback, but returning an empty plan without adjusting local prompt suppression or the route notice path can hide the approval entirely. The safer fix should fail closed to DMs while still surfacing a manual approval path in the origin when possible.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.82

Full review comments:

- **[P2] Preserve a manual approval path for origin misses:** `src/infra/approval-native-delivery.ts:113`
  - body: With this guard, an origin-only Discord approval whose origin cannot be resolved returns an empty delivery plan. Discord still suppresses the local `/approve` prompt whenever the native route runtime is active, and the route reporter only sends the manual failed-delivery notice when at least one native target was planned, so this case can leave the originating chat with no approval instructions at all. Keep the no-DM behavior, but also make this path emit a same-chat manual notice when possible or avoid suppressing the local prompt for empty origin-only plans.
  - confidence: 0.87
