The PR adds `YYYY-MM-DD` substitution for configured file log paths in `src/logging/logger.ts` and adds logger settings plus log-tail regression coverage.

## Best Possible Solution

## Solution Assessment

No. The feature direction is narrow and maintainable, but this branch should move placeholder resolution into the shared log-file-path surface and make the tests honor host-local date semantics before merge.
