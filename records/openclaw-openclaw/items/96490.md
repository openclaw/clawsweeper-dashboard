## What This Changes

The issue asks to replace the Base64-decoded Anthropic OAuth client id with an explicit public string constant and remove the decode helper.

## Best Possible Solution

Replace public OAuth client ids with plain constants and short comments where appropriate, preserving the existing values and OAuth flow behavior without adding config or migration surface.

## Solution Assessment

Yes. A plain constant with a short public-client-id comment is the narrowest maintainable fix, with a sibling check for the GitHub Copilot OAuth helper to avoid leaving the same pattern behind.
