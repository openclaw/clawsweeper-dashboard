## Best Possible Solution

Add a WhatsApp-owned allowlisted CTWA referral extractor that carries only approved fields through untrusted structured context or plugin metadata, and persist it only if maintainers explicitly approve durable session storage.

## Reproduction Assessment

## Solution Assessment

Unclear as proposed. Exposing the value is useful, but copying referral data broadly into the preamble or durable session metadata is not the narrowest maintainable fix until maintainers choose a typed, privacy-aware surface.
