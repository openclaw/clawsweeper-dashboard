The issue requests decoupling cron and isolated-agent delivery of produced assistant output from non-final ancillary background tool failures.

## Best Possible Solution

Define one terminal-outcome contract that still delivers valid final assistant text when only ancillary background diagnostics failed, while preserving true fatal run failures, denials, and any explicit fail-on-tool-failure opt-in.
