Full review comments:

- **[P2] Fold diagnostics into the gateway-aware health path:** `src/commands/doctor.ts:328`
  - body: This new call runs after noteMemorySearchHealth but has no gatewayMemoryProbe context. A running gateway can report embeddings ready while CLI-local credentials are unavailable, so doctor can emit a hard invalid-config failure after the existing health check accepted the same setup.
  - confidence: 0.89
- **[P2] Validate agent-only memorySearch configs:** `src/config/validation.memory-search.ts:117-118`
  - body: validateMemorySearchProviderConfig returns early when defaults memorySearch is absent or disabled, so configs that only define agents.list entries with memorySearch never reach the per-agent loop and can still fail later at runtime.
  - confidence: 0.88
- **[P2] Use stored cron run durations:** `src/cron/reliability.ts:147`
  - body: Completed cron jobs clear runningAtMs and persist lastDurationMs on the current service contract. Subtracting runningAtMs from lastRunAtMs yields timestamp-sized averages instead of real execution durations.
  - confidence: 0.86
