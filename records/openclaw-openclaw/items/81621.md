## Summary

Keep open: the PR targets a real agents runtime bug family, but the lifecycle fix is still placed after an embedded lifecycle event can already mark the run ended, so the false-success path can survive. The external proof is also not sufficient for merge because it is from a hotpatched older install rather than the latest-main PR code.

## What This Changes

The PR changes agent/subagent completion handling so unresolved tool-boundary endings become lifecycle errors, output selection waits through tool/yield turns, and related regression tests and announce prompt copy are updated.

## Best Possible Solution

Classify unresolved tool-boundary terminal metadata at the lifecycle event source or before setting lifecycleEnded, keep the output-selection hardening, add regression coverage for both paths, and provide redacted latest-main Telegram or log proof.

## Reproduction Assessment

Yes, source-level: current main can emit an embedded lifecycle end with stopReason "tool_calls" before the agent-command backstop runs. I did not execute a live current-main repro in this read-only review.

## Solution Assessment

No; the direction is narrow and relevant, but the guard is placed after lifecycleEnded can already suppress it. The safer fix is to classify unresolved tool-boundary terminal events before marking lifecycle completion, or inside the embedded lifecycle terminal path itself.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.82

Full review comments:

- **[P2] Classify tool-boundary lifecycle events before the backstop:** `src/agents/agent-command.ts:1125-1133`
  - body: The new pending-tool guard only runs when lifecycleEnded is false. In the embedded path, the lifecycle handler can emit phase "end" with stopReason "tool_calls", and onAgentEvent marks lifecycleEnded = true before this block, so the guard is skipped and the run can still be reported as successful. Move the classification into the lifecycle event path or intercept the terminal event before marking the lifecycle ended.
  - confidence: 0.83
