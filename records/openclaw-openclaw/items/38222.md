## Summary

Keep open. The prompt-context hardening gap is real, but this branch is not merge-ready against current main because it drops public hook result fields, caps the wrong boundary, leaves current prompt-visible sources uncapped, and still lacks after-fix real behavior proof.

## What This Changes

The PR adds a 50,000-character clamp for selected plugin hook `systemPrompt` and `prependContext` strings, plus focused truncation tests.

## Best Possible Solution
