  - body: Internal hook handlers are runtime code, so a malformed non-string entry in `event.messages` would throw on `.trim()` and can abort command or hook reply delivery. Normalize to strings before trimming and skip invalid entries instead of letting one bad hook output crash the path.
  - confidence: 0.82
