  - body: This new hook delivery runs before the existing `resolveSendPolicy` decision and calls `routeReply` directly. A session configured with `sendPolicy: deny` can still send hook-generated replies, bypassing the suppression path used by normal final, tool, block, and fast-abort replies.
  - confidence: 0.92

  - body: Runtime hook handlers are JS-extensible even though the TypeScript event type is `string[]`. If a hook pushes a non-string value, this `.trim()` call throws from the delivery helper and can abort command or inbound processing instead of safely skipping malformed hook output.
  - confidence: 0.84
