  - body: This new hook-drain path calls deliverInternalHookMessages before dispatch computes sendPolicy, and routeReply does not enforce that policy itself.   - confidence: 0.92

  - body: For a direct inbound message, ctx.To is commonly the bot address and ctx.From is the user who sent the message. Falling back to ctx.To before ctx.From sends hook replies to the bot identity; the new test currently locks in that wrong destination.
  - confidence: 0.9
