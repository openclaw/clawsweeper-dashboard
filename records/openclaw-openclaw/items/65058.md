## Summary

Keep open. The PR targets a real Google Chat parser bug that is still present on current main, but this external branch lacks inspectable after-fix real behavior proof and regression tests before it should merge.

## What This Changes

The PR updates the Google Chat webhook parser to unwrap Workspace Add-on `addedToSpacePayload` and `removedFromSpacePayload` envelopes and map them to lifecycle event types.

## Best Possible Solution

Land a focused Google Chat plugin parser fix that accepts lifecycle add-on envelopes, acknowledges non-message events without agent dispatch, adds regression tests, and stays coordinated with the open Google Chat parser cluster.

## Solution Assessment
