## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Resolve string SecretRefs before using apiKey:** `extensions/openai/openai-compatible-embedding-provider.ts:129-131`
  - body: `normalizeSecretInputString(value)` returns every non-empty string before `coerceSecretRef` runs, so an apiKey configured as `${ZHIPU_API_KEY}` or `$ZHIPU_API_KEY` is sent literally as `Authorization: Bearer ${ZHIPU_API_KEY}`.   - confidence: 0.94
