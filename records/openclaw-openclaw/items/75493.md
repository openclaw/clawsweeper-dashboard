## Summary

Keep open. The PR is the right narrow fix path for the MCP loopback cron case-preservation bug, and the current branch does forward `currentChannelId` through the cache boundary, but it also introduces a literal NUL byte in `src/gateway/mcp-http.runtime.ts`, causing Git/GitHub to treat that TypeScript source file as binary. That should be fixed before merge.

This PR threads a case-preserved `currentChannelId` from the CLI/MCP loopback request into gateway tool resolution and cron delivery context, adds a regression test, and records the Matrix cron fix in the changelog.
