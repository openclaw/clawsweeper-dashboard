## Best Possible Solution

## Solution Assessment

Yes, subject to proof. The diff is the narrowest maintainable direction I found: allow public rooms to use `autoThreadId` in `all` mode while preserving shared sessions for DMs, group DMs, and non-`all` public-room modes.
