## What This Changes

The issue reports that requests-in-flight heartbeat skips retry too aggressively after the heartbeat drift fix, causing heartbeat runs to block active Telegram and other channel conversations.

## Best Possible Solution

Keep this issue open until the linked fix PR is repaired and merged, or replace that PR with an equivalent fix that preserves per-target retry timing and includes focused tests plus real channel proof.
