  - body: The sandboxProfile branch returns before the normal host-exec wrapping that applies `tools.exec.pathPrepend` and shell snapshot handling. Any macOS call using this option can behave differently from ordinary exec with the same config; build the sandbox argv around the already-wrapped command path instead.
