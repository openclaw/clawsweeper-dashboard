## What This Changes

The PR updates OpenRouter Anthropic prompt-cache retention resolution and cache marker insertion so explicit `cacheRetention` values can produce `ttl: "1h"` or suppress markers, with focused unit tests.

## Best Possible Solution

Repair the wrapper/caller contract so the same effective params that drive transport retention also drive verified OpenRouter Anthropic marker insertion, while preserving endpoint-class gating and `none` opt-out behavior.
