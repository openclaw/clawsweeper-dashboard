## Summary

## What This Changes

The PR broadens embedded session-fence benign-change handling for queued user transcript appends, same-length linear transcript rewrites, unchanged text refreshes, and nested active-lock writes.

## Best Possible Solution

Land a narrow session-fence fix that permits queued OpenClaw-owned user transcript writes while still rejecting valid external user-message appends, with regression coverage and real behavior proof.
