## Best Possible Solution

Keep the copied-snippet protection, but preserve existing target-mode reaction binding by emitting a canonical header for that payload or supporting one narrow verified existing fallback shape, then require redacted real Signal proof before merge.

## Reproduction Assessment

## Solution Assessment

No in current form: header gating is the right mitigation for copied snippets, but it must preserve existing target-mode reaction binding before merge. The safer repair is to emit the canonical header for that payload or support one narrow verified fallback shape.
