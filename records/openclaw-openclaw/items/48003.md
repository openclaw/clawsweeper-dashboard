## What This Changes

Fix steer and steer-backlog so messages arriving during an active main-session run are injected into that run instead of waiting as followups.

## Best Possible Solution

Land or replace the focused lifecycle-aware steering fix, then close this issue after current main and a release contain the corrected behavior and aligned tests/docs.

## Solution Assessment
