## Best Possible Solution

Keep this as the canonical core API/security tracker and resolve it by reviewing and landing https://github.com/openclaw/openclaw/pull/78441 or a replacement that threads an additive narrowing policy through schema, spawn params, session metadata/protocol, runtime filtering, tests, and docs.
