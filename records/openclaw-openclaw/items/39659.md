## Summary

Keep open: current main still lacks the requested first-class Docker instance lifecycle. The existing Docker Compose, setup script, ClawDock helper, and multi-gateway docs provide manual path, port, profile, and update primitives, but they do not implement named create/list/delete/update semantics or an official instance registry.

## What This Changes

Add first-class Docker install commands for named OpenClaw instance create/list/delete/update with isolated directories, compose names, ports, updates, and cleanup.

## Best Possible Solution

Define the official Docker instance model and safety rules first, then implement the chosen lifecycle in the CLI/setup scripts with Docker docs and focused coverage.
