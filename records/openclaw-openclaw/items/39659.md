## What This Changes

The issue requests first-class Docker multi-instance lifecycle commands for creating, listing, deleting, and updating named OpenClaw instances with isolated ports, Compose names, config, workspace, and persistent directories.

## Best Possible Solution

Define the official Docker instance model and safety rules first, then implement the selected lifecycle across CLI registration, Docker setup/Compose handling, docs, and fresh-install plus upgrade validation.
