  - body: This adds `@zeroaltitude/openclaw-claude-bridge` to `minimumReleaseAgeExclude`, but the same PR makes that personal-scope package an executable subprocess that handles Anthropic credentials and network calls.   - body: Codex sends the terminal `assistant` event through `emitCodexAppServerEvent`, which attaches `sessionKey` and calls `params.onAgentEvent`; this block calls only the global emitter. That means Claude final assistant events can bypass per-attempt channel/progress subscribers and session-scoped consumers, even though the comment says it is matching Codex.
  - confidence: 0.88
- **[P2] Declare the /claude command owner in the manifest:** `extensions/claude/openclaw.plugin.json:5-8`
  - body: The runtime registers a reserved `/claude` command, but the manifest never declares a `commandAliases` entry like the Codex plugin does. Control-plane metadata builds command ownership from manifest aliases, so diagnostics, activation planning, and discoverability can miss `/claude` before the runtime has already loaded.
  - confidence: 0.82
