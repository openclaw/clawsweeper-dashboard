## Summary

Keep open. Current main has browser tab tracking plus idle/cap cleanup and lifecycle cleanup for cron, subagent, and reset paths, but it still does not close primary-session browser tabs at ordinary task completion; the remaining change needs ownership and preservation rules before implementation.

## What This Changes

Request that browser tabs/windows opened only for temporary automation be closed automatically after task completion while preserving user-owned or explicitly kept pages.

## Best Possible Solution

Define run-owned ephemeral browser tab metadata and hook primary-run completion cleanup to close only tabs created for that run, preserving explicit user-owned or kept pages and documenting the resulting behavior.

## Reproduction Assessment

Yes. Source inspection on current main shows browser opens are tracked by session, primary cleanup is idle/cap based, and ordinary chat completion has no browser cleanup call.

## Solution Assessment

Unclear as requested. The cleanup goal is sound, but the maintainable solution is ownership-based task cleanup rather than a blanket close-all hook or generic auto-close switch.
