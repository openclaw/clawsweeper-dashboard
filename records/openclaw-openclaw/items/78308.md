## Summary

## What This Changes

Request channel-mediated approval for bundle-MCP tool calls via a consent envelope that triggers the existing `/approve` pipeline before re-calling the MCP tool with a confirmation token.

## Best Possible Solution

Land a maintainer-approved bundle-MCP approval contract, likely through #78303 or a replacement, with schema/docs/tests aligned and a clear decision on how it coexists with MCP-native elicitation.
