## What This Changes

Request an opt-in MCP consent envelope that routes mutating bundle-MCP tool calls through existing channel/plugin approvals and adds an `mcp.approvals` config surface.

## Best Possible Solution

Review and land, or replace, https://github.com/openclaw/openclaw/pull/78303 with a security-approved bundle-MCP approval design that documents its relationship to MCP-native elicitations.

## Solution Assessment
