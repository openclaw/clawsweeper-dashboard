## Summary

Keep open: the bug being targeted is plausible, but the patch has blocking prompt/performance issues and the external PR still lacks real after-fix behavior proof.

## What This Changes

The PR injects the latest transcript compaction summary into post-compaction context and passes a session file through the reply-agent compaction call sites.

## Best Possible Solution

Plumb the compaction summary through a bounded path, preferably from the compaction result or a tail/checkpoint reader, and make the prompt wording conditional on an actually injected summary.

## Reproduction Assessment

No high-confidence runtime reproduction was established. Source inspection shows current main does not inject the transcript summary in this helper, and docs/source confirm compaction summaries are transcript entries, but no real continuation flow was run.

## Solution Assessment

No. The direction is plausible, but this implementation rereads the full transcript and tells the agent to use a summary even when none was injected; a bounded summary source plus conditional prose is safer.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Avoid reading the whole transcript after compaction:** `src/auto-reply/reply/post-compaction-context.ts:79-80`
  - body: This path runs immediately after compaction, when the session JSONL is large enough to trigger compaction in the first place. Loading the whole file and splitting it can allocate multiple large copies or fail before the refresh prompt is injected; use the compaction result/checkpoint or a bounded tail scan instead.
  - confidence: 0.87
- **[P2] Only mention a summary when one is injected:** `src/auto-reply/reply/post-compaction-context.ts:211-217`
  - body: When sessionFile is omitted, unreadable, or has no compaction entry, compactionSummary is null and no <compaction-summary> block is emitted, but the prompt still says the summary is below and should be used. That keeps the fallback path misleading instead of gracefully degrading.
  - confidence: 0.9
