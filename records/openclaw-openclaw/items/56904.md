Full review comments:

- **[P1] Stop terminal escalations before later hooks run:** `src/plugins/hooks.ts:794`
  - body: `runBeforeToolCallHook` hard-blocks `decision: "escalate"` when no approval is present, but this reducer only stops on `block` or `deny`. A lower-priority hook can still add `requireApproval`, turning the stricter escalation into user-approvable execution; stop on terminal escalations or otherwise freeze the result.
  - confidence: 0.9
- **[P2] Honor explicit empty-string subagent rewrites:** `src/agents/subagent-announce.ts:561-565`
  - body: These truthiness checks skip `resultText: ""` and `replyInstruction: ""`, so a guard that intentionally redacts child output to an empty string still delivers the original findings. Check for `undefined` instead so deliberate empty-string rewrites are preserved.
  - confidence: 0.86
