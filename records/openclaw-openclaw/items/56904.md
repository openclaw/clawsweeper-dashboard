  - confidence: 0.88

  - body: The PR exports `redact` as a guard decision and ranks it in the reducer, but the delivery call sites only act on `warn`, `deny`, and `escalate`. A plugin that returns `decision: "redact"` without also knowing to rewrite the payload gets no redaction behavior, which is unsafe for a public security-facing contract.
  - confidence: 0.82
