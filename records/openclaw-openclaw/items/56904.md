Full review comments:

- **[P1] Stop terminal escalations before later hooks run:** `src/plugins/hooks.ts:794`
  - body: `runBeforeToolCallHook` treats `decision: "escalate"` without `requireApproval` as a hard block, but this reducer keeps running lower-priority hooks because `shouldStop` only stops on block/deny. A later hook can add `requireApproval`, turning the stricter escalation into user-approvable execution; stop on terminal escalations or otherwise freeze the result.
  - confidence: 0.9
- **[P2] Honor explicit empty-string subagent rewrites:** `src/agents/subagent-announce.ts:561-565`
  - body: These truthiness checks skip `resultText: ""` or `replyInstruction: ""`, so a guard that intentionally redacts child output to an empty string still delivers the original findings. Check for `undefined` instead so deliberate empty-string rewrites are preserved.
  - confidence: 0.86
