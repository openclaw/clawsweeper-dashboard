  - confidence: 0.88

  - body: The PR exports `redact` as a guard decision, but the delivery call sites only implement warn, deny, and escalate behavior. A plugin returning `decision: "redact"` without also rewriting the payload gets no redaction, which is unsafe for a public security-facing API.
