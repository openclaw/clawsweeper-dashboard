  - body: `runBeforeToolCall` now treats `decision: "escalate"` as stricter than warn/redact, but the reducer keeps invoking lower-priority hooks because `shouldStop` only stops on `block` or `deny`.   - confidence: 0.9

  - body: These truthy checks skip intentional empty-string rewrites. A guard hook that redacts with `resultText: ""` or clears `replyInstruction` will leave the original child findings or instruction in place, defeating the new hook’s suppression/redaction contract.
  - confidence: 0.88
