## What This Changes

The PR updates Feishu interactive-card parsing and tests to extract top-level titles plus nested post-format fallback text and link nodes from `elements` arrays.

## Best Possible Solution

Revise and land a narrow Feishu-plugin parser/test fix that handles nested fallback rows and title/link text while preserving later candidate fallthrough, then add redacted real Feishu read proof.
