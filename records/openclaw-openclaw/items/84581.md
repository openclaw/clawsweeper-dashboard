## Summary

Keep this PR open: the latest proof is now tied to head `555b6dcb`, but the patch still leaves legacy catalog-only plaintext `models.json` keys untouched when planning returns `skip` or `noop` before the new migration runs.

The PR strips plaintext provider `apiKey` values from generated agent `models.json`, migrates eligible legacy catalog-only keys into `auth-profiles.json`, preserves non-secret markers, and adds regression coverage.
