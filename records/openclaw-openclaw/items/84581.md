## What This Changes

The PR strips plaintext provider `apiKey` values from generated `models.json`, migrates legacy catalog-only keys into `auth-profiles.json`, and adds regression coverage for stripping, migration, and fail-closed writes.

## Best Possible Solution

Keep `models.json` prompt-safe while preserving source-managed env markers for custom providers, add a regression for custom env SecretRef providers with model rows, and refresh redacted live proof from the latest head.

## Reproduction Assessment

## Solution Assessment

No. The strip step needs to distinguish source-managed env markers from plaintext literals instead of treating every unknown non-built-in string as a secret value.
