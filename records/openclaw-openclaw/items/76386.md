## Summary

## What This Changes

The PR changes `scripts/install.sh` so SIGINT/SIGTERM abort installation, doctor exit 130 propagates as an interrupt, and dashboard opening is gated on successful doctor runs.

## Best Possible Solution

Fix the real terminal Ctrl+C path by making doctor cancellation observable to the installer, then explicitly accept or revise the new doctor-failure dashboard/plugin-update semantics before merge.

  - body: The reported failure says `openclaw doctor` can catch Ctrl+C and exit 0.
