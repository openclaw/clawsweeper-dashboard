## Best Possible Solution

Make a maintainer-owned API decision: either add a backward-compatible account-scoped lifecycle seam with SDK docs and contract tests, or explicitly consolidate the channel-plugin request into https://github.com/openclaw/openclaw/issues/49804 after the Feishu-specific bug is handled separately.
