## Best Possible Solution

- **[P1] Preserve explicit CLI provider full-catalog browse:** `src/auto-reply/reply/commands-models.ts:342-347`
  - body: In the `all` view, `resolveVisibleModelCatalog` returns the entire catalog and this loop records every canonical model as a prune key; because `hasAuth` is also forced true for `all`, the later prune removes `claude-cli`/`google-gemini-cli` rows even when the user explicitly asks to browse that provider's full catalog.
