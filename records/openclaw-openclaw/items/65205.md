  - confidence: 0.96

  - body: The injected Canvas helper imports executable JavaScript from mutable CDN URLs inside the Canvas origin. Bundle a pinned dependency or use an explicitly approved, constrained loading model with CSP/proof before shipping this security-sensitive helper.
  - confidence: 0.93
- **[P1] Stop putting Activity tokens in WebSocket URLs:** `src/canvas-host/a2ui/index.html:325-329`
  - body: The browser bridge reads `activityToken` from the page URL, forwards it in the WebSocket query string, and falls back to `ws://` on HTTP pages. Use a safer server-set cookie, exchange, or protocol path and fail closed for insecure transport so Activity tokens do not leak through URLs or cleartext sockets.
  - confidence: 0.9

  - confidence: 0.91

## Security Review

Status: needs_attention

Summary: The diff introduces Activity auth, token transport, third-party script loading, and WebSocket fanout concerns that need security and owner review before merge.

Concerns:

- **[high] Activity mode can bypass Gateway auth without a credential:** `src/gateway/a2ui-activity.ts:151`
  - body: With Activity enabled and no token configured, Activity-looking requests can be accepted before normal bearer or capability auth, exposing Canvas/A2UI over a public Activity path.
  - confidence: 0.96
- **[high] Canvas helper loads executable SDK code from mutable CDNs:** `src/canvas-host/a2ui.ts:173`
  - body: The injected helper imports the Discord embedded SDK from jsDelivr or esm.sh at runtime inside the Canvas origin, which creates a supply-chain execution path without a pinned local dependency or visible CSP constraint.
  - confidence: 0.93
- **[medium] Activity token is copied into URL query strings:** `src/canvas-host/a2ui/index.html:325`
  - body: The A2UI bridge forwards `activityToken` in the WebSocket URL and can use cleartext `ws://` on HTTP pages, which can leak credentials through logs, intermediaries, browser history, or local network traffic.
  - confidence: 0.9
- **[medium] A2UI Activity fanout is unbounded:** `src/gateway/a2ui-activity.ts:240`
  - body: The Activity hub retains and broadcasts arbitrary JSONL without size limits or backpressure checks, creating memory and event-loop risk for public Activity connections.
  - confidence: 0.91
