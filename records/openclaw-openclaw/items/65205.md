  - body: Current main treats root `canvasHost` as legacy input and migrates it into `plugins.entries.canvas.config.host`; adding `canvasHost.activity` reopens a retired root config surface and bypasses Canvas plugin ownership.   - confidence: 0.94

  - body: This helper dynamically imports `@discord/embedded-app-sdk` from jsDelivr or esm.sh and executes that mutable third-party JavaScript in the Canvas origin. Bundle the SDK with the owning plugin/assets or use a pinned owner-controlled loading path before exposing it to Activity pages.
  - confidence: 0.9
- **[P1] Stop carrying Activity tokens in WebSocket URLs:** `src/canvas-host/a2ui/index.html:325-329`
  - body: The bridge reads `activityToken` from the page URL and forwards it in the WebSocket query string, falling back to `ws://` on HTTP pages. Use a safer cookie/token-exchange/protocol path and fail closed for insecure transport so tokens do not leak through logs, history, intermediaries, or plaintext traffic.
  - confidence: 0.91
- **[P2] Bound Activity broadcasts and slow clients:** `src/gateway/a2ui-activity.ts:240-243`

  - confidence: 0.91
