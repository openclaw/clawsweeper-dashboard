## Review Findings

Full review comments:

- **[P1] Move Activity config to the Canvas plugin owner:** `src/config/zod-schema.ts:654-660`
  - body: This adds `canvasHost.activity` under the legacy root Canvas config surface, but current main moved Canvas host runtime/config ownership to the Canvas plugin and hides root `canvasHost` as migration input only. Reintroducing a root runtime config path would split the Canvas source of truth and break the current owner boundary.
  - confidence: 0.91
- **[P1] Require tokened Activity auth before bypassing Gateway auth:** `src/gateway/a2ui-activity.ts:150-154`
  - body: With activity mode enabled and no `activity.token`, any request carrying spoofable launch markers can pass `authorizeCanvasRequest` before normal Gateway auth. Activity launch context markers alone are not an authorization boundary, so this can expose Canvas/A2UI routes to unauthenticated callers.
  - confidence: 0.93
- **[P1] Bundle or pin the Discord SDK before injecting it:** `src/canvas-host/a2ui.ts:171-176`
  - body: The Canvas helper executes `@discord/embedded-app-sdk` from jsDelivr or esm.sh at runtime inside the Canvas origin. This adds mutable third-party script execution to a gateway-hosted page instead of using a pinned local dependency or an explicitly approved loading boundary.
  - confidence: 0.9
- **[P1] Bound Activity A2UI broadcasts before retaining payloads:** `src/gateway/a2ui-activity.ts:240-244`
  - body: `broadcastPush` stores arbitrary JSONL and sends it to every connected WebSocket without a byte limit or slow-client handling. A large or repeated A2UI payload can become retained memory and fanout work across all Activity clients.
  - confidence: 0.88
- **[P2] Clear failed SDK loads before retrying:** `src/canvas-host/a2ui.ts:202-208`
  - body: When `load()` fails once, the rejected `_loadPromise` stays cached and every later call awaits the same failed promise. A transient CDN, network, or `ready()` timeout failure makes the helper unrecoverable until a full page reload.
  - confidence: 0.86
