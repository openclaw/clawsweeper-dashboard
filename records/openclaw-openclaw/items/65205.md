Full review comments:

- **[P1] Require credentials before activity auth bypass:** `src/gateway/a2ui-activity.ts:151-154`
  - body: When Activity mode is enabled without a token, this returns true after only spoofable launch markers from the URL/cookie/referer, and `authorizeCanvasRequest` checks it before normal bearer or canvas-capability auth. Fail closed unless a configured token or verifiable Discord assertion is present.
  - confidence: 0.96
- **[P1] Move Activity config under the Canvas plugin contract:** `src/config/zod-schema.ts:660-667`
  - body: This adds `canvasHost.activity` under the legacy root config schema, but current main treats root `canvasHost` as doctor-migrated input for `plugins.entries.canvas.config.host`. Put Activity settings under the Canvas plugin config or add an explicit maintainer-approved migration/product path.
  - confidence: 0.94
- **[P1] Bundle the Discord SDK instead of loading CDNs:** `src/canvas-host/a2ui.ts:172-176`
  - body: The injected Canvas helper executes `@discord/embedded-app-sdk` from jsDelivr or esm.sh at runtime in the Canvas origin, outside the lockfile and normal review path. Bundle/pin the SDK or provide a maintainer-approved local asset path before exposing this helper.
  - confidence: 0.92
- **[P2] Bound A2UI activity fanout:** `src/gateway/a2ui-activity.ts:240-243`
  - body: `broadcastPush` stores arbitrary JSONL and sends it to every Activity WebSocket without a byte cap or `bufferedAmount`/slow-client handling, so one large update or slow clients can create gateway memory and event-loop pressure.
  - confidence: 0.91
- **[P2] Keep activity tokens out of WebSocket query URLs:** `src/canvas-host/a2ui/index.html:325-329`
  - body: The browser bridge copies `activityToken` into the WebSocket query string and falls back to `ws://` on HTTP pages, which can leak the token through logs, intermediaries, or cleartext local/proxy traffic. Use a secure cookie or another non-URL credential channel and fail closed for authenticated activity traffic over HTTP.
  - confidence: 0.82

## Security Review
