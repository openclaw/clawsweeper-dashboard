## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

No single best fix is proven yet: `openclaw doctor --fix` or `openclaw plugins update codex` may repair affected installs, but maintainers need to choose the intended runtime behavior for stale official plugin drift, provider catalog routing, and failover classification.
