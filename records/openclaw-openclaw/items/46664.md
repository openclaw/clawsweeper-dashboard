## Summary

Keep open. Current main still has partial offline-related pieces, but not the requested durable cross-platform mobile command outbox: Android voice uses an in-memory queue, iOS has scoped watch/pending-node paths, and typed/voice mobile commands still lack crash-persistent capture, replay metadata, idempotent sync, privacy controls, and queue-management UX.

## What This Changes

The issue requests a durable offline command queue for the iOS/Android mobile node apps that captures text and voice intents while disconnected and syncs them to the gateway when connectivity returns.

## Best Possible Solution

Define a staged mobile outbox contract first, then add durable encrypted local storage and idempotent reconnect replay for text/voice intents before expanding to queue editing, reordering, Wi-Fi-only behavior, and richer background sync UX.

## Reproduction Assessment

Not applicable: this is a feature request rather than broken existing behavior. Source inspection gives a high-confidence current-state check that existing queues are narrow, mostly process-local paths rather than a durable general mobile command outbox.

## Solution Assessment

No for implementing the full SQLite/background-sync/queue-management proposal as one sweep. The maintainable path is a staged payload, persistence, privacy, and replay contract that builds on the existing Android voice queue, iOS watch/reachability patterns, and gateway pending-work foundation deliberately.

## Risks / Open Questions

- Persisting raw text commands or audio blobs without encryption at rest, retention limits, delete controls, and log redaction would expose sensitive user data on mobile devices.
- At-least-once reconnect replay without a mobile-generated idempotency key, capturedAt metadata, and gateway dedupe/ack contract could duplicate or mis-time user commands after crashes or partial syncs.
- Mobile OS background execution limits mean “sync automatically when connectivity returns” needs explicit foreground/background expectations instead of an unconditional immediate flush promise.
