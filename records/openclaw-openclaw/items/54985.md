The PR adds OpenClaw session, attempt, fallback, turn-index, and turn-id headers to outbound LLM requests and adds fallback visibility to LLM output hook events.

## Best Possible Solution

Define one documented opt-in or trusted-target LLM request-correlation contract with bounded identifiers, retry-stable model-call IDs, and fallback state propagated from the fallback runner's actual candidate state.

## Solution Assessment
