## Security Review

Status: needs_attention

Summary: The diff introduces concrete privacy regressions in inline assistant markers and structured Claude tool events.

Concerns:

- **[medium] Inline markers can bypass session verbose changes:** `src/agents/cli-runner/execute.ts:467`
  - body: The marker decision is re-evaluated per tool, but it does not load the newer persisted session entry that current channel delivery uses. A mid-run verbose-off session change can stop structured channel tool messages while inline assistant markers continue revealing command/path/query details.
  - confidence: 0.9

  - confidence: 0.88
