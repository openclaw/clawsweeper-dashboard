## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.78

Full review comments:

- **[P1] Drain CLI tool progress deliveries before lifecycle completion:** `src/auto-reply/reply/agent-runner-cli-dispatch.ts:111`
  - body: The new `toolEventUnsub` listener invokes `params.onToolEvent` and ignores its returned promise. `agent-runner-execution` passes an async callback that awaits typing and `onToolStart`, while `emitAgentEvent` notifies listeners synchronously, so the CLI run can emit terminal assistant/lifecycle events or surface unhandled rejections before channel progress has finished. Mirror `createAssistantTextBridge` with a serialized delivery promise and drain it on success/error/finally.
  - confidence: 0.84
- **[P2] Document the new CLI backend inheritance contract:** `src/plugins/cli-backend.types.ts:190-203`
  - body: `CliBackendPlugin` is exported through `openclaw/plugin-sdk/cli-backend`, so adding `inheritUserConfigFrom` creates a public plugin API. Scoped plugin guidance requires SDK docs and contract coverage to move with new seams; without that, external plugin authors get undocumented config inheritance behavior that can drift from setup/fallback resolution. Either keep this internal to bundled Anthropic or update the CLI backend plugin docs and contract tests.
  - confidence: 0.8
