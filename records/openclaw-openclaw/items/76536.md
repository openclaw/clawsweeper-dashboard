## What This Changes

## Best Possible Solution

## Solution Assessment

Yes. The PR restores the previous scoped cold-load behavior only after a loaded-registry miss, preserves the startup-registry fast path, and adds focused regression coverage for the Fish Audio manifest-contract case.
