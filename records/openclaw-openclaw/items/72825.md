## Summary

Keep this PR open: current main still has the GitHub Copilot Claude Opus 1m capability bug, and the PR is a useful narrow fix, but the proposed context/session fallback overstates Copilot's advertised 1,000,000-token limit by returning the Anthropic 1,048,576-token constant. The PR adds GitHub Copilot known-capability overrides for Claude Opus 4.6/4.7 1m variants, enables xhigh for the 4.7 1m internal variant, adjusts session context fallback behavior, and adds targeted unit tests.

## Best Possible Solution

## Solution Assessment
