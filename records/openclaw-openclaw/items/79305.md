## What This Changes

Add a structured warning when the session writer cache detects that the on-disk session store changed between cache loads.

## Best Possible Solution

Confirm the diagnostic contract first, then add a fail-open structured signal in the writer cache path with focused coverage for stat mismatch, missing file, and normal queue serialization.
