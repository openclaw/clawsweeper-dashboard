  - body: Current main intentionally keeps the generated LaunchAgent throttle at 10s so crash loops back off while explicit kickstart restarts still recover promptly.   - body: This branch still accepts trailing text in `meta.lastTouchedVersion` via the unanchored version parser and then logs the raw value here. Current main anchors version parsing, so the PR should preserve that hardening or sanitize/reject the value before logging it.
  - confidence: 0.88

  - body: Current launchd runtime resolves supervisor logs under `~/Library/Logs/openclaw`, but this docs addition points CLI LaunchAgent users at `$OPENCLAW_STATE_DIR/logs`. That sends users to the wrong files when diagnosing or truncating macOS service log growth.
  - confidence: 0.9
