## Summary

The PR wraps the global Undici EnvHttpProxyAgent with OpenClaw's enhanced proxy-bypass matcher, changes ALL_PROXY detection in proxy-env helpers, and updates related proxy tests.

## Best Possible Solution

Keep the shared SSRF helper HTTP(S)-only, add a global-dispatcher-specific proxy decision/factory that preserves ALL_PROXY parity, and use it for bootstrap, timeout rebuild, and reset paths with focused tests plus redacted real runtime proof.
