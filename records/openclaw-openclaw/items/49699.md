The PR changes device-pairing token helpers, gateway device-token RPC handlers, websocket token issuance, and tests to return structured error objects instead of null or ok/reason denial shapes.

## Best Possible Solution

Keep public token-management denials generic unless maintainers approve a versioned structured-error contract; add richer diagnostics only internally and first repair the syntax blockers.

## Solution Assessment
