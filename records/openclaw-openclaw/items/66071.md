## Summary

The PR updates the OpenResponses HTTP SSE handler and tests to buffer assistant deltas by phase, suppress commentary deltas, and assert phase metadata on completed output items.

## Best Possible Solution

Keep the PR open and revise the streaming handler to maintain a final-answer-only visible buffer while preserving phase-tagged structured output, then add mixed-phase coverage and real SSE proof.

## Solution Assessment

No. Buffering by phase is the right direction, but the implementation needs a separate visible final-answer buffer and mixed-phase regression coverage before it is the maintainable fix.
