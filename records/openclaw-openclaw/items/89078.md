  - body: `canSubmitTuiChatMessage` now defaults an omitted `queueMode` to steer, but the runtime `canSubmitChatMessage` caller still does not pass `state.sessionInfo.queueMode`. Typed prompts in followup, collect, or interrupt mode can therefore remain blocked before `sendMessage` forwards them.

  - confidence: 0.93

  - confidence: 0.84
