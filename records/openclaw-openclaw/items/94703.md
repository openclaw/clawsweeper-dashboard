## What This Changes

## Best Possible Solution

Preserve the shell during transient reconnects, but reset the prior-success gate for structured auth failures and known 4001 auth/device revocation close reasons, with regression coverage and inspectable browser proof.

## Reproduction Assessment

## Solution Assessment
