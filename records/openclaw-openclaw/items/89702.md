## Reproduction Assessment

## Solution Assessment

Yes, if maintainers accept treating same-family macOS platform labels as display metadata like iOS/Android. The safer alternative is a locality-gated metadata-upgrade path for macOS node-mode, but that would be a different policy from the existing platform-family helper.
