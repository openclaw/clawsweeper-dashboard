## What This Changes

This PR changes `openclaw backup` archive timestamp formatting from UTC `Z` names to local-time names with a UTC offset, updates backup docs/help examples, and adds formatter coverage.

## Best Possible Solution

Keep UTC archive basenames as the default; if local-offset names are desired, add an explicit opt-in with docs, tests for both paths, and real backup-run proof.
