The PR changes backup archive basename generation to use a backup-specific local-time timestamp with an explicit UTC offset, updates backup docs and CLI examples, and adds formatter coverage.

## Best Possible Solution

Land the backup-specific local-offset timestamp contract with docs, CLI help, regression tests, and an Unreleased Fixes changelog entry, while leaving the shared session archive formatter unchanged.
