## What This Changes

The branch updates session-lock orphan classification so untracked current-process locks with matching starttime are reclaimed during acquisition and stale-lock cleanup, with regression coverage that preserves tracked active locks and fresh legacy current-PID locks.

## Best Possible Solution

Yes. ## Solution Assessment
