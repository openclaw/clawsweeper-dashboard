## Best Possible Solution

Land a provider-local Bedrock implementation that adds default Anthropic betas additively, composes with guardrails, service tier, and app-profile trait handling, and keeps configurable custom betas tracked by https://github.com/openclaw/openclaw/issues/39734.

## Solution Assessment
