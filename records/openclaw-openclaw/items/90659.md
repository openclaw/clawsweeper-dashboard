  - body: `runVoidHook` builds each handler argument from the same mutable event. If a conversation-access handler with higher priority copies `summary` into another property synchronously, the later metadata-only handler’s redacted copy will still include that new property. Build each handler payload from an immutable snapshot or clone/freeze per handler before invoking plugin code.
