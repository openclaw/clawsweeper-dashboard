Full review comments:

- **[P1] Preserve existing transcript permissions:** `src/transcripts/store.ts:137-141`
  - body: `writeTextAtomic` goes through `replaceFileAtomic` with a fixed file mode and the wrapper's default directory mode, and fs-safe v0.3.0 then chmods both the destination and parent directory.
