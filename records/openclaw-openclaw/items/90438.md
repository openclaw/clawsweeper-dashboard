- **[P1] Preserve explicit compact session files:** `src/context-engine/delegate.ts:67`
  - body: `delegateCompactionToRuntime` now omits `params.sessionFile` whenever a session key is present. A ContextEngine caller that supplies `sessionKey` plus an explicit `sessionFile` but no full runtime store context can be resolved through the configured/default session store instead of compacting the caller's file target.
  - confidence: 0.9
- **[P1] Use no-create opens for SQLite reads:** `src/config/sessions/session-accessor.sqlite.ts:104`
  - body: The read helpers call the normal agent database opener, which creates directories/files, applies schema DDL, caches the handle, and registers the DB. Preview-style reads need a no-create/read-only path so absent agent databases return empty results instead of being materialized.
  - confidence: 0.9
- **[P1] Require export before deleting transcript rows:** `src/config/sessions/session-accessor.sqlite.ts:751`
  - body: Deleting the `sessions` row cascades `transcript_events`, but this cleanup path only counts transcript state as archived and does not require a durable export receipt first. Once SQLite transcripts are live, lifecycle cleanup can silently drop transcript history.
  - confidence: 0.88
