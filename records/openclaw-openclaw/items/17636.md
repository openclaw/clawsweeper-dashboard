## Summary

Current `main` does not contain the requested `pnpm build:runtime` target, and I do not have strong enough evidence that an equivalent supported replacement has definitively landed. The closest current alternative is `build:docker`, but `build` remains separate and current `ciArtifacts` once again includes plugin-sdk declaration steps, so this PR should stay open rather than be closed as already implemented.
