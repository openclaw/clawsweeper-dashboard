## Best Possible Solution

## Solution Assessment

Yes, the proposed formatter-level normalization is the narrow maintainable fix for the observed `- ` double-prefix case. Broader Markdown marker handling can be considered separately if maintainers want to support indented, `*`, or `+` bullets too.
