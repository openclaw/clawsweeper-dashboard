- **[P1] Avoid timestamp-only stale-message rejection:** `extensions/feishu/src/dedup.ts:226-227`
  - body: The new create_time watermark rejects `createTimeMs <= lastTime`. Feishu create_time values can be second-normalized or delivered out of order, so legitimate same-second or delayed messages from the same chat/sender can be dropped before handling; add a message/event tie-breaker or a bounded retry-specific check.
  - confidence: 0.9
- **[P1] Scope contact cache rows by Feishu account:** `extensions/feishu/src/contacts.ts:22-23`
  - body: The contacts cache uses `open_id` as a global primary key and local search returns before account-specific sync. In multi-account setups, one tenant's contact can satisfy, prune, or drive send/calendar actions for another tenant; include account scope in schema, reads, writes, and prunes.
  - confidence: 0.88
- **[P1] Scope group cache rows by Feishu account:** `extensions/feishu/src/groups.ts:55-56`
  - body: The groups cache uses `chat_id` as a global primary key, and local list/search can return cached groups before syncing the requested account. In multi-account setups this can return or delete another tenant's group IDs; include account scope in schema, reads, writes, and cleanup.
  - confidence: 0.86
- **[P1] Preserve the held claim for direct abort messages:** `extensions/feishu/src/monitor.account.ts:373-381`
  - body: `dispatchFeishuMessageDirect` calls `handleFeishuMessage` without `processingClaimHeld: true`, but the receive handler already claimed the message ID before debounce. Normal stop messages can be treated as duplicates by finalization and return before aborting the in-flight run.
  - confidence: 0.84
- **[P2] Route ops tools through an explicit enabled account:** `extensions/feishu/src/feishu-ops-tools.ts:128-132`
  - body: The tools register after finding enabled Feishu accounts, but execute paths call contacts/groups/send/calendar helpers without an `accountId`. A multi-account config with a disabled or different default can fail or operate on the wrong tenant; select or require the intended account for each tool call.
  - confidence: 0.82
- **[P2] Gate proactive file sends explicitly:** `extensions/feishu/src/feishu-ops-tools.ts:64-65`
  - body: The new `feishu_send` file action accepts a local `file_path` and uploads it from the host. Bundling that tool without an explicit opt-in, path allowlist, or equivalent policy exposes a broad local-file upload surface to agent tool calls.
  - confidence: 0.78
