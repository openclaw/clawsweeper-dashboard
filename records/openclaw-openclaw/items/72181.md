Keep open. Current main still lacks the requested BlueBubbles trainer/supervised approval mode, and the feature changes private-message delivery plus durable correction logging, so it needs BlueBubbles maintainer/product/privacy review before an implementation PR is safe to queue.

## What This Changes

Request an opt-in BlueBubbles trainer/supervised mode that drafts replies after a full agent run, routes them to the owner for approval or correction, and blocks delivery to the original sender until approved.

## Best Possible Solution

Keep this BlueBubbles-owned; if maintainers approve the direction, implement schema-backed trainer settings, final-DM delivery interception, owner command handling, pending draft expiry, privacy-aware correction storage, docs, and focused delivery/config tests.

## Reproduction Assessment

## Solution Assessment
