## What This Changes

The issue reports that RPC-triggered post-core finalization can inherit stale source-config and compatibility-host environment variables from the running gateway process.

## Best Possible Solution

Land or reject the linked fix PR after review; once it merges, this issue can close as implemented by that PR.

## Reproduction Assessment

Yes. ## Solution Assessment

Yes. Clearing only the internal finalizer handoff env vars before adding current-run values is the narrow maintainable fix, and the linked open PR appears to target that boundary.
