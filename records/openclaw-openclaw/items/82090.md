## Summary

## What This Changes

The issue asks Slack replies that mention users to use Slack `<@USER_ID>` mention tokens instead of plain `@name` text.

## Best Possible Solution

Review and either land https://github.com/openclaw/openclaw/pull/82152 as the conservative prompt-guidance fix or replace it with a constrained Slack-local resolver that only mentions unambiguous known users.
