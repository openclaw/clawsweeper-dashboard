## What This Changes

The PR removes assistant-media `?token=` auth, changes Control UI local attachment rendering to use bearer-authenticated same-origin fetches with blob URLs, and adds gateway/UI tests plus docs and changelog coverage.

## Best Possible Solution

Land a reviewed version that rejects reusable credentials in assistant-media query strings while preserving local attachment rendering through authenticated same-origin fetches and regression coverage.
