The PR adds config CLI hint-selection logic and tests so active-agent config mutations can print a next-invocation hint while restart-required paths keep the gateway restart hint.

## Best Possible Solution

Land the narrow CLI hint change through normal maintainer review, preserving the conservative restart hint for non-agent, `agentRuntime`-touching, and mixed-batch mutations.

## Solution Assessment
