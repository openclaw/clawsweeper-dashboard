Extend the existing CI Windows gating in `.github/workflows/ci.yml` with a small `pwsh` task or matrix entry that runs selected high-value contributor commands such as `pnpm check:docs` and a minimal build smoke command, preserving the existing bash-based `pnpm test:windows:ci` lane.
