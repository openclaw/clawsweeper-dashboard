## Summary

Keep open. Current main still lacks the proposed service lifecycle harness, and the PR remains useful, but the current patch is not merge-ready because it exposes the new helper through the legacy testing barrel, diverges from production gateway_start context defaults, and lacks the required changelog entry. Land a revised version that keeps the generic harness and stateDir seam, exposes the helper through a focused SDK test subpath, mirrors production gateway_start context defaults, updates docs and API baselines, and records the user-facing SDK feature in the active changelog.

  - confidence: 0.9
