## Best Possible Solution

Land a warning-only doctor diagnostic after maintainer scope review, keeping runtime fallback semantics unchanged and aligning the warning predicate with the runtime resolver.

## Reproduction Assessment

## Solution Assessment

Unclear as written. The warning-only doctor direction is maintainable, but object-form warning scope is a product decision and the predicate should mirror runtime primary normalization before merge.
