Full review comments:

- **[P1] Resample Nova Sonic audio at the bridge boundary:** `extensions/amazon/nova-sonic/bridge.ts:74-82`
  - body: The bridge always treats input chunks as 8 kHz mu-law and declares the result to Nova Sonic as 16 kHz LPCM, then converts 24 kHz provider output directly to mu-law for every sink. `RealtimeVoiceBridgeCreateRequest.audioFormat` can also be 24 kHz PCM16, and sibling providers resample both directions, so this will produce wrong-rate audio in voice-call and gateway-relay paths.
  - confidence: 0.9
- **[P1] Do not stage dependencies from a bundled plugin:** `extensions/amazon/package.json:13-14`
  - body: This adds `openclaw.bundle.stageRuntimeDependencies` for a bundled plugin even though current dependency-resolution docs say bundled plugin manifests must not request dependency staging. With three new AWS SDK runtime clients, this needs an external package shape or explicit maintainer packaging exception before bundling.
  - confidence: 0.92
- **[P2] Mirror AWS env metadata into setup providers:** `extensions/amazon/openclaw.plugin.json:3-6`
  - body: The new manifest only uses deprecated `providerAuthEnvVars`, while current manifest docs say new plugins should put setup/status env metadata on `setup.providers[].envVars`. Starting the AWS provider on the compatibility surface creates immediate setup/auth-status migration debt.
  - confidence: 0.86
- **[P2] Declare bundled startup activation explicitly:** `extensions/amazon/openclaw.plugin.json:1-2`
  - body: Current bundled plugin metadata tests require every bundled manifest to declare `activation.onStartup` as a boolean. This new manifest omits `activation`, so the bundled metadata contract will fail once `extensions/amazon/openclaw.plugin.json` is included.
  - confidence: 0.88
- **[P2] Remove or use the Nova Sonic model list:** `extensions/amazon/nova-sonic/realtime-voice-provider.ts:9-10`
  - body: `SUPPORTED_MODELS` is declared but never read. `tsconfig.extensions.json` enables `noUnusedLocals` for extension source, so the extension typecheck will fail until this constant is removed or used for validation.
  - confidence: 0.94
