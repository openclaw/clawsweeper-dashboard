  - body: `createBridge` receives the caller's `audioFormat`, but this bridge always treats input as 8 kHz mu-law and then declares it to Nova Sonic as 16 kHz PCM; it also converts Nova Sonic PCM output straight to mu-law without resampling.   - confidence: 0.92

  - body: This bundled plugin asks OpenClaw to stage runtime dependencies even though current dependency docs say bundled plugin manifests must not request dependency staging.
