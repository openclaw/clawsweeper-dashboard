## Summary

## What This Changes

The issue asks Control UI/WebChat to stop rendering failed hidden internal tool calls as prominent chat errors when the assistant turn has a normal final reply.

## Best Possible Solution

Review and land https://github.com/openclaw/openclaw/pull/90122 if maintainers accept its projection-boundary fix, while keeping the separate direct-chat dispatch work in https://github.com/openclaw/openclaw/pull/89975 independent.

## Solution Assessment
