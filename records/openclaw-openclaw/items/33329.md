## Summary

Keep open. Current main has provider-level discovery toggles for Bedrock, GitHub Copilot, Hugging Face, and Ollama, but the public docs still only document Bedrock discovery and there is still no global model-discovery disable control.

## What This Changes

Document and expose controls to suppress implicit Bedrock, GitHub Copilot, Hugging Face, and Ollama model discovery, including a later-requested global disable switch.

## Best Possible Solution

Publicly document all plugin-owned discovery toggles, then add a maintainer-approved global model-discovery disable control with explicit precedence rules.

## Reproduction Assessment

Yes. Static inspection on current main proves the provider toggles exist, public docs only document Bedrock discovery, and the strict ModelsConfigSchema has no global discovery-disable field.

## Solution Assessment

Partially. Provider-owned plugin config is the right boundary for per-provider discovery, but the global kill switch needs a core config key and precedence policy before implementation is safe.
