## Summary

Keep open. Current main has runtime discovery gates for Bedrock, GitHub Copilot, Hugging Face, and Ollama, and Bedrock plus Copilot are documented, but Hugging Face, Ollama, and the shared config-tools reference still do not document the available opt-out path. The broader global/replace-mode discovery behavior remains separately tracked at https://github.com/openclaw/openclaw/issues/66957.

## What This Changes

Document and expose config toggles for implicit Bedrock, Copilot, Hugging Face, and Ollama model-discovery network calls.

## Best Possible Solution

Keep the plugin-owned runtime gates and add focused public docs for disabling Hugging Face and Ollama discovery, plus the shared config reference entries, while leaving global replace-mode semantics to the canonical related issue.
