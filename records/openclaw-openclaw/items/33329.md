Keep open. Current main implements plugin-owned `discovery.enabled` gates for Bedrock, GitHub Copilot, Hugging Face, and Ollama, and Bedrock plus Copilot now have public opt-out docs; Hugging Face, Ollama, and the shared config tools reference still lack the suppression path. Keep the plugin-owned discovery gates and add focused public docs for disabling Hugging Face and Ollama discovery, plus the shared config reference entries, while leaving global replace-mode behavior to the canonical related issue.
