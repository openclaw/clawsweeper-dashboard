## Summary

Keep open. Current main has the plugin-owned runtime gates for Bedrock, GitHub Copilot, Hugging Face, and Ollama discovery, and Bedrock plus Copilot have public opt-out docs, but Hugging Face, Ollama, and the shared config tools reference still do not document the suppression path. The broader global/replace-mode discovery concern is already tracked at https://github.com/openclaw/openclaw/issues/66957.

## What This Changes

Request to document and add per-provider config controls for implicit Bedrock, GitHub Copilot, Hugging Face, and Ollama discovery triggered by ambient credentials or local probes.

## Best Possible Solution

Keep the plugin-owned discovery gates and add focused public docs for disabling Hugging Face and Ollama discovery, plus shared config-tools entries for the provider discovery toggles; leave global replace-mode behavior to https://github.com/openclaw/openclaw/issues/66957.
