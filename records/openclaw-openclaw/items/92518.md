Full review comments:

- **[P1] Limit fallback to upload scope failures:** `extensions/googlechat/src/api.ts:278`
  - body: The helper returns true for any error message containing `403` or `PERMISSION_DENIED`, and the callers then send the remote URL as text.   - confidence: 0.91
