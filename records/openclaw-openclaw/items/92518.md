Overall correctness: patch is incorrect

Full review comments:

- **[P1] Limit fallback to upload scope failures:** `extensions/googlechat/src/api.ts:278`
  - body: The helper returns true for any message containing `403` or `PERMISSION_DENIED`, and the fallback then sends the remote URL as text. That is broader than the documented upload insufficient-scope case, so unrelated upload authorization, quota, membership, or policy denials can be reported as delivered; match the `Google Chat upload 403` insufficient-scope response and add negative tests.
  - confidence: 0.91
- **[P2] Preserve action send metadata on fallback:** `extensions/googlechat/src/actions.ts:169`
  - body: The normal media action path returns `...sent`, including `messageName` and `threadName`, but the fallback discards the result from `sendGoogleChatMessage`. Tool callers lose platform message/thread metadata only on fallback sends; return the send result alongside `{ ok, to }`.
