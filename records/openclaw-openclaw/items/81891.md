Keep open: the current main branch still has a credible SecretRef lifecycle gap for Google Meet realtime paths. Resolve Google Meet and voice-call realtime provider SecretRefs from the active runtime snapshot before provider config normalization, add the missing Google Meet SecretRef contract, and cover the command/tool paths with focused regression tests.
