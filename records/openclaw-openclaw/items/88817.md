## Solution Assessment

Yes, this is the right layer: converting only this known lookup miss at the agent tool boundary preserves direct RPC semantics while preventing channel warning surfaces from treating exploratory schema misses as tool failures. Suppressing channel chips alone would leave the model-facing tool result inconsistent.
