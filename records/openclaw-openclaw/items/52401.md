  - confidence: 0.9

  - body: This exposes a new public typed hook but does not update the hook catalog or conversation-access docs with its event shape, replacement semantics, timeout/access behavior, and proof expectations. Plugin authors need that contract before depending on it.
  - confidence: 0.82
