## Solution Assessment

Yes. The patch changes the owning extraction pipeline, bypasses only the min-length gate for glossary matches, and applies whole-token matching only to the short entries that need the bypass; broader alternatives like lowering the global minimum or whole-word matching all Latin glossary entries would regress existing behavior.
