Implement the requested WhatsApp disappearing-message expiration feature for https://github.com/openclaw/openclaw/issues/71157. Use https://github.com/openclaw/openclaw/pull/73080 and https://github.com/openclaw/openclaw/pull/73083 only as implementation context; do not edit `CHANGELOG.md`, and put release-note context in the PR body or commit message instead.
