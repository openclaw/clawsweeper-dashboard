Add configurable outbound domain allow/deny enforcement for OpenClaw-managed web tools.

## Best Possible Solution

Define an auditable outbound-policy contract with an explicit threat model and honest scope, then enforce it consistently at every OpenClaw-owned network chokepoint included in that contract while documenting paths that remain operator-controlled externally.
