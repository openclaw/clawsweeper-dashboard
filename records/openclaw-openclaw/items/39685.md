Keep open: current main and v2026.6.10 still do not expose a generic `allowedDomains`/`denyDomains` egress firewall for OpenClaw web tools. Add operator-configurable domain allow/deny egress controls for `web_fetch` and `web_search` using the existing SSRF policy primitives.

## Best Possible Solution

Keep this as the canonical web-tool egress-policy tracker until maintainers choose one public contract and land schema, runtime, cache, fallback, docs, audit, and search/fetch coverage around that contract.
