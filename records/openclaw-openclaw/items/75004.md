## Review Findings

Full review comments:

- **[P2] Convert Tree-sitter byte offsets before exposing spans:** `src/infra/command-explainer/extract.ts:118-120`
  - body: `web-tree-sitter` reports `startIndex` and `endIndex` as byte offsets, but the new `SourceSpan` values are used as JavaScript string indexes in tests and will be consumed that way by review surfaces. Any non-ASCII text before or inside a command can shift `source.slice(span.startIndex, span.endIndex)` and point at the wrong command; convert byte offsets to string offsets or make the span contract explicitly byte-based with matching callers/tests.
  - confidence: 0.9
- **[P2] Make the bash grammar dependency visible to Knip:** `package.json:1703`
  - body: Exact-head CI still has `check-dependencies` failing. The PR adds `tree-sitter-bash` as a production dependency, but the runtime resolves it only through a helper argument before `require.resolve`, leaving no static dependency reference for Knip; add an intentional static resolver/import or a scoped Knip ignore with a reason before merge.
  - confidence: 0.82
