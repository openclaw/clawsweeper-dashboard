Overall confidence: 0.88

Full review comments:

- **[P1] Keep Google Vertex ADC credential material:** `src/agents/model-auth.ts:345-349`
  - body: Returning oauth mode without apiKey makes resolveApiKeyForProvider report success, but auth-controller rejects non-aws-sdk modes with no apiKey before the Google transport can resolve ADC. Preserve the current marker/transport contract or update both sides together.
  - confidence: 0.88
- **[P2] Recheck converted HEIC image size:** `src/gateway/chat-attachments.ts:103`
  - body: This returns the converted JPEG without checking its byte length against the configured image caps. HEIC-to-JPEG conversion can expand the payload, so mirror the existing converted-size guard before returning it to the prompt path.
  - confidence: 0.9
- **[P2] Validate denyCommands against the full catalog:** `src/cli/config-cli.ts:354-363`
  - body: resolveNodeCommandAllowlist is the effective per-platform allowlist, not every valid node command. Using it here flags valid dangerous commands like camera.snap or screen.record as unknown when operators try to deny them.
  - confidence: 0.89
- **[P3] Assert the exact JS placeholder:** `src/auto-reply/reply/export-html/template.security.test.ts:89`
  - body: toContain("JS") still passes when only MARKED_JS or HIGHLIGHT_JS is present. Match the actual {{JS}} placeholder form so the test fails if the main script slot disappears.
  - confidence: 0.82
