## Summary

Keep open. The PR is still an incomplete fix: it changes only the first usage guard on an older branch shape, while current main still treats all-zero usage as missing in the resolver, totals, persistence, and status paths, and the PR still lacks after-fix real behavior proof.

The branch changes embedded assistant usage recording to accept any normalized usage object instead of only nonzero usage, and removes the now-unused `hasNonzeroUsage` import.

## Best Possible Solution
