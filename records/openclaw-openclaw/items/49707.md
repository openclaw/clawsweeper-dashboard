## What This Changes

The PR removes the nonzero-usage guard in `recordAssistantUsage` and drops the now-unused import on its branch.

## Best Possible Solution

## Solution Assessment

No. Removing only the first nonzero guard is not the best fix because downstream totals, accumulator, and persistence still conflate zero-valued usage with missing usage.
