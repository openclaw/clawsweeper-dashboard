## Summary

Keep this PR open, but do not merge it as-is. It targets the real Windows CLI lifecycle regression tracked in #74378, but a launcher-wide hard exit can preempt async version output and can truncate pending stdout/stderr output.

## What This Changes

The PR adds a final `process.exit(process.exitCode ?? 0)` to `openclaw.mjs` after launcher help handling or built entry import completes.

## Best Possible Solution

Revise the lifecycle fix so short-lived Windows npm commands terminate after cleanup while `--version`, help/JSON output, and foreground commands keep their existing output and lifetime guarantees.

## Reproduction Assessment

Yes for the PR regression: source inspection shows `openclaw --version` can finish entry module import before its async version output runs, then hit the new hard launcher exit. The underlying Windows hang has a concrete reporter repro in #74378, but it still needs Windows confirmation after revision.

## Solution Assessment

No. A launcher-wide `process.exit()` is broader than the bug and conflicts with the async root version fast path and Node's stream-drain guidance; a narrower cleanup or drained shutdown boundary is safer.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Preserve root version output:** `openclaw.mjs:317`
  - body: The new final `process.exit(...)` runs after `await import('./dist/entry.js')`, but the root `--version` path starts `resolveVersion().then(...)` and returns from module evaluation before that promise prints and exits. This can make `openclaw --version` exit 0 with no output; await that path or avoid a launcher-wide hard exit.
  - confidence: 0.91
- **[P2] Drain output before forcing launcher exit:** `openclaw.mjs:317`
  - body: This hard exit applies after help and routed command output. Node documents that `process.exit()` may terminate before pending stdout/stderr I/O completes, so piped help/status/JSON output can be truncated; fix the leaking handle or explicitly drain streams before exiting.
  - confidence: 0.86
