## What This Changes

The issue requests a model picker and drag-to-reorder fallback model selection in the Control UI Agents Overview page.

## Best Possible Solution

A maintainer should decide the Control UI UX contract, then implement a focused Agents Overview change that reuses the existing configured-model/catalog option source and preserves explicit fallback ordering through the current config Save flow unless auto-save is deliberately chosen.
