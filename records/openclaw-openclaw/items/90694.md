## What This Changes

The PR changes Amazon Bedrock memory credential detection so unconfigured auto-selection avoids AWS SDK IMDS probing while explicit Bedrock configuration can still use the default AWS credential chain.

## Best Possible Solution

Keep the Bedrock-local no-signal short-circuit, but isolate the two credential probe modes before merge so explicit Bedrock IMDS support remains reliable.
