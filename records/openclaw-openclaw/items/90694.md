## What This Changes

The PR changes the Amazon Bedrock memory credential probe to skip AWS SDK default-chain probing when no AWS credential signals exist, while allowing the full chain for explicitly configured Bedrock memory providers and adding regression tests.

## Best Possible Solution

Keep the plugin-local no-signal short-circuit and explicit-Bedrock default-chain path, then merge only after maintainers accept the documented explicit-config boundary or add a live AWS default-chain smoke.
