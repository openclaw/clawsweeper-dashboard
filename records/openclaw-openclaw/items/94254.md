## What This Changes

## Best Possible Solution

Review and land or replace https://github.com/openclaw/openclaw/pull/94256 so Apple app-password redaction is field/context-gated and the reported kebab identifiers are covered by regression tests.

## Reproduction Assessment

## Solution Assessment

Yes, the right fix direction is to narrow or context-gate Apple app-password redaction in `src/logging/redact.ts` and add focused redaction regression tests. The linked PR appears to be the active candidate, so this issue should stay open until that PR lands or is replaced.
