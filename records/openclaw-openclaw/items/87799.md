- **[P1] Redirect the normal gum spinner path:** `scripts/install.sh:459`
  - body: The fallback branches now use `/dev/null`, but the normal `gum spin -- "$@"` path just above still runs the child with gum's inherited stdin.   - confidence: 0.91

  - body: This blanket redirect applies to all shell-function fallback steps, including `install_homebrew` through `run_remote_bash`. Homebrew's installer treats non-TTY stdin as noninteractive and uses nonprompting sudo behavior, so a normal direct macOS install that needs Homebrew can stop prompting and fail; route only known noninteractive consumers to `/dev/null` and keep interactive installers on `/dev/tty`.
  - confidence: 0.88
