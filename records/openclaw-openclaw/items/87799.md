- **[P1] Redirect the normal gum spinner command:** `scripts/install.sh:453`
  - body: `run_with_spinner` still runs the primary gum path as `gum spin ... -- "$@"`, and gum v0.17.0 passes `os.Stdin` to the wrapped command.   - confidence: 0.92

  - confidence: 0.86
