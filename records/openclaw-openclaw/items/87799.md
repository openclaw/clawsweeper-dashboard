- **[P1] Preserve TTY stdin for Homebrew bootstrap:** `scripts/install.sh:513`
  - body: `install_homebrew` runs `run_remote_bash` through this shared fallback, and because shell functions skip the gum branch this new redirection feeds Homebrew's installer `/dev/null`. Upstream Homebrew switches to `NONINTERACTIVE` when stdin is not a TTY and uses non-interactive sudo behavior, so direct macOS installs without Homebrew can regress; make this wrapper/caller path preserve `/dev/tty` for interactive bootstrap scripts.
  - confidence: 0.89
- **[P2] Guard both gum raw-mode fallback reruns:** `scripts/install.sh:473`
  - body: This branch adds `/dev/null` to the failing-status raw-mode fallback, but the sibling raw-mode-warning fallback above still reruns `"$@"` with the original stdin. If gum emits a raw-mode warning while returning success, that rerun can still consume piped script bytes; apply the same stdin policy to both fallback branches.
  - confidence: 0.74
- **[P2] Split the labeler workflow change from this installer fix:** `.github/workflows/labeler.yml:145-153`
  - body: This `pull_request_target` workflow change makes size-label remove/add failures best-effort and warning-only, which is unrelated to the installer stdin bug.
