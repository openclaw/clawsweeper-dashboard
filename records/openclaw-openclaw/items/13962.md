## Summary

Keep open: current main and v2026.5.22 still have WhatsApp mention gating plus buffered group history, but not mention-state-aware model routing, background-model processing for skipped messages, or a per-group mention context size.

## What This Changes

The issue requests WhatsApp group controls for routing mentioned messages and unmentioned background context through different models, plus a per-group context window for later mentioned replies.

## Best Possible Solution

Design a maintainer-approved WhatsApp/group contract for mention-state-aware routing and per-group context sizing, then update config schema/types/docs and add focused tests for background processing, context visibility, and model-precedence behavior.
