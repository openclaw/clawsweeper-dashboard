## Summary

Keep open: current main and v2026.5.22 still only provide WhatsApp mention gating plus buffered group history, not per-mention model routing, background-model processing, or a per-group mention context window.

## What This Changes

Request per-group controls that route mentioned group messages to one model, unmentioned/background group messages to another model, and size the context window used when a mention triggers a reply.

## Best Possible Solution

Design a maintainer-approved WhatsApp/group contract for mention-state-aware routing and per-group context sizing, preserving group trust boundaries and model-precedence rules before implementation.
