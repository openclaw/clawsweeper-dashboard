## Summary

Keep open: current main and v2026.6.8 only buffer skipped WhatsApp group messages for later context; they still lack mention-state-aware model routing and per-group mention context sizing, and background processing of unmentioned group traffic needs maintainer product/security design.

## Best Possible Solution

Design this as an explicit WhatsApp/group capability or trusted pre-gating plugin/API hook that preserves safe defaults, defines routing precedence, and adds schema/docs/tests for background turns and per-group context limits.
