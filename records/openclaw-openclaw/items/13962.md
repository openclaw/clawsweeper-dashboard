## Summary

Keep open: current main and the latest release still lack mention-state-aware model routing and per-group mention context sizing for WhatsApp-style mention-gated groups. The existing path buffers skipped messages as context, but does not process unmentioned traffic through a background model, and the feature needs product/security design before implementation.

## What This Changes

Request to add per-group mention-state model routing and a configurable context window for group mentions.

## Best Possible Solution

Add a deliberate opt-in WhatsApp or shared channel contract for mention-state routing and per-group context sizing, with schema/docs updates, routing precedence rules, privacy boundaries, and focused tests.
