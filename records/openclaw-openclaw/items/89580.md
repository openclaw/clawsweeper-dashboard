This PR replaces JSON round-trip cloning with `structuredClone` in the agent code-mode worker and tool-search code-mode child bridge serializers.

## Best Possible Solution

The QuickJS side is source/dependency-proven as an unresolved risk because structured clone is a separate extension that OpenClaw does not load.

## Solution Assessment

No. A direct replacement is not the best fix unless both constrained runtimes are made to expose `structuredClone` and preserve the existing JSON-safe bridge contract.
