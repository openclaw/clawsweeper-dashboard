## What This Changes

Add Web UI support for selecting a cron job model during creation and visibly showing configured or default cron job models in cron review surfaces.

## Best Possible Solution

Implement a narrow UI parity change that reuses the existing cron `payloadModel`, model suggestions, and payload storage, without adding a new config key or runtime model-selection path.
