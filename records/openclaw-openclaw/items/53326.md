  - body: `readWorkspaceSetupStateForDir` falls back to `.openclaw/workspace-state.json` whenever the new root state has no markers, but `readWorkspaceSetupState()` rethrows non-`ENOENT` I/O errors.   - confidence: 0.88

Existing workspaces keep depending on `.openclaw/workspace-state.json`; mark fallback-loaded state dirty or explicitly persist it to the new path.
  - confidence: 0.86
