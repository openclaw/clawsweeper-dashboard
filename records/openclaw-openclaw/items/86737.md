## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P1] Do not collapse same-content external transcripts:** `apps/shared/OpenClawKit/Sources/OpenClawChatUI/ChatViewModel.swift:406-408`
  - body: `adoptLocalUserEcho` treats any incoming user `session.message` with the same content fingerprint as an echo of an existing row. Because that key ignores sender, message id, timestamp, and whether there is a confirmed pending local send, a second client saying the same text in the active group chat will replace the previous row instead of appending a distinct transcript message. Please only adopt a confirmed local echo and add a regression for same-content external user messages.
