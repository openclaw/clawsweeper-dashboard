Request plugin-defined SecretRef providers and broader schema coverage so users can reference external secret backends without repeated core schema PRs.

Review, land, replace, or close https://github.com/openclaw/openclaw/pull/72548 with a documented, security-reviewed plugin SecretRef provider contract, then track broad field coverage through focused follow-ups such as https://github.com/openclaw/openclaw/issues/72390.
