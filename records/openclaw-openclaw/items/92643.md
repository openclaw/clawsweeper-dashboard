## Summary

Keep this PR open, but it is not merge-ready: the central fallback direction is useful, yet the new direct text path can still send and mirror an unbounded child result, the added repro script currently fails lint, and the supplied proof does not show the real external-channel fallback behavior.

## What This Changes

The branch adds proactive and SessionWriteLock-error direct text delivery fallbacks for subagent completions, plus one regression test and a repro script.

## Best Possible Solution

Land a bounded direct-completion fallback that sends a concise capped or artifact-linked message, keeps transcript mirroring small, preserves generated-media recovery, and leaves the orphan self-lock cleanup as a separate tracked fix unless it is addressed here.

## Reproduction Assessment

## Solution Assessment

No, this is not yet the best way to solve the issue. The direction is plausible, but the fallback should be bounded before direct send/mirror, the added script must pass lint, and the separate orphan self-lock path should not be treated as resolved by this patch.
