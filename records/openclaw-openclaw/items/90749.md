## What This Changes

The PR moves realtime voice-call barge-in cancellation to provider `input_audio_buffer.speech_started`, keeps local MuLaw fallback for providers without that event, adds `emitsSpeechStartedEvent` provider/catalog metadata, and updates docs/tests.

## Best Possible Solution

Land the scoped provider speech-start cancellation after maintainer acceptance of the additive provider capability contract, while preserving local fallback for providers that omit the flag.
