## Summary

## What This Changes

The PR adds an in-memory provider/model circuit breaker to model fallback, a `circuit_open` failover status, test reset support, and focused unit coverage.

## Best Possible Solution

Add a bounded, scoped circuit integrated with existing auth-profile cooldown semantics, trip only on transient provider/model-wide failures, and prove skip plus recovery in a redacted real gateway/session run.
