## Summary

Keep open. ## What This Changes

Add an internal `session:end` hook event that fires when an agent session completes for workflow orchestration callbacks.

## Best Possible Solution

Define the supported session-completion lifecycle contract, then land one canonical implementation with docs and focused tests or document plugin `session_end` as the supported replacement and close the competing PRs.
