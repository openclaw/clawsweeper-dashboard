Keep this open. ## What This Changes

Fix WebChat so assistant text, tool calls, and tool results remain in execution order during live streaming without requiring a page refresh.

## Best Possible Solution

Represent live assistant segments and tool events with a shared monotonic sequence from the event ingestion path, preserve that sequence through chat-item construction, and add a regression proving that a multi-tool turn has identical visible order before and after history reload.
