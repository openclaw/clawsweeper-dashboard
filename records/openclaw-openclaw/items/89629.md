- **[P1] Carry `usageState` through routed durable deliveries:** `src/auto-reply/dispatch.ts:343`
  - body: This only attaches `usageState` from the direct dispatcher `beforeDeliver` path.   - confidence: 0.88

Current `resolveUsageProviderId` intentionally returns OpenAI usage only for OAuth/token and `undefined` for API-key auth, so an OpenAI API-key-backed turn can expose cached ChatGPT OAuth usage windows instead of no limits. Pass the selected auth profile or credential mode, or skip limits when the active turn used API-key auth.
  - confidence: 0.9
