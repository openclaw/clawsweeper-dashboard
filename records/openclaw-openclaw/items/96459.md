## Summary

## What This Changes

The PR adds a scoped `index-managers` mode to `closeActiveMemorySearchManager`, routes active-memory timeout cleanup through it, and teaches memory-core to release disposable index managers without closing the shared QMD manager.

## Best Possible Solution

Land this PR if maintainers accept the scoped lifecycle contract; otherwise ask for a separately named release facade while preserving the same QMD manager-liveness behavior and tests.
