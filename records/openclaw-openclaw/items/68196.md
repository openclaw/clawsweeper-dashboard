## Summary

Keep open. The late-reply compensation direction is still useful, but this branch is conflicting and predates current main’s terminal-timeout guard and delayed A2A delivery path, so it should not be merged as written.

## What This Changes

The PR adds a post-timeout history-read compensation helper and sessions_send tests to convert some wait-window timeouts into ok or accepted results.

## Best Possible Solution

Rebase onto current main and add only the missing late-reply compensation while preserving terminal-timeout handling and delayed A2A delivery.

## Reproduction Assessment

Yes. Source inspection and focused mocks can reproduce both the remaining late-reply gap and the PR regressions: make agent start succeed, make the first agent.wait return timeout, then vary terminal metadata, a post-baseline history reply, or no reply.

## Solution Assessment

No as written. The compensation boundary is narrow and plausible, but this patch must be rebased onto current main and must not turn terminal timeouts into accepted results or drop the delayed A2A wait path.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Preserve terminal timeouts before compensating:** `src/agents/tools/sessions-send-tool.ts:344-370`
  - body: Current main treats agent.wait timeout metadata such as endedAt, stopReason, or livenessState as a terminal run timeout. This PR compensates every timeout first and then returns accepted whenever chat.history succeeds, which can hide a hard run timeout from callers unless the history read itself fails.
  - confidence: 0.9
- **[P2] Keep delayed A2A alive after accepted compensation:** `src/agents/tools/sessions-send-tool.ts:364-370`
  - body: When no new reply is found, this accepted branch returns without calling startA2AFlow(undefined, runId). That drops the current delayed announce wait path, so a target run that finishes shortly after the first wait can complete without the existing follow-up delivery.
  - confidence: 0.87
