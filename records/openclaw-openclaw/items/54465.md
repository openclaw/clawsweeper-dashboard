## Summary

Keep open. Current `main` does not already contain this design doc or an equivalent namespace-plan implementation, and the underlying SDK shape the PR discusses is still present: the SDK export map is generated from a flat entrypoint list, `OpenClawPluginApi` is still a flat registration surface, and only `api.runtime` is namespaced. The PR also has an unresolved maintainer-direction discussion, so this is not conservative-cleanup closable.
