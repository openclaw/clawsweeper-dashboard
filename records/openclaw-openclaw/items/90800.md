The issue reports severe Chinese IME typing lag in the Control UI webchat textarea caused by synchronous textarea auto-resize work on each input event during composition.

## Best Possible Solution

Land a focused Control UI composer fix that coalesces textarea resize work during IME composition and proves smooth CJK input in real macOS Chrome/Safari testing.

## Solution Assessment
