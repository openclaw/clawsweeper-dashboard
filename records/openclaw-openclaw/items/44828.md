  - confidence: 0.93

  - body: The `!hasMedia` condition means a payload with `text: "NO_REPLY"` plus media still falls through to the normal text-send branch before media delivery. Treat the silent text as non-deliverable while continuing the media path.
  - confidence: 0.9
