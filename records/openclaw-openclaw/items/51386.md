Keep open. Current main and v2026.5.7 still keep session transcript indexing opt-in and experimental, while recent discussion and related open items show unresolved source/retrieval policy, privacy, QMD export, duplicate-indexing, and ranking questions that make this a maintainer product/defaults decision.

## What This Changes

Request to graduate experimental session transcript indexing into a stable, discoverable multi-channel continuity capability with privacy defaults, docs, and cost guidance.

## Best Possible Solution

Keep this as the sessionMemory graduation tracker until maintainers define the stable source/retrieval contract, privacy scope, QMD/export behavior, diagnostics, cost guidance, and docs, then land one coordinated code/test/docs change.
