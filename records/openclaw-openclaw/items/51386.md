Graduate `experimental.sessionMemory` to a stable/default-on memory-search capability so agents can recall transcript context across sessions and channels.

## Best Possible Solution

Keep this as the maintainer-owned graduation tracker until maintainers define the stable source, visibility, ranking, QMD, migration, privacy, and cost contracts, then land narrow fixes and docs before any default change.
