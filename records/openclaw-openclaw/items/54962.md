## Best Possible Solution

Land a rebased targeted fix that prefixes only generic callback fallback text, preserves approval/plugin/select/multiselect/native slash callback routes, adds regression coverage, and includes real Telegram proof.

Yes for the source-level routing contract: current main sends generic callback fallback text as raw data while the docs promise `callback_data: <value>`. I did not establish a live Telegram/webhook run in this read-only review.

## Solution Assessment
