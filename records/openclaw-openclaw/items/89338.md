Feishu MEDIA/image upload should resolve SecretRef-backed appSecret credentials before building the Feishu media client so Keychain/exec secrets still deliver images.

## Best Possible Solution

Land one canonical fix that resolves Feishu app credentials through the active async SecretRef runtime before all outbound media/client-build paths, with regression coverage for outbound, message-tool, and reply-dispatcher delivery.

## Solution Assessment
