## What This Changes

The issue requests investigation or post-upgrade health checks for host instability observed after OpenClaw upgrades, including a Linux Mint updater freeze and prior gateway/systemd problems.

## Best Possible Solution

Define any accepted work as a narrow OpenClaw-owned upgrade-health contract around update, doctor, gateway service state, restart handoffs, and clear diagnostics; keep Mint updater or arbitrary host-service probes out unless new evidence shows OpenClaw affecting them.

## Solution Assessment
