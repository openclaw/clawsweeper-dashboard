Add a docs-only fix for https://github.com/openclaw/openclaw/issues/66874. Document Docker-specific Claude CLI setup without changing Dockerfile, release workflows, package dependencies, config schema, or CHANGELOG.md. In docs/install/docker.md, and only if needed docs/providers/anthropic.md or docs/gateway/cli-backends.md, cover: installing @anthropic-ai/claude-code into a path that survives container replacement, persisting Claude Code state under ~/.claude or all of /home/node, using agents.defaults.cliBackends.claude-cli.command when the binary is not on PATH, running claude auth status --text and openclaw models auth login --provider anthropic --method cli --set-default inside the container, and using Anthropic API key auth as the fallback for production Docker.
