The issue requests `models.providers.<provider>.modelIdNormalization.prefixWhenBare` so native provider configs can send Cloudflare AI Gateway REST/Unified `author/model` wire ids.

## Best Possible Solution

Keep this issue open until maintainers choose the generic config versus Cloudflare bridge boundary, then land a revised implementation with schema/types/docs alignment, catalog/picker propagation, focused tests, and redacted Cloudflare proof.
