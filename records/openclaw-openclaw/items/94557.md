## Summary

## Best Possible Solution

Keep the issue open until maintainers choose the generic config versus Cloudflare provider bridge boundary, then land one aligned implementation with schema/types/docs, catalog propagation, tests, and redacted live proof.

Unclear. The config knob is plausible and narrow, but maintainers still need to decide whether the durable boundary is generic provider config or Cloudflare provider-owned bridge behavior.
