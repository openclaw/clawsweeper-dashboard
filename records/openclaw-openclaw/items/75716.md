## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Yes, if maintainers accept the compatibility change. The plugin-owned resolver and secret-contract cleanup are the right layer, and preserving shipped fallback is the safer alternative only if maintainers choose compatibility over documented precedence.
