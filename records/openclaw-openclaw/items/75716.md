## Summary

## Reproduction Assessment

## Solution Assessment

Yes, if maintainers accept the compatibility change. The plugin-owned resolver and secret-contract cleanup are the right layer; preserving shipped fallback is the safer alternative only if maintainers choose compatibility over documented precedence.
