## What This Changes

The PR adds optional `gateway.tailscale.preserveFunnel` config plus schema/docs/changelog updates, gateway startup wiring, Tailscale Funnel-status parsing, and tests so serve mode can skip reapplying Serve when an external Funnel route already covers the gateway port.

## Best Possible Solution
