## Summary

## What This Changes

The PR adds an `after_tools_resolved` plugin hook, emits resolved tool metadata from embedded agent attempts, and updates plugin docs, tests, API baselines, and changelog.

## Best Possible Solution

Land an additive resolved-tools plugin API after the new hook reuses the shared agent context helper, preserves cloned schema snapshots, and receives maintainer approval for the API shape.
