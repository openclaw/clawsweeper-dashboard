## Best Possible Solution

Ship one explicitly approved resolved-tools plugin contract that uses `buildAgentHookContextChannelFields`, provides immutable schema snapshots, documents per-attempt retry semantics, covers raw and normal session keys, refreshes generated SDK baselines, and leaves release notes to the release process.
