## Best Possible Solution

Land the focused formatter-boundary sanitizer with its regression tests after normal maintainer merge gates remain green.

## Reproduction Assessment

Yes, from source inspection. Current main strips only ESC before emitting a BEL-delimited OSC-8 sequence, so a BEL in the label or URL can terminate part of the generated terminal link; I did not execute code in the read-only checkout.

## Solution Assessment

Yes. Sanitizing at `formatTerminalLink()` covers the shared formatter and current docs-link callers, while reusing `sanitizeTerminalText()` would be too broad because it strips ANSI sequences and changes printable residue differently than the existing formatter contract.
