- **[P1] Preserve the cumulative stream baseline:** `ui/src/ui/controllers/chat.ts:694`
  - body: This trims `inFlightRun.text` down to the unpersisted suffix and stores that suffix in `state.chatStream`. Later `chat.delta` events carry a cumulative assistant snapshot, and `resolveDeltaChatStreamText` expects `chatStream` to match that snapshot prefix before appending; with only the suffix stored, the next delta falls back to the full snapshot and reintroduces text already present in history. The fix needs a canonical unpersisted-tail contract or a separate rendered-tail/cumulative-baseline reconciliation path.
  - confidence: 0.9
