## What This Changes

Prevent Codex app-server current-turn OpenClaw runtime context from being persisted as normal native Codex user history and replayed into later model and compaction inputs.

## Best Possible Solution

Keep the issue open until OpenClaw uses a supported non-persisted current-turn context path, or sanitizes stale decorated history while preserving the real request, with multi-turn and compaction regression coverage plus redacted app-server proof.
