## What This Changes

The PR records queued and active lane counts at enqueue time, appends them to lane wait warnings, and extends the command queue wait-warning test.

## Best Possible Solution

Keep the narrow diagnostic improvement, but default missing queue-entry counters for preserved legacy entries and add coverage for that shape before merge.

## Solution Assessment
