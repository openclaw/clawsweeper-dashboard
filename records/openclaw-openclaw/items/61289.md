## Best Possible Solution

Land the narrow alias mapping with a concise Telegram docs update and redacted live Telegram proof, preserving `mediaUrl` precedence and existing local-root guards while leaving the separate buffer/filename false-success bug to https://github.com/openclaw/openclaw/issues/41779.

## Solution Assessment
