Do not add a new config option and do not hard-code Feishu-specific behavior into core if a generic setup/activation seam covers it. Add regression tests for restrictive `plugins.allow`, browser status request behavior, and the channel setup interaction. Add a user-facing CHANGELOG.md fix entry.
