The PR raises the memory-core Dream Diary narrative timeout default to 240,000 ms, adds an `OPENCLAW_DREAMING_NARRATIVE_TIMEOUT_MS` env override, updates the timeout assertion, and adds an Unreleased changelog entry.

## Best Possible Solution

## Solution Assessment
