- **[P2] Document the maintenance commands:** `extensions/memory-lancedb/index.ts:873-904`
  - body: The patch registers user-facing `ltm reindex` and `ltm drop-index`, including a timeout default and a command that drops table indexes, but the Memory LanceDB docs still list only the existing commands. Users need the timeout/default and drop-index semantics documented before this ships.
