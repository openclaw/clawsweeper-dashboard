  - body: This registers user-facing `ltm reindex --timeout` and `ltm drop-index`, but the memory-lancedb docs still list only `list`, `search`, and `stats`. Add docs for the timeout default, empty-table result, and drop-index semantics before shipping the new maintenance surface.
