## What This Changes

The issue requests fixing the unscheduled-reminder guard so isolated cron jobs and same-turn cron adds preserved across compaction suppress the false disclaimer.

## Best Possible Solution

Land #43404 or an equivalent focused fix that passes agent identity into the guard lookup, treats enabled same-agent isolated cron jobs as coverage, preserves same-turn cron-add evidence across compaction retry cleanup, and adds regressions for both paths.
