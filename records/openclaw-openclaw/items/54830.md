- **[P2] Normalize bootstrap provenance source before rendering:** `src/agents/pi-embedded-helpers/bootstrap.ts:260`
  - body: This stores `file.name` directly in `provenance.source`, but the existing bootstrap helper tolerates malformed hook output where `name` is `undefined`. Normalize to a string fallback, such as the path basename, or omit provenance for malformed names.
