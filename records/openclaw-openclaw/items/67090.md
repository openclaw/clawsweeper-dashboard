The PR widens post-compaction AGENTS.md section extraction from H2/H3 headings to H1-H4 headings and adds regression tests for H1/H4 extraction, H1 boundaries, and nested headings.

## Best Possible Solution

If H1 support is accepted, narrow the accepted levels to the intended contract and update those surfaces together; otherwise keep the H2/H3 behavior and close this PR as not planned.
