## What This Changes

Request a Microsoft Teams plugin config/runtime path that enables native streaming for Copilot custom-engine conversations without enabling streaming for ordinary Teams DMs, groups, or channels.

## Best Possible Solution

Add a plugin-owned, backward-compatible msteams surface detector and Copilot streaming override after maintainers choose the config shape and validate the Copilot activity metadata contract.

## Solution Assessment

Unclear until maintainers choose the config contract. The likely best path is a narrow msteams plugin change, not a core streaming change, but the detector and schema need product/API review first.
