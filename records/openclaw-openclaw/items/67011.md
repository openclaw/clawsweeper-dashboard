## What This Changes

The PR changes diagnostic payload traversal to use active recursion-path tracking with WeakMap memoization and adds cache-trace regression tests for shared references.

## Best Possible Solution

Merge the focused active-path plus memoized sanitizer fix after redacted real diagnostic-output proof is added and the failed checks are rerun or triaged.

## Reproduction Assessment

Yes, source-level. Current main uses one traversal-wide WeakSet, so sanitizing a payload like `{ left: shared, right: shared }` visits `shared` once and then emits `"[Circular]"` for the sibling reference.

## Solution Assessment

Yes for the code diff. Active recursion-path tracking is the right cycle criterion, and WeakMap memoization avoids repeated traversal of shared DAG-shaped payloads without changing redaction policy.
