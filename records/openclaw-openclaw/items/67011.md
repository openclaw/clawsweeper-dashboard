## What This Changes

The branch changes diagnostic payload redaction to track only objects on the active recursion path, memoize sanitized objects, and adds cache-trace regression coverage for shared references.

## Best Possible Solution

Land the active-path plus memoized sanitizer fix after the contributor adds redacted real cache-trace proof and the draft is ready for maintainer review.

## Reproduction Assessment

Yes. Source inspection gives a high-confidence path: pass an options payload whose left and right fields reference the same object through current main's cache-trace sanitizer, and the second visit hits the traversal-wide WeakSet as circular.

## Solution Assessment

Yes, for the code change. Active recursion-path tracking plus WeakMap memoization is the narrow maintainable fix; moving this into downstream JSON stringification would leave sanitizeDiagnosticPayload callers with incorrect redacted data.
