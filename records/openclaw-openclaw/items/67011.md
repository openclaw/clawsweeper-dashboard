## Summary

Keep this PR open. Current main still has the traversal-wide object tracking that can mislabel shared sibling references as circular, while the latest PR branch is focused, addresses the earlier memoization concern, and has no blocking review findings from this read-only pass.

## What This Changes

The PR changes diagnostic payload redaction to use active recursion-path tracking plus a WeakMap memo and adds cache-trace regression tests for repeated shared references.

## Best Possible Solution

Land or lightly adapt the active-path plus memoized sanitizer fix after maintainer review, keeping the change scoped to diagnostic payload traversal and preserving the existing credential and image redaction guarantees.

## Reproduction Assessment

Yes. ## Solution Assessment

Yes. Active recursion-path tracking is the correct cycle criterion, and the added `WeakMap` memo is the safer version because it avoids repeated work on DAG-shaped diagnostic payloads while preserving sanitized outputs.
