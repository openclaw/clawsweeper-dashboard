## Summary

Keep open: current main still misclassifies shared acyclic object references during cache-trace diagnostic sanitization, and this PR is a narrow, correct fix, but the external contributor still needs after-fix real behavior proof before merge.

## What This Changes

The PR replaces traversal-wide `seen` tracking in diagnostic payload redaction with active-path tracking plus `WeakMap` memoization, and adds cache-trace regression coverage for shared acyclic references.

## Best Possible Solution

Land or lightly adapt the active-path plus memoized sanitizer fix after contributor proof and green or adjudicated checks, while preserving existing credential and image redaction behavior.

## Reproduction Assessment

Yes. Source inspection of current main shows `{ left: shared, right: shared }` reaches the traversal-wide `WeakSet` on the second sibling and is emitted as `[Circular]` through cache-trace diagnostic sanitization.

## Solution Assessment

Yes. Active recursion-path tracking is the right cycle criterion, and the PR's `WeakMap` memoization addresses the shared-DAG performance concern without broadening the sanitizer contract.
