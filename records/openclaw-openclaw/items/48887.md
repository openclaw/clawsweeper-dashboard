## Summary

Keep open. Current main still routes `pnpm check:docs` through one unbatched docs `oxfmt` invocation that is source-reproducibly too long for native Windows `cmd.exe`, while this PR adds a targeted batching fix and tests; the remaining blockers are real behavior proof and exact-head validation, not a code-review finding.

## What This Changes

This PR updates `scripts/format-docs.mjs` to use Windows-safe `oxfmt` invocation batching and adds focused tests for docs file discovery, batching, spawn specs, and launch failures.
