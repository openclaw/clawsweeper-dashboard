  - body: This writes OpenClaw-owned runtime queue state to `state/pending-messages.json`. The repo storage contract says runtime queues/state should use SQLite-backed stores, with old sidecars handled by doctor or migration code, so this adds a parallel source of truth with no upgrade owner.

  - confidence: 0.9

  - confidence: 0.86
