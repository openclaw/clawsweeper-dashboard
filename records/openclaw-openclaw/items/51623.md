  - body: `item.prompt` comes from queued channel messages, but the replay path formats it into a system event that later renders as `System:` lines.   - body: The hard five-minute age check discards persisted queues after ordinary slow downtime or repeated failed restarts, which reintroduces the message loss this PR is meant to prevent. Recovery needs a safer durable policy, not a short unconditional TTL.
