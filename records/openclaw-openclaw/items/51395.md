## What This Changes

## Best Possible Solution

Add an explicit opt-in custom upload or drop-file browser action that preserves upload-root validation, declares target and strategy semantics, and includes synthetic dropzone tests plus docs.

## Solution Assessment

Unclear. The requested mechanisms are plausible, but the safer maintainable direction is an explicit custom-upload action rather than a silent fallback after ordinary upload calls.
