## Best Possible Solution

Land a corrected, rebased route-scoped dedupe implementation that uses current helper names, defines or reuses a routed-provider resolver, carries origin thread context through the embedded subscription boundary, mirrors outbound message-send normalization, and includes real Telegram or equivalent runtime proof.
