  - body: `emitSessionRunCancel` records every cancelled target in `TERMINAL_CANCELS`, but this branch does not clear or bound that map in production. A long-running gateway can retain one entry per aborted run; add TTL/cap cleanup or wire cleanup into run teardown.
  - confidence: 0.86

  - body: `startGatewayEarlyRuntime` registers the module-global cancel requester even when `minimalTestGateway` is true. Minimal gateways can replace the live requester with test-local state, so plugin cancellation may route to the wrong gateway state.
  - confidence: 0.87
