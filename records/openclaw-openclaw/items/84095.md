## What This Changes

The issue requests that Feishu plugin tools such as feishu_doc be exposed to Feishu DM agent sessions when the channel uses documented multi-account credentials.

## Best Possible Solution

Make Feishu tool availability account-aware for named account credentials while preserving top-level credential support, tool policy filtering, and hiding tools when no usable Feishu credentials exist.

## Solution Assessment
