## Summary

## Best Possible Solution

Add a minimal workspace badge or breadcrumb beside the existing active-agent label using agents.list workspace metadata, with responsive truncation and full-path tooltip, while deferring optional dropdown, color/icon, and message-prefix settings.

## Solution Assessment
