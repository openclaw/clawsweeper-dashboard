## Summary

Keep open. Current main still hides trusted model lifecycle diagnostics from the public plugin diagnostic stream, and this patch is technically narrow with useful proof, but it adds a new public Plugin SDK telemetry contract that needs maintainer API/data-boundary approval before merge.

## What This Changes

The PR adds `onModelDiagnosticEvent` to `openclaw/plugin-sdk/diagnostic-runtime`, allowlists five trusted `model.*` event types, and updates focused tests plus SDK subpath docs.

## Best Possible Solution

Land the narrow subpath API only after maintainers explicitly approve the public telemetry contract and data boundary; otherwise route model observability through an existing permissioned plugin/service hook design.
