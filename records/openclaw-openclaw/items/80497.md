## What This Changes

Adds `onModelDiagnosticEvent` to the `openclaw/plugin-sdk/diagnostic-runtime` subpath, allowlisting five trusted `model.*` diagnostic events with tests and docs.

## Best Possible Solution

Land the narrow diagnostic-runtime subpath API if maintainers accept observability-only telemetry semantics; otherwise keep plugins on the existing `model_call_*` and `llm_output` hooks.
