## Summary

Keep open for review changes: the feature is useful and the failure mode is source-reproducible, but the current patch still exposes model telemetry through a plugin-forgeable trust gate and needs SDK documentation before merge.

## What This Changes

The PR adds `onModelDiagnosticEvent` as a diagnostic-runtime/compat SDK subscription for five trusted model lifecycle diagnostic events, with regression tests and a changelog entry.

## Best Possible Solution

Expose model diagnostics through a public SDK contract with non-forgeable core provenance, or explicitly document the stream as non-integrity telemetry and update the plugin SDK docs before merge.

## Reproduction Assessment

Yes. Current main source shows the broad plugin listener drops trusted events while the embedded runner and reply runner emit the relevant model lifecycle through trusted diagnostic paths; the PR body adds after-fix gateway logs showing delivery.

## Solution Assessment

No, not yet. A narrow opt-in listener is the right shape, but the current implementation still relies on `metadata.trusted` even though plugin code can publicly emit trusted diagnostics through the same SDK subpath.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Use non-forgeable provenance for model telemetry:** `src/infra/diagnostic-events.ts:934-941`
  - body: `onModelDiagnosticEvent` still treats `metadata.trusted` as runtime provenance, but `openclaw/plugin-sdk/diagnostic-runtime` publicly exports `emitTrustedDiagnosticEvent`. A plugin can emit an allowlisted `model.usage` or `model.call.completed` through that same public subpath and this listener will deliver it as trusted telemetry, corrupting cost/span attribution.
  - confidence: 0.91
- **[P3] Document the diagnostic runtime listener:** `src/plugin-sdk/diagnostic-runtime.ts:14`
  - body: This adds a public diagnostic-runtime SDK export, but the plugin SDK docs still only describe the subpath generically or as diagnostic gating helpers. Please document how plugins should import it and what trust limits apply to the stream.
  - confidence: 0.78
