- **[P1] Derive the OpenAI TTL from cache retention:** `src/agents/embedded-agent-runner/extensions.ts:151-153`
  - body: The new 30-minute floor is independent of the effective `cacheRetention` and model/provider policy.   - body: This script writes `Authorization: Bearer ${apiKey}` into a temp `headers.txt` file and never removes it. Even with `shell: false`, running the proof leaves the provider token on disk after exit; use `fetch`/SDK, or wrap temp files in `try/finally` cleanup with restrictive file permissions.
