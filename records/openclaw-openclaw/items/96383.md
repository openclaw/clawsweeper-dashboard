- **[P1] Align the OpenAI pruning floor with cache retention:** `src/agents/embedded-agent-runner/extensions.ts:151-153`
  - body: The new default-only 30-minute floor ignores the effective OpenAI `cacheRetention` path. Existing configs can use `cacheRetention: "long"`, which OpenClaw maps to `prompt_cache_retention: "24h"`, but this would still prune old tool-result context after 30 minutes when `contextPruning.ttl` is unset. Please derive the unset pruning TTL from the resolved cache-retention policy, or document and test a maintainer-approved shorter default.
  - confidence: 0.87
- **[P1] Avoid interpolating provider credentials into a shell command:** `scripts/repro/issue-95840-context-pruning-proof.mts:163-168`
  - body: This script places `baseUrl` and `apiKey` directly inside a shell command passed to `execSync`. That can expose the key in process arguments while `curl` runs and makes credential/base URL contents shell-sensitive. Use `execFile`/`spawn` argument arrays or the existing HTTP client path so the token is never interpolated through a shell.
  - confidence: 0.92
