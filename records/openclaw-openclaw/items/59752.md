## What This Changes

The PR changes auto-reply and followup delivery from blanket same-target reply suppression to text/media-level messaging-tool dedupe, adds regression tests, preserves followup account fallback routing, adjusts short exact-text dedupe, and adds a changelog entry.

## Best Possible Solution

Land this PR or an equivalent focused fix after restoring test registry state and adapting the branch to current main’s shared route-dedupe helper, then close #59743 as implemented with the existing changelog credit preserved.

## Reproduction Assessment

Yes. Current main has a deterministic unit-level reproduction: same-target messaging-tool metadata can set the suppression decision and drop all final payloads even when the final text differs from the messaging-tool text.

## Solution Assessment

No, not quite yet. Content-level text/media dedupe is the right maintainable direction, but this branch should fix the registry teardown leak and rebase onto the current route-dedupe helper before merge.
