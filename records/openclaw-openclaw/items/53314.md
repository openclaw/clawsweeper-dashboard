- **[P2] Honor completion hard expiry for pending descendants:** `src/agents/subagent-registry-lifecycle.ts:275`
  - body: The pending-descendants fast path only defers while `endedAgo <= ANNOUNCE_EXPIRY_MS`. For completion-message runs between the 5-minute announce expiry and the 30-minute completion hard-expiry, this falls through to `resolveDeferredCleanupDecision`, re-counts descendants, and can consume retry budget even though the announce was explicitly blocked by descendants. Carry the pending-descendants signal until the completion hard-expiry for completion-message runs.
  - confidence: 0.87
- **[P2] Restore finalize-error recovery for pending descendants:** `src/agents/subagent-registry-lifecycle.ts:415-417`
  - body: This new branch catches `finalizeSubagentCleanup` failures with an empty handler, unlike the normal finalize path that logs and clears `cleanupHandled` so the run can be retried. If the fallthrough give-up path, persistence, or hook work rejects, cleanup can remain handled but incomplete.
  - confidence: 0.82
