This PR imports `stripInboundMetadata` into the embedded runner LLM-boundary normalizer, strips user-message strings/text blocks there, updates the strip-helper docstring, and adds two focused unit tests.

## Best Possible Solution

Land a narrow replay-hygiene fix that strips historical user-turn inbound metadata, extends `stripInboundMetadata` for the delivery hint and chat-window/structured context blocks, preserves current-turn context, and proves it with helper/replay tests plus redacted real model-bound replay evidence.
