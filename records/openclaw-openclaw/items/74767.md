The issue requests that Feishu streaming-card replies be searchable in Feishu conversation search, or that the limitation be documented/worked around.

## Best Possible Solution

## Solution Assessment

Yes. A Feishu-specific opt-in fallback is the narrowest maintainable direction because the limitation is tied to Feishu interactive-card indexing, not a shared OpenClaw reply contract.
