The issue asks OpenClaw to make Feishu streaming-card reply content searchable in Feishu conversation search or document the limitation.

## Best Possible Solution

Decide the Feishu channel policy with live tenant proof, then either document the limitation or implement a Feishu-owned searchable fallback that preserves the intended streaming UX.
