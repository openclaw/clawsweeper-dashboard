## Best Possible Solution

Land the file-authoritative media runner fix after maintainer acceptance of the compatibility behavior, or narrow the local-whisper exception if core should remain known-command only.

## Reproduction Assessment

## Solution Assessment

Mostly yes. Making deterministic transcript-file outputs authoritative is the narrow repair for progress stdout contamination; the remaining question is whether maintainers want the local-whisper path exception in core.
