  - body: The new gate treats `hasAnyMention && !effectiveWasMentioned` as another agent being mentioned, but callers use generic platform mention signals such as Slack user/subteam mentions, Telegram `@username` entities, Teams mentions, and WhatsApp JIDs.   - confidence: 0.92

  - body: `pruneBundledDistPluginNodeModules` recursively removes `<extensionsDir>/<plugin>/node_modules` without validating that `extensionsDir` is a real package-owned dist directory. If `dist/extensions` is redirected, postinstall can delete outside the install tree; use the current dist realpath/lstat boundary checks or remove this helper.
  - confidence: 0.88

  - body: The PR branch's Discord preflight return payload drops `memberRoleIds`, while current main carries it into the built context and downstream routing/authorization paths. Merging this stale shape would lose sender role context for Discord inbound messages.
