## Label Justifications

- `P1`: The source-confirmed regression deterministically blocks real isolated cron workflows after an intentional generated-media yield.
- `impact:message-loss`: The generated-media callback arrives, but the remaining `sessions_send` or other workflow actions are suppressed.
- `impact:session-state`: The isolated cron session is recorded as aborted instead of remaining resumable for the callback.

## Summary

Current main still reproduces the reported lifecycle failure in source: an intentional `sessions_yield` abort can flow into the cron generated-media completion wait and become the exact `AbortError: aborted`, while two open PRs are competing to repair the remaining callback/session lifecycle. Keep the issue open as the canonical tracker until one complete fix is merged.

## What This Changes

Prevent isolated cron workflows from failing after `image_generate` and `sessions_yield`, and preserve the session so the generated-media callback can resume remaining actions.

## Best Possible Solution

Land one canonical repair that treats intentional yield as resumable control flow across the async-task wait, same-session generated-media callback continuation, and cron cleanup, with non-yield, timeout, cancellation, fallback-delivery, and delete-after-run regression coverage plus a real gateway cron proof.

## Reproduction Assessment

Yes at source level: current main can route the intentionally aborted `sessions_yield` signal into the cron generated-media wait, which constructs the exact reported `AbortError: aborted`. The reporter also supplied a deterministic runtime sequence, and the linked PR evidence includes before-fix confirmation, but this review did not run a new live provider reproduction.

## Solution Assessment

Yes, a bounded lifecycle repair is the right solution, not a new feature or configuration option. The complete fix should cover callback-resumable session lifetime and cleanup as well as skipping the already-aborted wait, so maintainers should reconcile the two open implementations before landing.
