The issue reports that agent-session image tool invocations lose or corrupt inbound image data, causing configured vision models to report that no image was provided.

Land a rebased or consolidated fix that preserves managed `media://inbound` references until the shared media resolver handles them, covers the agent-session `workspaceDir` path, and then close this issue with the related malformed-path report.
