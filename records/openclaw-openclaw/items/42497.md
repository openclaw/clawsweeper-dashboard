## Solution Assessment

  - body: This branch returns before `resolveGatewayOptions`, explicit scope handling, and approval-runtime token setup. Embedded calls that supply `gatewayToken`, approval methods, or explicit scopes can execute with different authorization than the existing WebSocket path.

  - confidence: 0.88

  - confidence: 0.88
- **[P2] Normalize params like the WebSocket frame:** `src/agents/tools/gateway.ts:154-156`
  - body: The WebSocket path validates and serializes a request frame, which drops object properties whose value is `undefined`; this local branch forwards the original params object directly. Optional fields that succeed over WebSocket can fail schema validation or behave differently in embedded local dispatch.
  - confidence: 0.78
