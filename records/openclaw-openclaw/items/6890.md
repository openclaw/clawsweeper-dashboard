## What This Changes

The issue requests a Ralph-style repeated-prompt loop with a per-agent max iteration default and explicit CLI, agent, global, default precedence.

## Best Possible Solution

Design an additive OpenClaw-native loop/default config contract, or a plugin/runtime API if maintainers reject core ownership, with strict schema, resolver/docs/runtime wiring, and tests for precedence, STOP, interrupts, disabled, finite, and unlimited behavior.
