## What This Changes

The PR adds `api.runtime.tasks.runs.*.lifecycle` create/progress/finalize APIs, exposes lifecycle task types through the Plugin SDK, updates task identity matching for `taskKind` and parent-linked rows, and documents plugin-owned task records.

## Best Possible Solution
