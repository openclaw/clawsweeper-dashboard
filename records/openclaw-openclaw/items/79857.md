- **[P2] Preserve Proxyline stream timeouts:** `src/infra/net/undici-global-dispatcher.ts:127-131`
  - body: When the global dispatcher is Proxyline-managed, this branch records the timeout and returns without applying it. @openclaw/proxyline@0.2.0 creates its managed Undici dispatchers without bodyTimeout/headersTimeout options, so global fetch/Undici streams can fall back to Undici defaults instead of OpenClaw's long-stream timeout.
  - confidence: 0.84
- **[P2] Keep HTTP/1-only reset dispatchers:** `src/infra/net/undici-global-dispatcher.ts:216-230`
  - body: `forceResetGlobalDispatcher()` now recreates Agent and EnvHttpProxyAgent without `allowH2: false`. Reset paths after proxy shutdown or env changes should keep the current HTTP/1-only hardening instead of restoring less predictable dispatcher defaults.
  - confidence: 0.88
- **[P2] Reject WebSocket proxy probe errors:** `src/infra/net/proxy/external-proxy.e2e.test.ts:424`
  - body: `websocketProbe()` still resolves on `error`, so this e2e test can pass when the proxy breaks the WebSocket upgrade. Reject the error so the test actually proves the managed WebSocket route connects through the operator proxy.
  - confidence: 0.9

  - body: Current main already exempts `@openclaw/proxyline@0.2.0`, but this PR adds an unversioned exemption too. That lets future Proxyline releases bypass the repository minimum-release-age guard automatically.
  - confidence: 0.93
