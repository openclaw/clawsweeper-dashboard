Overall confidence: 0.9

Full review comments:

- **[P2] Preserve Proxyline stream timeouts:** `src/infra/net/undici-global-dispatcher.ts:127-131`
  - body: When the dispatcher is Proxyline-managed, this records the timeout and returns. `@openclaw/proxyline@0.2.0` constructs plain Undici agents and exposes no timeout option, so global fetch/Undici streams can fall back to Undici defaults instead of OpenClaw's configured long-stream timeout.
  - confidence: 0.87
- **[P2] Keep HTTP/1-only reset dispatchers:** `src/infra/net/undici-global-dispatcher.ts:217-230`
  - body: `forceResetGlobalDispatcher()` now recreates direct and env-proxy dispatchers without `allowH2: false`. Reset paths after proxy shutdown or env changes should preserve the current HTTP/1-only hardening instead of restoring less predictable dispatcher defaults.
  - confidence: 0.88
- **[P2] Reject WebSocket probe errors:** `src/infra/net/proxy/external-proxy.e2e.test.ts:424`
  - body: `websocketProbe()` resolves on `error`, so this e2e can pass when the proxy breaks the WebSocket upgrade. Reject the error so the test actually proves managed WebSocket routing connects through the operator proxy.
  - confidence: 0.9
- **[P2] Pin the Proxyline release-age exemption:** `pnpm-workspace.yaml:26`
  - body: Current main already exempts `@openclaw/proxyline@0.2.0`; adding bare `@openclaw/proxyline` lets future Proxyline releases bypass the minimum release-age guard automatically, even though this package owns process-wide network routing.
  - confidence: 0.93
