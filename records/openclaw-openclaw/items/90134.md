## What This Changes

The issue reports that stable Docker Hub `alpine/openclaw` image tags stopped being published after `2026.5.27`, blocking stable Docker users from pulling `2026.5.28` or `2026.6.1`.

## Best Possible Solution

Make the stable image contract explicit and enforce it: either backfill and automate Docker Hub stable tags for `alpine/openclaw`, or document GHCR as the only supported prebuilt registry and steer Docker Hub users there.

## Reproduction Assessment

## Solution Assessment
