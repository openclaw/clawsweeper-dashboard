That drops explicit durable-memory requests such as `Can you remember that my backup day is Sunday?` or remember/recuerda requests ending in `?`; because this branch also captures only the latest user message, there is no later history pass to store the fact. Exempt explicit memory-request triggers before the generic question rejection.
  - confidence: 0.9

  - body: The bare `last`, `current`, and `recent` alternatives route non-temporal prompts such as `what is the current count?` into the 65% recency-weighted path. That widens recall and can make a newer but less relevant memory outrank the semantically best one, so require temporal/source context or drop the broad terms before applying recency reranking.
