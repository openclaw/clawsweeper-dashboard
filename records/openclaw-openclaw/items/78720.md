  - body: This says deletes run after targets and upserts, and that a still-referenced delete refuses to apply. Current `applyProviderPlanMutations` deletes first, then upserts, and provider mutations run before target mutation projection, so the published contract is wrong.

  - confidence: 0.9

  - body: This PR is scoped to the secrets plan contract, but this line changes avatar documentation already tracked by https://github.com/openclaw/openclaw/issues/65312 and its separate closing PR. Please remove it here or move it to that path.
