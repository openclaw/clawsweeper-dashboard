## What This Changes

The branch adds sentinel-token sanitization for bash exec, web-fetch, file/search tool arguments, a stream-level tool-argument sanitizer, tool-loop blocking, compaction budget changes, embedded-runner error handling, a portable-context Telegram command plugin, hallucination correction logic, and an operator PM2 wrapper.

## Best Possible Solution
