## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

No. The best fix is a narrow sanitizer at active tool boundaries; this branch also adds a separate sensitive Telegram export feature and changes shared guarded-fetch semantics, which are not the best way to solve the sentinel-token bug.
