## Summary

Keep open. The PR is the right narrow docs direction for the Notion allowlist bug, but the current helper snippet still needs a path guard before it is safe to land.

## What This Changes

The PR updates the bundled Notion skill docs and changelog to replace shell-local multiline `curl` examples with `NOTION_API_KEY` setup guidance and a file-backed `python3` helper flow.

## Best Possible Solution

Land the docs repair after adding a small helper guard that rejects paths outside `/v1/...`, keeping the fix limited to the Notion skill docs and release note.

## Reproduction Assessment

Yes. Source inspection on current main shows the documented `NOTION_KEY=$(cat ...)` plus multiline `curl` examples, and the exec allowlist docs/tests show why that command shape misses strict allowlists.

## Solution Assessment

No, not as written. The docs-only direction is the narrow maintainable fix, but the runnable helper should validate the `/v1/...` API path before constructing the request and attaching the token.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.92

Full review comments:

- **[P2] Reject non-Notion helper paths:** `skills/notion/SKILL.md:71`
  - body: The helper appends arbitrary `api_path` text to `https://api.notion.com` and then adds the bearer token. A copied command such as `.evil.example/v1/search` or `@evil.example/v1/search` can target another host with `NOTION_API_KEY`; reject anything that does not start with `/v1/` before constructing the `Request`.
  - confidence: 0.94

## Security Review

Status: needs_attention

Summary: The docs-only patch adds a copied helper that can send a Notion bearer token to a non-Notion host if called with a malformed path.

Concerns:

- **[medium] Unchecked helper path can misroute the Notion token:** `skills/notion/SKILL.md:71`
  - body: The helper builds `https://api.notion.com{api_path}` from CLI input and adds `Authorization: Bearer {api_key}`. Without a `/v1/` guard, malformed input can change the destination host before the token is sent.
  - confidence: 0.94
