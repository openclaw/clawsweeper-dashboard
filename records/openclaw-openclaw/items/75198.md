Full review comments:

- **[P1] Use a provider-scoped alias lookup:** `src/agents/model-selection-shared.ts:483-487`
  - body: This lookup asks the global `byAlias` map for `MoE`, but that map stores only one entry per alias string. Please search aliases by requested provider or index aliases by `(provider, alias)`, then add duplicate-alias coverage.
