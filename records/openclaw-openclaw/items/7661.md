Agree on a provider-owned channel-management authority model, then add Slack channel creation with explicit gates, required scopes, SecretRef-compatible credentials, runtime dispatch, CLI/docs, and focused tests while landing or replacing the narrower Discord CLI work separately.

## Solution Assessment
