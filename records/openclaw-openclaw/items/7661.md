Add message-tool and CLI support for creating channels, especially Slack workspace channels and Discord guild channels.

## Best Possible Solution

Keep this as the canonical feature until maintainers settle the authority model, then implement provider-owned Slack channel creation with explicit gates/scopes/docs while landing, replacing, or superseding the Discord CLI parity work.

## Solution Assessment

No automated implementation should start yet. The maintainable path is provider-owned channel creation after product/security review of gates, scopes, and defaults, not a generic core shortcut for administrative workspace changes.
