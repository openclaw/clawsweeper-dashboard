## Summary

Keep open. Current main supports Control UI image/non-video upload, paste/drop, preview/removal, and `chat.send` attachment serialization, but it still does not provide the requested screenshot/scissors button or region-capture entry point; that remaining work needs maintainer product and privacy review.

## What This Changes

Add a Control UI chat composer screenshot-capture button that captures a screen region, previews it, and sends it as an existing chat attachment.

## Best Possible Solution

If maintainers accept the feature, add a first-class Control UI capture control that reuses the existing attachment payload store, preview/removal UI, and `chat.send` serialization path with explicit permission, cancel, error, and privacy behavior.
