## Summary

Keep open. Current main and the latest release support Control UI image/file uploads through paste, file picker, drag/drop, previews, and `chat.send` serialization, but they still do not implement the requested screenshot/scissors capture button or region-capture flow.

## What This Changes

Request a Control UI chat composer screenshot capture button that captures a region, previews the image, and sends it through the existing attachment pipeline.

## Best Possible Solution

Keep the existing upload pipeline and, if maintainers want this UX, add a Control UI capture affordance that converts approved captures into `ChatAttachment` objects with explicit permission, cancel, preview, removal, and send tests.
