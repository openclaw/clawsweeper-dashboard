Full review comments:

- **[P1] Keep remote HTTP on a bound guarded fetch:** `packages/memory-host-sdk/src/host/remote-http.ts:23-24`
  - body: The plugin SDK embedding facade still re-exports package helpers, and `withRemoteHttpResponse` now falls back to `getMemoryHostServices().network.fetchWithSsrFGuard`; the default service throws unless a host binding is installed. Existing providers such as Google call this helper through the SDK without passing a guarded-fetch implementation, so remote embedding requests can fail at runtime instead of using the core SSRF-guarded fetch that main uses.
  - confidence: 0.91
- **[P1] Preserve AbortSignal forwarding in remote HTTP calls:** `packages/memory-host-sdk/src/host/remote-http.ts:13-18`
  - body: `withRemoteHttpResponse` used to accept `signal?: AbortSignal` and forward it to the guarded fetch on main. This PR removes that parameter from the public helper contract and never passes a cancellation signal to `MemoryHostGuardedFetch`, so existing remote embedding calls that rely on abort/cancellation can no longer stop in-flight requests.
  - confidence: 0.93
- **[P2] Use byte-sniffing MIME detection for memory files:** `packages/memory-host-sdk/src/host/services.ts:128-130`
  - body: The new default service ignores the supplied `buffer` and only maps a small image-extension set, but `buildFileEntry` relies on this hook to decide whether multimodal files are indexed. Standalone/default service users will drop valid audio/video or extensionless files and may accept renamed non-images as images instead of using the existing byte-sniffing detector.
  - confidence: 0.9
- **[P1] Scope service overrides per async operation:** `packages/memory-host-sdk/src/host/services.ts:82-107`
  - body: `withMemoryHostServices` writes a module-global `activeServices` value before awaiting arbitrary async work. Overlapping calls can observe each other's auth, network, redaction, or session hooks, and these override helpers are exported through package/plugin SDK surfaces.
  - confidence: 0.86
