Land this PR after maintainer review and required merge gates if validation remains green, then let the merge close https://github.com/openclaw/openclaw/issues/77307.

## Solution Assessment

Yes. Reusing `google-auth-library`'s own exported `Gaxios` and preserving `Request` data inside the existing SSRF-guarded transport is a narrow plugin-local fix with no core API or config change.
