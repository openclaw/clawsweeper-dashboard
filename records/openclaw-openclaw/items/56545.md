  - body: Returning `null` for `resolvedMode === "agent-turn"` makes the failure-destination path look unset; current gateway failure notification dispatch then falls back to primary announce delivery, so a configured agent-turn failure destination can still send an outbound alert instead of waking the agent.
  - confidence: 0.86

  - body: The new branch enqueues the event for `params.job.agentId` but calls `requestHeartbeatNow` without an agent or session target. Current heartbeat requests support scoped `agentId`/`sessionKey`; an unscoped wake can run unrelated agents and trigger extra autonomous work.
  - confidence: 0.88
