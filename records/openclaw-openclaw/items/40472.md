Overall correctness: patch is incorrect

- **[P2] Suppress pending chat runs on reset:** `src/tui/tui-event-handlers.ts:201-213`
  - body: Current main records the generated `chat.send` run id in `state.pendingChatRunId` before the first chat event arrives.   - confidence: 0.82
