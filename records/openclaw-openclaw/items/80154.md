The PR adds a Gateway-only `--timeout-ms` option to `openclaw infer model run`, validates it, and forwards it to both the Gateway client wait budget and agent timeout parameter.

## Best Possible Solution

Keep the scoped Gateway timeout flag, and add the same flag to `model.run` capability metadata with focused CLI test coverage before merge.

## Solution Assessment
