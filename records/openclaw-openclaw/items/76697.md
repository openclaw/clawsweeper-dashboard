## Summary

Keep open: the live official ClawHub package for Codex is still not stable-installable, while current source metadata and generated docs still advertise ClawHub as a Codex distribution route.

## What This Changes

The issue requests that `openclaw plugins install clawhub:@openclaw/codex` install the official Codex plugin on OpenClaw 2026.5.2 instead of failing with no installable version.

## Best Possible Solution

Publish and tag a stable ClawHub artifact for `@openclaw/codex`, or remove the ClawHub route from Codex release metadata and generated docs until stable ClawHub packaging is ready.
