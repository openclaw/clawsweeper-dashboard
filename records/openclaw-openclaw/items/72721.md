## What This Changes

The issue requests a Feishu plugin fix for websocket-mode gateway startup/connection timeouts where SDK axios calls to `open.feishu.cn/open-apis/bot/v1/openclaw_bot/ping` hang on Alibaba Cloud ECS with Node.js 24 while native HTTPS works.

## Best Possible Solution

Fix the Feishu plugin boundary with a consistent transport and timeout policy for startup identity probing, setup/status probing, and SDK HTTP calls, while preserving WebSocket as a supported default and webhook mode as an operational workaround. Maintainers should review, repair, or replace PR #67466 with deterministic SDK transport coverage or redacted live Feishu proof before closing this issue.
