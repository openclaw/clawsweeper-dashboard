## Best Possible Solution

If accepted, add an explicit memory-core preview/recovery command that reads selected session transcripts, groups candidate summaries by date, preserves provenance and redaction, and writes reviewed daily notes or short-term candidates instead of raw automatic ingestion.

## Solution Assessment
