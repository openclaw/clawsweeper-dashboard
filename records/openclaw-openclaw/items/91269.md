## What This Changes

Request a feature to recover missed daily memory by extracting conversation content from `.openclaw/agents/main/sessions/*.jsonl` and organizing it into dated memory files.

## Best Possible Solution

Decide whether memory-core should add an explicit, opt-in recovery/preview command that reads selected session transcripts, summarizes them into dated daily notes with provenance, and reuses existing dreaming/search paths rather than copying raw transcripts into memory.

## Solution Assessment
