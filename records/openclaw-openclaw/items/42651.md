## Summary

Keep open: current main has adjacent memory ingestion, promotion, dreaming, wiki, and alternate LanceDB memory work, but it still lacks the requested default remember/recall/review/edit/forget operator contract and structured recall match explanations. The related open memory MVP issues are prerequisites and coordination points, not a complete supersession.

## What This Changes

The issue asks for ingestion helpers and a small public memory operator surface with remember, recall, review, edit, and forget commands plus provenance and match explanations.

## Best Possible Solution

Define one v1 default memory operator contract, then implement the selected CLI/skill/tool surface with conservative write/review/edit/forget semantics, provenance-preserving storage, structured match explanations, docs, and focused tests.

## Reproduction Assessment

Not applicable as a bug reproduction; this is a feature request. The missing surface is source-verifiable on current main because the default memory CLI and tools expose only the existing search/read/promotion/dreaming commands described above.

## Solution Assessment

Unclear until maintainers settle the default memory contract. Adding commands directly is plausible, but the safer path is to resolve the write, review, edit, forget, and conflict semantics in the related memory MVP work before exposing public operator verbs.
