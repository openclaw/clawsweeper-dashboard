## Summary

Keep open. ## What This Changes

Requests deterministic bootstrap context injection through dynamic bootstrap content, session-type filtering, and built-in time context.

## Best Possible Solution

Define one maintainer-approved bootstrap context contract that separates deterministic file selection, safe dynamic context, and cache-stable time access, then reconcile it with the open tiered-bootstrap PR and narrower HEARTBEAT.md/arbitrary-file trackers.

## Solution Assessment
