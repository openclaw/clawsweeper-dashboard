Requests dynamic bootstrap-file context, configurable per-session bootstrap filtering, and built-in current time/timezone context for agent sessions.

## Best Possible Solution

Define an owner-approved bootstrap context contract that covers session-kind filtering, a documented dynamic context route, and stable time access without breaking prompt caching or plugin boundaries.

## Solution Assessment
