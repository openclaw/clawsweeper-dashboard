## Best Possible Solution

Preserve Feishu IDs for agent context and reply targeting, then add narrow user-facing sanitization plus regression coverage for copied Feishu `[message_id: ...]` and adjacent sender-label scaffolding.

## Solution Assessment

Yes. The narrowest maintainable fix is a targeted user-facing sanitizer or Feishu reply transform that preserves structured metadata, not a new default-visible `showMetadata` option for internal scaffolding.
