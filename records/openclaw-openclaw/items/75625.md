## Summary

Keep this PR open for maintainer review. Current main still has the image/video provider-inventory visibility bypass, and this PR is a focused implementation with regression coverage and changelog coverage; the remaining decision is whether OpenClaw wants this stricter behavior that suppresses provider inventory even in full verbose tool-output mode, especially because #75550 is an open competing fix that preserves verbose output.

## What This Changes

This PR changes the agent media tool-result handler to suppress image/video provider-inventory list output whenever `details.providers` is present, adds image/video hidden and verbose suppression regression tests, and adds a changelog entry.

## Best Possible Solution

Keep one canonical fix for #75166: decide the intended verbose/debug behavior for media provider inventory, then land either this stricter handler suppression or the narrower #75550 variant with the focused media-handler tests and changelog entry.

## Reproduction Assessment

Yes. ## Solution Assessment

Unclear pending maintainer policy. The code change is narrow and matches this PR's stated goal, but it intentionally hides inventory even in full verbose mode while #75550 preserves verbose output, so maintainers should choose the desired operator/debug behavior.
