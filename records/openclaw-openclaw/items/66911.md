  - body: `--json` logs the cleanup plan before mutating and later logs a separate `{ ok, removed }` result after the write.   - confidence: 0.92

  - confidence: 0.87

  - body: The JSON dry-run warning tells users to pass `--force`, but the command does not define a `--force` option and the non-dry path has no force escape. Either add a deliberate force flag or remove that remediation text so scripts do not surface an impossible next step.
  - confidence: 0.84
