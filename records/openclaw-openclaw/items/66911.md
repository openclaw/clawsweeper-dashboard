## Summary

Keep open. Current main does not implement the cleanup command, and the latest PR head still has a blocking JSON contract regression: a mutating `--json` run emits two top-level JSON documents. The author appears to have addressed earlier runtime-only external-profile feedback, so the remaining blockers are narrow.

## What This Changes

The PR adds `openclaw models auth clean` for pruning stale auth profiles, plus CLI registration, auth-store helpers, sanitizer coverage, tests, and a changelog entry.

## Best Possible Solution

Keep the feature, but make `--json` return a single stable result payload and add models CLI docs before normal maintainer review or merge.

## Reproduction Assessment

Yes. Static inspection gives a high-confidence reproduction: run `modelsAuthCleanCommand({ json: true })` with at least one removable profile and it logs the plan JSON, then logs a separate result JSON object after the write.

## Solution Assessment

No. The feature direction is reasonable, but the current patch should preserve the one-document `--json` CLI contract and document the new command before merging.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Emit one JSON document for --json:** `src/commands/models/auth-clean.ts:197-213`
  - body: `--json` logs the cleanup plan before mutating and later logs a second `{ ok, removed }` object after the write. A successful command therefore writes two top-level JSON values to stdout, which breaks scripts that parse the command result as one JSON document. Build and emit a single payload after the write, with dry-run and no-op paths using the same shape.
  - confidence: 0.92
- **[P3] Document the clean subcommand:** `src/cli/models-cli.ts:413-419`
  - body: This registers a user-facing auth cleanup command, but the models CLI docs still list only `add`, `login`, `setup-token`, and `paste-token`. Add `clean` plus its `--dry-run`, `--json`, and `--agent` behavior so the published CLI reference matches the new surface.
  - confidence: 0.87
