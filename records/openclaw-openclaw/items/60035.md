Keep open. The issue requests first-class system/custom HTTP and SOCKS proxy configuration across CLI config, environment fallback, agent overrides, tool overrides, and runtime HTTP/browser plumbing.

## Best Possible Solution

Keep this open for a maintainer-approved staged proxy design that extends the shipped HTTP/HTTPS managed proxy and provider-request proxy baseline without weakening SSRF, credential redaction, browser egress, or upgrade behavior.
