## Summary

Keep open. Current main has meaningful proxy work now, including managed HTTP/HTTPS proxy routing, ALL_PROXY-aware Undici env proxy options, provider env-proxy helpers, and documented browser guardrails, but the requested first-class SOCKS top-level config, noProxy/bypass contract, per-agent/per-tool/browser proxy fields, and security-reviewed credential policy are still missing.

## What This Changes

Request first-class OpenClaw proxy configuration for system and custom HTTP/SOCKS proxies, including env fallback and per-agent/per-tool overrides.

## Best Possible Solution
