Full review comments:

- **[P1] Honor explicitly rooted bootstrap globs inside ignored dirs:** `src/agents/workspace.ts:1192-1193`
  - body: The new walker skips any directory whose path contains an ignored segment before trying to match candidates. Existing configs can legally set `bootstrap-extra-files.paths` to workspace-relative globs such as `dist/**/AGENTS.md` or `.cache/**/TOOLS.md`; current `main` passes those globs to `fs.glob`, but this PR resolves the walk root to `dist` or `.cache`, skips it, falls back to the literal glob string, and drops the configured bootstrap context. Keep broad `**` scans from descending into these dirs, but still honor a user glob whose explicit root is under one of them.
