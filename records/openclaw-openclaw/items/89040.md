- **[P1] Preserve configured glob matches by default:** `src/agents/workspace.ts:1238`
  - body: The new walker now prunes ignored directory names for broad configured patterns, so `**/AGENTS.md` no longer reaches directories such as `dist` or `node_modules` unless the segment is named literally.   - confidence: 0.86

  - body: Returning as soon as `visitedEntries` exceeds 20,000 keeps the event loop healthier, but it also keeps only whatever configured glob matches happened to be found before filesystem order hit the cutoff. That can omit user-configured context and change session grounding; please make the cutoff opt-in/narrower, or document and get maintainer acceptance for the upgrade-visible behavior before merge.
  - confidence: 0.8
