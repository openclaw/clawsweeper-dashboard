## Summary

Keep open for maintainer review: the PR addresses a real source-reproducible bootstrap-context event-loop stall, but replacing `fs.glob` semantics and interleaving ordered bootstrap hooks remain compatibility/session-state tradeoffs that need owner acceptance before merge.

## Solution Assessment
