Full review comments:

- **[P1] Report capped extra-bootstrap matches before omitting context:** `src/agents/workspace.ts:1257-1258`
  - body: The new hard cap stops after 128 extra-bootstrap matches but returns the shortened list with no diagnostic. Existing users whose configured pattern matches more AGENTS.md/TOOLS.md files will silently lose injected bootstrap context after upgrade, and the hook only logs diagnostics when the loader reports them. Please preserve the configured files or surface a tested truncation diagnostic/warning before dropping context.
  - confidence: 0.92
- **[P2] Count walked entries before claiming a glob visit cap:** `src/agents/workspace.ts:1244-1249`
  - body: In the fs.glob branch, visitedEntries increments only when the iterator yields a matching path. Node's glob implementation traverses directory children internally and yields result paths, so a huge workspace with few or late matches can still traverse far beyond 20,000 entries and will not yield every 256 walked entries. Please enforce the limit in a walker that counts actual visited entries, or remove the claim for the fs.glob path.
  - confidence: 0.86
