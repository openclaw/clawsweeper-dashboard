## Review Findings

Full review comments:

- **[P1] Report capped extra-bootstrap matches before omitting context:** `src/agents/workspace.ts:1257-1258`
  - body: This hard cap returns after 128 matches for existing `bootstrap-extra-files` globs such as `packages/*/AGENTS.md`. The documented hook currently loads configured workspace-relative paths without a cap, and the caller only logs diagnostics returned from `loadExtraBootstrapFilesWithDiagnostics`, so packages beyond the cap would silently disappear from Project Context in monorepos. Surface a warning/diagnostic or preserve existing behavior unless maintainers explicitly accept the cap.
  - confidence: 0.88
- **[P3] Await routing before recording bootstrap timing:** `src/agents/embedded-agent-runner/run/attempt.ts:1279-1292`
  - body: `resolveAttemptWorkspaceBootstrapRouting` is async and awaits `isWorkspaceBootstrapPending`, but this wrapper records `bootstrap-routing` in `finally` before the returned promise settles. The new trace can therefore hide a slow routing probe; make the wrapper async and await the routing call before recording the substage duration.
  - confidence: 0.9
