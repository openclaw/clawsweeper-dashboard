- **[P1] Honor explicit globs rooted in ignored dirs:** `src/agents/workspace.ts:1192-1194`
  - body: The new walker skips `currentRelativeDir` whenever it contains an ignored segment. For a configured glob such as `node_modules/*/AGENTS.md`, `resolveGlobWalkRoot()` starts at `node_modules`, this branch skips the root before matching, and the configured bootstrap context is lost even though current main’s `fs.glob` path would match it. Please only prune ignored directories for broad traversal, or otherwise preserve user-targeted globs and add regression coverage.
  - confidence: 0.9
- **[P1] Gate traversal truncation behind an upgrade-safe path:** `src/agents/workspace.ts:1213-1217`
  - body: Returning as soon as `visitedEntries` exceeds 20,000 means existing broad `bootstrap-extra-files` globs can silently inject only the files found before that filesystem order point, with the warning emitted after context has already been dropped. Because this is a public config surface, maintainers need an explicit compatibility decision plus docs/tests, or the default behavior should preserve configured context while making bounded scanning opt-in or otherwise upgrade-safe.
  - confidence: 0.82
