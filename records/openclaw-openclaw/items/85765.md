## What This Changes

The issue reports Telegram rendering Anthropic extended-thinking content after the final assistant message, exposing internal reasoning and sending many small follow-up messages.

## Best Possible Solution

Keep Telegram reasoning hidden by default, then define and test one maintainer-approved visible-reasoning path that either uses the transient `/reasoning stream` preview or emits one coalesced durable reasoning message before the final answer.
