## Summary

Keep open: current main still lacks the requested opt-in invisible Anthropic non-turn keep-warm ping with adaptive timing, dirty cancellation, cache-hit verification, and cost cap. The related warmer proposal at https://github.com/openclaw/openclaw/issues/70418 was closed as a duplicate of this canonical issue, and the open session-maintenance PR at https://github.com/openclaw/openclaw/pull/56575 is adjacent compact/reset work rather than the requested ping mode.

## What This Changes

Request an opt-in Anthropic prompt-cache keep-warm mechanism that sends invisible non-turn pings between user turns to avoid TTL expiry.

## Best Possible Solution

If maintainers accept the tradeoff, add a documented opt-in Anthropic maintenance mode that reuses normal payload policy and cache telemetry, performs no transcript/tool/compaction activity, cancels on dirty/user-turn events, and enforces a hard cost cap.
