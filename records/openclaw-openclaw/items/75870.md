## What This Changes

Request support for Signal Note-to-Self linked-device messages so a personal Signal account can chat with the OpenClaw Signal channel without a separate bot number.

## Best Possible Solution

Review #75890 as the canonical path; if accepted, implement the Signal-owned Note-to-Self carve-out with regression tests and docs while preserving loop protection for third-party sent transcripts and direct own-account echoes.

Yes.
