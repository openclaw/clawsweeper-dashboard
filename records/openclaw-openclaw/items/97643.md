  - body: Dropping the unscoped read here means a browser profile that only has the shipped `openclaw.control.settings.v1` record no longer migrates its saved Control UI settings. `loadSettings()` will return defaults before any later save can copy the legacy data, so users upgrading from a pre-scoped profile can silently lose their saved Gateway URL, appearance/layout settings, and session selection.
