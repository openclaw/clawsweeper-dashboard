## Best Possible Solution

Add narrowly scoped auto-login guidance to the English macOS VM source doc, avoid broad generic runtime auto-login advice unless it is explicitly isolated-VM scoped and tested, and let the i18n pipeline regenerate localized docs.

## Reproduction Assessment

## Solution Assessment
