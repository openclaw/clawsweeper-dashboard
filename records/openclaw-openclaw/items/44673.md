## What This Changes

This PR adds macOS VM auto-login guidance to the English and zh-CN VM docs and adds an auto-login hint to the macOS LaunchAgent bootstrap error.

## Best Possible Solution

Add tightly scoped English macOS VM guidance after Setup Assistant with an isolated-VM caveat, leave localization to the publish/i18n pipeline, and keep runtime LaunchAgent diagnostics focused on logged-in session or LaunchDaemon guidance unless maintainers approve broader auto-login copy.
