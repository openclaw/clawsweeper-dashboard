## What This Changes

Request a context/session pruning hook that fires after tool-result pruning with metadata usable for follow-up indexing or search.

## Best Possible Solution

## Solution Assessment

Unclear. A pruning lifecycle hook is plausible, but maintainers still need to settle event naming and the privacy-safe payload shape before changing the current `pi-hooks` path.
