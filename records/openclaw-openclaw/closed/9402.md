## Summary

Close as implemented_on_main. Current main already provides a pre-LLM routing seam through the plugin hook system: `before_model_resolve` can override provider/model before resolution on each turn, and `before_agent_reply` can claim or silence trivial turns before any LLM call. The behavior is documented, wired into the runtime, covered by tests, and present in the latest release commit.

## Evidence
