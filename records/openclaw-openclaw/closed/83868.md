## What This Changes

The PR removes a second merge inside `FeishuStreamingSession.update()` and suppresses `onPartialReply` updates after Feishu block delivery takes over a streaming card.

## Best Possible Solution

Branch streaming updates the same way `close()` already does: append suffixes only when `nextText` starts with `sentText`, replace the card content for non-prefix rewrites, add focused regression coverage, and require redacted live Feishu proof before merge.
