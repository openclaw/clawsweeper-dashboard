## Summary

Current `main` already implements the underlying Kimi/OpenAI malformed tool-call argument preservation logic in a dedicated repair module, and it does so more conservatively than this PR by preserving valid/preexisting args while still invalidating stale repaired args when later deltas make the payload invalid.

## Evidence
