## Summary

Current main already contains a shipped fix for ByteString crashes caused by pasted non-Latin1 credential characters such as U+2502 (`│`). Shared secret-input normalization now strips those characters before auth values are used, and regression tests explicitly cover the reported `ByteString` failure shape.

## Evidence
