## Summary

Current `main` already supports heartbeat session routing and isolated heartbeat execution. The config schema accepts both `agents.defaults.heartbeat.session` and `isolatedSession`, the runner resolves explicit heartbeat session keys, and isolated runs execute in a dedicated `:heartbeat` session with regression coverage. The issue's documented-feature gap is no longer present on `main`.
