## Summary

Current `main` already contains the desktop exec-approval overflow fix, and the changelog/appcast show it shipped in `v2026.4.23`. The approval UI now caps the modal height, scrolls long command content inside the prompt, and collapses multiple pending approvals into a single active modal with a queue badge.
