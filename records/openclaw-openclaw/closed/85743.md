## Summary

## What This Changes

The issue requests a heartbeat replay fail-safe that clears orphaned pending final delivery state after a retry cap or age TTL instead of replaying cached final text indefinitely.

## Best Possible Solution

Review and land or reject https://github.com/openclaw/openclaw/pull/85770 as the canonical narrow fix, then close this issue once the bounded replay behavior is on main.
