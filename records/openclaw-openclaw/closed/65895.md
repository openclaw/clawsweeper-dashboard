## Summary

Current main and the shipped v2026.4.26 docs already solve the PR's central discoverability problem. Amazon Bedrock Mantle is listed in the provider directory and Mintlify Models navigation, the dedicated provider page exists, and the bundled provider registers with docsPath `/providers/bedrock-mantle`. The remaining proposed `docs/providers/models.md` line is no longer required because that page is explicitly a starter set and points readers to the full provider catalog.

## Best Possible Solution
