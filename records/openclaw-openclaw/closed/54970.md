## Summary

Close #54970 as superseded by #69725. The underlying `exec host=node` bug is still real on current main, but this older PR is stale against removed `nodes run`/agent-node-run surfaces, while #69725 is an open, newer PR that targets the surviving `src/agents/bash-tools.exec-host-node.ts` path with focused tests and a local-plan fallback.

## Best Possible Solution
