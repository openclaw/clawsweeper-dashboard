## Summary

The reported ShieldCortex tool-name mismatch is not a current OpenClaw core item. Current main does not ship ShieldCortex, OpenClaw’s documented scope routes optional skills and bundle-style integrations to ClawHub/plugins, and the existing bundle/plugin system already supports packaging skills together with their own MCP tools. The concrete mismatch is tracked in the external ShieldCortex project, so the core issue can close as plugin-owned scope.
