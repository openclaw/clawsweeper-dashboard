## Summary

Current `main` already contains the Discord stale-socket reconnect crash fix, and the changelog shows it shipped in `v2026.3.28`. The current code now buffers/suppresses late `reconnect-exhausted` gateway errors during teardown/dispose instead of letting them surface as uncaught process-killing exceptions.
