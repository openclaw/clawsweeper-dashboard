## Summary

Current `main` already fixes the reported Opus 4.7 Anthropic thinking regression: Opus 4.7 is recognized as adaptive-thinking capable in the Anthropic and Anthropic Vertex transports, `/think xhigh` maps to `output_config.effort: "xhigh"`, the Anthropic provider advertises `xhigh`/`adaptive`/`max` for Opus 4.7, and regression tests cover those paths. `CHANGELOG.md` records the Opus 4.7 reasoning support in the `2026.4.18` release, and a later issue comment reports the original 400-plus-fallback behavior fixed on `2026.4.22`.
