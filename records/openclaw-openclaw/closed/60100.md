## Summary

Current main and v2026.4.24 route explicit memory embedding provider requests through the requested adapter only. `provider: "ollama"` no longer enters the auto-select loop that can walk Gemini/Voyage/Mistral, and `fallback: "none"` prevents fallback activation.
