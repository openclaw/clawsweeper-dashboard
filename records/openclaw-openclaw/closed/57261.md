## Summary

Current `main` already implements the requested behavior for `gateway install --force`: resolved `gateway.auth.token` SecretRefs are warning-only, unresolved refs still block install, and forced reinstalls now validate/build against merged daemon-runtime environment rather than only the current shell environment.

## Evidence
