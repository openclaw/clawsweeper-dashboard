## Summary

Current main implements both parts of the reported ask: Bonjour/mDNS can be disabled for headless/VPS deployments, and the current Bonjour plugin path suppresses the ciao `CIAO PROBING CANCELLED` watchdog rejection instead of letting it reach the process-fatal unhandled rejection path.
