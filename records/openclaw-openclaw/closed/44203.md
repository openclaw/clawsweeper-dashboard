## Summary

Current main and the shipped v2026.4.24 tag implement the edit-tool contract as canonical `path` plus non-empty `edits[]` entries containing `oldText` and `newText`, wrap host and sandbox edit tools with that validation, and include regression coverage for both path-only malformed calls and valid nested edit payloads.
