## Summary

Current `main` already supports ordered model fallback/autofailover via `agents.defaults.model.{primary,fallbacks}`, with runtime candidate chaining, failover execution, and test coverage in place. The same implementation is also present in the released `v2026.4.22` tag.

## Evidence
