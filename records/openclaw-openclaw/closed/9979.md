## Summary

Current `main` already supports Google Vertex AI for Gemini as the implicit `google-vertex` provider variant, with auth resolution, provider lookup, model normalization, and docs already present; this was also present in release `v2026.4.22`.

## Evidence

- **Bundled Google plugin exposes Vertex alias:** The Google provider plugin registers `hookAliases: ["google-antigravity", "google-vertex"]`, so `google-vertex` is a supported provider variant handled by the bundled Google plugin.
  - file: [extensions/google/provider-registration.ts:19](https://github.com/openclaw/openclaw/blob/754acc4478fe361db1e5303fd60f589d52beea7a/extensions/google/provider-registration.ts#L19)

  - file: [src/agents/model-auth-env.ts:27](https://github.com/openclaw/openclaw/blob/754acc4478fe361db1e5303fd60f589d52beea7a/src/agents/model-auth-env.ts#L27)
