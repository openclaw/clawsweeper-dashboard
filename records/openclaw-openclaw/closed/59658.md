## Summary

Keep open. Current main has some of the intended documentation fixes, but the linked sub-agent docs work is not fully implemented, and the PR is not merge-ready unchanged because current source/tests contradict several proposed or remaining statements.

## Best Possible Solution

Keep this PR open for now, but require a refresh against current main. The best path is one focused docs update that preserves ACP-only `streamTo`, documents `announceTimeoutMs` with the current 120000ms default, generalizes nesting/tool-policy/cascade wording for `maxSpawnDepth` 1-5, and accurately separates default always-denied tools from leaf-only denies and config overrides. After that lands, close #47558 and any duplicate PRs against the canonical fix.
