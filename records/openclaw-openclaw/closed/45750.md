## Summary

Current `main` already contains a shared gateway-call fix that directly targets CLI WebSocket handshake starvation and teardown races, and the cron CLI still uses that shared path. The fix landed after `v2026.4.22`, so it is on `main` but not tied to a released tag from the available evidence.
