## Summary

Current `main` contains a targeted heartbeat fix: the runner explicitly reads the workspace `HEARTBEAT.md`, appends a guardrail telling the model to use that exact workspace file instead of `docs/heartbeat.md`, and has regression coverage for that path guidance. That directly addresses the report that heartbeat was ignoring workspace HEARTBEAT instructions and falling back to `HEARTBEAT_OK`.
