## Summary

Current main already implements this request: the gateway pre-auth WebSocket handshake timeout now defaults to 10s and can be overridden with `OPENCLAW_HANDSHAKE_TIMEOUT_MS`, with tests and a shipped changelog entry in `v2026.4.22`.
