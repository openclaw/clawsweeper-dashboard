## Summary

Issue #45133 is already fixed on current `main` and the fix shipped after `v2026.3.12`. The reported TDZ came from a module-level `ANTHROPIC_MODEL_ALIASES` table in the `v2026.3.12` `src/agents/model-selection.ts` path. Fix commit `868fd32ee77424af5d0d27bba9ceb570b11fa929` replaced that lookup with inline normalization, added regression tests for dated Anthropic refs, and the current `main` keeps the safe implementation in `src/agents/model-ref-shared.ts` plus config/model tests covering `anthropic/claude-sonnet-4-20250514`.
