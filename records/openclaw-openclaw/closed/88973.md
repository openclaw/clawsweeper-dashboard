## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Yes for the code shape: the PR fixes the sink, keeps trusted internal authority Telegram-local, and normalizes durable inbound missing scopes to `[]`. The remaining question is maintainer acceptance and real runtime proof, not a narrower code repair.
