## Summary

Current `main` already implements the requested behavior well enough. The local authenticated probe path now keeps loopback probes device-bound so detail RPCs do not get scope-limited, regression coverage asserts that authenticated local probes return `ok: true`, and the docs/release notes describe scope-limited probe results as degraded reachability rather than full failure.
