## Summary

Close as `clawhub`: the PR adds a repo-root continuity bridge package for an external AiRYA sidecar, but current `main` policy and docs route MCP servers and similar optional surfaces to external plugins/bundles published via ClawHub, and OpenClaw already ships a generic MCP bridge surface with `openclaw mcp serve`.

## Evidence
