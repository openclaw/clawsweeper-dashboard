## Summary

Close #53402 as a duplicate of #61285. Current main still lacks native macOS Simplified Chinese localization: menu/settings strings remain hardcoded in Swift and no macOS localization resources are present. #61285 is the open tracker for the same remaining macOS menu bar/settings localization gap; existing Control UI zh-CN support is a separate web surface.

## Best Possible Solution

Keep the native macOS Simplified Chinese localization work consolidated in #61285, or the broader maintainer localization tracker if maintainers prefer that canonical issue. The eventual fix should add a real macOS/SwiftUI localization path for menu, settings, onboarding/permissions strings, and native permission text rather than treating web Control UI zh-CN support as sufficient.
