## Summary

Current main still does not implement the PR's intended Telegram `/think` native menu fix. The shared command menu path passes only provider/model context, while Ollama's extra thinking levels depend on catalog `reasoning` metadata; the PR remains a focused implementation candidate for #73515.

## Best Possible Solution

Keep this PR open as the active implementation candidate for #73515. The best path is to review and land a catalog-aware shared command menu fix that threads configured model catalog metadata through `CommandArgChoiceContext`, keeps Telegram generic, and preserves the regression test for configured Ollama models with `reasoning: true`.
