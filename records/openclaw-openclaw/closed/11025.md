## Summary

Current main already auto-downscales/recompresses oversized images before provider calls, including browser screenshots, prompt/channel attachments, tool-result image blocks, and replayed session history. The same implementation is present in the `v2026.4.22` release tag for the relevant code paths.

## Evidence
