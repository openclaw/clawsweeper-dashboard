## Summary

The remaining ask is not missing core routing; it is optional cross-session/context recall on top of existing channel/session plumbing. Current main exposes plugin hooks and context-engine/memory plugin surfaces that can inject dynamic context before the model call, and the documented Honcho plugin already covers cross-session memory across channel switches. This should be handled as a ClawHub/plugin capability unless a future issue identifies a specific missing OpenClaw plugin API.
