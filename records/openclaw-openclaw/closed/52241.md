## Summary

Issue #52241 requests WeChat-specific behavior for Tencent's external openclaw-weixin channel plugin, not a missing OpenClaw core capability. Current main documents WeChat as an external plugin, keeps WeChat runtime/API behavior outside core, and the channel plugin SDK already makes inbound message handling channel-owned. This is appropriate for the external @tencent-weixin/openclaw-weixin plugin or ClawHub, so the core repo item can close.
