## Summary

Current main already implements direct MCP server configuration for agents: OpenClaw stores named server definitions under `mcp.servers`, exposes them through `openclaw mcp` and `/mcp`, and the embedded Pi agent runtime loads those definitions and connects to the servers as tools.
