## Summary

Close as implemented: current main already propagates resolved Feishu sender names into the direct-chat session label path that WebChat reads, and the same implementation is present in release `v2026.4.23`.

## Evidence

- **Feishu ingress resolves sender names when enabled:** `resolveSenderNames` gates a best-effort profile lookup, and successful lookups populate `ctx.senderName` before inbound context is finalized.
  - file: [extensions/feishu/src/bot.ts:342](https://github.com/openclaw/openclaw/blob/51dd4f288f67ef4df4c7a121f078f508535df1fd/extensions/feishu/src/bot.ts#L342)
  - command: `nl -ba extensions/feishu/src/bot.ts | sed -n '320,380p'`
  - sha: [51dd4f288f67](https://github.com/openclaw/openclaw/commit/51dd4f288f67ef4df4c7a121f078f508535df1fd)
- **Inbound Feishu context forwards SenderName into session metadata flow:** The finalized inbound context for Feishu DMs/groups includes `SenderName: ctx.senderName ?? ctx.senderOpenId`, so downstream session labeling has the friendly name available.
  - file: [extensions/feishu/src/bot.ts:997](https://github.com/openclaw/openclaw/blob/51dd4f288f67ef4df4c7a121f078f508535df1fd/extensions/feishu/src/bot.ts#L997)

  - sha: [51dd4f288f67](https://github.com/openclaw/openclaw/commit/51dd4f288f67ef4df4c7a121f078f508535df1fd)
- **Direct-chat labels prefer SenderName over raw sender id:** Conversation-label derivation returns `SenderName` for direct chats and only falls back to `From` when no display name is available.
  - file: [src/channels/conversation-label.ts:39](https://github.com/openclaw/openclaw/blob/51dd4f288f67ef4df4c7a121f078f508535df1fd/src/channels/conversation-label.ts#L39)

  - sha: [51dd4f288f67](https://github.com/openclaw/openclaw/commit/51dd4f288f67ef4df4c7a121f078f508535df1fd)
- **Derived conversation label is persisted to session origin.label:** Session metadata derivation calls `resolveConversationLabel(ctx)` and writes the result into `origin.label`, which is the field the report says WebChat reads.
  - file: [src/config/sessions/metadata.ts:63](https://github.com/openclaw/openclaw/blob/51dd4f288f67ef4df4c7a121f078f508535df1fd/src/config/sessions/metadata.ts#L63)
  - command: `nl -ba src/config/sessions/metadata.ts | sed -n '1,220p'`

  - file: [src/channels/conversation-label.test.ts:23](https://github.com/openclaw/openclaw/blob/51dd4f288f67ef4df4c7a121f078f508535df1fd/src/channels/conversation-label.test.ts#L23)
