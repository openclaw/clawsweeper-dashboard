## Summary

The reported crash is already addressed on current main and in released builds: OpenClaw now disables late exec `onUpdate` emissions after run exit/abort specifically to avoid the exact `Agent listener invoked outside active run` failure, and regression tests cover the exit/abort cases. The reporter’s affected version (`2026.4.5`) predates that fix; the first released version I found with the guard is `v2026.4.10`.

## Evidence

- **Current main contains an explicit guard for this exact crash:** `runExecProcess` tracks `updatesDisabled` and makes `emitUpdate` a no-op once the exec run has settled, with an inline comment naming the exact `Agent listener invoked outside active run` crash and issue `#62520`.
