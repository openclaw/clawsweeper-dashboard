## Summary

Viber is a useful channel request, but current OpenClaw scope and SDK make this a plugin/ClawHub item rather than core work. Current main does not bundle Viber, the channel plugin API supports webhook-based messaging channels, and a community Viber channel plugin by the reporter already exists.
