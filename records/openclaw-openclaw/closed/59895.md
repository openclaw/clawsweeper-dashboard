## Summary

Current OpenClaw already exposes Discord channel-history reads through the shared `message` tool, including automatic inference of the current Discord target in channel-context runs, and that implementation is present in the shipped `v2026.4.23` release.

## Evidence

- **Discord `read` action is wired to history reads:** The Discord message action handler maps `action === "read"` to `readMessages`, forwarding `limit`, `before`, `after`, and `around` to the runtime.
  - file: [extensions/discord/src/actions/handle-action.ts:184](https://github.com/openclaw/openclaw/blob/a9797214338ba31c52c796adbb75afb16e0684a9/extensions/discord/src/actions/handle-action.ts#L184)
  - sha: [a9797214338b](https://github.com/openclaw/openclaw/commit/a9797214338ba31c52c796adbb75afb16e0684a9)
- **Runtime fetches Discord channel messages:** `readMessages` calls `readMessagesDiscord(channelId, query, ...)` and returns the fetched messages to the agent/tool caller.
  - file: [extensions/discord/src/actions/runtime.messaging.ts:306](https://github.com/openclaw/openclaw/blob/a9797214338ba31c52c796adbb75afb16e0684a9/extensions/discord/src/actions/runtime.messaging.ts#L306)
  - sha: [a9797214338b](https://github.com/openclaw/openclaw/commit/a9797214338ba31c52c796adbb75afb16e0684a9)
- **Current-channel target is inferred automatically:** When a message action needs a target and none is supplied, normalization fills `target` from `toolContext.currentChannelId`, which covers reading the current Discord channel without manually passing `channel:<id>`.
  - file: [src/infra/outbound/message-action-normalization.ts:36](https://github.com/openclaw/openclaw/blob/a9797214338ba31c52c796adbb75afb16e0684a9/src/infra/outbound/message-action-normalization.ts#L36)
  - sha: [a9797214338b](https://github.com/openclaw/openclaw/commit/a9797214338ba31c52c796adbb75afb16e0684a9)
- **Documented as a supported Discord tool/action:** The Discord docs list `readMessages` among the supported messaging actions.
  - file: [docs/channels/discord.md:907](https://github.com/openclaw/openclaw/blob/a9797214338ba31c52c796adbb75afb16e0684a9/docs/channels/discord.md#L907)
  - sha: [a9797214338b](https://github.com/openclaw/openclaw/commit/a9797214338ba31c52c796adbb75afb16e0684a9)
