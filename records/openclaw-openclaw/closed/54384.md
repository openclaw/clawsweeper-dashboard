## Summary

Current `main` already fixes this Control UI regression path. The gateway now preserves image attachments for text-only session models by offloading them instead of dropping them, persists transcript `MediaPath`/`MediaPaths`, and the Control UI renders those persisted transcript images. That matches the MiniMax-vs-Codex symptom in #54384, where MiniMax sessions could lose thumbnails after the optimistic send because the session model was treated as text-only while Codex sessions kept image-capable history.

## Evidence
