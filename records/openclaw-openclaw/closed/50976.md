## Summary

Current `main` prepares Slack routing before dispatch, rewrites bound Slack conversations onto the bound session/agent, and adds streaming duplicate-delivery guards. The report matches older behavior but is now covered by current code, tests, and the follow-up streaming cleanup linked from the issue.
