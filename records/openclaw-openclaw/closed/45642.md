## Summary

Current `main` already avoids the reported `CopyCopied!` leakage: tool output renders in its own tool-card content block, while the copy control is now an icon-only button used on assistant markdown actions rather than inside tool-output text flow. The same implementation is present in released tag `v2026.4.22` (`00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5`).

## Evidence
