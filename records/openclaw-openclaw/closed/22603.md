## Summary

Current `main` already has Antigravity-specific Gemini 3.1 compatibility: the Google provider owns the `google-antigravity` alias, normalizes bare Antigravity Gemini 3.1 IDs to the required `-low` form, and resolves Gemini 3.1 pro/flash requests against Antigravity templates. The same implementation is present in the shipped `v2026.4.22` tag.
