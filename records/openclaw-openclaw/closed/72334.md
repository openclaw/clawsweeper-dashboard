## Summary

Current main addresses the specific v2026.4.24 follow-up crash: #71668 correctly fixed the plugin module-instance handler split, and current main adds the missing lifetime fix so CIAO process handlers stay registered across Bonjour advertiser watchdog restarts. The classifier covers `CIAO ANNOUNCEMENT CANCELLED` and `CIAO PROBING CANCELLED`, the global unhandled-rejection path returns before fatal exit when a registered handler handles the reason, and tests now cover the stuck/restart lifecycle. No release tag containing the follow-up fix is present in this checkout, so the fixed release is not known.

## Best Possible Solution

Close this issue as implemented on main and ship the current advertiser-lifetime CIAO handler registration in the next release. Keep #71668 as the earlier partial fix for plugin module-instance handler sharing; this follow-up covers the remaining late-cancellation-after-watchdog-restart failure mode reported here.
