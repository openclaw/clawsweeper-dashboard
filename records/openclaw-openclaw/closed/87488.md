## Best Possible Solution

Apply the same visible reply-chain policy to prompt media refs, add a group allowlist regression, and land with redacted live Telegram proof for the linked image-context issue.

## Reproduction Assessment

## Solution Assessment

Is this the best way to solve the issue? No, not as-is; the fix should gate hydrated prompt media refs on the same visible reply-chain policy and include live Telegram proof before merge.
