- **[P1] Avoid using tmpdir as the project-local PATH base:** `src/infra/path-env.ts:52`
  - body: When `process.cwd()` is unavailable, this substitutes `os.tmpdir()` for `cwd`; later the same helper appends `${cwd}/node_modules/.bin` when project-local bin probing is enabled.   - confidence: 0.88

Keeping this stale `safeCwd() ?? os.tmpdir()` shape during rebase would resolve relative entrypoint candidates from the temp directory instead of the module location, so drop this hunk or keep current main's helper.
