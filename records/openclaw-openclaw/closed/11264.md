## Summary

Issue #11264 asks for a new messaging integration, and current main explicitly routes that kind of work to the external plugin ecosystem rather than OpenClaw core. The repo documents channel plugins as the supported extension path and has an explicit maintainer auto-close reason for third-party plugin requests.
