## What This Changes

The issue reports that Memory Dreaming ranks high-scoring promotion candidates but writes none to MEMORY.md because every candidate fails rehydration.

## Best Possible Solution

Preserve the stale/deleted-snippet safety by default, add explicit skipped-rehydration counts and reasons, and only add fuzzy or range fallback behind a maintainer-approved conservative policy.
