Full review comments:

- **[P1] Avoid adding a second provider-config source of truth:** `src/config/zod-schema.ts:469`
  - body: This new top-level `llm` key re-expresses provider credentials, model catalog, default model, and params that are already supported through `models.providers` plus `agents.defaults.model`. Without an explicit maintainer-approved config contract and migration/tooling plan, this creates competing ways to configure the same auth/provider state.
  - confidence: 0.88
- **[P1] Preserve existing model defaults when seeding primary:** `src/config/defaults.ts:592`
  - body: If `agents.defaults.model` already contains `fallbacks` or other fields but no `primary`, this assignment replaces the whole object with `{ primary }`. Adding `llm` would silently remove failover behavior, so the existing model object needs to be spread before setting `primary`.
  - confidence: 0.91
- **[P2] Register the shorthand model before making it primary:** `src/config/defaults.ts:574-576`
  - body: When `provider_id` matches an existing provider, the branch preserves that provider unchanged but can still set `primary` to the shorthand model. If the model is not already in `models.providers[providerId].models`, runtime lookup will fail with a dangling model ref.
  - confidence: 0.88
- **[P2] Validate provider IDs before building model refs:** `src/config/zod-schema.core.ts:275`
  - body: `provider_id` only requires a non-empty string, but the value becomes the left side of `provider/model`. Values containing `/` or other non-round-trippable ref characters can create a provider key that later parses as a different provider/model pair.
  - confidence: 0.86
- **[P2] Validate provider protocols before casting them:** `src/config/zod-schema.core.ts:271`
  - body: `llm.provider` is free-form and then cast to `ModelProviderConfig['api']`, so typos like `openai-compatable` pass validation and fail later in provider transport selection. Restrict this to supported shorthands plus the existing `MODEL_APIS` values.
  - confidence: 0.84
