## Summary

Current main already fixes the false Feishu duplicate-id startup warning class by treating duplicate candidates that resolve to the same physical plugin directory as the same plugin and skipping the warning. Real distinct duplicate Feishu/global-vs-bundled copies are still reported by precedence, which is intentional and documented.
