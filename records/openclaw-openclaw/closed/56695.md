## Summary

Current main and the shipped `v2026.4.23` release already downgrade the Feishu tool-registration messages from `info` to `debug`, which resolves the reported excessive gateway log noise. The Feishu bundled channel still registers its tool set through `registerFull(...)`, but the specific noisy `Registered ...` lines cited in the issue are no longer emitted at info level.
