## Summary

Current `main` already contains the live-session/fallback protections needed for this report: fallback attempts now persist an `auto` session override before the attempt, the live-session reconciler suppresses and clears stale pending switches when the current model already matches persisted selection, and retry loops are bounded with regression tests covering the old ping-pong behavior.
