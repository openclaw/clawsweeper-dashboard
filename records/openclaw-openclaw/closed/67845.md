## Summary

Issue #67845 is a specific Chinese localization request for Telegram native command menu descriptions. Current main does not implement per-language Telegram command menus, but the request is superseded by the canonical i18n/localization tracker #3460, which explicitly says specific-language localization issues should be closed as duplicates and already tracks zh-CN. The reporter's follow-up is useful because it narrows the desired Telegram behavior to stable command tokens with localized visible descriptions.

## Best Possible Solution

Close #67845 as superseded by #3460. If maintainers later reopen localization work, the concrete implementation should extend the Telegram plugin's command-menu config/runtime to support language-keyed descriptions and call Telegram `setMyCommands` per `language_code`, with docs and tests for built-in, custom, plugin, and skill command descriptions.
