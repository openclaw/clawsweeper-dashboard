- **[P2] Gate MiMo replay through provider policy:** `src/agents/transcript-policy.ts:146`
  - body: This generic fallback runs for arbitrary custom OpenAI-compatible providers. Keying it on `modelId.includes("mimo")` means unrelated model names containing that substring stop stripping historical reasoning, so hidden thinking can be replayed where the current policy intentionally removes it. Use a Xiaomi/provider-owned hook or explicit replay capability instead.

Adding `reasoning_content` for assistant array-content messages sends an extra key to strict non-MiMo backends, which is the payload shape this path exists to avoid.
  - confidence: 0.86
