## Summary

Current `main` already isolates unresolved Feishu account SecretRefs from read-only account discovery, so a secondary `channels.feishu.accounts.*.appSecret` SecretRef no longer takes down the whole Feishu plugin during register/reload. The shipped `v2026.4.22` release contains the same implementation and regression tests.

## Evidence
