## Summary

This PR is superseded by the narrower merged fix in #73521. Current main now addresses the Discord read/search hang and inbound hook session-key fallback without taking this PR's broader target-schema changes, and this PR's diff also carried extension-boundary and compatibility risks called out in review.

## Best Possible Solution
