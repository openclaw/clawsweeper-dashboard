## Summary

Current `main` already streams tool start/update/result events and renders them live in Control UI with collapsible tool output; the changelog shows this shipped in `v2026.3.7`.

## Evidence

  - file: [src/gateway/server-chat.ts:932](https://github.com/openclaw/openclaw/blob/7e52223d327555c611e8c91ee2500897e9eab0bf/src/gateway/server-chat.ts#L932)
