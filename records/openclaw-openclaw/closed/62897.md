## Summary

Close as implemented on current `main`: the WhatsApp inbox path now forwards `@g.us` inbound messages into group chat handling, routes them onto `agent:<agentId>:whatsapp:group:<jid>` session keys, and current tests cover both inbound group upserts and group-session routing. The shipped `v2026.4.23` release also includes WhatsApp fixes for scoped group-session handling and auth backup recovery that match the symptoms in this report.

## Evidence
