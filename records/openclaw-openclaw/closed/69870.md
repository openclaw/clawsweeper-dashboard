## Summary

Close as `clawhub`. The proposal is a broad external orchestration layer for agent flows, and current `main` explicitly steers that kind of capability into plugins/ClawHub rather than core. OpenClaw already ships plugin, Task Flow, Lobster, and webhook surfaces that let an external system like Flowise drive workflows without replacing core architecture.
