## Summary

PR #48092 proposes adding ModelScope as another bundled model-provider plugin. The current main branch does not include ModelScope support, but the existing provider plugin SDK, manifest auth metadata, and ClawHub/npm install path are sufficient for this API-key OpenAI-compatible provider without a core change or missing extension API.

## Best Possible Solution

Close this core PR and carry the useful ModelScope work into an external provider plugin published through ClawHub/npm. Reopen a core discussion only if building that plugin exposes a concrete missing provider SDK or manifest API.
