## Summary

Current `main` already honors isolated cron `agentTurn` model overrides, forwards the resolved provider/model into execution, and has dedicated regression coverage for Gemini-style overrides. I did not find a released tag in the local checkout that definitively ships this yet, so this looks fixed on `main` but not attributable to a released version from the available evidence.
