## Summary

Current `main` contains targeted fixes for long-session tool execution stalls in the Pi/compaction path. The compaction runner now preserves the filtered tool allowlist through compaction-session creation and re-applies active tools, matching a shipped changelog fix in `2026.4.21`, and the embedded runner also hardens cleanup around pending tool results plus surfaces incomplete-turn failures instead of silently wedging.
