## What This Changes

The PR adds a real-behavior-proof PR-template section, a required external-PR workflow check, Barnacle proof labels/actions, contributor/changelog docs, and regression tests.

## Best Possible Solution

Tighten the classifier so test/lint/typecheck/build/snapshot/mock/CI evidence remains `mock_only` unless separate live OpenClaw evidence is present, while preserving the trusted-base workflow design and override label.

## Reproduction Assessment

Yes for the PR finding. Source-evaluating the PR-head policy with a `Terminal transcript:` that only contains `pnpm test` output returns `passed`, which reproduces the policy bypass without needing to run the workflow.

## Solution Assessment

No. The workflow shape is reasonable, but the current classifier should reject test/CI-only transcripts before accepting descriptor-plus-residual text as real behavior proof.
