## Summary

Keep open. Current `main` still does not thread global `browser.ssrfPolicy` into the sandbox browser bridge, so this is not an `implemented_on_main` or `cannot_reproduce` cleanup case. The PR targets a real gap, but it still appears to need maintainer review rather than closure.

## Evidence

  - file: [src/agents/sandbox/context.ts:174](https://github.com/openclaw/openclaw/blob/7e49cc87f9e9cdad47d0572331d7a7a47281a83e/src/agents/sandbox/context.ts#L174)

  - file: [src/agents/sandbox/browser.ts:78](https://github.com/openclaw/openclaw/blob/7e49cc87f9e9cdad47d0572331d7a7a47281a83e/src/agents/sandbox/browser.ts#L78)
