## Summary

Current `main` no longer documents or implements the legacy Chrome-extension relay flow. The browser docs now describe the managed `openclaw` profile plus Chrome MCP `user` attach flow, doctor migrates legacy extension config to `existing-session`, and the remaining relay helper is an explicit no-op stub. The same docs are present in the tagged `v2026.4.22` release tree, so the issue is already resolved on shipped code.
