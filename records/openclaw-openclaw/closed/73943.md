## Summary

Keep open. The PR targets a real ACPX/Codex auth regression, and current main already handles the separate Codex timeout rejection, but this patch still has a security-sensitive backup exposure when the auth bridge falls back to copying credentials plus a remaining test isolation bug.

## What This Changes

The PR bridges host Codex `auth.json` into ACPX's isolated Codex home, documents the behavior, updates the changelog, and adds ACPX/Codex regression coverage.

## Best Possible Solution

Keep the auth bridge direction, but make it backup-safe before merge and isolate tests from both `CODEX_HOME` and the default home Codex auth path.

## Reproduction Assessment

Yes. Source inspection gives a high-confidence path: the PR copy fallback writes `auth.json` under `stateDir/acpx/codex-home`, and current backup creation archives the state directory with only plugin `node_modules` excluded.

## Solution Assessment

No. Bridging only `auth.json` while keeping Codex config hooks isolated is the narrow direction, but copying credentials into the default backup scope is not the safest implementation.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Keep copied Codex auth out of backups:** `extensions/acpx/src/codex-auth-bridge.ts:280`
  - body: When symlink creation fails, this fallback writes the host Codex `auth.json` into `stateDir/acpx/codex-home`. `openclaw backup create` archives the state directory and the current filter only skips plugin `node_modules`, so routine backups can contain Codex credentials.
  - confidence: 0.91
- **[P2] Isolate the no-auth test from the default Codex home:** `extensions/acpx/src/codex-auth-bridge.test.ts:92`
  - body: Deleting `CODEX_HOME` is not enough for this new no-auth assertion because the bridge falls back to the process home `.codex/auth.json`. On an authenticated developer or CI home, `generated.authPath` will be created and this test will fail; point `CODEX_HOME` at an auth-free temp directory or otherwise isolate the home lookup.
  - confidence: 0.82
