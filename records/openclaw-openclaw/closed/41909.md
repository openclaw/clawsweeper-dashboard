## Summary

Current `main` already contains a targeted fix for Anthropic signed-thinking replay with subagent/tool continuations. The new replay path preserves provider-owned thinking turns for Claude 4.6-class models and drops unsafe `sessions_spawn` assistant turns instead of mutating them, which addresses the reported HTTP 400 condition. The fix is on `main` in `29d0e930bb3de4886b2e799781eb5620d973d1be` and is not in the latest tagged release `v2026.4.22`.
