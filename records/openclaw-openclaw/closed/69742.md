## Summary

Close as `clawhub`: current main already exposes live per-session usage over Gateway RPC, and OpenClaw’s documented plugin seams already support adding authenticated Gateway HTTP routes. The remaining ask is a transport-specific monitoring adapter (`/api/sessions`), which fits the repo’s plugin-first scope better than another built-in core endpoint.
