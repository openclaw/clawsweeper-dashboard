## Summary

Current `main` already routes embedded/non-default agent runs through runtime plugin loading before hook lookup, re-initializes the shared global hook runner on compatible plugin-registry activation, and stores hook-runner state on a `globalThis` singleton with regression coverage for shared-state reuse across module reloads. The changelog also records the hook-runner singleton hardening and embedded hook-context parity fixes.
