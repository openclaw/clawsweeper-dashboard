## Summary

Close as ClawHub/plugin scope. The proposal is an optional, opinionated system-prompt policy rather than a missing core capability. Current main already exposes plugin prompt-mutation hooks that can prepend or append stable system context, and the bundled Skill Workshop plugin demonstrates that prompt-guidance injection path. OpenClaw's vision explicitly keeps optional capabilities in plugins/ClawHub unless a missing extension API is identified.

## Best Possible Solution

Keep the default OpenClaw system prompt lean and let these exact Agent Behavior Principles ship as a ClawHub plugin or skill pack, using `before_prompt_build` with `prependSystemContext`/`appendSystemContext`. If a future implementation discovers a missing prompt-injection or policy-composition API, open a narrower core plugin-API issue for that gap.
