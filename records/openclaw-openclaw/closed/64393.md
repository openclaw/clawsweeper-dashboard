## Summary

Current `main` already supersedes this PR with a stronger CLI session-reuse fix: it keeps the raw `mcpConfigHash`, adds a port-stable `mcpResumeHash` for the OpenClaw loopback MCP bridge, uses that hash for resume decisions, and has regression coverage for both loopback-port churn and real MCP-surface changes.
