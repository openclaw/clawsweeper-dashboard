## Summary

Current `main` already contains Windows-safe bundled channel/plugin loading that covers the reported Feishu startup failure mode: bundled Feishu now loads through the shared channel-entry contract, the loader rewrites Windows absolute import paths to `file://` URLs, and there are explicit Windows regression tests for both paths.

## Evidence

- **Feishu uses bundled channel entry loading:** `extensions/feishu/channel-entry.ts:3` defines Feishu with `defineBundledChannelEntry(...)`, so startup goes through the shared bundled-entry loader rather than a direct raw-path ESM import.
  - file: [extensions/feishu/channel-entry.ts:3](https://github.com/openclaw/openclaw/blob/320d52a23e41a326287826219a20f1f29f334554/extensions/feishu/channel-entry.ts#L3)
