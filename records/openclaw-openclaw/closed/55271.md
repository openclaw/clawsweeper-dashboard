## Summary

Current `main` already routes isolated cron `--announce --channel telegram --to <chat-id>` through the explicit delivery target path instead of falling back to the last/heartbeat route, and `main` now has regression coverage for both explicit-target preservation and plain Telegram direct delivery.

## Evidence
