## Summary

Current `main` already supports local/self-hosted models for agent workflows. OpenClaw ships bundled Ollama support with `Local only` onboarding, automatic local model discovery, agent default model selection, and native tool-calling/streaming for agent turns; it also documents generic self-hosted local backends such as SGLang and inferrs. The same support is present in the latest release `v2026.4.22`.
