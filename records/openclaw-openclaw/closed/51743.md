## Summary

Issue #51743 is already implemented on current main: Google Gemini SSE responses map `usageMetadata` into normalized `usage`, the Google provider routes `google-generative-ai` models through that transport, and the regression test asserts nonzero token counts from Gemini-style `usageMetadata`. The same implementation is present in release tag `v2026.4.22`.

## Evidence
