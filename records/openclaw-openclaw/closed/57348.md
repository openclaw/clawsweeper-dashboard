## Summary

Current `main` and the shipped `v2026.4.23` release already contain ACP/Gateway changes that address the reported post-`sendPrompt` hang: pending ACP prompt RPCs no longer trigger the Gateway tick-timeout disconnect, and ACP prompt state now reconciles or rejects after disconnect instead of hanging indefinitely.

## Evidence
