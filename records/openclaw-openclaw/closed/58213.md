## Summary

Current `main` already handles the reported loop: model fallback now treats `LiveSessionModelSwitchError` as a failed fallback candidate and continues instead of rethrowing into the outer retry path, with regression tests covering the scenario.

## Evidence

- **Live session switch still originates in embedded runner:** The embedded runner throws `LiveSessionModelSwitchError` when the persisted session model differs from the active attempt and the run can be restarted for a live switch.
  - file: [src/agents/pi-embedded-runner/run.ts:985](https://github.com/openclaw/openclaw/blob/90877e0d4274c53fd91bf5787265c4d6889dc9a7/src/agents/pi-embedded-runner/run.ts#L985)
  - command: `nl -ba src/agents/pi-embedded-runner/run.ts | sed -n '940,1045p'`
  - sha: [90877e0d4274](https://github.com/openclaw/openclaw/commit/90877e0d4274c53fd91bf5787265c4d6889dc9a7)
- **Fallback loop now absorbs that switch error:** `runWithModelFallback` special-cases `LiveSessionModelSwitchError`, wraps it as a `FailoverError`, records the failed candidate, and `continue`s to the next candidate instead of rethrowing and restarting the outer loop.
  - file: [src/agents/model-fallback.ts:921](https://github.com/openclaw/openclaw/blob/90877e0d4274c53fd91bf5787265c4d6889dc9a7/src/agents/model-fallback.ts#L921)
  - command: `nl -ba src/agents/model-fallback.ts | sed -n '900,955p'`
  - sha: [90877e0d4274](https://github.com/openclaw/openclaw/commit/90877e0d4274c53fd91bf5787265c4d6889dc9a7)
- **Regression tests cover the reported failure mode:** Two tests explicitly cover the previous indefinite-restart behavior: one verifies the last candidate no longer rethrows `LiveSessionModelSwitchError`, and another verifies fallback advances to the next candidate after that error.
  - file: [src/agents/model-fallback.test.ts:644](https://github.com/openclaw/openclaw/blob/90877e0d4274c53fd91bf5787265c4d6889dc9a7/src/agents/model-fallback.test.ts#L644)
  - command: `nl -ba src/agents/model-fallback.test.ts | sed -n '640,686p'`
  - sha: [90877e0d4274](https://github.com/openclaw/openclaw/commit/90877e0d4274c53fd91bf5787265c4d6889dc9a7)
