## Summary

Keep open. Current `main` and the latest tagged release still serialize MiniMax TTS `voice_setting.speed`, `vol`, and `pitch` as raw numbers with float defaults, with no integer coercion on the request path. The bundled tests and docs also still expect float values, so the report matches the present implementation rather than an already-fixed bug.
