## Summary

Close #50488 as ClawHub/plugin-scoped. The requested triage/classifier gate is useful, but current main already exposes pre-model, pre-reply dispatch hooks and plugin runtime/model helpers that are sufficient for an optional classifier plugin to decide whether to silence a group message without invoking the primary agent. VISION.md explicitly keeps optional capability out of core when plugin surfaces can express it.
