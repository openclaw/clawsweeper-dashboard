## Summary

Current `main` already routes replies from live origin metadata instead of mutable session `lastChannel`, and the shared-session outbound path now prefers `turnSourceChannel` for `replyChannel: last`. The repository also contains direct regression tests for Feishu-origin routing and cross-channel shared-session reply safety, so issue #45514 is implemented on `main` but not yet in the latest tagged release.

## Evidence
