## Summary

Current `main` already makes the slug-generator embedded run honor configured agent timeout settings, and the remaining `15_000` call sites are wait/cleanup bounds rather than the actual embedded-run budget. The shipped changelog entry for 2026.4.14 matches this issue directly.
