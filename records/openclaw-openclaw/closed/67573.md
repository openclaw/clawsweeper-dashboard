## Summary

Current `main` still preserves user-set session model overrides across `/new` and `/reset`, so issue #67573 remains valid and should stay open. The behavior is explicit in both runtime code and tests, and the issue also has an open linked PR (#67591), which makes conservative cleanup inappropriate.
