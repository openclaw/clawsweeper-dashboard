## Summary

Current OpenClaw no longer needs the runtime PI model catalog on ordinary MiniMax allowlist-backed turns, and that behavior was already shipped in v2026.4.23. The bundled MiniMax onboarding path now seeds local model metadata, so the reported 82-second per-message delay path does not match current shipped behavior for the official MiniMax setup.

## Evidence
