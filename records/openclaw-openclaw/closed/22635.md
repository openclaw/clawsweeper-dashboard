## Summary

Current `main` already provides a supported way to suppress CLI taglines via `cli.banner.taglineMode: "off"`, documents that behavior, and covers it with regression tests. It also skips the banner entirely for non-TTY output, which addresses the scripting/piped-output concern in the report.
