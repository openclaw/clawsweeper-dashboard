## Summary

Current main already stages local `MEDIA:` paths through the outbound attachment pipeline, includes `/tmp/openclaw` in the default allowed media roots, and documents/tests that absolute local media can be sent when the agent's file-read policy allows it. The reported absolute-path delivery failure is obsolete on current main and in release `v2026.4.23`.

## Evidence
