## Summary

Current `main` already supports direct webchat rendering of assistant image/media output. The Control UI normalizes assistant `MEDIA:` output into attachment blocks, renders image attachments inline in chat, supports local assistant-media preview URLs, and has focused tests covering both assistant image rendering and generated-image fetch/display behavior. The changelog records this as shipped in the 2026.4.11 Control UI/webchat release.

## Evidence
