## Summary

Current main already fixes the reported Telegram approval bootstrap bug. The shared exec-approval gating now requires explicit enablement (`true` or `"auto"`), Telegram has regression coverage for `enabled: false`, and the Telegram monitor only registers the native approval runtime when that gate passes. The fix shipped in v2026.4.23 via commit 527d7211e03273f48541321feec137332993d67b.

## Evidence
