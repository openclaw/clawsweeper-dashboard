## Summary

Current main no longer matches the 2026.4.9 hang described here: the plugin-loader path called out in the discussion now carries `activationSourceConfig` correctly, CLI startup now loads the real primary command for `--help`, and the shipped changelog includes standalone memory CLI and one-shot CLI hang fixes by v2026.4.23.
