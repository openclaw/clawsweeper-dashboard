## Summary

Current `main` no longer silently truncates the skills catalog. The skills prompt builder now prepends an explicit truncation notice, that prompt is injected verbatim into the system prompt, and regression tests assert the warning text. The changelog shows the shipped prompt-budget/compact-fallback change in `v2026.3.22`.

## Evidence
