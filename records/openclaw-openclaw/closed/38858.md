## Summary

Current `main` already implements slash-command autocomplete in the Control UI chat composer: typing `/` opens a command menu, partial text filters commands, selecting/tab-completing fills the draft, fixed-arg commands offer a second-stage picker, and the menu is populated from live `commands.list` gateway data.
