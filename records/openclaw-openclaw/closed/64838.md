## Summary

Current `main` no longer has a Ctrl/Cmd-F override in the agent file editor surface. The files panel now renders the editor as a plain `<textarea>`, while the only remaining Ctrl/Cmd-F handler in the Control UI is scoped to chat search. The files-panel implementation landed on 2026-04-25, after the latest tagged release `v2026.4.23`, so this appears fixed on `main` but not yet in a released tag.
