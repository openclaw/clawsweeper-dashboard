## Summary

Current `main` already implements multi-account Feishu support end to end. The Feishu channel now accepts `channels.feishu.accounts` plus `defaultAccount`, resolves per-account credentials/routing, starts all enabled accounts at runtime, and documents the feature explicitly. This appears to have landed after the latest tagged release, so it is implemented on `main` but not yet shipped in `v2026.4.22`.
