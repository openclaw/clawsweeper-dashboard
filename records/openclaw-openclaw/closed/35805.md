## Summary

The issue is asking OpenClaw core to enforce a workspace-specific file-age policy for custom governance files, but current main only treats a fixed bootstrap file set as agent context. The requested mtime-threshold alerting is useful, but it fits the existing plugin/skill/hook automation model and ClawHub scope better than OpenClaw core.
