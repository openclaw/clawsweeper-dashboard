## Summary

Current main contains a shipped Telegram startup packaging fix that matches this report: Telegram setup/secret surfaces now load from packaged top-level sidecars, and the 2026.4.8 changelog explicitly says this stopped installed builds from importing missing `dist/extensions/telegram/src/*` files during gateway startup.

## Evidence
