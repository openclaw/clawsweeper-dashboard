## Summary

Signal group overrides are already implemented on current `main` and are present in the shipped `v2026.4.22` release. The config schema exposes `channels.signal.groups`, the Signal runtime resolves `requireMention` from group config for inbound group messages, the docs document the setting, and tests cover both schema acceptance and mention-gating behavior.

If you are still seeing Signal reply to every group message, please open a fresh bug with your exact `channels.signal.groups` config and logs from current OpenClaw.
