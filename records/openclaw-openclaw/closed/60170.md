## Summary

Current `main` already covers the restart-storm failure mode this PR targeted. The generated systemd unit now includes `StartLimitBurst=5`, `StartLimitIntervalSec=60`, and `RestartPreventExitStatus=78`, and gateway startup restores invalid config from last-known-good before validation when possible. The shipped release notes for `v2026.4.23` explicitly call out preventing gateway crash loops, so this PR is superseded on `main`.
