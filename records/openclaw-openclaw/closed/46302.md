## Summary

Current `main` already gives plugins a public path to register internal hooks, including `agent:bootstrap`, so the requested bootstrap hot-reload seam is implemented well enough via Option A. The remaining `clearBootstrapSnapshot` export is not necessary for this request because bootstrap overrides still run after cached snapshot lookup.
