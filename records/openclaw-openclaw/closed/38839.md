## Summary

Current main already implements the requested cron reliability pieces: restart catch-up for missed recurring jobs, transient retry policy for one-shot jobs, failure notifications/destinations, manual rerun surfaces, and managed gateway autostart. The same implementation is present in release v2026.4.22.
