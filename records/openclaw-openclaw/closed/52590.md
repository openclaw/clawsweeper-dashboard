## Summary

Keep open. Current `main` shows this PR would remove Feishu manifest-declared SKILL.md loading while leaving the actual Feishu tool-registration entrypoint unchanged, so it is not a conservative maintainer close candidate.

## Evidence

- **Feishu still registers tools from the runtime entrypoint:** The bundled Feishu package still points OpenClaw at `./index.ts`, and that entrypoint's `registerFull(...)` registers the doc/chat/wiki/drive/perm/bitable tool groups. Removing manifest `skills` would not change this registration path.
  - file: [extensions/feishu/package.json:22](https://github.com/openclaw/openclaw/blob/c070509b7f8fdbe479e5d829643782c272e30e83/extensions/feishu/package.json#L22)
  - command: `nl -ba extensions/feishu/package.json | sed -n '10,24p'; nl -ba extensions/feishu/index.ts | sed -n '56,81p'`
  - sha: [c070509b7f8f](https://github.com/openclaw/openclaw/commit/c070509b7f8fdbe479e5d829643782c272e30e83)
- **Manifest `skills` means skill directories, not tool registration:** The Feishu manifest still declares `"skills": ["./skills"]`, and the plugin manifest docs define `skills` as "Skill directories to load, relative to the plugin root."
  - file: [extensions/feishu/openclaw.plugin.json:12](https://github.com/openclaw/openclaw/blob/c070509b7f8fdbe479e5d829643782c272e30e83/extensions/feishu/openclaw.plugin.json#L12)
  - command: `nl -ba extensions/feishu/openclaw.plugin.json | sed -n '1,20p'; nl -ba docs/plugins/manifest.md | sed -n '156,164p'`
  - sha: [c070509b7f8f](https://github.com/openclaw/openclaw/commit/c070509b7f8fdbe479e5d829643782c272e30e83)
- **Current main loads manifest skills into the skill workspace:** `resolvePluginSkillDirs(...)` iterates `record.skills` from the manifest registry, and `workspace.ts` merges those directories into the loaded extra-skill roots. Deleting Feishu's manifest `skills` would therefore stop these Feishu skill folders from loading.
  - file: [src/agents/skills/plugin-skills.ts:71](https://github.com/openclaw/openclaw/blob/c070509b7f8fdbe479e5d829643782c272e30e83/src/agents/skills/plugin-skills.ts#L71)

- **Feishu skill folders contain substantive guidance:** The bundled Feishu skills are real SKILL.md guides with operational instructions, token extraction, and action examples; they are not empty placeholders.
  - file: [extensions/feishu/skills/feishu-doc/SKILL.md:1](https://github.com/openclaw/openclaw/blob/c070509b7f8fdbe479e5d829643782c272e30e83/extensions/feishu/skills/feishu-doc/SKILL.md#L1)
