## Summary

Current `main` still appears to have the reported bug path. `memory-core` still does best-effort narrative cleanup by calling `subagent.deleteSession()`, while the gateway plugin runtime still dispatches fallback `sessions.delete` calls without minting `operator.admin`, and the current test suite explicitly asserts that this fallback path fails with `missing scope: operator.admin`. The linked PR/commit from the issue timeline is not present in this checkout of `main`.
