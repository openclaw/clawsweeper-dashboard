## Summary

Current `main` already carries the packaged-install/browser-runtime repair path for bundled plugin deps, and the browser plugin’s `playwright-core` activation path is explicitly covered by release smoke, doctor repair, and troubleshooting docs. The reported 2026.3.23-2 Windows global-install failure mode is obsolete against current `main`.
