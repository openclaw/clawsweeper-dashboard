## Summary

#53327 asks for Feishu Calendar API CRUD/free-busy tools. Current main does not implement those tools, but the request is an optional service-specific workplace integration that can be built with the existing OpenClaw plugin tool API and distributed through ClawHub/npm. The existing bundled @openclaw/feishu plugin covers Feishu chat/doc/wiki/drive/permission/bitable surfaces, and there is no protected label, maintainer author association, or discussion showing a missing core plugin API seam.

## Best Possible Solution

Close this OpenClaw core issue and route Feishu Calendar API support to a ClawHub/npm plugin or externally maintained expansion of the Feishu plugin. The existing plugin API can register calendar CRUD/free-busy tools without new core seams; if implementation later exposes a concrete missing SDK capability, that narrower plugin API gap should be filed separately.
