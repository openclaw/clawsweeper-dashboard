## Summary

Current `main` does not replace the original `sessions_send` payload with `Agent-to-agent announce step.`. The code sends the caller payload first, then runs a separate internal A2A announce turn; the placeholder can appear in transcript history as an additional inter-session message, but that is not the delivered payload being overwritten.
