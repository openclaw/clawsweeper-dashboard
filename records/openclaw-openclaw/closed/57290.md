## Summary

Current `main` already implements a targeted fix for fake-IP proxy environments: `web_fetch` now has an opt-in `tools.web.fetch.ssrfPolicy.allowRfc2544BenchmarkRange` setting for `198.18.0.0/15`, the runtime wires it into the SSRF guard, tests cover both default-blocked and opt-in-allowed behavior, and the changelog says it shipped in `v2026.4.10`.

## Evidence
