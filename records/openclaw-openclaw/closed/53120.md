## Summary

Current `main` already addresses the reported approval-ID regression. Exec approvals are registered before the gateway returns the accepted response, and approvals are no longer auto-expired when the originating surface can handle `/approve` directly. The same implementation is present in release `v2026.4.22`.

## Evidence
