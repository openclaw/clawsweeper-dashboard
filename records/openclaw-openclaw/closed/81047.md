## What This Changes

The PR removes `systemPromptHash` from the Claude CLI live-session fingerprint, exports the helper for tests, and adds fingerprint unit coverage for volatile prompt changes plus model/workspace/static-prompt gates.

## Best Possible Solution

Carry a stable Claude live-session prompt identity that excludes volatile chat metadata but still invalidates the warm process when durable system instructions or stable prompt-building inputs change.

## Reproduction Assessment

## Solution Assessment

No. Removing the full prompt hash fixes volatile metadata resets, but the narrower maintainable solution is to hash stable prompt material or carry a stable prompt fingerprint instead of ignoring durable prompt changes.
