## Summary

Keep open. Current `main` still shows the same config-path/schema mismatch: the Comfy runtime reads `models.providers.comfy` and expects nested `image`/`video`/`music` config there, but core model-provider validation is still a strict schema that does not admit those keys, while the Comfy manifest schema is only merged onto `plugins.entries.comfy.config`.
