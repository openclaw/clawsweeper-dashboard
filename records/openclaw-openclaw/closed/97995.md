Full review comments:

- **[P2] Handle multi-backtick inline code spans:** `extensions/memory-wiki/src/markdown.ts:393`
  - body: The new inline-code mask only matches single-backtick spans. Valid Markdown like `` `[[not-a-link]]` `` leaves `[[not-a-link]]` in searchable text, so `wiki lint` can still report code-only text as a broken wikilink; use a delimiter-length-aware span scanner or pattern and add coverage.
  - confidence: 0.91
- **[P2] Allow whitespace on tilde fence closers:** `extensions/memory-wiki/src/markdown.ts:392`
  - body: The tilde fence replacement requires the closing `~~~` to be followed immediately by a newline or EOF. Markdown permits trailing spaces on closing fences, so `~~~   ` leaves `[fake](bad.md)` and `[[fake]]` inside the block searchable; allow closing-line whitespace and cover this case.
  - confidence: 0.88
