  - body: The new inline-code mask only handles single-backtick spans safely.   - body: The tilde fence replacement requires the closing delimiter to be followed immediately by a newline or EOF. Markdown permits trailing spaces on closing fences, so a block closed as `~~~   ` leaves fake wikilinks and Markdown links inside the block searchable.
