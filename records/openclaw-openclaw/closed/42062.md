## Summary

Current main fixes the reported GLM-5/Ollama path by forwarding OpenClaw thinking-off state to native Ollama `/api/chat` as top-level `think: false`, with regression coverage and a changelog entry. Broader provider-agnostic untagged preamble filtering remains separate and is tracked by #62121.
