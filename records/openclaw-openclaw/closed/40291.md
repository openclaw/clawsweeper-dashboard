## Summary

Keep open. Current `main` does not implement this request: voice-call auto-responses still hardcode `agentId = "main"`. The proposed PR direction is also not strong enough for conservative cleanup because OpenClaw routing semantics are richer than a first `match.channel === "voice-call"` lookup, so closing as implemented/cannot_reproduce would be incorrect.

## Evidence
