## Summary

Current `main` already contains a direct fix for the reported "stuck on compacting context" behavior: prompt/context overflow is now preflighted before dispatch, overflow recovery is bounded instead of looping indefinitely, and the Control UI clears the compaction indicator on failed retries or terminal lifecycle events.

## Evidence
