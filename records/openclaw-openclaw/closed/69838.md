## Summary

Current `main` already implements a pre-prompt compaction safeguard that checks the larger full-transcript estimate, including context-engine-windowed sessions, and routes overflowing tool-use loops into compaction/truncation before provider failure. The fix is present on `main` but still unreleased relative to `v2026.4.23`.

## Evidence

- **Pre-prompt recovery now uses full transcript pressure:** `runEmbeddedAttempt()` computes `preemptiveCompaction` before prompt submission, passes both assembled `messages` and `unwindowedMessages`, and converts an overflow precheck into early tool-result truncation or compaction by setting `skipPromptSubmission` and `promptError = PREEMPTIVE_OVERFLOW_ERROR_TEXT`.
  - file: [src/agents/pi-embedded-runner/run/attempt.ts:2476](https://github.com/openclaw/openclaw/blob/5fe333ada85afd0def9e85985ce57e36aa00108a/src/agents/pi-embedded-runner/run/attempt.ts#L2476)
  - sha: [5fe333ada85a](https://github.com/openclaw/openclaw/commit/5fe333ada85afd0def9e85985ce57e36aa00108a)
- **Estimator prefers larger unwindowed history:** `shouldPreemptivelyCompactBeforePrompt()` now accepts `unwindowedMessages`, recomputes the estimate on the unwindowed transcript, and uses the larger estimate plus the matching message set for tool-result reduction pressure. This directly covers cases where context-engine assembly/windowing made the visible prompt look smaller than the real session pressure.
  - file: [src/agents/pi-embedded-runner/run/preemptive-compaction.ts:41](https://github.com/openclaw/openclaw/blob/5fe333ada85afd0def9e85985ce57e36aa00108a/src/agents/pi-embedded-runner/run/preemptive-compaction.ts#L41)

- **Regression test for windowed-history compaction:** A dedicated test now asserts that when assembled history is small but the unwindowed transcript is larger, the precheck returns `shouldCompact = true` and `route = "compact_only"`.
  - file: [src/agents/pi-embedded-runner/run/preemptive-compaction.test.ts:96](https://github.com/openclaw/openclaw/blob/5fe333ada85afd0def9e85985ce57e36aa00108a/src/agents/pi-embedded-runner/run/preemptive-compaction.test.ts#L96)

Fixes #71325.`
