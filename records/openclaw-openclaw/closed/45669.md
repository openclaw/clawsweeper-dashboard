## Summary

Current `main` already implements deterministic skill-command tool dispatch, and Telegram native commands feed into that same path. The feature is present on `main` via commit `37c37eecfbe4bef413c33b8606db62cf51c2d1d4`, but it does not appear in the provided latest release tag `v2026.4.22`, so no shipped release could be confirmed.

## Evidence

- **Skill frontmatter now defines deterministic tool dispatch:** `buildWorkspaceSkillCommandSpecs` reads `command-dispatch: tool`, requires `command-tool`, and records `{ kind: "tool", toolName, argMode: "raw" }` on the slash-command spec.
  - file: [src/agents/skills/command-specs.ts:119](https://github.com/openclaw/openclaw/blob/37c37eecfbe4bef413c33b8606db62cf51c2d1d4/src/agents/skills/command-specs.ts#L119)
  - command: `nl -ba src/agents/skills/command-specs.ts | sed -n '100,170p'`
  - sha: [37c37eecfbe4](https://github.com/openclaw/openclaw/commit/37c37eecfbe4bef413c33b8606db62cf51c2d1d4)
- **Slash-command handler bypasses the model and executes the tool directly:** `handleInlineActions` resolves skill invocations, checks `dispatch?.kind === "tool"`, builds the OpenClaw tool set, executes the named tool directly with `{ command, commandName, skillName }`, and returns the tool result/error as the reply instead of rewriting the request for model handling.
  - file: [src/auto-reply/reply/get-reply-inline-actions.ts:245](https://github.com/openclaw/openclaw/blob/37c37eecfbe4bef413c33b8606db62cf51c2d1d4/src/auto-reply/reply/get-reply-inline-actions.ts#L245)

- **Telegram native commands route into the same slash-command pipeline:** Telegram native command handling reconstructs the slash prompt (`/skill ...` or `/<skillname> ...`), sets `CommandSource: "native"`, and dispatches through `dispatchReplyWithBufferedBlockDispatcher`, so Telegram native skill commands hit the same deterministic tool-dispatch logic above.
  - file: [extensions/telegram/src/bot-native-commands.ts:781](https://github.com/openclaw/openclaw/blob/37c37eecfbe4bef413c33b8606db62cf51c2d1d4/extensions/telegram/src/bot-native-commands.ts#L781)
