## Summary

Current main and the v2026.4.2 release cover the requested no-prompt exec path: gateway/node host exec defaults were changed to YOLO/no-prompt behavior, and the docs now explain that disabling prompts requires aligning both tools.exec.* and the host-local exec-approvals.json policy, with an exec-policy preset for the local case.
