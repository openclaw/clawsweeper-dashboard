## Summary

Issue #10161 asks for a new external skill/workflow integration (`service-ability-creator`) in core. Current `main` explicitly directs new skills and community capabilities to ClawHub instead of OpenClaw core, and the repo already documents and implements ClawHub-backed skill/plugin installation plus workspace skill generation.
