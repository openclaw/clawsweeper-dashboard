## Summary

Close as `clawhub`: this PR is a docs-only discoverability listing for the external Kudosity SMS plugin, while current main and the shipped community-plugin docs explicitly make ClawHub the canonical discovery surface and ask contributors not to open docs-only listing PRs. The plugin is already posted at https://clawhub.ai/plugins/kudosity-openclaw-sms, so the user-visible path is covered without changing OpenClaw source docs.

## Best Possible Solution

Close this docs-only listing PR and keep Kudosity SMS discoverable through its existing ClawHub/npm package. If maintainers want curated plugin promotion in OpenClaw docs, that should be a separate policy change to `docs/plugins/community.md`, not a one-off listing exception.
