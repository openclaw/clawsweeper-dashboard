## Summary

Keep open. Current `main` still shows the same schema/runtime mismatch: the comfy runtime reads `models.providers.comfy`, while plugin config schemas are merged under `plugins.entries.comfy.config`, and the core provider schema remains strict enough to reject capability keys like `image`, `video`, and `music` on `models.providers.comfy`.

## Evidence
