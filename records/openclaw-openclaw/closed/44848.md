## Summary

Current `main` no longer matches the reported repeated plugin-registration behavior. Root help now has a fast path that does not load plugins at all, CLI command bootstrap routes plugin loading through one shared loader, and the runtime registry loader tracks scope/load state so repeated internal calls do not keep reloading the same channel/plugin registry.
