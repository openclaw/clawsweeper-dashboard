## Summary

Current `main` already ships the built-in `openai` core plugin, and that plugin registers `openai-codex` auth plus the documented `openclaw models auth login --provider openai-codex` flow. The same implementation is present in release `v2026.4.22`, so the 2026.3.2 missing-provider report is obsolete on current OpenClaw.

## Evidence
