## What This Changes

The PR adds group, sandbox, and subagent policy inputs to dispatch-time `message` tool availability and adds group/sandbox fallback regression tests.

## Best Possible Solution

Compute dispatch-time `message` availability from the same runtime policy session key and policy set used by embedded execution, while preserving current main's group, subagent, and inherited-policy handling.

Yes, source-reproducible with high confidence. ## Solution Assessment

No as written. Extending dispatch availability toward the real tool-policy pipeline is the right direction, but the sandbox check must use the runtime policy key and preserve the newer inherited-policy logic already on main.
