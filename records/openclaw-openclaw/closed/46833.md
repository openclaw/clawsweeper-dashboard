## Summary

Close this issue as a duplicate of open #50404. The requested Feishu/channel conversation list in the Control UI left sidebar is a narrower case of the broader Control UI session-sidebar/history tracker, which remains open and has the useful follow-up/workaround discussion. Current main still has only partial switching through the chat header selector and the separate Sessions tab, not the requested sidebar implementation.

## Best Possible Solution

Close this duplicate and keep the remaining Control UI conversation/sidebar work centralized on #50404. The eventual implementation should add a compact left-sidebar session list backed by sessions.list/sessions.preview, use the existing switchChatSession path for selection, and preserve the separate Sessions tab for full management.
