## Summary

Current main already contains Discord gateway reconnect and health-monitor changes that address the reported 10-minute Discord restart loop: stale heartbeat timers are cleared before reconnect, and stale-socket restarts now require provider-proven transport activity instead of ordinary app-idle time.

## Evidence
