## Summary

Current `main` shows that WeChat image send support is owned by the external `@tencent-weixin/openclaw-weixin` channel plugin, not by OpenClaw core, so this is not actionable in `openclaw/openclaw`.

## Evidence

- **WeChat runtime is explicitly external:** `docs/channels/wechat.md` says OpenClaw connects to WeChat through Tencent's external `@tencent-weixin/openclaw-weixin` plugin, and that WeChat login, Tencent iLink API calls, media upload/download, context tokens, and account monitoring are owned by the external plugin.
  - file: [docs/channels/wechat.md:10](https://github.com/openclaw/openclaw/blob/8866544ffefb42ca234c66c337134fb60743d8d0/docs/channels/wechat.md#L10)
  - sha: [8866544ffefb](https://github.com/openclaw/openclaw/commit/8866544ffefb42ca234c66c337134fb60743d8d0)
- **Project scope prefers optional capability as plugins:** `VISION.md` says optional capability should usually ship as plugins, with the preferred path being npm package distribution and maintenance in the plugin's own repository rather than core.
  - file: [VISION.md:54](https://github.com/openclaw/openclaw/blob/8866544ffefb42ca234c66c337134fb60743d8d0/VISION.md#L54)
  - sha: [8866544ffefb](https://github.com/openclaw/openclaw/commit/8866544ffefb42ca234c66c337134fb60743d8d0)
- **No WeChat runtime exists in this checkout:** A repo-wide file search for `wechat|weixin|wx` found only docs and generic registry/test references, not a bundled WeChat implementation under `src/` or `extensions/`.
  - command: `rg --files | rg 'wechat|weixin|wx'`
  - sha: [8866544ffefb](https://github.com/openclaw/openclaw/commit/8866544ffefb42ca234c66c337134fb60743d8d0)
- **Core only knows the plugin id generically:** The only code reference in core is a registry helper test that resolves the external plugin id `openclaw-weixin`; it does not implement outbound media behavior.
  - file: [src/channels/registry.helpers.test.ts:43](https://github.com/openclaw/openclaw/blob/8866544ffefb42ca234c66c337134fb60743d8d0/src/channels/registry.helpers.test.ts#L43)
  - sha: [8866544ffefb](https://github.com/openclaw/openclaw/commit/8866544ffefb42ca234c66c337134fb60743d8d0)
