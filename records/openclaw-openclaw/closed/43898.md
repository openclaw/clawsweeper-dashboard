## Summary

Current `main` no longer feeds `before_prompt_build` recall context into the `agent_end` user-message stream that `memory-lancedb` auto-capture reads, so the PR’s reported failure mode is obsolete. The proposed regex fix also has an unaddressed correctness risk because it strips any `<relevant-memories>...</relevant-memories>` block anywhere in user-authored text.

## Evidence

- **Memory recall is now a `before_prompt_build` prependContext hook:** `memory-lancedb` registers recall on `before_prompt_build` and returns `prependContext`; its test explicitly asserts it no longer uses the legacy `before_agent_start` path.
