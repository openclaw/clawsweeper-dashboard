## Summary

Current main already preserves cron announce summary/output text and WhatsApp paragraph delivery boundaries. The issue’s related PR #42667 pointed at the right newline/chunking area, but the shipped code now has equivalent paragraph-boundary behavior plus regression coverage.

## Best Possible Solution
