## Summary

Current `main` does not implement opt-in schema enforcement for workspace markdown files. Workspace bootstrap reads still load raw file contents, write paths still accept arbitrary content without schema checks, and doctor/docs only cover bootstrap size/truncation plus the special-case `HEARTBEAT.md` `tasks:` block.
