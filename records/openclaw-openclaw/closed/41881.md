## Summary

Current main and the latest published Docker image already provide the requested official multi-architecture Docker tags for linux/amd64 and linux/arm64, with unified manifest tags for normal and slim images. The remaining arm/v7 mention does not block closure because the issue proposed that target as optional, current Raspberry Pi docs direct users to 64-bit ARM64 rather than armv7l, and the pinned Node 24 base image used by the Dockerfile does not publish linux/arm/v7.
