## Summary

Current `main` already addresses the reported silent bootstrap truncation: bootstrap size limits are configurable, truncation produces an agent-visible warning block by default, and docs describe `/context` visibility for raw vs injected sizes. This behavior is also present in the latest release `v2026.4.22`.
