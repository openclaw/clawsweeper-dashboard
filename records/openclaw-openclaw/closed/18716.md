## Summary

Current `main` already covers both behaviors from PR #18716 with a different and more complete MSTeams implementation: user-targeted sends now resolve only DM-safe conversation references, and outbound media for small personal-chat images is loaded through the shared media-access path and converted to inline `data:` URLs before send.
