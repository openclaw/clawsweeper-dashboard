## Summary

Current `main` already adds inbound-media staging before reply execution and keeps tool failures as structured tool results, which addresses the Feishu image-path regression described in #44778. The relevant implementation is present on `main` but is not in the latest tagged release `v2026.4.22`.

## Evidence
