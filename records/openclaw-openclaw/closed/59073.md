## Summary

Current `main` no longer manages or queries a `memory-alt-main` collection. The default QMD workspace setup now creates only `memory-root-<agent>` for `MEMORY.md` and `memory-dir-<agent>` for `memory/**/*.md`, the repair path rebinds/migrates stale managed collections, and regression tests explicitly assert that phantom `memory-alt-*` collections are not queried. The same implementation is present in release `v2026.4.23`.
