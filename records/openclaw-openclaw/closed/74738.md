## What This Changes

The branch wraps OpenAI Completions builder tool emission and tool-history fallback in `supportsModelTools(model)`, adds two regression tests, and adds a changelog entry.

## Best Possible Solution

## Solution Assessment

Yes. Reusing the existing `supportsModelTools(model)` helper inside `buildOpenAICompletionsParams()` is the narrow maintainable fix because it covers direct builder callers without changing default tool-capable behavior.
