## Summary

The reported Windows `openclaw config` crash matches a bundled-plugin loader regression that is already fixed on `main` and shipped in `v2026.4.23` via the Windows `file://` import normalization and non-native Jiti loader changes for bundled entries/configure flows.

## Evidence
