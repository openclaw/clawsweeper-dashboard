## Summary

Current `main` already routes cross-agent subagent workspace context to the target agent workspace, persists that workspace on the child session, and has regression coverage for the reported case.

## Evidence

- **Cross-agent spawn picks target workspace:** `spawnSubagentDirect()` resolves `spawnedMetadata.workspaceDir` through `resolveSpawnedWorkspaceInheritance(...)` and explicitly ignores the requester's inherited workspace for cross-agent spawns before patching `spawnedWorkspaceDir` onto the child session.
