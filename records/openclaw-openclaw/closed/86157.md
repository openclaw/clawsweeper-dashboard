- **[P1] Resolve access groups before accepting inbound webhooks:** `extensions/channel-broker/src/http-routes.ts:73-74`
  - body: When `allowFrom` contains an `accessGroup:*` entry, this gate returns `true` without checking whether the broker sender is actually in that group.
