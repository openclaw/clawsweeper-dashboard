## Summary

Current main now covers the reported direct-session context-window drift with a pre-prompt, reserve-based context-pressure check that fires before model submission, routes oversized sessions through compaction or safe tool-result truncation, and then retries. The path is generic to embedded runs, preserves session/channel metadata for direct chat sessions, and is covered by focused regression tests. This shipped in v2026.4.24.
