## Summary

Current `main` still contains the reported bad context-percentage math in the Control UI `/usage` path. `ui/src/ui/chat/slash-command-executor.ts:385-399` computes the displayed percentage as `inputTokens / contextTokens`, and `ui/src/ui/chat/slash-command-executor.node.test.ts:490-492` locks that behavior in with an expected `30%` result for `1200 input / 4000 context` even though the same row carries `totalTokens: 1500`. By contrast, the canonical status path on `main` uses `total / contextTokens` in `src/commands/status.summary.ts:211-214`, matching the issue’s expected behavior.
