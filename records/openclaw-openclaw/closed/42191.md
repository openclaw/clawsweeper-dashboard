## Summary

Current `main` already implements Feishu thread/current-conversation binding for ACP and subagent sessions, wires it into the Feishu monitor, documents `/acp ... --thread here` for Feishu DMs and topics, and the latest release tag `v2026.4.22` already contains that implementation.
