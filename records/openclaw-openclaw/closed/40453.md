## Summary

Keep this PR open. Current main still has the original short descriptions for the seven bundled skills, and those descriptions are active model-facing skill-selection metadata. The PR is useful partial work toward the broader open skill-description cleanup, but its current wording still needs copyedit because two changed descriptions point Telegram users at a nonexistent bundled skill.

## What This Changes

The PR rewrites the description frontmatter for seven bundled skills to add trigger phrases and negative-selection guidance.

## Best Possible Solution

Keep the PR open for focused copyedit or fold it into #43410. The final wording should keep the useful trigger/negative-trigger intent, stay concise and single-line, and point fallbacks only at existing skills, channels, tools, or documented extension surfaces.
