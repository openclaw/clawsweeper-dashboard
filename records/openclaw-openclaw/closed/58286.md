## Summary

Current `main` no longer does an unconstrained fresh semver resolve for bundled plugin runtime deps. It now stages from the installed root workspace graph when available and, for the fallback installer path, rewrites runtime dependency specs to exact installed versions before invoking `npm install`, which addresses the reported lockfile-bypass supply-chain risk.

## Evidence
