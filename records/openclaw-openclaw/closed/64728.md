## Summary

Close as `clawhub`: Anthropic Batch API support is provider-specific, optional runtime behavior that current OpenClaw architecture assigns to provider plugins and ClawHub distribution rather than OpenClaw core.

## Evidence

## Risks / Open Questions

- A ClawHub/provider-plugin implementation for long-turn batch jobs may still want additional SDK ergonomics around deferred completion and delivery.
- No existing ClawHub package was linked in the issue discussion, so this recommendation is scope-based rather than based on an already-published plugin.
