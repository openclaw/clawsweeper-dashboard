## Summary

Current `main` already contains the Feishu reasoning-preview guard and plain-text routing behavior this issue asks for. Streamed reasoning is only exposed when the session is explicitly `reasoning:stream`, and ordinary text sends still route to `sendMessageFeishu` unless card rendering is explicitly selected by presentation/render mode.
