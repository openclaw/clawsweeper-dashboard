  - body: The parser only runs when the raw string starts with `[` and ends with `]`, so a provider response with surrounding whitespace or a trailing newline still takes the old path and is returned as raw JSON.   - confidence: 0.88

Require a stronger chat content shape, such as at least one valid text block, before switching away from plain-string handling.
