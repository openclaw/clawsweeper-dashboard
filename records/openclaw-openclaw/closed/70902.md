## Summary

Current `main` does not implement Claude CLI external OAuth sync. The PR stays actionable: the sync source already exists, `anthropic:claude-cli` is a defined auth profile, but `src/agents/auth-profiles/external-cli-sync.ts` still registers only Codex and MiniMax, so stale Claude CLI credentials are not adopted automatically.
