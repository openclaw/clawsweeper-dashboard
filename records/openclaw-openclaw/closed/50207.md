## Summary

Current `main` already suppresses auto-delivered final assistant replies when the `message` tool sends to the same conversation, which addresses the duplicate/out-of-order/leaked narration problem described in #50207. The fix is present on `main` but does not appear to have shipped in `v2026.4.22` yet.

## Evidence
