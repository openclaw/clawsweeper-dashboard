## Summary

Current main no longer matches the reported Feishu/webchat coupling: stale-socket restarts now require explicit transport-liveness tracking, Feishu's current gateway startup path does not publish that tracking, channel stop/start does not close webchat sockets, and websocket mass-close is confined to gateway shutdown.
