## Summary

Current main already pre-checks prompt size against the selected model's effective context budget, triggers compaction/truncation before prompt submission when that smaller budget would overflow, and prevents model-fallback from advancing on context-overflow errors. The same implementation is present in release `v2026.4.23`.

## Evidence
