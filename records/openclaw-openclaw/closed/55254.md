## Summary

Current `main` already contains the shared-session routing fix for synthetic `heartbeat`/system-event pollution, with regression tests that specifically cover `agent:main:main` and preserving an existing Feishu route. The changelog shows this shipped in `v2026.4.14`.

## Evidence
