## Summary

Current main already treats Feishu credentials as channel config, not plugin-entry config, and the Feishu setup/docs now direct users to `channels.feishu`/`channels.feishu.accounts.*`. The remaining rejection of `plugins.entries.<id>.config.appId/appSecret` is intentional because the Feishu plugin manifest exposes an empty plugin-local config schema.

## Evidence

- **Feishu plugin manifest intentionally rejects plugin-local config keys:** The Feishu manifest still declares an empty strict `configSchema`, so `plugins.entries.feishu.config.appId/appSecret` is expected to fail validation rather than represent a missing supported surface.
