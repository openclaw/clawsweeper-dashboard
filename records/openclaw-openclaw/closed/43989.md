## Summary

Current `main` does not match the reported hanging behavior. The gateway no longer waits indefinitely when no approval route exists; it expires the approval immediately with `decision: null`, and host exec resolves that through `askFallback` instead of depending on a gateway-created Unix socket. The docs also clarify that the Unix socket is macOS node-service-to-app IPC, not a gateway-startup socket.
