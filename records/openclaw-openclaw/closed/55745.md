## Summary

Current `main` already exposes typed plugin lifecycle hooks through `OpenClawPluginApi.on(...)`, and the typed hook map includes both `before_compaction` and `after_compaction`. The registry wires that API into typed hook registration, the SDK docs publish it, and compaction hook execution is covered by tests. The same API is present in release `v2026.4.22` (`00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5`).
