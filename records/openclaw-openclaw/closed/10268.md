## Summary

The issue’s core claim no longer holds: the reporter later verified that GitHub Copilot’s Chat Completions endpoint silently ignores `web_search_options`, so there is no reproducible Copilot-hosted Bing search capability for OpenClaw to wire up on current main. Code inspection also shows native `web_search` injection exists only for OpenAI/OpenAI Codex paths, while the GitHub Copilot wrapper only adjusts headers/payload rewriting and does not expose a hosted-search hook.

## Evidence

- **Reporter invalidated the original premise:** The follow-up issue comment says live Copilot testing with debug instrumentation showed `web_search_options` is silently ignored, with no annotations or web-search indicators returned.
- **Copilot provider wrapper has no hosted-search injection path:** The GitHub Copilot stream wrapper only adds Copilot headers for Anthropic transport and rewrites payload connection-bound IDs for Responses transport; it does not add `web_search_options` or a native `web_search` tool.
  - file: [extensions/github-copilot/stream.ts:52](https://github.com/openclaw/openclaw/blob/754acc4478fe361db1e5303fd60f589d52beea7a/extensions/github-copilot/stream.ts#L52)
  - command: `nl -ba extensions/github-copilot/stream.ts | sed -n '1,220p'`
  - sha: [754acc4478fe](https://github.com/openclaw/openclaw/commit/754acc4478fe361db1e5303fd60f589d52beea7a)
- **OpenAI native web search is intentionally limited to OpenAI provider models:** `shouldEnableOpenAINativeWebSearch` only activates when the provider normalizes to `openai` and the model uses the OpenAI Responses API on an OpenAI base URL.
  - file: [extensions/openai/native-web-search.ts:19](https://github.com/openclaw/openclaw/blob/754acc4478fe361db1e5303fd60f589d52beea7a/extensions/openai/native-web-search.ts#L19)
  - command: `nl -ba extensions/openai/native-web-search.ts | sed -n '1,220p'`
  - sha: [754acc4478fe](https://github.com/openclaw/openclaw/commit/754acc4478fe361db1e5303fd60f589d52beea7a)
- **Codex native search is intentionally limited to Codex-capable models:** Codex native web search eligibility is restricted to `openai-codex` or `openai-codex-responses`, not `github-copilot`.
  - file: [src/agents/codex-native-web-search.ts:34](https://github.com/openclaw/openclaw/blob/754acc4478fe361db1e5303fd60f589d52beea7a/src/agents/codex-native-web-search.ts#L34)
  - command: `nl -ba src/agents/codex-native-web-search.ts | sed -n '1,240p'`
