## Summary

Current `main` already contains a targeted fix for the reported startup warning spam: deferred gateway plugin reloads no longer re-log plugin diagnostics, and manifest discovery now suppresses false-positive duplicate warnings when two candidates resolve to the same physical plugin directory. Real duplicate plugin ids are still intentionally reported once by precedence.

## Evidence
