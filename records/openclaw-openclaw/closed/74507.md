## Summary

Keep open. The underlying main-branch bug is real, but this branch is not ready because the new inline suppression call omits the provider config/baseUrl context required by conditional manifest suppressions, and it also carries an unrelated global ClawHub install/update surface that needs separate review.

## What This Changes

This PR adds an inline-row model suppression check and regression test for `openai-codex/gpt-5.4-mini`, while also adding `--global` ClawHub skills install/update CLI behavior with tests and docs.

## Best Possible Solution

Land a narrow resolver change that applies the same context-aware suppression check to inline rows with `api`, add coverage for both the OpenAI Codex stale-row case and a conditional Qwen-like case, and move the global skills install/update feature to its own reviewed PR.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Pass suppression context for inline matches:** `src/agents/pi-embedded-runner/model.ts:597-600`
  - body: The new inline-row check calls `shouldSuppressBuiltInModel` with only `provider` and `id`. Manifest suppressions can depend on the configured provider API/baseUrl, so inline rows with `api` can still bypass conditional suppressions. Mirror the existing check below by passing `baseUrl: providerConfig?.baseUrl` and `config: cfg`.
  - confidence: 0.88

## Security Review

Status: needs_attention

Summary: The model fix itself does not add secrets or network calls, but the branch also adds unrelated global ClawHub skill installation behavior that broadens third-party skill visibility.

Concerns:

- **[medium] Split the global ClawHub install surface:** `src/cli/skills-cli.ts:97`
  - body: `--global` installs or updates downloaded ClawHub skills under the shared managed skills directory, making them visible to all local agents by default. That is a security-sensitive behavior change unrelated to the model-suppression fix and should be split or explicitly reviewed before merging.
  - confidence: 0.86
