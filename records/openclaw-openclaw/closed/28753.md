## Summary

Current main already supports same-chat/native exec approvals for the requested channels. Slack, Discord, and Telegram each have native approval routing with origin-channel targeting, and the shared approval flow documents same-chat `/approve` plus origin-surface delivery via `channels.<channel>.execApprovals.target`.
