## What This Changes

The issue asks for `enforceChatHistoryFinalBudget` to preserve visible context or diagnostics when a single oversized message and its placeholder exceed the chat-history byte budget.

## Best Possible Solution

Keep the merged shared-helper sentinel and regression coverage; no new OpenClaw code path is needed for this report.

## Solution Assessment
