## Summary

Current `main` already honors subagent model selection during spawn, persists the chosen runtime model/provider onto the child session before launch, and has regression coverage for the exact reported symptom. The same implementation shipped in `v2026.4.22`.

## Evidence

- **Spawn plan carries requested model into child session patch:** `resolveSubagentModelAndThinkingPlan()` resolves the subagent model from config/override and includes it in `initialSessionPatch.model`, so explicit `sessions_spawn.model` and subagent defaults are applied up front.
  - file: [src/agents/subagent-spawn-plan.ts:42](https://github.com/openclaw/openclaw/blob/b453293349657dedd069d291fa5b078d910067b1/src/agents/subagent-spawn-plan.ts#L42)
- **Child session runtime model is persisted before run start:** `persistInitialChildSessionRuntimeModel()` writes `model` and `modelProvider` onto the child session entry, and `spawnSubagentDirect()` calls it before the `agent` run is started.
  - file: [src/agents/subagent-spawn.ts:204](https://github.com/openclaw/openclaw/blob/b453293349657dedd069d291fa5b078d910067b1/src/agents/subagent-spawn.ts#L204)
- **Spawn response only reports modelApplied after persistence succeeds:** The accepted response returns `modelApplied` from the persisted-model path, matching the issue’s complaint about false-positive `modelApplied: true` responses.
  - file: [src/agents/subagent-spawn.ts:1150](https://github.com/openclaw/openclaw/blob/b453293349657dedd069d291fa5b078d910067b1/src/agents/subagent-spawn.ts#L1150)
- **Regression test covers persisted child runtime model:** `spawnSubagentDirect runtime model persistence` asserts an explicit spawn model is written to the child session store before the `agent` call begins, matching the reported broken behavior.
  - file: [src/agents/subagent-spawn.model-session.test.ts:48](https://github.com/openclaw/openclaw/blob/b453293349657dedd069d291fa5b078d910067b1/src/agents/subagent-spawn.model-session.test.ts#L48)
- **Docs and shipped release reflect the fix:** Docs state that explicit `sessions_spawn.model` wins, and release tag `v2026.4.22` contains the subagent model-plan/persistence/test changes.
  - file: [docs/tools/subagents.md:84](https://github.com/openclaw/openclaw/blob/00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5/docs/tools/subagents.md#L84)
