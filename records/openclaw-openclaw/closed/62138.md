## Summary

Close as clawhub/plugin-scope. Current main explicitly treats alternative compaction behavior as a context-engine plugin concern, and the docs already point users to an existing community lossless-compaction plugin (`@martian-engineering/lossless-claw`). This PR also has unresolved review issues, so there is not strong evidence to carry a new built-in core compaction mode forward.
