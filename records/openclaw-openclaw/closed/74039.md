## Summary

Current main and the shipped v2026.4.26 release already handle the Bonjour/CIAO cancellation crash reported against 2026.4.24. The Bonjour plugin classifies `CIAO ANNOUNCEMENT CANCELLED` and `CIAO PROBING CANCELLED`, wires scoped process rejection handling, recovers/restarts stuck advertisers, and has regression coverage for the announcement-cancelled and stuck-announcing paths.

## Best Possible Solution

Close as implemented. Keep the shipped Bonjour/CIAO classifier and scoped process-error handling; if a user can still reproduce on v2026.4.26 or newer, that should be a fresh issue with updated logs, OpenClaw version, Node version, OS, and Bonjour/mDNS environment details.
