  - body: `normalizeEnvMap` only drops a small reserved-key set before passing caller/config env into a `bash` process. A setup runner should reuse or expose the existing host/skill env sanitizer policy so command-execution pivots and sensitive override names do not bypass current protections.
  - confidence: 0.9

## Security Review
