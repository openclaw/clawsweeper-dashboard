## Summary

Current main already uses provider runtime auth storage instead of process-global `PAPERCLIP_*` env mutation in `pi-embedded-runner`, so the PR's serialization patch targets a code path that no longer exists and the shipped release `v2026.4.23` already contains the replacement design.
