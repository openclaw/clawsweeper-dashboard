## Summary

Current main already has the AGENTS bootstrap reinjection and post-compaction refresh paths that address this report. AGENTS.md is injected on turns by default, continuation-skip is opt-in only, compaction invalidates continuation-skip so full bootstrap context is rebuilt, and queued/immediate followups append a post-compaction AGENTS refresh prompt. The changelog records the targeted fix as shipped in v2026.3.28, after the reported 2026.3.12/2026.3.13 versions.

## Evidence
