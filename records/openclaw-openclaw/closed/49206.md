## Summary

Keep open. The reporter's EACCES failure was fixed by switching to a user-managed Node/npm setup, but current main still runs the macOS/Linux installer through global `npm install -g openclaw@...` and the built-in npm-prefix permission repair is Linux-only. Since the original macOS reproduction can still fail when npm's global prefix is not user-writable, this is not strong enough to close as implemented, cannot-reproduce, or not-actionable.
