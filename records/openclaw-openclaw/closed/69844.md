## Summary

The request is a valid calendar integration idea, but current OpenClaw scope and docs place optional calendar/service integrations in the plugin or skill ecosystem via ClawHub rather than OpenClaw core.

## Evidence

## Risks / Open Questions

- There may still be strong user demand for an official Google Calendar or Outlook/365 integration, but the current evidence supports tracking that as plugin/ClawHub work rather than as a core issue.
- An eventual maintainer-owned official plugin could still be worthwhile; closing this core issue should not be read as rejecting the feature outright.
