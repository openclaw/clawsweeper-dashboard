## Summary

Close as implemented on main: the current Windows restart path for task-managed gateways is a Scheduled Task handoff with a hidden helper, not a normal-path visible `cmd.exe /k gateway.cmd` respawn, and the same implementation is already present in the latest release.

## Evidence
