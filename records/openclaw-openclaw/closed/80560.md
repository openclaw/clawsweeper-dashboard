## Summary

## What This Changes

This PR changes two built-in TUI slash-command descriptions in `src/tui/commands.ts` so `/new` describes spawning an isolated session and `/reset` describes resetting the current session.

## Best Possible Solution

Merge the narrow TUI description fix after normal maintainer landing; keep the separate documentation wording work in https://github.com/openclaw/openclaw/issues/76592 and https://github.com/openclaw/openclaw/pull/77192.
