## Summary

Current main already handles the reported Control UI/webchat MEDIA rendering path, and the latest release snapshot contains the same behavior. The remaining `channel=webchat` rejection is intentional: WebChat is an internal UI surface that uses `chat.send`, not the outbound message channel API.

## Evidence

- **Assistant MEDIA lines normalize into attachment blocks:** `ui/src/ui/chat/message-normalizer.ts:172` converts parsed `MEDIA:` segments into `attachment` content items instead of leaking or truncating the raw directive text.
  - file: [ui/src/ui/chat/message-normalizer.ts:172](https://github.com/openclaw/openclaw/blob/c1383680405879a3c3f43dceb20a694f54d3c13c/ui/src/ui/chat/message-normalizer.ts#L172)
  - sha: [c13836804058](https://github.com/openclaw/openclaw/commit/c1383680405879a3c3f43dceb20a694f54d3c13c)
- **Regression test covers local MEDIA attachment parsing:** `ui/src/ui/chat/message-normalizer.test.ts:208` asserts `Hello\nMEDIA:/tmp/openclaw/test-image.png\nWorld` becomes a text/attachment/text sequence, matching the issue’s repro shape.
  - file: [ui/src/ui/chat/message-normalizer.test.ts:208](https://github.com/openclaw/openclaw/blob/c1383680405879a3c3f43dceb20a694f54d3c13c/ui/src/ui/chat/message-normalizer.test.ts#L208)
  - sha: [c13836804058](https://github.com/openclaw/openclaw/commit/c1383680405879a3c3f43dceb20a694f54d3c13c)

  - file: [ui/src/ui/chat/grouped-render.ts:1117](https://github.com/openclaw/openclaw/blob/c1383680405879a3c3f43dceb20a694f54d3c13c/ui/src/ui/chat/grouped-render.ts#L1117)
  - sha: [c13836804058](https://github.com/openclaw/openclaw/commit/c1383680405879a3c3f43dceb20a694f54d3c13c)
- **UI test covers local image/PDF rendering in chat:** `ui/src/ui/chat/grouped-render.test.ts:850` verifies `MEDIA:/tmp/openclaw/test image.png` and `MEDIA:/tmp/openclaw/test-doc.pdf` render through `/__openclaw__/assistant-media` instead of showing raw `MEDIA:` text.
  - file: [ui/src/ui/chat/grouped-render.test.ts:850](https://github.com/openclaw/openclaw/blob/c1383680405879a3c3f43dceb20a694f54d3c13c/ui/src/ui/chat/grouped-render.test.ts#L850)
