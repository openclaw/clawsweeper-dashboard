## Summary

PR #44959 should close as superseded by the later open Qwen/Ollama inline tool-call PR #55416. Current main still does not execute Qwen-style embedded XML tool calls, but this older PR targets the pre-refactor Ollama path and includes unrelated lockfile churn; the remaining product work should be tracked and reworked in the canonical follow-up.

## Best Possible Solution

Close #44959 as superseded by #55416. Continue the actual fix in the canonical follow-up, ported to `extensions/ollama/src/stream.ts`, with focused tests for embedded Qwen XML promotion, argument typing, whitespace preservation, and false-positive examples; keep lockfile and unrelated formatting changes out of that patch.
