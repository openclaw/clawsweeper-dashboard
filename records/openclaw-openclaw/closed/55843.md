## Summary

Current `main` already implements Simplified Chinese localization for the Control UI/Web UI, including locale registration, a language picker in the Overview settings, shipped `zh-CN` strings, and regression coverage. The changelog records the multilingual Control UI work as shipped in `v2026.4.5`.
