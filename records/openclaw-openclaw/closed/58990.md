## Summary

Current `main` already implements Feishu typing feedback, streaming card updates, and DM tool-result delivery, so the reported "zero feedback during long tasks" no longer matches the shipped code. The separate ask for a per-message fresh DM scope is not implemented, but that is an enhancement request rather than evidence that the no-feedback bug is still present.
