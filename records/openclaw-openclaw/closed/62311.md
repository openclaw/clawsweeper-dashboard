## Summary

Current `main` no longer routes the Control UI tool-card path through Node-only home-dir/config helpers. The UI now uses a browser-safe local `shortenHomeInString` helper in `ui/src/ui/tool-display.ts`, `ui/src` has no remaining non-test imports of `src/utils.js` or `src/infra/home-dir.js`, and the same implementation is already present in release `v2026.4.23`.
