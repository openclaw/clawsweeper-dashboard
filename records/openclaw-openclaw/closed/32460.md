## Summary

Current main already emits the embedded-agent tool hooks for native tools. Embedded tools are wrapped with the pre-call hook, the embedded tool lifecycle emits the post-call hook, the SDK docs describe the supported hook names as `before_tool_call`/`after_tool_call`, and the shipped `v2026.4.22` tree already contains that implementation.
