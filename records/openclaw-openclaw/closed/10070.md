## Summary

Current main already implements the requested Windows-compatible A2UI bundling flow: `canvas:a2ui:bundle` runs a Node script, and that script uses the shared cross-platform pnpm runner with explicit Windows `cmd.exe`/`pnpm.cmd` handling.
