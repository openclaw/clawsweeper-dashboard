## Summary

Current `main` does not implement this PR’s ACP stall-classification fix. The spawned-run relay still treats silent ACP runs as generic interactive-input stalls, and both ACP event bridges still drop the intermediate ACP status/tool signals this PR is trying to preserve. The PR discussion also matters here: the author addressed multiple review findings in follow-up commits, so this remains a live, useful patch rather than an obsolete draft.
