## Summary

Current `main` already refreshes the gateway service definition from the post-update install root before restart, and that service install path writes the managed `OPENCLAW_SERVICE_VERSION` value. The submitted PR is therefore redundant, and its own review thread identified correctness gaps in the proposed restart-script env pinning.
