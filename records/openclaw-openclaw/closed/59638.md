## Summary

PR #59638 is a focused third-party Chuizi.AI model-provider plugin, and the existing OpenClaw plugin/provider SDK already supports this shape outside core. Current project direction keeps optional capabilities out of bundled core unless there is a strong maintainer/product reason, with plugin discovery, provenance, and promotion handled through ClawHub/npm.

## Best Possible Solution

Close the bundled-core PR and ask the contributor to publish Chuizi.AI as an external native provider plugin on ClawHub/npm using the existing provider plugin SDK. If that externalization exposes a concrete missing SDK or onboarding seam, track that as a narrow plugin API issue or PR.
