  - confidence: 0.93
- **[P2] Register APNs through node.event:** `apps/visionos/Sources/APNsManager.swift:59-62`
  - body: The Gateway handles APNs registration through `node.event` with event `push.apns.register`; `node.apns` is not in the supported method surface. This request will be rejected, so the node will not publish its APNs token.
  - confidence: 0.91
- **[P2] Guard delayed reconnects from active sessions:** `apps/visionos/Sources/NodeManager.swift:132-136`
  - body: A scheduled reconnect always calls `connect()` after the sleep. If the user reconnects manually or the app becomes healthy before the delay expires, this stale task can call `GatewaySocket.connect()`, whose first action is to disconnect the current socket.
  - confidence: 0.86
- **[P2] Prevent stale disconnect tasks from closing new sockets:** `apps/visionos/Sources/NodeManager.swift:117-120`
  - body: `disconnect()` waits inside an async task before calling `socket.disconnect()` on the shared socket instance. A quick reconnect can create a new WebSocket before that older task resumes, letting the stale task close the new healthy connection.
  - confidence: 0.87
