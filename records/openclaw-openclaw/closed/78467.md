Overall correctness: patch is incorrect

Full review comments:

- **[P2] Request full lsof command names before classifying Chrome:** `extensions/browser/src/browser/chrome-mcp.ts:1121`
  - body: On macOS, lsof's default COMMAND column is truncated to the first nine characters unless `+c0` is passed, but this parser compares that field against full names like `Google Chrome`.
