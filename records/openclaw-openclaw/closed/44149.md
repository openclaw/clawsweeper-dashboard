## Summary

Close as clawhub. The issue asks for a Feishu/Lark-specific agent tool to download message attachments by message_id/file_key. Current main already has inbound Feishu media download plumbing and tests, but the remaining standalone tool request is an optional service-specific plugin capability, not a missing OpenClaw core API or protected maintainer item. VISION.md directs optional capabilities and plugin promotion to external plugins/ClawHub, and the plugin docs show agent tools can be registered by plugins.
