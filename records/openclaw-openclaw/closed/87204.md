Full review comments:

- **[P1] Keep session refreshes out of the picker filter:** `ui/src/ui/app-chat.ts:108-110`
  - body: `createChatSessionsLoadOverrides` is used by general chat/session refresh paths, not only the picker. The picker request builder on current main already adds `agentId` for picker loads, so keep this scoping local to the picker path.
