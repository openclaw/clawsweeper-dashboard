## Summary

Current `main` already provides the requested pre-send hard gate via the `message_sending` plugin hook: it can cancel or rewrite outbound assistant text before channel delivery, and delivery code skips sending when the hook cancels.

## Evidence

- **Hook contract supports cancel or replacement:** `PluginHookMessageSendingResult` exposes `content?: string` and `cancel?: boolean`, matching the requested replacement/block behavior.
  - file: [src/plugins/hook-message.types.ts:45](https://github.com/openclaw/openclaw/blob/6b618f063516aa9ba85903aa4565c5a204c055b2/src/plugins/hook-message.types.ts#L45)

  - file: [src/infra/outbound/deliver.ts:567](https://github.com/openclaw/openclaw/blob/6b618f063516aa9ba85903aa4565c5a204c055b2/src/infra/outbound/deliver.ts#L567)
