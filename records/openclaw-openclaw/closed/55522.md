## Summary

#55522 is a request for OpenClaw maintainers to publish a public/sideloadable alpha iOS IPA. Current main already has source/manual iOS install and maintainer-only Fastlane beta archive/TestFlight upload paths, while the docs explicitly state that public iOS distribution is not available. The remaining request is release/signing/distribution administration and product policy, not an actionable source-repo code or docs fix.

## Best Possible Solution

Keep the current repo behavior: source/manual Xcode deploy remains available for users, maintainers can archive or upload beta builds through the existing Fastlane flow, and any future public IPA/AltStore distribution should be handled as an explicit maintainer release policy decision outside this issue.
