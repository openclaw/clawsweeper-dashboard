## Summary

Current `main` already addresses the reported noisy REM-theme tokens by expanding concept stop-word filtering and adding a regression test for the exact chat-scaffolding pattern described in the issue. The latest tagged release `v2026.4.23` predates this delta, so the fix appears present on `main` but not yet tied to a shipped release from the evidence available here.

## Evidence

- **Current main adds the reported chat-scaffolding stop words:** `LANGUAGE_STOP_WORDS` on current `main` now includes `assistant`, `system`, and `user` in `shared`, plus `the`, `you`, and `your` in `english`, which directly covers the garbage reflection examples from the report.
