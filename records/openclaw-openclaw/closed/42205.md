## Summary

Current main already implements the reported fix: Gateway hello/bootstrap version reporting now resolves from the runtime/package version ahead of stale service markers, the Control UI waits for bootstrap before its first websocket connect, and regression tests cover the version-precedence path. The same implementation is present in the tagged release `v2026.4.22`.
