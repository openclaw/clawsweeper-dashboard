## What This Changes

Request to keep redacted raw config text available for read-only Control UI display instead of returning `raw: null` and forcing Form mode when safe raw round-trip validation fails.

## Best Possible Solution

Keep one canonical tracker for the Raw mode-disabled regression in #59330, preserve this issue's read-only Raw view discussion by linking it there, and resolve the shared bug with a tested gateway/UI contract that separates safe editing from safe display only if maintainers approve that product/security direction.

## Reproduction Assessment
