## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.87

Full review comments:

- **[P1] Preserve symlink-parent rejection during rotation:** `src/agents/queued-file-writer.ts:87`
  - body: This rotation path runs raw `fs.rename`/`fs.rm` before `safeAppendFile`. The existing writer contract rejects symlinked parent directories, but with `maxFiles` enabled an oversized active file under a symlinked parent can be renamed or unlinked through that parent before `appendRegularFile` enforces `rejectSymlinkParents`; validate the path with the same safe regular-file contract before rotating, or skip rotation when any parent is a symlink.
  - confidence: 0.88
- **[P2] Remove the zero queue cap from the sample config:** `docs/reference/prompt-caching.md:317`
  - body: The sample documents `maxQueuedBytes: 0` as the unlimited default, but the writer drops whenever `queuedBytes + lineBytes > maxQueuedBytes`; with a zero cap every non-empty trace line is dropped. Leave this key unset in the example, or show a positive bound.
  - confidence: 0.96
