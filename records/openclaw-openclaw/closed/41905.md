## Summary

Current `main` already covers the reported Bailian/OpenAI-compatible streaming-usage failure: usage aliases are normalized, DashScope-compatible endpoints keep streaming-usage compat, WebSocket totals fall back when `total_tokens` is missing/zero, and `/status`/session usage now backfill from transcript usage instead of sticking at `0`.
