## Summary

This PR adds a RunPR-specific cold outreach automation bundle under a new root `agents/` tree. Current main already has the generic pieces needed to build this outside core: ClawHub/skill/plugin distribution, workspace skills, scheduled tasks, Exa search, Gmail via `gog`, and iMessage via `imsg`. The proposed diff does not identify a missing generic OpenClaw API, and it includes personal/account-specific operational details that make it a poor fit for the core repository.

## Best Possible Solution

Close this PR as ClawHub/external-bundle work. The useful path is to keep the RunPR-specific outreach logic in a private repository or publish it as a ClawHub skill/bundle/plugin that uses existing OpenClaw scheduled tasks, Exa search, `gog`, and `imsg`; any truly missing generic API should be proposed separately as a small plugin/automation API issue.
