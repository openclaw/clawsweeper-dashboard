## Summary

Current main already implements the reported Control UI image-upload path: pasted/uploaded images are previewed, sent via `chat.send` attachments, persisted into transcript media fields, and rendered back into chat bubbles. The same code is present in released `v2026.4.23`, so this is already shipped beyond the reporter's `2026.4.9` build.
