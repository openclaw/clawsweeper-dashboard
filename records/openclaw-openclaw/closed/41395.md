## Summary

Issue #41395 is a deployment-specific audit note about one Hostinger gateway being outdated, not an actionable OpenClaw bug report or feature request. Current main already documents the supported update paths for Hostinger and self-managed installs, and the code treats "behind main" as update-status information rather than a product defect.
