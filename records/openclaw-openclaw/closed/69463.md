  - body: The added helper imports `@mariozechner/pi-agent-core`, but current main no longer depends on that package and uses the local `src/agents/runtime` facade backed by `packages/llm-core`. - **[P2] Do not default session names to raw session keys:** `src/agents/observability-session-headers.ts:39`
  - body: When tracing is enabled and `sessionName` is omitted, this sends the raw `sessionKey` as `x-session-name` on every provider request.   - confidence: 0.9
- **[P2] Block reserved tracing header names:** `src/agents/observability-session-headers.ts:30-31`
  - body: The configurable header names are accepted as arbitrary strings, but current transports merge request `options.headers` last over provider defaults. A config value like `Authorization`, `x-api-key`, or native session headers could override auth/routing headers, so this needs validation or a denylist before merge.
  - confidence: 0.87
