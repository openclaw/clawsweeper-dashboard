## Summary

Issue #46421 does not match the current OpenClaw codebase. Current `main` and latest release use SQLite/FTS/vector memory search, not a Neo4j/entity-graph pipeline, and the repo/history contain no `neo4j`, `structuredGraphSearch`, or `entity_embedding_index` implementation to verify or fix.
