## Summary

## What This Changes

Split the combined chat session picker into separate Agent and Agent Session controls while keeping Model and Thinking bound to the active session.

## Best Possible Solution

Keep the current main implementation; the optional internal/transient-session visibility cleanup should remain tracked separately by #70851.
