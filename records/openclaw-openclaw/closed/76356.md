  - confidence: 0.91

  - body: The mapper recognizes startup-failed, but server-chat only finalizes, persists, and broadcasts end/error phases. If startup-failed becomes terminal, session subscribers and persisted session state will not see it; with the follow-up end, they see the wrong terminal state instead.
  - confidence: 0.86
