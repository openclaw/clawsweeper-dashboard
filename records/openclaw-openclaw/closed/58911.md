## Summary

The reported breakage is in a third-party plugin relying on channel-branded Telegram SDK helper seams that current OpenClaw docs treat as non-public/compatibility surfaces. Current project scope points external plugin compatibility and promotion to ClawHub/plugin maintenance, and the named plugin has since documented a compatible release path.
