## Summary

Keep open: current `main` still contains the reported scope mismatch. Dream-narrative cleanup unconditionally calls `deleteSession`, that runtime maps to admin-only `sessions.delete`, and gateway tests explicitly assert the non-admin fallback path throws `missing scope: operator.admin`, which matches the issue report rather than showing a fix.
