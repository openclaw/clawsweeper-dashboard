## Summary

Current main contains a shipped fix path for this regression: the packaged install was slimmed in the 2026.4.15 release, and release validation now explicitly blocks a repeat of the 2026.3.12 oversized npm pack that correlated with low-memory gateway/doctor OOMs.

## Evidence
