## Summary

Current main already gives iOS Talk Mode an on-device system TTS path backed by AVSpeechSynthesizer, and selecting a non-ElevenLabs Talk provider such as system routes iOS away from the ElevenLabs API path into that local speech fallback. The exact nativeVoiceId/nativeRate sketch is not implemented, but the core requested capability is present.
