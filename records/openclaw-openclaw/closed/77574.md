## Summary

## What This Changes

`clawdock-dashboard` should fetch the dashboard URL without starting or recreating the Compose gateway or changing the running gateway image.

## Best Possible Solution

Land https://github.com/openclaw/openclaw/pull/80494 or an equivalent focused main-branch change, then close this issue after the fix merges.

## Reproduction Assessment

## Solution Assessment

Yes. Adding `--no-deps` only to the dashboard URL lookup is the narrowest maintainable fix because it preserves the existing CLI path while preventing Compose dependency startup for a URL-only command.
