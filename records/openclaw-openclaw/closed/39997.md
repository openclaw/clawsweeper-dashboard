## Summary

Keep open. Current `main` still has both behaviors this PR targets: Feishu upload fallbacks can echo local media paths back to users, and Feishu is still treated as a native voice-note TTS channel. The code, tests, and changelog all show this is not already implemented on `main`, so there is no conservative close basis.
