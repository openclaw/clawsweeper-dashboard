## Summary

Keep open. Current `main` still enforces the same gating the report describes: `mode="session"` requires `thread=true`, and `thread=true` still fails when no channel plugin provides `subagent_spawning` hooks. Tests on `main` lock in those rejection paths, and the issue timeline already shows a follow-up PR (#67790) still open rather than a landed fix.
