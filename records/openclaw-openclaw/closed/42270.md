## Summary

Current main addresses the concrete failure mode the issue discussion narrowed to: restricted/Tailscale network-interface enumeration failures now degrade instead of crashing, and LM Studio is now a first-class local provider path. The unmerged PR #44501 is not the basis for closure; the close is based on current main code, tests, and changelog evidence.
