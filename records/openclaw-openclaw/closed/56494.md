## Summary

Current `main` already removes the repeated Feishu full-registration path the issue describes: gateway config schema loads are manifest-only instead of activating plugins, configured channel startup prefers setup-only loading before the deferred full pass, and gateway tool resolution reuses the active registry instead of reloading plugins.

## Evidence
