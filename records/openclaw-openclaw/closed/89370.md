## Best Possible Solution

## Solution Assessment

Yes on code shape: relaxing the shared ID lower bound is the narrowest maintainable fix because raw, path, query, stored-theme, and fetch normalization all use the same validator. Merge readiness is still blocked on real behavior proof rather than a code change.
