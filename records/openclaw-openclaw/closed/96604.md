## Best Possible Solution

## Solution Assessment

Yes. Reusing `readProviderJsonResponse` inside the Qwen plugin is the narrowest maintainable fix because the shared helper already owns the byte cap, cancellation, and malformed-JSON normalization; a custom Qwen reader would duplicate that policy.
