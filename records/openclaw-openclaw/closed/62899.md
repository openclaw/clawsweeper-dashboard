## Summary

Current main documents session-level model changes as persistent shared session state, exposes an agent-accessible reset via `session_status(model=default)`, and has tests covering reset flows for per-session, fallback-pinned, and `/reset`-driven model restoration. The reporter’s `sessions_spawn` child-session workaround matches the documented transient-isolation pattern.
