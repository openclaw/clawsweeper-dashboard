## Summary

Keep open. Current `main` still contains the reported failing path: `memory-core` dreaming best-effort cleanup calls `subagent.deleteSession()`, the gateway subagent runtime dispatches that as `sessions.delete`, `sessions.delete` is still admin-scoped, and current tests explicitly assert that fallback synthetic plugin subagent deletion must fail without minting `operator.admin`. The issue discussion also includes a fresh 2026-04-25 repro on `v2026.4.23`.
