## Summary

The reported Docker build break for v2026.3.13 is no longer present on current main: the missing `batchViaPlaywright` export is restored, the browser act request types now include the `selector`/`delayMs` shape that the reported TS errors complained about, regression tests cover those paths, and the current `build:docker` script no longer runs the older strict DTS step that surfaced the failure in the issue log.
