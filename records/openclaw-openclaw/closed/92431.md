## Summary

The PR addresses a real launch-time regression: current main rejects programmatic unsupported thinking overrides before its existing fallback can run. ## What This Changes

The PR makes unsupported programmatic thinking overrides use the existing supported-level fallback while preserving hard errors for interactive `--think` input.

## Best Possible Solution
