## Summary

Issue #44854 is already fixed on current `main`, and the shipped `v2026.4.22` tag also contains the fix. The config-defaults path no longer references any `ANTHROPIC_MODEL_ALIASES` symbol; Anthropic pruning/cache-retention defaults use a local ref parser instead, with regression coverage for the affected defaulting path.
