## Summary

Current `main` already fixes the reported duplicate model-picker behavior in the Control UI by normalizing server model refs and deduplicating rendered picker options, and the underlying issue `#51439` was already closed as implemented. This PR is therefore superseded, and it also carries unrelated workflow changes with unresolved review feedback.
