## Summary

Current main already supports Telegram per-topic model routing through the documented `channels.modelByChannel.telegram["<chatId>:topic:<threadId>"]` path, and current validation also recognizes `plugins.allow`. The reported `channels.telegram.groups.*.topics.*.model` key is still not part of the Telegram topic schema, but the issue's requested fallback of documenting the supported override location is already implemented and shipped.
