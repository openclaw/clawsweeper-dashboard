## Summary

## What This Changes

The issue asks whether runtime-managed model auth support is missing from OpenClaw 2026.5.12-beta.1 and how plugins should detect it.

## Best Possible Solution

Keep the existing `api.runtime.modelAuth` and `openclaw/plugin-sdk/provider-auth-runtime` contract; any remaining Windows log should be investigated as a stale install or plugin-loading repro with exact package paths.

## Solution Assessment
