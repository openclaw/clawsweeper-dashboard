## Summary

Issue #19396's requested scoped opt-in for private/self-hosted Mattermost media fetches is already implemented on current `main` and is present in the latest release. The shipped config key is `channels.mattermost.network.dangerouslyAllowPrivateNetwork` (with legacy `allowPrivateNetwork` migration support), and inbound Mattermost media downloads now pass the Mattermost base hostname through the SSRF policy so private/local Mattermost hosts can be allowed explicitly.
