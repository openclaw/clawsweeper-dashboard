## Summary

Current `main` now provides native voice input in the web Control UI chat composer: a microphone dictation button backed by browser speech recognition plus a realtime Talk button backed by gateway-issued browser voice sessions. That means the issue's core gap, lack of native webchat voice input, is already addressed on `main`.
