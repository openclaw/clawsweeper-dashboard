## Summary

The request is a valid feature idea, but current `main` treats ACP-backed providers/harnesses as plugin territory, with explicit external plugin and ClawHub guidance plus public SDK seams for provider+harness integration.

## Risks / Open Questions

- Third-party agent-harness support is still documented as experimental, so an external ACP-backed provider plugin may need to track SDK/runtime changes more closely than a normal provider plugin.
