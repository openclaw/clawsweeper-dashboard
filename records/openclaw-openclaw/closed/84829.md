## Summary

## What This Changes

Report asks OpenClaw to stop discarding completed embedded LLM responses when another entry point writes to the same session during the released prompt-lock window.

Keep one canonical session-takeover thread open, fold this report's concurrent same-session evidence into it, and land a single session-lock fix that prevents lost completed turns without weakening external takeover detection.
