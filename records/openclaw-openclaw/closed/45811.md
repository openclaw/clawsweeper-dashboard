## Summary

Keep open. Current `main` still contains the exact behaviors this PR proposes to fix, so it is neither implemented on main nor obsolete: browser tool params still use `Boolean(...)` coercion for `fullPage` and `accept`, and CDP screenshot capture still forces `captureBeyondViewport: true` even when full-page capture is not explicitly requested.
