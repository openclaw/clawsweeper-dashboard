## Summary

Current main solves the Signal agent-workspace attachment failure via the shared reply-media normalizer: Signal replies go through the generic dispatch path, local reply media is staged with agent-scoped media access before channel delivery, and the v2026.4.24 changelog records the shipped fix for workspace-rooted absolute media paths.
