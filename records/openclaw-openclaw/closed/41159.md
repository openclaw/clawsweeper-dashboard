## Summary

Close as `clawhub`. On current `main`, new model providers and web-search providers are expected to ship through the plugin system and ClawHub/npm distribution, while this PR targets core files that are now legacy integration points. The feature idea is still valid, but it should be rebuilt as a provider plugin rather than merged as direct core wiring.
