## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Do not rehydrate explicitly removed web config:** `src/secrets/runtime.ts:526-530`
  - body: The refresh hook restores omitted web-tool and plugin web config from the active snapshot before every refresh. Since config writes re-read and publish the canonical persisted source config, omission can be an intentional removal; this can keep old web provider or plugin credential metadata active until restart. Distinguish stripped-source refreshes from explicit removals, or use the superseding provenance-based fix.
  - confidence: 0.87
