## Summary

Current main implements the requested Control UI avatar behavior for workspace-relative/data/same-origin configured agent avatars. It normalizes configured avatar paths to Control UI avatar routes, serves avatar metadata and bytes through `/avatar/<agentId>`, renders authenticated local avatars as managed blob URLs, and allows those blobs in CSP. The later external-link favicon comment is a separate enhancement and is intentionally outside the current remote-image policy.
