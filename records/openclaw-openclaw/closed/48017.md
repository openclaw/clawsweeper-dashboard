## Summary

PR #48017 is a useful optional model-provider integration, but current main has a mature provider-plugin surface that covers GigaChat-style auth, onboarding, model catalog, token exchange, tool-schema compatibility, and custom streaming. It should be extracted and published as an external GigaChat provider plugin through ClawHub/npm rather than merged as a large core PR.

## Best Possible Solution

Close this core PR and ask the contributor to extract the useful GigaChat implementation into an external provider plugin with tests/docs, then publish it through ClawHub or npm. If extraction exposes a missing generic provider API, open a focused SDK/API proposal instead of merging GigaChat-specific core code.
