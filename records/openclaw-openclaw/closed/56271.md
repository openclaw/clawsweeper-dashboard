## Summary

This issue asks for opt-in browser auto-close after task completion. That same remaining browser lifecycle problem is already tracked by open #40694, which covers browser-opened temporary tabs/windows closing after task completion. Current main still has only idle/cap-based primary tab cleanup plus subagent/cron/session lifecycle cleanup, so the right maintainer path is to keep the canonical issue open and close this as a duplicate.

## Best Possible Solution

Close this issue as a duplicate and keep #40694 as the canonical open thread for browser task-completion cleanup, including any opt-in config shape and Control UI behavior.
