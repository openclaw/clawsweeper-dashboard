## Summary

Current main and the v2026.4.24 release address the reported npm global install failure pattern by removing node-llama-cpp from the default published dependency path and documenting/staging runtime dependencies outside packaged global installs. The issue's 2026.4.12 follow-up is consistent with this affecting older releases, but the latest release has the relevant packaging/runtime-dependency fix.
