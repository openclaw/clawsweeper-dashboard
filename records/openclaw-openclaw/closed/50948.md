## Summary

Current main already implements inline image rendering in WebChat/Control UI for assistant image attachments, with regression coverage for `MEDIA:` image directives, `image_url` blocks, managed/generated images, and local assistant attachments. The changelog shows this shipped in release `v2026.4.22`, with earlier structured media bubble support noted in `2026.4.11`.

## Evidence
