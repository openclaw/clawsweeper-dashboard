## Summary

Current `main` already addresses the reported embedded-run timeout behavior: the old hard 600s path is gone, agent/LLM timeouts are configurable, idle stalls are retried once then surfaced as explicit errors instead of leaving the session apparently hung, and the queueing behavior during an active run is now documented with Stop/Steer controls. This shipped by `v2026.4.22` (`00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5`).
