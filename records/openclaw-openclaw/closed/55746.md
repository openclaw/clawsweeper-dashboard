## Summary

Current `main` already implements this uninstall cleanup, and the same behavior is present in release `v2026.4.22`. The uninstall path removes plugin-owned `channels.*` config for installed plugins, preserves built-in/shared channel keys, shows the removal in CLI preview/output, and has direct test coverage for the reported scenarios.

## Evidence
