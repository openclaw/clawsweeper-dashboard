  - body: Current main callers pass media-source context as a third argument to `resolveMediaToolLocalRoots`, but this patch changes the helper to a two-argument signature. Rebase the extra-root threading through the current media contract so image, PDF, music, and video callers keep compiling and enforcing the same policy.
  - confidence: 0.78
