## Summary

Current main already contains the Control UI fix path described in the issue: the context notice now hides on stale totals and uses fresh `totalTokens`/`contextTokens`, the notice is rendered as an inline pill instead of an overlay, tracked chat runs refresh sessions on final events, and there is regression coverage for the stale-token case. The same implementation is present in the shipped `v2026.4.22` tree.

## Evidence
