## Summary

Current `main` already auto-routes GitHub Copilot Claude models to `anthropic-messages`, keeps non-Claude Copilot models on `openai-responses`, documents that behavior, and covers it with targeted tests. The issue discussion also shows the reporter narrowed the original timeout investigation to this transport-selection behavior before reopening under the new title.
