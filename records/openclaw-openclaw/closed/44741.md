## Summary

Current main already implements the requested Feishu document comment-collaboration capability, but it lives on `feishu_drive` rather than `feishu_doc`. The shipped surface includes `list_comments`, `list_comment_replies`, `add_comment`, and `reply_comment`, plus comment-thread handling for Feishu document workflows.
