## Summary

Close as ClawHub/plugin-surface work. The exact Internal Hooks `cron:*` contract is still absent, but current main now exposes a typed `cron_changed` plugin hook that carries the cron lifecycle actions needed for external audit/sync/observability plugins. Keep OpenClaw core focused on the generic plugin API and only reopen core work if a concrete missing plugin API prevents external implementation.
