## Summary

Current `main` already carries a shipped macOS LaunchAgent restart fix for both `openclaw gateway restart` and update-triggered restarts. The changelog places the relevant Gateway/macOS repair in release `2026.4.5`, and current code/tests still implement and cover the bootstrap-and-retry behavior needed for the reported regression.
