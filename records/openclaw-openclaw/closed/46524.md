## Summary

Current `main` no longer uses the `ANTHROPIC_MODEL_ALIASES` const in model normalization. Anthropic shorthand normalization is now implemented as an inline switch in a separate shared module, and current tests cover the alias paths the report described. The shipped `v2026.4.22` release already contains this implementation.
