## Summary

Current main already registers `message_sent` as a typed plugin hook, routes normal external-channel replies through the outbound delivery path, and emits `runMessageSent(...)` there with regression coverage. The same outbound hook files are unchanged between current main and release `v2026.4.22`, so this behavior is already shipped.
