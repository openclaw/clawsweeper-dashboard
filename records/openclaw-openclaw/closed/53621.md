## Summary

Current `main` already implements both parts of the report: `sandbox explain` infers the channel from agent session keys like `agent:main:discord:slash:<userId>`, and Discord native slash command contexts now carry `Provider: "discord"` into elevated gating, so `tools.elevated.allowFrom.discord` resolves on the normal runtime path.

## Evidence
