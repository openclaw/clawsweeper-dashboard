- **[P1] Fail closed when changed-path discovery fails:** `scripts/generate-npm-shrinkwrap.mjs:716-717`
  - body: `listCheckChangedPaths` catches every git error and returns an empty path list, but `--check` uses that list to decide whether to reuse current shrinkwrap pins.
