## Summary

Current `main` already routes Signal outbound send dependency resolution through the public Plugin SDK runtime seam, so the broken `../../../src/infra/outbound/send-deps.js` runtime import reported in #46609 is no longer present. The shipped `v2026.4.22` tag also contains the fixed import path and the public `plugin-sdk/outbound-runtime` export.
