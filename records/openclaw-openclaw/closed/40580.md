## Summary

The PR targets `docs/zh-CN/channels/feishu.md`, but current `main` no longer keeps localized docs in this repository. The docs policy now treats English docs in `docs/` as the source of truth and sends generated locale trees to the separate `openclaw/docs` publish repo, so this PR's change does not apply to the current codebase.
