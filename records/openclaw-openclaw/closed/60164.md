## Summary

Current `main` already routes CLI `console.*` output through the normal file logger, and that logger is level-filtered by `logging.level`, so `console.log`/`console.info` are dropped when file logging is set to `warn` or higher.

## Evidence

- **CLI still uses console capture:** The CLI enables console capture during startup, so the reported path is still the intended path to inspect.
  - file: [src/cli/run-main.ts:210](https://github.com/openclaw/openclaw/blob/a155b84cdadef068de50274318952032cd867788/src/cli/run-main.ts#L210)
  - command: `nl -ba src/cli/run-main.ts | sed -n '200,220p'`
  - sha: [a155b84cdade](https://github.com/openclaw/openclaw/commit/a155b84cdadef068de50274318952032cd867788)
- **console.log maps to info-level file logging:** `enableConsoleCapture()` forwards `console.log/info` to `resolvedLogger.info(...)`, `console.warn` to `warn`, and `console.error` to `error`, so captured console output goes through the normal logger level filter instead of bypassing it.
  - file: [src/logging/console.ts:244](https://github.com/openclaw/openclaw/blob/a155b84cdadef068de50274318952032cd867788/src/logging/console.ts#L244)

  - sha: [a155b84cdade](https://github.com/openclaw/openclaw/commit/a155b84cdadef068de50274318952032cd867788)
- **File logger minLevel comes from logging.level:** The logger resolves its file-log level from config and builds the tslog instance with `minLevel: levelToMinLevel(settings.level)`, which is the mechanism that filters out lower-severity records such as info when configured at warn.
  - file: [src/logging/logger.ts:123](https://github.com/openclaw/openclaw/blob/a155b84cdadef068de50274318952032cd867788/src/logging/logger.ts#L123)

- **Tests cover file-log level filtering:** The test suite asserts that with `level: "warn"`, an info log is omitted while a warn log is retained, and `isFileLogLevelEnabled()` separately verifies that `warn` disables `info`/`debug`/`trace`. That matches the expected behavior for captured `console.log` output.
  - file: [src/logger.test.ts:62](https://github.com/openclaw/openclaw/blob/a155b84cdadef068de50274318952032cd867788/src/logger.test.ts#L62)
