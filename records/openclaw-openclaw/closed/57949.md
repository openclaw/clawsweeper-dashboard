## Summary

The reported scoped-package ClawHub install failure is already fixed on current `main`: temp download targets now sanitize the archive filename, the ClawHub installer keeps the archive alive through installation, and there is a regression test covering a scoped package name. The same implementation is present in release `v2026.4.23`.

## Evidence
