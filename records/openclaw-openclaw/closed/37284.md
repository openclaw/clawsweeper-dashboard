## Summary

Keep open. Current `main` partially covers the request: managed browser flows already support right-click through the existing `click` action (`--button right`), but the public browser act/CLI surface still has no `mousedown`, `mousemove`, or `mouseup` actions. The issue asks for both right-click and discrete mouse events, so it is not fully implemented on `main`.

## Evidence
