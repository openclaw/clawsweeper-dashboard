## Summary

Issue #11940 is best closed as a ClawHub/community plugin request, not kept open for OpenClaw core. Current project scope explicitly keeps optional capability out of core, prefers externally hosted plugins, and uses ClawHub as the canonical discovery surface for community plugins. The requested feature is already framed as an external UI plugin in its own repository, and current main already exposes model/fallback/auth surfaces that such a plugin can build on.
