## Summary

Current `main` contains a focused set of fixes for the reported upgrade regression: it now remaps unsupported persisted thinking levels, prevents fallback persistence from clobbering legacy/user-selected model state, clears provider-scoped auth overrides when fallback switches providers, and hardens OpenAI Codex OAuth refresh fallback so stale auth state does not force re-onboarding.
