## Summary

Current main already has an optional bundled `active-memory` feature that performs a pre-reply memory recall pass and injects the result into prompt context before the main agent answers.

## Evidence

- **Pre-prompt hook already exists:** `extensions/active-memory/index.ts` registers a `before_prompt_build` hook, builds a query from the latest user message/recent turns, runs active recall, and returns `prependContext` when it finds relevant memory.
