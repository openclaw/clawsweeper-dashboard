## Summary

Close as superseded: https://github.com/openclaw/openclaw/pull/79959 is the earlier open canonical PR for the same `originAgent` feature and uses the safer sentinel-preserving design, while this branch overwrites `provider` and would break replay filtering.

## What This Changes

The PR adds optional `originAgent` threading for `chat.inject` provider attribution and also includes an unrelated embedded-runner usage fallback change.

## Best Possible Solution

Use https://github.com/openclaw/openclaw/pull/79959 as the canonical implementation path and keep `provider`/`model` as immutable replay sentinels while adding any display attribution separately.

## Reproduction Assessment

Yes. Source inspection shows a `chat.inject` call with non-empty `originAgent` would persist a `gateway-injected` assistant row whose provider is not `openclaw`, so current replay filtering would no longer drop it.

## Solution Assessment

No. Overriding `provider` is not the best fix because that field is part of the transcript-only replay contract; the canonical safer approach is a separate `originAgent` display field.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.92

Full review comments:

- **[P1] Preserve the gateway-injected replay sentinel:** `src/gateway/server-methods/chat-transcript-inject.ts:91`
  - body: Changing the persisted `provider` to `params.originAgent` makes injected assistant rows stop matching the replay filter, which only drops rows with `provider === "openclaw"` and model `gateway-injected`. A `chat.inject` call with `originAgent: "hermes"` would therefore leak transcript-only UI text back into the next model request.
  - confidence: 0.94
- **[P2] Remove the unrelated embedded-runner usage change:** `src/agents/pi-embedded-runner/run.ts:1311-1312`
  - body: This PR is scoped to gateway `chat.inject` attribution, but it also changes embedded-runner usage accounting. That unrelated change belongs in its own PR or the existing usage issue path so this attribution review can be reasoned about independently.
  - confidence: 0.85
