## Summary

Current `main` already handles env-proxy bootstrap through the guarded `ensureGlobalUndiciEnvProxyDispatcher()` path, the OpenAI Codex network paths call that helper before outbound requests, and strict SSRF behavior still intentionally refuses ambient env proxies unless `TRUSTED_ENV_PROXY` is selected. This PR is therefore superseded on `main`, and its `fetch-guard` change would regress a preserved strict-mode invariant.
