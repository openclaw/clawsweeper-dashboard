## Summary

Current main already contains the Slack Socket Mode fixes that match this report: inbound channel/private-channel events are routed through the unified `message` handler instead of invalid `message.channels` / `message.groups` Bolt registrations, and the `@slack/bolt` interop path now handles the `App is not a constructor` failure mode with regression coverage.
