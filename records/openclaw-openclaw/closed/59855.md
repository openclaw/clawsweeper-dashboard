## Summary

Current `main` already contains targeted Gateway pairing fixes for this regression class: trusted loopback shared-secret clients are treated as local for pairing, and shared-secret loopback CLI reconnects with stale metadata now auto-approve instead of failing with `1008 pairing required`. The current docs also explicitly document the no-prompt exec approval policy (`security: "full"`, `ask: "off"`) for trusted single-operator setups.
