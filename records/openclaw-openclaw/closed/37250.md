## Summary

Current `main` already preserves timeout-era sub-agent progress well enough for this request: timeout announce now derives partial-progress text from child chat history, timed-out sessions are kept until later archive, and `sessions_history` can still return tool-call/tool-result rows when `includeTools: true`. This appears implemented on `main` after the latest tagged release.
