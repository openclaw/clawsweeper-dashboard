## Summary

Current main has moved Feishu off the old 2026.3.24 registration path. Feishu now uses split bundled setup/full channel entries, the bundled channel contract only calls `registerFull` in `full` mode, and gateway bootstrap has provisional plugin-log suppression. That is sufficient evidence that the duplicate Feishu registration log path reported against 2026.3.24 is already addressed on main.
