## Summary

Current main already carries the fix for this regression. The changelog for `2026.4.20` explicitly calls out `#42194` and says OpenAI Codex now normalizes legacy `openai-completions` overrides on default OpenAI/Codex and GitHub Copilot-compatible hosts back to the native Codex Responses transport. The implementation is present in `extensions/openai/openai-codex-provider.ts`, with regression coverage in provider and embedded-runner tests, plus doctor warnings for stale legacy overrides.
