## Summary

The reported Windows `ERR_UNSUPPORTED_ESM_URL_SCHEME` onboarding/configure failure was already fixed after `2026.4.5`; current `main` still contains the Windows path-normalization guard and regression coverage, and the fix shipped in `v2026.4.7`.

## Evidence

- **Shipped release note:** The `2026.4.7` changelog explicitly says Windows onboarding/configure flows stop tripping `ERR_UNSUPPORTED_ESM_URL_SCHEME` in plugin loader paths, which matches this report against `2026.4.5`.
  - file: [CHANGELOG.md:1193](https://github.com/openclaw/openclaw/blob/245451b6a9c529464e177afa65013adca0d63eb5/CHANGELOG.md#L1193)
  - sha: [245451b6a9c5](https://github.com/openclaw/openclaw/commit/245451b6a9c529464e177afa65013adca0d63eb5)
- **Current Windows import guard:** Current `main` documents the exact Windows failure mode and converts raw absolute `C:\...` paths into `file://` URLs before import.
  - file: [src/plugins/loader.ts:419](https://github.com/openclaw/openclaw/blob/245451b6a9c529464e177afa65013adca0d63eb5/src/plugins/loader.ts#L419)
  - sha: [245451b6a9c5](https://github.com/openclaw/openclaw/commit/245451b6a9c529464e177afa65013adca0d63eb5)
- **Current loader uses normalized specifiers:** The loader applies the normalized import path at module-load call sites used for plugin runtime and plugin entry loading.
  - file: [src/plugins/loader.ts:1929](https://github.com/openclaw/openclaw/blob/245451b6a9c529464e177afa65013adca0d63eb5/src/plugins/loader.ts#L1929)
  - sha: [245451b6a9c5](https://github.com/openclaw/openclaw/commit/245451b6a9c529464e177afa65013adca0d63eb5)
- **Regression coverage on win32:** Current tests simulate `process.platform === "win32"` and assert `C:\Users\alice\plugin\index.mjs` becomes `file:///C:/Users/alice/plugin/index.mjs` instead of reaching the loader as a raw drive-letter path.
  - file: [src/plugins/loader.test.ts:6600](https://github.com/openclaw/openclaw/blob/245451b6a9c529464e177afa65013adca0d63eb5/src/plugins/loader.test.ts#L6600)
  - sha: [245451b6a9c5](https://github.com/openclaw/openclaw/commit/245451b6a9c529464e177afa65013adca0d63eb5)
