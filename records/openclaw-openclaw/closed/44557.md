## Summary

Current `main` already implements form-mode rendering for `accounts`-style channel config maps. The config-form analyzer now converts `additionalProperties: true` into an editable map schema, the renderer sends object maps through `renderMapField`, and there is a focused browser test covering an `accounts` object specifically.
