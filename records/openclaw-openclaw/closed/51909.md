## Summary

Issue is obsolete on current main: the bundled ACPX backend no longer shells out through the old external acpx CLI path, ACP spawn workspace handling was hardened, and later releases ship a much newer bundled acpx runtime than the reporter's 2026.3.13/0.1.16 setup.

## Evidence

- **Bundled ACPX now runs embedded runtime directly:** Current plugin service constructs `new AcpxRuntime(...)` directly instead of launching an external bundled acpx binary hop for the backend path described in the report.
  - file: [extensions/acpx/src/service.ts:43](https://github.com/openclaw/openclaw/blob/aa3a5f9ff7907c2a41d6259d24b529bda41f293a/extensions/acpx/src/service.ts#L43)
- **Release notes for embedded ACPX runtime change:** Changelog for 2026.4.5 says the bundled `acpx` plugin now embeds the ACP runtime directly and removes the extra external ACP CLI hop, matching the failing path in the issue.
  - file: [CHANGELOG.md:1209](https://github.com/openclaw/openclaw/blob/aa3a5f9ff7907c2a41d6259d24b529bda41f293a/CHANGELOG.md#L1209)
- **ACP spawn now inherits target agent workspace:** Current ACP spawn resolves runtime cwd from the target agent workspace and only falls back when the inherited path is missing, covering the cross-agent/runtime-cwd part called out by the linked PR and later docs.
  - file: [src/agents/acp-spawn.ts:1130](https://github.com/openclaw/openclaw/blob/aa3a5f9ff7907c2a41d6259d24b529bda41f293a/src/agents/acp-spawn.ts#L1130)
- **Regression test for cross-agent ACP workspace inheritance:** Test asserts cross-agent `sessions_spawn({ runtime: "acp" })` initializes the ACP session with the target agent workspace when `cwd` is omitted.
  - file: [src/agents/acp-spawn.test.ts:1161](https://github.com/openclaw/openclaw/blob/aa3a5f9ff7907c2a41d6259d24b529bda41f293a/src/agents/acp-spawn.test.ts#L1161)
- **Release notes for ACP workspace fix:** Changelog for 2026.4.5 explicitly records that ACP now inherits the target agent workspace for cross-agent ACP spawns and falls back safely when missing.
  - file: [CHANGELOG.md:1402](https://github.com/openclaw/openclaw/blob/aa3a5f9ff7907c2a41d6259d24b529bda41f293a/CHANGELOG.md#L1402)
- **Bundled ACPX version has moved far past reported runtime:** Current bundled plugin pins `acpx` 0.5.3, whereas the issue reported 0.1.16 on OpenClaw 2026.3.13.
  - file: [extensions/acpx/package.json:6](https://github.com/openclaw/openclaw/blob/aa3a5f9ff7907c2a41d6259d24b529bda41f293a/extensions/acpx/package.json#L6)
