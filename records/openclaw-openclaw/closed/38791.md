## Summary

Current main already defaults omitted delivery to `announce` for isolated `agentTurn` cron jobs, and the latest tagged release available in this checkout (`v2026.4.22`) contains the same implementation.

## Evidence

  - sha: [51186d272543](https://github.com/openclaw/openclaw/commit/51186d2725439646f7fea0e59bb466f957404c46)
