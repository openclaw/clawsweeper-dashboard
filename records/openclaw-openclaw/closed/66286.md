## Summary

Keep open. Current `main` still does not surface upstream request IDs through the diagnostics path this PR targets, and the only review feedback in the thread is a minor deduplication suggestion rather than a reason to close. This is not implemented on `main`, not obsolete, and not a cleanup-close candidate.
