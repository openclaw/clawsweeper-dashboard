## Summary

Keep open: this is useful approval-analysis cleanup with automerge enabled, but the latest head still has security-sensitive parser blockers around sudo carrier options and unbounded recursive carrier analysis.

## What This Changes

The PR moves command-carrier parsing into a shared infra helper, routes exec approval summaries/payload extraction/inline-eval checks through it, and adds focused tests plus a changelog entry.

## Best Possible Solution

Keep the shared parser direction, but merge only after it preserves command-executing sudo carrier forms and restores bounded carrier recursion with regression coverage.

## Reproduction Assessment

Yes, by source inspection. `sudo -h` shows `-r role` and `-t type` are command-executing value flags, while the PR-head parser omits them; the recursive inline-eval path also lacks the depth bound present on current main.

## Solution Assessment

No. Centralizing carrier parsing is the right direction, but the mergeable fix needs the missing sudo short options and a bounded or iterative recursive unwrap path before automerge.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Handle sudo SELinux short value flags:** `src/infra/command-carriers.ts:28-49`
  - body: `sudo -h` shows `-r role` and `-t type` are valid command-executing options, but this new value-option table only includes the long `--role`/`--type` forms. As a result, `sudo -r role python3 -c ...` is rejected by the carrier parser instead of being unwrapped, so strict inline-eval and `/approve` payload detection can miss the carried command.
  - confidence: 0.88
- **[P2] Restore a depth cap for carrier recursion:** `src/infra/command-analysis/risks.ts:99`
  - body: The previous inline-eval carrier path stopped after a fixed carrier depth, but this recursive path only checks for repeated argv shapes. Long acyclic chains such as many nested `env` prefixes still recurse once per wrapper and can overflow the JS stack before an approval decision is reached.
  - confidence: 0.9
