  - confidence: 0.88

  - body: This section presents `ops-request`/`fswatch`/`agent-io-router` as the only authorized host channel for sandboxed agents, but there is no matching OpenClaw implementation or documented API in the repo. Publishing this would make a private convention look like a supported security boundary.
