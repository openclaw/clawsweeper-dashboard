## Best Possible Solution

Add focused regression coverage for denied `sessions_yield` across embedded runs, gateway scoped tools, tool-search/catalog exposure, and invocation failure; if any path bypasses filtering, enforce the effective tool policy before callback execution or before catalog/dispatch exposure.
