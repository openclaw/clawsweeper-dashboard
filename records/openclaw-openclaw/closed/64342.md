## Summary

Current `main` already has the post-turn isolated review capability requested here. Plugins can observe completed turns with `agent_end`, launch a separate embedded agent/session through the public plugin runtime, and the bundled `skill-workshop` plugin uses that exact pattern for silent post-turn skill review. The shipped implementation is present in release `v2026.4.23`.

## Risks / Open Questions

- There is no issue discussion clarifying whether the reporter wanted a brand-new named `post_turn` hook specifically, versus equivalent existing capability through `agent_end` plus embedded-agent runtime.
- The shipped bundled implementation is for skill capture; memory review is still plugin-owned, so this decision assumes the request is satisfied by the existing generic framework seam rather than a new built-in memory reviewer.
