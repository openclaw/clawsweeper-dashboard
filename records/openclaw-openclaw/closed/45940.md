## Summary

Current `main` has related gateway-probe hardening, but the exact false-negative path from #45940 is not clearly fixed. `openclaw gateway probe` still routes through `probeGateway`, and that path still marks post-connect non-scope failures/timeouts as unreachable; the Windows/event-loop yield fix is present in `callGateway`, not in the probe path used by `gateway probe`. That is not strong enough evidence to close conservatively.

## Evidence

- **`gateway probe` still uses the gateway-status probe path:** The CLI `gateway probe` command dispatches to `gatewayStatusCommand`, not to the separate `callGateway` path.
  - file: [src/cli/gateway-cli/register.ts:621](https://github.com/openclaw/openclaw/blob/c1383680405879a3c3f43dceb20a694f54d3c13c/src/cli/gateway-cli/register.ts#L621)
  - command: `nl -ba src/cli/gateway-cli/register.ts | sed -n '612,626p'`
- **Gateway status/probe still calls `probeGateway`:** `gatewayStatusCommand` probes each target with `probeGateway(...)`, so the issue hinges on `src/gateway/probe.ts` behavior.
  - file: [src/commands/gateway-status/probe-run.ts:124](https://github.com/openclaw/openclaw/blob/c1383680405879a3c3f43dceb20a694f54d3c13c/src/commands/gateway-status/probe-run.ts#L124)
  - command: `nl -ba src/commands/gateway-status/probe-run.ts | sed -n '118,140p'`
- **Post-connect timeout still settles as `ok: false`:** After `onHelloOk`, the probe records `connectLatencyMs`, but its follow-up detail timer still settles the result as `{ ok: false, error: "timeout" }` rather than a reachable-but-degraded state.
  - file: [src/gateway/probe.ts:284](https://github.com/openclaw/openclaw/blob/c1383680405879a3c3f43dceb20a694f54d3c13c/src/gateway/probe.ts#L284)

  - file: [src/commands/gateway-status/helpers.ts:279](https://github.com/openclaw/openclaw/blob/c1383680405879a3c3f43dceb20a694f54d3c13c/src/commands/gateway-status/helpers.ts#L279)
  - command: `nl -ba src/commands/gateway-status/helpers.ts | sed -n '267,310p'`

  - file: [src/commands/gateway-status/helpers.test.ts:251](https://github.com/openclaw/openclaw/blob/c1383680405879a3c3f43dceb20a694f54d3c13c/src/commands/gateway-status/helpers.test.ts#L251)
