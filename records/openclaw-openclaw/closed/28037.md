## Summary

Current `main` contains Slack Socket Mode inbound-routing fixes that match this report: socket mode now uses `SocketModeReceiver`, inbound messages are registered on the generic `message` event instead of subscription labels like `message.channels`, `groupPolicy:"open"` no longer blocks unlisted channels when channel config exists, and regression tests/docs were added alongside the implementation.

## Evidence
