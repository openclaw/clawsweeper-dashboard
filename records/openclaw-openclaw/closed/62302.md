## Summary

The reported bundled-plugin startup path from OpenClaw 2026.4.5 is already addressed on current `main`, and the relevant Jiti loader fixes shipped by `v2026.4.23` with regression coverage for module-relative loader filenames and Windows native-Jiti behavior.

## Evidence
