## Summary

Current main already implements the issue's acceptable simpler alternative: a Control UI chat toggle that hides tool calls and tool results, persists that preference locally, and filters those transcript items out of the rendered chat. The changelog shows this shipped in release 2026.1.9 as 'hide tool cards.'
