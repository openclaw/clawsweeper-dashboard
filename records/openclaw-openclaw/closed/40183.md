## Summary

Current `main` already has the requested backend cleanup: agent-scoped media roots are centralized, outbound delivery now carries agent context into media access resolution, and local media source normalization handles file URLs / absolute paths / Windows-drive paths in one place. The issue reads as already implemented on `main`, but not yet in the latest tagged release available in this checkout.
