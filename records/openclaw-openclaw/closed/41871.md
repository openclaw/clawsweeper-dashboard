## Summary

Current `main` contains multiple Ollama-specific fixes that match the reported hang modes: embedded runs now raise Undici stream/body timeouts to the run timeout instead of failing around 60s, native Ollama `/api/chat` requests forward top-level `think` control, Ollama wire model ids strip the `ollama/` prefix, and compaction now caps its reserve floor against the actual context window to avoid small-context infinite compaction loops.
