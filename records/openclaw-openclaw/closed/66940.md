## Summary

Current main no longer has the reported missing Streamable HTTP POST Accept header problem. OpenClaw delegates streamable-http to the pinned MCP SDK, and the pinned SDK version sets `accept: application/json, text/event-stream` on POST requests. The same SDK pin is present in the v2026.4.24 release tag.
