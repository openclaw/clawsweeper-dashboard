## Summary

Close as `clawhub`: current `main` already provides plugin hooks and fail-closed tool-call interception for governance plugins, and the issue itself links an existing `sift-openclaw-plugin`, so this belongs in the plugin/ClawHub ecosystem rather than OpenClaw core.
