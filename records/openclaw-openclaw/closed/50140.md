## Summary

Current `main` no longer ships or uses the legacy Chrome extension relay path that this stale-token bug targeted. The relay entrypoint is now an intentional no-op, doctor migrates legacy `driver: "extension"` / `browser.relayBindHost` configs to Chrome MCP attach, and the changelog records that removal in the 2026.3.22 breaking release.
