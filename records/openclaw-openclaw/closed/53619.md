## Summary

Current main already provides the requested Control UI tool-output control well enough: the chat toolbar has a persisted tool calls/results toggle, hidden mode suppresses tool result output, and visible mode renders tool output behind collapsed disclosure rows by default. The same surface is present in v2026.4.24. The remaining default-expanded preference is narrower and already tracked by #54478/#54838.

## Best Possible Solution

Keep the shipped Control UI toolbar toggle, persisted chatShowToolCalls setting, and collapsed disclosure rendering. Leave any separate default-expanded preference to #54478/#54838 instead of keeping this broader request open.
