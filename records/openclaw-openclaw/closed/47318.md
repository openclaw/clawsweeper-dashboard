  - body: This checks `operation.value` with a direct `parseModelRef` before `normalizeConfigMutationModelRefs(next as OpenClawConfig)`, so valid values that are accepted after config normalization or runtime resolution can still warn as missing.   - confidence: 0.86
- **[P2] Sanitize model refs before printing warnings:** `src/cli/config-cli.ts:793-794`
  - body: Both warning messages interpolate config-derived model text directly into terminal output.   - confidence: 0.88
