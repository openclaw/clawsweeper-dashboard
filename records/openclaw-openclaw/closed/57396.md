## Summary

Current `main` already carries tool-result media into outbound replies and now threads agent/requester delivery context into scoped local-media access, which addresses the reported auto-delivery miss for allowlist-based channels. The implementation is present on `main` but does not appear in a tagged release yet.

## Evidence
