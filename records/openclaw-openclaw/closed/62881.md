## Summary

Current `main` already routes the Windows gateway Codex app-server startup through wrapper resolution that avoids the raw `.cmd` spawn path behind the Node.js v24 DEP0190 warning, and it adds a regression test for that case.

## Evidence

- **Gateway startup now resolves Codex Windows shims before spawning:** `CodexAppServerClient.start()` still uses stdio transport for gateway startup, but `createStdioTransport()` now calls `resolveCodexAppServerSpawnInvocation()` first. That resolver runs `resolveWindowsSpawnProgram()` with `packageName: "@openai/codex"` and passes the resolved command/argv into `spawn()`, instead of spawning `options.command` directly.
  - file: [extensions/codex/src/app-server/client.ts:91](https://github.com/openclaw/openclaw/blob/245451b6a9c529464e177afa65013adca0d63eb5/extensions/codex/src/app-server/client.ts#L91)
  - command: `nl -ba extensions/codex/src/app-server/client.ts | sed -n '91,101p' && nl -ba extensions/codex/src/app-server/transport-stdio.ts | sed -n '21,60p'`
  - sha: [245451b6a9c5](https://github.com/openclaw/openclaw/commit/245451b6a9c529464e177afa65013adca0d63eb5)
- **Windows wrapper resolver avoids shell fallback by default:** On Windows, `.cmd`/`.bat` commands are now inspected and converted to either a direct `.exe` entrypoint or `node <entrypoint>` invocation. `shell: true` only appears in the explicit `allowShellFallback` escape hatch; the Codex app-server path does not opt into that fallback.
  - file: [src/plugin-sdk/windows-spawn.ts:229](https://github.com/openclaw/openclaw/blob/245451b6a9c529464e177afa65013adca0d63eb5/src/plugin-sdk/windows-spawn.ts#L229)

  - sha: [245451b6a9c5](https://github.com/openclaw/openclaw/commit/245451b6a9c529464e177afa65013adca0d63eb5)
- **Regression test covers the reported Windows npm shim case:** `transport-stdio.test.ts` creates a `codex.cmd` shim plus `@openai/codex/bin/codex.js`, resolves the spawn invocation on `win32`, and asserts the result is `node.exe <entryPath> ...` with `shell: undefined`. That is the DEP0190-avoiding behavior the issue requested.
