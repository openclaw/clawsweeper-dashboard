## Summary

Current `main` already supersedes this PR with a broader subagent-completion delivery fix: completions now fall back to the external requester route, while Telegram topic conversations already resolve back to real outbound `{to, threadId}` targets. The original topic-binding mismatch described in the PR discussion is therefore covered well enough on `main` without landing this patch as-is.
