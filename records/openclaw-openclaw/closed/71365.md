## Summary

Keep open. Current `main` does not contain a repo-level `.editorconfig`, so the PR's stated change is not already implemented. Existing formatting controls are split across `.gitattributes`, `oxfmt`, pre-commit hooks, and language-specific tooling, which is not equivalent to editor-facing defaults. The bot review also identified a real completeness gap for Python and Go files, so there is not a strong conservative basis to auto-close this PR.
