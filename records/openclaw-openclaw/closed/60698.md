- **[P1] Keep explicit extra params outside provider defaults:** `src/agents/pi-embedded-runner/extra-params.ts:510`
  - body: This moves `applyPrePluginStreamWrappers` before `wrapProviderStreamFn`, reversing the current precedence contract.   - confidence: 0.86
- **[P1] Sanitize Bedrock metadata before sending it:** `src/agents/pi-embedded-runner/extra-params.ts:282-285`
  - body: `agentId` and especially `agentName` come from local config, but AWS `requestMetadata` values only allow a restricted character set and are used to filter invocation logs. Names containing characters like emoji or apostrophes can fail every Bedrock stream call for that agent, and display names may expose sensitive labels in AWS logs.
  - confidence: 0.84
