## Summary

Current `main` already routes Slack DMs through the shared agent/account router instead of hard-coding them into `agent:main:main`, and then stores the turn under the routed agent/session. The implementation is present on `main` in commit `6f04eee2a12e665fb4b0e3b0021dbbf90efdf2a2`, but I could not verify a released tag that contains it.
