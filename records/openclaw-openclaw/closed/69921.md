## Summary

The requested post-turn persistence/enforcement mechanism belongs on the existing plugin surface, not in OpenClaw core. Current main already exposes post-turn/runtime hook points (`agent_end` and context-engine `afterTurn`), and bundled plugins use them for turn-end persistence.
