## Summary

Current `main` already carries the isolated-cron Telegram delivery fix. The cron isolated-agent path now passes the job’s explicit delivery target into delivery resolution, preserves/normalizes that target for outbound send, and has a regression test covering Telegram target writeback during manual `cron.run`.
