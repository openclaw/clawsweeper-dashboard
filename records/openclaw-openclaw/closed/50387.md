## Summary

This is a useful task-routing policy, but it belongs as a ClawHub/code-plugin capability rather than OpenClaw core. Current main already provides the generic pre-model-resolution hook needed for a plugin to classify the current prompt and return a provider/model override, and the project vision explicitly keeps optional policy features out of core when plugin surfaces are sufficient.
