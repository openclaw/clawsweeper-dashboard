## Summary

Close #49073 as a duplicate of the earlier open tracker #42986. Current main still does not implement native Telegram thread-bound subagent session spawning; #42986 already tracks that same remaining Telegram `sessions_spawn(mode="session", thread=true)` forum-topic binding gap. The implemented ACP/ACPX Telegram topic work from related #47673 is adjacent but does not satisfy this native subagent request.
