## Summary

Current `main` already implements the requested `/usage` behavior for reset/deleted transcripts, ignores `.bak` artifacts, and includes archived-only fallback for per-session detail queries. The changelog shows this shipped in `v2026.3.22` via PR `#43215`, so this open PR is duplicate work.
