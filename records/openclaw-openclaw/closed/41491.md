## Summary

Current main and the shipped v2026.4.22 release already contain the WhatsApp/Groq STT path for parameterized WhatsApp audio MIME (`audio/ogg; codecs=opus`) plus explicit regression coverage for this exact scenario, so the reported behavior is implemented on main.
