## Summary

Current main already implements the reported fix: Twilio conversation-mode greetings are deferred until media-stream connect, the initial message is only cleared after successful playback, and TwiML fallback is blocked once a stream is active. Regression tests in the voice-call plugin cover the reported race and retry behavior.

## Evidence
