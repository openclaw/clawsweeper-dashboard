## Summary

Current `main` resolves cron `job.agentId` into agent-specific defaults, applies per-job `payload.model` ahead of stored session overrides/defaults, and has targeted regression tests that cover both the selection and forwarding paths. The relevant cron code is unchanged between `v2026.4.23` and current `main`, so this is already shipped.
