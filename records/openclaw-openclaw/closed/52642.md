## What This Changes

The branch changes embedded Pi reasoning stream gating, replaces the agent command barrel with a copied command implementation that forwards stored reasoning, and adds scheduled-reset/session-memory metadata handling with tests.

## Best Possible Solution

Close this broad branch as superseded, land the focused gateway reasoning work through https://github.com/openclaw/openclaw/pull/79687 and UI follow-up through https://github.com/openclaw/openclaw/pull/59086, and keep reset-hook work on https://github.com/openclaw/openclaw/pull/61675.

## Reproduction Assessment

Yes, source-level. Current main callback-gates `streamReasoning` on `onReasoningStream`, while Gateway `chat.send` does not pass that callback, so callback-less WebSocket clients cannot receive live thinking events from this path.

## Solution Assessment

No. The branch addresses real symptoms, but it combines unrelated fixes and introduces regressions; the narrower maintainable solution is the focused subscriber-boundary PR plus separate reset-hook work.

- **[P1] Import reset hooks before scheduled reset fallback:** `src/auto-reply/reply/get-reply.ts:341`
  - body: When `scheduledResetTriggered` is true, this branch calls `emitResetCommandHooks()` before importing it, so the first scheduled rollover can throw instead of flushing hooks.
  - confidence: 0.96
- **[P1] Remove duplicate transcript helper declarations:** `src/hooks/bundled/session-memory/handler.ts:54`

  - confidence: 0.95
- **[P1] Preserve per-run provider and model overrides:** `src/commands/agent.ts:936-942`
  - body: The copied command implementation seeds provider/model only from defaults and stored session overrides, never applying authorized `opts.provider` or `opts.model`, so one-shot gateway and local override runs can silently use the wrong model.
  - confidence: 0.91
- **[P2] Reinstate the ingress model-override guard:** `src/commands/agent.ts:1246-1254`
  - body: `agentCommandFromIngress()` now validates only `senderIsOwner` and no longer requires `allowModelOverride`, weakening the network ingress trust-boundary contract for untyped callers.
  - confidence: 0.9
- **[P2] Preserve reasoning visibility gates when streaming:** `src/agents/pi-embedded-subscribe.ts:52`
  - body: Setting `streamReasoning` to only `reasoningMode === "stream"` drops the current `canShowReasoning` guard, so a rebased branch could broadcast thinking content when visibility is disabled.
  - confidence: 0.84
- **[P2] Keep scheduled reset hooks behind authorization:** `src/auto-reply/reply/get-reply.ts:340-350`
  - body: The scheduled-reset branch fires reset hooks before checking `command.isAuthorizedSender`, so an unauthorized post-expiry message in a shared session could trigger reset-hook automation.
  - confidence: 0.84
