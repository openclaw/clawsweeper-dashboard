## Summary

Current OpenClaw code and shipped release `v2026.4.22` already route Docker gateway startup through `--bind ${OPENCLAW_GATEWAY_BIND:-lan}` and resolve `gateway.bind=lan` to `0.0.0.0`, with regression tests covering container and explicit `lan` cases.

## Evidence
