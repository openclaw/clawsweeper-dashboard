## Summary

Current main already contains the routing and Feishu card-path changes that address the reported cron-only Feishu failures: main-session cron wakes now force heartbeat delivery to the last active channel/target, missing message-tool targets are inferred from current channel context, and Feishu card payloads are built in schema 2.0 with sanitized templates.
