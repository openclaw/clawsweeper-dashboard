## Summary

Current `main` already implements the substance of this request via route-first CLI execution, per-command plugin-load policies, existing `--json` machine output, and gateway handshake paths that no longer wait behind heavy startup. There is not a separate `--agent` / `OPENCLAW_AGENT_MODE` alias, but the cited programmatic use cases are already covered.
