## Summary

Issue #42412 requests optional skill usage analytics. Current main does not implement a core `openclaw skills usage` command, but the requested tracking/reporting can live as a ClawHub/plugin capability using existing hook, CLI, gateway, and skill-command surfaces, and VISION.md explicitly keeps optional capabilities out of core when plugin APIs are sufficient.
