## Summary

Keep this PR open. Current main still has the blanket same-target messaging-tool suppression that drops final replies regardless of whether the tool send contained text, so the PR is not obsolete. The PR is still an implementation candidate for #59743, but its latest review context shows a remaining functional issue to fix before landing. Security review pass found no supply-chain-sensitive file changes in the provided PR file list.

## Best Possible Solution

Keep the PR open or consolidate it with the other open implementation candidates for #59743. The landed fix should tie the text/media distinction to the matching origin target, include caption and whitespace handling, preserve same-target media dedupe, and add regression coverage for media-only origin sends plus mixed-origin-media/cross-target-text sends in the reply payload and follow-up delivery paths.
