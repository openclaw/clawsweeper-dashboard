## Summary

Current `main` already fixes the reported FTS-only (`provider=none`) memory regression. The memory manager now writes `fts-only` chunks even with no embedding provider, keyword search stops filtering out FTS-only rows when `providerModel` is absent, and focused tests cover both indexing/search and forced reindex behavior in no-provider mode.
