## Summary

Current main already contains the requests-in-flight heartbeat retry/scheduling fix and regression coverage for the silent-skip/drift path described in #40113.

## Evidence

- **Wake layer preserves retry cooldown and requeues busy heartbeats:** `src/infra/heartbeat-wake.ts:130` keeps an existing `retry` timer as a hard minimum delay, and `src/infra/heartbeat-wake.ts:178` requeues `requests-in-flight` wakes then schedules `DEFAULT_RETRY_MS` instead of dropping them.
  - file: [src/infra/heartbeat-wake.ts:130](https://github.com/openclaw/openclaw/blob/bfa6708c03f24c7caa5d0a96a298c292be0d85c1/src/infra/heartbeat-wake.ts#L130)
- **Runner no longer overrides retry with immediate reschedule:** `src/infra/heartbeat-runner.ts:1479` tracks `requestsInFlight`, and `src/infra/heartbeat-runner.ts:1533` returns early on that skip reason without advancing cadence; `src/infra/heartbeat-runner.ts:1554` skips `scheduleNext()` in that case so the wake-layer retry owns recovery.
  - file: [src/infra/heartbeat-runner.ts:1479](https://github.com/openclaw/openclaw/blob/bfa6708c03f24c7caa5d0a96a298c292be0d85c1/src/infra/heartbeat-runner.ts#L1479)
- **Regression tests cover the reported failure mode:** `src/infra/heartbeat-wake.test.ts:71` asserts a `requests-in-flight` heartbeat retries after the default delay, and `src/infra/heartbeat-runner.scheduler.test.ts:214` asserts the runner reschedules correctly and does not drift under repeated busy skips.
  - file: [src/infra/heartbeat-runner.scheduler.test.ts:214](https://github.com/openclaw/openclaw/blob/bfa6708c03f24c7caa5d0a96a298c292be0d85c1/src/infra/heartbeat-runner.scheduler.test.ts#L214)
- **Shipped changelog entry matches this fix:** `CHANGELOG.md:3205` documents the heartbeat/requests-in-flight scheduling fix: stop advancing `nextDueMs` and avoid immediate `scheduleNext()` overrides so retry cooldowns are honored and cadence does not drift.
  - file: [CHANGELOG.md:3205](https://github.com/openclaw/openclaw/blob/bfa6708c03f24c7caa5d0a96a298c292be0d85c1/CHANGELOG.md#L3205)
