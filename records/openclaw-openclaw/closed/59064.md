## Summary

Current `main` already contains targeted fixes and regression coverage for the reported 429 fallback/live-session race, and it now emits user-facing lifecycle error text for rate-limit failures instead of going silent. Later comments broaden the thread into separate approval-migration and stale-PATH topics, but the original item's core fallback/notification behavior is implemented on `main`.
