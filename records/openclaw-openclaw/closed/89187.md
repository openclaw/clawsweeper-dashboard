Use one safe tool-name collection contract for session allowlists and prompt-cache diagnostics so malformed descriptors cannot crash either path.

No. The non-string prompt-cache branch is repaired, but the unreadable-name branch should be fixed at the shared tool-name collection boundary because the session allowlist reads the same descriptors first.
