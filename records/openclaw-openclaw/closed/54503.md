## Summary

Current main already handles provider-qualified slash-containing model refs in the Control UI and gateway session model plumbing. The picker normalizes nested model IDs to fully qualified refs, sends that ref through `sessions.patch`, and gateway/session resolution preserves nested model IDs under the selected provider. The same implementation is already present in release `v2026.4.22`.

## Evidence
