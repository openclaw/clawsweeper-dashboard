## Summary

Current main and the shipped changelog cover the reported silent-drop path: Control UI still sends image attachments through `chat.send`, Gateway detects text-only session models and offloads those images as `media://inbound/*` refs instead of discarding them, and media understanding now prefers configured `agents.defaults.imageModel` entries before the active text model. Existing regression tests cover the text-only WebChat/Gateway path.
