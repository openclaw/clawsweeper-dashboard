## Summary

Close as `clawhub`: current `main` intentionally keeps optional capability out of core when plugins can express it, and the plugin SDK already supports plugin-owned gateway RPC methods. Detailed remote host telemetry fits that existing plugin/ClawHub extension path better than a new core `system.metrics` method.
