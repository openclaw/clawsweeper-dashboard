## Summary

Telegram forum-topic binding routing is implemented on current `main` and shipped in `v2026.4.22`. The current code canonicalizes topic conversations as `-100…:topic:<threadId>`, routes inbound forum messages with that peer id plus the parent group fallback, and has regression tests covering both parent-group and topic-specific bindings.

## Evidence
