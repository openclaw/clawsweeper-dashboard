## Summary

Keep open. Current `main` still starts Bonjour advertising regardless of `gateway.bind`, and the bundled advertiser watchdog on `main` still re-advertises/recreates services when they remain in non-`announced` states. The code and tests show the flapping/restart behavior is still present as an active recovery path, not an obsolete report or an implemented fix.
