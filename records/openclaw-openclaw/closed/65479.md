## Summary

Close as duplicate/superseded by open #63101. Current main still has the same Feishu legacy-config validation gap, and #63101 is the broader canonical tracker for strict Feishu schema failures after upgrade, including `threadSession` and other stale Feishu keys. The `footer` portion overlaps with #56882/current Feishu behavior: current schema/docs do not support a Feishu footer config, so any no-op removal or migration belongs with the broader Feishu cleanup in #63101.
