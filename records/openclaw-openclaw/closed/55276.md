## Summary

Current main contains a targeted fix for embedded-run prompt/replay corruption that removes orphaned trailing user leaves, merges their text back into the next prompt in a controlled way, and adds regression coverage for structured/heartbeat cases. That matches the reported symptom class of prior content leaking into later file-tool path generation.
