## Summary

#71737 should close as superseded by open PR #72082, which is the focused implementation tracker for the same generic plugin host-hook contracts. Current main has adjacent raw event/cron/prompt hook surfaces, but not the requested typed lifecycle-owned SDK contract, so this is not implemented-on-main.

## Best Possible Solution
