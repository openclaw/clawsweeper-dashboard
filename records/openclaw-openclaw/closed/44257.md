## Summary

Current main already contains the isolated-cron deadlock fix and coverage for the exact visible symptom: manual `cron.run` can return `{ ok: true, enqueued: true, runId }`, the queued run completes, and `cron runs` shows the finished entry. The changelog ships the underlying fix in `v2026.3.13` as a cron isolated-session deadlock fix.
