## Summary

Current main implements the intended Windows exec path by invoking PowerShell directly and passing the user command as a single -Command argument. The latest failing example is a nested powershell -Command wrapper, which current tool guidance explicitly tells agents not to use; a direct PowerShell pipeline with $_ and -f works under the current invocation shape.
