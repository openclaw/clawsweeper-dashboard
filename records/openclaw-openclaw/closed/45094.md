## Summary

Issue #45094 is proposing a third-party enhanced LanceDB memory plugin, and current OpenClaw scope explicitly routes optional plugin work like this to external plugins published on ClawHub/npm rather than adding or tracking it in core.

## Evidence

- **Scope anchor in vision:** Project vision says optional capability should usually ship as plugins, plugin authors should host and maintain plugins in their own repository, and the bar for adding optional plugins to core is intentionally high.
