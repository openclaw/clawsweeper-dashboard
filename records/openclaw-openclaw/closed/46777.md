## Summary

Current `main` already implements the cron fallback behavior this PR is trying to add: isolated cron runs compute an explicit fallback override for cron model/payload cases, pass it into `runWithModelFallback`, and ship regression coverage plus docs for the preserved default fallback chain behavior.
