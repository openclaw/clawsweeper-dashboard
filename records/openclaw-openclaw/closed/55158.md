## Summary

Current main does not show a core bug here: default per-user location belongs in injected `USER.md`, and custom weather-skill instructions belong in workspace `skills/weather/SKILL.md`, which already overrides the bundled skill. The bundled weather skill still uses `London` only in example commands, which explains the reported confusion.

## Evidence

- **`USER.md` is the supported per-user context surface:** The agent docs define `USER.md` as the user profile and preferred-address file, and say its contents are injected directly into a new session's context.
  - file: [docs/concepts/agent.md:34](https://github.com/openclaw/openclaw/blob/34896839ba22f3a71a3d5a8fb4ca5f4c6989a534/docs/concepts/agent.md#L34)
  - sha: [34896839ba22](https://github.com/openclaw/openclaw/commit/34896839ba22f3a71a3d5a8fb4ca5f4c6989a534)
- **Workspace skills already override bundled skills:** The docs list skill precedence with `<workspace>/skills` first, ahead of bundled skills.
  - file: [docs/concepts/agent.md:59](https://github.com/openclaw/openclaw/blob/34896839ba22f3a71a3d5a8fb4ca5f4c6989a534/docs/concepts/agent.md#L59)
  - sha: [34896839ba22](https://github.com/openclaw/openclaw/commit/34896839ba22f3a71a3d5a8fb4ca5f4c6989a534)
- **Code matches the documented precedence:** `loadWorkspaceSkillEntries()` loads bundled, managed, project, and workspace skills, then merges them with workspace last; the comment explicitly states workspace has highest precedence.
  - file: [src/agents/skills/workspace.ts:534](https://github.com/openclaw/openclaw/blob/34896839ba22f3a71a3d5a8fb4ca5f4c6989a534/src/agents/skills/workspace.ts#L534)
  - sha: [34896839ba22](https://github.com/openclaw/openclaw/commit/34896839ba22f3a71a3d5a8fb4ca5f4c6989a534)
- **Bundled weather skill uses London as sample text, not persisted config:** The bundled weather skill says to include a city/region/airport code, then shows `wttr.in/London` in example commands and quick responses. That is example content, not a separate default-location setting.
  - file: [skills/weather/SKILL.md:49](https://github.com/openclaw/openclaw/blob/34896839ba22f3a71a3d5a8fb4ca5f4c6989a534/skills/weather/SKILL.md#L49)
  - sha: [34896839ba22](https://github.com/openclaw/openclaw/commit/34896839ba22f3a71a3d5a8fb4ca5f4c6989a534)

## Risks / Open Questions
