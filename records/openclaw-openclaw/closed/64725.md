## Summary

Current `main` already supports per-agent hybrid memory-search tuning for vector/text weights, MMR lambda, and temporal decay, and the same surface is present in the shipped `v2026.4.23` release. The issue’s premise that these values are fixed in core no longer matches current behavior.

## Evidence
