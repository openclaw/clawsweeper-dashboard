## Summary

Current main implements the reported fix for new claude-cli live sessions: the live-session argv builder now pairs `--input-format stream-json` with `--output-format stream-json`, and regression tests cover fresh live-session argv even when backend args omit the output-format flag. The fix is on main but not in a tagged release I could identify.

## Best Possible Solution

Keep the mainline fix from b81eaf8a4e89bc13ab3065a28ac5eca1d905cafd, ship it in the next release, and leave the separate doctor/downgrade behavior tracked by #72715 rather than reopening this Claude CLI argv bug.

## Evidence
