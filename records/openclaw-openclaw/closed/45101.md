## Summary

Current `main` has no bundled DashScope/Qwen speech provider or `timbre` TTS surface, so this report does not map to an implemented OpenClaw feature and is not actionable as a current OpenClaw bug.

  - file: [extensions/qwen/openclaw.plugin.json:5](https://github.com/openclaw/openclaw/blob/4630ce3d9e2f089062ccd386b0ae241325961d1a/extensions/qwen/openclaw.plugin.json#L5)
  - sha: [4630ce3d9e2f](https://github.com/openclaw/openclaw/commit/4630ce3d9e2f089062ccd386b0ae241325961d1a)

  - file: [docs/providers/qwen.md:227](https://github.com/openclaw/openclaw/blob/4630ce3d9e2f089062ccd386b0ae241325961d1a/docs/providers/qwen.md#L227)
