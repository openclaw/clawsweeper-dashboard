## Summary

Keep open. Current `main` does not contain the PR's docs boundary work: `docs/logging.md` documents diagnostics export basics, but it does not define diagnostics as the telemetry ownership surface, does not include the hook-vs-diagnostics decision table described in the PR, and its event catalog is still incomplete relative to current diagnostic event types on `main`.
