## Summary

Current `main` no longer has an unconditional cron/plugin bootstrap path: the `cron` sub-CLI is lazy-loaded, gateway RPC runtime is imported only on first use, and there is now a regression contract that guards hot runtime/config imports from triggering plugin-registry work. The latest shipped tag `v2026.4.23` contains the same implementation.

## Evidence
