## Summary

Current main already maps Ollama `thinking`/`reasoning` payloads into structured thinking blocks, and the Control UI only shows those blocks when session reasoning visibility is enabled. The report does not include `/reasoning on|stream`, so it no longer matches current documented behavior closely enough to keep open as an active bug.

## Evidence
