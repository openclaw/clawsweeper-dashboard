## Summary

Current `main` no longer applies the exec script-preflight rejection layer in default host YOLO mode (`host=gateway`, `security=full`, `ask=off`), so the reported compound-shell rejection is implemented away on `main` even though it does not appear to be in the latest tagged release yet.

## Evidence
