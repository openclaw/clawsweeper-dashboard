## Summary

Keep open. Current `main` has already absorbed the handshake-timeout/startup side of this PR through other changes, but it does not implement the PR’s second behavior change of granting `operator.read` to device-less token/password auth. Current tests and docs still assert the narrower scope-stripping behavior, so closing this PR as implemented would be inaccurate.
