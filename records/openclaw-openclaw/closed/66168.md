  - confidence: 0.91

  - body: `McpLoopbackToolCache.resolve` syncs resources from every active cache entry into one global registry. Because `mcp.resources.list/read` serve that registry without filtering by the request's session/account context, one client can enumerate or read resources declared by another session's tools, including file-backed resources. Scope the registry or filter reads to the caller's resolved tool set.
