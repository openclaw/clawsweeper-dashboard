## Summary

Current `main` already routes embedded and subagent runs through the parent agent’s effective model. The subagent selection code now prefers `agents.list[].model` before global subagent/default settings, the spawn path persists that resolved model onto child sessions, and embedded runs resolve their default model from the session agent id. Regression tests and docs both describe the inherited-caller behavior. The proving commit is `972c7112a292f2776b8008b8c0e311e64569236f`, and it is not contained in any local tag, so no shipped release could be confirmed.
