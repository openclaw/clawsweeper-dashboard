## Summary

The reported paths do not match OpenClaw's published package layout. The reported npm version `openclaw@2026.3.23-2` already shipped workspace templates under `docs/reference/templates/`, and current/main code resolves templates from the package root plus root `package.json`, not from `dist/docs/...` or `dist/package.json`. Current release validation also explicitly requires those template files in the npm pack.
