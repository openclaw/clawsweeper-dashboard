## Summary

Current `main` no longer has the TDZ-prone `ANTHROPIC_MODEL_ALIASES` startup path. Anthropic model normalization is now a pure inline normalization path, `applyContextPruningDefaults()` no longer reparses configured model refs during gateway config load, and regression tests cover Anthropic alias normalization plus canonical allowlist insertion.
