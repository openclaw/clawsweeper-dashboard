## Summary

Keep open. Current `main` has not implemented this PR’s full removal of browser `evaluate`: the action is still part of the browser act contract, CLI, and docs, and the server only blocks it when `browser.evaluateEnabled=false`. The repository’s current security/docs posture treats browser evaluate as an intentional trusted-operator capability when enabled, so this PR is not proven obsolete or already shipped.
