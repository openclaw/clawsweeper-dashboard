## Summary

Current `main` fixes the reported dreaming-cron behavior by running managed dreaming in an isolated session, updating stale managed jobs toward that shape, and providing a doctor migration for persisted legacy jobs. The latest tagged release `v2026.4.23` predates that change, so this is implemented on `main` but not yet confirmed in a shipped release.
