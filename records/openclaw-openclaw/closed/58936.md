## Summary

The reported `wecommcp` SmartSheet persistence bug is in the externally maintained WeCom plugin surface, not in OpenClaw core on `main`. Current OpenClaw only catalogs and installs WeCom as an external community plugin, and project scope explicitly keeps optional channel behavior in plugins/ClawHub rather than core.
