  - body: This reads display text from the original tool-call arguments, but the send path sanitizes and normalizes the delivered `sourceReply` payload before it is safe to display.   - confidence: 0.9

  - body: The fallback treats `channel` as a harmless hint, while current `message-action-runner` treats `channel` as an explicit route before using the internal source-reply sink.
