## Summary

Issue #44904 requests new functionality for the external `@wecom/wecom-openclaw-plugin` plugin, not OpenClaw core. Current main scope/docs direct optional plugin work to community-owned plugins hosted and maintained outside this repo, so this should be pursued in the WeCom plugin's own repository/ClawHub path instead of `openclaw/openclaw`.
