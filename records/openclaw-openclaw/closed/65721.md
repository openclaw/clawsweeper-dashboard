## Summary

Current `main` already addresses the reported failure mode. Invalid non-plugin-local config now auto-recovers from last-known-good on startup and reload, and OpenClaw-owned config writes that would clobber the file are rejected before persistence. This behavior is documented, tested, and shipped in `v2026.4.23`; broader hardening ideas from the issue remain feature work rather than an open crash bug.
