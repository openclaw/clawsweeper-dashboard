## Summary

Close as implemented: current main and v2026.6.6 no longer have the old bundled skill-workshop plugin/reviewer path; stale skill-workshop plugin config now warns and returns, while Skill Workshop runs as a built-in tool.

## What This Changes

The issue asks for Skill Workshop to avoid crashing when the old skill-workshop plugin reviewer receives Matrix room IDs as channel context.

## Best Possible Solution

Keep the shipped built-in Skill Workshop path and removed-plugin warning; users on older releases should upgrade and remove stale plugins.entries.skill-workshop config.

## Reproduction Assessment

No high-confidence current-main failure path remains: the old bundled skill-workshop plugin directory is absent, stale plugin config is ignored with a warning, and the current built-in tool has no embedded reviewer path. The original 2026.5.x Matrix reproduction was credible but is now obsolete.

## Solution Assessment

Yes. Retiring the old plugin reviewer and keeping Skill Workshop as a built-in guarded tool is cleaner than relaxing bundled plugin dirName validation or adding a Matrix-specific runtime shim.
