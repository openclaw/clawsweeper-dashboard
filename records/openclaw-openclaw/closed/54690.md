## Summary

Keep open. Current `main` still has the failure shape: the parent relay only emits completion when it sees ACP lifecycle `end`/`error`, and oneshot ACP cleanup still skips the post-turn runtime-status reconcile that would resolve terminal ACP identity/session metadata before close. The repo docs also still list this as a known ACP stall symptom rather than a shipped fix.
