## Summary

Current `main` no longer has the reported Anthropic alias TDZ startup path. Config/model normalization now uses a static helper in `src/agents/model-ref-shared.ts`, `parseModelRef` routes through that helper before optional runtime normalization, and the changelog records the startup fix as shipped in `v2026.4.22`.

## Evidence
