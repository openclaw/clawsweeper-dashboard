## Summary

Current `main` already addresses the reported workspace-path bootstrap issue: the embedded runner now passes the effective workspace as the tool workspace, resolves bootstrap state from the canonical workspace, remaps injected bootstrap file paths onto the active workspace, and the system prompt explicitly states the working directory and path-handling guidance.
