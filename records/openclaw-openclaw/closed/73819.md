## Summary

Current main fixes the reported v2026.4.26 memory-core/chokidar runtime-deps failure by detecting root dist chunks that import plugin-owned deps, staging `chokidar@^5.0.0` into the package-level plugin runtime deps root before gateway plugin loading, and covering the memory-core root-dist import case in tests. The fix is on main only; it is not in the latest release tag `v2026.4.26`.

## Best Possible Solution

Keep the current main implementation and close this report as fixed on main. The remaining maintainer action is release communication or shipping the next release so affected `v2026.4.26` npm users can upgrade.
