## Summary

Current `main` now carries native subagent workspace inheritance through the spawn path, routes cross-agent spawns to the target agent workspace, preserves same-agent inherited workspace, and documents that `cwd` is ACP-only while native subagents only inject `AGENTS.md` + `TOOLS.md`. That matches the reported gap closely enough to treat the issue as implemented on `main`.
