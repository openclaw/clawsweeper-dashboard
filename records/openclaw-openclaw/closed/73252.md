## Summary

## What This Changes

Yes. The original bug is source-reproducible from the pre-fix path described in the PR and prior review: Mattermost previously had a strict schema without `streaming` and the inbound monitor created draft previews unconditionally; current main now contains the corrected schema and runtime gating.
