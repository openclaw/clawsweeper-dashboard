## Summary

The requested pre-turn keyword model routing now fits the supported plugin path on current `main`: OpenClaw exposes a `before_model_resolve` hook that can inspect the prompt before session/model resolution and return a provider/model override, and the project vision says optional capabilities that can be expressed as plugins should ship through ClawHub rather than as new core config.
