## Summary

Current `main` already strips assistant-side `<relevant-memories>` scaffolding from webchat-visible text, with regression coverage matching the reported leak pattern. The fix appears on `main` after the latest tagged release, so the report is valid for older builds but can be closed as already implemented upstream.
