## Summary

Current `main` routes both manual and watch-triggered memory indexing through the same `MemoryIndexManager.sync(...)` path, and the guarded remote-fetch stack used by memory embeddings now forces HTTP/1.1 (`allowH2: false`) to avoid the Undici 8 `TypeError: fetch failed` regression documented as fixed in the `2026.4.7` changelog.

## Evidence
