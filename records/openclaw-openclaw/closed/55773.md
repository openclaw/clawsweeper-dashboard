## Summary

Keep open. Current `main` does not implement this PR: the bundled browser plugin still registers `browser.request` with `scope: "operator.write"`, the gateway method-scope tests still expect `browser.request` to resolve to `operator.write`, and gateway authorization allows write-scoped callers to invoke methods registered at that scope. The security/docs context still supports the PR’s rationale because browser control is documented as sensitive and effectively operator-level access.
