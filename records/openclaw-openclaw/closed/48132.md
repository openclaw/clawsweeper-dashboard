## Summary

Keep open. Current `main` still does not support downloads for existing-session / Chrome MCP browser connections, so this PR is not already implemented or obsolete. The repo also still documents that limitation, and the proposed patch as described in the PR context has unresolved blocking issues.

## Evidence

That means current main still treats CDP/existing-session downloads as unsupported.
