## Summary

Keep open. Current `main` does not implement this WhatsApp fix: tool lifecycle payloads are still unconditionally suppressed, and the current regression test still asserts that all tool payloads are dropped. The PR discussion also surfaced a real follow-up gap in the proposed patch: tool-only media turns would still be reported as silent because `didQueueVisibleReply` ignores `counts.tool`.
