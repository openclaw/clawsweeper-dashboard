## Summary

Current `main` no longer has the TDZ-prone `ANTHROPIC_MODEL_ALIASES` top-level binding on the config-load path, and the issue-closing fix commit `cf5b181e40fbf684bd125b3de87efae0b6bbcee8` is included before release tag `v2026.3.13-1`. This issue is already fixed on `main` and shipped after `2026.3.12`.
