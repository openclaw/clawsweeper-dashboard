## Summary

Current `main` already implements the requested fix: bundled channel plugins explicitly enabled through channel config are treated as explicit activation, bypass restrictive `plugins.allow` checks, are preserved during gateway startup, and the behavior is covered by tests and documented in config help. The latest locally available release tag `v2026.4.23` contains this behavior.

## Evidence
