## Summary

Current main already implements and documents a supported way to inspect which credential source OpenClaw is using for OpenAI. `openclaw models status` and `openclaw models status --json` expose per-provider effective auth source, and the docs explain the precedence that can make an older stored profile win over a newly exported env key.

## Evidence
