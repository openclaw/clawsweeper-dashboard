## Summary

Current `main` already addresses this report well enough: gateway model-pricing bootstrap now runs asynchronously, pricing fetch timeouts were extended to 30 seconds, and refreshes are scheduled on a 24-hour cadence instead of creating frequent warning churn. The fix is explicitly documented as shipped in `v2026.4.23`, so this issue can be conservatively closed as implemented on main even though no explicit disable toggle was added.
