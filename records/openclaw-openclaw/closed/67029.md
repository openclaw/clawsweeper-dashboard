## Summary

Keep open. Current `main` still routes memory-core dreaming narrative cleanup through the gateway `sessions.delete` path, which remains `operator.admin`-scoped, so the reported `missing scope: operator.admin` failure path still exists. There is an unreleased cleanup hardening entry for leaked dreaming sub-sessions, but it does not remove the admin-scoped cleanup call or the warning path described in this issue.
