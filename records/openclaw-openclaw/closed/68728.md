## Summary

Keep open. Current main still does not mark Ollama `gemma4` or `/api/show` `capabilities: ["thinking"]` models as reasoning-capable. The linked PRs are useful context, but none of that work is merged into the checked-out main SHA.

## Best Possible Solution

Keep #68728 open until a narrow Ollama fix lands on main. The likely fix belongs in `extensions/ollama/src/provider-models.ts`: set `reasoning: true` when `/api/show` capabilities include `thinking` or when the model id matches `gemma4` variants, with regression coverage in `extensions/ollama/src/provider-models.test.ts` and discovery/setup tests as needed. Treat Hugging Face and self-hosted heuristic expansion as separate scope unless maintainers explicitly want the broader provider change.
