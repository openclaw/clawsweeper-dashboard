## Summary

Current `main` already suppresses the automatic final reply when a messaging tool send targets the same conversation, and the suppression path includes Feishu/Lark target normalization. The report appears to describe pre-fix behavior from `2026.4.22`; I could not tie the fix to a released tag, so `fixedRelease` is null.
