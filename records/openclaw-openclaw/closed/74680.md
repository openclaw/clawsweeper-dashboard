  - body: `openclaw/plugin-sdk/llm-task` is not exported by `package.json` and has no matching `src/plugin-sdk` entrypoint on current main, so this new plugin will fail to resolve/build.   - body: This package is ESM and the new AIRI module uses ESM imports, so `require` is undefined when `handleUserSpeech` runs. A speech message will crash before transcription and cleanup; use top-level ESM imports or `await import()` instead.
  - confidence: 0.9

## Security Review

Status: needs_attention

Summary: The diff adds unrelated security-sensitive plugin code, including real-money Freqtrade trading tools, to a PR whose stated purpose is only removing a bundled skill.

- **[medium] Separate real-money trading tools from this cleanup:** `extensions/freqtrade/src/tools.ts:97`
  - body: The new Freqtrade plugin registers `freqtrade_forcebuy` and `freqtrade_forcesell` tools that can place or close live trading positions when `dryRun` is disabled. That may be acceptable only with dedicated product/security review and should not be bundled into a no-op skill-removal PR.

  - confidence: 0.78
