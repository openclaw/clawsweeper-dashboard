## Summary

Issue #45822 matches the plugin HTTP-route regression that current `main` already fixes. The gateway now pins plugin webhook routes to the startup registry and resolves both auth and dispatch from the same live HTTP-route registry, with regression tests covering the exact 'active registry changed later' case. The changelog shipped that fix in `v2026.3.22`.
