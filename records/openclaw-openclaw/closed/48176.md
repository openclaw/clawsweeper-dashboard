## Summary

Current `main` already contains the browser-safety changes for this Control UI blank-screen class: the logging path now avoids eager temp-dir resolution in browser-like imports, the UI has local session-key helpers, and the tools UI imports the browser-safe shared policy surface. The changelog explicitly records this blank-screen fix as shipped in `v2026.4.2`.
