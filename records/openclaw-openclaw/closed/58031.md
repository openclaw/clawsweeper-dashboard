## Summary

Current `main` already supports much larger AGENTS/SOUL workspace context than the issue requests: bootstrap injection defaults to 12,000 chars per file and 60,000 chars total, workspace bootstrap reads allow up to 2 MiB, and the gateway file-set schema does not impose a 4 KB / 3 KB write cap.
