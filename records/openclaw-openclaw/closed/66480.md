## Summary

Current `main` already implements Control UI image attachments for file upload, clipboard paste, and drag/drop, and the Gateway accepts/sends those attachments through `chat.send`. The latest shipped release also contains the same code paths, so this feature request is already satisfied on current OpenClaw.
