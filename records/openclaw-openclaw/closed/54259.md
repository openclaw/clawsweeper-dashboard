## Summary

The request is a useful multi-agent coordination idea, but current main treats cross-agent visibility as an optional orchestration layer and already exposes plugin/hook seams that can implement it outside OpenClaw core.

## Risks / Open Questions

- A community/plugin implementation may still want a narrow additional seam for persistence or configuration ergonomics, but the requested behavior does not need to be a built-in core feature.
- Core main does not ship this multi-agent transcript feature today; closing here only reflects scope, not that the idea lacks value.
