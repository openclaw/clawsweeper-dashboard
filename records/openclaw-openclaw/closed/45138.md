## Summary

Current main already contains the fix path for this report: Anthropic model normalization no longer uses an `ANTHROPIC_MODEL_ALIASES` constant, and `applyContextPruningDefaults` is now gated behind an Anthropic auth signal with focused regression coverage. The same implementation is present in the released tag `v2026.4.22`.
