## Summary

Current main already contains explicit session-switch avatar refresh/reset logic for Control UI chat, plus regression coverage for stale avatar responses after switching sessions. The latest available release tag in this checkout, `v2026.4.22` (`00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5`), also contains that implementation, so the report against `2026.3.12` appears fixed in later builds.

## Evidence
