The linked fix attempts for this issue are closed/unmerged, so maintainers should keep the implementation discussion and eventual fix under #41879 rather than splitting the same heartbeat-scope bug across two open issues.

## Best Possible Solution

Close this issue as a duplicate of #41879. Keep #41879 open as the canonical implementation/product-policy thread, then fix the heartbeat scope contract in `src/infra/heartbeat-summary.ts` and `src/infra/heartbeat-runner.ts`, update the scheduler/runtime skip paths together, and adjust the regression tests and heartbeat docs to match the chosen policy.
