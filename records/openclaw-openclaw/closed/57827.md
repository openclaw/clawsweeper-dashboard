## Summary

Current `main` already avoids shell interpolation in `config.openFile`: it builds a command/argv tuple, executes it with `execFile`, and has regression tests that pass shell metacharacters through `OPENCLAW_CONFIG_PATH` as literal path data. The same implementation is present in the shipped `v2026.4.23` release.

## Evidence
