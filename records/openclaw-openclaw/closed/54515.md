## Summary

Do not close #54515 yet. Current main fixed the exact deepgram/deepseek/groq 2026.3.14 examples and normalizes bundled plugin versions in status output, but the broader packaging contract remains unresolved: current main still has at least one packaged bundled plugin version older than the root package, the npm release check does not enforce bundled plugin version alignment, and PR #54572 proposed that guard but was closed unmerged.

## Best Possible Solution

Keep #54515 open until maintainers choose and enforce the bundled plugin version contract. Either align packaged bundled plugin metadata during release/prepack with a tested release-check guard, or explicitly document which bundled plugins may have independent package versions and ensure config/status/update diagnostics consistently report the host OpenClaw version separately from plugin package versions.
