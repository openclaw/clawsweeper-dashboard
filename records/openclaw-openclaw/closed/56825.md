## Summary

Current main already uses a direct directory watch for default workspace memory indexing, with a regression test covering that behavior, and the same implementation is present in release v2026.4.23.

## Evidence

- **Default memory watcher no longer uses a glob:** The default memory watcher now seeds chokidar with `MEMORY.md` and the `memory` directory itself, then marks the index dirty on add/change/unlink. The reported `path.join(this.workspaceDir, "memory", "**", "*.md")` pattern is not used for the built-in workspace memory path.
