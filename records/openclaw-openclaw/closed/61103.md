## Summary

Current `main` no longer drops Control UI image attachments when the session model is text-only. The gateway now offloads those images to `media://inbound/*` refs and regression tests cover that exact path. The issue timeline’s cross-reference to PR #63783 matches the implementation now present on `main`. The checkout history is shallow, so the exact fix commit and first released version could not be determined confidently.
