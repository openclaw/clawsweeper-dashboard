## Summary

Current `main` already has the one-shot gateway RPC changes that address this report: the CLI cron path is still a direct gateway RPC call, but the gateway call wrapper now yields before socket start to avoid handshake starvation, ignores intentional teardown closes, and waits for WebSocket shutdown before resolving.

## Evidence

- **`cron list` still uses the direct one-shot gateway RPC path:** `openclaw cron list` calls `callGatewayFromCli("cron.list", ...)`, and the CLI runtime forwards that to `callGateway(...)` rather than doing local plugin work first.
  - file: [src/cli/cron-cli/register.cron-simple.ts:32](https://github.com/openclaw/openclaw/blob/56fe2aab9c87596515f2280c8060dc740ea64a88/src/cli/cron-cli/register.cron-simple.ts#L32)

- **Gateway call wrapper now avoids handshake starvation from heavy synchronous loading:** `callGateway` explicitly yields to the event loop before opening the WebSocket, with an inline note that heavy synchronous module loading could otherwise starve `connect.challenge` handling and cause handshake timeouts.
  - file: [src/gateway/call.ts:479](https://github.com/openclaw/openclaw/blob/f85806b6d78698fb6b6060d09e16cb09ae40131b/src/gateway/call.ts#L479)

- **Intentional teardown closes are suppressed instead of surfacing as user-facing failures:** After a request completes, `callGateway` sets `ignoreClose = true`, clears the wrapper timeout, and only then tears the client down; the `onClose` handler exits early when teardown is intentional.
  - file: [src/gateway/call.ts:487](https://github.com/openclaw/openclaw/blob/f85806b6d78698fb6b6060d09e16cb09ae40131b/src/gateway/call.ts#L487)

- **Gateway client teardown now waits for socket close instead of racing its own normal close:** `GatewayClient.beginStop()` creates a pending stop, calls `ws.close()`, and waits for shutdown completion/terminate fallback, which prevents the one-shot client from treating its own close as an intermittent failure path.
  - file: [src/gateway/client.ts:381](https://github.com/openclaw/openclaw/blob/f85806b6d78698fb6b6060d09e16cb09ae40131b/src/gateway/client.ts#L381)
