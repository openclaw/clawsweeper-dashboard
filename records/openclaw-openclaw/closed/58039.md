## Summary

Current `main` already decouples `/model` from the failing provider path by persisting model overrides through session state, marking a pending live switch, and applying that switch at the next safe retry boundary. The present code and tests cover both the control-plane update and the recovery path that avoids getting stuck on the unavailable provider.

## Evidence
