## Summary

Current main, and shipped release v2026.4.24, no longer build the reported LCM compaction payload with a hardcoded `temperature: 0.2`. OpenClaw now pins `@mariozechner/pi-coding-agent`/`pi-ai` to 0.70.2, whose `generateSummary()` completion options omit temperature, and OpenClaw's compaction wrapper does not add it before reaching the Anthropic transport.

## Best Possible Solution

Close as implemented. Keep the current 0.70.2 compaction dependency path and avoid reintroducing default sampling params in future compaction wrappers; add a focused regression test if the summarization wrapper or Anthropic transport options are changed again.
