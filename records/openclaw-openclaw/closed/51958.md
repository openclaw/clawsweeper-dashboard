## Summary

Current `main` already supersedes PR #51958. Exec-approval followups now compute an external-only delivery target, preserve `webchat` as session-only origin metadata, and the gateway downgrades best-effort internal delivery to session-only instead of throwing channel-selection errors.

## Evidence
