## Summary

Current `main` does not implement file-backed cron `agentTurn` prompts. The cron payload type still requires inline `message`, the CLI only exposes `--message`, docs still document `--message` as required for isolated jobs, and the timer passes `job.payload.message` directly into isolated runs. This is a coherent, still-actionable feature request, so there is not strong evidence to close it.
