## Summary

Keep open. Current `main` does not already implement this PR: the Linux fallback executable lists still omit the `/usr/lib/...` and `/opt/...` Chromium/Chrome/Brave paths from the PR, and the launch error still lacks the proposed `browser.executablePath` hint. The underlying Linux detection gap remains plausible on `main`.
