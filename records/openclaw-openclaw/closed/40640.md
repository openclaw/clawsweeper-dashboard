## Summary

PR #40640 proposes a bundled optional `computer-use` plugin scaffold, but current OpenClaw direction keeps this kind of optional capability out of core and points it to external plugin distribution instead. Current docs already show that optional tool plugins should be published to ClawHub or npm, and `main` does not contain an `extensions/computer-use` plugin.

## Evidence
