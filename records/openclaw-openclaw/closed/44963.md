## Summary

Nexos is not implemented on current main, but this PR is an optional third-party model gateway/provider integration. Current main already has the ClawHub/npm plugin path and generic provider-plugin APIs for API-key auth, model catalogs, dynamic discovery, manifest-driven onboarding flags, and OpenAI-compatible endpoints, so this does not need to be bundled into OpenClaw core.

## Best Possible Solution

Do not merge this as a bundled core provider. The best path is for Nexos to publish and maintain a provider plugin on ClawHub/npm using the existing provider-plugin and manifest auth surfaces; open a focused SDK/API issue only if that external plugin hits a concrete missing generic seam.
