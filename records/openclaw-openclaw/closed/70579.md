## Summary

Keep open. Current `main` still does not implement Telegram `replyToModeByChatType`, and Telegram ingress/native-command delivery still uses a single reply-mode path, so the requested behavior is not already shipped. The PR also has concrete blocking review findings from its own discussion, so there is not strong evidence to close it as obsolete or implemented.

## Evidence
