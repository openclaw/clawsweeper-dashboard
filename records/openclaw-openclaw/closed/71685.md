## Summary

Keep PR #71685 open. Current main does not already implement an Azure Speech TTS provider, and the PR contains potentially useful provider work, but it is not a conservative auto-close candidate because the PR changes a live plugin/provider surface, has unresolved review feedback, and likely needs maintainer judgment on whether Azure TTS should be a bundled plugin, external plugin, or ClawHub item.
