## Summary

Current main already exposes Feishu document/wiki/drive functionality as agent plugin tools. The expected surface is runtime agent tools, not a top-level `openclaw feishu_doc` CLI command, and the shipped Feishu wiki skill documents the wiki-link-to-doc-read workflow.

## Best Possible Solution

Close as implemented on current main. Users should verify availability with `/tools` or `/tools verbose` in the Feishu conversation; if a current install still cannot see or invoke `feishu_doc`, open a fresh issue with the redacted Feishu config, `/tools verbose` output, OpenClaw version, gateway logs, and whether the message is a normal chat, wiki link, or document-comment thread.
