Full review comments:

- **[P2] Preserve the current model when no scoped rows exist:** `src/flows/model-picker.ts:811`
  - body: `maybeFilterModelsByProvider` only enforces the preferred provider when matching rows exist; otherwise the picker intentionally falls back to the unscoped list. This condition hides the configured current model whenever `preferredProvider` is set, so a non-catalog current model becomes unselectable in fallback-unscoped cases. Gate this on `hasPreferredProvider` as well.
  - confidence: 0.88
- **[P2] Fall back to a model option, not the first action:** `src/flows/model-picker.ts:829`
  - body: When the computed initial value is hidden, `options[0]` can be `__manual__`, a provider setup action, or undefined after auth filtering rather than a selectable model. Prefer the first displayed model value and return the manual flow when no options remain, so the picker cannot default to an action or call select with an empty list.
  - confidence: 0.83
