## Summary

Current `main` no longer emits a hardcoded `Subagent main finished`-style completion message. Subagent completion delivery now goes through typed internal `task_completion` events and a parent-agent/user-facing render path, and regression tests explicitly assert that the old `✅ Subagent` banner is absent.

## Evidence
