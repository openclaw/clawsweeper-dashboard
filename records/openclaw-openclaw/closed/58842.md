## Summary

Current main, shipped in v2026.4.26, now covers the local Ollama/local-model hang class reported here: native Ollama no longer forces full configured context unless `params.num_ctx` is explicit, Ollama capabilities suppress unsupported tools, native thinking is opt-in/configurable, provider-scoped request timeouts reach guarded fetches, and a lean local-model profile exists for small-model/tool-surface pressure.
