## Summary

PR #69385 adds an optional bundled AJ Router plugin. Current main already has the external plugin path, model-routing hook, plugin config, command, and ClawHub install surfaces needed for this to live outside core, while VISION.md sets a high bar for adding optional plugins to the repository. Existing PR review comments also identify correctness issues to fix before publishing externally.

## Best Possible Solution

Close this core PR as ClawHub scope. AJ Router should be maintained as an external ClawHub/npm plugin using the existing before_model_resolve, command, config, and install surfaces; if external implementation proves that model-resolution needs a cleaner fail-closed abort result, open a focused Plugin SDK proposal for that seam instead of bundling the whole router.
