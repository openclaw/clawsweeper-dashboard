## Summary

Current `main` does not implement this PR. The compact-digit timestamp handling is still unchanged, the cron store migration/inference work now lives in `openclaw doctor --fix` rather than runtime load-time rewrites, and the added cron observability logs are not present. The PR may need rework against current architecture, but conservative cleanup should keep it open.
