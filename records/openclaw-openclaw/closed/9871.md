## Summary

`PluginCommandContext` on current `main` already includes `sessionKey`, plugin command dispatch forwards it, and there is a regression test covering the pass-through. The same implementation is present in the tagged release commit `v2026.4.22` (`00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5`).
