## Summary

Keep open. I did not find strong evidence on current `main` that #56670 is fixed or obsolete. The `openai-completions` streaming usage path is still materially the same as the latest release, and I did not find an end-to-end regression proof that streamed usage survives all the way into persisted session transcript messages.
