- **[P1] Scope task-route leases beyond runId:** `src/tasks/task-route-lease.ts:140-146`
  - body: `runId` is not a globally unique delivery identity in the existing task registry: current code scopes same-runId records by runtime/session/owner and tests cover different requester scopes sharing a runId.
