## Summary

Close as a duplicate of the still-open broader TUI footer customization tracker #53774. Current main has not implemented footer display modes; the remaining work is the same footer customization surface that #53774 already tracks, with #70290 as a possible compact-rendering input rather than a shipped fix.

## Best Possible Solution

Close this as duplicate/superseded by #53774 and keep #53774 as the canonical TUI footer customization tracker. The eventual implementation should add a reviewed footer display/config surface in `src/tui/tui.ts`, `src/tui/tui-types.ts`, CLI/config/schema/docs, and focused tests, while considering compact rendering ideas from #70290.
