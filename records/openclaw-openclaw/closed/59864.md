## Summary

Current `main` already covers the two failure modes described in #59864: Telegram DM voice notes now run audio preflight transcription instead of reaching the agent as raw `<media:audio>`, and the bundled Deepgram media-understanding provider is enabled by default and registered under the `deepgram` id.

## Evidence
