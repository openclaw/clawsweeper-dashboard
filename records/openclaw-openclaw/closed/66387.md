## Summary

Current main supersedes this PR with an approved custom-theme architecture: built-in Control UI themes remain `claw`, `knot`, and `dash`, while user-supplied aesthetics now flow through a single browser-local `custom` theme slot imported from tweakcn. That means the hard-coded Matrix/Cyberpunk additions proposed here no longer match the maintained direction.
