## Summary

Current `main` already ships this feature as the context-engine plugin slot: plugins register `api.registerContextEngine(...)`, users select it with `plugins.slots.contextEngine`, the runner replaces the model message array from `assemble(...)`, and post-turn capture runs through `afterTurn`/`ingestBatch`. The changelog says this landed in release 2026.3.7 via #22201.

## Risks / Open Questions

- The shipped method names are `assemble` and `afterTurn`/`ingestBatch`, not the exact sketch names `buildContext` and `captureExchange`, but the behavior is present.
