## Summary

Keep this issue open. Current main still does not emit a gateway startup/runtime-auth warning for legacy `CLAWDBOT_*` or `MOLTBOT_*` environment variables, and open maintainer PR #73445 now supersedes the earlier closed-unmerged PR #53667 as the implementation candidate linked with closing syntax.

## Best Possible Solution

Keep this issue open and review, land, or replace PR #73445. The best fix belongs in gateway startup/runtime-auth diagnostics, with focused tests that detect `CLAWDBOT_*` and `MOLTBOT_*`, improve missing-auth guidance, and preserve the intentional removal of legacy fallback support.
