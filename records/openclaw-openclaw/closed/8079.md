## Summary

Current `main` already supports the requested browser proxy routing via `browser.extraArgs`, which accepts Chromium flags like `--proxy-server=...`, carries them through resolved browser config, and appends them to Chrome launch args. The same implementation is present in release `v2026.4.22`.
