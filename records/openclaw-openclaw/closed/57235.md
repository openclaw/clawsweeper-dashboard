## Summary

Current OpenClaw already ships the requested web multi-session chat capability: the Control UI has a dedicated new-session button, in-chat session switching, and a Sessions screen for browsing/managing prior sessions. The implementation is present on current `main` and already shipped in `v2026.4.23`.
