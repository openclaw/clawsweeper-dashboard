## Summary

Current `main` already provides channel-visible pre-exec/tool-start logging through verbose tool summaries. Users can enable it per session with `/verbose on` or by default with `agents.defaults.verboseDefault: "on"`; exec tool starts include the command args and are forwarded to messaging surfaces before completion.
