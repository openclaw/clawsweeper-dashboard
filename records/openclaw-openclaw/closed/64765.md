## Summary

Current `main` includes a post-report Active Memory/QMD search-path fix. Active Memory now snapshots its own `qmd.searchMode` into the embedded sub-agent config, `memory_search` detects Active Memory sessions and forwards that override, and the QMD manager uses the correct QMD 1.1+ `query` call shape without sending unsupported `minScore`. The added tests cover both inheriting the main QMD mode and the v2/v1 query-tool fallback, which matches the issue’s reported Active Memory vs manual-search mismatch.

## Evidence
