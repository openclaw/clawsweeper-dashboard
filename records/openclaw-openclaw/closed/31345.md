## Summary

The request is for bundling a third-party WeChat channel into core/default channels, but current OpenClaw direction is to keep optional channel integrations as external plugins. Main already documents WeChat as an external plugin path, and community plugins are meant to be distributed via ClawHub or npm rather than merged into core by default.
