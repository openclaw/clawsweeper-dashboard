## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Gate before_assemble as prompt mutation:** `src/plugins/hook-types.ts:184-185`
  - body: `before_assemble` can replace model-visible history, but the PR only adds it to the conversation hook list. A non-bundled plugin with `allowConversationAccess=true` and `allowPromptInjection=false` could still rewrite prompt history, bypassing the documented prompt-mutation opt-out.
  - confidence: 0.93
- **[P2] Pass isolated messages into assembly hooks:** `src/agents/pi-embedded-runner/run/attempt.ts:3721`
  - body: The hook payload hands `activeSession.messages` directly to plugin code. A handler can mutate the array or nested message content and then return `void` or throw, changing session state outside the returned-result contract; clone and normalize the payload before exposing it.
  - confidence: 0.91
- **[P2] Validate replacements before storing messages:** `src/agents/pi-embedded-runner/run/attempt.ts:3739`
  - body: Returned `messages` are cast to `AgentMessage[]` and installed without schema validation or replay repair. A malformed plugin result can corrupt `activeSession` and fail later outside hook error handling; validate and normalize replacements before assignment.
  - confidence: 0.88
- **[P2] Refresh the pre-prompt boundary after replacement:** `src/agents/pi-embedded-runner/run/attempt.ts:3739`
  - body: `prePromptMessageCount` is captured before `before_assemble` can remove history. If a hook shrinks the message list, later current-turn lookup and context-engine finalization can slice at a stale boundary and miss the assistant turn or prompt-cache metadata.
  - confidence: 0.84
- **[P2] Report the submitted prompt to after_assemble:** `src/agents/pi-embedded-runner/run/attempt.ts:4213-4214`
  - body: The event claims to observe final model input, but it reports `systemPromptText` and `promptSubmission.prompt` instead of the runtime-context-aware `systemPromptForHook` and `promptForModel` that this path submits. Observability plugins will miss context actually visible to the model.
  - confidence: 0.88
