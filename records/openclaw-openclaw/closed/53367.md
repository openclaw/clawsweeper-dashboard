## Summary

Current `main` already fixes the stale OAuth overwrite bug through auth-store reload and recovery logic, and the shipped `v2026.4.23` release contains that implementation plus regression coverage. This PR's runtime-snapshot invalidation approach is no longer needed to address the reported behavior.
