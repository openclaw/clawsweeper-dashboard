## Best Possible Solution

Run direct macOS npm-global LaunchAgent proof for the Control UI or `gateway call update.run` latest-version case; if it passes, close as covered by the merged hardening stack, and if it fails, fix the durable handoff versus explicit safe refusal contract.

## Solution Assessment
