Request to expose managed TaskFlow creation, child-task linkage, wait/resume, and finish/fail operations to agents via CLI commands, an agent tool, or `sessions_spawn` flow binding.

Keep core TaskFlow CLI/tooling focused on inspection and cancellation, and publish any generic `taskflow` agent tool as a ClawHub plugin built on `api.runtime.tasks.managedFlows`.
