## Summary

Issue #57319 requests a new Lossless Claw (LCM) capability, but current OpenClaw main treats LCM as a community plugin concern rather than core-maintained behavior. The repo already exposes generic context-engine seams for compaction/assembly, and docs explicitly list Lossless Claw as a third-party plugin to install from npm/ClawHub.
