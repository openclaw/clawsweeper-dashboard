- **[P2] Skip managers that are still in use:** `extensions/memory-core/src/memory/manager-cache.ts:153-164`
  - body: `lastAccessAt` is refreshed when the manager is acquired, but a memory search or sync can keep using that same instance for longer than `idleMs`. Track active use or refresh idleness after operation completion before allowing eviction.
