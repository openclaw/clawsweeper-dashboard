## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.93

Full review comments:

- **[P1] Remove registry-driven git installs from core:** `skills/plugin-marketplace/scripts/marketplace.py:181-191`
  - body: The marketplace script installs registry entries by running `git clone`/`git pull` from remote metadata into the user's skill directory. That creates a parallel supply-chain path without ClawHub package validation, scan state, provenance, or pinned artifacts, so it should not be added to core.
  - confidence: 0.93
- **[P1] Escape user text before GUI command dispatch:** `skills/desktop-automation/scripts/desktop_auto.py:55`
  - body: The desktop automation script interpolates user-provided text directly into AppleScript and PowerShell command strings. If this were bundled, crafted text could break out of the intended keystroke/clipboard operation; use safe argument passing or literal escaping before any core distribution.
  - confidence: 0.88
