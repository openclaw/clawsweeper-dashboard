## Summary

Keep open. Current `main` does not implement the relay attach selection UX, and the PR is not safely obsolete. The checked-out code intentionally removed the old relay path, still throws generic tab-selection errors, and the current browser config contract has no `relayAttachUx` field. The PR also still appears to carry unresolved dependency/config-contract risk in the provided GitHub context.
