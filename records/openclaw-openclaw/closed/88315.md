## What This Changes

The PR adds an `ownsNativeCompaction` CLI backend capability, propagates it through backend resolution, marks `claude-cli` as owning compaction, skips OpenClaw compaction for owning non-harness CLI sessions, and documents/tests the contract.

## Best Possible Solution

Land the generic backend hook only after agent/plugin owners accept the public SDK contract and `claude-cli` defer semantics, with the focused compaction tests and plugin SDK checks staying green.

## Solution Assessment
