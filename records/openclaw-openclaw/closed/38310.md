## Summary

Keep open. Current `main` still does not support `SecretRef` values in `sandbox.docker.env`, so this PR has not been implemented on `main` and there is no strong basis to close it conservatively.

## Evidence

- **Config type still string-only:** `SandboxDockerSettings.env` is still typed as `Record<string, string>`, not `SecretInput`, so the requested config shape is not present on current `main`. See `src/config/types.sandbox.ts:20`-`21`.

  - file: [src/config/zod-schema.agent-runtime.ts:102](https://github.com/openclaw/openclaw/blob/eb6e1245ac881566a7f31adc7d94d9e3c721a6ac/src/config/zod-schema.agent-runtime.ts#L102)
