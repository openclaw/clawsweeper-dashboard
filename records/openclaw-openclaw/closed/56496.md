## Summary

Current main no longer exposes a top-level `buttons` parameter on the shared message tool, so the reported validation failure for non-send actions is obsolete. Telegram now advertises rich message UI through shared `presentation` capability instead, and tests explicitly assert that `properties.buttons` is absent while `react`/`poll` actions remain available.

## Evidence

- **Shared message tool uses presentation buttons, not top-level buttons:** `src/agents/tools/message-tool.ts:63` defines button data only inside `presentation.blocks[].buttons`, and `src/agents/tools/message-tool.ts:90` builds send-only schema fields from that presentation payload.
  - file: [src/agents/tools/message-tool.ts:63](https://github.com/openclaw/openclaw/blob/6b618f063516aa9ba85903aa4565c5a204c055b2/src/agents/tools/message-tool.ts#L63)
  - sha: [6b618f063516](https://github.com/openclaw/openclaw/commit/6b618f063516aa9ba85903aa4565c5a204c055b2)
- **Message tool schema assembly has no raw `buttons` property:** `src/agents/tools/message-tool.ts:380` composes the shared schema props, and `src/agents/tools/message-tool.ts:412` returns a flat `Type.Object({ action, ...props })` built from those props. There is no top-level `buttons` field to become erroneously required for `react`/`delete`/`edit`/`poll`.
  - file: [src/agents/tools/message-tool.ts:380](https://github.com/openclaw/openclaw/blob/6b618f063516aa9ba85903aa4565c5a204c055b2/src/agents/tools/message-tool.ts#L380)
  - sha: [6b618f063516](https://github.com/openclaw/openclaw/commit/6b618f063516aa9ba85903aa4565c5a204c055b2)
- **Regression test covers the reported shape:** `src/agents/tools/message-tool.test.ts:429` creates scoped message tools for Telegram/Discord/Slack, then `src/agents/tools/message-tool.test.ts:446-452` asserts `presentation` exists, `properties.buttons` is undefined, and the action enum still includes non-send actions such as `react` and `poll`.
  - file: [src/agents/tools/message-tool.test.ts:429](https://github.com/openclaw/openclaw/blob/6b618f063516aa9ba85903aa4565c5a204c055b2/src/agents/tools/message-tool.test.ts#L429)
  - sha: [6b618f063516](https://github.com/openclaw/openclaw/commit/6b618f063516aa9ba85903aa4565c5a204c055b2)

  - file: [extensions/telegram/src/channel-actions.ts:147](https://github.com/openclaw/openclaw/blob/6b618f063516aa9ba85903aa4565c5a204c055b2/extensions/telegram/src/channel-actions.ts#L147)
