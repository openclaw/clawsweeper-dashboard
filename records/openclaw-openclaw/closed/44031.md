## Summary

Current `main` already has an escape-hatch recovery path for compaction timeout/overflow hangs: it grants one compaction grace window, falls back to the pre-compaction snapshot, converts unrecoverable overflow/compaction failures into a fresh-session reset, and rewrites queued followups onto the new session so the lane can continue.
