## Summary

Current `main` already covers the request well enough: OpenClaw has a session-delete API, the Control UI exposes delete actions for selected sessions, and deleted transcript archives are cleaned up by session-maintenance retention / `openclaw sessions cleanup`, so cleared sessions no longer need to accumulate indefinitely.
