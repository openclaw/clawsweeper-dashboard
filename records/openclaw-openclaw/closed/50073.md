## Summary

Issue #50073 is a focused version of the still-open multi-agent capability profiling work already tracked in #35203. Current main has per-agent skill allowlists and target-agent discovery, but not the requested capability index or skill-based router, so this should consolidate under the canonical open RFC rather than close as implemented.
