## Summary

Current `main` does not match the reported failure mode: the bundled Google and MiniMax plugin schemas are small manifest JSON objects with no recursive `$ref` structure, plugin config validation reads those manifest objects directly, and the changelog records a post-`2026.4.5` fix for plugin-validation crashes. I could not find evidence on `main` that these two bundled plugins still expose the reported AJV stack-overflow path.

## Evidence

- **Google plugin schema is acyclic manifest JSON:** The bundled Google plugin declares a small literal `configSchema` with only `webSearch.apiKey` and `webSearch.model`; there is no `$ref`, `definitions`, or recursive branch in the schema.
  - file: [extensions/google/openclaw.plugin.json:76](https://github.com/openclaw/openclaw/blob/245451b6a9c529464e177afa65013adca0d63eb5/extensions/google/openclaw.plugin.json#L76)

  - sha: [245451b6a9c5](https://github.com/openclaw/openclaw/commit/245451b6a9c529464e177afa65013adca0d63eb5)
- **MiniMax plugin schema is acyclic manifest JSON:** The bundled MiniMax plugin likewise declares a small literal `configSchema` with only `webSearch.apiKey` and `webSearch.region`; there is no `$ref`, `definitions`, or recursive branch in the schema.
  - file: [extensions/minimax/openclaw.plugin.json:107](https://github.com/openclaw/openclaw/blob/245451b6a9c529464e177afa65013adca0d63eb5/extensions/minimax/openclaw.plugin.json#L107)

  - sha: [245451b6a9c5](https://github.com/openclaw/openclaw/commit/245451b6a9c529464e177afa65013adca0d63eb5)
- **Plugin manifests load config schemas from manifest JSON objects:** `loadPluginManifest` parses the on-disk manifest and only accepts an object-valued `configSchema` from that manifest payload before returning the normalized plugin manifest.
  - file: [src/plugins/manifest.ts:866](https://github.com/openclaw/openclaw/blob/245451b6a9c529464e177afa65013adca0d63eb5/src/plugins/manifest.ts#L866)

  - sha: [245451b6a9c5](https://github.com/openclaw/openclaw/commit/245451b6a9c529464e177afa65013adca0d63eb5)

  - file: [src/config/validation.ts:1155](https://github.com/openclaw/openclaw/blob/245451b6a9c529464e177afa65013adca0d63eb5/src/config/validation.ts#L1155)
