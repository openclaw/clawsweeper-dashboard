## Summary

Current `main` no longer uses a `cmd.exe`/`findstr` shell pipeline for Windows gateway port probing. The gateway startup/restart path now routes through direct PowerShell/netstat helpers with `windowsHide: true` and no `shell: true`+args pattern, so the reported console flash and `DEP0190` path are already removed.
