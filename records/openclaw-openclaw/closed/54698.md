## Summary

Current `main` already routes embedded/followup/heartbeat runs through the shared model-fallback path, and it has dedicated embedded-run regression coverage for overloaded-primary fallback behavior. The fix landed after the latest tagged release, so it is implemented on `main` but not yet tied to a shipped release tag.

## Evidence
