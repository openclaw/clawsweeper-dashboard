## Summary

Current `main` already documents the requested guidance in the newer browser-control docs: ARIA snapshots are inspection-only, interactive snapshots use `e12` role refs, and the debug workflow tells users to retry failed actions from `snapshot --interactive` and prefer role refs. This PR is superseded by the current docs surface on `main`.

## Evidence
