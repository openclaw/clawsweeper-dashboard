## Summary

Issue #62711 is a duplicate of the still-open Signal SSE tracking issue #61434. Both reports describe OpenClaw failing to keep the Signal `/api/v1/events` SSE stream alive with signal-cli 0.14.x while JSON-RPC/sending still works, causing inbound Signal messages to be unavailable. Current main still routes Signal receiving through the same SSE fetch loop, so this should stay tracked in the canonical open issue rather than as a separate report.
