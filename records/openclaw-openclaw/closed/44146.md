## Summary

Keep open. Current `main` at `50e36983bb2d827adc617282c090c0a0359c6eee` still uses the pre-PR `JSONSerialization` response bridge in `MacNodeBrowserProxy`, and the macOS tests still lack the scalar-response regression this PR adds. The earlier request-body crash fix from #43069 is present, but the response-side/browser-proxy JSON bridge change described in PR #44146 is not implemented on `main` or in the latest release tag I inspected.
