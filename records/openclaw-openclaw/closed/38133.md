## Summary

Keep open. Current `main` still computes the `/usage` line in the reply pipeline but the agent-event chat path finalizes from the streamed buffer, not the finalized reply payload, so the reported CC Chat mismatch is still present in code. Control UI now reloads transcript/history after final events and can render usage metadata from transcript messages, but that is a separate path and does not prove `/usage tokens|full` semantics are implemented for agent chat finals.
