## Summary

Current main already contains the requested guardrails: missing or partial usage snapshots are normalized safely, `totalTokens` is derived from component fields when absent, prior session `totalTokens` is preserved when a provider omits usage, and the parent-session fork path now checks that the parent entry exists before reading `.totalTokens`.
