## Summary

Current main now handles the reported Control UI context-full failure mode: it shows a high-context warning with a Compact action, routes that action through the normal session compaction RPC and refresh path, hard-caps oversized chat.history payloads so the transcript response cannot break the browser view, and falls back to clear overflow recovery/error behavior in the runner.
