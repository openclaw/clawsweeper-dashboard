## Summary

Current main already covers this class of failure. Top-level subagent completion delivery no longer depends on a live parent session row, subagent runs are tracked in a durable SQLite-backed task ledger with persisted requester routing, and terminal task updates can be delivered directly to the originating channel with session-queue fallback.

## Evidence
