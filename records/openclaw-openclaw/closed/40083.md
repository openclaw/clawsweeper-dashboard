## Summary

Current `main` already covers the requested troubleshooting and recovery surface for broken `~/.openclaw` state: `openclaw doctor` now performs targeted state-integrity checks and guided repairs, `openclaw backup create|verify` provides a first-class backup path, and docs cover migration/recovery workflows.
