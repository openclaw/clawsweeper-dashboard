## Summary

Close as superseded. Current main has not implemented this PR's one-line `isControlUiVisible` guard removal, and merging it as written would widen a documented hidden-run chat/node fanout boundary for external-channel sessions. The same TUI live-session problem is now covered by the open maintainer PR #73402, which uses the safer `sessions.messages.subscribe` / `session.message` transcript path instead of broadcasting hidden external-channel runs through the general chat fanout.

## Best Possible Solution

Close this PR as superseded by #73402. Review and land #73402, or an equivalent implementation, that subscribes the TUI to `sessions.messages.subscribe` / `session.message` for the active session while keeping hidden external-channel chat/node fanout gated by `isControlUiVisible` and covering both active-session filtering and hidden-run fanout in tests.
