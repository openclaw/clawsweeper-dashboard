## Summary

Current `main` now makes the ClawHub plugin contract explicit: external code plugins still require `openclaw.compat.pluginApi`, the documented format is a date-style semver range, and install/runtime code still validates that field.

## Evidence

- **Plugin setup docs now define the ClawHub publish baseline:** The plugin setup guide explicitly shows `openclaw.compat.pluginApi` and says external ClawHub publishes require the `compat` and `build` blocks.
  - file: [docs/plugins/sdk-setup.md:44](https://github.com/openclaw/openclaw/blob/68315792677f9628d1370d6869366442f1d3ecad/docs/plugins/sdk-setup.md#L44)
  - sha: [68315792677f](https://github.com/openclaw/openclaw/commit/68315792677f9628d1370d6869366442f1d3ecad)
- **ClawHub docs repeat the required package metadata:** The ClawHub guide says code plugins must include `openclaw.compat.pluginApi` and shows the expected date-style range format.
  - file: [docs/tools/clawhub.md:290](https://github.com/openclaw/openclaw/blob/68315792677f9628d1370d6869366442f1d3ecad/docs/tools/clawhub.md#L290)
  - sha: [68315792677f](https://github.com/openclaw/openclaw/commit/68315792677f9628d1370d6869366442f1d3ecad)
- **Shared publish contract still requires pluginApi:** The package-contract helper lists `openclaw.compat.pluginApi` as a required field and emits the exact ClawHub validation message when it is missing.
  - file: [packages/plugin-package-contract/src/index.ts:23](https://github.com/openclaw/openclaw/blob/68315792677f9628d1370d6869366442f1d3ecad/packages/plugin-package-contract/src/index.ts#L23)
  - sha: [68315792677f](https://github.com/openclaw/openclaw/commit/68315792677f9628d1370d6869366442f1d3ecad)
- **Install/runtime still enforce plugin API compatibility:** ClawHub plugin install rejects packages whose advertised `pluginApiRange` does not satisfy the active runtime version.
  - file: [src/plugins/clawhub.ts:662](https://github.com/openclaw/openclaw/blob/3a64aa49a9014518fa1977fbc023f3e0b978ae5d/src/plugins/clawhub.ts#L662)
  - sha: [3a64aa49a901](https://github.com/openclaw/openclaw/commit/3a64aa49a9014518fa1977fbc023f3e0b978ae5d)
- **Tests cover the current date-style plugin API contract:** Regression coverage expects `pluginApiRange: >=2026.3.22` and the matching runtime-version error path, which matches the current documented contract rather than the old exact `1.2.0` form.
