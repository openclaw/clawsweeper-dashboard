## Summary

Current `main` already fixes the reported packaging bug: the build config now emits `plugins/runtime/index`, tests lock that entry into the unified dist graph, and a built-dist smoke test loads plugins from the built artifact and asserts they register successfully. The same fix is present in release `v2026.4.22`.
