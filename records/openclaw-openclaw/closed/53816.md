## Summary

Current main no longer schedules the reported test pair into the same Windows CI process. `src/plugin-sdk/channel-policy.test.ts` is now classified into the `unit-fast` lane, while `src/cli/daemon-cli/lifecycle-core.test.ts` stays in the agentic CLI/daemon lane, and CI runs those shards separately.
