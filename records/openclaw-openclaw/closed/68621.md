## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.84

Full review comments:

- **[P2] Preserve legacy user-pinned profiles:** `src/agents/pi-embedded-runner/run/setup.ts:134-137`
  - body: This guard only protects profiles when `requestedAuthProfileIdSource` is explicitly `"user"`. Older persisted `/model ...@profile` selections can have an auth profile without a source, and this branch would let a hook override that legacy user pin. Treat source-less requested profiles as user-selected unless the legacy compaction marker identifies them as automatic.
  - confidence: 0.86
