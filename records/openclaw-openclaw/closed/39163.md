## Summary

Current `main` appears to have made this PR obsolete by removing the separate-bundle singleton condition for bundled root plugins. The logging transport `Set` is still local in `src/logging/logger.ts`, but bundled root plugins and `plugin-sdk` now ship in one unified dist graph and bundled public artifacts reuse one shared Jiti loader, which addresses the split-registry failure mode this PR targeted.
