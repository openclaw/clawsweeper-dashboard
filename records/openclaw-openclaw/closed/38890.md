## Summary

Current `main` already supports suppressing exec lifecycle chat noise via `tools.exec.notifyOnExit`, and that setting is exposed through the schema-backed Control UI config surface. This draft PR's single-file `ui/src/ui/storage.ts` change is obsolete and incomplete relative to the shipped implementation path.
