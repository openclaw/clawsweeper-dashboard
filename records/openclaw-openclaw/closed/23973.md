## Summary

Issue #23973 requests an optional novelty skill/plugin. Current project scope explicitly routes new skills and most optional capabilities to ClawHub/community plugins rather than OpenClaw core, so this belongs in the external skill/plugin ecosystem, not the main repo backlog.
