## Summary

Current main and the shipped v2026.4.23 release no longer advertise `openclaw qa` to packaged npm users. The QA CLI is now gated to source checkouts only, while npm packaging continues to intentionally omit `qa-lab` artifacts.

## Evidence

- **Private QA CLI is source-checkout only:** `loadPrivateQaCliModule` only resolves when `OPENCLAW_ENABLE_PRIVATE_QA_CLI=1` and the package root also has `.git`, `src`, and `dist/plugin-sdk/qa-lab.js`; otherwise it throws `Private QA CLI is only available from an OpenClaw source checkout.`
  - file: [src/cli/program/private-qa-cli.ts:8](https://github.com/openclaw/openclaw/blob/9a0b26cafca76d267dd231b5eaceadb071ec749d/src/cli/program/private-qa-cli.ts#L8)

  - file: [src/cli/program/subcli-descriptors.ts:175](https://github.com/openclaw/openclaw/blob/9a0b26cafca76d267dd231b5eaceadb071ec749d/src/cli/program/subcli-descriptors.ts#L175)
