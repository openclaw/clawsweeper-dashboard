## Summary

Current `main` already handles Feishu group slash commands by stripping the bot mention before command parsing, normalizing mention-prefixed group commands for command detection, and covering that behavior with regression tests. The changelog records the shipped fix in `v2026.3.7`.

## Evidence

  - file: [extensions/feishu/src/bot.ts:504](https://github.com/openclaw/openclaw/blob/7d9172d5e093435d37ed3e00e476a4eaf84a252b/extensions/feishu/src/bot.ts#L504)
