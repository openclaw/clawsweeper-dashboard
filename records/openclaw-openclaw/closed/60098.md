## Summary

Current main does not show evidence of a remaining core regression here. The reported symptom matches a config-path/env-name mismatch more than a gateway bug: main still accepts exact localhost origins, the active config path is `~/.openclaw/openclaw.json` (or `OPENCLAW_CONFIG_PATH`), and current docs/install flows explicitly support SSH-tunneled dashboard access with `gateway.controlUi.allowedOrigins` set to localhost/127.0.0.1.

## Evidence
