## Summary

Keep open. The issue asks for the gateway/GUI to default to 0.0.0.0 for network access, but current main still defaults bare-metal and VM hosts to loopback. Main only auto-binds to 0.0.0.0 inside detected containers, and non-loopback use still requires auth plus Control UI allowed origins. That is a partial, environment-specific improvement, not the requested default change.
