## Best Possible Solution

Repair the node-host plugin registry or installed-service discovery path so an enabled file-transfer plugin contributes the `file` cap and all four commands before `GatewayClient` construction, with handshake-level regression coverage.

## Reproduction Assessment

## Solution Assessment
