## Summary

Current `main` keeps core model selection intentionally simple (primary + fallbacks), while exposing a public plugin hook specifically for deterministic model/provider overrides before resolution. Repo vision also says optional capability should usually ship as plugins and that ClawHub is the canonical surface for community plugins, so this request fits a plugin/ClawHub solution rather than OpenClaw core.
