## Summary

Current main already implements the isolated cron agentTurn deadlock fix that matches this report: cron-triggered embedded work is remapped from the occupied cron lane onto the nested lane, covered by regression tests, and recorded in the 2026.3.13 changelog after the reported 2026.3.8 version.

## Evidence
