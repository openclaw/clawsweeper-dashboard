## Summary

Current `main` already contains a same-snapshot plugin-loader reentry guard, runtime short-circuiting to avoid recursive reloads, and regression coverage for the exact recursion pattern described in #62320. The shipped `v2026.4.23` release also contains those protections, so this 2026.4.5 regression is no longer current.

## Evidence

- **Shipped loader now hard-stops same-cache-key reentry:** `PluginLoadReentryError` is defined and `loadOpenClawPlugins` throws it when the same registry cache key is already in flight, preventing recursive re-entry instead of recursing until stack overflow/hang.
