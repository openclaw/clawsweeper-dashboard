## Summary

Current `main` already exposes the requested steer/abort capability to plugins through the public SDK: `openclaw/plugin-sdk/agent-harness` re-exports `queueEmbeddedPiMessage` as `queueAgentHarnessMessage` and `abortEmbeddedPiRun` as `abortAgentHarnessRun`, and that subpath is in the package exports/entrypoint list used for dist JS generation.
