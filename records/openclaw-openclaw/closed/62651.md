## Summary

Current `main` already contains the managed-browser CDP reuse fix for the follow-up `PortInUseError` race, routes `snapshot`/`navigate`/`act` through the same profile/CDP path, and ships a newer browser plugin with `playwright-core` 1.59.1. The fix is also present in release `v2026.4.23` (`a9797214338ba31c52c796adbb75afb16e0684a9`).

## Evidence
