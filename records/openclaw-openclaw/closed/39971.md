## Summary

Current `main` already contains the Kimi Coding fix for plain-text pseudo tool calls and shipped it in the `2026.3.12` release notes. The provider is now configured for native `anthropic-messages`, wraps Kimi streams through a provider-specific normalizer, and has regression tests that convert tagged/plain-text Kimi tool markup into structured tool calls.
