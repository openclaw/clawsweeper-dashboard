## Summary

Current `main` already fixes the cached-rejected-promise behavior for the LanceDB loader and the related database-initialization path, with regression tests in-tree. This PR is therefore obsolete, and its second commit would regress current gateway-token scoping behavior in the Control UI.

## Evidence
