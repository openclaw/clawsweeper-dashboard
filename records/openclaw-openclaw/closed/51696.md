## Summary

Current `main` already implements a web chat toggle to hide tool calls/results, persists that preference, and suppresses tool-result rendering when the toggle is off. The same implementation is present in the released `v2026.4.22` tag.

## Evidence

  - file: [ui/src/ui/storage.ts:51](https://github.com/openclaw/openclaw/blob/aa3a5f9ff7907c2a41d6259d24b529bda41f293a/ui/src/ui/storage.ts#L51)
