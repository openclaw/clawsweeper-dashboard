## Summary

Current `main` already contains Gemini-specific incomplete-turn recovery that retries reasoning-only, empty, and planning-only Gemini turns and surfaces terminal error/liveness state instead of leaving the session in a silent stuck state. The changelog also records this exact behavior change as fixing Gemini sessions that silently stall.

## Evidence
