## Summary

Keep open. Current `main` has partial outbound hook wiring, but it does not implement this PR's claimed session-context propagation or its chunk-level outbound lifecycle behavior well enough to treat the PR as already implemented on main.

## Evidence

- **Plugin hook message context still lacks session attribution fields:** `PluginHookMessageContext` on current `main` only exposes `channelId`, `accountId`, and `conversationId`; there is no `sessionKey` or `sessionId` in the plugin hook context type.
  - file: [src/plugins/hook-message.types.ts:1](https://github.com/openclaw/openclaw/blob/3361593442e295e9bb1f55cbfb38861ea9413e45/src/plugins/hook-message.types.ts#L1)
  - command: `nl -ba src/plugins/hook-message.types.ts | sed -n '1,220p'`
  - sha: [3361593442e2](https://github.com/openclaw/openclaw/commit/3361593442e295e9bb1f55cbfb38861ea9413e45)
- **Outbound hooks are invoked without sessionKey/sessionId in plugin context:** `applyMessageSendingHook()` passes only `{ channelId, accountId, conversationId }` to `runMessageSending`, and `createMessageSentEmitter()` builds the plugin hook context from the same reduced canonical fields. The session key is only used for internal `message:sent` hooks, not plugin hooks.
  - file: [src/infra/outbound/deliver.ts:522](https://github.com/openclaw/openclaw/blob/3361593442e295e9bb1f55cbfb38861ea9413e45/src/infra/outbound/deliver.ts#L522)

  - sha: [3361593442e2](https://github.com/openclaw/openclaw/commit/3361593442e295e9bb1f55cbfb38861ea9413e45)
- **Current outbound delivery still applies message_sending before chunking and emits one message_sent after chunked delivery:** `applyMessageSendingHook()` runs once per normalized payload before `sendTextChunks()`. After chunked delivery, `emitMessageSent()` is called once with the payload text, not once per actual chunk/send attempt.
  - file: [src/infra/outbound/deliver.ts:779](https://github.com/openclaw/openclaw/blob/3361593442e295e9bb1f55cbfb38861ea9413e45/src/infra/outbound/deliver.ts#L779)

  - file: [src/infra/outbound/deliver.test.ts:810](https://github.com/openclaw/openclaw/blob/3361593442e295e9bb1f55cbfb38861ea9413e45/src/infra/outbound/deliver.test.ts#L810)
