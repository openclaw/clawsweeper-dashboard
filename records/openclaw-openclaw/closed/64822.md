## Summary

Current `main` no longer matches the failure mode described in PR #64822: dev-channel update preflight now evaluates candidate commits inside a temporary worktree instead of mutating the main checkout, while `.bundle.hash` remains intentionally tracked in the repository and current maintainer guidance.

## Evidence
