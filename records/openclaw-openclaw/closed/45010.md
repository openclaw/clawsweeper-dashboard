## Summary

Keep open. Current main still preserves Bedrock/Anthropic thinking blocks during replay and does not show a targeted fix that strips invalid or empty `thinkingSignature` values before replay. The existing transport fallback only converts non-redacted empty-signature thinking blocks to text at send time, which is narrower than the issue report and narrower than the proposed replay-sanitization fix.
