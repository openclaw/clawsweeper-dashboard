## Summary

This is a third-party provider integration request that current OpenClaw scope routes to external provider plugins published on ClawHub or npm, not new core-bundled support. The repo already documents provider-plugin primitives for API-key auth, OpenAI-compatible catalogs, and onboard/model-ref UX, and current main has no built-in `clawapi` integration.

## Evidence
