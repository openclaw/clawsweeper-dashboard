## Summary

Keep open. Current `main` still builds the gateway service PATH from hard-coded common user/version-manager directories and does not include `NIX_PROFILES` or `~/.nix-profile/bin`, so this PR is neither implemented on `main` nor obsolete. The linked review thread also still leaves maintainership questions about Nix edge cases, so there is not strong evidence for a conservative close.
