## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Apply reset mode without cache-eligibility gating:** `src/agents/pi-embedded-runner/cache-ttl.ts:170-172`
  - body: `resolveCacheTtlMs` returns `null` before reading the configured mode whenever the provider is not prompt-cache eligible, and the sweep skips maintenance when the TTL is null. That makes `timeBasedContextCompact: "reset"` silently do nothing for non-cache providers even though reset only needs idle timestamps.
  - confidence: 0.95
- **[P1] Use the newest activity timestamp for expiry:** `src/gateway/session-cache-maintenance.ts:53-57`
  - body: `resolveCacheTouchAt` always prefers `lastCacheTouchAt` over `lastAssistantMessageAt`. If a session has an older cache touch and a newer assistant reply, the sweep can treat it as expired and reset or compact a session that is not actually idle past the configured window.
  - confidence: 0.91
- **[P2] Deduplicate aliases before maintenance actions:** `src/gateway/session-cache-maintenance.ts:311`
  - body: The sweep iterates the raw session-store entries and acts on each key from the same snapshot. If canonical and legacy aliases point at the same session, one sweep can run reset or compaction more than once for the same underlying session.
  - confidence: 0.86
- **[P2] Fix the Moonshot eligibility mock:** `src/agents/pi-embedded-runner/cache-ttl.test.ts:10-11`
  - body: The mock still returns `true` for native Moonshot while the changed expectations assert `false`, so the targeted cache-TTL test file cannot pass as written and the PR validation claim is contradicted by the source.
  - confidence: 0.98
- **[P2] Record user activity only for model turns:** `src/agents/agent-command.ts:540`
  - body: This block marks an explicit command override as user activity before any assistant/model activity is guaranteed. Command-only paths can leave `lastUserMessageAt` newer than `lastAssistantMessageAt`, causing idle maintenance to believe the session is not awaiting the user and skip later reset/compact work.
  - confidence: 0.78
