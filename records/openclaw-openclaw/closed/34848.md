## Summary

Issue #34848 is a specific Simplified Chinese localization request and is explicitly covered by the maintainer-owned canonical i18n/localization tracker #3460, which lists zh-CN and says specific-language localization issues should be closed as duplicates. Current main has Control UI zh-CN support, but not native macOS `.lproj` localization, so this is not implemented-on-main.
