## Summary

Keep open. Current `main` does not implement this Feishu fix, and the proposed PR change is likely still incomplete for the split-tag case it targets. On `main`, Feishu preview streaming still forwards partial snapshots directly, upstream tag stripping only recognizes complete `<think...>` tags, and Feishu's snapshot merge logic keeps the older text when a later snapshot becomes shorter, which is exactly how a leaked `<thi` prefix can stick on screen.

## Evidence
