## Summary

Current main already contains shipped fixes for the reported GPT-5/Codex behavior: unconfigured OpenAI-family GPT-5 runs now auto-enter the strict no-stall execution contract, plan-only turns are retried with an act-now steer instead of silently ending, and OpenAI/Codex tool schemas are normalized so parameter-free tools do not degrade into plain-text non-tool behavior. The changelog shows these fixes shipped in 2026.4.10.

## Evidence
