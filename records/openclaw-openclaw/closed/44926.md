## Summary

Current main implements the Telegram forum-topic routing needed by #44926: message sends accept explicit thread IDs, preserve auto-resolved thread IDs through outbound dispatch, parse topic-qualified Telegram targets, and pass Telegram `message_thread_id` into sends. The remaining human-readable topic-name-to-ID lookup concern from E122 is narrower and already tracked separately in #43231.
