## Summary

Keep open. Current `main` still builds exec-event prompts with the user-facing wording the report calls out, still formats async exec completions as `System`/`System (untrusted)` lines, still injects queued system events into the user prompt path, and the Control UI history filter does not hide generated system-wrapper user messages.
