## Summary

Close as `cannot_reproduce`: on current `main`, the bundled Amazon Bedrock path does not construct the reported legacy `{ type: "text", text: ... }` Converse payload shape. The OpenClaw-owned Bedrock wrapper only adds `cacheRetention`, optional `guardrailConfig`, and Bedrock `cachePoint` markers, and current tests assert tagged-union-style `{ text: ... }` content for this path.
