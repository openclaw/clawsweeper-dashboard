## Summary

Close as superseded: the linked empty-history ACP issue is already closed as implemented on main, the remaining startup-reconcile warning is tracked by https://github.com/openclaw/openclaw/issues/72013, and this PR’s specific merge semantics can regress fresher resolved session IDs.

## What This Changes

The PR changes ACP ensure-derived session identities to become resolved when `backendSessionId` or `agentSessionId` is present, adds focused identity tests, and updates an ACP spawn fixture.

## Best Possible Solution

Leave current merge semantics intact, keep https://github.com/openclaw/openclaw/issues/72013 as the canonical startup-warning fix, and avoid merging an ensure-state shortcut that can regress session IDs.

## Reproduction Assessment

Yes. Source inspection shows current ensure identities are pending, startup reconciliation targets pending identities with stable IDs, and existing tests model stale ensure IDs being corrected by fresher status IDs; I did not run live ACP because this was a read-only review.

## Solution Assessment

No. The narrow maintainable direction is to preserve status/event authority and fix the startup-warning path in https://github.com/openclaw/openclaw/issues/72013; this PR makes ensure IDs authoritative too early.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Preserve resolved IDs when merging ensure results:** `src/acp/runtime/session-identity.ts:160`
  - body: With this change, `createIdentityFromEnsure()` returns a resolved identity as soon as `ensureSession()` exposes IDs. `ensureRuntimeHandle()` then feeds that identity into `mergeSessionIdentity()`, where resolved incoming values can replace an already-resolved current identity; if the ensure response is older and status refresh is skipped or fails, fresher persisted `acpxSessionId`/`agentSessionId` values can be regressed. Keep ensure-sourced IDs non-authoritative or make the merge source-aware before marking them resolved.
  - confidence: 0.91
