## Summary

Current `main` already isolates loopback browser-origin auth throttling by normalized origin, preserves the `browser-origin:` namespace through rate-limit normalization, and includes regression tests for independent browser-origin buckets. The same implementation is present in released tag `v2026.4.23`, so this PR is redundant on current OpenClaw.

## Evidence
