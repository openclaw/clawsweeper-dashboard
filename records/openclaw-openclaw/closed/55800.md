## Summary

Current `main` already implements the Windows symlink-fallback behavior this PR proposes, and it does so in a broader form than the PR patch. The runtime staging script now falls back to copying the source path on Windows symlink failures, so there is no distinct change left to merge from #55800.

## Evidence
