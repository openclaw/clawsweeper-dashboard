## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.94

Full review comments:

- **[P1] Preserve backticks inside quoted fenced code blocks:** `extensions/slack/src/format.ts:158`
  - body: This regex runs after rendering and treats any paired backticks on a quoted line as inline-code delimiters. A blockquoted fenced code block containing literal backticks, such as JavaScript template strings, would be rewritten into bold markers and corrupt code that should stay verbatim. Move the workaround to the IR/style-aware layer or explicitly exclude code-block ranges, and add a regression for this case.
  - confidence: 0.96
