## What This Changes

The issue requests that Active Memory stop discarding successful verbose sub-agent memory summaries as unavailable and produce clearer diagnostics when recall parsing or embedded-runner handling fails.

## Best Possible Solution

Add focused regression coverage for a successful but verbose Active Memory sub-agent response and adjust the active-memory or embedded-runner path so non-empty assistant text is normalized and truncated before unavailable classification, while preserving explicit diagnostics for true memory-tool unavailability.
