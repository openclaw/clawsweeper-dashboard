## Summary

Current main already contains the Control UI/webchat protections that address disappearing assistant replies: oversized markdown now falls back to sanitized plain-text rendering instead of disappearing, and chat final-message handling preserves assistant replies even when the final payload is sparse or text-only. The shipped `v2026.4.22` tag contains these code paths and regression coverage.

## Evidence
