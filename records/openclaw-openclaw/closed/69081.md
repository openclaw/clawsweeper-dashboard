Keep open. Current main still arms the TUI streaming watchdog only from chat deltas, active-run tool and lifecycle events do not refresh watchdog liveness, reconnect reloads history without a reconnect-specific watchdog/state resync, and the issue is now paired with open draft PR #74224 using GitHub closing syntax for this issue.

Keep this issue open and review/land #74224 or an equivalent narrow TUI fix.
