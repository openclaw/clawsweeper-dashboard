## Summary

Current `main` already supports the described cost-isolation workflow through agent-scoped auth profiles plus cron job agent targeting. There is not a dedicated per-cron `--auth-profile` flag, but the requested per-agent override path exists and cron runs resolve credentials from the selected agent’s auth store.
