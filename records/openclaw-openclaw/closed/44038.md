## Summary

Keep open. Current `main` has native Windows install and onboarding improvements, but it does not implement the requested standalone Windows GUI quick installer. This also does not fit the `clawhub` close bucket, because ClawHub is scoped to skills/plugins, while the proposal is an external installer/app. The repo vision is still terminal-first, but it explicitly keeps setup reliability/first-run UX and Windows companion apps in scope, so this remains a product-direction request rather than a conservatively closable item.
