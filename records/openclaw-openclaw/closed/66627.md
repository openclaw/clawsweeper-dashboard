## Summary

Keep open. Current `main` still conflates Windows unknown-listener fallback PIDs with actionable stale gateway PIDs, and the restart/retry paths still act on that combined list. PR #66627 addresses a real, still-unfixed behavior gap rather than something already implemented or obsolete.
