## Summary

Close as implemented on current `main`: isolated cron runs now resolve per-job `payload.model` ahead of stored session/default model selection, pass that resolved model into the actual execution path, and include explicit regression tests for the reported override bug. The fix appears to be on `main` after the latest tagged release, so no shipped release tag could be confirmed from this checkout.
