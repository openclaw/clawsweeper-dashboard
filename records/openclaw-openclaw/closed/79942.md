Keep open. The PR has blocking review findings: its unrelated git installer change breaks valid ref checkouts, and the zh-CN Gateway glossary edit conflicts with the current docs-i18n prompt.

## What This Changes

The branch updates zh-CN and zh-TW docs glossary JSON entries and also changes git plugin ref checkout/tests to insert an option separator before the requested ref.

## Best Possible Solution

Split or drop the git installer change in favor of https://github.com/openclaw/openclaw/pull/79901, then merge a docs-only glossary update after the Gateway terminology policy is made consistent across the glossary and prompt.

## Reproduction Assessment

Yes. Current main source shows the reported glossary values, and an isolated Git check reproduces that the PR's checkout argv rejects a normal tag ref.

## Solution Assessment

No. The git installer fix should follow the already-open git switch direction in https://github.com/openclaw/openclaw/pull/79901, and the glossary change should not conflict with the docs-i18n prompt.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.95

Full review comments:

- **[P2] Use a checkout form that accepts valid refs:** `src/plugins/git-install.ts:290`
  - body: This changes the argv to `git checkout --detach -- <ref>`, but real Git treats the token after `--` as a pathspec in checkout mode; `git checkout --detach -- v1.2.3` exits 128, so every `git:<repo>@<tag>` install with an explicit ref would fail. Use the `git switch --detach -- <ref>` direction from https://github.com/openclaw/openclaw/pull/79901 or remove this unrelated change.
  - confidence: 0.98
- **[P2] Align the Gateway terminology sources:** `docs/.i18n/glossary.zh-CN.json:84`
  - body: The docs-i18n prompt still mandates `Gateway 网关` for OpenClaw Gateway, while this changes the zh-CN glossary target to `网关`. Those are both mandatory guidance for generated docs, so the PR needs a deliberate terminology update to the prompt too, or the glossary should stay aligned.
  - confidence: 0.9
