## Summary

Current `main` no longer uses the legacy node shell wrapper described in the report. Node shell execution is now routed through `exec host=node`, the `nodes` CLI explicitly blocks `system.run`/`system.run.prepare`, and headless-node approvals are documented and implemented as local `~/.openclaw/exec-approvals.json` policy rather than requiring a Linux approval socket daemon.

## Evidence
