## Summary

Current main already rebuilds agent FS tools from config on each embedded run and explicitly allows host read/write/edit outside the agent workspace when `tools.fs.workspaceOnly` is `false`; the regression now has direct coverage on main. The fixing change appears on main in commit `51186d2725439646f7fea0e59bb466f957404c46`, which landed after the latest tagged release `v2026.4.22`, so this looks fixed on main but not yet released.
