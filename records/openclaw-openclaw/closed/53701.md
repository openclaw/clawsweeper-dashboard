Close #53701 as a duplicate of the older open #7406. Current main still has only partial Telegram topic-name plumbing, so the remaining work should stay tracked by #7406 rather than keeping multiple issues open for the same Control UI session-label request.

## Best Possible Solution

Close #53701 as a duplicate and keep #7406 as the canonical tracker.
