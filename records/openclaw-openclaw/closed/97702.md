## Best Possible Solution

Land the narrow helper swap after normal CI completion, keeping WHAM JSON parsing on the shared bounded provider response helper.

## Solution Assessment

Yes. Reusing the existing bounded provider JSON reader at the single successful WHAM parse is the narrowest maintainable fix and preserves the existing probe-failure fallback behavior.
