## Summary

Current main already ships first-class browser automation as the bundled `browser` plugin, enabled by default, with Playwright-backed runtime support. The issue's stated current-state assumptions are obsolete; the remaining manual Docker Chromium install step is environment-specific setup, not absence of a default browser capability.
