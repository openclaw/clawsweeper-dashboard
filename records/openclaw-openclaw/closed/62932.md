## Summary

Current `main` already contains a model-agnostic fix for post-tool turns that end with no visible assistant text: OpenClaw now retries silent error completions and, if they still fail, surfaces an explicit user-facing error instead of ending silently. The same implementation is present in release `v2026.4.23`, which is newer than the reporter's `2026.4.7` version.

## Evidence
