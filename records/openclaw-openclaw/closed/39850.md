## Summary

Current `main` already supersedes this PR with a Kimi-owned replay-policy path. `resolveTranscriptPolicy` now prefers plugin replay policy over core transport fallbacks, the bundled Kimi plugin registers `buildReplayPolicy`, and that policy disables signature preservation for Kimi. The transcript-policy test suite also carries an explicit `#39798` Kimi case. This makes the PR's one-line core fallback change obsolete.
