## Summary

The requested simplified external-notify API already exists. OpenClaw exposes authenticated HTTP hook ingress with `POST /hooks/agent`, which runs an agent turn from an external service using a bearer token, optional `agentId`, and guarded `sessionKey` routing. The endpoint is implemented on current `main`, documented, tested, and already present in release `v2026.4.22`.

## Evidence

- **Current main implements HTTP agent hook ingress:** `server-http.ts` handles `POST /hooks/agent`, validates the payload, resolves optional `sessionKey`, applies allowlists/prefix checks, dispatches the agent run, and returns `{ ok: true, runId }`.
  - file: [src/gateway/server-http.ts:664](https://github.com/openclaw/openclaw/blob/bae7b54a85b17d180e4bfc28973a7e4fd1646a93/src/gateway/server-http.ts#L664)
