## Summary

Current `main` still intentionally supports OpenAI/Codex GPT-5.4-family models and OpenAI `service_tier` handling, with follow-up fixes layered on top of the original GPT-5.4 work. This March 2026 revert no longer matches current `main` and would now remove supported behavior rather than fix a reproducible regression.
