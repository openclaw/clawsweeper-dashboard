## Summary

Current `main` already fixes this: the memory host session scan now includes usage-counted archived transcripts (`.jsonl.reset.*` and `.jsonl.deleted.*`) instead of only bare `.jsonl` files, and the fix is covered by a direct regression test and shipped in `v2026.4.23`.

## Evidence
