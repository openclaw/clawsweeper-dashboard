Current main implements the Control UI per-message metadata path end to end: assistant usage is captured from streaming completion events and message_end, chat.history preserves sanitized assistant usage/cost metadata, and the grouped chat renderer displays token/cache/cost/context/model metadata when that gateway metadata is present.
