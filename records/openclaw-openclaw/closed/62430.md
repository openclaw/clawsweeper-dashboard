## Summary

Current main has the provider-hook reentrancy guard and regression coverage that address the 2026.4.5 Windows Google/MiniMax stack-overflow path, and the fix is recorded in the 2026.4.7 changelog. The TUI startup path also does not request eager plugin loading by default.
