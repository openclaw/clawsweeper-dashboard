## Summary

Current `main` already contains an unreleased fix for same-session heartbeat interruption: `runHeartbeatOnce` now skips heartbeat execution when either the main lane or the resolved session lane is busy, and the wake layer retries instead of interrupting the active turn.

## Evidence

- **Heartbeat now skips a busy target session lane:** `runHeartbeatOnce` checks the resolved session lane and returns `skipped: requests-in-flight` specifically to avoid interrupting an active streaming turn.
