The issue requests wiring the documented Plugin SDK `before_agent_finalize` hook into normal OpenClaw agent finalization, not just native relay harnesses.

## Best Possible Solution

Wire the existing `runAgentHarnessBeforeAgentFinalizeHook` contract into the normal embedded OpenClaw final-answer path, preserving native relay behavior and adding focused tests for continue, finalize, and revise outcomes.
