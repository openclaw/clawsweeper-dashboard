## What This Changes

This PR adds a ClawDock helper to read and normalize `OPENCLAW_GATEWAY_PORT` from `.env`, rewrites `clawdock-dashboard` URLs to `localhost:<host_port>`, and adds a changelog entry.

## Best Possible Solution

Keep the ClawDock URL rewrite, make the port parser shell-portable and aligned with `parseGatewayPortEnvValue`, then require after-fix `clawdock-dashboard` output for a remapped port before merge.

## Solution Assessment
