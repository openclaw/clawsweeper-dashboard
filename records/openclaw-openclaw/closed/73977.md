## Summary

Close as duplicate/superseded. The reported ACPX/Codex auth regression is real on current main, but the same remaining work is already tracked by open canonical issue #73910 and open fix PR #73943. Current main still forces an isolated Codex ACP CODEX_HOME and tests still assert source Codex auth is not copied, so this should not be closed as implemented.
