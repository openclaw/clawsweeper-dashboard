## Summary

Keep open. Current `main` has landed a narrower OTEL lifecycle-span subset, but this PR still carries materially different scope: GenAI-style inference/content capture work, outbound trace-context propagation, and a different event contract. The branch is not implemented on `main`, and the PR also still conflicts with current compatibility expectations around `model.usage` listeners.
