## Summary

Current `main` already contains the loopback-gateway diagnostic fixes described in #46100: authenticated loopback probes keep device identity, status reporting separates reachability from read capability, and the shipped changelog for `v2026.4.22` explicitly says scope-limited sessions no longer appear fully healthy.
