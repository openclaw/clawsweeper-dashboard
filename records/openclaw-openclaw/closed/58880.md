## Summary

Current `main` already handles the reported OpenRouter mandatory-reasoning failure: it strips proxy reasoning fields when thinking is off, has regression coverage for that behavior, and retries reasoning-required errors with `minimal` instead of surfacing the 400. The same implementation is present in released `v2026.4.23`.

## Evidence
