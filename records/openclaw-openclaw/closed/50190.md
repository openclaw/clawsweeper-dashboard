## Summary

Current `main` already supports assistant image/video output in the Control UI via `MEDIA:` attachments, and the feature shipped in `v2026.4.11` via PR `#64104` (`cc5c691f0069f1d23ce0cde6cd258f5027d40957`). Image and video generation tools emit `MEDIA:` lines, the chat normalizer converts them into structured attachments, and the renderer displays images inline and videos in a playable card.
