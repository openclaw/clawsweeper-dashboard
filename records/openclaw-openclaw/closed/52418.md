## Summary

#52418 asks OpenClaw core to add Feishu Task API CRUD/collaborator/due-date tools. Current main does not implement Feishu task tools, but this is an optional service-specific Feishu surface that fits the existing plugin tool API and ClawHub/npm distribution path. The bundled @openclaw/feishu plugin already covers chat/docs/wiki/drive/permission/bitable, and there is no protected label, maintainer author association, or discussion showing a missing core plugin API seam.

## Best Possible Solution

Close the core issue and direct Feishu Task API support to a ClawHub/npm plugin that registers Feishu task tools with api.registerTool. If implementation discovers a concrete missing OpenClaw plugin API seam, open a narrow SDK/API issue for that blocker rather than keeping this service-specific core feature request open.
