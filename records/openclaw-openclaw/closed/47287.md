## Summary

Current `main` already prevents a workspace skill that shares a bundled skill name from being reported as bundled, and it includes a regression test for that exact case. This PR's user-visible fix is therefore already present on `main`, even though the implementation retains a legacy `unknown`-source compatibility fallback.
