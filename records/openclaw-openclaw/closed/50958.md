## Summary

Current `main` already implements inline image rendering in webchat for structured image blocks, assistant `MEDIA:` attachments, transcript media history, and managed generated images. The changelog shows the webchat media-bubble work shipped in `v2026.4.10`, with a later follow-up in `v2026.4.22` to preserve generated images across history reloads.
