## Summary

Keep open. Current `main` still scopes Telegram forum topics to their own sessions, but the active model context remains bounded by the model context window and compaction, and Telegram-specific extra history injection is still only pending mention-gated group history. I found no implementation of the requested safety signals or fallback history fetches, so there is not strong evidence that issue #37301 is already fixed or obsolete.
