## Summary

Current main already enforces startup context well enough to cover this request: workspace bootstrap files are injected into Project Context on normal runs, bare `/new` and `/reset` runs prepend recent daily memory plus an explicit Session Startup instruction before the first reply, and `MEMORY.md` is filtered out of subagent/cron bootstrap injection.
