## Summary

Current `main` already implements Chinese Control UI localization, including `zh-CN`/`zh-TW` locale support, a persisted language picker in the Overview dashboard, shipped Chinese locale bundles, docs, and regression tests. The changelog shows this shipped in release `v2026.4.5`.

## Evidence

- **Chinese locales are part of the supported Control UI locale contract:** The UI i18n type includes both `zh-CN` and `zh-TW` as first-class supported locales.
  - file: [ui/src/i18n/lib/types.ts:3](https://github.com/openclaw/openclaw/blob/350d3221800a18283915cd2f3b612ae04ddd490c/ui/src/i18n/lib/types.ts#L3)
  - sha: [350d3221800a](https://github.com/openclaw/openclaw/commit/350d3221800a18283915cd2f3b612ae04ddd490c)
- **Runtime locale loader includes Simplified and Traditional Chinese:** The locale registry lazy-loads both Chinese bundles and maps browser `zh*` locales to `zh-CN`/`zh-TW`.
  - file: [ui/src/i18n/lib/registry.ts:13](https://github.com/openclaw/openclaw/blob/350d3221800a18283915cd2f3b612ae04ddd490c/ui/src/i18n/lib/registry.ts#L13)
  - sha: [350d3221800a](https://github.com/openclaw/openclaw/commit/350d3221800a18283915cd2f3b612ae04ddd490c)
- **Dashboard already exposes a language selector:** The Overview -> Gateway Access card renders a `Language` dropdown, applies `i18n.setLocale(...)`, and persists the chosen locale in settings.
  - file: [ui/src/ui/views/overview.ts:252](https://github.com/openclaw/openclaw/blob/350d3221800a18283915cd2f3b612ae04ddd490c/ui/src/ui/views/overview.ts#L252)
  - sha: [350d3221800a](https://github.com/openclaw/openclaw/commit/350d3221800a18283915cd2f3b612ae04ddd490c)
- **Chinese translations are present in shipped locale bundles:** Both Simplified and Traditional Chinese locale files exist with translated UI strings and language labels.
  - file: [ui/src/i18n/locales/zh-CN.ts:4](https://github.com/openclaw/openclaw/blob/350d3221800a18283915cd2f3b612ae04ddd490c/ui/src/i18n/locales/zh-CN.ts#L4)
  - sha: [350d3221800a](https://github.com/openclaw/openclaw/commit/350d3221800a18283915cd2f3b612ae04ddd490c)
- **Tests cover Chinese locale behavior:** Regression tests verify loading saved `zh-CN` at startup and keeping the Chinese locale selected in the dashboard language dropdown.
  - file: [ui/src/i18n/test/translate.test.ts:62](https://github.com/openclaw/openclaw/blob/350d3221800a18283915cd2f3b612ae04ddd490c/ui/src/i18n/test/translate.test.ts#L62)
  - sha: [350d3221800a](https://github.com/openclaw/openclaw/commit/350d3221800a18283915cd2f3b612ae04ddd490c)
