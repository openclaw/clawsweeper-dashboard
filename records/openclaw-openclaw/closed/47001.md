## Summary

The report is about `clawhub install` behavior and China-specific mirror/proxy handling for the ClawHub registry flow, not an OpenClaw core bug. Current `main` documents native OpenClaw skill installs via `openclaw skills install`, and the in-repo install path talks to ClawHub archive/search endpoints directly rather than cloning from GitHub.
