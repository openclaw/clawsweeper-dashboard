## What This Changes

The issue reports a 2026.5.19 regression where `before_prompt_build` hook `prependContext` was routed into system prompt space instead of user/model prompt space.

## Best Possible Solution

Keep the shipped prompt-boundary implementation and close this report as fixed by the merged replacement PR.

## Reproduction Assessment

Yes. The merged fix separates transcript/session prompt text from model prompt text, which is the narrow maintainable boundary for keeping hook context model-visible without persisting it as user text.
