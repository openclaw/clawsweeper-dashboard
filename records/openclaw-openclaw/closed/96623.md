## Summary

Current main and shipped npm artifacts already satisfy the requested external-install path for Feishu: the root package intentionally excludes bundled Feishu runtime, but ships the official external channel catalog, and the Feishu npm package contains compiled runtime output.

## What This Changes

The issue reports that Feishu is excluded from the root npm tarball and allegedly unavailable as an official external channel plugin for npm-installed users.

## Best Possible Solution

Keep Feishu externalized and rely on the shipped official catalog plus @openclaw/feishu npm package rather than rebundling Feishu into the root package.

## Reproduction Assessment

No. The reported 2026.5.26 npm artifacts already include the external channel catalog entry and a published @openclaw/feishu package with compiled runtime output.

## Solution Assessment

Yes. The external npm plugin path is the narrowest maintainable solution because OpenClaw policy prefers optional channel runtime ownership in plugins instead of root package bundling.
