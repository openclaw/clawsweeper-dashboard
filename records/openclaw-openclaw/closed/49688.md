## Summary

The PR adds Octen as a new bundled web-search plugin, but current `main` explicitly pushes new optional/commercial integrations to external plugins distributed through ClawHub or npm. OpenClaw already documents web-search provider hooks and external plugin install/publish flows, so this belongs in ClawHub rather than core.

## Evidence
