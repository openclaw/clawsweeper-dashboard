## What This Changes

The issue reports that the macOS curl installer fails during global npm installation when npm cannot write to `/usr/local/lib/node_modules/openclaw`.

## Best Possible Solution

Extend the default installer's npm-prefix handling to cover unwritable macOS global prefixes, or fail with clear no-root guidance, while preserving the separate `install-cli.sh` local-prefix path.
