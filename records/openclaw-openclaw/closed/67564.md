## Summary

Issue #67564 reports a failure inside the external `@tencent-weixin/openclaw-weixin` channel plugin’s long-poll monitor, and current `main` documents WeChat/Weixin as plugin-owned runtime outside the OpenClaw core repo. The issue timeline already points to the plugin-side tracker at Tencent/openclaw-weixin#75, so the core issue can be conservatively closed as out-of-scope for OpenClaw core.

## Evidence
