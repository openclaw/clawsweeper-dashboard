## Summary

Close #50453 as a duplicate of the broader open #62328. Current main still supports Node >=22.14, still uses node:sqlite for the builtin memory database, still creates keyword search with SQLite FTS5, and still degrades to vector-only or empty results when FTS is unavailable. The remaining Windows/node:sqlite FTS5 fallback work should stay centralized on #62328.
