## Summary

Current `main` and the latest shipped release already add the config-recovery path this issue asks for: suspicious clobbered reads now preserve a `.clobbered.*` copy and restore `openclaw.json.bak` when the config loses metadata, loses `gateway.mode`, or shrinks sharply. The reporter is on `2026.4.5`; the shipped fix is documented in `v2026.4.23`.
