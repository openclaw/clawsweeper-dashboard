## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Yes; this is the narrow wrapper-owned fix for the observed failure mode, and it preserves the sibling slug and owned-key paths. A future upstream Crabbox validation hook could be cleaner, but it is not required for this focused OpenClaw wrapper guard.
