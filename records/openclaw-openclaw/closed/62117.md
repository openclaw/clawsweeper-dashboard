## Summary

Current `main` already includes the Microsoft speech plugin, capability-provider compatibility loading, and regression coverage for config-backed Microsoft TTS registration. The changelog shows the bundled Microsoft speech provider auto-enable fix shipped in `v2026.4.15`, so this 2026.4.5 regression is already resolved on main.
