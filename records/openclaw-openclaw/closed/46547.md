## Summary

Current `main` already implements the requested behavior: OpenClaw keeps its managed `web_search` tool available for OpenRouter xAI/Grok models, and the Perplexity web-search provider supports the OpenRouter/Sonar compatibility path via `OPENROUTER_API_KEY` or explicit `baseUrl`/`model` overrides.
