## Summary

Current main already includes a dedicated multi-agent setup flow. Users no longer need to hand-edit `agents.list`/`bindings`: `openclaw agents add` creates the agent entry, workspace, and agentDir, and the wizard can configure auth/channels and apply bindings. The remaining separate-workspace requirement is still the documented isolation model for multi-agent setups, not a missing configuration function.
