## Summary

Current `main` already includes the requested prerelease core-workflow QA framework: repo-backed QA scenarios for memory/model/subagent/workspace flows, release-check workflows that run install smoke plus QA/live/E2E lanes before release approval, and docs for extending the suite. The main implementation landed in `ac063568d354c14e0a0a43bcffb9725bb3fca651` after this issue was filed and has not yet shipped in a tagged release.
