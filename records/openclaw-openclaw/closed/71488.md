## Summary

Keep open. Current `main` still strips LM Studio `@iq*` suffixes in `splitTrailingAuthProfile`, and there is no matching regression coverage on `main`, so this PR appears to address a real unfixed bug rather than an item suitable for cleanup closing.

## Evidence
