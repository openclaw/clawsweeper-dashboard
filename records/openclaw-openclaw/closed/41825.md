## Summary

Current main already contains explicit protections for this routing bug class: heartbeat/cron/exec turns no longer overwrite shared-session delivery metadata, and reply delivery prefers the live turn source over stale session routing when multiple channels share one session. The behavior is covered by regression tests and was shipped in the 2026.4.14 release notes.
