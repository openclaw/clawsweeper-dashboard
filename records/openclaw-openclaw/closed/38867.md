## Summary

Current main no longer matches the report as filed. The Brave provider still sets the auth header, Node's fetch stack normalizes it to lowercase on the actual request, and current main now hardens the known Brave 422 paths by normalizing unsupported country values and rejecting invalid language codes before the upstream call.
