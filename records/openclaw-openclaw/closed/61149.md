## Summary

Current `main` already routes MiniMax image generation to the dedicated `https://api.minimax.io/v1/image_generation` endpoint, with regression coverage for both default and `/anthropic`-configured base URLs. The same implementation is present in release `v2026.4.23`, so this issue is already fixed well enough to close.
