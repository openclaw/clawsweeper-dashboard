## Summary

Current `main` routes Dream Diary narratives into `DREAMS.md`/`dreams.md`, suppresses subagent delivery to the terminal, and has explicit managed-cron regression coverage for that path. The same write path is already present in the shipped `v2026.4.23` release tree, so the titled stdout-vs-`DREAMS.md` bug is obsolete against current supported builds.

## Evidence
