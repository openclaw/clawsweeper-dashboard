## Summary

Current main already honors `OPENCLAW_STATE_DIR` for state resolution, and the Windows-specific gateway/service path helpers explicitly preserve Windows absolute paths. The latest release snapshot `v2026.4.23` contains the same production code in this area, and the issue timeline already links the closed Windows follow-up PRs.
