## Summary

Keep this PR open. Current main still has the beta-header behavior this PR is trying to remove, the active Anthropic implementation now lives in the bundled plugin path rather than the PR's old production file path, and docs/tests still assert beta-gated behavior. It is also paired with the same-author open tracking issue #45550, with #45613 as another open implementation candidate, so maintainer consolidation is better than cleanup closure.

## Best Possible Solution

Keep this PR open for maintainer consolidation. The best path is to rebase or replace it against `extensions/anthropic/stream-wrappers.ts`, update `extensions/anthropic/stream-wrappers.test.ts`, `src/agents/pi-embedded-runner-extraparams.test.ts`, and the Anthropic/token-use docs, preserve `context1m` context-window sizing in `src/agents/context.ts`, and make an explicit compatibility decision for legacy configured `context-1m-2025-08-07` beta headers under OAuth auth. Maintainers should then decide whether #45613 is the canonical implementation for #45550.
