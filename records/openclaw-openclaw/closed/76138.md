Keep open. Current main and the latest shipped release still expose the composer-toolbar plus-labelled New session control and route one click directly into fresh session creation; the older in-place transcript-loss risk is mitigated, but the reported ambiguous, send-adjacent affordance remains unresolved.

The issue requests a safer Control UI New Session affordance through confirmation, icon or placement changes, or an optional hide setting.

## Best Possible Solution
