## Summary

Current `main` already implements Anthropic Claude 4.6 adaptive thinking and effort mapping. The Anthropic provider advertises `adaptive` for Claude 4.6 and defaults that family to `adaptive`, the Anthropic transport emits `thinking: { type: "adaptive" }` plus `output_config.effort`, and regression tests cover the Opus 4.6 payload shape.

## Evidence

- **Anthropic provider exposes adaptive for Claude 4.6 and defaults to it:** The provider thinking profile for Anthropic includes `off/minimal/low/medium/high` and adds `adaptive` for Claude 4.6; its default level for Claude 4.6 is `adaptive`. The provider test asserts `claude-opus-4-6` contains `adaptive` and defaults to `adaptive`.

  - file: [extensions/anthropic/index.test.ts:239](https://github.com/openclaw/openclaw/blob/754acc4478fe361db1e5303fd60f589d52beea7a/extensions/anthropic/index.test.ts#L239)
