## Summary

Current main fixes the reported v2026.4.24 crash-loop path. The issue log exits on `CIAO PROBING CANCELLED` after the Bonjour watchdog restarts the advertiser; main now keeps ciao cancellation handlers registered across advertiser restarts and has tests/changelog/docs for that behavior.

## Best Possible Solution

Close this report as fixed on main. Keep the current bounded Bonjour behavior, ship the unreleased fix, and point affected v2026.4.24 users to the Bonjour troubleshooting docs or `OPENCLAW_DISABLE_BONJOUR=1` until they update.
