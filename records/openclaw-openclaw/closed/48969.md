## Summary

The reported root cause does not match current or latest-shipped code. OpenClaw already requests Discord's non-privileged `Guilds` gateway intent unconditionally; the `intents` config/probe surfaces only model privileged intents. The reported `channels unresolved: guild:...` message is emitted by allowlist/config resolution code that uses Discord REST lookups, not gateway intent negotiation.
