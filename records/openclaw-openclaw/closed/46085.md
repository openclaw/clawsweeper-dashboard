## Summary

Current main already avoids the reported false-error path for default/no-target heartbeats: unset heartbeat delivery resolves to `none`, cron/exec heartbeat prompts stay internal-only when delivery is disabled, and regression coverage asserts the run finishes successfully without outbound sends.
