## Summary

Current `main` already preserves the embedded session stream function and injects the resolved runtime API key into provider-owned/custom-provider stream functions, which covers the reported `codex-lb` auth-loss path. The same implementation is present in the shipped `v2026.4.23` release.

## Evidence
