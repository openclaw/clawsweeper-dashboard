## Summary

Current main already implements the requested post-compaction orientation path: OpenClaw re-injects configurable AGENTS.md sections as a post-compaction system event before the next turn, and the config surface documents `agents.defaults.compaction.postCompactionSections` for choosing that injected text. This is a stronger version of the requested `compaction.suffix`/post-system-message behavior.
