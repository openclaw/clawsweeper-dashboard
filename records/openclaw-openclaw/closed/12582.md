## Summary

Keep open. Current `main` has overlapping public plugin lifecycle coverage via `gateway_stop`, but it does not implement this PR’s internal `gateway:shutdown` / `gateway:pre-restart` events, related internal-hooks docs, or shutdown-event tests. That is not strong enough for an `implemented_on_main` close.
