## Summary

Keep open. Current `main` does not implement the requested `docs/runbooks/` structure, and the request is still a coherent docs feature rather than something we can confidently auto-close. The repo already has public docs for MiniMax, TTS, and Gateway runbooks, but the issue asks for a broader documentation reorganization plus external-system runbooks, which needs maintainer product/scope judgment rather than conservative cleanup.
