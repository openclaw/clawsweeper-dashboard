## What This Changes

## Best Possible Solution

Implement a doctor-owned Memory Core sidecar import into the existing per-agent DB, preserve cache/vector rows and non-memory tables, update fixtures/tests, and add real upgrade proof; otherwise let https://github.com/openclaw/openclaw/pull/95631 become the canonical fix after it is proof-positive.
