## Summary

Keep open. Current main still does not omit `dist/extensions/<plugin>/.openclaw-install-stage/**` or `dist/extensions/<plugin>/.openclaw-install-stage-*/**` from packaged dist inventory traversal, and global install verification consumes that traversal before reporting `unexpected packaged dist file ...` errors.

  - file: [src/infra/package-dist-inventory.ts:26](https://github.com/openclaw/openclaw/blob/dcfd5913fd24a7fce8fe804bf687f4d4856a6652/src/infra/package-dist-inventory.ts#L26)
