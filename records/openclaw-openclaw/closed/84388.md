## Best Possible Solution

## Solution Assessment

Mostly yes: failing closed by default is the conservative mitigation, and the exec-server bridge is a reasonable explicit opt-in. It is not the complete containment design if maintainers require the whole Codex app-server/orchestrator inside the sandbox.
