## Summary

Keep open. Current main still routes normal Feishu agent replies through a Feishu-specific dispatcher that sends directly via Feishu helpers, while the `message:sent` hook emission lives in the shared outbound delivery path that this dispatcher bypasses. The issue comment pointing at the Feishu dispatch path is consistent with the current code.
