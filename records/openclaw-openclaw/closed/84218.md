Make `heartbeat.isolatedSession=true` prevent prior heartbeat context-engine/session replay from entering later heartbeat prompts.

## Best Possible Solution

Review and land, or replace, the linked fix so isolated heartbeat runs keep stable delivery routing but use a fresh bounded context-engine identity with regression coverage.

## Solution Assessment

Yes. The narrow maintainable direction is to honor the existing docs by splitting fresh context-engine identity from stable heartbeat delivery routing, rather than adding a new config knob as the primary fix.
