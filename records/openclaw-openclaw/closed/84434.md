## Best Possible Solution

## Reproduction Assessment

## Solution Assessment

Yes for the bundled default path. Using Claude Code's deny-rule surface in the Anthropic plugin is narrower than core tool filtering; forcing the deny rule into custom user argv overrides would be a separate compatibility decision.
