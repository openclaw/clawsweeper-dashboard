## Summary

Current `main` already addresses this: compaction notices are silent by default, Telegram handles compaction via channel callbacks/status reactions instead of sending `🧹 Compacting context...` into chat, and the agent runner now bypasses the block-reply pipeline for the rare opt-in notice path so final replies are not dropped on non-streaming turns.
