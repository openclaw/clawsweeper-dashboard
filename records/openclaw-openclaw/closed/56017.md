## Summary

Current `main` already ships Simplified Chinese Control UI localization, including `zh-CN` registration, a language picker, a generated `zh-CN` locale bundle, and tests covering locale loading/alignment. The latest release `v2026.4.22` also contains the implementation, so this PR is superseded.
