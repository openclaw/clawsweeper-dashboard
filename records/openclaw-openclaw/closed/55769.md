## Summary

Current main already routes spawned-session/subagent replies through post-run `chat.history` reads, prefers phased `final_answer` text, and has regression coverage for empty-assistant fallback paths. The reported March 2026 behavior is implemented/fixed in shipped code and does not match current main.
