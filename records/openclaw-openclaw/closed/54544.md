## Summary

Current main already includes archived session transcripts in session-memory indexing. The memory host scans files accepted by the shared session artifact helper, which treats primary `.jsonl` files plus `.jsonl.reset.*` and `.jsonl.deleted.*` archives as searchable session transcripts while excluding `.jsonl.bak.*` backups. This behavior is covered by tests and documented in the `2026.4.5` changelog.
