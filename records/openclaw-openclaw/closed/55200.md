## Summary

Current `main` now has explicit MiniMax VLM routing for both `minimax` and `minimax-portal`, plus a fallback path that resolves provider auth even when `MiniMax-VL-01` is only manually configured or not present in the discovered text catalog. The issue matched an older gap and is implemented on `main` after the latest release.
