## Summary

Current `main` already implements the requested Feishu group-member lookup. The bundled Feishu plugin registers `feishu_chat`, its `members` action calls the Feishu chat-members API for a `chat_id`, the tool schema exposes the needed parameters, and tests cover the members flow. The same implementation is present in release `v2026.4.22`.
