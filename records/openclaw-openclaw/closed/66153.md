## Summary

Current main already guards against this case: WhatsApp inbound access control drops outbound `fromMe` DMs unless they are true same-phone self-chat, and the repo includes regression coverage for ignoring outbound DMs. The same implementation is present in the shipped `v2026.4.23` release.

## Evidence
