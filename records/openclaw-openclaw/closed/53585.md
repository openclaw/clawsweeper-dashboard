## Summary

Keep open: current `main` does not implement this PR's background-first/direct-notification rewrite. The bundled `coding-agent` skill now exists on `main`, but it still documents a bash-first workflow with optional backgrounding and uses `openclaw system event` plus heartbeat wake for completion, not explicit `openclaw message send` routing.
