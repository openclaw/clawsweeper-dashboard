## Summary

Current `main` and the shipped `v2026.4.23` tag already address both reported packaging problems: workspace templates are explicitly packed and resolved from the package root, and `memory-lancedb` now resolves `@lancedb/lancedb` from the bundled plugin manifest path used by flattened dist chunks.

## Evidence
