## Summary

Current `main` contains targeted gateway/UI changes and tests that address the reported lingering `pondering` state for active chat runs, and the codex warmup path now falls back to async model resolution before warning. The fix appears on `main` after `v2026.4.23`, so it is implemented but not tied to a shipped release tag in this checkout.
