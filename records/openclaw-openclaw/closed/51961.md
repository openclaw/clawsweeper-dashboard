## Summary

Current main already fixes the user-visible #51887 failure path through explicit read-scope probing and least-privilege gateway calls for `status --all`, `security audit --deep`, and `gateway status --require-rpc`. This PR's exact `GatewayClient` fallback-scope edit is not on main, but the reported status/deep-audit regression is already addressed well enough in shipped code.
