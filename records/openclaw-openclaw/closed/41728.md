## Summary

Current main already implements a persisted Control UI toggle that hides tool calls and tool results from chat, and the same wiring is present in the released v2026.4.22 tag.

## Evidence

- **Control UI toggle exists:** The chat controls compute `showToolCalls` from persisted settings and expose a toggle button that flips `chatShowToolCalls`; the mobile chat controls expose the same toggle.
  - file: [ui/src/ui/app-render.helpers.ts:185](https://github.com/openclaw/openclaw/blob/a53fea39050beba21ac5468d876c37323cfdd2e6/ui/src/ui/app-render.helpers.ts#L185)

- **Setting is persisted:** `UiSettings` includes `chatShowToolCalls`, defaults it to `true`, and reloads persisted values from browser storage.
  - file: [ui/src/ui/storage.ts:60](https://github.com/openclaw/openclaw/blob/a53fea39050beba21ac5468d876c37323cfdd2e6/ui/src/ui/storage.ts#L60)
  - sha: [a53fea39050b](https://github.com/openclaw/openclaw/commit/a53fea39050beba21ac5468d876c37323cfdd2e6)
