## Summary

Current `main` already handles this case: configured bundled channels are auto-enabled and, when `plugins.allow` is restrictive, the owning bundled plugin is appended to the allowlist before plugin loading. The repo also has regression coverage for Telegram specifically, including the exact `plugins.allow: ["browser"]` + configured Telegram scenario.

## Evidence
