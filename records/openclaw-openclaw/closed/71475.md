## Summary

Current `main` already fixes the Telegram tool-progress leak by suppressing separate default tool-progress messages and routing tool activity into sanitized draft-preview updates, with regression coverage that includes the Telegram preview path and forum-topic threading. The fix appears to have landed after `v2026.4.23`, so I could not tie it to a newer shipped release tag.

## Evidence
