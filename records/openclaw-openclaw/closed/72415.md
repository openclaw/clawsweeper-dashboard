## Summary

Current main fixes the observable #72415 failure. Bundled plugins loaded from an external runtime-deps stage now get a mirror-side `dist/extensions/node_modules/openclaw` package with `./plugin-sdk/*` exports, so Telegram/Discord-style staged files resolve `openclaw/plugin-sdk/*` without mutating the packaged install. Current main also anchors npm runtime-deps installs with an OpenClaw-owned package manifest, covering the related #72374 package-root failure mode.

## Best Possible Solution

Close #72415 as implemented. Keep the shipped mirror-side plugin SDK alias and the current package-root hardening; leave any new reproduction on current release/main to a fresh issue with current logs and stage-directory state.

## Evidence
