## What This Changes

The PR updates shared capability-provider discovery so enabled manifest-backed image, video, and music generation providers can be merged with an already-active generation provider, and adds a fal/xai regression plus changelog entry.

## Best Possible Solution

Land this focused shared resolver fix after normal exact-head merge checks, keeping the behavior generic to generation capability families rather than adding fal-specific handling.

## Solution Assessment
