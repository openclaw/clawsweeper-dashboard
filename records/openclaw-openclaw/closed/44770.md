## Summary

Current main already fixes the reported macOS agent-context restart failure by detecting when restart is invoked from the managed LaunchAgent and handing the restart off to a detached helper that survives the caller shutdown.

## Evidence

- **LaunchAgent restart avoids self-termination:** `restartLaunchAgent` detects when the current process belongs to the same launchd label and returns a scheduled detached handoff instead of running `launchctl kickstart -k` in-process.
