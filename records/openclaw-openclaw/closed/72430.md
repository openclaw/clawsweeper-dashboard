## Summary

Current main already fixes the reported v2026.4.24 Windows Slack startup failure and the accompanying Bonjour/CIAO startup crash. The shared plugin loader now avoids native Jiti on Windows, keeps Windows import specifiers file-URL safe, and the Bonjour plugin registers nonfatal ciao process-error handlers with regression coverage.

## Best Possible Solution

Keep the current shared loader and Bonjour fixes on main. Users affected on v2026.4.24 should move to the next release or a main build; a new issue should be opened only if the same Windows Slack or CIAO failure still reproduces there with logs.
