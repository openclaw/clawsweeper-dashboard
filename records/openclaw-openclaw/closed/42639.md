## Summary

This is a useful Perplexity Agent API integration request, but current main has enough public provider-plugin surface to implement it outside OpenClaw core. The existing bundled Perplexity plugin is web-search-only; a separate Perplexity Agent provider plugin can register catalog rows and use provider stream hooks to rewrite `model` to `preset`.
