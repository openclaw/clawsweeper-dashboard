## Summary

Current `main` already resolves isolated cron `payload.model`, carries it into the live selection, forwards it through the fallback runner, and passes the resulting provider/model into the embedded cron agent path. Dedicated regression tests now assert both `runWithModelFallback` and `runEmbeddedPiAgent` receive the per-job override.
