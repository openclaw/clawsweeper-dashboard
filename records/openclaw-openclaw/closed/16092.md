## Summary

Current `main` already implements config-backed compaction prompt customization via `agents.defaults.compaction.customInstructions`, which is threaded into safeguard compaction summaries and covered by tests.

## Close Comment

This Codex review found this request is already implemented on `main`. OpenClaw now accepts `agents.defaults.compaction.customInstructions` in config (`src/config/types.agent-defaults.ts:436`, `src/config/zod-schema.agent-defaults.ts:177`), threads it into safeguard compaction (`src/agents/pi-embedded-runner/extensions.ts:105`, `src/agents/pi-hooks/compaction-safeguard.ts:770`), and appends that text into the compaction summarization prompt in `src/agents/compaction.ts:85`. There is also test coverage proving the custom instructions are preserved in compaction summaries (`src/agents/compaction.identifier-preservation.test.ts:78`, `src/agents/pi-hooks/compaction-safeguard.test.ts:1282`). This landed in commit `b22bf36bc4165dfe89b563960498e27e8e4e5034`. Closing as implemented on `main`. If you still need full prompt replacement rather than appended instructions, please open a new issue scoped specifically to that behavior.
