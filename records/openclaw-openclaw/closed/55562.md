## Summary

Current `main` already avoids the reported stale runtime-snapshot overwrite path: locked auth-store updates reload from disk, cache reads are invalidated on `auth-profiles.json`/`auth-state.json` mtime changes, and regression coverage explicitly checks that edited per-agent auth files are reloaded instead of being overwritten.

## Evidence
