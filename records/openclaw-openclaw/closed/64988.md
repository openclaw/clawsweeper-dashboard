Overall confidence: 0.89

Full review comments:

- **[P1] Honor explicit providers before reparsing slash model refs:** `src/gateway/session-utils.ts:1049-1052`
  - body: This branch reparses any selected model containing `/` before checking `entry.providerOverride`. For wrapper selections such as `providerOverride: "openrouter"` with `modelOverride: "anthropic/claude-haiku-4.5"`, that can turn the selected provider into `anthropic` and violate the existing wrapper-provider preservation contract.
  - confidence: 0.9
- **[P2] Do not let stored providers suppress bare-model inference:** `src/gateway/session-utils.ts:1073-1078`
  - body: After finding a unique configured provider, the helper returns `entry.modelProvider` first. Sessions that already ran and have stored provider metadata can keep routing a bare selected model through the stale provider, so the new inference path silently misses the upgrade-shaped case.
  - confidence: 0.87
- **[P2] Infer configured providers for non-OpenAI defaults:** `src/gateway/session-utils.ts:1079-1084`
  - body: The inference branch only runs when the resolved provider is empty or `openai`. If an agent defaults to another provider, a bare override that uniquely matches configured models still remains pinned to that default provider instead of the configured match.
  - confidence: 0.86
