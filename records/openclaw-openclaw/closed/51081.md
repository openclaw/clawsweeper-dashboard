## Summary

Keep open. Current `main` has partial TTS-path work, but the WhatsApp auto-reply path still sends PTT audio with a caption in a single media send, and the current regression test explicitly expects that behavior. That matches the still-open user-visible text-loss part of this issue, so there is not strong enough evidence for a conservative close.
