## Summary

Current `main` already resolves Control UI model selections against the catalog-backed provider instead of trusting a stale session/default provider, and the UI sends that qualified ref unchanged via `sessions.patch`. The regression is covered by focused tests, so this issue is implemented on `main` but not yet in the latest tagged release.

## Evidence
