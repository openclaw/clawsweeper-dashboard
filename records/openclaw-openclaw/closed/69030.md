## Summary

Current main solves the reported raw-inference problem for `openclaw infer model run`. The default local path now bypasses `agentCommand` and calls the simple-completion provider path directly with only the supplied user message, while gateway model probes forward `modelRun: true` and `promptMode: "none"` so AGENTS/bootstrap context, tools, context-engine assembly, and bundled MCP are suppressed. The exact `--raw` flag was not added, but the central user-visible behavior is implemented by making `model run` raw/lean by default.

## Best Possible Solution

Close this as implemented on current main. Keep the lean local `infer model run` behavior as the default raw inference path, keep the gateway raw-probe safeguards, and leave JSON Schema support (#67306) plus broader session-lock/fallback behavior (#66646) to their own canonical issues.
