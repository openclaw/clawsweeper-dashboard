## Summary

Current main already carries the Control UI image-upload path end to end, including gateway normalization for nested base64 attachment payloads and regression coverage for `chat.send` image attachments. The fix is present in shipped release `v2026.4.22` via `00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5`.
