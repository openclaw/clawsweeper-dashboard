## Summary

Current `main` already ships the generic external `exec` SecretRef path, docs, interactive/non-interactive setup, and secure runtime handling that this AuthVault docs PR is trying to establish. The related alignment issue from the same author was closed as implemented after Codex review, so this PR is best treated as superseded rather than an outstanding core gap.
