## Summary

The reported gateway crash is already fixed on current `main`: the 2026.4.10 changelog calls out this exact error signature, the exec runtime now suppresses late update callbacks after run end/abort, and regression tests cover both exit and abort paths.

## Evidence

- **Release note matches the reported crash:** The 2026.4.10 changelog includes an exact fix entry for this issue: preventing the gateway crash with the message "Agent listener invoked outside active run" when a subagent exec tool emits stdout/stderr after the run has ended or been aborted.
  - file: [CHANGELOG.md:1011](https://github.com/openclaw/openclaw/blob/51dd4f288f67ef4df4c7a121f078f508535df1fd/CHANGELOG.md#L1011)

  - sha: [51dd4f288f67](https://github.com/openclaw/openclaw/commit/51dd4f288f67ef4df4c7a121f078f508535df1fd)
- **Runtime guard on settled exec runs:** `runExecProcess` now tracks `updatesDisabled` and makes `emitUpdate()` a no-op once the exec promise has settled, with an inline comment explicitly tying the guard to the outside-active-run crash.
  - file: [src/agents/bash-tools.exec-runtime.ts:561](https://github.com/openclaw/openclaw/blob/51dd4f288f67ef4df4c7a121f078f508535df1fd/src/agents/bash-tools.exec-runtime.ts#L561)

- **Abort and exit paths disable updates before late output can race in:** The runtime disables updates before marking the session exited, and the exec tool's abort handler also calls `run.disableUpdates()` before cancellation so late stdout/stderr cannot enqueue post-run update promises.
  - file: [src/agents/bash-tools.exec.ts:1715](https://github.com/openclaw/openclaw/blob/51dd4f288f67ef4df4c7a121f078f508535df1fd/src/agents/bash-tools.exec.ts#L1715)

- **Regression tests cover the failure mode:** Tests assert that `onUpdate` is not invoked after a foreground exec exits and that updates are suppressed after an abort signal fires, covering the race that previously produced the unhandled rejection.
  - file: [src/agents/bash-tools.test.ts:926](https://github.com/openclaw/openclaw/blob/51dd4f288f67ef4df4c7a121f078f508535df1fd/src/agents/bash-tools.test.ts#L926)
