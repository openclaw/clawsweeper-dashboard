## Summary

Close as duplicate/superseded by the existing open Linux/Windows companion-app tracking work. Current main still does not ship a native Windows companion app, but the remaining product/design/code work is already tracked by the open maintainer-owned #75 and overlapping Tauri desktop PRs #44013/#59075, so #69823 would split the same discussion.

## Best Possible Solution

Close #69823 as duplicate/superseded and keep the remaining native Windows companion-app work consolidated on #75, with implementation review continuing on the active Tauri desktop PRs or a future maintainer-approved replacement. The eventual solution should add a reviewed Windows desktop app under the chosen app structure, document gateway lifecycle/security/install/update behavior, and retain existing CLI Scheduled Task/Startup fallback support until the native app ships.
