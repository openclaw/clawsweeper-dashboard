## Summary

Close #52242 as a duplicate of the older open tracker #47597. Current main still lacks native subagent progress relay: `sessions_spawn` rejects `streamTo: "parent"` unless `runtime` is `acp`, and docs describe native subagents as non-blocking with completion-only announce behavior. #52242’s central request is progress from `runtime="subagent"`, and its Option C is exactly the remaining work tracked by #47597.
