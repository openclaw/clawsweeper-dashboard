## Summary

Issue #17344 is a valid observability/security integration idea, but current OpenClaw scope points this kind of optional, deployment-specific monitoring to external plugins/integrations rather than core. Main already stops at app-layer OTLP export via `diagnostics-otel`, and the reporter linked an external `openclaw-observability-plugin` repo that already hosts the Tetragon policies/docs.
