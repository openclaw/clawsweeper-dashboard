## Summary

The requested 'session health governance' work is better scoped as a ClawHub skill/plugin than additional OpenClaw core behavior. Current main already provides the extension points and several built-in guardrails for tool-heavy session recovery, and the reporter’s own follow-up comment describes a skill-shaped solution.
