## Summary

Current main already exposes the public ACP runtime and plugin-service seams needed for a remote ACP backend, while repo policy and user-facing plugin docs say optional capability like this should ship as an external plugin package via ClawHub/npm rather than as a new bundled core extension.

## Evidence
