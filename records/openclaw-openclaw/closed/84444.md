## Summary

Keep open: this PR is a focused, source-backed fix with sufficient terminal proof and passing PR checks. It still needs maintainer merge judgment because it intentionally stops recall artifact writes in the documented default/off dreaming state.

## What This Changes

The PR gates memory-core recall-tracking artifact writes behind `dreaming.enabled` for tool and CLI search paths and adds regression coverage for enabled and disabled dreaming behavior.

## Best Possible Solution

Land this PR as the canonical fix if maintainers accept the documented master-switch semantics, then close https://github.com/openclaw/openclaw/issues/84436 and supersede https://github.com/openclaw/openclaw/pull/84445.
