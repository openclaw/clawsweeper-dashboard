- **[P1] Register real hooks before startup activation:** `extensions/openclaw-governance/src/index.ts:65-70`
  - body: The manifest startup-loads this plugin, but `register(api)` only voids the API and leaves the model/output/message wiring for later.   - confidence: 0.92
- **[P1] Avoid storing raw DLP matches:** `extensions/openclaw-governance/src/store/sqlite.ts:107`
  - body: The new DLP table requires `matched_snippet` and the scanner returns exact matched text, so enabling DLP persistence would store SSNs, cards, emails, or other sensitive values in cleartext. Store redacted/hash-only evidence by default before this can be a security boundary.
  - confidence: 0.9
- **[P2] Accept cache price overrides in config:** `extensions/openclaw-governance/openclaw.plugin.json:81-83`
  - body: `CostLedgerOptions.pricesPerMillion` accepts `cacheReadUsd` and `cacheWriteUsd`, but the plugin schema only allows `inputUsd` and `outputUsd`. Operators cannot override the cache prices the ledger now charges, so custom rate cards will undercount or ignore cached-token pricing.
  - confidence: 0.86
