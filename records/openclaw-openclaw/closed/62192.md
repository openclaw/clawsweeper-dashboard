## Summary

Current `main` already implements the requested documentation path: OpenClaw now documents Ollama `Cloud only` setup against `https://ollama.com`, explicitly warns that `/v1` OpenAI-compatible mode breaks or degrades tool calling, and the Ollama setup/runtime code and tests match that guidance.
