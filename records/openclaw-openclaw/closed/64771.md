## Summary

Current OpenClaw already implements automatic Control UI websocket reconnects for non-auth disconnects, restores chat/UI state after reconnect, and ships test coverage for those paths; the same behavior is present in the released v2026.4.23 tree, so this issue no longer matches current main or current release behavior.

## Evidence
