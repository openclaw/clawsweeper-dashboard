## Summary

Current `main` already contains the gateway startup race fix that matches the reporter’s follow-up diagnosis, and the same code shipped in `v2026.4.23`. The memory CLI still depends on gateway `secrets.resolve`, so the original symptom path is credible, but the half-open WebSocket startup window is now explicitly closed and covered by regression tests.
