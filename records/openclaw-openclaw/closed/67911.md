## Summary

This request fits OpenClaw's existing plugin/hook extension model rather than a new core config surface. Current main already exposes `agent_end`/`session_end` hooks, emits session-end lifecycle events, and ships bundled memory-capture examples, so the remaining work is packaging a workflow-specific hook/plugin for ClawHub.
