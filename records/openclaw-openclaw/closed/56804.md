## Summary

Current main already addresses the reported unattended-heartbeat quota drain: seeded workspaces now get a comment-only `HEARTBEAT.md` scaffold that causes heartbeat runs to skip API calls when left empty, and heartbeat now has explicit low-cost controls (`model`, `lightContext`, `isolatedSession`) for intentional runs.
