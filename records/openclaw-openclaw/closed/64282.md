## Summary

Close as implemented: current main, and the shipped v2026.4.23 release, retry failed global npm updates with `--omit=optional`, which matches the reported `EBADPLATFORM` failure from `@lancedb/lancedb` on Raspberry OS `linux/arm`.

## Evidence
