## Summary

Current `main` already implements the routing protections behind this report. Shared `agent:main:main` DMs are documented default behavior under `session.dmScope: "main"`, and current code explicitly preserves external Telegram delivery metadata from webchat/internal turns while binding reply delivery to the current turn source instead of stale session state.
