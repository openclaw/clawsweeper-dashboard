## Summary

Current `main` does not show the reported write-failure behavior for the config baseline script. The write path now propagates file I/O errors directly instead of silently continuing, so this PR no longer matches the code it targets.

## Evidence

  - file: [src/config/doc-baseline.ts:601](https://github.com/openclaw/openclaw/blob/6b618f063516aa9ba85903aa4565c5a204c055b2/src/config/doc-baseline.ts#L601)
