- **[P2] Normalize structured content before appending text:** `src/agents/openai-transport-stream.ts:1218-1225`
  - body: The new fallback only runs after `choice.delta.content` is falsy. Structured Mistral content under `delta.content` still enters the earlier text append path and can be coerced to `[object Object]`, so normalize or reject non-string content before appending.
  - confidence: 0.86

  - confidence: 0.84
