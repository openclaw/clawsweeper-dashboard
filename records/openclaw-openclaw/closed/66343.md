## Summary

Current `main` already has heartbeat visibility controls that suppress healthy heartbeat chatter and prompt-echo noise by default while still allowing real heartbeat alerts to be delivered.

## Evidence

- **Default heartbeat visibility is silent for OK chatter:** `resolveHeartbeatVisibility` defaults to `showOk: false`, `showAlerts: true`, and `useIndicator: true`, which matches the requested behavior of hiding routine heartbeat noise while preserving problem notifications.
