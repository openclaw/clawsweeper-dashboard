## Summary

Current main no longer silently drops async plugin registration. OpenClaw now enforces a synchronous `register(api)` contract, has regression coverage for the exact async-route case, and that behavior was already present in shipped release `v2026.4.23`.

## Evidence

## Risks / Open Questions

- This closes the silent-failure bug. It does not add first-class support for async `register(api)`; that would be a separate feature request from the now-enforced synchronous contract.
