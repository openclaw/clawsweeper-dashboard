## Summary

Current main has explicit stale-run handling for the two observable parts of this report: the TUI no longer leaves an active streamed run displayed forever when final events go missing, and the background task ledger reconciles orphaned active tasks to `lost` with audit/maintenance visibility.

## Evidence
