## Summary

Current `main` does not have a code path that would discover `workspace/memory/dreaming/*.md` as agents for the Web UI switcher. The UI loads the switcher from `agents.list`, and the gateway builds that list from configured agent IDs plus state-dir `agents/` directories, not workspace markdown files.

## Evidence

- **Web UI agent switcher source:** The UI controller populates its agent list only from the gateway `agents.list` RPC response.
  - file: [ui/src/ui/controllers/agents.ts:60](https://github.com/openclaw/openclaw/blob/5fe333ada85afd0def9e85985ce57e36aa00108a/ui/src/ui/controllers/agents.ts#L60)
  - command: `nl -ba ui/src/ui/controllers/agents.ts | sed -n '54,76p'`
  - sha: [5fe333ada85a](https://github.com/openclaw/openclaw/commit/5fe333ada85afd0def9e85985ce57e36aa00108a)
- **Gateway handler for `agents.list`:** `agents.list` returns `listAgentsForGateway(cfg)` directly; there is no alternate workspace scan in the handler.
  - file: [src/gateway/server-methods/agents.ts:429](https://github.com/openclaw/openclaw/blob/5fe333ada85afd0def9e85985ce57e36aa00108a/src/gateway/server-methods/agents.ts#L429)
  - command: `nl -ba src/gateway/server-methods/agents.ts | sed -n '420,440p'`
  - sha: [5fe333ada85a](https://github.com/openclaw/openclaw/commit/5fe333ada85afd0def9e85985ce57e36aa00108a)
- **Agent discovery inputs on current main:** `listConfiguredAgentIds` adds IDs from config and `stateDir/agents` directories only, and `listAgentsForGateway` maps those IDs into the response. This code does not inspect workspace `memory/` or `.md` files.
  - file: [src/gateway/session-utils.ts:668](https://github.com/openclaw/openclaw/blob/5fe333ada85afd0def9e85985ce57e36aa00108a/src/gateway/session-utils.ts#L668)
  - command: `nl -ba src/gateway/session-utils.ts | sed -n '668,820p'`
  - sha: [5fe333ada85a](https://github.com/openclaw/openclaw/commit/5fe333ada85afd0def9e85985ce57e36aa00108a)
- **Regression coverage for list scope:** A test asserts `listAgentsForGateway` keeps the explicit `agents.list` scope and excludes unrelated disk-only agent entries, reinforcing that the list is bounded and not sourced from arbitrary workspace files.
  - file: [src/gateway/session-utils.test.ts:680](https://github.com/openclaw/openclaw/blob/5fe333ada85afd0def9e85985ce57e36aa00108a/src/gateway/session-utils.test.ts#L680)
