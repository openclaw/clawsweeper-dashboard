## Summary

Current `main` already contains the packaged bundled-runtime mirror fix and regression coverage for symlinked/global-install cases where bundled extensions import `openclaw/plugin-sdk/*`. The latest tagged release (`v2026.4.23`) predates that fix, so this is implemented on `main` but not yet tied to a shipped release tag.
