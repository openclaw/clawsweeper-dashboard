## Summary

Current `main` already has a pre-execution tool hook via plugin `before_tool_call`, with documented block/approval semantics and runtime enforcement; the internal-hooks docs explicitly point users to plugin hooks for this exact need, and the capability shipped in `v2026.3.28` before the issue was opened on April 3, 2026.
