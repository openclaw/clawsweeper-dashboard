## Summary

Close as `clawhub`: the request is a useful multi-agent project-room/orchestration UX, but current OpenClaw direction keeps this kind of optional heavy coordination layer out of core when existing plugin, command, hook, and subagent surfaces can support experimentation externally. Related issues #34999 and #49509 were already closed on the same ClawHub/plugin-scope basis, and #19929 remains the narrower open core tracker for shared channel/session grouping.
