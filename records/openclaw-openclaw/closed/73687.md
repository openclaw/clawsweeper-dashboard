## Summary

Current main no longer ships the reported unusable default fallback chain. The Google onboarding/default path now writes only `google/gemini-3.1-pro-preview` as the primary model, with tests asserting that shape, while the 16k context-window guard remains in place. The same default is present in the v2026.4.26 release tag, newer than the reported 2026.4.24 environment.

## Best Possible Solution

Close this as implemented for current main and v2026.4.26. Keep the current Google default behavior; if maintainers want automatic cleanup of already-persisted 2026.4.24 configs, track that separately as a narrow doctor/migration follow-up.
