## Summary

Close as implemented on main. Current OpenClaw no longer ships the legacy Chrome extension relay path that this PR hardens, so the change is already superseded by a broader removal and migration on main.

## Evidence

- **Changelog shipped the removal:** The 2026.3.22 release notes explicitly say OpenClaw removed the legacy Chrome extension relay path, bundled extension assets, `driver: "extension"`, and `browser.relayBindHost`.
  - file: [CHANGELOG.md:2207](https://github.com/openclaw/openclaw/blob/c1383680405879a3c3f43dceb20a694f54d3c13c/CHANGELOG.md#L2207)
- **Runtime relay hook is now a stub:** `ensureExtensionRelayForProfiles` is intentionally a no-op because the Chrome extension relay path has been removed.
  - file: [extensions/browser/src/browser/server-lifecycle.ts:13](https://github.com/openclaw/openclaw/blob/c1383680405879a3c3f43dceb20a694f54d3c13c/extensions/browser/src/browser/server-lifecycle.ts#L13)
- **Legacy relay profile is gone:** Browser config tests assert that `chrome-relay` is no longer auto-created on current main.
  - file: [extensions/browser/src/browser/config.test.ts:57](https://github.com/openclaw/openclaw/blob/c1383680405879a3c3f43dceb20a694f54d3c13c/extensions/browser/src/browser/config.test.ts#L57)
- **Legacy extension driver is migrated away:** Doctor normalizers delete `browser.relayBindHost` and rewrite `driver: "extension"` to `"existing-session"`, showing current main treats the relay path as retired compatibility state.
  - file: [src/commands/doctor/shared/legacy-config-core-normalizers.ts:27](https://github.com/openclaw/openclaw/blob/c1383680405879a3c3f43dceb20a694f54d3c13c/src/commands/doctor/shared/legacy-config-core-normalizers.ts#L27)
- **New profile creation rejects the old driver:** Current browser server contract tests expect `driver: "extension"` to fail with `unsupported profile driver "extension"`.
  - file: [extensions/browser/src/browser/server.agent-contract-core.test.ts:547](https://github.com/openclaw/openclaw/blob/c1383680405879a3c3f43dceb20a694f54d3c13c/extensions/browser/src/browser/server.agent-contract-core.test.ts#L547)
- **Old relay implementation files are absent from main:** A repo file search found no current `extension-relay.ts` or `assets/chrome-extension/background-utils.js`, which were the surfaces discussed in the PR and review comment.
