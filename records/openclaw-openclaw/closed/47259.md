## Risks / Open Questions

- A ClawHub implementation still needs a concrete retrieval strategy for Feishu KB content, since `feishu_wiki` currently supports browse/get flows but not full text search.
- No OpenClaw core change can truly prove 'never use model prior knowledge'; that behavior is inherently prompt/workflow constrained, so expectations should be set accordingly.
