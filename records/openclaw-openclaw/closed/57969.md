## Summary

Current `main` already covers the reported gap: bundle MCP now supports direct remote MCP servers over HTTP/SSE and `streamable-http`, exposes a per-server `connectionTimeoutMs` override for both stdio and HTTP, documents those paths, and also tears down timed-out stdio process trees to avoid the orphaned-subprocess failure mode described later in the thread.
