## Summary

Current main already covers this request through the reset-hook pipeline: `/new` and `/reset` emit reset hooks, the bundled `session-memory` hook persists the pre-reset session into `memory/YYYY-MM-DD-slug.md`, docs expose it, and the changelog says it shipped in `v2026.2.21`.
