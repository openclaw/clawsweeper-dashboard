## Summary

This request is an optional orchestration/memory layer that fits the repo’s plugin/ClawHub path, not OpenClaw core. Current main already provides core primitives for sub-agent orchestration and durable task flows, while the project vision explicitly rejects new default agent-hierarchy frameworks and heavy orchestration layers in core.
