## Summary

Current `main` already wires Anthropic into the companion-app usage surface. The gateway now serves `usage.status`, the macOS companion app renders provider usage rows from that summary, and the bundled Anthropic plugin resolves usage auth plus usage snapshots, including a Claude web fallback when setup-token scope is missing. This landed after the issue was filed and is on `main`, but it is not in the latest tagged release `v2026.4.23` yet.
