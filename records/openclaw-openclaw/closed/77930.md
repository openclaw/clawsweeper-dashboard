## What This Changes

The issue reports that Discord stopped appearing as a loaded channel in 2026.5.x because the Discord plugin is no longer bundled and must be installed/enabled separately.

## Best Possible Solution

Land or revise https://github.com/openclaw/openclaw/pull/78090 so the Discord setup docs clearly require installing `@openclaw/discord`, then enabling/restarting or using `openclaw doctor --fix` when Discord is configured but missing.

## Solution Assessment
