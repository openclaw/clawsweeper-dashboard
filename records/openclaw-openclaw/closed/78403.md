Keep open: this maintainer-labeled PR addresses a valid regression, but current head still has blocking correctness and validation issues in paginated self-list lookup and generated prompt snapshots.

## What This Changes

This PR allows restricted cron-triggered runs to call `cron.status` and a self-filtered `cron.list`, with regression tests, cron docs, and a changelog entry.

## Best Possible Solution

Land one canonical fix, this PR or https://github.com/openclaw/openclaw/pull/78286, that permits read-only self-introspection, keeps mutations self-remove-only, handles paginated self-list lookup, and updates generated prompt snapshots/docs.

Yes. Current main source and tests show self-remove-scoped `status` and `list` are rejected before gateway dispatch, and https://github.com/openclaw/openclaw/issues/78208 includes matching installed v2026.5.4 gateway logs.

## Solution Assessment

No, not yet. The read-only permission boundary is appropriate, but this PR must find the current job across pagination or by id and refresh generated prompt snapshots before merge.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.87

Full review comments:

- **[P2] Fetch enough pages before filtering self lists:** `src/agents/tools/cron-tool.ts:711-716`
  - body: In self-remove scope, this calls `cron.list` once and filters only that returned page to `selfRemoveOnlyJobId`. The gateway caps each list page at 200 jobs, so an agent with more jobs can have the current job outside the first page and receive an empty self-list even though the job exists. Page until the job is found or use an id-targeted lookup before returning the filtered response.
  - confidence: 0.88
- **[P2] Update prompt snapshots for the new cron text:** `src/agents/tools/cron-tool.ts:675-676`
  - body: The PR adds new cron tool description text, but the generated prompt snapshot fixtures are not updated and still lack the `RESTRICTED CRON RUNS` section. Regenerate or update the prompt snapshots so `prompt:snapshots:check` captures the changed tool prompt before merge.
  - confidence: 0.82
