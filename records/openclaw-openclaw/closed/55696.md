## Summary

Current `main` already supersedes this PR. The embedded attempt runner no longer imports Telegram helpers from either `extensions/telegram/api.js` or a `plugin-sdk/telegram` barrel; it now uses generic channel prompt seams, with Telegram-specific behavior supplied by the Telegram channel plugin itself.
