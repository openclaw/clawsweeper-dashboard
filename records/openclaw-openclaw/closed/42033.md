## Summary

Current main contains the missing zh-CN Control UI Agents/Cron labels for schedule, last run, and reset, and the Agents/Cron view renders those exact i18n keys. The same keys are present in release v2026.4.24.

  - file: [ui/src/ui/views/cron.ts:483](https://github.com/openclaw/openclaw/blob/e640c0a95fcbd06a1311f18fc635518d011d45da/ui/src/ui/views/cron.ts#L483)
