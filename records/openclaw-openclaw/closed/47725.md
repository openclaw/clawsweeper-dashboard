## Summary

Issue #47725 describes an optional multi-agent orchestration system. Current main scope explicitly routes this kind of capability to plugins/skills published on ClawHub rather than OpenClaw core, and the vision doc also excludes agent-hierarchy and heavy orchestration frameworks as default core architecture.
