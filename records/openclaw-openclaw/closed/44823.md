## Summary

Current `main` already implements Feishu ACP thread/topic bindings. The Feishu channel normalizes ACP conversations for direct chats and Feishu topic threads, thread-binding tests cover Feishu topic conversations, ACP command-context tests build Feishu topic conversation ids from group chat + root message id, and the Feishu docs document persistent ACP bindings plus `/acp spawn ... --thread here` for DMs and Feishu/Lark threads. This support is already present in release `v2026.4.22` (`00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5`).
