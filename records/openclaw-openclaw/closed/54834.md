## Summary

Current `main` no longer matches the reported default ambiguous path. Isolated `cron add` still defaults to announce delivery, but the CLI no longer persists `delivery.bestEffort: true` by default, unresolved implicit `channel: "last"` routes are treated as fail-closed unless best-effort was explicitly requested, and `cron list/show` plus docs now surface whether `last` resolves or will fail closed.

## Evidence
