## Summary

Current main no longer exposes the `openclaw nodes run` wrapper this PR modifies. The `nodes` CLI now rejects `system.run` on `nodes invoke`, and docs/tests direct node shell execution through the `exec` tool with `host=node`, so the PR's target bug is obsolete on the reviewed surface.
