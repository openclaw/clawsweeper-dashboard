## Summary

Current main and the shipped `v2026.4.22` tree already carry the WhatsApp plugin as `@openclaw/whatsapp` with a valid `openclaw.extensions` manifest, and the release pipeline now blocks publishing plugin packages that omit that field. The reported `whatsapp@0.0.5-Alpha` package shape is obsolete relative to the supported install path on main.
