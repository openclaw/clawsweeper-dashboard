## Summary

Current `main` already wires `/restart` through the restart-sentinel flow, and the same implementation is present in shipped release `v2026.4.22`. The slash command now builds a routed sentinel from the session delivery context, writes it before restart, startup schedules sentinel wake, and regression tests cover both normal and `SIGUSR1` restart paths.
