## Summary

Current `main` and the latest shipped release are aligned: the legacy Chrome extension relay was intentionally removed, the supported browser surface is the bundled browser plugin, and the docs now explain why `openclaw browser` / `browser.request` can be missing and how to migrate to managed `openclaw` or `user` profiles.
