## Summary

The reported March 2026 loopback gateway regression is already fixed on current `main`: the gateway pre-auth handshake timeout is now 10 seconds with an `OPENCLAW_HANDSHAKE_TIMEOUT_MS` override, matching a changelog entry that explicitly says it stops busy local gateways from dropping healthy CLI connections at 3 seconds.

## Evidence

  - file: [src/gateway/handshake-timeouts.ts:1](https://github.com/openclaw/openclaw/blob/c1383680405879a3c3f43dceb20a694f54d3c13c/src/gateway/handshake-timeouts.ts#L1)
