## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.82

Full review comments:

- **[P2] Report integrity-check errors:** `src/security/audit-extra.async.ts:1350-1352`
  - body: `verifySkillIntegrity` returns `status: "error"` when hashing fails, but this collector only handles `missing_hash` and `hash_mismatch`. If a tampered skill becomes unreadable or otherwise breaks hashing, `openclaw security audit` can silently omit any integrity finding and make the install look clean.
  - confidence: 0.86
- **[P2] Hash runtime-relevant hidden files:** `src/agents/skills-clawhub.ts:486-488`
  - body: The integrity digest skips every dotfile and dot-directory. Because install-time and audit-time hashes both omit those paths, changing a hidden file that the skill uses at runtime will still verify as `ok`; the hasher should include skill contents except explicitly excluded OpenClaw metadata paths.
  - confidence: 0.78
