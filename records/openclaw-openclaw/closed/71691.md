## Summary

Current main addresses the reported heartbeat/background-work compaction failure mode. Heartbeat prompts are no longer persisted as visible transcript prompts, runtime-only system work is carried separately, compaction detects heartbeat/silent/tool/reasoning-only windows, and tests cover the transcript, compaction, and session-reset boundaries.

## Best Possible Solution

Close as implemented on current main. Keep the shipped runtime-only prompt/transcript split and real-conversation compaction guards; if a new reproduction appears after the next release, track it as a fresh bug with logs showing the remaining compaction window and session rollover path.
