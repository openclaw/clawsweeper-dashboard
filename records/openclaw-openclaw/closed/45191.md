## Summary

Issue #45191 is implemented on current main and in the latest shipped release. The config-load path no longer parses Anthropic model refs during `applyContextPruningDefaults`, and the reported `ANTHROPIC_MODEL_ALIASES` symbol is absent from the source entirely, removing the TDZ failure mode described in the report.
