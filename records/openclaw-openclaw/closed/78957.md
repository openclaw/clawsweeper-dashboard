## What This Changes

Reports that Android cannot recover from a renewed gateway TLS certificate because stored TLS fingerprints survive QR/setup-code rescans.

## Best Possible Solution

Keep one canonical tracker for a user-confirmed Android stale TLS pin recovery flow that preserves fail-closed pinning safety.
