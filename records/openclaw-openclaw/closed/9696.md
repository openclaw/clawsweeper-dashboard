## Summary

Current `main` already shows estimated cost in the shared `/status` status card, and `session_status` uses that same status-card path. The released docs also describe `session_status` as `/status`-equivalent and document estimated cost from model pricing config.

## Evidence

- **Shared status card renders cost:** `buildStatusMessage()` computes a pricing config with `resolveModelCostConfig()`, estimates usage cost, formats it as `💵 Cost: ...`, and includes that in the status output.
  - file: [src/status/status-message.ts:780](https://github.com/openclaw/openclaw/blob/754acc4478fe361db1e5303fd60f589d52beea7a/src/status/status-message.ts#L780)
  - sha: [754acc4478fe](https://github.com/openclaw/openclaw/commit/754acc4478fe361db1e5303fd60f589d52beea7a)
- **session_status uses shared status builder:** `session_status` calls `buildStatusText()` for its response, so it inherits the same usage/cost rendering as `/status`.
  - file: [src/agents/tools/session-status-tool.ts:547](https://github.com/openclaw/openclaw/blob/754acc4478fe361db1e5303fd60f589d52beea7a/src/agents/tools/session-status-tool.ts#L547)
  - sha: [754acc4478fe](https://github.com/openclaw/openclaw/commit/754acc4478fe361db1e5303fd60f589d52beea7a)
- **Status builder passes model auth into status card:** `buildStatusText()` resolves model auth and invokes `buildStatusMessage()` with `modelAuth`, which is the gate used by the cost line logic.
  - file: [src/status/status-text.ts:165](https://github.com/openclaw/openclaw/blob/754acc4478fe361db1e5303fd60f589d52beea7a/src/status/status-text.ts#L165)
  - sha: [754acc4478fe](https://github.com/openclaw/openclaw/commit/754acc4478fe361db1e5303fd60f589d52beea7a)
- **Tests cover cost in status output:** Status tests assert that the rendered status text contains `Cost: $0.0020`, confirming the shared status card already includes estimated cost.
  - file: [src/auto-reply/status.test.ts:88](https://github.com/openclaw/openclaw/blob/754acc4478fe361db1e5303fd60f589d52beea7a/src/auto-reply/status.test.ts#L88)
  - sha: [754acc4478fe](https://github.com/openclaw/openclaw/commit/754acc4478fe361db1e5303fd60f589d52beea7a)
- **Docs describe the shipped behavior:** Docs say `session_status` is the lightweight `/status` equivalent, and `/status` shows estimated cost from model pricing config when available.
  - file: [docs/concepts/session-tool.md:81](https://github.com/openclaw/openclaw/blob/00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5/docs/concepts/session-tool.md#L81)
  - sha: [00bd2cf7a376](https://github.com/openclaw/openclaw/commit/00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5)
- **Released docs cover cost source:** The cost docs state `/status` shows estimated cost for the last reply and that costs are estimated from `models.providers.<provider>.models[].cost`.
  - file: [docs/reference/api-usage-costs.md:17](https://github.com/openclaw/openclaw/blob/00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5/docs/reference/api-usage-costs.md#L17)
