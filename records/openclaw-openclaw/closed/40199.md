## Summary

Current `main` does not add weekday to the system prompt, but it now implements the underlying need on the preferred path: the system prompt stays cache-stable with timezone only, and gateway/chat message handling injects a weekday-prefixed timestamp into user-authored messages. That gives agents day-of-week context without reintroducing daily system-prompt churn.
