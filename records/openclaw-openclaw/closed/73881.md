## Summary

Current main fixes the no-gateway `openclaw infer model run` hang path reported against v2026.4.26. The shipped 4.26 code routed local model runs through `agentCommand`, while current main routes the default/local path through a lean simple-completion provider call with `skipPiDiscovery: true`, avoiding the full agent/gateway/tool/MCP initialization surface. The broader gateway/`secrets.resolve` family remains separately tracked by #73759.

## Best Possible Solution

Close this issue as fixed on current main for the local/no-gateway `infer model run` path, keep the lean simple-completion implementation and regression coverage, and continue any gateway/`secrets.resolve` follow-up under #73759.
