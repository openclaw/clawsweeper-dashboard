## Summary

Current main already implements macOS Universal Binary packaging for Intel and Apple Silicon, and the same implementation is present in the shipped `v2026.4.22` release tag.

  - file: [scripts/package-mac-app.sh:24](https://github.com/openclaw/openclaw/blob/c6684af682bcb59bd1ee85cb49f72d4074007e2c/scripts/package-mac-app.sh#L24)
