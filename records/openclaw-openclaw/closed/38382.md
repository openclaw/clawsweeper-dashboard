## Summary

Issue #38382 asks OpenClaw core to automatically recurse through symlinked workspace directories and inject large external corpora into Project Context. Current main keeps bootstrap injection fixed and explicit, already exposes opt-in hook/plugin and memory indexing surfaces, and the broad symlink-vault behavior fits ClawHub/plugin scope rather than core safe defaults.
