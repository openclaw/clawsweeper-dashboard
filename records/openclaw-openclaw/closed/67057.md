## Summary

Issue #67057 reports Dreaming narrative session accumulation blocking Telegram on OpenClaw 2026.4.14. Current main already contains a direct Dreaming narrative leak fix, and that fix shipped in v2026.4.23 with matching code paths and regression tests for stale session-store entry cleanup, orphan transcript cleanup, and per-workspace session isolation.
