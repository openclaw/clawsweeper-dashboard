## Summary

Current `main` already contains a shipped fix for the empty post-startup channel-registry regression described here. The gateway now pins the startup channel registry so later plugin-registry churn cannot evict configured channels, and regression tests explicitly cover Telegram surviving a post-boot registry replacement.
