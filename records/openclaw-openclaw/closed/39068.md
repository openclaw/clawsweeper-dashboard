## Summary

Issue #39068 is already addressed on current `main`: fresh local onboarding now defaults to the `coding` tool profile instead of leaving new installs messaging-only, and the Control UI now creates a missing per-agent config entry before applying tool-profile and tool-override changes, so the main agent toggles/presets no longer no-op.

## Evidence
