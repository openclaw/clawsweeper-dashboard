## Summary

Current `main` already implements Anthropic Claude Opus 4.7 resolution instead of leaving `anthropic/claude-opus-4-7` in a `configured,missing` state. The Anthropic plugin now forward-resolves Opus 4.7 from the 4.6 template family, treats it as a modern Anthropic model, and keeps Opus 4.7 thinking defaults separate from the older adaptive-thinking default. Repo tests cover the explicit Opus 4.7 resolution path, and the changelog shows the support shipped in `v2026.4.15` with follow-up Opus 4.7 reasoning hardening in `v2026.4.18`.
