## Summary

## What This Changes

Request that Gateway WebSocket failure logs include the last completed handshake phase, such as upgrade, auth validation, session attach, subscription registration, or ready.

## Best Possible Solution

Land or replace the open linked PR with a narrow Gateway-only diagnostics change that records redacted last-completed handshake phase metadata on failure logs, with focused regression coverage and real runtime log proof.
