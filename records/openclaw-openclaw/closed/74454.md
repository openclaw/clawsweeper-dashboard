## Summary

Keep open: current main still lets the registry helper prefer env-derived Windows root candidates before executing the selected registry client. The patch addresses that remaining command-resolution exposure, but it is security-sensitive Windows compatibility hardening and should land through maintainer/security review rather than cleanup closure.

## What This Changes

The PR pins Windows registry probing to a fixed System32 `reg.exe`, updates registry-helper and workspace-dotenv tests, and adds a changelog entry.

## Best Possible Solution

## Solution Assessment
