## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Preserve symlink boundaries when mirroring dist roots:** `src/plugins/bundled-runtime-root.ts:256-270`
  - body: The new root-entry mirror calls `fs.statSync(sourcePath)`, which follows symlinks, then copies/links files or recursively refreshes directories. A symlink under `dist/` or `dist-runtime/` can therefore cause the runtime mirror to materialize target contents outside the package instead of preserving or skipping the link. Use `lstatSync` or the `Dirent` symlink state before copying.
  - confidence: 0.84
- **[P1] Freeze pnpm runtime dependency resolution:** `src/plugins/bundled-runtime-deps.ts:1077-1078`
  - body: The new pnpm path installs generated manifests with `--config.frozen-lockfile=false` and `--config.minimum-release-age=0`, while runtime specs still accept `^`/`~` ranges from package metadata. Startup or doctor repair can resolve newer package versions than the OpenClaw release was built and tested with. Pin the plan or ship and enforce lockfiles before enabling pnpm staging.
  - confidence: 0.88
