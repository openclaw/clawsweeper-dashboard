## Summary

The reported `2026.3.2` npm packaging miss is already resolved on current `main`: Spanish is wired into the Control UI locale registry, has a generated locale bundle, is covered by registry tests, and the publish pipeline now rejects packages that omit Control UI assets. The latest shipped tag available in this checkout (`v2026.4.22`, `00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5`) also contains that implementation.
