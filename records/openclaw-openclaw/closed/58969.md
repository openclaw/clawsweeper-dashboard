## Summary

Current `main` already fixes the reported underscore-stripping bug in `stripMarkdown()`: the shared implementation preserves underscores inside identifiers, the public `plugin-sdk/text-runtime` export points plugins at that implementation, and regression tests cover the reported cases. The fix is present on `main` in commit `4f4288e3b5be2e55abfce5213f0bcf182a252f5e`, but I did not find a release tag containing that commit yet.

## Evidence

- **Fixed underscore regex on main:** `src/shared/text/strip-markdown.ts:12` now uses `/(?<![\p{L}\p{N}])_(?!_)(.+?)(?<!_)_(?![\p{L}\p{N}])/gu`, which only strips `_italic_` markers when they are not embedded in letter/digit sequences.
