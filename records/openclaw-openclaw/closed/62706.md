## Summary

Current `main` no longer supports the failing argument combination from the report: `existing-session` profiles reject explicit CDP endpoints, Chrome MCP is launched with `--autoConnect` plus optional `--userDataDir` only, and the browser status route is now Chrome MCP-specific with regression coverage.

## Evidence
