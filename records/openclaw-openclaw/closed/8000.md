## Summary

Current main intentionally routes niche messaging integrations to the external channel-plugin path. An Agora relay integration matches the documented ClawHub/npm plugin model better than OpenClaw core or a new bundled plugin.

## Evidence

- **Project scope:** The vision doc says optional capability should usually ship as plugins, should be hosted and maintained in the plugin author's own repository, and the bar for adding optional plugins to core is intentionally high.
  - file: [VISION.md:54](https://github.com/openclaw/openclaw/blob/cec34821754ad935c780805f3116f99449d46ac2/VISION.md#L54)
  - sha: [cec34821754a](https://github.com/openclaw/openclaw/commit/cec34821754ad935c780805f3116f99449d46ac2)
- **External plugin guidance:** The plugin getting-started guide explicitly says you do not need to add a plugin to the OpenClaw repository; publish to ClawHub or npm and install it with `openclaw plugins install`.
  - file: [docs/plugins/building-plugins.md:16](https://github.com/openclaw/openclaw/blob/cec34821754ad935c780805f3116f99449d46ac2/docs/plugins/building-plugins.md#L16)
  - sha: [cec34821754a](https://github.com/openclaw/openclaw/commit/cec34821754ad935c780805f3116f99449d46ac2)
- **Channel-plugin path exists:** OpenClaw has a dedicated guide for building a messaging channel plugin that connects OpenClaw to a messaging platform, which is exactly the shape requested by the issue.
  - file: [docs/plugins/sdk-channel-plugins.md:11](https://github.com/openclaw/openclaw/blob/cec34821754ad935c780805f3116f99449d46ac2/docs/plugins/sdk-channel-plugins.md#L11)
  - sha: [cec34821754a](https://github.com/openclaw/openclaw/commit/cec34821754ad935c780805f3116f99449d46ac2)
- **Runtime support for external/custom channels:** The channel adapter types explicitly call out external channel plugins for custom integrations, and the read-only discovery tests cover trusted external channel plugins loaded from a workspace.
  - file: [src/channels/plugins/types.adapters.ts:266](https://github.com/openclaw/openclaw/blob/cec34821754ad935c780805f3116f99449d46ac2/src/channels/plugins/types.adapters.ts#L266)
  - sha: [cec34821754a](https://github.com/openclaw/openclaw/commit/cec34821754ad935c780805f3116f99449d46ac2)
