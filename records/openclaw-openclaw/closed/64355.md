## Summary

Close as `clawhub`. The Campfire work is substantive and already uses the plugin boundary correctly, but current `main` policy and docs explicitly push new optional channel integrations toward externally maintained plugins published through ClawHub/npm rather than adding another bundled OpenClaw repo plugin.
