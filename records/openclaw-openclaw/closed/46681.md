## Summary

Issue #46681 is a progress update for a gstack-style workflow/agent orchestration system that is neither present on current `main` nor aligned with OpenClaw core scope. Current repo guidance routes reusable workflows to skills/plugins, with new skills expected to ship via ClawHub first, and explicitly says not to merge default agent-hierarchy or heavy orchestration layers into core.
