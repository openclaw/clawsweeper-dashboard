## Summary

Current `main` already implements the reported fix: heartbeat runs resolve `agents.defaults.heartbeat.model` up front, mark that a heartbeat override was resolved, and then skip persisted session model overrides. The same logic and regression coverage are present in the shipped `v2026.4.23` release commit.
