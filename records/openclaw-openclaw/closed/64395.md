## Summary

Issue #64395 reports behavior in the WeChat/Weixin integration, but current `main` documents that integration as the external `@tencent-weixin/openclaw-weixin` channel plugin, with WeChat-specific runtime owned outside `openclaw/openclaw`. That makes this a plugin-scope item rather than an OpenClaw core issue.
