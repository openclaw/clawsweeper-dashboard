## Summary

Current `main` already covers this path: CLI operator connects include `operator.talk.secrets` in the default scope bundle, `talk.config(includeSecrets)` accepts either `operator.talk.secrets` or `operator.admin`, device-pair approval tests cover requests that include `operator.talk.secrets`, and the changelog records the CLI default-scope fix in release `2026.4.5`.

## Evidence
