## Summary

#55489 is the same restart/version-reset skills snapshot bug now tracked by canonical maintainer PR #71497. Current main still has the process-local skills snapshot version seeded at 0, so this is not implemented on main yet; the right cleanup is to close the issue as superseded by the open maintainer/dedupe-parent PR rather than keep parallel tracking alongside older contributor PR #55495.

## Best Possible Solution

Close #55489 as superseded by maintainer/dedupe-parent PR #71497, and keep the remaining code, test, and changelog work concentrated there until it lands. Older contributor PR #55495 can remain secondary to the broader shared refresh-state fix in #71497.
