## Summary

Current `main` no longer matches the reported Control UI image-upload failure. The Control UI still serializes selected images into `chat.send` attachments, the gateway accepts attachment-only webchat sends, and the reply path now treats inbound images as sufficient input for a run. The issue thread also contains April 1, 2026 user confirmation that the upload path was working.

## Evidence
