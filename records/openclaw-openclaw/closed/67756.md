## Summary

Current `main` already preserves cron `payload.model` as a full provider/model selection and forwards that resolved provider/model through the isolated-cron dispatch path, with targeted regression tests and docs covering the behavior. The report matches an older behavior, but the current code path no longer supports the silent OpenAI fallthrough described in #67756.
