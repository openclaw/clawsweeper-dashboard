## Summary

Current main already treats WhatsApp outbound as a gateway-owned delivery path, centralizes live connection ownership per account, differentiates auth state from runtime-disconnected state in status/health, and the changelog shows the relevant WhatsApp gateway handoff/outbound-readiness fixes shipped after the reporter’s 2026.4.2 build.

## Evidence
