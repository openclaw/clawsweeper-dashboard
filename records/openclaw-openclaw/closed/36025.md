## Summary

Close as `clawhub`. This request is for custom keyword-triggered automation, and current `main` already treats that as a skill/plugin responsibility rather than an OpenClaw core feature. The repo vision pushes optional capability to plugins/ClawHub, and the SDK already exposes the needed extension points to bypass the LLM and own dispatch.
