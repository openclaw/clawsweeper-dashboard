## Summary

Keep open. Current `main` does not include HexaClaw, so this is not `implemented_on_main`, and the repo docs/vision support both bundled provider plugins and external ClawHub plugins. That policy guidance alone is not strong enough to auto-close a live provider-plugin PR as `clawhub` without clearer maintainer direction or an already-linked external plugin path.
