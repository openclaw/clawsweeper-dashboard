This PR is not a good merge candidate because it bundles unrelated Matrix, model-selection, Ollama, Windows schtasks, and Discord changes, and the existing review thread flagged a TypeScript redeclaration in the branch.

Close this PR as superseded by the focused config-validation path in #50076.
