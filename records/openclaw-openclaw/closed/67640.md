## Summary

This is a useful optional Tencent YuanBao channel integration, but current main's product direction and plugin SDK support make it a ClawHub/npm plugin rather than a new bundled OpenClaw core plugin. The linked PR #67657 was closed unmerged, and current main does not contain a YuanBao channel, docs page, or labeler surface. The PR/body also indicate the implementation can live as a plugin using the public channel SDK, with no core changes required.

## Best Possible Solution

Close the core bundled-plugin request and ask the contributor to publish the YuanBao integration as an external channel plugin on ClawHub/npm, using PR #67657 as the starting implementation. If that work exposes a concrete missing public channel SDK seam, open a narrow follow-up for the API gap rather than tracking the entire YuanBao integration in core.
