## Summary

Current main already avoids using `toolResult` transcript entries for the TUI session-popover preview. The TUI still displays `lastMessagePreview`, but gateway preview generation now only accepts `user` and `assistant` messages, so raw external-content security wrappers from tool results are not eligible to appear there.

## Evidence
