## Summary

Issue requests a core Anthropic Claude Agent SDK migration, but current main documents this class of runtime as a plugin/provider-harness concern rather than OpenClaw core. The repo has explicit plugin and agent-harness extension points for native agent daemons, and VISION.md says optional capability should usually ship as plugins while heavy orchestration layers duplicating existing agent/tool infrastructure are out of scope for core.

## Evidence
