## Summary

Keep open. Current `main` only cleans tracked browser tabs in the normal `completeSubagentRun(... triggerCleanup=true)` lifecycle path; the explicit `releaseSubagentRun` and `markSubagentRunTerminated` terminal paths still do not invoke browser-tab cleanup, so the PR's requested behavior is not fully implemented on `main`.
