## Summary

Keep open. Current main has active Node/npm reporting and PATH troubleshooting, but it does not implement the requested install-script warning for multiple Node environments or a doctor check for duplicate global OpenClaw installs. The linked PR #40850 directly addresses the request, but it is still open and unmerged, so the issue is not implemented or safely superseded on main.

## Best Possible Solution

Keep #40839 open as the canonical user-facing request. Maintainers should either revive/replace PR #40850 with tests for `scripts/install.sh`, or split the work into a narrow installer warning plus a separate doctor check that scans known global npm roots and reports duplicate OpenClaw versions without mutating installs unexpectedly.
