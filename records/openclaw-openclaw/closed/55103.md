## Summary

Current `main` already contains the compaction-safeguard fixes for this report, and the changelog shows they shipped in `v2026.2.24`. The old generic `"Summary unavailable due to context limits"` path was replaced with runtime-model fallback, explicit cancel-on-summarization-failure behavior, and more defensive oversize-aware summarization.

## Evidence

- **Shipped changelog entry:** `2026.2.24` explicitly notes two relevant fixes: passing model metadata through the embedded runtime to avoid repeated `"Summary unavailable due to context limits"` fallbacks, and cancelling safeguard compaction when summary generation cannot run so history is preserved instead of being truncated to fallback text.
  - file: [CHANGELOG.md:4091](https://github.com/openclaw/openclaw/blob/34896839ba22f3a71a3d5a8fb4ca5f4c6989a534/CHANGELOG.md#L4091)
- **Runtime model fallback on current main:** The safeguard now resolves the summarization model as `ctx.model ?? runtime?.model`, covering the embedded `compact.ts` path where `ctx.model` may be missing.
  - file: [src/agents/pi-hooks/compaction-safeguard.ts:860](https://github.com/openclaw/openclaw/blob/34896839ba22f3a71a3d5a8fb4ca5f4c6989a534/src/agents/pi-hooks/compaction-safeguard.ts#L860)
- **Compaction cancels instead of silently injecting fallback summary:** When summarization still fails, the safeguard logs the error, records a cancel reason, and returns `{ cancel: true }` rather than persisting a generic fallback summary.
  - file: [src/agents/pi-hooks/compaction-safeguard.ts:1118](https://github.com/openclaw/openclaw/blob/34896839ba22f3a71a3d5a8fb4ca5f4c6989a534/src/agents/pi-hooks/compaction-safeguard.ts#L1118)
- **Oversize-aware fallback summarization:** Current compaction first tries full summarization, then retries with oversized messages omitted and annotated, instead of only producing the old generic fallback text.
  - file: [src/agents/compaction.ts:399](https://github.com/openclaw/openclaw/blob/34896839ba22f3a71a3d5a8fb4ca5f4c6989a534/src/agents/compaction.ts#L399)
- **Regression coverage for root cause:** A test explicitly documents the root-cause fix for the `compact.ts` workflow: when `ctx.model` is undefined, `runtime.model` is used instead.
  - file: [src/agents/pi-hooks/compaction-safeguard.test.ts:1718](https://github.com/openclaw/openclaw/blob/34896839ba22f3a71a3d5a8fb4ca5f4c6989a534/src/agents/pi-hooks/compaction-safeguard.test.ts#L1718)
