## Summary

Keep open. Current `main` still forwards raw audio transcription text into agent context with only whitespace/length trimming; there is no built-in guard for known Whisper subtitle-credit hallucinations, so the PR addresses behavior that is not yet implemented on `main`.

## Evidence

- **Audio transcription output is still passed through with trim-only handling:** `runProviderEntry()` returns `audio.transcription` text via `trimOutput(result.text, maxChars)` and does not sanitize known subtitle-credit strings first.
  - file: [src/media-understanding/runner.entries.ts:634](https://github.com/openclaw/openclaw/blob/7d471837369885c12d540d6bafa319ff41129b89/src/media-understanding/runner.entries.ts#L634)
  - sha: [7d4718373698](https://github.com/openclaw/openclaw/commit/7d471837369885c12d540d6bafa319ff41129b89)
- **Only generic trimming exists in the media runner:** The only nearby text cleanup helper is `trimOutput()`, which strips whitespace and enforces `maxChars`; there is no transcript-specific hallucination filter in this file.
  - file: [src/media-understanding/runner.entries.ts:70](https://github.com/openclaw/openclaw/blob/7d471837369885c12d540d6bafa319ff41129b89/src/media-understanding/runner.entries.ts#L70)
  - sha: [7d4718373698](https://github.com/openclaw/openclaw/commit/7d471837369885c12d540d6bafa319ff41129b89)
- **Transcript text is injected into user/agent context unchanged:** `applyMediaUnderstanding()` takes `audio.transcription` outputs, formats them into `transcript`, then assigns that value to `ctx.Transcript`, `ctx.CommandBody`, and `ctx.RawBody`. A hallucinated credit line would therefore propagate into prompt context on current `main`.
  - file: [src/media-understanding/apply.ts:533](https://github.com/openclaw/openclaw/blob/7d471837369885c12d540d6bafa319ff41129b89/src/media-understanding/apply.ts#L533)

  - file: [src/media-understanding/format.ts:93](https://github.com/openclaw/openclaw/blob/7d471837369885c12d540d6bafa319ff41129b89/src/media-understanding/format.ts#L93)
  - sha: [7d4718373698](https://github.com/openclaw/openclaw/commit/7d471837369885c12d540d6bafa319ff41129b89)
