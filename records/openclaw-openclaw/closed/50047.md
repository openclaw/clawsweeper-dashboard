## Summary

Current `main` already addresses the reported startup crash by forcing source plugin entries through Jiti's transform path instead of native loading. The Discord runtime shims still use `.js` specifiers in source, but the loader now explicitly disables native loading for source entries so those sibling `.js` imports are rewritten against the source graph.
