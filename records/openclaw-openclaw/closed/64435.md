## Summary

The issue proposes a large in-core multi-agent orchestration subsystem, but current main already covers delegated subagents, ACP-backed external harness sessions, and durable task orchestration. OpenClaw's current vision explicitly routes optional orchestration toward plugins/ClawHub and says agent-hierarchy frameworks and heavy orchestration layers are out of scope for core for now.
