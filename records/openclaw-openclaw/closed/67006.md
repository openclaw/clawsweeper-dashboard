## Summary

Current `main` already changed `runtime-postbuild` to stage bundled plugin runtime deps from the installed root workspace graph before any fallback installer runs, and coverage now explicitly exercises that source-build path. The exact fallback dependency chain from the report is also obsolete on current `main` because the relevant plugin manifests/root deps have changed.
