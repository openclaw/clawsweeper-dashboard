- **[P1] Preserve stale-run reconciliation when no terminal arrives:** `ui/src/ui/app-gateway.ts:1175-1177`
  - body: Current main starts a `loadSessions` refresh for an active-run `session.message` and can replay the deferred history reload after that refresh clears a stale run.
