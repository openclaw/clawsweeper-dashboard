## Summary

Current `main` already suppresses WhatsApp quoted self-reply implicit mentions in shared-number/self-chat group flows, and includes targeted regression coverage for the reported bypass.

## Evidence

- **Fix in group gating:** `applyGroupGating` now computes `implicitReplyToSelf` and only treats a quoted reply as `quoted_bot` when the quoted sender overlaps self but the current sender is not the same self identity. That prevents self-replies from counting as implicit mentions.
