## Summary

Issue #44940 is already implemented on current `main` for the reported isolated `agentTurn` cron path. The merged fix changed isolated embedded cron runs to execute as non-owner senders, which removes owner-only `cron` and `gateway` tools from the model surface and addresses the reported self-rewrite mechanism.
