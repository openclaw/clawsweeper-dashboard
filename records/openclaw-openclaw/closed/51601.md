## Summary

Current main already has layered protections for the reported leak modes: MiniMax Anthropic streaming disables implicit thinking to prevent `reasoning_content` leakage, Discord suppresses reasoning payloads and strips reasoning tags from streamed previews, assistant-visible text strips leaked tool/tool-result XML, and session-history sanitization drops orphaned reasoning/tool-result replay after session resets or transcript mismatches.
