## Summary

Current `main` already guards MiniMax Anthropic-compatible streaming by disabling implicit thinking for both `minimax` and `minimax-portal`, which matches the reported `minimax-portal/MiniMax-M2.7` path and prevents leaked `reasoning_content` from surfacing as stray visible text. The fix is covered by targeted tests and documented publicly, but it does not appear to have shipped in `v2026.4.23` yet.
