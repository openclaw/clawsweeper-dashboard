## Summary

Current main already supports opt-in cross-agent memory access through per-agent memory scope configuration: `memorySearch.extraPaths` for selected external memory files/directories and `memorySearch.qmd.extraCollections` for selected cross-agent QMD transcript collections. The shipped design is scope-based rather than adding `allowReadFrom` and per-call `agent` selectors, but it covers the requested capability well enough.
