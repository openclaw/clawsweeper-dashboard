## Summary

Current `main` already contains the loopback/auth/probe fixes that match this report: local shared-secret CLI calls stay device-bound so read scopes are preserved, scope-limited probe results are now treated as degraded-but-reachable instead of unreachable, and the gateway call path yields before websocket connect to avoid handshake-timeout races.
