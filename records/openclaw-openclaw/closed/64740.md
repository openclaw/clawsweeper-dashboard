## Summary

Current `main` already mitigates the reported Claude Code `HEARTBEAT.md` prompt rejection by stripping the exact heartbeat policy quote from injected system context while preserving heartbeat behavior through a separate generic Heartbeats section, so the proposed filename rename is no longer needed.
