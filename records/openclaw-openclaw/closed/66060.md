## Summary

Current `main` no longer feeds active-memory the full embedded-agent bootstrap context that the report described. The active-memory run is constrained to `memory_search`/`memory_get`, lightweight bootstrap now resolves to empty context for default runs, and the embedded runner switches allowlisted runs to a minimal system prompt with the skills catalog stripped.

## Evidence
