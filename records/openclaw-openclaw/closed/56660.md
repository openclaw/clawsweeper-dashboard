## Summary

Current `main` already implements the compaction-model override for the recovery paths this PR targets. `runEmbeddedPiAgent` now routes both timeout and overflow compaction through `buildEmbeddedCompactionRuntimeContext`, and that helper eagerly applies `agents.defaults.compaction.model` plus the provider-switch auth-profile drop. Tests also cover the override behavior. The PR is therefore redundant on current `main`.
