## Summary

The reported runtime-path failure is convincingly demonstrated, but the launcher still has merge-blocking upgrade and startup compatibility problems. ## Best Possible Solution

Keep the narrowly selected managed-launcher injection, but preserve or explicitly migrate prior Codex HOME state, handle existing profile content without preventing app-server startup, and prove both fresh-install and upgrade behavior.

No, not yet. Launcher-based runtime injection is the narrowest plausible Nix-owned fix, but its HOME migration and fail-closed profile handling are not upgrade-safe without further changes or explicit maintainer acceptance.
