## What This Changes

The issue requests Node UI controls for configuring sandboxing of gateway tool calls.

## Best Possible Solution

Define the gateway-side sandbox policy, storage, and API contract first, then add Node UI controls wired to that contract while keeping the current Windows-node `system.run` scope explicit.

## Solution Assessment
