## Best Possible Solution

Add a maintainer-approved diagnostics sharing flow that reuses the existing bundle UI, routes all exported content through the agreed scrub boundary, and either opens a prefilled GitHub issue with an attachable file or performs an authenticated upload with explicit user confirmation.
