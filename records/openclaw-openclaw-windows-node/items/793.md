## Summary

## What This Changes

## Best Possible Solution

Keep the shared-token auth path, but scope the effective browser-control endpoint to the active GatewayRecord/tunnel contract, align diagnostics with that endpoint, and require redacted live split-gateway proof before merge.

## Solution Assessment

No. A port override may be useful, but the safer solution is to bind the effective endpoint to the active gateway/tunnel contract and keep diagnostics aligned with the same endpoint.
