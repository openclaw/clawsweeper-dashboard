## What This Changes

This PR replaces hardcoded tray/node handshake versions with AppVersionInfo, adds node-only startup reconnect for persisted NodeDeviceToken installs, rewrites exec approval prompts as a native Win32 window with fallback, and adds regression coverage.

## Best Possible Solution

Land this approach only after Windows validation proves node identity survives restart/reboot/gateway-restart flows and the native exec prompt preserves deny, allow-once, always-allow, and stale/drift protections.
