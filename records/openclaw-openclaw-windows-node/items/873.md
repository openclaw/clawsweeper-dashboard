- **[P1] Fail closed for main-session toast actions:** `src/OpenClaw.Tray.WinUI/App.xaml.cs:1132-1134`
  - body: The toast/action handler only considers `_appState.Sessions` and `client.MainSessionKey` when deciding whether a delete targets the main session.
