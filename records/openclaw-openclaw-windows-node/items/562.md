Full review comments:

- **[P2] Handle setup-log enumeration failures:** `src/OpenClaw.Tray.WinUI/Services/DiagnosticsBundleBuilder.cs:139-143`
  - body: `AppendLatestSetupLogs` checks that the directory exists, then enumerates `*.jsonl` without catching `IOException`, `UnauthorizedAccessException`, or a race where the directory disappears. Unlike the other log-tail sections, that aborts the whole diagnostics bundle and falls back to a generic failure string, so an inaccessible setup-log folder can prevent users from exporting diagnostics; emit a sanitized status line for setup logs instead.
  - confidence: 0.86
- **[P2] Align localized copy with the copied bundle:** `src/OpenClaw.Tray.WinUI/Strings/fr-fr/Resources.resw:3544`
  - body: The direct copy handler still uses `BuildDebugBundle`, and that bundle includes `Recent Tray Log`, but this localized description says log excerpts are excluded. In these locales users can copy log content while being told it is not included, which undercuts the preview-before-log-sharing boundary; either remove the tray log from the copied bundle or align all localized descriptions with the English behavior.
  - confidence: 0.88
