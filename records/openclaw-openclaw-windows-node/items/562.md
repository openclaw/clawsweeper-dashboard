  - body: `Build()` calls `NormalizePersistedDiagnosticsLogs()` before creating the bundle, and that path writes sanitized text back to existing tray, JSONL, crash, auxiliary, and setup logs.   - confidence: 0.9
- **[P2] Guard setup-log enumeration before startup/export:** `src/OpenClaw.Tray.WinUI/Services/DiagnosticsBundleBuilder.cs:267`
  - body: `NormalizeLatestSetupLogs()` enumerates the setup-log directory after an `Exists` check without catching enumeration failures, and the same pattern appears in setup-log tail/cache-key enumeration. If the directory is deleted or becomes inaccessible between those calls, startup normalization or bundle preview can throw instead of producing a failed diagnostics section.
  - confidence: 0.86

  - body: The constructor now saves tool and attachment metadata caches immediately when legacy escaping is detected, so normal chat startup can rewrite persisted local metadata before the user performs an explicit action.   - confidence: 0.78
