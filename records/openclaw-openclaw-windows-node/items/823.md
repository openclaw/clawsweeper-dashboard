## What This Changes

Land the focused MIME-format hardening after the draft is ready, required checks pass, real behavior proof is added, and maintainers accept the documented-format fail-closed behavior.

Yes; validating `png`/`jpeg`, normalizing `jpg`, rejecting unsupported formats before capture, and deriving MIME from the normalized value is the narrowest maintainable fix for the hardening gap.
