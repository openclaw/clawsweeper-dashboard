Full review comments:

- **[P2] Make Dispose claim teardown before running callbacks:** `src/OpenClaw.Shared/WebSocketClientBase.cs:684-685`
  - body: This now delays `_disposed = true` until after `OnDisposing()`, but `Dispose()` still only checks `_disposed`. Claim teardown with an atomic/early guard before invoking subclass cleanup.
