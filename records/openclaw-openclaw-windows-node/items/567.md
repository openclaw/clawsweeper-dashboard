- **[P2] Avoid logging stable hashes of chat text:** `src/OpenClaw.Tray.WinUI/Chat/OpenClawChatDataProvider.cs:1158-1160`
  - body: This trace writes `h={ChatTraceHash(traceText)}` for every chat message.   - confidence: 0.86
