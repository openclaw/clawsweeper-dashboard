## Best Possible Solution

Land the narrow isolation fix after normal repository validation, keeping gateway credential precedence and device-token behavior unchanged.

## Reproduction Assessment

## Solution Assessment

Yes: using invariant formatting matches existing dot-decimal assertions and local invariant-format precedent, and isolating identityPath before construction fixes constructor-time host state leakage. Reusing the existing CreateTempIdentityPath helper would be minor cleanup, not a correctness blocker.
