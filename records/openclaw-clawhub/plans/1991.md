## Plan

Fix issue https://github.com/openclaw/clawhub/issues/1991 by adding a narrow Convex repair path for existing ClawPack/npm-pack package releases whose persisted release.files are metadata-only while the stored tarball contains more entries. Preserve artifact bytes and existing trust/moderation policy. Add regression coverage for old metadata-only rows, package inspect/raw-file behavior, and scan requeue behavior, and update specs/security-moderation.md to describe the supported scan scope after the repair.
