Keep open: this is a fresh product/UX/API RFC, current main still lacks plugin category browse controls and category API support, and the linked implementation at https://github.com/openclaw/clawhub/pull/2126 remains open and conflicting.

The RFC requests plugin browse categories and clearer sort controls while preserving verified and executes-code filters and deferring stats-based sorting.
