Keep open: the named skill now appears clean in the public API on engine v2.4.24, but this issue also asks for a class-level defensive-scanner policy, and current main still has no approved metadata/category/allowlist path while the related implementation at https://github.com/openclaw/clawhub/pull/1132 remains open and conflicted.
