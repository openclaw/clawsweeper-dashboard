## What This Changes

The issue asks why the `skillpick` and `threeperson` skills do not appear in global web search after the reporter believed moderation had cleared.

## Best Possible Solution

Keep the global suspicious-skill filter in place, then have staff complete moderation review or an explicit override for the production records; improve owner-facing diagnostics if they still make suspicious records look cleared.
