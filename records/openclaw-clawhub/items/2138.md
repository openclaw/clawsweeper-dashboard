  - body: `/api/v1/skills/export` lives inside the documented `/api/v1/skills/{slug}` namespace, and `export` is not reserved by the slug validator.   - confidence: 0.94
- **[P1] Throttle export auth failures before token lookup:** `convex/httpApiV1/skillsV1.ts:1571-1574`
  - body: The handler returns from `requireExportAuth` before calling `applyRateLimit`, so missing, invalid, and non-admin export attempts do not consume the new export bucket. Put an IP/export throttle before the privileged token lookup or make the auth helper consume the appropriate failure bucket.
  - confidence: 0.88
- **[P2] Add an aggregate export size budget:** `convex/httpApiV1/skillsV1.ts:1693`
  - body: The handler queues the page's file blobs, fetches them, materializes each as a `Uint8Array`, and only then compresses the archive. Without a total file/byte cap, a large page can exceed Convex action memory even when the item count limit is respected.
  - confidence: 0.86
