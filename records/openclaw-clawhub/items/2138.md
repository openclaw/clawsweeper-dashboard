## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Preserve public visibility before exporting files:** `convex/skills.ts:10413`
  - body: The export page is built from `skillSearchDigest.by_active_created` with only the soft-delete prefix, then the handler reads internal version blobs directly. That bypasses the current public list/detail/download gates for hidden, removed, suspicious, or malware-blocked skills, so a mirror token could receive content the public API intentionally withholds.
  - confidence: 0.93
- **[P1] Move export out of the skill slug namespace:** `convex/http.ts:77-79`
  - body: Registering `/api/v1/skills/export` as an exact route means a real skill with slug `export` no longer resolves through the documented `/api/v1/skills/{slug}` detail route. Put export under a reserved namespace such as `/-/export`, or add an explicit reserved-slug migration and compatibility test.
  - confidence: 0.88
- **[P1] Page mirror sync by updates and tombstones:** `convex/skills.ts:10408-10419`
  - body: The PR is meant for hourly mirror sync, but this query selects by `createdAt`. Any older skill that is republished, renamed, hidden, removed, restored, or otherwise moderated after its creation window will not be included in later incremental exports, so mirrors can silently diverge.
  - confidence: 0.9
- **[P2] Rate-limit failed export authentication:** `convex/httpApiV1/skillsV1.ts:1571-1575`
  - body: `requireExportAuth` runs before `applyRateLimit`, so missing, invalid, or non-admin tokens return without consuming the export bucket. That conflicts with the existing rate-limit model for invalid tokens and leaves token validation/admin checks exposed to unthrottled retry traffic.
  - confidence: 0.86
- **[P2] Cap aggregate export bytes before zipping:** `convex/httpApiV1/skillsV1.ts:1693`
  - body: The handler allows up to 1000 skills, fetches every file blob, materializes each blob into a `Uint8Array`, and then creates another ZIP buffer without a total byte guard. Large skills or many files can turn an authorized export into a Convex action memory/availability failure.
  - confidence: 0.87
