Define the mirror-export contract first, then implement a bounded endpoint that exports only the approved public surface with rate limits, auditability, tests, and redacted runtime proof.

  - body: The handler allows 1000 skills per page and later loads every selected blob into memory before zipping. Since current publish limits allow a 50 MB skill bundle, a valid admin export can attempt a page far beyond Convex action memory; sum version file sizes and reject or shrink pages before calling `ctx.storage.get()`/`blob.arrayBuffer()`.
