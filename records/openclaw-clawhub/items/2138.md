## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Filter exports through the public visibility boundary:** `convex/skills.ts:10411-10421`
  - body: The export page comes straight from `skillSearchDigest.by_active_created` and returns those rows without calling `toPublicSkill`, checking the owner is public, or honoring the suspicious/hidden filters used by the public list/detail APIs. An admin export token would therefore bulk-copy rows that the documented public API says mirrors must not bypass; build the export from the same public visibility rules before exposing file blobs.
  - confidence: 0.93
- **[P1] Avoid shadowing a valid skill slug with the export route:** `convex/http.ts:76-80`
  - body: `export` is not a reserved skill slug on current main, and `/api/v1/skills/{slug}` is the documented detail route. Adding an exact `/api/v1/skills/export` route makes that slug resolve to the archive endpoint instead of the skill detail API, so existing or future skills named `export` lose their public API URL.
  - confidence: 0.86
- **[P1] Rate-limit failed export authentication attempts:** `convex/httpApiV1/skillsV1.ts:1570-1575`
  - body: The handler calls `requireExportAuth` before `applyRateLimit`, so missing, invalid, and non-admin bearer-token attempts return without consuming the new export bucket. That leaves the expensive token lookup and role-check path unthrottled for a security-sensitive endpoint; apply an IP-scoped limit before or around failed auth too.
  - confidence: 0.89
- **[P1] Bound the total export payload before reading blobs:** `convex/httpApiV1/skillsV1.ts:1693-1741`
  - body: The code accepts up to 1000 skills and fetches every listed blob into memory before building another ZIP buffer, but it has no aggregate byte or file-count ceiling. A small number of large published skills can exceed Convex action memory or runtime limits and make the endpoint unavailable; enforce a total budget before reading or while streaming entries.
  - confidence: 0.86
