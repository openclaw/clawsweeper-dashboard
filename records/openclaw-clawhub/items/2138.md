  - body: `listByDateRange` pages `skillSearchDigest` by `by_active_created` and returns those rows directly to the export action.   - body: The handler calls `requireExportAuth` before `applyRateLimit`, so missing or invalid bearer attempts return 401 before consuming the export/IP bucket.   - confidence: 0.86

  - body: The export always adds `${slug}/_meta.json`, but published skill files can already include that relative path. When such a skill appears in a page, `buildMergedExportZip` throws on the duplicate path and turns an otherwise valid export request into a server error.
  - confidence: 0.83
- **[P2] Bound export bytes before reading blobs:** `convex/httpApiV1/skillsV1.ts:1625`
  - body: The endpoint can select up to 1000 skills and then read every selected blob into memory before zipping. Valid skill bundles can be large enough for a normal page to exceed action memory or response limits; paginate by a byte budget or lower the effective limit before fetching blobs.
  - confidence: 0.78
