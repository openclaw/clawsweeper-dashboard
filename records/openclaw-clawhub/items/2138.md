That can break fetching an existing or future skill with that slug; use a reserved namespace such as `/api/v1/skills/-/export` or add an explicit reservation/migration.
  - confidence: 0.86
- **[P2] Throttle failed export auth attempts before token lookup:** `convex/httpApiV1/skillsV1.ts:1571-1574`
  - body: The handler calls `requireExportAuth` before `applyRateLimit`, so invalid or non-admin tokens return 401/403 without consuming the export bucket while still forcing auth lookup work on a privileged public endpoint.   - confidence: 0.86
- **[P2] Cap aggregate export size before reading blobs:** `convex/httpApiV1/skillsV1.ts:1693`
  - body: The handler accepts up to 1000 skills, collects every file blob, converts each blob to an `ArrayBuffer`, and then builds one ZIP without any total byte or file-count cap.
