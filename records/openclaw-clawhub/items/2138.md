Full review comments:

- **[P1] Filter exports through public visibility gates:** `convex/skills.ts:10411-10413`
  - body: `listByDateRange` returns raw `skillSearchDigest` rows and the handler exports their files directly. Existing public list and download paths filter hidden, non-active, and malware-blocked skills, so this can bulk expose content that is intentionally absent from public mirror/download surfaces.
  - confidence: 0.92
- **[P2] Rate-limit export attempts before auth lookup:** `convex/httpApiV1/skillsV1.ts:1571-1574`
  - body: The handler calls `requireExportAuth` before `applyRateLimit`, so missing or invalid export credentials never consume the export IP bucket while still forcing token parsing, hashing, and lookup work. Apply a cheap pre-auth limiter or move the existing export limiter ahead of auth so failed attempts are throttled too.
  - confidence: 0.86
- **[P2] Cap export bytes before reading blobs:** `convex/httpApiV1/skillsV1.ts:1693`
  - body: The only hard cap is the number of skills, but each selected version can add multiple blobs and the handler materializes all blob bytes plus the ZIP in memory. Add aggregate byte/file limits before or during blob reads so a valid admin export cannot exhaust Convex action memory or runtime.
  - confidence: 0.88
