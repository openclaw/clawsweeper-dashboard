  - confidence: 0.88

  - body: The handler limits only the number of skills, then reads every selected blob into memory and builds one ZIP. A few large valid skills or many-file skills can exceed action memory or latency budgets, so the export needs a total byte/file budget before `blob.arrayBuffer()` and ZIP assembly.
  - confidence: 0.86
