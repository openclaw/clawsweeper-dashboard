  - confidence: 0.88

  - confidence: 0.9
- **[P1] Bound export ZIP memory before reading blobs:** `convex/httpApiV1/skillsV1.ts:1740-1741`
  - body: A valid request can ask for up to 1000 skills, enqueue every file blob, convert each blob to a `Uint8Array`, keep all entries in memory, and then create the merged ZIP with no aggregate byte or file-count guard. This can exhaust the Convex action before rate limits help.
  - confidence: 0.86
