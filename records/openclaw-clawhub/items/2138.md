  - body: This endpoint calls `requireExportAuth` before `applyRateLimit`, so missing/invalid tokens and non-admin tokens return 401/403 without consuming the export IP bucket.   - confidence: 0.9
- **[P1] Use an update-aware cursor for mirror sync:** `convex/skills.ts:10413-10418`
  - body: The export query ranges over `createdAt`, so an hourly mirror that advances this cursor will miss existing skills whose latest version, file set, moderation state, rename, restore, or deletion changes after the skill was first created. The feed needs an `updatedAt`/change-log cursor or another explicit delta source for the stated sync workflow.
  - confidence: 0.88
- **[P1] Bound the aggregate ZIP before loading blobs:** `convex/httpApiV1/skillsV1.ts:1693-1741`
  - body: A request can select up to 1000 skills, fetch every referenced storage blob, convert all blobs to `ArrayBuffer`s, and then build one ZIP in memory without a total byte or file-count cap. Use the stored file sizes to reject or paginate oversized exports before loading blobs so one valid admin request cannot exhaust the action runtime.
