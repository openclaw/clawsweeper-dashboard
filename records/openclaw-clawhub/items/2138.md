  - confidence: 0.91

  - body: `requireExportAuth` runs before `applyRateLimit`, so missing or invalid credentials can repeatedly exercise bearer parsing, token hashing, and auth lookup without consuming the export IP bucket.   - confidence: 0.87

  - body: The handler accepts up to 1000 skills, fetches their blobs, converts each blob into memory, and only then creates one ZIP.
