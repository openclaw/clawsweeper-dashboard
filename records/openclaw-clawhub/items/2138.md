## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.93

Full review comments:

- **[P1] Filter exports through public visibility:** `convex/skills.ts:10413`
  - body: This query only pages `skillSearchDigest.by_active_created`, which constrains `softDeletedAt` but not `moderationStatus` or `blocked.malware`; the export handler then fetches each latest version file and zips it. A mirror export should reuse public visibility filtering or be explicitly approved as a staff-only moderation export, otherwise hidden or malware-blocked content can leave the service.
  - confidence: 0.95
- **[P1] Cursor by updates, not creation time:** `convex/skills.ts:10413`
  - body: The endpoint is positioned as hourly mirror sync, but `startDate` and `endDate` are applied to `createdAt`. An existing skill that publishes a new version, is renamed, or has moderation or metadata changed after its original creation date will not appear in incremental sync windows.
  - confidence: 0.93
- **[P1] Rate-limit failed auth before token lookup:** `convex/httpApiV1/skillsV1.ts:1571`
  - body: The handler calls `requireExportAuth` before `applyRateLimit`, so invalid bearer-token attempts can force token hashing and token lookup before consuming the export IP bucket. Put anonymous/export throttling ahead of the privileged auth lookup or otherwise consume a failure bucket.
  - confidence: 0.9
- **[P1] Avoid occupying a valid skill slug path:** `packages/schema/src/routes.ts:30`
  - body: Adding `/api/v1/skills/export` as an exact route takes over the current detail route for a valid `export` skill slug, and `export` is not in the current reserved slug list. Put bulk operations under a reserved control namespace or reserve and migrate the slug explicitly.
  - confidence: 0.87
- **[P2] Cap total archive work per request:** `convex/httpApiV1/skillsV1.ts:1740`
  - body: The handler allows up to 1000 skills, loads every file blob into memory with `arrayBuffer()`, then builds a full ZIP buffer, but it never enforces total bytes or file count. Large skills can exhaust Convex action memory or stall workers even when request count limits are respected.
  - confidence: 0.91
