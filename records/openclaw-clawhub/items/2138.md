## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.92

Full review comments:

- **[P1] Preserve the existing skill visibility boundary:** `convex/skills.ts:10412`
  - body: `listByDateRange` reads active digest rows and the handler then fetches internal latest-version blobs directly. That skips the current public/download checks that hide pending, removed, hidden, and malware-blocked skills, so the export can leak files that normal catalog and download APIs withhold.
  - confidence: 0.93
- **[P1] Avoid shadowing the export skill slug:** `packages/schema/src/routes.ts:30`
  - body: Adding `/api/v1/skills/export` inside the documented `/api/v1/skills/{slug}` namespace can make a normal or legacy `export` skill resolve to this privileged endpoint instead of its detail response. Use a reserved control path such as `/api/v1/skills/-/export` and reserve or migrate the namespace explicitly.
  - confidence: 0.86
- **[P2] Page mirror exports by updates, not creation time:** `convex/skills.ts:10412`
  - body: The PR uses the `by_active_created` index for incremental mirror sync. Existing skills that are republished, renamed, moderated, restored, or otherwise changed after creation will not be included in a later hourly created-date window, so mirrors can silently drift.
  - confidence: 0.9
- **[P2] Throttle failed export authentication:** `convex/httpApiV1/skillsV1.ts:1568`
  - body: The handler calls `requireExportAuth` before `applyRateLimit`, so missing or invalid bearer tokens return before consuming the export rate-limit bucket. Put an IP or failed-auth limiter in front of token lookup so the new privileged route cannot be used for unthrottled auth probing or token-hash lookup load.
  - confidence: 0.88
- **[P2] Cap exported bytes before reading blobs:** `convex/httpApiV1/skillsV1.ts:1665`
  - body: For up to 1000 skills the handler queues every file, reads each blob with `arrayBuffer()`, then zips and copies the result without an aggregate file or byte limit. A few large skills can exceed Convex action memory/time limits, so enforce aggregate caps before loading blobs.
  - confidence: 0.84
