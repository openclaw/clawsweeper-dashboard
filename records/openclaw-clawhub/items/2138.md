  - body: The handler runs `requireExportAuth` before `applyRateLimit`, so missing, invalid, and non-admin tokens return before the export bucket is consumed. Invalid-token traffic can still force token hashing and token/user lookups outside the intended throttling model for this expensive endpoint.
  - confidence: 0.88

  - body: The endpoint may select 1000 skills, fetch every file blob, convert every blob to an `ArrayBuffer`, and then build one ZIP, but it never caps total files or bytes. A set of large but valid skills can push the action over memory/runtime limits and fail the export path.
