## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Preserve public visibility filtering before exporting blobs:** `convex/skills.ts:10411-10413`
  - body: `listByDateRange` returns raw `skillSearchDigest` rows from `by_active_created`, and the handler then fetches blobs for every row. That index only constrains `softDeletedAt`, so hidden or blocked-malware skills can be archived despite the existing public API path filtering them out and the docs forbidding mirrors from bypassing moderation boundaries.
  - confidence: 0.92
- **[P2] Throttle failed export auth attempts:** `convex/httpApiV1/skillsV1.ts:1571-1574`
  - body: The handler returns `requireExportAuth` failures before `applyRateLimit`, so missing or invalid tokens never consume the new export bucket and do not receive rate-limit headers. Current API docs say missing/invalid token behavior falls back to IP enforcement, which this new path bypasses.
  - confidence: 0.9
- **[P2] Paginate mirror sync by updates, not creation time:** `convex/skills.ts:10408-10413`
  - body: The PR describes hourly incremental mirror sync, but this query pages `by_active_created`. Republish/update of an older skill changes `updatedAt` and `latestVersionId` without changing `createdAt`, so incremental exports can silently miss changed skills.
  - confidence: 0.88
- **[P2] Add an aggregate export size budget:** `convex/httpApiV1/skillsV1.ts:1740-1741`
  - body: A single allowed request can fetch every file blob for up to 1000 skills, convert each blob to an ArrayBuffer, keep all ZIP entries in memory, and then zip them, with no total byte or file-count cap. Large skills can exhaust Convex action memory before rate limits help.
  - confidence: 0.86
- **[P2] Avoid shadowing the skill slug route:** `convex/http.ts:77-79`
  - body: Registering exact `/api/v1/skills/export` makes that path an export endpoint instead of the documented `/api/v1/skills/{slug}` detail route, and I did not find a slug reservation for `export`. Existing or grandfathered skills with that slug would lose their API detail URL.
  - confidence: 0.76
