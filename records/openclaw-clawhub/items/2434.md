- **[P2] Handle empty search results before computing widths:** `packages/clawhub/src/cli/commands/skills.ts:162`
  - body: If the registry returns no matches, `rows` is empty and `maxColumnWidth(rows.map(...))` calls `reduce` without an initial value, so `clawhub search <rare-query>` throws instead of cleanly showing no results. Return 0 for empty columns or skip the table when there are no rows, and add a regression test.
  - confidence: 0.93

  - body: This new API field reads `skill.stats.downloads`, but the stats migration makes top-level `statsDownloads` the source of truth with nested stats only as fallback. During or after migration the CLI can display stale or zero downloads; include the top-level field from the search result and prefer it, or use `readCanonicalStat` where a full skill doc is available.
  - confidence: 0.88
