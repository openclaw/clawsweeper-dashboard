Preserve indexed queries and existing digest update behavior; do not replace the path with unbounded `.collect()`. Update `scheduleOwnerPublisherDigestSync` to start both rebuilds with the initial cursor shape, and add focused Vitest coverage in `convex/functions.test.ts` for continuation scheduling and one-page processing.
