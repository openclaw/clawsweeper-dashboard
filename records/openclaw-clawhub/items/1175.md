## What This Changes

Publishing an updated skill should no longer fail when personal-publisher creation triggers owner-publisher digest syncs that previously exceeded Convex's single-paginated-query limit.

## Best Possible Solution

## Solution Assessment

Yes. Paging one owner-publisher digest page per internal mutation with a continuation cursor is the narrow Convex-safe fix; the closed collect-based PR would be less bounded for large publishers.
