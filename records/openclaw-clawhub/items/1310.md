## Summary

Keep open. This is an open ClawHub issue, and current main still has security/moderation surfaces plus a narrow publish-time content quality gate rather than the requested public multidimensional quality scoring model.

## What This Changes

Request to add public multidimensional quality signals for skills, informed by a top-50 static-analysis study of quality and trust gaps beyond security scanning.

## Best Possible Solution

Define a product-owned public quality model distinct from security/moderation, then add agreed schema, API/search support, backfill or recomputation, and browse/detail UI presentation.
