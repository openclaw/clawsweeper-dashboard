## Summary

## What This Changes

The issue proposes public quality signals for ClawHub skills based on a top-50 static analysis that found trust gaps beyond existing security scans.

## Best Possible Solution

Define explainable quality dimensions separately from security and moderation verdicts, then add schema/API/UI/backfill support only after maintainers decide scoring authority, freshness, visibility, and rollout semantics.
