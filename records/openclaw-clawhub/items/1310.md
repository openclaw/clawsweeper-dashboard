## Summary

Keep open: ClawHub cleanup policy keeps issues open, and current main still lacks the requested public multidimensional skill quality-scoring model beyond security/moderation signals and a narrow publish-time quality hold. ## What This Changes

The issue requests public quality signals for ClawHub skills based on external top-50 analysis showing trust gaps beyond existing security scanning.

## Best Possible Solution

Define a product-owned quality-signal model distinct from security/moderation, reconcile it with https://github.com/openclaw/clawhub/issues/2042, then add approved schema, indexed API/search support, backfill, browse/detail UI, and operator documentation.
