## What This Changes

Publishing a skill through GitHub import fails with an import-during-publish GitHub API rate-limit error that waiting does not resolve.

## Best Possible Solution

Keep the 14-day immutable GitHub account-age gate, verify production GITHUB_TOKEN and 403/429 Convex logs, then reduce avoidable first-publish GitHub lookups or add safer retry/reset guidance without allowing failed lookups to pass.
