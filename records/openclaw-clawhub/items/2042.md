Keep open: this is a new ClawHub curation/tooling proposal, and repo policy keeps ClawHub issues open for maintainer review. Current main already has heuristic skill quality gates and moderation surfaces, but it does not integrate or document the proposed `agent-skill-infra` CLI modules.

## What This Changes

The issue proposes evaluating the external `agent-skill-infra` CLI as a complementary automated quality scoring and behavior testing tool for ClawHub skill curation.

## Best Possible Solution

Maintainers should decide whether this belongs as documented optional reviewer tooling or a narrow staff-only integration that feeds the existing quality/moderation surfaces without changing publish-time policy by default.
