## What This Changes

The issue proposes using the external agent-skill-infra CLI for automated skill quality scoring, behavior tests, and version-drift checks in ClawHub curation.

## Best Possible Solution

Define a product-owned quality-signal model distinct from security/moderation, decide whether agent-skill-infra is an optional curator tool or integrated service, then add approved schema, indexed API/search support, UI, backfill, docs, and evaluator data-boundary policy.
