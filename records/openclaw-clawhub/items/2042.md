Keep open: this is a ClawHub feature/workflow adoption proposal, not a current-main bug. ## What This Changes

## Risks / Open Questions

- Endorsing a third-party Python CLI needs package provenance, version pinning, sandboxing, and update-policy review.

- Adding another quality signal without a narrow contract could conflict with existing quality gate, scanner, moderation, search, and public trust semantics.
- Any public quality sort/filter should use denormalized indexed fields to avoid Convex query and bandwidth regressions.
