## What This Changes

The issue asks ClawHub to consider an external automated skill quality-scoring CLI for curation, including semantic scoring, behavior tests, and version-drift checks.

## Best Possible Solution

Maintainers should decide whether this belongs as optional reviewer documentation, staff-only tooling, CI guidance, or a formal ClawHub quality signal, then define supply-chain, privacy, schema/API, and UI boundaries before implementation.
