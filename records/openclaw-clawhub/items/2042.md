## Summary

Keep open: this is a ClawHub workflow/tooling adoption proposal, not a current-main bug. Current main has native publish-time quality and scan/moderation paths, but no `agent-skill-infra` integration or reviewer workflow, and ClawHub cleanup policy keeps issues open for maintainer product/security judgment.

## What This Changes

The issue proposes using the external `agent-skill-infra` CLI modules as an automated first-pass quality scoring, behavior testing, and version-drift tool for ClawHub skill curation.

## Best Possible Solution

Maintainers should decide whether this belongs as optional reviewer documentation, staff-only tooling, CI guidance, or a formal ClawHub signal, then define the privacy, supply-chain, schema/API, and UI boundaries before implementation.
