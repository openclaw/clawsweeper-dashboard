Keep open: ClawHub cleanup policy keeps issues open, and current main has structural quality gating plus security scan paths but no `agent-skill-infra` integration or semantic, behavior-test, and version-drift curation workflow. ## What This Changes

The issue proposes evaluating the external `agent-skill-infra` CLI for ClawHub curation, including semantic quality scoring, eval-based behavior tests, and version-drift checks.

## Best Possible Solution

Maintainers should decide whether this belongs as optional reviewer documentation, staff-only tooling, CI guidance, or a formal ClawHub quality signal, then define supply-chain, privacy, schema/API, and UI boundaries before implementation.
