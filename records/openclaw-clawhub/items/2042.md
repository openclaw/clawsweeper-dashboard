## Summary

Keep open: this is a ClawHub issue and a new curation/tooling adoption proposal, not a current-main bug. Current main has a narrower publish-time quality gate and security/moderation scans, but no `agent-skill-infra` integration or agreed reviewer workflow, so maintainer product and security judgment is still needed.

## What This Changes

The issue proposes adopting or documenting the external `agent-skill-infra` CLI as an automated first-pass quality scorer, behavior tester, and version-drift checker for ClawHub skill curation.

## Best Possible Solution

Maintain the current quality and moderation pipeline until maintainers decide whether `agent-skill-infra` should become optional documentation, reviewer tooling, CI guidance, or a narrow staff-only signal after privacy and supply-chain review.
