Keep open: this is a new ClawHub curation/tooling proposal, not an already-implemented fix. Current main has heuristic quality gates, moderation state, scanner evidence, and staff tooling, but it does not integrate or document the proposed external `agent-skill-infra` modules; the related OpenClaw core request at https://github.com/openclaw/openclaw/issues/75958 was redirected here for ClawHub maintainer judgment.

Request to consider `agent-skill-infra` as automated semantic scoring, behavior testing, and version-awareness tooling for ClawHub skill curation.

## Best Possible Solution

Maintainers should decide whether this belongs as documented optional reviewer tooling or as a narrow staff-only integration that feeds existing quality and moderation evidence without changing publish-time policy by default.
