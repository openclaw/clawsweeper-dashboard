## Summary

Current main only implements deterministic anti-spam checks for thin or templated skills; the proposed semantic scoring, eval-driven behavior testing, and version-baseline workflow remain unimplemented and require a maintainer product and security decision.

## What This Changes

Assess or integrate the external agent-skill-infra CLI as a complementary automated skill-quality, behavior-testing, and version-drift signal for ClawHub curation.

## Best Possible Solution

Define one advisory quality-signal contract through the related product discussion, then either document this CLI as optional non-authoritative reviewer tooling or integrate vetted, explainable outputs while keeping deterministic enforcement separate.
