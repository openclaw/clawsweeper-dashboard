## What This Changes

The issue proposes using `agent-skill-infra` as an automated skill quality scoring, behavior testing, and version-awareness tool for ClawHub curation workflows.

## Best Possible Solution

Define a product-owned quality-signal model distinct from security and moderation, decide whether `agent-skill-infra` is an optional curator tool or integrated service, then add approved schema, indexed API/search support, UI, backfill, docs, and evaluator data-boundary policy.
