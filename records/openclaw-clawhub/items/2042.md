## Summary

Keep open: this ClawHub issue proposes a new curation and trust-signal workflow, and current main still has only a narrow publish-time quality gate plus separate security audit surfaces. The linked agent-skill-infra toolkit is relevant, but adoption needs maintainer product and security policy decisions rather than cleanup closure.

## What This Changes

Request to consider agent-skill-infra as an automated skill quality, behavior-test, and version-awareness tool for ClawHub’s curation workflow.

## Best Possible Solution

Define a product-owned quality-signal model distinct from security and moderation, decide whether agent-skill-infra is an optional curator tool or integrated ClawHub service, then add approved schema, indexed API/search support, UI, backfill, operator docs, and evaluator data-boundary policy.
