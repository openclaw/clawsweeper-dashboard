## Summary

Keep open: current main has deterministic publish-time quality gating plus ClawScan, SkillSpector, VirusTotal, plugin-inspector, and `clawhub scan` surfaces, but it still lacks the requested agent-skill-infra-style semantic scoring, eval-based behavior testing, and version-drift curation workflow. ## What This Changes

The issue proposes evaluating agent-skill-infra as complementary automated semantic, behavior-test, and version-drift quality tooling for ClawHub skill curation.

## Best Possible Solution

Define one advisory quality-signal contract and data-handling boundary, then decide whether to document agent-skill-infra as optional reviewer tooling or integrate vetted outputs separately from security verdicts and enforcement decisions.
