## What This Changes

The issue proposes using or adapting `agent-skill-infra` for automated skill quality scoring, behavior testing, and version-awareness in ClawHub curation.

## Best Possible Solution

Define a product-owned quality-signal model distinct from security and moderation, coordinate it with https://github.com/openclaw/clawhub/issues/1310, then add approved schema, indexed API/search support, UI, backfill, docs, and evaluator data-boundary policy.
