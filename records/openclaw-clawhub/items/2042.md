The issue proposes evaluating the external `agent-skill-infra` CLI as a complementary ClawHub curation tool for semantic skill scoring, behavior evals, and version-drift checks.

## Best Possible Solution

Maintainers should decide whether this belongs as optional reviewer documentation, staff-only tooling, CI guidance, or a formal ClawHub quality signal, then define supply-chain, privacy, schema/API, UI, and moderation boundaries before implementation.
