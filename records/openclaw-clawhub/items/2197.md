The issue reports that the production homepage GitHub sign-in button does nothing because Chrome reports CSP blocking JavaScript eval before OAuth starts.

## Best Possible Solution

Preserve the strict CSP, verify the affected production/regional login path with browser diagnostics and Convex/GitHub OAuth logs, then make the narrowest auth UI or configuration fix that lets OAuth start or surfaces an actionable error.
