## What This Changes

The reporter asks maintainers to restore a deleted ClawHub account or remove its GitHub OAuth binding so the same GitHub identity can sign up again.

## Best Possible Solution

A maintainer should handle the specific account through support/admin tooling, then update code and docs only if ClawHub decides deleted GitHub identities should be unlinkable or reusable.
