## Best Possible Solution

Keep a canonical OAuth/session tracker open until maintainers verify production GitHub sign-in end to end for fresh and existing active accounts, inspect production OAuth/Convex/Vercel settings and logs, fix the confirmed root cause, and add redacted smoke proof plus focused regression coverage.
