## What This Changes

## Best Possible Solution

Define a supported deleted-account lifecycle: either keep deletion terminal with clearer support messaging, or add an audited, identity-verified restore/unlink/recreate flow that preserves deletion, OAuth, moderation, token, and audit safeguards.

## Reproduction Assessment

No for a direct automated clear or ordinary unban. The maintainable path needs maintainer security/product judgment plus audited identity-verified recovery, identity unlinking, account recreation, or an explicit terminal-support policy.
