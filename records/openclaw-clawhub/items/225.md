## Best Possible Solution

Define a maintainer-approved shared skill preflight layer that browser upload, direct CLI publish, and local scanner tooling can reuse, with stable lint/dry-run output, hosted scanner parity, ignore guidance, and moderation-approved abuse-policy wording.
