## Summary

## Best Possible Solution

Keep the shipped admin repair-name path for one-off migrations, and design any reusable runtimeId correction as an explicit moderated/admin or plugin-settings workflow with dry-run planning, authority checks, collision rejection, audit logs, and focused tests.
