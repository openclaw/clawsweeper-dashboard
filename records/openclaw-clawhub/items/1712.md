Current main partially addresses the report by counting successful package archive downloads, but package install counts still have no ClawHub-owned writer and skill install counts remain tied to authenticated sync telemetry rather than immediate `skill add` activity; ClawHub cleanup policy also keeps issues open unless they are PRs implemented on main.
