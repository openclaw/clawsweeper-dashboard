## Summary

Keep open. Current main has the package archive download counter fix, but package installs/stars still lack defined writer semantics and direct skill installs are still counted only through documented sync telemetry, so the remaining work needs maintainer product/privacy direction.

## What This Changes

Request that package download/install/star counters and skill install counters increment from real ClawHub/OpenClaw usage instead of staying zero.

## Best Possible Solution

Keep the shipped package download counter, then document the intended package install/star and skill install telemetry semantics in specs/docs before adding explicit signals that preserve the privacy controls.
