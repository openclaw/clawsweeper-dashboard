## What This Changes

Make homepage/unified search and `/skills` return consistent skill results for the same query, or clearly disclose that suspicious-skill filtering differs.

## Best Possible Solution

Review and land or close https://github.com/openclaw/clawhub/pull/2079; once current main clearly aligns the defaults or discloses hidden suspicious filtering, this can close as implemented.

Yes. Preserving the safer unified-search default while adding clear disclosure is the narrowest maintainable direction, but it should land through the linked PR after review and conflict resolution.
