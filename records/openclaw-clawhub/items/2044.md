## What This Changes

The report asks homepage/unified search and `/skills` search to return consistent skill results or visibly explain differing suspicious-skill filters.

## Best Possible Solution

Land or revise https://github.com/openclaw/clawhub/pull/2079 so public search keeps a documented moderation-visibility contract and clearly discloses when suspicious skill results are hidden.
