## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P1] Preserve multi-value response headers when wrapping routes:** `convex/lib/httpRouteRateLimit.ts:103-106`
  - body: Wrapping `auth.addHttpRoutes` means successful OAuth/dev-auth responses now pass through this reconstruction. `mergeHeaders` converts `response.headers` into a plain object via `Headers.entries()`, which collapses multiple `Set-Cookie` values into one comma-joined value, so callbacks that set more than one cookie can send invalid cookies and fail sign-in. Preserve multi-value headers when adding rate-limit headers, or avoid rebuilding auth responses, and add coverage for a wrapped response with multiple `Set-Cookie` headers.
  - confidence: 0.86
