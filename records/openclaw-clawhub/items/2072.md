## Summary

Keep open: current main does not implement this PR, and the latest patch still has a source-visible stale digest bug, a missing required changelog entry, and no after-fix real behavior proof.

## What This Changes

This PR enforces scoped package publishes against matching publisher ownership and adds package-owner transfer support across Convex APIs, the HTTP API, CLI, schema artifacts, tests, and the plugin detail UI.

## Best Possible Solution

Merge only after ownership transfers keep package and capability search digests synchronized, the user-facing change is documented in the changelog, and the contributor posts redacted after-fix proof.

## Reproduction Assessment

Yes, from source inspection: the new transfer helper patches the packages row while catalog/search owner fields are served from denormalized digest rows. A live check would transfer a package and compare detail owner data against catalog/search owner data.

## Solution Assessment

No, not yet. The transfer implementation should update packageSearchDigest and packageCapabilitySearchDigest in the same mutation, then add the required changelog entry and real behavior proof.
