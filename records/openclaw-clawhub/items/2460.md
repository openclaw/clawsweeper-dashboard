## Summary

The feature is not present on current main and contains meaningful unique work, so it should stay open, but the current draft is not merge-ready: its bounded source reads can produce incomplete attested feeds, the public trust-lane contract lacks the repository-required durable spec, and no real Convex-backed behavior proof is attached.

## What This Changes

The PR adds five public ClawHub feed endpoints that expose all, official, community, and reviewed skill/plugin catalog lanes with deterministic entries, source revisions, SHA-256 attestations, tests, route constants, and HTTP API documentation.

## Best Possible Solution

Define the feed contract in a focused `specs/` document, reuse canonical catalog visibility and installability predicates, and serve a cursor-paginated or materialized snapshot whose completeness, truncation, revision, and attestation semantics are explicit and proven on real Convex data.
