## What This Changes

Adds five public HTTP feed endpoints for discovering all, official, community, and reviewed ClawHub skills/plugins, with feed hashing, tests, schema routes, and API documentation.

## Best Possible Solution

Approve a shared feed contract first, then implement each lane from canonical ClawHub eligibility fields/helpers with indexed cursor pagination or cached materialization, document the durable trust and attestation guarantees in `specs/`, and prove representative live responses on real Convex data.
