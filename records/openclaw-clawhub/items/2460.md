## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Reuse public skill visibility for feed lanes:** `convex/feeds.ts:12-31`
  - body: The feed matcher applies lane rules directly to skill digests, so `all`, `official`, and `community` can expose hidden or removed skills, and the digest shape here cannot exclude `blocked.malware` flags. Existing catalog output goes through the public skill visibility gate; apply that same gate before lane-specific matching.
  - confidence: 0.93
- **[P1] Keep blocked packages out of public feeds:** `convex/feeds.ts:43-53`
  - body: For package feeds, `all`, `official`, and `community` only exclude private packages, but current package reads also hide packages blocked from public visibility by scan status unless the viewer has owner access. This would advertise blocked packages through a new unauthenticated endpoint.
  - confidence: 0.91
- **[P2] Apply read rate limits to feed endpoints:** `convex/httpApiV1/feedsV1.ts:368-385`
  - body: `feedV1Handler` runs the catalog query and returns a cacheable response without the normal read rate-limit path used by existing public API handlers. Add read rate limiting and carry those headers into the response before exposing these endpoints.
  - confidence: 0.87
- **[P2] Enforce the requested limit after merging feed entries:** `convex/feeds.ts:68-91`
  - body: With both types enabled, `limit=1` sets `perKindLimit` to 1 and can return one skill plus one plugin, so the feed can exceed its advertised limit and misstate truncation. Merge and sort candidates, then slice to the total limit and compute truncation from over-fetch or cursors.
  - confidence: 0.86
- **[P2] Avoid broad digest scans for every feed lane:** `convex/feeds.ts:70-83`
  - body: Each request can read 5,000 active skill digests and 5,000 active package digests, then filter by lane in JavaScript even when indexed fields like channel, official status, or suspicious status are available. Use indexed, paginated lane-specific reads so the public feed does not become a bandwidth hotspot.
  - confidence: 0.82
