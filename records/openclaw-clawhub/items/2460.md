- **[P1] Keep root feed reads behind rate limiting:** `convex/feeds.ts:59-65`
  - body: This registers `rootFeed` as a public Convex query, and the HTTP handler calls it without `applyRateLimit`. A caller can invoke the Convex query directly, or hit the new feed routes, to make the backend read the recent skill window plus both plugin-family windows without consuming the documented read quota.
