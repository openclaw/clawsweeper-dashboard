## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Gate skill fallback checks on a true fresh cursor:** `convex/skills.ts:6122`
  - body: Here `!cursor` treats any decoded cursor whose underlying Convex cursor is null as a fresh request. That includes a valid in-progress skill catalog cursor with `offset > 0`; if recommendation scores become missing or stale between page requests, the next page switches from the recommended index to updated while reusing the old offset, which can skip or duplicate rows. Mirror the fresh-start state check used in `packagesV1.ts` and add a regression for an in-progress recommended cursor without `recommendedFallback`.
  - confidence: 0.9
- **[P2] Align the visible filtered sort with the loader default:** `src/routes/plugins/index.tsx:171`
  - body: When filtered browse has no explicit sort, this now sends `sort: updated` to the API, but `activeSort` still defaults to `recommended` for the sidebar. The page can show Recommended selected while results are loaded as Recently updated; compute the visible default with the same helper or cover the implicit filtered state in the route test.
  - confidence: 0.92
