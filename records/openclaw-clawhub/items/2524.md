## Summary

The issue asks ClawHub to use an official ClawHub/OpenClaw-owned GitHub OAuth App so the web and CLI consent screen no longer shows personal ClawdHub branding.

## Best Possible Solution

Transfer or recreate the GitHub OAuth App under official OpenClaw/ClawHub control, preserve the deployed callback contract, rotate the Convex production credentials, and capture before/after web plus CLI auth proof.
