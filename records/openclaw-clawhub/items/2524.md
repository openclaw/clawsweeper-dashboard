## What This Changes

GitHub OAuth sign-in should show official ClawHub/OpenClaw-owned OAuth app branding instead of a personal ClawdHub app on the consent screen.

## Best Possible Solution

Have a maintainer create or transfer the production GitHub OAuth App to the official owner, rotate Convex production credentials together, and verify web plus CLI sign-in against the live consent screen.
