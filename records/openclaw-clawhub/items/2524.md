## Summary

## What This Changes

The issue asks for ClawHub GitHub sign-in to use an official ClawHub/OpenClaw-owned OAuth App identity instead of a personal ClawdHub app identity on the consent screen.

## Best Possible Solution

Move or recreate the GitHub OAuth App under official OpenClaw/ClawHub control, preserve the production callback contract, rotate Convex production credentials, and capture fresh web plus CLI auth proof.
