## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.78

Full review comments:

- **[P2] Verify the fetched GitHub archive before installing it:** `packages/clawhub/src/cli/commands/skills.ts:914-917`
  - body: The resolver returns a verifier-produced `contentHash`, but the CLI currently downloads the codeload zip and extracts it without checking that the installed folder matches that hash. If the archive source or override returns different bytes for the pinned commit, the CLI can install content that ClawHub did not verify; recompute the same folder hash after extraction and fail before writing origin/lock metadata on mismatch.
  - confidence: 0.73
- **[P2] Batch each GitHub source sync before writing skills:** `convex/githubSkillSync.ts:364-367`
  - body: The scheduled action pages sources, but each source still collects every existing source-backed skill and then patches/inserts every discovered skill and content row in a single mutation. A large official repo can exceed Convex transaction limits and make the 15-minute cron fail repeatedly; page or cap per-source work and schedule continuation.
  - confidence: 0.78
