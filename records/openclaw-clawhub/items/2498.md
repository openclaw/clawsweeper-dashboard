## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P1] Verify the GitHub archive before accepting the install:** `packages/clawhub/src/cli/commands/skills.ts:917`
  - body: `resolution.github.contentHash` is returned by the install resolver, but this path fetches the codeload zip and extracts it without comparing the installed folder to that verified hash. A bad archive, mirror override, or extraction mismatch would still be written as a successful install/update, so compute the same source-folder hash before accepting the install and fail on mismatch.
  - confidence: 0.92
- **[P2] Batch source skill writes before syncing large repos:** `convex/githubSkillSync.ts:413`
  - body: A source sync still reads every existing skill for the source and then patches/inserts every planned skill in one mutation. If an official source grows large enough, the 15-minute cron can exceed Convex transaction limits and keep failing; split the per-source skill work into cursor-sized mutation batches instead of applying the entire plan at once.
  - confidence: 0.84
