## What This Changes

Prevent the skill detail view from overflowing or becoming unusable on iOS Safari and other narrow mobile viewports.

## Best Possible Solution

## Solution Assessment

Yes. Constraining oversized skill content to its parent container with local scrolling, then guarding the page in a real mobile browser test, is the narrowest durable solution.
