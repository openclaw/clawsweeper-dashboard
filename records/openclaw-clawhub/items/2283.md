## What This Changes

The PR hides personal package/plugin rows during user bans or account deletion, revokes related package publish tokens, restores only matching ban-hidden packages on unban, and documents/tests that behavior.

## Best Possible Solution

Land the security fix after maintainer review and redacted live proof shows ban, delete, and unban flows hide or restore personal packages and revoke tokens without affecting org-owned packages.
