## What This Changes

Restore GitHub OAuth access for an existing ClawHub account after its GitHub username changes.

## Best Possible Solution

Verify the reporter through the immutable GitHub numeric ID, inspect the corresponding production auth account and user state, then perform the narrowest audited repair that preserves content ownership; only use destructive purge after explicit confirmation and a clear impact inventory.
