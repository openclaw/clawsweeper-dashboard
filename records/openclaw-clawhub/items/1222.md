The issue reports skill publishing failing with a GitHub API rate-limit error during upload.

Keep the account-age gate intact, verify production `GITHUB_TOKEN` and 403/429 logs first, then reduce avoidable publish-time GitHub lookups or safely backfill/cache `githubCreatedAt` if maintainers choose a code mitigation.
