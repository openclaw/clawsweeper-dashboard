## What This Changes

The issue reports that publishing a skill fails with a GitHub API rate-limit error during upload/publish.

## Best Possible Solution

Keep the account-age gate, verify production `GITHUB_TOKEN` quota and Convex 403/429 logs, then reduce avoidable publish-time GitHub lookups through safe caching/backfill or clearer retry/reset handling without weakening immutable-ID account-age enforcement.
