  - body: `rawFsNamespaces` starts with `fs`, so package-local code such as `const fs = makeVirtualFilesystem(); fs.readFile(path)` would be reported as raw Node filesystem usage without any `fs` or `node:fs` import. Start empty and add a regression for virtual fs calls.
