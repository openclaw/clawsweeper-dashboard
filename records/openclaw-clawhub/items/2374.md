## What This Changes

Fix hybrid skill search scoring so lexical deduplication does not discard the same skill's vector similarity contribution.

## Best Possible Solution

## Reproduction Assessment

Yes. Current-main source provides a deterministic reproduction: direct-first deduplication can retain an entry without embeddingId, and the subsequent score calculation assigns zero vector contribution despite a vector match for the same skill.

## Solution Assessment
