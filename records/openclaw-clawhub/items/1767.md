## Best Possible Solution

Define a maintainer-approved local scanner contract that reuses shared manifest/schema validation and deterministic static moderation logic, adds missing plugin/SKILL metadata checks, documents hosted parity, and exposes stable human plus JSON output through the chosen public CLI/package surface.
