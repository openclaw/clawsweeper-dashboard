## What This Changes

Update skill-page install codeblocks and copied prompts to use the canonical owner-scoped `@owner/slug` install reference.

## Best Possible Solution

Adopt one maintainer-approved `@owner/slug` grammar across OpenClaw support, ClawHub CLI parsing, the skill-page install surface, documentation, and focused compatibility tests while retaining intentional legacy slug-only behavior.

## Reproduction Assessment

## Solution Assessment
