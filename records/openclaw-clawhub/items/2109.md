Keep open. The issue asks ClawHub to expose scanner finding details such as rule ID and file:line in package moderation-status or inspect output, and to document server-side scanner rule IDs for false-positive triage.

Define an owner/staff-safe diagnostic contract using existing staticScan code/file/line/message fields, document the stable CLI/API behavior, and keep broader scanner internals behind an approved disclosure policy.
