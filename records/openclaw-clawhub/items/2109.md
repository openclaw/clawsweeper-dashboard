## What This Changes

## Best Possible Solution

Expose a narrow owner/staff-supported path for existing staticScan code/file/line/message details in moderation-status and human inspect, document the stable fields and current version-JSON workaround, and require security approval before exposing raw evidence, regexes, LLM/VT internals, or broader rule catalogs.
