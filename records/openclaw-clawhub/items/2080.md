The issue requests a maintainer-supported recovery path or clearer deletion semantics when a UI-deleted plugin package remains reserved or mapped to a stale publisher.

Maintainers should inspect the affected production package rows, choose an audit-logged restore, transfer, identity repair, or deletion policy, and then clarify owner-facing deletion and recovery semantics.
