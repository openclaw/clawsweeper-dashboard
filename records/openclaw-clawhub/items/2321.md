Keep this open: this collaborator-authored moderation/security RFC is still an active policy decision. ## What This Changes

The issue asks maintainers to define content-policy review categories and default consequences for spam, crypto/blockchain/finance/trading content, scanner-detected malicious artifacts, visibility, installability, upload review, and account enforcement.
