## What This Changes

The issue reports that starring skills and downloading skills through the CLI or website do not update the visible skill metrics.

## Best Possible Solution

The desired end state is a documented, tested metrics path where star and download events reliably reach visible counters within an explicit freshness target without creating hot-document contention.
