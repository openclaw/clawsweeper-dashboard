## What This Changes

Re-link the existing `@gingiris` personal publisher to the reporter’s current GitHub account, or provide a staff-assisted way to publish the corrected skill versions.

## Best Possible Solution

After trusted identity verification against the linked moderation history, use a narrowly scoped, audited staff operation that atomically reassigns the `@gingiris` personal publisher and its ownership records to the verified current user, then verify profile ownership and a safe publish while keeping ordinary cross-account checks unchanged.
