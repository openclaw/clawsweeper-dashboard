Fix the silent `clawhub package publish` hang reported in https://github.com/openclaw/clawhub/issues/2219. Implement a narrow reliability fix so non-interactive package publishes emit useful progress or diagnostics before the live publish request and stalled multipart publishes fail with a clear bounded error.
