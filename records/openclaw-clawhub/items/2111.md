  - body: `mergeTargetSlug` is initialized from `ownedSkills[0]` only on the first render. When the owned-skill query resolves from empty to non-empty, the Select gets a value that matches no item and the Update button stays disabled because `mergeTargetSlug` is still empty.
  - confidence: 0.82

  - body: `toHaveBeenCalledBefore` is not a Vitest matcher installed by this repo’s setup, so this new test will fail at runtime or during type checking. Compare `mock.invocationCallOrder` or add an explicit local matcher.
  - confidence: 0.9
