  - body: This narrows `DetailTab` to README/files/install metadata while deleting the compare/version rendering path. Current skill pages let users compare releases, inspect older changelogs, and download older ZIPs; after this change those established flows have no replacement.
  - confidence: 0.9

  - confidence: 0.88

  - body: `toHaveBeenCalledBefore` is not a Vitest matcher installed by this repo's setup, so this new test will fail at runtime or type checking. Compare `mock.invocationCallOrder` or add an explicit local matcher.
  - confidence: 0.91

  - body: `mergeTargetSlug` is initialized from `ownedSkills[0]` only on the first render. When `ownedSkills` loads later, the Select can hold an empty value that matches no item and keeps the merge action disabled despite available targets.
  - confidence: 0.86
- **[P3] Collapse the duplicate skill breadcrumb segment:** `src/components/SkillHeader.tsx:227-230`
  - body: The skill hero breadcrumb renders `skills / owner / skills / slug`, so the same collection segment appears twice and the second link sends users back to the same listing hierarchy. Render the collection segment once.
  - confidence: 0.8

  - body: The security report breadcrumb renders `skills/plugins / owner / skills/plugins / artifact / scanner`, which duplicates the collection segment and adds a confusing dead-end navigation step. Render a single collection crumb before the artifact.
  - confidence: 0.8
