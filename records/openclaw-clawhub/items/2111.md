  - body: `mergeTargetSlug` is initialized from `ownedSkills[0]` only on the first render. When `ownedSkills` loads later, the Select can hold an empty value that matches no item and keeps the merge action disabled despite available targets.
  - confidence: 0.86

  - body: The skill hero breadcrumb renders `skills / owner / skills / slug`, so the same collection segment appears twice and the second link sends users back to the same listing hierarchy. Render the collection segment once.
