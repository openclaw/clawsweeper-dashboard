- **[P2] Restore the skill history and compare tabs:** `src/components/SkillDetailTabs.tsx:18`
  - body: This narrows the skill detail tabs to README, files, and install metadata, while the PR also deletes SkillDiffCard, SkillVersionsPanel, and the diffing helpers.   - confidence: 0.88

  - body: `tabFromHash()` accepts install-tab hashes unconditionally, but `SkillDetailTabs` only renders those tabs when metadata exists. A stale `#runtime`, `#dependencies`, `#install`, or `#links` URL on a skill without that metadata leaves the pane blank instead of showing README or Files.
  - confidence: 0.86
- **[P2] Replace the non-standard matcher:** `src/components/DevPersonaFab.test.tsx:124`
  - body: `toHaveBeenCalledBefore` is not a built-in Vitest matcher, and the repo setup does not install an extension that provides it. This test will fail at runtime; compare `mock.invocationCallOrder` values instead.
  - confidence: 0.84
- **[P3] Sync the merge target after owned skills load:** `src/components/SkillOwnershipPanel.tsx:212`
  - body: `mergeTargetSlug` is initialized from the first render only. When `ownedSkills` starts empty and later populates, the Select can keep an empty value that matches no item and leaves the merge action disabled until the user manually reselects a target.
  - confidence: 0.77
