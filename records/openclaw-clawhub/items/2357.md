## What This Changes

Plugin detail pages should rewrite relative README image/resource URLs to a resolvable package or stable source asset location instead of leaving them route-relative.

## Best Possible Solution

Choose and implement a version-stable, sanitized README asset resolution path, then rewrite relative plugin README image/resource URLs and cover it with regression tests.

## Solution Assessment

Unclear. Rewriting relative README assets is the right target, but maintainers need to choose between a safe package asset endpoint and a stable source/raw URL strategy before implementation.
