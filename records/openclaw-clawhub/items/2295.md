## Summary

Keep open. Current main still installs skill files and ClawHub metadata only, while automatic OpenClaw hook registration would add a security-sensitive runtime handoff that needs maintainer policy first.

## Best Possible Solution

Define a maintainer-approved hook registration contract first, then implement explicit detection, disclosure, consent, registration, docs, and tests in the chosen ClawHub/OpenClaw install path.
