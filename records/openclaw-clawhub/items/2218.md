This PR should stay open because the tooltip feature is useful and mostly scoped, but the new preview copy ignores the existing staff-cleared suppression path and can surface raw suspicious scanner summaries under a Cleared label.

## Best Possible Solution

Keep the tooltip direction, but make preview copy honor cleared/suppressed scan state and add a regression test before normal maintainer merge.
