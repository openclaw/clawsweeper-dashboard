Keep this PR open: the visual proof and CI look useful, but the new preview text bypasses the existing staff-cleared scan suppression path and can expose raw suspicious scanner summaries in a public tooltip.

## What This Changes

The branch adds hover/focus audit preview tooltips with capped overview text, a full-audit CTA, tests, styles, and a changelog entry.

## Best Possible Solution

Keep the tooltip feature, but make its overview text honor `suppressScanResults` with public-safe copy and add regression coverage for staff-cleared previews.

## Reproduction Assessment

## Solution Assessment

No; the preview implementation should reuse suppressed/public-safe text when scan results are cleared, rather than reading raw scanner summaries unconditionally.
