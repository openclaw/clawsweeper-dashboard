## Summary

Keep this PR open: the tooltip UX is useful and has visual proof, but the current patch can expose suppressed scanner summaries in staff-cleared audit previews.

## What This Changes

Adds hover/focus preview tooltips with capped scanner overview text and a Read full audit CTA to compact audit cards, plus tests, CSS, and a changelog note.

## Best Possible Solution

Keep the tooltip UX, but gate preview overview copy on suppressScanResults and add focused regression coverage for the staff-cleared state before merge.

## Reproduction Assessment

Yes. Source inspection gives a high-confidence path: render DetailSecuritySummary on the PR head with suppressScanResults and suspicious scanner summaries, then focus an audit card; the card status is Cleared while the preview text comes from raw scanner summaries.

## Solution Assessment

No, not as currently written. The narrow maintainable fix is to make tooltip overview text honor scan suppression, using the cleared/suppressed public copy or generic cleared copy instead of raw scanner summaries.
