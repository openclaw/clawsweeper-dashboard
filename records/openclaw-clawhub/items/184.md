## Summary

Keep open. This is a ClawHub issue, and the cleanup policy only permits auto-closing ClawHub pull requests that current main certainly implements. Substantively, current main has ordinary badges and staff manual-clean overrides, but no paid expedited human-review certificate or badge program.

## What This Changes

Add a paid expedited human security-review pipeline for skills, with a public certificate or badge after manual approval.

## Best Possible Solution

Maintainers should first decide whether ClawHub should offer paid expedited human review at all. If approved, define the trust model separately from verified-publisher identity, then implement it end-to-end across schema/payment state, review queue, moderator workflow, badge/API display, audit trail, and docs.
