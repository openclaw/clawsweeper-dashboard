Keep open. ClawHub cleanup policy keeps issues open, and current main still lacks the requested paid expedited human-review certificate or skill badge workflow; the related verified-publisher work in #192 is complementary trust work, not a paid code-review approval product.

## What This Changes

The issue requests a paid expedited manual security review and approval pipeline for skills, with a public certificate or badge for approved skills.

## Best Possible Solution

Maintainers should first decide whether ClawHub should offer paid expedited human review; if approved, implement it end to end across schema/payment state, review queue, staff workflow, public badge/API display, auditability, and docs.
