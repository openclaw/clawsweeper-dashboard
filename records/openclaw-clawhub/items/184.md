## Summary

Keep open. Current main has ordinary skill badges, official/trusted publisher primitives, security scanning, and moderator manual overrides, but not a paid expedited human-review certificate; the remaining work is security-sensitive product/payment policy rather than cleanup.

## What This Changes

Add a paid expedited human-review pipeline and public certificate/badge for skills that pass manual security review.

## Best Possible Solution

Maintainers should first decide whether paid expedited human review should exist, then implement payment state, review queues, moderator workflow, public badge/API display, audit trails, revocation/appeals, and docs as one coherent feature.
