## Summary

Keep open. ClawHub cleanup policy keeps issues open, and current main still has ordinary skill badges, scanner/moderation state, and moderator manual-clean overrides rather than a paid expedited human-review certificate program.

## What This Changes

Request a paid expedited human review and approval badge or certificate for ClawHub skills that pass manual security review.

## Best Possible Solution

Maintainers should first decide whether ClawHub should offer paid expedited human review; if approved, capture the trust model in specs before implementing payment state, review queues, moderator workflow, public badge/API display, audit trails, appeals, and docs.
