## Summary

Keep open. Current main has ordinary skill badges, trusted-publisher primitives, security scanning, package moderation queues, and moderator manual overrides, but it does not implement a paid expedited human-review certificate or badge program.

## What This Changes

Add a paid expedited manual security-review approval pipeline and public certificate or badge for skills that pass human review.

## Best Possible Solution

Maintainers should first decide whether paid expedited human review should exist, then define the trust, payment, review, audit, revocation, and public wording model before implementing schema, queues, UI, API display, and docs.
