Keep open. This ClawHub issue asks for a paid security-trust feature, and current main still only has ordinary skill badges, staff manual-clean overrides, and publisher/package verification primitives rather than a paid expedited human-review certificate program.

Maintainers should first decide whether ClawHub should offer paid expedited human review; if approved, implement the trust model end-to-end across schema, payment state, review workflow, public badge/API display, audit trail, and docs.
