## Summary

Keep this RFC open: current main has improved all-time plugin and skill ranking, but it still lacks the package and publisher daily rollups, monthly leaderboards, snapshot-pinned pagination, and coordinated period controls needed to solve the central cross-surface discovery request.

## What This Changes

Add weekly, monthly, and all-time popularity modes for publisher, skill, and plugin discovery using consistent temporal statistics and stable pagination.

## Best Possible Solution

Approve a simple V1 metric and ownership-attribution rule, persist those invariants in a focused spec, then implement indexed daily package and publisher rollups plus snapshot-stable entity leaderboards before adding synchronized period controls to the three browse surfaces.
