## What This Changes

The issue reports that CLI and web skill publishing fail because personal publisher resolution can hit Convex multiple-pagination errors and leave the owner dropdown without a usable personal publisher.

## Best Possible Solution

A maintainer should deploy or confirm the current backend, smoke-test CLI and web publishing for both a first-time account and an affected existing publisher, then close or consolidate the related reports if production passes. If large-owner digest pagination still fails, the remaining implementation should batch owner-publisher digest rebuilds through cursor-carrying scheduled mutations.
