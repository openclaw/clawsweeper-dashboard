## What This Changes

## Best Possible Solution

Land the feature only after the moderated getBySlug fallback preserves icon, regression coverage covers that update flow, conflicts are resolved, and the contributor adds redacted real publish/update proof.

No; the overall direction is plausible, but the current diff must preserve icon in the owner-visible moderated fallback and include real behavior proof before it is a complete solution.
