Display the reported skill's actual MIT license instead of the platform MIT-0/no-attribution license, and check whether the mismatch affects other skills.

## Best Possible Solution

Maintainers should choose one license model and implement it consistently across publish consent, storage/schema, public APIs, listing UI, CLI output, docs, migrations/backfills, and tests.

## Solution Assessment
