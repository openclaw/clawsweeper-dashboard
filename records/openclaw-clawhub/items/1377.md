## Best Possible Solution

Maintainers should choose one license model and implement it consistently across publish consent, storage, API schemas, UI, CLI output, docs, and any migration or backfill behavior.

## Reproduction Assessment

Yes. Source inspection shows current main still hard-codes MIT-0 across schema, publish storage, docs, and listing UI, and the live API for the reported slug still returns MIT-0.

## Solution Assessment

Unclear as a code-only fix. Displaying MIT alone is not the safest solution unless maintainers first decide whether ClawHub should honor upstream package licenses or continue enforcing a platform MIT-0 contract.
