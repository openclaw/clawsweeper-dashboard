## Summary

This PR addresses a plausible OAuth failure, but it is not merge-ready: one production GitHub sign-in path bypasses the required redirect marker, the branch conflicts with newer auth handling on main, and no after-fix runtime evidence establishes safe callback timing or CLI compatibility.

## Best Possible Solution
