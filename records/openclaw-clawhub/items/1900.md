## What This Changes

The issue requests restoration or purging of residual data after a deleted ClawHub account can no longer sign in with the same GitHub identity.

## Best Possible Solution

Maintainers should verify GitHub ownership and production account state, choose an audited recovery/unlink/recreate policy for user-initiated deletions, keep banned or abuse-disabled accounts blocked, and then update auth code, operator tooling, docs/specs, and tests together.

## Solution Assessment
