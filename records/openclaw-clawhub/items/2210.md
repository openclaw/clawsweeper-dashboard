## Summary

Keep open: current main still uses the shared Convex proof backend, and this PR is not merge-ready because the proposed full-stack runner has concrete env-handling defects, missing real behavior proof, and a failing local-auth check.

## What This Changes

The PR changes `proof:ui` to run lane-local Convex by default, adds `--dev-auth`, repeated `--env`, and `--seed-command`, expands runner tests, and documents the invariant.

## Best Possible Solution

Land a revised full-stack proof runner only after reserved backend env keys are protected, env files or secrets are kept out of artifacts, checks pass, and redacted runtime proof shows the lane-local Convex path working.

## Reproduction Assessment

Not applicable: this is a feature PR for proof tooling rather than a bug report. I source-checked current main, the PR head, live checks, and PR comments instead of reproducing a failing current-main path.

## Solution Assessment

No: the direction may be useful, but the current implementation is not the best mergeable solution while reserved env keys can bypass lane-local backends and generated env values can be copied into artifacts.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Keep generated env files out of proof artifacts:** `scripts/ui-proof.mjs:304-305`
  - body: `lane_env_file` is written under `$remote_out`, and the artifact copy later rsyncs that directory wholesale. With the new `--env KEY=VALUE` flag, tokens or private endpoints can be retained in `.artifacts/clawhub-ui-proof/.../.env.local`; write this file outside the copied artifact tree or scrub it before artifact collection.
  - confidence: 0.9
- **[P2] Reserve the lane-local Convex environment keys:** `scripts/ui-proof.mjs:269`
  - body: Because `opts.env` is spread after the lane-local Convex URL/deployment fields, a caller can pass `--env VITE_CONVEX_URL=...` or `--env CONVEX_DEPLOYMENT=...` and silently route proof to a shared or production-like backend. Reject those reserved keys or ensure the lane-local values always win.
  - confidence: 0.87
