## Summary

The PR changes `proof:ui` to start lane-local Convex per proof lane, adds explicit `--dev-auth`, `--env`, and `--seed-command` options, and updates tests, specs, and the ClawHub UI proof skill for the full-stack invariant.

## Best Possible Solution

Land a revised full-stack proof runner after backend-control env keys are reserved, generated env files are excluded or scrubbed from artifacts, local-auth CI is green, redacted runtime proof is attached, and maintainers accept the shared-backend removal.
