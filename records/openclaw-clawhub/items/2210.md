Full review comments:

- **[P2] Keep generated env files out of proof artifacts:** `scripts/ui-proof.mjs:304-305`
  - body: `lane_env_file` is created inside `$remote_out`, and `copyRemoteArtifacts` later rsyncs that directory wholesale, so arbitrary `--env` values or local deployment data can be retained in `.artifacts/clawhub-ui-proof/.../.env.local`. Write this temp file outside the copied artifact tree or scrub it before artifact collection.
  - confidence: 0.9
- **[P2] Reserve the lane-local Convex environment keys:** `scripts/ui-proof.mjs:269`
  - body: Because `opts.env` is spread after the lane-local Convex URL and deployment fields, a caller can pass `--env VITE_CONVEX_URL=...` or `--env CONVEX_DEPLOYMENT=...` and silently route proof away from the local backend. Reject those reserved keys or ensure the lane-local values always win.
  - confidence: 0.88
