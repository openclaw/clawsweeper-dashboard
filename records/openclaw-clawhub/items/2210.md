Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Reserve backend env keys for lane-local proof:** `scripts/ui-proof.mjs:269`
  - body: `opts.env` is spread after the lane-local Convex settings, so `--env VITE_CONVEX_URL=...` or `--env CONVEX_DEPLOYMENT=...` can make a run advertised as full-stack build or execute against a shared backend.   - confidence: 0.89
- **[P2] Keep generated env files out of proof artifacts:** `scripts/ui-proof.mjs:304-305`
  - body: This writes `.env.local` under `$remote_out`, and the lane later copies that whole directory back as proof output. Since `--env` can carry secrets or private endpoints, put the generated env file outside the copied artifact tree or delete it before artifact copy.
