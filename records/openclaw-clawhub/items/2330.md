Keep open: current main still supports single-skill install plus skill/package browsing or download, but not publisher-scoped batch installation across skills and plugins; this unresolved feature needs a maintainer product/API decision.

## What This Changes

Request publisher-scoped install support so users can install all skills and plugins from a given publisher in one command, similar to `npx skills add mattpocock/skills`.

## Best Possible Solution

Define a first-class publisher-scoped catalog and install contract, then implement CLI/OpenClaw support that preserves per-item safety checks, clear prompts, lockfile metadata, and partial-failure reporting.

Not applicable. This is a feature request, and source inspection confirms current main supports single-target skill install plus catalog browsing/download rather than publisher-wide batch install.

## Solution Assessment
