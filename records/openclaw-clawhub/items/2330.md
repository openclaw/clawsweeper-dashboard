## What This Changes

Request support for installing every skill and plugin published by a given publisher, similar to `npx skills add owner/skills`.

## Best Possible Solution

Define a first-class publisher-scoped catalog and install contract, then implement CLI/OpenClaw support that preserves per-item moderation, confirmation, lockfile metadata, and partial-failure reporting.
