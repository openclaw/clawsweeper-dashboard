## What This Changes

The issue requests a CLI or installer flow to install all skills/plugins published by a given publisher, similar to `npx skills add mattpocock/skills`.

## Best Possible Solution

Define a first-class publisher-scoped install contract, then add indexed catalog support and a CLI flow that preserves per-item safety checks, user confirmation, lockfile metadata, and clear partial-failure reporting.
