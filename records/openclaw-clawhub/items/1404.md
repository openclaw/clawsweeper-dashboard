## Best Possible Solution

A maintainer should answer or close manually with current guidance: the download arrow means downloads, the package/box means versions/packages, and literal `yescan` recall is now handled by indexed prefix search.

Yes. The current implementation is the narrow maintainable solution: keep icon semantics distinct in the UI and use indexed digest prefix candidates for literal slug/name recall.
