Keep open for maintainer follow-up under the ClawHub issue cleanup rule. Current main and the live search endpoint now cover the two observable support concerns: download/version icons are semantically split, and `yescan` returns the reporter's skill through the search API.

## What This Changes

## Best Possible Solution
