## Summary

Keep open. This is a ClawHub issue, which the cleanup policy says not to auto-close, and current main still lacks an end-to-end owner-facing package lifecycle: package-wide DELETE exists, but restore/hide/unhide, package CLI commands, owner UI, docs, and version-level delisting remain unresolved.

## What This Changes

Add owner-facing hide/delete/undelete controls for ClawHub plugin packages across API, CLI, UI, and documentation.

## Best Possible Solution

## Solution Assessment
