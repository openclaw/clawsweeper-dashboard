Keep open. ClawHub policy keeps issues open, and current main still lacks a deterministic CLI/API path for exact-slug or slug-prefix skill enumeration; recent search-prefix recall improves ranked search but does not provide cursor-paginated registry discovery.

Add a narrow registry discovery surface that preserves ranked search while exposing exact slug lookup and slug-prefix enumeration through indexed, cursor-paginated backend queries, public v1 API support, CLI JSON/human output, and focused docs/tests.
