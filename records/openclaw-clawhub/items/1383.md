## Summary

## What This Changes

The issue requests a registry-oriented ClawHub CLI discovery path for exact slug lookup and prefix/list-style skill enumeration.

## Best Possible Solution

Preserve relevance-ranked search and add one backward-compatible registry discovery contract with exact slug handling, indexed slug-prefix filtering, cursor pagination, JSON output, docs, and focused CLI/API tests.

## Solution Assessment

Partly: the requested direction is useful, but the issue lists alternatives rather than one stable contract. The best maintainable path is a maintainer-chosen CLI/API shape that reuses exact detail lookup where appropriate and adds indexed prefix pagination without changing ranked search semantics.
