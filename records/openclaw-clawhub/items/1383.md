Keep open. Repository policy keeps ClawHub issues open, and current main still lacks deterministic exact-slug or slug-prefix skill enumeration; the recent prefix-recall work improves ranked search but does not provide a complete list-style discovery path.

## What This Changes

The issue requests registry-oriented CLI skill discovery for exact slug, slug prefix, or hub-list enumeration instead of relying only on relevance search.

## Best Possible Solution

Add a backward-compatible CLI/API discovery surface for exact slug and slug-prefix enumeration using indexed, cursor-paginated digest queries, while preserving existing relevance search.

## Solution Assessment
