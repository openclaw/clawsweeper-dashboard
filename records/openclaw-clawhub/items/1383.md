Request a CLI/API registry discovery path for exact or prefix-based skill enumeration instead of relying only on relevance-ranked search.

Add a documented cursor-pageable registry discovery API for exact slug and slug-prefix filters, then expose it through an explicit CLI mode while preserving relevance-ranked search behavior.

## Solution Assessment
