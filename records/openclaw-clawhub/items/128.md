## Summary

## What This Changes

The issue requests a `--dry-run` option for `clawhub update` so users can preview available skill updates without changing installed files or lock state.

## Best Possible Solution

Add a no-write planning path inside `cmdUpdate` that reuses existing registry, moderation, fingerprint, semver, pin, and version checks, documents the flag, and tests both preview output and absence of writes.

## Reproduction Assessment

## Solution Assessment
