## Summary

Keep open: current main still lacks a clear latest-version ZIP download action on ordinary skill detail pages, while the backend endpoint and per-version ZIP links remain available.

## What This Changes

Restore the removed download button so users can manually download the latest skill ZIP from a skill detail page.

## Best Possible Solution

Restore a clear latest-version `Download zip` action for ordinary non-Nix skill detail pages, backed by the existing latest-download endpoint and leaving per-version ZIP links in the Versions tab.

## Reproduction Assessment

Yes. Source inspection on current main shows ordinary skill pages expose CLI/prompt install controls and per-version ZIP links, but no clear latest-version ZIP action in the primary detail-page surface.

## Solution Assessment

Yes. A focused UI restoration using `/api/v1/download?slug=<slug>` is the narrowest maintainable fix because the backend endpoint and docs already support latest-version downloads.
