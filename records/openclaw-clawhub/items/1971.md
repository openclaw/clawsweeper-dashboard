## Summary

Keep open: current main still supports ZIP downloads, but the above-the-fold skill detail UI only exposes install/copy flows, leaving the reported manual download workflow hard to discover.

## What This Changes

The issue requests restoring a direct download affordance for skill files so users can manually copy skills or start from the latest published version.

## Best Possible Solution

Add a visible latest `Download ZIP` affordance for ordinary skills using the existing download endpoint, while keeping Nix/plugin-bundle pages on their bundle-specific install path.

## Reproduction Assessment

Yes. A source-level check shows the primary skill hero renders only CLI/Prompt install controls, while the only ZIP link is under the Versions tab and the backend endpoint still exists.

## Solution Assessment

Yes. The narrowest maintainable fix is a UI restoration on the skill detail page, not backend work, because `/api/v1/download` already supports latest-version ZIP downloads.
