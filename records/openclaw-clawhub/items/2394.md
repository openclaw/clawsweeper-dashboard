## What This Changes

Request to make API-key-required classifications auditable by exposing why a skill was flagged.

## Best Possible Solution

Define structured, sanitized reason codes with bounded evidence for the credential classifier, then persist and expose them on the appropriate publisher or staff-facing surfaces.

## Reproduction Assessment

## Solution Assessment

Yes in intent, but not as raw model text. The safer maintainable path is structured reason codes plus sanitized bounded evidence with an explicit disclosure policy.
