## Summary

Keep this feature request open for maintainer product and security design: current main provides skill, code-plugin, and bundle-plugin catalogs, but it does not provide a shareable cron-template artifact or a safe installation and activation contract.

## What This Changes

Add a browsable catalog of reusable cron job templates with configuration previews and installation into an OpenClaw instance.

## Best Possible Solution

Define a versioned, inspectable automation artifact that can reuse ClawHub discovery and moderation while requiring local parameter review and explicit confirmation before creating or enabling any scheduled job.

## Reproduction Assessment

Not applicable: this issue requests a new catalog and installation capability rather than reporting broken existing behavior.

## Solution Assessment
