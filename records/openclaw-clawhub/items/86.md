## Summary

## What This Changes

Add a ClawHub section for publishing, browsing, previewing, categorizing, and installing reusable OpenClaw cron-job templates.

## Best Possible Solution

Define a constrained, versioned automation-template artifact with human-readable preview, immutable installed snapshots, explicit schedule/recipient/tool/secret binding, moderation and trust metadata, and deliberate operator confirmation before writing an OpenClaw job.
