## Summary

Current main does not provide shareable scheduled-job templates or a compatible declarative artifact type. The proposal fits ClawHub’s ecosystem direction, but recurring agent execution needs a maintainer-approved product and security contract before implementation.

## What This Changes

Add a ClawHub catalog for browsing, previewing, sharing, and installing reusable cron job templates.

## Best Possible Solution

Define a declarative automation-template artifact containing the schedule, prompt or action, required capabilities, variables, compatibility, and provenance, then provide preview-first, explicit import into OpenClaw rather than silently activating recurring jobs.

## Reproduction Assessment

Not applicable; this issue requests a new catalog and import capability rather than reporting broken current behavior.

## Solution Assessment

No. A skills-like browsing experience is reasonable, but one-click installation is incomplete until templates have a declarative schema, capability disclosure, security review, and explicit operator confirmation.
