- `impact:security`: The requested feature would distribute recurring executable agent configurations that may retain access to sensitive tools, credentials, channels, and external services.

## Summary

Current main does not implement shareable cron-job templates: ClawHub supports skills and plugin package families, but no automation artifact, cron-template catalog, or installer exists. The idea remains useful and vision-aligned, but it needs explicit maintainer product and security decisions because installing a community-authored recurring job creates durable execution, permission, moderation, and update boundaries.

## What This Changes

Add a browsable, shareable catalog of OpenClaw cron-job templates with preview and one-click installation.

## Best Possible Solution

Design a versioned automation-template artifact with an immutable install snapshot, human-readable effective schedule/prompt/tool/destination preview, explicit operator confirmation, publisher and moderation policy, and opt-in updates before building the catalog and installation UI.

## Reproduction Assessment

Not applicable; this issue requests a new catalog and installation capability rather than reporting broken existing behavior.

## Solution Assessment

No, not as currently specified; a skill-like one-click installer is too underspecified for recurring execution, and the artifact, consent, moderation, revocation, and update contracts should be decided first.
