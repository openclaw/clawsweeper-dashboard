## What This Changes

The issue requests that `clawhub sync` handle a slug conflict without preventing updates for the remaining skills.

Add focused CLI sync error handling that records known slug-taken publish failures per skill, continues the batch, and reports an accurate partial-success result while preserving backend slug uniqueness.
