## What This Changes

Add discoverable owner-facing web controls to soft-delete and restore managed skills while preserving existing moderation and security restrictions.

## Best Possible Solution

Keep the shipped delete action and add an owner-dashboard deleted-skills view showing reservation expiry, with restoration delegated to the existing provenance-aware server lifecycle and validated through real Convex and local-auth browser coverage.
