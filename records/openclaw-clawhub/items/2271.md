## Best Possible Solution

Use the existing skill report/moderation workflow for human security triage, then require the publisher to fail closed in non-interactive agent contexts or hide the listing until that risk is resolved.

## Reproduction Assessment

## Solution Assessment

No auto-close or automated fix is appropriate. The narrowest maintainable path is human security/moderation triage using the existing report/hide surfaces, with a separate maintainer decision on scanner coverage if needed.
