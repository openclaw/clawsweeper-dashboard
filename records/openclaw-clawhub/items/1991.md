## Best Possible Solution

Add a cursor-based admin repair that parses stored ClawPack tarballs, rebuilds missing release file metadata, resets or requeues affected package ClawScan/LLM results, and updates specs/docs so inspect, raw-file, and scanner surfaces agree with artifact truth.

## Solution Assessment
