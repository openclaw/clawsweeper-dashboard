## Summary

Keep open: current main still has the duplicated local helpers, so this PR is not obsolete or implemented on main. The proposed deduplication is useful, but the branch still has a concrete behavior regression already raised in review: `skills.ts` and `souls.ts` currently round finite public pagination limits, while the new shared helper truncates them.

## What This Changes

The PR adds a shared Convex `clampInt` helper in `convex/lib/math.ts`, adds tests for it, and replaces local helper definitions across 11 Convex modules with imports from the shared helper.

## Best Possible Solution

Keep the helper deduplication direction, but land it only after either preserving existing rounded semantics for public skills/souls pagination or explicitly standardizing truncation with focused tests that cover public pagination and internal batch-size behavior.
