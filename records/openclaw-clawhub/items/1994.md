## Summary

Keep open. ## What This Changes

The issue asks how a community skill and code plugin can become Verified or otherwise visibly trusted in ClawHub.

## Best Possible Solution

Use the broader publisher-verification discussion to define one public community policy covering eligibility, review ownership, badge wording, revocation, and the distinction between artifact scans and publisher endorsement.
