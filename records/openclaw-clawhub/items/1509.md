Fix `clawhub publish` so a valid root `SKILL.md` or `SKILLS.md` is uploaded for new skill publishes instead of failing with `SKILL.md required`.

## Best Possible Solution

Review, land, or supersede #1856 with a CLI fix that preserves only the required root manifest during publish while keeping normal ignore behavior for other files.

## Reproduction Assessment

## Solution Assessment
