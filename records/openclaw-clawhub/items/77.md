The issue asks ClawHub to keep skill titles/display names short by adding a length or concision policy.

## Best Possible Solution

Define the public skill display-name length policy first, then enforce it consistently in shared backend publish validation, web publish/import UI, GitHub import, CLI/API schemas, docs, and tests.
