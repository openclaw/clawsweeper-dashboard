## What This Changes

The issue requests a maximum length or concision policy for public skill titles/display names.

## Best Possible Solution

Maintainers should choose the public display-name maximum and legacy behavior, then enforce one shared backend rule with matching UI, CLI/API schemas, docs, and tests.

## Reproduction Assessment

## Solution Assessment
