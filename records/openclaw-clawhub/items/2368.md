## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.87

Full review comments:

- **[P1] Strip credentials from trust-card source URLs:** `convex/githubImport.ts:247`
  - body: This passes the GitHub import `source` into the public trust card, but `source.url` is `resolved.originalUrl`. The current URL parser only checks protocol and hostname, so a pasted URL like `https://token@github.com/owner/repo?x=...` is accepted and would later be exposed by `/api/v1/skills/:slug/trust-card`. Build the trust-card URL from the resolved owner/repo/ref/path, or strip userinfo/search/hash before storing it.
  - confidence: 0.87
