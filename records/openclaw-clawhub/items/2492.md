## What This Changes

The PR makes legacy hosted artifact LLM evaluation advisory-only, redirects legacy manual and backfill scans to Codex-backed security jobs, propagates preserve-mode through worker completion, updates local diagnostic output, and records the moderation invariant.

## Best Possible Solution

Keep bounded hosted evaluation available only as clearly labeled diagnostics, route authoritative artifact verdicts through the materialized-workspace Codex worker, and merge only after a refreshed current-main review plus real Convex evidence for normal and preserve-mode completion.

## Reproduction Assessment

Yes, from current-main source inspection: the bounded hosted evaluator can write stored analysis that feeds moderation, while the branch removes that authority. The after-fix queue and completion behavior has not been reproduced in a real Convex deployment.

## Solution Assessment

Yes in architectural direction: making bounded evaluation advisory and retaining the full-workspace Codex worker as the sole authority is the narrowest maintainable trust boundary. Merge readiness still requires runtime proof and explicit maintainer acceptance of the legacy-call compatibility change.
