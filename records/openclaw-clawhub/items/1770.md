## What This Changes

The issue asks maintainers to purge the soft-deleted `creator-scraper-cv` skill slug so an org-owned skill can be renamed to it.

## Best Possible Solution

A maintainer should verify account and org control, inspect production slug, alias, and reservation state, then use the least destructive audited admin path to release or reclaim the slug and confirm the org rename succeeds.
