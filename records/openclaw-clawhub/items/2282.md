## What This Changes

This PR adds moderation guards for direct and accepted skill ownership transfers, with regression tests, a changelog entry, and an ownership-transfer spec note.

## Best Possible Solution

Land this PR or an equivalent guard after adding `blocked.malware` flag coverage to both transfer guards and posting redacted live proof for rejected direct and accepted transfers.

No, not yet. The guard placement is the right narrow approach, but it should include `blocked.malware` in both helper copies and still needs real setup proof before merge.
