## What This Changes

Report that the published `hogar23/home-assistant-control` skill's generic `scripts/ha_call.sh` can POST directly to Home Assistant service endpoints without the confirmations implemented in `scripts/ha_safe_action.sh`.

## Best Possible Solution

Security triage should decide whether to hide or quarantine the current published skill, ask the owner to route high-impact service calls through one guarded path, and separately consider a ClawScan static rule for raw API escape hatches.
