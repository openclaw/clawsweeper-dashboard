## Summary

Keep this draft PR open for normal review; it adds useful OG image work that is not on current main, but the branch introduces a blocking server-side avatar fetch risk that should be fixed before merge.

## What This Changes

The branch refreshes dynamic OG rendering for skills, plugins, and publisher profiles with shared SVG renderers, new PNG routes, metadata helpers, a local watermark asset, and tests.

## Best Possible Solution

Ship the OG refresh after hardening remote avatar loading, preserving the visual renderer/tests while preventing public OG endpoints from fetching arbitrary server-side URLs.

## Reproduction Assessment

Yes for the blocking finding from source inspection: the new OG routes pass avatar query or metadata URLs to fetchImageDataUrl, which fetches arbitrary http(s) targets server-side. I did not execute a live SSRF probe.

## Solution Assessment

No, the PR is not mergeable as-is; the maintainable path is to keep the OG renderer work but remove public arbitrary avatar fetches or restrict them with trusted-host/non-private-target and redirect checks plus regression tests.

## Repair Work Prompt

On the PR 2180 branch, fix the SSRF risk introduced by dynamic OG avatar loading. Public /og/skill, /og/plugin, and /og/profile currently pass query-supplied or metadata image URLs into server/og/fetchImageDataUrl.ts, which accepts arbitrary http(s) URLs. Either remove public remote avatar URL support or add robust target validation: reject loopback, link-local, private RFC1918/ULA, metadata-service, localhost, and other non-public targets; handle redirects so they cannot land on disallowed hosts; prefer https and trusted image hosts if that fits existing product behavior; and avoid reading unbounded bodies before enforcing size limits where practical. Add focused regression tests for blocked localhost, 127.0.0.1, ::1, 169.254.169.254, private IPv4 ranges, and redirect-to-private behavior, plus an allowed public image case. Do not change the intended OG visual design except for safe fallback avatar behavior. Update tests only for the touched behavior.
