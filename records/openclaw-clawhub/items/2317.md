## What This Changes

The issue requests moving most root-level skill commands under a `clawhub skills` namespace now that ClawHub also has plugin/package commands.

## Best Possible Solution

Adopt a maintainer-approved skill namespace, preserve root commands as compatibility aliases during migration, and update CLI help, docs, and command coverage with an explicit deprecation plan.

## Reproduction Assessment

Not applicable as a bug reproduction. Source and docs inspection confirm current main still has root-level skill commands and no plural `clawhub skills` namespace.

## Solution Assessment
