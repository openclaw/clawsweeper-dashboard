## What This Changes

Request a usable follow-up path when static-analyzer dispute issues are auto-closed after rescan guidance.

## Best Possible Solution

Update the rescan-guidance/analyzer appeal flow so non-collaborators can continue a static-analysis dispute after rescanning, either by keeping appeals open, routing them to a writable queue, or adding an approved suppression path.
