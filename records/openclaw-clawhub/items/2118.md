Keep open because this is a fresh product/UX RFC; current main has not implemented the proposed plugin categories or sort semantics, and the taxonomy/API questions need maintainer direction.

## What This Changes

Request to add plugin browse categories and clearer sort controls while preserving verified and executes-code filters.

Keep the RFC open, choose taxonomy and sort semantics, then implement plugin categories and sort controls through BrowseSidebar and package catalog APIs/indexes where accurate.
