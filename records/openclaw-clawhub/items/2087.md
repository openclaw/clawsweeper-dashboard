## Summary

Keep open: current main still exposes the public surface as `/users`, and this PR is a user-facing product-language change that is not already implemented on main. ## Best Possible Solution

Not applicable for the rename itself; this is a user-facing naming and route compatibility PR rather than a broken-behavior report. The reserved-slug mismatch is source-reproducible from the PR diff plus the validator’s documented route-sync rule.

## Solution Assessment
