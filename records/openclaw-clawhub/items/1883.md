Current main partially answers the discussion through dependency declarations, explicit lockfile pinning, and popularity-based discovery, but it does not implement skill-to-skill conflict metadata or task-context recommendations. Those remaining ideas are meaningful product choices, so this issue is still necessary but should be narrowed before implementation.

## What This Changes

Explore ways to reduce agent confusion from large skill collections through task-context recommendations, skill conflict or dependency metadata, loading guidance, and version pinning.

## Best Possible Solution
