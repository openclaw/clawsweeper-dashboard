Overall correctness: patch is incorrect

Overall confidence: 0.87

Full review comments:

- **[P2] Clear stale auto changelogs on generation failure:** `src/routes/publish-plugin.tsx:275-278`
  - body: If a previous auto-generated changelog exists, selecting another package and then hitting a failed preview request leaves the old text in the textarea while only changing the status to error. Submitting after that can publish release notes for the wrong package, so clear auto-sourced text or mark it unusable when the active request fails.
  - confidence: 0.88
- **[P2] Distinguish prior releases without README files:** `convex/lib/changelog.ts:268`
  - body: `oldReadmeText` is `null` both when there is no previous package release and when the previous release has no README. Since `generateFallback` treats `null` as an initial release, updates to an existing package that previously shipped without a README get misleading initial-release notes; track previous-release existence separately.
  - confidence: 0.84
