## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.87

Full review comments:

- **[P2] Clear stale auto changelogs on preview failure:** `src/routes/publish-plugin.tsx:275-278`
  - body: When a previous auto-generated changelog exists, selecting another package and then hitting a failed preview request only sets the status to error while leaving the old text in the textarea. A publish can then submit release notes for the wrong package, so clear auto-sourced text or mark it unusable in this failure path.
  - confidence: 0.88
- **[P2] Track previous releases separately from README text:** `convex/lib/changelog.ts:268`
  - body: `oldReadmeText` is `null` both when there is no previous package release and when the previous release has no README. Since the fallback treats null as an initial release, updates to an existing package without a prior README get misleading initial-release notes; track prior-release existence separately.
  - confidence: 0.84
