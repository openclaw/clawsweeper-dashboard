Overall correctness: patch is incorrect

Overall confidence: 0.87

Full review comments:

- **[P2] Clear stale auto changelogs on preview failure:** `src/routes/publish-plugin.tsx:275-278`
  - body: After a successful auto preview, selecting different files and hitting a failed preview request leaves the previous auto text in the textarea while only setting `changelogStatus` to `"error"`. A submit can publish release notes for the wrong package, so clear auto-sourced text or block it when the active request fails.
  - confidence: 0.88
- **[P2] Track previous package releases without READMEs:** `convex/lib/changelog.ts:268`
  - body: `oldReadmeText` is `null` both when no prior package release exists and when the prior release has no README, and `generateFallback` treats that as an initial release. Existing packages that previously shipped without a README will get misleading `Initial release` notes; track previous-release existence separately from README availability.
  - confidence: 0.84
