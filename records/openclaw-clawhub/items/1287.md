  - body: If a previous auto-generated changelog exists, selecting another package and then hitting a failed preview request leaves the old text in the textarea while only changing the status to error.   - confidence: 0.88

Since `generateFallback` treats `null` as an initial release, updates to an existing package that previously shipped without a README get misleading initial-release notes; track prior-release existence separately from readable README text.
  - confidence: 0.84
