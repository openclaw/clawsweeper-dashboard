## Summary

Current main does not provide Anthropic Sandbox Runtime or an equivalent local process-policy provider. The delegated-run architecture can accommodate it, but built-in inclusion, supported platforms, and one-shot lifecycle semantics require maintainer product judgment before implementation.

## What This Changes

Add Anthropic Sandbox Runtime as a local delegated-run provider with configuration, CLI flags, doctor support, tests, and documentation.

## Best Possible Solution

Define a v1 macOS/Linux one-shot local delegated-provider contract, then implement it entirely behind provider hooks with explicit unsupported lifecycle operations, secret-safe environment forwarding, non-mutating doctor checks, focused tests, and provider documentation.

## Solution Assessment

Unclear pending maintainer scope decisions; a delegated-provider adapter is the narrowest maintainable architecture, but core inclusion, supported platforms, and lifecycle/session behavior must be chosen before coding.
