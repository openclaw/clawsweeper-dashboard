## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.92

Full review comments:

- **[P1] Send the bridge's argv exec request:** `internal/providers/cloudflaresandbox/client.go:66-70`
  - body: The official bridge requires `argv` plus optional `timeout_ms` and `cwd`, but this client sends `command`, `workingDir`, `env`, and `timeoutSecs`. Real exec calls will not match the documented bridge contract even though the fake bridge accepts them.
  - confidence: 0.94
- **[P1] Parse the bridge's SSE payloads:** `internal/providers/cloudflaresandbox/client.go:233-235`
  - body: The upstream bridge sends stdout/stderr SSE data as raw base64 strings and exit as `{"exit_code": N}`. This parser JSON-decodes every event and only accepts `exitCode`, so live streamed output will fail or miss the exit status.
  - confidence: 0.93
- **[P1] Use the documented file upload route:** `internal/providers/cloudflaresandbox/client.go:341-343`
  - body: The bridge exposes raw writes as `PUT /v1/sandbox/{id}/file/{path}` and archive hydrate as `POST /v1/sandbox/{id}/hydrate`; there is no documented `POST /files/write?path=...&encoding=raw` endpoint. Archive sync will fail before the command runs.
  - confidence: 0.92
- **[P1] Do not require metadata the bridge never returns:** `internal/providers/cloudflaresandbox/backend.go:596-598`
  - body: The official `POST /v1/sandbox` response contains only an `id`, but this code requires returned ownership metadata and falls back to an unsupported get route before accepting the sandbox. Warmup and one-shot runs will fail immediately against the official bridge.
  - confidence: 0.91
- **[P1] Avoid undocumented sandbox inventory endpoints:** `internal/providers/cloudflaresandbox/client.go:153-164`
  - body: The official bridge has create, delete, exec, file, hydrate, persist, and per-sandbox running routes, but not `GET /v1/sandbox`, `GET /v1/sandbox/{id}`, or `GET /v1/sandbox/running`. List, status, and cleanup need to use supported routes plus local claims or land an explicit bridge extension.
  - confidence: 0.88
