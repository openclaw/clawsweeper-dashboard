Full review comments:

- **[P1] Keep runner tokens out of command arguments:** `internal/cli/actions.go:2447`
  - body: This Windows branch writes RUNNER_TOKEN into the encoded PowerShell command passed over SSH, making the registration token visible to command-line inspection even though the repo policy says not to pass secrets as command-line arguments. Please move the token through stdin, a restricted temp file, or another non-argv handoff before starting the installer.
  - confidence: 0.84
- **[P2] Normalize imported MSYS path values:** `internal/cli/sync_windows_target.go:239-244`
  - body: Import-CrabboxEnvFile converts the env-file path itself from /c/... to C:\..., but it leaves path-valued variables such as PNPM_HOME or RUNNER_TOOL_CACHE in MSYS form. Add-CrabboxPath then uses Test-Path on those raw values, so a bash-written hydrate env file can still leave node or pnpm off PATH on native Windows.
  - confidence: 0.77
