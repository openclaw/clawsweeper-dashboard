- **[P1] Preserve claim state when Tart release fails:** `internal/providers/tart/backend.go:272-276`

  - confidence: 0.91

  - body: This IP guard only checks `inst.Running`, but the branch already handles Tart JSON where `Running` is omitted and `State` is `running`. In that case an empty or `--` IP returns a LeaseTarget with no SSH target instead of failing non-status resolves, so SSH-target commands can proceed with unusable connection data; use the normalized running-state check here too.
  - confidence: 0.86
