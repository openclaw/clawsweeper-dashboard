- **[P2] Prune stale Tart claims before deleting missing VMs:** `internal/providers/tart/backend.go:279-280`
  - body: When `ReleaseLease` is called with only a lease ID whose claim points at a VM no longer present in `tart list`, `resolveInstance` returns `State: "missing"`; this block still calls `tart delete` and returns that not-found error before `removeLeaseClaim`/`removeStoredTestboxKey`.
