- **[P1] Preserve generic SSH and workRoot config for Tart:** `internal/cli/config.go:777-785`
  - body: `baseConfig` now gives `Tart.User` and `Tart.WorkRoot` non-empty defaults, and this block copies those fields back into the generic SSH/work-root fields. A config that sets only the generic `ssh.user` or top-level `workRoot` is silently reset to `admin` and `/Users/admin/crabbox`; please fall back to the generic values when Tart-specific fields were not explicitly set, while keeping explicit `tart.user`/`tart.workRoot` higher precedence.
  - confidence: 0.86
- **[P2] Honor cancellation before launching kept VMs:** `internal/providers/tart/backend.go:410-434`
  - body: For `keep=true`, this starts `tart run` with `exec.Command` and the following wait only observes process exit or a timer. If the context is already canceled, or cancels during the observation window, Crabbox can still launch a retained local VM; check `ctx` before `Start` and include `ctx.Done()` in the startup wait so canceled warmups cannot leave abandoned Tart instances.
  - confidence: 0.84
