Full review comments:

- **[P1] Honor cancellation before kept Tart starts:** `internal/providers/tart/backend.go:410`
  - body: In the keep path, startVM builds exec.Command and reaches cmd.Start without checking ctx. If the acquire context has already been canceled, a kept warmup can still launch tart run before any lease claim exists; if the Crabbox process exits at that point, cleanup has no recorded lease to reconcile. Please check ctx before starting the kept process and cover the canceled-keep path.
  - confidence: 0.9
- **[P3] Align the Tart disk flag help with validation:** `internal/providers/tart/flags.go:24`
  - body: The flag help says `0 = use clone default`, but applyFlags rejects `--tart-disk 0` as invalid. Users following `--help` will get an error, so either remove the zero guidance or accept explicit zero as the clone-default sentinel.
  - confidence: 0.95
