## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Make SSH key injection honor tart.user:** `internal/providers/tart/backend.go:452`
  - body: The PR exposes `tart.user`/`CRABBOX_TART_USER` and later uses the stored `ssh_user` label to build the SSH target, but key injection always runs `tart exec ... bash -c` and writes `~/.ssh/authorized_keys` for whichever user the guest agent uses. On the default Cirrus images that is `admin`, so setting `CRABBOX_TART_USER=root` makes Crabbox wait for SSH as `root` even though the key was injected for `admin`, causing acquisition to hang or fail after bootstrap looked successful. Either reject unsupported users or inject into the configured user's home before using it as `SSHUser`.
  - confidence: 0.93
- **[P2] Reject non-positive Tart resource sizes:** `internal/providers/tart/flags.go:35-43`
  - body: `applyFlags` copies explicit `--tart-cpu`, `--tart-memory`, and `--tart-disk` values directly, and `applyDefaults` later treats `<=0` as unset and silently rewrites them to defaults. That makes dry-run/config accept typos like `--tart-cpu 0` and proceed with a different VM size; for disk, the explicit marker can also survive and turn an invalid value into `tart set --disk-size 50`. Return a config error for non-positive Tart resource values from CLI/env/YAML before defaults run.
  - confidence: 0.9
- **[P2] Apply Tart defaults before config show renders:** `internal/cli/config_cmd.go:164-171`
  - body: `config show` is documented to include provider defaults, but the new Tart output is rendered from `loadConfig()` while `applyProviderConfigDefaults` has no Tart branch. With `provider=tart` selected through env or YAML, text/JSON output can still show Linux defaults such as `target=linux`, `sshUser=crabbox`, port `2222`, and `/work/crabbox` even though actual Tart leases use macOS/admin/22 and `/Users/admin/crabbox`. Add Tart to the effective-config default path or render the provider-configured view.
  - confidence: 0.88
