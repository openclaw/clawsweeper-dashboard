## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Preserve generic SSH and work-root overrides:** `internal/cli/config.go:777-785`
  - body: The Tart branch copies the provider defaults back over `cfg.SSHUser` and `cfg.WorkRoot`, so a config with top-level `ssh.user` or `workRoot` still resolves to `admin` and `/Users/admin/crabbox`. Track explicit `tart.user`/`tart.workRoot` values and let the generic settings flow through when the Tart-specific values were not set.
  - confidence: 0.9
- **[P1] Preserve kept VMs after post-start failures:** `internal/providers/tart/backend.go:132-135`
  - body: When `--keep` is set, a VM that has already started should remain available for inspection or reuse after IP, SSH injection, readiness, claim, or endpoint failures. These branches unconditionally stop and delete the VM after `startVM` succeeds, defeating the user-visible keep contract.
  - confidence: 0.93
- **[P1] Honor cancellation before launching kept VMs:** `internal/providers/tart/backend.go:409-418`
  - body: The keep path uses plain `exec.Command` and starts `tart run` even if the acquire context is already canceled. Check `ctx.Err()` before spawning and handle `ctx.Done()` during the observation window so abandoned runs do not boot background macOS VMs.
  - confidence: 0.89
- **[P2] Use Tart state when Running is omitted:** `internal/providers/tart/backend.go:185-186`
  - body: `Running` is an optional decoded bool, so older or different `tart list` JSON that omits it makes a `State:"running"` VM look stopped to non-status Resolve callers. Use the normalized state helper at this boundary so existing claimed running VMs remain reusable.
  - confidence: 0.86
- **[P3] Align the tart disk zero contract:** `internal/providers/tart/flags.go:24`
  - body: The flag help advertises `0 = use clone default`, but explicit `--tart-disk 0` is rejected. Either accept zero as the documented clone-default sentinel or change the help/docs so the CLI does not advertise an unusable value.
  - confidence: 0.91
