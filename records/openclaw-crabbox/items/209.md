  - body: `ReleaseLease` discards both `stopVM` and `deleteVM` errors and then removes the lease claim/key.   - confidence: 0.94
- **[P2] Preserve kept Tart VMs after post-start failures:** `internal/providers/tart/backend.go:162-163`
  - body: This branch runs after `tart run` has started the VM, but the cleanup still stops and deletes it unconditionally. With `--keep`, failures from SSH readiness, claim persistence, or endpoint update can destroy the VM despite the kept-lease contract; gate these cleanups on `!req.Keep` or otherwise preserve the started kept VM.
  - confidence: 0.9
