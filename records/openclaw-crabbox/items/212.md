## Risks / Open Questions

- Merging this PR makes Crabbox own a built-in Windows host Hyper-V matrix: Hyper-V availability, PowerShell Direct behavior, guest auto-logon timing, DHCP, OpenSSH install, guest internet access, and MinGit bootstrap can all fail outside CI.
