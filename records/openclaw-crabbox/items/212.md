- Acquisition can fail when Hyper-V, PowerShell Direct, guest DHCP, Windows Features-on-Demand, or guest internet access is unavailable; users may need to repair the host or pre-bake OpenSSH and git.

- With `--hyperv-init-password`, the chosen password is intentionally present inside the lease guest's RunOnce command until first auto-logon executes it; the host-side command keeps it off argv, but the guest exposure remains part of the feature contract.
