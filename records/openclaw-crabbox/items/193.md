## Summary

Keep open: current main and the latest release still expose lease/run/job primitives, not a Station lifecycle, stationProfile, or modelAccess API. The linked roadmap/docs work in https://github.com/openclaw/crabbox/pull/346 is open and intentionally does not close the runtime Station work.

## What This Changes

Request a new Station lifecycle primitive, an agent station profile, and later scoped modelAccess credential delivery for supervised long-running workloads.

## Best Possible Solution

Keep this umbrella open, review the roadmap/security PR on its own merits, then split a generic SSH-backed Station MVP with no model credentials before any agent profile or modelAccess work.
