## Summary

Keep open: current main and the latest release still expose warmup/run/job/status/logs/stop and generic profiles, but there is no Station command/config/evidence surface or scoped Reasoning Access. This is new lifecycle, auth, credential, and evidence roadmap work that needs maintainer product and security decisions before implementation.

## What This Changes

The issue requests a new Station lifecycle primitive, an agent station profile, and explicit scoped Reasoning Access for long-running repo-owned workloads.

## Best Possible Solution

Keep this as a roadmap item: stabilize harness evidence first, implement generic Station without model credentials, then add opt-in Reasoning Access only after the token scope, redaction, revocation, budget, and egress model is approved and tested.
