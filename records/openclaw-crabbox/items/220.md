Full review comments:

- **[P1] Decode snake_case exec exit codes:** `internal/providers/smolvm/client.go:59`
  - body: `encoding/json` only uses the first `json` tag here, so an API response like `{"exit_code":7}` leaves `ExitCode` at zero. That can make failed archive injection or user commands look successful; add an explicit fallback/custom decode path and a regression test for snake_case failure responses.
  - confidence: 0.88
- **[P2] Send SmolVM stderr to stderr:** `internal/providers/smolvm/client.go:255-257`
  - body: `Run` passes only `Runtime.Stdout` into `ExecStream`, and this branch writes `result.Stderr` to that same writer. Commands that write diagnostics to stderr will be emitted on stdout, breaking redirection/capture semantics compared with other delegated providers; thread both stdout and stderr writers through this API.
  - confidence: 0.94
