Full review comments:

- **[P1] Validate SmolVM API origins before adding the bearer token:** `internal/providers/smolvm/client.go:151-152`
  - body: `newAPI` trusts `cfg.Smolvm.BaseURL` directly, and that value can come from repo config, env, or `--smolvm-base-url`; every API request then gets the SmolVM bearer token.
