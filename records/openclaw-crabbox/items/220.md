## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Decode snake_case exec exit codes:** `internal/providers/smolvm/client.go:59`
  - body: `ExitCode` has two `json` keys in the struct tag, but Go's JSON decoder only uses the first `json` tag value. If the API returns `exit_code`, nonzero exec failures stay at zero and `crabbox run` can report success for a failed command.
  - confidence: 0.9
- **[P2] Preserve exec stderr separately:** `internal/providers/smolvm/client.go:255-257`
  - body: `ExecStream` receives only a stdout writer and writes `result.Stderr` to it; `Run` passes `b.rt.Stdout`, so command diagnostics are mixed into stdout. Pass a stderr writer through this path or emit stderr from the backend to `b.rt.Stderr`.
  - confidence: 0.92
