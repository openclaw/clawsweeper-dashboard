- **[P1] Validate SmolVM API URLs before adding bearer credentials:** `internal/providers/smolvm/client.go:151`
  - body: `newAPI` stores `cfg.Smolvm.BaseURL` after trimming only, and later requests add `Authorization: Bearer ...` plus command/archive/env bodies.   - confidence: 0.92
- **[P1] Decode snake_case SmolVM exec exit codes:** `internal/providers/smolvm/client.go:59`
  - body: The duplicate struct tag only gives `encoding/json` the `exitCode` name for `ExitCode`, so a response with `exit_code: 1` leaves the field at zero. `ExecStream` then returns success and `Run` skips failure handling; add an explicit snake_case field or custom unmarshal plus a nonzero `exit_code` regression test.
  - confidence: 0.95
