Full review comments:

- **[P1] Decode snake_case exec exit codes:** `internal/providers/smolvm/client.go:59`
  - body: `encoding/json` will only honor the first `json` tag here, so a SmolVM response that returns `exit_code` leaves `ExitCode` at zero.   - confidence: 0.94
- **[P1] Route remote stderr to stderr:** `internal/providers/smolvm/client.go:255-256`
  - body: `ExecStream` only receives a stdout writer and writes `result.Stderr` into it, so commands that emit diagnostics can corrupt machine-readable stdout and bypass stderr-specific handling. Thread a stderr writer through the API/backend wiring and update the regression to assert stderr separation.
  - confidence: 0.96
