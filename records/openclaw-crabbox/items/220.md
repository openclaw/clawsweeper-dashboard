Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Decode snake_case exec exit codes:** `internal/providers/smolvm/client.go:59`
  - body: `ExitCode` is tagged as `json:"exitCode" json:"exit_code"`, but `encoding/json` only uses one JSON tag name for a field. If the API returns `exit_code`, a failed command decodes as zero and `Run` can report success; add an alias/custom unmarshal path plus a regression test for the snake_case response.
  - confidence: 0.93
- **[P2] Send SmolVM stderr to stderr:** `internal/providers/smolvm/client.go:255-257`
  - body: `ExecStream` appends `result.Stderr` to the stdout writer, so remote diagnostics end up in local stdout instead of stderr. This breaks normal redirection and output consumers; thread a stderr writer through the path or write response stderr to `b.rt.Stderr`.
  - confidence: 0.86
