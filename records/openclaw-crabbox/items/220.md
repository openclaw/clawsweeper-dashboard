## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Decode snake-case exec exit codes:** `internal/providers/smolvm/client.go:59`
  - body: `execResult.ExitCode` has two `json` tags, but Go's decoder only uses the first `json:"exitCode"` mapping for this field. If the SmolVM exec API returns `exit_code`, nonzero delegated commands decode as exit code 0, so `ExecStream` and `Run` can report failed commands as successful; add a separate snake-case field or custom unmarshal plus a regression test for `exit_code: 7`.
  - confidence: 0.91
