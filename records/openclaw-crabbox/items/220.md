  - body: `CRABBOX_SMOLVM_BASE_URL`, config, and `--smolvm-base-url` are trimmed directly into `c.base`, and every request later attaches `Authorization` plus may carry archive or exec payloads.   - confidence: 0.9

  - body: The duplicate struct tag only decodes `exitCode`, not `exit_code`; snake_case exec responses leave `ExitCode` at zero. If SmolVM returns `exit_code` for a failed command, Crabbox can report success, so add an explicit fallback field or custom unmarshal with a failing-command regression test.
  - confidence: 0.86
