Overall correctness: patch is incorrect

Full review comments:

- **[P1] Validate the SmolVM base URL before sending bearer tokens:** `internal/providers/smolvm/client.go:151-152`
  - body: `cfg.Smolvm.BaseURL` can be set by repo config, env, or flag and is stored here without validating the origin; every later request built from that base gets `Authorization: Bearer <smk_...>`.
