## Summary

The PR gates `localContainer.dockerSocket` file config behind trusted config paths, adds focused config tests, and updates local-container docs.

## Best Possible Solution

Land the trust-boundary fix with the included documentation that repo-local socket config is ignored and trusted operator-controlled paths remain supported.
