- **[P1] Escape the WSL probe host and port:** `internal/cli/ssh.go:887`
  - body: `target.Host` and `target.Port` are inserted directly into a string that is executed by `wsl bash -c`. Build the probe without shell interpolation or quote these values and add regression coverage for hostile strings.
