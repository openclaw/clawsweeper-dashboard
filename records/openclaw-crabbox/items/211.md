- **[P1] Reject symlinked WASI guest roots before copying:** `internal/providers/wasi/backend.go:426-427`
  - body: `copyManifestToGuest` calls `os.MkdirAll(target)` before any no-follow validation of the guest workdir itself.   - confidence: 0.91
- **[P2] Gate the WASI live smoke behind provider selection:** `scripts/live-smoke.sh:460`
  - body: Every other smoke in this script is guarded by `has_provider`, but the new `wasi_smoke` is called unconditionally. That means `CRABBOX_LIVE_PROVIDERS=aws`, `kubevirt`, or any other selected subset will also run the local WASI smoke, which breaks the operator's provider-selection contract and can make live automation fail for the wrong provider.
  - confidence: 0.97
