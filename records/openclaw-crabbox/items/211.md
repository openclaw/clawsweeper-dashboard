- **[P1] Check guest symlink parents before creating directories:** `internal/providers/wasi/backend.go:484-487`
  - body: `os.MkdirAll(parent)` runs before `rejectSymlinkComponents`, but `MkdirAll` follows symlinked intermediate directories. Please validate existing components before creating missing ones, or create each component with no-follow checks, and add a nested-parent regression.
