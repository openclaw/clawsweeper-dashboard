## Review Findings

Overall confidence: 0.9

Full review comments:

- **[P1] Validate guest parents before creating them:** `internal/providers/wasi/backend.go:484-487`
  - body: `os.MkdirAll(parent)` runs before `rejectSymlinkComponents`, but `MkdirAll` follows symlinked intermediate directories. In a kept guest workdir, a module can leave `crabbox/dir -> /some/host/path`; syncing a manifest entry like `dir/sub/file.txt` will create `/some/host/path/sub` before the later rejection fires, so the host-side write has already escaped the guest. Validate existing components before creating missing ones, or create each component with no-follow checks, and add a nested-parent regression that asserts no outside side effect.
  - confidence: 0.91
