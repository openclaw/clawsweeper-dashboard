The branch downloads the Chocolatey installer to a temporary PowerShell file, verifies a SHA-256 digest before execution, adds installer URL/digest environment overrides, and adds a Node static regression test.

Land a maintainer-approved version of this Chocolatey verification flow, explicitly accept or document the digest update path, let https://github.com/openclaw/crabbox/issues/721 close on merge, and keep Node/Docker hardening scoped to https://github.com/openclaw/crabbox/issues/686.
