- **[P2] Allow Windows npm-pack paths through validation:** `scripts/behavior-eval.mjs:2149-2150`
  - body: `resolveNpmPackPluginPath()` now absolutizes the override with `path.resolve()`, but the next validation step still calls `isSafeNpmCoordinate()`.
