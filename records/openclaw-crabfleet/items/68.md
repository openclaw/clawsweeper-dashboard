## Summary

## What This Changes

The PR requires owner proof for every GitHub Actions action-session registration or resume, updates the public endpoint contract, and adds focused regression coverage.

## Best Possible Solution

## Solution Assessment

Yes. Requiring a resolved owner before both insert and resume is the narrowest maintainable fix, and the branch updates docs, tests, and real endpoint proof for that contract.
