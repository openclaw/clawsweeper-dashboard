## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Skip button fallback when no file candidate exists:** `src/browser/chatgptFiles.ts:779-781`
  - body: `collectChatGptFileArtifacts` calls the button fallback when `saved.savedFiles.length < Math.max(allFiles.length, 1)`. With no detected file links, `allFiles.length` is 0, so every normal ChatGPT browser run with no downloadable file still polls for download buttons for up to 60 seconds before finishing. Gate the fallback on a detected current-turn file candidate instead.
  - confidence: 0.92
- **[P2] Keep button downloads scoped to the target turn:** `src/browser/chatgptFiles.ts:482-486`
  - body: When the scoped assistant-turn search finds no button, this fallback clicks the last global matching download buttons on the page. In a reused or multi-turn conversation, that can save an older generated file as the current session's artifact. Drop the global fallback or filter it by the same turn/minTurnIndex evidence.
  - confidence: 0.84
