Full review comments:

- **[P1] Resolve symlinked output parents before accepting paths:** `src/mcp/utils.ts:32-38`
  - body: The new MCP boundary approves any lexical path under `ORACLE_HOME_DIR`, but `path.resolve` does not resolve symlinks.
