## Summary

Keep open for maintainer product direction: current main and v0.2.7 already support encrypted backup of the current archive snapshot, but they do not provide durable retention for messages that later disappear from WhatsApp Desktop.

## What This Changes

The issue asks whether wacrawl should become a durable WhatsApp backup alternative beyond mirroring WhatsApp Desktop state.

## Best Possible Solution

Define an explicit opt-in durable-history mode that merges newly observed records into retained storage while documenting deletion policy, media handling, integrity checks, restore semantics, and unchanged default snapshot behavior.

## Reproduction Assessment

Not applicable: this is a feature request for a new durable-retention guarantee, and source/docs inspection already shows current behavior is snapshot replacement rather than a broken existing contract.

## Solution Assessment

Yes in direction, but not as an unconditional sync change; an explicit opt-in retention mode is safer because it changes deletion, privacy, storage, and restore expectations.
