## Summary

Keep open for maintainer product direction: current main and v0.2.7 already support encrypted backup of the current wacrawl archive snapshot, but the archive still mirrors WhatsApp Desktop snapshot replacement and does not provide a durable-retention guarantee for messages that disappear upstream.

## What This Changes

The issue asks whether wacrawl should become a durable alternative to WhatsApp iCloud/Google Drive backups that preserves chat history after WhatsApp Desktop drops it.

## Best Possible Solution

Add an explicit opt-in durable-history mode that merges newly observed records into a retained archive, with documented deletion policy, copied-media handling, integrity verification, snapshot recovery, and unchanged default replacement behavior.

## Reproduction Assessment

Not applicable: this is a feature request for a new durable-retention guarantee, and the source/docs inspection already shows current behavior is snapshot replacement rather than a broken existing contract.

## Solution Assessment

Yes in direction, but not as an unconditional sync change; an explicit opt-in retention mode is safer because it changes deletion, privacy, storage, and restore expectations.
