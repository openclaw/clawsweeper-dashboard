## Summary

Keep open for maintainer product direction: current main and v0.2.7 support encrypted backups of the current wacrawl archive snapshot, but they do not guarantee durable retention for messages that disappear from WhatsApp Desktop.

## What This Changes

The issue requests turning wacrawl into a durable WhatsApp backup alternative rather than only mirroring the current WhatsApp Desktop archive snapshot.

## Best Possible Solution

Add an explicit opt-in durable-history mode that merges newly observed records into a retained archive, with documented deletion policy, copied-media handling, integrity verification, snapshot recovery, and unchanged default replacement behavior.

## Reproduction Assessment

Not applicable: this is a feature request for a new durable-retention guarantee, not a broken existing contract. Source and docs inspection already show current behavior is snapshot replacement.

## Solution Assessment

Yes in direction, but not as an unconditional sync change; an explicit opt-in retention mode is safer because it changes deletion, privacy, storage, and restore expectations.
